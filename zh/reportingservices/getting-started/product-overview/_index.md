## **欢迎使用 Aspose.Slides for Reporting Services！**

Aspose.Slides for Reporting Services 是市场上唯一能够在 Microsoft SQL Server 2005、2008、2012、2016 和 2017 Reporting Services（32 位和 64 位）中生成真实 PPT 和 PPS 报告的解决方案。所有 RDL 报表功能，包括表格、矩阵、图表和图像，都以最高精度转换为 Microsoft PowerPoint 演示文稿。

## **产品概述**

Microsoft SQL Server Reporting Services 本身不具备将报表导出为 Microsoft PowerPoint 演示文稿的功能，但在服务器上安装 Aspose.Slides for Reporting Services 后，您可以使用以下额外的导出格式：

- PPT - 通过 Aspose.Slides 的 PowerPoint 演示文稿
- PPS - 通过 Aspose.Slides 的 PowerPoint 幻灯片放映
- PPTX - 通过 Aspose.Slides 的 PowerPoint 2007 演示文稿
- PPSX - 通过 Aspose.Slides 的 PowerPoint 2007 幻灯片放映

Aspose.Slides for Reporting Services 在服务器上创建演示文稿，无需使用 Microsoft PowerPoint。Aspose.Slides for Reporting Services 在内部使用 Aspose.Slides for .NET —— 业界领先的服务器端演示文稿处理组件。

**Aspose.Slides for Reporting Services 使得可以将任何报表导出为 PPT、PPS、PPTX 或 PPSX 格式。**
