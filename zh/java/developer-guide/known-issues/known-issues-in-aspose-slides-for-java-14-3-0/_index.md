Aspose.Slides for Java 14.3.0 (14.4.0) 提供了全新的 PPT 处理实现。功能改进很多，支持 PPTX 到 PPT 的部分转换。但仍有一些未实现的功能：

- 某些形状在序列化的 PPT 文档中几何形状不正确（标注）
- 并非所有 PPTX 文本格式化功能在 PPT 序列化时都受支持
- 序列化的 PPT 文档中缺少文本语言和拼写设置的信息
- 并非所有 PPTX 主题特性在 PPT 序列化时都受支持

**与 Aspose.Slides for Java 8.6.0 相比，存在一些差异：**

- 已知在 OLE/ActiveX PPT 序列化为 PPT 时存在问题

**与 Aspose.Slides for .NET 14.3.0 相比，存在一些差异：**

- 目前 Aspose.Slides for Java 不支持演示文稿打印
