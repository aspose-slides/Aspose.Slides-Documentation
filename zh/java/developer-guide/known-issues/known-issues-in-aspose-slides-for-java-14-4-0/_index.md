{{% alert color="primary" %}} 

Aspose.Slides for Java 14.4.0 为 PowerPoint 文档处理提供了新的决定。存在一些限制和已知问题，这些将在后续版本中移除：

- 某些形状在序列化的 PPT 文档中几何形状错误（弧形、循环箭头、标注）。
- 并非所有 PPTX 文本格式化功能在 PPT 序列化中受支持（制表符、缩进和段落格式限制）。
- 序列化的 PPT 文档中不包含文本语言和拼写设置的信息。
- 并非所有 PPTX 主题功能在 PPT 序列化中受支持（仅支持填充格式、线条格式和字体的序列化）。
- 在 OLE/ActiveX PPT 序列化为 PPT 时存在已知问题。
- 不支持 WordArt 的序列化和渲染。

{{% /alert %}}
