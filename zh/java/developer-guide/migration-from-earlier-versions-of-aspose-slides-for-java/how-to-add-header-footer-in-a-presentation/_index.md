新发布了[Aspose.Slides for Java API](https://docs.aspose.com/slides/java/)，现在这个单一产品支持从头生成 PowerPoint 文档和编辑现有文档的功能。

{{% /alert %}} 
## **对旧代码的支持**
为了使用使用 Aspose.Slides for Java 13.x 之前版本开发的旧代码，您需要对代码进行一些小的修改，代码将按以前的方式工作。所有在旧版 Aspose.Slides for Java 的 Aspose.Slide 和 Aspose.Slides.Pptx 命名空间下存在的类现在已合并到单一的 Aspose.Slides 命名空间中。请查看以下简单的代码片段，了解如何在旧版 Aspose.Slides API 中添加演示文稿的页眉和页脚，并按照描述如何迁移到新的合并 API 的步骤进行操作。
