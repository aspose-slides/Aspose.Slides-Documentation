## **支持旧版代码**
要使用在 Aspose.Slides for Java 13.x 之前版本中开发的旧版代码，您只需对代码进行少量修改，代码即可像以前一样工作。旧版 Aspose.Slides for Java 中位于 Aspose.Slide 和 Aspose.Slides.Pptx 命名空间的所有类现已合并到单一的 Aspose.Slides 命名空间。请查看下面的简单代码示例，该示例演示如何在旧版 Aspose.Slides API 中创建 Hello World 演示文稿，并按照步骤了解如何迁移到新的合并 API。
