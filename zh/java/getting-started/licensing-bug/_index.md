本文描述了在 Aspose.Slides for Java 版本 23.2、23.3、23.4、23.5、23.6 和 23.7 中遇到的许可错误。该错误导致在订阅到期后，永久许可暂时变得无法使用。

## 症状 ##

在您的永久许可订阅到期后，Aspose.Slides for Java 版本 23.2 - 23.7 可能会报告许可证已过期，所有功能进入评估模式。
这种行为是不正确的，并且不影响 23.2 之前和 23.7 之后的版本。

## 解决方案 ##

Aspose.Slides for Java 已解决此问题，并发布了更新版本（23.2.1、23.3.1、23.4.1、23.5.1、23.6.1、23.7.1）修复此问题。

如果您在项目中使用受影响版本的 Aspose.Slides for Java，请切换到已更正的版本。

如果您在使用 Aspose.Slides for Java 时遇到任何许可问题：

- 检查您正在使用的版本，确保它不受上述错误的影响。
- 参考 [Aspose 文档](https://docs.aspose.com/slides/java/getting-started/) 获取故障排除步骤和已知的许可问题。
- 联系 [Aspose 支持](https://forum.aspose.com/) 以获得进一步的帮助。

## 受影响的产品和版本 ##

请注意，此错误仅影响 Aspose.Slides for Java 版本 23.2 到 23.7。**其他 Aspose 产品和其他版本的 Aspose.Slides 不受影响**。
