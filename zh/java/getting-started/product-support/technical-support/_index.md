## **技术支持**

{{% alert color="primary" %}} 

Aspose 为其所有产品提供无限制的免费技术支持。该支持对所有用户均可使用，包括评估版用户。

{{% /alert %}} 

如果您需要 Aspose.Slides for Java 的帮助，请考虑以下内容：

- 支持的主要渠道是 Aspose.Forums。请在 Aspose.Slides 论坛中发布您的问题，通常会在几小时内得到回答。问题由 Aspose.Slides 支持团队直接答复。
- 请注意，Aspose 不提供电话技术支持。电话支持仅用于销售和购买相关问题。
- 在论坛等待回复时，请考虑时区差异。

如果您在使用 Aspose.Slides for Java 时遇到问题，请遵循以下简单步骤，以确保以最高效的方式解决问题：

- 在报告问题之前，请确保使用最新的 Aspose.Slides for Java 版本，参见 Aspose.Slides 下载以了解最新版本。
- 在报告问题之前，请浏览论坛、本文档以及 API 参考；也许您的问题已经得到解答。
- 报告问题时，请包括原始文档以及可能导致问题的代码片段。如果需要附加多个文件，请将它们压缩为一个 zip。将文档上传至 Aspose.Forums 是安全的，因为只有您和 Aspose 开发人员可以访问这些附件。
- 请尝试每个主题只报告一个问题。如果还有其他问题，请在单独的主题中报告。

## **FAQ**

**提供哪些付费支持等级，它们有何区别？**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services)。它们在包含的事件数量、响应优先级和可用的沟通渠道方面有所不同。

**我可以直接将 bug 升级给工程团队吗？**

可以。[付费支持](https://helpdesk.aspose.com/)允许在问题需要更深入调查时升级给开发人员。

**是否支持旧版本库，并为其发布修复？**

您可以使用旧版本，但修复仅提供给当前发布的版本；不会为旧版发布补丁。
