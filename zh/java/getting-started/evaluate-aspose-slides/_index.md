您可以轻松下载 Aspose.Slides 进行评估。评估下载与购买的下载相同。评估版本只需添加几行代码以应用许可证，即可变为许可证版本。

Aspose.Slides 的评估版本（未指定许可证）提供完整的产品功能，但在打开和保存时会在文档顶部插入评估水印，并在从演示幻灯片中提取文本时限制为一张幻灯片。
