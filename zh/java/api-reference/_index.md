{{% alert color="primary" %}} 

Aspose.Slides for Java 是一个类库，使 Java 应用程序能够读取和写入演示文稿，而无需使用 Microsoft PowerPoint® 或其他软件。Aspose.Slides for Java 允许管理文本、形状、表格和动画，向幻灯片添加音频和视频，预览幻灯片，将幻灯片导出为 SVG、PDF 格式等。

[Aspose.Slides for Java API 参考.](https://reference.aspose.com/slides/java)
