## **技术支持**

{{% alert color="primary" %}} 
Aspose 为其所有产品提供无限制的免费技术支持。该支持面向所有用户（包括使用评估包的用户）。
{{% /alert %}} 

当您需要帮助使用 Aspose.Slides for Android via Java 时，请考虑以下事项：

- 支持的主要渠道是 Aspose.Forums。将在 Aspose.Slides 论坛中发布您的问题——它将在数小时内得到答复。Aspose.Slides 支持团队会直接回答论坛上发布的问题。
- 请注意，Aspose 不提供电话技术支持。电话支持仅适用于销售和购买问题。
- 在论坛期待回复时，请耐心等待，并考虑时区差异。

当您在使用 Aspose.Slides for Android via Java 时遇到问题，请遵循以下指南以最高效的方式解决：

- 在报告问题之前，请检查并确认您使用的是最新的 Aspose.Slides for Android via Java 版本。也许您遇到的问题已在更新中得到解决。

  请参阅 Aspose.Slides 下载以获取最新版本信息。

- 在报告问题之前，您可能想先浏览我们的论坛、本文件和 API 参考。也许您的问题已经得到了解答。

- 报告问题时，请包括原始文档和（如果可能）与问题相关的代码片段。如果需要附加多个文件，请将它们打包为 ZIP。

  您可以在 Aspose.Forums 安全地附加文档，因为只有您和 Aspose 开发人员可以访问这些附件。

- 针对特定问题创建并使用一个主题。如果您遇到其他问题，需要创建另一个主题进行报告。

## **常见问题**

**有哪些付费支持层级以及它们有何区别？**

[Developer、Business 和 Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services)。它们在包含的事件数量、响应优先级以及可用的沟通渠道上有所不同。

**我可以直接将错误上报给工程团队吗？**

是的。[付费支持](https://helpdesk.aspose.com/) 在问题需要更深入调查时允许升级至开发人员。

**是否支持旧版库，且会为其发布修复？**

您可以使用旧版本，但修复仅针对当前发布的版本；不会为旧版提供补丁。
