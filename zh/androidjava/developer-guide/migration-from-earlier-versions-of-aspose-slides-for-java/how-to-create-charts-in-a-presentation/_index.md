## **对旧代码的支持**
为了使用在 14.x.x 之前版本的 Aspose.Slides for Java 开发的旧代码，您需要对代码进行一些小的更改，代码将正常工作。所有在旧版 Aspose.Slides for Java 中属于 com.aspose.slides 和 com.aspose.slides.pptx 命名空间的类现在已合并到单一的 com.aspose.slides 命名空间中。请查看以下简单代码片段，了解如何使用旧版 Aspose.Slides API 从头在演示文稿中创建普通图表，并遵循描述如何迁移到新的合并 API 的步骤。
