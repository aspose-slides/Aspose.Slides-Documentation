## **技术支持**
{{% alert color="primary" %}}

Aspose 为所有产品提供无限的免费技术支持。该支持对所有用户均可使用（包括使用评估版的用户）。

{{% /alert %}} 

当您需要 Aspose.Slides for Python via .NET 的帮助时，请考虑以下内容：

- 获取支持的主要途径是 [Aspose.Forums](https://forum.aspose.com/c/slides/11)。在 Aspose.Slides 论坛中发布您的问题——它将在几小时内得到回答。Aspose.Slides 支持团队会直接回复论坛上的提问。
- 请注意，Aspose 不提供电话技术支持。电话支持仅适用于销售和购买问题。
- 在论坛期待回复时，请耐心等待并考虑时区差异。

当您遇到 Aspose.Slides for Python via .NET 的问题时，请按照以下指南以最高效的方式解决：

- 在报告问题之前，请检查并确认您使用的是最新的 Aspose.Slides for Python via .NET 版本。也许您遇到的问题已在更新中得到解决。

  请参阅 Aspose.Slides 下载以获取最新版本的信息。

- 在报告问题之前，您可能需要浏览我们的论坛、本文档和 API 参考。也许您的问题答案已经提供。

- 报告问题时，请包括原始文档以及（如果可能）与问题相关的代码片段。如需附加多个文件，请将它们压缩为 ZIP 包。

  您可以安全地在 Aspose.Forums 中附加文档，因为 **only** 您和 Aspose 开发人员可以访问这些附件。

- 为特定问题创建并使用一个主题。如果您遇到另一个问题，需要另建主题进行报告。

## **常见问题**

**有哪些付费支持层级，它们有何区别？**

[Developer、Business 和 Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services)。它们在包含的事件数量、响应优先级以及可用的沟通渠道上有所不同。

**我可以直接将漏洞升级给工程团队吗？**

可以。[付费支持](https://helpdesk.aspose.com/) 在问题需要深入调查时允许升级到开发人员。

**是否支持旧版本库，且会为其发布修复吗？**

您可以使用旧版本，但修复仅提供给当前发布的版本；不会为旧版发布补丁。
