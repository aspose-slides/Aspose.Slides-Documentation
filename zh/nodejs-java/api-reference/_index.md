{{% alert color="primary" %}} 

Aspose.Slides for Node.js via Java 是一个类库，它使 Node.js 应用程序能够读取和写入演示文档，而无需使用 Microsoft PowerPoint® 或其他软件。Aspose.Slides for Node.js via Java 允许您管理文本、形状、表格和动画，向幻灯片添加音频和视频，预览幻灯片，将幻灯片导出为 SVG、PDF 格式，并执行更多操作。

Aspose.Slides for Node.js via Java 使用与 [Aspose.Slides for Java](https://reference.aspose.com/slides/nodejs-java/) 相同的 API 参考。 

{{% /alert %}}
