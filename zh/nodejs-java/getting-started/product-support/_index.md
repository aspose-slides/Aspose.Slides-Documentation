## **技术支持**
{{% alert color="primary" %}}

Aspose 为其所有产品提供无限的免费技术支持。该支持对所有用户均可使用（包括使用评估版的用户）。

{{% /alert %}} 

当您需要 Aspose.Slides for Node.js via Java 的帮助时，请考虑以下内容：

- 支持的主要途径是 [Aspose.Forums](https://forum.aspose.com/c/slides/11)。在 Aspose.Slides 论坛中发布您的问题——它将在几小时内得到答复。Aspose.Slides 支持团队会直接回答论坛上发布的问题。
- 请注意，Aspose 不提供电话技术支持。电话支持仅适用于销售和购买相关问题。
- 在论坛等待回复时，请保持耐心，并考虑时区差异。

当您在使用 Aspose.Slides for Node.js via Java 时遇到问题，请遵循以下指南以最有效的方式解决：

- 在报告问题之前，请检查并确认您使用的是最新的 Aspose.Slides for Node.js via Java 版本。也许您遇到的问题已经在更新中得到解决。 

  请参阅 [Aspose.Slides download](https://releases.aspose.com/slides/nodejs-java/) 以获取最新版本信息。

- 在报告问题之前，您可能需要查看我们的论坛、本文件以及 API 参考。也许您的问题已有答案。 

- 报告问题时，请附上原始文档以及（如果可能）与问题相关的代码片段。如果需要附加多个文件，请将它们压缩为 ZIP 包。您可以安全地在 Aspose.Forums 中附加文档，因为 **仅** 您和 Aspose 开发人员可以访问这些附件。

- 针对特定问题创建并使用一个主题。如果您遇到其他问题，需要创建另一个主题来报告。

## **常见问答**

**有哪些付费支持层级，它们有何区别？**

[Developer、Business 和 Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services)。它们在包含的事件数量、响应优先级以及可用的沟通渠道上有所不同。

**我可以直接将 Bug 升级到工程团队吗？**

可以。[付费支持](https://helpdesk.aspose.com/) 在问题需要更深入调查时允许升级到开发人员。

**是否支持旧版本库，并为其发布修复？**

您可以使用旧版本，但仅对当前发布提供修复；不会为旧版发布补丁。
