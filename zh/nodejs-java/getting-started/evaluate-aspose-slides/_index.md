您可以轻松下载 Aspose.Slides 进行评估。评估包与购买的包完全相同。评估版本在您添加几行代码以应用许可证后即可变为许可版本。

Aspose.Slides 的评估版本（未指定许可证）提供完整的产品功能，但在打开和保存文档时，会在文档顶部插入评估水印。在从演示文稿幻灯片中提取文本时，您也仅限使用一张幻灯片。
