Aspose.Slides for Node.js via Java 不需要安装任何第三方产品，例如 Microsoft PowerPoint。Aspose.Slides 本身是一个引擎，可用于创建、修改、转换和渲染各种格式的文档，包括 Microsoft PowerPoint 演示文稿格式。

## **受支持的操作系统**

Aspose.Slides for Node.js via Java 支持任何运行 Java 运行时的 32 位或 64 位操作系统，包括但不限于：

### **Windows**
- Microsoft Windows 2003 Server ( x64, x86)
- Microsoft Windows 2008 Server ( x64, x86)
- Microsoft Windows 2012 Server ( x64, x86)
- Microsoft Windows 2012 R2 Server ( x64, x86)
- Microsoft Windows 2016 Server ( x64, x86)
- Microsoft Windows 2019 Server ( x64, x86)
- Microsoft Windows Vista ( x64, x86)
- Microsoft Windows XP ( x64, x86)
- Microsoft Windows 7 ( x64, x86)
- Microsoft Windows 8, 8.1 ( x64, x86)
- Microsoft Windows 10 ( x64, x86)

### **Linux**
- Linux (Ubuntu, CentOS and others)

### **Mac**
- Mac OS X

## **常见问题**

**转换和渲染是否需要安装 Microsoft PowerPoint？**

No, PowerPoint is not required; Aspose.Slides is a standalone engine for [创建](/slides/zh/nodejs-java/create-presentation/), modifying, [转换](/slides/zh/nodejs-java/convert-presentation/), and [渲染](/slides/zh/nodejs-java/convert-powerpoint-to-png/) presentations.

**正确渲染需要哪些字体？**

In practice, the fonts used in the presentation or proper [替代字体](/slides/zh/nodejs-java/font-substitution/) must be available. To ensure consistent rendering on Linux/macOS, it is advisable to install common font packages.
