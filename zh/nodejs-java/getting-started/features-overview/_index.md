## **支持的平台**
Aspose.Slides for Node.js via Java 是一个平台无关的 API。它可以在任何安装了 Node.js 7 或更高版本的平台（Windows、Linux、MacOS 等）上使用。Oracle JDK 8 或更高版本是另一个要求。

## **文件格式和转换**
Aspose.Slides for Node.js via Java 支持大多数 PowerPoint 文档格式。它还允许您将这些文档导出为组织内部常用并共享的文件格式。请查看以下详情：

|**功能**|**描述**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via Java 为此演示文档格式提供最快的处理速度。|
|PPT 到 PPTX 转换|Aspose.Slides for Node.js via Java 支持 PPT 到 PPTX 的转换。|
|可移植文档格式 (PDF)|您可以使用单一方法将所有受支持的文件格式导出为 Adobe 可移植文档格式 (PDF) 文档。|
|XML 页面规范 (XPS)|您可以使用单一方法将所有受支持的文件格式导出为 XML 页面规范 (XPS) 文档。|
|标记图像文件格式 (TIFF)|您可以将所有受支持的演示文稿文件格式导出为标记图像文件格式 (TIFF)。|
|PPTX 到 HTML 转换|Aspose.Slides for Node.js via Java 支持将 PresentationEx 转换为 HTML 格式。|

## **FAQ**

**我需要在服务器/电脑上安装 Microsoft PowerPoint 才能使用该库吗？**

不需要。PowerPoint 并非必需；Aspose.Slides 是一个独立的引擎，能够创建、编辑、转换和渲染演示文稿。

**多线程是如何工作的？可以并行处理吗？**

在不同线程中处理不同文档是安全的；同一[presentation](https://reference.aspose.com/slides/nodejs-java/aspose.slides/presentation/)对象不应被[multiple threads](/slides/zh/nodejs-java/multithreading/)同时使用。

**是否支持文件密码和加密？**

是的。您可以[/slides/nodejs-java/password-protected-presentation/]打开受密码保护的演示文稿，设置或移除打开和写入密码，并检查保护状态。

**在 Linux 容器中需要关心字体包吗？**

是的。建议安装常用字体包，或在应用程序中显式[specify font directories](/slides/zh/nodejs-java/custom-font/)，以避免意外的字体替换。

**评估版有哪些限制？**

在[evaluation mode](/slides/zh/nodejs-java/licensing/)下，输出会添加水印并且存在若干限制；您可以获取[30 天临时许可证](https://purchase.aspose.com/temporary-license/)以进行完整功能测试。

**是否支持将外部格式导入演示文稿（PDF/HTML → PPTX）？**

是的。您可以将[PDF 页面和 HTML 内容](/slides/zh/nodejs-java/import-presentation/)添加到演示文稿中，将其转换为幻灯片。
