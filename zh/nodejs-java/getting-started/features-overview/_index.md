## **支持的平台**
Aspose.Slides for Node.js via Java 是一个平台独立的 API。它可以在安装了 Node.js 7 或更高版本的任何平台（Windows、Linux、MacOS 等）上使用。另一个要求是 Oracle JDK 8 或更高版本。

## **文件格式和转换**
Aspose.Slides for Node.js via Java 支持大多数 PowerPoint 文档格式。它还允许您将它们导出为组织间常用的文件格式。请查看以下详细信息：

|**特性**|**描述**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via Java 为此演示文档格式提供最快的处理速度。|
|PPT 到 PPTX 转换|Aspose.Slides for Node.js via Java 支持 PPT 到 PPTX 转换。|
|可移植文档格式 (PDF)|您可以使用一个方法将所有支持的文件格式导出为 Adobe 可移植文档格式 (PDF) 文档。|
|XML 解析器规范 (XPS)|您可以使用一个方法将所有支持的文件格式导出为 XML 解析器规范 (XPS) 文档。|
|标记图像文件格式 (TIFF)|您可以将所有支持的演示文件格式导出为标记图像文件格式 (TIFF)。|
|PPTX 到 HTML 转换|Aspose.Slides for Node.js via Java 支持将 PresentationEx 转换为 HTML 格式。|

## **渲染和打印**
Aspose.Slides for Node.js via Java 支持以高保真度将演示文稿中的幻灯片渲染为不同的图形格式。请查看以下详细信息：

|**特性**|**描述**|
| :- | :- |
|支持的 Java 图像格式|使用 Aspose.Slides for Node.js via Java，您可以将演示幻灯片和幻灯片上的图像渲染为所有 Java 支持的图形格式，如 TIFF、PNG、BMP、JPEG、GIF。|
|SVG 格式|Aspose.Slides for Node.js via Java 还提供内置方法，允许您将演示幻灯片导出为可缩放矢量图形 (SVG) 格式。|
|演示文稿打印|最新版本的 Aspose.Slides for Node.js via Java 提供了具有不同选项的内置打印方法。|
## **内容特性**
Aspose.Slides for Node.js via Java 允许您访问、修改或创建几乎所有演示文档的项目或内容。请查看以下详细信息：

|**特性**|**描述**|
| :- | :- |
|母版幻灯片|母版幻灯片定义普通幻灯片的布局。Aspose.Slides for Node.js via Java 允许您访问和修改演示文档的母版幻灯片。|
|普通幻灯片|使用 Aspose.Slides for Node.js via Java，您可以创建不同类型的新幻灯片；您还可以访问和修改演示文稿中现有的幻灯片。|
|克隆/复制幻灯片|Aspose.Slides for Node.js via Java 提供了允许您在演示文稿中克隆或复制现有幻灯片的内置方法。您还可以将复制和克隆的幻灯片用于其它演示文稿。由于幻灯片从母版幻灯片继承其布局，因此内置克隆方法在克隆时会自动复制母版。|
|管理幻灯片部分|在演示文稿中以不同部分组织幻灯片的方法。|
|占位符和文本框|您可以访问幻灯片中的占位符和文本框。此外，您可以使用适当的方法从头开始创建带有文本框的幻灯片。|
|页眉和页脚|Aspose.Slides for Node.js via Java 支持处理幻灯片中的页眉/页脚。|
|幻灯片中的笔记|使用 Aspose.Slides for Node.js via Java，您可以访问和修改与幻灯片关联的笔记，并添加新笔记。|
|查找形状|您还可以使用与形状关联的替代文本在幻灯片中查找特定形状。|
|背景|Aspose.Slides for Node.js via Java 允许您处理与演示文稿中的母版幻灯片或普通幻灯片关联的背景。|
|文本框|文本框可以从头创建。您可以访问现有的文本框。您还可以修改其文本而不丢失原始文本格式。|
|矩形形状|您可以使用 Aspose.Slides for Node.js via Java 创建或修改矩形形状。|
|折线形状|您可以使用 Aspose.Slides for Node.js via Java 创建或修改折线形状。|
|椭圆形状|您可以使用 Aspose.Slides for Node.js via Java 创建或修改椭圆形状。|
|组合形状|Aspose.Slides for Node.js via Java 支持组合形状。|
|自动形状|Aspose.Slides for Node.js via Java 支持自动形状。|
|SmartArt|Aspose.Slides for Node.js via Java 提供对 MS PowerPoint 中 SmartArt 形状的支持。|
|图表|Aspose.Slides for Node.js via Java 提供对 PowerPoint 中 MSO 图表的支持。|
|形状序列化|Aspose.Slides for Node.js via Java 支持大量形状。当 Aspose.Slides for Node.js via Java 不支持某个形状时，您可以使用序列化方法通过现有幻灯片序列化该形状。以这种方式，您可以根据需要进一步使用该形状。|
|图片框|您可以使用 Aspose.Slides for Node.js via Java 管理图片框中的图片。|
|音频框|您可以使用 Aspose.Slides for Node.js via Java 在幻灯片的音频框中链接或嵌入音频文件。|
|视频框|您可以在视频框中处理视频文件。Aspose.Slides for Node.js via Java 还支持链接和嵌入的视频。|
|OLE 框|您可以使用 Aspose.Slides for Node.js via Java 管理 OLE 框中的 OLE 对象。|
|表格|Aspose.Slides for Node.js via Java 支持幻灯片中的表格。|
|ActiveX 控件|支持 ActiveX 控件。|
|VBA 宏|支持在演示文稿中管理 VBA 宏。|
|文本框架|您可以通过与该形状关联的文本框访问该形状中的文本。|
|文本扫描|您可以通过内置扫描方法在演示文稿或幻灯片级别扫描文本。|
|动画|您可以在形状上应用动画。|
|幻灯片放映|Aspose.Slides for Node.js via Java 支持幻灯片放映和幻灯片切换。|
