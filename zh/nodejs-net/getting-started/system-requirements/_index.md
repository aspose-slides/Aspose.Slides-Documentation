Aspose.Slides for Node.js via .NET 不需要安装任何第三方产品，如 Microsoft PowerPoint。Aspose.Slides本身是一个用于创建、修改、转换和渲染各种格式文档的引擎，包括 Microsoft PowerPoint 演示文稿格式。

## 支持的操作系统

Aspose.Slides for .NET 支持任何运行 .NET 运行时的 32 位或 64 位操作系统，包括但不限于：

### **Windows**
- Microsoft Windows 2003 服务器 (x64, x86)
- Microsoft Windows 2008 服务器 (x64, x86)
- Microsoft Windows 2012 服务器 (x64, x86)
- Microsoft Windows 2012 R2 服务器 (x64, x86)
- Microsoft Windows 2016 服务器 (x64, x86)
- Microsoft Windows 2019 服务器 (x64, x86)
- Microsoft Windows Vista (x64, x86)
- Microsoft Windows XP (x64, x86)
- Microsoft Windows 7 (x64, x86)
- Microsoft Windows 8, 8.1 (x64, x86)
- Microsoft Windows 10 (x64, x86)
- Microsoft Windows 11 (x64, x86)

### **Linux**
- Linux (Ubuntu、CentOS 等)

### **Mac**
- Mac OS X
