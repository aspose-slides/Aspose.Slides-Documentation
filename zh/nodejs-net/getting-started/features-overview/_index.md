## **支持的平台**
Aspose.Slides for Node.js 通过 .NET 是一个平台独立的 API。它可以在安装了 Node.js 7 或更高版本的任何平台（Windows、Linux、MacOS 等）上使用。Oracle JDK 8 或更高版本是另一个要求。

## **文件格式和转换**
Aspose.Slides for Node.js 通过 .NET 支持大多数 PowerPoint 文档格式。它还允许您将其导出为组织使用和共享的流行文件格式。请查看以下详细信息：

|**特性**|**描述**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js 通过 .NET 为此演示文档格式提供最快的处理速度。|
|PPT 转 PPTX 转换|Aspose.Slides for Node.js 通过 .NET 支持 PPT 到 PPTX 的转换|
|可移植文档格式 (PDF)|您可以通过单个方法将所有支持的文件格式导出为 Adobe 可移植文档格式 (PDF) 文档。|
|XML 解析器规范 (XPS)|您可以通过单个方法将所有支持的文件格式导出为 XML 解析器规范 (XPS) 文档。|
|标记图像文件格式 (TIFF)|您可以将所有支持的演示文件格式导出为标记图像文件格式 (TIFF)。|
|PPTX 到 HTML 转换|Aspose.Slides for Node.js 通过 .NET 支持将 PresentationEx 转换为 HTML 格式。|

## **渲染和打印**
Aspose.Slides for Node.js 通过 .NET 支持以高保真度将演示文稿中的幻灯片渲染为不同的图形格式。请查看以下详细信息：

|**特性**|**描述**|
| :- | :- |
|.NET 支持的图像格式|使用 Aspose.Slides for Node.js 通过 .NET，您可以将演示幻灯片和幻灯片上的图像渲染为所有 .NET 支持的图形格式，如 TIFF、PNG、BMP、JPEG、GIF。|
|SVG 格式|Aspose.Slides for Node.js 通过 .NET 还提供内置方法，允许您将演示幻灯片导出为可扩展矢量图形 (SVG) 格式。|
|演示文稿打印|Aspose.Slides for Node.js 通过 .NET 的最新版本提供了具有不同选项的内置打印方法。|

## **内容特性**
Aspose.Slides for Node.js 通过 .NET 允许您访问、修改或创建几乎所有演示文档的项目或内容。请查看以下详细信息：

|**特性**|**描述**|
| :- | :- |
|母版幻灯片|母版幻灯片定义了正常幻灯片的布局。Aspose.Slides for Node.js 通过 .NET 允许您访问和修改演示文档的母版幻灯片|
|正常幻灯片|使用 Aspose.Slides for Node.js 通过 .NET，您可以创建不同类型的新幻灯片；您还可以访问和修改演示文稿中现有的幻灯片|
|克隆/复制幻灯片|Aspose.Slides for Node.js 通过 .NET 提供了内置方法，允许您在演示文稿中克隆或复制现有幻灯片。您还可以将复制和克隆的幻灯片从一个演示文稿使用到另一个。由于幻灯片从母版幻灯片继承其布局，因此内置的克隆方法在克隆时会自动复制母版|
|管理幻灯片部分|在演示文稿内部以不同部分组织幻灯片的方法|
|占位符和文本框|您可以访问幻灯片中的占位符和文本框。此外，您可以使用适当的方法从头开始创建带有文本框的幻灯片|
|页眉和页脚|Aspose.Slides for Node.js 通过 .NET 方便幻灯片中页眉/页脚的处理|
|幻灯片中的注释|使用 Aspose.Slides for Node.js 通过 .NET，您可以访问和修改与幻灯片相关的注释，也可以添加新的注释|
|查找形状|您还可以使用与形状相关的替代文本从幻灯片中找到特定形状|
|背景|Aspose.Slides for Node.js 通过 .NET 允许您处理与演示文稿中的母版或正常幻灯片相关的背景|
|文本框|可以从头开始创建文本框。您可以访问现有的文本框。您还可以修改它们的文本，而不会丢失原始文本格式|
|矩形形状|您可以使用 Aspose.Slides for Node.js 通过 .NET 创建或修改矩形形状|
|多段形状|您可以使用 Aspose.Slides for Node.js 通过 .NET 创建或修改多段形状|
|椭圆形状|您可以使用 Aspose.Slides for Node.js 通过 .NET 创建或修改椭圆形状|
|组合形状|Aspose.Slides for Node.js 通过 .NET 支持组合形状|
|自动形状|Aspose.Slides for Node.js 通过 .NET 支持自动形状|
|SmartArt|Aspose.Slides for Node.js 通过 .NET 提供对 MS PowerPoint 中 SmartArt 形状的支持|
|图表|Aspose.Slides for Node.js 通过 .NET 提供对 PowerPoint 中 MSO 图表的支持|
|形状序列化|Aspose.Slides for Node.js 通过 .NET 支持大量形状。当 Aspose.Slides for Node.js 通过 .NET 缺乏对某一形状的支持时，您可以使用序列化方法通过该方法对现有幻灯片中的形状进行序列化。通过这种方式，您可以根据需要进一步使用该形状|
|图片框|您可以使用 Aspose.Slides for Node.js 通过 .NET 管理图片框中的图片|
|音频框|您可以在幻灯片上的音频框中链接或嵌入音频文件，使用 Aspose.Slides for Node.js 通过 .NET|
|视频框|您可以在视频框中处理视频文件。Aspose.Slides for Node.js 通过 .NET 还提供对链接和嵌入视频的支持|
|OLE 框|您可以在 OLE 框中管理 OLE 对象，使用 Aspose.Slides for Node.js 通过 .NET|
|表格|Aspose.Slides for Node.js 通过 .NET 支持幻灯片中的表格|
|ActiveX 控件|支持 ActiveX 控件|
|VBA 宏|支持管理演示文稿中的 VBA 宏。|
|文本框|您可以通过与形状相关的文本框访问该形状的文本|
|文本扫描|您可以通过内置扫描方法在演示文稿或幻灯片级别扫描文本。|
|动画|您可以对形状应用动画|
|幻灯片放映|Aspose.Slides for Node.js 通过 .NET 支持幻灯片放映和幻灯片切换|

## **格式化特性**
使用 Aspose.Slides for Node.js 通过 .NET，您可以格式化演示文稿幻灯片上的文本和形状。请查看以下详细信息：

|**特性**|**描述**|
| :- | :- |
|文本格式化|<p>在 Aspose.Slides for Node.js 通过 .NET 中，您可以通过与形状相关的文本框管理文本。因此，您可以使用与文本框相关的段落和部分格式化文本。这些文本元素可以通过 Aspose.Slides for Node.js 通过 .NET 进行格式化。</p><p>- 字体类型</p><p>- 字体大小</p><p>- 字体颜色</p><p>- 字体阴影</p><p>- 段落对齐</p><p>- 段落项目符号</p><p>- 段落方向</p>|
|形状格式化|<p>在 Aspose.Slides for Node.js 通过 .NET 中，幻灯片的基本元素是形状。您可以使用 Aspose.Slides for Node.js 通过 .NET 格式化这些形状元素：</p><p>- 位置</p><p>- 尺寸</p><p>- 线条</p><p>- 填充（包括图案、渐变、实色）</p><p>- 文本</p><p>- 图像</p>|
