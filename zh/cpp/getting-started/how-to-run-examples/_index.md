## **软件需求**
请确保在下载和运行示例之前满足以下要求。

1. Visual Studio 2015 或更高版本。

## **从 GitHub 下载 Aspose.Slides**
Aspose.Slides for C++ 的所有示例均托管在 [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-C)。

- 您可以使用喜爱的 GitHub 客户端克隆仓库，或从 [here](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/refs/heads/master.zip) 下载 ZIP 文件。
- 将 ZIP 文件的内容解压到计算机上的任意文件夹。所有示例位于 **Examples** 文件夹中。
- 有一个 C++ Visual Studio 解决方案文件。
- 项目是在 Visual Studio 2015 中创建的，但解决方案文件兼容 Visual Studio 2015 及更高版本。
- 在 Visual Studio 中打开解决方案文件并构建项目。
- 从 [Download Section](https://downloads.aspose.com/slides/cpp) 下载 Aspose.Slides for C++ 的最新发布版本。
- 解压发布包并将 **Include** 和 **Bin** 目录复制到 **Examples** 文件夹内。
- 打开 **main.cpp** 文件，所有示例均从此处调用。
- 取消注释您想要运行的示例。
- 构建并编译解决方案

如果在设置或运行示例时遇到任何问题，请通过我们的论坛联系。
