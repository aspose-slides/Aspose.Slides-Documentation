## **对遗留代码的支持**
为了使用与 Aspose.Slides for PHP via Java 早于 13.x 版本开发的遗留代码，您需要对代码进行一些小的修改，代码将像以前一样工作。之前在 Aspose.Slide 和 Aspose.Slides.Pptx 命名空间下的旧 Aspose.Slides for PHP via Java 中的所有类现在已合并到单一的 Aspose.Slides 命名空间中。请查看以下简单代码片段，了解如何在遗留 Aspose.Slides API 中创建 Hello World 演示文档，并按照描述迁移到新合并 API 的步骤进行操作。
