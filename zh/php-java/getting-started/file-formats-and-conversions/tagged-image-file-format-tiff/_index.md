## **关于 TIFF**

[标签图像文件格式](https://en.wikipedia.org/wiki/TIFF) 最初由 Aldus 创建，因其能够在一个文档中存储多个图像而广受欢迎。该格式在扫描、传真和其他图像处理应用程序中得到了广泛支持。

## **在 Aspose.Slides for PHP via Java 中的 TIFF**

任何可以在 Aspose.Slide for Java 中加载的文档，也可以直接由 Aspose.Slides for PHP via Java 转换为 TIFF 文档，从而消除了对任何第三方组件的需求。此外，您可以选择定义生成的 TIFF 文档中图像的大小。有关通过 Aspose.Slides for PHP via Java 导出演示文档到 TIFF 文档的信息，请参见[此主题](/slides/php-java/converting-a-presentation/)。
