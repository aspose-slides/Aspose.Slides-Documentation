## **关于 TIFF**
[标记图像文件格式](https://en.wikipedia.org/wiki/TIFF) 以能够在一个文档中存储多个图像而闻名，最初由 Aldus 创建。该格式被扫描、传真以及其他图像处理应用广泛支持。

## **在 Aspose.Slides for PHP via Java 中的 TIFF**
任何可以在 Aspose.Slide for Java 中加载的文档，都可以直接通过 Aspose.Slides for PHP via Java 转换为 TIFF 文档，无需任何第三方组件。此外，您还可以可选地定义生成的 TIFF 文档中图像的大小。有关通过 Aspose.Slides for PHP via Java 将演示文稿导出为 TIFF 文档的信息，请参阅[此主题](/slides/zh/php-java/convert-powerpoint-to-tiff/)。
