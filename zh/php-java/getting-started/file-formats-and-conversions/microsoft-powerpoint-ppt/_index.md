{{% alert color="primary" %}} 

[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) 是一种演示文稿文件格式，能够由不同版本的 Microsoft PowerPoint 创建、读取、操作和写入。它是 Microsoft 开发的二进制演示文稿格式。

{{% /alert %}} 

## **在 Aspose.Slides for PHP via Java 中的 PPT**
Aspose.Slides for PHP via Java 能够读取以下软件创建的 PPT 文件。

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

同样，Aspose.Slides for PHP via Java 创建的 PPT 文件也可以被上述软件读取。

## **对 PPT 的全面支持**
Aspose.Slides for PHP via Java 提供几乎所有 PPT 文档文件格式支持的特性。它不仅覆盖了不同 Microsoft PowerPoint 版本提供的基础和高级功能，还包括 Microsoft PowerPoint 本身不支持的功能。使用 Aspose.Slides for PHP via Java API 库的主要优势在于处理这些功能的简易性。

除了创建、读取和写入 PPT 文档文件的基本任务外，Aspose.Slides for PHP via Java 还提供以下功能：

- 将其他 Microsoft Office 文件格式导入为[OLE 对象到 PPT 文档](/slides/zh/php-java/manage-ole/)。
- [将 PPT 文档导出为 PDF](/slides/zh/php-java/convert-powerpoint-to-pdf/)。
- 将 PPT 文档中的幻灯片导出为 SVG 格式。
- 将幻灯片渲染为 Java 框架支持的任何图像格式。
- 设置 PPT 文档中幻灯片的尺寸。
- 管理形状上的动画。
- 管理幻灯片放映。
- [格式化幻灯片上的文本](/slides/zh/php-java/text-formatting/)。
- 从 PPT 文档中提取文本。
- [处理幻灯片上的表格](/slides/zh/php-java/powerpoint-table/)。
- 使用[克隆功能](/slides/zh/php-java/clone-slides/)自动复制母版。
