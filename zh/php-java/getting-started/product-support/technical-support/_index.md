## **技术支持**

{{% alert color="primary" %}} 

Aspose 为所有产品提供无限的免费技术支持。该支持对所有用户均可用，包括评估版用户。

{{% /alert %}} 

如果您需要 Aspose.Slides for PHP via Java 的帮助，请参考以下内容：

- 支持的主要渠道是 Aspose.Forums。请在 Aspose.Slides 论坛中发布您的问题，几个小时内会得到答复。问题由 Aspose.Slides 支持团队直接解答。
- 请注意，Aspose 不提供电话技术支持。电话支持仅针对销售和购买问题。
- 在论坛中期待回复时，请考虑时区差异。

如果您在使用 Aspose.Slides for PHP via Java 时遇到问题，请遵循以下简单步骤，以确保以最高效的方式解决问题：

- 在报告问题之前，请确保使用最新的 Aspose.Slides for PHP via Java 版本，可访问 Aspose.Slides 下载页面了解最新版本。
- 在报告问题之前，请浏览论坛、本文档和 API 参考；也许您的问题已经得到答案。
- 报告问题时，请附上原始文档以及可能导致问题的代码片段。如果需要附加多个文件，请将它们压缩成一个 zip。将文档上传至 Aspose.Forums 是安全的，因为只有您和 Aspose 开发人员能够访问这些附件。
- 请尝试在每个主题中报告一个问题。如果有其他问题，请在单独的主题中报告。

## **常见问题**

**有哪些付费支持层级，它们有何区别？**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services)。它们在包含的事件数量、响应优先级和可用的沟通渠道方面有所不同。

**我可以直接将错误升级给工程团队吗？**

是的。[Paid support](https://helpdesk.aspose.com/) 在问题需要更深入调查时允许升级至开发人员。

**是否支持较旧的库版本，且会发布修复吗？**

您可以使用旧版本，但仅为当前发布提供修复；不会为旧版发布补丁。
