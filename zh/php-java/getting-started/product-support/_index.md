## **技术支持**
{{% alert color="primary" %}}

Aspose 为所有产品提供无限制的免费技术支持。支持适用于所有用户（包括使用评估包的用户）。

{{% /alert %}} 

当您需要 Aspose.Slides for PHP via Java 的帮助时，请考虑以下几点：

- 支持的主要渠道是 [Aspose.Forums](https://forum.aspose.com/c/slides/11)。在 Aspose.Slides 论坛中发布您的问题——几小时内会有回复。Aspose.Slides 支持团队会直接回答在论坛上发布的问题。
- 请注意，Aspose 不提供电话技术支持。电话支持仅适用于销售和购买问题。
- 当您期待在论坛中得到回复时，请耐心等待，并考虑时区差异。

当您在使用 Aspose.Slides for PHP via Java 时遇到问题时，请遵循以下指南以最有效的方式解决：

- 报告问题之前，请检查并确认您正在使用最新版本的 Aspose.Slides for PHP via Java。您遇到的问题可能在某个更新中已解决。 

  请查看 [Aspose.Slides 下载](https://downloads.aspose.com/slides/php-java) 以获取有关最新版本的信息。

- 在报告问题之前，您可能想先浏览我们的论坛、此文档和 API 参考。您的问题可能已有答案。 

- 报告问题时，请附上原始文档和（如果可能）与问题相关的代码片段。如果需要附加多个文件，请将它们放入 ZIP 包中。您可以安全地在 Aspose.Forums 中附加您的文档，因为 **只有** 您和 Aspose 开发人员可以访问附加的文件。

- 针对特定问题创建并使用一个主题。如果您遇到另一个问题，您需要创建另一个主题来报告该问题。
