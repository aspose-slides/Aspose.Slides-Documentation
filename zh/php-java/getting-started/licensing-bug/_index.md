本文描述了在 Aspose.Slides for PHP via Java 版本 23.2、23.3、23.4、23.5、23.6 和 23.7 中遇到的许可错误。此错误导致永久许可证在其订阅到期后暂时无法使用。

## **症状**

在永久许可证订阅到期后，Aspose.Slides for PHP via Java 版本 23.2 - 23.7 可能会报告许可证已过期，并将所有功能切换到评估模式。此行为不正确，且不影响 23.2 之前和 23.7 之后的版本。

## **解决方案**

Aspose.Slides for PHP via Java 解决了此问题并发布了带有修复的更新版本 (23.2.1、23.3.1、23.4.1、23.5.1、23.6.1、23.7.1)。

如果您在项目中使用受影响的 Aspose.Slides for PHP via Java 版本，请切换到使用已修正的版本。

如果您在使用 Aspose.Slides for PHP via Java 时遇到任何许可问题：

- 检查您使用的版本，确保它不受上述错误的影响。
- 参考 [Aspose 文档](https://docs.aspose.com/slides/php-java/getting-started/) 获取故障排除步骤和已知许可问题。
- 联系 [Aspose 支持](https://forum.aspose.com/) 获取进一步帮助。

## **受影响的产品和版本**

请注意，此错误仅影响 Aspose.Slides for PHP via Java 版本 23.2 到 23.7。**其他 Aspose 产品和其他版本的 Aspose.Slides 不受影响**。
