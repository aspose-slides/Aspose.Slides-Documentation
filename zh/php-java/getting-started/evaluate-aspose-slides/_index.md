## **Aspose.Slides 评估**

您可以轻松下载 Aspose.Slides 进行评估。评估包与购买的包相同。只需添加几行代码应用许可证，评估版本即可转为正式授权。

Aspose.Slides 的评估版本（未指定许可证）提供完整的产品功能，但在打开和保存文档时会在文档顶部插入评估水印。提取演示文稿文本时也仅限于一张幻灯片。

## **常见问题**

**我可以在评估模式下在不同线程中并行测试多个演示文稿吗？**

可以。您可以并行处理不同的文档；但不应在多个线程之间共享同一 Presentation 对象[跨线程](/slides/zh/php-java/multithreading/)。评估模式不会影响此行为。

**我需要在服务器或 CI 环境中安装 Microsoft PowerPoint 来评估此库吗？**

不需要。Aspose.Slides 是独立的引擎，无论是评估还是生产环境都不需要安装 PowerPoint。

**我可以在评估模式下完整测试 PPT/PPTX 转 PDF 和图像的转换吗？**

可以。此[转换器](/slides/zh/php-java/convert-presentation/)可用；输出中会包含水印。

**我可以使用临时许可证进行无水印的负载测试吗？**

可以。30 天临时许可证可消除评估模式的限制，允许进行无水印的测试。
