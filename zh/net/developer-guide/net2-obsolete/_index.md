作为我们持续提升产品质量和安全性的努力的一部分，我们计划在 **版本 25.11** 中进行以下更改：

- 针对 **.NET Framework 2.0**、**3.5** 和 **4.0** 的程序集将从发行包中移除。  
- 相反，我们将引入一个针对 **.NET Framework 4.6.2** 的新程序集，提供对现代平台功能的更好支持并提升安全性。  
- 对 **.NET Standard 2.1** 的支持将停止。以后我们将改用更广泛兼容的 **.NET Standard 2.0** 程序集。

如果您的项目目前依赖于已弃用的平台，建议您规划迁移到受支持的版本。这些更改将使我们能够专注于提升性能、安全性以及与最新平台的兼容性。
