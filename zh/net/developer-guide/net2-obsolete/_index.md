作为我们持续改进产品质量和安全性的努力的一部分，我们计划在 **version 25.11** 中进行以下更改：

- 将从发布包中移除目标为 **.NET Framework 2.0**、**3.5** 和 **4.0** 的程序集。  
- 取而代之的是引入一个目标为 **.NET Framework 4.6.2** 的新程序集，提供对现代平台特性的更好支持和改进的安全性。
- 将停止对 **.NET Standard 2.1** 的支持。今后我们将使用更具兼容性的 **.NET Standard 2.0** 程序集。

如果您的项目当前依赖于已弃用的平台，建议计划迁移到受支持的版本。这些更改将使我们能够专注于提升性能、安全性以及与最新平台的兼容性。
