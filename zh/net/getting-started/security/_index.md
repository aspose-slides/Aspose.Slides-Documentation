## **Aspose.Slides 中的安全性**

Aspose 在开发其产品时遵循最佳实践。

* Aspose.Slides for .NET 用于在其他格式的文件中操作和转换演示文稿。它不会在演示文稿中运行脚本。Aspose.Slides 仅解析演示文稿结构，并允许终端用户的代码以便捷的方式操作对象模型。
* Aspose.Slides 作为一个库，在解析和解释文档时不执行远程代码。所有 Aspose 产品都在您的机器上运行。它们不会将任何数据传输给 Aspose。唯一例外是使用 [计量许可](https://purchase.aspose.com/faqs/licensing/metered) 时，即使如此（如果您使用计量许可），也仅会处理您的 API 使用信息。
* Aspose 组件在与常规应用程序相同的用户上下文中运行。因此，Aspose 组件不会对关键系统资源构成风险。此外，当文档被 Aspose 组件打开时，宏不会自动运行。
* 与 Microsoft Office 套件固有或相关的风险不适用于 Aspose 组件，因此 Aspose 产品非常安全。

## **常见问题**

**使用哪种系统监控 Aspose 代码中的漏洞？**

我们对每个 Aspose.Slides 发行版进行静态代码分析。我们可以提供安全报告，证明 Aspose.Slides 代码通过了 OWASP Top 10。

**使用哪种系统监控外部包中的漏洞？**

我们不使用系统来检查外部包的漏洞，因为 Aspose.Slides 不包含或使用任何外部包。Aspose.Slides 仅依赖于 .NET 框架本身。
