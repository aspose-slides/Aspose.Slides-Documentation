## **Aspose.Slides 安全性**

Aspose 在开发其产品时遵循最佳实践。

* Aspose.Slides for .NET 用于在其他格式的文件中操作和转换演示文稿。它不在演示文稿中运行脚本。Aspose.Slides 仅解析演示文稿结构，并允许最终用户的代码以方便的方式操作对象模型。
* Aspose.Slides 作为一个库在解析和解释文档时不执行远程代码。所有 Aspose 产品都在您的机器上运行。它们不向 Aspose 传输任何数据。唯一例外是涉及 [metered license](https://purchase.aspose.com/faqs/licensing/metered) 时，即使使用计量许可证，也仅处理您的 API 使用信息。
* Aspose 组件在与常规应用程序相同的用户上下文中运行。因此，Aspose 组件不会对关键系统资源构成风险。此外，当文档由 Aspose 组件打开时，宏不会自动运行。
* Microsoft Office 套件固有或相关的风险不适用于 Aspose 组件，因此 Aspose 产品非常安全。

## 问答

> 使用什么系统监控 Aspose 代码中的漏洞？

我们对每个 Aspose.Slides 版本进行静态代码分析。我们可以提供安全报告，证明 Aspose.Slides 代码通过了 OWASP Top 10。

> 您使用什么系统来监控外部包中的漏洞？

我们不使用系统检查外部包的漏洞，因为 Aspose.Slides 不包含或使用任何外部包。Aspose.Slides 仅依赖 ONLY .NET 框架本身。
