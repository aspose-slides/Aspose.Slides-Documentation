## **介绍**
Microsoft Visual Studio 是微软推出的集成开发环境（IDE）。它用于为 Microsoft Windows 开发计算机程序，以及网站、Web 应用程序和 Web 服务。Visual Studio 使用 Microsoft 软件开发平台，如 Windows API、Windows Forms、Windows Presentation Foundation、Windows Store 和 Microsoft Silverlight。它既可以生成本机代码，也可以生成托管代码。
### **Aspose.Slides for .NET**
Aspose.Slides for .NET 是一个独特的 PowerPoint® 管理 API，使 .NET 应用程序能够在不使用 Microsoft PowerPoint 的情况下读取、写入和操作 PowerPoint 文档。

Aspose.Slides for .NET 是首个也是唯一的 .NET PowerPoint API，提供在您自己的应用程序中管理 PowerPoint 文档的功能。与所有 Aspose .NET 组件一样，Aspose.Slides 使用托管 C# 编写，价格极具竞争力且速度极快。

欲了解更多关于 [Aspose.Slides for .NET API](http://www.aspose.com/.net/powerpoint-component.aspx) 的信息
### **使用 Visual Studio 探索 Aspose.Slides for .NET 示例**
Aspose Visual Studio 插件 - **探索 Aspose.Slides for .NET 示例** 是一个极佳的工具，可快速下载并浏览 Aspose.Slides for .NET API 示例。它通过提供一种非常简单的方式，无需离开 Visual Studio 即可无缝选择、下载并打开最新的示例项目，从而节省大量时间和精力。

此版本提供以下功能：

- 支持 Visual Studio 2010、Visual Studio 2012 和 Visual Studio 2013。
- 可从 Visual Studio 中的 **文件 > Aspose** 菜单轻松启动
- 节省时间并降低学习曲线。
- 让您能够选择并打开托管在 Github.com 上的最新 Aspose.Slides for .NET API 示例。
- 允许您打开 **C#** 或 **VB.NET** 示例项目
- 所选项目会自动在 Visual Studio 中打开，并设置所有必需的引用，提供即开即用的环境。
- 自动从 **NuGet** 下载最新的 API 库，并从 **GitHub** 下载示例。

## **安装**
安装 Aspose Visual Studio 插件非常简单，请按以下步骤操作：

- 双击下载的 msi 文件
- 按步骤操作，您可以在每个界面上直接点击 “Next” 使用默认设置完成安装。

安装完成后会出现成功对话框。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_2)

**注意:** 请确保重新启动 Visual Studio 以使更改生效。
## **使用**
安装后，您可以按照下面的说明轻松启动并使用交互式插件：

**1.** 在 Visual Studio 中选择 **文件 > Aspose**，随后选择 **New Aspose.Slides Example Project** 打开插件。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_3)

**2.** 第一次打开插件时，会显示如图所示的 API 选择对话框。它将花费几分钟时间从 **NuGet** 下载最新的 API 及其库，并从 **GitHub** 下载最新的示例源代码。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_4)

**3.** 在 “Select an API to View Examples” 对话框中，选择或更改示例项目的创建位置，并选择或更改示例项目类型为 **C#** 或 **VB.NET**，随后点击 **Continue** 在 Visual Studio 中打开示例项目。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_1)

**4.** 根据在 Visual Studio 中选择的项目类型（**C#** 或 **VB.NET**），生成新的示例项目。

#### **如何配置源码**
打开并扩展源码前，需要安装以下软件：

- Visual Studio 2010

请按以下简单步骤开始：

1. 下载/克隆源码。
1. 打开 Visual Studio 2010，选择 **文件 > 打开项目**。
1. 浏览到您下载的最新源码并打开 **AsposeVisualStudioPlugin.sln**。

#### **源码概览**
解决方案中包含两个项目：

- **AsposeVisualStudioPlugin** - 包含与 Visual Studio 的集成、Aspose API 的选择与自动下载以及示例的获取。
- **Installer** - 使用 Wix 的安装程序项目，用于生成插件的 msi 安装文件。
