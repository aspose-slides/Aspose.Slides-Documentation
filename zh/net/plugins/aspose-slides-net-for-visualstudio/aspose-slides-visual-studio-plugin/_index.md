## **简介**
Microsoft Visual Studio 是 Microsoft 提供的一体化开发环境（IDE）。它用于为 Microsoft Windows 开发计算机程序，以及网站、Web 应用程序和 Web 服务。Visual Studio 使用 Microsoft 软件开发平台，如 Windows API、Windows Forms、Windows Presentation Foundation、Windows Store 和 Microsoft Silverlight。它既可以生成本机代码，也可以生成托管代码。

### **Aspose.Slides for .NET**
Aspose.Slides for .NET 是一个独特的 PowerPoint® 管理 API，能够让 .NET 应用程序在不使用 Microsoft PowerPoint 的情况下读取、写入和操作 PowerPoint 文档。

Aspose.Slides for .NET 是首个也是唯一的 .NET PowerPoint API，提供在您自己的应用程序中管理 PowerPoint 文档的功能。和所有 Aspose .NET 组件一样，Aspose.Slides 使用托管 C# 编写，定价极具竞争力，速度极快。

欲了解更多关于 [Aspose.Slides for .NET API](http://www.aspose.com/.net/powerpoint-component.aspx) 的信息

### **使用 Visual Studio 探索 Aspose.Slides for .NET 示例**
Aspose Visual Studio 插件 - **Explore Aspose.Slides for .NET Examples** 是一款能够快速下载并浏览 Aspose.Slides for .NET API 示例的强大工具。它通过提供一个非常简便的方式，无需离开 Visual Studio 即可无缝选择、下载并打开最新的示例项目，从而节省大量时间和精力。

此版本提供以下功能：

- 支持 Visual Studio 2010、Visual Studio 2012 和 Visual Studio 2013。
- 可从 Visual Studio 中的 **File > Aspose** 菜单轻松启动
- 节约时间并降低学习曲线
- 让您能够选择并打开托管在 Github.com 上的最新 Aspose.Slides for .NET API 示例
- 支持打开 **C#** 或 **VB.NET** 示例项目
- 所选项目会自动在 Visual Studio 中打开，并设置所有必需的引用，为您提供一个可直接运行并探索的环境
- 自动从 **NuGet** 下载最新的 API 库，并从 **GitHub** 下载示例

## **使用**
安装后，您可以按以下描述轻松启动并使用交互式插件：

**1.** 在 Visual Studio 中选择 **File > Aspose**，随后选择 **New Aspose.Slides Example Project** 打开插件。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_3)

**2.** 第一次打开插件时，会出现如图所示的 API 选择对话框。系统会耗时几分钟从 **NuGet** 下载最新的 API 及其库，并从 **GitHub** 下载最新的示例源码。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_4)

**3.** 在 “Select an API to View Examples” 对话框中，选择或更改示例项目的创建位置，并选择示例项目的语言类型为 **C#** 或 **VB.NET**，然后点击 **Continue** 在 Visual Studio 中打开示例项目。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_1)

**4.** 根据所选的项目类型（**C#** 或 **VB.NET**）在 Visual Studio 中生成新的示例项目。

#### **源代码概览**
解决方案中包含两个项目

- **AsposeVisualStudioPlugin** - 包含与 Visual Studio 的集成、Aspose API 的选择与自动下载以及示例的管理。
- **Installer** - Wix 安装程序项目，用于生成插件的 msi 安装文件。
