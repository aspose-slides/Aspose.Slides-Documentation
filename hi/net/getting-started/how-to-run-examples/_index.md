## **सॉफ़्टवेयर आवश्यकताएँ**
उदाहरणों को डाउनलोड करने और चलाने से पहले, कृपया जांचें और पुष्टि करें कि आपका सेटअप इन आवश्यकताओं को पूरा करता है: 

- Visual Studio 2010 या उससे आगे।
- Visual Studio में NuGet पैकेज मैनेजर स्थापित है। सुनिश्चित करें कि Visual Studio में नवीनतम NuGet API संस्करण स्थापित है। 

NuGet पैकेज मैनेजर को स्थापित करने के निर्देशों के लिए, इस पृष्ठ पर जाएँ: https://docs.microsoft.com/en-us/nuget/install-nuget-client-tools

1. **Tools** > **Options** > **NuGet Package Manager** पर जाएँ।

1. **NuGet Package Manager** को विस्तारित करें (उस पर दो बार क्लिक करके) और फिर **Package Sources** चुनें। 

1. जांचें और पुष्टि करें कि nuget.org पैरामीटर चयनित है। 

उदाहरण प्रोजेक्ट NuGet Automatic Package Restore सुविधा का उपयोग करता है, इसलिए आपको एक सक्रिय इंटरनेट कनेक्शन की आवश्यकता है। 

यदि आपके पास उस मशीन पर सक्रिय इंटरनेट कनेक्शन नहीं है जहाँ आप उदाहरण चलाने की योजना बना रहे हैं, तो कृपया [Installation](https://docs.aspose.com/slides/hi/net/installation/) देखें और (हाथ से) Aspose.Slides.dll के लिए रेफ़रेंस जोड़ें। 
## **GitHub से Aspose.Slides डाउनलोड करें**
सभी Aspose.Slides for .NET उदाहरणों की मेजबानी [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-.NET) पर की गई है।

आप अपने पसंदीदा GitHub क्लाइंट का उपयोग करके रिपॉज़िटरी को क्लोन कर सकते हैं या ZIP फ़ाइल को [यहाँ](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/master.zip) डाउनलोड कर सकते हैं।

1. यदि आप ZIP फ़ाइल डाउनलोड करते हैं, तो आपको उसकी सामग्री को अपने कंप्यूटर पर किसी फ़ोल्डर में निकालना होगा। 

सभी उदाहरण **Examples** फ़ोल्डर में संग्रहीत हैं।

एक C# Visual Studio सॉल्यूशन फ़ाइल मौजूद है। प्रोजेक्ट्स Visual Studio 2013 में बनाए गए हैं, लेकिन सॉल्यूशन फ़ाइलें Visual Studio 2010 SP1 और उसके बाद के संस्करणों के साथ संगत हैं।

2. Visual Studio में सॉल्यूशन फ़ाइल खोलें और प्रोजेक्ट बनाएं।

पहली बार चलाने पर, निर्भरताएँ स्वचालित रूप से NuGet के माध्यम से डाउनलोड हो जाती हैं।

**Examples** के मूल फ़ोल्डर में स्थित **Data** फ़ोल्डर में C# उदाहरणों में उपयोग की जाने वाली इनपुट फ़ाइलें हैं। आपको उदाहरण प्रोजेक्ट के साथ **Data** फ़ोल्डर डाउनलोड करना होगा।

3. RunExamples.cs फ़ाइल खोलें। सभी उदाहरण यहां से बुलाए जाते हैं।

4. प्रोजेक्ट के भीतर उन उदाहरणों की टिप्पणी हटाएँ जिन्हें आप चलाना चाहते हैं।

यदि आपको सेटअप करने या उदाहरण चलाने में समस्याएँ आती हैं, तो कृपया हमारे फ़ोरम का उपयोग करके संपर्क करें।
## **योगदान दें**
आप एक उदाहरण जोड़कर या सुधार कर परियोजना में योगदान दे सकते हैं। रिपॉजिटरी में सभी उदाहरण और शॉकेस प्रोजेक्ट ओपन-सोर्स हैं, इसलिए आप (और अन्य लोग) इन्हें एप्लिकेशन में स्वतंत्र रूप से उपयोग कर सकते हैं।

योगदान देने के लिए, आप रिपॉजिटरी को फोर्क कर सकते हैं, स्रोत कोड को संपादित कर सकते हैं, और एक पुल रिक्वेस्ट बना सकते हैं। हम बदलावों की समीक्षा करेंगे। यदि वे उपयोगी पाए जाते हैं, तो हम उन्हें रिपॉजिटरी में जोड़ देंगे।
