## **OpenXML क्या है?**
OpenXML SDK, Microsoft का .NET SDK है जो Microsoft Office 2007 और उससे आगे के फ़ाइल फ़ॉर्मैट्स को प्रोसेस करने के लिए है। ये फ़ाइल फ़ॉर्मैट मूलतः ज़िप्ड पैकेज होते हैं और Open XML SDK for Office, Open XML पैकेज और पैकेज के भीतर मौजूद Open XML स्कीमा तत्वों को नियंत्रित करने का काम आसान बनाता है। Open XML SDK की क्लासेस उन कई सामान्य कार्यों को संलग्न करती हैं जिन्हें डेवलपर्स Open XML पैकेज पर करते हैं, जिससे आप केवल कुछ कोड लाइनों से जटिल ऑपरेशन कर सकते हैं।

जब आप Open XML SDK 2.5 स्थापित कर लेते हैं, तो Visual Studio में अपने मौजूदा प्रोजेक्ट या एप्लिकेशन को खोलें, या नया प्रोजेक्ट या एप्लिकेशन बनाएँ। फिर, अपने प्रोजेक्ट या एप्लिकेशन में, निम्नलिखित घटकों के रेफ़रेंसेज़ जोड़ें:
- DocumentFormat.OpenXml
- WindowsBase
## **Aspose.Slides for .NET**
Aspose.Slides for .NET एक अनोखा PowerPoint® प्रबंधन घटक है जो .NET एप्लिकेशन को Microsoft PowerPoint का उपयोग किए बिना PowerPoint दस्तावेज़ों को पढ़ने, लिखने और संशोधित करने में समर्थ बनाता है।

Aspose.Slides for .NET पहला और एकमात्र .NET घटक है जो आपके अपने एप्लिकेशन में PowerPoint दस्तावेज़ों को प्रबंधित करने की कार्यक्षमता प्रदान करता है। सभी Aspose .NET घटकों की तरह, Aspose.Slides प्रबंधित C# में लिखा गया है, अत्यधिक किफायती है और अत्यंत तेज़ है।

## **परिचय**
फ़ाइल फ़ॉर्मैट लाइब्रेरी खोजने वाले डेवलपर्स को बाजार में उपलब्ध लाइब्रेरीज़ की सुविधाओं को देखना पड़ सकता है। मुख्यतः, विकल्प चुनते समय दो कारकों पर विचार किया जाता है:
- सुविधाएँ
- लाइब्रेरी उपयोग करने में आवश्यक प्रयास

यदि आप एक .NET डेवलपर हैं जो तेज़, आसान और हल्की फ़ाइल फ़ॉर्मैट लाइब्रेरी की तलाश में हैं फ़ाइलों को प्रोसेस करने के लिए, तो आपको VSTO (Visual Studio Tool for Office) की तुलना Aspose से करनी पड़ सकती है। इस मामले में, यह प्रोजेक्ट आपको दोनों लाइब्रेरीज़ की सुविधाओं और कोड की तुलना करने में मदद करेगा।

## **समर्थन, विस्तार और योगदान**
### **समर्थन**
हम मुफ्त समर्थन प्रदान करते हैं। हमारा उत्पाद उपयोग करने वाला कोई भी व्यक्ति, चाहे उसने इसे खरीदा हो या मूल्यांकन के रूप में उपयोग कर रहा हो, हमारे पूरे ध्यान और सम्मान का हकदार है।

आप निम्नलिखित प्लेटफ़ॉर्म का उपयोग करके Aspose .NET for OpenXML से संबंधित किसी भी समस्या या सुझाव को लॉग कर सकते हैं:
- [Github](http://goo.gl/UHkCJe)
- [Sourceforge](http://goo.gl/f8zdLU)
- [Bitbucket](http://goo.gl/1Eo1DQ)
### **विस्तार और योगदान**
Aspose.Slides for .NET और OpenXML स्रोत कोड के लिए तुलना कोड नीचे सूचीबद्ध प्रमुख सामाजिक कोडिंग वेबसाइटों पर उपलब्ध है। डेवलपर्स को सलाह दी जाती है कि वे अपनी आवश्यकताओं के अनुसार OpenXML से Aspose.Slides में माइग्रेट करने के लिए स्रोत कोड डाउनलोड करें।

कृपया शुरू करने के लिए इन सरल चरणों का पालन करें
- स्रोत कोड डाउनलोड/क्लोन करें।
- Visual Studio 2010 खोलें और File > Open Project चुनें।
- उस नवीनतम स्रोत कोड को ब्राउज़ करें जिसे आपने डाउनलोड किया है और .sln फ़ाइल खोलें।
