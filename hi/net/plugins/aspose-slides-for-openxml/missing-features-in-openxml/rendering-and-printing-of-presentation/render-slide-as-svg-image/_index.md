SVG—Scalable Vector Graphics का संक्षिप्त रूप है—एक मानक ग्राफ़िक प्रकार या फ़ॉर्मेट है जिसका उपयोग दो‑आयामी छवियों को रेंडर करने के लिए किया जाता है। SVG छवियों को XML में वेक्टर के रूप में संग्रहीत करता है, जिसमें उनके व्यवहार या रूप‑रंग को परिभाषित करने वाले विवरण होते हैं।

SVG कुछ ही छवि फ़ॉर्मेट्स में से एक है जो इन पहलुओं में अत्यंत उच्च मानकों को पूरा करता है: स्केलेबिलिटी, इंटरएक्टिविटी, प्रदर्शन, एक्सेसेबिलिटी, प्रोग्रामेबिलिटी और अन्य। इन कारणों से, यह वेब विकास में सामान्यतः उपयोग किया जाता है।

आप निम्नलिखित स्थितियों में SVG फ़ाइलों का उपयोग कर सकते हैं:

- जब आप अपनी प्रस्तुति को बहुत बड़े फ़ॉर्मेट में प्रिंट करने की योजना बनाते हैं। SVG छवियों को किसी भी रिज़ॉल्यूशन या स्तर पर स्केल किया जा सकता है। आप आवश्यकतानुसार SVG छवियों का आकार बदल सकते हैं बिना गुणवत्ता खोए।
- जब आप अपनी स्लाइड्स से चार्ट और ग्राफ़ को विभिन्न माध्यमों या प्लेटफ़ॉर्म पर उपयोग करने का इरादा रखते हैं। अधिकांश रीडर SVG फ़ाइलों को समझ सकते हैं।
- जब आपको छवियों के सबसे छोटे संभावित आकार की आवश्यकता होती है। SVG फ़ाइलें आमतौर पर अन्य फ़ॉर्मेट की उच्च‑रिज़ॉल्यूशन समकक्षों से छोटी होती हैं, विशेषकर उन फ़ॉर्मेट्स की जो बिटमैप (JPEG या PNG) पर आधारित हैं।

Aspose.Slides for .NET आपको अपनी प्रस्तुतियों की स्लाइड्स को **SVG** छवियों के रूप में एक्सपोर्ट करने की अनुमति देता है। किसी भी स्लाइड से SVG छवि उत्पन्न करने के लिए, निम्न करें:

- Presentation क्लास का एक इंस्टेंस बनाएँ।
- प्रस्तुति में सभी स्लाइड्स पर इटरेट करें।
- प्रत्येक स्लाइड को FileStream के माध्यम से उसकी स्वयं की SVG फ़ाइल में लिखें।

{{% alert color="primary" %}} 
आप हमारे [नि:शुल्क वेब एप्लिकेशन](https://products.aspose.app/slides/hi/conversion/ppt-to-svg) को आज़मा सकते हैं, जिसमें हमने Aspose.Slides for .NET से PPT को SVG में बदलने की सुविधा लागू की है।
{{% /alert %}} 

C# में यह नमूना कोड दर्शाता है कि Aspose.Slides का उपयोग करके PPT को SVG में कैसे परिवर्तित किया जाए:

``` csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    for (var index = 0; index < pres.Slides.Count; index++)
    {
        ISlide slide = pres.Slides[index];

        using (FileStream fileStream = new FileStream($"slide-{index}.svg", FileMode.Create, FileAccess.Write))
        {
            slide.WriteAsSvg(fileStream);   
        }
    }
}
```
