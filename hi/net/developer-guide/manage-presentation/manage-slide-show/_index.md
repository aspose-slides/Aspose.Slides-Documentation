## **परिचय**

Microsoft PowerPoint में, **Slide Show** सेटिंग्स पेशेवर प्रस्तुतियों को तैयार करने और देने के लिए एक मुख्य टूल हैं। इस अनुभाग की सबसे महत्वपूर्ण सुविधाओं में से एक **Set Up Show** है, जो आपको प्रस्तुति को विशेष परिस्थितियों और दर्शकों के अनुसार अनुकूलित करने की अनुमति देती है, जिससे लचीलापन और सुविधा सुनिश्चित होती है। इस फ़ीचर के साथ, आप शो प्रकार चुन सकते हैं (जैसे, वक्ता द्वारा प्रस्तुत, व्यक्तिगत रूप से ब्राउज़ किया गया, या कियोस्क पर ब्राउज़ किया गया), लूपिंग सक्षम या अक्षम कर सकते हैं, प्रदर्शन के लिए विशिष्ट स्लाइड्स चुन सकते हैं, और टाइमिंग का उपयोग कर सकते हैं। तैयारी का यह चरण आपकी प्रस्तुति को अधिक प्रभावी और पेशेवर बनाने में महत्वपूर्ण है।

`SlideShowSettings` `Presentation` वर्ग का एक प्रॉपर्टी है, जिसका प्रकार `SlideShowSettings` है, जो PowerPoint प्रस्तुति में स्लाइड शो सेटिंग्स को प्रबंधित करने की सुविधा देता है। इस लेख में, हम इस प्रॉपर्टी का उपयोग करके स्लाइड शो सेटिंग्स के विभिन्न पहलुओं को कैसे कॉन्फ़िगर और नियंत्रित किया जा सकता है, इसे जानेंगे।

## **शो प्रकार चुनें**

`SlideShowSettings.SlideShowType` स्लाइड शो के प्रकार को निर्धारित करता है, जो निम्नलिखित क्लासों में से एक का उदाहरण हो सकता है: `PresentedBySpeaker`, `BrowsedByIndividual`, या `BrowsedAtKiosk`। इस प्रॉपर्टी का उपयोग करके आप विभिन्न उपयोग परिदृश्यों के लिए प्रस्तुति को अनुकूलित कर सकते हैं, जैसे स्वचालित कियोस्क या मैन्युअल प्रस्तुतियाँ।

नीचे दिया गया कोड उदाहरण नई प्रस्तुति बनाता है और शो प्रकार को “Browsed by an individual” पर सेट करता है, बिना स्क्रॉलबार दिखाए।

```cs
using var presentation = new Presentation();

var showType = new BrowsedByIndividual
{
    ShowScrollbar = false
};

presentation.SlideShowSettings.SlideShowType = showType;

presentation.Save("output.pptx", SaveFormat.Pptx);
```

## **शो विकल्प सक्षम करें**

`SlideShowSettings.Loop` निर्धारित करता है कि स्लाइड शो को मैन्युअल रूप से बंद करने तक लूप में दोहराया जाए या नहीं। यह निरंतर चलने वाली स्वचालित प्रस्तुतियों के लिए उपयोगी है। `SlideShowSettings.ShowNarration` निर्धारित करता है कि स्लाइड शो के दौरान आवाज़ीय विवरण चलाए जाएँ या नहीं। यह उन स्वचालित प्रस्तुतियों के लिए उपयोगी है जिनमें दर्शकों के लिए आवाज़ मार्गदर्शन शामिल है। `SlideShowSettings.ShowAnimation` निर्धारित करता है कि स्लाइड वस्तुओं में जोड़े गए एनीमेशन चलाए जाएँ या नहीं। यह प्रस्तुति के पूर्ण दृश्य प्रभाव को प्रदान करने में मदद करता है।

नीचे दिया गया कोड उदाहरण नई प्रस्तुति बनाता है और स्लाइड शो को लूप में चलाता है।

```cs
using var presentation = new Presentation();

presentation.SlideShowSettings.Loop = true;

presentation.Save("output.pptx", SaveFormat.Pptx);
```

## **दिखाने के लिए स्लाइड चुनें**

`SlideShowSettings.Slides` प्रॉपर्टी आपको प्रस्तुति के दौरान प्रदर्शित की जाने वाली स्लाइड्स की एक रेंज चुनने की अनुमति देती है। यह तब उपयोगी है जब आप पूरी प्रस्तुति के बजाय केवल भाग को दिखाना चाहते हैं। नीचे दिया गया कोड उदाहरण नई प्रस्तुति बनाता है और स्लाइड रेंज को `2` से `9` तक सेट करता है।

```cs
using var presentation = new Presentation();

var slideRange = new SlidesRange 
{
    Start = 2,
    End = 9
};

presentation.SlideShowSettings.Slides = slideRange;

presentation.Save("output.pptx", SaveFormat.Pptx);
```

## **आगे की स्लाइड्स का उपयोग करें**

`SlideShowSettings.UseTimings` प्रॉपर्टी आपको प्रत्येक स्लाइड के लिए पूर्वनिर्धारित टाइमिंग के उपयोग को सक्षम या अक्षम करने की अनुमति देती है। यह पूर्वनिर्धारित डिस्प्ले अवधि के साथ स्वचालित रूप से स्लाइड्स दिखाने के लिए उपयोगी है। नीचे दिया गया कोड उदाहरण नई प्रस्तुति बनाता है और टाइमिंग के उपयोग को अक्षम करता है।

```cs
using var presentation = new Presentation();

presentation.SlideShowSettings.UseTimings = false;

presentation.Save("output.pptx", SaveFormat.Pptx);
```

## **मीडिया नियंत्रण दिखाएँ**

`SlideShowSettings.ShowMediaControls` प्रॉपर्टी निर्धारित करती है कि मल्टीमीडिया सामग्री (जैसे वीडियो या ऑडियो) चलाते समय स्लाइड शो के दौरान मीडिया नियंत्रण (जैसे प्ले, पॉज़, स्टॉप) प्रदर्शित किए जाएँ या नहीं। यह तब उपयोगी है जब आप प्रस्तुतकर्ता को प्रस्तुति के दौरान मीडिया प्लेबैक पर नियंत्रण देना चाहते हैं।

नीचे दिया गया कोड उदाहरण नई प्रस्तुति बनाता है और मीडिया नियंत्रण को प्रदर्शित करने के लिए सक्षम करता है।

```cs
using var presentation = new Presentation();

presentation.SlideShowSettings.ShowMediaControls = true;

presentation.Save("output.pptx", SaveFormat.Pptx);
```

## **FAQ**

**क्या मैं प्रस्तुति को इस प्रकार सहेज सकता हूँ कि वह सीधे स्लाइड शो मोड में खुल जाए?**

हाँ। फ़ाइल को PPSX या PPSM के रूप में सहेजें; ये फ़ॉर्मेट PowerPoint में खुले पर सीधे स्लाइड शो मोड में लॉन्च होते हैं। Aspose.Slides में, निर्यात के दौरान उपयुक्त सहेजने के फ़ॉर्मेट का चयन करें [/slides/hi/net/save-presentation/]।

**क्या मैं व्यक्तिगत स्लाइड्स को शो से बाहर कर सकता हूँ बिना उन्हें फ़ाइल से हटाए?**

हाँ। स्लाइड को [Hidden](https://reference.aspose.com/slides/hi/net/aspose.slides/slide/hidden/) के रूप में चिह्नित करें। छिपी हुई स्लाइड्स प्रस्तुति में बनी रहती हैं लेकिन स्लाइड शो के दौरान प्रदर्शित नहीं होतीं।

**क्या Aspose.Slides स्लाइड शो चला सकता है या स्क्रीन पर लाइव प्रस्तुति को नियंत्रित कर सकता है?**

नहीं। Aspose.Slides प्रस्तुति फ़ाइलों को संपादित, विश्लेषण और रूपांतरण करता है; वास्तविक प्लेबैक PowerPoint जैसे व्यूअर एप्लिकेशन द्वारा संभाला जाता है।
