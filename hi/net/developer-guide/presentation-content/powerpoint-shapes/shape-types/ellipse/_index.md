## **परिचय**

यह लेख Aspose.Slides का उपयोग करके PowerPoint स्लाइड्स में अंडाकार आकार जोड़ने का तरीका दिखाता है। यह सरल अंडाकार बनाना, स्वरूपित अंडाकार बनाना, और अद्यतन प्रस्तुति को PPTX फ़ाइल के रूप में सहेजना शामिल करता है। यह अंडाकार की स्थिति और आकार के साथ काम करना, स्टैकिंग क्रम को नियंत्रित करना, और एनीमेशन प्रभाव लागू करने जैसे संबंधित प्रश्नों को भी छूता है।

## **अंडाकार बनाएँ**
एक चयनित स्लाइड में सरल अंडाकार जोड़ने के लिए, नीचे दिए गए चरणों का पालन करें:

1. [Presentation ](https://reference.aspose.com/slides/hi/net/aspose.slides/presentation) क्लास का एक इंस्टेंस बनाएं
1. इंडेक्स का उपयोग करके स्लाइड का संदर्भ प्राप्त करें
1. IShapes ऑब्जेक्ट द्वारा प्रकट किए गए AddAutoShape मेथड का उपयोग करके Ellipse प्रकार का AutoShape जोड़ें
1. संशोधित प्रस्तुति को PPTX फ़ाइल के रूप में सहेजें

नीचे दिए गए उदाहरण में, हमने पहली स्लाइड में एक अंडाकार जोड़ा है।

```c#
// PPTX को दर्शाने वाली Presentation क्लास का उदाहरण बनाएँ
using (Presentation pres = new Presentation())
{

    // पहले स्लाइड प्राप्त करें
    ISlide sld = pres.Slides[0];

    // अंडाकार प्रकार का ऑटोशेप जोड़ें
    sld.Shapes.AddAutoShape(ShapeType.Ellipse, 50, 150, 150, 50);

    //डिस्क पर PPTX फ़ाइल लिखें
    pres.Save("EllipseShp1_out.pptx", SaveFormat.Pptx);
}
```



## **स्वरूपित अंडाकार बनाएँ**
एक बेहतर स्वरूपित अंडाकार स्लाइड में जोड़ने के लिए, नीचे दिए गए चरणों का पालन करें:

1. [Presentation ](https://reference.aspose.com/slides/hi/net/aspose.slides/presentation) क्लास का एक इंस्टेंस बनाएं।
1. इंडेक्स का उपयोग करके स्लाइड का संदर्भ प्राप्त करें।
1. IShapes ऑब्जेक्ट द्वारा प्रकट किए गए AddAutoShape मेथड का उपयोग करके Ellipse प्रकार का AutoShape जोड़ें।
1. अंडाकार का Fill Type को Solid सेट करें।
1. IShape ऑब्जेक्ट से जुड़ी FillFormat ऑब्जेक्ट द्वारा प्रकट SolidFillColor.Color प्रॉपर्टी का उपयोग करके अंडाकार का रंग सेट करें।
1. अंडाकार की रेखाओं का रंग सेट करें।
1. अंडाकार की रेखाओं की चौड़ाई सेट करें।
1. संशोधित प्रस्तुति को PPTX फ़ाइल के रूप में सहेजें।

नीचे दिए गए उदाहरण में, हमने प्रस्तुति की पहली स्लाइड में एक स्वरूपित अंडाकार जोड़ा है।

```c#
// PPTX का प्रतिनिधित्व करने वाली Presentation क्लास का उदाहरण बनाएं
using (Presentation pres = new Presentation())
{

    // पहले स्लाइड प्राप्त करें
    ISlide sld = pres.Slides[0];

    // अंडाकार प्रकार का ऑटोशेप जोड़ें
    IShape shp = sld.Shapes.AddAutoShape(ShapeType.Ellipse, 50, 150, 150, 50);

    // अंडाकार आकार पर कुछ स्वरूपण लागू करें
    shp.FillFormat.FillType = FillType.Solid;
    shp.FillFormat.SolidFillColor.Color = Color.Chocolate;

    // अंडाकार की रेखा पर कुछ स्वरूपण लागू करें
    shp.LineFormat.FillFormat.FillType = FillType.Solid;
    shp.LineFormat.FillFormat.SolidFillColor.Color = Color.Black;
    shp.LineFormat.Width = 5;

    //डिस्क पर PPTX फ़ाइल लिखें
    pres.Save("EllipseShp2_out.pptx", SaveFormat.Pptx);
}
```

## **अक्सर पूछे जाने वाले प्रश्न**

**मैं स्लाइड की इकाइयों के संबंध में अंडाकार की सटीक स्थिति और आकार कैसे सेट कर सकता हूँ?**

निर्देशांक और आकार आमतौर पर **पॉइंट्स** में निर्दिष्ट किए जाते हैं। विश्वसनीय परिणामों के लिये, अपने गणनाओं को स्लाइड के आकार पर आधार बनाएं और मान असाइन करने से पहले आवश्यक मिलीमीटर या इंच को पॉइंट्स में परिवर्तित करें।

**मैं अंडाकार को अन्य वस्तुओं के ऊपर या नीचे कैसे रख सकता हूँ (स्टैकिंग क्रम नियंत्रित करना)?**

ऑब्जेक्ट के ड्रॉइंग क्रम को सामने लाकर या पीछे भेजकर समायोजित करें। इससे अंडाकार अन्य वस्तुओं के ऊपर ओवरलैप कर सकता है या उनके नीचे की वस्तुओं को उजागर कर सकता है।

**मैं अंडाकार की उपस्थिति या ज़ोर देने के लिए एनीमेशन कैसे लागू करूँ?**

[Apply](/slides/hi/net/shape-animation/) प्रवेश, ज़ोर या निकास प्रभाव को आकार पर लागू करें, और ट्रिगर व टाइमिंग को कॉन्फ़िगर करें ताकि एनीमेशन कब और कैसे चले, तय किया जा सके।
