## **परिचय**

यह लेख Aspose.Slides का उपयोग करके प्रस्तुतियों से फ़्लैश ऑब्जेक्ट्स निकालने का तरीका समझाता है। यह दिखाता है कि स्लाइड के कंट्रोल्स संग्रह में नाम से फ़्लैश कंट्रोल कैसे खोजें और एम्बेडेड SWF ऑब्जेक्ट डेटा के साथ कैसे कार्य करें।

## **प्रस्तुतियों से फ़्लैश ऑब्जेक्ट्स निकालना**
Aspose.Slides for .NET प्रस्तुतियों से फ़्लैश ऑब्जेक्ट्स निकालने की सुविधा प्रदान करता है। आप नाम द्वारा फ़्लैश कंट्रोल तक पहुँच सकते हैं और इसे प्रस्तुतियों से निकाल सकते हैं तथा संग्रहीत SWF ऑब्जेक्ट डेटा को शामिल कर सकते हैं।

```c#
using (Presentation pres = new Presentation("withFlash.pptm"))
{
    IControlCollection controls = pres.Slides[0].Controls;
    Control flashControl = null;
    foreach (IControl control in controls)
    {
        if (control.Name == "ShockwaveFlash1")
        {
            flashControl = (Control)control;
        }
    }
}
```

## **अक्सर पूछे जाने वाले प्रश्न**

**फ़्लैश सामग्री निकालते समय कौन से प्रस्तुति फ़ॉर्मेट समर्थित हैं?**

[Aspose.Slides supports](/slides/hi/net/supported-file-formats/) मुख्य PowerPoint फ़ॉर्मेट जैसे PPT और PPTX को, क्योंकि यह इन कंटेनरों को लोड कर सकता है और उनके कंट्रोल्स तक पहुँच सकता है, जिसमें फ़्लैश‑संबंधित ActiveX तत्व शामिल हैं।

**क्या मैं फ़्लैश वाले प्रस्तुति को HTML5 में बदल सकता हूँ और फ़्लैश इंटरएक्टिविटी को संरक्षित रख सकता हूँ?**

नहीं। Aspose.Slides SWF सामग्री को निष्पादित नहीं करता या उसकी इंटरएक्टिविटी को बदलता नहीं है। जबकि [HTML](/slides/hi/net/convert-powerpoint-to-html/)/[HTML5](/slides/hi/net/export-to-html5/) निर्यात को समर्थन मिलता है, फ़्लैश आधुनिक ब्राउज़रों में समर्थन समाप्त होने के कारण नहीं चलेगा। अनुशंसित तरीका यह है कि निर्यात से पहले फ़्लैश को वीडियो या HTML5 एनीमेशन जैसे विकल्पों से बदल दिया जाए।

**सुरक्षा के दृष्टिकोण से, क्या Aspose.Slides प्रस्तुति पढ़ते समय SWF फ़ाइलें निष्पादित करता है?**

नहीं। Aspose.Slides फ़्लैश को फ़ाइल में एम्बेडेड बाइनरी डेटा के रूप में व्यवहार करता है और प्रसंस्करण के दौरान SWF सामग्री को निष्पादित नहीं करता।

**मैं उन प्रस्तुतियों को कैसे संभालूँ जो OLE के माध्यम से फ़्लैश के साथ अन्य एम्बेडेड फ़ाइलें भी शामिल करती हैं?**

Aspose.Slides [extracting embedded OLE objects](/slides/hi/net/manage-ole/) को समर्थन देता है, इसलिए आप सभी संबंधित एम्बेडेड सामग्री को एक ही पास में प्रोसेस कर सकते हैं, फ़्लैश कंट्रोल और अन्य OLE‑एम्बेडेड दस्तावेज़ों को साथ‑साथ संभालते हुए।
