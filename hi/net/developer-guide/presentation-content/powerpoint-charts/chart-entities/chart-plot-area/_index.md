## **अवलोकन**

यह लेख Aspose.Slides में चार्ट के प्लॉट एरिया के साथ काम करने का तरीका दर्शाता है। यह चार्ट लेआउट को वैध करके और फिर उसकी X, Y, चौड़ाई, और ऊँचाई मान पढ़कर प्लॉट एरिया की वास्तविक स्थिति और आकार प्राप्त करने की विधि समझाता है।

यह यह भी दर्शाता है कि जब लेआउट को मैन्युअल रूप से सेट किया गया हो तो प्लॉट एrea के लेआउट मोड को कैसे कॉन्फ़िगर किया जाए, `LayoutTargetType` का उपयोग करके यह निर्धारित किया जाए कि प्लॉट एरिया को उसके आंतरिक क्षेत्र द्वारा या अक्षों और अक्ष लेबल सहित बाहरी क्षेत्र द्वारा गणना किया जाए।

## **चार्ट प्लॉट एरिया की चौड़ाई और ऊँचाई प्राप्त करें**
Aspose.Slides for .NET एक सरल API प्रदान करता है।

1. एक नया [Presentation](https://reference.aspose.com/slides/hi/net/aspose.slides/presentation) क्लास का उदाहरण बनाएं।
1. पहली स्लाइड तक पहुंचें।
1. डिफ़ॉल्ट डेटा के साथ चार्ट जोड़ें।
1. वास्तविक मान प्राप्त करने के लिए IChart.ValidateChartLayout() मेथड को कॉल करें।
1. चार्ट तत्व का वास्तविक X स्थान (बायां) चार्ट के बाएँ ऊपर कोने के सापेक्ष प्राप्त करता है।
1. चार्ट तत्व का वास्तविक शीर्ष चार्ट के बाएँ ऊपर कोने के सापेक्ष प्राप्त करता है।
1. चार्ट तत्व की वास्तविक चौड़ाई प्राप्त करता है।
1. चार्ट तत्व की वास्तविक ऊँचाई प्राप्त करता है।

```c#
using (Presentation pres = new Presentation("test.Pptx"))
{
    Chart chart = (Chart)pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 100, 100, 500, 350);
    chart.ValidateChartLayout();

    double x = chart.PlotArea.ActualX;
    double y = chart.PlotArea.ActualY;
    double w = chart.PlotArea.ActualWidth;
    double h = chart.PlotArea.ActualHeight;
	
	// चार्ट के साथ प्रस्तुति सहेजें
	pres.Save("Chart_out.pptx", SaveFormat.Pptx);
}
```

## **चार्ट प्लॉट एरिया का लेआउट मोड सेट करें**
Aspose.Slides for .NET चार्ट प्लॉट एरिया के लेआउट मोड को सेट करने के लिए एक सरल API प्रदान करता है। **LayoutTargetType** प्रॉपर्टी **ChartPlotArea** और **IChartPlotArea** क्लासों में जोड़ी गई है। यदि प्लॉट एरिया का लेआउट मैन्युअल रूप से परिभाषित किया गया हो, तो यह प्रॉपर्टी यह निर्धारित करती है कि प्लॉट एरिया को उसके अंदर (अक्ष और अक्ष लेबल को शामिल नहीं) या बाहर (अक्ष और अक्ष लेबल सहित) लेआउट किया जाए। दो संभावित मान **LayoutTargetType** enum में परिभाषित हैं।

- **LayoutTargetType.Inner** - निर्दिष्ट करता है कि प्लॉट एरिया का आकार प्लॉट एरिया के आकार को निर्धारित करेगा, जिसमें टिक मार्क और अक्ष लेबल शामिल नहीं होंगे।
- **LayoutTargetType.Outer** - निर्दिष्ट करता है कि प्लॉट एरिया का आकार प्लॉट एरिया, टिक मार्क और अक्ष लेबल के आकार को निर्धारित करेगा।

नीचे नमूना कोड दिया गया है।

```c#
using (Presentation presentation = new Presentation())
{
    ISlide slide = presentation.Slides[0];
    IChart chart = slide.Shapes.AddChart(ChartType.ClusteredColumn, 20, 100, 600, 400);
    chart.PlotArea.AsILayoutable.X = 0.2f;
    chart.PlotArea.AsILayoutable.Y = 0.2f;
    chart.PlotArea.AsILayoutable.Width = 0.7f;
    chart.PlotArea.AsILayoutable.Height = 0.7f;
    chart.PlotArea.LayoutTargetType = LayoutTargetType.Inner;

    presentation.Save("SetLayoutMode_outer.pptx", SaveFormat.Pptx);
}
```

## **अक्सर पूछे जाने वाले प्रश्न**

**ActualX, ActualY, ActualWidth, और ActualHeight किस इकाइयों में लौटाए जाते हैं?**  
पॉइंट्स में; 1 इंच = 72 पॉइंट्स। ये Aspose.Slides समन्वय इकाइयाँ हैं।

**सामग्री के संदर्भ में प्लॉट एरिया और चार्ट एरिया में कैसे अंतर है?**  
प्लॉट एरिया डेटा ड्रॉइंग क्षेत्र है (सीरीज़, ग्रिडलाइन, ट्रेंडलाइन आदि); जबकि चार्ट एरिया में आसपास के तत्व (शीर्षक, लीजेंड आदि) शामिल होते हैं। 3D चार्ट्स में, प्लॉट एरिया में दीवारें/फ़्लोर और अक्ष भी शामिल होते हैं।

**जब लेआउट मैन्युअल हो तो प्लॉट एरिया के X, Y, Width, और Height कैसे व्याख्या किए जाते हैं?**  
ये चार्ट के कुल आकार के फ्रैक्शन (0–1) होते हैं; इस मोड में, ऑटो‑पोजिशनिंग अक्षम हो जाता है और आपने जो फ़्रैक्शन सेट किए हैं, उनका उपयोग किया जाता है।

**लेजेंड जोड़ने/हिलाने के बाद प्लॉट एरिया की स्थिति क्यों बदल गई?**  
लेजेंड चार्ट एरिया में प्लॉट एरिया के बाहर स्थित होता है लेकिन लेआउट और उपलब्ध स्थान को प्रभावित करता है, इसलिए ऑटो‑पोजिशनिंग सक्रिय होने पर प्लॉट एरिया स्थानांतरित हो सकता है। (यह PowerPoint चार्ट्स के लिए सामान्य व्यवहार है।)
