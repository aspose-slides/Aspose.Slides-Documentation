## **अवलोकन**

Aspose.Slides आपको प्रस्तुति से एक चार्ट को छवि के रूप में निर्यात करने की अनुमति देता है। यह लेख दिखाता है कि चार्ट से छवि कैसे प्राप्त करें और उसे सहेजें, जो तब उपयोगी होता है जब आपको PowerPoint प्रस्तुति के बाहर चार्ट दृश्यों को पुन: उपयोग करना हो।

बुनियादी छवि निर्यात कार्यप्रवाह के अलावा, लेख सामान्य निर्यात‑संबंधी प्रश्नों को भी संबोधित करता है, जिसमें चार्ट सामग्री को SVG में सहेजना, रेंडरिंग विकल्पों के माध्यम से आउटपुट आकार को नियंत्रित करना, लेबल और लेजेंड की उपस्थिति को संरक्षित करने के लिए फॉन्ट लोड करना, और रेंडरिंग के दौरान मूल प्रस्तुति फ़ॉर्मेटिंग जैसे थीम, शैलियाँ, भराव और प्रभाव को बनाए रखना शामिल है।

## **चार्ट छवि प्राप्त करें**
Aspose.Slides for .NET विशिष्ट चार्ट की छवि निकालने के लिए समर्थन प्रदान करता है। नीचे एक नमूना उदाहरण दिया गया है।

```c#
using (Presentation presentation = new Presentation("test.pptx"))
{
    ISlide slide = presentation.Slides[0];
    IChart chart = slide.Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 600, 400);

    using (IImage image = chart.GetImage())
    {
        image.Save("image.png", ImageFormat.Png);
    }
}
```

## **अक्सर पूछे जाने वाले प्रश्न**

**क्या मैं चार्ट को रास्टर छवि के बजाय वेक्टर (SVG) के रूप में निर्यात कर सकता हूँ?**

हाँ। एक चार्ट एक आकार (shape) है, और उसकी सामग्री को SVG में सहेजा जा सकता है using the [shape-to-SVG सहेजने की विधि](https://reference.aspose.com/slides/hi/net/aspose.slides/shape/writeassvg/)।

**मैं निर्यातित चार्ट का सटीक आकार पिक्सेल में कैसे निर्धारित करूँ?**

इमेज‑रेंडरिंग ओवरलोड्स का उपयोग करें जो आपको आकार या स्केल निर्दिष्ट करने की अनुमति देते हैं—लाइब्रेरी दिए गए आयाम/स्केल के साथ वस्तुओं को रेंडर करना समर्थन करती है।

**निर्यात के बाद लेबल और लेजेंड में फॉन्ट गलत दिख रहे हैं तो मुझे क्या करना चाहिए?**

चार्ट रेंडरिंग मीट्रिक और टेक्स्ट उपस्थिति को संरक्षित करने के लिए आवश्यक फॉन्ट को [फ़ॉन्ट लोड करें](/slides/hi/net/custom-font/) via [FontsLoader](https://reference.aspose.com/slides/hi/net/aspose.slides/fontsloader/)।

**क्या निर्यात PowerPoint थीम, शैलियों और प्रभावों का सम्मान करता है?**

हाँ। Aspose.Slides का रेंडरर प्रस्तुति की फ़ॉर्मेटिंग (थीम, शैलियाँ, भराव, प्रभाव) का पालन करता है, इसलिए चार्ट की उपस्थिति संरक्षित रहती है।

**चार्ट छवियों के अलावा उपलब्ध रेंडरिंग/निर्यात क्षमताएँ कहाँ देख सकते हैं?**

आउटपुट लक्ष्यों ([PDF](/slides/hi/net/convert-powerpoint-to-pdf/), [SVG](/slides/hi/net/render-a-slide-as-an-svg-image/), [XPS](/slides/hi/net/convert-powerpoint-to-xps/), [HTML](/slides/hi/net/convert-powerpoint-to-html/), आदि) और संबंधित रेंडरिंग विकल्पों के लिए [API](/slides/hi/net/api)/[दस्तावेज़ीकरण](/slides/hi/net/convert-powerpoint/) के निर्यात अनुभाग देखें।
