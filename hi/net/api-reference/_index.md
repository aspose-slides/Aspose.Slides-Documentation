Aspose.Slides for .NET एक क्लास लाइब्रेरी है जो .NET एप्लिकेशन को Microsoft PowerPoint® या अन्य टूल्स का उपयोग किए बिना प्रेज़ेंटेशन दस्तावेज़ पढ़ने और लिखने में सक्षम बनाती है। Aspose.Slides for .NET टेक्स्ट, शेप्स, टेबल्स और एनीमेशन को प्रबंधित करने, स्लाइड्स में ऑडियो और वीडियो जोड़ने, स्लाइड्स का प्रीव्यू करने, स्लाइड्स को SVG, PDF फ़ॉर्मेट में एक्सपोर्ट करने और अधिक करने की अनुमति देती है।

नवीनतम API रेफ़रेंस [Aspose.Slides for .NET API Reference](https://reference.aspose.com/slides/hi/net) पर पाया जा सकता है।
