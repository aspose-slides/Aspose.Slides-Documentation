## **परिचय**

PowerPoint प्रस्तुतियों में, स्लाइड पृष्ठभूमि कई तत्वों से बन सकती है, जिसमें स्लाइड पृष्ठभूमि छवि, प्रस्तुति थीम, रंग योजना, और मास्टर स्लाइड या लेआउट स्लाइड पर रखे गये ऑब्जेक्ट्स शामिल हैं।

यह लेख Aspose.Slides का उपयोग करके स्लाइड की पूरी पृष्ठभूमि को एक छवि के रूप में निकालने का तरीका दिखाता है। चूँकि इस कार्य के लिए कोई एकल विधि उपलब्ध नहीं है, इसलिए चयनित स्लाइड को एक अस्थायी प्रेजेंटेशन में क्लोन किया जाता है, स्लाइड के आकार को हटाया जाता है, और फिर परिणामी पृष्ठभूमि को छवि में परिवर्तित किया जाता है।

## **स्लाइड की पूरी पृष्ठभूमि प्राप्त करें**

Aspose.Slides for PHP via Java पूरी प्रेजेंटेशन स्लाइड पृष्ठभूमि को छवि के रूप में निकालने के लिए कोई सरल विधि प्रदान नहीं करता, लेकिन आप नीचे दिए गए चरणों का पालन करके यह कर सकते हैं:
1. प्रेजेंटेशन को [प्रेजेंटेशन](https://reference.aspose.com/slides/hi/php-java/aspose.slides/presentation/) क्लास का उपयोग करके लोड करें।
1. प्रेजेंटेशन से स्लाइड आकार प्राप्त करें।
1. एक स्लाइड का चयन करें।
1. एक अस्थायी प्रेजेंटेशन बनाएँ।
1. अस्थायी प्रेजेंटेशन में समान स्लाइड आकार सेट करें।
1. चयनित स्लाइड को अस्थायी प्रेजेंटेशन में क्लोन करें।
1. क्लोन की गई स्लाइड से आकार हटाएँ।
1. क्लोन की गई स्लाइड को छवि में परिवर्तित करें।

निम्नलिखित कोड उदाहरण पूरी प्रेजेंटेशन स्लाइड पृष्ठभूमि को एक छवि के रूप में निकालता है।
```php
$slideIndex = 0;
$imageScale = 1;

$presentation = new Presentation("sample.pptx");

$slideSize = $presentation->getSlideSize()->getSize();
$slide = $presentation->getSlides()->get_Item($slideIndex);

$tempPresentation = new Presentation();

$slideWidth = $slideSize->getWidth();
$slideHeight = $slideSize->getHeight();
$tempPresentation->getSlideSize()->setSize($slideWidth, $slideHeight, SlideSizeScaleType::DoNotScale);

$clonedSlide = $tempPresentation->getSlides()->addClone($slide);
$clonedSlide->getShapes()->clear();

$background = clonedSlide->getImage($imageScale, $imageScale);
$background->save("output->png", ImageFormat::Png);

$tempPresentation->dispose();
$presentation->dispose();
```

## **अक्सर पूछे जाने वाले प्रश्न**

**क्या मास्टर स्लाइड से जटिल ग्रेडिएंट, टेक्सचर, या चित्र भराव परिणामस्वरूप पृष्ठभूमि छवि में संरक्षित रहेंगे?**

हाँ। Aspose.Slides स्लाइड, लेआउट, या मास्टर पर परिभाषित ग्रेडिएंट, चित्र, और टेक्सचर भराव को रेंडर करता है। यदि आपको विरासती मास्टर से लुक अलग करना है, तो निर्यात करने से पहले वर्तमान स्लाइड पर [अपना पृष्ठभूमि सेट करें](/slides/hi/php-java/presentation-background/)।

**क्या मैं परिणामस्वरूप पृष्ठभूमि छवि को सहेजने से पहले उसमें वॉटरमार्क जोड़ सकता हूँ?**

हाँ। आप [एक वॉटरमार्क जोड़ें](/slides/hi/php-java/watermark/) आकार या चित्र को काम कर रहे [स्लाइड की प्रति](/slides/hi/php-java/clone-slides/) (अन्य सामग्री के पीछे रखकर) पर जोड़ सकते हैं और फिर निर्यात करें। यह आपको वॉटरमार्क बेक्ड इन वाली पृष्ठभूमि छवि उत्पन्न करने देता है।

**क्या मैं किसी विशिष्ट लेआउट या मास्टर की पृष्ठभूमि को बिना किसी मौजूदा स्लाइड से जुड़ाए प्राप्त कर सकता हूँ?**

हाँ। इच्छित मास्टर या लेआउट तक पहुंचें, आवश्यक आकार के साथ एक [अस्थायी स्लाइड](/slides/hi/php-java/clone-slides/) पर लागू करें, और उस स्लाइड को निर्यात करके लेआउट या मास्टर से निकाली गई पृष्ठभूमि प्राप्त करें।

**क्या ऐसी कोई लाइसेंसिंग सीमाएँ हैं जो छवि निर्यात को प्रभावित करती हैं?**

रेंडरिंग सुविधाएँ पूरी तरह से एक [वैध लाइसेंस](/slides/hi/php-java/licensing/) के साथ उपलब्ध हैं। मूल्यांकन मोड में, आउटपुट में वॉटरमार्क जैसी सीमाएँ हो सकती हैं। बैच निर्यात चलाने से पहले प्रक्रिया में एक बार लाइसेंस को सक्रिय करें।
