## **अवलोकन**

प्रस्तुति सुलभता सुनिश्चित करती है कि सहायक तकनीकों—जैसे स्क्रीन रीडर, ब्रेल डिस्प्ले, या केवल कीबोर्ड नेविगेशन—का उपयोग करने वाले लोग आपके स्लाइड्स को दृष्टि वाले, माउस‑उपयोगकर्ता दर्शकों की तरह ही प्रभावी रूप से समझ और नेविगेट कर सकें। अच्छी प्रथा स्पष्ट पढ़ने क्रम, सूचनात्मक दृश्यों के लिए सार्थक वैकल्पिक पाठ, पर्याप्त रंग कंट्रास्ट, पठनीय टाइपोग्राफी, वर्णनात्मक लिंक टेक्स्ट, और केवल रंग या स्थिति से अर्थ न देने पर केंद्रित होती है। जब सुलभता को प्रारंभ से ही योजना में शामिल किया जाता है, तो परिणाम एक साफ़ संरचना, अधिक सुसंगत दृश्य, और ऐसी सामग्री होती है जो हर दर्शक तक बिना वर्कअराउंड के पहुँचती है।

## **सजावटी के रूप में चिह्नित करें**

सजावटी के रूप में चिह्नित करना केवल सजावटी दृश्यों को चिन्हित करता है ताकि स्क्रीन रीडर उन्हें छोड़ दें, शोर को कम करें और सार्थक सामग्री पर ध्यान केंद्रित रखें। इसे पृष्ठभूमियों, सजावटी तत्वों और स्पेसर पर लागू करें—कभी भी चार्ट, आइकन या ऐसी छवियों पर नहीं जो जानकारी प्रदान करती हों। Aspose.Slides इस फ़्लैग को पहचान और मान्य करने के लिए उपलब्ध कराता है, जिससे स्वचालित सुलभता जाँच और सफ़ाई संभव होती है।

![सजावटी के रूप में चिह्नित करें](mark_as_decorative.png)

निम्नलिखित कोड नमूना दर्शाता है कि कैसे निर्धारित किया जाए कि कोई आकार सजावटी के रूप में चिह्नित है।

```php
$presentation = new Presentation("sample.pptx");
try {
    $shape = $presentation->getSlides()->get_Item(0)->getShapes()->get_Item(0);
    echo "Is shape decorative: " . ($shape->isDecorative() ? "true" : "false") . "\n";
} finally {
    $presentation->dispose();
}
```
