## **सारांश**

यह लेख Aspose.Slides में एक प्रस्तुति कैसे बनाएं, स्लाइड में सरल सामग्री जोड़ें, और परिणाम को फ़ाइल के रूप में सहेजें, यह दिखाता है। यह यह भी प्रदर्शित करता है कि नई प्रस्तुति कैसे बनाएं और सहेजें, समर्थनयुक्त फ़ॉर्मेट में मौजूदा प्रस्तुति को कैसे खोलें, और उसे किसी अन्य फ़ॉर्मेट में सहेजें। अतिरिक्त रूप से, लेख में फ़ॉर्मेट्स, टेम्प्लेट्स, स्लाइड आकार, इकाइयाँ, मेमोरी उपयोग, थ्रेडिंग, लाइसेंसिंग, डिजिटल हस्ताक्षर, और VBA समर्थन से संबंधित सामान्य प्रश्नों को कवर करने वाला संक्षिप्त FAQ शामिल है।

## **प्रस्तुति बनाएं**

एक चयनित स्लाइड में सरल साधारण रेखा जोड़ने के लिए, नीचे दिए गए चरणों को पालन करें:

1. Presentation क्लास का एक उदाहरण बनाएं।
2. स्लाइड का संदर्भ उसके इंडेक्स का उपयोग करके प्राप्त करें।
3. Shapes ऑब्जेक्ट द्वारा प्रदान किए गए addAutoShape मेथड का उपयोग करके लाइन प्रकार का AutoShape जोड़ें।
4. संशोधित प्रस्तुति को PPTX फ़ाइल के रूप में लिखें।

नीचे दिए गए उदाहरण में, हमने प्रस्तुति की पहली स्लाइड में एक रेखा जोड़ दी है।

```php
  # एक Presentation ऑब्जेक्ट को इनस्टैंशिएट करें जो एक प्रस्तुति फ़ाइल का प्रतिनिधित्व करता है
  $pres = new Presentation();
  try {
    # पहली स्लाइड प्राप्त करें
    $slide = $pres->getSlides()->get_Item(0);
    # लाइन प्रकार का एक ऑटोषेप जोड़ें
    $slide->getShapes()->addAutoShape(ShapeType::Line, 50, 150, 300, 0);
    $pres->save("NewPresentation_out.pptx", SaveFormat::Pptx);
  } finally {
    if (!java_is_null($pres)) {
      $pres->dispose();
    }
  }
```

## **अक्सर पूछे जाने वाले प्रश्न**

**नई प्रस्तुति को किन फ़ॉर्मेट्स में सहेजा जा सकता है?**

आप इसे [PPTX, PPT, और ODP](/slides/hi/php-java/save-presentation/) में सहेज सकते हैं, और [PDF](/slides/hi/php-java/convert-powerpoint-to-pdf/), [XPS](/slides/hi/php-java/convert-powerpoint-to-xps/), [HTML](/slides/hi/php-java/convert-powerpoint-to-html/), [SVG](/slides/hi/php-java/convert-powerpoint-to-png/), तथा [images](/slides/hi/php-java/convert-powerpoint-to-png/) सहित अन्य फ़ॉर्मेट्स में निर्यात कर सकते हैं।

**क्या मैं टेम्प्लेट (POTX/POTM) से शुरू करके नियमित PPTX के रूप में सहेज सकता हूँ?**

हां। टेम्प्लेट लोड करें और इच्छित फ़ॉर्मेट में सहेजें; POTX/POTM/PPTM और समान फ़ॉर्मेट्स [समर्थित](/slides/hi/php-java/supported-file-formats/) हैं।

**प्रस्तुति बनाते समय स्लाइड आकार/आस्पेक्ट अनुपात कैसे नियंत्रित करें?**

[slide size](/slides/hi/php-java/slide-size/) निर्धारित करें (4:3 और 16:9 जैसे प्रीसैट या कस्टम आयाम सहित) और तय करें कि सामग्री कैसे स्केल होनी चाहिए।

**आकार और निर्देशांक किस इकाइयों में मापे जाते हैं?**

पॉइंट्स में: 1 इंच = 72 यूनिट्स।

**बहुत बड़ी प्रस्तुतियों (कई मीडिया फ़ाइलों के साथ) को मेमोरी उपयोग कम करने के लिए कैसे संभालें?**

[BLOB management strategies](/slides/hi/php-java/manage-blob/) का उपयोग करें, अस्थायी फ़ाइलों के माध्यम से इन‑मेमोरी स्टोरेज को सीमित करें, और शुद्ध इन‑मेमोरी स्ट्रीम्स की बजाय फ़ाइल‑आधारित वर्कफ़्लो को प्राथमिकता दें।

**क्या मैं समानांतर में प्रस्तुतियों को बना/सहेज सकता हूँ?**

आप समान [Presentation](https://reference.aspose.com/slides/hi/php-java/aspose.slides/presentation/) इंस्टेंस को [multiple threads](/slides/hi/php-java/multithreading/) से संचालित नहीं कर सकते। प्रत्येक थ्रेड या प्रक्रिया के लिए अलग, अलगाव वाले इंस्टेंस चलाएँ।

**ट्रायल वॉटरमार्क और प्रतिबंधों को कैसे हटाएँ?**

[Apply a license](/slides/hi/php-java/licensing/) को प्रत्येक प्रक्रिया में एक बार लागू करें। लाइसेंस XML को अपरिवर्तित रखना ज़रूरी है, और यदि कई थ्रेड्स शामिल हों तो लाइसेंस सेटअप को समन्वित करना चाहिए।

**क्या मैं बनाई गई PPTX को डिजिटल रूप से साइन कर सकता हूँ?**

हां। प्रस्तुतियों के लिए [Digital signatures](/slides/hi/php-java/digital-signature-in-powerpoint/) (जोड़ना और सत्यापित करना) समर्थित हैं।

**क्या बनाई गई प्रस्तुतियों में मैक्रो (VBA) समर्थित हैं?**

हां। आप [create/edit VBA projects](/slides/hi/php-java/presentation-via-vba/) कर सकते हैं और PPTM/PPSM जैसे मैक्रो‑सक्षम फ़ाइलें सहेज सकते हैं।
