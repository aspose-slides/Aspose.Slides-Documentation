## **सारांश**

यह FAQ Aspose.Slides के बारे में सामान्य प्रश्नों के उत्तर प्रदान करता है। यह समर्थित फ़ाइल फ़ॉर्मेट, बड़े प्रस्तुतियों के साथ काम करते समय अपवादों को संभालना, स्लाइड आकार बदलना, स्लाइडों का पूर्वावलोकन, प्रस्तुतियों से टेक्स्ट प्राप्त करना, तालिका की सीमाओं का स्वरूपण, चित्र रखने, और PDF या छवियों में प्रस्तुतियों को कनवर्ट करते समय फ़ॉन्ट से संबंधित समस्याओं को हल करना आदि शामिल है।

## **समर्थित फ़ाइल फ़ॉर्मेट**

**Q:** Aspose.Slides for PHP via Java किन फ़ाइल फ़ॉर्मेट को समर्थन करता है?  

**A:** Aspose.Slides for PHP via Java उन फ़ाइल फ़ॉर्मेट को समर्थन करता है जो [Supported File Formats](/slides/hi/php-java/supported-file-formats/) में वर्णित हैं।

## **अपवाद**

**Q:** मैं बड़ी PPT फ़ाइल को इमेजेस के साथ लोड करते समय out of memory अपवाद प्राप्त कर रहा हूँ। क्या Aspose.Slides में फ़ाइल आकार की कोई सीमा है?  

**A:** Aspose.Slides द्वारा समर्थित प्रस्तुति आकार की गणना के लिए कोई विशिष्ट सूत्र नहीं है। संपूर्ण प्रस्तुति संरचना और इमेजेस को मेमोरी में रखने के लिए पर्याप्त स्थान होना चाहिए। सामान्यतः, मेमोरी में इमेजेस हार्ड डिस्क से अधिक स्थान घेरती हैं, विशेषकर जब इमेजेस में अतिरिक्त प्रभाव हों। सामान्य रूप से, Aspose.Slides for PHP via Java लगभग 300 MB के आकार की प्रस्तुति फ़ाइलों को 4 GB RAM वाले सर्वर पर आसानी से संभाल सकता है।

## **स्लाइड्स के साथ कार्य करना**

**Q:** क्या मैं प्रस्तुति में स्लाइड्स के आकार को बदल सकता हूँ?  

**A:** आप प्रस्तुति में स्लाइड्स के आकार को परिभाषित करने के लिए [Presentation](https://reference.aspose.com/slides/hi/php-java/aspose.slides/presentation/) क्लास द्वारा प्रदान किए गए `getSlideSize` मेथड का उपयोग कर सकते हैं।

**Q:** क्या प्रस्तुति में विभिन्न आकार की स्लाइड्स परिभाषित करने का कोई तरीका है?  

**A:** चूँकि स्लाइड्स का आकार Microsoft PowerPoint दस्तावेज़ में प्रस्तुति स्तर पर परिभाषित होता है, इसलिए इसे अलग-अलग आकार देने का कोई तरीका नहीं है।

**Q:** क्या Aspose.Slides for PHP via Java स्लाइड को सहेजने से पहले उसके पूर्वावलोकन का समर्थन करता है?  

**A:** आप प्रस्तुति स्लाइड्स को इमेजेस में रेंडर कर सकते हैं और इन इमेजेस का उपयोग स्लाइड्स के पूर्वावलोकन के लिए कर सकते हैं।

## **टेक्स्ट के साथ कार्य करना**

**Q:** क्या प्रस्तुति से सम्पूर्ण टेक्स्ट प्राप्त करना संभव है?  

**A:** Aspose.Slides for PHP via Java [SlideUtil](https://reference.aspose.com/slides/hi/php-java/aspose.slides/slideutil/) क्लास प्रदान करता है जो प्रस्तुतियों से सम्पूर्ण टेक्स्ट प्राप्त करने के विभिन्न मेथड्स देता है।

**Q:** Windows और Linux ऑपरेटिंग सिस्टम पर पैराग्राफ आकार अलग क्यों होते हैं?  

**A:** पैराग्राफ आकार की गणना उस टेक्स्ट आकार पर आधारित होती है जो दिए गए पैराग्राफ को दर्शाता है। टेक्स्ट आकार की गणना PowerPoint प्रस्तुति में निर्दिष्ट फ़ॉन्ट के मीट्रिक्स पर आधारित होती है। यदि निर्दिष्ट फ़ॉन्ट उपलब्ध नहीं है, तो उसे सबसे समान फ़ॉन्ट से बदल दिया जाता है, लेकिन इस फ़ॉन्ट के मीट्रिक्स मूल फ़ॉन्ट से अलग होते हैं। परिणामस्वरूप, विभिन्न सिस्टमों में पैराग्राफ आकार की गणना अलग परिणाम देती है, जो स्थापित फ़ॉन्ट्स के सेट पर निर्भर करता है। विभिन्न ऑपरेटिंग सिस्टमों पर समान परिणाम पाने के लिए, आपको सिस्टमों पर एक ही फ़ॉन्ट्स स्थापित करने होंगे या उन्हें रनटाइम पर [external fonts](/slides/hi/php-java/custom-font/) के रूप में लोड करना होगा।

## **फ़ॉर्मेटिंग और छवियां**

**Q:** मैं तालिका की सीमा का रंग कैसे सेट कर सकता हूँ?  

**A:** आप सभी तालिका सीमाओं का रंग या पूरी तालिका की सीमा का रंग बदल सकते हैं। सभी सीमाओं को बदलने के लिए कृपया [Cell](https://reference.aspose.com/slides/hi/php-java/aspose.slides/cell/) क्लास के `getCellFormat` मेथड का उपयोग करें। पूरी तालिका की सीमा के लिए, आपको सेल्स पर इटेरेट करना होगा और बाहरी सीमाओं का रंग बदलना होगा।

**Q:** Aspose.Slides for PHP via Java चित्र रखने के लिए कौन सी माप इकाई उपयोग करता है?  

**A:** स्लाइड्स पर सभी शाक्यों के समन्वय और आकार पॉइंट्स (72 dpi) में मापे जाते हैं।

## **फ़ॉन्ट्स के साथ कार्य करना**

**Q:** PPT को PDF या छवियों में कनवर्ट करते समय आउटपुट दस्तावेज़ों में फ़ॉन्ट्स अलग क्यों होते हैं?  

**A:** यह समस्या इस बात को दर्शा सकती है कि प्रस्तुति में उपयोग किए गए फ़ॉन्ट्स उस ऑपरेटिंग सिस्टम पर मौजूद नहीं हैं जहाँ कोड चलाया गया था। आपको फ़ॉन्ट्स को ऑपरेटिंग सिस्टम पर स्थापित करना चाहिए या उन्हें बाहरी फ़ॉन्ट्स के रूप में [FontsLoader](https://reference.aspose.com/slides/hi/php-java/aspose.slides/fontsloader/) क्लास का उपयोग करके लोड करना चाहिए जैसा नीचे दिखाया गया है:
```php
$folders = ["path_to_a_folder_with_fonts"];
FontsLoader::loadExternalFonts($folders);
```
