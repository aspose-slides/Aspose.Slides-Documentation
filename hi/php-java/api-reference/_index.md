Aspose.Slides for PHP via Java एक क्लास लाइब्रेरी है जो PHP एप्लीकेशन को माइक्रोसॉफ्ट पावरपॉइंट® या अन्य को उपयोग किए बिना प्रेजेंटेशन दस्तावेज़ पढ़ने और लिखने की अनुमति देती है। Aspose.Slides for PHP via Java टेक्स्ट, शैप्स, टेबल्स और एनीमेशन को मैनेज करने, स्लाइड्स में ऑडियो और वीडियो जोड़ने, स्लाइड्स का प्रीव्यू करने, स्लाइड्स को SVG, PDF फॉर्मेट में एक्सपोर्ट करने और भी बहुत कुछ करने की सुविधा देती है।

नवीनतम API रेफरेंस यहाँ पाया जा सकता है: [Aspose.Slides for PHP via Java API Reference](https://reference.aspose.com/slides/hi/php-java/).
