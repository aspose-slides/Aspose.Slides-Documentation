## **Aspose.Slides मूल्यांकन**

आप आसानी से Aspose.Slides को मूल्यांकन हेतु डाउनलोड कर सकते हैं। मूल्यांकन पैकेज खरीदे गए पैकेज के समान ही है। मूल्यांकन संस्करण केवल कुछ पंक्तियों का कोड लाइसेंस लागू करने के बाद लाइसेंस प्राप्त बन जाता है।

Aspose.Slides का मूल्यांकन संस्करण (बिना लाइसेंस निर्दिष्ट किए) पूर्ण उत्पाद कार्यक्षमता प्रदान करता है, लेकिन यह खोलने और सहेजने पर दस्तावेज़ के शीर्ष पर एक मूल्यांकन वॉटरमार्क डालता है। प्रस्तुति स्लाइड्स से पाठ निकालते समय आप केवल एक स्लाइड तक सीमित हैं।

## **FAQ**

**Can I test multiple presentations in parallel across different threads in evaluation mode?**  
हां। आप विभिन्न दस्तावेज़ों को समानांतर रूप से प्रोसेस कर सकते हैं; आपको वही प्रस्तुति ऑब्जेक्ट [थ्रेड्स के बीच](/slides/hi/php-java/multithreading/) साझा नहीं करना चाहिए। मूल्यांकन मोड इसका प्रभाव नहीं डालता।

**Do I need to install Microsoft PowerPoint to evaluate the library on a server or in CI?**  
नहीं। Aspose.Slides एक स्वतंत्र इंजन है और मूल्यांकन या उत्पादन दोनों के लिए PowerPoint स्थापित करने की आवश्यकता नहीं है।

**Can I fully test conversion of PPT/PPTX to PDF and images in evaluation mode?**  
हां। [कनवर्टर](/slides/hi/php-java/convert-presentation/) काम करते हैं; आउटपुट में एक वॉटरमार्क शामिल होगा।

**Can I use a temporary license for load testing without a watermark?**  
हां। 30 दिन का अस्थायी लाइसेंस मूल्यांकन मोड की सीमाओं को हटा देता है और वॉटरमार्क के बिना परीक्षण करने की अनुमति देता है।
