## **परिचय**

यह लेख Aspose.Slides के लिए समर्थन प्राप्त करने और मुद्दों की प्रभावी रिपोर्टिंग के बारे में बताता है। यह मुख्य समर्थन चैनल का वर्णन करता है, समस्या सबमिट करने की सिफ़ारिशें देता है, और उत्पाद पेज, ब्लॉग, डाउनलोड, समर्थन फ़ोरम, दस्तावेज़, और API रेफ़रेंस जैसे उपयोगी तकनीकी संसाधनों की ओर संकेत करता है। लेख में पेड समर्थन, मुद्दा एस्केलेशन, और पुराने लाइब्रेरी संस्करणों के समर्थन के बारे में सामान्य प्रश्नों के उत्तर भी शामिल हैं।

## **तकनीकी समर्थन**

{{% alert color="primary" %}} 
Aspose अपने सभी उत्पादों के लिए असीमित मुफ्त तकनीकी समर्थन प्रदान करता है। समर्थन सभी उपयोगकर्ताओं के लिए उपलब्ध है, जिसमें मूल्यांकन शामिल है।
{{% /alert %}} 

यदि आपको Aspose.Slides for PHP via Java में मदद चाहिए, तो निम्नलिखित पर विचार करें:

- समर्थन के लिए मुख्य माध्यम Aspose.Forums है। अपना प्रश्न Aspose.Slides फ़ोरम में पोस्ट करें और इसे कुछ घंटों के भीतर उत्तर मिलेगा। प्रश्नों का उत्तर सीधे Aspose.Slides समर्थन टीम द्वारा दिया जाता है।
- कृपया ध्यान दें, Aspose फोन पर तकनीकी समर्थन प्रदान नहीं करता। फोन समर्थन केवल बिक्री और खरीद प्रश्नों के लिए उपलब्ध है।
- फ़ोरम में उत्तर की अपेक्षा करते समय, कृपया टाइम ज़ोन अंतर को ध्यान में रखें।

यदि आपके पास Aspose.Slides for PHP via Java में कोई समस्या है, तो इसे सबसे प्रभावी तरीके से हल करने के लिए नीचे दिए गए सरल चरणों का पालन करें:

- समस्या रिपोर्ट करने से पहले सुनिश्चित करें कि आप सबसे नवीनतम Aspose.Slides for PHP via Java संस्करण का उपयोग कर रहे हैं, नवीनतम संस्करण जानने के लिए Aspose.Slides डाउनलोड देखें।
- फ़ोरम, इस दस्तावेज़ और API रेफ़रेंस को देखें; संभवतः आपका प्रश्न पहले ही उत्तर दिया गया है।
- समस्या रिपोर्ट करते समय, कृपया मूल दस्तावेज़ और संभवतः वह कोड का अंश शामिल करें जो समस्या पैदा करता है। यदि आपको कई फाइलें संलग्न करनी हों, तो उन्हें एक ज़िप फ़ाइल में संकुचित करें। यह Aspose.Forums में सुरक्षित है क्योंकि केवल आप और Aspose डेवलपर्स को ही संलग्न फाइलों तक पहुंच होगी।
- कृपया एक थ्रेड में केवल एक ही मुद्दा रिपोर्ट करें। यदि आपके पास दूसरा मुद्दा है, तो उसे अलग थ्रेड में रिपोर्ट करें।

## **FAQ**

**कौन से पेड समर्थन टियर्स उपलब्ध हैं और वे कैसे अलग हैं?**

[डेवलपर, बिज़नेस, और एंटरप्राइस](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). वे शामिल घटनाओं की संख्या, प्रतिक्रिया प्राथमिकता, और उपलब्ध संचार चैनलों के आधार पर अलग होते हैं।

**क्या मैं बग को सीधे इंजीनियरिंग टीम को एस्केलेट कर सकता हूँ?**

हाँ। [पेड समर्थन](https://helpdesk.aspose.com/) गहरी जांच की आवश्यकता वाले मुद्दों के लिए डेवलपर्स को एस्केलेशन की अनुमति देता है।

**क्या पुराने लाइब्रेरी संस्करणों को समर्थन मिलता है, और क्या आप उनके लिए फिक्स जारी करते हैं?**

आप पुराने संस्करणों का उपयोग कर सकते हैं, लेकिन फिक्स केवल वर्तमान रिलीज़ के लिए प्रदान किए जाते हैं; लेगेसी संस्करणों के लिए पैच जारी नहीं किए जाते।
