## **समर्थित प्लेटफ़ॉर्म**
Aspose.Slides for PHP via Java एक प्लेटफ़ॉर्म स्वतंत्र API है। इसे किसी भी प्लेटफ़ॉर्म (Windows, Linux, macOS आदि) पर उपयोग किया जा सकता है जहाँ PHP 7 या उससे नया स्थापित हो। Oracle JDK 7 या उससे नया भी एक आवश्यकता है।

## **फ़ाइल फ़ॉर्मेट और रूपांतरण**
Aspose.Slides for PHP via Java अधिकांश PowerPoint दस्तावेज़ फ़ॉर्मेट को समर्थन देता है। यह आपको इन्हें उन लोकप्रिय फ़ाइल फ़ॉर्मेट में निर्यात करने की सुविधा भी देता है जो संगठनों द्वारा उपयोग और शेयर किए जाते हैं। नीचे विवरण देखें:

|**फ़ीचर**|**विवरण**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for PHP via Java इस प्रेज़ेंटेशन दस्तावेज़ फ़ॉर्मेट के लिए सबसे तेज़ प्रोसेसिंग प्रदान करता है।|
|PPT to PPTX conversion|Aspose.Slides for PHP via Java PPT से PPTX रूपांतरण को समर्थन देता है।|
|Portable Document Format (PDF)|आप सभी समर्थित फ़ाइल फ़ॉर्मेट को एक ही मेथड से Adobe Portable Document Format (PDF) दस्तावेज़ों में निर्यात कर सकते हैं।|
|XML Parser Specification (XPS)|आप सभी समर्थित फ़ाइल फ़ॉर्मेट को एक ही मेथड से XML Parser Specification (XPS) दस्तावेज़ों में निर्यात कर सकते हैं।|
|Tagged Image File Format (TIFF)|आप सभी समर्थित प्रेज़ेंटेशन फ़ाइल फ़ॉर्मेट को Tagged Image File Format (TIFF) में निर्यात कर सकते हैं।|
|PPTX To HTML Conversion|Aspose.Slides for PHP via Java PresentationEx को HTML फ़ॉर्मेट में रूपांतरित करने का समर्थन करता है।|

|**फ़ीचर**|**विवरण**|
| :- | :- |
|Java Supported Image Formats|Aspose.Slides for PHP via Java का उपयोग करके आप सभी Java समर्थित ग्राफ़िक्स फ़ॉर्मेट जैसे TIFF, PNG, BMP, JPEG, GIF में प्रेज़ेंटेशन स्लाइड्स और स्लाइड्स पर चित्र रेंडर कर सकते हैं।|
|SVG Format|Aspose.Slides for PHP via Java बिल्ट‑इन मेथड्स प्रदान करता है जो प्रेज़ेंटेशन स्लाइड्स को Scalable Vector Graphics (SVG) फ़ॉर्मेट में निर्यात करने की अनुमति देते हैं।|
|Presentation Printing|Aspose.Slides for PHP via Java के नवीनतम संस्करण विभिन्न विकल्पों के साथ बिल्ट‑इन प्रिंट मेथड्स प्रदान करते हैं।|

## **FAQ**

**क्या लाइब्रेरी के काम करने के लिए सर्वर/पीसी पर Microsoft PowerPoint स्थापित करना आवश्यक है?**

नहीं। PowerPoint आवश्यक नहीं है; Aspose.Slides एक स्वतंत्र इंजन है जो प्रेज़ेंटेशन बनाने, संशोधित करने, रूपांतरित करने और रेंडर करने के लिए कार्य करता है।

**मल्टी‑थ्रेडिंग कैसे काम करती है? क्या प्रोसेसिंग को समांतर किया जा सकता है?**

विभिन्न थ्रेड्स में विभिन्न दस्तावेज़ों को प्रोसेस करना सुरक्षित है; वही [प्रेज़ेंटेशन](https://reference.aspose.com/slides/hi/php-java/aspose.slides/presentation/) ऑब्जेक्ट को एक साथ कई [थ्रेड्स](/slides/hi/php-java/multithreading/) द्वारा उपयोग नहीं किया जाना चाहिए।

**क्या फ़ाइल पासवर्ड और एन्क्रिप्शन समर्थित हैं?**

हां। आप एन्क्रिप्टेड प्रेज़ेंटेशन खोल सकते हैं, खोलने और लिखने के पासवर्ड सेट या हटाए जा सकते हैं, और सुरक्षा स्थिति की जाँच कर सकते हैं। ([आप कर सकते हैं](/slides/hi/php-java/password-protected-presentation/))

**क्या Linux कंटेनर में फ़ॉन्ट पैकेजों के बारे में ध्यान रखना चाहिए?**

हां। सामान्य फ़ॉन्ट पैकेज स्थापित करने या अपने एप्लिकेशन में स्पष्ट रूप से [फ़ॉन्ट डायरेक्टरी निर्दिष्ट](/slides/hi/php-java/custom-font/) करने की सलाह दी जाती है, ताकि अप्रत्याशित प्रतिस्थापन से बचा जा सके।

**क्या मूल्यांकन संस्करण में सीमाएँ हैं?**

[मूल्यांकन मोड](/slides/hi/php-java/licensing/) में आउटपुट में एक वाटरमार्क जोड़ा जाता है और कुछ सीमाएँ लागू होती हैं; पूर्ण‑फ़ीचर परीक्षण के लिए एक [30‑दिन का अस्थायी लाइसेंस](https://purchase.aspose.com/temporary-license/) उपलब्ध है।

**क्या बाहरी फ़ॉर्मेट (PDF/HTML → PPTX) को प्रेज़ेंटेशन में आयात करना समर्थित है?**

हां। आप [PDF पृष्ठ और HTML सामग्री](/slides/hi/php-java/import-presentation/) को प्रेज़ेंटेशन में जोड़ सकते हैं, जिससे वे स्लाइड्स में परिवर्तित हो जाएँ।
