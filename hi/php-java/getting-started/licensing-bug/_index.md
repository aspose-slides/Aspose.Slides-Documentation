## **सारांश**

यह लेख Aspose.Slides for PHP via Java संस्करण 23.2, 23.3, 23.4, 23.5, 23.6 और 23.7 में सामना किए गए लाइसेंसिंग बग का विवरण देता है। इस बग के कारण स्थायी लाइसेंसों का सब्सक्रिप्शन समाप्त होने के बाद वे अस्थायी रूप से अनुपयोगी हो गए।

## **लक्षण**

जब आपका स्थायी लाइसेंस सब्सक्रिप्शन समाप्त हो जाता है, तो Aspose.Slides for PHP via Java संस्करण 23.2 - 23.7 लाइसेंस को समाप्त बताया सकता है, जिससे सभी फ़ंक्शन मूल्यांकन मोड में चले जाते हैं। यह व्यवहार गलत है और संस्करण 23.2 से पहले तथा 23.7 के बाद के संस्करणों को प्रभावित नहीं करता।

## **समाधान**

Aspose.Slides for PHP via Java ने इस समस्या को ठीक किया और फिक्स के साथ अपडेटेड संस्करण (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) जारी किए।

यदि आप अपने प्रोजेक्ट में Aspose.Slides for PHP via Java के प्रभावित संस्करणों में से किसी एक का उपयोग कर रहे हैं, तो कृपया सुधारे गए संस्करण का उपयोग करने के लिए स्विच करें।

यदि आप Aspose.Slides for PHP via Java के साथ कोई लाइसेंसिंग समस्या का सामना करते हैं:

- अपनी उपयोग की गई संस्करण की जाँच करें और सुनिश्चित करें कि वह ऊपर वर्णित बग से प्रभावित नहीं है।
- समाधान के कदम और ज्ञात लाइसेंसिंग समस्याओं के लिए [Aspose दस्तावेज़ीकरण](https://docs.aspose.com/slides/hi/php-java/getting-started/) देखें।
- अधिक सहायता के लिए [Aspose support](https://forum.aspose.com/) से संपर्क करें।

## **प्रभावित उत्पाद और संस्करण**

कृपया ध्यान दें कि यह बग केवल Aspose.Slides for PHP via Java संस्करण 23.2 से 23.7 तक को ही प्रभावित करता है। **अन्य Aspose उत्पाद और Aspose.Slides के अन्य संस्करण प्रभावित नहीं होते**।
