## **TIFF के बारे में**

[Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) जो एक दस्तावेज़ में कई छवियों को संग्रहीत करने के लिए जाना जाता है, मूल रूप से Aldus द्वारा बनाया गया था। इस फ़ॉर्मेट को स्कैनिंग, फ़ैक्सिंग और अन्य छवि प्रोसेसिंग अनुप्रयोगों द्वारा व्यापक रूप से समर्थन प्राप्त है।
