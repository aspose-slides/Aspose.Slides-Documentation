Portable Document Format (PDF) एक फ़ाइल फ़ॉर्मेट है जिसे Adobe Systems ने संगठनों के बीच दस्तावेज़ों का आदान-प्रदान करने के लिए बनाया है। इस फ़ॉर्मेट का उद्देश्य सामग्री और लेआउट को समान रखना था, चाहे वह किसी भी प्लेटफ़ॉर्म पर देखा जाए। Aspose.Slides for PHP via Java आपको प्रेज़ेंटेशन फ़ाइलों को PDF में बदलने की अनुमति देता है।

{{% /alert %}}

## **Aspose.Slides for PHP via Java में PDF**
Aspose.Slides for PHP via Java में लोड की जा सकने वाली कोई भी प्रेज़ेंटेशन आपके चयन के अनुसार PDF 1.5, PDF/A-1a, PDF/A-1b या PDF/UA के अनुरूप PDF में बदली जा सकती है। Aspose.Slides for PHP via Java प्रेज़ेंटेशन को PDF में निर्यात करता है और अधिकांश मामलों में, उत्पन्न PDF बिल्कुल मूल प्रेज़ेंटेशन जैसा दिखता है।

Aspose.Slides PDF में रूपांतरित करते समय निम्नलिखित प्रेज़ेंटेशन सुविधाओं को समर्थन करता है:

- छवियाँ, टेक्स्ट बॉक्स और अन्य आकृतियों।
- टेक्स्ट और फ़ॉर्मेटिंग।
- पैरा और फ़ॉर्मेटिंग।
- हाइपरलिंक।
- हेडर और फुटर।
- बुलेट्स।
- टेबल्स।

आप Aspose.Slides for PHP via Java का उपयोग करके सीधे प्रेज़ेंटेशन को PDF में निर्यात कर सकते हैं: आपको किसी अन्य घटक की आवश्यकता नहीं है। इसके अतिरिक्त, आप [Converting to PDF](/slides/hi/php-java/convert-powerpoint-to-pdf/) में समझाए अनुसार विभिन्न विकल्पों के साथ प्रेज़ेंटेशन से PDF निर्यात को अनुकूलित कर सकते हैं।
