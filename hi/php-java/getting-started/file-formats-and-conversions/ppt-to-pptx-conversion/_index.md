Aspose.Slides for PHP via Java द्वारा PPT से PPTX रूपांतरण समर्थित है। अधिकांश प्रस्तुति सुविधाएँ - मास्टर स्लाइड्स, संरचना आदि - एक फ़ॉर्मेट से दूसरे में परिवर्तित होने पर बरकरार रहती हैं, लेकिन वहाँ [कुछ सीमाएं](/slides/hi/php-java/ppt-to-pptx-conversion/) हैं।

## **रूपांतरण में समर्थित विशेषताएँ**
Aspose.Slides for PHP via Java PPT फ़ाइल फ़ॉर्मेट को PPTX में बदलने के लिए आंशिक समर्थन प्रदान करता है। रूपांतरण के लिए समर्थन अभी-अभी Aspose.Slides for PHP via Java में प्रस्तुत किया गया है, इसलिए इसमें कुछ सीमाएँ हैं और यह सरल प्रस्तुतियों के लिए सबसे उपयुक्त है। जब PPT को PPTX में रूपांतरित किया जाता है तो Aspose.Slides for PHP via Java का मुख्य लाभ यह है कि API का उपयोग कितना आसान है। कोड के उदाहरण देखने के लिए, पढ़ें [PPT को PPTX में रूपांतरण](). नीचे, सूचियाँ बताती हैं कि कौन-सी सुविधाएँ समर्थित हैं और कौन-सी नहीं हैं PPT से PPTX रूपांतरण के लिए।

## **समर्थित विशेषताएँ**
निम्नलिखित विशेषताएँ रूपांतरण के लिए समर्थित हैं:

- मास्टर, लेआउट और स्लाइड्स की संरचना का रूपांतरण।
- चार्ट का रूपांतरण।
- समूह आकार।
- आयत और अण्डाकार सहित ऑटॉशेप्स का रूपांतरण।
- कस्टम ज्यामिति वाले आकार।
- ऑटॉशेप्स के लिए टेक्सचर और चित्र भराव शैली।
- प्लेसहोल्डर का रूपांतरण।
- रेखाओं और पॉलीलाइन का रूपांतरण।
- लाइन और भराव स्वरूप।
- ग्रेडिएंट भराव शैली।
- OLE फ़्रेम, तालिकाएँ, वीडियो और ऑडियो फ़्रेम, आदि।
- एनिमेशन और स्लाइडशो गुण।
- टेक्स्ट फ़्रेम और टेक्स्ट होल्डर्स में टेक्स्ट का रूपांतरण।
