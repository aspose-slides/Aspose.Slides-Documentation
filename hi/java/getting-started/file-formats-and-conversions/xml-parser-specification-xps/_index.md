[XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) एक पेज वर्णन भाषा और स्थिर‑डॉक्यूमेंट फ़ॉर्मेट है जो मूल रूप से माइक्रोसॉफ़्ट द्वारा विकसित किया गया था। PDF की तरह, XPS दस्तावेज़ की सत्यता को बनाए रखने और डिवाइस‑स्वतंत्र दस्तावेज़ स्वरूप प्रदान करने के लिए डिज़ाइन किया गया है। 
{{% /alert %}} 

## **Aspose.Slides for Java में XPS**
Aspose.Slides for Java द्वारा लोड किया जा सकने वाला कोई भी प्रेजेंटेशन दस्तावेज़ XPS फ़ॉर्मेट में परिवर्तित किया जा सकता है। Aspose.Slides for Java उच्च‑विश्वसनीयता पेज लेआउट और रेंडरिंग इंजन का उपयोग करके स्थिर‑लेआउट XPS दस्तावेज़ फ़ॉर्मेट में आउटपुट बनाता है।
आप Aspose.Slides for Java के माध्यम से प्रेजेंटेशन दस्तावेज़ों को XPS में निर्यात करने के बारे में [Converting to XPS](https://docs.aspose.com/slides/hi/java/convert-powerpoint-to-xps/) में सीख सकते हैं।
