The [Portable Document Format](https://en.wikipedia.org/wiki/PDF) एडोब सिस्टम्स द्वारा निर्मित एक फ़ाइल फ़ॉर्मैट है, जिसका उपयोग संगठनों के बीच दस्तावेज़ों का आदान‑प्रदान करने के लिए किया जाता है। इस फ़ॉर्मैट का उद्देश्य सामग्री और लेआउट को समान रखना था, चाहे इसे किसी भी प्लेटफ़ॉर्म पर देखा जाए। Aspose.Slides for Java आपको प्रेज़ेंटेशन फ़ाइलों को PDF में बदलने की सुविधा देता है।
{{% /alert %}} 

## **Aspose.Slides for Java में PDF**
Aspose.Slides for Java में लोड की जा सकने वाली कोई भी प्रस्तुति आपके चयन के अनुसार PDF 1.5, PDF/A-1a, PDF/A-1b या PDF/UA का अनुरूप PDF में परिवर्तित की जा सकती है। Aspose.Slides for Java प्रस्तुतियों को PDF में निर्यात करता है और अधिकांश मामलों में, उत्पन्न PDF मूल प्रस्तुति के समान दिखता है।

Aspose.Slides PDF में परिवर्तित करते समय निम्नलिखित प्रस्तुति सुविधाओं का समर्थन करता है:

- छवियाँ, टेक्स्ट बॉक्स और अन्य आकार।
- टेक्स्ट और फ़ॉर्मेटिंग।
- अनुच्छेद और फ़ॉर्मेटिंग।
- हाइपरलिंक।
- हेडर और फुटर।
- बुलेट।
- टेबल।

आप Aspose.Slides for Java का उपयोग करके प्रस्तुतियों को सीधे PDF में निर्यात कर सकते हैं: आपको किसी अन्य घटक की आवश्यकता नहीं है। आगे, आप [Converting to PDF](/slides/hi/java/converting-a-presentation/) में समझाए गए विभिन्न विकल्पों के साथ प्रस्तुति से PDF निर्यात को कस्टमाइज़ कर सकते हैं।
