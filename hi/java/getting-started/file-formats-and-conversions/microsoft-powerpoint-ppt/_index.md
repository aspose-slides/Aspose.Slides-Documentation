[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) माइक्रोसॉफ्ट पावरपॉइंट के विभिन्न संस्करणों द्वारा निर्मित, पढ़ी, संशोधित और लिखी जा सकने वाला प्रस्तुति दस्तावेज़ फ़ाइल फ़ॉर्मेट है। यह माइक्रोसॉफ्ट द्वारा विकसित प्रस्तुति दस्तावेज़ों के लिए बाइनरी फ़ॉर्मेट है।

{{% /alert %}} 

## **Aspose.Slides for Java में PPT**
Aspose.Slides for Java नीचे सूचीबद्ध सॉफ़्टवेयर द्वारा निर्मित PPT फ़ाइलों को पढ़ सकता है।

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

इसी प्रकार, Aspose.Slides for Java द्वारा निर्मित PPT फ़ाइलों को ऊपर दिए गए सॉफ़्टवेयर द्वारा पढ़ा जा सकता है।

## **PPT के लिए व्यापक समर्थन**
Aspose.Slides for Java PPT दस्तावेज़ फ़ाइल फ़ॉर्मेट द्वारा समर्थित लगभग सभी सुविधाओं के लिए समर्थन प्रदान करता है। यह न केवल विभिन्न माइक्रोसॉफ्ट पावरपॉइंट संस्करणों द्वारा प्रदान की गई बुनियादी और उन्नत सुविधाओं को कवर करता है, बल्कि ऐसी सुविधाएँ भी शामिल करता है जो माइक्रोसॉफ्ट पावरपॉइंट द्वारा समर्थित नहीं हैं। Aspose.Slides for Java API लाइब्रेरी को उपयोग करने का मुख्य लाभ इन सुविधाओं को संभालने में आसानी है।

PPT दस्तावेज़ फ़ाइलों के निर्माण, पढ़ने और लिखने से संबंधित बुनियादी कार्यों के अतिरिक्त, Aspose.Slides for Java कई सुविधाएँ प्रदान करता है:

- अन्य Microsoft Office फ़ाइल फ़ॉर्मेट को [OLE objects into PPT documents]() के रूप में आयात करें।
- [PPT दस्तावेज़ को PDF में निर्यात करें](/slides/hi/java/convert-powerpoint-ppt-and-pptx-to-pdf/).
- PPT दस्तावेज़ों में स्लाइड्स को SVG फ़ॉर्मेट में निर्यात करें।
- जावा फ़्रेमवर्क द्वारा समर्थित किसी भी इमेज फ़ॉर्मेट में स्लाइड्स को रेंडर करें।
- PPT दस्तावेज़ों में स्लाइड्स का आकार सेट करें।
- शेप्स पर एनीमेशन प्रबंधित करें।
- स्लाइड शोज़ प्रबंधित करें।
- [स्लाइड्स पर टेक्स्ट फ़ॉर्मेट करें]().
- PPT दस्तावेज़ों से टेक्स्ट निकालें।
- [स्लाइड्स पर टेबल्स को संभालें]().
- स्वचालित रूप से मास्टर कॉपी करें [the cloning feature]() का उपयोग करके।
