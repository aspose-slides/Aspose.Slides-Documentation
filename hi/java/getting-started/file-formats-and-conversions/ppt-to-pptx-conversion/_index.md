Aspose.Slides for Java में PPT से PPTX रूपांतरण समर्थित है। अधिकांश प्रस्तुति सुविधाएँ - मास्टर स्लाइड्स, संरचना आदि - एक फ़ॉर्मेट से दूसरे में बदलने पर बरकरार रहती हैं, परन्तु कुछ सीमाएँ हैं। 
{{% /alert %}} 
## **रूपांतरण में समर्थित सुविधाएँ**
Aspose.Slides for Java PPT फ़ाइल फ़ॉर्मेट को PPTX में बदलने के लिए आंशिक समर्थन प्रदान करता है। रूपांतरण के लिए समर्थन अभी अभी Aspose.Slides for Java में जोड़ा गया है, इसलिए इसमें कुछ सीमाएँ हैं और यह सरल प्रस्तुतियों के लिए सबसे अच्छा कार्य करता है। PPT को PPTX में बदलते समय Aspose.Slides for Java का मुख्य लाभ यह है कि API का उपयोग बहुत आसान है। कोड के उदाहरण देखने के लिए, [PPT को PPTX में बदलना](). नीचे सूची बताती है कि कौन‑सी सुविधाएँ समर्थित हैं और कौन‑सी नहीं हैं PPT से PPTX रूपांतरण के लिए।

## **समर्थित सुविधाएँ**
रूपांतरण के लिए निम्न सुविधाएँ समर्थित हैं:

- मास्टर, लेआउट और स्लाइड्स की संरचना का रूपांतरण।
- चार्ट्स का रूपांतरण।
- समूह आकार।
- आयत और दीर्घवृत्त सहित ऑटोशेप्स का रूपांतरण।
- कस्टम ज्योमेट्री वाले आकार।
- ऑटोशेप्स के लिए टेक्सचर और चित्र भराव शैलियाँ।
- प्लेसहोल्डर्स का रूपांतरण।
- लाइन और पॉलीलाइन का रूपांतरण।
- लाइन और भराव स्वरूप।
- ग्रेडिएंट भराव शैलियाँ।
- OLE फ्रेम, तालिकाएँ, वीडियो और ऑडियो फ्रेम, आदि।
- एनीमेशन और स्लाइड शो गुण।
- टेक्स्ट फ्रेम और टेक्स्ट होल्डर्स में टेक्स्ट का रूपांतरण।
