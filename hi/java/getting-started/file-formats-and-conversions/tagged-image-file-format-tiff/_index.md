## **TIFF के बारे में**

[Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) जिसे एक दस्तावेज़ में कई छवियों को संग्रहीत करने के लिए जाना जाता है, मूल रूप से Aldus द्वारा बनाया गया था। यह फॉर्मेट स्कैनिंग, फैक्सिंग और अन्य छवि संपादन एप्लिकेशनों द्वारा व्यापक रूप से समर्थित है।

## **Aspose.Slides for Java में TIFF**

Aspose.Slides for Java में लोड किया जा सकने वाला कोई भी दस्तावेज़ सीधे TIFF दस्तावेज़ में परिवर्तित किया जा सकता है, जिससे किसी भी तृतीय‑पक्ष घटक की आवश्यकता समाप्त हो जाती है। आगे, आप परिणामी TIFF दस्तावेज़ में छवियों का आकार वैकल्पिक रूप से निर्धारित कर सकते हैं। आप Aspose.Slides for Java के माध्यम से प्रस्तुति दस्तावेज़ों को TIFF दस्तावेज़ों में निर्यात करने के बारे में जानकारी [इस विषय](/slides/hi/java/converting-a-presentation/) में पा सकते हैं।
