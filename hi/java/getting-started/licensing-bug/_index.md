## **अवलोकन**

यह लेख Aspose.Slides for Java संस्करण 23.2, 23.3, 23.4, 23.5, 23.6 और 23.7 में पाए गए लाइसेंसिंग बग का वर्णन करता है। यह बग स्थायी लाइसेंसों को उनकी सदस्यता समाप्त होने के बाद अस्थायी रूप से अनुपयोगी बना देता था।

## **लक्षण**

जब आपकी स्थायी लाइसेंस सब्सक्रिप्शन समाप्त हो जाती है, तो Aspose.Slides for Java संस्करण 23.2 - 23.7 लाइसेंस को समाप्त रिपोर्ट कर सकता है, जिससे सभी फ़ंक्शन मूल्यांकन मोड में बदल जाते हैं। यह व्यवहार गलत है और संस्करण 23.2 से पहले और 23.7 के बाद के संस्करणों को प्रभावित नहीं करता।

## **समाधान**

Aspose.Slides for Java ने इस समस्या को हल किया और सुधार के साथ अपडेटेड संस्करण (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) जारी किए।

यदि आप अपने प्रोजेक्ट में Aspose.Slides for Java के प्रभावित संस्करणों में से किसी का उपयोग कर रहे हैं, तो कृपया सुधारित संस्करण में स्विच करें।

यदि आपको Aspose.Slides for Java के साथ किसी भी लाइसेंसिंग समस्या का सामना करना पड़े:

- उपयोग किए जा रहे संस्करण की जाँच करें और सुनिश्चित करें कि वह ऊपर वर्णित बग से प्रभावित नहीं है।
- समस्या निवारण चरणों और ज्ञात लाइसेंसिंग मुद्दों के लिए [Aspose documentation](https://docs.aspose.com/slides/hi/java/getting-started/) देखें।
- आगे सहायता के लिए [Aspose support](https://forum.aspose.com/) से संपर्क करें।

## **प्रभावित उत्पाद और संस्करण**

कृपया ध्यान दें कि यह बग केवल Aspose.Slides for Java संस्करण 23.2 से 23.7 तक को प्रभावित करता है। **अन्य Aspose उत्पाद और Aspose.Slides के अन्य संस्करण प्रभावित नहीं होते**।
