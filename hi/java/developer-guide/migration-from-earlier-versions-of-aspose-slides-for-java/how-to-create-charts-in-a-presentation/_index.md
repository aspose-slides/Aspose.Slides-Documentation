{{% alert color="primary" %}} 
एक नया [Aspose.Slides for Java API](https://docs.aspose.com/slides/hi/java/) जारी किया गया है और अब यह एकल उत्पाद मूल से PowerPoint दस्तावेज़ बनाने और मौजूदा दस्तावेज़ों को संपादित करने की क्षमता का समर्थन करता है।
{{% /alert %}} 
## **पुराने कोड के लिए समर्थन**
14.x.x से पहले के Aspose.Slides for Java संस्करणों के साथ विकसित किए गए लेगेसी कोड का उपयोग करने के लिए, आपको अपने कोड में कुछ छोटे बदलाव करने होंगे और कोड पहले की तरह काम करेगा। पुराने Aspose.Slides for Java में com.aspose.slides और com.aspose.slides.pptx नामस्थान के तहत मौजूद सभी क्लासें अब एकल com.aspose.slides नामस्थान में मिल गई हैं। कृपया नीचे दिया गया सरल कोड स्निपेट देखें जो लेगेसी Aspose.Slides API का उपयोग करके प्रेज़ेंटेशन में मूल से एक सामान्य चार्ट बनाने के लिए है और नई मिलीजुली API में माइग्रेट करने के चरणों का पालन करें।
