एक नया [Aspose.Slides for Java API](/slides/hi/java/) जारी किया गया है और अब यह एकल उत्पाद प्रारंभ से PowerPoint दस्तावेज़ उत्पन्न करने और मौजूदा दस्तावेज़ों को संपादित करने की क्षमता प्रदान करता है।

{{% /alert %}} 
## **विरासत कोड के लिए समर्थन**
13.x से पहले के Aspose.Slides for Java संस्करणों में विकसित किए गए विरासत कोड का उपयोग करने के लिए, आपको अपने कोड में कुछ छोटे बदलाव करने होंगे और कोड पहले की तरह काम करेगा। Aspose.Slide और Aspose.Slides.Pptx नामस्थानों के तहत पुराने Aspose.Slides for Java में मौजूद सभी क्लासें अब एकल Aspose.Slides नामस्थान में मिल गई हैं। कृपया नीचे दिया गया सरल कोड स्निपेट देखें जो विरासत Aspose.Slides API में Hello World प्रस्तुति दस्तावेज़ बनाने के लिए है और नई सम्मिलित API में माइग्रेट करने के चरणों का पालन करें।
