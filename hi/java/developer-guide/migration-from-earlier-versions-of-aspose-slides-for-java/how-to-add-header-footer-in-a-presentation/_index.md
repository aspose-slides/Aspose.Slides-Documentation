एक नया [Aspose.Slides for Java API](https://docs.aspose.com/slides/hi/java/) जारी किया गया है और अब यह एकल उत्पाद शुरुआत से PowerPoint दस्तावेज़ बनाने और मौजूदा दस्तावेज़ों को संपादित करने की क्षमता प्रदान करता है।

{{% /alert %}} 
## **पुराने कोड के लिए समर्थन**
Aspose.Slides for Java के 13.x से पहले संस्करणों के साथ विकसित किए गए पुराने कोड का उपयोग करने के लिए, आपको अपने कोड में कुछ छोटे परिवर्तन करने होंगे और कोड पहले जैसा ही काम करेगा। पुराने Aspose.Slides for Java में Aspose.Slide और Aspose.Slides.Pptx नामस्थान के तहत मौजूद सभी क्लासेज अब एक ही Aspose.Slides नामस्थान में मिलाई गई हैं। कृपया नीचे प्रस्तुत सरल कोड स्निपेट देखें जो लेगसी Aspose.Slides API में प्रस्तुति में हेडर फ़ुटर जोड़ने के लिए है और नई मर्ज्ड API में माइग्रेट करने के चरणों का पालन करें।
