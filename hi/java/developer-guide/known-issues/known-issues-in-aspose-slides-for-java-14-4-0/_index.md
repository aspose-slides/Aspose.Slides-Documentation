Aspose.Slides for Java 14.4.0 PowerPoint दस्तावेज़ प्रोसेसिंग के लिए नई क्षमताएँ प्रदान करता है। कुछ प्रतिबंध और ज्ञात मुद्दे हैं, जिन्हें अगले रिलीज़ में हटाया जाएगा:

- कुछ शेप्स में क्रमबद्ध PPT दस्तावेज़ों में गलत ज्यामिति होती है (आर्क, सर्कुलर एरो, कॉलआउट्स)।
- सभी PPTX टेक्स्ट फ़ॉर्मेटिंग सुविधाएँ PPT क्रमबद्धता में समर्थित नहीं हैं (टैबुलेशन, इंडेंटेशन और पैराग्राफ फ़ॉर्मेटिंग में सीमाएँ)।
- टेक्स्ट भाषा और स्पेलिंग सेटिंग्स की जानकारी क्रमबद्ध PPT दस्तावेज़ों में उपलब्ध नहीं है।
- सभी PPTX थीम सुविधाएँ PPT क्रमबद्धता में समर्थित नहीं हैं (केवल फ़िल फ़ॉर्मेट, लाइन फ़ॉर्मेट और फ़ॉन्ट का क्रमबद्धन)।
- OLE/ActiveX के PPT से PPT क्रमबद्धन में ज्ञात समस्याएँ मौजूद हैं।
- WordArt का क्रमबद्धन और रेंडरिंग समर्थित नहीं है।
