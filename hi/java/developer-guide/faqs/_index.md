## **अवलोकन**

यह FAQ Aspose.Slides से संबंधित सामान्य प्रश्नों के उत्तर प्रदान करती है। यह समर्थित फ़ाइल फ़ॉर्मेट, बड़े प्रेज़ेंटेशन के साथ काम करते समय अपवादों को संभालना, स्लाइड आकार बदलना, स्लाइड का पूर्वावलोकन, प्रेज़ेंटेशन से टेक्स्ट प्राप्त करना, तालिका सीमाओं का फ़ॉर्मेटिंग, चित्र रखने और प्रेज़ेंटेशन को PDF या छवियों में बदलते समय फ़ॉन्ट‑संबंधित समस्याओं का समाधान शामिल करता है।

## **समर्थित फ़ाइल फ़ॉर्मेट**

**Q: Aspose.Slides for Java कौन से फ़ाइल फ़ॉर्मेट्स का समर्थन करता है?**

**A**: Aspose.Slides for Java वह फ़ाइल फ़ॉर्मेट समर्थन करता है जिसका विवरण [समर्थित फ़ाइल फ़ॉर्मेट](/slides/hi/java/supported-file-formats/) में दिया गया है।

## **अपवाद**

**Q: मैं बड़ी PPT फ़ाइल जिसमें चित्र हैं, लोड करते समय out of memory अपवाद प्राप्त कर रहा हूँ। क्या फ़ाइल आकार के संबंध में Aspose.Slides में कोई सीमा है?**

**A**: Aspose.Slides द्वारा समर्थित प्रेज़ेंटेशन आकार की गणना के लिए कोई विशिष्ट सूत्र नहीं है। मेमोरी में संपूर्ण प्रेज़ेंटेशन संरचना और चित्रों को समायोजित करने के लिए पर्याप्त स्थान होना चाहिए। सामान्यतः, मेमोरी में चित्र हार्ड डिस्क की तुलना में अधिक स्थान लेते हैं, विशेष रूप से जब चित्रों पर अतिरिक्त प्रभाव लागू हों।

सामान्यतः, Aspose.Slides for Java 4 GB RAM वाले सर्वर पर लगभग 300 MB आकार की प्रेज़ेंटेशन फ़ाइलों को आसानी से संभाल सकता है।

## **स्लाइड्स के साथ काम करना**

**Q: क्या मैं प्रेज़ेंटेशन में स्लाइड्स का आकार बदल सकता हूँ?**

**A**: आप `getSlideSize` मेथड का उपयोग कर सकते हैं जो कि [Presentation](https://reference.aspose.com/slides/hi/java/com.aspose.slides/presentation/) क्लास द्वारा प्रदान किया गया है, ताकि प्रेज़ेंटेशन में स्लाइड्स का आकार निर्धारित किया जा सके।

**Q: क्या प्रेज़ेंटेशन में विभिन्न आकार की स्लाइड्स निर्धारित करने का कोई तरीका है?**

**A**: चूँकि स्लाइड्स का आकार Microsoft PowerPoint दस्तावेज़ में प्रेज़ेंटेशन स्तर पर निर्धारित होता है, इसलिए इसे अलग‑अलग आकार में सेट करने का कोई तरीका नहीं है।

**Q: क्या Aspose.Slides for Java स्लाइड को सहेजने से पहले उसके पूर्वावलोकन का समर्थन करता है?**

**A**: आप प्रेज़ेंटेशन स्लाइड्स को छवियों में रेंडर कर सकते हैं और इन छवियों का उपयोग स्लाइड का पूर्वावलोकन करने के लिए कर सकते हैं।

## **पाठ के साथ काम करना**

**Q: क्या प्रेज़ेंटेशन से सभी टेक्स्ट प्राप्त करना संभव है?**

**A**: Aspose.Slides for Java वह [SlideUtil](https://reference.aspose.com/slides/hi/java/com.aspose.slides/slideutil/) क्लास प्रदान करता है जो प्रेज़ेंटेशन से पूर्ण टेक्स्ट प्राप्त करने के विभिन्न मेथड्स उपलब्ध कराता है।

**Q: Windows और Linux ऑपरेटिंग सिस्टम पर पैराग्राफ आकार अलग‑अलग क्यों होते हैं?**

**A**: पैराग्राफ आकार की गणना उस पैराग्राफ के प्रतिनिधित्व वाले टेक्स्ट आकार पर आधारित होती है। टेक्स्ट आकार की गणना PowerPoint प्रेज़ेंटेशन में निर्दिष्ट फ़ॉन्ट के मेट्रिक्स पर निर्भर करती है। यदि निर्दिष्ट फ़ॉन्ट अनुपलब्ध हो, तो उसे सबसे समान फ़ॉन्ट से प्रतिस्थापित किया जाता है, जिसके मेट्रिक्स मूल फ़ॉन्ट से भिन्न होते हैं। परिणामस्वरूप, विभिन्न प्रणालियों में स्थापित फ़ॉन्ट सेट के आधार पर पैराग्राफ आकार की गणना अलग‑अलग परिणाम देती है। विभिन्न ऑपरेटिंग सिस्टम पर समान परिणाम प्राप्त करने के लिए, आपको उन सिस्टम पर समान फ़ॉन्ट स्थापित करने होंगे या उन्हें रन‑टाइम पर [बाहरी फ़ॉन्ट्स](/slides/hi/java/custom-font/) के रूप में लोड करना होगा।

## **फ़ॉर्मेटिंग और छवियां**

**Q: मैं तालिका सीमा का रंग कैसे सेट कर सकता हूँ?**

**A**: आप सभी तालिका सीमाओं या पूरी तालिका की बाहरी सीमा का रंग बदल सकते हैं। सभी सीमाओं को बदलने के लिए कृपया [ICell](https://reference.aspose.com/slides/hi/java/com.aspose.slides/icell/) इंटरफ़ेस से `getCellFormat` मेथड का उपयोग करें। पूरी तालिका की बाहरी सीमा के लिए, आपको प्रत्येक सेल पर इटररेट करके बाहरी सीमाओं का रंग बदलना होगा।

**Q: चित्र रखने के लिए Aspose.Slides for Java कौन सी माप इकाई उपयोग करता है?**

**A**: स्लाइड पर सभी आकारों और समन्वयों को पॉइंट्स (72 dpi) में मापा जाता है।

## **फ़ॉन्ट्स के साथ काम करना**

**Q: PPT को PDF या छवियों में बदलते समय आउटपुट दस्तावेज़ों में फ़ॉन्ट अलग क्यों दिखते हैं?**

**A**: यह समस्या संकेत कर सकती है कि प्रेज़ेंटेशन में उपयोग किए गये फ़ॉन्ट को उस ऑपरेटिंग सिस्टम पर स्थापित नहीं किया गया है जहाँ कोड चलाया गया था। आपको ऑपरेटिंग सिस्टम पर फ़ॉन्ट स्थापित करना चाहिए या उन्हें नीचे दर्शाए गए अनुसार [FontsLoader](https://reference.aspose.com/slides/hi/java/com.aspose.slides/fontsloader/) क्लास का उपयोग करके बाहरी फ़ॉन्ट्स के रूप में लोड करना चाहिए:
```cs
var folders = new String[] { "path_to_a_folder_with_fonts" };
FontsLoader.loadExternalFonts(folders);
```
