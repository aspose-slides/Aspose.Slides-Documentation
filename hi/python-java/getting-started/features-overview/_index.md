## **समर्थित प्लेटफ़ॉर्म**
Aspose.Slides for Python via Java एक प्लेटफ़ॉर्म स्वतंत्र API है। इसे किसी भी प्लेटफ़ॉर्म (Windows, Linux, MacOS, आदि) पर इस्तेमाल किया जा सकता है जहाँ Python 3.7 या उससे ऊपर स्थापित हो। JDK 8 या उससे ऊपर एक अतिरिक्त आवश्यकता है।

## **फ़ाइल फ़ॉर्मैट और रूपांतरण**
Aspose.Slides for Python via Java अधिकांश PowerPoint दस्तावेज़ फ़ॉर्मैट्स को समर्थन देता है। यह आपको इन्हें उन लोकप्रिय फ़ाइल फ़ॉर्मैट्स में निर्यात करने की अनुमति देता है जो संगठन उपयोग करते हैं और आपस में साझा करते हैं। नीचे दिए गए विवरण देखें:

|**फ़ीचर**|**विवरण**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Python via Java इस प्रस्तुति दस्तावेज़ फ़ॉर्मैट के लिए सबसे तेज़ प्रोसेसिंग प्रदान करता है।|
|PPT to PPTX conversion|Aspose.Slides for Python via Java PPT से PPTX रूपांतरण का समर्थन करता है|
|Portable Document Format (PDF)|आप एक ही विधि से सभी समर्थित फ़ाइल फ़ॉर्मैट्स को Adobe Portable Document Format (PDF) दस्तावेज़ों में निर्यात कर सकते हैं।|
|XML Parser Specification (XPS)|आप एक ही विधि से सभी समर्थित फ़ाइल फ़ॉर्मैट्स को XML Parser Specification (XPS) दस्तावेज़ों में निर्यात कर सकते हैं।|
|Tagged Image File Format (TIFF)|आप सभी समर्थित प्रस्तुति फ़ाइल फ़ॉर्मैट्स को Tagged Image File Format (TIFF) में निर्यात कर सकते हैं।|
|PPTX To HTML Conversion|Aspose.Slides for Python via Java PresentationEx को HTML फ़ॉर्मैट में रूपांतरित करने का समर्थन करता है।|

## **रेंडरिंग और प्रिंटिंग**
Aspose.Slides for Python via Java प्रस्तुति में स्लाइड्स की उच्च-विश्वसनीयता रेंडरिंग को विभिन्न ग्राफ़िक्स फ़ॉर्मैट्स में समर्थन देता है। नीचे दिए गए विवरण देखें:

|**फ़ीचर**|**विवरण**|
| :- | :- |
|Java Supported Image Formats|Aspose.Slides for Python via Java का उपयोग करके, आप सभी Java-समर्थित ग्राफ़िक्स फ़ॉर्मैट्स जैसे TIFF, PNG, BMP, JPEG, GIF में प्रस्तुति स्लाइड्स और स्लाइड्स पर छवियों को रेंडर कर सकते हैं।|
|SVG Format|Aspose.Slides for Python via Java अंतर्निर्मित विधियाँ भी प्रदान करता है जो आपको प्रस्तुति स्लाइड्स को Scalable Vector Graphics (SVG) फ़ॉर्मैट में निर्यात करने की अनुमति देती हैं।|
|Presentation Printing|Aspose.Slides for Python via Java के नवीनतम संस्करण विभिन्न विकल्पों के साथ अंतर्निर्मित प्रिंट विधियाँ प्रदान करते हैं।|

## **सामग्री विशेषताएँ**
Aspose.Slides for Python via Java आपको प्रस्तुति दस्तावेज़ों के लगभग सभी आइटम या सामग्री तक पहुँचने, संशोधित करने या बनाने की अनुमति देता है। नीचे दिए गए विवरण देखें:

|**फ़ीचर**|**विवरण**|
| :- | :- |
|Master Slides|मास्टर स्लाइड्स सामान्य स्लाइड्स की लेआउट को परिभाषित करती हैं। Aspose.Slides for Python via Java आपको प्रस्तुति दस्तावेज़ों की मास्टर स्लाइड्स तक पहुँचने और उन्हें संशोधित करने की अनुमति देता है।|
|Normal Slides|Aspose.Slides for Python via Java के साथ आप विभिन्न प्रकार की नई स्लाइड्स बना सकते हैं; आप प्रस्तुतियों में मौजूदा स्लाइड्स तक पहुँच सकते हैं और उन्हें संशोधित कर सकते हैं।|
|Cloning / Copying Slides|Aspose.Slides for Python via Java द्वारा प्रदान की गई अंतर्निर्मित विधियाँ आपको एक प्रस्तुति के भीतर मौजूदा स्लाइड्स को क्लोन या कॉपी करने की अनुमति देती हैं। आप एक प्रस्तुति से दूसरी में कॉपी और क्लोन किए गए स्लाइड्स का उपयोग भी कर सकते हैं। क्योंकि स्लाइड अपना लेआउट मास्टर स्लाइड से विरासत में प्राप्त करता है, क्लोनिंग के दौरान अंतर्निर्मित विधियाँ स्वचालित रूप से मास्टर को कॉपी कर देती हैं।|
|Managing Slides sections|प्रस्तुति के भीतर विभिन्न सेक्शनों में स्लाइड्स को व्यवस्थित करने की विधियाँ|
|Place Holders and Text Holders|आप स्लाइड में प्लेसहोल्डर्स और टेक्स्ट होल्डर्स तक पहुँच सकते हैं। आगे, आप उचित विधि का उपयोग करके शून्य से टेक्स्ट होल्डर्स वाले स्लाइड बना सकते हैं।|
|Header and Footers|Aspose.Slides for Python via Java स्लाइड्स में हेडर/फ़ूटर को संभालने में सुविधा प्रदान करता है।|
|Notes in Slides|Aspose.Slides for Python via Java के साथ आप स्लाइड से जुड़ी नोट्स तक पहुँच सकते हैं, उन्हें संशोधित कर सकते हैं और नई नोट्स भी जोड़ सकते हैं।|
|Finding a Shape|आप एक स्लाइड से आकार का वैकल्पिक टेक्स्ट उपयोग करके विशेष आकार भी खोज सकते हैं।|
|Backgrounds|Aspose.Slides for Python via Java आपको प्रस्तुति में मास्टर या सामान्य स्लाइड से जुड़े बैकग्राउंड पर काम करने की अनुमति देता है।|
|Text Boxes|टेक्स्ट बॉक्स शून्य से बनाए जा सकते हैं। आप मौजूदा टेक्स्ट बॉक्स तक पहुँच सकते हैं। आप उनके टेक्स्ट को मूल फ़ॉर्मेट खोए बिना भी संशोधित कर सकते हैं।|
|Rectangle Shapes|आप Aspose.Slides for Python via Java के साथ आयताकार आकार बना या संशोधित कर सकते हैं।|
|Poly Line Shapes|आप Aspose.Slides for Python via Java के साथ पॉलीलाइन आकार बना या संशोधित कर सकते हैं।|

|Group Shapes|Aspose.Slides for Python via Java समूह आकारों का समर्थन करता है।|
|Auto Shapes|Aspose.Slides for Python via Java ऑटो शैप्स का समर्थन करता है।|
|SmartArt|Aspose.Slides for Python via Java MS PowerPoint में SmartArt आकारों के लिए समर्थन प्रदान करता है।|
|Charts|Aspose.Slides for Python via Java PowerPoint में MSO चार्ट्स के लिए समर्थन प्रदान करता है।|
|Shapes Serialization|Aspose.Slides for Python via Java बड़ी संख्या में आकारों का समर्थन करता है। जब कोई आकार समर्थित नहीं होता, तो आप एक सीरियलाइज़ेशन विधि का उपयोग कर सकते हैं जिससे आप उस आकार को मौजूदा स्लाइड से सीरियलाइज़ कर सकते हैं। इस प्रकार, आप आवश्यकता अनुसार उस आकार को आगे उपयोग कर सकते हैं।|
|Picture Frames|आप Aspose.Slides for Python via Java के साथ पिक्चर फ्रेम में चित्रों को प्रबंधित कर सकते हैं।|
|Audio Frames|आप Aspose.Slides for Python via Java के साथ स्लाइड्स में ऑडियो फ्रेम में ऑडियो फ़ाइलों को लिंक या एम्बेड कर सकते हैं।|

|OLE Frame|आप Aspose.Slides for Python via Java के साथ OLE फ्रेम में OLE ऑब्जेक्ट्स को प्रबंधित कर सकते हैं।|

|VBA Macros|प्रस्तुतियों के अंदर VBA मैक्रोज़ को प्रबंधित करने के लिए समर्थन।|
|Text Frame|आप किसी भी आकार के साथ जुड़े टेक्स्ट फ्रेम के माध्यम से उस आकार का टेक्स्ट एक्सेस कर सकते हैं।|

|Animations|आप आकारों पर एनीमेशन लागू कर सकते हैं।|
|Slide Shows|Aspose.Slides for Python via Java स्लाइड शो और स्लाइड ट्रांज़िशन का समर्थन करता है।|

Aspose.Slides for Python via Java के साथ आप प्रस्तुतियों में स्लाइड्स पर टेक्स्ट और आकारों को फ़ॉर्मेट कर सकते हैं। नीचे दिए गए विवरण देखें:

|**फ़ीचर**|**विवरण**|
| :- | :- |
|Text Formatting|<p>Aspose.Slides for Python via Java में, आप शैप्स से जुड़े टेक्स्ट फ्रेम्स के माध्यम से टेक्स्ट प्रबंधित कर सकते हैं। इसलिए, आप पैराग्राफ और हिस्सों के माध्यम से टेक्स्ट फ़ॉर्मेट कर सकते हैं। ये टेक्स्ट तत्व Aspose.Slides for Python via Java द्वारा फ़ॉर्मेट किए जा सकते हैं।</p><p>- फ़ॉन्ट प्रकार</p><p>- फ़ॉन्ट आकार</p><p>- फ़ॉन्ट रंग</p><p>- फ़ॉन्ट शेड्स</p><p>- पैराग्राफ संरेखण</p><p>- पैराग्राफ बुलेटिंग</p><p>- पैराग्राफ अभिविन्यास</p>|
|Shape Formatting|<p>Aspose.Slides for Python via Java में, स्लाइड का मूल तत्व एक आकार (शेप) है। आप इन आकार तत्वों को Aspose.Slides for Python via Java के साथ फ़ॉर्मेट कर सकते हैं:</p><p>- स्थिति</p><p>- आकार</p><p>- रेखा</p><p>- भराई (पैटर्न, ग्रेडिएंट, सॉलिड सहित)</p><p>- टेक्स्ट</p><p>- छवि</p>|
