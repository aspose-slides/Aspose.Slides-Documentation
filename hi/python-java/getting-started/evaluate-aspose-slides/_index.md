आप आसानी से Aspose.Slides को मूल्यांकन के लिए डाउनलोड कर सकते हैं। मूल्यांकन पैकेज खरीदी गई पैकेज के समान है। मूल्यांकन संस्करण केवल कुछ पंक्तियों कोड जोड़ने के बाद लाइसेंस लागू करने पर लाइसेंस प्राप्त हो जाता है। 

Aspose.Slides का मूल्यांकन संस्करण (बिना लाइसेंस निर्दिष्ट किए) पूर्ण उत्पाद कार्यक्षमता प्रदान करता है, लेकिन यह दस्तावेज़ को खोलने और सहेजने पर शीर्ष पर एक मूल्यांकन वॉटरमार्क डालता है। प्रेजेंटेशन स्लाइड्स से टेक्स्ट निकालते समय आपको केवल एक स्लाइड तक सीमित किया जाता है।
