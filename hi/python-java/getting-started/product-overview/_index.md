## **उत्पाद विवरण**

Aspose.Slides for Python via Java एक क्रॉस‑प्लेटफ़ॉर्म क्लास लाइब्रेरी है जो आपके अनुप्रयोगों को Microsoft PowerPoint® का उपयोग किए बिना विभिन्न प्रकार के दस्तावेज़ प्रोसेसिंग कार्य करने में सक्षम बनाती है।

### **आपको Aspose.Slides for Python via Java क्यों उपयोग करना चाहिए**

- Aspose.Slides for Python via Java पहला और एकमात्र घटक है जो PowerPoint® दस्तावेज़ों को प्रबंधित करने के लिए आवश्यक कार्यक्षमताएँ प्रदान करता है।  
- Aspose.Slides for Python via Java में कई प्रमुख सुविधाएँ हैं जैसे टेक्स्ट, शैलियों, तालिकाएँ और एनीमेशन प्रबंधन, स्लाइडों में ऑडियो और वीडियो जोड़ना, स्लाइडों का पूर्वावलोकन, स्लाइडों को SVG, PDF और अन्य फ़ॉर्मैट में निर्यात करना, तथा कई अन्य क्षमताएँ।  
- Aspose.Slides for Python via Java न केवल विभिन्न स्रोतों से PowerPoint फ़ाइलें खोलता है बल्कि आपके प्रस्तुतियों को कई तरीकों से सहेजने की भी अनुमति देता है। उदाहरण के लिए, आप अपने प्रस्तुतियों को PPT के रूप में सहेज सकते हैं; आप अपनी स्लाइडों को छवियों के रूप में भी सहेज सकते हैं।  
- Aspose.Slides for Python via Java आपको लगभग सभी फीचर प्रदान करता है जो आप Microsoft PowerPoint में या तो पा सकते हैं या नहीं। Aspose.Slides for Python via Java की बहुमुखी प्रतिभा के कारण, तालिकाओं और विभिन्न प्रकार की शैलियों के अतिरिक्त, आप इसे विभिन्न प्रकार के फ्रेम—टेक्स्ट, ऑडियो, और वीडियो फ्रेम—स्लाइडों में जोड़ने के लिए उपयोग कर सकते हैं।  

## **Aspose.Slides विशेषताएँ**

Aspose.Slides for Python via Java की विशेषताओं को इन समूहों में विभाजित किया गया है:

- प्रस्तुति सुविधाएँ।  
- स्लाइड सुविधाएँ।  
- ग्राफ़िक्स और मल्टीमीडिया सुविधाएँ।  
- तालिका सुविधाएँ।  
- टेक्स्ट और पैराग्राफ सुविधाएँ  
- उन्नत सुविधाएँ  

### **प्रस्तुति सुविधाएँ**

- विभिन्न स्रोतों से Microsoft PowerPoint प्रस्तुतियों को खोलना।  
- विभिन्न Microsoft PowerPoint फ़ॉर्मैट में प्रस्तुतियों को सहेजना।  
- शून्य से प्रस्तुतियाँ बनाना।  
- किसी अन्य घटक का उपयोग किए बिना प्रस्तुतियों को PDF दस्तावेज़ों में परिवर्तित करना।  
- प्रस्तुति के दस्तावेज़ गुणों को पढ़ना या संशोधित करना।  
- प्रस्तुति पर सुरक्षा लागू करना।  
- प्रस्तुति को PDF में निर्यात करना।  
- प्रस्तुति को TIFF में निर्यात करना।  
- प्रस्तुति को XPS में निर्यात करना।  
- प्रस्तुति को SWF में निर्यात करना।  
- प्रस्तुति को PDF नोट / Tiff नोट्स में निर्यात करना।  
- प्रस्तुति को HTML में निर्यात करना।  

### **स्लाइड सुविधाएँ**

- शून्य से नई स्लाइडें बनाना और उन्हें प्रस्तुतियों में जोड़ना।  
- प्रस्तुति में किसी भी स्लाइड तक उसकी स्थिति के माध्यम से पहुँचना।  
- स्लाइडों को समान या अन्य प्रस्तुति में कॉपी या क्लोन करना।  
- स्लाइडों को संपादित और हटाना।  
- स्लाइडों के साथ मास्टर को स्वचालित रूप से कॉपी करना।  
- स्लाइडों का पूर्वावलोकन करना।  
- स्लाइड का नाम प्राप्त करना या सेट करना।  
- स्लाइडों को छवियों के रूप में सहेजना।  
- स्लाइडों को SVG फ़ॉर्मैट में निर्यात करना।  
- प्रस्तुतियों में कई मास्टर के साथ काम करना।  
- स्लाइड पृष्ठभूमि को किसी भी समर्थित शैली से भरना।  
- नई स्लाइड नोट्स जोड़ना या प्रबंधित करना।  
- नई स्लाइड मास्टर और शैलियों को गतिशील रूप से सेट करना।  
- स्लाइड शो ट्रांज़िशन को जोड़ना या संशोधित करना।  
- मोर्फ़ ट्रांज़िशन जोड़ना।  

### **ग्राफ़िक्स और मल्टीमीडिया सुविधाएँ**

- रेखाएँ, आयत, दीर्घवृत्त और पॉलीलाइन जैसी शैलियों को बनाना या प्रबंधित करना।  
- शैलियों में रेखा शैलियों को प्रबंधित करना।  
- किसी shape का वैकल्पिक टेक्स्ट प्राप्त करना।  
- शैलियों को घुमाना या फ़्लिप करना।  
- विभिन्न शैलियों में shape को भरना—उदाहरण के लिए ठोस, ग्रेडिएंट, पैटर्न, टेक्सचर, और चित्र।  
- 48 पूर्वनिर्धारित पैटर्न और 24 टेक्सचर में से किसी का उपयोग करना।  
- शैलियों को लिंक असाइन करना।  
- स्लाइडों और शैलियों में नई एनीमेशन प्रभाव पढ़ना, संशोधित करना और जोड़ना।  
- प्रस्तुतियों में नई तस्वीरें जोड़ना और मौजूदा तस्वीरों का प्रबंधन करना।  
- प्रस्तुतियों में नई तस्वीर बुलेट्स जोड़ना और मौजूदा बुलेट्स का प्रबंधन करना।  
- प्रस्तुतियों में नई ऑडियो फ्रेम जोड़ना और मौजूदा फ्रेम का प्रबंधन करना।  
- प्रस्तुतियों में नई वीडियो फ्रेम जोड़ना और मौजूदा फ्रेम का प्रबंधन करना।  

### **तालिका सुविधाएँ**

- शून्य से तालिकाएँ बनाना और उन्हें स्लाइडों में जोड़ना।  
- तालिकाओं का आकार बदलना और उन्हें स्थानांतरित करना।  
- तालिका में किसी भी सेल तक पहुँचना।  
- किसी भी सेल के फ़ॉर्मैट और बॉर्डर को बदलना।  
- कॉलम और पंक्तियों का आकार बदलना।  
- कॉलम और पंक्तियों को जोड़ना और हटाना।  
- पूरी तालिका के लिए बॉर्डर की चौड़ाई और रंग सेट करना।  

### **टेक्स्ट और पैराग्राफ सुविधाएँ**

- टेक्स्ट प्लेसहोल्डर और टेक्स्ट फ्रेम में टेक्स्ट प्राप्त करना और सेट करना।  
- प्लेसहोल्डर का प्रकार प्राप्त करना।  
- टेक्स्ट फ्रेम में टेक्स्ट फ़ॉर्मेटिंग बदलना।  
- नई फ़ॉन्ट्स जोड़ना या प्रस्तुति में पहले से उपयोग की गई फ़ॉन्ट्स को प्रबंधित करना।  
- टेक्स्ट और टेक्स्ट फ़ॉर्मेटिंग को बदलना।  
- टेक्स्ट के लिंक बनाना या प्रबंधित करना।  
- टैब और प्रभावी टैब का समर्थन।  
- लगभग किसी भी प्रकार की shape में नई टेक्स्ट फ्रेम जोड़ना।  
- टेक्स्ट फ्रेम के मार्जिन और टेक्स्ट रैप सेटिंग्स को एक्सेस और संशोधित करना।  
- पैराग्राफ इंडेंटेशन को एक्सेस और संशोधित करना।  

### **उन्नत सुविधाएँ**

- प्रस्तुतियों में एम्बेडेड OLE ऑब्जेक्ट्स तक पहुँचना।  
- प्रस्तुतियों को लॉक करना।

- समर्थन के मुख्य माध्यम हैं [Aspose Forums](https://forum.aspose.com/)। अपना प्रश्न [Aspose.Slides forum](https://forum.aspose.com/c/slides/hi/11) में पोस्ट करें—और इसे कुछ घंटों के भीतर उत्तर मिलेगा। Aspose.Slides समर्थन टीम सीधे फ़ोरम पर पोस्ट किए गए प्रश्नों का उत्तर देती है।  
- कृपया ध्यान दें कि Aspose फोन पर तकनीकी समर्थन प्रदान नहीं करता है। फोन समर्थन केवल बिक्री और खरीद प्रश्नों के लिए उपलब्ध है।  
- फ़ोरम में उत्तर की अपेक्षा करते समय, कृपया धैर्य रखें और समय क्षेत्र भिन्नताओं के लिए मूल्यांकन करें।  

यदि आपके पास Aspose.Slides for Python via Java के साथ कोई समस्या है, तो इसे सबसे प्रभावी ढंग से हल करने के लिए नीचे दिए गए दिशानिर्देशों का पालन करें:

  नवीनतम उत्पाद संस्करण की जानकारी के लिए [Aspose.Slides download](https://releases.aspose.com/slides/hi/python-java/) देखें।
