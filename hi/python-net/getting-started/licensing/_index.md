## **परिचय**

Aspose.Slides को मूल्यांकन मोड में या वैध लाइसेंस के साथ उपयोग किया जा सकता है। मूल्यांकन संस्करण लाइसेंस प्राप्त संस्करण के समान कार्यक्षमता प्रदान करता है, लेकिन यह प्रस्तुतियों को खोलने या सहेजने पर मूल्यांकन वॉटरमार्क जोड़ता है और टेक्स्ट निष्कर्षण को एक स्लाइड तक सीमित करता है।

## **Aspose.Slides का मूल्यांकन करें**

आप **Aspose.Slides for Python via .NET** का मूल्यांकन संस्करण इसके [डाउनलोड पृष्ठ](https://pypi.org/project/Aspose.Slides/) से डाउनलोड कर सकते हैं। मूल्यांकन संस्करण लाइसेंस प्राप्त उत्पाद के समान सुविधाएँ प्रदान करता है। मूल्यांकन पैकेज खरीदे गए पैकेज के समान है और लाइसेंस लागू करने के लिए कुछ कोड पंक्तियों को जोड़ने के बाद लाइसेंस प्राप्त हो जाता है।

जब आप **Aspose.Slides** के मूल्यांकन से संतुष्ट हों, तो आप [लाइसेंस खरीदें](https://purchase.aspose.com/buy) सकते हैं। हम उपलब्ध सब्सक्रिप्शन विकल्पों की समीक्षा करने की सिफारिश करते हैं। यदि आपके कोई प्रश्न हैं, तो Aspose बिक्री टीम से संपर्क करें।

हर Aspose लाइसेंस में एक वर्ष की सब्सक्रिप्शन शामिल होती है, जिसमें इस अवधि के दौरान जारी किए गए नए संस्करणों और फिक्सेस के मुफ्त अपग्रेड शामिल हैं। लाइसेंस प्राप्त और मूल्यांकन उपयोगकर्ता दोनों को मुफ्त, असीमित तकनीकी समर्थन मिलता है।

* Aspose.Slides for Python via .NET आमतौर पर लाइसेंस को इन स्थानों में खोजता है:
  * आप द्वारा प्रदान किया गया स्पष्ट पथ
  * उस फ़ोल्डर जिसमें वह Python स्क्रिप्ट है जो Aspose.Slides for Python via .NET को कॉल करती है

एक लाइसेंस **फ़ाइल**, **स्ट्रीम**, या **एम्बेडेड रिसोर्स** से लोड किया जा सकता है।

Aspose.Slides लाइसेंसिंग को संभालने के लिए [License](https://reference.aspose.com/slides/hi/python-net/aspose.slides/license/) क्लास प्रदान करता है।

{{% alert color="warning" %}}
नए लाइसेंस केवल संस्करण 21.4 या बाद में Aspose.Slides को सक्रिय कर सकते हैं। पहले के संस्करण अलग लाइसेंसिंग सिस्टम का उपयोग करते हैं और इन लाइसेंसों को पहचानेंगे नहीं।
{{% /alert %}}

### **फ़ाइल**

लाइसेंस सेट करने का सबसे आसान तरीका यह है कि लाइसेंस फ़ाइल को घटक की DLL के समान फ़ोल्डर में रखें और केवल फ़ाइल नाम (बिना पथ के) निर्दिष्ट करें।

निम्नलिखित Python कोड लाइसेंस फ़ाइल को सेट करने का तरीका दर्शाता है:

```py
import aspose.slides as slides

# लाइसेंस क्लास का उदाहरण बनाता है। 
license = slides.License()

# लाइसेंस फ़ाइल पथ सेट करता है। 
license.set_license("Aspose.Slides.lic")
```

{{% alert color="warning" %}}
यदि आप लाइसेंस फ़ाइल को किसी अलग डायरेक्टरी में रखते हैं, तो जब आप [License.set_license()](https://reference.aspose.com/slides/hi/python-net/aspose.slides/license/set_license/#str) को कॉल करते हैं, तो स्पष्ट पथ के अंत में फ़ाइल नाम आपका लाइसेंस फ़ाइल का नाम होना चाहिए।

उदाहरण के लिए, आप लाइसेंस फ़ाइल का नाम *Aspose.Slides.lic.xml* रख सकते हैं। फिर, अपने कोड में, उस फ़ाइल का पूरा पथ (जो Aspose.Slides.lic.xml पर समाप्त होता है) [License.set_license()](https://reference.aspose.com/slides/hi/python-net/aspose.slides/license/set_license/#str) मेथड को पास करें।
{{% /alert %}}

### **स्ट्रीम**

आप स्ट्रीम से लाइसेंस लोड कर सकते हैं। निम्नलिखित Python उदाहरण स्ट्रीम से लाइसेंस लागू करने का तरीका दर्शाता है:

```py
import aspose.slides as slides

# लाइसेंस क्लास का उदाहरण बनाता है।
license = slides.License()

# स्ट्रीम से लाइसेंस सेट करता है।
license.set_license(stream)
```

## **लाइसेंस वैधता जाँच**

यह सुनिश्चित करने के लिए कि लाइसेंस सही ढंग से लागू हुआ है, आप इसे वैधता जाँच सकते हैं। निम्नलिखित Python कोड लाइसेंस को वैधता जाँचने का तरीका दर्शाता है:

```py
import aspose.slides as slides

license = slides.License()

license.set_license("Aspose.Slides.lic")

if license.is_licensed():
    print("License is good!")
```

## **अक्सर पूछे जाने वाले प्रश्न**

**क्या मैं लाइसेंस को पूरी तरह ऑफ़लाइन पर्यावरण (इंटरनेट एक्सेस नहीं) में लागू कर सकता हूँ?**

हाँ। लाइसेंस वैधता स्थानीय रूप से लाइसेंस फ़ाइल का उपयोग करके की जाती है; इंटरनेट कनेक्शन की आवश्यकता नहीं होती।

**एक वर्ष की सब्सक्रिप्शन समाप्त होने के बाद क्या होता है? क्या लाइब्रेरी काम करना बंद कर देगी?**

नहीं। लाइसेंस कायम है: आप सब्सक्रिप्शन समाप्ति तिथि से पहले जारी किए गए संस्करणों का उपयोग जारी रख सकते हैं; लेकिन नवीनीकरण के बिना नए रिलीज़ का उपयोग नहीं कर पाएंगे।
