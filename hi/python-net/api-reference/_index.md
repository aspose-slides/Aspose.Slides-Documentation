Aspose.Slides for Python via .NET एक क्लास लाइब्रेरी है जो Python अनुप्रयोगों को Microsoft PowerPoint® या अन्य सॉफ़्टवेयर का उपयोग किए बिना प्रेजेंटेशन दस्तावेज़ पढ़ने और लिखने में सक्षम बनाती है। Aspose.Slides for Python via .NET आपको टेक्स्ट, शैप्स, टेबल्स और एनीमेशन को प्रबंधित करने; स्लाइड्स में ऑडियो और वीडियो जोड़ने; स्लाइड्स का प्रीव्यू देखने; और स्लाइड्स को SVG, PDF फ़ॉर्मेट आदि में एक्सपोर्ट करने की अनुमति देती है।

नवीनतम API रेफ़रेंस यहाँ पाया जा सकता है: [Aspose.Slides for Python via .NET API Reference](https://reference.aspose.com/slides/hi/python-net/).
