## **सर्वर आवश्यकताएँ**
{{% alert color="primary" %}} 

Aspose.Slides for SharePoint को स्थापित करने के लिए, निम्नलिखित में से किसी एक को चलाने वाला सिस्टम आवश्यक है:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft Office SharePoint Server 2010
- Microsoft Office SharePoint Server 2013
- Microsoft Office SharePoint Server 2016
- Microsoft Office SharePoint Server 2019

{{% /alert %}} {{% alert color="primary" %}} 

इन उत्पादों के लिए सर्विस पैक 2 या उसके बाद वाला संस्करण स्थापित करने की सलाह दी जाती है।

{{% /alert %}} 
## **क्लाइंट आवश्यकताएँ**
{{% alert color="primary" %}} 

SharePoint साइटों तक वेब ब्राउज़र के माध्यम से पहुंचा जाता है। Aspose.Slides for SharePoint को निम्नलिखित ब्राउज़रों के साथ परीक्षण किया गया है:

- Microsoft Internet Explorer 6, 7, 8
- Mozilla FireFox 3.5

{{% /alert %}}
