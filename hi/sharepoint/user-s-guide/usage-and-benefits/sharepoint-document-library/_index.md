एक लाइब्रेरी एक ऐसी कंटेंट संरचना है जो फ़ाइलें रखती है। एक लाइब्रेरी में केवल एक प्रकार की फ़ाइल या कई विभिन्न प्रकार की फ़ाइलें हो सकती हैं, जिसमें दस्तावेज़, स्प्रेडशीट और प्रस्तुतियाँ शामिल हैं। एक SharePoint लाइब्रेरी एक एकल स्थान बनाती है जहाँ लोग दस्तावेज़ अपलोड और एक्सेस कर सकते हैं। दस्तावेज़ SQL Server डेटाबेस में द्विआधारी स्वरूप में संग्रहीत होते हैं, जिसका उपयोग SharePoint साइट करती है। ये दस्तावेज़ अन्य SharePoint उपयोगकर्ताओं के लिए हाइपरलिंक की सूची के रूप में उपलब्ध होते हैं।
