## **अवलोकन**

Aspose.Slides का उपयोग मूल्यांकन मोड में या वैध लाइसेंस के साथ किया जा सकता है। मूल्यांकन संस्करण लाइसेंस वाले संस्करण के समान कार्यक्षमता प्रदान करता है, लेकिन प्रस्तुतियों को खोलते या सहेजते समय एक मूल्यांकन वॉटरमार्क जोड़ता है तथा टेक्स्ट निकाला जाने पर केवल एक स्लाइड तक सीमा लगाता है।

यह लेख Aspose.Slides में लाइसेंस कैसे काम करता है और लाइब्रेरी का उपयोग करने से पहले लाइसेंस कैसे लागू किया जाता है, समझाता है। लाइसेंस `License` क्लास का उपयोग करके फ़ाइल, स्ट्रीम या एम्बेडेड रिसोर्स से लोड किया जा सकता है। लेख यह भी दिखाता है कि लाइसेंस सही ढंग से लागू हुआ है या नहीं, इसे कैसे सत्यापित किया जाए।

## **Aspose.Slides का मूल्यांकन**

{{% alert color="primary" %}} 

आप **Aspose.Slides for Android via Java** का मूल्यांकन संस्करण उसके [download page](https://releases.aspose.com/slides/hi/androidjava/) से डाउनलोड कर सकते हैं। मूल्यांकन संस्करण उत्पाद के लाइसेंस वाले संस्करण के समान कार्यक्षमताएँ प्रदान करता है। मूल्यांकन पैकेज खरीदे गए पैकेज के समान ही होता है। मूल्यांकन संस्करण केवल कुछ लाइन कोड जोड़ने (लाइसेंस लागू करने) के बाद लाइसेंस्ड बन जाता है।

हर Aspose लाइसेंस के साथ एक वर्ष की सब्सक्रिप्शन मिलती है, जिससे इस अवधि के भीतर रिलीज़ किए गए नए संस्करणों या फ़िक्स के लिए मुफ्त अपग्रेड मिलते हैं। लाइसेंस्ड उत्पाद (या यहाँ तक कि मूल्यांकन संस्करण) वाले उपयोगकर्ताओं को मुफ्त और असीमित तकनीकी समर्थन मिलता है।

## **Aspose.Slides में लाइसेंसिंग**

* एक मूल्यांकन संस्करण लाइसेंस प्राप्त करने के बाद और कुछ कोड लाइनें जोड़ने (लाइसेंस लागू करने) के बाद लाइसेंस्ड बन जाता है।  
* लाइसेंस एक साधारण टेक्स्ट XML फ़ाइल है जिसमें उत्पाद का नाम, लाइसेंस प्राप्त डेवलपर्स की संख्या, सब्सक्रिप्शन समाप्ति तिथि आदि विवरण होते हैं।  
* लाइसेंस फ़ाइल डिजिटल रूप से हस्ताक्षरित होती है, इसलिए आपको फ़ाइल में कोई बदलाव नहीं करने चाहिए। फ़ाइल की सामग्री में अनजाने में अतिरिक्त लाइन ब्रेक जोड़ने से भी यह अमान्य हो जाएगी।  
* Aspose.Slides for Android via Java सामान्यतः लाइसेंस को इन स्थानों में खोजता है:  
  * एक स्पष्ट पथ  
  * वह फ़ोल्डर जिसमें Aspose.Slides.jar स्थित है  
* मूल्यांकन संस्करण से जुड़ी सीमाओं से बचने के लिए, आपको **Aspose.Slides** का उपयोग करने से पहले लाइसेंस सेट करना आवश्यक है। आपको प्रत्येक एप्लिकेशन या प्रोसेस के लिये लाइसेंस केवल एक बार सेट करना होता है।

## **लाइसेंस लागू करना**

लाइसेंस को **फ़ाइल** या **स्ट्रीम** से लोड किया जा सकता है।

{{% alert color="primary" %}}

Aspose.Slides लाइसेंसिंग ऑपरेशन्स के लिये [License](https://reference.aspose.com/slides/hi/androidjava/com.aspose.slides/license/) क्लास प्रदान करता है।

{{% /alert %}} 

{{% alert color="warning" %}}

नये लाइसेंस केवल संस्करण 21.4 या बाद के साथ ही Aspose.Slides को सक्रिय कर सकते हैं। पुराने संस्करण अलग लाइसेंसिंग सिस्टम का उपयोग करते हैं और इन लाइसेंसों को पहचानेंगे नहीं।

{{% /alert %}}

### **फ़ाइल**

लाइसेंस सेट करने की सबसे आसान विधि यह है कि लाइसेंस फ़ाइल को Aspose.Slides.jar वाले फ़ोल्डर या आपके एप्लिकेशन की jar में रखें।

यह Java कोड दिखाता है कि लाइसेंस फ़ाइल कैसे सेट की जाए:

``` java
// License वर्ग का एक उदाहरण बनाता है
com.aspose.slides.License license = new com.aspose.slides.License();

// लाइसेंस फ़ाइल पथ सेट करता है
license.setLicense("Aspose.Slides.Android.via.Java.lic");
```

{{% alert color="warning" %}} 

यदि आप लाइसेंस फ़ाइल को किसी अलग डायरेक्टरी में रखते हैं, तो जब आप [SetLicense](https://reference.aspose.com/slides/hi/androidjava/com.aspose.slides/license/#setLicense-java.lang.String-) मेथड को कॉल करते हैं, तो निर्दिष्ट स्पष्ट पथ के अंत में लाइसेंस फ़ाइल का नाम आपके लाइसेंस फ़ाइल के नाम के समान होना चाहिए।

उदाहरण के तौर पर, आप लाइसेंस फ़ाइल का नाम *Aspose.Slides.Android.via.Java.lic.xml* में बदल सकते हैं। फिर, अपने कोड में, आपको फ़ाइल का पथ (अंत में *Aspose.Slides.Android.via.Java.lic.xml* वाला) को [SetLicense](https://reference.aspose.com/slides/hi/androidjava/com.aspose.slides/license/#setLicense-java.lang.String-) मेथड को पास करना होगा।

{{% /alert %}}

### **स्ट्रीम**

आप लाइसेंस को स्ट्रीम से लोड कर सकते हैं। यह Java कोड दिखाता है कि स्ट्रीम से लाइसेंस कैसे लागू किया जाए:

``` java
// License वर्ग का एक उदाहरण बनाता है
com.aspose.slides.License license = new com.aspose.slides.License();

// स्ट्रीम के माध्यम से लाइसेंस सेट करता है
license.setLicense(new java.io.FileInputStream("Aspose.Slides.Android.via.Java.lic"));
```

## **लाइसेंस का सत्यापन**

यह जांचने के लिए कि लाइसेंस सही ढंग से सेट हुआ है या नहीं, आप इसे सत्यापित कर सकते हैं। यह Java कोड दिखाता है कि लाइसेंस कैसे सत्यापित किया जाए:

```java
License license = new License();
license.setLicense("Aspose.Slides.Android.via.Java.lic");

if (License.isLicensed()) 
{
    System.out.println("License is good!");
}
```

## **अक्सर पूछे जाने वाले प्रश्न**

**क्या मैं लाइसेंस को पूरी तरह ऑफ़लाइन वातावरण (कोई इंटरनेट एक्सेस नहीं) में लागू कर सकता हूँ?**

हां। लाइसेंस सत्यापन स्थानीय रूप से लाइसेंस फ़ाइल का उपयोग करके किया जाता है; इंटरनेट कनेक्शन की कोई आवश्यकता नहीं है।

**एक वर्ष की सब्सक्रिप्शन समाप्त होने के बाद क्या होता है? क्या लाइब्रेरी काम करना बंद कर देगी?**

नहीं। लाइसेंस स्थायी है: आप अपनी सब्सक्रिप्शन समाप्ति तिथि से पहले जारी किए गए संस्करणों का उपयोग जारी रख सकते हैं; केवल नए रिलीज़ के लिये आपको नवीनीकरण करना होगा।
