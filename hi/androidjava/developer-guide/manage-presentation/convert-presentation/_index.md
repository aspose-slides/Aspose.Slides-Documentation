## **परिचय**

Aspose.Slides for Android via Java PowerPoint और OpenDocument प्रस्तुतियों को लोड कर सकता है और उन्हें कई अन्य प्रारूपों में सहेज या रेंडर कर सकता है, बिना Microsoft PowerPoint, OpenOffice या LibreOffice के। आप पुरानी PPT फ़ाइलों को आधुनिक PPTX में बदल सकते हैं, प्रस्तुतियों को PDF और XPS जैसे फिक्स्ड‑लेआउट दस्तावेज़ों में निर्यात कर सकते हैं, स्लाइडों को HTML के रूप में प्रकाशित कर सकते हैं, या पूर्वावलोकन, थंबनेल और अभिलेखों के लिए स्लाइडों को छवि फ़ाइलों के रूप में रेंडर कर सकते हैं।

अधिकांश दस्तावेज़ रूपांतरण समान सामान्य कार्य‑प्रवाह का उपयोग करते हैं: स्रोत फ़ाइल को लोड करें, आवश्यक आउटपुट फ़ॉर्मेट चुनें, और आवश्यक होने पर फ़ॉर्मेट‑विशिष्ट विकल्प लागू करें। छवि फ़ॉर्मेट के लिए, प्रत्येक स्लाइड को अलग‑अलग रेंडर किया जाता है और फिर उसे रास्टर या वेक्टर छवि के रूप में सहेजा जाता है। नीचे लिंक किए गए समर्पित लेख प्रत्येक स्थिति के कार्यान्वयन विवरण प्रदान करते हैं।

## **PowerPoint और OpenDocument रूपांतरण**

Aspose.Slides for Android via Java PPT, PPTX, PPS, PPSX, POT, POTX और ODP जैसे सामान्यतः प्रयुक्त प्रस्तुति फ़ॉर्मेटों के बीच रूपांतरण का समर्थन करता है। PowerPoint और OpenDocument फ़ाइलों के लिए समान रूपांतरण API उपयोग किया जाता है, इसलिए PPTX फ़ाइल को PDF में सहेजने वाला वर्कफ़्लो आमतौर पर केवल इनपुट फ़ाइल बदल कर ODP फ़ाइल पर लागू किया जा सकता है।

ODP फ़ाइलों को बदलते समय याद रखें कि PowerPoint और OpenDocument अनुप्रयोग प्रत्येक लेआउट और फ़ॉर्मेटिंग सुविधा को बिल्कुल समान तरीके से समर्थन नहीं करते। यदि ODP फ़ाइल LibreOffice या OpenOffice Impress में बनाई गई थी, तो आउटपुट की समीक्षा करें और [OpenDocument प्रस्तुतियों को बदलें](/slides/hi/androidjava/convert-openoffice-odp/) में वर्णित विकल्पों का उपयोग करें जब आपको फ़ॉर्मेट‑विशिष्ट मार्गदर्शन चाहिए।

## **PPT से PPTX रूपांतरण**

PPT पुराना बाइनरी PowerPoint फ़ॉर्मेट है, जबकि PPTX आधुनिक Office Open XML फ़ॉर्मेट है। Aspose.Slides for Android via Java जटिल प्रस्तुति संरचनाओं जैसे मास्टर, लेआउट, स्लाइड, चार्ट, ग्रुप्ड शेप, प्लेसहोल्डर, टेक्स्ट फ्रेम, टेक्सचर और पिक्चर फ़िल्स को संरक्षित रखते हुए उच्च‑सटीकता वाला PPT से PPTX रूपांतरण समर्थन करता है।

विवरण के लिए देखें [PPT को PPTX में बदलें](/slides/hi/androidjava/convert-ppt-to-pptx/) और [PPT बनाम PPTX](/slides/hi/androidjava/ppt-vs-pptx/)।

## **फ़िक्स्ड‑लेआउट निर्यात**

PDF, XPS और TIFF उपयोगी होते हैं जब आउटपुट को सभी उपकरणों पर समान दिखना चाहिए और इसे प्रस्तुति के रूप में संपादित नहीं किया जाना चाहिए। समर्पित PDF, XPS और TIFF लेख compliance, hidden slides, notes, image quality, compression, pixel format और output size को नियंत्रित करने के तरीकों को समझाते हैं।

## **HTML और छवि निर्यात**

HTML और HTML5 निर्यात ब्राउज़र दर्शन, वेब प्रकाशन और हल्के साझा करने के लिए उपयोगी हैं। छवि निर्यात तब उपयोगी है जब प्रत्येक स्लाइड को अलग‑अलग पूर्वावलोकन, थंबनेल या रास्टर संपत्ति बनाना हो। फ़ॉर्मेट‑विशिष्ट रेंडरिंग मार्गदर्शन के लिए PNG, JPG और SVG लेख देखें।

## **अक्सर पूछे जाने वाले प्रश्न**

**क्या प्रस्तुतियों को बदलने के लिए मुझे Microsoft PowerPoint की आवश्यकता है?**

नहीं। Aspose.Slides for Android via Java एक स्टैंडअ्लोन लाइब्रेरी है और Microsoft PowerPoint या Office ऑटोमेशन की आवश्यकता नहीं होती।

**क्या मैं कई प्रस्तुतियों को बैच में बदल सकता हूँ?**

हां। प्रत्येक प्रस्तुति को लोड करें, आवश्यक फ़ॉर्मेट में सहेजें, और प्रोसेसिंग के बाद प्रस्तुति ऑब्जेक्ट को डिस्पोज़ कर दें। समानांतर प्रोसेसिंग के लिए अलग‑अलग प्रस्तुति इंस्टेंस उपयोग करें और [멀티थ्रेडिंग](/slides/hi/androidjava/multithreading/) मार्गदर्शन का पालन करें।

**क्या मैं केवल चयनित स्लाइडों को निर्यात कर सकता हूँ?**

हां। कई निर्यात विधियां आपको स्लाइड इंडेक्स पास करने या आउटपुट फ़ॉर्मेट के आधार पर व्यक्तिगत स्लाइड रेंडर करने की अनुमति देती हैं। लक्ष्य फ़ॉर्मेट के लिए समर्पित लेख देखें।

**क्या PDF या XPS निर्यात करते समय छिपी हुई स्लाइडों को शामिल किया जा सकता है?**

हां। PDF और XPS रूपांतरण लेखों में वर्णित hidden‑slide निर्यात सेटिंग्स का उपयोग करें।

**क्या मैं PDF/A आउटपुट बना सकता हूँ?**

हां। PDF निर्यात के लिए compliance सेटिंग्स उपलब्ध हैं। विवरण के लिए देखें [PowerPoint को PDF में बदलें](/slides/hi/androidjava/convert-powerpoint-to-pdf/)।

**रूपांतरण के दौरान फ़ॉन्ट्स कैसे संभाले जाते हैं?**

Aspose.Slides एम्बेडेड फ़ॉन्ट्स, फ़ॉन्ट फ़ॉलबैक और फ़ॉन्ट सब्स्टीट्यूशन सेटिंग्स का उपयोग कर सकता है। देखें [Embedded Font](/slides/hi/androidjava/embedded-font/), [Fallback Font](/slides/hi/androidjava/fallback-font/) और [Font Substitution](/slides/hi/androidjava/font-substitution/).
