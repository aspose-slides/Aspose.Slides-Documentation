**Aspose.Slides for Node.js via .NET में आपका स्वागत है**

![Aspose.Slides for Node.js via .NET उत्पाद लोगो](aspose_slides-for-nodejs-via-net.png)

Aspose.Slides for Node.js via .NET एक क्लास लाइब्रेरी है जो आपके अनुप्रयोगों को Microsoft PowerPoint® का उपयोग किए बिना PowerPoint® दस्तावेज़ पढ़ने और लिखने में सक्षम बनाती है।

Aspose.Slides for Node.js via .NET पहला और केवल एकमात्र घटक है जो PowerPoint® दस्तावेज़ों को प्रबंधित करने की कार्यक्षमता प्रदान करता है।

Aspose.Slides for Node.js via .NET कई प्रमुख सुविधाएँ प्रदान करता है जैसे कि टेक्स्ट, शैलियां, तालिकाओं और एनीमेशन का प्रबंधन, स्लाइड्स में ऑडियो और वीडियो जोड़ना, स्लाइड्स का पूर्वावलोकन, स्लाइड्स को SVG, PDF प्रारूप में निर्यात करना और अन्य कई चीजें।
