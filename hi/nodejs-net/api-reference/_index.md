Aspose.Slides for Node.js via .NET एक क्लास लाइब्रेरी है जो Node.js एप्लिकेशन को माइक्रोसॉफ्ट PowerPoint® या अन्य उपकरणों का उपयोग किए बिना प्रेज़ेंटेशन दस्तावेज़ पढ़ने और लिखने में सक्षम बनाती है। Aspose.Slides for Node.js via .NET आपको टेक्स्ट, शेप्स, टेबल्स एवं एनीमेशन को प्रबंधित करने, स्लाइड्स में ऑडियो और वीडियो जोड़ने, स्लाइड्स का प्रीव्यू करने, स्लाइड्स को SVG, PDF फ़ॉर्मेट में निर्यात करने और बहुत कुछ करने की सुविधा देता है।

Aspose.Slides for Node.js via .NET uses the [Aspose.Slides .NET के समान API रेफ़रेंस](https://reference.aspose.com/slides/hi/nodejs-net/).
