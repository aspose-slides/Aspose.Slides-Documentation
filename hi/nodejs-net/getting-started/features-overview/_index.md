## **समर्थित प्लेटफ़ॉर्म**
Aspose.Slides for Node.js via .NET एक प्लेटफ़ॉर्म स्वतंत्र API है। इसे किसी भी प्लेटफ़ॉर्म (Windows, Linux, MacOS, आदि) पर उपयोग किया जा सकता है जहाँ Node.js 7 या उससे अधिक स्थापित हो। Oracle JDK 8 या उससे अधिक एक अन्य आवश्यकता है।

## **फ़ाइल फ़ॉर्मेट और रूपांतरण**
Aspose.Slides for Node.js via .NET अधिकांश PowerPoint दस्तावेज़ फ़ॉर्मेट का समर्थन करता है। यह आपको उन्हें उन लोकप्रिय फ़ाइल फ़ॉर्मेट में निर्यात करने की सुविधा देता है जो संगठन उपयोग करते हैं और आपस में साझा करते हैं। विवरण देखें:

|**फ़ीचर**|**विवरण**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via .NET इस प्रेज़ेंटेशन दस्तावेज़ फ़ॉर्मेट के लिए सबसे तेज़ प्रोसेसिंग प्रदान करता है।|
|PPT to PPTX conversion|Aspose.Slides for Node.js via .NET PPT से PPTX रूपांतरण का समर्थन करता है।|
|Portable Document Format (PDF)|आप एक ही मेथड से सभी समर्थित फ़ाइल फ़ॉर्मेट को Adobe Portable Document Format (PDF) दस्तावेज़ों में निर्यात कर सकते हैं।|
|XML Parser Specification (XPS)|आप एक ही मेथड से सभी समर्थित फ़ाइल फ़ॉर्मेट को XML Parser Specification (XPS) दस्तावेज़ों में निर्यात कर सकते हैं।|
|Tagged Image File Format (TIFF)|आप सभी समर्थित प्रेज़ेंटेशन फ़ाइल फ़ॉर्मेट को Tagged Image File Format (TIFF) में निर्यात कर सकते हैं।|
|PPTX To HTML Conversion|Aspose.Slides for Node.js via .NET PresentationEx को HTML फ़ॉर्मेट में रूपान्तरण का समर्थन करता है।|

## **रेंडरिंग और प्रिंटिंग**
Aspose.Slides for Node.js via .NET विभिन्न ग्राफ़िक्स फ़ॉर्मेट में प्रेज़ेंटेशन स्लाइड्स की उच्च फ़िडेलिटी रेंडरिंग का समर्थन करता है। विवरण देखें:

|**फ़ीचर**|**विवरण**|
| :- | :- |
|.NET Supported Image Formats|Aspose.Slides for Node.js via .NET का उपयोग करके आप प्रेज़ेंटेशन स्लाइड्स और स्लाइड्स पर छवियों को सभी .NET समर्थित ग्राफ़िक्स फ़ॉर्मेट जैसे TIFF, PNG, BMP, JPEG, GIF में रेंडर कर सकते हैं।|
|SVG Format|Aspose.Slides for Node.js via .NET अंतर्निर्मित मेथड प्रदान करता है जो प्रेज़ेंटेशन स्लाइड्स को Scalable Vector Graphics (SVG) फ़ॉर्मेट में निर्यात करने की अनुमति देता है।|
|Presentation Printing|Aspose.Slides for Node.js via .NET के नवीनतम संस्करण विभिन्न विकल्पों के साथ अंतर्निर्मित प्रिंट मेथड प्रदान करते हैं।|

|**फ़ीचर**|**विवरण**|
| :- | :- |
