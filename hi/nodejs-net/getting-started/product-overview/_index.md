## **उत्पाद विवरण**

Aspose.Slides for Node.js via .NET एक क्रॉस‑प्लेटफ़ॉर्म क्लास लाइब्रेरी है जो आपके अनुप्रयोगों को Microsoft PowerPoint® का उपयोग किए बिना विभिन्न दस्तावेज़ प्रसंस्करण कार्यों को करने में सक्षम बनाती है।

### **आपको Aspose.Slides for Node.js via .NET क्यों उपयोग करना चाहिए**

- Aspose.Slides for Node.js via .NET वह पहला और एकमात्र घटक है जो PowerPoint® दस्तावेज़ों को प्रबंधित करने के लिए आवश्यक कार्यक्षमताएँ प्रदान करता है।  
- Aspose.Slides for Node.js via .NET कई प्रमुख सुविधाएँ प्रदान करता है जैसे पाठ, आकार, तालिकाएँ एवं एनीमेशन प्रबंधन, स्लाइड्स में ऑडियो और वीडियो जोड़ना, स्लाइड्स का पूर्वावलोकन, स्लाइड्स को SVG, PDF और अन्य फ़ॉर्मैट में निर्यात करना, और कई अन्य क्षमताएँ।  
- Aspose.Slides for Node.js via .NET न केवल विभिन्न स्रोतों से PowerPoint फ़ाइलें खोलता है बल्कि आपकी प्रस्तुतियों को विभिन्न तरीकों से सहेजने की अनुमति भी देता है। उदाहरण के लिए, आप अपनी प्रस्तुतियों को PPT में सहेज सकते हैं; आप स्लाइड्स को इमेज के रूप में भी सहेज सकते हैं।  
- Aspose.Slides for Node.js via .NET आपको लगभग सभी सुविधाएँ देता है जो आप Microsoft PowerPoint में पा सकते हैं या नहीं पा सकते। Aspose.Slides for Node.js via .NET की बहुमुखी प्रतिभा के साथ, तालिकाओं और विभिन्न प्रकार के आकारों के अलावा, आप अपने स्लाइड्स में विभिन्न प्रकार के फ्रेम—पाठ, ऑडियो और वीडियो फ्रेम—जोड़ सकते हैं।  

## **Aspose.Slides विशेषताएँ**

Aspose.Slides for Node.js via .NET की विशेषताएँ इन समूहों में विभाजित की जा सकती हैं:

- Presentation विशेषताएँ।  
- Slides विशेषताएँ।  
- Graphics और Multimedia विशेषताएँ।  
- Table विशेषताएँ।  
- Text & Paragraph विशेषताएँ  
- Advanced विशेषताएँ  

### **Presentation विशेषताएँ**

- विभिन्न स्रोतों से Microsoft PowerPoint प्रस्तुतियों को खोलना।  
- विभिन्न Microsoft PowerPoint फ़ॉर्मैट में प्रस्तुतियों को सहेजना।  
- शून्य से प्रस्तुतियाँ बनाना।  
- किसी अन्य घटक का उपयोग किए बिना प्रस्तुतियों को PDF दस्तावेज़ में परिवर्तित करना।  
- प्रस्तुति के डॉक्युमेंट प्रॉपर्टी पढ़ना या संशोधित करना।  
- प्रस्तुति पर सुरक्षा लागू करना।  
- प्रस्तुति को PDF में निर्यात करना।  
- प्रस्तुति को TIFF में निर्यात करना।  
- प्रस्तुति को XPS में निर्यात करना।  
- प्रस्तुति को SWF में निर्यात करना।  
- प्रस्तुति को PDF नोट / Tiff नोट्स में निर्यात करना।  
- प्रस्तुति को HTML में निर्यात करना।  

### **Slides विशेषताएँ**

- शून्य से नई स्लाइड्स बनाना और उन्हें प्रस्तुतियों में जोड़ना।  
- प्रस्तुति में किसी स्लाइड की स्थिति के माध्यम से उसका एक्सेस करना।  
- स्लाइड को उसी या किसी अन्य प्रस्तुति में कॉपी या क्लोन करना।  
- स्लाइड्स को संपादित और हटाना।  
- स्लाइड्स के साथ मास्टर को स्वचालित रूप से कॉपी करना।  
- स्लाइड्स का पूर्वावलोकन करना।  
- स्लाइड का नाम प्राप्त या निर्धारित करना।  
- स्लाइड्स को इमेज के रूप में सहेजना।  
- स्लाइड्स को SVG फ़ॉर्मैट में निर्यात करना।  
- प्रस्तुतियों में कई मास्टर के साथ काम करना।  
- किसी भी समर्थित शैली से स्लाइड पृष्ठभूमि को भरना।  
- नई स्लाइड नोट्स जोड़ना या प्रबंधित करना।  
- नई स्लाइड मास्टर और शैलियों को गतिशील रूप से सेट करना।  
- स्लाइड शो ट्रांज़िशन को जोड़ना या संशोधित करना।  
- Morph ट्रांज़िशन जोड़ना।  

### **Graphics & Multimedia विशेषताएँ**

- रेखाएँ, आयतें, अण्डाकार और पॉलीलाइन जैसे आकार बनाना या प्रबंधित करना।  
- आकारों में रेखा शैलियों को प्रबंधित करना।  
- किसी आकार के वैकल्पिक पाठ को प्राप्त करना।  
- आकारों को घुमाना या फ़्लिप करना।  
- विभिन्न शैलियों में आकारों को भरना—जैसे ठोस, ग्रेडिएंट, पैटर्न, टेक्सचर, और चित्र।  
- 48 पूर्वनिर्धारित पैटर्न और 24 टेक्सचर का उपयोग करना।  
- आकारों को लिंक असाइन करना।  
- स्लाइड्स और आकारों में नई एनीमेशन प्रभाव पढ़ना, संशोधित करना और जोड़ना।  
- प्रस्तुतियों में नई तस्वीरें जोड़ना और मौजूदा तस्वीरों को प्रबंधित करना।  
- प्रस्तुतियों में नई चित्र बुलेट्स जोड़ना और मौजूदा बुलेट्स को प्रबंधित करना।  
- प्रस्तुतियों में नई ऑडियो फ्रेम जोड़ना और मौजूदा फ्रेम को प्रबंधित करना।  
- प्रस्तुतियों में नई वीडियो फ्रेम जोड़ना और मौजूदा फ्रेम को प्रबंधित करना।  

### **Table विशेषताएँ**

- शून्य से तालिकाएँ बनाना और उन्हें स्लाइड्स में जोड़ना।  
- तालिकाओं का आकार बदलना और उन्हें स्थानांतरित करना।  
- तालिका में किसी भी सेल तक पहुँच।  
- किसी भी सेल के फ़ॉर्मेट और बॉर्डर बदलना।  
- कॉलम और पंक्तियों का आकार बदलना।  
- कॉलम और पंक्तियों को जोड़ना और हटाना।  
- पूरी तालिका के लिए बॉर्डर की चौड़ाई और रंग सेट करना।  

### **Text & Paragraph विशेषताएँ**

- टेक्स्ट प्लेसहोल्डर और टेक्स्ट फ्रेम में टेक्स्ट प्राप्त और सेट करना।  
- प्लेसहोल्डर का प्रकार प्राप्त करना।  
- टेक्स्ट फ्रेम में टेक्स्ट फ़ॉर्मेटिंग बदलना।  
- नई फ़ॉन्ट जोड़ना या प्रस्तुतियों में पहले से उपयोग की गई फ़ॉन्ट को प्रबंधित करना।  
- टेक्स्ट और टेक्स्ट फ़ॉर्मेटिंग बदलना।  
- टेक्स्ट के लिंक बनाना या प्रबंधित करना।  
- टैब और प्रभावी टैब का समर्थन।  
- लगभग किसी भी प्रकार के आकार में नई टेक्स्ट फ्रेम जोड़ना।  
- टेक्स्ट फ्रेम के मार्जिन और टेक्स्ट रैप सेटिंग्स को एक्सेस और संशोधित करना।  
- पैराग्राफ इंडेंटेशन को एक्सेस और संशोधित करना।  

### **Advanced विशेषताएँ**

- प्रस्तुतियों में एम्बेडेड OLE ऑब्जेक्ट्स तक पहुँच।  
- प्रस्तुतियों को लॉक करना।  

## **तकनीकी समर्थन**

Aspose अपने सभी उत्पादों के लिए असीमित मुफ्त तकनीकी समर्थन प्रदान करता है। समर्थन सभी उपयोगकर्ताओं (जिसमें मूल्यांकन पैकेज वाले उपयोगकर्ता भी शामिल हैं) के लिए उपलब्ध है। यदि आपको Aspose.Slides for Node.js via .NET के साथ मदद की आवश्यकता है, तो नीचे दिए गए बिंदुओं पर विचार करें:

- समर्थन का मुख्य माध्यम [Aspose फ़ोरम्स](https://forum.aspose.com/) है। अपने प्रश्न को [Aspose.Slides फ़ोरम](https://forum.aspose.com/c/slides/hi/11) में पोस्ट करें—और इसे कुछ घंटों के भीतर उत्तर मिल जाएगा। Aspose.Slides समर्थन टीम फ़ोरम पर पोस्ट किए गए प्रश्नों के सीधे उत्तर देती है।  
- कृपया ध्यान दें कि Aspose फ़ोन पर तकनीकी समर्थन प्रदान नहीं करता। फ़ोन समर्थन केवल बिक्री और खरीद संबंधी प्रश्नों के लिए उपलब्ध है।  
- फ़ोरम में उत्तर की प्रतीक्षा करते समय, कृपया धैर्य रखें और समय क्षेत्र अंतर के अनुसार मूल्यांकन करें।  

यदि आपके पास Aspose.Slides for Node.js via .NET से संबंधित कोई समस्या है, तो इसे सबसे कुशल तरीके से हल करने के लिए नीचे दिए गए दिशानिर्देशों का पालन करें:

- किसी समस्या की रिपोर्ट करने से पहले सुनिश्चित करें कि आप सबसे नवीनतम Aspose.Slides for Node.js via .NET संस्करण का उपयोग कर रहे हैं। संभवतः आपकी समस्या का समाधान किसी अपडेट में पहले ही हो चुका हो।  

  नवीनतम उत्पाद संस्करण की जानकारी के लिए देखें [Aspose.Slides डाउनलोड](https://releases.aspose.com/slides/hi/nodejs-net/)।  

- समस्या रिपोर्ट करने से पहले, आप हमारे फ़ोरम, इस दस्तावेज़ और API रेफ़रेंस को देख सकते हैं। संभवतः आपका प्रश्न पहले ही उत्तरित हो चुका है।  

- समस्या रिपोर्ट करते समय, कृपया मूल दस्तावेज़ और (यदि संभव हो) समस्या से संबंधित कोड का एक भाग शामिल करें। यदि आपको कई फ़ाइलें संलग्न करनी हों, तो उन्हें ZIP पैकेज में रखें।  

  आप अपने दस्तावेज़ Aspose.Forums में सुरक्षित रूप से संलग्न कर सकते हैं क्योंकि केवल आप और Aspose डेवलपर्स को संलग्न फ़ाइलों तक पहुंच होगी।  

- विशिष्ट समस्या के लिए एक थ्रेड बनाएँ और उसका उपयोग करें। यदि आपको कोई अन्य समस्या आती है, तो उसे रिपोर्ट करने के लिए एक नया थ्रेड बनाएँ।  

यह तालिका Aspose.Slides for Node.js via .NET से संबंधित महत्वपूर्ण तकनीकी संसाधनों को सूचीबद्ध करती है।
