## **Aspose.Slides का मूल्यांकन करें**
आप आसानी से Aspose.Slides को मूल्यांकन के लिए डाउनलोड कर सकते हैं। मूल्यांकन पैकेज खरीदे गए पैकेज के समान है। मूल्यांकन संस्करण बस कुछ पंक्तियों का कोड जोड़ने के बाद लाइसेंस लागू करने पर लाइसेंस प्राप्त हो जाता है।

## **मूल्यांकन संस्करण की सीमाएं**
Aspose.Slides का मूल्यांकन संस्करण (बिना निर्दिष्ट लाइसेंस के) पूर्ण उत्पाद कार्यक्षमता प्रदान करता है, लेकिन यह खोलने और सहेजने पर दस्तावेज़ के शीर्ष पर मूल्यांकन वॉटरमार्क डालता है। प्रस्तुति स्लाइडों से टेक्स्ट निकालते समय आप केवल एक स्लाइड तक सीमित हैं।

{{% alert color="primary" %}} 
यदि आप मूल्यांकन संस्करण की सीमाओं के बिना Aspose.Slides का परीक्षण चाहते हैं, तो आप **30-दिन का अस्थायी लाइसेंस** अनुरोध कर सकते हैं। अधिक जानकारी के लिए कृपया [अस्थायी लाइसेंस कैसे प्राप्त करें?](https://purchase.aspose.com/temporary-license) देखें।

## **लाइसेंस के बारे में**
आप आसानी से Aspose.Slides का Node.js हेतु .NET से [डाउनलोड पृष्ठ](https://releases.aspose.com/slides/hi/nodejs-net/) से मूल्यांकन संस्करण डाउनलोड कर सकते हैं। मूल्यांकन संस्करण Aspose.Slides के लाइसेंस प्राप्त संस्करण के बिल्कुल **उसी क्षमताओं** को प्रदान करता है। आगे, मूल्यांकन संस्करण सिर्फ लाइसेंस खरीदने और लाइसेंस लागू करने के लिए कुछ पंक्तियों का कोड जोड़ने पर लाइसेंस प्राप्त हो जाता है।

लाइसेंस एक साधारण-टेक्स्ट XML फ़ाइल है जिसमें उत्पाद का नाम, लाइसेंस प्राप्त डेवलपर्स की संख्या, सदस्यता समाप्ति तिथि आदि विवरण होते हैं। फ़ाइल डिजिटल रूप से साइन की गई है, इसलिए इसे संशोधित न करें। फ़ाइल की सामग्री में अनजाने में एक अतिरिक्त लाइन ब्रेक जोड़ना भी इसे अमान्य कर देगा।

मूल्यांकन संस्करण से जुड़ी सीमाओं से बचने के लिए, आपको **Aspose.Slides** का उपयोग करने से पहले लाइसेंस सेट करना आवश्यक है। आपको प्रत्येक एप्लिकेशन या प्रोसेस में केवल एक बार लाइसेंस सेट करना होगा।

### **Node.js के लिए Aspose.Slides में .NET के माध्यम से लाइसेंस सेट करना**
लाइसेंस निम्न स्थानों से लागू किए जा सकते हैं:
* स्पष्ट पथ
* स्ट्रीम
* मीटरड लाइसेंस के रूप में – एक नया लाइसेंसिंग तंत्र

{{% alert color="primary" %}}
किसी घटक को लाइसेंस करने के लिए **setLicense** मेथड का उपयोग करें।

जबकि **setLicense** को कई बार कॉल करना हानिकारक नहीं है, यह संसाधनों (प्रोसेसर) की बर्बादी है।
{{% /alert %}}

{{% alert color="warning" %}}
नए लाइसेंस केवल संस्करण 21.4 या बाद के साथ Aspose.Slides को सक्रिय कर सकते हैं। पूर्व संस्करण एक अलग लाइसेंसिंग सिस्टम का उपयोग करते हैं और इन लाइसेंसों को पहचान नहीं पाएंगे।
{{% /alert %}}

#### **फ़ाइल का उपयोग करके लाइसेंस लागू करना**
यह कोड स्निपेट लाइसेंस फ़ाइल सेट करने के लिए उपयोग किया जाता है:

**Node.js**

```javascript
// Aspose.Slides मॉड्यूल को PowerPoint फ़ाइल हेरफ़ेर के लिए आयात करें
const asposeSlides = require('aspose.slides.via.net');

// This function sets up the Aspose.Slides library with a license
function setupAsposeSlidesLicense() {
	
    // Aspose.Slides मॉड्यूल से License क्लास को प्रारम्भ करें
    var license = new asposeSlides.License();
    
    // फ़ाइल से लाइसेंस लागू करें
    // "your_license_file.lic" को अपनी वास्तविक लाइसेंस फ़ाइल के पथ से बदलें
    license.setLicense("your_license_file.lic");
}

// Aspose.Slides के लिए लाइसेंस सेट करने हेतु फ़ंक्शन चलाएँ
setupAsposeSlidesLicense();
```
{{% alert color="primary" %}}
setLicense मेथड को कॉल करते समय, लाइसेंस नाम आपके लाइसेंस फ़ाइल के समान होना चाहिए। उदाहरण के लिए, आप लाइसेंस फ़ाइल का नाम "Aspose.Slides.lic.xml" में बदल सकते हैं। फिर, कोड में आपको नया लाइसेंस नाम (Aspose.Slides.lic.xml) setLicense मेथड को पास करना होगा।
{{% /alert %}}
