## **Aspose.Slides for Reporting Services में आपका स्वागत है!**

Aspose.Slides for Reporting Services एकमात्र समाधान है जो Microsoft SQL Server 2005, 2008, 2012, 2016 और 2017 Reporting Services (32‑bit और 64‑bit) में वास्तविक PPT और PPS रिपोर्टों का जनरेट करना संभव बनाता है। सभी RDL रिपोर्ट सुविधाएँ, जैसे तालिकाएँ, मैट्रिक्स, चार्ट और छवियाँ, उच्चतम सटीकता के साथ Microsoft PowerPoint प्रस्तुतियों में परिवर्तित की जाती हैं।

## **उत्पाद अवलोकन**

Microsoft SQL Server Reporting Services में मूल रूप से रिपोर्टों को Microsoft PowerPoint प्रस्तुतियों के रूप में निर्यात करने की सुविधा नहीं है, लेकिन आपके सर्वर पर Aspose.Slides for Reporting Services स्थापित करने के बाद आपको अतिरिक्त निर्यात स्वरूपों तक पहुंच मिलती है:

- PPT - Aspose.Slides के माध्यम से PowerPoint प्रस्तुति
- PPS - Aspose.Slides के माध्यम से PowerPoint स्लाइडशो
- PPTX - Aspose.Slides के माध्यम से PowerPoint 2007 प्रस्तुति
- PPSX - Aspose.Slides के माध्यम से PowerPoint 2007 स्लाइडशो

Aspose.Slides for Reporting Services सर्वर पर प्रस्तुतियों को Microsoft PowerPoint का उपयोग किए बिना बनाता है। Aspose.Slides for Reporting Services आंतरिक रूप से Aspose.Slides for .NET – सर्वर‑साइड प्रस्तुतियों की प्रोसेसिंग के लिए विश्व‑स्तरीय घटक – का उपयोग करता है।

**Aspose.Slides for Reporting Services किसी भी रिपोर्ट को PPT, PPS, PPTX या PPSX स्वरूप में निर्यात करना संभव बनाता है।**
