{{% alert color="primary" %}} 
Microsoft SQL Server Reporting Services में रिपोर्ट को Microsoft PowerPoint प्रस्तुतियों के रूप में निर्यात करने की अंतर्निहित क्षमताएँ नहीं हैं, लेकिन अपने सर्वर पर Aspose.Slides for Reporting Services स्थापित करने के बाद, आप दो अतिरिक्त निर्यात फ़ॉर्मेट तक पहुँच प्राप्त करेंगे: 

- Microsoft PowerPoint Presentation (PPT)
- Microsoft PowerPoint SlideShow (PPS)
- Microsoft PowerPoint 2007 Presentation (PPTX)
- Microsoft PowerPoint 2007 SlideShow (PPSX)

{{% /alert %}} 

इन फ़ॉर्मेट में दस्तावेज़ बनाने के लिए, Aspose.Slides for Reporting Services एक बिल्ट‑इन संस्करण पर निर्भर करता है [Aspose.Slides for .NET](http://www.aspose.com/Products/Aspose.Slides/), जो Aspose का बाजार में अग्रणी प्रस्तुति‑प्रसंस्करण घटक है। दस्तावेज़ उत्पन्न करने के लिए Microsoft PowerPoint का उपयोग नहीं किया जाता है। 

एक ही रिपोर्ट की PPT और PPS प्रतियाँ एक-दूसरे के समान दिखेंगी, Aspose.Slides द्वारा प्रदान किए गए अंतर्निहित फ़ाइल फ़ॉर्मेट के उच्च स्तर के समर्थन के कारण।
