Aspose.Slides for C++ एक क्लास लाइब्रेरी है जो C++ अनुप्रयोगों को प्रस्तुति दस्तावेज़ों को पढ़ने और लिखने में सक्षम बनाती है, बिना Microsoft PowerPoint® या अन्य टूल का उपयोग किए। Aspose.Slides for C++ पाठ, आकार, तालिकाएँ और एनिमेशन को प्रबंधित करने, स्लाइड्स में ऑडियो और वीडियो जोड़ने, स्लाइड्स का पूर्वावलोकन करने, स्लाइड्स को SVG, PDF स्वरूप में निर्यात करने और अधिक करने की अनुमति देता है।

नवीनतम API रेफ़रेंस यहाँ प्राप्त किया जा सकता है: [Aspose.Slides for C++ API Reference](https://reference.aspose.com/slides/hi/cpp/).
