## **सॉफ़्टवेयर आवश्यकताएँ**
कृपया डाउनलोड करने और उदाहरण चलाने से पहले नीचे दी गई आवश्यकताओं को पूरा करना सुनिश्चित करें।

1. Visual Studio 2015 या उससे अधिक।

## **GitHub से Aspose.Slides डाउनलोड करें**
Aspose.Slides for C++ के सभी उदाहरण [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-C) पर होस्ट किए गए हैं।

- आप अपने पसंदीदा GitHub क्लाइंट का उपयोग करके रिपॉज़िटरी को क्लोन कर सकते हैं या ZIP फ़ाइल को [यहाँ](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/refs/heads/master.zip) से डाउनलोड कर सकते हैं।
- ZIP फ़ाइल की सामग्री को अपने कंप्यूटर में किसी भी फ़ोल्डर में निकालें। सभी उदाहरण **Examples** फ़ोल्डर में स्थित हैं।
- एक C++ Visual Studio समाधान फ़ाइल मौजूद है।
- प्रोजेक्ट्स Visual Studio 2015 में बनाए गए हैं, लेकिन समाधान फ़ाइलें Visual Studio 2015 और उससे ऊपर के संस्करणों के साथ संगत हैं।
- Solution फ़ाइल को Visual Studio में खोलें और प्रोजेक्ट को बिल्ड करें.
- Aspose.Slides for C++ का नवीनतम रिलीज़ [Download Section](https://downloads.aspose.com/slides/hi/cpp) से डाउनलोड करें।
- रिलीज़ को निकालें और **Examples** फ़ोल्डर के अंदर **Include** और **Bin** डायरेक्टरीज़ को कॉपी करें
- **main.cpp** फ़ाइल खोलें, सभी उदाहरण यहाँ से कॉल किए जाते हैं।
- प्रोजेक्ट के भीतर उन उदाहरणों की टिप्पणी हटाएँ जिन्हें आप चलाना चाहते हैं।
- समाधान को बिल्ड और कंपाइल करें

यदि आपको सेटअप या उदाहरण चलाने में कोई समस्या आती है, तो हमारे फ़ोरम के माध्यम से निःसंकोच संपर्क करें।

## **योगदान दें**
यदि आप कोई उदाहरण जोड़ना या सुधारना चाहते हैं, तो हम आपको परियोजना में योगदान करने के लिए प्रोत्साहित करते हैं। इस रिपॉज़िटरी में सभी उदाहरण और शोकेस प्रोजेक्ट ओपन सोर्स हैं और आपके अपने एप्लिकेशन में स्वतंत्र रूप से उपयोग किए जा सकते हैं।

योगदान देने के लिए, आप रिपॉज़िटरी को फ़ोर्क कर सकते हैं, स्रोत कोड को संपादित कर सकते हैं और एक पुल अनुरोध बना सकते हैं। हम परिवर्तन की समीक्षा करेंगे और यदि वह उपयोगी पाया गया तो रिपॉज़िटरी में शामिल करेंगे।
