## **XPS के बारे में**
[XML पार्सर विशिष्टता](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) एक पेज विवरण भाषा और एक नियत-डॉक्यूमेंट फ़ॉर्मैट है जिसे मूल रूप से माइक्रोसॉफ्ट द्वारा विकसित किया गया था। PDF की तरह, XPS एक नियत-लेआउट डॉक्यूमेंट फ़ॉर्मैट है जिसे दस्तावेज़ की सटीकता को बनाए रखने और डिवाइस-स्वतंत्र दस्तावेज़ रूप दिखाने के लिए डिज़ाइन किया गया है।
## **Aspose.Slides for C++ में XPS**
कोई भी प्रस्तुति दस्तावेज़ जिसे Aspose.Slides for C++ द्वारा लोड किया जा सकता है, XPS फ़ॉर्मैट में परिवर्तित किया जा सकता है। Aspose.Slides for C++ उच्च-विश्वसनीयता पेज लेआउट और रेंडरिंग इंजन का उपयोग करके नियत-लेआउट XPS दस्तावेज़ फ़ॉर्मैट में आउटपुट उत्पन्न करता है।
यह उल्लेख करने योग्य है कि Aspose.Slides for C++ सीधे XPS उत्पन्न करता है बिना Windows Presentation Foundation (WPF) वर्गों पर निर्भर हुए, जो C++ Framework 3.5 के साथ पैकेज किए गए हैं, इस प्रकार Aspose.Slides for C++ को C++ Framework संस्करण 3.5 से पहले के संस्करणों पर चलने वाली मशीनों पर भी XPS दस्तावेज़ उत्पन्न करने की अनुमति मिलती है। आप Aspose.Slides for C++ के माध्यम से प्रस्तुतिकरण दस्तावेज़ों को XPS दस्तावेज़ों में निर्यात करने के बारे में [इस विषय](/slides/hi/net/) में सीख सकते हैं।
