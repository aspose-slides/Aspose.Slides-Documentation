## **समीक्षा**

Aspose.Slides का उपयोग मूल्यांकन मोड में या वैध लाइसेंस के साथ किया जा सकता है। मूल्यांकन संस्करण लाइसेंस प्राप्त संस्करण के समान कार्यक्षमता प्रदान करता है, लेकिन प्रस्तुति खोलने या सहेजने पर एक मूल्यांकन वॉटरमार्क जोड़ता है और टेक्स्ट एक्सट्रैक्शन को केवल एक स्लाइड तक सीमित करता है।

## **Aspose.Slides का मूल्यांकन करें**

{{% alert color="primary" %}} 
आप **Aspose.Slides for C++** का एक मूल्यांकन संस्करण [its NuGet download page](https://www.nuget.org/packages/Aspose.Slides.CPP/) से डाउनलोड कर सकते हैं। मूल्यांकन संस्करण लाइसेंस प्राप्त उत्पाद के समान कार्यक्षमता प्रदान करता है। वास्तव में, मूल्यांकन पैकेज खरीदे गए पैकेज के समान ही है—जब आप लाइसेंस लागू करने के लिए कुछ पंक्तियों का कोड जोड़ते हैं तो यह लाइसेंस प्राप्त हो जाता है।

जब आप **Aspose.Slides** के अपने मूल्यांकन से संतुष्ट हों, तो आप [एक लाइसेंस खरीदें](https://purchase.aspose.com/buy)। हम अनुशंसा करते हैं कि उपलब्ध सब्सक्रिप्शन प्रकारों की समीक्षा करें। यदि आपके कोई प्रश्न हों, तो कृपया Aspose बिक्री टीम से संपर्क करें।

प्रत्येक Aspose लाइसेंस में एक वर्ष की सदस्यता शामिल होती है, जो मुफ्त अपग्रेड प्रदान करती है, जिसमें इस अवधि के दौरान जारी नई संस्करण और बग फिक्स शामिल हैं। चाहे आप लाइसेंस प्राप्त संस्करण उपयोग कर रहे हों या मूल्यांकन संस्करण, आपको मुफ्त और असीमित तकनीकी समर्थन मिलता है।

## **Aspose.Slides में लाइसेंसिंग**

* मूल्यांकन संस्करण लाइसेंस खरीदने और कोड की कुछ पंक्तियों को जोड़कर इसे लागू करने के बाद लाइसेंस प्राप्त हो जाता है।
* लाइसेंस एक साधारण टेक्स्ट XML फ़ाइल है जिसमें उत्पाद का नाम, लाइसेंस प्राप्त डेवलपर्स की संख्या, सब्सक्रिप्शन समाप्ति तिथि आदि विवरण होते हैं।
* लाइसेंस फ़ाइल डिजिटल रूप से साइन की गई होती है, इसलिए इसे संशोधित नहीं किया जाना चाहिए। एक आकस्मिक परिवर्तन—जैसे लाइन ब्रेक जोड़ना—भी फ़ाइल को अमान्य कर देगा।
* Aspose.Slides for C++ typically looks for the license file in the following locations:
  * आपके कोड में स्पष्ट रूप से निर्दिष्ट पथ
  * घटक की DLL (Aspose.Slides में शामिल) वाली फ़ोल्डर
  * जिस एसेम्बली से घटक की DLL को कॉल किया जाता है, उसकी फ़ोल्डर
* मूल्यांकन संस्करण की सीमाओं से बचने के लिए, आपको Aspose.Slides का उपयोग करने से पहले लाइसेंस सेट करना चाहिए। एक लाइसेंस को केवल एक बार प्रति एप्लिकेशन या प्रक्रिया सेट करने की आवश्यकता होती है।

## **लाइसेंस लागू करें**

लाइसेंस को **फ़ाइल**, **स्ट्रीम**, या **एम्बेडेड रिसोर्स** से लोड किया जा सकता है।

{{% alert color="primary" %}}
Aspose.Slides लाइसेंसिंग ऑपरेशनों के लिए [License](https://reference.aspose.com/slides/hi/cpp/class/aspose.slides.license/) क्लास प्रदान करता है।
{{% /alert %}} 

{{% alert color="warning" %}}
नए लाइसेंस केवल संस्करण 21.4 या बाद के साथ Aspose.Slides को सक्रिय कर सकते हैं। पहले के संस्करण अलग लाइसेंसिंग सिस्टम का उपयोग करते हैं और ये लाइसेंस नहीं पहचानेंगे।
{{% /alert %}}

### **फ़ाइल**

लाइसेंस सेट करने का सबसे आसान तरीका है लाइसेंस फ़ाइल को घटक की DLL (Aspose.Slides में शामिल) के समान फ़ोल्डर में रखना और केवल फ़ाइल नाम निर्दिष्ट करना, पथ के बिना।

निम्नलिखित C++ कोड दिखाता है कि लाइसेंस फ़ाइल कैसे सेट की जाए:

```c++
#include <Util/License.h>

using namespace Aspose::Slides;

int main()
{
    auto license = MakeObject<License>();
    license->SetLicense(u"Aspose.Slides.lic");

    return 0;
}
```

{{% alert color="warning" %}} 
यदि आप लाइसेंस फ़ाइल को किसी अन्य निर्देशिका में रखते हैं, तो जब आप [License::SetLicense](https://reference.aspose.com/slides/hi/cpp/aspose.slides/license/setlicense/) मेथड को कॉल करते हैं, तो निर्दिष्ट स्पष्ट पथ के अंत में फ़ाइल नाम बिल्कुल आपके लाइसेंस फ़ाइल के नाम से मेल खाना चाहिए।

उदाहरण के लिए, यदि आप अपनी लाइसेंस फ़ाइल का नाम *Aspose.Slides.lic.xml* रखते हैं, तो आपको अपने कोड में [License::SetLicense](https://reference.aspose.com/slides/hi/cpp/aspose.slides/license/setlicense/) मेथड को पूर्ण पथ देना होगा जो *Aspose.Slides.lic.xml* पर समाप्त हो।
{{% /alert %}}

### **स्ट्रीम**

आप स्ट्रीम से लाइसेंस लोड कर सकते हैं। निम्नलिखित C++ कोड दिखाता है कि स्ट्रीम से लाइसेंस कैसे लागू किया जाए:

```c++
auto license = MakeObject<License>();

auto stream = File::OpenRead(u"Aspose.Slides.lic");

license->SetLicense(stream);
```

## **लाइसेंस की वैधता जाँचें**

यह जांचने के लिए कि लाइसेंस सही ढंग से सेट किया गया है या नहीं, आप इसे वैधता जांच सकते हैं। निम्नलिखित C++ कोड दिखाता है कि लाइसेंस को कैसे वैधता जांचा जाए:

```c++
auto license = MakeObject<License>();

license->SetLicense(u"Aspose.Slides.lic");

if (license->IsLicensed())
{
    Console::WriteLine(u"License is good!");
    Console::ReadKey();
}
```

## **थ्रेड सुरक्षा**

{{% alert title="Note" color="warning" %}} 
[License::SetLicense](https://reference.aspose.com/slides/hi/cpp/aspose.slides/license/setlicense/) मेथड **थ्रेड-सुरक्षित नहीं** है। यदि आपको इस मेथड को कई थ्रेड्स से एक साथ कॉल करने की आवश्यकता है, तो संभावित समस्याओं से बचने के लिए सिंक्रोनाइज़ेशन प्रिमिटिव्स (जैसे लॉक) का उपयोग करने की सलाह दी जाती है।
{{% /alert %}}

## **अक्सर पूछे जाने वाले प्रश्न**

**क्या मैं लाइसेंस को पूरी तरह ऑफ़लाइन वातावरण में (कोई इंटरनेट एक्सेस नहीं) लागू कर सकता हूँ?**

हाँ। लाइसेंस वैधता लाइसेंस फ़ाइल का उपयोग करके स्थानीय रूप से की जाती है; इंटरनेट कनेक्शन की आवश्यकता नहीं है।

**एक‑साल की सब्सक्रिप्शन समाप्त होने के बाद क्या होता है? क्या लाइब्रेरी काम करना बंद कर देगी?**

नहीं। लाइसेंस स्थायी है: आप अपनी सब्सक्रिप्शन समाप्ति तिथि से पहले जारी किए गए संस्करणों का उपयोग जारी रख सकते हैं; केवल बिना नवीकरण के आप नई रिलीज़ इस्तेमाल नहीं कर पाएँगे।
