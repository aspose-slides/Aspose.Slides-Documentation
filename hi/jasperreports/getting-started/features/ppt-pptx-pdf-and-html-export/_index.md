{{% alert color="primary" %}} 

JasperReports और JasperServer में रिपोर्टों को Microsoft PowerPoint प्रस्तुतियों के रूप में निर्यात करने की अंतर्निहित क्षमताएँ नहीं हैं, लेकिन Aspose.Slides for JasperReports के साथ, आपको अतिरिक्त निर्यात स्वरूपों तक पहुंच मिलती है:

- Microsoft PowerPoint Presentation (PPT)
- Microsoft PowerPoint Presentation (PPTX)
- HTML
- PDF

{{% /alert %}} 

इन स्वरूपों में दस्तावेज़ बनाने के लिए, Aspose.Slides for JasperReports एक अंतर्निहित संस्करण पर निर्भर करता है जो [Aspose.Slides for Java](https://products.aspose.com/slides/hi/java/) से है, जो Aspose का बाजार में अग्रणी प्रस्तुति‑प्रसंस्करण लाइब्रेरी है। दस्तावेज़ बनाने के लिए Microsoft PowerPoint का उपयोग नहीं किया जाता है।
