Aspose.Slides for JasperReports निम्न के साथ काम करता है:

**Aspose.Slides for JasperReports 17.5 और पहले के संस्करण:**

- Java 2 Platform Standard Edition 1.4 या उच्चतर।
- JasperReports 2.0, 3.0 या उच्चतर।

**Aspose.Slides for JasperReports 17.6 और नए संस्करण:**

- Java 2 Platform Standard Edition 6.0 या उच्चतर।
- JasperReports 3.7.2 या उच्चतर।
