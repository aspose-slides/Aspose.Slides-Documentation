## **समर्थित प्लेटफ़ॉर्म**
Aspose.Slides for Node.js via Java एक प्लेटफ़ॉर्म स्वतंत्र API है। इसे किसी भी प्लेटफ़ॉर्म (Windows, Linux, MacOS, आदि) पर उपयोग किया जा सकता है जहाँ Node.js 7 या उससे अधिक स्थापित हो। Oracle JDK 8 या उससे अधिक भी एक आवश्यकता है।

## **फ़ाइल फ़ॉर्मेट और रूपांतरण**
Aspose.Slides for Node.js via Java अधिकांश PowerPoint दस्तावेज़ फ़ॉर्मेट का समर्थन करता है। यह आपको उन्हें उन लोकप्रिय फ़ाइल फ़ॉर्मेट में निर्यात करने देता है जो संगठनों द्वारा उपयोग और साझा किए जाते हैं। नीचे विवरण देखें:

|**फ़ीचर**|**वर्णन**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via Java इस प्रस्तुति दस्तावेज़ फ़ॉर्मेट के लिए सबसे तेज़ प्रोसेसिंग प्रदान करता है।|
|PPT to PPTX conversion|Aspose.Slides for Node.js via Java PPT से PPTX रूपांतरण का समर्थन करता है।|
|Portable Document Format (PDF)|आप एक ही मेथड से सभी समर्थित फ़ाइल फ़ॉर्मेट को Adobe Portable Document Format (PDF) दस्तावेज़ों में निर्यात कर सकते हैं।|
|XML Parser Specification (XPS)|आप एक ही मेथड से सभी समर्थित फ़ाइल फ़ॉर्मेट को XML Parser Specification (XPS) दस्तावेज़ों में निर्यात कर सकते हैं।|
|Tagged Image File Format (TIFF)|आप सभी समर्थित प्रस्तुति फ़ाइल फ़ॉर्मेट को Tagged Image File Format (TIFF) में निर्यात कर सकते हैं।|
|PPTX To HTML Conversion|Aspose.Slides for Node.js via Java PresentationEx को HTML फ़ॉर्मेट में रूपांतरित करने का समर्थन करता है।|

## **रेंडरिंग और प्रिंटिंग**
Aspose.Slides for Node.js via Java विभिन्न ग्राफ़िक फ़ॉर्मेट में स्लाइडों की उच्च फ़िडेलिटी रेंडरिंग का समर्थन करता है। नीचे विवरण देखें:

|**फ़ीचर**|**वर्णन**|
| :- | :- |
|Java Supported Image Formats|Aspose.Slides for Node.js via Java का उपयोग करके आप प्रस्तुति स्लाइडों और स्लाइडों पर छवियों को सभी Java समर्थित ग्राफ़िक फ़ॉर्मेट जैसे TIFF, PNG, BMP, JPEG, GIF में रेंडर कर सकते हैं।|
|SVG Format|Aspose.Slides for Node.js via Java निर्मित मेथड प्रदान करता है जो प्रस्तुति स्लाइडों को Scalable Vector Graphics (SVG) फ़ॉर्मेट में निर्यात करने की अनुमति देते हैं।|
|Presentation Printing|Aspose.Slides for Node.js via Java के नवीनतम संस्करण विभिन्न विकल्पों के साथ निर्मित प्रिंट मेथड प्रदान करते हैं।|

## **सामग्री विशेषताएँ**

|**फ़ीचर**|**वर्णन**|
| :- | :- |

|Normal Slides|Aspose.Slides for Node.js via Java के साथ आप विभिन्न प्रकार की नई स्लाइडें बना सकते हैं; आप प्रस्तुतियों में मौजूदा स्लाइडों तक भी पहुँच और संशोधन कर सकते हैं।|
|Cloning / Copying Slides|Aspose.Slides for Node.js via Java द्वारा प्रदान किए गए निर्मित मेथड आपको प्रस्तुति में मौज़ूद स्लाइडों को क्लोन या कॉपी करने की अनुमति देते हैं। आप एक प्रस्तुति से दूसरी में कॉपी और क्लोन की गई स्लाइडों का उपयोग भी कर सकते हैं। चूँकि एक स्लाइड अपने लेआउट को मास्टर स्लाइड से प्राप्त करती है, इसलिए निर्मित क्लोनिंग मेथड स्वचालित रूप से क्लोनिंग पर मास्टर को कॉपी कर देते हैं।|

## **FAQ**

**क्या लाइब्रेरी के काम करने के लिए सर्वर/पीसी पर Microsoft PowerPoint स्थापित होना आवश्यक है?**

नहीं। PowerPoint आवश्यक नहीं है; Aspose.Slides एक स्टैंडअलोन इंजन है जो प्रस्तुतियों को बनाने, संपादित करने, रूपांतरित करने और रेंडर करने के लिए उपयोग किया जाता है।

**멀티थ्रेडिंग कैसे कार्य करता है? क्या प्रोसेसिंग को समानांतर किया जा सकता है?**

विभिन्न थ्रेड्स में अलग‑अलग दस्तावेज़ों को प्रोसेस करना सुरक्षित है; एक ही [प्रेजेंटेशन] (https://reference.aspose.com/slides/hi/nodejs-java/aspose.slides/presentation/) ऑब्जेक्ट को [कई थ्रेंड](/slides/hi/nodejs-java/multithreading/) एक साथ उपयोग नहीं किया जाना चाहिए।

**क्या फ़ाइल पासवर्ड और एन्क्रिप्शन समर्थित हैं?**

हां। आप एन्क्रिप्टेड प्रस्तुतियों को खोल सकते हैं, ओपन और राइट पासवर्ड सेट या हटाए जा सकते हैं, और सुरक्षा स्थिति जाँच सकते हैं। [आप यह कर सकते हैं](/slides/hi/nodejs-java/password-protected-presentation/)।

**क्या Linux कंटेनरों में फ़ॉन्ट पैकेजों की पुष्टि करनी चाहिए?**

हां। अप्रत्याशित प्रतिस्थापन से बचने के लिये सामान्य फ़ॉन्ट पैकेज स्थापित करना और/या अपने अनुप्रयोग में स्पष्ट रूप से [फ़ॉन्ट डायरेक्टरी निर्दिष्ट](/slides/hi/nodejs-java/custom-font/) करना अनुशंसित है।

**क्या मूल्यांकन संस्करण में सीमाएँ हैं?**

[मूल्यांकन मोड](/slides/hi/nodejs-java/licensing/) में आउटपुट में वॉटरमार्क जोड़ा जाता है और कुछ सीमाएँ लागू होती हैं; पूर्ण‑फ़ीचर परीक्षण के लिये एक [30‑दिन का अस्थायी लाइसेंस] (https://purchase.aspose.com/temporary-license/) उपलब्ध है।

**क्या बाहरी फ़ॉर्मेट (PDF/HTML → PPTX) को प्रस्तुति में आयात करना समर्थित है?**

हां। आप एक प्रस्तुति में [PDF पेज और HTML सामग्री](/slides/hi/nodejs-java/import-presentation/) जोड़ सकते हैं, जिससे वे स्लाइडों में परिवर्तित हो जाएँ।
