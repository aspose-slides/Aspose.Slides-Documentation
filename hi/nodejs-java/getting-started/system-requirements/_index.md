## **परिचय**

Aspose.Slides for Node.js via Java को किसी भी थर्ड पार्टी प्रोडक्ट जैसे Microsoft PowerPoint को स्थापित करने की आवश्यकता नहीं है। Aspose.Slides स्वयं विभिन्न फ़ॉर्मैट्स में दस्तावेज़ बनाने, संशोधित करने, परिवर्तित करने और रेंडर करने के लिए एक इंजन है, जिसमें Microsoft PowerPoint प्रस्तुति फ़ॉर्मैट भी शामिल हैं।

## **समर्थित ऑपरेटिंग सिस्टम**

Aspose.Slides for Node.js via Java किसी भी 32-बिट या 64-बिट ऑपरेटिंग सिस्टम को समर्थन देता है जो Java रनटाइम चलाता है, जिसमें लेकिन इन तक सीमित नहीं है:

### **विंडोज**
- Microsoft Windows 2003 Server ( x64, x86)
- Microsoft Windows 2008 Server ( x64, x86)
- Microsoft Windows 2012 Server ( x64, x86)
- Microsoft Windows 2012 R2 Server ( x64, x86)
- Microsoft Windows 2016 Server ( x64, x86)
- Microsoft Windows 2019 Server ( x64, x86)
- Microsoft Windows Vista ( x64, x86)
- Microsoft Windows XP ( x64, x86)
- Microsoft Windows 7 ( x64, x86)
- Microsoft Windows 8, 8.1 ( x64, x86)
- Microsoft Windows 10 ( x64, x86)

### **लिनक्स**
- Linux (Ubuntu, CentOS और अन्य)

### **Mac**
- Mac OS X

## **अक्सर पूछे जाने वाले प्रश्न**

**क्या मुझे रूपांतरण और रेंडरिंग के लिए Microsoft PowerPoint स्थापित करने की आवश्यकता है?**

नहीं, PowerPoint आवश्यक नहीं है; Aspose.Slides एक स्टैंडअलोन इंजन है जो प्रस्तुतियों को [बनाने](/slides/hi/nodejs-java/create-presentation/), संशोधित करने, [परिवर्तित करने](/slides/hi/nodejs-java/convert-presentation/), और [रेंडर करने](/slides/hi/nodejs-java/convert-powerpoint-to-png/) के लिए है।

**सही रेंडरिंग के लिए कौन फ़ॉन्ट्स आवश्यक हैं?**

व्यावहारिक रूप से, प्रस्तुति में उपयोग किए गए फ़ॉन्ट्स या उपयुक्त [स्थानापन्न](/slides/hi/nodejs-java/font-substitution/) उपलब्ध होने चाहिए। Linux/macOS पर सुसंगत रेंडरिंग सुनिश्चित करने के लिए सामान्य फ़ॉन्ट पैकेज स्थापित करना सलाहनीय है।

**Linux पर कस्टम फ़ॉन्ट फॉलबैक या गायब पाठ के रूप में क्यों रेंडर होता है?**

यदि फ़ॉन्ट फ़ाइल में असंगत या दूषित name-table प्रविष्टियां होती हैं, तो Linux फ़ॉन्ट‑मैचिंग स्टैक (FreeType/fontconfig) एक अमान्य रिकॉर्ड चुन सकता है, जिससे फ़ॉन्ट अनसुलझा रह जाता है। सुधारित name-table रिकॉर्ड वाली फ़ॉन्ट संस्करण का उपयोग करने या एक सुसंगत विकल्प स्थापित करने से समस्या हल हो जाती है।
