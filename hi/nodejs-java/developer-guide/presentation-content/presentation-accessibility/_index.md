## **Overview**

प्रेजेंटेशन एक्सेसेबिलिटी यह सुनिश्चित करती है कि सहायक तकनीकों—जैसे स्क्रीन रीडर, ब्रेल डिस्प्ले, या केवल कीबोर्ड नेविगेशन—का उपयोग करने वाले लोग आपके स्लाइड्स को उतनी ही प्रभावी रूप से समझ और नेविगेट कर सकें जितना कि दृष्टि वाले, माउस उपयोगकर्ता दर्शक। अच्छा अभ्यास स्पष्ट पढ़ने का क्रम, सूचनात्मक विज़ुअल्स के लिए अर्थपूर्ण वैकल्पिक पाठ, पर्याप्त रंग कंट्रास्ट, पठनीय टाइपोग्राफी, वर्णनात्मक लिंक टेक्स्ट, और केवल रंग या स्थिति के आधार पर अर्थ व्यक्त करने से बचना पर केंद्रित होता है। जब एक्सेसेबिलिटी की शुरुआत से ही योजना बनाई जाती है, तो परिणाम एक साफ़ संरचना, अधिक सुसंगत विज़ुअल्स, और ऐसा कंटेंट होता है जो हर दर्शक तक बिना वर्कअराउंड के पहुँचता है।

## **Mark as Decorative**

Mark as Decorative फ़्लैग शुद्ध सजावटी विज़ुअल्स को चिह्नित करता है ताकि स्क्रीन रीडर उन्हें स्किप कर दें, शोर कम हो और अर्थपूर्ण कंटेंट पर फोकस बना रहे। इसे बैकग्राउंड, फ़्लॉरिश, और स्पेसर पर लागू करें—कभी भी चार्ट, आइकन, या ऐसी इमेज पर नहीं जो जानकारी देती हों। Aspose.Slides इस फ़्लैग को डिटेक्शन और वैलीडेशन के लिए एक्सपोज़ करता है, जिससे स्वचालित एक्सेसेबिलिटी चेक और क्लीनअप संभव होता है।

![Mark as Decorative](mark_as_decorative.png)

निम्न कोड नमूना दिखाता है कि कैसे यह निर्धारित किया जाए कि कोई शैप Mark as Decorative के रूप में चिह्नित है।

```js
let presentation = new aspose.slides.Presentation("sample.pptx");
try {
    let shape = presentation.getSlides().get_Item(0).getShapes().get_Item(0);
    console.log("Is shape decorative:", shape.isDecorative());
} finally {
    presentation.dispose();
}
```
