## **अवलोकन**

यह FAQ Aspose.Slides के बारे में सामान्य प्रश्नों के उत्तर प्रदान करता है। यह समर्थित फ़ाइल फ़ॉर्मेट, बड़े प्रेजेंटेशन के साथ काम करते समय अपवादों को संभालना, स्लाइड आकार बदलना, स्लाइड का पूर्वावलोकन, प्रेजेंटेशन से टेक्स्ट प्राप्त करना, तालिका सीमा को स्वरूपित करना, चित्र रखना, और प्रेजेंटेशन को PDF या छवियों में परिवर्तित करते समय फ़ॉन्ट‑संबंधी समस्याओं को हल करना आदि को कवर करता है।

## **समर्थित फ़ाइल फ़ॉर्मेट**

**प्रश्न:** Aspose.Slides for Node.js via Java कौन‑से फ़ाइल फ़ॉर्मेट का समर्थन करता है?  
**उत्तर:** Aspose.Slides for Node.js via Java उन फ़ाइल फ़ॉर्मेट का समर्थन करता है जो [समर्थित फ़ाइल फ़ॉर्मेट](/slides/hi/nodejs-java/supported-file-formats/) में वर्णित हैं।

## **अपवाद**

**प्रश्न:** मैं बड़ी PPT फ़ाइल जिसमें चित्र हों, लोड करते समय out of memory अपवाद प्राप्त कर रहा हूँ। क्या फ़ाइल आकार के संबंध में Aspose.Slides में कोई सीमा है?  
**उत्तर:** Aspose.Slides द्वारा समर्थित प्रेजेंटेशन आकार की कोई विशिष्ट सूत्र नहीं है। मेमोरी में पूरी प्रेजेंटेशन संरचना और चित्रों को रखने के लिए पर्याप्त स्थान होना चाहिए। सामान्यतः, मेमोरी में चित्रों का आकार हार्ड डिस्क की तुलना में अधिक होता है, विशेषकर जब चित्रों पर अतिरिक्त प्रभाव लागू होते हैं।

सामान्यतः, Aspose.Slides for Node.js via Java 4 GB RAM वाले सर्वर पर लगभग 300 MB के प्रेजेंटेशन फ़ाइलों को आसानी से संभाल सकता है।

## **स्लाइड्स के साथ काम करना**

**प्रश्न:** क्या मैं प्रेजेंटेशन में स्लाइड्स का आकार बदल सकता हूँ?  
**उत्तर:** आप `getSlideSize` मेथड का उपयोग कर सकते हैं जो [Presentation](https://reference.aspose.com/slides/hi/nodejs-java/aspose.slides/presentation/) क्लास द्वारा प्रदान किया गया है, ताकि प्रेजेंटेशन में स्लाइड्स का आकार निर्धारित किया जा सके।

**प्रश्न:** क्या प्रेजेंटेशन में विभिन्न आकार की स्लाइड्स निर्धारित करने का कोई तरीका है?  
**उत्तर:** क्योंकि स्लाइड्स का आकार माइक्रोसॉफ़्ट पावरपॉइंट दस्तावेज़ों में प्रेजेंटेशन स्तर पर निर्धारित होता है, इसका कोई तरीका नहीं है।

**प्रश्न:** क्या Aspose.Slides for Node.js via Java स्लाइड को सहेजने से पहले पूर्वावलोकन का समर्थन करता है?  
**उत्तर:** आप प्रेजेंटेशन स्लाइड्स को छवियों में रेंडर कर सकते हैं और इन छवियों का उपयोग स्लाइड्स का पूर्वावलोकन करने के लिए कर सकते हैं।

## **टेक्स्ट के साथ काम करना**

**प्रश्न:** क्या प्रेजेंटेशन से सभी टेक्स्ट प्राप्त करना संभव है?  
**उत्तर:** Aspose.Slides for Node.js via Java [SlideUtil](https://reference.aspose.com/slides/hi/nodejs-java/aspose.slides/slideutil/) क्लास प्रदान करता है जो प्रेजेंटेशनों से पूरा टेक्स्ट प्राप्त करने के विभिन्न मेथड्स प्रदान करता है।

**प्रश्न:** विंडोज़ और लिनक्स ऑपरेटिंग सिस्टम में पैराग्राफ आकार अलग क्यों होते हैं?  
**उत्तर:** पैराग्राफ आकार की गणना उस पैराग्राफ के प्रतिनिधित्व वाले टेक्स्ट आकार पर आधारित होती है। टेक्स्ट आकार की गणना प्रेजेंटेशन में निर्दिष्ट फ़ॉन्ट के मीट्रिक्स पर निर्भर करती है। यदि निर्दिष्ट फ़ॉन्ट अनुपलब्ध है, तो इसे सबसे समान फ़ॉन्ट से बदला जाता है, लेकिन उस फ़ॉन्ट के मीट्रिक्स मूल फ़ॉन्ट से भिन्न होते हैं। परिणामस्वरूप, विभिन्न सिस्टमों में स्थापित फ़ॉन्ट्स के सेट के आधार पर पैराग्राफ आकार की गणना अलग-अलग परिणाम देती है। विभिन्न ऑपरेटिंग सिस्टम पर समान परिणाम पाने के लिए आपको सिस्टम पर समान फ़ॉन्ट्स इंस्टॉल करने या उन्हें रनटाइम पर [बाहरी फ़ॉन्ट](/slides/hi/nodejs-java/custom-font/) के रूप में लोड करने की जरूरत है।

## **स्वरूपण और छवियां**

**प्रश्न:** तालिका सीमा का रंग कैसे सेट करूँ?  
**उत्तर:** आप सभी तालिका सीमाओं या पूरे तालिका के चारों ओर की सीमा का रंग बदल सकते हैं। सभी सीमाओं को बदलने के लिए कृपया [Cell](https://reference.aspose.com/slides/hi/nodejs-java/aspose.slides/cell/) क्लास से `getCellFormat` मेथड का उपयोग करें। पूरे तालिका की सीमा के लिए, आपको कोशिकाएँ इटरैट कर बाहरी सीमाओं का रंग बदलना होगा।

**प्रश्न:** Aspose.Slides for Node.js via Java चित्र रखने के लिए कौन-से माप का उपयोग करता है?  
**उत्तर:** स्लाइड्स पर सभी आकृतियों के निर्देशांक और आकार पॉइंट्स (72 dpi) में मापे जाते हैं।

## **फ़ॉन्ट्स के साथ काम करना**

**प्रश्न:** PPT को PDF या छवियों में परिवर्तित करते समय आउटपुट दस्तावेज़ों में फ़ॉन्ट्स अलग क्यों दिखते हैं?  
**उत्तर:** यह समस्या यह संकेत दे सकती है कि प्रेजेंटेशन में उपयोग किए गए फ़ॉन्ट्स उस ऑपरेटिंग सिस्टम में मौजूद नहीं हैं जहाँ कोड निष्पादित हुआ था। आपको फ़ॉन्ट्स को ऑपरेटिंग सिस्टम पर इंस्टॉल करना चाहिए या उन्हें [FontsLoader](https://reference.aspose.com/slides/hi/nodejs-java/aspose.slides/fontsloader/) क्लास का उपयोग करके बाहरी फ़ॉन्ट के रूप में लोड करना चाहिए जैसा कि नीचे दिखाया गया है:
```javascript
var folders = java.newArray("java.lang.String", ["path_to_a_folder_with_fonts"]));
java.callStaticMethodSync("com.aspose.slides.FontsLoader", "loadExternalFonts", folders);
```
