## **अवलोकन**

Aspose.Slides for Node.js via Java PowerPoint और OpenDocument प्रस्तुतियों को लोड कर सकता है और उन्हें कई अन्य फ़ॉर्मेट में सहेज या रेंडर कर सकता है, बिना Microsoft PowerPoint, OpenOffice या LibreOffice के। आप पुरानी PPT फ़ाइलों को आधुनिक PPTX में परिवर्तित कर सकते हैं, प्रस्तुतियों को PDF और XPS जैसे स्थिर‑लेआउट दस्तावेज़ों में निर्यात कर सकते हैं, स्लाइड्स को HTML के रूप में प्रकाशित कर सकते हैं, या प्रीव्यू, थंबनेल और अभिलेखों के लिए स्लाइड्स को इमेज फ़ाइलों के रूप में रेंडर कर सकते हैं।

अधिकांश दस्तावेज़ रूपांतरण एक ही सामान्य वर्कफ़्लो का उपयोग करते हैं: स्रोत फ़ाइल लोड करें, आवश्यक आउटपुट फ़ॉर्मेट चुनें, और आवश्यक होने पर फ़ॉर्मेट‑विशिष्ट विकल्प लागू करें। इमेज फ़ॉर्मेट के लिए, प्रत्येक स्लाइड को अलग‑अलग रेंडर किया जाता है और फिर रास्टर या वेक्टर इमेज के रूप में सहेजा जाता है। नीचे लिंक किए गए समर्पित लेख प्रत्येक केस के कार्यान्वयन विवरण प्रदान करते हैं।

## **PowerPoint और OpenDocument परिवर्तन**

Aspose.Slides for Node.js via Java सामान्य रूप से उपयोग किए जाने वाले प्रस्तुति फ़ॉर्मेट जैसे PPT, PPTX, PPS, PPSX, POT, POTX, और ODP से रूपांतरण का समर्थन करता है। PowerPoint और OpenDocument फ़ाइलों के लिए समान रूपांतरण API उपयोग किया जाता है, इसलिए एक वर्कफ़्लो जो PPTX को PDF में सहेजता है, अक्सर केवल इनपुट फ़ाइल बदलकर ODP पर भी लागू किया जा सकता है।

ODP फ़ाइलों को बदलते समय याद रखें कि PowerPoint और OpenDocument अनुप्रयोग प्रत्येक लेआउट और फ़ॉर्मेटिंग सुविधा को बिल्कुल समान तरीके से समर्थन नहीं करते। यदि ODP फ़ाइल LibreOffice या OpenOffice Impress में बनाई गई थी, तो आउटपुट की समीक्षा करें और जब आपको फ़ॉर्मेट‑विशिष्ट मार्गदर्शन की आवश्यकता हो तो [OpenDocument प्रस्तुतियों को बदलें](/slides/hi/nodejs-java/convert-openoffice-odp/) में वर्णित विकल्पों का उपयोग करें।

## **PPT से PPTX परिवर्तन**

PPT पुराना बाइनरी PowerPoint फ़ॉर्मेट है, जबकि PPTX आधुनिक Office Open XML फ़ॉर्मेट है। Aspose.Slides for Node.js via Java उच्च‑वफ़ादारी PPT से PPTX परिवर्तन का समर्थन करता है और जटिल प्रस्तुति संरचनाओं जैसे मास्टर, लेआउट, स्लाइड, चार्ट, ग्रुप्ड शैप, प्लेसहोल्डर, टेक्स्ट फ्रेम, टेक्सचर और पिक्चर फ़िल को सुरक्षित रखता है।

विवरण के लिए देखें [PPT को PPTX में बदलें](/slides/hi/nodejs-java/convert-ppt-to-pptx/) और [PPT बनाम PPTX](/slides/hi/nodejs-java/ppt-vs-pptx/)।

## **स्थिर‑लेआउट निर्यात**

PDF, XPS, और TIFF उपयोगी होते हैं जब आउटपुट को सभी उपकरणों पर समान दिखना चाहिए और इसे प्रस्तुति के रूप में संपादित नहीं किया जाना चाहिए। समर्पित PDF, XPS, और TIFF लेख समझाते हैं कि अनुपालन, छिपी स्लाइड्स, नोट्स, इमेज क्वालिटी, कंप्रेशन, पिक्सेल फ़ॉर्मेट और आउटपुट साइज को कैसे नियंत्रित किया जाए।

## **HTML और इमेज निर्यात**

HTML और HTML5 निर्यात ब्राउज़र में देखना, वेब पर प्रकाशित करना और हल्का साझा करना आसान बनाते हैं। इमेज निर्यात तब उपयोगी होता है जब प्रत्येक स्लाइड को अलग‑अलग प्रीव्यू, थंबनेल या रास्टर एसेट बनाना हो। फ़ॉर्मेट‑विशिष्ट रेंडरिंग मार्गदर्शन के लिए PNG, JPG और SVG लेख देखें।

## **अक्सर पूछे जाने वाले प्रश्न**

**क्या प्रस्तुति को बदलने के लिए मुझे Microsoft PowerPoint की आवश्यकता है?**

नहीं। Aspose.Slides for Node.js via Java एक स्वतंत्र लाइब्रेरी है और इसे Microsoft PowerPoint या Office ऑटोमेशन की आवश्यकता नहीं होती।

**क्या मैं कई प्रस्तुतियों को बैच में बदल सकता हूँ?**

हां। प्रत्येक प्रस्तुति लोड करें, आवश्यक फ़ॉर्मेट में सहेजें, और प्रोसेसिंग के बाद प्रस्तुति ऑब्जेक्ट को डिस्पोज़ करें। समानांतर प्रोसेसिंग के लिए अलग‑अलग प्रस्तुति इंस्टेंस का उपयोग करें और [बहु‑थ्रेडिंग](/slides/hi/nodejs-java/multithreading/) मार्गदर्शन का पालन करें।

**क्या मैं केवल चयनित स्लाइड्स को निर्यात कर सकता हूँ?**

हां। कई निर्यात विधियां आपको स्लाइड इंडेक्स पास करने या व्यक्तिगत स्लाइड रेंडर करने देती हैं, यह आउटपुट फ़ॉर्मेट पर निर्भर करता है। लक्षित फ़ॉर्मेट के समर्पित लेख देखें।

**क्या मैं PDF या XPS निर्यात करते समय छिपी स्लाइड्स शामिल कर सकता हूँ?**

हां। [PDF](/slides/hi/nodejs-java/convert-powerpoint-to-pdf/) और [XPS](/slides/hi/nodejs-java/convert-powerpoint-to-xps/) परिवर्तन लेखों में वर्णित छिपी‑स्लाइड निर्यात सेटिंग्स का उपयोग करें।

**क्या मैं PDF/A आउटपुट बना सकता हूँ?**

हां। PDF निर्यात के लिए PDF अनुपालन सेटिंग्स उपलब्ध हैं। विवरण के लिए देखें [PowerPoint को PDF में बदलें](/slides/hi/nodejs-java/convert-powerpoint-to-pdf/)।

**रूपांतरण के दौरान फ़ॉन्ट कैसे संभाले जाते हैं?**

Aspose.Slides एंबेडेड फ़ॉन्ट, फ़ॉलबैक फ़ॉन्ट और फ़ॉन्ट प्रतिस्थापन सेटिंग्स का उपयोग कर सकता है। देखें [एंबेडेड फ़ॉन्ट](/slides/hi/nodejs-java/embedded-font/), [फ़ॉलबैक फ़ॉन्ट](/slides/hi/nodejs-java/fallback-font/), और [फ़ॉन्ट प्रतिस्थापन](/slides/hi/nodejs-java/font-substitution/)।
