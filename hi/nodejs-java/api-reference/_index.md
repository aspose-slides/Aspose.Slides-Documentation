Aspose.Slides for Node.js via Java एक क्लास लाइब्रेरी है जो Node.js अनुप्रयोगों को Microsoft PowerPoint® या अन्य सॉफ़्टवेयर का उपयोग किए बिना प्रस्तुति दस्तावेज़ को पढ़ने और लिखने में सक्षम बनाती है। Aspose.Slides for Node.js via Java आपको टेक्स्ट, शेप्स, टेबल्स और एनिमेशन का प्रबंधन करने, स्लाइड्स में ऑडियो और वीडियो जोड़ने, स्लाइड्स का पूर्वावलोकन करने, स्लाइड्स को SVG, PDF फ़ॉर्मेट में निर्यात करने, तथा बहुत कुछ करने की अनुमति देता है।

नवीनतम API रेफ़रेंस यहाँ पाया जा सकता है: [Aspose.Slides for Node.js via Java API Reference](https://reference.aspose.com/slides/hi/nodejs-java/)।
