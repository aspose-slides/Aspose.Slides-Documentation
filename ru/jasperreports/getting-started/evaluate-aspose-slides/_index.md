Вы можете легко скачать Aspose.Slides для оценки. Загружаемый файл для оценки такой же, как и покупаемый. Оценочная версия просто становится лицензированной, когда вы добавляете несколько строк кода для применения лицензии.

Оценочная версия Aspose.Slides (без указанной лицензии) предоставляет полную функциональность продукта, но вставляет водяной знак оценки в верхней части документа при открытии и сохранении, а также ограничивает извлечение текста из слайдов презентации одним слайдом.
