{{% alert color="primary" %}} 

Aspose предоставляет неограниченную бесплатную техническую поддержку для всех своих продуктов. Поддержка доступна всем пользователям, включая оценочные версии.

{{% /alert %}} 

Если вам нужна помощь с Aspose.Slides для JasperReports, рассмотрите следующие рекомендации:

- Основной канал поддержки — это [**Aspose.Forum**](https://forum.aspose.com/c/slides/11). Опубликуйте свой вопрос на форуме Aspose.Slides, и на него ответят в течение нескольких часов. Вопросы отвечают непосредственно сотрудники службы поддержки Aspose.Slides.
- Обратите внимание, что Aspose не предоставляет техническую поддержку по телефону. Телефонная поддержка доступна только для вопросов о продаже и покупке.
- При ожидании ответа на форумах учитывайте разницу во времени.


Если у вас возникла проблема с Aspose.Slides для JasperReports, выполните следующие простые шаги, чтобы решить её наиболее эффективным способом:

- Убедитесь, что вы используете последнюю версию Aspose.Slides для JasperReports перед тем, как сообщить о проблеме; посмотрите загрузку Aspose.Slides, чтобы узнать о последней версии.
- Просмотрите форумы, эту документацию и справочник API перед тем, как сообщить о проблеме; возможно, на ваш вопрос уже был дан ответ.
- При сообщении о проблеме, пожалуйста, включите оригинальный документ и, возможно, фрагмент вашего кода, который вызывает проблему. Если вам нужно прикрепить несколько файлов, упакуйте их в один архив. Безопасно прикреплять ваши документы в Aspose.Forums, так как только вы и разработчики Aspose будете иметь доступ к прикрепленным файлам.
- Пожалуйста, старайтесь сообщать об одной проблеме в одном потоке. Если у вас есть другая проблема, сообщите о ней в отдельном потоке.
