## **О PPT**
[PPT](https://ru.wikipedia.org/wiki/Microsoft_PowerPoint) — это формат файла презентационного документа, который может быть создан, прочитан, изменен и записан различными версиями Microsoft PowerPoint. Это двоичный формат для презентационных документов, разработанный Microsoft.
## **PPT в Aspose.Slides для C++**
Aspose.Slides для C++ может читать файлы PPT, созданные программным обеспечением, перечисленным ниже.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Аналогичным образом, файлы PPT, созданные Aspose.Slides для C++, могут быть прочитаны указанным выше набором программного обеспечения.
## **Всеобъемлющая поддержка для PPT**
Aspose.Slides для C++ предоставляет поддержку почти всех функций, связанных с форматом файла PPT документа. Он охватывает не только основные/расширенные функции, предоставляемые различными версиями Microsoft PowerPoint для манипуляций с документами PPT, но и некоторые функции, которые даже не поддерживаются Microsoft PowerPoint. Основное преимущество использования библиотеки API Aspose.Slides для C++ заключается в простоте использования для обработки таких функций.

В дополнение к основным задачам, связанным с созданием, чтением и записью файлов документов PPT, есть несколько функций, предоставляемых Aspose.Slides для C++, таких как:

- Импорт других форматов файлов MS Office как OLE-объекты в документы PPT
- Экспорт документов PPT в форматы PDF, TIFF, XPS
- Экспорт слайдов в документах PPT в форматы SVG
- Рендеринг слайдов в любой формат изображения, поддерживаемый C++ Framework
- Установка размера слайдов в документе PPT
- Управление анимациями на формах
- Управление показами слайдов
- Форматирование текста на слайдах
- Сканирование текста из документов PPT
- Обработка таблиц на слайдах
- Автоматическое копирование мастеров с помощью функции клонирования
