## **Добавить Ошибочную Полосу**
Aspose.Slides для C++ предоставляет простой API для управления значениями ошибочной полосы. Пример кода применяется при использовании пользовательского типа значения. Чтобы указать значение, используйте свойство **ErrorBarCustomValues** конкретной точки данных в коллекции **DataPoints** серии:

1. Создайте экземпляр класса [Presentation](https://reference.aspose.com/slides/net/aspose.slides/presentation).
1. Добавьте пузырьковую диаграмму на нужном слайде.
1. Получите доступ к первой серии диаграммы и установите формат ошибочной полосы X.
1. Получите доступ к первой серии диаграммы и установите формат ошибочной полосы Y.
1. Установите значения и формат полос.
1. Запишите изменённую презентацию в файл PPTX.

## **Добавить Пользовательскую Ошибочную Полосу**
Aspose.Slides для C++ предоставляет простой API для управления значениями пользовательской ошибочной полосы. Пример кода применяется, когда свойство **IErrorBarsFormat.ValueType** равно **Custom**. Чтобы указать значение, используйте свойство **ErrorBarCustomValues** конкретной точки данных в коллекции **DataPoints** серии:

1. Создайте экземпляр класса [Presentation](https://reference.aspose.com/slides/net/aspose.slides/presentation).
1. Добавьте пузырьковую диаграмму на нужном слайде.
1. Получите доступ к первой серии диаграммы и установите формат ошибочной полосы X.
1. Получите доступ к первой серии диаграммы и установите формат ошибочной полосы Y.
1. Получите доступ к индивидуальным точкам данных серии диаграммы и установите значения ошибочной полосы для отдельной точки данных серии.
1. Установите значения и формат полос.
1. Запишите изменённую презентацию в файл PPTX.
