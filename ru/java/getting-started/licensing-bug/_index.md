В этой статье описывается ошибка лицензирования, возникшая в версиях Aspose.Slides для Java 23.2, 23.3, 23.4, 23.5, 23.6 и 23.7. Эта ошибка привела к тому, что бессрочные лицензии временно стали недоступными после истечения срока подписки.

## Симптомы ##

После истечения срока подписки на вашу бессрочную лицензию Aspose.Slides для Java версии 23.2 - 23.7 могут сообщить о том, что лицензия истекла, переводя все функции в режим оценки. Это поведение является неправильным и не затрагивает версии до 23.2 и после 23.7.

## Решение ##

Aspose.Slides для Java решила эту проблему и выпустила обновленные версии (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) с исправлением.

Если вы используете одну из затронутых версий Aspose.Slides для Java в вашем проекте, пожалуйста, перейдите на использование исправленной версии.

Если вы столкнулись с какими-либо проблемами лицензирования с Aspose.Slides для Java:

- Проверьте версию, которую вы используете, и убедитесь, что она не затронута вышеописанной ошибкой.
- Обратитесь к [документации Aspose](https://docs.aspose.com/slides/java/getting-started/) для получения шагов по устранению неполадок и известным проблемам лицензирования.
- Свяжитесь с [поддержкой Aspose](https://forum.aspose.com/) для получения дополнительной помощи.

## Затронутые продукты и версии ##

Обратите внимание, что эта ошибка затрагивает только версии Aspose.Slides для Java с 23.2 по 23.7. **Другие продукты Aspose и другие версии Aspose.Slides не затронуты**.
