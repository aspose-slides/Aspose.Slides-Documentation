Вы можете легко скачать Aspose.Slides для оценки. Загрузка для оценки такая же, как и при покупке. Оценочная версия просто становится лицензированной, когда вы добавляете несколько строк кода для применения лицензии.

Оценочная версия Aspose.Slides (без указанной лицензии) предоставляет полную функциональность продукта, но вставляет водяной знак оценки в верхней части документа при открытии и сохранении, и ограничивает извлечение текста из слайдов презентации одной страницей.
