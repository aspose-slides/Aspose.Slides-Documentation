## **Оценка Aspose.Slides**

Вы можете легко скачать Aspose.Slides для оценки. Загрузка для оценки такая же, как и покупная загрузка. Версия оценки просто становится лицензированной, когда вы добавляете несколько строк кода для применения лицензии.

Версия оценки Aspose.Slides (без указания лицензии) предоставляет полный функционал продукта, но вставляет водяной знак оценки в верхнюю часть документа при открытии и сохранении, а также ограничивает извлечение текста из слайдов одной страницей.

## **Часто задаваемые вопросы**

**Могу ли я тестировать несколько презентаций параллельно в разных потоках в режиме оценки?**

Да. Вы можете обрабатывать разные документы параллельно; не следует использовать один и тот же объект презентации [в разных потоках](/slides/ru/java/multithreading/). Режим оценки не влияет на это.

**Нужно ли устанавливать Microsoft PowerPoint для оценки библиотеки на сервере или в CI?**

Нет. Aspose.Slides — это автономный движок и не требует установленного PowerPoint ни для оценки, ни для продакшна.

**Могу ли я полностью протестировать конвертацию PPT/PPTX в PDF и изображения в режиме оценки?**

Да. [Конвертеры](/slides/ru/java/convert-presentation/) работают; вывод будет содержать водяной знак.

**Можно ли использовать временную лицензию для нагрузочного тестирования без водяного знака?**

Да. 30‑дневная временная лицензия снимает ограничения режима оценки и позволяет тестировать без водяного знака.
