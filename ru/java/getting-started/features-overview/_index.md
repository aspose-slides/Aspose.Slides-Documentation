|**Функция**|**Описание**|
| :- | :- |

|[Конверсия PPT в PPTX](/slides/ru/java/ppt-to-pptx-conversion/)|Aspose.Slides for Java поддерживает конверсию из PPT в PPTX.|

## **Форматирование**
Можно форматировать текст и фигуры на слайдах презентации, используя Aspose.Slides for Java.

|**Функция**|**Описание**|
| :- | :- |
|Форматирование текста|<p>В Aspose.Slides for Java текст управляется через текстовые кадры, связанные с фигурами. Таким образом, текст форматируется с помощью абзацев и частей, связанных с текстовыми кадрами. Можно форматировать следующие элементы текста.</p><p>- Тип шрифта.</p><p>- Размер шрифта.</p><p>- Цвет шрифта.</p><p>- Тени шрифта.</p><p>- Выравнивание абзаца.</p><p>- Маркирование абзаца.</p><p>- Ориентация абзаца.</p>|
|Форматирование фигур|<p>В Aspose.Slides for Java базовым элементом слайда является фигура. Следующие элементы фигуры могут быть отформатированы с помощью Aspose.Slides for Java:</p><p>- Позиция</p><p>- Размер</p><p>- Линия</p><p>- Заливка (включая узор, градиент и сплошную).</p><p>- Текст</p><p>- Изображение</p>|

## **Часто задаваемые вопросы**

**Нужно ли устанавливать Microsoft PowerPoint на сервер/ПК для работы библиотеки?**

Нет. PowerPoint не требуется; Aspose.Slides — автономный механизм для создания, редактирования, конвертации и рендеринга презентаций.

**Как работает многопоточность? Можно ли параллелизировать обработку?**

Безопасно обрабатывать разные документы в разных потоках; один и тот же объект [презентации](https://reference.aspose.com/slides/java/com.aspose.slides/presentation/) не должен использоваться [несколькими потоками](/slides/ru/java/multithreading/) одновременно.

**Поддерживаются ли пароли и шифрование файлов?**

Да. [Вы можете](/slides/ru/java/password-protected-presentation/) открывать зашифрованные презентации, задавать или удалять пароль для открытия и записи, а также проверять статус защиты.

**Нужно ли заботиться о шрифтовых пакетах в Linux‑контейнерах?**

Да. Рекомендуется установить общие пакеты шрифтов и/или явно [указать каталоги шрифтов](/slides/ru/java/custom-font/) в вашем приложении, чтобы избежать неожиданных замен.

**Есть ли ограничения в оценочной версии?**

В [режиме оценки](/slides/ru/java/licensing/) к выводу добавляется водяной знак и применяются определённые ограничения; доступна [30‑дневная временная лицензия](https://purchase.aspose.com/temporary-license/) для полного тестирования возможностей.

**Поддерживается ли импорт внешних форматов в презентацию (PDF/HTML → PPTX)?**

Да. Вы можете добавить [страницы PDF и HTML‑контент](/slides/ru/java/import-presentation/) в презентацию, превратив их в слайды.
