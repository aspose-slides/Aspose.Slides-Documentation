## **Обзор**
Aspose.Slides for Java не требует установки Microsoft PowerPoint, так как Aspose.Slides представляет собой движок для создания, конвертации, разметки страниц и рендеринга документов Microsoft PowerPoint.

## **Поддерживаемые операционные системы**
Aspose.Slides for Java поддерживает любые 32‑разрядные или 64‑разрядные операционные системы, на которых работает среда выполнения Java, включая, но не ограничиваясь:

### **Windows**
- Microsoft Windows 2003 Server ( x64, x86)
- Microsoft Windows 2008 Server ( x64, x86)
- Microsoft Windows 2012 Server ( x64, x86)
- Microsoft Windows 2012 R2 Server ( x64, x86)
- Microsoft Windows 2016 Server ( x64, x86)
- Microsoft Windows 2019 Server ( x64, x86)
- Microsoft Windows Vista ( x64, x86)
- Microsoft Windows XP ( x64, x86)
- Microsoft Windows 7 ( x64, x86)
- Microsoft Windows 8, 8.1 ( x64, x86)
- Microsoft Windows 10 ( x64, x86)

### **Linux**
- Linux (Ubuntu, OpenSUSE, CentOS and others)

### **Mac**
- Mac OS X

## **Поддерживаемые версии Java**
Aspose.Slides for Java поддерживает J2SE 6.0 (Java 1.6) и более новые версии.

## **Часто задаваемые вопросы**

**Нужен ли установленный Microsoft PowerPoint для конвертации и рендеринга?**

Нет, PowerPoint не требуется; Aspose.Slides — это автономный движок для [создания](/slides/ru/java/create-presentation/), изменения, [конвертации](/slides/ru/java/convert-presentation/) и [рендеринга](/slides/ru/java/convert-powerpoint-to-png/) презентаций.

**Какие шрифты требуются для корректного рендеринга?**

На практике должны быть доступны шрифты, использованные в презентации, или их подходящие [замены](/slides/ru/java/font-substitution/). Чтобы обеспечить согласованный рендеринг в Linux/macOS, рекомендуется установить общие пакеты шрифтов.

**Почему пользовательский шрифт отображается как запасной или отсутствующий текст в Linux?**

Если в файле шрифта имеются неконсистентные или повреждённые записи в таблице имён, стек сопоставления шрифтов Linux (FreeType/fontconfig) может выбрать недействительную запись, что приводит к невозможности разрешения шрифта. Использование версии шрифта с исправленными записями таблицы имён или установка согласованной замены решает проблему.
