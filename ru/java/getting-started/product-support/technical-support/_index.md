## **Техническая поддержка**

{{% alert color="primary" %}} 
Aspose предоставляет неограниченную бесплатную техническую поддержку для всех своих продуктов. Поддержка доступна всем пользователям, включая оценочную версию.
{{% /alert %}} 

Если вам нужна помощь с Aspose.Slides for Java, рассмотрите следующее:

- Основным каналом поддержки являются Aspose.Forums. Опубликуйте ваш вопрос в форуме Aspose.Slides, и он будет отвечен в течение нескольких часов. На вопросы отвечает непосредственно команда поддержки Aspose.Slides.
- Обратите внимание, Aspose не предоставляет техническую поддержку по телефону. Телефонная поддержка доступна только для вопросов по продажам и покупке.
- Ожидая ответ на форумах, пожалуйста, учитывайте разницу часовых поясов.

Если у вас возникла проблема с Aspose.Slides for Java, выполните следующие простые шаги, чтобы решить её максимально эффективно:

- Убедитесь, что вы используете последнюю версию Aspose.Slides for Java перед сообщением об ошибке, см. загрузку Aspose.Slides, чтобы узнать о последней версии.
- Просмотрите форумы, эту документацию и справочник API перед сообщением об ошибке; возможно, ваш вопрос уже был отвечен.
- При сообщении об ошибке, пожалуйста, включите оригинальный документ и, возможно, фрагмент вашего кода, вызывающего проблему. Если нужно прикрепить несколько файлов, упакуйте их в один ZIP. Прикреплять документы в Aspose.Forums безопасно, так как только вы и разработчики Aspose будут иметь доступ к вложенным файлам.
- Пожалуйста, старайтесь сообщать об одной проблеме в каждом обсуждении. Если у вас есть другая проблема, создайте отдельное обсуждение.

## **Часто задаваемые вопросы**

**Какие уровни платной поддержки доступны и чем они отличаются?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Они различаются по количеству включённых инцидентов, приоритету ответов и доступным каналам связи.

**Могу ли я эскалировать баг напрямую к инженерной команде?**

Да. [Paid support](https://helpdesk.aspose.com/) позволяет эскалировать к разработчикам, когда проблема требует более детального расследования.

**Поддерживаются ли более старые версии библиотеки и выпускаются ли для них исправления?**

Вы можете использовать более старые версии, но исправления предоставляются только для текущего релиза; патчи для устаревших версий не выпускаются.
