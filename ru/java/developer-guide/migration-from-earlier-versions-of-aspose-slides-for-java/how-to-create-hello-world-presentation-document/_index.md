## **Поддержка устаревшего кода**
Для использования устаревшего кода, разработанного с Aspose.Slides для Java версий до 13.x, вам нужно внести некоторые небольшие изменения в ваш код, и он будет работать как раньше. Все классы, которые были представлены в старой версии Aspose.Slides для Java в пространствах имен Aspose.Slide и Aspose.Slides.Pptx, теперь объединены в одно пространство имен Aspose.Slides. Пожалуйста, посмотрите следующий простой фрагмент кода для создания документа презентации Hello World в устаревшем API Aspose.Slides и следуйте шагам, описывающим, как перейти на новый объединённый API.
