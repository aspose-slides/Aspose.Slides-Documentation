## **Поддержка устаревшего кода**
Чтобы использовать устаревший код, разработанный для Aspose.Slides для Java версий ранее 13.x, необходимо внести небольшие изменения в ваш код, и он будет работать как и прежде. Все классы, которые находились в старой версии Aspose.Slides для Java в пространствах имен Aspose.Slide и Aspose.Slides.Pptx, теперь объединены в единственное пространство имен Aspose.Slides. Пожалуйста, ознакомьтесь со следующим простым примером кода для создания презентации Hello World в устаревшем API Aspose.Slides и следуйте шагам, описывающим миграцию на новое объединённое API.
