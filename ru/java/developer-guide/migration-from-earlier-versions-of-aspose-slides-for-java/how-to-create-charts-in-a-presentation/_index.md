## **Поддержка устаревшего кода**
Чтобы использовать устаревший код, разработанный с помощью Aspose.Slides для Java версий до 14.x.x, вам нужно внести некоторые небольшие изменения в ваш код, и он будет работать как раньше. Все классы, которые были в старых версиях Aspose.Slides для Java в пространствах имен com.aspose.slides и com.aspose.slides.pptx, теперь объединены в одно пространство имен com.aspose.slides.
