## **Поддержка устаревшего кода**
Чтобы использовать устаревший код, написанный для Aspose.Slides for Java версии ниже 14.x.x, вам нужно внести небольшие изменения в ваш код, и он будет работать так же, как раньше. Все классы, которые находились в старом Aspose.Slides for Java в пространствах имён com.aspose.slides и com.aspose.slides.pptx, теперь объединены в едином пространстве имён com.aspose.slides.
