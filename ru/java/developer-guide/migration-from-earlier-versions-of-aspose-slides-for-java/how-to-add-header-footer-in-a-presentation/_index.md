Выпущен новый [Aspose.Slides для Java API](https://docs.aspose.com/slides/java/), который теперь поддерживает возможность создания документов PowerPoint с нуля и редактирования существующих документов.

{{% /alert %}} 
## **Поддержка устаревшего кода**
Чтобы использовать устаревший код, разработанный с использованием Aspose.Slides для Java версий ранее 13.x, вам необходимо внести некоторые незначительные изменения в ваш код, и он будет работать как и прежде. Все классы, которые присутствовали в старом Aspose.Slides для Java в пространствах имен Aspose.Slide и Aspose.Slides.Pptx, теперь объединены в одно пространство имен Aspose.Slides. Пожалуйста, ознакомьтесь со следующим простым фрагментом кода для добавления верхнего и нижнего колонтитулов в презентацию в устаревшем API Aspose.Slides и следуйте шагам, описывающим, как перейти на новый объединенный API.
