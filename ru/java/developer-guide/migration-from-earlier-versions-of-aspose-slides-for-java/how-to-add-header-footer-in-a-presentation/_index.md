Выпущен новый [Aspose.Slides for Java API](https://docs.aspose.com/slides/java/) и теперь этот единый продукт поддерживает возможность создавать документы PowerPoint с нуля и редактировать существующие.

{{% /alert %}} 
## **Поддержка устаревшего кода**
Чтобы использовать устаревший код, разработанный для Aspose.Slides for Java версий ранее 13.x, необходимо внести небольшие изменения в ваш код, и он будет работать как прежде. Все классы, которые ранее находились в пространствах имён Aspose.Slide и Aspose.Slides.Pptx в старой версии Aspose.Slides for Java, теперь объединены в единственное пространство имён Aspose.Slides. Пожалуйста, ознакомьтесь со следующим простым фрагментом кода для добавления верхнего и нижнего колонтитулов в презентацию в устаревшем API Aspose.Slides и следуйте шагам, описывающим миграцию к новому объединённому API.
