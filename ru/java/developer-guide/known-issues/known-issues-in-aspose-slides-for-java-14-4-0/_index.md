{{% alert color="primary" %}} 

Aspose.Slides for Java 14.4.0 предоставляет новое решение для обработки документов PowerPoint. Существуют некоторые ограничения и известные проблемы, которые будут устранены в будущих выпусках:

- Некоторые фигуры имеют неправильную геометрию в сериализованных PPT-документах (дуга, круглая стрелка, всплывающие подсказки).
- Не все функции форматирования текста PPTX поддерживаются при сериализации в PPT (ограничения табуляции, отступов и форматирования абзацев).
- Информация о языке текста и настройках орфографии отсутствует в сериализованных PPT-документах.
- Не все возможности темы PPTX поддерживаются при сериализации в PPT (включена только сериализация форматов заливки, линий и шрифтов).
- Известны проблемы при сериализации OLE/ActiveX PPT в PPT.
- Сериализация и рендеринг WordArt не поддерживаются.

{{% /alert %}}
