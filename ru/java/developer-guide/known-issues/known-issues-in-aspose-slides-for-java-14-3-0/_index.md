Aspose.Slides для Java 14.3.0 (14.4.0) предоставляет полностью новую реализацию обработки PPT. Есть множество улучшений, частичное преобразование PPTX в PPT. Но есть некоторые не реализованные функции:

- Некоторые фигуры имеют неправильную геометрию в сериализованных PPT‑документах (выноски)
- Не все возможности форматирования текста PPTX поддерживаются при сериализации в PPT
- Информация о языке текста и настройках орфографии отсутствует в сериализованных PPT‑документах
- Не все возможности тем PPTX поддерживаются при сериализации в PPT

**Есть некоторые различия по сравнению с Aspose.Slides for Java 8.6.0:**

- Известны проблемы при сериализации OLE/ActiveX PPT в PPT

**Есть некоторые различия по сравнению с Aspose.Slides for .NET 14.3.0:**

- Поддержка печати презентаций в настоящее время недоступна в Aspose.Slides for Java
