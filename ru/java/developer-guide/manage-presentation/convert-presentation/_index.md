## **Обзор**

Aspose.Slides for Java может загружать презентации PowerPoint и OpenDocument и сохранять или визуализировать их в многие другие форматы без Microsoft PowerPoint, OpenOffice или LibreOffice. Вы можете конвертировать устаревшие файлы PPT в современные PPTX, экспортировать презентации в документы фиксированного макета, такие как PDF и XPS, публиковать слайды как HTML или визуализировать слайды в виде изображений для предварительного просмотра, миниатюр и архивов.

Большинство конвертаций документов используют один и тот же общий рабочий процесс: загрузить исходный файл, выбрать требуемый формат вывода и при необходимости применить параметры, специфичные для формата. Для графических форматов каждый слайд визуализируется отдельно, а затем сохраняется как растровое или векторное изображение. Специализированные статьи, указанные ниже, предоставляют детали реализации для каждого случая.

## **Конвертация PowerPoint и OpenDocument**

Aspose.Slides for Java поддерживает конвертацию из часто используемых форматов презентаций, таких как PPT, PPTX, PPS, PPSX, POT, POTX и ODP. Один и тот же API конвертации используется для файлов PowerPoint и OpenDocument, поэтому рабочий процесс, сохраняющий файл PPTX в PDF, обычно можно применить к файлу ODP, изменив только входной файл.

При конвертации файлов ODP помните, что приложения PowerPoint и OpenDocument не поддерживают каждую макетную и форматировочную функцию одинаково. Если файл ODP был создан в LibreOffice или OpenOffice Impress, проверьте результат и используйте параметры, описанные в [Конвертировать презентации OpenDocument](/slides/ru/java/convert-openoffice-odp/), когда требуется руководство по конкретному формату.

## **Конвертация PPT в PPTX**

PPT — старый бинарный формат PowerPoint, тогда как PPTX — современный формат Office Open XML. Aspose.Slides for Java поддерживает высокоточные конвертации PPT в PPTX с сохранением сложных структур презентации, таких как мастера, макеты, слайды, диаграммы, сгруппированные объекты, заполнители, текстовые рамки, текстуры и заливки изображениями.

Подробности см. в [Конвертировать PPT в PPTX](/slides/ru/java/convert-ppt-to-pptx/) и [PPT vs PPTX](/slides/ru/java/ppt-vs-pptx/).

## **Экспорт с фиксированным макетом**

PDF, XPS и TIFF полезны, когда вывод должен выглядеть одинаково на всех устройствах и не должен редактироваться как презентация. Специализированные статьи по PDF, XPS и TIFF объясняют, как управлять соответствием стандартам, скрытыми слайдами, заметками, качеством изображений, сжатием, форматом пикселей и размером вывода.

## **Экспорт в HTML и изображения**

Экспорт в HTML и HTML5 полезен для просмотра в браузере, веб‑публикаций и лёгкого обмена. Экспорт изображений полезен, когда каждый слайд должен стать отдельным предварительным просмотром, миниатюрой или растровым ресурсом. Используйте статьи о PNG, JPG и SVG для получения рекомендаций по рендерингу для конкретных форматов.

## **Вопросы и ответы**

**Нужен ли мне Microsoft PowerPoint для конвертации презентаций?**

Нет. Aspose.Slides for Java является автономной библиотекой и не требует Microsoft PowerPoint или автоматизации Office.

**Могу ли я пакетно конвертировать много презентаций?**

Да. Загружайте каждую презентацию, сохраняйте её в требуемый формат и освобождайте объект презентации после обработки. Для параллельной обработки используйте отдельные экземпляры презентаций и следуйте рекомендациям по [multithreading](/slides/ru/java/multithreading/).

**Могу ли я экспортировать только выбранные слайды?**

Да. Несколько методов экспорта позволяют передавать индексы слайдов или визуализировать отдельные слайды, в зависимости от формата вывода. См. специализированную статью для целевого формата.

**Могу ли я включать скрытые слайды при экспорте в PDF или XPS?**

Да. Используйте настройки экспорта скрытых слайдов, описанные в статьях о [PDF](/slides/ru/java/convert-powerpoint-to-pdf/) и [XPS](/slides/ru/java/convert-powerpoint-to-xps/).

**Могу ли я создать вывод PDF/A?**

Да. Параметры соответствия PDF доступны при экспорте в PDF. Смотрите [Конвертировать PowerPoint в PDF](/slides/ru/java/convert-powerpoint-to-pdf/) для деталей.

**Как обрабатываются шрифты во время конвертации?**

Aspose.Slides может использовать встроенные шрифты, резервные шрифты и настройки замены шрифтов. См. [Встроенный шрифт](/slides/ru/java/embedded-font/), [Резервный шрифт](/slides/ru/java/fallback-font/) и [Замена шрифтов](/slides/ru/java/font-substitution/).
