## **Обзор**

Aspose.Slides for Java может загружать презентации PowerPoint и OpenDocument и сохранять или рендерить их во многие другие форматы без Microsoft PowerPoint, OpenOffice или LibreOffice. Вы можете конвертировать устаревшие файлы PPT в современные PPTX, экспортировать презентации в фиксированные документы, такие как PDF и XPS, публиковать слайды как HTML или рендерить слайды в виде файлов изображений для предварительного просмотра, миниатюр и архивов.

Большинство конвертаций документов используют один и тот же общий рабочий процесс: загрузить исходный файл, выбрать необходимый формат вывода и при необходимости применить параметры, специфичные для формата. Для форматов изображений каждый слайд рендерится отдельно, а затем сохраняется как растровое или векторное изображение. Специальные статьи, указанные ниже, предоставляют детали реализации для каждого случая.

## **Конвертация PowerPoint и OpenDocument**

Aspose.Slides for Java поддерживает конвертацию из часто используемых форматов презентаций, таких как PPT, PPTX, PPS, PPSX, POT, POTX и ODP. Один и тот же API конвертации используется для файлов PowerPoint и OpenDocument, поэтому рабочий процесс, который сохраняет файл PPTX в PDF, обычно можно применить к файлу ODP, изменив только входной файл.

При конвертации файлов ODP помните, что приложения PowerPoint и OpenDocument не поддерживают каждую компоновку и форматирование одинаково. Если файл ODP создан в LibreOffice или OpenOffice Impress, проверьте результат и используйте параметры, описанные в [Конвертировать презентации OpenDocument](/slides/ru/java/convert-openoffice-odp/), когда требуется руководство по конкретному формату.

## **Конвертация PPT в PPTX**

PPT — это старый бинарный формат PowerPoint, тогда как PPTX — современный формат Office Open XML. Aspose.Slides for Java поддерживает высокоточную конвертацию PPT в PPTX, сохраняя сложные структуры презентации, такие как шаблоны, макеты, слайды, диаграммы, сгруппированные фигуры, заполнители, текстовые кадры, текстуры и заливки изображениями.

Подробности см. в [Конвертировать PPT в PPTX](/slides/ru/java/convert-ppt-to-pptx/) и [PPT vs PPTX](/slides/ru/java/ppt-vs-pptx/).

## **Экспорт фиксированного макета**

PDF, XPS и TIFF полезны, когда вывод должен выглядеть одинаково на разных устройствах и не должен редактироваться как презентация. Специализированные статьи о PDF, XPS и TIFF объясняют, как управлять соответствием, скрытыми слайдами, заметками, качеством изображения, сжатием, форматом пикселей и размером вывода.

## **Экспорт HTML и изображений**

Экспорт в HTML и HTML5 полезен для просмотра в браузере, публикации в вебе и лёгкого обмена. Экспорт изображений полезен, когда каждый слайд должен стать отдельным превью, миниатюрой или растровым ресурсом. Используйте статьи о PNG, JPG и SVG для получения рекомендаций по рендерингу, специфичным для формата.

## **Вопросы и ответы**

**Нужен ли мне Microsoft PowerPoint для конвертации презентаций?**

Нет. Aspose.Slides for Java — это автономная библиотека и не требует Microsoft PowerPoint или автоматизации Office.

**Могу ли я пакетно конвертировать много презентаций?**

Да. Загружайте каждую презентацию, сохраняйте её в требуемый формат и освобождайте объект презентации после обработки. Для параллельной обработки используйте отдельные экземпляры презентаций и следуйте рекомендациям по [многопоточности](/slides/ru/java/multithreading/).

**Могу ли я экспортировать только выбранные слайды?**

Да. Несколько методов экспорта позволяют передавать индексы слайдов или рендерить отдельные слайды в зависимости от формата вывода. Смотрите специальную статью для целевого формата.

**Могу ли я включать скрытые слайды при экспорте в PDF или XPS?**

Да. Используйте настройки экспорта скрытых слайдов, описанные в статьях о [PDF](/slides/ru/java/convert-powerpoint-to-pdf/) и [XPS](/slides/ru/java/convert-powerpoint-to-xps/).

**Могу ли я создать вывод PDF/A?**

Да. Параметры соответствия PDF доступны для экспорта в PDF. Смотрите [Конвертировать PowerPoint в PDF](/slides/ru/java/convert-powerpoint-to-pdf/) для подробностей.

**Как обрабатываются шрифты при конвертации?**

Aspose.Slides может использовать вложенные шрифты, резервные шрифты и настройки подстановки шрифтов. Смотрите [Встроенный шрифт](/slides/ru/java/embedded-font/), [Резервный шрифт](/slides/ru/java/fallback-font/), и [Подстановка шрифтов](/slides/ru/java/font-substitution/).
