{{% alert color="primary" %}} 

Aspose.Slides for Java — это библиотека классов, позволяющая Java-приложениям читать и записывать презентационные документы без использования Microsoft PowerPoint® или других программ. Aspose.Slides for Java позволяет управлять текстом, фигурами, таблицами и анимациями, добавлять аудио и видео в слайды, просматривать слайды, экспортировать слайды в форматы SVG, PDF и многое другое.

Последнюю справочную документацию API можно найти на
[Справка API Aspose.Slides for Java.](https://reference.aspose.com/slides/java)
