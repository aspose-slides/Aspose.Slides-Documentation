{{% alert color="primary" %}} 
[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) — это файловый формат презентационных документов, который может создаваться, читаться, изменяться и записываться различными версиями Microsoft PowerPoint. Это бинарный формат презентационных документов, разработанный компанией Microsoft.
{{% /alert %}} 

## **PPT в Aspose.Slides for PHP via Java**
Aspose.Slides for PHP via Java может читать файлы PPT, созданные перечисленным ниже программным обеспечением.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Аналогично, файлы PPT, созданные Aspose.Slides for PHP via Java, могут быть прочитаны указанным выше набором программного обеспечения.

## **Полная поддержка PPT**
Aspose.Slides for PHP via Java предоставляет поддержку практически всех функций, поддерживаемых форматом файлов PPT. Он охватывает не только базовые и расширенные возможности, предоставляемые различными версиями Microsoft PowerPoint для работы с документами PPT, но и функции, которые даже не поддерживаются Microsoft PowerPoint. Основным преимуществом использования библиотеки Aspose.Slides for PHP via Java API является простота работы с такими функциями.

Помимо базовых задач, связанных с созданием, чтением и записью файлов PPT, Aspose.Slides for PHP via Java предоставляет несколько функций:
- Импортировать другие форматы файлов Microsoft Office в виде [OLE-объектов в документы PPT](/slides/ru/php-java/manage-ole/).
- [Экспортировать документы PPT в PDF](/slides/ru/php-java/convert-powerpoint-to-pdf/).
- Экспортировать слайды в документах PPT в форматы SVG.
- Отображать слайды в любой графический формат, поддерживаемый Java Framework.
- Устанавливать размер слайдов в документах PPT.
- Управлять анимацией фигур.
- Управлять демонстрациями слайдов.
- [Форматировать текст на слайдах](/slides/ru/php-java/text-formatting/).
- Извлекать текст из документов PPT.
- [Работать с таблицами на слайдах](/slides/ru/php-java/powerpoint-table/).
- Автоматически копировать шаблоны с помощью [функции клонирования](/slides/ru/php-java/clone-slides/).
