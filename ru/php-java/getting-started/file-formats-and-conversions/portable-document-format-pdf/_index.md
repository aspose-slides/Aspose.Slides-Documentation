{{% alert color="primary" %}}

[Portable Document Format](https://en.wikipedia.org/wiki/PDF) — это файловый формат, созданный компанией Adobe Systems для обмена документами между организациями. Цель формата заключалась в том, чтобы содержимое и макет оставались одинаковыми независимо от платформы, на которой они просматриваются. Aspose.Slides for PHP via Java позволяет конвертировать файлы презентаций в PDF.

{{% /alert %}}

## **PDF в Aspose.Slides for PHP via Java**
Любая презентация, которую можно загрузить в Aspose.Slides for PHP via Java, может быть преобразована в PDF, соответствующий [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) или [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA) в зависимости от вашего выбора. Aspose.Slides for PHP via Java экспортирует презентации в PDF, и в большинстве случаев полученный PDF выглядит точно так же, как оригинальная презентация.

Aspose.Slides поддерживает следующие возможности презентаций при конвертации в PDF:

- Изображения, текстовые блоки и другие фигуры.
- Текст и форматирование.
- Абзацы и форматирование.
- Гиперссылки.
- Колонтитулы.
- Маркированные списки.
- Таблицы.

Вы можете экспортировать презентации в PDF напрямую, используя Aspose.Slides for PHP via Java: вам не нужен какой‑либо другой компонент. Кроме того, вы можете настроить экспорт презентации в PDF с помощью различных параметров, как описано в [Конвертация в PDF](/slides/ru/php-java/convert-powerpoint-to-pdf/).
