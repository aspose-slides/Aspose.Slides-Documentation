## **О TIFF**

[Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF), известный тем, что позволяет хранить несколько изображений в одном документе, изначально был создан компанией Aldus. Этот формат широко поддерживается сканирующими, факсимильными и другими приложениями для обработки изображений.

## **TIFF в Aspose.Slides for PHP via Java**

Любой документ, который можно загрузить в Aspose.Slide for Java, также может быть напрямую преобразован в документ TIFF с помощью Aspose.Slides for PHP via Java, без необходимости в сторонних компонентах. Кроме того, при желании можно задать размер изображений в получающемся документе TIFF. Информацию об экспорте презентаций в документы TIFF через Aspose.Slides for PHP via Java можно найти в [этой теме](/slides/ru/php-java/convert-powerpoint-to-tiff/).
