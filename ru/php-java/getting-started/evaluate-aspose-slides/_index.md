## **Оценка Aspose.Slides**

Вы можете легко загрузить Aspose.Slides для оценки. Пакет оценки идентичен приобретённому пакету. Версия оценки просто становится лицензированной после того, как вы добавите несколько строк кода для применения лицензии.

Версия оценки Aspose.Slides (без указания лицензии) предоставляет полный функционал продукта, но вставляет водяной знак оценки в верхнюю часть документа при открытии и сохранении. При извлечении текста из слайдов презентации вы также ограничены одним слайдом.

## **FAQ**

**Могу ли я тестировать несколько презентаций одновременно в разных потоках в режиме оценки?**

Да. Вы можете обрабатывать разные документы параллельно; не следует использовать один и тот же объект презентации [across threads](/slides/ru/php-java/multithreading/). Режим оценки не влияет на это.

**Нужно ли устанавливать Microsoft PowerPoint для оценки библиотеки на сервере или в CI?**

Нет. Aspose.Slides — это автономный движок и не требует установленного PowerPoint ни для оценки, ни для производства.

**Могу ли я полностью протестировать конвертацию PPT/PPTX в PDF и изображения в режиме оценки?**

Да. [Конвертеры](/slides/ru/php-java/convert-presentation/) работают; в результате будет отображён водяной знак.

**Можно ли использовать временную лицензию для нагрузочного тестирования без водяного знака?**

Да. 30‑дневная временная лицензия устраняет ограничения режима оценки и позволяет тестировать без водяного знака.
