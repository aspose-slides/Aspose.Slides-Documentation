## **Technical support**

{{% alert color="primary" %}} 

Aspose предоставляет неограниченную бесплатную техническую поддержку для всех своих продуктов. Поддержка доступна всем пользователям, включая оценочную версию.

{{% /alert %}} 

If you need help with Aspose.Slides for PHP via Java, consider the following:

- Основным каналом поддержки являются форумы Aspose.Forums. Опубликуйте свой вопрос в форуме Aspose.Slides, и вам ответят в течение нескольких часов. На вопросы отвечает непосредственно команда поддержки Aspose.Slides.
- Обратите внимание, Aspose не предоставляет техническую поддержку по телефону. Телефонная поддержка доступна только для вопросов по продажам и покупке.
- Ожидая ответа на форумах, учитывайте разницу часовых поясов.

If you have an issue with Aspose.Slides for PHP via Java, follow these simple steps to make sure it is resolved in the most efficient way:

- Убедитесь, что вы используете последнюю версию Aspose.Slides for PHP via Java перед тем, как сообщать о проблеме; см. Aspose.Slides download, чтобы узнать о последней версии.
- Просмотрите форумы, эту документацию и справочник API перед тем, как сообщать о проблеме; возможно, ваш вопрос уже был отвечен.
- При сообщении о проблеме, пожалуйста, приложите оригинальный документ и, при необходимости, фрагмент кода, вызывающего проблему. Если нужно прикрепить несколько файлов, упакуйте их в один архив zip. Прикреплять документы в Aspose.Forums безопасно, так как доступ к вложениям имеют только вы и разработчики Aspose.
- Старайтесь описывать одну проблему в одном потоке обсуждения. Если у вас есть другая проблема, создайте отдельный поток.

## **FAQ**

**Какие уровни платной поддержки доступны и чем они отличаются?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Они различаются количеством включенных инцидентов, приоритетом ответа и доступными каналами коммуникации.

**Могу ли я эскалировать ошибку напрямую к инженерной команде?**

Да. [Paid support](https://helpdesk.aspose.com/) позволяет эскалировать к разработчикам, когда проблема требует более глубокого исследования.

**Поддерживаются ли более старые версии библиотек и выпускаются ли исправления для них?**

Вы можете использовать более старые версии, но исправления предоставляются только для текущего выпуска; патчи для устаревших версий не выпускаются.
