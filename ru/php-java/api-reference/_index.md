{{% alert color="primary" %}}

Aspose.Slides for PHP via Java является библиотекой классов, которая позволяет PHP-приложениям читать и записывать презентационные документы без использования Microsoft PowerPoint® или других. Aspose.Slides for PHP via Java позволяет управлять текстом, фигурами, таблицами и анимациями, добавлять аудио и видео в слайды, просматривать слайды, экспортировать слайды в форматы SVG, PDF и многое другое.

Последнюю справочную документацию API можно найти по адресу [Aspose.Slides for PHP via Java API Reference](https://reference.aspose.com/slides/ru/php-java/).
