## **Поддержка устаревшего кода**
Для того чтобы использовать устаревший код, разработанный с Aspose.Slides для PHP через Java версии ранее 13.x, вам нужно внести небольшие изменения в ваш код, и он будет работать как раньше. Все классы, которые были в старых версиях Aspose.Slides для PHP через Java в пространствах имен Aspose.Slide и Aspose.Slides.Pptx, теперь объединили в одно пространство имен Aspose.Slides. Пожалуйста, посмотрите следующий простой фрагмент кода для создания документа презентации Hello World в старом API Aspose.Slides и следуйте шагам, описывающим, как мигрировать на новый объединенный API.
