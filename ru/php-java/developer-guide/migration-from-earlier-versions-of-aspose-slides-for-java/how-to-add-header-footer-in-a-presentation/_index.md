## **Поддержка устаревшего кода**
Чтобы использовать устаревший код, разработанный с помощью Aspose.Slides для PHP через Java версий ниже 13.x, вам нужно внести некоторые незначительные изменения в ваш код, и он будет работать как раньше. Все классы, которые были в старом Aspose.Slides для PHP через Java в пространствах имен Aspose.Slide и Aspose.Slides.Pptx, теперь объединены в одно пространство имен Aspose.Slides. Пожалуйста, ознакомьтесь со следующим простым фрагментом кода для добавления заголовка и нижнего колонтитула в презентацию в устаревшем API Aspose.Slides и следуйте шагам, описывающим, как мигрировать на новый объединенный API.
