Таблицы широко используются для отображения данных на слайдах презентаций. В этой статье показано, как программно создать таблицу размером 15 x 15 с размером шрифта 10, сначала используя [VSTO 2008](/slides/php-java/создание-таблицы-на-слайде-powerpoint/), а затем [Aspose.Slides для PHP через Java](/slides/php-java/создание-таблицы-на-слайде-powerpoint/).

{{% /alert %}} 
## **Создание таблиц**
### **Пример VSTO 2008**
Следующие шаги добавляют таблицу на слайд Microsoft PowerPoint при помощи VSTO:

1. Создайте презентацию.
1. Добавьте пустой слайд в презентацию.
1. Добавьте таблицу размером 15 x 15 на слайд.
1. Добавьте текст в каждую ячейку таблицы с размером шрифта 10.
1. Сохраните презентацию на диск.
