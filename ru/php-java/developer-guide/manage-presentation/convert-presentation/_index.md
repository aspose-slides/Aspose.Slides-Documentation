## **Обзор**

Aspose.Slides for PHP via Java может загружать презентации PowerPoint и OpenDocument и сохранять или визуализировать их в многие другие форматы без Microsoft PowerPoint, OpenOffice или LibreOffice. Вы можете преобразовать устаревшие файлы PPT в современные PPTX, экспортировать презентации в документы фиксированного макета, такие как PDF и XPS, публиковать слайды как HTML или рендерить слайды в файлы изображений для превью, эскизов и архивов.

Большинство преобразований документов используют один и тот же общий рабочий процесс: загрузить исходный файл, выбрать требуемый формат вывода и при необходимости применить параметры, специфичные для формата. Для форматов изображений каждый слайд рендерится отдельно, а затем сохраняется как растровое или векторное изображение. Ниже приведены статьи, содержащие детали реализации для каждого случая.

## **Преобразование PowerPoint и OpenDocument**

Aspose.Slides for PHP via Java поддерживает преобразование из часто используемых форматов презентаций, таких как PPT, PPTX, PPS, PPSX, POT, POTX и ODP. Один и тот же API преобразования используется как для файлов PowerPoint, так и для OpenDocument, поэтому рабочий процесс, который сохраняет PPTX в PDF, обычно можно применить к ODP, изменив лишь входной файл.

При преобразовании файлов ODP помните, что приложения PowerPoint и OpenDocument не поддерживают каждую компоновку и параметр форматирования точно одинаково. Если файл ODP был создан в LibreOffice или OpenOffice Impress, проверьте результат и используйте параметры, описанные в [Convert OpenDocument Presentations](/slides/ru/php-java/convert-openoffice-odp/) при необходимости получения рекомендаций, специфичных для формата.

## **Преобразование PPT в PPTX**

PPT — это старый бинарный формат PowerPoint, тогда как PPTX — современный формат Office Open XML. Aspose.Slides for PHP via Java поддерживает высокоточное преобразование PPT в PPTX с сохранением сложных структур презентации, таких как мастера, макеты, слайды, диаграммы, сгруппированные фигуры, заполнители, текстовые кадры, текстуры и заливки изображениями.

Подробности см. в [Convert PPT to PPTX](/slides/ru/php-java/convert-ppt-to-pptx/) и [PPT vs PPTX](/slides/ru/php-java/ppt-vs-pptx/).

## **Экспорт фиксированного макета**

PDF, XPS и TIFF полезны, когда вывод должен выглядеть одинаково на всех устройствах и не подлежит редактированию как презентация. Специализированные статьи по PDF, XPS и TIFF объясняют, как управлять соответствием стандартам, скрытыми слайдами, заметками, качеством изображений, сжатием, форматом пикселей и размером вывода.

## **Экспорт в HTML и изображения**

Экспорт в HTML и HTML5 полезен для просмотра в браузере, веб‑публикаций и лёгкого обмена. Экспорт изображений нужен, когда каждый слайд должен стать отдельным превью, эскизом или растровым ресурсом. Используйте статьи о PNG, JPG и SVG для рекомендаций по рендерингу, специфичным для формата.

## **FAQ**

**Нужен ли мне Microsoft PowerPoint для преобразования презентаций?**

Нет. Aspose.Slides for PHP via Java — автономная библиотека и не требует Microsoft PowerPoint или автоматизации Office.

**Можно ли пакетно преобразовывать множество презентаций?**

Да. Загружайте каждую презентацию, сохраняйте её в необходимый формат и после обработки освобождайте объект презентации. Для параллельной обработки используйте отдельные экземпляры презентаций и следуйте рекомендациям по [multithreading](/slides/ru/php-java/multithreading/).

**Можно ли экспортировать только выбранные слайды?**

Да. Несколько методов экспорта позволяют передать индексы слайдов или рендерить отдельные слайды, в зависимости от формата вывода. См. специализированную статью для нужного формата.

**Можно ли включать скрытые слайды при экспорте в PDF или XPS?**

Да. Используйте параметры экспорта скрытых слайдов, описанные в статьях о [PDF](/slides/ru/php-java/convert-powerpoint-to-pdf/) и [XPS](/slides/ru/php-java/convert-powerpoint-to-xps/).

**Можно ли создавать вывод PDF/A?**

Да. Для экспорта PDF доступны параметры соответствия стандарту. См. детали в [Convert PowerPoint to PDF](/slides/ru/php-java/convert-powerpoint-to-pdf/).

**Как обрабатываются шрифты во время преобразования?**

Aspose.Slides может использовать внедрённые шрифты, резервные шрифты и настройки подстановки шрифтов. См. [Embedded Font](/slides/ru/php-java/embedded-font/), [Fallback Font](/slides/ru/php-java/fallback-font/) и [Font Substitution](/slides/ru/php-java/font-substitution/).
