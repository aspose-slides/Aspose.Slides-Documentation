## **Обзор**

Aspose.Slides for PHP via Java может загружать презентации PowerPoint и OpenDocument и сохранять их или отображать в многих других форматах без Microsoft PowerPoint, OpenOffice или LibreOffice. Вы можете конвертировать устаревшие файлы PPT в современные PPTX, экспортировать презентации в документы фиксированного макета, такие как PDF и XPS, публиковать слайды как HTML или рендерить слайды в виде файлов изображений для предварительных просмотров, миниатюр и архивов.

Большинство конвертаций документов используют один и тот же общий рабочий процесс: загрузить исходный файл, выбрать требуемый формат вывода и при необходимости применить параметры, специфичные для формата. Для форматов изображений каждый слайд рендерится отдельно, а затем сохраняется как растровое или векторное изображение. Специальные статьи, ссылки на которые приведены ниже, предоставляют детали реализации для каждого случая.

## **Конвертация PowerPoint и OpenDocument**

Aspose.Slides for PHP via Java поддерживает конвертацию из часто используемых форматов презентаций, таких как PPT, PPTX, PPS, PPSX, POT, POTX и ODP. Один и тот же API конвертации используется для файлов PowerPoint и OpenDocument, поэтому рабочий процесс, сохраняющий файл PPTX в PDF, обычно можно применить к файлу ODP, изменив лишь входной файл.

При конвертации файлов ODP помните, что приложения PowerPoint и OpenDocument не поддерживают каждый элемент макета и форматирования одинаково. Если файл ODP был создан в LibreOffice или OpenOffice Impress, проверьте результат и используйте параметры, описанные в [Конвертировать OpenDocument‑презентации](/slides/ru/php-java/convert-openoffice-odp/), когда требуется руководство по конкретному формату.

## **Конвертация PPT в PPTX**

PPT — это старый бинарный формат PowerPoint, тогда как PPTX — современный формат Office Open XML. Aspose.Slides for PHP via Java поддерживает высокоточное преобразование PPT в PPTX с сохранением сложных структур презентации, таких как шаблоны, макеты, слайды, диаграммы, группированные фигуры, заполнители, текстовые рамки, текстуры и заливки изображениями.

Подробности см. в [Конвертировать PPT в PPTX](/slides/ru/php-java/convert-ppt-to-pptx/) и [PPT против PPTX](/slides/ru/php-java/ppt-vs-pptx/).

## **Экспорт фиксированного макета**

PDF, XPS и TIFF полезны, когда вывод должен выглядеть одинаково на разных устройствах и не должен редактироваться как презентация. Специальные статьи по PDF, XPS и TIFF объясняют, как управлять соответствием, скрытыми слайдами, заметками, качеством изображения, сжатием, форматом пикселей и размером вывода.

## **Экспорт HTML и изображений**

Экспорт в HTML и HTML5 полезен для просмотра в браузере, публикации в вебе и легковесного обмена. Экспорт изображений полезен, когда каждый слайд должен стать отдельным предварительным просмотром, миниатюрой или растровым ресурсом. Используйте статьи по PNG, JPG и SVG для рекомендаций по рендерингу, специфичному для формата.

## **FAQ**

**Нужен ли мне Microsoft PowerPoint для конвертации презентаций?**

Нет. Aspose.Slides for PHP via Java — это автономная библиотека и не требует Microsoft PowerPoint или автоматизации Office.

**Могу ли я пакетно конвертировать множество презентаций?**

Да. Загружайте каждую презентацию, сохраняйте её в требуемый формат и освобождайте объект презентации после обработки. Для параллельной обработки используйте отдельные экземпляры презентаций и следуйте рекомендациям по [многопоточности](/slides/ru/php-java/multithreading/).

**Могу ли я экспортировать только выбранные слайды?**

Да. Несколько методов экспорта позволяют передать индексы слайдов или отрисовать отдельные слайды, в зависимости от формата вывода. См. специальную статью для целевого формата.

**Могу ли я включать скрытые слайды при экспорте в PDF или XPS?**

Да. Используйте параметры экспорта скрытых слайдов, описанные в статьях по конвертации в [PDF](/slides/ru/php-java/convert-powerpoint-to-pdf/) и [XPS](/slides/ru/php-java/convert-powerpoint-to-xps/).

**Могу ли я создать вывод PDF/A?**

Да. Параметры соответствия PDF доступны для экспорта в PDF. Подробнее см. в статье [Конвертировать PowerPoint в PDF](/slides/ru/php-java/convert-powerpoint-to-pdf/).

**Как обрабатываются шрифты во время конвертации?**

Aspose.Slides может использовать встроенные шрифты, резервные шрифты и параметры замены шрифтов. Смотрите статьи [Встроенный шрифт](/slides/ru/php-java/embedded-font/), [Резервный шрифт](/slides/ru/php-java/fallback-font/), и [Замена шрифтов](/slides/ru/php-java/font-substitution/).
