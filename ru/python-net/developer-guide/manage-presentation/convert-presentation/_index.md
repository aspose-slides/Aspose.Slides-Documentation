## **Обзор**

Aspose.Slides for Python via .NET может загружать презентации PowerPoint и OpenDocument и сохранять или визуализировать их в множество других форматов без Microsoft PowerPoint, OpenOffice или LibreOffice. Вы можете преобразовать устаревшие файлы PPT в современные PPTX, экспортировать презентации в фиксированные документы, такие как PDF и XPS, публиковать слайды в виде HTML или визуализировать их как файлы изображений для предварительного просмотра, миниатюр и архивов.

Большинство преобразований документов используют один и тот же общий рабочий процесс: загрузить исходный файл, выбрать требуемый формат вывода и при необходимости применить параметры, специфичные для формата. Для форматов изображений каждый слайд визуализируется отдельно, а затем сохраняется как растровое или векторное изображение. Ниже приведены статьи, подробно описывающие реализацию для каждого случая.

## **Конвертация PowerPoint и OpenDocument**

Aspose.Slides for Python via .NET поддерживает конвертацию из часто используемых форматов презентаций, таких как PPT, PPTX, PPS, PPSX, POT, POTX и ODP. Один и тот же API используется как для файлов PowerPoint, так и для OpenDocument, поэтому рабочий процесс, сохраняющий PPTX в PDF, обычно можно применить к ODP, изменив только входной файл.

При конвертации ODP‑файлов помните, что приложения PowerPoint и OpenDocument не поддерживают каждый макет и каждый параметр форматирования одинаково. Если ODP был создан в LibreOffice или OpenOffice Impress, проверьте результат и используйте параметры, описанные в [Convert OpenDocument Presentations](/slides/ru/python-net/convert-openoffice-odp/), когда требуется руководство по формату.

## **Конвертация PPT в PPTX**

PPT — старый бинарный формат PowerPoint, а PPTX — современный формат Office Open XML. Aspose.Slides for Python via .NET поддерживает высокоточное преобразование PPT в PPTX с сохранением сложных структур презентации, таких как мастеры, макеты, слайды, диаграммы, сгруппированные фигуры, заполнители, текстовые кадры, текстуры и заливка изображениями.

Подробности см. в [Convert PPT to PPTX](/slides/ru/python-net/convert-ppt-to-pptx/) и [PPT vs PPTX](/slides/ru/python-net/ppt-vs-pptx/).

## **Экспорт фиксированного макета**

PDF, XPS и TIFF полезны, когда вывод должен выглядеть одинаково на всех устройствах и не должен редактироваться как презентация. Специализированные статьи по PDF, XPS и TIFF объясняют, как управлять соответствием стандартам, скрытыми слайдами, заметками, качеством изображений, сжатием, форматом пикселей и размером вывода.

## **Экспорт в HTML и изображения**

Экспорт в HTML и HTML5 удобен для просмотра в браузере, веб‑публикаций и лёгкого обмена. Экспорт изображений полезен, когда каждый слайд должен стать отдельным превью, миниатюрой или растровым ресурсом. Используйте статьи по PNG, JPG и SVG для рекомендаций по рендерингу, специфичному для формата.

## **FAQ**

**Нужен ли мне Microsoft PowerPoint для конвертации презентаций?**

Нет. Aspose.Slides for Python via .NET — это автономная библиотека, не требующая Microsoft PowerPoint или автоматизации Office.

**Можно ли пакетно конвертировать множество презентаций?**

Да. Загружайте каждую презентацию, сохраняйте её в требуемый формат и освобождайте объект презентации после обработки. Для параллельной обработки используйте отдельные экземпляры презентаций и следуйте рекомендациям по [multithreading](/slides/ru/python-net/multithreading/).

**Можно ли экспортировать только выбранные слайды?**

Да. Некоторые методы экспорта позволяют передать индексы слайдов или визуализировать отдельные слайды, в зависимости от формата вывода. См. специализированную статью для нужного формата.

**Можно ли включать скрытые слайды при экспорте в PDF или XPS?**

Да. Используйте параметры экспорта скрытых слайдов, описанные в статьях по [PDF](/slides/ru/python-net/convert-powerpoint-to-pdf/) и [XPS](/slides/ru/python-net/convert-powerpoint-to-xps/).

**Можно ли создать вывод PDF/A?**

Да. Для экспорта PDF доступны настройки соответствия стандарту PDF. См. [Convert PowerPoint to PDF](/slides/ru/python-net/convert-powerpoint-to-pdf/) для деталей.

**Как обрабатываются шрифты при конвертации?**

Aspose.Slides может использовать встроенные шрифты, резервные шрифты и параметры замены шрифтов. См. [Embedded Font](/slides/ru/python-net/embedded-font/), [Fallback Font](/slides/ru/python-net/fallback-font/) и [Font Substitution](/slides/ru/python-net/font-substitution/).
