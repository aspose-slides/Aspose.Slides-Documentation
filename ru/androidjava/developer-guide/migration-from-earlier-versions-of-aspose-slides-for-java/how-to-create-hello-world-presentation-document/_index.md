## **Поддержка устаревшего кода**
Чтобы использовать устаревший код, разработанный с использованием Aspose.Slides для Java версий ранее 13.x, вам нужно внести некоторые незначительные изменения в ваш код, и он будет работать как раньше. Все классы, которые были представлены в старом Aspose.Slides для Java в пространствах имен Aspose.Slide и Aspose.Slides.Pptx, теперь объединены в одно пространство имен Aspose.Slides. Пожалуйста, посмотрите на следующий простой фрагмент кода для создания документа презентации Hello World в устаревшем API Aspose.Slides и следуйте шагам, описывающим, как мигрировать на новый объединенный API.
