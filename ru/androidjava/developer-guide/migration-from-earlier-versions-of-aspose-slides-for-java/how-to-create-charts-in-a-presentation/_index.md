## **Поддержка устаревшего кода**
Чтобы использовать устаревший код, разработанный с Aspose.Slides для Java версий ранее 14.x.x, вам нужно внести некоторые небольшие изменения в ваш код, и он будет работать как раньше. Все классы, которые были присутствуют в старом Aspose.Slides для Java под пространствами имен com.aspose.slides и com.aspose.slides.pptx, теперь объединены в одно пространство имен com.aspose.slides.
