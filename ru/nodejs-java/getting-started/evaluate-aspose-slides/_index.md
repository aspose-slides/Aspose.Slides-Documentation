## **Оценка Aspose.Slides**

Вы можете легко скачать Aspose.Slides для оценки. Пакет оценки идентичен купленному пакету. Версия оценки просто становится лицензированной после того, как вы добавите несколько строк кода для применения лицензии.

Версия оценки Aspose.Slides (без указания лицензии) предоставляет полный функционал продукта, но вставляет водяной знак «evaluation» в верхнюю часть документа при открытии и сохранении. При извлечении текста из слайдов презентации вы также ограничены одним слайдом.

## **FAQ**

**Могу ли я параллельно тестировать несколько презентаций в разных потоках в режиме оценки?**

Да. Вы можете обрабатывать разные документы параллельно; не следует использовать один и тот же объект презентации [в разных потоках](/slides/ru/net/multithreading/). Режим оценки не влияет на это.

**Нужно ли устанавливать Microsoft PowerPoint для оценки библиотеки на сервере или в CI?**

Нет. Aspose.Slides — это автономный движок и не требует установки PowerPoint ни для оценки, ни для продакшн‑использования.

**Могу ли я полностью протестировать конвертацию PPT/PPTX в PDF и изображения в режиме оценки?**

Да. [Конвертеры](/slides/ru/net/convert-presentation/) работают; в выводе будет присутствовать водяной знак.

**Можно ли использовать временную лицензию для нагрузочного тестирования без водяного знака?**

Да. 30‑дневная временная лицензия устраняет ограничения режима оценки и позволяет тестировать без водяного знака.
