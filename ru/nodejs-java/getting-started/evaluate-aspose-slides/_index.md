## **Aspose.Slides Evaluation**

Вы можете легко скачать Aspose.Slides для оценки. Оценочный пакет идентичен приобретённому пакету. Оценочная версия просто становится лицензированной после того, как вы добавите несколько строк кода для применения лицензии.

Оценочная версия Aspose.Slides (без указания лицензии) предоставляет полный набор функций продукта, но вставляет водяной знак оценки в верхнюю часть документа при открытии и сохранении. Кроме того, при извлечении текста из слайдов презентации вы ограничены одним слайдом.

## **Часто задаваемые вопросы**

**Могу ли я тестировать несколько презентаций одновременно в разных потоках в режиме оценки?**

Да. Вы можете обрабатывать разные документы параллельно; не следует использовать один и тот же объект презентации [между потоками](/slides/ru/net/multithreading/). Режим оценки не влияет на это.

**Нужна ли установка Microsoft PowerPoint для оценки библиотеки на сервере или в CI?**

Нет. Aspose.Slides — автономный движок и не требует установленного PowerPoint ни для оценки, ни для боевого использования.

**Могу ли я полностью протестировать конвертацию PPT/PPTX в PDF и изображения в режиме оценки?**

Да. [Конвертеры](/slides/ru/net/convert-presentation/) работают; результат будет содержать водяной знак.

**Могу ли я использовать временную лицензию для нагрузочного тестирования без водяного знака?**

Да. 30‑дневная временная лицензия снимает ограничения режима оценки и позволяет проводить тестирование без водяного знака.
