## **Поддерживаемые платформы**
Aspose.Slides for Node.js via Java — это независимый от платформы API. Его можно использовать на любой платформе (Windows, Linux, macOS и т.д.), где установлен Node.js 7 или выше. Требуется также Oracle JDK 8 или выше.

## **Форматы файлов и конверсия**
Aspose.Slides for Node.js via Java поддерживает большинство форматов документов PowerPoint. Он также позволяет экспортировать их в популярные форматы файлов, которые организации используют и обмениваются между собой. Подробнее:

|**Функция**|**Описание**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via Java обеспечивает самую быструю обработку этого формата презентаций.|
|Конверсия PPT в PPTX|Aspose.Slides for Node.js via Java поддерживает конверсию PPT в PPTX.|
|Portable Document Format (PDF)|Вы можете экспортировать все поддерживаемые форматы файлов в документы Adobe Portable Document Format (PDF) одним методом.|
|XML Parser Specification (XPS)|Вы можете экспортировать все поддерживаемые форматы файлов в документы XML Parser Specification (XPS) одним методом.|
|Tagged Image File Format (TIFF)|Вы можете экспортировать все поддерживаемые форматы презентаций в Tagged Image File Format (TIFF).|
|Конверсия PPTX в HTML|Aspose.Slides for Node.js via Java поддерживает конверсию PresentationEx в формат HTML.|

## **FAQ**

**Нужен ли мне устанавливать Microsoft PowerPoint на сервер/ПК, чтобы библиотека работала?**

Нет. PowerPoint не требуется; Aspose.Slides — это автономный движок для создания, редактирования, конвертации и визуализации презентаций.

**Как работает многопоточность? Можно ли параллелизовать обработку?**

Безопасно обрабатывать разные документы в разных потоках; один и тот же [презентация](https://reference.aspose.com/slides/nodejs-java/aspose.slides/presentation/) объект не должен использоваться одновременно [несколькими потоками](/slides/ru/nodejs-java/multithreading/).

**Поддерживаются ли пароли файлов и шифрование?**

Да. [Вы можете](/slides/ru/nodejs-java/password-protected-presentation/) открыть зашифрованные презентации, установить или удалить пароль для открытия и записи, а также проверить статус защиты.

**Нужно ли заботиться о пакетах шрифтов в Linux‑контейнерах?**

Да. Рекомендуется установить общие пакеты шрифтов и/или явно [указать каталоги шрифтов](/slides/ru/nodejs-java/custom-font/) в вашем приложении, чтобы избежать непредвиденных замен.

**Есть ли ограничения в оценочной версии?**

В [режиме оценки](/slides/ru/nodejs-java/licensing/) к результату добавляется водяной знак и действуют определённые ограничения; доступна [30‑дневная временная лицензия](https://purchase.aspose.com/temporary-license/) для полного тестирования функций.

**Поддерживается ли импорт внешних форматов в презентацию (PDF/HTML → PPTX)?**

Да. Вы можете добавить [страницы PDF и содержимое HTML](/slides/ru/nodejs-java/import-presentation/) в презентацию, превратив их в слайды.
