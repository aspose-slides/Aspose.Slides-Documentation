## **Техническая поддержка**
{{% alert color="primary" %}}
Aspose предоставляет неограниченную бесплатную техническую поддержку для всех своих продуктов. Поддержка доступна всем пользователям (включая пользователей с оценочными пакетами).
{{% /alert %}} 

Когда вам нужна помощь с Aspose.Slides for Node.js via Java, учитывайте следующее:

- Основным каналом поддержки является [Aspose.Forums](https://forum.aspose.com/c/slides/11). Опубликуйте свой вопрос в форуме Aspose.Slides — и он будет отвечен в течение нескольких часов. Команда поддержки Aspose.Slides напрямую отвечает на вопросы, размещённые на форуме.
- Обратите внимание, что Aspose не предоставляет техническую поддержку по телефону. Телефонная поддержка доступна только для вопросов по продажам и покупке.
- Ожидая ответа на форуме, пожалуйста, будьте терпеливы и учитывайте разницу во времени.


Если вы столкнулись с проблемой в Aspose.Slides for Node.js via Java, следуйте этим рекомендациям, чтобы решить её максимально эффективно:

- Проверьте и убедитесь, что вы используете последнюю версию Aspose.Slides for Node.js via Java перед тем, как сообщать о проблеме. Возможно, возникшая у вас проблема уже решена в обновлении.  

  Смотрите [Aspose.Slides download](https://releases.aspose.com/slides/nodejs-java/) чтобы получить информацию о последней версии.

- Прежде чем сообщать о проблеме, вы можете просмотреть наши форумы, эту документацию и справочник API. Возможно, ответ на ваш вопрос уже был дан. 

- Сообщая о проблеме, пожалуйста, приложите оригинальный документ и (по возможности) фрагмент вашего кода, относящийся к проблеме. Если нужно приложить несколько файлов, упакуйте их в архив ZIP. Вы можете безопасно прикреплять свои документы в Aspose.Forums, так как **только** вы и разработчики Aspose имеют доступ к вложенным файлам.

- Создавайте и используйте одну ветку обсуждения для конкретной проблемы. Если возникает другая проблема, необходимо создать отдельную ветку для её сообщения.

## **FAQ**

**Какие уровни платной поддержки доступны и чем они отличаются?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Они различаются по количеству включённых инцидентов, приоритету ответов и доступным каналам коммуникации.

**Могу ли я напрямую эскалировать баг в инженерную команду?**

Да. [Платная поддержка](https://helpdesk.aspose.com/) позволяет эскалировать проблему разработчикам, когда требуется более глубокое исследование.

**Поддерживаются ли старые версии библиотеки и выпускаются ли исправления для них?**

Вы можете использовать старые версии, но исправления предоставляются только для текущего релиза; патчи для устаревших версий не выпускаются.
