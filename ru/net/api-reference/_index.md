{{% alert color="primary" %}} 

Aspose.Slides for .NET — это библиотека классов, позволяющая .NET‑приложениям читать и записывать документы презентаций без использования Microsoft PowerPoint® и других приложений. Aspose.Slides for .NET позволяет управлять текстом, фигурами, таблицами и анимациями, добавлять аудио и видео на слайды, просматривать слайды, экспортировать их в форматы SVG, PDF и многое другое.

Последнюю справочную информацию по API можно найти в [Aspose.Slides for .NET API Reference.](https://reference.aspose.com/slides/net)
