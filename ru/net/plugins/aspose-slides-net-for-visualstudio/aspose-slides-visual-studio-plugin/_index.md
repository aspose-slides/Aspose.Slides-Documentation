## **Введение**
Microsoft Visual Studio — интегрированная среда разработки (IDE) от Microsoft. Она используется для разработки программ для Microsoft Windows, а также веб‑сайтов, веб‑приложений и веб‑служб. Visual Studio использует платформы разработки Microsoft, такие как Windows API, Windows Forms, Windows Presentation Foundation, Windows Store и Microsoft Silverlight. Она может создавать как нативный код, так и управляемый код.

### **Aspose.Slides for .NET**
Aspose.Slides for .NET — уникальный API управления PowerPoint®, который позволяет .NET‑приложениям читать, записывать и изменять документы PowerPoint без использования Microsoft PowerPoint.

Aspose.Slides for .NET — первый и единственный .NET API для PowerPoint, предоставляющий возможности управления документами PowerPoint внутри ваших собственных приложений. Как и все компоненты Aspose .NET, Aspose.Slides написан на управляемом C#, имеет невероятно выгодную цену и работает со скоростью молнии.

Для получения дополнительной информации о [Aspose.Slides for .NET API](http://www.aspose.com/.net/powerpoint-component.aspx)

### **Исследуйте примеры Aspose.Slides for .NET с помощью Visual Studio**
Aspose Visual Studio Plugin — **Explore Aspose.Slides for .NET Examples** — отличный инструмент для быстрого скачивания и изучения примеров API Aspose.Slides for .NET. Он экономит много времени и усилий, предоставляя очень простой способ без проблем выбирать, скачивать и открывать новейшие примеры проектов, не покидая Visual Studio.

Эта версия предоставляет следующие возможности:
- Поддерживает Visual Studio 2010, Visual Studio 2012 и Visual Studio 2013.
- Легко запускается из меню **File > Aspose** в Visual Studio
- Экономит время и уменьшает кривую обучения.
- Позволяет выбирать и открывать новейшие примеры API Aspose.Slides for .NET, размещённые на Github.com.
- Позволяет открыть примеры проектов **C#** или **VB.NET**
- Выбранный проект автоматически открывается в Visual Studio со всеми необходимыми ссылками, предоставляя готовую к запуску и исследованию среду.
- Автоматически загружает новейшие библиотеки API из **NuGet** и примеры из **GitHub**.

## **Установка**
Установка Aspose Visual Studio Plugin очень простая, пожалуйста, выполните следующие шаги
- Дважды щёлкните загруженный файл msi
- Следуйте инструкциям, просто нажимайте Next на каждом экране, чтобы установить плагин с использованием стандартных настроек.

После завершения установки появится окно с сообщением об успешной установке.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_2)

**Примечание:** Пожалуйста, перезапустите Visual Studio, чтобы изменения вступили в силу.

## **Использование**
После установки вы можете легко запустить и использовать интерактивный плагин, как описано ниже:
**1.** Откройте плагин, выбрав **File > Aspose**, а затем **New Aspose.Slides Example Project** в Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_3)

**2.** При первом открытии плагина вы увидите диалог выбора API, как показано ниже. Загрузка новейшего API и его библиотек из **NuGet**, а также исходного кода примеров из **GitHub**, займёт несколько минут.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_4)

**3.** В диалоговом окне «Select an API to View Examples» выберите или измените расположение, где будут созданы примерные проекты, а также выберите тип проекта примеров — **C#** или **VB.NET**, и нажмите **Continue**, чтобы открыть примерный проект в Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_1)

**4.** Новые проекты‑примеры создаются в соответствии с выбранным типом проекта **C#** или **VB.NET** в Visual Studio.

Последний исходный код можно получить по одному из следующих адресов
- [Github ](https://github.com/aspose-slides/Aspose.Slides-for-.NET)
#### **Как настроить исходный код**
Чтобы открыть и расширить исходный код, необходимо установить следующее
- Visual Studio 2010

Пожалуйста, выполните следующие простые шаги, чтобы начать
1. Скачайте/клонируйте исходный код.
2. Откройте Visual Studio 2010 и выберите **File** > **Open Project**
3. Найдите последнюю версию скачанного кода и откройте **AsposeVisualStudioPlugin.sln**
#### **Обзор исходного кода**
В решении присутствуют два проекта
- **AsposeVisualStudioPlugin** — Содержит интеграцию с Visual Studio, выбор и автоматическую загрузку API Aspose и примеров.
- **Installer** — Проект установщика Wix для создания msi‑файла плагина.
