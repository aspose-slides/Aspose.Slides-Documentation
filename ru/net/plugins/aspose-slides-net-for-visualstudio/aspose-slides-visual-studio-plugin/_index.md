## **Introduction**
Microsoft Visual Studio — это интегрированная среда разработки (IDE) от компании Microsoft. Она используется для разработки программного обеспечения для Microsoft Windows, а также веб‑сайтов, веб‑приложений и веб‑служб. Visual Studio использует платформы разработки Microsoft, такие как Windows API, Windows Forms, Windows Presentation Foundation, Windows Store и Microsoft Silverlight. Она может создавать как нативный код, так и управляемый код.

### **Aspose.Slides for .NET**
Aspose.Slides for .NET — это уникальный API управления PowerPoint®, который позволяет .NET‑приложениям читать, записывать и изменять документы PowerPoint без использования Microsoft PowerPoint.

Aspose.Slides for .NET — первый и единственный .NET API для PowerPoint, предоставляющий возможности управления документами PowerPoint внутри ваших собственных приложений. Как и все компоненты Aspose .NET, Aspose.Slides написан на управляемом C#, имеет невероятно доступную цену и молниеносную скорость.

Для получения дополнительных сведений о [Aspose.Slides for .NET API](http://www.aspose.com/.net/powerpoint-component.aspx)

### **Explore Aspose.Slides for .NET Examples Using Visual Studio**
Aspose Visual Studio Plugin — **Explore Aspose.Slides for .NET Examples** — это отличный инструмент для быстрого скачивания и изучения примеров API Aspose.Slides for .NET. Он экономит массу времени и усилий, предоставляя простую возможность беспрепятственно выбирать, загружать и открывать новейшие проекты‑примеры, не покидая Visual Studio.

Эта версия предоставляет следующие возможности:

- Поддерживает Visual Studio 2010, Visual Studio 2012 и Visual Studio 2013.
- Легко запускается из меню **File > Aspose** в Visual Studio
- Экономит время и сокращает кривую обучения.
- Позволяет выбирать и открывать новейшие примеры API Aspose.Slides for .NET, размещённые на Github.com.
- Позволяет открыть проекты‑примеры **C#** или **VB.NET**
- Выбранный проект автоматически открывается в Visual Studio со всеми необходимыми ссылками, предоставляя готовую к запуску и изучению среду.
- Автоматически загружает новейшие библиотеки API из **NuGet** и примеры из **GitHub**.

Установить Aspose Visual Studio Plugin очень просто, выполните следующие шаги

- Дважды щёлкните загруженный файл msi
- Следуйте инструкциям, просто нажимайте Next на каждом экране, чтобы установить плагин со стандартными настройками.

После завершения установки появится диалоговое окно с подтверждением успеха.

## **Using**
После установки вы можете легко запустить и использовать интерактивный плагин, как описано ниже:

**1.** Откройте плагин, выбрав **File > Aspose**, а затем **New Aspose.Slides Example Project** в Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_3)

**2.** При первом открытии плагина вы увидите диалог выбора API, как показано ниже. Загрузка новейшего API и их библиотек из **NuGet**, а также исходного кода последних примеров из **GitHub**, займет несколько минут.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_4)

**3.** В диалоговом окне «Select an API to View Examples» выберите или измените место, где будут создаваться проекты‑примеры, а также выберите тип проекта‑примера **C#** или **VB.NET** и нажмите **Continue**, чтобы открыть проект‑пример в Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_1)

**4.** Новые проекты‑примеры создаются в соответствии с выбранным типом проекта **C#** или **VB.NET** в Visual Studio.

#### **Source code overview**
В решении (solution) находятся два проекта

- **AsposeVisualStudioPlugin** — содержит интеграцию с Visual Studio, выбор и автоматическую загрузку API Aspose и примеров.
- **Installer** — проект установщика Wix для создания msi‑файла плагина.
