**Добро пожаловать в Aspose.Slides for .NET**

![Логотип продукта Aspose.Slides for .NET](home_1.png)

Aspose.Slides for .NET — это API управления Microsoft PowerPoint®, который позволяет .NET приложениям читать и записывать документы PowerPoint® без использования Microsoft PowerPoint®.

Aspose.Slides for .NET — первый и единственный компонент, предоставляющий функции, необходимые для управления документами PowerPoint®.

Aspose.Slides for .NET предлагает множество ключевых возможностей, таких как управление текстом, фигурами, таблицами и анимацией, добавление аудио и видео к слайдам, просмотр слайдов, экспорт слайдов в SVG, PDF и различные форматы, а также многие другие возможности.
