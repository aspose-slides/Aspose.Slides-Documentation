## **Обзор**
Aspose.Slides для .NET не требует установки Microsoft PowerPoint, так как Aspose.Slides является независимым движком для создания, конвертации, верстки и рендеринга документов Microsoft PowerPoint.
## **Поддерживаемые операционные системы**
Aspose.Slides для .NET поддерживает любую 32-битную или 64-битную операционную систему, на которой установлена среда .NET или Mono, включая (но не ограничиваясь):
### **Windows**
- Microsoft Windows 2000 Server (x64, x86)
- Microsoft Windows 2003 Server (x64, x86)
- Microsoft Windows 2022 Server
- Microsoft Windows Vista (x64, x86)
- Microsoft Windows XP (x64, x86)
- Microsoft Windows 7 (x64, x86)
- Microsoft Windows 8, 8.1 (x64, x86)
- Microsoft Windows 10 (x64, x86)
- Microsoft Azure

### **Linux**
- Linux (Ubuntu, OpenSUSE, CentOS и другие)

{{%  alert  title="Заметки"  color="primary"  %}} 

1. В версии GLIBC CentOS 7 — 2.14, но Aspose.Slides для .NET 6 и .NET 7 требует Linux x86_x64 с GLIBC 2.23 и выше. Это требование также применимо к Aspose.Slides кроссплатформенной версии, которая может быть использована только для .NET 6, .NET 7 или выше.

2. Если ваша система не соответствует требованию в пункте 1, используйте Aspose Slides для .NETStandard.

{{% /alert %}} 

### **Mac**
- Mac OS X

- Microsoft Visual Studio 2008
- Microsoft Visual Studio 2010
- Microsoft Visual Studio 2012
