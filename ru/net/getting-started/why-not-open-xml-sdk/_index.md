## **Что такое Open XML SDK?**
Иногда нам задают вопрос: *Почему следует использовать продукты Aspose, а не бесплатный Open XML SDK?* 

Нам легко ответить на этот вопрос, описав функции и возможности. 

Согласно [MSDN Library](https://docs.microsoft.com/en-us/office/open-xml/open-xml-sdk), Open XML SDK определяется так: 

> "Open XML SDK 2.0 упрощает задачу манипулирования пакетами Open XML и подлежащими им элементами схемы Open XML внутри пакета. Open XML SDK 2.0 инкапсулирует многие распространённые задачи, которые разработчики выполняют с пакетами Open XML, так что вы можете выполнять сложные операции всего лишь несколькими строками кода. Документы OOXML по сути являются сжатыми XML‑файлами, а Open XML SDK — это набор классов, позволяющих работать с содержимым документов OOXML типобезопасным способом. Вместо того чтобы распаковывать файл для извлечения XML, загружать этот XML в дерево DOM и работать напрямую с элементами и атрибутами XML, Open XML SDK предоставляет классы для выполнения этих действий."

## **Что такое Aspose.Slides?**
Aspose.Slides — это библиотека классов, позволяющая приложениям выполнять следующие задачи обработки презентаций: 

- Программирование с использованием объектной модели презентации.  

- Высококачественное преобразование во все популярные поддерживаемые форматы PowerPoint, включая конвертацию в PDF, XPS, TIFF и печать.  

- Генерация миниатюр слайдов в известных форматах, таких как PNG, JPEG и BMP, а также экспорт слайдов в SVG.  

- Создание презентаций с нуля или путём комбинирования элементов из одного или нескольких документов.  

- Добавление анимаций, OLE‑фреймов, таблиц, создание и управление диаграммами.  

- Управление (расширенное управление) и настройка форматирования текста на уровнях TextFrames, Paragraphs и Portions.  

  Подробнее о доступных функциях см. страницу [Aspose.Slides Features](/slides/ru/net/product-overview/).

## **Заключение**
Open XML SDK и Aspose.Slides не являются прямыми конкурентами, поскольку они решают существенно разные задачи и ориентированы на разные аудитории. 

{{% alert color="primary" %}} 

Open XML SDK — это библиотека классов, предоставляющая типобезопасный способ работы с OOXML‑документами, тогда как Aspose.Slides — чрезвычайно полезная библиотека обработки презентаций, обеспечивающая отличную поддержку практически всех форматов файлов Microsoft PowerPoint. 

{{% /alert %}} 

Если ваш рабочий процесс представляет собой базовую программную операцию с документом PPTX, то Open XML SDK может стать хорошим выбором. С Open XML SDK вы сможете легко выполнять простые задачи, такие как создание простого PPTX‑документа, удаление комментариев, колонтитулов, извлечение изображений и т.п. Некоторые задачи можно выполнить с помощью Open XML SDK, но не с Aspose.Slides. Например, если необходимо напрямую работать с элементами и атрибутами XML OOXML‑документа, следует использовать Open XML SDK. 

Если вам требуется выполнять сложные задачи с документами — такие как задачи из списка ниже — тогда Aspose.Slides будет лучшим вариантом. 

- Операции с более старыми форматами PowerPoint (и с PPTX).  
- Копирование или клонирование фигур внутри слайдов с одновременным объединением объектов, стилей и других элементов форматирования надлежащим образом.  
- Замена отформатированного или неотформатированного текста.  
- Применение анимаций и использование соединителей с фигурами.  
- Конвертация документа в PDF, TIFF или XPS с результатом, идентичным конвертации в Microsoft PowerPoint.  
- Разработка .NET или Java‑приложений как для настольных, так и для веб‑окружений.
