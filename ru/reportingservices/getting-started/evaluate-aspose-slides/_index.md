{{% alert color="primary" %}} 

Вы можете легко загрузить Aspose.Slides для оценки. Загрузка для оценки такая же, как и для покупки. Оценочная версия просто становится лицензированной, когда вы добавляете несколько строк кода для применения лицензии.

Оценочная версия Aspose.Slides (без указанной лицензии) предоставляет полную функциональность продукта, но вставляет оценочный водяной знак на каждый слайд созданной презентации.
