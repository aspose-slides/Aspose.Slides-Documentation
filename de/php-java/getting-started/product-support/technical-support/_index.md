## **Technischer Support**

{{% alert color="primary" %}} 

Aspose bietet unbegrenzten kostenlosen technischen Support für alle seine Produkte. Der Support steht allen Benutzern zur Verfügung, einschließlich der Evaluation.

{{% /alert %}} 

Wenn Sie Hilfe zu Aspose.Slides für PHP via Java benötigen, beachten Sie Folgendes:

- Der Hauptweg zum Support ist das Aspose.Forums. Stellen Sie Ihre Frage im Aspose.Slides-Forum und Sie erhalten innerhalb weniger Stunden eine Antwort. Die Fragen werden direkt vom Aspose.Slides-Supportteam beantwortet.
- Bitte beachten Sie, dass Aspose keinen telefonischen technischen Support anbietet. Telefonischer Support steht nur für Verkaufs‑ und Kauf‑Fragen zur Verfügung.
- Wenn Sie eine Antwort im Forum erwarten, berücksichtigen Sie bitte Zeitzonenunterschiede.

Wenn Sie ein Problem mit Aspose.Slides für PHP via Java haben, befolgen Sie diese einfachen Schritte, um sicherzustellen, dass es auf effizienteste Weise gelöst wird:

- Stellen Sie sicher, dass Sie die neueste Version von Aspose.Slides für PHP via Java verwenden, bevor Sie das Problem melden; siehe Aspose.Slides-Download, um die neueste Version zu erfahren.
- Durchsuchen Sie die Foren, diese Dokumentation und die API‑Referenz, bevor Sie das Problem melden; vielleicht wurde Ihre Frage bereits beantwortet.
- Wenn Sie ein Problem melden, fügen Sie bitte das Originaldokument und ggf. einen Code‑Ausschnitt, der das Problem verursacht, bei. Wenn Sie mehrere Dateien anhängen müssen, komprimieren Sie diese zu einer ZIP‑Datei. Es ist sicher, Ihre Dokumente im Aspose.Forums anzuhängen, da nur Sie und die Aspose‑Entwickler Zugriff auf die angehängten Dateien haben.
- Bitte versuchen Sie, pro Thread nur ein Problem zu melden. Wenn Sie ein weiteres Problem haben, melden Sie es in einem separaten Thread.

## **FAQ**

**Welche kostenpflichtigen Support‑Stufen sind verfügbar und wie unterscheiden sie sich?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Sie unterscheiden sich durch die Anzahl der enthaltenen Vorfälle, die Priorität der Antwort und die verfügbaren Kommunikationskanäle.

**Kann ich einen Fehler direkt an das Engineering‑Team eskalieren?**

Ja. [Paid support](https://helpdesk.aspose.com/) ermöglicht die Eskalation an die Entwickler, wenn ein Problem einer tieferen Untersuchung bedarf.

**Werden ältere Bibliotheksversionen unterstützt und werden dafür Updates veröffentlicht?**

Sie können ältere Versionen verwenden, aber Updates werden nur für die aktuelle Version bereitgestellt; Patches für ältere Versionen werden nicht veröffentlicht.
