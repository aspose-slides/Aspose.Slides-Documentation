## **Produktbeschreibung**
Aspose.Slides for PHP via Java ist eine plattformübergreifende Klassenbibliothek, die es Ihren Anwendungen ermöglicht, eine Vielzahl von Dokumentenverarbeitungsaufgaben durchzuführen, ohne Microsoft PowerPoint® zu verwenden.

### **Warum Sie Aspose.Slides for PHP via Java verwenden sollten**

- Aspose.Slides for PHP via Java ist die erste und einzige Komponente, die die zur Verwaltung von PowerPoint®-Dokumenten erforderlichen Funktionalitäten bereitstellt. 
- Aspose.Slides for PHP via Java bietet zahlreiche wichtige Funktionen, wie das Verwalten von Texten, Formen, Tabellen & Animationen, das Hinzufügen von Audio und Video zu Folien, die Vorschau von Folien, das Exportieren von Folien nach SVG, PDF und anderen Formaten sowie viele weitere Möglichkeiten. 
- Aspose.Slides for PHP via Java ermöglicht nicht nur das Öffnen von PowerPoint-Dateien aus verschiedenen Quellen, sondern erlaubt Ihnen auch, Ihre Präsentationen auf verschiedene Weise zu speichern. Beispielsweise können Sie Ihre Präsentationen im PPT-Format speichern; Sie können Ihre Folien auch als Bilder speichern. 
- Aspose.Slides for PHP via Java bietet Ihnen fast alle Funktionen, die Sie in Microsoft PowerPoint finden (oder nicht finden) könnten. Aufgrund der Vielseitigkeit von Aspose.Slides for PHP via Java können Sie neben Tabellen und verschiedenen Formen auch unterschiedliche Rahmen—Text-, Audio- und Video‑Frames—zu Ihren Folien hinzufügen. 

## **Aspose.Slides‑Funktionen**

Die Funktionen von Aspose.Slides for PHP via Java können in die folgenden Gruppen unterteilt werden:

- Präsentationsfunktionen.
- Folienfunktionen.
- Grafik‑ und Multimedia‑Funktionen.
- Tabellenfunktionen.
- Text‑ und Absatz‑Funktionen
- Erweiterte Funktionen

### **Präsentationsfunktionen**

- Öffnen von Microsoft PowerPoint‑Präsentationen aus verschiedenen Quellen.  
- Speichern von Präsentationen in verschiedenen Microsoft PowerPoint‑Formaten.  
- Erstellen von Präsentationen von Grund auf.  
- Konvertieren von Präsentationen in PDF‑Dokumente ohne Verwendung einer anderen Komponente.  
- Lesen oder Ändern der Dokumenteigenschaften einer Präsentation.  
- Schutz für die Präsentation anwenden.  
- Exportieren der Präsentation nach PDF.  
- Exportieren der Präsentation nach TIFF.  
- Exportieren der Präsentation nach XPS.  
- Exportieren der Präsentation nach SWF.  
- Exportieren von Notizen der Präsentation nach PDF / TIFF.  
- Exportieren der Präsentation nach HTML.  

### **Folienfunktionen**

- Erstellen neuer Folien von Grund auf und Hinzufügen zu Präsentationen.  
- Zugriff auf jede Folie über ihre Position in einer Präsentation.  
- Kopieren oder Klonen von Folien in dieselbe oder eine andere Präsentation.  
- Bearbeiten und Löschen von Folien.  
- Automatisches Kopieren von Master‑Folien mit Folien.  
- Vorschau von Folien.  
- Abrufen oder Festlegen des Namens einer Folie.  
- Speichern von Folien als Bilder.  
- Exportieren von Folien ins SVG‑Format.  
- Arbeiten mit mehreren Master‑Folien in Präsentationen.  
- Füllen des Folienhintergrunds mit beliebigem unterstützten Stil.  
- Hinzufügen neuer oder Verwalten von Foliennotizen.  
- Dynamisches Festlegen neuer Folienmaster und -stile.  
- Hinzufügen oder Ändern von Folienübergängen.  
- Hinzufügen von Morph‑Übergängen  

### **Grafik‑ und Multimedia‑Funktionen**

- Erstellen oder Verwalten von Formen wie Linien, Rechtecken, Ellipsen und Polylinien.  
- Verwalten von Linienstilen in Formen.  
- Abrufen des Alternativtexts einer Form.  
- Drehen und Spiegeln von Formen.  
- Füllen von Formen in verschiedenen Stilen—z. B. einfarbig, Verlauf, Muster, Textur und Bild.  
- Verwenden von 48 vordefinierten Mustern und 24 Texturen.  
- Zuweisen von Links zu Formen.  
- Lesen, Ändern und Hinzufügen neuer Animationseffekte zu Folien und Formen.  
- Hinzufügen neuer Bilder zu Präsentationen und Verwalten vorhandener Bilder.  
- Hinzufügen neuer Bild‑Aufzählungszeichen zu Präsentationen und Verwalten vorhandener Aufzählungszeichen.  
- Hinzufügen neuer Audio‑Frames zu Präsentationen und Verwalten vorhandener Frames.  
- Hinzufügen neuer Video‑Frames zu Präsentationen und Verwalten vorhandener Frames.  

### **Tabellenfunktionen**

- Erstellen von Tabellen von Grund auf und Hinzufügen zu Folien.  
- Größenänderung und Verschieben von Tabellen.  
- Zugriff auf jede Zelle einer Tabelle.  
- Ändern des Formats und der Rahmen einer Zelle.  
- Größenänderung von Spalten und Zeilen.  
- Hinzufügen und Löschen von Spalten und Zeilen.  
- Festlegen von Rahmenbreite und -farbe für die gesamte Tabelle.  

### **Text‑ und Absatzfunktionen**

- Abrufen und Festlegen von Texten in Text‑Platzhaltern und Text‑Frames.  
- Abrufen des Typs eines Platzhalters.  
- Ändern der Textformatierung in einem Text‑Frame.  
- Hinzufügen neuer Schriften oder Verwalten bereits in einer Präsentation verwendeter Schriften.  
- Ändern von Texten und Textformatierung.  
- Erstellen oder Verwalten von Links im Text.  
- Unterstützung von Tabulatoren und effektiven Tabulatoren.  
- Hinzufügen neuer Text‑Frames zu fast jedem Formtyp.  
- Zugriff auf und Ändern von Rändern und Textumbruch‑Einstellungen von Text‑Frames.  
- Zugriff auf und Ändern von Absatz‑Einrückungen.  

### **Erweiterte Funktionen**

- Zugriff auf in Präsentationen eingebettete OLE‑Objekte.  
- Sperren von Präsentationen.

## **FAQ**

**Unterstützt es PDF‑Konformitätsstufen für Archivierung und Barrierefreiheit (PDF/A und PDF/UA)?**

Ja. Sie können PDFs mit PDF/A‑2a/2b/2u, PDF/A‑3a/3b sowie PDF/UA speichern, indem Sie die [PDF‑Export‑Optionen](https://reference.aspose.com/slides/php-java/aspose.slides/pdfoptions/) konfigurieren.

**Gibt es einen Schriftart‑Ersatzmechanismus und Unterstützung für benutzerdefinierte Schriftarten, um korrektes Rendering sicherzustellen?**

Ja. Die Bibliothek ermöglicht das [Laden benutzerdefinierter Schriftarten](/slides/de/php-java/custom-font/) und das [Definieren von Ersatzregeln](/slides/de/php-java/fallback-font/), sodass fehlende Glyphen beim Konvertieren und Rendern zuverlässig ersetzt werden.

**Kann ich erkennen, ob eine Datei passwortgeschützt ist, ohne sie vollständig zu öffnen?**

Ja. Sie können eine Präsentation [untersuchen](/slides/de/php-java/examine-presentation/), um festzustellen, ob sie ein Passwort erfordert, bevor das gesamte Dokument geladen wird.

**Ist Microsoft PowerPoint für die Verarbeitung und Konvertierung erforderlich?**

Nein. Aspose.Slides ist eine eigenständige Engine; PowerPoint wird auf Server oder Workstation nicht benötigt.

**Ist es sicher, Präsentationen aus mehreren Threads zu verarbeiten?**

Ja, Sie können verschiedene Dokumente parallel in separaten Threads verarbeiten; verwenden Sie jedoch nicht dieselbe Präsentationsinstanz gleichzeitig [über Threads hinweg](/slides/de/php-java/multithreading/).

**Werden Makros beibehalten und kann ich VBA in PPTM/PPSM‑Dateien verwalten?**

Ja. Präsentationen mit Makros werden [unterstützt](/slides/de/php-java/presentation-via-vba/), und Sie können die VBA‑Projekte in diesen Dateien [untersuchen und verwalten](https://reference.aspose.com/slides/php-java/aspose.slides/presentation/getvbaproject/).

**Kann ich PDF oder HTML zurück in PowerPoint‑Folien konvertieren?**

Ja. Sie können [PDF‑Seiten oder HTML‑Inhalte importieren](/slides/de/php-java/import-presentation/), um Folien in einer Präsentation zu erstellen oder zu befüllen.

**Wird der XPS‑Export unterstützt und kann ich die Qualität und den Inhalt der XPS‑Ausgabe steuern?**

Ja. Der [Export nach XPS](/slides/de/php-java/convert-powerpoint-to-xps/) ist verfügbar, und die [Speicheroptionen](https://reference.aspose.com/slides/php-java/aspose.slides/xpsoptions/) ermöglichen es, die Ausgabequalität und den enthaltenen Inhalt anzupassen.

**Kann ich Folien in Bilder konvertieren und die Ausgabequalität steuern?**

Ja. Folien können in [PNG, JPEG, GIF, BMP, TIFF](/slides/de/php-java/convert-powerpoint-to-png/) und andere Formate gerendert werden, wobei Sie die Größe und Qualität fein granular steuern können.

**Werden mathematische Formeln unterstützt (MathML/MathText) und können sie exportiert werden?**

Ja. Sie können [mathematischen Text erstellen und bearbeiten](/slides/de/php-java/powerpoint-math-equations/) und [Formeln exportieren](/slides/de/php-java/exporting-math-equations/), auch nach MathML.

**Wie geht die Bibliothek mit Verschlüsselung und Passwörtern um (öffnen, setzen, entfernen)?**

[Sie unterstützt](/slides/de/php-java/password-protected-presentation/) das Öffnen verschlüsselter Präsentationen, das Setzen oder Entfernen von Passwörtern zum Öffnen und Ändern sowie das Prüfen des Schutzstatus einer Datei.
