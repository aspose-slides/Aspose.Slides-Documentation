{{% alert color="primary" %}}

[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) ist das Präsentationsdokumentdateiformat, das von verschiedenen Versionen von Microsoft PowerPoint erstellt, gelesen, manipuliert und geschrieben werden kann. Dies ist das binäre Format für Präsentationsdokumente, das von Microsoft entwickelt wurde.

{{% /alert %}}

## **PPT in Aspose.Slides für PHP via Java**
Aspose.Slides for PHP via Java kann PPT-Dateien lesen, die mit der unten aufgeführten Software erstellt wurden.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Ebenso können PPT-Dateien, die mit Aspose.Slides für PHP via Java erstellt wurden, von der oben genannten Software gelesen werden.

## **Umfassende Unterstützung für PPT**
Aspose.Slides for PHP via Java bietet Unterstützung für fast alle Funktionen, die vom PPT-Dokumentformat unterstützt werden. Es deckt nicht nur die grundlegenden und erweiterten Funktionen ab, die von verschiedenen Microsoft PowerPoint‑Versionen für die Manipulation von PPT‑Dokumenten bereitgestellt werden, sondern auch Funktionen, die von Microsoft PowerPoint nicht einmal unterstützt werden. Der Hauptvorteil der Verwendung der Aspose.Slides for PHP via Java‑API‑Bibliothek liegt in der einfachen Handhabung solcher Funktionen.

Zusätzlich zu den grundlegenden Aufgaben beim Erstellen, Lesen und Schreiben von PPT‑Dokumentdateien bietet Aspose.Slides for PHP via Java mehrere Funktionen:

- Importieren anderer Microsoft Office-Dateiformate als [OLE-Objekte in PPT-Dokumente](/slides/de/php-java/manage-ole/).
- [Exportieren von PPT-Dokumenten nach PDF](/slides/de/php-java/convert-powerpoint-to-pdf/).
- Exportieren von Folien in den PPT-Dokumenten in SVG-Formate.
- Rendern von Folien in jedes Bildformat, das vom Java-Framework unterstützt wird.
- Festlegen der Größe von Folien in PPT-Dokumenten.
- Verwalten von Animationen auf Formen.
- Verwalten von Bildschirmanzeigen.
- [Text auf Folien formatieren](/slides/de/php-java/text-formatting/).
- Extrahieren von Text aus PPT-Dokumenten.
- [Tabellen auf Folien verarbeiten](/slides/de/php-java/powerpoint-table/).
- Kopieren von Masterfolien automatisch mit [der Klonfunktion](/slides/de/php-java/clone-slides/).
