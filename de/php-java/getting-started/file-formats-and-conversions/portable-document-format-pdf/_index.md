{{% alert color="primary" %}}

Der [Portable Document Format](https://en.wikipedia.org/wiki/PDF) ist ein Dateiformat, das von Adobe Systems für den Austausch von Dokumenten zwischen Organisationen erstellt wurde. Der Zweck des Formats war es, Inhalt und Layout unverändert zu lassen, unabhängig von der Plattform, auf der es angezeigt wurde. Aspose.Slides for PHP via Java ermöglicht das Konvertieren von Präsentationsdateien in PDF.

{{% /alert %}}

## **PDF in Aspose.Slides for PHP via Java**
Jede Präsentation, die in Aspose.Slides for PHP via Java geladen werden kann, lässt sich in ein PDF konvertieren, das entweder [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) oder [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA) entspricht, je nach Ihrer Auswahl. Aspose.Slides for PHP via Java exportiert Präsentationen nach PDF und in den meisten Fällen sieht das ausgegebene PDF genau wie die ursprüngliche Präsentation aus.

Aspose.Slides unterstützt beim Konvertieren nach PDF folgende Präsentationsfunktionen:

- Bilder, Textfelder und andere Formen.
- Text und Formatierung.
- Absätze und Formatierung.
- Hyperlinks.
- Kopf‑ und Fußzeilen.
- Aufzählungszeichen.
- Tabellen.

Sie können Präsentationen direkt mit Aspose.Slides for PHP via Java in PDFs exportieren: Sie benötigen keine weitere Komponente. Darüber hinaus können Sie den Export von Präsentation zu PDF mit verschiedenen Optionen anpassen, wie in [Converting to PDF](/slides/de/php-java/convert-powerpoint-to-pdf/) erklärt.
