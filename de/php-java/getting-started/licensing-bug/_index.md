Dieser Artikel beschreibt einen Lizenzierungsfehler, der in Aspose.Slides for PHP via Java Versionen 23.2, 23.3, 23.4, 23.5, 23.6 und 23.7 aufgetreten ist. Dieser Fehler führte dazu, dass unbefristete Lizenzen nach Ablauf ihres Abonnements vorübergehend nicht mehr nutzbar waren.

## **Symptome**

Nachdem das Abonnement Ihrer unbefristeten Lizenz abgelaufen ist, können Aspose.Slides for PHP via Java Versionen 23.2 - 23.7 die Lizenz als abgelaufen melden und alle Funktionen in einen Evaluierungsmodus überführen. Dieses Verhalten ist fehlerhaft und betrifft nicht die Versionen vor 23.2 und nach 23.7.

## **Lösung**

Aspose.Slides for PHP via Java hat dieses Problem behoben und aktualisierte Versionen (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) mit dem Fix veröffentlicht.

Wenn Sie eine der betroffenen Versionen von Aspose.Slides for PHP via Java in Ihrem Projekt verwenden, wechseln Sie bitte zur korrigierten Version.

- Prüfen Sie die von Ihnen verwendete Version und stellen Sie sicher, dass sie nicht von dem oben beschriebenen Fehler betroffen ist.
- Lesen Sie die [Aspose-Dokumentation](https://docs.aspose.com/slides/php-java/getting-started/) für Fehlersuchschritte und bekannte Lizenzierungsprobleme.

## **Betroffene Produkte und Versionen**

Bitte beachten Sie, dass dieser Fehler nur Aspose.Slides for PHP via Java Versionen 23.2 bis 23.7 betrifft. **Andere Aspose-Produkte und andere Versionen von Aspose.Slides sind nicht betroffen**.
