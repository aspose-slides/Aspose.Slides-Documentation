## **Unterstützte Plattformen**
Aspose.Slides for PHP via Java ist eine plattformunabhängige API. Sie kann auf jeder Plattform (Windows, Linux, macOS usw.) verwendet werden, auf der PHP 7 oder höher installiert ist. Oracle JDK 7 oder höher ist eine weitere Voraussetzung.

## **Dateiformate und Konvertierungen**
Aspose.Slides for PHP via Java unterstützt die meisten PowerPoint‑Dateiformate. Es ermöglicht zudem den Export in die gängigen Dateiformate, die Organisationen verwenden und untereinander austauschen. Details:

|**Funktion**|**Beschreibung**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for PHP via Java bietet die schnellste Verarbeitung für dieses Präsentationsdokumentformat.|
|PPT‑zu‑PPTX‑Konvertierung|Aspose.Slides for PHP via Java unterstützt die PPT‑zu‑PPTX‑Konvertierung.|
|Portable Document Format (PDF)|Sie können alle unterstützten Dateiformate mit einer einzigen Methode in Adobe Portable Document Format (PDF) Dokumente exportieren.|
|XML Parser Specification (XPS)|Sie können alle unterstützten Dateiformate mit einer einzigen Methode in XML Parser Specification (XPS) Dokumente exportieren.|
|Tagged Image File Format (TIFF)|Sie können alle unterstützten Präsentationsdateiformate in Tagged Image File Format (TIFF) exportieren.|
|PPTX‑zu‑HTML‑Konvertierung|Aspose.Slides for PHP via Java unterstützt die Konvertierung von PresentationEx in HTML‑Format.|

## **FAQ**

**Muss ich Microsoft PowerPoint auf dem Server/PC installieren, damit die Bibliothek funktioniert?**

Nein. PowerPoint ist nicht erforderlich; Aspose.Slides ist eine eigenständige Engine zum Erstellen, Bearbeiten, Konvertieren und Rendern von Präsentationen.

**Wie funktioniert Multithreading? Kann die Verarbeitung parallelisiert werden?**

Es ist sicher, verschiedene Dokumente in unterschiedlichen Threads zu verarbeiten; das gleiche [Präsentation](https://reference.aspose.com/slides/php-java/aspose.slides/presentation/)‑Objekt darf nicht gleichzeitig von [mehreren Threads](/slides/de/php-java/multithreading/) verwendet werden.

**Werden Dateipasswörter und Verschlüsselung unterstützt?**

Ja. [Sie können](/slides/de/php-java/password-protected-presentation/) verschlüsselte Präsentationen öffnen, ein Öffnungs‑ und Schreibpasswort setzen oder entfernen und den Schutzstatus prüfen.

**Muss ich mich in Linux‑Containern um Schriftpakete kümmern?**

Ja. Es wird empfohlen, gängige Schriftpakete zu installieren und/oder explizit [Schriftverzeichnisse angeben](/slides/de/php-java/custom-font/) in Ihrer Anwendung, um unerwartete Ersetzungen zu vermeiden.

**Gibt es Einschränkungen in der Evaluierungsversion?**

Im [Evaluierungsmodus](/slides/de/php-java/licensing/) wird dem Ausgabedokument ein Wasserzeichen hinzugefügt und bestimmte Einschränkungen gelten; eine [30‑tägige temporäre Lizenz](https://purchase.aspose.com/temporary-license/) steht für umfassende Tests zur Verfügung.

**Wird das Importieren externer Formate in eine Präsentation (PDF/HTML → PPTX) unterstützt?**

Ja. Sie können [PDF‑Seiten und HTML‑Inhalt](/slides/de/php-java/import-presentation/) zu einer Präsentation hinzufügen und sie in Folien umwandeln.
