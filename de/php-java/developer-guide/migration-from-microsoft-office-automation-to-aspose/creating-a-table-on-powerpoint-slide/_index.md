{{% alert color="primary" %}} 

Tabellen werden häufig verwendet, um Daten auf Präsentationsfolien anzuzeigen. Dieser Artikel zeigt, wie man programmgesteuert eine 15 x 15 Tabelle mit einer Schriftgröße von 10 erstellt, zunächst mit [VSTO 2008](/slides/php-java/creating-a-table-on-powerpoint-slide/) und dann mit [Aspose.Slides für PHP über Java](/slides/php-java/creating-a-table-on-powerpoint-slide/).

{{% /alert %}} 
## **Tabellen erstellen**
### **VSTO 2008 Beispiel**
Die folgenden Schritte fügen einer Microsoft PowerPoint-Folie mithilfe von VSTO eine Tabelle hinzu:

1. Eine Präsentation erstellen.
1. Eine leere Folie zur Präsentation hinzufügen.
1. Eine 15 x 15 Tabelle zur Folie hinzufügen.
1. Text in jede Zelle der Tabelle mit einer Schriftgröße von 10 hinzufügen.
1. Die Präsentation auf der Festplatte speichern.
