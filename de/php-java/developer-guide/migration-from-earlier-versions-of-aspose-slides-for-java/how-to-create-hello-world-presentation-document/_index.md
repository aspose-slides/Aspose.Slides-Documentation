## **Unterstützung für Legacy-Code**
Um den Legacy-Code, der mit vorherigen Versionen von Aspose.Slides für PHP über Java (vor 13.x) entwickelt wurde, zu verwenden, müssen Sie einige kleinere Änderungen in Ihrem Code vornehmen, und der Code wird wie zuvor funktionieren. Alle Klassen, die in der alten Aspose.Slides für PHP über Java unter den Namespaces Aspose.Slide und Aspose.Slides.Pptx vorhanden waren, sind jetzt im einzelnen Aspose.Slides-Namespace zusammengeführt. Bitte werfen Sie einen Blick auf das folgende einfache Code-Snippet zur Erstellung eines Hello World Präsentationsdokuments in der Legacy Aspose.Slides API und folgen Sie den Schritten, die beschreiben, wie man zur neuen zusammengeführten API migriert.
