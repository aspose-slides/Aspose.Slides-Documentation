[**Aspose.Slides API**](https://products.aspose.com/slides/php-java/) ermöglicht das Konvertieren von OpenOffice ODP-Präsentationen in viele Formate. Die API zum Konvertieren von ODP in andere Formate ist die gleiche wie für PowerPoint-Formate. Sie können die folgenden Beispiele verwenden, um ODP zu konvertieren, indem Sie einfach die Quelldatei in die ODP-Präsentationsdatei ändern:

- [ODP zu HTML konvertieren](/slides/de/php-java/convert-powerpoint-ppt-and-pptx-to-html/)
- [ODP zu PDF konvertieren](/slides/de/php-java/convert-powerpoint-ppt-and-pptx-to-pdf/)
- [ODP zu TIFF konvertieren](/slides/de/php-java/convert-powerpoint-ppt-and-pptx-to-tiff/)
- [ODP zu SWF Flash konvertieren](/slides/de/php-java/convert-powerpoint-ppt-and-pptx-to-swf-flash/)
- [ODP zu XPS konvertieren](/slides/de/php-java/convert-powerpoint-ppt-and-pptx-to-microsoft-xps-document/)
- [ODP zu PDF mit Notizen konvertieren](/slides/de/php-java/convert-powerpoint-ppt-and-pptx-to-pdf-notes/)
- [ODP zu TIFF mit Notizen konvertieren](/slides/de/php-java/convert-powerpoint-ppt-and-pptx-to-tiff-with-notes/)
