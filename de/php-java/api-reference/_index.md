{{% alert color="primary" %}} 

Aspose.Slides for PHP via Java ist eine Klassenbibliothek, die PHP-Anwendungen ermöglicht, Präsentationsdokumente zu lesen und zu schreiben, ohne Microsoft PowerPoint® oder andere zu verwenden. Aspose.Slides for PHP via Java erlaubt die Verwaltung von Text, Formen, Tabellen und Animationen, das Hinzufügen von Audio und Video zu Folien, die Vorschau von Folien, den Export von Folien in SVG-, PDF-Format und mehr.

Die aktuelle API-Referenz finden Sie unter [Aspose.Slides for PHP via Java API Reference](https://reference.aspose.com/slides/de/php-java/).
