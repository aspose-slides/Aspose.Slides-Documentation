## Aspose.Slides für .NET

{{% alert color="primary" %}}

![Aspose.Slides für .NET Produktlogo](home_1.png)

Aspose.Slides für .NET ist eine Microsoft PowerPoint® Verwaltungs-API, die .NET-Anwendungen ermöglicht, PowerPoint®-Dokumente zu lesen und zu schreiben, ohne Microsoft PowerPoint® zu verwenden. Aspose.Slides für .NET ist die erste und einzige Komponente, die die Funktionalität zur Verwaltung von PowerPoint®-Dokumenten bereitstellt. Aspose.Slides für .NET bietet viele wichtige Funktionen, wie z. B. das Verwalten von Text, Formen, Tabellen und Animationen, das Hinzufügen von Audio und Video zu Folien, das Vorschau von Folien, das Exportieren von Folien in SVG-, PDF-Format und mehr.
