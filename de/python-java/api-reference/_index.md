{{% alert color="primary" %}} 

Aspose.Slides für Python über Java ist eine Klassenbibliothek, die es Python-Anwendungen ermöglicht, Präsentationsdokumente zu lesen und zu schreiben, ohne Microsoft PowerPoint® oder andere zu verwenden. Aspose.Slides für Python über Java ermöglicht Ihnen die Verwaltung von Text, Formen, Tabellen und Animationen, das Hinzufügen von Audio und Video zu Folien, die Vorschau von Folien, den Export von Folien in SVG-, PDF-Format und vieles mehr.

Aspose.Slides für Python über Java verwendet die [gleiche API-Referenz wie Aspose.Slides für Java](https://reference.aspose.com/slides/python-java/). 

{{% /alert %}}
