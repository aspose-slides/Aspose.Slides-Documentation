## **Serveranforderungen**
{{% alert color="primary" %}} 

Um Aspose.Slides für SharePoint zu installieren, wird ein System benötigt, das eines der folgenden betreibt:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft Office SharePoint Server 2010
- Microsoft Office SharePoint Server 2013
- Microsoft Office SharePoint Server 2016
- Microsoft Office SharePoint Server 2019

{{% /alert %}} {{% alert color="primary" %}} 

Es wird empfohlen, Service Pack 2 oder höher für diese Produkte zu installieren.

{{% /alert %}} 
## **Clientanforderungen**
{{% alert color="primary" %}} 

SharePoint-Websites werden über einen Webbrowser aufgerufen. Aspose.Slides für SharePoint wurde mit den folgenden Browsern getestet:

- Microsoft Internet Explorer 6, 7, 8
- Mozilla FireFox 3.5

{{% /alert %}}
