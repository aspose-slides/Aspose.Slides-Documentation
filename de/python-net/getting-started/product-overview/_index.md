## **Produktbeschreibung**
Aspose.Slides for Python via .NET ist eine plattformübergreifende Klassenbibliothek, die es Ihren Anwendungen ermöglicht, eine Vielzahl von Dokumentenverarbeitungsaufgaben ohne Verwendung von Microsoft PowerPoint® durchzuführen.

### Warum Sie Aspose.Slides for Python via .NET verwenden sollten

- Aspose.Slides for Python via .NET ist die erste und einzige Komponente, die die für die Verwaltung von PowerPoint®-Dokumenten erforderlichen Funktionen bereitstellt. 
- Aspose.Slides for Python via .NET bietet zahlreiche Schlüssel‑Features wie die Verwaltung von Texten, Formen, Tabellen und Animationen, das Hinzufügen von Audio‑ und Videodateien zu Folien, die Vorschau von Folien, das Exportieren von Folien nach SVG, PDF und anderen Formaten sowie viele weitere Möglichkeiten. 
- Aspose.Slides for Python via .NET ermöglicht nicht nur das Öffnen von PowerPoint‑Dateien aus verschiedenen Quellen, sondern erlaubt Ihnen auch, Ihre Präsentationen auf vielfältige Weise zu speichern. Beispielsweise können Sie Ihre Präsentationen im PPT‑Format speichern; Sie können Ihre Folien auch als Bilder speichern. 

- Aspose.Slides for Python via .NET bietet Ihnen fast alle Funktionen, die Sie in Microsoft PowerPoint finden (oder nicht finden) könnten. Aufgrund der Vielseitigkeit von Aspose.Slides for Python via .NET können Sie, neben Tabellen und verschiedenen Formen, unterschiedliche Rahmen—Text‑, Audio‑ und Video‑Frames—zu Ihren Folien hinzufügen. 
- Aspose.Slides for Python via .NET wurde mit Managed Python entwickelt. Sie können Aspose.Slides for Python via .NET mit jeder .NET‑Sprache wie Python, VB.NET, J# usw. verwenden. Aspose.Slides for Python via .NET kann zudem mit jeder Art von Anwendung—Web‑ oder Windows‑Anwendung—eingesetzt werden. 
- Aspose.Slides for Python via .NET stellt außerdem umfassende Demos und funktionierende Beispiele bereit, um Entwicklern ein besseres Verständnis der API zu ermöglichen.

## **Aspose.Slides‑Funktionen**
Die Funktionen von Aspose.Slides for Python via .NET können in diese Gruppen unterteilt werden:

1. Präsentations‑Funktionen.
1. Folien‑Funktionen.
1. Grafik‑ und Multimedia‑Funktionen.
1. Tabellen‑Funktionen.
1. Text‑ und Absatz‑Funktionen
1. Erweiterte Funktionen

### **Präsentations‑Funktionen**
- Öffnen von Microsoft PowerPoint‑Präsentationen aus verschiedenen Quellen.
- Speichern von Präsentationen in verschiedenen Microsoft PowerPoint‑Formaten.
- Erstellen von Präsentationen von Grund auf.
- Konvertieren von Präsentationen in PDF‑Dokumente ohne Verwendung einer anderen Komponente
- Lesen oder Ändern der Dokumenteigenschaften einer Präsentation
- Anwenden von Schutz auf Präsentationen
- Exportieren von Präsentationen nach PDF
- Exportieren von Präsentationen nach TIFF
- Exportieren von Präsentationen nach XPS
- Exportieren von Präsentationen nach SWF
- Exportieren von Präsentationen zu PDF‑Notizen / TIFF‑Notizen
- Exportieren von Präsentationen nach HTML

### **Folien‑Funktionen**
- Erstellen neuer Folien von Grund auf und Hinzufügen zu Präsentationen.
- Zugriff auf beliebige Folien über deren Position in einer Präsentation.
- Kopieren oder Klonen von Folien in dieselbe oder eine andere Präsentation.
- Bearbeiten und Löschen von Folien.
- Automatisches Kopieren von Master‑Folien.
- Vorschau von Folien.
- Abrufen oder Festlegen des Namens einer Folie.
- Speichern von Folien als Bilder.
- Exportieren von Folien im SVG‑Format.
- Arbeiten mit mehreren Master‑Folien in Präsentationen.
- Füllen des Folienhintergrunds mit beliebigem unterstütztem Stil.
- Hinzufügen neuer oder Verwalten von Foliennotizen.
- Dynamisches Festlegen neuer Folienmaster und -stile.
- Hinzufügen oder Ändern von Folienübergängen.
- Hinzufügen von Morph‑Übergängen

### **Grafik‑ & Multimedia‑Funktionen**
- Erstellen oder Verwalten von Formen wie Linien, Rechtecken, Ellipsen und Polylinien.
- Verwalten von Linienstilen in Formen.
- Abrufen des Alternativtexts einer Form.
- Drehen und Spiegeln von Formen.
- Füllen von Formen in verschiedenen Stilen—z. B. einfarbig, Verlauf, Muster, Textur und Bild.
- Verwendung von 48 vordefinierten Mustern und 24 Texturen.
- Zuweisen von Links zu Formen.
- Lesen, Ändern und Hinzufügen neuer Animationseffekte zu Folien und Formen.
- Hinzufügen neuer Bilder zu Präsentationen und Verwalten vorhandener Bilder.
- Hinzufügen neuer Bild‑Aufzählungszeichen zu Präsentationen und Verwalten vorhandener.
- Hinzufügen neuer Audio‑Frames zu Präsentationen und Verwalten vorhandener.
- Hinzufügen neuer Video‑Frames zu Präsentationen und Verwalten vorhandener.

### **Tabellen‑Funktionen**
- Erstellen von Tabellen von Grund auf und Hinzufügen zu Folien.
- Größenänderung und Verschieben von Tabellen.
- Zugriff auf beliebige Zelle in einer Tabelle.
- Ändern des Formats und der Rahmen einer Zelle.
- Größenänderung von Spalten und Zeilen.
- Hinzufügen und Löschen von Spalten und Zeilen.
- Festlegen von Rahmenbreite und -farbe für die gesamte Tabelle.

### **Text‑ & Absatz‑Funktionen**
- Abrufen und Festlegen von Texten in Text‑Platzhaltern und Text‑Frames.
- Abrufen des Typs eines Platzhalters.
- Ändern der Textformatierung in einem Text‑Frame.
- Hinzufügen neuer Schriftarten oder Verwalten der bereits in einer Präsentation verwendeten Schriftarten.
- Ändern von Texten und Textformatierung.
- Erstellen oder Verwalten von Links zum Text.
- Unterstützung von Tabulatoren und effektiven Tabulatoren.
- Hinzufügen neuer Text‑Frames zu fast jedem Formtyp.
- Zugriff auf und Ändern von Rändern und Textumbruch‑Einstellungen von Text‑Frames.
- Zugriff auf und Ändern von Absatz‑Einrückungen.

### **Erweiterte Funktionen**
- Zugriff auf in Präsentationen eingebettete OLE‑Objekte.
- Sperren von Präsentationen.

## **FAQ**

**Unterstützt es PDF‑Konformitätsstufen für Archivierung und Barrierefreiheit (PDF/A und PDF/UA)?**

Ja. Sie können mit PDF‑Export‑Optionen[PDF export options](https://reference.aspose.com/slides/python-net/aspose.slides.export/pdfoptions/) nach PDF mit PDF/A‑2a/2b/2u, PDF/A‑3a/3b sowie PDF/UA speichern.

**Gibt es einen Schriftart‑Ersetzungsmechanismus und Unterstützung für benutzerdefinierte Schriftarten, um ein korrektes Rendering sicherzustellen?**

Ja. Die Bibliothek lässt Sie [benutzerdefinierte Schriftarten laden](/slides/de/python-net/custom-font/) und [Fallback‑Regeln definieren](/slides/de/python-net/fallback-font/), sodass fehlende Glyphen zuverlässig während Konvertierung und Rendering ersetzt werden.

**Kann ich erkennen, ob eine Datei passwortgeschützt ist, ohne sie vollständig zu öffnen?**

Ja. Sie können [eine Präsentation prüfen](/slides/de/python-net/examine-presentation/), um festzustellen, ob ein Passwort erforderlich ist, bevor das gesamte Dokument geladen wird.

**Ist Microsoft PowerPoint für die Verarbeitung und Konvertierung erforderlich?**

Nein. Aspose.Slides ist eine eigenständige Engine; PowerPoint wird auf dem Server oder Arbeitsplatz nicht benötigt.

**Ist es sicher, Präsentationen aus mehreren Threads zu verarbeiten?**

Ja, Sie können verschiedene Dokumente parallel in separaten Threads verarbeiten; vermeiden Sie lediglich die gleichzeitige Nutzung derselben Präsentationsinstanz über Threads hinweg[across threads](/slides/de/python-net/multithreading/).

**Werden Makros erhalten und kann ich VBA in PPTM/PPSM‑Dateien verwalten?**

Ja. Präsentationen mit Makros [werden unterstützt](/slides/de/python-net/presentation-via-vba/), und Sie können [VBA‑Projekte prüfen und verwalten](https://reference.aspose.com/slides/python-net/aspose.slides.vba/) in diesen Dateien.

**Kann ich PDF oder HTML zurück in PowerPoint‑Folien konvertieren?**

Ja. Sie können [PDF‑Seiten oder HTML‑Inhalte importieren](/slides/de/python-net/import-presentation/) to create or populate slides in a presentation.

**Wird XPS‑Export unterstützt und kann ich die Qualität und den Inhalt der XPS‑Ausgabe steuern?**

Ja. [Export nach XPS](/slides/de/python-net/convert-powerpoint-to-xps/) ist verfügbar, und [Speicher‑Optionen](https://reference.aspose.com/slides/python-net/aspose.slides.export/xpsoptions/) ermöglichen die Feinabstimmung von Qualität und enthaltenen Inhalten.

**Kann ich Folien in Bilder konvertieren und die Ausgabequalität steuern?**

Ja. Folien können [in PNG, JPEG, GIF, BMP, TIFF rendern](/slides/de/python-net/convert-powerpoint-to-png/) und andere Formate mit feiner Kontrolle über Größe und Qualität exportiert werden.

**Werden mathematische Formeln unterstützt (MathML/MathText) und können sie exportiert werden?**

Ja. Sie können [mathematischen Text erstellen und bearbeiten](/slides/de/python-net/powerpoint-math-equations/) und [Formeln exportieren](/slides/de/python-net/exporting-math-equations/), einschließlich nach MathML.

**Wie geht die Bibliothek mit Verschlüsselung und Passwörtern um (öffnen, setzen, entfernen)?**

[Unterstützt](/slides/de/python-net/password-protected-presentation/) das Öffnen verschlüsselter Präsentationen, das Setzen oder Entfernen von Passwörtern zum Öffnen und Ändern sowie das Prüfen des Schutzstatus einer Datei.
