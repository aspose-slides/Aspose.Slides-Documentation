{{% alert color="primary" %}}
Aspose.Slides for .NET ist eine Klassenbibliothek, die .NET-Anwendungen das Lesen und Schreiben von Präsentationsdokumenten ermöglicht, ohne Microsoft PowerPoint® oder andere zu verwenden. Aspose.Slides for .NET ermöglicht die Verwaltung von Text, Formen, Tabellen & Animationen, das Hinzufügen von Audio und Video zu Folien, die Vorschau von Folien, den Export von Folien nach SVG, PDF und mehr.

Die aktuelle API-Referenz finden Sie unter [Aspose.Slides for .NET API-Referenz.](https://reference.aspose.com/slides/net)
{{% /alert %}}
