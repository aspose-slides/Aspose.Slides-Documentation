{{% alert color="primary" %}} 

Aspose.Slides für .NET ist eine Klassenbibliothek, die .NET‑Anwendungen ermöglicht, Präsentationsdokumente zu lesen und zu schreiben, ohne Microsoft PowerPoint® oder andere Programme zu verwenden. Aspose.Slides für .NET erlaubt die Verwaltung von Text, Formen, Tabellen und Animationen, das Hinzufügen von Audio und Video zu Folien, die Vorschau von Folien, den Export von Folien in das SVG‑ oder PDF‑Format und mehr.

Die aktuelle API‑Referenz finden Sie unter [Aspose.Slides for .NET API Reference.](https://reference.aspose.com/slides/net)
