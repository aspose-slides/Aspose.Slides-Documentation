{{% alert color="primary" %}} 

Aspose.Slides for .NET ist eine Klassenbibliothek, die .NET-Anwendungen ermöglicht, Präsentationsdokumente zu lesen und zu schreiben, ohne Microsoft PowerPoint® oder andere Programme zu verwenden. Aspose.Slides for .NET ermöglicht das Verwalten von Text, Formen, Tabellen & Animationen, das Hinzufügen von Audio und Video zu Folien, die Vorschau von Folien, den Export von Folien in SVG, PDF-Format und mehr.

Die aktuelle API-Referenz finden Sie unter [Aspose.Slides for .NET API Reference.](https://reference.aspose.com/slides/net)
