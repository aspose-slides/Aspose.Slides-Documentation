**Aspose.Slides** für .NET ermöglicht es Entwicklern jetzt, die PPT über die Instanz der Klasse Presentation zuzugreifen und diese in das entsprechende PPTX-Format zu konvertieren. Derzeit unterstützt es die teilweise Konvertierung von PPT in PPTX. Für weitere Details darüber, welche Funktionen in der Konvertierung von PPT zu PPTX unterstützt und nicht unterstützt werden, besuchen Sie bitte diesen Dokumentationslink.

**Aspose.Slides** für .NET bietet die Klasse Presentation, die eine PPTX-Präsentationsdatei darstellt. Die Präsentationsklasse kann jetzt auch PPT über Presentation zugreifen, wenn das Objekt instanziiert wird.

``` csharp

 //Instanziieren Sie ein Präsentationsobjekt, das eine PPTX-Datei darstellt

PresentationEx pres = new PresentationEx("Conversion.ppt");

//Speichern der PPTX-Präsentation im PPTX-Format

pres.Save(MyDir +"Converted.pptx", SaveFormat.Pptx);

```
