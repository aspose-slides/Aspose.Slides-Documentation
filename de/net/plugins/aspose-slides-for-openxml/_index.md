## **Was ist OpenXML?**
OpenXML SDK ist ein .NET SDK von Microsoft, um Dateiformate von Microsoft Office 2007 und höher zu verarbeiten. Diese Dateiformate sind im Wesentlichen gezippte Pakete und das Open XML SDK für Office vereinfacht die Aufgabe, Open XML‑Pakete und die zugrunde liegenden Open XML‑Schemata‑Elemente in einem Paket zu manipulieren. Die Klassen im Open XML SDK kapseln viele gängige Aufgaben, die Entwickler an Open XML‑Paketen ausführen, sodass Sie komplexe Vorgänge mit wenigen Codezeilen erledigen können.

Nachdem Sie das Open XML SDK 2.5 installiert haben, öffnen Sie Ihr bestehendes Projekt oder Ihre Anwendung in Visual Studio oder erstellen ein neues Projekt bzw. eine neue Anwendung. Dann fügen Sie in Ihrem Projekt oder Ihrer Anwendung Verweise zu den folgenden Komponenten hinzu:

- DocumentFormat.OpenXml
- WindowsBase
## **Aspose.Slides für .NET**
Aspose.Slides für .NET ist eine einzigartige PowerPoint®‑Verwaltungskomponente, die .NET‑Anwendungen das Lesen, Schreiben und Manipulieren von PowerPoint‑Dokumenten ermöglicht, ohne Microsoft PowerPoint zu verwenden.

Aspose.Slides für .NET ist die erste und einzige .NET‑Komponente, die Funktionalitäten zum Verwalten von PowerPoint‑Dokumenten in Ihren eigenen Anwendungen bereitstellt. Wie alle Aspose‑.NET‑Komponenten ist Aspose.Slides in verwaltetem C# geschrieben, preislich erstaunlich und blitzschnell.

## **Einleitung**
Entwickler, die nach Bibliotheken für Dateiformate suchen, müssen sich die von den auf dem Markt verfügbaren Bibliotheken angebotenen Funktionen ansehen. Im Wesentlichen werden bei der Auswahl einer Option zwei Faktoren berücksichtigt:

- Funktionen
- Aufwand, der für die Nutzung der Bibliothek erforderlich ist

Wenn Sie ein .NET‑Entwickler sind und nach einer schnelleren, einfachen und leichten Bibliothek für Dateiformate zur Verarbeitung von Dateien suchen, sollten Sie VSTO (Visual Studio Tools for Office) mit Aspose vergleichen. In diesem Fall wird Ihnen dieses Projekt helfen, die Funktionen und den Code beider Bibliotheken zu vergleichen.
