## **Was ist OpenXML?**
OpenXML SDK ist ein .NET SDK von Microsoft, um Microsoft Office 2007 und neuere Dateiformate zu verarbeiten. Diese Dateiformate sind im Grunde ZIP‑Pakete und das Open XML SDK for Office vereinfacht das Manipulieren von Open XML‑Paketen und den zugrunde liegenden Open XML‑Schematenelementen innerhalb eines Pakets. Die Klassen im Open XML SDK kapseln viele gängige Aufgaben, die Entwickler mit Open XML‑Paketen ausführen, sodass Sie komplexe Vorgänge mit wenigen Codezeilen erledigen können.

Nachdem Sie das Open XML SDK 2.5 installiert haben, öffnen Sie Ihr bestehendes Projekt oder Ihre Anwendung in Visual Studio oder erstellen ein neues Projekt bzw. eine neue Anwendung. Fügen Sie dann in Ihrem Projekt oder Ihrer Anwendung Verweise auf die folgenden Komponenten hinzu:
- DocumentFormat.OpenXml
- WindowsBase

## **Aspose.Slides for .NET**
Aspose.Slides for .NET ist eine einzigartige PowerPoint®‑Verwaltungskomponente, die .NET‑Anwendungen das Lesen, Schreiben und Manipulieren von PowerPoint‑Dokumenten ermöglicht, ohne Microsoft PowerPoint zu verwenden.

Aspose.Slides for .NET ist die erste und einzige .NET‑Komponente, die Funktionalität zum Verwalten von PowerPoint‑Dokumenten innerhalb Ihrer eigenen Anwendungen bereitstellt. Wie alle Aspose .NET‑Komponenten ist Aspose.Slides in verwaltetem C# geschrieben, preiswert und blitzschnell.
