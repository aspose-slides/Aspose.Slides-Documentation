## **Willkommen bei Aspose.Slides for Reporting Services!**

Aspose.Slides for Reporting Services ist die einzige Lösung auf dem Markt, die es ermöglicht, echte PPT- und PPS-Berichte in Microsoft SQL Server 2005, 2008, 2012, 2016 und 2017 Reporting Services (32-Bit und 64-Bit) zu erstellen. Alle RDL-Berichtsfunktionen, einschließlich Tabellen, Matrizen, Diagramme und Bilder, werden mit höchster Präzision in Microsoft PowerPoint-Präsentationen konvertiert.

## **Produktübersicht**

Microsoft SQL Server Reporting Services verfügt nicht über integrierte Funktionen zum Exportieren von Berichten als Microsoft PowerPoint-Präsentationen, aber nach der Installation von Aspose.Slides for Reporting Services auf Ihrem Server erhalten Sie Zugriff auf zusätzliche Exportformate:

- PPT - PowerPoint-Präsentation über Aspose.Slides
- PPS - PowerPoint-SlideShow über Aspose.Slides
- PPTX - PowerPoint 2007-Präsentation über Aspose.Slides
- PPSX - PowerPoint 2007-SlideShow über Aspose.Slides

Aspose.Slides for Reporting Services erstellt Präsentationen auf dem Server, ohne Microsoft PowerPoint zu verwenden. Aspose.Slides for Reporting Services verwendet intern Aspose.Slides für .NET – die erstklassige Komponente für die serverseitige Verarbeitung von Präsentationen.

**Aspose.Slides for Reporting Services ermöglicht den Export jedes Berichts im PPT-, PPS-, PPTX- oder PPSX-Format.**
