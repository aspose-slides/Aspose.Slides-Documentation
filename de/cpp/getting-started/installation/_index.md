## **Windows**
NuGet bietet den einfachsten Weg zum Herunterladen und Installieren der Aspose-APIs für C++ auf PCs. 

### **Option One: Install or Update Aspose.Slides for C++ from the NuGet Package Manager**

1. Öffnen Sie Microsoft Visual Studio. 
2. Erstellen Sie eine einfache Konsolenanwendung. Oder Sie können Ihr bevorzugtes Projekt öffnen. 
3. Navigieren Sie zu **Tools** > **NuGet package manager**.
4. Unter **Browse** geben Sie *Aspose.Slides.Cpp* in das Textfeld ein. 

![todo:image_alt_text](installation_1.png)

3. Klicken Sie auf die benötigte Version **Aspose.Slides.Cpp** und dann auf **Install**. 
   * Wenn Sie Aspose.Slides aktualisieren möchten – Sie haben es bereits installiert – klicken Sie stattdessen auf **Update**. 

Die ausgewählte API wird heruntergeladen und in Ihrem Projekt referenziert.

### **Option 2: Install or Update Aspose.Slides Through the Package Manager Console**

Um die [Aspose.Slides API](https://www.nuget.org/packages/Aspose.Slides.Cpp/) über die Package Manager Console zu referenzieren, führen Sie Folgendes aus:

1. Öffnen Sie Ihre Lösung/Projekt in Visual Studio.

2. Navigieren Sie zu **Tools** > **NuGet Package Manager** > **Package Manager Console**. 

   Die Package Manager Console öffnet sich. 

![todo:image_alt_text](installation_2.png)

4. Geben Sie diesen Befehl ein: `Install-Package Aspose.Slides.Cpp` 
> Wenn Sie die x86-Version installieren möchten, verwenden Sie das Paket Aspose.Slides.Cpp.x86: `Install-Package Aspose.Slides.Cpp.x86`

5. Drücken Sie die Eingabetaste.

   Die neueste vollständige Version wird in Ihre Anwendung installiert. 

   * Alternativ können Sie dem Befehl das Präfix `-prerelease` hinzufügen, um anzugeben, dass die neueste Version (einschließlich Hotfixes) ebenfalls installiert werden soll.

![todo:image_alt_text](installation_3.png)

Sobald der Download abgeschlossen ist, sollten Sie einige Bestätigungsnachrichten sehen.  

![todo:image_alt_text](installation_4.png)

Falls Sie mit der [Aspose EULA](https://about.aspose.com/legal/eula) nicht vertraut sind, möchten Sie möglicherweise die in der URL referenzierte Lizenz lesen.  

In der Package Manager Console können Sie den Befehl `Update-Package Aspose.Slides.Cpp` ausführen, um nach Updates für das Aspose.Slides-Paket zu suchen. Updates (falls gefunden) werden automatisch installiert. Sie können auch das Präfix `-prerelease` verwenden, um die neueste Version zu aktualisieren.


### **Using Include and lib Folders**
1. [Download](https://downloads.aspose.com/slides/cpp) die neueste Aspose.Slides für C++-Version.
2. Entpacken Sie den Ordner in die Produktionsumgebung.
3. Um Aspose.Slides für C++ zu verwenden, referenzieren Sie die Include- und lib-Ordner in Ihrem Projekt

## **FAQ**

**Gibt es eine kostenlose Version oder eine Einschränkung in der Testphase?**

Ja, standardmäßig läuft Aspose.Slides im Evaluierungsmodus, der Wasserzeichen einfügt und möglicherweise weitere Einschränkungen hat. Um die Beschränkungen zu entfernen, müssen Sie eine gültige [license](/slides/de/cpp/licensing/) anwenden.
