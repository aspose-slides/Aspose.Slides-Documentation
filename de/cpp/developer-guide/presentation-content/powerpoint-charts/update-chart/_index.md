## **Diagramm aktualisieren**
Aspose.Slides für C++ hat die einfachste API bereitgestellt, um Diagramme auf die einfachste Weise zu aktualisieren. Um ein Diagramm in einer Folie zu aktualisieren:

- Öffnen Sie eine Instanz der Presentation Klasse, die das Diagramm enthält.
- Erhalten Sie die Referenz einer Folie, indem Sie ihren Index verwenden.
- Durchlaufen Sie alle Formen, um das gewünschte Diagramm zu finden.
- Greifen Sie auf das Arbeitsblatt der Diagrammdaten zu.
- Ändern Sie die Datenreihen des Diagramms, indem Sie die Werte der Reihen ändern.
- Fügen Sie eine neue Reihe hinzu und befüllen Sie die Daten darin.
- Schreiben Sie die modifizierte Präsentation als PPTX-Datei.
