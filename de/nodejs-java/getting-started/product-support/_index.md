## **Technischer Support**
{{% alert color="primary" %}}

Aspose bietet unbegrenzten kostenlosen technischen Support für alle seine Produkte. Der Support steht allen Benutzern zur Verfügung (einschließlich Benutzern mit Evaluierungspaketen).

{{% /alert %}} 

Wenn Sie Hilfe zu Aspose.Slides für Node.js via Java benötigen, berücksichtigen Sie Folgendes:

- Der Hauptweg für Support ist das [Aspose.Forums](https://forum.aspose.com/c/slides/11). Stellen Sie Ihre Frage im Aspose.Slides‑Forum—sie wird innerhalb weniger Stunden beantwortet. Das Aspose.Slides‑Support‑Team beantwortet Fragen, die im Forum gestellt werden, direkt.
- Bitte beachten Sie, dass Aspose keinen technischen Support per Telefon anbietet. Telefonischer Support ist nur für Verkaufs‑ und Kauffragen verfügbar.
- Wenn Sie eine Antwort im Forum erwarten, haben Sie bitte Geduld und berücksichtigen Sie Zeitunterschiede.

Wenn Sie ein Problem mit Aspose.Slides für Node.js via Java haben, folgen Sie diesen Richtlinien, um es auf effizienteste Weise zu lösen:

- Stellen Sie sicher, dass Sie die neueste Version von Aspose.Slides für Node.js via Java verwenden, bevor Sie das Problem melden. Möglicherweise wurde das von Ihnen erlebte Problem bereits in einem Update behoben. 

  Siehe [Aspose.Slides download](https://releases.aspose.com/slides/nodejs-java/), um Informationen zur neuesten Version zu erhalten.

- Bevor Sie ein Problem melden, sollten Sie ggf. unsere Foren, diese Dokumentation und die API‑Referenz durchsuchen. Möglicherweise wurde die Antwort auf Ihre Frage bereits bereitgestellt. 

- Wenn Sie ein Problem melden, fügen Sie bitte das Originaldokument und (wenn möglich) einen Code‑Auszug, der das Problem betrifft, bei. Wenn Sie mehrere Dateien anhängen müssen, packen Sie sie in ein ZIP‑Archiv. Sie können Ihre Dokumente sicher im Aspose.Forums anhängen, da **nur** Sie und die Aspose‑Entwickler Zugriff auf die beigefügten Dateien haben.

- Erstellen und verwenden Sie einen Thread pro spezifischem Problem. Wenn Sie ein weiteres Problem haben, müssen Sie dafür einen neuen Thread anlegen.

## **FAQ**

**Welche kostenpflichtigen Support‑Stufen stehen zur Verfügung und wie unterscheiden sie sich?**

[Developer, Business und Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Sie unterscheiden sich durch die Anzahl der enthaltenen Vorfälle, die Priorität der Antwort und die verfügbaren Kommunikationskanäle.

**Kann ich einen Fehler direkt an das Engineering-Team eskalieren?**

Ja. [Kostenpflichtiger Support](https://helpdesk.aspose.com/) ermöglicht eine Eskalation zu den Entwicklern, wenn ein Problem eine tiefere Untersuchung erfordert.

**Werden ältere Bibliotheksversionen unterstützt und werden dafür Korrekturen veröffentlicht?**

Sie können ältere Versionen verwenden, aber Korrekturen werden nur für die aktuelle Version bereitgestellt; Patches für ältere Versionen werden nicht veröffentlicht.
