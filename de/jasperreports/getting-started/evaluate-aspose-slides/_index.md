Sie können Aspose.Slides ganz einfach zur Evaluierung herunterladen. Der Evaluierungsdownload ist identisch mit dem gekauften Download. Die Evaluierungsversion wird einfach lizenziert, wenn Sie ein paar Zeilen Code hinzufügen, um die Lizenz anzuwenden.

Die Evaluierungsversion von Aspose.Slides (ohne angegebene Lizenz) bietet die volle Produktfunktionalität, fügt jedoch beim Öffnen und Speichern ein Evaluierungs-Wasserzeichen am oberen Rand des Dokuments ein und beschränkt sich beim Extrahieren des Textes aus Präsentationsfolien auf eine Folie.
