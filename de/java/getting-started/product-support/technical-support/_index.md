## **Technischer Support**

{{% alert color="primary" %}} 

Aspose bietet unbegrenzten kostenlosen technischen Support für alle seine Produkte. Der Support steht allen Benutzern zur Verfügung, einschließlich der Evaluierung.

{{% /alert %}} 

Wenn Sie Hilfe zu Aspose.Slides für Java benötigen, beachten Sie Folgendes:

- Der Hauptweg für Support ist das Aspose.Forums. Stellen Sie Ihre Frage im Aspose.Slides-Forum und sie wird innerhalb weniger Stunden beantwortet. Die Fragen werden direkt vom Aspose.Slides-Supportteam beantwortet.
- Bitte beachten Sie, dass Aspose keinen telefonischen technischen Support anbietet. Telefonischer Support ist nur für Vertriebs- und Kauffragen verfügbar.
- Wenn Sie eine Antwort im Forum erwarten, berücksichtigen Sie bitte die Zeitverschiedenheiten.

Wenn Sie ein Problem mit Aspose.Slides für Java haben, befolgen Sie diese einfachen Schritte, um sicherzustellen, dass es auf effizienteste Weise gelöst wird:

- Stellen Sie sicher, dass Sie die neueste Aspose.Slides für Java-Version verwenden, bevor Sie das Problem melden; siehe Aspose.Slides-Download, um die neueste Version zu erfahren.
- Werfen Sie einen Blick in die Foren, diese Dokumentation und die API-Referenz, bevor Sie das Problem melden; vielleicht wurde Ihre Frage bereits beantwortet.
- Wenn Sie ein Problem melden, fügen Sie bitte das Originaldokument und ggf. einen Codeausschnitt bei, der das Problem verursacht. Wenn Sie mehrere Dateien anhängen müssen, zippen Sie sie zu einer Datei. Es ist sicher, Ihre Dokumente im Aspose.Forums anzuhängen, da nur Sie und die Aspose‑Entwickler Zugriff auf die angehängten Dateien haben.
- Bitte versuchen Sie, pro Thread nur ein Problem zu melden. Wenn Sie ein weiteres Problem haben, melden Sie es in einem separaten Thread.

## **FAQ**

**Welche kostenpflichtigen Support‑Stufen sind verfügbar und wie unterscheiden sie sich?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Sie unterscheiden sich nach der Anzahl der enthaltenen Vorfälle, der Priorität der Antwort und den verfügbaren Kommunikationskanälen.

**Kann ich einen Fehler direkt an das Engineering‑Team weiterleiten?**

Ja. [Paid support](https://helpdesk.aspose.com/) ermöglicht die Eskalation an die Entwickler, wenn ein Problem einer tieferen Untersuchung bedarf.

**Werden ältere Bibliotheksversionen unterstützt und veröffentlichen Sie Updates dafür?**

Sie können ältere Versionen verwenden, aber Fehlerbehebungen werden nur für die aktuelle Version bereitgestellt; Patches für Legacy‑Versionen werden nicht veröffentlicht.
