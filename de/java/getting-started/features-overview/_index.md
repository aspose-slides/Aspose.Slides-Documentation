## **Unterstützte Plattformen**
Aspose.Slides for Java unterstützt die beliebtesten Entwicklungs- und Bereitstellungsplattformen.

|**Funktion**|**Beschreibung**|
| :- | :- |
|Desktop-Anwendungen|Aspose.Slides für Java kann zur Entwicklung von Windows Forms-Anwendungen verwendet werden|
|Enterprise-Webanwendungen|Die Verwendung von Aspose.Slides für Java hilft beim Erstellen von Webanwendungen. Unterstützung für die Nutzung von Aspose.Slides für Java mit PHP wird ebenfalls bereitgestellt.|
|Linux/Unix|Aspose.Slides für Java ist eine plattformunabhängige API und kann in einer Linux- und Unix-Umgebung arbeiten.|

|Form finden|Sie können eine bestimmte Form auf einer Folie mithilfe des alternativen Textes der Form finden.|

|Textfelder|Textfelder können von Grund auf neu erstellt werden. Auf vorhandene Textfelder kann zugegriffen und ihr Text kann geändert werden, ohne das ursprüngliche Textformat zu verlieren.|
|Rechteckformen|Rechteckformen können mit Aspose.Slides für Java erstellt oder geändert werden.|

## **FAQ**

**Muss ich Microsoft PowerPoint auf dem Server/PC installieren, damit die Bibliothek funktioniert?**

Nein. PowerPoint ist nicht erforderlich; Aspose.Slides ist eine eigenständige Engine zum Erstellen, Bearbeiten, Konvertieren und Rendern von Präsentationen.

**Wie funktioniert Multithreading? Kann die Verarbeitung parallelisiert werden?**

Es ist sicher, verschiedene Dokumente in unterschiedlichen Threads zu verarbeiten; dasselbe [presentation](https://reference.aspose.com/slides/java/com.aspose.slides/presentation/)‑Objekt darf nicht gleichzeitig von [multiple threads](/slides/de/java/multithreading/) verwendet werden.

**Werden Dateipasswörter und Verschlüsselung unterstützt?**

Ja. [You can](/slides/de/java/password-protected-presentation/) verschlüsselte Präsentationen öffnen, ein Öffnungs‑ und Schreibpasswort setzen oder entfernen und den Schutzstatus prüfen.

**Muss ich mich um Schriftpakete in Linux‑Containern kümmern?**

Ja. Es wird empfohlen, gängige Schriftpakete zu installieren und/oder in Ihrer Anwendung ausdrücklich [specify font directories](/slides/de/java/custom-font/) anzugeben, um unerwartete Ersetzungen zu vermeiden.

**Gibt es Einschränkungen in der Evaluierungs‑Version?**

Im [evaluation mode](/slides/de/java/licensing/) wird dem Output ein Wasserzeichen hinzugefügt und bestimmte Einschränkungen gelten; eine [30‑day temporary license](https://purchase.aspose.com/temporary-license/) ist für Tests mit vollem Funktionsumfang verfügbar.

**Wird das Importieren externer Formate in eine Präsentation (PDF/HTML → PPTX) unterstützt?**

Ja. Sie können [PDF pages and HTML content](/slides/de/java/import-presentation/) zu einer Präsentation hinzufügen und in Folien umwandeln.
