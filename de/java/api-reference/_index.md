{{% alert color="primary" %}} 

Aspose.Slides for Java ist eine Klassenbibliothek, die es Java-Anwendungen ermöglicht, Präsentationsdokumente zu lesen und zu schreiben, ohne Microsoft PowerPoint® oder andere zu verwenden. Aspose.Slides for Java erlaubt die Verwaltung von Text, Formen, Tabellen & Animationen, das Hinzufügen von Audio und Video zu Folien, die Vorschau von Folien, den Export von Folien nach SVG, PDF und mehr.

Die aktuelle API-Referenz finden Sie unter
[Aspose.Slides for Java API-Referenz.](https://reference.aspose.com/slides/java)
