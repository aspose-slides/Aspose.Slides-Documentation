## **Unterstützung für Legacy-Code**
Um den Legacy-Code, der mit Aspose.Slides für Java-Versionen vor 13.x entwickelt wurde, zu verwenden, müssen Sie einige kleinere Änderungen an Ihrem Code vornehmen, und der Code funktioniert wie zuvor. Alle Klassen, die in der alten Aspose.Slides für Java unter den Namensräumen Aspose.Slide und Aspose.Slides.Pptx vorhanden waren, sind jetzt in einem einzigen Aspose.Slides-Namespace zusammengeführt worden. Bitte werfen Sie einen Blick auf den folgenden einfachen Codeausschnitt zur Erstellung eines Hello World Präsentationsdokuments in der Legacy Aspose.Slides API und folgen Sie den Schritten, die beschreiben, wie man zur neuen zusammengeführten API migriert.
