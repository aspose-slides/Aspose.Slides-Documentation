Aspose.Slides for Java 14.3.0 (14.4.0) bietet eine völlig neue Implementierung der PPT-Verarbeitung. Es gibt zahlreiche Verbesserungen, eine teilweise Konvertierung von PPTX zu PPT. Es gibt jedoch einige nicht implementierte Funktionen:

- Einige Formen haben in serialisierten PPT-Dokumenten eine falsche Geometrie (Call-outs)
- Nicht alle Textformatierungsfunktionen von PPTX werden bei der PPT-Serialisierung unterstützt
- Informationen zur Textsprache und zu Rechtschreiboptionen fehlen in serialisierten PPT-Dokumenten
- Nicht alle Funktionen von PPTX-Themes werden bei der PPT-Serialisierung unterstützt

**Es gibt einige Unterschiede im Vergleich zu Aspose.Slides for Java 8.6.0:**

- Es gibt bekannte Probleme bei der OLE/ActiveX-PPT-Serialisierung zu PPT

**Es gibt einige Unterschiede im Vergleich zu Aspose.Slides for .NET 14.3.0:**

- Die Druckunterstützung für Präsentationen ist derzeit in Aspose.Slides for Java nicht verfügbar
