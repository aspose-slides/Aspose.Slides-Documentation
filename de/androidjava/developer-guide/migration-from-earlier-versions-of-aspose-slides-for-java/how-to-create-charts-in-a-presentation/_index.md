Eine neue [Aspose.Slides für Java API](https://docs.aspose.com/slides/androidjava/) wurde veröffentlicht und jetzt unterstützt dieses einzelne Produkt die Fähigkeit, PowerPoint-Dokumente von Grund auf zu erstellen und bestehende Dokumente zu bearbeiten.

{{% /alert %}} 
## **Unterstützung für Legacy-Code**
Um den Legacy-Code zu verwenden, der mit früheren Aspose.Slides for Java-Versionen vor 14.x.x entwickelt wurde, müssen Sie einige kleinere Änderungen an Ihrem Code vornehmen, und der Code wird wie zuvor funktionieren. Alle Klassen, die in der alten Aspose.Slides for Java unter den Namespaces com.aspose.slides und com.aspose.slides.pptx vorhanden waren, sind jetzt im einzelnen Namespace com.aspose.slides zusammengefasst. Bitte werfen Sie einen Blick auf den folgenden einfachen Code-Schnipsel zum Erstellen eines normalen Diagramms von Grund auf in der Präsentation unter Verwendung der Legacy Aspose.Slides API und folgen Sie den Schritten zur Migration zur neuen zusammengefassten API.
