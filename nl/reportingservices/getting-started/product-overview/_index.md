## **Welkom bij Aspose.Slides for Reporting Services!**

Aspose.Slides for Reporting Services is de enige oplossing op de markt die het mogelijk maakt om echte PPT- en PPS-rapporten te genereren in Microsoft SQL Server 2005, 2008, 2012, 2016 en 2017 Reporting Services (32-bit en 64-bit). Alle RDL-rapportfuncties, inclusief tabellen, matrixen, grafieken en afbeeldingen, worden met de hoogste precisie omgezet naar Microsoft PowerPoint‑presentaties.

## **Productoverzicht**

Microsoft SQL Server Reporting Services heeft geen ingebouwde mogelijkheden om rapporten te exporteren als Microsoft PowerPoint‑presentaties, maar na het installeren van Aspose.Slides for Reporting Services op uw server krijgt u toegang tot extra exportformaten:

- PPT – PowerPoint‑presentatie via Aspose.Slides
- PPS – PowerPoint‑diavoorstelling via Aspose.Slides
- PPTX – PowerPoint 2007‑presentatie via Aspose.Slides
- PPSX – PowerPoint 2007‑diavoorstelling via Aspose.Slides

Aspose.Slides for Reporting Services maakt presentaties op de server zonder gebruik van Microsoft PowerPoint. Aspose.Slides for Reporting Services maakt intern gebruik van Aspose.Slides for .NET – de eersteklas component voor server‑side presentatieverwerking.

**Aspose.Slides for Reporting Services maakt het mogelijk om elk rapport te exporteren in PPT-, PPS-, PPTX- of PPSX‑formaat.**
