Er waren in het verleden vragen over Aspose.Slides voor Reporting Services‑integratie met SharePoint. In dit artikel richten we ons op SharePoint 2010. Er wordt aangenomen dat er al een SharePoint‑farmomgeving is geïnstalleerd. De voorbeelden die we in dit artikel zullen volgen, zijn gebaseerd op een volledige SharePoint‑cloud, maar de stappen zijn vergelijkbaar voor een SharePoint Foundation‑server. Voordat we verder gaan, laten we beginnen met enkele belangrijke documentatie die je als referentie kunt gebruiken.

- [Overzicht van Reporting Services en SharePoint‑technologie‑integratie](https://docs.microsoft.com/en-us/previous-versions/sql/sql-server-2008-r2/bb326358(v=sql.105))
- [Reporting Services configureren voor SharePoint 2010‑integratie](https://docs.microsoft.com/en-us/previous-versions/sql/)

{{% /alert %}} 
#### **Omgevingsconfiguratie**
De configuratie bestaat uit **4 servers**. Dit omvat een **Domain Controller**, een **SQL Server**, een **SharePoint Server** en een server voor **Reporting Services**. Je kunt ervoor kiezen om SharePoint en Reporting Services op dezelfde machine te plaatsen.
