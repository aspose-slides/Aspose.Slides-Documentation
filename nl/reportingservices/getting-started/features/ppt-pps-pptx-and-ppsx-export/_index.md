{{% alert color="primary" %}} 

Microsoft SQL Server Reporting Services heeft geen ingebouwde mogelijkheden om rapporten te exporteren als Microsoft PowerPoint‑presentaties, maar na het installeren van Aspose.Slides for Reporting Services op uw server krijgt u toegang tot twee extra exportformaten: 

- Microsoft PowerPoint‑presentatie (PPT)
- Microsoft PowerPoint‑diavoorstelling (PPS)
- Microsoft PowerPoint 2007‑presentatie (PPTX)
- Microsoft PowerPoint 2007‑diavoorstelling (PPSX)

{{% /alert %}} 

Om documenten in deze formaten te maken, maakt Aspose.Slides for Reporting Services gebruik van een ingebouwde versie van [Aspose.Slides for .NET](http://www.aspose.com/Products/Aspose.Slides/), de marktleidende component voor het verwerken van presentaties van Aspose. Microsoft PowerPoint wordt niet gebruikt om de documenten te genereren. 


PPT‑ en PPS‑kopieën van hetzelfde rapport zullen er identiek uitzien, dankzij de hoge mate van ondersteuning voor de onderliggende bestandsformaten, aangeboden door Aspose.Slides.
