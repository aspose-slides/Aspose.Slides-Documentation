{{% alert color="primary" %}} 

Aspose.Slides for Node.js via Java is een klassenbibliotheek die Node.js‑toepassingen in staat stelt presentatiedocumenten te lezen en te schrijven zonder Microsoft PowerPoint® of andere software te gebruiken. Aspose.Slides for Node.js via Java stelt je in staat om tekst, vormen, tabellen en animaties te beheren, audio en video aan dia’s toe te voegen, dia’s te bekijken, dia’s naar SVG of PDF te exporteren en nog veel meer.

De nieuwste API‑referentie kun je vinden op [Aspose.Slides for Node.js via Java API Reference](https://reference.aspose.com/slides/nl/nodejs-java/).
