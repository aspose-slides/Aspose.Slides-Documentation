{{% alert color="primary" %}} 

Aspose.Slides for C++ is een class library die C++‑toepassingen in staat stelt presentatiedocumenten te lezen en te schrijven zonder Microsoft PowerPoint® of andere te gebruiken. Aspose.Slides for C++ maakt het mogelijk om tekst, vormen, tabellen en animaties te beheren, audio en video aan dia’s toe te voegen, dia’s vooraf te bekijken, dia’s te exporteren naar SVG, PDF‑formaat en meer.

De nieuwste API‑referentie is te vinden op [Aspose.Slides for C++ API Reference](https://reference.aspose.com/slides/nl/cpp/).
