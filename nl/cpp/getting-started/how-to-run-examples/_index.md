## **Softwarevereisten**
Zorg ervoor dat u aan de volgende vereisten voldoet voordat u de voorbeelden downloadt en uitvoert.

1. Visual Studio 2015 of hoger.
## **Aspose.Slides downloaden van GitHub**
Alle voorbeelden van Aspose.Slides voor C++ staan op [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-C).

- U kunt de repository klonen met uw favoriete GitHub-client of het ZIP-bestand downloaden vanaf [hier](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/refs/heads/master.zip).
- Pak de inhoud van het ZIP-bestand uit naar een map op uw computer. Alle voorbeelden bevinden zich in de map **Examples**.
- Er is een Visual Studio-oplossingsbestand voor C++.
- De projecten zijn gemaakt in Visual Studio 2015, maar de oplossingsbestanden zijn compatibel met Visual Studio 2015 en hoger.
- Open het oplossingsbestand in Visual Studio en bouw het project.
- Download de nieuwste release van Aspose.Slides voor C++ via de [Download Section](https://downloads.aspose.com/slides/nl/cpp).
- Pak de release uit en kopieer de mappen **Include** en **Bin** naar de map **Examples**.
- Open het bestand **main.cpp**; alle voorbeelden worden hier vanaf aangeroepen.
- Verwijder de commentaartekens bij de voorbeelden die u wilt uitvoeren binnen het project.
- Bouw en compileer de oplossing

Neem gerust contact op via onze forums als u problemen ondervindt bij het installeren of uitvoeren van de voorbeelden.
## **Bijdragen**
Als u een voorbeeld wilt toevoegen of verbeteren, moedigen wij u aan een bijdrage te leveren aan het project. Alle voorbeelden en showcase-projecten in deze repository zijn open source en kunnen vrij worden gebruikt in uw eigen toepassingen.

Om bij te dragen, kunt u de repository forken, de broncode bewerken en een pull request aanmaken. Wij zullen de wijzigingen beoordelen en opnemen in de repository indien ze nuttig blijken te zijn.
