**Welkom bij Aspose.Slides for Java!**

Aspose.Slides for Java is een Microsoft PowerPoint®-beheers-API die Java‑applicaties in staat stelt PowerPoint®-documenten te lezen en te schrijven zonder Microsoft PowerPoint®. Aspose.Slides for Java is de eerste en enige component die de functionaliteit biedt om PowerPoint®‑documenten te beheren. Aspose.Slides for Java biedt tal van belangrijke functies, zoals het beheren van tekst, vormen, tabellen en animaties, het toevoegen van audio‑ en video‑materiaal aan dia's, het voorvertonen van dia's, en het exporteren van dia's naar SVG, PDF en andere formaten.
