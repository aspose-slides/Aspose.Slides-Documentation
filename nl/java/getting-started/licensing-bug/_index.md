## **Overzicht**

Dit artikel beschrijft een licentie‑bug die is aangetroffen in Aspose.Slides for Java versies 23.2, 23.3, 23.4, 23.5, 23.6 en 23.7. Deze bug zorgde ervoor dat eeuwigdurende licenties tijdelijk onbruikbaar werden nadat hun abonnement was verlopen.

## **Symptomen**

Nadat uw abonnement op een eeuwigdurende licentie verloopt, kan Aspose.Slides for Java versies 23.2 - 23.7 de licentie als verlopen melden, waardoor alle functies overgaan naar een evaluatiemodus. Dit gedrag is onjuist en treft geen versies vóór 23.2 en na 23.7.

## **Oplossing**

Aspose.Slides for Java heeft dit probleem verholpen en bijgewerkte versies (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) met de fix uitgebracht.

Als u een van de getroffen versies van Aspose.Slides for Java in uw project gebruikt, schakelt u over naar de gecorrigeerde versie.

## **Betroffen producten en versies**

Houd er rekening mee dat deze bug alleen invloed heeft op Aspose.Slides for Java versies 23.2 tot en met 23.7. **Andere Aspose‑producten en andere versies van Aspose.Slides zijn niet getroffen**.
