## **Over TIFF**

Het [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) dat bekend staat om het opslaan van meerdere afbeeldingen in één document, werd oorspronkelijk gemaakt door Aldus. Dit formaat wordt breed ondersteund door scan‑, fax‑ en andere beeldbewerkingsapplicaties.

## **TIFF in Aspose.Slides for Java**

Elk document dat kan worden geladen in Aspose.Slides for Java kan ook direct worden geconverteerd naar een TIFF‑document door Aspose.Slides for Java, waardoor het gebruik van een derde‑partij component overbodig is. Bovendien kunt u optioneel de grootte van de afbeeldingen in het resulterende TIFF‑document bepalen. U vindt informatie over het exporteren van presentatiedocumenten naar TIFF‑documenten via Aspose.Slides for Java in [deze topic](/slides/nl/java/converting-a-presentation/).
