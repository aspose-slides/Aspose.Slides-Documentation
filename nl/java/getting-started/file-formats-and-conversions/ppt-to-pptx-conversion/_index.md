PPT‑naar‑PPTX‑conversie wordt ondersteund door Aspose.Slides for Java. De meeste presentatiefuncties – master‑dia’s, structuur enzovoort – blijven behouden bij conversie van het ene formaat naar het andere, maar er zijn [een paar beperkingen](/slides/nl/java/ppt-to-pptx-conversion/).

## **Ondersteunde functies bij conversie**
Aspose.Slides for Java biedt gedeeltelijke ondersteuning voor het converteren van het PPT‑bestandformaat naar PPTX. De ondersteuningsfunctionaliteit voor conversie is net geïntroduceerd in Aspose.Slides for Java, waardoor er enkele beperkingen zijn en het het beste werkt voor eenvoudige presentaties. Het belangrijkste voordeel dat Aspose.Slides for Java biedt bij het converteren van PPT naar PPTX, is hoe eenvoudig de API te gebruiken is. Zie voorbeelden van de code in [PPT naar PPTX converteren](). Hieronder leggen de lijsten uit welke functies worden ondersteund en welke niet bij PPT‑naar‑PPTX‑conversie.

## **Ondersteunde functies**
De volgende functies worden ondersteund voor conversie:

- De structuur van masters, lay-outs en dia’s converteren.
- Grafieken converteren.
- Groepsvormen.
- Auto‑vormen converteren, inclusief rechthoeken en ellipsen. 
- Vormen met aangepaste geometrie.
- Texturen‑ en afbeeldingsvullingsstijlen voor auto‑vormen.
- Placeholders converteren.
- Lijnen en poly‑lijnen converteren.
- Lijn‑ en vulformaten.
- Verloopvullingsstijlen.
- OLE‑frames, tabellen, video‑ en audio‑frames, enz.
- Animatie‑ en diavoorstelling‑eigenschappen.
- Tekst in tekst‑frames en tekst‑containers converteren.
