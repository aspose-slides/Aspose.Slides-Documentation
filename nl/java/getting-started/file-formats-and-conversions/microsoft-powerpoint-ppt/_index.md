{{% alert color="primary" %}} 
[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) is het presentatiedocumentformaat dat kan worden aangemaakt, gelezen, bewerkt en weggeschreven door verschillende versies van Microsoft PowerPoint. Het betreft het binaire formaat voor presentatiedocumenten ontwikkeld door Microsoft.
{{% /alert %}} 

## **PPT in Aspose.Slides for Java**
Aspose.Slides for Java kan PPT‑bestanden lezen die zijn aangemaakt met de onderstaande software.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Evenzo kunnen PPT‑bestanden die met Aspose.Slides for Java zijn aangemaakt, door de bovenstaande software worden gelezen.

## **Uitgebreide ondersteuning voor PPT**
Aspose.Slides for Java biedt ondersteuning voor bijna alle functies die door het PPT‑documentformaat worden ondersteund. Het dekt niet alleen de basis‑ en geavanceerde functies die door verschillende versies van Microsoft PowerPoint worden geleverd voor het manipuleren van PPT‑documenten, maar ook functies die zelfs niet door Microsoft PowerPoint worden ondersteund. Het belangrijkste voordeel van het gebruik van de Aspose.Slides for Java API‑bibliotheek is het gebruiksgemak bij het behandelen van dergelijke functies.

Naast de basis taken met betrekking tot het aanmaken, lezen en schrijven van PPT‑documentbestanden, biedt Aspose.Slides for Java verschillende functies:

- Importeer andere Microsoft Office‑bestandsformaten als [OLE‑objecten in PPT‑documenten]().
- [Exporteer PPT‑documenten naar PDF](/slides/nl/java/convert-powerpoint-ppt-and-pptx-to-pdf/).
- Exporteer dia's in de PPT‑documenten naar SVG‑formaten.
- Render dia's naar elk afbeeldingsformaat dat door het Java‑framework wordt ondersteund.
- Stel de afmetingen van dia's in PPT‑documenten in.
- Beheer animaties op vormen.
- Beheer diavoorstellingen.
- [Formatteer tekst op dia's]().
- Extraheer tekst uit PPT‑documenten.
- [Verwerk tabellen op dia's]().
- Kopieer automatisch masters met behulp van [de kloonfunctie]().
