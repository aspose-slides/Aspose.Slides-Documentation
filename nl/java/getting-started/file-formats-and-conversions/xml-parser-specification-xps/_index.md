{{% alert color="primary" %}}

De [XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) is een paginabeschrijvingstaal en een vast-documentformaat dat oorspronkelijk door Microsoft is ontwikkeld. Net als PDF is XPS ontworpen om de documentgetrouwheid te behouden en een apparaatonafhankelijke weergave van documenten te bieden.

{{% /alert %}}

## **XPS in Aspose.Slides voor Java**
Elk presentatiedocument dat door Aspose.Slides for Java kan worden geladen, kan naar XPS‑formaat worden geconverteerd. Aspose.Slides for Java maakt gebruik van de high-fidelity paginalay-out‑ en renderengine om output in het vaste‑layout XPS‑documentformaat te produceren.
U kunt meer te weten komen over het exporteren van presentatiedocumenten naar XPS‑documenten via Aspose.Slides for Java in [Omzetten naar XPS](https://docs.aspose.com/slides/nl/java/convert-powerpoint-to-xps/).
