Er is een nieuwe [Aspose.Slides for Java API](/slides/nl/java/) uitgebracht en nu ondersteunt dit enkele product de mogelijkheid om PowerPoint‑documenten vanaf nul te genereren en bestaande documenten te bewerken.

## **Ondersteuning voor legacy‑code**
Om de legacy‑code te kunnen gebruiken die ontwikkeld is met Aspose.Slides for Java versies ouder dan 13.x, moet je enkele kleine aanpassingen in je code doen en zal de code werken zoals voorheen. Alle klassen die aanwezig waren in het oude Aspose.Slides for Java onder de namespaces Aspose.Slide en Aspose.Slides.Pptx zijn nu samengevoegd in één Aspose.Slides namespace. Bekijk de volgende eenvoudige code‑fragment voor het maken van een Hello World‑presentatiedocument in de legacy Aspose.Slides API en volg de stappen die beschrijven hoe je migreert naar de nieuwe samengevoegde API.
