Een nieuwe [Aspose.Slides for Java API](https://docs.aspose.com/slides/nl/java/) is uitgebracht en nu ondersteunt dit enkele product de mogelijkheid om PowerPoint‑documenten vanaf nul te genereren en bestaande te bewerken.

{{% /alert %}} 
## **Ondersteuning voor legacy-code**
Om de legacy‑code te gebruiken die ontwikkeld is met Aspose.Slides for Java versies eerder dan 14.x.x, moet je enkele kleine wijzigingen in je code aanbrengen en zal de code net als eerder werken. Alle klassen die aanwezig waren in de oude Aspose.Slides for Java onder de com.aspose.slides en com.aspose.slides.pptx namespaces zijn nu samengevoegd in één com.aspose.slides namespace.
