Een veelvoorkomende taak die ontwikkelaars moeten uitvoeren, is dynamisch tekst toevoegen aan dia’s. In dit artikel vind je code‑voorbeelden voor het dynamisch toevoegen van tekst met [VSTO](/slides/nl/java/adding-text-dynamically-using-vsto-and-aspose-slides-for-java/) en [Aspose.Slides for Java](/slides/nl/java/adding-text-dynamically-using-vsto-and-aspose-slides-for-java/).

## **Tekst dynamisch toevoegen**
Beide methoden volgen deze stappen:

1. Maak een presentatie.  
2. Voeg een lege dia toe.  
3. Voeg een tekstvak toe.  
4. Stel wat tekst in.  
5. Schrijf de presentatie weg.
