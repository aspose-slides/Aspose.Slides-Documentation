Aspose.Slides for Java 14.3.0 (14.4.0) biedt een volledig nieuwe implementatie van PPT‑verwerking. Er zijn veel verbeteringen, gedeeltelijke conversie van PPTX naar PPT. Maar er zijn enkele niet‑geïmplementeerde functies:

- Sommige vormen hebben een onjuiste geometrie in geserialiseerde PPT‑documenten (Call outs)
- Niet alle tekstopmaakfuncties van PPTX worden ondersteund bij PPT‑serialisatie
- Informatie over tekstaal en spellingsinstellingen is niet aanwezig in geserialiseerde PPT‑documenten
- Niet alle functies van PPTX‑thema’s worden ondersteund bij PPT‑serialisatie

**Er zijn enkele verschillen ten opzichte van Aspose.Slides for Java 8.6.0:**

- Er zijn bekende problemen bij OLE/ActiveX PPT‑serialisatie naar PPT

**Er zijn enkele verschillen ten opzichte van Aspose.Slides for .NET 14.3.0:**

- Ondersteuning voor het afdrukken van presentaties is momenteel niet beschikbaar in Aspose.Slides for Java
