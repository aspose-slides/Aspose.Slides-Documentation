{{% alert color="primary" %}}

Aspose.Slides for Java 14.4.0 biedt nieuwe mogelijkheden voor het verwerken van PowerPoint-documenten. Er zijn enkele beperkingen en bekende problemen, die in komende releases zullen worden opgelost:

- Sommige vormen hebben een verkeerde geometrie in geserialiseerde PPT-documenten (boog, cirkelvormige pijl, bijschriften).
- Niet alle PPTX-tekstopmaakfuncties worden ondersteund bij PPT-serialisatie (tabulatie, inspringing en alinea-opmaakbeperkingen).
- Informatie over tekstaal en spellinginstellingen ontbreekt in geserialiseerde PPT-documenten.
- Niet alle PPTX-thema-functies worden ondersteund bij PPT-serialisatie (alleen serialisatie van opvul-formaten, lijn-formaten en lettertype).
- Er zijn bekende problemen bij OLE/ActiveX PPT-serialisatie naar PPT.
- WordArt-serialisatie en weergave worden niet ondersteund.

{{% /alert %}}
