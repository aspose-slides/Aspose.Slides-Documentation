{{% alert color="primary" %}} 

Aspose.Slides voor Java is een class library die Java-applicaties in staat stelt presentatiedocumenten te lezen en te schrijven zonder Microsoft PowerPoint® of andere programma's te gebruiken. Aspose.Slides voor Java maakt het mogelijk tekst, vormen, tabellen & animaties te beheren, audio en video aan dia's toe te voegen, dia's te previewen, dia's te exporteren naar SVG, PDF-formaat en meer.

De nieuwste API-referentie is te vinden op [Aspose.Slides for Java API-referentie](https://reference.aspose.com/slides/nl/java).

{{% /alert %}}
