{{% alert color="primary" %}} 

Aspose.Slides for Android via Java is een klassebibliotheek die Java‑toepassingen voor Android in staat stelt presentatiedocumenten te lezen en te schrijven zonder Microsoft PowerPoint® of andere software te gebruiken. Aspose.Slides for Android via Java stelt u in staat tekst, vormen, tabellen en animaties te beheren; audio en video toe te voegen aan dia's; dia's te bekijken; en dia's te exporteren naar SVG, PDF‑formaat en meer.

De nieuwste API-referentie is te vinden op [Aspose.Slides for Android via Java API-referentie](https://reference.aspose.com/slides/nl/androidjava/).
