**Welkom bij Aspose.Slides for Android via Java**

Aspose.Slides for Android via Java is een Microsoft PowerPoint®-beheers-API die Android-toepassingen in staat stelt PowerPoint®-documenten te lezen en te schrijven zonder Microsoft PowerPoint®. Aspose.Slides for Android via Java is het eerste en enige component dat de functionaliteit biedt die nodig is om PowerPoint®-documenten op Android-platformen te beheren. Aspose.Slides for Android biedt tal van belangrijke functies waarmee u tekst, vormen, tabellen & animaties, audio- en video-bestanden in dia's kunt toevoegen, wijzigen en manipuleren.
