## **Ondersteunde platforms**
Aspose.Slides for Node.js via .NET is een platformonafhankelijke API. Het kan op elk platform (Windows, Linux, macOS, enz.) worden gebruikt waar Node.js 7 of hoger is geïnstalleerd. Oracle JDK 8 of hoger is een andere vereiste.

## **Bestandsformaten en conversies**
Aspose.Slides for Node.js via .NET ondersteunt de meeste PowerPoint‑documentformaten. Het stelt u ook in staat ze te exporteren naar de populaire bestandsformaten die organisaties gebruiken en onderling delen. Bekijk de volgende details:

|**Functie**|**Beschrijving**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via .NET biedt de snelste verwerking voor dit presentatiedocumentformaat.|
|PPT to PPTX conversion|Aspose.Slides for Node.js via .NET ondersteunt conversie van PPT naar PPTX.|
|Portable Document Format (PDF)|U kunt alle ondersteunde bestandsformaten met één methode exporteren naar Adobe Portable Document Format (PDF)-documenten.|
|XML Parser Specification (XPS)|U kunt alle ondersteunde bestandsformaten met één methode exporteren naar XML Parser Specification (XPS)-documenten.|
|Tagged Image File Format (TIFF)|U kunt alle ondersteunde presentatiedocumentformaten exporteren naar Tagged Image File Format (TIFF).|
|PPTX To HTML Conversion|Aspose.Slides for Node.js via .NET ondersteunt de conversie van PresentationEx naar HTML-indeling.|

Aspose.Slides for Node.js via .NET ondersteunt accurate weergave van dia's in een presentatie naar verschillende grafische formaten. Bekijk de volgende details:

|**Functie**|**Beschrijving**|
| :- | :- |
|.NET Supported Image Formats|Met Aspose.Slides for Node.js via .NET kunt u presentatiedia's en afbeeldingen op dia's renderen naar alle door .NET ondersteunde grafische formaten zoals TIFF, PNG, BMP, JPEG, GIF.|
|SVG Format|Aspose.Slides for Node.js via .NET biedt bovendien ingebouwde methoden waarmee u presentatiedia's kunt exporteren naar Scalable Vector Graphics (SVG)-formaten.|
|Presentation Printing|De nieuwste versies van Aspose.Slides for Node.js via .NET bieden ingebouwde afdrukmethoden met verschillende opties.|

## **Inhoudsfuncties**
Aspose.Slides for Node.js via .NET stelt u in staat bijna alle items of inhoud van presentatiedocumenten te benaderen, wijzigen of creëren. Bekijk de volgende details:

|**Functie**|**Beschrijving**|
| :- | :- |

|Normal Slides|Met Aspose.Slides for Node.js via .NET kunt u nieuwe dia's van verschillende types creëren; u kunt ook bestaande dia's in de presentaties benaderen en wijzigen.|
|Cloning / Copying Slides|Er zijn ingebouwde methoden van Aspose.Slides for Node.js via .NET die u in staat stellen bestaande dia's in een presentatie te klonen of te kopiëren. U kunt gekopieerde en gekloonde dia's van de ene presentatie naar de andere gebruiken. Omdat een dia zijn lay-out van de masterdia erft, kopiëren de ingebouwde kloonmethoden de master automatisch bij het klonen.|
|Managing Slides sections|Methoden om dia's in verschillende secties binnen een presentatie te organiseren.|
|Place Holders and Text Holders|U kunt de plaatsaanduidingen en tekstaanduidingen in een dia benaderen. Bovendien kunt u een dia met tekstaanduidingen vanaf nul creëren met de juiste methode.|

|Text Boxes|Tekstvakken kunnen vanaf het begin worden gemaakt. U kunt bestaande tekstvakken benaderen. U kunt ook hun teksten wijzigen zonder het oorspronkelijke tekstformaat te verliezen.|
|Rectangle Shapes|U kunt rechthoekige vormen creëren of wijzigen met Aspose.Slides for Node.js via .NET.|
|Poly Line Shapes|U kunt polylijnvormen creëren of wijzigen met Aspose.Slides for Node.js via .NET.|
