Aspose.Slides for Node.js via .NET is een klassikale bibliotheek die Node.js‑toepassingen in staat stelt presentatie‑documenten te lezen en te schrijven zonder Microsoft PowerPoint® of andere software te gebruiken. Aspose.Slides for Node.js via .NET stelt je in staat tekst, vormen, tabellen en animaties te beheren, audio en video aan dia’s toe te voegen, dia’s te bekijken, dia’s te exporteren naar SVG, PDF‑formaat, en nog veel meer.

Aspose.Slides for Node.js via .NET gebruikt dezelfde API‑referentie als Aspose.Slides .NET([same API reference as Aspose.Slides .NET](https://reference.aspose.com/slides/nl/nodejs-net/)).

{{% /alert %}}
