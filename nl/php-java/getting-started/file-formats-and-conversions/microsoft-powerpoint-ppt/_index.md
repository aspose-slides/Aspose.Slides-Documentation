{{% alert color="primary" %}} 

[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) is het presentatie‑documentbestandsformaat dat kan worden gemaakt, gelezen, bewerkt en geschreven door verschillende versies van Microsoft PowerPoint. Het is het binaire formaat voor presentatiedocumenten ontwikkeld door Microsoft.

{{% /alert %}} 

## **PPT in Aspose.Slides voor PHP via Java**
Aspose.Slides for PHP via Java kan PPT‑bestanden lezen die zijn gemaakt met de onderstaande software.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Evenzo kunnen PPT‑bestanden die zijn gemaakt door Aspose.Slides for PHP via Java gelezen worden door de bovengenoemde software.

## **Uitgebreide ondersteuning voor PPT**
Aspose.Slides for PHP via Java biedt ondersteuning voor bijna alle functies die door het PPT‑documentformaat worden ondersteund. Het dekt niet alleen de basis‑ en geavanceerde functies die verschillende versies van Microsoft PowerPoint bieden voor PPT‑documentbewerkingen, maar ook functies die zelfs niet door Microsoft PowerPoint worden ondersteund. Het belangrijkste voordeel van het gebruik van de Aspose.Slides for PHP via Java‑API‑bibliotheek is het gebruiksgemak bij het afhandelen van dergelijke functies.

Naast de basis taken met betrekking tot het maken, lezen en schrijven van PPT‑documentbestanden, biedt Aspose.Slides for PHP via Java verschillende extra functies:

- Importeer andere Microsoft Office‑bestandsformaten als [OLE‑objecten in PPT‑documenten](/slides/nl/php-java/manage-ole/).
- [Exporteer PPT‑documenten naar PDF](/slides/nl/php-java/convert-powerpoint-to-pdf/).
- Exporteer dia's in de PPT‑documenten naar SVG‑formaten.
- Render dia's naar elk afbeeldingsformaat dat wordt ondersteund door het Java‑framework.
- Stel de grootte van dia's in PPT‑documenten in.
- Beheer animaties op vormen.
- Beheer diavoorstellingen.
- [Formatteer tekst op dia's](/slides/nl/php-java/text-formatting/).
- Extraheer tekst uit PPT‑documenten.
- [Verwerk tabellen op dia's](/slides/nl/php-java/powerpoint-table/).
- Kopieer masters automatisch met behulp van [de kloon‑functie](/slides/nl/php-java/clone-slides/).
