## **Ondersteunde functies bij conversie**
Aspose.Slides for PHP via Java biedt gedeeltelijke ondersteuning voor het converteren van het PPT‑bestandsformaat naar PPTX. Ondersteuning voor conversie is onlangs geïntroduceerd in Aspose.Slides for PHP via Java, waardoor er enkele beperkingen zijn en het het beste werkt voor eenvoudige presentaties. Het belangrijkste voordeel dat Aspose.Slides for PHP via Java biedt bij het converteren van PPT naar PPTX, is hoe eenvoudig de API te gebruiken is. Lees meer over [PPT naar PPTX converteren](). Hieronder leggen lijsten uit welke functies ondersteund worden en welke niet bij PPT‑naar‑PPTX‑conversie.

## **Ondersteunde functies**
The following features are supported for conversion:

- Converteren van de structuur van masters, lay-outs en dia's.
- Converteren van grafieken.
- Groepsvormen.
- Converteren van autovormen, inclusief rechthoeken en ellipsen. 
- Vormen met aangepaste geometrie.
- Textuur- en afbeeldingenvullingsstijlen voor autovormen.
- Converteren van placeholder‑objecten.
- Converteren van lijnen en polylijnen.
- Lijn- en vulformaten.
- Gradientvullingsstijlen.
- OLE‑frames, tabellen, video‑ en audio‑frames, enzovoort.
- Animatie‑ en diavoorstellingseigenschappen.
- Converteren van tekst in tekst‑frames en tekst‑containers.
