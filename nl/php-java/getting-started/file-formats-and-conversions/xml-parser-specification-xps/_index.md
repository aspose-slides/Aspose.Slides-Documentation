{{% alert color="primary" %}} 

De [XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) is een paginabeschrijvingstaal en een vaste‑documentindeling die oorspronkelijk door Microsoft is ontwikkeld. Net als PDF is XPS ontworpen om de getrouwe weergave van documenten te behouden en een apparaatonafhankelijke weergave van het document te bieden. 

{{% /alert %}} 

## **XPS in Aspose.Slides for PHP via Java**
Elk presentatiedocument dat kan worden geladen door Aspose.Slides for PHP via Java, kan worden omgezet naar het XPS‑formaat. Aspose.Slides for PHP via Java gebruikt de high‑fidelity paginalay‑out‑ en renderengine om output te produceren in een vaste‑lay‑out XPS‑documentindeling.  
U kunt meer te weten komen over het exporteren van presentatiedocumenten naar XPS‑documenten via Aspose.Slides for PHP via Java in [Converting to XPS](https://docs.aspose.com/slides/nl/php-java/convert-powerpoint-to-xps/).
