{{% alert color="primary" %}} 

Het [Portable Document Format](https://en.wikipedia.org/wiki/PDF) is een bestandstype dat door Adobe Systems is gecreëerd voor het uitwisselen van documenten tussen organisaties. Het doel van het formaat was om de inhoud en lay‑out hetzelfde te houden, ongeacht op welk platform het werd bekeken. Aspose.Slides for PHP via Java stelt u in staat presentaties te converteren naar PDF.

{{% /alert %}} 

## **PDF in Aspose.Slides for PHP via Java**
Elke presentatie die kan worden geladen in Aspose.Slides for PHP via Java kan worden geconverteerd naar een PDF die voldoet aan [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) of [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA), afhankelijk van uw keuze. Aspose.Slides for PHP via Java exporteert presentaties naar PDF en in de meeste gevallen ziet de gegenereerde PDF er precies uit als de oorspronkelijke presentatie.

Aspose.Slides ondersteunt de volgende presentatie‑eigenschappen bij het converteren naar PDF:

- Afbeeldingen, tekstvakken en andere vormen.
- Tekst en opmaak.
- Alinea’s en opmaak.
- Hyperlinks.
- Koppen en voetteksten.
- Opsommingstekens.
- Tabellen.

U kunt presentaties rechtstreeks naar PDF exporteren met Aspose.Slides for PHP via Java: u heeft geen andere component nodig. Bovendien kunt u de export van presentatie naar PDF aanpassen met verschillende opties zoals beschreven in [Converting to PDF](/slides/nl/php-java/convert-powerpoint-to-pdf/).
