## **Inleiding**

Aspose.Slides for PHP via Java vereist geen enkel extern product, zoals Microsoft PowerPoint, dat geïnstalleerd moet zijn. Aspose.Slides zelf is een engine voor het maken, wijzigen, converteren en renderen van documenten in verschillende formaten, inclusief de Microsoft PowerPoint-presentatieformaten.

## **FAQ**

**Moet ik Microsoft PowerPoint geïnstalleerd hebben voor conversies en rendering?**

Nee, PowerPoint is niet vereist; Aspose.Slides is een zelfstandige engine voor [aanmaken](/slides/nl/php-java/create-presentation/), wijzigen, [converteren](/slides/nl/php-java/convert-presentation/) en [renderen](/slides/nl/php-java/convert-powerpoint-to-png/) van presentaties.

**Welke lettertypen zijn nodig voor correcte weergave?**

In de praktijk moeten de lettertypen die in de presentatie worden gebruikt of passende [substituten](/slides/nl/php-java/font-substitution/) beschikbaar zijn. Om consistente weergave op Linux/macOS te garanderen, is het aan te raden algemene lettertype‑pakketten te installeren.

**Waarom wordt een aangepast lettertype op Linux weergegeven als een fallback of ontbrekende tekst?**

Als het lettertypebestand inconsistenties of corrupte naam‑tabel‑items bevat, kan de Linux‑lettertype‑matching‑stack (FreeType/fontconfig) een ongeldig record selecteren, waardoor het lettertype niet wordt herkend. Het gebruik van een lettertype‑versie met gecorrigeerde naam‑tabel‑records of het installeren van een consistent vervangend lettertype lost het probleem op.
