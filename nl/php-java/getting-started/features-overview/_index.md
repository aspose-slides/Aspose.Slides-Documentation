## **Ondersteunde platformen**
Aspose.Slides for PHP via Java is a platform independent API. It can be used on any platform (Windows, Linux, MacOS, etc.) where PHP 7 or higher is installed. Oracle JDK 7 or higher is another requirement.

## **Bestandsformaten en conversies**
Aspose.Slides for PHP via Java supports most PowerPoint document formats. It also lets you export them to the popular file formats organizations use and share among themselves. Go through these details:

|**Functie**|**Beschrijving**|

## **FAQ**

**Moet ik Microsoft PowerPoint installeren op de server/pc zodat de bibliotheek werkt?**

Nee. PowerPoint is niet vereist; Aspose.Slides is een zelfstandige engine voor het maken, bewerken, converteren en renderen van presentaties.

**Hoe werkt multithreading? Kan verwerking worden geparalleliseerd?**

Het is veilig om verschillende documenten in verschillende threads te verwerken; hetzelfde [presentatie](https://reference.aspose.com/slides/nl/php-java/aspose.slides/presentation/)‑object mag niet door [meerdere threads](/slides/nl/php-java/multithreading/) tegelijk worden gebruikt.

**Worden bestandswachtwoorden en encryptie ondersteund?**

Ja. [U kunt](/slides/nl/php-java/password-protected-presentation/) versleutelde presentaties openen, een open‑ en schrijf‑wachtwoord instellen of verwijderen, en de beschermingsstatus controleren.

**Moet ik rekening houden met lettertypepakketten in Linux‑containers?**

Ja. Het wordt aanbevolen om algemene lettertypepakketten te installeren en/of expliciet [lettertype‑mappen op te geven](/slides/nl/php-java/custom-font/) in uw applicatie om onverwachte vervangingen te voorkomen.

**Zijn er beperkingen in de evaluatie‑versie?**

In [evaluatiemodus](/slides/nl/php-java/licensing/) wordt een watermerk aan de output toegevoegd en gelden bepaalde beperkingen; een [30‑daagse tijdelijke licentie](https://purchase.aspose.com/temporary-license/) is beschikbaar voor volledige functionaliteitstesten.

**Wordt het importeren van externe formaten in een presentatie (PDF/HTML → PPTX) ondersteund?**

Ja. U kunt [PDF‑pagina's en HTML‑inhoud](/slides/nl/php-java/import-presentation/) aan een presentatie toevoegen, waardoor ze worden omgezet naar dia's.
