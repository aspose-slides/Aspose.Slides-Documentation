## **Overzicht**

Dit artikel beschrijft een licentie‑fout die zich voordoet in Aspose.Slides for PHP via Java versies 23.2, 23.3, 23.4, 23.5, 23.6 en 23.7. Deze fout zorgde ervoor dat eeuwigdurende licenties tijdelijk niet bruikbaar waren nadat hun abonnementsperiode was verlopen.

## **Symptomen**

Nadat uw eeuwigdurende licentie‑abonnement is verlopen, kan Aspose.Slides for PHP via Java versies 23.2‑23.7 de licentie als verlopen melden, waardoor alle functies in een evaluatiemodus gaan.  
Dit gedrag is onjuist en treft geen versies vóór 23.2 en na 23.7.

## **Oplossing**

Aspose.Slides for PHP via Java heeft dit probleem aangepakt en bijgewerkte versies (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) uitgebracht met de oplossing.

Als u een van de getroffen versies van Aspose.Slides for PHP via Java in uw project gebruikt, schakel dan over naar de gecorrigeerde versie.

- Controleer de versie die u gebruikt en zorg ervoor dat deze niet getroffen wordt door de bovenstaande fout.
- Raadpleeg de [Aspose‑documentatie](https://docs.aspose.com/slides/nl/php-java/getting-started/) voor stappen voor probleemoplossing en bekende licentie‑problemen.

## **Betrokken producten en versies**

Let op: deze fout heeft alleen invloed op Aspose.Slides for PHP via Java versies 23.2 tot en met 23.7. **Andere Aspose‑producten en andere versies van Aspose.Slides worden niet geraakt**.
