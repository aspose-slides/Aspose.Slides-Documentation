{{% alert color="primary" %}} 

Aspose.Slides for PHP via Java is een class library die PHP-applicaties in staat stelt presentatiedocumenten te lezen en te schrijven zonder Microsoft PowerPoint® of andere programma's te gebruiken. Aspose.Slides for PHP via Java maakt het mogelijk om tekst, vormen, tabellen en animaties te beheren, audio en video aan dia's toe te voegen, dia's te previewen, dia's te exporteren naar SVG, PDF-formaat en meer.

De nieuwste API-referentie is te vinden op [Aspose.Slides for PHP via Java API Reference](https://reference.aspose.com/slides/nl/php-java/).
