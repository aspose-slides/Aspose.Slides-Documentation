{{% alert color="primary" %}} 

Aspose.Slides for Python via .NET is een class library die Python-toepassingen in staat stelt presentatiedocumenten te lezen en te schrijven zonder Microsoft PowerPoint® of andere software te gebruiken. Met Aspose.Slides for Python via .NET kunt u tekst, vormen, tabellen en animaties beheren; audio en video aan dia's toevoegen; een voorbeeld van dia's bekijken; en dia's exporteren naar SVG, PDF-formaat en meer.

De nieuwste API-referentie is te vinden op [Aspose.Slides for Python via .NET API Reference](https://reference.aspose.com/slides/nl/python-net/).
