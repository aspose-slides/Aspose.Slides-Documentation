## **Ondersteunde platforms**
De platforms Aspose.Slides for Python via .NET kunnen worden gebruikt op Windows x64 of x86 en een breed scala aan Linux‑distributies met Python 3.5 of later geïnstalleerd. Er zijn extra vereisten voor het doel‑Linux‑platform:
- GCC‑6 runtime‑bibliotheken (of nieuwer)
- Afhankelijkheden van .NET Core Runtime. Het installeren van .NET Core Runtime zelf is NIET vereist
- Voor Python 3.5‑3.7: de `pymalloc`‑build van Python is nodig. De Python‑build‑optie `--with-pymalloc` is standaard ingeschakeld. Meestal wordt de `pymalloc`‑build van Python gemarkeerd met de `m`‑achtervoeging in de bestandsnaam.
- `libpython` gedeelde Python‑bibliotheek. De Python‑build‑optie `--enable-shared` is standaard uitgeschakeld; sommige Python‑distributies bevatten de `libpython` gedeelde bibliotheek niet. Voor sommige Linux‑platforms kan de `libpython` bibliotheek via de pakketbeheerder worden geïnstalleerd, bijvoorbeeld: `sudo apt-get install libpython3.7`. Het veelvoorkomende probleem is dat de `libpython` bibliotheek op een andere locatie dan de standaard systeemlocatie voor gedeelde bibliotheken wordt geïnstalleerd. Het probleem kan worden verholpen door de Python‑build‑opties te gebruiken om alternatieve bibliotheekpaden in te stellen bij het compileren van Python, of door een symbolische link naar het `libpython`‑bibliotheekbestand te maken in de standaard systeemlocatie voor gedeelde bibliotheken. Typisch is de bestandsnaam van de `libpython` gedeelde bibliotheek `libpythonX.Ym.so.1.0` voor Python 3.5‑3.7, of `libpythonX.Y.so.1.0` voor Python 3.8 of later (bijvoorbeeld: `libpython3.7m.so.1.0`, `libpython3.9.so.1.0`).

Als u ondersteuning voor meer platforms nodig heeft, kijk dan naar de “tweelingbroer”-producten Aspose.Slides for .NET of Aspose.Slides for Java.

## **FAQ**

**Moet ik Microsoft PowerPoint op de server/PC installeren voor dat de bibliotheek werkt?**

Nee. PowerPoint is niet vereist; Aspose.Slides is een zelfstandige engine voor het maken, bewerken, converteren en renderen van presentaties.

**Hoe werkt multithreading? Kan verwerking worden geparallelliseerd?**

Het is veilig om verschillende documenten in verschillende threads te verwerken; hetzelfde [presentation](https://reference.aspose.com/slides/nl/python-net/aspose.slides/presentation/) object mag niet gelijktijdig door [multiple threads](/slides/nl/python-net/multithreading/) worden gebruikt.

**Worden bestandswachtwoorden en versleuteling ondersteund?**

Ja. [U kunt](/slides/nl/python-net/password-protected-presentation/) versleutelde presentaties openen, een open‑ en schrijf‑wachtwoord instellen of verwijderen, en de beschermingsstatus controleren.

**Moet ik rekening houden met lettertype‑pakketten in Linux‑containers?**

Ja. Het wordt aanbevolen om veelvoorkomende lettertype‑pakketten te installeren en/of expliciet [lettertype‑mappen op te geven](/slides/nl/python-net/custom-font/) in uw applicatie om onverwachte substituties te voorkomen.

**Zijn er beperkingen in de evaluatieversie?**

In [evaluatiemodus](/slides/nl/python-net/licensing/) wordt een watermerk aan de uitvoer toegevoegd en gelden bepaalde beperkingen; een [30‑daagse tijdelijke licentie](https://purchase.aspose.com/temporary-license/) is beschikbaar voor volledige functionaliteitstesten.

**Wordt het importeren van externe formaten in een presentatie (PDF/HTML → PPTX) ondersteund?**

Ja. U kunt [PDF‑pagina’s en HTML‑inhoud](/slides/nl/python-net/import-presentation/) aan een presentatie toevoegen, waardoor ze in dia's worden omgezet.
