{{% alert color="primary" %}} 

Aspose.Slides for .NET is een class library die .NET‑applicaties in staat stelt presentatiedocumenten te lezen en te schrijven zonder Microsoft PowerPoint® of andere programma’s te gebruiken. Aspose.Slides for .NET maakt het mogelijk om tekst, vormen, tabellen en animaties te beheren, audio en video aan dia’s toe te voegen, dia’s te bekijken, dia’s te exporteren naar SVG, PDF‑formaat en meer.

De nieuwste API‑referentie is te vinden op [Aspose.Slides for .NET API Reference](https://reference.aspose.com/slides/nl/net).
