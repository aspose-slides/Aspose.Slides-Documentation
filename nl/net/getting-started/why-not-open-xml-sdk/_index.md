## **Overzicht**

Dit artikel legt uit wanneer ontwikkelaars mogelijk kiezen voor Open XML SDK of Aspose.Slides voor het werken met presentatiedocumenten. Het beschrijft Open XML SDK als een bibliotheek voor het manipuleren van OOXML‑pakketten en hun onderliggende XML‑elementen, terwijl Aspose.Slides wordt gepresenteerd als een presentatie‑verwerkingsbibliotheek met een hoog‑niveau objectmodel en ondersteuning voor vele PowerPoint‑gerelateerde taken.

Het artikel vergelijkt beide opties op basis van ondersteunde formaten, programmeermodel, render‑ en afdrukmogelijkheden, platformondersteuning en veelvoorkomende use‑cases. Het verduidelijkt tevens dat Open XML SDK geschikt kan zijn voor basis‑PPTX‑bewerkingen of directe toegang tot OOXML‑elementen, terwijl Aspose.Slides meer geschikt is voor complexe presentatietaken zoals werken met meerdere PowerPoint‑formaten, het kopiëren of klonen van vormen, het vervangen van tekst, het toepassen van animaties en het converteren van presentaties naar PDF, TIFF of XPS.

## **Wat is Open XML SDK?**
Soms krijgen we de vraag: *Waarom zouden we Aspose‑producten gebruiken in plaats van de gratis Open XML SDK?*

We vinden het makkelijk om deze vraag te beantwoorden aan de hand van functies en mogelijkheden.

Volgens de [MSDN Library](https://docs.microsoft.com/en-us/office/open-xml/open-xml-sdk) wordt Open XML SDK als volgt gedefinieerd:

> "The Open XML SDK 2.0 simplifies the task of manipulating Open XML packages and the underlying Open XML schema elements within a package. The Open XML SDK 2.0 encapsulates many common tasks that developers perform on Open XML packages, so that you can perform complex operations with just a few lines of code. OOXML documents are essentially zipped XML files and Open XML SDK is a collection of classes that allows you to work with the content of OOXML documents in a strongly-typed way. That is instead of unzipping a file to extract XML, loading that XML into a DOM tree, and working with XML elements and attributes directly, Open XML SDK provides classes to do that."

## **Wat is Aspose.Slides?**
Aspose.Slides is een class library die toepassingen in staat stelt de volgende presentatie‑verwerkingstaken uit te voeren:

- Programmeren met een presentatie‑objectmodel.
- Hoogwaardige conversies met alle populaire ondersteunde PowerPoint‑presentatieformaten, inclusief conversie naar PDF, XPS, TIFF en afdrukken.
- Genereren van dia‑miniaturen in bekende formaten zoals PNG, JPEG en BMP naast export naar SVG.
- Presentaties van de grond af opbouwen of door elementen uit één of meerdere documenten te combineren.
- Animaties, OLE‑frames, tabellen, grafieken toevoegen en beheren.
- Uitgebreide controle over en beheer van tekstopmaak in TextFrames, Paragraphs en Portions.

Voor meer details over de beschikbare functies, zie de [Aspose.Slides Features](/slides/nl/net/product-overview/) pagina.

## **Conclusie**
Open XML SDK en Aspose.Slides concurreren niet direct omdat ze aanzienlijk verschillende behoeften vervullen en zich op verschillende doelgroepen richten.

{{% alert color="primary" %}} 

Open XML SDK is een class library die een sterk getypeerde manier biedt om met OOXML‑documenten te werken, terwijl Aspose.Slides een extreem nuttige presentatie‑verwerkingsbibliotheek is die uitstekende ondersteuning biedt voor bijna alle Microsoft PowerPoint‑bestandsformaten. 

{{% /alert %}} 

Als uw workflow een basis‑programmeertaken op een PPTX‑document betreft, kan Open XML SDK een goede keuze zijn. Met Open XML SDK moet u in staat zijn eenvoudige taken uit te voeren, zoals het genereren van een eenvoudig PPTX‑document of het verwijderen van opmerkingen, kop‑/voetteksten, het extraheren van afbeeldingen, enzovoort. Bepaalde taken kunnen worden uitgevoerd met Open XML SDK maar niet met Aspose.Slides. Bijvoorbeeld, als u direct toegang nodig heeft tot de XML‑elementen en attributen van een OOXML‑document, moet u Open XML SDK gebruiken.

Als u complexe taken op documenten moet uitvoeren — zoals de taken in de lijst hieronder — dan is Aspose.Slides uw beste optie.

- Operaties met oudere PowerPoint‑formaten (en ook PPTX).
- Vormen binnen dia's kopiëren of klonen op een manier die objecten, stijlen en andere opmaakelementen op gepaste wijze combineert.
- Opmaak‑ of onopgemaakte tekst vervangen.
- Animaties toepassen en connectoren gebruiken met vormen.
- Een document converteren naar PDF, TIFF of XPS zodat het eruitziet alsof Microsoft PowerPoint de conversie heeft uitgevoerd.
- Een .NET‑ of Java‑applicatie ontwikkelen in zowel desktop‑ als web‑omgevingen.
