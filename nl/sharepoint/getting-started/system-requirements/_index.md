## **Serververeisten**
{{% alert color="primary" %}} 

Om Aspose.Slides for SharePoint te installeren, is een systeem dat een van de volgende draait vereist:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft Office SharePoint Server 2010
- Microsoft Office SharePoint Server 2013
- Microsoft Office SharePoint Server 2016
- Microsoft Office SharePoint Server 2019

{{% /alert %}} {{% alert color="primary" %}} 

Het installeren van Service Pack 2 of hoger voor deze producten wordt aanbevolen.

{{% /alert %}} 
## **Clientvereisten**
{{% alert color="primary" %}} 

SharePoint-sites worden benaderd via een webbrowser. Aspose.Slides for SharePoint is getest met de volgende browsers:

- Microsoft Internet Explorer 6, 7, 8
- Mozilla FireFox 3.5

{{% /alert %}}
