Een bibliotheek is een contentstructuur die bestanden bevat. Een bibliotheek kan slechts één type bestand bevatten of vele verschillende types, waaronder documenten, spreadsheets en presentaties. Een SharePoint‑bibliotheek creëert één locatie waar mensen documenten kunnen uploaden en openen. Documenten worden opgeslagen in binair formaat in de SQL‑Server‑database die door de SharePoint‑site wordt gebruikt. De documenten zijn beschikbaar voor andere SharePoint‑gebruikers als een lijst met hyperlinks.
