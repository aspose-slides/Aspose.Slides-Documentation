{{% alert color="primary" %}} 

JasperReports en JasperServer hebben geen ingebouwde mogelijkheden om rapporten te exporteren als Microsoft PowerPoint‑presentaties, maar met Aspose.Slides for JasperReports krijg je toegang tot extra exportformaten:

- Microsoft PowerPoint‑presentatie (PPT)
- Microsoft PowerPoint‑presentatie (PPTX)
- HTML
- PDF

{{% /alert %}} 

Om documenten in deze formaten te maken, maakt Aspose.Slides for JasperReports gebruik van een ingebouwde versie van [Aspose.Slides for Java](https://products.aspose.com/slides/nl/java/), de marktleidende bibliotheek voor het verwerken van presentaties van Aspose. Microsoft PowerPoint wordt niet gebruikt om de documenten te genereren.
