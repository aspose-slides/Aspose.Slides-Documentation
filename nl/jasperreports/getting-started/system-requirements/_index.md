Aspose.Slides for JasperReports werkt met:

**Aspose.Slides for JasperReports 17.5 en eerdere versies:**

- Java 2 Platform Standard Edition 1.4 of hoger.
- JasperReports 2.0, 3.0 of hoger.

**Aspose.Slides for JasperReports 17.6 en nieuwere versies:**

- Java 2 Platform Standard Edition 6.0 of hoger.
- JasperReports 3.7.2 of hoger.
