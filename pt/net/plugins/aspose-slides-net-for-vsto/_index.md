## **O que é VSTO?**
VSTO, ou Visual Studio Tools for Office, ou Microsoft Visual Studio Tools, faz parte do conjunto de ferramentas Visual Studio .NET da Microsoft e oferece suporte ao Microsoft .NET Framework. É usado por desenvolvedores para escrever código que conecta os aplicativos Microsoft Office Word, Excel e Outlook. Desenvolvedores que utilizam VSTO podem empregar as linguagens C# ou Visual Basic. É uma alternativa ao Visual Basic for Applications, ou VBA.

## **Aspose.Slides for .NET**
Aspose.Slides for .NET é um componente exclusivo de gerenciamento de PowerPoint® que permite que aplicações .NET leiam, escrevam e manipulem documentos PowerPoint sem usar o Microsoft PowerPoint.

Aspose.Slides for .NET é o primeiro e único componente .NET que fornece funcionalidade para gerenciar documentos PowerPoint dentro de suas próprias aplicações. Assim como todos os componentes Aspose .NET, Aspose.Slides é escrito em C# gerenciado, tem um preço incrível e é extremamente rápido.

## **Introdução**
Os desenvolvedores que buscam bibliotecas de formatos de arquivo podem precisar analisar os recursos fornecidos pelas bibliotecas disponíveis no mercado. Principalmente, dois fatores são considerados ao escolher uma opção:

- Recursos
- Esforço necessário para usar a biblioteca

Se você é um desenvolvedor .NET em busca de uma biblioteca de formatos de arquivo mais rápida, fácil e leve para processar arquivos, pode ser necessário comparar VSTO (Visual Studio Tools for Office) com Aspose. ## **Requisitos de Sistema e Plataformas Compatíveis**
### **Requisitos de Sistema**
Uma ou mais das seguintes edições do Visual Studio Tools for Office:

- Microsoft Visual Studio 2005 Tools for the Microsoft Office System
- Microsoft Visual Studio 2005 Tools for the 2007 Microsoft Office System
- Visual Studio 2008 Professional Edition
- Visual Studio 2008 Team Suite Edition
- Uma versão do Microsoft Office:
- Microsoft Office Professional 2003 SP1
- 2007 Microsoft Office system
### **Sistemas Operacionais Compatíveis**
Windows 2000; Windows Server 2003; Windows Vista; Windows XP
### **Versões do .NET Framework Compatíveis**
apenas .NET Framework 2.0 ou superior.
