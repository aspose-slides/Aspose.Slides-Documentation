## **O Que é OpenXML?**
OpenXML SDK é um SDK .NET da Microsoft para processar formatos de arquivo do Microsoft Office 2007 e posteriores. Esses formatos de arquivo são basicamente pacotes zipados e o Open XML SDK for Office simplifica a tarefa de manipular pacotes Open XML e os elementos do esquema Open XML subjacentes dentro de um pacote. As classes no Open XML SDK encapsulam muitas tarefas comuns que os desenvolvedores realizam em pacotes Open XML, permitindo que você execute operações complexas com poucas linhas de código.

Depois de instalar o Open XML SDK 2.5, abra seu projeto ou aplicação existente no Visual Studio, ou crie um novo projeto ou aplicação. Em seguida, no seu projeto ou aplicação, adicione referências aos seguintes componentes:

- DocumentFormat.OpenXml
- WindowsBase

## **Aspose.Slides para .NET**
Aspose.Slides para .NET é um componente exclusivo de gerenciamento de PowerPoint® que permite que aplicações .NET leiam, escrevam e manipulem documentos PowerPoint sem usar o Microsoft PowerPoint.

Aspose.Slides para .NET é o primeiro e único componente .NET que fornece funcionalidades para gerenciar documentos PowerPoint dentro de suas próprias aplicações. Como todos os componentes Aspose .NET, Aspose.Slides é escrito em C# gerenciado, possui preço incrível e é extremamente rápido.

## **Introdução**
Os desenvolvedores que buscam bibliotecas de formatos de arquivo podem precisar analisar os recursos oferecidos pelas bibliotecas disponíveis no mercado. Principalmente, dois fatores são considerados ao escolher uma opção:

- Recursos
- Esforço necessário para usar a biblioteca

Se você é um desenvolvedor .NET procurando uma biblioteca de formatos de arquivo mais rápida, fácil e leve para processar arquivos, pode ser necessário comparar VSTO (Visual Studio tool for Office) com Aspose. Nesse caso, este projeto ajudará você a comparar recursos e código em ambas as bibliotecas.
