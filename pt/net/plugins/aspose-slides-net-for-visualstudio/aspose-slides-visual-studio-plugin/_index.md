## **Introdução**
Microsoft Visual Studio é um ambiente de desenvolvimento integrado (IDE) da Microsoft. É usado para desenvolver programas de computador para Microsoft Windows, bem como sites, aplicativos web e serviços web. O Visual Studio utiliza plataformas de desenvolvimento da Microsoft como Windows API, Windows Forms, Windows Presentation Foundation, Windows Store e Microsoft Silverlight. Ele pode produzir código nativo e código gerenciado.
### **Aspose.Slides para .NET**
Aspose.Slides para .NET é uma API única de gerenciamento de PowerPoint® que permite que aplicações .NET leiam, escrevam e manipulem documentos PowerPoint sem usar o Microsoft PowerPoint.

Aspose.Slides para .NET é a primeira e única API PowerPoint para .NET que fornece funcionalidade para gerenciar documentos PowerPoint dentro das suas próprias aplicações. Como todos os componentes Aspose .NET, Aspose.Slides é escrito em C# gerenciado, tem preço incrível e é extremamente rápido.

Para mais detalhes sobre [API Aspose.Slides para .NET](http://www.aspose.com/.net/powerpoint-component.aspx)
### **Explore Exemplos do Aspose.Slides para .NET Usando o Visual Studio**
Aspose Visual Studio Plugin - **Explore Exemplos do Aspose.Slides para .NET** é uma ótima ferramenta para baixar rapidamente e explorar exemplos da API Aspose.Slides para .NET. Ela economiza muito tempo e esforço ao fornecer uma opção muito simples para selecionar, baixar e abrir os projetos de exemplo mais recentes sem precisar sair do Visual Studio.

Esta versão oferece os seguintes recursos:

- Suporta Visual Studio 2010, Visual Studio 2012 e Visual Studio 2013.
- Fácil de iniciar a partir do menu **File > Aspose** no Visual Studio
- Economiza tempo e reduz a curva de aprendizado.
- Permite selecionar e abrir os exemplos mais recentes da API Aspose.Slides para .NET hospedados no Github.com.
- Permite abrir projetos de exemplo em **C#** ou **VB.NET**
- O projeto selecionado é aberto automaticamente no Visual Studio com todas as referências necessárias configuradas, oferecendo um ambiente pronto para execução e exploração.
- Faz download automaticamente das bibliotecas mais recentes da API a partir do **NuGet** e dos exemplos do **GitHub**.

## **Requisitos de Sistema e Plataformas Suportadas**
Para instalar e usar o Aspose Visual Studio Plugin você precisa ter uma das seguintes versões do Visual Studio instaladas

- Visual Studio 2010, 2012, 2013

Sinta‑se à vontade para nos contatar se encontrar algum problema ao instalar ou usar este plugin.

## **Instalação**
É muito simples instalar o Aspose Visual Studio Plugin, siga estes passos:

- Clique duas vezes no arquivo msi baixado
- Siga as etapas, basta pressionar **Next** em cada tela para instalá‑lo usando as configurações padrão.

Um diálogo de sucesso será exibido quando a instalação for concluída.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_2)

**Nota:** Certifique‑se de reiniciar o Visual Studio para que as alterações tenham efeito.
## **Uso**
Depois de instalado, você pode iniciar e usar o plugin interativo conforme descrito abaixo:
**1.** Abra o plugin selecionando **File > Aspose** seguido de **New Aspose.Slides Example Project** no Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_3)

**2.** Na primeira vez que abrir o plugin, será exibida a caixa de diálogo de seleção da API como mostrada abaixo. Levará alguns minutos para baixar a API mais recente e suas bibliotecas do **NuGet** e o código‑fonte dos exemplos mais recentes do **GitHub**.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_4)

**3.** Na caixa de diálogo **Select an API to View Examples**, selecione ou altere o local onde os projetos de exemplo serão criados e também selecione ou altere o tipo de projeto de exemplo como **C#** ou **VB.NET** e clique em **Continue** para abrir o projeto de exemplo no Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_1)

**4.** Novos projetos de exemplos conforme a seleção de tipo de projeto **C#** ou **VB.NET** no Visual Studio.

#### **Visão geral do código‑fonte**
Existem dois projetos na solução

- **AsposeVisualStudioPlugin** – Contém a integração com o Visual Studio, seleção e download automático das APIs Aspose e dos exemplos.
- **Installer** – Projeto Wix para gerar o arquivo msi instalador do plugin.
