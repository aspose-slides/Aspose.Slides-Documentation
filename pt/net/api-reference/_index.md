{{% alert color="primary" %}} 

Aspose.Slides for .NET é uma biblioteca de classes que permite que aplicações .NET leiam e escrevam documentos de apresentação sem usar o Microsoft PowerPoint® ou outros. Aspose.Slides for .NET permite gerenciar texto, formas, tabelas e animações, adicionar áudio e vídeo aos slides, visualizar slides, exportar slides para SVG, formato PDF e muito mais.

A referência da API mais recente pode ser encontrada em [Aspose.Slides for .NET API Reference](https://reference.aspose.com/slides/pt/net).
