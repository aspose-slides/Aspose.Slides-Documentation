{{% alert color="primary" %}} 

Aspose.Slides for Python via .NET é uma biblioteca de classes que permite que aplicativos Python leiam e escrevam documentos de apresentação sem usar o Microsoft PowerPoint® ou outro software. Aspose.Slides for Python via .NET permite gerenciar texto, formas, tabelas e animações; adicionar áudio e vídeo aos slides; visualizar slides; e exportar slides para SVG, formato PDF e muito mais.

Referência de API mais recente pode ser encontrada em [Referência de API do Aspose.Slides for Python via .NET](https://reference.aspose.com/slides/pt/python-net/).
