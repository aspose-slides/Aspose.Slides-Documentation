## **Plataformas Suportadas**
Aspose.Slides for Python via Java é uma API independente de plataforma. Pode ser usada em qualquer sistema (Windows, Linux, macOS, etc.) onde o Python 3.7 ou superior esteja instalado. JDK 8 ou superior é outro requisito.

## **Formatos de Arquivo e Conversões**
Aspose.Slides for Python via Java suporta a maioria dos formatos de documentos PowerPoint. Também permite exportá-los para os formatos de arquivo populares que as organizações utilizam e compartilham entre si. Veja os detalhes abaixo:

|**Recurso**|**Descrição**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Python via Java oferece o processamento mais rápido para este formato de documento de apresentação.|
|PPT to PPTX conversion|Aspose.Slides for Python via Java suporta conversão de PPT para PPTX|
|Portable Document Format (PDF)|Você pode exportar todos os formatos de arquivo suportados para documentos Adobe Portable Document Format (PDF) com um único método.|
|XML Parser Specification (XPS)|Você pode exportar todos os formatos de arquivo suportados para documentos XML Parser Specification (XPS) com um único método.|
|Tagged Image File Format (TIFF)|Você pode exportar todos os formatos de arquivo de apresentação suportados para Tagged Image File Format (TIFF).|
|PPTX To HTML Conversion|Aspose.Slides for Python via Java suporta a conversão de PresentationEx para o formato HTML.|

## **Renderização e Impressão**
Aspose.Slides for Python via Java suporta renderização de alta fidelidade dos slides de uma apresentação para diferentes formatos gráficos. Veja os detalhes abaixo:

|**Recurso**|**Descrição**|
| :- | :- |
|Java Supported Image Formats|Usando Aspose.Slides for Python via Java, você pode renderizar slides de apresentação e imagens nos slides para todos os formatos gráficos suportados pelo Java, como TIFF, PNG, BMP, JPEG, GIF.|
|SVG Format|Aspose.Slides for Python via Java também fornece métodos internos que permitem exportar slides de apresentação para formatos Scalable Vector Graphics (SVG).|
|Presentation Printing|As versões mais recentes de Aspose.Slides for Python via Java fornecem métodos de impressão internos com diferentes opções.|

## **Recursos de Conteúdo**
Aspose.Slides for Python via Java permite acessar, modificar ou criar quase todos os itens ou conteúdos de documentos de apresentação. Veja os detalhes abaixo:

|**Recurso**|**Descrição**|
| :- | :- |
|Master Slides|Os Master Slides definem o layout dos slides normais. Aspose.Slides for Python via Java permite acessar e modificar os Master Slides de documentos de apresentação.|
|Normal Slides|Com Aspose.Slides for Python via Java, você pode criar novos slides de diferentes tipos; também pode acessar e modificar slides existentes nas apresentações.|
|Cloning / Copying Slides|Existem métodos internos fornecidos por Aspose.Slides for Python via Java que permitem clonar ou copiar slides existentes dentro de uma apresentação. Você também pode usar slides copiados e clonados de uma apresentação para outra. Como um slide herda seu layout do master slide, os métodos internos de clonagem copiam automaticamente o master ao clonar.|
|Managing Slides sections|Métodos para organizar slides em diferentes seções dentro de uma apresentação.|
|Place Holders and Text Holders|Você pode acessar os placeholders e text holders em um slide. Além disso, pode criar um slide com text holders do zero usando o método apropriado.|
|Header and Footers|Aspose.Slides for Python via Java facilita o manuseio de cabeçalhos/rodapés nos slides.|
|Notes in Slides|Com Aspose.Slides for Python via Java, você pode acessar e modificar notas associadas a um slide e também adicionar novas notas.|
|Finding a Shape|Você também pode encontrar uma forma específica em um slide usando o texto alternativo associado à forma.|
|Backgrounds|Aspose.Slides for Python via Java permite trabalhar com planos de fundo associados a um master ou slide normal em uma apresentação.|
|Text Boxes|Caixas de texto podem ser criadas do zero. Você pode acessar caixas de texto existentes. Também pode modificar seus textos sem perder o formato original.|
|Rectangle Shapes|Você pode criar ou modificar formas retangulares com Aspose.Slides for Python via Java.|
|Poly Line Shapes|Você pode criar ou modificar formas de polilinha com Aspose.Slides for Python via Java.|
|Ellipse Shapes|Você pode criar ou modificar formas de elipse com Aspose.Slides for Python via Java.|
|Group Shapes|Aspose.Slides for Python via Java suporta formas agrupadas.|
|Auto Shapes|Aspose.Slides for Python via Java suporta auto shapes.|
|SmartArt|Aspose.Slides for Python via Java oferece suporte a formas SmartArt no MS PowerPoint.|
|Charts|Aspose.Slides for Python via Java oferece suporte a gráficos MSO no PowerPoint.|
|Shapes Serialization|Aspose.Slides for Python via Java suporta um grande número de formas. Quando o Aspose.Slides for Python via Java não oferece suporte a uma forma, você pode usar um método de serialização que permite serializar essa forma a partir de um slide existente. Dessa forma, você pode reutilizar a forma conforme suas necessidades.|
|Picture Frames|Você pode gerenciar imagens em quadros de imagem com Aspose.Slides for Python via Java.|
|Audio Frames|Você pode vincular ou incorporar arquivos de áudio em quadros de áudio nos slides com Aspose.Slides for Python via Java.|
|Video Frames|Você pode manipular arquivos de vídeo em quadros de vídeo. Aspose.Slides for Python via Java também oferece suporte a vídeos vinculados e incorporados.|
|OLE Frame|Você pode gerenciar objetos OLE em quadros OLE com Aspose.Slides for Python via Java.|
|Tables|Aspose.Slides for Python via Java suporta tabelas em slides.|
|ActiveX Controls|Suporte a controles ActiveX.|
|VBA Macros|Suporte ao gerenciamento de macros VBA dentro de apresentações.|
|Text Frame|Você pode acessar o texto de qualquer forma através do quadro de texto associado a essa forma.|
|Text Scanning|Você pode escanear texto em uma apresentação, ao nível da apresentação ou do slide, através de métodos de varredura internos.|
|Animations|Você pode aplicar animações em formas.|
|Slide Shows|Aspose.Slides for Python via Java suporta apresentações de slides e transições de slides.|

## **Recursos de Formatação**
Com Aspose.Slides for Python via Java, você pode formatar textos e formas nos slides de apresentações. Veja os detalhes abaixo:

|**Recurso**|**Descrição**|
| :- | :- |
|Text Formatting|<p>No Aspose.Slides for Python via Java, você pode gerenciar textos através dos quadros de texto associados às formas. Dessa forma, você pode formatar textos usando os parágrafos e as porções associadas aos quadros de texto. Esses elementos de texto podem ser formatados com o Aspose.Slides for Python via Java.</p><p>- Tipo de Fonte</p><p>- Tamanho da Fonte</p><p>- Cor da Fonte</p><p>- Tons da Fonte</p><p>- Alinhamento de Parágrafo</p><p>- Marcadores de Parágrafo</p><p>- Orientação de Parágrafo</p>|
|Shape Formatting|<p>No Aspose.Slides for Python via Java, o elemento básico de um slide é uma forma. Você pode formatar esses elementos de forma com o Aspose.Slides for Python via Java:</p><p>- Posição</p><p>- Tamanho</p><p>- Linha</p><p>- Preenchimento (incluindo Padrão, Gradiente, Sólido)</p><p>- Texto</p><p>- Imagem</p>|
