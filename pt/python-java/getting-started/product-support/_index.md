## **Suporte técnico**
{{% alert color="primary" %}}

A Aspose fornece suporte técnico gratuito e ilimitado para todos os seus produtos. O suporte está disponível para todos os usuários (incluindo usuários com pacotes de avaliação).

{{% /alert %}} 

Quando precisar de ajuda com Aspose.Slides for Python via Java, considere o seguinte:

- O principal canal de suporte é o [Aspose.Forums](https://forum.aspose.com/c/slides/pt/11). Poste sua pergunta no fórum Aspose.Slides—ela será respondida em poucas horas. A equipe de suporte do Aspose.Slides responde diretamente às perguntas postadas no fórum.
- Observe que a Aspose não fornece suporte técnico por telefone. O suporte telefônico está disponível apenas para questões de vendas e compra.
- Ao aguardar uma resposta nos fóruns, seja paciente e leve em conta as diferenças de fuso horário.


Quando você encontrar um problema com Aspose.Slides for Python via Java, siga estas diretrizes para resolvê‑lo da maneira mais eficiente:

- Verifique e confirme que você está usando a versão mais recente do Aspose.Slides for Python via Java antes de relatar o problema. Talvez o problema que você encontrou já tenha sido resolvido em uma atualização. 

  Consulte [Aspose.Slides download](https://releases.aspose.com/slides/pt/python-java/) para obter informações sobre a versão mais recente.

- Antes de relatar um problema, você pode querer consultar nossos fóruns, esta documentação e a Referência da API. Talvez a resposta para sua pergunta já tenha sido fornecida. 

- Ao relatar um problema, inclua o documento original e (se possível) um fragmento do seu código relacionado ao problema. Se precisar anexar vários arquivos, coloque‑os em um pacote ZIP. Você pode anexar seus documentos com segurança no Aspose.Forums, pois **apenas** você e os desenvolvedores da Aspose têm acesso aos arquivos anexados.

- Crie e use um único tópico para um problema específico. Se você encontrar outro problema, deverá criar outro tópico para relatá‑lo.
