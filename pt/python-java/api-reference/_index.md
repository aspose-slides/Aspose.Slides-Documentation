Aspose.Slides for Python via Java é uma biblioteca de classes que permite que aplicativos Python leiam e gravem documentos de apresentação sem usar o Microsoft PowerPoint® ou outros. Aspose.Slides for Python via Java permite gerenciar texto, formas, tabelas e animações, adicionar áudio e vídeo aos slides, visualizar slides, exportar slides para SVG, formato PDF e fazer muito mais.

Aspose.Slides for Python via Java usa a [referência de API igual à do Aspose.Slides para Java](https://reference.aspose.com/slides/pt/python-java/).
