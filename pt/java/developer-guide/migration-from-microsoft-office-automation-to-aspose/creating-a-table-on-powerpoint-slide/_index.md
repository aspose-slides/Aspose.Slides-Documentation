As tabelas são amplamente usadas para exibir dados em slides de apresentação. Este artigo mostra como criar programaticamente uma tabela 15 x 15 com tamanho de fonte 10 usando primeiro [VSTO 2008](/slides/pt/java/creating-a-table-on-powerpoint-slide/) e depois [Aspose.Slides for Java](/slides/pt/java/creating-a-table-on-powerpoint-slide/).

{{% /alert %}} 
## **Criando Tabelas**
### **Exemplo VSTO 2008**
Os passos a seguir adicionam uma tabela a um slide do Microsoft PowerPoint usando VSTO:

1. Criar uma apresentação.  
1. Adicionar um slide vazio à apresentação.  
1. Adicionar uma tabela de 15 x 15 ao slide.  
1. Adicionar texto a cada célula da tabela com tamanho de fonte 10.  
1. Salvar a apresentação no disco.
