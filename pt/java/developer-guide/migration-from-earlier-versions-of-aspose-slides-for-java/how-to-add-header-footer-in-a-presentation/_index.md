## **Suporte a Código Legado**
Para usar o código legado desenvolvido com versões do Aspose.Slides for Java anteriores à 13.x, você precisa fazer algumas alterações menores no seu código e ele funcionará como antes. Todas as classes que estavam presentes no antigo Aspose.Slides for Java nos namespaces Aspose.Slide e Aspose.Slides.Pptx agora foram mescladas em um único namespace Aspose.Slides. Por favor, examine o seguinte trecho de código simples para adicionar cabeçalho e rodapé em uma apresentação na API legacy do Aspose.Slides e siga os passos que descrevem como migrar para a nova API mesclada.
