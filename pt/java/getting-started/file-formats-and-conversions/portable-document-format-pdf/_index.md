O [Formato de Documento Portátil](https://en.wikipedia.org/wiki/PDF) é um formato de arquivo criado pela Adobe Systems para troca de documentos entre organizações. O objetivo do formato era manter o conteúdo e o layout iguais, independentemente da plataforma em que fosse visualizado. Aspose.Slides for Java permite converter arquivos de apresentação para PDF.

{{% /alert %}} 

## **PDF no Aspose.Slides for Java**
Qualquer apresentação que possa ser carregada no Aspose.Slides for Java pode ser convertida para um PDF que segue o [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) ou [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA), dependendo da sua escolha. Aspose.Slides for Java exporta apresentações para PDF e, na maioria dos casos, o PDF resultante tem exatamente a mesma aparência da apresentação original.

Aspose.Slides suporta os seguintes recursos de apresentação ao converter para PDF:

- Imagens, caixas de texto e outras formas.
- Texto e formatação.
- Parágrafos e formatação.
- Hiperenlaces.
- Cabeçalhos e rodapés.
- Marcadores.
- Tabelas.

Você pode exportar apresentações para PDFs diretamente usando Aspose.Slides for Java: não é necessário nenhum outro componente. Além disso, pode personalizar a exportação de apresentação para PDF com várias opções, conforme explicado em [Convertendo para PDF](/slides/pt/java/converting-a-presentation/).
