## **Sobre TIFF**

O [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) que é conhecido por armazenar múltiplas imagens em um único documento foi originalmente criado pela Aldus. Esse formato é amplamente suportado por scanners, fax e outras aplicações de manipulação de imagens.

## **TIFF no Aspose.Slides for Java**

Qualquer documento que possa ser carregado no Aspose.Slides for Java também pode ser convertido diretamente em documento TIFF pelo Aspose.Slides for Java, eliminando a necessidade de qualquer componente de terceiros. Além disso, você pode definir opcionalmente o tamanho das imagens no documento TIFF resultante. Você pode encontrar informações sobre a exportação de documentos de apresentação para documentos TIFF através do Aspose.Slides for Java em [este tópico](/slides/pt/java/converting-a-presentation/).
