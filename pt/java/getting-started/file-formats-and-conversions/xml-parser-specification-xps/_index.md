A [Especificação do Analizador XML](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) é uma linguagem de descrição de página e um formato de documento fixo originalmente desenvolvido pela Microsoft. Assim como o PDF, o XPS foi projetado para preservar a fidelidade do documento e fornecer uma aparência independente de dispositivo. 

{{% /alert %}} 

## **XPS no Aspose.Slides for Java**
Qualquer documento de apresentação que possa ser carregado pelo Aspose.Slides for Java pode ser convertido para o formato XPS. O Aspose.Slides for Java usa o mecanismo de layout de página e renderização de alta fidelidade para produzir saída no formato de documento XPS de layout fixo.  
Você pode aprender sobre a exportação de documentos de apresentação para documentos XPS através do Aspose.Slides for Java em [Convertendo para XPS](https://docs.aspose.com/slides/pt/java/convert-powerpoint-to-xps/).
