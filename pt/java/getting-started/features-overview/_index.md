## **Plataformas Suportadas**
Aspose.Slides for Java suporta as plataformas de desenvolvimento e implantação mais populares.

|**Recurso**|**Descrição**|
| :- | :- |
|Aplicativos Desktop|Aspose.Slides for Java pode ser usado para desenvolver aplicações Windows Forms|
|Aplicações Web Corporativas|Usar Aspose.Slides for Java ajuda a criar aplicações web direcionadas. O suporte ao uso de Aspose.Slides for Java com PHP também é fornecido.|
|Linux/Unix|Aspose.Slides for Java é uma API independente de plataforma e pode funcionar em ambientes Linux e Unix.|

## **Formatos de Arquivo e Conversões**
Aspose.Slides for Java suporta a maioria dos formatos de documento do Microsoft PowerPoint e os exporta para formatos populares amplamente usados por organizações.

|**Recurso**|**Descrição**|
| :- | :- |

|[PresentationML (PPTX, XML)](/slides/pt/java/presentationml-pptx-xml/)|Aspose.Slides for Java suporta o processamento do formato de apresentação OOXML (também conhecido como PresentationML ou PPTX).|
|[PPT to PPTX conversion](/slides/pt/java/ppt-to-pptx-conversion/)|Aspose.Slides for Java suporta a conversão de PPT para PPTX.|
|[Portable Document Format (PDF)](/slides/pt/java/developer-guide/)|Os formatos de arquivo suportados podem ser exportados para documentos Adobe Portable Document Format (PDF) com um único método.|
|[XML Parser Specification (XPS)](/slides/pt/java/xml-parser-specification-xps/)|Todos os formatos de arquivo suportados podem ser exportados para documentos XML Parser Specification (XPS) com um único método.|
|[Tagged Image File Format (TIFF)](https://docs.aspose.com/slides/pt/java/convert-powerpoint-to-tiff/)|Os formatos de arquivo de apresentação suportados por Aspose.Slides for Java também podem ser exportados para Tagged Image File Format (TIFF).|
|[ODP to PPTX Conversion](https://docs.aspose.com/slides/pt/java/convert-odp-to-pptx/)|Aspose.Slides for Java suporta o carregamento de OpenDocument Presentation (ODP) e a conversão para PPTX.|
|[Presentation To HTML Conversion](https://docs.aspose.com/slides/pt/java/convert-powerpoint-to-html/)|Aspose.Slides for Java suporta a conversão de PresentationEx para o formato HTML.|

## **FAQ**

**Preciso instalar o Microsoft PowerPoint no servidor/PC para que a biblioteca funcione?**

Não. O PowerPoint não é necessário; Aspose.Slides é um motor autônomo para criar, editar, converter e renderizar apresentações.

**Como o multithreading funciona? O processamento pode ser paralelizado?**

É seguro processar documentos diferentes em threads diferentes; o mesmo objeto de [apresentação](https://reference.aspose.com/slides/pt/java/com.aspose.slides/presentation/) não deve ser usado por [múltiplas threads](/slides/pt/java/multithreading/) ao mesmo tempo.

**Senhas de arquivo e criptografia são suportadas?**

Sim. [Você pode](/slides/pt/java/password-protected-presentation/) abrir apresentações criptografadas, definir ou remover uma senha de abertura e gravação, e verificar o status da proteção.

**Preciso me preocupar com pacotes de fontes em contêineres Linux?**

Sim. É recomendado instalar pacotes de fontes comuns e/ou especificar explicitamente [diretórios de fontes](/slides/pt/java/custom-font/) em sua aplicação para evitar substituições inesperadas.

**Existem limitações na versão de avaliação?**

No [modo de avaliação](/slides/pt/java/licensing/), uma marca d'água é adicionada à saída e certas limitações se aplicam; uma [licença temporária de 30 dias](https://purchase.aspose.com/temporary-license/) está disponível para teste com todos os recursos.

**A importação de formatos externos para uma apresentação (PDF/HTML → PPTX) é suportada?**

Sim. Você pode adicionar [páginas PDF e conteúdo HTML](/slides/pt/java/import-presentation/) a uma apresentação, transformando-as em slides.
