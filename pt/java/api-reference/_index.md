Aspose.Slides for Java é uma biblioteca de classes que permite que aplicações Java leiam e escrevam documentos de apresentação sem usar o Microsoft PowerPoint® ou outros. Aspose.Slides for Java permite gerenciar texto, formas, tabelas & animações, adicionar áudio e vídeo aos slides, visualizar slides, exportar slides para SVG, formato PDF e muito mais.

A referência mais recente da API pode ser encontrada em [Referência da API do Aspose.Slides for Java](https://reference.aspose.com/slides/pt/java).
