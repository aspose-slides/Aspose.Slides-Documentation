{{% alert color="primary" %}}
Aspose.Slides for PHP via Java é uma biblioteca de classes que permite que aplicações PHP leiam e escrevam documentos de apresentação sem usar o Microsoft PowerPoint® ou outros. Aspose.Slides for PHP via Java permite gerenciar texto, formas, tabelas e animações, adicionar áudio e vídeo aos slides, visualizar slides, exportar slides para SVG, PDF e mais.

A referência mais recente da API pode ser encontrada em [Aspose.Slides for PHP via Java API Reference](https://reference.aspose.com/slides/pt/php-java/).
