## **Visão geral**

Este artigo explica como obter suporte para Aspose.Slides e como relatar problemas de forma eficaz. Descreve o canal principal de suporte, fornece recomendações para enviar um problema e aponta recursos técnicos úteis, como a página do produto, blog, downloads, fórum de suporte, documentação e Referência da API. O artigo também responde a perguntas comuns sobre suporte pago, escalonamento de incidentes e suporte a versões mais antigas da biblioteca.

## **Suporte técnico**

{{% alert color="primary" %}} 

A Aspose oferece suporte técnico ilimitado e gratuito para todos os seus produtos. O suporte está disponível para todos os usuários, inclusive avaliação.

{{% /alert %}} 

Se precisar de ajuda com Aspose.Slides for PHP via Java, considere o seguinte:

- O principal canal de suporte é o Aspose.Forums. Publique sua pergunta no fórum Aspose.Slides e ela será respondida em algumas horas. As questões são respondidas diretamente pela equipe de suporte do Aspose.Slides.
- Note que a Aspose não oferece suporte técnico por telefone. O suporte telefônico está disponível apenas para perguntas de vendas e compra.
- Ao aguardar uma resposta nos fóruns, considere as diferenças de fuso horário.

Se você tem um problema com Aspose.Slides for PHP via Java, siga estas etapas simples para garantir que ele seja resolvido da maneira mais eficiente:

- Certifique‑se de usar a versão mais recente do Aspose.Slides for PHP via Java antes de relatar o problema; consulte o download do Aspose.Slides para saber a versão mais recente.
- Consulte os fóruns, esta documentação e a Referência da API antes de relatar o problema; talvez sua pergunta já tenha sido respondida.
- Ao relatar um problema, inclua o documento original e, se possível, um trecho do seu código que cause o erro. Se precisar anexar vários arquivos, compacte‑os em um único zip. É seguro anexar seus documentos no Aspose.Forums, pois somente você e os desenvolvedores da Aspose terão acesso aos arquivos anexados.
- Tente relatar um único problema por tópico. Se houver outro problema, relate‑o em um tópico separado.

## **FAQ**

**Quais níveis de suporte pago estão disponíveis e como eles diferem?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Eles diferem pelo número de incidentes incluídos, prioridade de resposta e canais de comunicação disponíveis.

**Posso escalar um bug diretamente para a equipe de engenharia?**

Sim. [Paid support](https://helpdesk.aspose.com/) permite escalonamento para os desenvolvedores quando um problema requer investigação mais aprofundada.

**Versões mais antigas da biblioteca recebem suporte e correções?**

É possível usar versões mais antigas, mas as correções são fornecidas apenas para a versão atual; patches para versões legadas não são emitidos.
