## **Plataformas Suportadas**
Aspose.Slides for PHP via Java é uma API independente de plataforma. Pode ser usada em qualquer plataforma (Windows, Linux, macOS, etc.) onde o PHP 7 ou superior esteja instalado. O Oracle JDK 7 ou superior é outro requisito.

## **Formatos de Arquivo e Conversões**
Aspose.Slides for PHP via Java suporta a maioria dos formatos de documentos do PowerPoint. Também permite exportá-los para os formatos de arquivo populares que as organizações utilizam e compartilham entre si. Confira os detalhes:

|**Recurso**|**Descrição**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for PHP via Java fornece o processamento mais rápido para este formato de documento de apresentação.|
|Conversão de PPT para PPTX|Aspose.Slides for PHP via Java suporta a conversão de PPT para PPTX|
|Portable Document Format (PDF)|Você pode exportar todos os formatos de arquivo suportados para documentos Adobe Portable Document Format (PDF) com um único método.|
|XML Parser Specification (XPS)|Você pode exportar todos os formatos de arquivo suportados para documentos XML Parser Specification (XPS) com um único método.|
|Tagged Image File Format (TIFF)|Você pode exportar todos os formatos de arquivo de apresentação suportados para Tagged Image File Format (TIFF).|
|Conversão de PPTX para HTML|Aspose.Slides for PHP via Java suporta a conversão de PresentationEx para o formato HTML.|

## **FAQ**

**Preciso instalar o Microsoft PowerPoint no servidor/PC para que a biblioteca funcione?**

Não. O PowerPoint não é necessário; Aspose.Slides é um mecanismo autônomo para criar, editar, converter e renderizar apresentações.

**Como o multithreading funciona? O processamento pode ser paralelizado?**

É seguro processar documentos diferentes em threads distintas; o mesmo objeto [apresentação](https://reference.aspose.com/slides/pt/php-java/aspose.slides/presentation/) não deve ser usado por [múltiplas threads](/slides/pt/php-java/multithreading/) ao mesmo tempo.

**Suportam-se senhas de arquivo e criptografia?**

Sim. [Você pode](/slides/pt/php-java/password-protected-presentation/) abrir apresentações criptografadas, definir ou remover uma senha de abertura e gravação, e verificar o status da proteção.

**Preciso me preocupar com pacotes de fontes em contêineres Linux?**

Sim. Recomenda-se instalar pacotes de fontes comuns e/ou especificar explicitamente [diretórios de fontes](/slides/pt/php-java/custom-font/) em sua aplicação para evitar substituições inesperadas.

**Existem limitações na versão de avaliação?**

No [modo de avaliação](/slides/pt/php-java/licensing/), uma marca d'água é adicionada à saída e certas limitações se aplicam; uma [licença temporária de 30 dias](https://purchase.aspose.com/temporary-license/) está disponível para testes com todos os recursos.

**A importação de formatos externos para uma apresentação (PDF/HTML → PPTX) é suportada?**

Sim. Você pode adicionar [páginas PDF e conteúdo HTML](/slides/pt/php-java/import-presentation/) a uma apresentação, convertendo-os em slides.
