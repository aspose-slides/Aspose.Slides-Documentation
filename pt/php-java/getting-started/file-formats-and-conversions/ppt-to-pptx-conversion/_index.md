A conversão de PPT para PPTX é suportada pelo Aspose.Slides for PHP via Java. A maioria dos recursos de apresentação - slides mestre, estrutura e assim por diante - são mantidos ao converter de um formato para o outro, mas há [algumas limitações](/slides/pt/php-java/ppt-to-pptx-conversion/).

{{% /alert %}} 
## **Recursos Compatíveis na Conversão**
Aspose.Slides for PHP via Java oferece suporte parcial para converter o formato de arquivo PPT para PPTX. O suporte à conversão foi recentemente introduzido no Aspose.Slides for PHP via Java, portanto possui algumas limitações e funciona melhor para apresentações simples. A principal vantagem que o Aspose.Slides for PHP via Java oferece ao converter PPT para PPTX é a facilidade de uso da API. Para ver exemplos de código, leia sobre [Convertendo PPT para PPTX](). Abaixo, listas explicam quais recursos são suportados e quais não são para a conversão de PPT para PPTX.

## **Recursos Compatíveis**
Os seguintes recursos são suportados na conversão:

- Conversão da estrutura de mestres, layouts e slides.
- Conversão de gráficos.
- Agrupamento de formas.
- Conversão de autoshapes, incluindo retângulos e elipses. 
- Formas com geometria personalizada.
- Texturas e estilos de preenchimento de imagens para autoshapes.
- Conversão de placeholders.
- Conversão de linhas e polilinhas.
- Formatos de linha e preenchimento.
- Estilos de preenchimento gradiente.
- Quadros OLE, tabelas, quadros de vídeo e áudio, etc.
- Propriedades de animação e apresentação de slides.
- Conversão de texto em quadros de texto e contêineres de texto.
