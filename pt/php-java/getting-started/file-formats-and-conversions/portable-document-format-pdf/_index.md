O [Portable Document Format](https://en.wikipedia.org/wiki/PDF) é um formato de arquivo criado pela Adobe Systems para a troca de documentos entre organizações. O objetivo do formato era manter o conteúdo e o layout iguais, independentemente da plataforma em que fosse visualizado. Aspose.Slides for PHP via Java permite converter arquivos de apresentação para PDF.

{{% /alert %}} 

## **PDF no Aspose.Slides for PHP via Java**
Qualquer apresentação que possa ser carregada no Aspose.Slides for PHP via Java pode ser convertida em um PDF que segue o [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) ou [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA) dependendo da sua escolha. Aspose.Slides for PHP via Java exporta apresentações para PDF e, na maioria dos casos, o PDF resultante tem a mesma aparência da apresentação original.

Aspose.Slides suporta os seguintes recursos de apresentação ao converter para PDF:

- Imagens, caixas de texto e outras formas.
- Texto e formatação.
- Parágrafos e formatação.
- Hiperlinks.
- Cabeçalhos e rodapés.
- Marcadores.
- Tabelas.

Você pode exportar apresentações para PDFs diretamente usando Aspose.Slides for PHP via Java: não precisa de nenhum outro componente. Além disso, pode personalizar a exportação de apresentação para PDF com várias opções, conforme explicado em [Convertendo para PDF](/slides/pt/php-java/convert-powerpoint-to-pdf/).
