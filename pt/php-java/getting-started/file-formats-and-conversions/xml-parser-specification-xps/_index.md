A [Especificação do Analisador XML](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) é uma linguagem de descrição de página e um formato de documento fixo originalmente desenvolvido pela Microsoft. Como o PDF, o XPS foi projetado para preservar a fidelidade do documento e fornecer aparência de documento independente de dispositivo. 

{{% /alert %}} 

## **XPS no Aspose.Slides para PHP via Java**
Qualquer documento de apresentação que possa ser carregado pelo Aspose.Slides para PHP via Java pode ser convertido para o formato XPS. O Aspose.Slides para PHP via Java usa o mecanismo de layout de página e renderização de alta fidelidade para produzir saída no formato de documento XPS de layout fixo.
Você pode aprender sobre a exportação dos documentos de apresentação para documentos XPS através do Aspose.Slides para PHP via Java em [Convertendo para XPS](https://docs.aspose.com/slides/pt/php-java/convert-powerpoint-to-xps/).
