## **Sobre TIFF**

O [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF), conhecido por armazenar múltiplas imagens em um único documento, foi criado originalmente pela Aldus. Este formato é amplamente suportado por aplicativos de digitalização, fax e outras manipulações de imagem.

## **TIFF no Aspose.Slides para PHP via Java**

Qualquer documento que possa ser carregado no Aspose.Slides para Java também pode ser convertido diretamente em documento TIFF pelo Aspose.Slides para PHP via Java, eliminando a necessidade de qualquer componente de terceiros. Além disso, você pode opcionalmente definir o tamanho das imagens no documento TIFF resultante. Você pode encontrar informações sobre a exportação de documentos de apresentação para documentos TIFF através do Aspose.Slides para PHP via Java neste [este tópico](/slides/pt/php-java/convert-powerpoint-to-tiff/).
