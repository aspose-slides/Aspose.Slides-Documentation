{{% alert color="primary" %}} 

Aspose.Slides for C++ é uma biblioteca de classes que permite que aplicações C++ leiam e escrevam documentos de apresentação sem usar o Microsoft PowerPoint® ou outros. Aspose.Slides for C++ permite gerenciar texto, formas, tabelas e animações, adicionar áudio e vídeo aos slides, visualizar slides, exportar slides para SVG, formato PDF e muito mais.

A referência mais recente da API pode ser encontrada em [Aspose.Slides for C++ API Reference](https://reference.aspose.com/slides/pt/cpp/).
