**Bem-vindo ao Aspose.Slides for C++**

Aspose.Slides for C++ é uma API de gerenciamento do Microsoft PowerPoint® que permite que aplicações C++ leiam e escrevam documentos PowerPoint® sem usar o Microsoft PowerPoint®. Aspose.Slides for C++ é o primeiro e único componente que fornece a funcionalidade de gerenciar documentos PowerPoint®. Aspose.Slides for C++ oferece muitos recursos principais, como gerenciamento de texto, formas, tabelas e animações, adição de áudio e vídeo aos slides, visualização de slides, exportação de slides para SVG, formato PDF e muito mais.
