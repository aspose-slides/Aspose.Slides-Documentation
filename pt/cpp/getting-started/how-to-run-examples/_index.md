## **Requisitos de Software**
Certifique-se de atender aos seguintes requisitos antes de baixar e executar os exemplos.

1. Visual Studio 2015 ou superior.

## **Baixar Aspose.Slides do GitHub**
Todos os exemplos do Aspose.Slides para C++ estão hospedados no [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-C).

- Você pode clonar o repositório usando seu cliente GitHub favorito ou baixar o arquivo ZIP [aqui](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/refs/heads/master.zip).
- Extraia o conteúdo do arquivo ZIP para qualquer pasta em seu computador. Todos os exemplos estão localizados na pasta **Examples**.
- Existe um arquivo de solução do Visual Studio para C++.
- Os projetos foram criados no Visual Studio 2015, mas os arquivos de solução são compatíveis com o Visual Studio 2015 e superior.
- Abra o arquivo de solução no Visual Studio e compile o projeto.
- Baixe a versão mais recente do Aspose.Slides para C++ na [Seção de Download](https://downloads.aspose.com/slides/pt/cpp).
- Extraia a versão e copie os diretórios **Include** e **Bin** para dentro da pasta **Examples**.
- Abra o arquivo **main.cpp**, todos os exemplos são chamados a partir dele.
- Descomente os exemplos que deseja executar dentro do projeto.
- Compile e construa a solução

Sinta-se à vontade para nos contatar através de nossos Fóruns se tiver algum problema ao configurar ou executar os exemplos.

## **Contribuir**
Se você gostaria de adicionar ou melhorar um exemplo, incentivamos a contribuir com o projeto. Todos os exemplos e projetos de demonstração neste repositório são de código aberto e podem ser usados livremente em suas próprias aplicações.

Para contribuir, você pode fazer um fork do repositório, editar o código-fonte e criar um pull request. Revisaremos as alterações e as incluiremos no repositório se forem úteis.
