{{% alert color="primary" %}} 

Todos os componentes Aspose C++ exigem o conjunto de permissões Full Trust. O motivo é que os componentes Aspose C++ precisam acessar configurações do registro, arquivos do sistema além do diretório virtual para certas operações, como análise de fontes etc. Além disso, os Componentes Aspose C++ são baseados em classes de sistema C++ centrais que também exigem o conjunto de permissões Full Trust em muitos casos. 

{{% /alert %}}
