## **Sobre o XPS**
A [Especificação do Parser XML](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) é uma linguagem de descrição de página e um formato de documento fixo originalmente desenvolvido pela Microsoft. Assim como o PDF, o XPS é um formato de documento de layout fixo projetado para preservar a fidelidade do documento e fornecer aparência de documento independente do dispositivo.

## **XPS no Aspose.Slides para C++**
Qualquer documento de apresentação que pode ser carregado pelo Aspose.Slides for C++ pode ser convertido para o formato XPS. Aspose.Slides for C++ usa o mecanismo de layout de página e renderização de alta fidelidade para produzir saída no formato de documento XPS de layout fixo.

Vale a pena mencionar que o Aspose.Slides for C++ gera XPS diretamente sem depender das classes do Windows Presentation Foundation (WPF) que são distribuídas com o .NET Framework 3.5, permitindo assim que o Aspose.Slides for C++ produza documentos XPS em máquinas que executam versões do .NET Framework anteriores à versão 3.5. Você pode aprender sobre a exportação de documentos de apresentação para documentos XPS através do Aspose.Slides for C++ em [este tópico](/slides/pt/net/).
