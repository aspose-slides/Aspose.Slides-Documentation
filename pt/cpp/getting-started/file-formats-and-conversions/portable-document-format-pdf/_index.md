## **Sobre PDF**
O [Formato de Documento Portátil](https://en.wikipedia.org/wiki/PDF) é um formato de arquivo criado pela Adobe Systems para a troca de documentos entre diferentes organizações. O objetivo deste formato era possibilitar que o conteúdo dos documentos fosse representado de forma que sua aparência visual não dependesse da plataforma em que estivesse sendo visualizado.

## **PDF no Aspose.Slides para C++**
Qualquer documento de apresentação que possa ser carregado no Aspose.Slides para C++ pode ser convertido em um documento PDF que pode estar em conformidade com [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A) ou [PDF /A-1b](https://en.wikipedia.org/wiki/PDF/A), dependendo da sua escolha. O Aspose.Slides para C++ exporta os documentos de apresentação para PDF de modo que, na maioria das vezes, o documento PDF exportado se assemelha quase ao documento de apresentação original. A solução Aspose oferece suporte aos seguintes recursos dos documentos de apresentação ao converter para documentos PDF:

- Imagens, caixas de texto e outras formas
- Texto e formatação
- Parágrafos e formatação
- Hipervínculos
- Cabeçalhos e rodapés
- Marcadores
- Tabelas

Você pode exportar os documentos de apresentação para documentos PDF diretamente usando apenas o componente Aspose.Slides para C++. Ou seja, você não precisa de nenhum outro componente de terceiros ou do Aspose.Pdf para isso. Além disso, é possível personalizar a exportação de apresentação para PDF com diferentes opções, conforme explicado neste[tópico](http://docs.aspose.com/display/slidesnet/Converting+to+PDF+File).
