## **Formatos de Arquivo e Conversões**
Aspose.Slides for C++ oferece suporte à maioria dos formatos de documentos PowerPoint e também os exporta para formatos populares amplamente usados para intercâmbio entre organizações. A tabela a seguir enumera esses formatos:

|**Recurso**|**Descrição**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/pt/cpp/microsoft-powerpoint-ppt/)|Aspose.Slides for C++ fornece o processamento mais rápido para este formato de documento de apresentação.|
|[Portable Document Format (PDF)](/slides/pt/cpp/portable-document-format-pdf/)|Os formatos de arquivo suportados podem ser exportados para documentos Adobe Portable Document Format (PDF) com um único método.|
|[XML Parser Specification (XPS)](/slides/pt/cpp/xml-parser-specification-xps/)|Todos os formatos de arquivo suportados podem ser exportados para documentos XML Parser Specification (XPS) com um único método.|
|[PresentationML (PPTX, XML)](/slides/pt/cpp/presentationml-pptx-xml/)|Aspose.Slides for C++ oferece suporte ao processamento do formato de apresentação OOXML, também conhecido como PresentationML ou PPTX.|
|[PPTX To HTML Conversion](https://docs.aspose.com/slides/pt/cpp/convert-powerpoint-to-html/)|Aspose.Slides for C++ oferece suporte à conversão de apresentações para o formato HTML.|

## **Renderização**
Aspose.Slides for C++ oferece renderização de alta fidelidade dos slides nos documentos de apresentação para vários formatos gráficos, conforme listado abaixo:

|**Recurso**|**Descrição**|
| :- | :- |
|Formatos de Imagem Compatíveis com C++|Usando Aspose.Slides for C++, você poderá renderizar não apenas slides de apresentação, mas também imagens nos slides para todos os formatos gráficos compatíveis com C++, como PNG, BMP e JPEG.|
|Formato SVG|Aspose.Slides for C++ também fornece um método interno para exportar slides de apresentação para formatos Scalable Vector Graphics (SVG).|

## **Recursos de Conteúdo**
Aspose.Slides for C++ permite acessar, modificar ou criar quase todo o conteúdo possível dos documentos de apresentação.

|**Recurso**|**Descrição**|

|Slides Mestres|Os Slides Mestres definem o layout dos slides normais. Aspose.Slides for C++ permite acessar e modificar os Slides Mestres dos documentos de apresentação.|

|Clonagem / Cópia de Slides|Existem métodos internos fornecidos por Aspose.Slides for C++ para clonar ou copiar slides existentes, não apenas dentro de uma apresentação, mas também de uma apresentação para outra. Como um slide herda seu layout do slide mestre, os métodos de clonagem internos copiam automaticamente o mestre também ao clonar.|
|Place Holders e Text Holders|Você também pode acessar os place holders e text holders em um slide. Além disso, pode criar um slide com text holders a partir do zero usando o método apropriado.|

|Anotações em Slides|Com Aspose.Slides for C++, você pode não apenas acessar e modificar notas associadas a um slide, mas também adicionar notas.|
|Encontrar uma Forma|Você também pode encontrar uma forma específica em um slide usando o texto alternativo associado à forma.|
|Fundos|Aspose.Slides for C++ permite trabalhar com o fundo associado a um slide mestre ou normal da apresentação.|
|Caixas de Texto|As caixas de texto podem ser criadas a partir do zero. As caixas de texto existentes podem ser acessadas e seu texto também pode ser modificado sem perder o formato original.|
|Formas Retangulares|Formas retangulares podem ser criadas ou modificadas por Aspose.Slides for C++.|
|Formas de Polilinha|Formas de polilinha também podem ser criadas ou modificadas por Aspose.Slides for C++.|
|Formas de Elipse|Formas de elipse também são criadas ou modificadas por Aspose.Slides for C++.|
|Formas Agrupadas|Aspose.Slides for C++ também oferece suporte a formas agrupadas.|
|Formas Automáticas|Formas automáticas também são suportadas por Aspose.Slides for C++.|

|Quadros de Imagem|As imagens são gerenciadas em quadros de imagem usando Aspose.Slides for C++.|

## **Recursos de Formatação**
É possível formatar texto e formas em slides de um documento de apresentação usando Aspose.Slides for C++.

|**Recurso**|**Descrição**|
| :- | :- |
|Formatação de Texto|<p>In Aspose.Slides for C++, o texto é gerenciado por meio de quadros de texto associados às formas. Assim, o texto pode ser formatado usando os parágrafos e trechos associados aos quadros de texto. Os seguintes elementos de texto podem ser formatados por meio do Aspose.Slides for C++.</p><p>- Tipo de Fonte</p><p>- Tamanho da Fonte</p><p>- Cor da Fonte</p><p>- Tons da Fonte</p><p>- Alinhamento de Parágrafo</p><p>- Marcadores de Parágrafo</p><p>- Orientação de Parágrafo</p>|
|Formatação de Forma|<p>In Aspose.Slides for C++, o elemento básico de um slide é a forma. Os seguintes elementos de Forma podem ser formatados usando Aspose.Slides for C++:</p><p>- Posição</p><p>- Tamanho</p><p>- Linha</p><p>- Preenchimento (incluindo Padrão, Gradiente, Sólido)</p><p>- Texto</p><p>- Imagem</p>|

## **FAQ**

**Preciso instalar o Microsoft PowerPoint no servidor/PC para que a biblioteca funcione?**

Não. O PowerPoint não é necessário; Aspose.Slides é um mecanismo autônomo para criar, editar, converter e renderizar apresentações.

**Como funciona o multithreading? O processamento pode ser paralelizado?**

É seguro processar diferentes documentos em diferentes threads; o mesmo [presentation](https://reference.aspose.com/slides/pt/cpp/aspose.slides/presentation/) objeto não deve ser usado por [múltiplas threads](/slides/pt/cpp/multithreading/) ao mesmo tempo.

**Senhas de arquivo e criptografia são suportadas?**

Sim. [Você pode](/slides/pt/cpp/password-protected-presentation/) abrir apresentações criptografadas, definir ou remover uma senha de abertura e gravação, e verificar o status de proteção.

**Preciso me preocupar com pacotes de fontes em contêineres Linux?**

Sim. É recomendado instalar pacotes de fontes comuns e/ou especificar explicitamente [diretórios de fontes](/slides/pt/cpp/custom-font/) em sua aplicação para evitar substituições inesperadas.

**Existem limitações na versão de avaliação?**

No [modo de avaliação](/slides/pt/cpp/licensing/), uma marca d'água é adicionada à saída e certas limitações se aplicam; uma [licença temporária de 30 dias](https://purchase.aspose.com/temporary-license/) está disponível para testes com todos os recursos.

**A importação de formatos externos em uma apresentação (PDF/HTML → PPTX) é suportada?**

Sim. Você pode adicionar [páginas PDF e conteúdo HTML](/slides/pt/cpp/import-presentation/) a uma apresentação, transformando-as em slides.
