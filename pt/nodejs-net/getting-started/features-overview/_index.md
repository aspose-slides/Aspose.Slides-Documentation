## **Plataformas Suportadas**
Aspose.Slides for Node.js via .NET é uma API independente de plataforma. Pode ser usada em qualquer plataforma (Windows, Linux, MacOS, etc.) onde o Node.js 7 ou superior esteja instalado. O Oracle JDK 8 ou superior é outro requisito.

## **Formatos de Arquivo e Conversões**
Aspose.Slides for Node.js via .NET suporta a maioria dos formatos de documento do PowerPoint. Também permite exportá-los para os formatos de arquivo populares que as organizações usam e compartilham entre si. Veja os detalhes abaixo:

|**Recurso**|**Descrição**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via .NET fornece o processamento mais rápido para este formato de documento de apresentação.|
|Conversão de PPT para PPTX|Aspose.Slides for Node.js via .NET suporta a conversão de PPT para PPTX|
|Formato de Documento Portátil (PDF)|Você pode exportar todos os formatos de arquivo suportados para documentos Adobe Portable Document Format (PDF) com um único método.|
|Especificação de Analisador XML (XPS)|Você pode exportar todos os formatos de arquivo suportados para documentos XML Parser Specification (XPS) com um único método.|
|Formato de Arquivo de Imagem Etiquetado (TIFF)|Você pode exportar todos os formatos de arquivo de apresentação suportados para Tagged Image File Format (TIFF).|
|Conversão de PPTX para HTML|Aspose.Slides for Node.js via .NET suporta a conversão de PresentationEx para formato HTML.|

## **Renderização e Impressão**
Aspose.Slides for Node.js via .NET suporta renderização de alta fidelidade de slides em apresentações para diferentes formatos gráficos. Veja os detalhes abaixo:

|**Recurso**|**Descrição**|
| :- | :- |
|Formatos de Imagem Compatíveis com .NET|Usando Aspose.Slides for Node.js via .NET, você pode renderizar slides de apresentação e imagens nos slides para todos os formatos gráficos compatíveis com .NET, como TIFF, PNG, BMP, JPEG, GIF.|
|Formato SVG|Aspose.Slides for Node.js via .NET também fornece métodos internos que permitem exportar slides de apresentação para formatos Scalable Vector Graphics (SVG).|
|Impressão de Apresentação|As versões mais recentes de Aspose.Slides for Node.js via .NET fornecem métodos de impressão internos com diferentes opções.|

## **Recursos de Conteúdo**
Aspose.Slides for Node.js via .NET permite acessar, modificar ou criar quase todos os itens ou conteúdos de documentos de apresentação. Veja os detalhes abaixo:

|**Recurso**|**Descrição**|
| :- | :- |
|Slides Mestre|Os Slides mestre definem o layout dos slides normais. Aspose.Slides for Node.js via .NET permite acessar e modificar os Slides Mestre de documentos de apresentação|
|Slides Normais|Com Aspose.Slides for Node.js via .NET, você pode criar novos slides de diferentes tipos; também pode acessar e modificar slides existentes nas apresentações|
|Clonagem / Copiar Slides|Existem métodos internos fornecidos por Aspose.Slides for Node.js via .NET que permitem clonar ou copiar slides existentes dentro de uma apresentação. Você também pode usar slides copiados e clonados de uma apresentação para outra. Como um slide herda seu layout do slide mestre, os métodos internos de clonagem copiam automaticamente o mestre durante a clonagem|
|Gerenciamento de Seções de Slides|Métodos para organizar slides em diferentes seções dentro de uma apresentação|
|Place Holders e Text Holders|Você pode acessar os place holders e text holders em um slide. Além disso, pode criar um slide com text holders do zero usando o método apropriado|
|Cabeçalhos e Rodapés|Aspose.Slides for Node.js via .NET facilita o tratamento de cabeçalhos/rodapés nos slides|
|Notas em Slides|Com Aspose.Slides for Node.js via .NET, você pode acessar e modificar notas associadas a um slide e também adicionar novas notas|
|Localizar uma Forma|Você também pode localizar uma forma específica em um slide usando o texto alternativo associado à forma|
|Fundos|Aspose.Slides for Node.js via .NET permite trabalhar com fundos associados a um slide mestre ou normal em uma apresentação|
|Caixas de Texto|Caixas de texto podem ser criadas do zero. Você pode acessar caixas de texto existentes. Também pode modificar seus textos sem perder o formato original do texto|
|Formas Retângulo|Você pode criar ou modificar formas retângulo com Aspose.Slides for Node.js via .NET|
|Formas Poly Line|Você pode criar ou modificar formas poly line com Aspose.Slides for Node.js via .NET|
|Formas Elipse|Você pode criar ou modificar formas elipse com Aspose.Slides for Node.js via .NET|
|Formas em Grupo|Aspose.Slides for Node.js via .NET suporta formas em grupo|
|Auto Shapes|Aspose.Slides for Node.js via .NET suporta auto shapes|
|SmartArt|Aspose.Slides for Node.js via .NET fornece suporte para formas SmartArt no MS PowerPoint|
|Gráficos|Aspose.Slides for Node.js via .NET fornece suporte para Gráficos MSO no PowerPoint|
|Serialização de Formas|Aspose.Slides for Node.js via .NET suporta um grande número de formas. Quando o Aspose.Slides for Node.js via .NET não possui suporte a uma forma, você pode usar um método de serialização que permite serializar essa forma de um slide existente. Dessa forma, você pode usar a forma posteriormente conforme suas necessidades|
|Molduras de Imagem|Você pode gerenciar imagens em molduras de imagem com Aspose.Slides for Node.js via .NET|
|Molduras de Áudio|Você pode vincular ou incorporar arquivos de áudio em molduras de áudio nos slides com Aspose.Slides for Node.js via .NET|
|Molduras de Vídeo|Você pode manipular arquivos de vídeo em molduras de vídeo. Aspose.Slides for Node.js via .NET também fornece suporte para vídeos vinculados e incorporados|
|Moldura OLE|Você pode gerenciar objetos OLE em molduras OLE com Aspose.Slides for Node.js via .NET|
|Tabelas|Aspose.Slides for Node.js via .NET suporta tabelas em slides|
|Controles ActiveX|Suporte para controles ActiveX|
|Macros VBA|Suporte para gerenciamento de macros VBA dentro de apresentações.|
|Quadro de Texto|Você pode acessar o texto de qualquer forma através do quadro de texto associado a essa forma|
|Escaneamento de Texto|Você pode escanear texto em uma apresentação no nível da apresentação ou do slide através de métodos de escaneamento internos.|
|Animações|Você pode aplicar animações em formas|
|Apresentações de Slides|Aspose.Slides for Node.js via .NET suporta apresentações de slides e transições de slides|

## **Recursos de Formatação**
Com Aspose.Slides for Node.js via .NET, você pode formatar textos e formas em slides de apresentações. Veja os detalhes abaixo:

|**Recurso**|**Descrição**|
| :- | :- |
|Text Formatting|<p>No Aspose.Slides for Node.js via .NET, você pode gerenciar textos através dos quadros de texto associados às formas. Assim, você pode formatar textos usando os parágrafos e trechos associados aos quadros de texto. Esses elementos de texto podem ser formatados através do Aspose.Slides for Node.js via .NET.</p><p>- Tipo de Fonte</p><p>- Tamanho da Fonte</p><p>- Cor da Fonte</p><p>- Tons da Fonte</p><p>- Alinhamento de Parágrafo</p><p>- Marcadores de Parágrafo</p><p>- Orientação do Parágrafo</p>|
|Shape Formatting|<p>No Aspose.Slides for Node.js via .NET, o elemento básico de um slide é uma forma. Você pode formatar esses elementos de forma com Aspose.Slides for Node.js via .NET:</p><p>- Posição</p><p>- Tamanho</p><p>- Linha</p><p>- Preenchimento (incluindo Padrão, Gradiente, Sólido)</p><p>- Texto</p><p>- Imagem</p>|
