Aspose.Slides for Node.js via .NET é uma biblioteca de classes que permite que aplicações Node.js leiam e escrevam documentos de apresentação sem usar o Microsoft PowerPoint® ou outros. Aspose.Slides for Node.js via .NET permite gerenciar texto, formas, tabelas e animações, adicionar áudio e vídeo aos slides, visualizar slides, exportar slides para SVG, formato PDF, e fazer muito mais.

Aspose.Slides for Node.js via .NET usa a [mesma referência de API que o Aspose.Slides .NET](https://reference.aspose.com/slides/pt/nodejs-net/). 

{{% /alert %}}
