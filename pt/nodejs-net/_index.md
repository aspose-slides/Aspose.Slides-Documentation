**Bem-vindo ao Aspose.Slides for Node.js via .NET**

![Logotipo do produto Aspose.Slides for Node.js via .NET](aspose_slides-for-nodejs-via-net.png)

Aspose.Slides for Node.js via .NET é uma biblioteca de classes que permite que suas aplicações leiam e gravem documentos PowerPoint® sem usar o Microsoft PowerPoint®.

Aspose.Slides for Node.js via .NET é o primeiro e único componente que fornece a funcionalidade de gerenciar documentos PowerPoint®.

Aspose.Slides for Node.js via .NET oferece muitos recursos principais, como gerenciamento de texto, formas, tabelas e animações, adição de áudio e vídeo aos slides, visualização de slides, exportação de slides para SVG, formato PDF e muito mais.
