## Aspose.Slides for .NET

{{% alert color="primary" %}}

![Logotipo do Produto Aspose.Slides for .NET](home_1.png)

Aspose.Slides for .NET é uma API de gerenciamento do Microsoft PowerPoint® que permite que aplicativos .NET leiam e gravem documentos PowerPoint® sem usar o Microsoft PowerPoint®. Aspose.Slides for .NET é o primeiro e único componente que oferece funcionalidade para gerenciar documentos PowerPoint®. Aspose.Slides for .NET fornece muitos recursos principais, como gerenciamento de texto, formas, tabelas e animações, adição de áudio e vídeo aos slides, visualização de slides, exportação de slides para SVG, PDF e muito mais.
