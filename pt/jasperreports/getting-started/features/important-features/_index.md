Esta seção fornece uma descrição detalhada dos recursos do Aspose.Slides for JasperReports. Clique nos links na tabela para saber mais sobre os recursos. 

|**Recurso** |**Descrição** |
| :- | :- |
|[Exportação PPT, PPS, PPTX e PPSX](/slides/pt/jasperreports/ppt-pptx-pdf-and-html-export/)|Exporte relatórios como apresentações do Microsoft PowerPoint a partir de sua aplicação sem usar o Microsoft PowerPoint. |
|Alta Fidelidade ao Design do Relatório|Relatórios PPT e PPS têm o mesmo layout e aparência que PDF e outros formatos. |
|Implantação Fácil e Leve|Aspose.Slides for JasperReport é um exportador de JasperReport e JasperServer para o formato Microsoft PowerPoint. |
|[Licenciamento Simples e Acessível](/slides/pt/jasperreports/licensing/)|Uma licença por servidor ou uma licença para todos os seus servidores. Desenvolvedores ilimitados. |
|[Suporte Técnico Gratuito de Classe Mundial](/slides/pt/jasperreports/technical-support/)|Suporte técnico ilimitado é fornecido diretamente pelos desenvolvedores que trabalham no produto|
