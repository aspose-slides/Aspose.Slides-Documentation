Todas as demonstrações fornecidas com Aspose.Slides para JasperReports são demonstrações padrão alteradas. É melhor copiar todas as demonstrações para a pasta de demonstrações do JasperReports:
...\jasperreports-x.x.x\demo\samples\

Use a sequência padrão de comandos para compilar e exportar relatórios:

- ant javac
- ant compile
- ant fill
- ant ppt

{{% alert color="primary" %}} 
Por favor, não se esqueça de executar o HSQLDB com o banco de dados de teste para preencher os relatórios com dados e copiar aspose.slides.jasperreports.library-xx.x.jar da pasta \lib\JasperReports X.X.X - X.X.X do arquivo aspose-slides-xx.x-jasperreports.zip para &#60;InstallDir&#62;\lib directory.
{{% /alert %}} 

A maioria das demonstrações (exceto Charts) já possui apresentações geradas, portanto você pode pular todas as etapas “ant” e verificar os resultados imediatamente.
