Para usar o Aspose.Slides para JasperReports a partir da sua aplicação, copie aspose.slides.jasperreports.library-xx.x.jar da pasta \lib\JasperReports X.X.X - X.X.X do arquivo aspose-slides-xx.x-jasperreports.zip para o diretório JasperReports\lib ou para uma pasta de bibliotecas da sua aplicação. Depois disso, você pode acessar os exportadores programaticamente.
