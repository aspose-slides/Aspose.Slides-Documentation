Aspose.Slides for JasperReports funciona com:

**Aspose.Slides for JasperReports 17.5 e versões anteriores:**

- Java 2 Platform Standard Edition 1.4 ou superior.
- JasperReports 2.0, 3.0 ou superior.

**Aspose.Slides for JasperReports 17.6 e versões mais recentes:**

- Java 2 Platform Standard Edition 6.0 ou superior.
- JasperReports 3.7.2 ou superior.
