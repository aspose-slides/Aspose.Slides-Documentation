{{% alert color="primary" %}} 

Com o Aspose.Slides for SharePoint, você pode converter documentos entre vários formatos populares de documentos do Office a partir de uma biblioteca de documentos do SharePoint. As conversões são realizadas com alta fidelidade e precisão. 

{{% /alert %}} 
## **Formatos de Entrada Compatíveis**
Aspose.Slides for SharePoint suporta os seguintes formatos de entrada: 

- PPT – Apresentação Microsoft PowerPoint 97 - 2003
- PPS – Apresentação de Slides Microsoft PowerPoint 97 - 2003
- POT – Modelo Microsoft PowerPoint 97 - 2003
- PPTX – Apresentação Office Open XML
- PPSX – Apresentação de Slides Office Open XML
- POTX – Modelo Office Open XML

{{% alert color="primary" %}} 

Para gerar documentos, o Aspose.Slides for SharePoint depende de uma versão incorporada do [Aspose.Slides for .NET](http://www.aspose.com/categories/.net-components/aspose.slides-for-.net/default.aspx), o único componente de processamento de documentos PowerPoint da Aspose. 

{{% /alert %}}
## **Formatos de Saída Compatíveis**
Os seguintes formatos de saída são suportados pelo Aspose.Slides for SharePoint: 

- PDF – Formato de Documento Portátil
- TIFF – Pacote de Imagens
- XPS – Especificação de Papel XML
