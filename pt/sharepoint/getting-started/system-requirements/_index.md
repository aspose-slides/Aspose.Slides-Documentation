## **Requisitos do Servidor**
{{% alert color="primary" %}} 

Para instalar o Aspose.Slides for SharePoint, é necessário um sistema que execute um dos seguintes:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft Office SharePoint Server 2010
- Microsoft Office SharePoint Server 2013
- Microsoft Office SharePoint Server 2016
- Microsoft Office SharePoint Server 2019

{{% /alert %}} {{% alert color="primary" %}} 

É recomendada a instalação do Service Pack 2 ou posterior para esses produtos.

{{% /alert %}} 
## **Requisitos do Cliente**
{{% alert color="primary" %}} 

Os sites do SharePoint são acessados através de um navegador web. O Aspose.Slides for SharePoint foi testado com os seguintes navegadores:

- Microsoft Internet Explorer 6, 7, 8
- Mozilla FireFox 3.5

{{% /alert %}}
