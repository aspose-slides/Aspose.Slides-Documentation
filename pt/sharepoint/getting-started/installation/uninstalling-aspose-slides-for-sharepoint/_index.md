Para desinstalar o Aspose.Slides for SharePoint:

1. Execute o programa de instalação.
   Se o Aspose.Slides for SharePoint já estiver instalado, o programa de instalação sugere removê-lo ou repará-lo.
1. Selecione **Remove** para desinstalar o Aspose.Slides for SharePoint.
