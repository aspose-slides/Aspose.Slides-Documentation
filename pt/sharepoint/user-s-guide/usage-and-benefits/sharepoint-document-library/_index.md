Uma biblioteca é uma estrutura de conteúdo que contém arquivos. Uma biblioteca pode conter apenas um tipo de arquivo ou muitos tipos diferentes, incluindo documentos, planilhas e apresentações. Uma biblioteca do SharePoint cria um local único onde as pessoas podem enviar e acessar documentos. Os documentos são armazenados em formato binário no banco de dados SQL Server usado pelo site do SharePoint. Os documentos ficam disponíveis para outros usuários do SharePoint como uma lista de hyperlinks.
