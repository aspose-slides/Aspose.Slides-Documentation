{{% alert color="primary" %}} 

Os sites do SharePoint são projetados para colaboração, comunicação e armazenamento de conteúdo, a fim de atender a necessidades comerciais específicas. Eles permitem que um grupo de indivíduos alcance um objetivo comum, facilitando o compartilhamento de informações e a comunicação. As informações geralmente estão disponíveis como um site e são acessadas por meio de um navegador web. 

{{% /alert %}} 
## **O que é SharePoint e Aspose.Slides?**
Aspose.Slides for SharePoint é uma solução flexível para converter documentos ~Microsoft PowerPoint para outros formatos como PPT, POT, PPS, PPTX, PPSX e POTX. Aspose.Slides for SharePoint permite ler e converter arquivos PowerPoint em uma aplicação SharePoint sem usar o Microsoft PowerPoint.
