## **Convertendo uma Apresentação**
Para converter um documento Microsoft PowerPoint de uma biblioteca de documentos do SharePoint: 

1. Selecione um documento Microsoft PowerPoint em uma biblioteca de documentos.
2. Clique na seta para baixo para revelar um menu e clique em **Convert via Aspose.Slides.SharePoint**. 

   **O menu do arquivo Presentation 2 mostrando a opção Convert via Aspose.Slides** 

![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_2.png)




3. Selecione o formato de saída desejado no formulário. Se desejar, altere o nome do arquivo de saída e a pasta de destino.
4. Clique em **Convert** para converter o arquivo. 

   **O formulário de conversão permite selecionar o formato do arquivo de saída, nome e destino** 

![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_3.png)




5. Quando a conversão estiver concluída, uma mensagem de sucesso será exibida. 

   **A conversão foi bem-sucedida** 

![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_4.png)




6. Clique em **Source Library** (para ir ao diretório de origem) ou em **Destination Library** (para ir ao diretório onde o arquivo foi salvo). 

   O documento convertido aparece na biblioteca de documentos. 

   **O documento convertido mostrado na biblioteca onde foi salvo** 

![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_5.png)
