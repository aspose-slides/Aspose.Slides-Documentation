**Bem-vindo ao Aspose.Slides for Reporting Services**

O Aspose.Slides for Reporting Services é a única solução no mercado que possibilita gerar verdadeiros relatórios PPT e PPS no Microsoft SQL Server 2005, 2008, 2012, 2016 e 2017 Reporting Services (32 bits e 64 bits). Todos os recursos de relatórios RDL, incluindo tabelas, matrizes, gráficos e imagens, são convertidos com o mais alto grau de precisão para apresentações Microsoft PowerPoint.
