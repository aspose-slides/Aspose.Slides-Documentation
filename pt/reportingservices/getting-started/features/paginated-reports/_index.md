{{% alert color="primary" %}} 

Aspose.Slides for Reporting Services implementou suporte para Power BI. 

{{% /alert %}} 

Relatórios paginados são relatórios formatados para caber bem em uma página. Esses relatórios são ideais para operações de exportação e impressão. Os relatórios paginados são projetados para fornecer quantas páginas forem necessárias para imprimir ou exportar cada linha de dados da sua tabela. 

Por exemplo, se você deseja criar um relatório para que os proprietários desse relatório possam visualizar a quantidade de produto, o preço de venda e o lucro de cada item vendido nos últimos 12 meses, então um relatório paginado é o que você precisa. 

Para permitir que você crie relatórios paginados, a Microsoft fornece o Power BI Report Builder no Power BI. Depois de criar um relatório paginado e salvá-lo em sua máquina, o arquivo resultante recebe a extensão .rdl.
