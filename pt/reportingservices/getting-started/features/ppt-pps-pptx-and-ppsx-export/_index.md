{{% alert color="primary" %}} 

Microsoft SQL Server Reporting Services não possui recursos integrados para exportar relatórios como apresentações do Microsoft PowerPoint, mas após instalar o Aspose.Slides for Reporting Services no seu servidor, você terá acesso a dois formatos de exportação adicionais: 

- Microsoft PowerPoint Presentation (PPT)
- Microsoft PowerPoint SlideShow (PPS)
- Microsoft PowerPoint 2007 Presentation (PPTX)
- Microsoft PowerPoint 2007 SlideShow (PPSX)

{{% /alert %}} 

Para criar documentos nesses formatos, o Aspose.Slides for Reporting Services depende de uma versão incorporada do [Aspose.Slides for .NET](http://www.aspose.com/Products/Aspose.Slides/), o principal componente de processamento de apresentações do mercado da Aspose. O Microsoft PowerPoint não é usado para gerar os documentos. 


Cópias PPT e PPS do mesmo relatório aparecerão idênticas entre si, graças ao alto grau de suporte aos formatos de arquivo subjacentes fornecido pelo Aspose.Slides.
