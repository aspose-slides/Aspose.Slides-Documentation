{{% alert color="primary" %}} 

Aspose.Slides for Reporting Services suporta totalmente todas as especificações RDL. Isso significa duas grandes coisas: 

- Não é necessário redesenhar relatórios existentes. Você pode exportar qualquer relatório RDL existente como uma apresentação Microsoft PowerPoint e ele aparecerá exatamente de acordo com o design RDL.
- Não é necessário usar um designer de relatórios específico. Você pode usar qualquer designer de relatórios RDL e o relatório será exportado exatamente como você o projetou.

{{% /alert %}} 

Aspose.Slides for Reporting Services suporta os seguintes elementos RDL: 

- Página, cabeçalhos, rodapés
- Caixas de texto
- Imagens
- Sub‑relatórios
- Gráficos
- Listas
- Tabelas
- Matrizes
- Estilos
- Retângulos
