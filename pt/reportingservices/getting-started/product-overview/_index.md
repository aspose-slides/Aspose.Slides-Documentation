## **Bem-vindo ao Aspose.Slides for Reporting Services!**

Aspose.Slides for Reporting Services é a única solução no mercado que possibilita gerar relatórios PPT e PPS verdadeiros no Microsoft SQL Server 2005, 2008, 2012, 2016 e 2017 Reporting Services (32 bits e 64 bits). Todos os recursos de relatórios RDL, incluindo tabelas, matrizes, gráficos e imagens, são convertidos com o mais alto grau de precisão para apresentações Microsoft PowerPoint.

## **Visão geral do produto**

Microsoft SQL Server Reporting Services não dispõe de recursos nativos para exportar relatórios como apresentações Microsoft PowerPoint, mas após instalar o Aspose.Slides for Reporting Services no seu servidor, você terá acesso a formatos de exportação adicionais:

- PPT - Apresentação PowerPoint via Aspose.Slides
- PPS - Apresentação de Slides PowerPoint via Aspose.Slides
- PPTX - Apresentação PowerPoint 2007 via Aspose.Slides
- PPSX - Apresentação de Slides PowerPoint 2007 via Aspose.Slides

Aspose.Slides for Reporting Services cria apresentações no servidor sem usar o Microsoft PowerPoint. Internamente, o Aspose.Slides for Reporting Services utiliza o Aspose.Slides para .NET – o componente de classe mundial para processamento de apresentações no lado do servidor.

**Aspose.Slides for Reporting Services permite exportar qualquer relatório nos formatos PPT, PPS, PPTX ou PPSX.**
