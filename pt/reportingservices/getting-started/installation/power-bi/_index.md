## **Power BI**
Você pode instalar o Aspose.Slides for Reporting Services em um servidor Power BI usando os mesmos processos de instalação:

Veja

* [Instalar com Instalador MSI](https://docs.aspose.com/slides/pt/reportingservices/install-with-msi-installer/#installation). O programa de instalação detecta automaticamente o Power BI instalado no servidor, instala o produto e, em seguida, o configura para o seu ambiente de trabalho. 
* [Instalar manualmente](https://docs.aspose.com/slides/pt/reportingservices/install-manually/).

{{% alert title="Nota" color="warning" %}} 

**Aspose.Slides for Reporting Services** requer a instalação do **.NET Framework 3.5** na máquina host. 

{{% /alert %}}

Aspose.Slides for SSRS suporta relatórios paginados (cdl).
