## **Suporte de licença**
{{% alert color="primary" %}} 

Você pode baixar uma versão de avaliação do **Aspose.Slides for Reporting Services** na [sua página de lançamentos](https://releases.aspose.com/slides/pt/reportingservices/). A versão de avaliação fornece as mesmas funcionalidades da versão licenciada do produto. O pacote de avaliação é o mesmo que o pacote adquirido. A versão de avaliação simplesmente se torna licenciada após você adicionar algumas linhas de código a ela (para aplicar a licença).

Quando estiver satisfeito com sua avaliação do produto, você pode [comprar uma licença](https://purchase.aspose.com/buy). Recomendamos que você analise os diferentes tipos de assinatura. Se tiver dúvidas, entre em contato com a equipe de vendas da Aspose.

{{% /alert %}} 

## **Licenciamento no Aspose.Slides for Reporting Service**

* Uma versão de avaliação torna-se licenciada após você comprar uma licença e adicionar algumas linhas de código a ela (para aplicar a licença).
* A licença é um arquivo XML de texto simples que contém detalhes como o nome do produto, o número de desenvolvedores a que está licenciada, a data de expiração da assinatura, etc.
* O arquivo de licença é assinado digitalmente, portanto você não deve modificá-lo. Mesmo a adição inadvertida de uma quebra de linha extra ao conteúdo do arquivo o invalidará.
* Para evitar as limitações associadas à versão de avaliação, você precisa definir uma licença antes de usar o Aspose.Slides for Reporting Service. 

Baixe a licença para o seu computador e copie‑a para a pasta **C:\Program Files\Microsoft SQL Server\<Instance>\Reporting Services\ReportServer\bin**, que é onde o **Aspose.Slides.ReportingServices.dll** está instalado. 

Para confirmar que a licença foi instalada corretamente, exporte qualquer relatório como uma apresentação do Microsoft PowerPoint. Se o documento não contiver marca d'água, a licença foi ativada com sucesso. 

![todo:image_alt_text](license-aspose-slides-for-reporting-services_1.png)

Quando um arquivo Aspose.Slides.ReportingServices.lic válido existe na pasta *ReportServer\bin*, não há marca d'água de avaliação. 

**Modo de avaliação**

O Aspose.Slides for Reporting Services injeta uma marca d'água quando está operando no modo de avaliação (sem licença)
