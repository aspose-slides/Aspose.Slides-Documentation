{{% alert color="primary" %}} 

Aspose.Slides for Node.js via Java é uma biblioteca de classes que permite que aplicativos Node.js leiam e escrevam documentos de apresentação sem usar o Microsoft PowerPoint® ou outros. Aspose.Slides for Node.js via Java permite gerenciar texto, formas, tabelas e animações, adicionar áudio e vídeo aos slides, visualizar slides, exportar slides para SVG, formato PDF e fazer muito mais.

A referência mais recente da API pode ser encontrada em [Aspose.Slides for Node.js via Java API Reference](https://reference.aspose.com/slides/pt/nodejs-java/).
