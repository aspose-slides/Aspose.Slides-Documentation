## **Plataformas Compatíveis**
Aspose.Slides for Node.js via Java é uma API independente de plataforma. Pode ser usada em qualquer plataforma (Windows, Linux, MacOS, etc.) onde o Node.js 7 ou superior esteja instalado. O Oracle JDK 8 ou superior é outro requisito.

## **Formatos de Arquivo e Conversões**
Aspose.Slides for Node.js via Java suporta a maioria dos formatos de documentos PowerPoint. Também permite exportá-los para os formatos de arquivo populares que as organizações utilizam e compartilham entre si. Veja os detalhes:

|**Recurso**|**Descrição**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via Java fornece o processamento mais rápido para este formato de documento de apresentação.|
|Conversão PPT para PPTX|Aspose.Slides for Node.js via Java suporta conversão de PPT para PPTX|
|Portable Document Format (PDF)|Você pode exportar todos os formatos de arquivo suportados para documentos Adobe Portable Document Format (PDF) com um único método.|
|XML Parser Specification (XPS)|Você pode exportar todos os formatos de arquivo suportados para documentos XML Parser Specification (XPS) com um único método.|
|Tagged Image File Format (TIFF)|Você pode exportar todos os formatos de apresentação suportados para Tagged Image File Format (TIFF).|
|Conversão PPTX para HTML|Aspose.Slides for Node.js via Java suporta a conversão de PresentationEx para formato HTML.|

## **FAQ**

**Preciso instalar o Microsoft PowerPoint no servidor/PC para que a biblioteca funcione?**

Não. PowerPoint não é necessário; Aspose.Slides é um mecanismo independente para criar, editar, converter e renderizar apresentações.

**Como o multithreading funciona? O processamento pode ser paralelizado?**

É seguro processar documentos diferentes em threads diferentes; o mesmo [apresentação](https://reference.aspose.com/slides/pt/nodejs-java/aspose.slides/presentation/) object must not be used by [várias threads](/slides/pt/nodejs-java/multithreading/) at the same time.

**Senhas de arquivos e criptografia são suportadas?**

Sim. [Você pode](/slides/pt/nodejs-java/password-protected-presentation/) abrir apresentações criptografadas, definir ou remover uma senha de abertura e escrita, e verificar o status da proteção.

**Preciso me preocupar com pacotes de fontes em contêineres Linux?**

Sim. É recomendado instalar pacotes de fontes comuns e/ou especificar explicitamente [especificar diretórios de fontes](/slides/pt/nodejs-java/custom-font/) em sua aplicação para evitar substituições inesperadas.

**Existem limitações na versão de avaliação?**

Em [modo de avaliação](/slides/pt/nodejs-java/licensing/), uma marca d'água é adicionada ao output e certas limitações se aplicam; uma [licença temporária de 30 dias](https://purchase.aspose.com/temporary-license/) está disponível para teste de todos os recursos.

**A importação de formatos externos para uma apresentação (PDF/HTML → PPTX) é suportada?**

Sim. Você pode adicionar [páginas PDF e conteúdo HTML](/slides/pt/nodejs-java/import-presentation/) a uma apresentação, transformando-as em slides.
