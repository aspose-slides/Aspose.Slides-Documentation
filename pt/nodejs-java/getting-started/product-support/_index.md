## **Visão geral**

Este artigo explica como obter suporte para Aspose.Slides e como relatar problemas de forma eficaz. Ele descreve o canal principal de suporte, fornece recomendações para enviar um problema e aponta recursos técnicos úteis, como a página do produto, blog, downloads, fórum de suporte, documentação e Referência da API. O artigo também responde a perguntas comuns sobre suporte pago, escalonamento de problemas e suporte a versões antigas da biblioteca.

## **Suporte técnico**
{{% alert color="primary" %}}

Aspose fornece suporte técnico gratuito e ilimitado para todos os seus produtos. O suporte está disponível para todos os usuários (incluindo usuários com pacotes de avaliação).

{{% /alert %}} 

Quando precisar de ajuda com Aspose.Slides for Node.js via Java, considere o seguinte:

- O principal canal de suporte é o [Fórum Aspose](https://forum.aspose.com/c/slides/pt/11). Publique sua pergunta no fórum Aspose.Slides — ela será respondida em poucas horas. A equipe de suporte do Aspose.Slides responde diretamente às perguntas postadas no fórum.
- Observe que a Aspose não fornece suporte técnico por telefone. O suporte telefônico está disponível apenas para perguntas de vendas e compras.
- Ao aguardar uma resposta nos fóruns, seja paciente e leve em consideração as diferenças de fusos horários.


Quando você encontrar um problema com Aspose.Slides for Node.js via Java, siga estas diretrizes para resolvê‑lo da maneira mais eficiente:

- Verifique e confirme que está usando a versão mais recente do Aspose.Slides for Node.js via Java antes de relatar o problema. Talvez o problema que você encontrou já tenha sido resolvido em uma atualização. 

  Consulte o [download do Aspose.Slides](https://releases.aspose.com/slides/pt/nodejs-java/) para obter informações sobre a versão mais recente.

- Antes de relatar um problema, talvez queira consultar nossos fóruns, esta documentação e a Referência da API. Talvez a resposta para sua pergunta já tenha sido fornecida. 

- Ao relatar um problema, inclua o documento original e (se possível) um fragmento do seu código relacionado ao problema. Se precisar anexar vários arquivos, coloque‑os em um pacote ZIP. Você pode anexar seus documentos com segurança no Fórum Aspose, pois **somente** você e os desenvolvedores da Aspose têm acesso aos arquivos anexados.

- Crie e use um único tópico para um problema específico. Se você encontrar outro problema, deve criar outro tópico para relatar.

## **FAQ**

**Quais níveis de suporte pago estão disponíveis e como eles diferem?**

[Desenvolvedor, Business e Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Eles diferem pelo número de incidentes incluídos, prioridade de resposta e canais de comunicação disponíveis.

**Posso escalar um bug diretamente para a equipe de engenharia?**

Sim. O [suporte pago](https://helpdesk.aspose.com/) permite a escalonamento para os desenvolvedores quando um problema requer investigação mais aprofundada.

**Versões antigas da biblioteca são suportadas e vocês lançam correções para elas?**

Você pode usar versões antigas, mas correções são fornecidas apenas para a versão atual; patches para versões legadas não são lançados.
