## **Power BI**
You can install Aspose.Slides for Reporting Services on a Power BI server through the same installation processes:

See 

* [Install with MSI Installer](https://docs.aspose.com/slides/reportingservices/install-with-msi-installer/#installation). The installation program automatically detects the installed Power BI on the server, installs the product, and then configures it for your work environment. 
* [Install Manually](https://docs.aspose.com/slides/reportingservices/install-manually/).

{{% alert title="Note" color="warning" %}} 

**Aspose.Slides for Reporting Services** requires the installation of **.NET Framework 3.5** on the host machine. 

{{% /alert %}}

Aspose.Slides for SSRS supports paginated reports (cdl).
