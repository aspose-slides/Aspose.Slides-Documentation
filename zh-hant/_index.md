## Aspose.Slides for .NET

{{% alert color="primary" %}}

![Aspose.Slides for .NET Product Logo](home_1.png)

Aspose.Slides for .NET 是一套 Microsoft PowerPoint® 管理 API，可讓 .NET 應用程式在不使用 Microsoft PowerPoint® 的情況下讀寫 PowerPoint® 檔案。Aspose.Slides for .NET 是首個且唯一提供管理 PowerPoint® 文件功能的元件。Aspose.Slides for .NET 提供許多關鍵功能，包括管理文字、圖形、表格與動畫、向投影片新增音訊與影片、預覽投影片、將投影片匯出為 SVG、PDF 格式等。
