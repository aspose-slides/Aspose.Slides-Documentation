---
title: 安裝 Aspose.Slides for JasperReports
type: docs
weight: 40
url: /zh-hant/jasperreports/installing-aspose-slides-for-jasperreports/
---
要在您的應用程式中使用 Aspose.Slides for JasperReports，請將 aspose.slides.jasperreports.library-xx.x.jar 從 aspose-slides-xx.x-jasperreports.zip 的 \lib\JasperReports X.X.X - X.X.X 資料夾，複製到 JasperReports\lib 目錄或您的應用程式的 library 資料夾。之後，您即可以程式方式存取匯出器。
