Aspose.Slides for JasperReports 支援以下環境：

**Aspose.Slides for JasperReports 17.5 及之前的版本：**

- Java 2 Platform Standard Edition 1.4 或更新版本。
- JasperReports 2.0、3.0 或更新版本。

**Aspose.Slides for JasperReports 17.6 及更新的版本：**

- Java 2 Platform Standard Edition 6.0 或更新版本。
- JasperReports 3.7.2 或更新版本。
