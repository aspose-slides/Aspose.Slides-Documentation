{{% alert color="primary" %}} 

JasperReports 和 JasperServer 本身不具備將報表匯出為 Microsoft PowerPoint 簡報的功能，但使用 Aspose.Slides for JasperReports，您即可取得額外的匯出格式：

- Microsoft PowerPoint 簡報 (PPT)
- Microsoft PowerPoint 簡報 (PPTX)
- HTML
- PDF

{{% /alert %}} 

要產生這些格式的文件，Aspose.Slides for JasperReports 依賴內建的 [Aspose.Slides for Java](https://products.aspose.com/slides/zh-hant/java/)，這是 Aspose 的市場領先簡報處理函式庫。Microsoft PowerPoint 不會被用來產生文件。
