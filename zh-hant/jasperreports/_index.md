**歡迎使用 Aspose.Slides for JasperReports**

Aspose.Slides for JasperReports 是一個專為需要在 Java 應用程序中輕鬆將 JasperReports 報表匯出為 Microsoft PowerPoint 簡報（PPT）和 Microsoft PowerPoint 幻燈片放映（PPS）格式的開發人員設計和開發的函式庫。所有報表功能均以最高精度轉換為 Microsoft PowerPoint 簡報。Aspose.Slides for JasperReports 支援 JasperReports 5 以上版本。
