Aspose 為所有使用者提供免費支援。若公司希望其問題被優先處理，或想影響我們的產品開發路線圖，可使用如 [advanced support options](https://helpdesk.aspose.com/kb/faq/2-Developer-Business-Support-Key-Benefits-Conditions) 等進階支援方案，如 Priority Support 與 Enterprise Support。
