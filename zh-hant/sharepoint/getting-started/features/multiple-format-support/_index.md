{{% alert color="primary" %}} 

使用 Aspose.Slides for SharePoint，您可以在 SharePoint 文件庫中在多種流行的 Office 文件格式之間進行轉換。轉換具有高度的忠實度和精確性。 

{{% /alert %}} 
## **支援的輸入格式**
Aspose.Slides for SharePoint 支援以下輸入格式： 

- PPT – Microsoft PowerPoint 簡報 97 - 2003
- PPS – Microsoft PowerPoint 投影片放映 97 - 2003
- POT – Microsoft PowerPoint 範本 97 - 2003
- PPTX – Office Open XML 簡報
- PPSX – Office Open XML 投影片放映
- POTX – Office Open XML 範本

{{% alert color="primary" %}} 

若要產生文件，Aspose.Slides for SharePoint 依賴內建版本的 [Aspose.Slides for .NET](http://www.aspose.com/categories/.net-components/aspose.slides-for-.net/default.aspx)，這是 Aspose 唯一的 PowerPoint 文件處理元件。

{{% /alert %}}
## **支援的輸出格式**
Aspose.Slides for SharePoint 支援以下輸出格式： 

- PDF – 可攜式文件格式
- TIFF – 圖像套件
- XPS – XML 紙張規範

**從「轉換設定」畫面中選取輸出格式**
