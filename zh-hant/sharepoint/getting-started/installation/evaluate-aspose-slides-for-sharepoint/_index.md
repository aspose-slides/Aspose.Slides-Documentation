請務必善加利用免費的 Aspose.Words for SharePoint 評估版：它沒有時間限制，且提供免費的技術支援。我們鼓勵使用者在購買前先評估我們的產品，讓您確信該產品能如期實現您的需求。

{{% /alert %}} 

評估版與付費版的 Aspose.Slides for SharePoint 使用相同的下載檔案。只需從下載頁面[下載 Aspose.Slides for SharePoint](http://www.aspose.com/community/files/73/sharepoint-components/aspose.slides-for-sharepoint/default.aspx) [安裝它]()，預設即以評估模式運作。 

評估模式會在匯出文件的每一張投影片上加入評估水印。當您購買授權後，只需在已安裝的 Aspose.Slides for SharePoint 評估版上安裝授權方案，即可切換至授權模式。
