## **轉換簡報**
從 SharePoint 文件庫轉換 Microsoft PowerPoint 文件的步驟如下： 

1. 在文件庫中選取一個 Microsoft PowerPoint 文件。  
2. 點擊向下箭頭以展開功能表，然後點擊 **Convert via Aspose.Slides.SharePoint**。  

   **Presentation 2 檔案的功能表顯示 Convert via Aspose.Slides 選項**  

   ![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_2.png)

3. 從表單中選取所需的輸出格式。如需，亦可更改輸出檔案名稱及目的資料夾。  
4. 點擊 **Convert** 以轉換檔案。  

   **此轉換表單讓您選擇輸出檔案格式、名稱與儲存位置**  

   ![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_3.png)

5. 轉換完成後，會顯示成功訊息。  

   **轉換成功**  

   ![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_4.png)

6. 點擊 **Source Library**（前往原始目錄）或 **Destination Library**（前往檔案儲存的目錄）。  

   轉換後的文件會出現在文件庫中。  

   **已儲存至的文件庫中顯示的轉換文件**  

   ![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_5.png)
