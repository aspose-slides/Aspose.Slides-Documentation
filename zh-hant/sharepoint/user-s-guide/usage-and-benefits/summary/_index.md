{{% alert color="primary" %}} 

安裝與使用 Aspose.Slides for SharePoint 非常簡單。它不僅能讓伺服器直接轉換 Microsoft PowerPoint 檔案，還免除使用者必須安裝 Microsoft PowerPoint 的需求。

此外，開發人員可以使用 Aspose.Slides for SharePoint 的公共 API 來開發自己的 SharePoint 解決方案。

換句話說，Aspose.Slides for SharePoint 可惠及 SharePoint 網站管理員、開發人員與最終使用者。

{{% /alert %}}
