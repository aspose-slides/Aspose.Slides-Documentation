資料庫是一種包含檔案的內容結構。資料庫可能只包含單一類型的檔案，或包含多種不同類型的檔案，包括文件、試算表和簡報。SharePoint 資料庫建立一個單一位置，讓使用者上傳與存取文件。文件以二進位格式儲存在 SharePoint 網站所使用的 SQL Server 資料庫中。這些文件會以超連結清單的形式提供給其他 SharePoint 使用者。
