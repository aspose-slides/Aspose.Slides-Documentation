{{% alert color="primary" %}} 

Aspose.Slides for Java 是一個類別庫，使 Java 應用程式能在不使用 Microsoft PowerPoint® 或其他軟體的情況下讀取與寫入簡報文件。Aspose.Slides for Java 允許管理文字、圖形、表格與動畫，向投影片添加音訊和視訊，預覽投影片，將投影片匯出為 SVG、PDF 格式等。

最新的 API 參考可在 [Aspose.Slides for Java API Reference](https://reference.aspose.com/slides/zh-hant/java) 找到。
