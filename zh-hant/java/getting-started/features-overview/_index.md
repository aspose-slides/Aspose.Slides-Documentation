## **支援平台**
Aspose.Slides for Java 支援最受歡迎的開發與部署平台。

|**功能**|**說明**|
| :- | :- |
|桌面應用程式|Aspose.Slides for Java 可用於開發 Windows Forms 應用程式|
|企業網路應用程式|使用 Aspose.Slides for Java 可協助建立針對... 的 Web 應用程式。亦提供使用 Aspose.Slides for Java 搭配 PHP 的支援。|

## **檔案格式與轉換**
Aspose.Slides for Java 支援大多數 Microsoft PowerPoint 檔案格式，並將它們匯出為組織廣泛使用的流行格式。

|**功能**|**說明**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/zh-hant/java/microsoft-powerpoint-ppt/)|Aspose.Slides for Java 為此簡報文件格式提供最快速的處理速度。|
|[PresentationML (PPTX, XML)](/slides/zh-hant/java/presentationml-pptx-xml/)|Aspose.Slides for Java 支援處理 OOXML 簡報格式（亦稱為 PresentationML 或 PPTX）。|
|[PPT to PPTX conversion](/slides/zh-hant/java/ppt-to-pptx-conversion/)|Aspose.Slides for Java 支援從 PPT 轉換為 PPTX。|
|[Portable Document Format (PDF)](/slides/zh-hant/java/developer-guide/)|支援的檔案格式可透過單一方法匯出為 Adobe 可攜式文件格式 (PDF) 文件。|
|[XML Parser Specification (XPS)](/slides/zh-hant/java/xml-parser-specification-xps/)|所有支援的檔案格式可透過單一方法匯出為 XML Parser Specification (XPS) 文件。|
|[Tagged Image File Format (TIFF)](https://docs.aspose.com/slides/zh-hant/java/convert-powerpoint-to-tiff/)|Aspose.Slides for Java 支援的簡報檔案格式亦可匯出為 Tagged Image File Format (TIFF)。|
|[ODP to PPTX Conversion](https://docs.aspose.com/slides/zh-hant/java/convert-odp-to-pptx/)|Aspose.Slides for Java 支援載入 OpenDocument Presentation (ODP) 並將其轉換為 PPTX。|
|[Presentation To HTML Conversion](https://docs.aspose.com/slides/zh-hant/java/convert-powerpoint-to-html/)|Aspose.Slides for Java 支援將 PresentationEx 轉換為 HTML 格式。|

|**功能**|**說明**|
| :- | :- |
|支援的影像格式|使用 Aspose.Slides for Java，您不僅可以渲染簡報投影片，還可以渲染投影片上的影像，並匯出為受支援的常見圖形格式，如 TIFF、PNG、BMP、JPEG、GIF 與 metafiles。|

## **常見問題**

**是否需要在伺服器/電腦上安裝 Microsoft PowerPoint 才能使用此函式庫？**

不需要。PowerPoint 不是必須的；Aspose.Slides 是一個獨立的引擎，用於建立、編輯、轉換和渲染簡報。

**多執行緒如何運作？能否平行處理？**

在不同執行緒中處理不同的文件是安全的；同一個 [presentation](https://reference.aspose.com/slides/zh-hant/java/com.aspose.slides/presentation/) 物件不得同時被 [multiple threads](/slides/zh-hant/java/multithreading/) 使用。

**是否支援檔案密碼與加密？**

是的。[您可以](/slides/zh-hant/java/password-protected-presentation/) 開啟受加密的簡報、設定或移除開啟與寫入密碼，並檢查保護狀態。

**在 Linux 容器中需要關注字型套件嗎？**

是的。建議安裝常見的字型套件，或在應用程式中明確 [specify font directories](/slides/zh-hant/java/custom-font/) 以避免意外的字型替換。

**評估版有什麼限制？**

在 [評估模式](/slides/zh-hant/java/licensing/) 中，輸出會加上浮水印，且有某些限制；可使用 [30 天的暫時授權](https://purchase.aspose.com/temporary-license/) 進行完整功能測試。

**是否支援將外部格式匯入簡報（PDF/HTML → PPTX）？**

是的。您可以將 [PDF 頁面與 HTML 內容](/slides/zh-hant/java/import-presentation/) 加入簡報，將它們轉換為投影片。
