{{% alert color="primary" %}} 

The [XML 解析器規範](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) 是一種頁面描述語言，也是最初由 Microsoft 開發的固定文件格式。與 PDF 類似，XPS 設計用於保留文件真實度並提供與裝置無關的文件外觀。

{{% /alert %}} 

## **在 Aspose.Slides for Java 中的 XPS**
任何可由 Aspose.Slides for Java 載入的簡報文件皆可轉換為 XPS 格式。Aspose.Slides for Java 使用高保真頁面佈局與渲染引擎，以固定版面 XPS 文件格式產出輸出。您可以在 [轉換為 XPS](https://docs.aspose.com/slides/zh-hant/java/convert-powerpoint-to-xps/) 中了解如何透過 Aspose.Slides for Java 將簡報文件匯出為 XPS 文件。
