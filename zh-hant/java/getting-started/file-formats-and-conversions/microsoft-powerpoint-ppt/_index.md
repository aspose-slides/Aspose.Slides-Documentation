{{% alert color="primary" %}} 

[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) 是一種簡報文件格式，可由不同版本的 Microsoft PowerPoint 建立、讀取、操作與寫入。這是 Microsoft 開發的二進位簡報文件格式。

{{% /alert %}} 

## **在 Aspose.Slides for Java 中的 PPT**
Aspose.Slides for Java 可以讀取以下列出的軟體所建立的 PPT 檔案。

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

同樣地，由 Aspose.Slides for Java 建立的 PPT 檔案亦可被上述軟體讀取。

## **對 PPT 的完整支援**
Aspose.Slides for Java 提供幾乎所有 PPT 文件格式所支援的功能。它不僅涵蓋不同版本 Microsoft PowerPoint 提供的基本與進階功能，還包括 Microsoft PowerPoint 本身未支援的功能。使用 Aspose.Slides for Java API 函式庫的主要優勢在於處理此類功能的便利性。

除了建立、讀取與寫入 PPT 文件的基本工作外，Aspose.Slides for Java 還提供以下多項功能：

- 將其他 Microsoft Office 檔案格式匯入為 [OLE 物件至 PPT 文件]().
- [匯出 PPT 文件至 PDF](/slides/zh-hant/java/convert-powerpoint-ppt-and-pptx-to-pdf/).
- 將 PPT 文件中的投影片匯出為 SVG 格式。
- 將投影片渲染為 Java 框架支援的任何影像格式。
- 設定 PPT 文件中投影片的大小。
- 管理形狀上的動畫。
- 管理投影片放映。
- [格式化投影片文字]().
- 從 PPT 文件中擷取文字。
- [處理投影片中的表格]().
- 使用 [複製功能]() 自動複製母版。
