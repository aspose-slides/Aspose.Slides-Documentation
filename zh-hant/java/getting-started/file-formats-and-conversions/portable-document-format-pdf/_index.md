{{% alert color="primary" %}} 

The [可攜式文件格式](https://en.wikipedia.org/wiki/PDF) 是由 Adobe Systems 所創建的檔案格式，用於在組織之間交換文件。此格式的目的是不論在何種平台上檢視，都能保持內容與版面相同。Aspose.Slides for Java 讓您可以將簡報檔案轉換為 PDF.

{{% /alert %}} 

## **Aspose.Slides for Java 中的 PDF**
任何可在 Aspose.Slides for Java 中載入的簡報，都可以依照您的選擇轉換為符合 PDF 1.5、PDF/A-1a、PDF/A-1b 或 PDF/UA 的 PDF。Aspose.Slides for Java 會將簡報匯出為 PDF，且在大多數情況下，輸出的 PDF 與原始簡報完全相同。

在轉換為 PDF 時，Aspose.Slides 支援下列簡報功能：

- 影像、文字方塊及其他圖形。
- 文字與格式設定。
- 段落與格式設定。
- 超連結。
- 頁首與頁尾。
- 項目符號。
- 表格。

您可以直接使用 Aspose.Slides for Java 將簡報匯出為 PDF：不需要任何其他元件。此外，您還可以依據在 [轉換為 PDF](/slides/zh-hant/java/converting-a-presentation/) 中說明的各種選項自訂簡報至 PDF 的匯出行為。
