## **概覽**

本文說明如何取得 Aspose.Slides 的支援以及如何有效回報問題。它描述主要的支援管道，提供提交問題的建議，並指向有用的技術資源，例如產品頁面、部落格、下載、支援論壇、文件與 API 參考。本文亦回答有關付費支援、問題升級以及舊版函式庫支援的常見問題。

## **技術支援**

{{% alert color="primary" %}} 

Aspose 為所有產品提供無限制的免費技術支援。此支援對所有使用者皆可使用，包括評估版。

{{% /alert %}} 

如果您需要 Aspose.Slides for Java 的協助，請考慮以下事項：

- 主要的支援管道是 Aspose.Forums。請在 Aspose.Slides 論壇發佈您的問題，問題將在數小時內得到回覆。這些問題由 Aspose.Slides 支援團隊直接回答。
- 請注意，Aspose 不提供電話技術支援。電話支援僅限於銷售與購買相關問題。
- 在論壇期待回覆時，請考慮時區差異。

如果您在使用 Aspose.Slides for Java 時遇到問題，請遵循以下簡單步驟，以確保問題以最有效的方式得到解決：

- 在回報問題之前，請確保您使用的是最新的 Aspose.Slides for Java 版本，請參閱 Aspose.Slides 下載以了解最新版本。
- 在回報問題前，請先查看論壇、此文件與 API 參考；或許您的問題已經有答案。
- 回報問題時，請附上原始文件以及可能導致問題的程式碼片段。如需附加多個檔案，請將它們壓縮為一個 zip。將文件附加在 Aspose.Forums 是安全的，因為只有您與 Aspose 開發人員能存取這些檔案。
- 請盡量在每個討論串中只回報一個問題。如有其他問題，請另開新討論串回報。

## **常見問題**

**有哪些付費支援層級？它們有何不同？**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services)。它們在包含的事件數量、回應優先順序以及可用的通訊管道上有所不同。

**我可以直接將錯誤升級給工程團隊嗎？**

是的。[付費支援](https://helpdesk.aspose.com/) 允許在問題需要更深入的調查時升級給開發人員。

**是否支援舊版函式庫，且會釋出修補程式嗎？**

您可以使用舊版，但修正僅提供給目前的發行版；不會為舊版提供修補程式。
