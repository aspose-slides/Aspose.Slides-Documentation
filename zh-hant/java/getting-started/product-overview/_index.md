## **歡迎使用 Aspose.Slides for Java！**

Aspose.Slides for Java 是一個 Microsoft PowerPoint® 管理 API，讓 Java 應用程式在沒有 Microsoft PowerPoint® 的情況下讀寫 PowerPoint® 文件。

Aspose.Slides for Java 是首個也是唯一提供管理 PowerPoint® 文件所需功能的元件。

Aspose.Slides for Java 提供許多核心功能，如管理文字、圖形、表格與動畫、在投影片中加入音訊與視訊、預覽投影片、將投影片匯出為 SVG、PDF 以及其他格式。

## **產品說明**

Aspose.Slides for Java 不僅能從不同來源開啟 PowerPoint 檔案，還能以多種方式儲存簡報。例如，您可以將簡報儲存為 PPT；也可以將投影片另存為圖片。

Aspose.Slides for Java 為您提供幾乎所有在 Microsoft PowerPoint 中可能找到的功能。除了表格與各種圖形外，您還可以在投影片中加入不同類型的框架──文字、音訊與視訊框架。

Aspose.Slides for Java 亦提供功能完整的示範與範例，讓開發人員更深入了解 API。

### **簡報功能**

- 從不同來源開啟 Microsoft PowerPoint 簡報。
- 以不同的 Microsoft PowerPoint 格式儲存簡報。
- 從頭建立簡報。
- 在不使用其他元件的情況下將簡報轉換為 PDF 文件。
- 讀取 Open Document Presentation (ODP)。
- 將 PPT 轉換為 PPTX 或反向轉換。
- 讀取或修改簡報的文件屬性。
- 匯出簡報為 PDF
- 匯出簡報為 TIFF
- 匯出簡報為 XPS
- 匯出簡報為 SWF
- 匯出簡報為 PDF 註解 / Tiff 註解
- 匯出簡報為 HTML

### **投影片功能**

- 從頭建立新投影片並加入簡報。
- 透過投影片在簡報中的位置存取任意投影片。
- 複製或克隆投影片至相同或其他簡報。
- 編輯與刪除投影片。
- 自動複製含投影片的母片。
- 預覽投影片。
- 取得或設定投影片名稱。
- 將投影片儲存為圖片。
- 將投影片匯出為 SVG 格式。
- 在簡報中使用多個母片。
- 使用任何支援的樣式填充投影片背景。
- 新增或管理投影片備註。
- 動態設定新母片與樣式。
- 新增或修改投影片切換效果。
- 新增 Morph 轉場

### **圖形與多媒體功能**

- 建立或管理線條、矩形、橢圓與折線等圖形。
- 管理圖形的線條樣式。
- 取得圖形的替代文字。
- 旋轉與翻轉圖形。
- 以不同樣式填充圖形，例如實心、漸層、圖案、紋理與圖片。
- 使用 48 種預定義圖案與 24 種紋理。
- 為圖形指派連結。
- 讀取、修改並為投影片與圖形新增動畫效果。
- 新增圖片至簡報並管理現有圖片。
- 為簡報新增圖片項目符號並管理現有項目符號。
- 為簡報新增音訊框架並管理現有音訊框架。
- 為簡報新增視訊框架並管理現有視訊框架。

### **表格功能**

- 從頭建立表格並加入投影片。
- 調整表格大小與移動表格。
- 存取表格中的任意儲存格。
- 變更任意儲存格的格式與邊框。
- 調整欄與列的寬度與高度。
- 新增與刪除欄與列。
- 為整個表格設定邊框寬度與顏色。
- 套用預設樣式至表格

### **文字與段落功能**

- 取得與設定文字佔位符與文字框中的文字。
- 取得佔位符的類型。
- 變更文字框中的文字格式。
- 新增字型或管理簡報中已使用的字型。
- 變更文字與文字格式。
- 建立或管理文字連結。
- 支援 Tab 與有效的 Tab。
- 為幾乎所有類型的圖形新增文字框。
- 存取與修改文字框的邊距與文字換行設定。
- 存取與修改段落縮排。

### **進階功能**

- 存取嵌入於簡報中的 OLE 物件。
- 鎖定簡報。
- 為簡報新增浮水印。
- 為投影片內的圖形套用動畫。

## **常見問題**

**它是否支援用於封存與可及性的 PDF 相容等級 (PDF/A 與 PDF/UA)？**

是的。您可透過設定[PDF匯出選項](https://reference.aspose.com/slides/zh-hant/java/com.aspose.slides/pdfoptions/)將檔案儲存為 PDF/A-2a/2b/2u、PDF/A-3a/3b，以及 PDF/UA。

**是否有字型替代機制並支援自訂字型以確保正確呈現？**

是的。此函式庫允許您[載入自訂字型](/slides/zh-hant/java/custom-font/)並[定義備援規則](/slides/zh-hant/java/fallback-font/)，在轉換與呈現過程中可靠地替代缺失字形。

**我可以在未完全開啟檔案的情況下偵測檔案是否受密碼保護嗎？**

是的。您可以[檢查簡報](/slides/zh-hant/java/examine-presentation/)以判斷是否需要密碼才能載入完整文件。

**處理與轉換是否需要安裝 Microsoft PowerPoint？**

不需要。Aspose.Slides 為獨立引擎，伺服器或工作站上不需要 PowerPoint。

**在多執行緒環境中處理簡報是否安全？**

是的，您可在不同執行緒中平行處理不同文件；只需避免在[跨執行緒](/slides/zh-hant/java/multithreading/)同時使用同一簡報實例。

**巨集是否會被保留，且我能否在 PPTM/PPSM 檔案中管理 VBA？**

是的。支援含巨集的簡報[如文件所示](/slides/zh-hant/java/presentation-via-vba/)，您亦可[檢查與管理 VBA 專案](https://reference.aspose.com/slides/zh-hant/java/com.aspose.slides/presentation/#getVbaProject--)。

**我能將 PDF 或 HTML 轉回 PowerPoint 投影片嗎？**

是的。您可[匯入 PDF 頁面或 HTML 內容](/slides/zh-hant/java/import-presentation/)以建立或填充簡報中的投影片。

**是否支援 XPS 匯出，且能控制 XPS 輸出的品質與內容？**

是的。[匯出至 XPS](/slides/zh-hant/java/convert-powerpoint-to-xps/)可用，且[儲存選項](https://reference.aspose.com/slides/zh-hant/java/com.aspose.slides/xpsoptions/)允許您調整輸出品質與包含的內容。

**我可以將投影片轉換為圖片並控制輸出品質嗎？**

是的。投影片可[轉換為 PNG、JPEG、GIF、BMP、TIFF](/slides/zh-hant/java/convert-powerpoint-to-png/)等格式，並可細緻控制大小與品質。

**是否支援數學公式 (MathML/MathText) 並能匯出？**

是的。您可[建立與編輯數學文字](/slides/zh-hant/java/powerpoint-math-equations/)並[匯出方程式](/slides/zh-hant/java/exporting-math-equations/)，包括匯出為 MathML。

**函式庫如何處理加密與密碼（開啟、設定、移除）？**

[它支援](/slides/zh-hant/java/password-protected-presentation/)開啟加密簡報、設定或移除開啟與修改的密碼，並檢查檔案的保護狀態。
