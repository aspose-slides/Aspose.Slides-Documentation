Aspose.Slides for Java 14.3.0（14.4.0）提供了全新的 PPT 處理實作。新增了許多改進與 PPTX 轉換為 PPT 的部分功能。但仍有一些未實作的功能：

- 某些圖形在序列化的 PPT 文件中幾何形狀錯誤（Call outs）
- 不是所有 PPTX 文字格式功能都在 PPT 序列化中受到支援
- 序列化的 PPT 文件中未包含文字語言與拼寫設定資訊
- 不是所有 PPTX 主題的功能都在 PPT 序列化中受到支援

**與 Aspose.Slides for Java 8.6.0 相比，有一些差異：**

- 已知 OLE/ActiveX PPT 序列化為 PPT 時存在問題

**與 Aspose.Slides for .NET 14.3.0 相比，有一些差異：**

- 目前在 Aspose.Slides for Java 中不支援簡報列印功能
