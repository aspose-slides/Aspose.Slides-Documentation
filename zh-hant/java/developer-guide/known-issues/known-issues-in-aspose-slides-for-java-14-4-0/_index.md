{{% alert color="primary" %}} 
Aspose.Slides for Java 14.4.0 為 PowerPoint 文件處理提供了新的決策。以下是一些限制與已知問題，這些問題將在未來的版本中移除：

- 某些形狀在序列化的 PPT 文件中幾何形狀不正確（弧形、圓形箭頭、標註）。
- 並非所有 PPTX 文字格式設定功能在 PPT 序列化時都受支援（制表符、縮排和段落格式的限制）。
- 序列化的 PPT 文件中不包含文字語言與拼寫設定資訊。
- 並非所有 PPTX 主題功能在 PPT 序列化時都受支援（僅序列化填充格式、線條格式和字型）。
- OLE/ActiveX PPT 序列化為 PPT 時存在已知問題。
- 不支援 WordArt 的序列化與呈現。
{{% /alert %}}
