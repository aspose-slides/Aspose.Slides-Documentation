## **支援舊版程式碼**
為了使用在 13.x 之前的 Aspose.Slides for Java 版本中開發的舊版程式碼，您需要對程式碼做少量修改，程式碼即可如同以前般運作。舊版 Aspose.Slides for Java 中位於 Aspose.Slide 和 Aspose.Slides.Pptx 命名空間的所有類別，現在已合併到單一的 Aspose.Slides 命名空間。請檢視以下用於在舊版 Aspose.Slides API 中建立 Hello World 簡報文件的簡易程式碼片段，並依照說明步驟遷移至新的合併 API。
