## **支援舊版程式碼**
若要使用在 14.x.x 之前的 Aspose.Slides for Java 版本開發的舊版程式碼，您只需對程式碼做少量修改，即可如同之前般正常運作。舊版 Aspose.Slides for Java 中位於 com.aspose.slides 與 com.aspose.slides.pptx 名稱空間的所有類別，現在已合併到單一的 com.aspose.slides 名稱空間。請參考以下簡單的程式碼片段，瞭解如何使用舊版 Aspose.Slides API 從頭建立一般圖表，並依照說明步驟遷移至新合併的 API。
