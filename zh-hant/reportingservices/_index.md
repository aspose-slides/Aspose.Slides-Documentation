**歡迎使用 Aspose.Slides for Reporting Services**

Aspose.Slides for Reporting Services 是市場上唯一能在 Microsoft SQL Server 2005、2008、2012、2016 和 2017 Reporting Services（32 位元和 64 位元）中產生真正 PPT 與 PPS 報表的解決方案。所有 RDL 報表功能，包括表格、矩陣、圖表和影像，都以最高精度轉換為 Microsoft PowerPoint 簡報。
