## **歡迎使用 Aspose.Slides for Reporting Services！**

Aspose.Slides for Reporting Services 是市場上唯一能夠在 Microsoft SQL Server 2005、2008、2012、2016 及 2017 Reporting Services（32 位元與 64 位元）中產生真正 PPT 與 PPS 報表的解決方案。所有 RDL 報表功能，包括表格、矩陣、圖表與影像，都會以最高精度轉換為 Microsoft PowerPoint 簡報。

## **產品概覽**

Microsoft SQL Server Reporting Services 本身不具備將報表匯出為 Microsoft PowerPoint 簡報的功能，但在伺服器上安裝 Aspose.Slides for Reporting Services 後，即可使用以下額外的匯出格式：

- PPT - 透過 Aspose.Slides 的 PowerPoint 簡報
- PPS - 透過 Aspose.Slides 的 PowerPoint 投影片放映
- PPTX - 透過 Aspose.Slides 的 PowerPoint 2007 簡報
- PPSX - 透過 Aspose.Slides 的 PowerPoint 2007 投影片放映

Aspose.Slides for Reporting Services 在伺服器上建立簡報，無需使用 Microsoft PowerPoint。Aspose.Slides for Reporting Services 內部使用 Aspose.Slides for .NET——這是一個世界級的伺服器端簡報處理元件。

**Aspose.Slides for Reporting Services 使任何報表都能匯出為 PPT、PPS、PPTX 或 PPSX 格式。**
