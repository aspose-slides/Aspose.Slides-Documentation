{{% alert color="primary" %}} 

本文說明如何在伺服器上安裝 Aspose.Slides for Reporting Services。

{{% /alert %}} 
### **安裝選項**
Aspose.Slides for Reporting Services 可以透過兩種方式部署：

* 使用 MSI 安裝程式自動安裝
* 手動複製組件並修改配置檔案

自動安裝的 MSI 安裝程式 (Aspose.Slides for Reporting Services XX.XX) 以及手動安裝的 zip 套件 (Aspose.Slides for Reporting Services XX.XX DLL Only) 可從 [產品發布頁面](https://releases.aspose.com/slides/zh-hant/reportingservices/) 下載。

{{% alert color="primary" %}} 

Aspose.Slides for Reporting Services 可以安裝在 Power BI 伺服器上。

{{% /alert %}}
