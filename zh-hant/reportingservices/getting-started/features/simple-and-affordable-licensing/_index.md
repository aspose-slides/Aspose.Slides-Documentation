## 授權類型
{{% alert color="primary" %}} 目前提供兩種授權類型：Server Subscription Site Subscription {{% /alert %}}
每個授權皆附帶一年免費升級至任何新版本，以及一年無限次免費技術支援。

Aspose.Slides for Reporting Services 的評估版與付費版使用相同的下載。只需下載 Aspose.Slides for Reporting Services 並安裝到伺服器，即可在評估模式下使用。

請務必利用 Aspose.Slides for Reporting Services 的評估版，因為它沒有時間限制，且會提供給評估使用者免費技術支援。

評估模式會在匯出的報告中加入浮水印。當您[購買授權](https://purchase.aspose.com/buy)時，只需將授權檔案複製到相應資料夾，Aspose.Slides for Reporting Services 即會切換至授權模式。
## **Server Subscription**
Server subscription 提供以下功能：

- 每台實體或虛擬伺服器都需要一個授權。
- 處理器或核心數量不限。
- Microsoft SQL Server 執行個體數量不限。
- 開發人員數量不限。
- 1 年內免費提供新版本與 hot fix。
- 1 年內免費技術支援。
- 多筆購買可享折扣。
## **Site Subscription**
Site subscription 提供以下功能：

- 每個實體站點（校區）需要一個授權。
- 實體或虛擬伺服器數量不限。
- 處理器或核心數量不限。
- Microsoft SQL Server 執行個體數量不限。
- 開發人員數量不限。
- 1 年內免費提供新版本與 hot fix。
- 1 年內免費技術支援。
- 多筆購買可享折扣。
