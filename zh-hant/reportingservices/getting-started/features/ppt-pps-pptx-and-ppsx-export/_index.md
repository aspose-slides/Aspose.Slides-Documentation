{{% alert color="primary" %}} 

Microsoft SQL Server Reporting Services 沒有內建將報告匯出為 Microsoft PowerPoint 簡報的功能，但在您的伺服器上安裝 Aspose.Slides for Reporting Services 後，您將可使用另外兩種匯出格式：

- Microsoft PowerPoint 簡報 (PPT)
- Microsoft PowerPoint 投影片放映 (PPS)
- Microsoft PowerPoint 2007 簡報 (PPTX)
- Microsoft PowerPoint 2007 投影片放映 (PPSX)

{{% /alert %}} 

要在這些格式中建立文件，Aspose.Slides for Reporting Services 依賴內建的 [Aspose.Slides for .NET](http://www.aspose.com/Products/Aspose.Slides/) 版本，這是 Aspose 所提供的市場領先簡報處理元件。Microsoft PowerPoint 並未用於產生這些文件。 

由於 Aspose.Slides 提供對底層檔案格式的高度支援，同一報告的 PPT 與 PPS 檔案會完全相同。
