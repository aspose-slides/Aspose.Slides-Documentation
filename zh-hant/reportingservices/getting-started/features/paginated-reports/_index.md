{{% alert color="primary" %}} 

Aspose.Slides for Reporting Services 已實作對 Power BI 的支援。 

{{% /alert %}} 

分頁報表是針對頁面進行格式化的報表。此類報表非常適合匯出作業和列印。分頁報表的設計方式是提供足夠的頁數，以列印或匯出表格中的每一列資料。 

例如，若您想建立一份報表，讓該報表的擁有者能檢視過去 12 個月內每項商品的數量、銷售價格與利潤，則分頁報表就是您需要的。 

為了讓您能建立分頁報表，Microsoft 在 Power BI 中提供了 Power BI Report Builder。當您建立分頁報表並將其儲存至本機後，產生的檔案會使用 .rdl 副檔名。
