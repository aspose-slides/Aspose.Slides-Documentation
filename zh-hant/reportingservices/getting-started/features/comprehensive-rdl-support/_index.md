{{% alert color="primary" %}} 

Aspose.Slides for Reporting Services 完全支援所有 RDL 規範。這意味著兩件好事：

- 無需重新設計現有的報告。您可以將任何現有的 RDL 報告匯出為 Microsoft PowerPoint 簡報，且它將完全依照 RDL 設計呈現。
- 無需使用特定的報告設計器。您可以使用任何 RDL 報告設計器，報告將以您設計的方式完整匯出。

{{% /alert %}} 

Aspose.Slides for Reporting Services 支援以下 RDL 元素：

- 頁面、頁首、頁尾
- 文字方塊
- 圖像
- 子報告
- 圖表
- 清單
- 表格
- 矩陣
- 樣式
- 矩形

**一個包含頁首、頁尾、圖像、子報告、表格、文字方塊和矩形，並匯出為 Microsoft PowerPoint (PPT) 簡報的報告範例。**
