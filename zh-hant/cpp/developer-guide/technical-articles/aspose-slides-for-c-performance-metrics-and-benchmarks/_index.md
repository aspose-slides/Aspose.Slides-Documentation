## **目的**
效能通常是選擇元件時的首要關鍵因素。本文測量了 Aspose.Slides for C++ 與 VSTO 2008 的效能。這些簡單測試在相似的作業系統、硬體元件和組態下執行。 

本文呈現包括 **Aspose.Slides for C++** 與 **VSTO 2008** 在內的產品效能測量。此處提供的效能估計旨在協助您了解在相似組態、商用硬體及廣泛使用的作業系統下，不同元件在常見使用情境中的表現。自然地，您的應用程式效能取決於資料、資料存取模式、快取大小、其他組態參數、作業系統與硬體等。此基準測試旨在說明元件在最低硬體條件下的表現，硬體越快，元件處理任務的速度亦會越快。

## **測試方法**
所有效能測試皆在相同的硬體與作業系統組合上執行，未使用自訂組態、調校或其他效能提升技術。所有測試均在同一系統上執行元件安裝，該系統在其他情況下處於閒置狀態。為獲得精確的讀數，我們一次執行所有任務兩至三次，以更好地評估元件並取得準確的結果。
