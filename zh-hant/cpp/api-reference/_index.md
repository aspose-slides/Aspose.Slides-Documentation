{{% alert color="primary" %}} 

Aspose.Slides for C++ 是一個類別庫，讓 C++ 應用程式能在不使用 Microsoft PowerPoint® 或其他軟體的情況下讀寫簡報文件。Aspose.Slides for C++ 可用於管理文字、形狀、表格與動畫，將音訊與影片加入投影片，預覽投影片，將投影片匯出為 SVG、PDF 格式等。

最新的 API 參考可於 [Aspose.Slides for C++ API Reference](https://reference.aspose.com/slides/zh-hant/cpp/) 找到。
