## **歡迎使用 Aspose.Slides for C++！**

Aspose.Slides for C++ 是一套 Microsoft PowerPoint® 管理 API，讓 C++ 應用程式能在不使用 Microsoft PowerPoint® 的情況下讀寫 PowerPoint® 文件。Aspose.Slides for C++ 是首個亦唯一提供 PowerPoint® 文件管理功能的元件。Aspose.Slides for C++ 提供許多關鍵功能，如管理文字、圖形、表格與動畫、在投影片中加入音訊與影片、預覽投影片、將投影片匯出為 SVG、PDF 等格式。

## **產品說明**
Aspose.Slides for C++ 使用原生 C++ 開發。現在已編譯為 Windows x64 環境，也即將支援 Linux 環境。

Aspose.Slides for C++ 不僅可以開啟來自不同來源的 PowerPoint 檔案，還允許您以多種方式儲存簡報。例如，您不僅可以儲存為 PPT，還可以儲存為 PDF 格式。而且，您也可以將投影片儲存為影像。Aspose.Slides for C++ 為您提供幾乎所有在 Microsoft PowerPoint 中可找到的功能。Aspose.Slides for C++ 的多樣性在於，您不僅可以在投影片中新增表格與各種圖形，還可以新增各種框架（包括文字、音訊與影片框架）。

## **豐富功能集**
Aspose.Slides for C++ 的功能可分為以下幾個領域：

1. 簡報功能。
1. 投影片功能。
1. 圖形與多媒體功能。
1. 表格功能。
1. 文字與段落功能。
1. 進階功能。

### **簡報功能**
- 從不同來源開啟 Microsoft PowerPoint 簡報。
- 以不同的 Microsoft PowerPoint 格式儲存簡報。
- 從頭建立簡報。
- 在不使用其他元件的情況下將簡報轉換為 PDF 文件。
- 讀取或修改簡報的文件屬性。
- 為簡報套用保護。
- 將簡報匯出為 PDF。
- 將簡報匯出為 XPS。
- 將簡報匯出為 SWF。
- 將簡報匯出為 PDF 註解。
- 將簡報匯出為 HTML。

### **投影片功能**
- 從頭建立新投影片並加入簡報。
- 透過投影片在簡報中的位置存取任意投影片。
- 複製或克隆投影片至相同或其他簡報。
- 編輯與刪除投影片。
- 自動複製含投影片的母片。
- 預覽投影片。
- 取得或設定投影片名稱。
- 將投影片儲存為影像。
- 將投影片匯出為 SVG 格式。
- 在簡報中使用多個母片。
- 以任何支援的樣式填滿投影片背景。
- 新增或管理投影片註解。
- 動態設定新母片與樣式。
- 新增或修改投影片放映過渡效果。
- 新增 Morph 轉場。

### **圖形與多媒體功能**
- 建立或管理線條、矩形、橢圓與折線等圖形。
- 管理圖形的線條樣式。
- 取得圖形的替代文字。
- 旋轉或翻轉圖形。
- 以各種樣式填充圖形，如實色、漸層、圖案、紋理與圖片。
- 使用 48 種預定義圖案與 24 種紋理。
- 為圖形指派連結。
- 讀取、修改或新增投影片與圖形的動畫效果。
- 新增圖片至簡報或管理現有圖片。
- 為簡報新增圖片項目符號或管理現有項目符號。
- 為簡報新增音訊框架或管理現有音訊框架。
- 為簡報新增影片框架或管理現有影片框架。

### **表格功能**
- 從頭建立表格並加入投影片。
- 調整表格大小與位置。
- 存取表格中的任意儲存格。
- 更改任儲存格的格式與邊框。
- 調整欄與列的寬高。
- 新增與刪除欄與列。
- 為整個表格設定邊框寬度與顏色。

### **文字與段落功能**
- 取得與設定文字佔位符與文字框中的文字。
- 取得佔位符的類型。
- 更改文字框中的文字格式。
- 新增或管理簡報中使用的字型。
- 更改文字與文字格式化。
- 建立或管理文字連結。
- 支援定位點與有效定位點。
- 為幾乎所有圖形類型新增文字框。
- 取得與修改文字框的邊距與換行設定。
- 取得與修改段落縮排。

### **進階功能**
- 存取簡報中嵌入的 OLE 物件。
- 鎖定簡報。

## **支援平台**
- Windows x64/x86（Microsoft Visual Studio 2017 或更新版本）。
- Linux（Clang 3.9 或更新版本、GCC 6.1 或更新版本）。
- macOS（Xcode 13.4 或更新版本）。

## **常見問與答**

**是否支援 PDF 封存與無障礙等級（PDF/A 與 PDF/UA）？**

是的。您可以透過設定 [PDF 匯出選項](https://reference.aspose.com/slides/zh-hant/cpp/aspose.slides.export/pdfoptions/) 以 PDF/A-2a/2b/2u、PDF/A-3a/3b 以及 PDF/UA 格式儲存。

**是否提供字型替代機制並支援自訂字型以確保正確渲染？**

是的。此函式庫允許您 [載入自訂字型](/slides/zh-hant/cpp/custom-font/) 並 [定義回退規則](/slides/zh-hant/cpp/fallback-font/)，在轉換與渲染時可靠地替代缺失的字形。

**我可以在不完整開啟檔案的情況下檢測檔案是否受密碼保護嗎？**

是的。您可以 [檢查簡報](/slides/zh-hant/cpp/examine-presentation/) 以判斷是否需要密碼才能載入完整文件。

**處理與轉換是否必須安裝 Microsoft PowerPoint？**

不需要。Aspose.Slides 為獨立引擎，伺服器或工作站上無需安裝 PowerPoint。

**多執行緒處理簡報是否安全？**

是的，您可以在不同執行緒中平行處理不同文件；只需避免在多執行緒間同時使用同一簡報實例，請參考 [跨執行緒使用](/slides/zh-hant/cpp/multithreading/)。

**巨集是否會被保留，且我能管理 PPTM/PPSM 檔案中的 VBA 嗎？**

是的。支援含巨集的簡報，您可以 [檢查與管理 VBA 專案](https://reference.aspose.com/slides/zh-hant/cpp/aspose.slides.vba/)，相關檔案亦受支援。

**我可以將 PDF 或 HTML 轉回 PowerPoint 投影片嗎？**

可以。您可以 [匯入 PDF 頁面或 HTML 內容](/slides/zh-hant/cpp/import-presentation/) 以建立或填充簡報投影片。

**是否支援 XPS 匯出，且可調整 XPS 輸出的品質與內容？**

是的。提供 [匯出為 XPS](/slides/zh-hant/cpp/convert-powerpoint-to-xps/)，且透過 [儲存選項](https://reference.aspose.com/slides/zh-hant/cpp/aspose.slides.export/xpsoptions/) 調整輸出品質與包含內容。

**我可以將投影片轉為影像並控制輸出品質嗎？**

可以。投影片可 [轉換為 PNG、JPEG、GIF、BMP、TIFF](/slides/zh-hant/cpp/convert-powerpoint-to-png/) 等格式，並可細緻控制大小與品質。

**是否支援數學公式（MathML/MathText）且可匯出？**

是的。您可以 [建立與編輯數學文字](/slides/zh-hant/cpp/powerpoint-math-equations/) 並 [匯出公式](/slides/zh-hant/cpp/exporting-math-equations/)，包括匯出為 MathML。

**函式庫如何處理加密與密碼（開啟、設定、移除）？**

[支援](/slides/zh-hant/cpp/password-protected-presentation/) 開啟加密簡報、設定或移除開啟與修改的密碼，並可檢查檔案的保護狀態。
