## **Aspose.Slides 評估**

您可以輕鬆下載 Aspose.Slides 進行評估。評估版下載與購買版下載相同。當您加入少量程式碼以套用授權時，評估版會直接變為已授權。

Aspose.Slides 的評估版本（未指定授權）提供完整的產品功能，但在開啟或儲存文件時會在文件頂部插入評估水印，且在從簡報投影片中擷取文字時僅限制為一張投影片。

## **常見問題**

**我可以在評估模式下於不同執行緒中同時測試多個簡報嗎？**

是的。您可以平行處理不同的文件；不應該在不同執行緒間共享相同的簡報物件[跨執行緒](/slides/zh-hant/cpp/multithreading/)。評估模式不會受到影響。

**我需要在伺服器或 CI 上安裝 Microsoft PowerPoint 以評估此函式庫嗎？**

不需要。Aspose.Slides 為獨立的引擎，無論在評估或正式環境均不需要安裝 PowerPoint。

**我可以在評估模式下完整測試 PPT/PPTX 轉換為 PDF 以及影像的功能嗎？**

是的。此[轉換器](/slides/zh-hant/cpp/convert-presentation/)可以正常運作；輸出結果會包含水印。

**我可以使用臨時授權進行負載測試且不顯示水印嗎？**

是的。30 天的臨時授權會移除評估模式的限制，允許在沒有水印的情況下進行測試。
