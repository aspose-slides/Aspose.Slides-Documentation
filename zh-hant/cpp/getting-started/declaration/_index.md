{{% alert color="primary" %}} 

所有 Aspose C++ 元件都需要 Full Trust 權限設定。原因在於，Aspose C++ 元件需要存取登錄檔設定、系統檔案（虛擬目錄之外）的特定操作，例如字型解析等。此外，Aspose C++ 元件是基於核心 C++ 系統類別，在許多情況下也需要 Full Trust 權限設定。

{{% /alert %}}
