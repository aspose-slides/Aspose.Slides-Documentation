## **軟體需求**
請確保在下載與執行範例之前符合以下需求。

1. Visual Studio 2015 或更高版本。

## **從GitHub下載Aspose.Slides**
所有 Aspose.Slides for C++ 的範例皆托管於[GitHub](https://github.com/aspose-slides/Aspose.Slides-for-C)。

- 您可以使用喜愛的 GitHub 用戶端克隆儲存庫，或從[here](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/refs/heads/master.zip)下載 ZIP 檔案。
- 將 ZIP 檔案的內容解壓縮到電腦上的任意資料夾。所有範例位於 **Examples** 資料夾中。
- 提供 C++ Visual Studio 解決方案檔案。
- 專案是以 Visual Studio 2015 建立，但解決方案檔案相容於 Visual Studio 2015 及以上版本。
- 在 Visual Studio 中開啟解決方案檔案並建置專案。
- 從[Download Section](https://downloads.aspose.com/slides/zh-hant/cpp)下載 Aspose.Slides for C++ 最新發行版。
- 解壓縮發行版，並將 **Include** 與 **Bin** 目錄複製到 **Examples** 資料夾內。
- 開啟 **main.cpp** 檔案，所有範例皆從此呼叫。
- 取消註解您想在專案中執行的範例。
- 建置並編譯解決方案。

如在設定或執行範例時遇到任何問題，歡迎透過我們的論壇聯絡我們。

## **貢獻**
若您想新增或改進範例，我們鼓勵您為專案貢獻。此儲存庫中的所有範例與示範專案皆為開源，您可自由在自己的應用程式中使用。

要貢獻，您可以 Fork 此儲存庫、編輯原始碼並建立 Pull Request。我們會審查變更，若有幫助將納入儲存庫。
