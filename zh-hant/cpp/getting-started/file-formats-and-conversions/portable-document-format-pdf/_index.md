## **關於 PDF**
[Portable Document Format](https://en.wikipedia.org/wiki/PDF) 是由 Adobe System 所建立的檔案格式，旨在於不同組織之間交換文件。此格式的目的是讓文件內容的視覺外觀不受檢視平台限制。

## **Aspose.Slides for C++ 中的 PDF**
任何能載入至 Aspose.Slides for C++ 的簡報文件，都可以轉換為符合 [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A) 或 [PDF /A-1b](https://en.wikipedia.org/wiki/PDF/A) 的 PDF 文件，具體依您的選擇而定。Aspose.Slides for C++ 以一種方式匯出簡報文件為 PDF，使得大多數情況下，匯出的 PDF 檔案看起來與原始簡報文件幾乎相同。

Aspose 解決方案在將簡報文件轉換為 PDF 時支援以下簡報功能：

- 圖像、文字方塊與其他形狀
- 文字與格式設定
- 段落與格式設定
- 超連結
- 頁首與頁尾
- 项目符号
- 表格

您可以僅使用 Aspose.Slides for C++ 元件直接將簡報文件匯出為 PDF，無需任何其他第三方或 Aspose.Pdf 元件。此外，您還可以依照 [this topic](http://docs.aspose.com/display/slidesnet/Converting+to+PDF+File) 中說明的不同選項，自訂簡報到 PDF 的匯出方式。
