## **關於 PPT**
[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) 是一種演示文件格式，可由不同版本的 Microsoft PowerPoint 建立、讀取、操作與寫入。這是 Microsoft 開發的二進位演示文件格式。 
## **Aspose.Slides for C++ 中的 PPT**
Aspose.Slides for C++ 能讀取以下軟體所建立的 PPT 檔案。 

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

同樣地，由 Aspose.Slides for C++ 建立的 PPT 檔案也可被上述軟體讀取。 
## **對 PPT 的全面支援**
Aspose.Slides for C++ 為幾乎所有與 PPT 文件格式相關的功能提供支援。它不僅涵蓋不同 Microsoft PowerPoint 版本所提供的基本/進階功能，還包括 Microsoft PowerPoint 本身尚未支援的功能。使用 Aspose.Slides for C++ API 函式庫的主要優點在於處理這些功能的便利性。 

除了建立、讀取與寫入 PPT 文件的基本任務外，Aspose.Slides for C++ 還提供以下多項功能： 

- 將其他 MS Office 檔案格式匯入為 PPT 文件中的 OLE 物件
- 將 PPT 文件匯出為 PDF、TIFF、XPS 格式
- 將 PPT 文件中的投影片匯出為 SVG 格式
- 將投影片渲染為 C++ 框架支援的任何影像格式
- 設定 PPT 文件中投影片的大小
- 管理圖形的動畫
- 管理投影片放映
- 格式化投影片上的文字
- 從 PPT 文件掃描文字
- 處理投影片上的表格
- 使用複製功能自動複製母片
