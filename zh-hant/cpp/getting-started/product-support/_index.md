## **概觀**

本文說明如何取得 Aspose.Slides 的支援以及如何有效回報問題。它描述主要的支援管道，提供提交問題的建議，並指向產品頁面、部落格、下載、支援論壇、文件與 API 參考等實用技術資源。本文亦回答關於付費支援、問題升級以及舊版函式庫支援的常見問題。

## **技術支援**
Aspose 為其所有產品提供無限制的免費技術支援。此支援對所有使用者皆可使用，包含評估版。  
若您需要 Aspose.Slides for C++ 的協助，請考慮以下事項：

- 主要的支援管道是 Aspose 論壇。將您的問題發布於 Aspose.Slides 論壇，幾小時內即可得到回覆。問題由 Aspose.Slides 支援團隊直接回答。
- 請注意，Aspose 並未提供電話技術支援。電話支援僅限於銷售與購買相關問題。
- 在論壇等待回覆時，請考慮時區差異。  
  若您在 Aspose.Slides for C++ 上遇到問題，請遵循以下簡單步驟，以確保以最高效的方式解決問題：
  - 在回報問題前，請確保您使用的是最新的 Aspose.Slides for C++ 版本，請參閱 Aspose.Slides 下載頁面以取得最新版本資訊。
- 在回報問題前，請先瀏覽論壇、本文檔與 API 參考；您的問題可能已被回答。
- 回報問題時，請附上原始文件，並盡可能提供導致問題的程式碼片段。若需附加多個檔案，請將它們壓縮成一個 zip。將文件上傳至 Aspose.Forums 是安全的，因為只有您與 Aspose 開發人員可以存取這些附件。
- 請盡量在每個討論串中僅回報一個問題。如有其他問題，請另開新討論串。

## **常見問題**

**有哪些付費支援等級？它們有何不同？**

[Developer、Business、Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services)。它們依據包含的事件數量、回應優先順序以及可使用的通訊管道而有所差異。

**我可以直接將錯誤升級給工程團隊嗎？**

可以。[付費支援](https://helpdesk.aspose.com/) 在問題需要更深入調查時，允許升級至開發人員。

**舊版函式庫是否仍受支援？會釋出修補嗎？**

您可以使用舊版，但僅提供目前版本的修正；不會為舊版發布修補程式。
