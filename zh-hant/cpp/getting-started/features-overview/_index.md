## **檔案格式與轉換**
Aspose.Slides for C++ 支援大多數 PowerPoint 文件格式，並將其匯出為在組織間廣泛使用的流行格式。以下表格列出這些格式：

|**功能**|**說明**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/zh-hant/cpp/microsoft-powerpoint-ppt/)|Aspose.Slides for C++ 為此簡報文件格式提供最快速的處理速度。|
|[Portable Document Format (PDF)](/slides/zh-hant/cpp/portable-document-format-pdf/)|支援的檔案格式可透過單一方法匯出為 Adobe Portable Document Format (PDF) 文件。|
|[XML Parser Specification (XPS)](/slides/zh-hant/cpp/xml-parser-specification-xps/)|所有支援的檔案格式可透過單一方法匯出為 XML Parser Specification (XPS) 文件。|
|[PresentationML (PPTX, XML)](/slides/zh-hant/cpp/presentationml-pptx-xml/)|Aspose.Slides for C++ 支援 OOXML 簡報格式，也稱為 PresentationML 或 PPTX 的處理。|
|[PPTX To HTML Conversion](https://docs.aspose.com/slides/zh-hant/cpp/convert-powerpoint-to-html/)|Aspose.Slides for C++ 支援將簡報轉換為 HTML 格式。|

## **呈現**
Aspose.Slides for C++ 支援以高保真度將簡報文件中的投影片呈現在各種圖形格式，如下所述：

|**功能**|**說明**|
| :- | :- |
|C++ 支援的影像格式|使用 Aspose.Slides for C++，您不僅能將簡報投影片呈現，還能將投影片上的影像以所有 C++ 支援的圖形格式（如 PNG、BMP 與 JPEG）進行呈現。|
|SVG 格式|Aspose.Slides for C++ 亦提供內建方法，將簡報投影片匯出為可擴充向量圖形（SVG）格式。|

## **內容功能**
Aspose.Slides for C++ 讓您能存取、修改或建立簡報文件中幾乎所有可能的內容。

|**功能**|**說明**|
| :- | :- |
|母片投影片|母片投影片定義一般投影片的版面配置。Aspose.Slides for C++ 讓您能存取並修改簡報文件的母片投影片。|
|一般投影片|使用 Aspose.Slides for C++，您不僅能建立不同類型的新投影片，亦可存取與修改簡報文件中現有的投影片。|
|克隆 / 複製投影片|Aspose.Slides for C++ 提供內建方法，可克隆或複製現有投影片，不僅限於同一簡報內，也可在不同簡報之間進行。由於投影片會繼承其母片的版面配置，內建的克隆方法在克隆時會自動一併複製母片。|
|占位元件與文字占位元件|您也可以存取投影片中的占位元件與文字占位元件。此外，您還能使用相應方法從頭建立包含文字占位元件的投影片。|
|頁首與頁尾|Aspose.Slides for C++ 亦便利於在投影片中處理頁首／頁尾。|
|投影片備註|使用 Aspose.Slides for C++，您不僅能存取與修改投影片相關的備註，亦可新增備註。|
|尋找形狀|您亦可利用形狀的替代文字，從投影片中尋找特定形狀。|
|背景|Aspose.Slides for C++ 可處理與母片或一般投影片相關的背景。|
|文字方塊|文字方塊可以從頭建立。現有的文字方塊可以存取，且其文字也能在不失去原始格式的情況下進行修改。|
|矩形形狀|Aspose.Slides for C++ 可建立或修改矩形形狀。|
|折線形狀|Aspose.Slides for C++ 亦可建立或修改折線形狀。|
|橢圓形狀|Aspose.Slides for C++ 亦可建立或修改橢圓形狀。|
|群組形狀|Aspose.Slides for C++ 亦支援群組形狀。|
|自動圖案|Aspose.Slides for C++ 亦支援自動圖案。|
|形狀序列化|雖然 Aspose.Slides for C++ 支援大量形狀，但若某個形狀未被支援，Aspose.Slides for C++ 提供序列化方法，您可以將該形狀從現有投影片序列化，依需求供 Aspose.Slides for C++ 後續使用。|
|圖片框架|圖片透過 Aspose.Slides for C++ 在圖片框架中管理。|
|音訊框架|音訊檔案可由 Aspose.Slides for C++ 以連結或嵌入方式放置於投影片的音訊框架中。|
|視訊框架|視訊檔案透過 Aspose.Slides for C++ 在視訊框架中處理。|
|OLE 框架|OLE 物件由 Aspose.Slides for C++ 在 OLE 框架中管理。|
|表格|投影片上的表格亦受到 Aspose.Slides for C++ 支援。|
|文字框架|任何形狀相關的文字皆可透過該形狀的文字框架存取。|
|文字掃描|可透過內建掃描方法，在簡報或投影片層級掃描文字。|
|動畫|可對形狀套用動畫。|
|投影片放映|亦支援投影片放映、投影片切換與變形切換。|

使用 Aspose.Slides for C++，可以對簡報文件中投影片的文字與形狀進行格式化。

|**功能**|**說明**|
| :- | :- |
|文字格式化|<p>在 Aspose.Slides for C++ 中，文字透過與形狀關聯的文字框架管理。因此，可使用文字框架中的段落與文字區塊來格式化文字。以下文字元素可透過 Aspose.Slides for C++ 進行格式化。</p><p>- 字型類型</p><p>- 字型大小</p><p>- 字型顏色</p><p>- 字型色階</p><p>- 段落對齊</p><p>- 段落項目符號</p><p>- 段落方向</p>|

## **常見問題**

**我需要在伺服器/電腦上安裝 Microsoft PowerPoint 才能使用此函式庫嗎？**

不需要。PowerPoint 並非必備；Aspose.Slides 為獨立的引擎，提供簡報的建立、編輯、轉換與呈現功能。

**多執行緒如何運作？可以平行處理嗎？**

相同的 [簡報](https://reference.aspose.com/slides/zh-hant/cpp/aspose.slides/presentation/) 物件不得同時被 [多執行緒](/slides/zh-hant/cpp/multithreading/) 使用。

**是否支援檔案密碼與加密？**

是的。[您可以](/slides/zh-hant/cpp/password-protected-presentation/) 開啟受加密的簡報，設定或移除開啟與寫入密碼，並檢查保護狀態。

**我需要關注 Linux 容器中的字型套件嗎？**

是的。建議在您的應用程式中安裝常見的字型套件，和/或明確 [指定字型目錄](/slides/zh-hant/cpp/custom-font/) 以避免意外的字型替換。

**評估版是否有任何限制？**

在 [評估模式](/slides/zh-hant/cpp/licensing/) 中，輸出會加上浮水印，且會套用某些限制；可取得 [30 天臨時授權](https://purchase.aspose.com/temporary-license/) 以完整測試所有功能。

**是否支援將外部格式匯入簡報（PDF/HTML → PPTX）？**

是的。您可以將 [PDF 頁面與 HTML 內容](/slides/zh-hant/cpp/import-presentation/) 新增至簡報，並將其轉換為投影片。
