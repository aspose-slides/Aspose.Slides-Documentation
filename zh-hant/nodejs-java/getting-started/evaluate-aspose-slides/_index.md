## **Aspose.Slides 評估**

您可以輕鬆下載 Aspose.Slides 進行評估。評估套件與購買套件相同。只要在程式碼中加入幾行授權程式，就能讓評估版轉為正式授權版。

Aspose.Slides 的評估版（未指定授權）提供完整的產品功能，但在開啟和儲存文件時會在文件頂部插入評估水印。從簡報投影片提取文字時也僅限於一張投影片。

## **常見問題**

**我可以在評估模式下於不同執行緒平行測試多個簡報嗎？**

可以。您可以平行處理不同的文件；不應該在執行緒間共享同一個簡報物件 [跨執行緒](/slides/zh-hant/net/multithreading/)。評估模式不會影響此行為。

**我需要在伺服器或 CI 上安裝 Microsoft PowerPoint 來評估此函式庫嗎？**

不需要。Aspose.Slides 為獨立的引擎，無論在評估或正式環境都不需要安裝 PowerPoint。

**我能在評估模式下完整測試 PPT/PPTX 轉換為 PDF 及影像嗎？**

可以。 [轉換器](/slides/zh-hant/net/convert-presentation/) 可正常運作；輸出結果會包含水印。

**我可以使用臨時授權進行負載測試而不出現水印嗎？**

可以。30 天的臨時授權會移除評估模式的限制，允許在測試時不顯示水印。
