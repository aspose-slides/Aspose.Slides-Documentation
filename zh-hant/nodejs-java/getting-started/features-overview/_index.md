## **支援平台**
Aspose.Slides for Node.js via Java 是跨平台的 API。它可在任何平台（Windows、Linux、MacOS 等）上使用，只要已安裝 Node.js 7 或更高版本。Oracle JDK 8 或更高版本是另一項需求。

## **檔案格式與轉換**
Aspose.Slides for Node.js via Java 支援大多數 PowerPoint 文件格式。它亦允許您將其匯出為組織間常用且共享的流行檔案格式。請查閱以下詳細資訊：

|**功能**|**說明**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via Java 為此簡報文件格式提供最快的處理速度。|
|PPT 轉換為 PPTX|Aspose.Slides for Node.js via Java 支援 PPT 轉換為 PPTX。|
|Portable Document Format (PDF)|您可以使用單一方法將所有支援的檔案格式匯出為 Adobe 可攜式文件格式（PDF）文件。|
|XML Parser Specification (XPS)|您可以使用單一方法將所有支援的檔案格式匯出為 XML 解析規格（XPS）文件。|
|Tagged Image File Format (TIFF)|您可以將所有支援的簡報檔案格式匯出為標記圖像檔案格式（TIFF）。|
|PPTX 轉換為 HTML|Aspose.Slides for Node.js via Java 支援將 PresentationEx 轉換為 HTML 格式。|

## **渲染與列印**
Aspose.Slides for Node.js via Java 支援高保真度地將簡報中的投影片渲染為不同的圖形格式。請查閱以下詳細資訊：

|**功能**|**說明**|
| :- | :- |
|支援的 Java 圖像格式|使用 Aspose.Slides for Node.js via Java，您可以將簡報投影片及投影片中的圖像渲染為所有 Java 支援的圖形格式，如 TIFF、PNG、BMP、JPEG、GIF。|
|SVG 格式|Aspose.Slides for Node.js via Java 亦提供內建方法，允許您將簡報投影片匯出為可縮放向量圖形（SVG）格式。|
|簡報列印|Aspose.Slides for Node.js via Java 的最新版本提供具備不同選項的內建列印方法。|

## **常見問題**

**我是否需要在伺服器/電腦上安裝 Microsoft PowerPoint 才能使用此函式庫？**

否。PowerPoint 並非必要；Aspose.Slides 是用於建立、編輯、轉換與呈現簡報的獨立引擎。

**多執行緒如何運作？處理程序能平行化嗎？**

在不同執行緒中處理不同文件是安全的；同一個 [presentation](https://reference.aspose.com/slides/zh-hant/nodejs-java/aspose.slides/presentation/) 物件不得同時被 [多個執行緒](/slides/zh-hant/nodejs-java/multithreading/) 使用。

**是否支援檔案密碼與加密？**

是的。[您可以](/slides/zh-hant/nodejs-java/password-protected-presentation/) 開啟加密的簡報、設定或移除開啟與寫入密碼，並檢查保護狀態。

**在 Linux 容器中需要注意字型套件嗎？**

是的。建議在您的應用程式中安裝常見字型套件和/或明確 [指定字型目錄](/slides/zh-hant/nodejs-java/custom-font/) ，以避免意外的替換。

**評估版有什麼限制？**

在 [評估模式](/slides/zh-hant/nodejs-java/licensing/) 中，輸出會加入浮水印且會套用某些限制；可取得 [30 天臨時授權](https://purchase.aspose.com/temporary-license/)，以完整功能測試。

**是否支援將外部格式匯入簡報（PDF/HTML → PPTX）？**

是的。您可以將 [PDF 頁面與 HTML 內容](/slides/zh-hant/nodejs-java/import-presentation/) 加入簡報，將其轉換為投影片。
