## **概覽**

本文說明如何取得 Aspose.Slides 的支援以及如何有效回報問題。它描述主要的支援管道、提供提交問題的建議，並指向有用的技術資源，如產品頁面、部落格、下載、支援論壇、文件與 API 參考。本文亦回答有關付費支援、問題升級以及舊版函式庫支援的常見問題。

## **技術支援**
{{% alert color="primary" %}}

Aspose 為其所有產品提供無限制的免費技術支援。此支援對所有使用者開放（包括使用評估套件的使用者）。

{{% /alert %}} 

當您需要於 Node.js via Java 使用 Aspose.Slides 時，可考慮以下事項：

- 主要的支援渠道是 [Aspose.Forums](https://forum.aspose.com/c/slides/zh-hant/11)。在 Aspose.Slides 論壇發表問題——通常會在數小時內得到回覆。Aspose.Slides 支援團隊會直接回覆論壇上的問題。
- 請注意，Aspose 不提供電話技術支援。電話支援僅限於銷售與購買相關問題。
- 在等待論壇回覆時，請保持耐心，並考量時區差異。

當您在 Node.js via Java 使用 Aspose.Slides 時遇到問題，請遵循以下指南以最高效的方式解決：

- 在回報問題前，請先確認您已使用最新的 Aspose.Slides for Node.js via Java 版本。也許您遇到的問題已在更新中解決。 

  請參閱 [Aspose.Slides download](https://releases.aspose.com/slides/zh-hant/nodejs-java/) 取得最新版本資訊。

- 在回報問題之前，建議先瀏覽我們的論壇、此文件與 API 參考。可能已經有答案。

- 回報問題時，請附上原始文件以及（若可能）與問題相關的程式碼片段。如需附加多個檔案，請以 ZIP 壓縮。您可以安全地在 Aspose.Forums 附加文件，因為 **只有** 您與 Aspose 開發人員可以存取這些檔案。

- 為每個特定問題建立並使用單一討論串。如果您遇到其他問題，必須另開新討論串進行回報。

## **常見問題**

**提供哪些付費支援等級？它們有何差異？**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services)。它們在包含的事件數量、回應優先順序與可使用的通訊管道上有所不同。

**我可以直接將 Bug 升級給工程團隊嗎？**

可以。[Paid support](https://helpdesk.aspose.com/) 允許在問題需要更深入調查時升級給開發人員。

**舊版函式庫是否仍受支援？會釋出修復嗎？**

您可以使用舊版，但修復僅提供給目前的發行版本；不會對舊版釋出修補程式。
