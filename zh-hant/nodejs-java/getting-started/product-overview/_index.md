## **歡迎使用 Aspose.Slides for Node.js via Java！**

Aspose.Slides 是一個 Microsoft PowerPoint® 管理 API，允許應用程式在不使用 Microsoft PowerPoint® 的情況下讀取、寫入、建立和修改 PowerPoint 簡報。該庫支援處理簡報、投影片、形狀、表格、文字、動畫、音訊、視訊以及其他簡報元件。

使用 Aspose.Slides，開發人員可以從不同來源開啟簡報，將其儲存為各種 PowerPoint 格式，將簡報匯出為 PDF、TIFF、XPS、SWF、HTML、SVG 等格式，並將投影片另存為影像。API 亦提供管理文件屬性、套用保護、預覽投影片、編輯投影片內容、處理投影片備註，以及處理嵌入式 OLE 物件等進階簡報元素的功能。

## **為何應使用 Aspose.Slides for Node.js via Java**

- Aspose.Slides for Node.js via Java 是首個也是唯一提供管理 PowerPoint® 文件所需功能的元件。 
- Aspose.Slides for Node.js via Java 提供許多關鍵功能，包括管理文字、形狀、表格與動畫、在投影片中加入音訊與視訊、預覽投影片、將投影片匯出為 SVG、PDF 等其他格式，以及更多功能。 
- Aspose.Slides for Node.js via Java 不僅能讓您從不同來源開啟 PowerPoint 檔案，還能以多種方式儲存簡報。例如，您可以將簡報儲存為 PPT；亦可將投影片另存為影像。 
- Aspose.Slides for Node.js via Java 提供幾乎所有在 Microsoft PowerPoint 中能找到或找不到的功能。憑藉其多樣性，除了表格與各種形狀外，您還可以使用它在投影片中加入不同類型的框架——文字、音訊與視訊框架。

## **Aspose.Slides 功能**

Aspose.Slides for Node.js via Java 的功能可分為以下幾組：

- 簡報功能。
- 投影片功能。
- 圖形與多媒體功能。
- 表格功能。
- 文字與段落功能
- 進階功能

### **簡報功能**

- 從不同來源開啟 Microsoft PowerPoint 簡報。
- 以不同的 Microsoft PowerPoint 格式儲存簡報。
- 從零開始建立簡報。
- 將簡報轉換為 PDF 文件，且不需使用其他元件。
- 讀取或修改簡報的文件屬性。
- 為簡報套用保護。
- 將簡報匯出為 PDF。
- 將簡報匯出為 TIFF。
- 將簡報匯出為 XPS。
- 将簡報匯出為 SWF。
- 将簡報匯出為 PDF 註解 / TIFF 註解。
- 将簡報匯出為 HTML。

### **投影片功能**

- 從零開始建立新投影片並將其加入簡報。
- 依照投影片在簡報中的位置存取任意投影片。
- 複製或克隆投影片至相同或其他簡報。
- 編輯與刪除投影片。
- 自動將投影片母版一起複製。
- 預覽投影片。
- 取得或設定投影片名稱。
- 將投影片另存為影像。
- 將投影片匯出為 SVG 格式。
- 在簡報中使用多個母版。
- 使用任何支援的樣式填充投影片背景。
- 新增或管理投影片備註。
- 動態設定新的投影片母版與樣式。
- 新增或修改投影片放映過渡效果。
- 新增形態過渡 (Morph Transition)。

### **圖形與多媒體功能**

- 建立或管理線條、矩形、橢圓形與多段線等形狀。
- 管理形狀的線條樣式。
- 取得形狀的替代文字。
- 旋轉或翻轉形狀。
- 以不同樣式填充形狀，例如實色、漸層、圖案、紋理與圖片。
- 使用 48 種預定義圖案與 24 種紋理。
- 為形狀指定超連結。
- 讀取、修改與新增投影片及形狀的動畫效果。
- 新增圖片至簡報並管理已存在的圖片。
- 新增圖片項目符號至簡報並管理已存在的圖片項目符號。
- 新增音訊框架至簡報並管理已存在的音訊框架。
- 新增視訊框架至簡報並管理已存在的視訊框架。

### **表格功能**

- 從零開始建立表格並將其加入投影片。
- 調整表格大小與位置。
- 存取表格中的任意儲存格。
- 變更任意儲存格的格式與邊框。
- 調整欄與列的大小。
- 新增與刪除欄與列。
- 設定整個表格的邊框寬度與顏色。

### **文字與段落功能**

- 取得與設定文字佔位符與文字框中的文字。
- 取得佔位符的類型。
- 變更文字框中的文字格式。
- 新增字型或管理簡報中已使用的字型。
- 變更文字與文字格式。
- 建立或管理文字的超連結。
- 支援 Tab 與有效的 Tab。
- 在幾乎所有形狀上新增文字框。
- 取得與修改文字框的邊距與文字環繞設定。
- 取得與修改段落縮排。

### **進階功能**

- 存取簡報中嵌入的 OLE 物件。
- 鎖定簡報。

## **常見問題**

**是否支援 PDF 保存與無障礙存取的符合等級（PDF/A 與 PDF/UA）？**

是的。您可以透過設定 [PDF export options](https://reference.aspose.com/slides/zh-hant/nodejs-java/aspose.slides/pdfoptions/) 將檔案儲存為 PDF/A-2a/2b/2u、PDF/A-3a/3b，以及 PDF/UA。

**是否提供字型替代機制與自訂字型支援，以確保正確渲染？**

是的。此函式庫允許您 [load custom fonts](/slides/zh-hant/nodejs-java/custom-font/) 並 [define fallback rules](/slides/zh-hant/nodejs-java/fallback-font/)，在轉換與渲染過程中可靠地替代缺失的字形。

**是否能在未完整開啟檔案的情況下偵測檔案是否受密碼保護？**

是的。您可以 [inspect a presentation](/slides/zh-hant/nodejs-java/examine-presentation/) 以判斷在載入完整檔案前是否需要密碼。

**處理與轉換是否需要安裝 Microsoft PowerPoint？**

不需要。Aspose.Slides 為獨立引擎，伺服器或工作站上不必安裝 PowerPoint。

**從多執行緒同時處理簡報是否安全？**

是的，您可以在不同執行緒中平行處理不同文件；只要避免同時在多個執行緒中使用同一個簡報實例即可 [across threads](/slides/zh-hant/nodejs-java/multithreading/)。

**巨集是否會被保留，且能在 PPTM/PPSM 檔案中管理 VBA 嗎？**

是的。支援含巨集的簡報 [are supported](/slides/zh-hant/nodejs-java/presentation-via-vba/)，您可檢視並管理這些檔案中的 VBA 專案。

**是否能將 PDF 或 HTML 轉回 PowerPoint 投影片？**

是的。您可以 [import PDF pages or HTML content](/slides/zh-hant/nodejs-java/import-presentation/) 以建立或填充簡報中的投影片。

**是否支援 XPS 匯出，且能控制 XPS 輸出的品質與內容？**

是的。提供 [Export to XPS](/slides/zh-hant/nodejs-java/convert-powerpoint-to-xps/)，且可透過 [save options](https://reference.aspose.com/slides/zh-hant/nodejs-java/aspose.slides/xpsoptions/) 調整輸出品質與包含的內容。

**是否能將投影片轉換為影像並控制輸出品質？**

是的。投影片可 [rendered to PNG, JPEG, GIF, BMP, TIFF](/slides/zh-hant/nodejs-java/convert-powerpoint-to-png/) 及其他格式，並可細緻控制尺寸與品質。

**是否支援數學公式（MathML/MathText），且能匯出？**

是的。您可以 [create and edit mathematical text](/slides/zh-hant/nodejs-java/powerpoint-math-equations/) 並 [export equations](/slides/zh-hant/nodejs-java/exporting-math-equations/)，包括匯出為 MathML。

**函式庫如何處理加密與密碼（開啟、設定、移除）？**

[It supports](/slides/zh-hant/nodejs-java/password-protected-presentation/) 開啟加密簡報、設定或移除開啟與修改的密碼，並檢查檔案的保護狀態。
