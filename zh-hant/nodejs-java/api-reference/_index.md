{{% alert color="primary" %}} 

Aspose.Slides for Node.js via Java 是一個類別庫，允許 Node.js 應用程式在不使用 Microsoft PowerPoint® 或其他軟體的情況下讀取和寫入簡報文件。Aspose.Slides for Node.js via Java 允許您管理文字、圖形、表格與動畫，向投影片加入音訊和影片，預覽投影片，將投影片匯出為 SVG、PDF 格式，並執行更多操作。

最新的 API 參考可在 [Aspose.Slides for Node.js via Java API Reference](https://reference.aspose.com/slides/zh-hant/nodejs-java/) 中找到。
