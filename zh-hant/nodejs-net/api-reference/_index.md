Aspose.Slides for Node.js via .NET 是一個類別庫，讓 Node.js 應用程式能在不使用 Microsoft PowerPoint® 或其他軟體的情況下讀取和寫入簡報文件。Aspose.Slides for Node.js via .NET 允許您管理文字、圖形、表格與動畫，向投影片加入音訊和視訊，預覽投影片，將投影片匯出為 SVG、PDF 格式，並且還能做更多操作。

Aspose.Slides for Node.js via .NET 使用[相同的 API 參考與 Aspose.Slides .NET](https://reference.aspose.com/slides/zh-hant/nodejs-net/)。
