## **產品說明**

Aspose.Slides for Node.js via .NET 是一個跨平台類別庫，讓您的應用程式在不使用 Microsoft PowerPoint® 的情況下執行各種文件處理工作。

### **為什麼要使用 Aspose.Slides for Node.js via .NET**

- Aspose.Slides for Node.js via .NET 是首個也是唯一提供管理 PowerPoint® 文件所需功能的元件。
- Aspose.Slides for Node.js via .NET 提供許多關鍵功能，例如管理文字、圖形、表格與動畫、向投影片加入音訊與影片、預覽投影片、將投影片匯出為 SVG、PDF 以及其他格式，等多種能力。
- Aspose.Slides for Node.js via .NET 不僅能從不同來源開啟 PowerPoint 檔案，還允許您以多種方式儲存簡報。例如，您可以將簡報儲存為 PPT，也可以將投影片儲存為影像。
- Aspose.Slides for Node.js via .NET 為您提供幾乎所有在 Microsoft PowerPoint 中可以或不可以找到的功能。鑑於 Aspose.Slides for Node.js via .NET 的多功能性，除了表格和各種圖形外，您還可以使用它向投影片加入不同類型的框架——文字、音訊與影片框架。

## **Aspose.Slides 功能**

Aspose.Slides for Node.js via .NET 的功能可分為以下幾類：

- 簡報功能。
- 投影片功能。
- 圖形與多媒體功能。
- 表格功能。
- 文字與段落功能
- 進階功能

### **簡報功能**

- 從不同來源開啟 Microsoft PowerPoint 簡報。
- 以不同的 Microsoft PowerPoint 格式儲存簡報。
- 從頭建立簡報。
- 將簡報轉換為 PDF 文件，無需使用其他元件。
- 讀取或修改簡報的文件屬性。
- 為簡報套用保護。
- 將簡報匯出為 PDF。
- 將簡報匯出為 TIFF。
- 將簡報匯出為 XPS。
- 將簡報匯出為 SWF。
- 將簡報匯出為 PDF Note / Tiff Notes。
- 將簡報匯出為 HTML。

### **投影片功能**

- 從頭建立新投影片並將其加入簡報。
- 依投影片在簡報中的位置存取任意投影片。
- 複製或克隆投影片至相同或其他簡報。
- 編輯與刪除投影片。
- 自動複製帶投影片的母版。
- 預覽投影片。
- 取得或設定投影片名稱。
- 將投影片儲存為影像。
- 將投影片匯出為 SVG 格式。
- 在簡報中使用多個母版。
- 使用任何支援的樣式填滿投影片背景。
- 新增或管理投影片備註。
- 動態設定新投影片母版與樣式。
- 新增或修改投影片切換效果。
- 新增 Morph 轉場。

### **圖形與多媒體功能**

- 建立或管理線條、矩形、橢圓、多邊線等圖形。
- 管理圖形的線條樣式。
- 取得圖形的替代文字。
- 旋轉與翻轉圖形。
- 以不同樣式填充圖形，例如實色、漸層、圖案、紋理與圖片。
- 使用 48 種預定義圖案與 24 種紋理。
- 為圖形指定超連結。
- 讀取、修改並新增投影片與圖形的動畫效果。
- 新增圖片至簡報並管理已存在的圖片。
- 新增圖片項目符號至簡報並管理已存在的項目符號。
- 新增音訊框架至簡報並管理已存在的音訊框架。
- 新增影片框架至簡報並管理已存在的影片框架。

### **表格功能**

- 從頭建立表格並將其加入投影片。
- 調整表格大小與位置。
- 存取表格中的任何儲存格。
- 更改任何儲存格的格式與邊框。
- 調整欄寬與列高。
- 新增與刪除欄與列。
- 為整個表格設定邊框寬度與顏色。

### **文字與段落功能**

- 取得與設定文字佔位符與文字框內的文字。
- 取得佔位符的類型。
- 更改文字框內的文字格式。
- 新增字型或管理簡報中已使用的字型。
- 更改文字與文字格式。
- 建立或管理文字的超連結。
- 支援 Tab 與有效 Tab。
- 為幾乎任何類型的圖形新增文字框。
- 存取與修改文字框的邊距與文字環繞設定。
- 存取與修改段落縮排。

### **進階功能**

- 存取嵌入簡報中的 OLE 物件。
- 鎖定簡報。
