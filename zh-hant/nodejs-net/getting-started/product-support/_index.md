## **技術支援**
{{% alert color="primary" %}}

Aspose 為其所有產品提供無限免費的技術支援。此支援對所有使用者皆可使用（包括使用評估套件的使用者）。

{{% /alert %}} 

當您需要 Aspose.Slides for Node.js via .NET 的協助時，請考慮以下事項：

- 支援的主要管道是 [Aspose.Forums](https://forum.aspose.com/c/slides/zh-hant/11)。在 Aspose.Slides 論壇發佈您的問題—它將在數小時內得到回覆。Aspose.Slides 支援團隊會直接回答論壇上發佈的問題。
- 請注意，Aspose 不提供電話技術支援。電話支援僅適用於銷售與購買相關的問題。
- 在論壇期待回覆時，請耐心等待，並考慮時區差異。

當您在使用 Aspose.Slides for Node.js via .NET 時遇到問題，請遵循以下指南以最有效的方式解決：

- 在回報問題前，請檢查並確認您使用的是最新的 Aspose.Slides for Node.js via .NET 版本。也許您遇到的問題已在更新中得到解決。

  請參閱 [Aspose.Slides download](https://releases.aspose.com/slides/zh-hant/nodejs-net/) 以取得最新版本資訊。

- 在回報問題前，您可能想先瀏覽我們的論壇、此文件以及 API 參考。也許您的問題已在其中得到解答。

- 回報問題時，請附上原始文件以及（如果可能）與問題相關的程式碼片段。若需要附加多個檔案，請將它們壓縮為 ZIP 包。您可以安全地在 Aspose.Forums 上附加文件，因為 **只有** 您與 Aspose 開發人員可以存取這些附件。

- 針對特定問題建立並使用一個討論串。若您遇到其他問題，必須另開討論串進行回報。
