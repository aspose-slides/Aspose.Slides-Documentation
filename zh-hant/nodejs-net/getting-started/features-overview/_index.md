## **支援平台**
Aspose.Slides for Node.js via .NET 是一個平台無關的 API。它可在任何安裝了 Node.js 7 或更高版本的平台（Windows、Linux、MacOS 等）上使用。Oracle JDK 8 或更高版本也是必須的。

## **檔案格式與轉換**
Aspose.Slides for Node.js via .NET 支援大多數 PowerPoint 文件格式。它亦讓您將這些文件匯出為組織常用且相互分享的流行檔案格式。請參閱以下詳細資訊：

|**功能**|**說明**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via .NET 為此簡報文件格式提供最快的處理速度。|
|PPT to PPTX conversion|Aspose.Slides for Node.js via .NET 支援 PPT 轉換為 PPTX。|
|Portable Document Format (PDF)|您可以使用單一方法將所有支援的檔案格式匯出為 Adobe 可攜式文件格式 (PDF) 文件。|
|XML Parser Specification (XPS)|您可以使用單一方法將所有支援的檔案格式匯出為 XML 解析器規格 (XPS) 文件。|
|Tagged Image File Format (TIFF)|您可以將所有支援的簡報檔案格式匯出為標籤圖像檔案格式 (TIFF)。|
|PPTX To HTML Conversion|Aspose.Slides for Node.js via .NET 支援將 PresentationEx 轉換為 HTML 格式。|

## **呈現與列印**
Aspose.Slides for Node.js via .NET 支援將簡報中的投影片高忠實度呈現為不同的圖形格式。請參閱以下詳細資訊：

|**功能**|**說明**|
| :- | :- |
|.NET Supported Image Formats|使用 Aspose.Slides for Node.js via .NET，您可以將簡報投影片及投影片上的影像呈現為所有 .NET 支援的圖形格式，例如 TIFF、PNG、BMP、JPEG、GIF。|
|SVG Format|Aspose.Slides for Node.js via .NET 亦提供內建方法，讓您將簡報投影片匯出為可縮放向量圖形 (SVG) 格式。|
|Presentation Printing|最新版本的 Aspose.Slides for Node.js via .NET 提供具備不同選項的內建列印方法。|

## **內容功能**
Aspose.Slides for Node.js via .NET 允許您存取、修改或建立幾乎所有簡報文件的項目或內容。請參閱以下詳細資訊：

|**功能**|**說明**|
| :- | :- |
|Master Slides|母片 (Master Slides) 定義普通投影片的版面配置。Aspose.Slides for Node.js via .NET 允許您存取與修改簡報文件的母片。|
|Normal Slides|使用 Aspose.Slides for Node.js via .NET，您可以建立不同類型的新投影片；亦可存取與修改簡報中現有的投影片。|
|Cloning / Copying Slides|Aspose.Slides for Node.js via .NET 提供內建方法，可在簡報內部複製或克隆現有投影片。您也可以將複製或克隆的投影片從一個簡報使用於另一個簡報。由於投影片的版面配置繼承自母片，內建的克隆方法會自動在克隆時複製母片。|
|Managing Slides sections|提供方法將投影片組織為簡報內的不同區段。|
|Place Holders and Text Holders|您可以存取投影片中的版位持有者與文字持有者。此外，您亦可使用相應方法從頭建立具有文字持有者的投影片。|
|Header and Footers|Aspose.Slides for Node.js via .NET 方便在投影片中處理頁首/頁尾。|
|Notes in Slides|使用 Aspose.Slides for Node.js via .NET，您可以存取與修改與投影片關聯的備註，亦可新增備註。|
|Finding a Shape|您亦可透過形狀的替代文字，在投影片中找到特定形狀。|
|Backgrounds|Aspose.Slides for Node.js via .NET 允許您處理簡報中母片或普通投影片的背景。|
|Text Boxes|文字方塊可從頭建立。您可以存取現有的文字方塊。亦可修改其文字而不失去原始的文字格式。|
|Rectangle Shapes|您可以使用 Aspose.Slides for Node.js via .NET 建立或修改矩形形狀。|
|Poly Line Shapes|您可以使用 Aspose.Slides for Node.js via .NET 建立或修改多段線形狀。|
|Ellipse Shapes|您可以使用 Aspose.Slides for Node.js via .NET 建立或修改橢圓形狀。|
|Group Shapes|Aspose.Slides for Node.js via .NET 支援群組形狀。|
|Auto Shapes|Aspose.Slides for Node.js via .NET 支援自動形狀。|
|SmartArt|Aspose.Slides for Node.js via .NET 提供對 MS PowerPoint 中 SmartArt 形狀的支援。|
|Charts|Aspose.Slides for Node.js via .NET 提供對 PowerPoint 中 MSO 圖表的支援。|
|Shapes Serialization|Aspose.Slides for Node.js via .NET 支援大量形狀。當 Aspose.Slides for Node.js via .NET 缺乏對某形狀的支援時，您可以使用序列化方法，將該形狀從現有投影片序列化。如此，您即可依需求再次使用該形狀。|
|Picture Frames|您可以使用 Aspose.Slides for Node.js via .NET 管理圖片框中的圖片。|
|Audio Frames|您可以在投影片的音訊框中連結或嵌入音訊檔案，使用 Aspose.Slides for Node.js via .NET。|
|Video Frames|您可以在影片框中處理影片檔案。Aspose.Slides for Node.js via .NET 亦支援連結與嵌入式影片。|
|OLE Frame|您可以使用 Aspose.Slides for Node.js via .NET 管理 OLE 框中的 OLE 物件。|
|Tables|Aspose.Slides for Node.js via .NET 支援投影片中的表格。|
|ActiveX Controls|支援 ActiveX 控制項。|
|VBA Macros|支援在簡報中管理 VBA 巨集。|
|Text Frame|您可以透過與形狀關聯的文字框存取其文字。|
|Text Scanning|您可以使用內建的掃描方法，在簡報或投影片層級掃描文字。|
|Animations|您可以對形狀套用動畫。|
|Slide Shows|Aspose.Slides for Node.js via .NET 支援投影片秀與投影片過場效果。|

## **格式化功能**
使用 Aspose.Slides for Node.js via .NET，您可以格式化簡報投影片上的文字與形狀。請參閱以下詳細資訊：

|**功能**|**說明**|
| :- | :- |
|Text Formatting|<p>在 Aspose.Slides for Node.js via .NET 中，您可以透過與形狀關聯的文字框來管理文字。因此，您可以使用文字框中的段落與文字區段來格式化文字。這些文字元素可透過 Aspose.Slides for Node.js via .NET 進行格式設定。</p><p>- 字型類型</p><p>- 字型大小</p><p>- 字體顏色</p><p>- 字型色階</p><p>- 段落對齊</p><p>- 段落項目符號</p><p>- 段落方向</p>|
|Shape Formatting|<p>在 Aspose.Slides for Node.js via .NET 中，投影片的基本元素是形狀。您可以使用 Aspose.Slides for Node.js via .NET 對這些形狀元素進行格式設定：</p><p>- 位置</p><p>- 大小</p><p>- 線條</p><p>- 填充（包括圖案、漸層、純色）</p><p>- 文字</p><p>- 圖像</p>|
