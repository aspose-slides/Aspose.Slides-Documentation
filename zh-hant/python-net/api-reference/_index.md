Aspose.Slides for Python via .NET 是一個類別庫，讓 Python 應用程式在不使用 Microsoft PowerPoint® 或其他軟體的情況下讀寫簡報文件。Aspose.Slides for Python via .NET 允許您管理文字、形狀、表格和動畫；將音訊與視訊加入投影片；預覽投影片；以及將投影片匯出為 SVG、PDF 等格式。

最新的 API 參考可於 [Aspose.Slides for Python via .NET API Reference](https://reference.aspose.com/slides/zh-hant/python-net/) 找到。
