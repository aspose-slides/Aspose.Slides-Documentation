Aspose.Slides for Python via .NET 包含廣泛的功能，例如處理文字、圖形、表格和動畫；添加音訊和影片；預覽投影片；以及將投影片匯出為 SVG、PDF 等格式。

## 常見問題

### 什麼是 Aspose.Slides for Python via .NET？

Aspose.Slides for Python via .NET 是一個功能強大的 Python 程式庫，允許您在未安裝 Microsoft PowerPoint 的情況下，以程式方式建立、編輯和轉換 PowerPoint 簡報（PPT、PPTX、ODP）。

### Aspose.Slides 支援哪些簡報功能？

此程式庫支援管理文字、圖形、表格、圖表、動畫、母片、音訊、影片等。它亦提供投影片預覽、渲染、列印，以及匯出為 PDF、SVG、HTML 和影像等格式的功能。

### 我可以使用 Aspose.Slides 將簡報轉換為其他格式嗎？

可以。Aspose.Slides 能將 PowerPoint 檔案高保真且高效能地轉換為 PDF、SVG、HTML、JPG、PNG、TIFF 等多種格式。

### 使用 Aspose.Slides 是否需要 Microsoft PowerPoint？

不需要。Aspose.Slides 是獨立的 API，無需 Microsoft Office 或任何第三方軟體。

### Aspose.Slides for Python via .NET 支援哪些平台？

它是跨平台的，可在 Windows、Linux 與 macOS 環境執行。

### 如何開始使用 Aspose.Slides for Python？

您可以透過 PyPi 安裝，並參考 [開發者指南](/slides/zh-hant/python-net/developer-guide/) 開始使用範例、API 參考與教學。
