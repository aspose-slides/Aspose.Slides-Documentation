## **概覽**

本文說明如何取得 Aspose.Slides 的支援以及如何有效回報問題。內容描述主要支援管道、提供提交問題的建議，並列出產品頁面、部落格、下載、支援論壇、文件與 API 參考等技術資源。本文亦回答有關付費支援、問題升級以及舊版函式庫支援的常見問題。

## **技術支援**
{{% alert color="primary" %}}

Aspose 為所有產品提供無限制的免費技術支援。此支援適用於所有使用者（包括使用評估套件的使用者）。

{{% /alert %}} 

當您需要 Aspose.Slides for Python via .NET 的協助時，請考慮以下事項：

- 支援的主要管道是 [Aspose.Forums](https://forum.aspose.com/c/slides/zh-hant/11)。在 Aspose.Slides 論壇發表您的問題，通常會在數小時內獲得回覆。Aspose.Slides 支援團隊會直接回答論壇上的問題。
- 請注意，Aspose 不提供電話技術支援。電話支援僅限於銷售與購買相關問題。
- 在論壇等待回覆時，請保持耐心，並考量時區差異。

當您在 Aspose.Slides for Python via .NET 中遇到問題時，請遵循以下指引以最有效的方式解決：

- 在回報問題之前，先確認您正在使用最新的 Aspose.Slides for Python via .NET 版本。也許您遇到的問題已在更新中修復。

  請參閱 Aspose.Slides 下載以取得最新版本資訊。

- 回報問題前，您可以先瀏覽我們的論壇、此文件與 API 參考。或許您的問題已得到答案。

- 回報問題時，請附上原始文件並（如果可能）提供與問題相關的程式碼片段。若需附加多個檔案，請將它們壓縮成 ZIP 包。

  您可以安全地在 Aspose.Forums 上上傳文件，因為 **only** 您與 Aspose 開發人員能存取這些附件。

- 為每個特定問題建立並使用一個討論串。若您遇到其他問題，必須另開討論串進行回報。

## **常見問題**

**提供哪些付費支援層級，且它們有何差異？**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services)。它們在包含的事件數量、回應優先順序以及可用的通訊管道上有所不同。

**我可以直接將 Bug 升級給工程團隊嗎？**

可以。[付費支援](https://helpdesk.aspose.com/)允許在問題需要更深入調查時升級給開發人員。

**舊版函式庫是否仍受支援，且會釋出修補程式嗎？**

您可以使用較舊的版本，但修補程式僅提供給目前的發行版；不會為舊版發佈補丁。
