## **Aspose.Slides 評估版**

您可以輕鬆下載 Aspose.Slides 進行評估。評估套件與購買的套件相同。評估版只要加入幾行程式碼套用授權即可變成正式授權。

未指定授權的 Aspose.Slides 評估版提供完整功能，但會在開啟或儲存文件時於文件頂部插入評估水印。從簡報投影片中擷取文字時亦限制只能擷取一張投影片。

## **常見問與答**

**我可以在評估模式下於不同執行緒平行測試多個簡報嗎？**

可以。您可以平行處理不同的文件；不應在[執行緒間共享相同的簡報物件](/slides/zh-hant/python-net/multithreading/)。評估模式不會影響此行為。

**我需要在伺服器或 CI 上安裝 Microsoft PowerPoint 來評估此函式庫嗎？**

不需要。Aspose.Slides 為獨立引擎，無論是評估還是正式環境皆不需安裝 PowerPoint。

**我可以在評估模式下完整測試 PPT/PPTX 轉 PDF 與影像嗎？**

可以。[轉換器](/slides/zh-hant/python-net/convert-presentation/) 可正常運作；輸出檔案會包含水印。

**我可以使用暫時授權進行負載測試且不顯示水印嗎？**

可以。30 天暫時授權會移除評估模式的限制，並允許測試時不顯示水印。
