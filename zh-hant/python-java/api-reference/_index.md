Aspose.Slides for Python via Java 是一個類別庫，使 Python 應用程式能夠讀取和寫入簡報文件，而不需要使用 Microsoft PowerPoint® 或其他軟體。Aspose.Slides for Python via Java 允許您管理文字、圖形、表格與動畫，向投影片加入音訊與影片，預覽投影片，將投影片匯出為 SVG、PDF 格式，還有更多功能。

Aspose.Slides for Python via Java 使用[與 Aspose.Slides for Java 相同的 API 參考](https://reference.aspose.com/slides/zh-hant/python-java/)。
