## **支援平台**
Aspose.Slides for Python via Java 是一個平台無關的 API。它可在任何安裝了 Python 3.7 或更高版本的平台（Windows、Linux、MacOS 等）上使用。JDK 8 或更高版本是另一項需求。

## **檔案格式與轉換**
Aspose.Slides for Python via Java 支援大多數 PowerPoint 文件格式。它還允許您將它們匯出為組織間常用並共享的檔案格式。請參閱以下細節：

|**功能**|**說明**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Python via Java 為此簡報文件格式提供最快的處理速度。|
|PPT 轉換為 PPTX|Aspose.Slides for Python via Java 支援 PPT 轉換為 PPTX。|
|Portable Document Format (PDF)|您可以使用單一方法將所有支援的檔案格式匯出為 Adobe Portable Document Format（PDF）文件。|
|XML Parser Specification (XPS)|您可以使用單一方法將所有支援的檔案格式匯出為 XML Parser Specification（XPS）文件。|
|Tagged Image File Format (TIFF)|您可以將所有支援的簡報檔案格式匯出為 Tagged Image File Format（TIFF）。|
|PPTX To HTML Conversion|Aspose.Slides for Python via Java 支援將 PresentationEx 轉換為 HTML 格式。|

## **渲染與列印**
Aspose.Slides for Python via Java 支援將簡報投影片高保真度渲染為不同的圖形格式。請參閱以下細節：

|**功能**|**說明**|
| :- | :- |
|Java 支援的影像格式|使用 Aspose.Slides for Python via Java，您可以將簡報投影片與投影片上的影像渲染為所有 Java 支援的圖形格式，如 TIFF、PNG、BMP、JPEG、GIF。|
|SVG 格式|Aspose.Slides for Python via Java 亦提供內建方法，讓您將簡報投影片匯出為 Scalable Vector Graphics（SVG）格式。|
|簡報列印|最新版本的 Aspose.Slides for Python via Java 提供具備多種選項的內建列印方法。|

## **內容功能**
Aspose.Slides for Python via Java 允許您存取、修改或建立簡報文件中幾乎所有的項目或內容。請參閱以下細節：

|**功能**|**說明**|
| :- | :- |
|母版投影片|母版投影片定義普通投影片的版面配置。Aspose.Slides for Python via Java 允許您存取與修改簡報文件的母版投影片。|
|普通投影片|使用 Aspose.Slides for Python via Java，您可以建立不同類型的新投影片；亦可存取並修改簡報中現有的投影片。|
|克隆 / 複製投影片|Aspose.Slides for Python via Java 提供內建方法，允許您在同一簡報中克隆或複製現有投影片。您亦可將複製或克隆的投影片從一個簡報移至另一個簡報。由於投影片會從母版繼承版面配置，內建的克隆方法會自動複製母版。|
|管理投影片分節|用於在簡報內將投影片組織到不同分節的方法。|
|占位符與文字占位符|您可以存取投影片中的占位符與文字占位符。此外，您可使用相應方法從頭建立帶有文字占位符的投影片。|
|頁首與頁尾|Aspose.Slides for Python via Java 方便處理投影片中的頁首/頁尾。|
|投影片備註|使用 Aspose.Slides for Python via Java，您可以存取與修改投影片關聯的備註，亦可新增備註。|
|尋找形狀|您亦可使用形狀的備用文字在投影片中找到特定形狀。|
|背景|Aspose.Slides for Python via Java 允許您處理與母版或普通投影片關聯的背景。|
|文字方塊|文字方塊可從頭建立。您可以存取現有文字方塊，亦可在不失去原始文字格式的情況下修改其文字。|
|矩形形狀|您可以使用 Aspose.Slides for Python via Java 建立或修改矩形形狀。|
|折線形狀|您可以使用 Aspose.Slides for Python via Java 建立或修改折線形狀。|
|橢圓形狀|您可以使用 Aspose.Slides for Python via Java 建立或修改橢圓形狀。|
|群組形狀|Aspose.Slides for Python via Java 支援群組形狀。|
|自動圖形|Aspose.Slides for Python via Java 支援自動圖形。|
|SmartArt|Aspose.Slides for Python via Java 提供對 Microsoft PowerPoint 中 SmartArt 形狀的支援。|
|Charts|Aspose.Slides for Python via Java 提供對 PowerPoint 中 MSO 圖表的支援。|
|形狀序列化|Aspose.Slides for Python via Java 支援大量形狀。當缺少對某形狀的支援時，您可以使用序列化方法，將該形狀從現有投影片序列化，之後依需求再次使用該形狀。|
|圖片框架|您可以使用 Aspose.Slides for Python via Java 管理圖片框架中的圖片。|
|音訊框架|您可以在投影片的音訊框架中連結或嵌入音訊檔案，使用 Aspose.Slides for Python via Java。|
|視訊框架|您可以在視訊框架中處理視訊檔案。Aspose.Slides for Python via Java 亦支援連結與嵌入式視訊。|
|OLE 框架|您可以使用 Aspose.Slides for Python via Java 管理 OLE 框架中的 OLE 物件。|
|表格|Aspose.Slides for Python via Java 支援投影片中的表格。|
|ActiveX 控制項|支援 ActiveX 控制項。|
|VBA 巨集|支援在簡報內管理 VBA 巨集。|
|文字框|您可以透過與形狀關聯的文字框存取該形狀中的文字。|
|文字掃描|您可以使用內建掃描方法，在簡報或投影片層級掃描文字。|
|動畫|您可以對形狀套用動畫。|
|投影片放映|Aspose.Slides for Python via Java 支援投影片放映與投影片切換。|

## **格式化功能**
使用 Aspose.Slides for Python via Java，您可以格式化簡報投影片中的文字與形狀。請參閱以下細節：

|**功能**|**說明**|
| :- | :- |
|文字格式化|<p>在 Aspose.Slides for Python via Java 中，您可以透過形狀關聯的文字框管理文字。因此，您可以使用文字框關聯的段落與文字區塊來格式化文字。這些文字元素可透過 Aspose.Slides for Python via Java 進行格式化。</p><p>- 字型類型</p><p>- 字型大小</p><p>- 字型顏色</p><p>- 字型色階</p><p>- 段落對齊</p><p>- 段落項目符號</p><p>- 段落方向</p>|
|形狀格式化|<p>在 Aspose.Slides for Python via Java 中，投影片的基本元素是形狀。您可以使用 Aspose.Slides for Python via Java 為這些形狀元素設定格式：</p><p>- 位置</p><p>- 大小</p><p>- 線條</p><p>- 填色（包括圖案、漸層、純色）</p><p>- 文字</p><p>- 圖像</p>|
