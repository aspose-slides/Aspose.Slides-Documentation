您可以輕鬆下載 Aspose.Slides 以進行評估。評估套件與購買的套件相同。只要加入幾行程式碼套用授權，評估版即可直接變為授權版。

未指定授權的 Aspose.Slides 評估版提供完整的產品功能，但在開啟和儲存文件時會在文件頂部插入評估水印。 在從簡報投影片中擷取文字時，亦只能處理一張投影片。
