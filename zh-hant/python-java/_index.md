**歡迎使用 Aspose.Slides for Python via Java**

![Aspose.Slides for Python via Java Product Logo](aspose_slides-for-python-via-java.png)

Aspose.Slides for Python via Java 是一個類別庫，可讓您的應用程式在不使用 Microsoft PowerPoint® 的情況下讀寫 PowerPoint® 文件。

Aspose.Slides for Python via Java 是第一個也是唯一提供管理 PowerPoint® 文件功能的元件。

Aspose.Slides for Python via Java 提供許多關鍵功能，例如管理文字、圖形、表格與動畫、向投影片加入音訊與視訊、預覽投影片、將投影片匯出為 SVG、PDF 格式等。
