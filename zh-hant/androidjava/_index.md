**歡迎使用 Aspose.Slides for Android via Java**

Aspose.Slides for Android via Java 是一套 Microsoft PowerPoint® 管理 API，讓 Android 應用程式在沒有 Microsoft PowerPoint® 的情況下即可讀寫 PowerPoint® 文件。Aspose.Slides for Android via Java 是首個亦唯一提供在 Android 平台上管理 PowerPoint® 文件所需功能的元件。Aspose.Slides for Android 提供多項關鍵功能，讓您能在投影片中新增、修改與操作文字、圖形、表格與動畫、音訊與影片。
