## **概述**

本文說明如何取得 Aspose.Slides 的支援以及如何有效回報問題。它描述了主要的支援管道，提供提交問題的建議，並指向有用的技術資源，如產品頁面、部落格、下載、支援論壇、文件以及 API 參考。本文亦回答了關於付費支援、問題升級以及舊版函式庫支援的常見問題。

## **技術支援**

{{% alert color="primary" %}} 

Aspose 為所有產品提供無限制的免費技術支援。此支援對所有使用者皆可使用（包括使用評估套件的使用者）。

{{% /alert %}} 

當您需要 Aspose.Slides for Android via Java 的協助時，請考慮以下事項：

- 支援的主要管道是 Aspose.Forums。將您的問題發表在 Aspose.Slides 論壇——問題將在數小時內得到回覆。Aspose.Slides 支援團隊會直接回答論壇上的問題。
- 請注意，Aspose 不提供電話技術支援。電話支援僅限於銷售與購買相關問題。
- 在等待論壇回覆時，請保持耐心，並考慮時區差異。

當您在使用 Aspose.Slides for Android via Java 時遇到問題，請依照以下指引以最有效的方式解決：

- 在回報問題之前，先確認您已使用最新的 Aspose.Slides for Android via Java 版本。也許您遇到的問題已在更新中解決。

  請參閱 Aspose.Slides 下載以取得最新版本資訊。

- 回報問題前，您可以先瀏覽我們的論壇、此文件與 API 參考。也許您的問題答案已經提供。
- 回報問題時，請附上原始文件，並（若可能）提供與問題相關的程式碼片段。如需附加多個檔案，請將它們壓縮成 ZIP 套件。

  您可以安全地在 Aspose.Forums 上附加文件，因為只有您與 Aspose 開發人員能存取這些檔案。
- 為特定問題建立並使用單一討論串。若您遇到另一個問題，必須再建立新討論串以回報。

## **常見問題**

**有哪些付費支援等級可供選擇，且它們有何不同？**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services)。它們在包含的事件數量、回應優先級以及可用的通訊渠道上有所不同。

**我可以直接將錯誤升級給工程團隊嗎？**

可以。[Paid support](https://helpdesk.aspose.com/) 允許在問題需要深入調查時升級給開發人員。

**是否支援較舊的函式庫版本，且會釋出修復嗎？**

您可以使用較舊的版本，但修復僅提供給目前的發行版本；不會為舊版釋出補丁。
