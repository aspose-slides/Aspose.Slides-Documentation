## **歡迎使用 Aspose.Slides for Android via Java！**

Aspose.Slides for Android via Java 是一個 Microsoft PowerPoint® 管理 API，讓 Java 應用程式在不需要 Microsoft PowerPoint® 的情況下讀寫 PowerPoint® 文件。

Aspose.Slides for Android via Java 是唯一提供管理 PowerPoint® 文件所需功能的元件。

Aspose.Slides for Android via Java 提供許多關鍵功能，如管理文字、圖形、表格與動畫、在投影片中加入音訊與視訊、預覽投影片、將投影片匯出為 SVG、PDF 及其他格式。

## **產品說明**
Aspose.Slides for Android via Java 不僅可以從不同來源開啟 PowerPoint 檔案，還允許您以多種方式儲存簡報。例如，您可以將簡報儲存為 PPT；也可以將投影片儲存為影像。

Aspose.Slides for Android via Java 為您提供幾乎所有在 Microsoft PowerPoint 中可以或不可以找到的功能。藉由 Aspose.Slides for Android via Java 的多樣性，除了表格和各種圖形之外，您還可以在投影片中加入不同類型的框架──文字、音訊和視訊框架。

Aspose.Slides for Android via Java 也提供完整的示範和可執行範例，協助開發人員更深入了解 API。

### **簡報功能**
- 從不同來源開啟 Microsoft PowerPoint 簡報。
- 以不同的 Microsoft PowerPoint 格式儲存簡報。
- 從頭建立簡報。
- 在不使用其他元件的情況下將簡報轉換為 PDF 文件。
- 讀取 Open Document Presentation (ODP)。
- 能夠在 PPT 與 PPTX 之間相互轉換。
- 讀取或修改簡報的文件屬性。
- 將簡報匯出為 PDF
- 將簡報匯出為 TIFF
- 將簡報匯出為 XPS
- 將簡報匯出為 SWF
- 將簡報匯出為 PDF Note / Tiff Notes
- 將簡報匯出為 HTML

## **常見問題**

**它是否支援 PDF 存檔與無障礙等級 (PDF/A 與 PDF/UA)？**

Yes. You can save to PDF with PDF/A-2a/2b/2u, PDF/A-3a/3b, as well as PDF/UA by configuring [PDF export options](https://reference.aspose.com/slides/zh-hant/androidjava/com.aspose.slides/pdfoptions/).

**是否有字型替代機制並支援自訂字型以確保正確呈現？**

Yes. The library lets you [load custom fonts](/slides/zh-hant/androidjava/custom-font/) and [define fallback rules](/slides/zh-hant/androidjava/fallback-font/) so missing glyphs are substituted reliably during conversion and rendering.

**我能在未完整開啟檔案的情況下偵測檔案是否受密碼保護嗎？**

Yes. You can [inspect a presentation](/slides/zh-hant/androidjava/examine-presentation/) to determine if it requires a password before loading the full document.

**處理與轉換是否需要安裝 Microsoft PowerPoint？**

No. Aspose.Slides is a standalone engine; PowerPoint is not needed on the server or workstation.

**在多執行緒中處理簡報是否安全？**

Yes, you can process different documents in parallel in separate threads; just avoid using the same presentation instance concurrently [across threads](/slides/zh-hant/androidjava/multithreading/).

**巨集是否會被保留，我可以管理 PPTM/PPSM 檔案中的 VBA 嗎？**

Yes. Presentations with macros [are supported](/slides/zh-hant/androidjava/presentation-via-vba/), and you can [inspect and manage VBA projects](https://reference.aspose.com/slides/zh-hant/androidjava/com.aspose.slides/presentation/#getVbaProject--) in those files.

**我能將 PDF 或 HTML 轉回 PowerPoint 投影片嗎？**

Yes. You can [import PDF pages or HTML content](/slides/zh-hant/androidjava/import-presentation/) to create or populate slides in a presentation.

**是否支援 XPS 匯出，且我能控制 XPS 輸出的品質與內容嗎？**

Yes. [Export to XPS](/slides/zh-hant/androidjava/convert-powerpoint-to-xps/) is available, and [save options](https://reference.aspose.com/slides/zh-hant/androidjava/com.aspose.slides/xpsoptions/) allow you to tune output quality and included content.

**我能將投影片轉換為影像並控制輸出品質嗎？**

Yes. Slides can be [rendered to PNG, JPEG, GIF, BMP, TIFF](/slides/zh-hant/androidjava/convert-powerpoint-to-png/) and other formats with fine-grained control over size and quality.

**是否支援數學公式 (MathML/MathText)，且能匯出嗎？**

Yes. You can [create and edit mathematical text](/slides/zh-hant/androidjava/powerpoint-math-equations/) and [export equations](/slides/zh-hant/androidjava/exporting-math-equations/), including to MathML.

**程式庫如何處理加密與密碼 (開啟、設定、移除)？**

[It supports](/slides/zh-hant/androidjava/password-protected-presentation/) opening encrypted presentations, setting or removing passwords for opening and modifying, and checking a file’s protection status.
