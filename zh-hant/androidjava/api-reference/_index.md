{{% alert color="primary" %}} 

Aspose.Slides for Android via Java 是一個類別庫，使 Android 上的 Java 應用程式能在不使用 Microsoft PowerPoint® 或其他軟體的情況下讀寫簡報文件。Aspose.Slides for Android via Java 允許您管理文字、圖形、表格和動畫；向投影片加入音訊和視訊；預覽投影片；以及將投影片匯出為 SVG、PDF 等格式。

最新的 API 參考可於 [Aspose.Slides for Android via Java API Reference](https://reference.aspose.com/slides/zh-hant/androidjava/) 找到。
