## **概述**

本文說明如何取得 Aspose.Slides 的支援以及如何有效回報問題。內容包含主要支援管道、提交問題的建議，以及產品頁面、部落格、下載、支援論壇、文件與 API 參考等技術資源。文章同時回答有關付費支援、問題升級與舊版函式庫支援的常見問題。

## **技術支援**
{{% alert color="primary" %}}
Aspose 為所有產品提供無限制的免費技術支援。此支援對所有使用者皆開放（包括使用評估套件的使用者）。
{{% /alert %}}

當您需要 Aspose.Slides for .NET 的協助時，請考慮以下事項：

- 主要的支援管道是 [Aspose.Forums](https://forum.aspose.com/c/slides/zh-hant/11)。在 Aspose.Slides 論壇發問，通常會在數小時內得到回覆。Aspose.Slides 支援團隊會直接回答論壇上的問題。
- 請注意，Aspose 不提供電話技術支援。電話支援僅限於銷售與購買相關問題。
- 在論壇等待回覆時，請保持耐心，並考慮時區差異。

當您在使用 Aspose.Slides for .NET 時遇到問題，請依照以下指引，以最高效率解決：

- 在回報問題前，先確認您使用的是最新的 Aspose.Slides for .NET 版本。可能您遇到的問題已在更新中修復。 

  請參閱 Aspose.Slides 下載頁面取得最新版本資訊。

- 回報問題前，建議先瀏覽我們的論壇、此文件與 API 參考。也許您的問題已有人解答。

- 回報問題時，請附上原始文件，並（如果可能）提供與問題相關的程式碼片段。若需附加多個檔案，請將它們壓縮成 ZIP 包。

  您可以安心在 Aspose.Forums 上上傳文件，因為 **只有**您與 Aspose 開發人員可以存取這些附件。

- 為每個特定問題建立並使用一個討論串。如果您遇到另一個問題，請另開新討論串進行回報。

## **常見問題**

**有哪些付費支援層級？其差異為何？**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services)。它們在包含的事件數量、回應優先級和可用的通訊管道上有所不同。

**我可以直接將錯誤升級給開發團隊嗎？**

可以。[付費支援](https://helpdesk.aspose.com/) 允許在問題需要深入調查時升級至開發人員。

**舊版函式庫有支援嗎？會針對它們發布修補程式嗎？**

您可以使用舊版，但修補程式僅提供給目前的發行版本；不會為舊版發佈補丁。
