## **Aspose.Slides 的安全性**

Aspose 在開發其產品時遵循最佳實務。

* Aspose.Slides for .NET 用於操作和轉換其他格式的簡報檔案。它不會在簡報中執行腳本。Aspose.Slides 僅解析簡報結構，並讓最終使用者的程式碼以方便的方式操作物件模型。
* Aspose.Slides 作為函式庫在解析與解譯文件的情境中運作，且不執行遠端程式碼。所有 Aspose 產品皆在您的機器上執行，並不會將任何資料傳送至 Aspose。唯一的例外是使用[計量授權](https://purchase.aspose.com/faqs/licensing/metered)時，即使如此（若您使用計量授權），也僅會處理您的 API 使用資訊。
* Aspose 元件在與一般應用程式相同的使用者情境下執行。因此，Aspose 元件不會對關鍵系統資源構成風險。此外，當文件由 Aspose 元件開啟時，巨集不會自動執行。
* Microsoft Office 套件固有或相關的風險並不適用於 Aspose 元件，因此 Aspose 產品相當安全。

## **常見問題**

**使用哪些系統來監控 Aspose 程式碼中的漏洞？**

我們對每個 Aspose.Slides 版本都執行靜態程式碼分析。我們可以提供安全報告，證明 Aspose.Slides 程式碼符合 OWASP Top 10。

**使用哪些系統來監控外部套件的漏洞？**

我們不使用系統來檢查外部套件的漏洞，因為 Aspose.Slides 不包含或使用任何外部套件。Aspose.Slides 僅依賴 .NET 框架本身。
