## **常見問題**

**我需要在伺服器/電腦上安裝 Microsoft PowerPoint 才能使用此函式庫嗎？**

不需要。PowerPoint 並非必要；Aspose.Slides 是一個獨立的引擎，用於建立、編輯、轉換與算繪簡報。

**多執行緒如何運作？可以平行處理嗎？**

在不同執行緒中處理不同文件是安全的；同一個 [presentation](https://reference.aspose.com/slides/zh-hant/net/aspose.slides/presentation/) 物件不可同時被 [multiple threads](/slides/zh-hant/net/multithreading/) 使用。

**是否支援檔案密碼與加密？**

是。您可以 [開啟](/slides/zh-hant/net/password-protected-presentation/) 加密的簡報，設定或移除開啟與寫入密碼，並檢查保護狀態。

**在 Linux 容器中需要關注字型套件嗎？**

是。建議在您的應用程式中安裝常見字型套件和/或明確 [指定字型目錄](/slides/zh-hant/net/custom-font/)，以避免意外的字型替換。

**評估版有什麼限制？**

在 [評估模式](/slides/zh-hant/net/licifying/) 下，輸出會加上浮水印，且會有某些限制；可取得 [30 天臨時授權](https://purchase.aspose.com/temporary-license/) 以進行完整功能測試。

**是否支援將外部格式匯入簡報（PDF/HTML → PPTX）？**

是。您可以將 [PDF 頁面與 HTML 內容](/slides/zh-hant/net/import-presentation/) 新增至簡報，將它們轉換為投影片。
