## **歡迎使用 Aspose.Slides for .NET!**

Aspose.Slides 是一個 Microsoft PowerPoint® 管理 API，允許應用程式在不使用 Microsoft PowerPoint® 的情況下讀取、寫入、建立與修改 PowerPoint 簡報。此函式庫支援處理簡報、投影片、圖形、表格、文字、動畫、音訊、視訊以及其他簡報元素。

使用 Aspose.Slides，開發人員可以從不同來源開啟簡報，將其儲存為各種 PowerPoint 格式，將簡報匯出為 PDF、TIFF、XPS、SWF、HTML、SVG 等格式，並將投影片儲存為圖片。API 亦提供管理文件屬性、套用保護、預覽投影片、編輯投影片內容、處理投影片備註，以及處理嵌入 OLE 物件等進階簡報元素的功能。

## **為何應該使用 Aspose.Slides for .NET**

- Aspose.Slides for .NET 是首個也是唯一提供管理 PowerPoint® 文件所需功能的元件。  
- Aspose.Slides for .NET 提供許多關鍵功能，如管理文字、圖形、表格與動畫、為投影片加入音訊與視訊、預覽投影片、將投影片匯出為 SVG、PDF 以及其他格式，還有更多功能。  
- Aspose.Slides for .NET 不僅能從不同來源開啟 PowerPoint 檔案，還允許以多種方式儲存簡報。例如，您可以將簡報儲存為 PPT；也可以將投影片儲存為圖片。  

- Aspose.Slides for .NET 為您提供幾乎所有在 Microsoft PowerPoint 中可找到或不可找到的功能。除了表格和各種圖形外，您還可以使用它在投影片上加入不同類型的框架——文字、音訊與視訊框架。  
- Aspose.Slides for .NET 使用 Managed C# 開發。您可以使用任意 .NET 語言（如 C#、VB.NET、J# 等）搭配 Aspose.Slides for .NET。此元件亦可用於任何類型的應用程式——無論是 Web 還是 Windows 應用程式。  
- Aspose.Slides for .NET 也提供完整的示範與實作範例，協助開發人員更好地了解 API。

## **Aspose.Slides 功能**
Aspose.Slides for .NET 的功能可分為以下幾組：

1. 簡報功能。  
1. 投影片功能。  
1. 圖形與多媒體功能。  
1. 表格功能。  
1. 文字與段落功能。  
1. 進階功能。

### **簡報功能**
- 從不同來源開啟 Microsoft PowerPoint 簡報。  
- 以不同的 Microsoft PowerPoint 格式儲存簡報。  
- 從頭建立簡報。  
- 在不使用其他元件的情況下將簡報轉換為 PDF 文件。  
- 讀取或修改簡報的文件屬性。  
- 為簡報套用保護。  
- 匯出簡報為 PDF。  
- 匯出簡報為 TIFF。  
- 匯出簡報為 XPS。  
- 匯出簡報為 SWF。  
- 匯出簡報為 PDF 註記 / Tiff 註記。  
- 匯出簡報為 HTML。

### **投影片功能**
- 從頭建立新投影片並將其加入簡報。  
- 透過投影片在簡報中的位置存取任意投影片。  
- 複製或克隆投影片至同一或其他簡報。  
- 編輯與刪除投影片。  
- 自動複製含投影片的母片。  
- 預覽投影片。  
- 取得或設定投影片名稱。  
- 將投影片儲存為圖片。  
- 將投影片匯出為 SVG 格式。  
- 在簡報中使用多個母片。  
- 使用任何支援的樣式填滿投影片背景。  
- 新增或管理投影片備註。  
- 動態設定新的投影片母片與樣式。  
- 新增或修改投影片放映過渡效果。  
- 新增 Morph 過渡效果。

### **圖形與多媒體功能**
- 建立或管理線條、矩形、橢圓形與折線等圖形。  
- 管理圖形的線條樣式。  
- 取得圖形的替代文字。  
- 旋轉或翻轉圖形。  
- 以不同樣式填滿圖形，例如實色、漸層、圖案、紋理與圖片。  
- 使用 48 種預定義圖案與 24 種紋理。  
- 為圖形指派超連結。  
- 讀取、修改並為投影片與圖形新增動畫效果。  
- 為簡報新增圖片並管理現有圖片。  
- 為簡報新增圖片項目符號並管理現有項目符號。  
- 為簡報新增音訊框架並管理現有音訊框架。  
- 為簡報新增視訊框架並管理現有視訊框架。

### **表格功能**
- 從頭建立表格並將其加入投影片。  
- 調整表格大小與位置。  
- 存取表格中的任意儲存格。  
- 變更任意儲存格的格式與邊框。  
- 調整欄與列的大小。  
- 新增與刪除欄與列。  
- 為整個表格設定邊框寬度與顏色。

### **文字與段落功能**
- 取得與設定文字佔位符與文字框中的文字。  
- 取得佔位符的類型。  
- 變更文字框內的文字格式。  
- 新增字型或管理簡報中已使用的字型。  
- 變更文字與文字格式。  
- 建立或管理文字的超連結。  
- 支援 Tab 與有效的 Tab。  
- 為幾乎任何類型的圖形新增文字框。  
- 存取與修改文字框的邊距與文字換行設定。  
- 存取與修改段落縮排。

### **進階功能**
- 存取簡報中嵌入的 OLE 物件。  
- 鎖定簡報。

## **聲明**
所有 Aspose .NET 元件皆需要 Full Trust 權限設定。原因在於 Aspose .NET 元件需要最高權限來存取登錄設定、系統檔案與虛擬目錄，以執行字型解析等操作。此外，Aspose .NET 元件基於 .NET 核心系統類別，許多情況也需要 Full Trust 權限設定。

提供多家公司應用程式的網際服務供應商大多強制使用 Medium Trust 安全等級。在 .NET 2.0 的情況下，該安全等級會套用以下限制：

- OleDbPermission 不可用。這表示您無法使用 ADO.NET 管理式 OLE DB 資料提供元存取資料庫。  
- EventLogPermission 不可用。這表示您無法存取 Windows 事件記錄。  
- ReflectionPermission 不可用。這表示您無法使用反射。  
- RegistryPermission 不可用。這表示您無法存取登錄。  
- WebPermission 受限。這表示您的應用程式只能與您在 <trust> 元素中定義的位址或位址範圍通訊。  
- FileIOPermission 受限。這表示您只能存取位於應用程式虛擬目錄層級中的檔案。

基於上述原因，Aspose .NET 元件只能在授予 Full Trust 權限設定的伺服器上使用。

## **常見問答**

**是否支援 PDF 符合存檔與無障礙標準 (PDF/A 與 PDF/UA)？**

是。您可以透過設定 [PDF 匯出選項](https://reference.aspose.com/slides/zh-hant/net/aspose.slides.export/pdfoptions/)，將檔案儲存為 PDF/A-2a/2b/2u、PDF/A-3a/3b，以及 PDF/UA。

**是否提供字型替換機制並支援自訂字型以確保正確呈現？**

是。此函式庫允許您 [載入自訂字型](/slides/zh-hant/net/custom-font/) 並 [定義備援規則](/slides/zh-hant/net/fallback-font/)，以在轉換與呈現時可靠地替換遺失的字形。

**我能在未完整開啟檔案的情況下偵測檔案是否受密碼保護嗎？**

是。您可以 [檢查簡報](/slides/zh-hant/net/examine-presentation/) 以判斷是否需要密碼才能載入完整文件。

**處理與轉換時是否需要安裝 Microsoft PowerPoint？**

不需要。Aspose.Slides 為獨立引擎，伺服器或工作站上皆不須安裝 PowerPoint。

**在多執行緒環境下處理簡報是否安全？**

是，您可以在不同執行緒中平行處理不同文件；只要避免在多執行緒間同時使用同一個簡報實例即可，請參考 [跨執行緒使用](/slides/zh-hant/net/multithreading/)。

**巨集是否會被保留，且能否管理 PPTM/PPSM 檔案中的 VBA？**

是。支援含巨集的簡報，您可以 [檢查與管理 VBA 專案](/slides/zh-hant/net/presentation-via-vba/)。

**我可以將 PDF 或 HTML 轉回 PowerPoint 投影片嗎？**

是。您可以 [匯入 PDF 頁面或 HTML 內容](/slides/zh-hant/net/import-presentation/) 以建立或填充簡報中的投影片。

**是否支援 XPS 匯出，且能控制 XPS 輸出的品質與內容？**

是。提供 [匯出為 XPS](/slides/zh-hant/net/convert-powerpoint-to-xps/)，並可透過 [儲存選項](https://reference.aspose.com/slides/zh-hant/net/aspose.slides.export/xpsoptions/) 調整輸出品質與包含的內容。

**我能將投影片轉換為圖片並控制輸出品質嗎？**

是。投影片可 [轉換為 PNG、JPEG、GIF、BMP、TIFF](/slides/zh-hant/net/convert-powerpoint-to-png/) 等格式，且可細緻控制尺寸與品質。

**是否支援數學公式 (MathML/MathText) 並可匯出？**

是。您可以 [建立與編輯數學文字](/slides/zh-hant/net/powerpoint-math-equations/) 並 [匯出方程式](/slides/zh-hant/net/exporting-math-equations/)，包括匯出為 MathML。

**函式庫如何處理加密與密碼（開啟、設定、移除）？**

[它支援](/slides/zh-hant/net/password-protected-presentation/) 開啟加密的簡報、設定或移除開啟與修改的密碼，並可檢查檔案的保護狀態。
