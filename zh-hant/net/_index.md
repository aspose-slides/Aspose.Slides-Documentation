**歡迎使用 Aspose.Slides for .NET**

![Aspose.Slides for .NET 產品標誌](home_1.png)

Aspose.Slides for .NET 是一個 Microsoft PowerPoint® 管理 API，讓 .NET 應用程式能在不使用 Microsoft PowerPoint® 的情況下讀寫 PowerPoint® 文件。

Aspose.Slides for .NET 是首個亦唯一提供管理 PowerPoint® 文件所需功能的元件。

Aspose.Slides for .NET 提供許多關鍵功能，例如管理文字、圖形、表格與動畫、將音訊和影片加入投影片、投影片預覽、將投影片匯出為 SVG、PDF 以及其他各種格式，還有許多其他能力。
