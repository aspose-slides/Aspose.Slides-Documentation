## **什麼是 OpenXML？**
OpenXML SDK 是 Microsoft 提供的 .NET SDK，用於處理 Microsoft Office 2007 及以上版本的檔案格式。這些檔案格式基本上是壓縮封包，Open XML SDK for Office 簡化了在封包中操作 Open XML 封包以及其底層 Open XML 架構元素的工作。Open XML SDK 中的類別封裝了開發人員在 Open XML 封包上執行的許多常見任務，使您只需少量程式碼即可執行複雜的操作。

安裝 Open XML SDK 2.5 後，於 Visual Studio 開啟現有的專案或應用程式，或建立新專案或應用程式。接著，在您的專案或應用程式中，加入下列元件的參考：

- DocumentFormat.OpenXml
- WindowsBase

## **Aspose.Slides for .NET**
Aspose.Slides for .NET 是一套獨特的 PowerPoint® 管理元件，使 .NET 應用程式能在不使用 Microsoft PowerPoint 的情況下讀取、寫入與操作 PowerPoint 文件。

Aspose.Slides for .NET 是首個也是唯一提供在您自己的應用程式中管理 PowerPoint 文件功能的 .NET 元件。與所有 Aspose .NET 元件一樣，Aspose.Slides 採用受管理的 C# 編寫，價格極具競爭力且執行速度極快。

## **簡介**
尋找檔案格式函式庫的開發人員可能需要瀏覽市面上可取得函式庫所提供的功能。主要在選擇時會考慮兩個因素：

- 功能
- 使用函式庫所需的工作量

如果您是尋求更快、簡易且輕量級檔案格式函式庫以處理檔案的 .NET 開發人員，可能需要比較 VSTO（Visual Studio tool for Office）與 Aspose。在此情況下，本專案將協助您比較兩個函式庫的功能與程式碼。
