## **什麼是 VSTO？**
VSTO（Visual Studio Tools for Office，或 Microsoft Visual Studio Tools）是 Microsoft 的 Visual Studio .NET 工具套件的一部分，支援 Microsoft .NET Framework。開發人員使用它編寫連接 Microsoft Office Word、Excel 與 Outlook 應用程式的程式碼。使用 VSTO 的開發者可以使用 C# 或 Visual Basic 語言。它是 Visual Basic for Applications（VBA）的替代方案。

## **Aspose.Slides for .NET**
Aspose.Slides for .NET 是一個獨特的 PowerPoint® 管理元件，使 .NET 應用程式能在不使用 Microsoft PowerPoint 的情況下讀取、寫入與操作 PowerPoint 文件。

Aspose.Slides for .NET 是首個也是唯一提供在自行開發的應用程式中管理 PowerPoint 文件功能的 .NET 元件。與所有 Aspose .NET 元件一樣，Aspose.Slides 使用受管理的 C# 編寫，價格極具競爭力且效能極快。

## **系統需求與支援平台**
### **系統需求**
以下任一 Visual Studio Tools for Office 版本：

- Microsoft Visual Studio 2005 Tools for the Microsoft Office System
- Microsoft Visual Studio 2005 Tools for the 2007 Microsoft Office System
- Visual Studio 2008 Professional Edition
- Visual Studio 2008 Team Suite Edition
- 任一版本的 Microsoft Office：
- Microsoft Office Professional 2003 SP1
- 2007 Microsoft Office system

### **支援的作業系統**
Windows 2000; Windows Server 2003; Windows Vista; Windows XP

### **支援的 .NET Framework 版本**
僅支援 .NET Framework 2.0 或更高版本。
