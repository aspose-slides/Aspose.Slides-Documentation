## **簡介**
Microsoft Visual Studio 是 Microsoft 提供的整合式開發環境 (IDE)。它可用於開發 Microsoft Windows 平台的電腦程式，以及網站、Web 應用程式與 Web 服務。Visual Studio 使用 Microsoft 軟體開發平台，如 Windows API、Windows Forms、Windows Presentation Foundation、Windows Store 與 Microsoft Silverlight。它能產生原生程式碼與受管理程式碼。

### **Aspose.Slides for .NET**
Aspose.Slides for .NET 是一套獨特的 PowerPoint® 管理 API，讓 .NET 應用程式能在不使用 Microsoft PowerPoint 的情況下讀取、寫入與操作 PowerPoint 文件。

Aspose.Slides for .NET 是首個也是唯一的 .NET PowerPoint API，提供在您自己的應用程式內管理 PowerPoint 文件的功能。與所有 Aspose .NET 元件一樣，Aspose.Slides 以受管理的 C# 編寫，價格親民且執行速度極快。

[關於 Aspose.Slides for .NET API 的更多資訊](http://www.aspose.com/.net/powerpoint-component.aspx)

### **使用 Visual Studio 探索 Aspose.Slides for .NET 範例**
Aspose Visual Studio Plugin - **Explore Aspose.Slides for .NET Examples** 是一個快速下載與探索 Aspose.Slides for .NET API 範例的優秀工具。它提供極簡的方式讓您不必離開 Visual Studio，即可選取、下載並開啟最新的範例專案，節省大量時間與精力。

此版本提供以下功能：

- 支援 Visual Studio 2010、Visual Studio 2012 與 Visual Studio 2013。
- 可從 Visual Studio 中的 **File > Aspose** 功能表輕鬆啟動。
- 節省時間，降低學習曲線。
- 讓您選取並開啟托管於 Github.com 的最新 Aspose.Slides for .NET API 範例。
- 可開啟 **C#** 或 **VB.NET** 範例專案。
- 選取的專案會自動在 Visual Studio 中開啟，並設定好所有必要的參考，讓您立即可執行與探索。
- 自動從 **NuGet** 下載最新的 API 程式庫，並從 **GitHub** 下載範例。

## **系統需求與受支援平台**
為了安裝與使用 Aspose Visual Studio Plugin，您必須安裝以下任一版本的 Visual Studio：

- Visual Studio 2010、2012、2013

若在安裝或使用此外掛程式時遇到任何問題，歡迎隨時與我們聯繫。

## **安裝**
安裝 Aspose Visual Studio Plugin 非常簡單，請依照以下步驟執行：

- 雙擊下載的 msi 檔案
- 按照步驟操作，您可以在每個畫面直接按 **Next**，以標準設定完成安裝。

安裝完成後會出現成功對話方塊。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_2)

**注意：** 請確保重新啟動 Visual Studio 以使變更生效。

## **使用**
安裝完成後，您即可依下列說明輕鬆啟動並使用互動式外掛程式：

**1.** 於 Visual Studio 中選取 **File > Aspose**，接著點選 **New Aspose.Slides Example Project** 開啟外掛程式。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_3)

**2.** 第一次開啟外掛程式時會顯示 API 選取對話方塊，如下圖所示。系統會花費幾分鐘時間，從 **NuGet** 下載最新的 API 與程式庫，並從 **GitHub** 取得最新的範例原始碼。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_4)

**3.** 在「Select an API to View Examples」對話方塊中，選擇或變更要建立範例專案的存放位置，並選取 **C#** 或 **VB.NET** 作為範例專案類型，然後點選 **Continue** 以在 Visual Studio 中開啟範例專案。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_1)

**4.** 依所選的 **C#** 或 **VB.NET** 專案類型，產生新的範例專案。

#### **如何設定原始碼**
開啟並擴充原始碼前，請先安裝以下軟體：

- Visual Studio 2010

請依照以下簡易步驟開始：

1. 下載/克隆原始碼。
2. 開啟 Visual Studio 2010，選取 **File** > **Open Project**。
3. 瀏覽至您剛下載的最新原始碼，開啟 **AsposeVisualStudioPlugin.sln**。

#### **原始碼概觀**
解決方案中包含兩個專案：

- **AsposeVisualStudioPlugin** - 包含與 Visual Studio 的整合、Aspose API 與範例的選取與自動下載功能。
- **Installer** - 使用 Wix 建立的安裝程式專案，用以產生外掛程式的 msi 安裝檔。
