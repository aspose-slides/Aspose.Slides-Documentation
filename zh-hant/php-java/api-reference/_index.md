{{% alert color="primary" %}} 
Aspose.Slides for PHP via Java 是一個類別庫，使 PHP 應用程式能在不使用 Microsoft PowerPoint® 或其他軟體的情況下讀寫簡報文件。Aspose.Slides for PHP via Java 允許管理文字、形狀、表格與動畫，向投影片加入音訊與影片，預覽投影片，將投影片匯出為 SVG、PDF 格式等。

最新的 API 參考請參閱 [Aspose.Slides for PHP via Java API 參考](https://reference.aspose.com/slides/zh-hant/php-java/)。
