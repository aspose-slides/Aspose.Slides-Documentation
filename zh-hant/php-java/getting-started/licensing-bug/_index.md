## **概述**

本文說明在 Aspose.Slides for PHP via Java 版本 23.2、23.3、23.4、23.5、23.6 與 23.7 中遭遇的授權錯誤。此錯誤會在永久授權的訂閱到期後，使其暫時無法使用。

## **症狀**

當您的永久授權訂閱到期後，Aspose.Slides for PHP via Java 版本 23.2 - 23.7 可能會將授權顯示為已過期，並將所有功能切換為評估模式。此行為不正確，且不會影響 23.2 之前及 23.7 之後的版本。

## **解決方案**

Aspose.Slides for PHP via Java 已針對此問題進行修正，並發布了含修復的更新版本（23.2.1、23.3.1、23.4.1、23.5.1、23.6.1、23.7.1）。

如果您的專案中使用了受影響的 Aspose.Slides for PHP via Java 版本，請改用已修正的版本。

如果您在使用 Aspose.Slides for PHP via Java 時遇到任何授權問題：

- 檢查您所使用的版本，確保它未受到上述錯誤的影響。
- 參考 [Aspose documentation](https://docs.aspose.com/slides/zh-hant/php-java/getting-started/) 以取得除錯步驟與已知授權問題。
- 聯絡 [Aspose support](https://forum.aspose.com/) 以獲得進一步協助。

## **受影響的產品與版本**

請注意，此錯誤僅影響 Aspose.Slides for PHP via Java 版本 23.2 至 23.7。**其他 Aspose 產品及其他版本的 Aspose.Slides 不受影響**。
