## **Aspose.Slides 評估**

您可以輕鬆下載 Aspose.Slides 以進行評估。評估套件與購買套件相同。只要在程式碼中加入幾行以套用授權，評估版即可轉為正式授權。

未指定授權的 Aspose.Slides 評估版提供完整的產品功能，但在開啟和儲存文件時會在文件頂部插入評估水印。從簡報投影片中擷取文字時，亦僅限於一張投影片。

## **常見問題**

**我可以在評估模式下於不同執行緒中平行測試多個簡報嗎？**

可以。您可以平行處理不同文件；不應在[跨執行緒](/slides/zh-hant/php-java/multithreading/)共享同一個簡報物件。評估模式不會影響此行為。

**我需要在伺服器或 CI 上安裝 Microsoft PowerPoint 來評估此函式庫嗎？**

不需要。Aspose.Slides 是獨立的引擎，評估或正式環境皆不需安裝 PowerPoint。

**我可以在評估模式下完整測試 PPT/PPTX 轉 PDF 與影像的轉換嗎？**

可以。[轉換器](/slides/zh-hant/php-java/convert-presentation/) 可以正常運作；輸出會包含水印。

**我可以使用臨時授權進行負載測試而不顯示水印嗎？**

可以。30 天臨時授權會移除評估模式限制，允許在沒有水印的情況下測試。
