## **Overview**

本文說明了如何取得 Aspose.Slides 的支援以及如何有效回報問題。它描述了主要的支援渠道，提供提交問題的建議，並指向有用的技術資源，如產品頁面、部落格、下載、支援論壇、文件以及 API 參考。本文亦回答了關於付費支援、問題升級以及舊版函式庫支援的常見問題。

## **Technical support**

{{% alert color="primary" %}} 
Aspose 為其所有產品提供無限制的免費技術支援。此支援對所有使用者皆開放，包括評估版使用者。
{{% /alert %}} 

如果您需要 Aspose.Slides for PHP via Java 的協助，請考慮以下方式：

- 主要的支援管道是 Aspose.Forums。請將您的問題發佈在 Aspose.Slides 論壇，問題會在數小時內得到回覆。此回覆由 Aspose.Slides 支援團隊直接提供。
- 請注意，Aspose 不提供電話技術支援。電話支援僅限於銷售與購買相關問題。
- 在等待論壇回覆時，請考慮時區差異。

如果您在使用 Aspose.Slides for PHP via Java 時遇到問題，請遵循以下簡單步驟，以確保問題得到最有效的解決：

- 在回報問題前，請確保您使用的是最新的 Aspose.Slides for PHP via Java 版本，請參閱 Aspose.Slides 下載以了解最新版本。
- 在回報問題之前，請先查看論壇、此文件與 API 參考；也許您的問題已經得到解答。
- 回報問題時，請附上原始文件以及可能導致問題的程式碼片段。若需附加多個檔案，請將它們壓縮為一個 zip 檔。將文件附加於 Aspose.Forums 是安全的，因為只有您與 Aspose 開發人員能存取這些檔案。
- 請盡量在同一討論串中只回報一個問題。若有其他問題，請另開新討論串回報。

## **FAQ**

**有哪些付費支援等級可供選擇，且它們有何差異？**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services)。它們在包含的事件數量、回應優先級以及可用的溝通渠道上有所不同。

**我可以直接將錯誤升級至工程團隊嗎？**

是的。[Paid support](https://helpdesk.aspose.com/) 允許在問題需要更深入調查時升級至開發人員。

**舊版函式庫是否受到支援，且會釋出修補程式嗎？**

您可以使用舊版，但修復僅提供給最新發行版本；不會為舊版提供修補程式。
