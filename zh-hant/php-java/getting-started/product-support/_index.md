## **技術支援**
{{% alert color="primary" %}}

Aspose 為其所有產品提供無限制的免費技術支援。支援對所有使用者皆可使用（包括使用評估套件的使用者）。

{{% /alert %}} 

當您需要 Aspose.Slides for PHP via Java 的協助時，請考慮以下事項：

- 支援的主要管道是 [Aspose.Forums](https://forum.aspose.com/c/slides/zh-hant/11)。在 Aspose.Slides 論壇發表問題—將在數小時內得到回覆。Aspose.Slides 支援團隊會直接回答論壇上的提問。
- 請注意，Aspose 並不提供電話技術支援。電話支援僅限於銷售與購買相關問題。
- 在論壇等待回覆時，請保持耐心，並考慮時區差異。

當您在 Aspose.Slides for PHP via Java 中遇到問題時，請遵循以下指南，以最高效率解決問題：

- 在回報問題前，先確認您使用的是最新的 Aspose.Slides for PHP via Java 版本。也許您所遇到的問題已在更新中得到解決。  

  請參閱 [Aspose.Slides download](https://downloads.aspose.com/slides/zh-hant/php-java) 取得最新版本資訊。

- 在回報問題之前，您可以先瀏覽我們的論壇、此文件及 API 參考。或許您的問題已在此得到答案。

- 回報問題時，請附上原始文件與（若可能）與問題相關的程式碼片段。如需附加多個檔案，請將它們壓縮成 ZIP 檔。您可以安全地在 Aspose.Forums 上上傳檔案，因為 **只有** 您與 Aspose 開發人員能存取這些附件。

- 為每個特定問題建立並使用單一討論串。若您遇到其他問題，必須另建討論串進行回報。
