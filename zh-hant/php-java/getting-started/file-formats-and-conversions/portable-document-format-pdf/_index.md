{{% alert color="primary" %}} 

[可攜式文件格式](https://en.wikipedia.org/wiki/PDF) 是由 Adobe Systems 創建的檔案格式，用於在組織之間交換文件。此格式的目的是無論在何種平台上檢視，都能保持內容與版面配置相同。Aspose.Slides for PHP via Java 允許您將簡報檔案轉換為 PDF。

{{% /alert %}} 

## **PDF 於 Aspose.Slides for PHP via Java**
任何可載入至 Aspose.Slides for PHP via Java 的簡報，都可以依照您的選擇轉換為符合 [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A)、[PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A)、[PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) 或 [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA) 的 PDF。Aspose.Slides for PHP via Java 會將簡報匯出為 PDF，且在大多數情況下，輸出的 PDF 與原始簡報完全相同。

Aspose.Slides 在轉換為 PDF 時支援以下簡報功能：

- 圖片、文字方塊與其他圖形。
- 文字與格式設定。
- 段落與格式設定。
- 超連結。
- 頁首與頁尾。
- 項目符號。
- 表格。

您可以直接使用 Aspose.Slides for PHP via Java 將簡報匯出為 PDF：不需要任何其他元件。另外，您也可以依照 [轉換為 PDF](/slides/zh-hant/php-java/convert-powerpoint-to-pdf/) 中說明的各種選項，自訂簡報至 PDF 的匯出設定。
