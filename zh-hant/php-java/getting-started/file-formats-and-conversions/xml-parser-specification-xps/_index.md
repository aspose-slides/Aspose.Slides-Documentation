{{% alert color="primary" %}} 
XML 解析器規範是一種頁面描述語言，也是最初由微軟開發的固定文件格式。與 PDF 類似，XPS 設計用於保留文件的忠實度，並提供與裝置無關的文件外觀。 
{{% /alert %}} 

## **在 Aspose.Slides for PHP via Java 中的 XPS**
任何可由 Aspose.Slides for PHP via Java 載入的簡報文件皆可轉換為 XPS 格式。Aspose.Slides for PHP via Java 使用高保真度的頁面佈局與渲染引擎，以產生固定版面的 XPS 文件格式。  
您可以在[Converting to XPS](https://docs.aspose.com/slides/zh-hant/php-java/convert-powerpoint-to-xps/) 中了解透過 Aspose.Slides for PHP via Java 將簡報文件匯出為 XPS 文件的方法。
