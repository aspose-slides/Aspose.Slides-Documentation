Aspose.Slides for PHP via Java 支援將 PPT 轉換為 PPTX。大多數簡報功能──母片、結構等──在從一種格式轉換為另一種格式時均會保留，但仍有[a few limitations](/slides/zh-hant/php-java/ppt-to-pptx-conversion/)。

{{% /alert %}} 
## **轉換支援的功能**
Aspose.Slides for PHP via Java 為 PPT 檔案格式轉換為 PPTX 提供部份支援。轉換功能剛在 Aspose.Slides for PHP via Java 中加入，因此有少數限制，最適合用於簡單的簡報。Aspose.Slides for PHP via Java 在將 PPT 轉換為 PPTX 時的主要優勢是 API 使用非常簡便。若要查看程式碼範例，請閱讀[轉換 PPT 為 PPTX](). 以下列表說明了 PPT 轉換為 PPTX 時哪個功能受支援，哪個功能不受支援。

## **支援的功能**
以下功能在轉換時受到支援：

- 轉換母片、版面配置與投影片的結構。
- 轉換圖表。
- 群組圖形。
- 轉換包含矩形與橢圓形的自動圖形。
- 具有自訂幾何圖形的圖形。
- 自動圖形的紋理與圖片填充樣式。
- 轉換佔位符。
- 轉換直線與折線。
- 線條與填充格式。
- 漸層填充樣式。
- OLE 框架、表格、影片與音訊框架等。
- 動畫與投影片屬性。
- 轉換文字框與文字佔位符中的文字。
