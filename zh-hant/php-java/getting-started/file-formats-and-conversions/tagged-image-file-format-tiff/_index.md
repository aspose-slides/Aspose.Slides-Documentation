## **關於 TIFF**

[標記圖像檔案格式](https://en.wikipedia.org/wiki/TIFF) 是一種以在單一文件中儲存多張影像聞名的格式，最初由 Aldus 所創建。此格式被掃描、傳真及其他影像處理應用程式廣泛支援。

## **在 Aspose.Slides for PHP via Java 中的 TIFF**

任何可在 Aspose.Slides for Java 中載入的文件，都能直接由 Aspose.Slides for PHP via Java 轉換為 TIFF 文件，無需任何第三方元件。此外，您還可以選擇性地定義產生之 TIFF 文件中影像的大小。您可以在[this topic](/slides/zh-hant/php-java/convert-powerpoint-to-tiff/) 中找到有關透過 Aspose.Slides for PHP via Java 將簡報文件匯出為 TIFF 文件的資訊。
