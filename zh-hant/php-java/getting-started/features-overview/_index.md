## **支援平台**
Aspose.Slides for PHP via Java 是一個平台無關的 API。它可在任何安裝了 PHP 7 或更高版本的平台（Windows、Linux、macOS 等）上使用。Oracle JDK 7 或更高版本是另一個需求。

## **檔案格式與轉換**
Aspose.Slides for PHP via Java 支援大多數 PowerPoint 文件格式。它還讓您將它們匯出為組織常用並彼此分享的流行檔案格式。請參閱以下詳細資訊：

|**功能**|**說明**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for PHP via Java 為此簡報文件格式提供最快的處理速度。|
|PPT 轉換為 PPTX|Aspose.Slides for PHP via Java 支援 PPT 轉換為 PPTX|
|Portable Document Format (PDF)|您可以使用單一方法將所有支援的檔案格式匯出為 Adobe Portable Document Format (PDF) 文件。|
|XML Parser Specification (XPS)|您可以使用單一方法將所有支援的檔案格式匯出為 XML Parser Specification (XPS) 文件。|
|Tagged Image File Format (TIFF)|您可以將所有支援的簡報檔案格式匯出為 Tagged Image File Format (TIFF)。|
|PPTX 轉換為 HTML|Aspose.Slides for PHP via Java 支援將 PresentationEx 轉換為 HTML 格式。|

## **簡報渲染**
Aspose.Slides for PHP via Java 支援高保真度的幻燈片渲染，可將簡報中的幻燈片渲染為不同的圖形格式。請參閱以下詳細資訊：

|**功能**|**說明**|
| :- | :- |
|Java 支援的影像格式|使用 Aspose.Slides for PHP via Java，您可以將簡報幻燈片以及幻燈片上的影像渲染為所有 Java 支援的圖形格式，如 TIFF、PNG、BMP、JPEG、GIF。|
|SVG Format|Aspose.Slides for PHP via Java 亦提供內建方法，允許您將簡報幻燈片匯出為可縮放向量圖形 (SVG) 格式。|
|簡報列印|最新版本的 Aspose.Slides for PHP via Java 提供具備不同選項的內建列印方法。|

## **FAQ**

**我需要在伺服器/電腦上安裝 Microsoft PowerPoint 才能使用此函式庫嗎？**

不需要。PowerPoint 並非必須；Aspose.Slides 是一個獨立的引擎，可用於建立、編輯、轉換與渲染簡報。

**多執行緒是如何運作的？處理程序可以平行化嗎？**

在不同執行緒中處理不同文件是安全的；同一個 [presentation](https://reference.aspose.com/slides/zh-hant/php-java/aspose.slides/presentation/) 物件不得同時被 [multiple threads](/slides/zh-hant/php-java/multithreading/) 使用。

**是否支援檔案密碼與加密？**

是的。您可以 [開啟](/slides/zh-hant/php-java/password-protected-presentation/) 加密的簡報、設定或移除開啟與寫入密碼，並檢查保護狀態。

**在 Linux 容器中我需要注意字型套件嗎？**

是的。建議安裝常見的字型套件，或在應用程式中明確 [指定字型目錄](/slides/zh-hant/php-java/custom-font/)，以避免意外的字型替換。

**評估版有什麼限制嗎？**

在 [evaluation mode](/slides/zh-hant/php-java/licensing/) 中，輸出會加上浮水印，且會有特定限制；可使用 [30 天暫時授權](https://purchase.aspose.com/temporary-license/) 進行完整功能測試。

**是否支援將外部格式匯入簡報 (PDF/HTML → PPTX)？**

是的。您可以將 [PDF 頁面和 HTML 內容](/slides/zh-hant/php-java/import-presentation/) 新增至簡報，將它們轉換為投影片。
