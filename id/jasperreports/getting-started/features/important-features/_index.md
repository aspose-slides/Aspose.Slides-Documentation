Bagian ini memberikan deskripsi rinci tentang fitur‑fitur Aspose.Slides untuk JasperReports. Klik tautan dalam tabel untuk mempelajari lebih lanjut tentang fitur‑fitur tersebut. 

|**Fitur** |**Deskripsi** |
| :- | :- |
|[Ekspor PPT, PPS, PPTX, dan PPSX](/slides/id/jasperreports/ppt-pptx-pdf-and-html-export/)|Ekspor laporan sebagai presentasi Microsoft PowerPoint dari aplikasi Anda tanpa menggunakan Microsoft PowerPoint. |
|Kesetiaan Tinggi terhadap Desain Laporan|Laporan PPT dan PPS memiliki tata letak dan tampilan yang sama dengan PDF dan format lainnya. |
|Penerapan yang Mudah dan Ringan|Aspose.Slides untuk JasperReport adalah eksportir JasperReport dan JasperServer ke format Microsoft PowerPoint. |
|[Lisensi yang Sederhana dan Terjangkau](/slides/id/jasperreports/licensing/)|Satu lisensi per server atau satu lisensi untuk semua server Anda. Pengembang tak terbatas. |
|[Dukungan Teknis Gratis Kelas Dunia](/slides/id/jasperreports/technical-support/)|Dukungan teknis tak terbatas disediakan langsung oleh pengembang yang bekerja pada produk ini. |
