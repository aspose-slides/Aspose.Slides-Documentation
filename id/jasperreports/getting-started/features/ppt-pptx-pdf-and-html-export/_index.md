{{% alert color="primary" %}} 

JasperReports dan JasperServer tidak memiliki kemampuan bawaan untuk mengekspor laporan sebagai presentasi Microsoft PowerPoint, tetapi dengan Aspose.Slides for JasperReports, Anda mendapatkan akses ke format ekspor tambahan:

- Presentasi Microsoft PowerPoint (PPT)
- Presentasi Microsoft PowerPoint (PPTX)
- HTML
- PDF

{{% /alert %}} 

Untuk membuat dokumen dalam format ini, Aspose.Slides for JasperReports mengandalkan versi bawaan dari [Aspose.Slides for Java](https://products.aspose.com/slides/id/java/), perpustakaan pemrosesan presentasi terkemuka di pasar dari Aspose. Microsoft PowerPoint tidak digunakan untuk menghasilkan dokumen.
