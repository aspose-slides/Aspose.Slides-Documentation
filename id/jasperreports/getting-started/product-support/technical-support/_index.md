{{% alert color="primary" %}} 

Aspose menyediakan dukungan teknis gratis tak terbatas untuk semua produk mereka. Dukungan tersedia untuk semua pengguna, termasuk versi evaluasi.

{{% /alert %}} 

Jika Anda membutuhkan bantuan dengan Aspose.Slides untuk JasperReports, pertimbangkan hal berikut:

- Jalur utama untuk dukungan adalah [**Aspose.Forum**](https://forum.aspose.com/c/slides/id/11). Kirim pertanyaan Anda di forum Aspose.Slides dan akan dijawab dalam beberapa jam. Pertanyaan dijawab langsung oleh tim dukungan Aspose.Slides.
- Harap dicatat, Aspose tidak menyediakan dukungan teknis melalui telepon. Dukungan telepon hanya tersedia untuk pertanyaan penjualan dan pembelian.
- Ketika menunggu balasan di forum, harap perhatikan perbedaan zona waktu.

Jika Anda mengalami masalah dengan Aspose.Slides untuk JasperReports, ikuti langkah-langkah sederhana berikut untuk memastikan masalah terselesaikan dengan cara paling efisien:

- Pastikan Anda menggunakan versi terbaru Aspose.Slides untuk JasperReports sebelum melaporkan masalah, lihat unduhan Aspose.Slides untuk mengetahui versi terkini.
- Lihatlah forum, dokumentasi ini, dan Referensi API sebelum melaporkan masalah; mungkin pertanyaan Anda sudah terjawab.
- Saat melaporkan masalah, sertakan dokumen asli dan mungkin potongan kode yang menyebabkan masalah. Jika Anda perlu melampirkan beberapa file, zip menjadi satu. Aman melampirkan dokumen Anda di Aspose.Forums karena hanya Anda dan pengembang Aspose yang dapat mengakses file yang dilampirkan.
- Harap laporkan satu masalah per thread. Jika Anda memiliki masalah lain, laporkan di thread terpisah.
