---
Untuk menggunakan Aspose.Slides untuk JasperReports dari aplikasi Anda, salin aspose.slides.jasperreports.library-xx.x.jar dari folder \lib\JasperReports X.X.X - X.X.X dalam aspose-slides-xx.x-jasperreports.zip ke direktori JasperReports\lib atau ke folder pustaka aplikasi Anda. Setelah itu, Anda dapat mengakses pengekspor secara program.
