Semua demo yang disediakan dengan Aspose.Slides untuk JasperReports adalah demo standar yang diubah. Lebih baik menyalin semua demo ke folder demo JasperReports:
...\jasperreports-x.x.x\demo\samples\

Gunakan urutan perintah standar untuk membangun dan mengekspor laporan:

- ant javac
- ant compile
- ant fill
- ant ppt

{{% alert color="primary" %}} 
Harap jangan lupa menjalankan HSQLDB dengan basis data uji untuk mengisi laporan dengan data dan menyalin aspose.slides.jasperreports.library-xx.x.jar dari \lib\JasperReports X.X.X - X.X.X folder dalam aspose-slides-xx.x-jasperreports.zip ke &#60;InstallDir&#62;\lib directory.
{{% /alert %}} 

Sebagian besar demo (kecuali Charts) sudah memiliki presentasi yang dihasilkan sehingga Anda dapat melewatkan semua langkah “ant” dan memeriksa hasilnya segera.
