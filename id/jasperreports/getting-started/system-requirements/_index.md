Aspose.Slides for JasperReports bekerja dengan:

**Aspose.Slides for JasperReports 17.5 dan versi sebelumnya:**

- Java 2 Platform Standard Edition 1.4 atau lebih tinggi.
- JasperReports 2.0, 3.0 atau lebih tinggi.

**Aspose.Slides for JasperReports 17.6 dan versi yang lebih baru:**

- Java 2 Platform Standard Edition 6.0 atau lebih tinggi.
- JasperReports 3.7.2 atau lebih tinggi.
