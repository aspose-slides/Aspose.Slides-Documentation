**Selamat datang di Aspose.Slides for JasperReports**

Aspose.Slides for JasperReports adalah pustaka yang dirancang khusus dan dikembangkan untuk pengembang yang perlu dengan mudah mengekspor laporan dari JasperReports ke format Microsoft PowerPoint Presentation (PPT) dan Microsoft PowerPoint Show (PPS) dalam aplikasi Java mereka. Semua fitur laporan dikonversi dengan tingkat presisi tertinggi ke presentasi Microsoft PowerPoint. Aspose.Slides for JasperReports mencakup dukungan untuk JasperReports 5+.
