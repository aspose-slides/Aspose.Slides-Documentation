{{% alert color="primary" %}}

Semua komponen Aspose C++ memerlukan set izin Full Trust. Alasannya, komponen Aspose C++ perlu mengakses pengaturan registri, file sistem selain direktori virtual untuk operasi tertentu seperti parsing font, dll. Selain itu, Komponen Aspose C++ didasarkan pada kelas sistem C++ inti yang juga memerlukan set izin Full Trust dalam banyak kasus.
