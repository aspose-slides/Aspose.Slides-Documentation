## **Unduh Aspose.Slides dari GitHub**
Semua contoh Aspose.Slides untuk C++ dihosting di [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-C).

- Anda dapat meng-clone repositori menggunakan klien GitHub favorit Anda atau mengunduh file ZIP dari [sini](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/refs/heads/master.zip).
- Ekstrak isi file ZIP ke folder mana saja di komputer Anda. Semua contoh berada di folder **Examples**.
- Terdapat file solusi Visual Studio C++.
- Proyek-proyek dibuat di Visual Studio 2015, tetapi file solusi kompatibel dengan Visual Studio 2015 dan yang lebih baru.
- Buka file solusi di Visual Studio dan bangun proyek.
- Unduh rilis terbaru Aspose.Slides untuk C++ dari [Bagian Unduhan](https://downloads.aspose.com/slides/id/cpp).
- Ekstrak rilis dan salin direktori **Include** dan **Bin** ke dalam folder **Examples**.
- Buka file **main.cpp**, semua contoh dipanggil dari sini.
- Hapus komentar pada contoh yang ingin Anda jalankan dalam proyek.
- Bangun dan kompilasi solusi

Jangan ragu menghubungi kami melalui Forum jika Anda mengalami masalah dalam menyiapkan atau menjalankan contoh.
