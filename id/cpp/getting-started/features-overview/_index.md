## **Format File dan Konversi**
Aspose.Slides for C++ mendukung sebagian besar format dokumen PowerPoint dan juga mengekspor ke format populer yang banyak digunakan untuk pertukaran antar organisasi. Tabel berikut mencantumkan format-format tersebut:

|**Fitur**|**Deskripsi**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/id/cpp/microsoft-powerpoint-ppt/)|Aspose.Slides for C++ menyediakan pemrosesan tercepat untuk format dokumen presentasi ini.|
|[Portable Document Format (PDF)](/slides/id/cpp/portable-document-format-pdf/)|Format file yang didukung dapat diekspor ke dokumen Adobe Portable Document Format (PDF) dengan satu metode.|
|[XML Parser Specification (XPS)](/slides/id/cpp/xml-parser-specification-xps/)|Semua format file yang didukung dapat diekspor ke dokumen XML Parser Specification (XPS) dengan satu metode.|
|[PresentationML (PPTX, XML)](/slides/id/cpp/presentationml-pptx-xml/)|Aspose.Slides for C++ mendukung pemrosesan format presentasi OOXML yang juga dikenal sebagai PresentationML atau PPTX.|
|[Konversi PPTX ke HTML](https://docs.aspose.com/slides/id/cpp/convert-powerpoint-to-html/)|Aspose.Slides for C++ mendukung konversi presentasi ke format HTML.|

| :- | :- |
|Format Gambar yang Didukung C++|Dengan Aspose.Slides for C++, Anda dapat merender tidak hanya slide presentasi, tetapi juga gambar pada slide ke semua format grafis yang didukung C++ seperti PNG, BMP, dan JPEG.|

## **FAQ**

**Apakah saya perlu menginstal Microsoft PowerPoint di server/PC agar pustaka ini berfungsi?**

Tidak. PowerPoint tidak diperlukan; Aspose.Slides adalah mesin mandiri untuk membuat, mengedit, mengkonversi, dan merender presentasi.

**Bagaimana cara kerja multithreading? Apakah pemrosesan dapat diparalelkan?**

Aman untuk memproses dokumen yang berbeda di thread yang berbeda; objek [presentasi](https://reference.aspose.com/slides/id/cpp/aspose.slides/presentation/) yang sama tidak boleh digunakan oleh [banyak thread](/slides/id/cpp/multithreading/) secara bersamaan.

**Apakah kata sandi file dan enkripsi didukung?**

Ya. Anda dapat [membuka](/slides/id/cpp/password-protected-presentation/) presentasi yang terenkripsi, mengatur atau menghapus kata sandi buka dan tulis, serta memeriksa status perlindungan.

**Apakah saya perlu memperhatikan paket font di kontainer Linux?**

Ya. Disarankan untuk menginstal paket font umum dan/atau secara eksplisit [menentukan direktori font](/slides/id/cpp/custom-font/) dalam aplikasi Anda untuk menghindari substitusi yang tidak terduga.

**Apakah ada batasan dalam versi evaluasi?**

Dalam [mode evaluasi](/slides/id/cpp/licensing/), watermark ditambahkan ke output dan beberapa batasan berlaku; [lisensi sementara 30 hari](https://purchase.aspose.com/temporary-license/) tersedia untuk pengujian fitur lengkap.

**Apakah impor format eksternal ke dalam presentasi (PDF/HTML → PPTX) didukung?**

Ya. Anda dapat menambahkan [halaman PDF dan konten HTML](/slides/id/cpp/import-presentation/) ke sebuah presentasi, mengubahnya menjadi slide.
