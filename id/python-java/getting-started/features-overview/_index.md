## **Platform yang Didukung**
Aspose.Slides for Python via Java adalah API yang independen platform. Itu dapat digunakan pada platform apa pun (Windows, Linux, MacOS, dll.) dimana Python 3.7 atau yang lebih tinggi terinstal. JDK 8 atau yang lebih tinggi merupakan persyaratan lain.

## **Format File dan Konversi**
Aspose.Slides for Python via Java mendukung sebagian besar format dokumen PowerPoint. Ini juga memungkinkan Anda mengekspor mereka ke format file populer yang digunakan organisasi dan dibagikan di antara mereka. Lihat detail berikut:

|**Fitur**|**Deskripsi**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Python via Java menyediakan pemrosesan tercepat untuk format dokumen presentasi ini.|
|PPT to PPTX conversion|Aspose.Slides for Python via Java mendukung konversi PPT ke PPTX|
|Portable Document Format (PDF)|Anda dapat mengekspor semua format file yang didukung ke dokumen Adobe Portable Document Format (PDF) dengan satu metode.|
|XML Parser Specification (XPS)|Anda dapat mengekspor semua format file yang didukung ke dokumen XML Parser Specification (XPS) dengan satu metode.|
|Tagged Image File Format (TIFF)|Anda dapat mengekspor semua format file presentasi yang didukung ke Tagged Image File Format (TIFF).|
|PPTX To HTML Conversion|Aspose.Slides for Python via Java mendukung konversi PresentationEx ke format HTML.|

## **Rendering dan Pencetakan**
Aspose.Slides for Python via Java mendukung rendering berfidelity tinggi dari slide dalam presentasi ke berbagai format grafik. Lihat detail berikut:

|**Fitur**|**Deskripsi**|
| :- | :- |
|Java Supported Image Formats|Dengan menggunakan Aspose.Slides for Python via Java, Anda dapat merender slide presentasi dan gambar pada slide ke semua format grafik yang didukung Java seperti TIFF, PNG, BMP, JPEG, GIF.|
|SVG Format|Aspose.Slides for Python via Java juga menyediakan metode bawaan yang memungkinkan Anda mengekspor slide presentasi ke format Scalable Vector Graphics (SVG).|
|Presentation Printing|Versi terbaru Aspose.Slides for Python via Java menyediakan metode cetak bawaan dengan berbagai opsi.|

|Rectangle Shapes|Anda dapat membuat atau memodifikasi bentuk persegi panjang dengan Aspose.Slides for Python via Java.|
