{{% alert color="primary" %}}

Aspose.Slides for .NET adalah pustaka kelas yang memungkinkan aplikasi .NET membaca dan menulis dokumen presentasi tanpa menggunakan Microsoft PowerPoint® atau lainnya. Aspose.Slides for .NET memungkinkan mengelola teks, bentuk, tabel & animasi, menambahkan audio dan video ke slide, meninjau slide, mengekspor slide ke format SVG, PDF, dan lainnya.

Referensi API terbaru dapat ditemukan di [Aspose.Slides for .NET API Reference](https://reference.aspose.com/slides/id/net).

{{% /alert %}}
