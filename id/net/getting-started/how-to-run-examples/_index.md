## **Persyaratan Perangkat Lunak**
Sebelum Anda mengunduh dan menjalankan contoh, harap periksa dan pastikan bahwa pengaturan Anda memenuhi persyaratan berikut: 

- Visual Studio 2010 atau lebih tinggi.
- NuGet Package Manager terpasang di Visual Studio. Verifikasi bahwa versi API NuGet terbaru terpasang di Visual Studio. 

Untuk petunjuk menginstal NuGet package manager, kunjungi halaman ini: https://docs.microsoft.com/en-us/nuget/install-nuget-client-tools

1. Buka **Tools** > **Options** > **NuGet Package Manager**.

1. Perluas **NuGet Package Manager** (dengan mengklik dua kali) dan kemudian pilih **Package Sources**. 

1. Periksa dan pastikan bahwa parameter nuget.org dipilih. 

   Proyek contoh menggunakan fitur NuGet Automatic Package Restore, sehingga Anda memerlukan koneksi internet yang aktif. 

   Jika Anda tidak memiliki koneksi internet aktif pada mesin tempat Anda akan menjalankan contoh, silakan periksa [Installation](https://docs.aspose.com/slides/id/net/installation/) dan (secara manual) tambahkan referensi ke Aspose.Slides.dll dalam proyek contoh.
## **Unduh Aspose.Slides dari GitHub**
Semua contoh Aspose.Slides untuk .NET dihosting di [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-.NET).

Anda dapat mengkloning repositori menggunakan klien GitHub favorit Anda atau mengunduh file ZIP [here](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/master.zip).

1. Jika Anda mengunduh file ZIP, Anda harus mengekstrak isinya ke folder di komputer Anda. 

Semua contoh disimpan dalam folder **Examples**.

Ada file solusi Visual Studio C#. Proyek-proyek dibuat di Visual Studio 2013, tetapi file solusi kompatibel dengan Visual Studio 2010 SP1 dan yang lebih tinggi.

2. Buka file solusi di Visual Studio dan bangun (build) proyek.

   Pada menjalankan pertama, dependensi akan diunduh secara otomatis melalui NuGet.

Folder **Data** di folder akar **Examples** berisi file input yang digunakan dalam contoh C#. Anda harus mengunduh folder **Data** bersamaan dengan proyek contoh.

3. Buka file RunExamples.cs. Semua contoh dipanggil dari sini.

4. Hapus komentar pada contoh yang ingin Anda jalankan dalam proyek.

Silakan hubungi kami melalui forum jika Anda mengalami masalah dalam menyiapkan atau menjalankan contoh.
## **Berkontribusi**
Anda dapat berkontribusi pada proyek dengan menambahkan atau memperbaiki contoh. Semua contoh dan proyek showcase dalam repositori bersifat open-source, sehingga Anda (dan orang lain) dapat menggunakannya secara bebas dalam aplikasi.

Untuk berkontribusi, Anda dapat fork repositori, mengedit kode sumber, dan membuat pull request. Kami akan meninjau perubahan tersebut. Jika kami menemukan mereka berguna, kami akan menambahkannya ke repositori.
