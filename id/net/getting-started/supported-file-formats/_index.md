## **Gambaran Umum**

Aspose.Slides mendukung file presentasi dari Microsoft PowerPoint 97 hingga Office 365, termasuk Microsoft PowerPoint untuk Mac. Artikel ini mencantumkan versi PowerPoint yang didukung oleh perpustakaan dan menyediakan tabel format file yang dapat dimuat, disimpan, atau keduanya.

Artikel ini juga menjawab pertanyaan umum tentang kepatuhan PDF, penyisipan font, file yang dilindungi password, font khusus, fallback font, dan opsi ekspor XPS.

## **FAQ**

**Apakah saya dapat menyimpan presentasi ke PDF yang memenuhi standar arsip dan aksesibilitas (PDF/A dan PDF/UA)?**

Ya. Aspose.Slides mendukung mengekspor ke PDF dengan tingkat kepatuhan seperti PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b, serta PDF/UA melalui pengaturan [compliance](https://reference.aspose.com/slides/id/net/aspose.slides.export/pdfoptions/compliance/) dalam [PDF export options](https://reference.aspose.com/slides/id/net/aspose.slides.export/pdfoptions/).

**Apakah perpustakaan mendukung penyisipan font saat mengekspor ke PDF, dengan kontrol detail atas apa yang disisipkan?**

Ya. Anda dapat mengontrol apakah font disisipkan sepenuhnya atau hanya subset (hanya glif yang digunakan), menentukan bagaimana font sistem umum diperlakukan, dan mengonfigurasi perilaku teks ASCII melalui [PDF export options](https://reference.aspose.com/slides/id/net/aspose.slides.export/pdfoptions/).

**Apakah saya dapat mendeteksi apakah sebuah file dilindungi password sebelum benar‑benar memuatnya?**

Ya. Dengan menggunakan [factory-based inspection API](https://reference.aspose.com/slides/id/net/aspose.slides/presentationfactory/), Anda dapat mengkueri file presentasi untuk menentukan apakah ia dilindungi password tanpa membuka sepenuhnya.

**Apakah ada mekanisme fallback font dan dukungan untuk font khusus?**

Ya. Perpustakaan mendukung [loading](/slides/id/net/custom-font/) dan [embedding](/slides/id/net/embedded-font/) font khusus serta menyediakan [fallback rules](/slides/id/net/fallback-font/) font untuk mencegah hilangnya glif saat rendering dan konversi.

**Apakah saya dapat mengekspor slide ke XPS, dan apakah ada opsi untuk menyesuaikan output XPS?**

Ya. [Export to XPS](/slides/id/net/convert-powerpoint-to-xps/) didukung, dan Anda dapat menyesuaikan [save options](https://reference.aspose.com/slides/id/net/aspose.slides.export/xpsoptions/) yang relevan untuk mengontrol kualitas output dan konten dokumen XPS.
