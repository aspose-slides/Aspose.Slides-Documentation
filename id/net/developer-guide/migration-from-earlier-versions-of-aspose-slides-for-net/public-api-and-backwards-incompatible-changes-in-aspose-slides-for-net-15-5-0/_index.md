## **Perubahan API Publik**
#### **Kelas CommonSlideViewProperties dan Antarmuka ICommonSlideViewProperties Telah Ditambahkan**
Kelas Aspose.Slides.CommonSlideViewProperties dan antarmuka Aspose.Slides.ICommonSlideViewProperties mewakili properti tampilan slide umum (saat ini opsi skala tampilan).
#### **Properti IAxis.LabelOffset Telah Ditambahkan**
Properti IAxis.LabelOffset menentukan jarak label dari sumbu. Diterapkan pada sumbu kategori atau tanggal.
#### **Properti IChartTextBlockFormat.AutofitType Telah Ditambahkan**
Mengubah properti ini dapat memberikan pengaruh tertentu hanya pada bagian diagram berikut: DataLabel dan DataLabelFormat (dukungan penuh di PowerPoint 2013; di PowerPoint 2007 tidak berpengaruh pada render).
#### **Properti IChartTextBlockFormat.WrapText Telah Ditambahkan**
Mengubah properti ini dapat memberikan pengaruh tertentu hanya pada bagian diagram berikut: DataLabel dan DataLabelFormat (dukungan penuh di PowerPoint 2007/2013).
#### **Properti Margin Telah Ditambahkan ke IChartTextBlockFormat**
Mengubah properti ini dapat memberikan pengaruh tertentu hanya pada bagian diagram berikut: DataLabel dan DataLabelFormat (dukungan penuh di PowerPoint 2013; di PowerPoint 2007 tidak berpengaruh pada render).
#### **Properti ViewProperties.NotesViewProperties Telah Ditambahkan**
Properti Aspose.Slides.ViewProperties.NotesViewProperties telah ditambahkan. Ini menentukan properti tampilan umum yang terkait dengan mode tampilan catatan.
#### **Properti ViewProperties.SlideViewProperties Telah Ditambahkan**
Properti Aspose.Slides.ViewProperties.SlideViewProperties telah ditambahkan. Ini menentukan properti tampilan umum yang terkait dengan mode tampilan slide.
