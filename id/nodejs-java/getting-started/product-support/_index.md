## **Ringkasan**

Artikel ini menjelaskan cara mendapatkan dukungan untuk Aspose.Slides dan cara melaporkan masalah secara efektif. Artikel ini menjabarkan saluran dukungan utama, memberikan rekomendasi untuk mengirimkan masalah, dan menunjuk ke sumber daya teknis yang berguna seperti halaman produk, blog, unduhan, forum dukungan, dokumentasi, dan Referensi API. Artikel ini juga menjawab pertanyaan umum tentang dukungan berbayar, eskalasi masalah, dan dukungan untuk versi perpustakaan yang lebih lama.

## **Dukungan teknis**
{{% alert color="primary" %}}

Aspose menyediakan dukungan teknis gratis tak terbatas untuk semua produknya. Dukungan tersedia untuk semua pengguna (termasuk pengguna dengan paket evaluasi).

{{% /alert %}} 

Saat Anda memerlukan bantuan dengan Aspose.Slides for Node.js via Java, pertimbangkan hal berikut:

- Saluran utama untuk dukungan adalah [Aspose.Forums](https://forum.aspose.com/c/slides/id/11). Ajukan pertanyaan Anda di forum Aspose.Slides—dan pertanyaan tersebut akan dijawab dalam beberapa jam. Tim dukungan Aspose.Slides secara langsung menjawab pertanyaan yang diposting di forum.
- Harap dicatat bahwa Aspose tidak menyediakan dukungan teknis melalui telepon. Dukungan telepon hanya tersedia untuk pertanyaan penjualan dan pembelian.
- Saat menunggu balasan di forum, harap bersabar dan pertimbangkan perbedaan zona waktu.

Saat Anda mengalami masalah dengan Aspose.Slides for Node.js via Java, ikuti panduan berikut untuk menyelesaikannya dengan cara paling efisien:

- Periksa dan pastikan bahwa Anda menggunakan versi terbaru Aspose.Slides for Node.js via Java sebelum melaporkan masalah. Mungkin masalah yang Anda alami sudah teratasi dalam pembaruan. 

  Lihat [Aspose.Slides download](https://releases.aspose.com/slides/id/nodejs-java/) untuk mendapatkan informasi tentang versi terbaru.

- Sebelum melaporkan masalah, Anda dapat menelusuri forum kami, dokumentasi ini, dan Referensi API. Mungkin jawaban atas pertanyaan Anda sudah tersedia. 

- Saat melaporkan masalah, sertakan dokumen asli dan (jika memungkinkan) potongan kode yang berkaitan dengan masalah tersebut. Jika Anda perlu melampirkan banyak file, kemas dalam paket ZIP. Anda dapat dengan aman melampirkan dokumen Anda di Aspose.Forums karena **only** Anda dan pengembang Aspose yang memiliki akses ke file yang dilampirkan.

- Buat dan gunakan satu thread untuk masalah tertentu. Jika Anda mengalami masalah lain, Anda harus membuat thread lain untuk melaporkannya.

## **FAQ**

**Tier dukungan berbayar apa yang tersedia dan bagaimana perbedaannya?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Mereka berbeda berdasarkan jumlah insiden yang termasuk, prioritas respons, dan saluran komunikasi yang tersedia.

**Apakah saya dapat mengeskalasikan bug langsung ke tim rekayasa?**

Ya. [Paid support](https://helpdesk.aspose.com/) memungkinkan eskalasi ke pengembang ketika sebuah isu memerlukan penyelidikan lebih mendalam.

**Apakah versi perpustakaan yang lebih lama didukung, dan apakah Anda merilis perbaikan untuknya?**

Anda dapat menggunakan versi yang lebih lama, tetapi perbaikan hanya disediakan untuk rilis saat ini; patch untuk versi legacy tidak diterbitkan.
