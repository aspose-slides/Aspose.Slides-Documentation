## **Platform yang Didukung**
Aspose.Slides for Node.js via Java adalah API yang independen dari platform. Dapat digunakan pada platform apa pun (Windows, Linux, MacOS, dll.) yang telah menginstal Node.js 7 atau lebih tinggi. Oracle JDK 8 atau lebih tinggi juga diperlukan.

## **Format File dan Konversi**
Aspose.Slides for Node.js via Java mendukung sebagian besar format dokumen PowerPoint. Ia juga memungkinkan Anda mengekspor ke format file populer yang digunakan organisasi untuk berbagi. Lihat detail berikut:

|**Fitur**|**Deskripsi**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via Java menyediakan pemrosesan tercepat untuk format dokumen presentasi ini.|
|Konversi PPT ke PPTX|Aspose.Slides for Node.js via Java mendukung konversi PPT ke PPTX|
|Portable Document Format (PDF)|Anda dapat mengekspor semua format file yang didukung ke dokumen Adobe Portable Document Format (PDF) dengan satu metode.|
|XML Parser Specification (XPS)|Anda dapat mengekspor semua format file yang didukung ke dokumen XML Parser Specification (XPS) dengan satu metode.|
|Tagged Image File Format (TIFF)|Anda dapat mengekspor semua format file presentasi yang didukung ke Tagged Image File Format (TIFF).|
|Konversi PPTX ke HTML|Aspose.Slides for Node.js via Java mendukung konversi PresentationEx ke format HTML.|

## **FAQ**

**Apakah saya perlu menginstal Microsoft PowerPoint di server/PC agar perpustakaan ini berfungsi?**

Tidak. PowerPoint tidak diperlukan; Aspose.Slides adalah mesin mandiri untuk membuat, mengedit, mengonversi, dan merender presentasi.

**Bagaimana multithreading bekerja? Apakah pemrosesan dapat diparalelkan?**

Aman untuk memproses dokumen yang berbeda di thread yang berbeda; objek [presentasi](https://reference.aspose.com/slides/id/nodejs-java/aspose.slides/presentation/) yang sama tidak boleh digunakan oleh [multiple threads](/slides/id/nodejs-java/multithreading/) pada waktu yang bersamaan.

**Apakah kata sandi file dan enkripsi didukung?**

Ya. [Anda dapat](/slides/id/nodejs-java/password-protected-presentation/) membuka presentasi yang dienkripsi, menentukan atau menghapus kata sandi buka dan tulis, serta memeriksa status perlindungan.

**Apakah saya perlu memperhatikan paket font di kontainer Linux?**

Ya. Disarankan untuk menginstal paket font umum dan/atau secara eksplisit [menentukan direktori font](/slides/id/nodejs-java/custom-font/) dalam aplikasi Anda untuk menghindari substitusi yang tidak terduga.

**Apakah ada keterbatasan dalam versi evaluasi?**

Dalam [mode evaluasi](/slides/id/nodejs-java/licensing/), watermark ditambahkan pada output dan beberapa keterbatasan diterapkan; [lisensi sementara 30 hari](https://purchase.aspose.com/temporary-license/) tersedia untuk pengujian penuh fitur.

**Apakah mengimpor format eksternal ke dalam presentasi (PDF/HTML → PPTX) didukung?**

Ya. Anda dapat menambahkan [halaman PDF dan konten HTML](/slides/id/nodejs-java/import-presentation/) ke sebuah presentasi, mengubahnya menjadi slide.
