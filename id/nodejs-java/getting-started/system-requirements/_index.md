## **Pendahuluan**

Aspose.Slides for Node.js via Java tidak memerlukan produk pihak ketiga apa pun seperti Microsoft PowerPoint untuk diinstal. Aspose.Slides sendiri merupakan mesin untuk membuat, memodifikasi, mengonversi, dan merender dokumen dalam berbagai format, termasuk format presentasi Microsoft PowerPoint.

## **Sistem Operasi yang Didukung**

## **FAQ**

**Apakah saya perlu menginstal Microsoft PowerPoint untuk konversi dan rendering?**

Tidak, PowerPoint tidak diperlukan; Aspose.Slides adalah mesin mandiri untuk [membuat](/slides/id/nodejs-java/create-presentation/), memodifikasi, [mengonversi](/slides/id/nodejs-java/convert-presentation/), dan [merender](/slides/id/nodejs-java/convert-powerpoint-to-png/) presentasi.

**Font apa yang dibutuhkan untuk rendering yang tepat?**

Secara praktik, font yang digunakan dalam presentasi atau [pengganti](/slides/id/nodejs-java/font-substitution/) yang tepat harus tersedia. Untuk memastikan rendering yang konsisten pada Linux/macOS, disarankan memasang paket font umum.

**Mengapa font khusus dirender sebagai fallback atau teks yang hilang di Linux?**

Jika berkas font memiliki entri tabel nama yang tidak konsisten atau rusak, tumpukan pencocokan font Linux (FreeType/fontconfig) dapat memilih rekaman yang tidak valid, sehingga font tidak dapat diidentifikasi. Menggunakan versi font dengan entri tabel nama yang telah diperbaiki atau memasang pengganti yang konsisten menyelesaikan masalah tersebut.
