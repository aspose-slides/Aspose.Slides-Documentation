{{% alert color="primary" %}} 

Aspose.Slides for Node.js via Java adalah perpustakaan kelas yang memungkinkan aplikasi Node.js membaca dan menulis dokumen presentasi tanpa menggunakan Microsoft PowerPoint® atau lainnya. Aspose.Slides for Node.js via Java memungkinkan Anda mengelola teks, bentuk, tabel & animasi, menambahkan audio dan video ke slide, meninjau slide, mengekspor slide ke format SVG, PDF, dan banyak lagi.

Referensi API terbaru dapat ditemukan di [Aspose.Slides for Node.js via Java API Reference](https://reference.aspose.com/slides/id/nodejs-java/).
