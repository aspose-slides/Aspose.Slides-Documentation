{{% alert color="primary" %}} 

Aspose.Slides for Reporting Services adalah sebuah [ekstensi rendering](http://msdn2.microsoft.com/en-us/library/ms154606.aspx) untuk Microsoft SQL Server Reporting Services. 
Aspose.Slides for Reporting Services disediakan sebagai satu paket instalasi MSI yang dapat dipasang pada komputer yang menjalankan salah satu hal berikut: 

- Microsoft SQL Server 2005 Reporting Services (32-bit dan 64-bit)
- Microsoft SQL Server 2008 Reporting Services (32-bit dan 64-bit)

Selain itu, mudah untuk menyebarkan dan mengelola Aspose.Slides for Reporting Services secara manual, karena hanya terdiri dari satu assembly .NET *Aspose.Slides* *.ReportingServices.dll* , yang ditulis sepenuhnya dalam C#, mematuhi CLS, dan hanya berisi kode terkelola yang aman. 

{{% /alert %}} 

Paket instalasi MSI dan unduhan ZIP mencakup Aspose.Slides for ReportingServices: 

- Bin\SSRS2005\Aspose.Slides.ReportingServices.dll – dibangun untuk Microsoft SQL Server 2005 dan .NET Framework 2.0 (gunakan untuk x86 dan x64)
- Bin\SSRS2008\Aspose.Slides.ReportingServices.dll – dibangun untuk Microsoft SQL Server 2008 dan .NET Framework 2.0 (gunakan untuk x86 dan x64)

Saat menginstal, Aspose.Slides.ReportingServices.dll disalin ke direktori ReportServer\bin dan file konfigurasi diperbarui sehingga Reporting Services menyadari adanya ekstensi rendering baru. Langkah-langkah ini dilakukan oleh instalator Aspose.Slides for Reporting Services, tetapi Anda juga dapat melakukannya secara manual seperti yang dijelaskan lebih lanjut dalam dokumentasi ini.
