{{% alert color="primary" %}} 

Microsoft SQL Server Reporting Services tidak memiliki kemampuan bawaan untuk mengekspor laporan sebagai presentasi Microsoft PowerPoint, tetapi setelah menginstal Aspose.Slides untuk Reporting Services di server Anda, Anda akan mendapatkan akses ke format ekspor tambahan berikut: 

- Microsoft PowerPoint Presentation (PPT)
- Microsoft PowerPoint SlideShow (PPS)
- Microsoft PowerPoint 2007 Presentation (PPTX)
- Microsoft PowerPoint 2007 SlideShow (PPSX)

{{% /alert %}} 

Untuk membuat dokumen dalam format ini, Aspose.Slides untuk Reporting Services mengandalkan versi bawaan dari [Aspose.Slides for .NET](http://www.aspose.com/Products/Aspose.Slides/), komponen pemrosesan presentasi terkemuka di pasar dari Aspose. Microsoft PowerPoint tidak digunakan untuk menghasilkan dokumen. 


Salinan PPT dan PPS dari laporan yang sama akan terlihat identik satu sama lain, berkat dukungan tingkat tinggi untuk format file dasar, yang disediakan oleh Aspose.Slides.
