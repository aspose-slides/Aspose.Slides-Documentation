{{% alert color="primary" %}} 

Aspose.Slides for Reporting Services sepenuhnya mendukung semua spesifikasi RDL. Ini berarti dua hal hebat:

- Tidak perlu mendesain ulang laporan yang sudah ada. Anda dapat mengekspor laporan RDL yang ada menjadi presentasi Microsoft PowerPoint dan akan muncul persis sesuai desain RDL.
- Tidak perlu menggunakan perancang laporan tertentu. Anda dapat menggunakan perancang laporan RDL apa pun dan laporan akan diekspor persis seperti yang Anda rancang.

{{% /alert %}} 

Aspose.Slides for Reporting Services mendukung elemen RDL berikut:

- Halaman, header, footer
- Kotak teks
- Gambar
- Sub-laporan
- Grafik
- Daftar
- Tabel
- Matriks
- Gaya
- Persegi panjang

**Contoh laporan dengan header, footer, gambar, sub-laporan, tabel, kotak teks, dan persegi panjang yang diekspor sebagai presentasi Microsoft PowerPoint (PPT).**
