## Jenis Lisensi 
{{% alert color="primary" %}} Saat ini tersedia dua jenis lisensi: Server Subscription Site Subscription {{% /alert %}}
Setiap lisensi mencakup upgrade gratis selama 1 tahun ke versi baru produk apa pun yang dirilis dan dukungan teknis gratis tanpa batas selama 1 tahun. 

Versi evaluasi dan berbayar Aspose.Slides for Reporting Services menggunakan unduhan yang sama. Cukup unduh Aspose.Slides for Reporting Services dan instal di server, maka akan berjalan dalam mode evaluasi. 


Pastikan memanfaatkan evaluasi Aspose.Slides for Reporting Services karena tidak memiliki batas waktu dan dukungan teknis gratis disediakan untuk pengguna evaluasi. 


Mode evaluasi menyisipkan watermark ke dalam laporan yang diekspor. Saat Anda [membeli lisensi](https://purchase.aspose.com/buy), cukup salin lisensi ke folder yang tepat dan Aspose.Slides for Reporting Services akan mulai beroperasi dalam mode berlisensi. 
## **Server Subscription**
Server subscription provides following features:

- Satu lisensi dibutuhkan untuk setiap server fisik atau virtual.
- Jumlah prosesor atau inti tanpa batas.
- Jumlah instance Microsoft SQL Server tanpa batas.
- Jumlah pengembang tanpa batas.
- Rilis baru dan perbaikan cepat gratis selama 1 tahun.
- Dukungan teknis gratis selama 1 tahun.
- Diskon berlaku untuk pembelian banyak.
## **Site Subscription**
Site subscription provides following features:

- Satu lisensi dibutuhkan per satu lokasi fisik (kampus).
- Jumlah server fisik atau virtual tanpa batas.
- Jumlah prosesor atau inti tanpa batas.
- Jumlah instance Microsoft SQL Server tanpa batas.
- Jumlah pengembang tanpa batas.
- Rilis baru dan perbaikan cepat gratis selama 1 tahun.
- Dukungan teknis gratis selama 1 tahun.
- Diskon berlaku untuk pembelian banyak.
