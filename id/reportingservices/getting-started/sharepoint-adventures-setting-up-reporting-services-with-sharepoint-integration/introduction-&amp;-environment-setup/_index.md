#### **Pengaturan Lingkungan**
Pengaturan yang akan kami gunakan terdiri dari **4 server**. Itu mencakup **Domain Controller**, **SQL Server**, **Server SharePoint**, dan server untuk **Reporting Services**. Anda dapat memilih untuk menempatkan SharePoint dan Reporting Services pada satu mesin.
