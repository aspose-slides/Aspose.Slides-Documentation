## **Power BI**
Anda dapat menginstal Aspose.Slides untuk Reporting Services pada server Power BI melalui proses instalasi yang sama:

Lihat

* [Instal dengan MSI Installer](https://docs.aspose.com/slides/id/reportingservices/install-with-msi-installer/#installation). Program instalasi secara otomatis mendeteksi Power BI yang terpasang di server, menginstal produk, dan kemudian mengonfigurasinya untuk lingkungan kerja Anda.
* [Instal Secara Manual](https://docs.aspose.com/slides/id/reportingservices/install-manually/).

{{% alert title="Note" color="warning" %}} 
**Aspose.Slides for Reporting Services** memerlukan instalasi **.NET Framework 3.5** pada mesin host. 
{{% /alert %}}

Aspose.Slides untuk SSRS mendukung laporan berpaginasi (cdl).
