## **Installation**
Anda dapat menginstal Aspose.Slides untuk Reporting Services melalui installer MSI. 

{{% alert title="Catatan" color="warning" %}} 

**Aspose.Slides untuk Reporting Services** memerlukan instalasi **.NET Framework 3.5** pada mesin host. 

{{% /alert %}}

Jalankan ***Aspose.Slides.ReportingServices.msi*** dan ikuti langkah-langkah yang diberikan oleh installer. 

Installer akan menyalin assembly dan file lainnya ke direktori yang ditentukan serta menginstal produk pada instance Reporting Services default. Anda tidak perlu menyalin atau memodifikasi file secara manual kecuali ingin menambahkan parameter konfigurasi khusus. 

Instalasi menggunakan installer MSI adalah pilihan terbaik dalam kebanyakan kasus. Namun, Anda mungkin ingin menginstal produk secara manual dalam beberapa situasi: 

- Instalasi otomatis gagal karena masalah keamanan atau alasan lainnya. 
- produk harus diinstal pada instance Reporting Services yang bernama (bukan default) atau pada banyak instance. 
- setelah memperbarui ke versi terbaru, Anda hanya ingin mengganti assembly alih-alih mencopot versi lama dan menginstal yang baru menggunakan installer MSI. **Catatan** bahwa Anda mungkin akan memiliki file lain dalam kasus ini.
