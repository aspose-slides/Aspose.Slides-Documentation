{{% alert color="primary" %}} 

Artikel ini berfokus pada instalasi Aspose.Slides for Reporting Services di server.

{{% /alert %}} 
### **Opsi Instalasi**
Aspose.Slides for Reporting Services dapat diterapkan dengan dua cara: 

* secara otomatis dengan menggunakan installer MSI
* secara manual dengan menyalin assembly dan memodifikasi file konfigurasi. 

Installer MSI untuk instalasi otomatis (Aspose.Slides for Reporting Services XX.XX) dan paket zip yang berisi file untuk instalasi manual (Aspose.Slides for Reporting Services XX.XX DLL Only) dapat diunduh dari [halaman rilis produk](https://releases.aspose.com/slides/id/reportingservices/). 

{{% alert color="primary" %}} 

Aspose.Slides for Reporting Services dapat diinstal pada server Power BI.
