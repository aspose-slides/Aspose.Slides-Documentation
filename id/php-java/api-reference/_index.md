{{% alert color="primary" %}} 

Aspose.Slides for PHP via Java adalah perpustakaan kelas yang memungkinkan aplikasi PHP untuk membaca dan menulis dokumen presentasi tanpa menggunakan Microsoft PowerPoint® atau lainnya. Aspose.Slides for PHP via Java memungkinkan mengelola teks, bentuk, tabel & animasi, menambahkan audio dan video ke slide, menampilkan pratinjau slide, mengekspor slide ke format SVG, PDF, dan lainnya.

Referensi API terbaru dapat ditemukan di [Aspose.Slides for PHP via Java API Reference](https://reference.aspose.com/slides/id/php-java/).
