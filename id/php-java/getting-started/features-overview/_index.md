## **Platform yang Didukung**
Aspose.Slides for PHP via Java adalah API yang independen platform. Ini dapat digunakan pada platform apa pun (Windows, Linux, MacOS, dll.) dimana PHP 7 atau lebih tinggi telah terinstal. Oracle JDK 7 atau lebih tinggi adalah persyaratan lain.

## **Format File dan Konversi**
Aspose.Slides for PHP via Java mendukung sebagian besar format dokumen PowerPoint. Ini juga memungkinkan Anda mengekspor mereka ke format file populer yang digunakan dan dibagikan oleh organisasi. Lihat detail berikut:

|**Fitur**|**Deskripsi**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for PHP via Java menyediakan pemrosesan tercepat untuk format dokumen presentasi ini.|
|Konversi PPT ke PPTX|Aspose.Slides for PHP via Java mendukung konversi PPT ke PPTX.|
|Portable Document Format (PDF)|Anda dapat mengekspor semua format file yang didukung ke dokumen Adobe Portable Document Format (PDF) dengan satu metode.|
|XML Parser Specification (XPS)|Anda dapat mengekspor semua format file yang didukung ke dokumen XML Parser Specification (XPS) dengan satu metode.|
|Tagged Image File Format (TIFF)|Anda dapat mengekspor semua format file presentasi yang didukung ke Tagged Image File Format (TIFF).|
|Konversi PPTX ke HTML|Aspose.Slides for PHP via Java mendukung konversi PresentationEx ke format HTML.|

## **FAQ**

**Apakah saya perlu menginstal Microsoft PowerPoint di server/PC agar pustaka ini berfungsi?**

Tidak. PowerPoint tidak diperlukan; Aspose.Slides adalah engine mandiri untuk membuat, mengedit, mengonversi, dan merender presentasi.

**Bagaimana cara kerja multithreading? Apakah pemrosesan dapat diparalelkan?**

Aman untuk memproses dokumen yang berbeda dalam thread yang berbeda; objek [presentasi](https://reference.aspose.com/slides/id/php-java/aspose.slides/presentation/) yang sama tidak boleh digunakan oleh [beberapa thread](/slides/id/php-java/multithreading/) pada waktu yang bersamaan.

**Apakah kata sandi file dan enkripsi didukung?**

Ya. [Anda dapat](/slides/id/php-java/password-protected-presentation/) membuka presentasi terenkripsi, mengatur atau menghapus kata sandi buka dan tulis, serta memeriksa status perlindungan.

**Apakah saya perlu memperhatikan paket font di container Linux?**

Ya. Disarankan untuk menginstal paket font umum dan/atau secara eksplisit [menentukan direktori font](/slides/id/php-java/custom-font/) dalam aplikasi Anda untuk menghindari substitusi yang tidak terduga.

**Apakah ada keterbatasan pada versi evaluasi?**

Dalam [mode evaluasi](/slides/id/php-java/licensing/), watermark ditambahkan pada output dan beberapa keterbatasan berlaku; [lisensi sementara 30 hari](https://purchase.aspose.com/temporary-license/) tersedia untuk pengujian semua fitur.

**Apakah mengimpor format eksternal ke dalam presentasi (PDF/HTML → PPTX) didukung?**

Ya. Anda dapat menambahkan [halaman PDF dan konten HTML](/slides/id/php-java/import-presentation/) ke sebuah presentasi, mengubahnya menjadi slide.
