## **Pendahuluan**

Aspose.Slides untuk PHP via Java tidak memerlukan produk pihak ketiga apa pun seperti Microsoft PowerPoint yang harus diinstal. Aspose.Slides sendiri adalah mesin untuk membuat, memodifikasi, mengonversi, dan merender dokumen dalam berbagai format, termasuk format presentasi Microsoft PowerPoint.

## **Sistem Operasi yang Didukung**

## **FAQ**

**Apakah saya perlu menginstal Microsoft PowerPoint untuk konversi dan render?**

Tidak, PowerPoint tidak diperlukan; Aspose.Slides adalah mesin mandiri untuk [membuat](/slides/id/php-java/create-presentation/), memodifikasi, [mengonversi](/slides/id/php-java/convert-presentation/), dan [merender](/slides/id/php-java/convert-powerpoint-to-png/) presentasi.

**Font apa yang dibutuhkan untuk rendering yang benar?**

Secara praktik, font yang digunakan dalam presentasi atau [pengganti](/slides/id/php-java/font-substitution/) yang tepat harus tersedia. Untuk memastikan rendering konsisten pada Linux/macOS, disarankan menginstal paket font umum.

**Mengapa font khusus dirender sebagai fallback atau teks yang hilang di Linux?**

Jika file font memiliki entri tabel nama yang tidak konsisten atau rusak, stack pencocokan font Linux (FreeType/fontconfig) dapat memilih rekaman yang tidak valid, sehingga font tidak teratasi. Menggunakan versi font dengan rekaman tabel nama yang diperbaiki atau menginstal pengganti yang konsisten menyelesaikan masalah.
