## **Gambaran Umum**

Artikel ini menjelaskan cara mendapatkan dukungan untuk Aspose.Slides dan cara melaporkan masalah secara efektif. Artikel ini menggambarkan saluran dukungan utama, memberikan rekomendasi untuk mengirimkan masalah, serta menunjuk ke sumber daya teknis yang berguna seperti halaman produk, blog, unduhan, forum dukungan, dokumentasi, dan Referensi API. Artikel ini juga menjawab pertanyaan umum tentang dukungan berbayar, eskalasi masalah, dan dukungan untuk versi perpustakaan yang lebih lama.

## **Dukungan Teknis**

{{% alert color="primary" %}} 

Aspose menyediakan dukungan teknis gratis tak terbatas untuk semua produknya. Dukungan tersedia untuk semua pengguna, termasuk yang sedang mengevaluasi.

{{% /alert %}} 

Jika Anda memerlukan bantuan dengan Aspose.Slides for PHP via Java, pertimbangkan hal berikut:

- Saluran utama untuk dukungan adalah Aspose.Forums. Kirim pertanyaan Anda di forum Aspose.Slides dan akan dijawab dalam beberapa jam. Pertanyaan dijawab langsung oleh tim dukungan Aspose.Slides.
- Harap dicatat, Aspose tidak menyediakan dukungan teknis melalui telepon. Dukungan telepon hanya tersedia untuk pertanyaan penjualan dan pembelian.
- Saat mengharapkan balasan di forum, harap pertimbangkan perbedaan zona waktu.

Jika Anda memiliki masalah dengan Aspose.Slides for PHP via Java, ikuti langkah sederhana berikut untuk memastikan masalah terselesaikan secara efisien:

- Pastikan Anda menggunakan versi terbaru Aspose.Slides for PHP via Java sebelum melaporkan masalah, lihat Unduhan Aspose.Slides untuk mengetahui versi terbaru.
- Lihat forum, dokumentasi ini, dan Referensi API sebelum melaporkan masalah; mungkin pertanyaan Anda sudah terjawab.
- Saat melaporkan masalah, sertakan dokumen asli dan mungkin cuplikan kode yang menyebabkan masalah. Jika perlu melampirkan beberapa file, zip menjadi satu file. Aman melampirkan dokumen Anda di Aspose.Forums karena hanya Anda dan pengembang Aspose yang dapat mengakses file yang dilampirkan.
- Cobalah melaporkan satu masalah per thread. Jika Anda memiliki masalah lain, laporkan di thread terpisah.

## **FAQ**

**Tingkat dukungan berbayar apa yang tersedia dan bagaimana perbedaannya?**

[Pengembang, Bisnis, dan Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Perbedaannya terletak pada jumlah insiden yang termasuk, prioritas respons, dan saluran komunikasi yang tersedia.

**Apakah saya dapat mengeskalasikan bug langsung ke tim rekayasa?**

Ya. [Dukungan berbayar](https://helpdesk.aspose.com/) memungkinkan eskalasi ke pengembang ketika suatu masalah memerlukan penyelidikan lebih mendalam.

**Apakah versi perpustakaan yang lebih lama didukung, dan apakah Anda merilis perbaikan untuknya?**

Anda dapat menggunakan versi lama, tetapi perbaikan hanya disediakan untuk rilis saat ini; patch untuk versi legacy tidak dikeluarkan.
