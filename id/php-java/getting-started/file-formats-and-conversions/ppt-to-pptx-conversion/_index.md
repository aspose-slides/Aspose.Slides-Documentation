## **Fitur yang Didukung dalam Konversi**
Aspose.Slides untuk PHP via Java menyediakan dukungan parsial untuk mengonversi format file PPT ke PPTX. Dukungan untuk konversi baru saja diperkenalkan ke Aspose.Slides untuk PHP via Java, sehingga memiliki beberapa keterbatasan dan paling cocok untuk presentasi sederhana. Keuntungan utama yang diberikan Aspose.Slides untuk PHP via Java saat mengonversi PPT ke PPTX adalah betapa mudahnya API yang digunakan. Untuk melihat contoh kode, baca tentang [Mengonversi PPT ke PPTX](). Di bawah ini, daftar menjelaskan fitur mana yang didukung dan mana yang tidak untuk konversi PPT ke PPTX.

## **Fitur yang Didukung**
Fitur-fitur berikut didukung untuk konversi:

- Mengonversi struktur master, tata letak, dan slide.
- Mengonversi diagram.
- Bentuk grup.
- Mengonversi autoshape termasuk persegi panjang dan elips. 
- Bentuk dengan geometri khusus.
- Tekstur dan gaya isi gambar untuk autoshape.
- Mengonversi placeholder.
- Mengonversi garis dan polilin.
- Format garis dan isi.
- Gaya isi gradien.
- Bingkai OLE, tabel, bingkai video dan audio, dll.
- Properti animasi dan slideshow.
- Mengonversi teks dalam bingkai teks dan penampung teks.
