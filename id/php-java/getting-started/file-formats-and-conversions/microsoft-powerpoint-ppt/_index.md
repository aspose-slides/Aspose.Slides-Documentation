{{% alert color="primary" %}} 

PPT adalah format berkas dokumen presentasi yang dapat dibuat, dibaca, dimanipulasi, dan ditulis oleh berbagai versi Microsoft PowerPoint. Ini adalah format biner untuk dokumen presentasi yang dikembangkan oleh Microsoft.

{{% /alert %}} 

## **PPT di Aspose.Slides for PHP via Java**
Aspose.Slides for PHP via Java dapat membaca file PPT yang dibuat oleh perangkat lunak yang tercantum di bawah ini.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Demikian pula, file PPT yang dibuat oleh Aspose.Slides for PHP via Java dapat dibaca oleh perangkat lunak di atas.

## **Dukungan Komprehensif untuk PPT**
Aspose.Slides for PHP via Java menyediakan dukungan untuk hampir semua fitur yang didukung oleh format berkas dokumen PPT. Tidak hanya mencakup fitur dasar dan lanjutan yang disediakan oleh berbagai versi Microsoft PowerPoint untuk manipulasi dokumen PPT, tetapi juga fitur yang bahkan tidak didukung oleh Microsoft PowerPoint. Keuntungan utama menggunakan pustaka API Aspose.Slides for PHP via Java adalah kemudahan penggunaan dalam menangani fitur-fitur tersebut.

- Ekspor slide dalam dokumen PPT ke format SVG.
- Render slide ke format gambar apa pun yang didukung oleh Java Framework.
- Atur ukuran slide dalam dokumen PPT.
- Kelola animasi pada bentuk.
- Kelola pertunjukan slide.
- [Format teks pada slide](/slides/id/php-java/text-formatting/).
- Ekstrak teks dari dokumen PPT.
- [Tangani tabel pada slide](/slides/id/php-java/powerpoint-table/).
- Secara otomatis menyalin master menggunakan [fitur kloning](/slides/id/php-java/clone-slides/).
