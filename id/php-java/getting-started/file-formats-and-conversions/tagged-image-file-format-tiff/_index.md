## **Tentang TIFF**

The [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) yang dikenal karena menyimpan banyak gambar dalam satu dokumen awalnya dibuat oleh Aldus. Format ini didukung secara luas oleh aplikasi pemindaian, faks, dan manipulasi gambar lainnya.

## **TIFF di Aspose.Slides untuk PHP via Java**

Setiap dokumen yang dapat dimuat di Aspose.Slides untuk Java juga dapat dikonversi langsung menjadi dokumen TIFF oleh Aspose.Slides untuk PHP via Java, menghilangkan kebutuhan komponen pihak ketiga. Selanjutnya, Anda dapat secara opsional menentukan ukuran gambar dalam dokumen TIFF yang dihasilkan. Anda dapat menemukan informasi tentang mengekspor dokumen presentasi ke dokumen TIFF melalui Aspose.Slides untuk PHP via Java di [topik ini](/slides/id/php-java/convert-powerpoint-to-tiff/).
