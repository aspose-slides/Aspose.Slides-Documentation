{{% alert color="primary" %}} 

Portable Document Format (PDF) adalah format berkas yang dibuat oleh Adobe Systems untuk pertukaran dokumen antar organisasi. Tujuan format ini adalah untuk menjaga konten dan tata letak tetap sama, terlepas dari platform yang digunakan untuk melihatnya. Aspose.Slides for PHP via Java memungkinkan Anda mengonversi file presentasi ke PDF.

{{% /alert %}} 

## **PDF di Aspose.Slides for PHP via Java**
Setiap presentasi yang dapat dimuat ke Aspose.Slides for PHP via Java dapat dikonversi menjadi PDF yang sesuai dengan [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) atau [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA) tergantung pilihan Anda. Aspose.Slides for PHP via Java mengekspor presentasi ke PDF dan dalam kebanyakan kasus, PDF hasilnya terlihat persis seperti presentasi asli.

Aspose.Slides mendukung fitur presentasi berikut saat mengonversi ke PDF:

- Gambar, kotak teks, dan bentuk lainnya.
- Teks dan pemformatan.
- Paragraf dan pemformatan.
- Tautan hiper.
- Header dan footer.
- Bullet.
- Tabel.

Anda dapat mengekspor presentasi ke PDF secara langsung menggunakan Aspose.Slides for PHP via Java: Anda tidak memerlukan komponen lain. Selanjutnya, Anda dapat menyesuaikan ekspor presentasi ke PDF dengan berbagai opsi seperti dijelaskan dalam [Mengonversi ke PDF](/slides/id/php-java/convert-powerpoint-to-pdf/).
