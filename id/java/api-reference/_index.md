{{% alert color="primary" %}} 

Aspose.Slides for Java adalah pustaka kelas yang memungkinkan aplikasi Java untuk membaca dan menulis dokumen presentasi tanpa menggunakan Microsoft PowerPoint® atau yang lainnya. Aspose.Slides for Java memungkinkan mengelola teks, bentuk, tabel & animasi, menambahkan audio dan video ke slide, meninjau slide, mengekspor slide ke format SVG, PDF, dan lain-lain.

Referensi API terbaru dapat ditemukan di [Aspose.Slides for Java API Reference](https://reference.aspose.com/slides/id/java).
