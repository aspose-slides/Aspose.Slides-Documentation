{{% alert color="primary" %}} 

Aspose.Slides for Java 14.4.0 menyediakan keputusan baru untuk pemrosesan dokumen PowerPoint. Ada beberapa pembatasan dan masalah yang diketahui, yang akan dihapus pada rilis mendatang:

- Beberapa bentuk memiliki geometri yang salah dalam dokumen PPT yang diserialisasi (busur, panah melingkar, callouts).
- Tidak semua fitur pemformatan teks PPTX didukung dalam serialisasi PPT (tabulasi, indentasi, dan keterbatasan pemformatan paragraf).
- Info tentang bahasa teks dan pengaturan ejaan tidak ada dalam dokumen PPT yang diserialisasi.
- Tidak semua fitur tema PPTX didukung dalam serialisasi PPT (hanya serialisasi format isi, format garis, dan font).
- Ada masalah yang diketahui dalam serialisasi OLE/ActiveX PPT ke PPT.
- Serialisasi dan rendering WordArt tidak didukung.

{{% /alert %}}
