Aspose.Slides for Java 14.3.0 (14.4.0) menyediakan implementasi baru sepenuhnya untuk pemrosesan PPT. Ada banyak perbaikan, konversi parsial PPTX ke PPT. Namun ada beberapa fitur yang belum diimplementasikan:

- Beberapa bentuk memiliki geometri yang salah dalam dokumen PPT yang diserialisasi (Call outs)
- Tidak semua fitur pemformatan teks PPTX didukung dalam serialisasi PPT
- Informasi tentang bahasa teks dan pengaturan ejaan tidak ada dalam dokumen PPT yang diserialisasi
- Tidak semua fitur tema PPTX didukung dalam serialisasi PPT

**Ada beberapa perbedaan dibandingkan dengan Aspose.Slides for Java 8.6.0:**

- Ada masalah yang diketahui dalam serialisasi OLE/ActiveX PPT ke PPT

**Ada beberapa perbedaan dibandingkan dengan Aspose.Slides for .NET 14.3.0:**

- Dukungan pencetakan presentasi saat ini tidak tersedia di Aspose.Slides for Java
