## **Support for Legacy Code**
Untuk menggunakan kode warisan yang dikembangkan dengan versi Aspose.Slides for Java sebelum 13.x, Anda perlu melakukan beberapa perubahan kecil pada kode Anda agar kode tersebut tetap berfungsi seperti sebelumnya. Semua kelas yang ada di Aspose.Slides for Java lama di bawah namespace Aspose.Slide dan Aspose.Slides.Pptx kini digabungkan menjadi satu namespace Aspose.Slides. Silakan lihat contoh kode sederhana berikut untuk menambahkan header footer dalam presentasi menggunakan API Aspose.Slides legacy dan ikuti langkah‑langkah yang menjelaskan cara migrasi ke API yang telah digabung.
