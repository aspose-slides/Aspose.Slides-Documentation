## **Dukungan untuk Kode Warisan**
Untuk menggunakan kode warisan yang dikembangkan dengan Aspose.Slides for Java versi sebelum 14.x.x, Anda perlu membuat beberapa perubahan kecil pada kode Anda dan kode tersebut akan berfungsi seperti sebelumnya. Semua kelas yang ada di Aspose.Slides for Java lama di bawah namespace com.aspose.slides dan com.aspose.slides.pptx kini digabung menjadi satu namespace com.aspose.slides. Silakan lihat potongan kode sederhana berikut untuk membuat diagram normal dari awal dalam presentasi menggunakan API Aspose.Slides warisan dan ikuti langkah-langkah yang menjelaskan cara bermigrasi ke API baru yang digabung.
