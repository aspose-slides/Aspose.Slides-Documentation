## **Gambaran Umum**

Aspose.Slides mendukung file presentasi dari Microsoft PowerPoint 97 hingga Office 365, termasuk Microsoft PowerPoint untuk Mac. Artikel ini mencantumkan versi PowerPoint yang didukung oleh perpustakaan dan menyediakan tabel format file yang dapat dimuat, disimpan, atau keduanya.

Artikel ini juga menjawab pertanyaan umum tentang kepatuhan PDF, penyematan font, file yang dilindungi sandi, font khusus, fallback font, dan opsi ekspor XPS.

## **FAQ**

**Apakah saya dapat menyimpan presentasi ke PDF yang memenuhi standar arsip dan aksesibilitas (PDF/A dan PDF/UA)?**

Ya. Aspose.Slides mendukung ekspor ke PDF dengan tingkat kepatuhan seperti PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b, serta PDF/UA melalui pengaturan [kepatuhan](https://reference.aspose.com/slides/id/java/com.aspose.slides/pdfoptions/#setCompliance-int-) dalam [opsi ekspor PDF](https://reference.aspose.com/slides/id/java/com.aspose.slides/pdfoptions/).

**Apakah perpustakaan mendukung penyematan font saat mengekspor ke PDF, dengan kontrol detail atas apa yang disematkan?**

Ya. Anda dapat mengontrol apakah font disematkan secara penuh atau subset (hanya glif yang digunakan), menentukan perlakuan terhadap font sistem umum, dan mengonfigurasi perilaku untuk teks ASCII melalui [opsi ekspor PDF](https://reference.aspose.com/slides/id/java/com.aspose.slides/pdfoptions/).

**Bisakah saya mendeteksi apakah sebuah file dilindungi sandi sebelum memuatnya?**

Ya. Dengan menggunakan [API inspeksi berbasis pabrik](https://reference.aspose.com/slides/id/java/com.aspose.slides/presentationfactory/), Anda dapat memeriksa file presentasi untuk menentukan apakah file tersebut dilindungi sandi tanpa harus membuka secara penuh.

**Apakah ada mekanisme fallback font dan dukungan untuk font khusus?**

Ya. Perpustakaan mendukung [memuat](/slides/id/java/custom-font/) dan [menyematkan](/slides/id/java/embedded-font/) font khusus serta menyediakan [aturan fallback font](/slides/id/java/fallback-font/) untuk mencegah hilangnya glif selama proses rendering dan konversi.

**Bisakah saya mengekspor slide ke XPS, dan apakah ada opsi untuk menyesuaikan output XPS?**

Ya. [Ekspor ke XPS](/slides/id/java/convert-powerpoint-to-xps/) didukung, dan Anda dapat menyesuaikan [opsi penyimpanan](https://reference.aspose.com/slides/id/java/com.aspose.slides/xpsoptions/) yang relevan untuk mengontrol kualitas output serta konten dokumen XPS.
