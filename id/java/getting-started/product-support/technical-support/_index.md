## **Gambaran Umum**

Artikel ini menjelaskan cara mendapatkan dukungan untuk Aspose.Slides dan cara melaporkan masalah secara efektif. Artikel ini menjelaskan saluran dukungan utama, memberikan rekomendasi untuk mengirimkan masalah, dan menujuk ke sumber daya teknis yang berguna seperti halaman produk, blog, unduhan, forum dukungan, dokumentasi, dan Referensi API. Artikel ini juga menjawab pertanyaan umum tentang dukungan berbayar, eskalasi masalah, dan dukungan untuk versi perpustakaan yang lebih lama.

## **Dukungan Teknis**

{{% alert color="primary" %}} 

Aspose menyediakan dukungan teknis tak terbatas secara gratis untuk semua produknya. Dukungan tersedia untuk semua pengguna, termasuk evaluasi.

{{% /alert %}} 

Jika Anda membutuhkan bantuan dengan Aspose.Slides for Java, pertimbangkan hal berikut:

- Saluran utama untuk dukungan adalah Aspose.Forums. Kirim pertanyaan Anda di forum Aspose.Slides dan akan dijawab dalam beberapa jam. Pertanyaan dijawab langsung oleh tim dukungan Aspose.Slides.
- Harap dicatat, Aspose tidak menyediakan dukungan teknis melalui telepon. Dukungan telepon hanya tersedia untuk pertanyaan penjualan dan pembelian.
- Saat menunggu balasan di forum, harap perhatikan perbedaan zona waktu.

Jika Anda mengalami masalah dengan Aspose.Slides for Java, ikuti langkah sederhana berikut untuk memastikan masalah terselesaikan secara paling efisien:

- Pastikan Anda menggunakan versi terbaru Aspose.Slides for Java sebelum melaporkan masalah, lihat Unduhan Aspose.Slides untuk mengetahui versi terbaru.
- Lihatlah forum, dokumentasi ini, dan Referensi API sebelum melaporkan masalah; mungkin pertanyaan Anda sudah terjawab.
- Saat melaporkan masalah, sertakan dokumen asli dan kemungkinan potongan kode yang menyebabkan masalah. Jika Anda perlu melampirkan beberapa file, zip menjadi satu. Aman untuk melampirkan dokumen Anda di Aspose.Forums karena hanya Anda dan pengembang Aspose yang dapat mengakses file terlampir.
- Harap coba melaporkan satu masalah per thread. Jika Anda memiliki masalah lain, laporkan di thread terpisah.

## **FAQ**

**Tier dukungan berbayar apa yang tersedia dan bagaimana perbedaannya?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Mereka berbeda berdasarkan jumlah insiden yang termasuk, prioritas respons, dan saluran komunikasi yang tersedia.

**Apakah saya dapat mengeskalasikan bug langsung ke tim engineering?**

Ya. [Paid support](https://helpdesk.aspose.com/) memungkinkan eskalasi ke pengembang ketika masalah memerlukan penyelidikan lebih dalam.

**Apakah versi perpustakaan yang lebih lama didukung, dan apakah Anda merilis perbaikan untuknya?**

Anda dapat menggunakan versi lama, tetapi perbaikan hanya disediakan untuk rilis saat ini; patch untuk versi lama tidak diterbitkan.
