## **Platform yang Didukung**
Aspose.Slides for Java mendukung platform pengembangan dan penyebaran yang paling populer.

|**Fitur**|**Deskripsi**|
| :- | :- |
|Aplikasi Desktop|Aspose.Slides for Java dapat digunakan untuk mengembangkan aplikasi Windows Forms|
|Aplikasi Web Perusahaan|Menggunakan Aspose.Slides for Java membantu membangun aplikasi Web. Dukungan untuk menggunakan Aspose.Slides for Java dengan PHP juga disediakan.|
|Linux/Unix|Aspose.Slides for Java adalah API yang independen platform dan dapat bekerja di lingkungan Linux dan Unix.|

|[XML Parser Specification (XPS)](/slides/id/java/xml-parser-specification-xps/)|Semua format file yang didukung dapat diekspor ke dokumen XML Parser Specification (XPS) dengan satu metode.|
|[Tagged Image File Format (TIFF)](https://docs.aspose.com/slides/id/java/convert-powerpoint-to-tiff/)|Format file presentasi yang didukung oleh Aspose.Slides for Java juga dapat diekspor ke Tagged Image File Format (TIFF).|
|[ODP to PPTX Conversion](https://docs.aspose.com/slides/id/java/convert-odp-to-pptx/)|Aspose.Slides for Java mendukung memuat OpenDocument Presentation (ODP) dan mengkonversinya ke PPTX.|

## **FAQ**

**Apakah saya perlu menginstal Microsoft PowerPoint di server/PC agar perpustakaan ini berfungsi?**

Tidak. PowerPoint tidak diperlukan; Aspose.Slides adalah engine mandiri untuk membuat, mengedit, mengonversi, dan merender presentasi.

**Bagaimana cara kerja multithreading? Apakah pemrosesan dapat diparalelkan?**

Aman untuk memproses dokumen yang berbeda dalam thread yang berbeda; objek [presentasi](https://reference.aspose.com/slides/id/java/com.aspose.slides/presentation/) yang sama tidak boleh digunakan oleh [beberapa thread](/slides/id/java/multithreading/) secara bersamaan.

**Apakah kata sandi file dan enkripsi didukung?**

Ya. [Anda dapat](/slides/id/java/password-protected-presentation/) membuka presentasi terenkripsi, menetapkan atau menghapus kata sandi buka dan tulis, serta memeriksa status perlindungan.

**Apakah saya perlu memperhatikan paket font di kontainer Linux?**

Ya. Disarankan untuk menginstal paket font umum dan/atau secara eksplisit [menentukan direktori font](/slides/id/java/custom-font/) dalam aplikasi Anda untuk menghindari substitusi yang tidak terduga.

**Apakah ada keterbatasan pada versi evaluasi?**

Dalam [mode evaluasi](/slides/id/java/licensing/), watermark ditambahkan ke output dan beberapa keterbatasan berlaku; [lisensi sementara 30 hari](https://purchase.aspose.com/temporary-license/) tersedia untuk pengujian fitur lengkap.

**Apakah mengimpor format eksternal ke dalam presentasi (PDF/HTML → PPTX) didukung?**

Ya. Anda dapat menambahkan [halaman PDF dan konten HTML](/slides/id/java/import-presentation/) ke sebuah presentasi, mengubahnya menjadi slide.
