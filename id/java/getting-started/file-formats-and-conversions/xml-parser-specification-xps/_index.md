{{% alert color="primary" %}} 
Spesifikasi [XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) adalah bahasa deskripsi halaman dan format dokumen tetap yang awalnya dikembangkan oleh Microsoft. Seperti PDF, XPS dirancang untuk mempertahankan kesetiaan dokumen dan menyediakan tampilan dokumen yang independen dari perangkat. 
{{% /alert %}} 
## **XPS di Aspose.Slides untuk Java**
Setiap dokumen presentasi yang dapat dimuat oleh Aspose.Slides untuk Java dapat dikonversi ke format XPS. Aspose.Slides untuk Java menggunakan mesin tata letak halaman dan rendering dengan kesetiaan tinggi untuk menghasilkan output dalam format dokumen XPS berlayout tetap.
Anda dapat mempelajari cara mengekspor dokumen presentasi ke dokumen XPS melalui Aspose.Slides untuk Java di [Converting to XPS](https://docs.aspose.com/slides/id/java/convert-powerpoint-to-xps/).
