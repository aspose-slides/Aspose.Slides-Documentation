{{% alert color="primary" %}} 

[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) adalah format file dokumen presentasi yang dapat dibuat, dibaca, dimanipulasi, dan ditulis oleh berbagai versi Microsoft PowerPoint. Ini adalah format biner untuk dokumen presentasi yang dikembangkan oleh Microsoft.

{{% /alert %}} 

## **PPT dalam Aspose.Slides for Java**
Aspose.Slides for Java dapat membaca file PPT yang dibuat oleh perangkat lunak yang tercantum di bawah ini.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Demikian pula, file PPT yang dibuat oleh Aspose.Slides for Java dapat dibaca oleh perangkat lunak di atas.

## **Dukungan Komprehensif untuk PPT**
Aspose.Slides for Java menyediakan dukungan untuk hampir semua fitur yang didukung oleh format file dokumen PPT. Ini tidak hanya mencakup fitur dasar dan lanjutan yang disediakan oleh berbagai versi Microsoft PowerPoint untuk manipulasi dokumen PPT, tetapi juga fitur yang bahkan tidak didukung oleh Microsoft PowerPoint. Keuntungan utama menggunakan pustaka API Aspose.Slides for Java adalah kemudahan penggunaan dalam menangani fitur-fitur tersebut.

Selain tugas dasar terkait pembuatan, pembacaan, dan penulisan file dokumen PPT, terdapat beberapa fitur yang disediakan oleh Aspose.Slides for Java:

- Impor format file Microsoft Office lainnya sebagai [objek OLE ke dalam dokumen PPT]().
- [Ekspor dokumen PPT ke PDF](/slides/id/java/convert-powerpoint-ppt-and-pptx-to-pdf/).
- Ekspor slide dalam dokumen PPT ke format SVG.
- Render slide ke format gambar apa pun yang didukung oleh Java Framework.
- Atur ukuran slide dalam dokumen PPT.
- Kelola animasi pada bentuk.
- Kelola pertunjukan slide.
- [Format teks pada slide]().
- Ekstrak teks dari dokumen PPT.
- [Kelola tabel pada slide]().
- Secara otomatis menyalin master menggunakan [fitur cloning]().
