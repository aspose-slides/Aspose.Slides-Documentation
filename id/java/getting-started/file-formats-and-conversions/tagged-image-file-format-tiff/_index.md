## **Tentang TIFF**

[Format File Gambar Berlabel](https://en.wikipedia.org/wiki/TIFF) yang dikenal karena menyimpan banyak gambar dalam satu dokumen awalnya dibuat oleh Aldus. Format ini secara luas didukung oleh aplikasi pemindaian, faks, dan manipulasi gambar lainnya.

## **TIFF di Aspose.Slides for Java**

Setiap dokumen yang dapat dimuat di Aspose.Slide for Java juga dapat dikonversi langsung menjadi dokumen TIFF oleh Aspose.Slides for Java, menghilangkan kebutuhan akan komponen pihak ketiga. Selanjutnya, Anda dapat secara opsional menentukan ukuran gambar dalam dokumen TIFF yang dihasilkan. Anda dapat menemukan informasi tentang mengekspor dokumen presentasi ke dokumen TIFF melalui Aspose.Slides for Java pada [topik ini](/slides/id/java/converting-a-presentation/).
