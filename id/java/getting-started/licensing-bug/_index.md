## **Ikhtisar**

Artikel ini menjelaskan bug lisensi yang ditemukan pada Aspose.Slides for Java versi 23.2, 23.3, 23.4, 23.5, 23.6, dan 23.7. Bug ini menyebabkan lisensi seumur hidup menjadi tidak dapat digunakan sementara setelah langganannya kedaluwarsa.

## **Gejala**

Setelah langganan lisensi seumur hidup Anda kedaluwarsa, Aspose.Slides for Java versi 23.2 - 23.7 dapat melaporkan lisensi sebagai kedaluwarsa, mengalihkan semua fungsi ke mode evaluasi.
Perilaku ini tidak benar dan tidak memengaruhi versi sebelum 23.2 maupun setelah 23.7.

## **Solusi**

Aspose.Slides for Java telah menangani masalah ini dan merilis versi yang diperbarui (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) dengan perbaikan.

Jika Anda menggunakan salah satu versi yang terpengaruh dari Aspose.Slides for Java dalam proyek Anda, harap beralih ke versi yang telah diperbaiki.

Jika Anda mengalami masalah lisensi dengan Aspose.Slides for Java:

- Periksa versi yang Anda gunakan dan pastikan tidak terpengaruh oleh bug yang dijelaskan di atas.
- Rujuk ke [dokumentasi Aspose](https://docs.aspose.com/slides/id/java/getting-started/) untuk langkah pemecahan masalah dan masalah lisensi yang diketahui.
- Hubungi [dukungan Aspose](https://forum.aspose.com/) untuk bantuan lebih lanjut.

## **Produk dan versi yang terpengaruh**

Harap dicatat bahwa bug ini hanya memengaruhi Aspose.Slides for Java versi 23.2 hingga 23.7. **Produk Aspose lainnya dan versi lain dari Aspose.Slides tidak terpengaruh**.
