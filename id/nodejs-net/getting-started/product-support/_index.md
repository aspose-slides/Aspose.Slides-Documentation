## **Dukungan teknis**
{{% alert color="primary" %}}

Aspose menyediakan dukungan teknis gratis tak terbatas untuk semua produknya. Dukungan tersedia bagi semua pengguna (termasuk pengguna dengan paket evaluasi).

{{% /alert %}} 

Ketika Anda membutuhkan bantuan dengan Aspose.Slides for Node.js via .NET, pertimbangkan hal berikut:

- Jalur utama untuk dukungan adalah [Aspose.Forums](https://forum.aspose.com/c/slides/id/11). Kirim pertanyaan Anda di forum Aspose.Slides—dan akan dijawab dalam beberapa jam. Tim dukungan Aspose.Slides langsung menjawab pertanyaan yang diposting di forum.
- Harap dicatat bahwa Aspose tidak menyediakan dukungan teknis melalui telepon. Dukungan telepon hanya tersedia untuk pertanyaan penjualan dan pembelian.
- Saat menunggu balasan di forum, harap bersabar dan pertimbangkan perbedaan zona waktu.

Ketika Anda mengalami masalah dengan Aspose.Slides for Node.js via .NET, ikuti pedoman berikut untuk menyelesaikannya dengan cara paling efisien:

- Periksa dan pastikan bahwa Anda menggunakan versi terbaru Aspose.Slides for Node.js via .NET sebelum melaporkan masalah. Mungkin, masalah yang Anda alami sudah teratasi dalam pembaruan.

  Lihat [Aspose.Slides download](https://releases.aspose.com/slides/id/nodejs-net/) untuk mendapatkan informasi tentang versi terbaru.

- Sebelum melaporkan masalah, Anda dapat menelusuri forum kami, dokumentasi ini, dan Referensi API. Mungkin, jawaban atas pertanyaan Anda sudah tersedia.

- Saat melaporkan masalah, sertakan dokumen asli dan (jika memungkinkan) potongan kode Anda yang terkait dengan masalah tersebut. Jika Anda perlu melampirkan beberapa file, masukkan ke dalam paket ZIP. Anda dapat dengan aman melampirkan dokumen Anda di Aspose.Forums karena **hanya** Anda dan pengembang Aspose yang memiliki akses ke file yang dilampirkan.

- Buat dan gunakan satu thread untuk satu masalah spesifik. Jika Anda mengalami masalah lain, Anda harus membuat thread lain untuk melaporkannya.
