Aspose.Slides for Node.js via .NET adalah pustaka kelas yang memungkinkan aplikasi Node.js membaca dan menulis dokumen presentasi tanpa menggunakan Microsoft PowerPoint® atau lainnya. Aspose.Slides for Node.js via .NET memungkinkan Anda mengelola teks, bentuk, tabel & animasi, menambahkan audio dan video ke slide, meninjau slide, mengekspor slide ke format SVG, PDF, dan banyak lagi.

Aspose.Slides for Node.js via .NET menggunakan[sumber referensi API yang sama seperti Aspose.Slides .NET](https://reference.aspose.com/slides/id/nodejs-net/).
