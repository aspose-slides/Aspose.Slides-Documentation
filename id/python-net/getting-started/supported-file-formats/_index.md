## **Gambaran Umum**

Aspose.Slides mendukung file presentasi dari Microsoft PowerPoint 97 hingga Office 365, termasuk Microsoft PowerPoint untuk Mac. Artikel ini mencantumkan versi PowerPoint yang didukung oleh perpustakaan dan menyediakan tabel format file yang dapat dimuat, disimpan, atau keduanya.

Artikel ini juga menjawab pertanyaan umum tentang kepatuhan PDF, penyematan font, file yang dilindungi kata sandi, font khusus, fallback font, dan opsi ekspor XPS.

## **Tanya Jawab**

**Apakah saya dapat menyimpan presentasi ke PDF yang memenuhi standar pengarsipan dan aksesibilitas (PDF/A dan PDF/UA)?**

Ya. Aspose.Slides for Python via .NET mendukung ekspor ke PDF dengan tingkat kepatuhan seperti PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b, serta PDF/UA melalui pengaturan [compliance](https://reference.aspose.com/slides/id/python-net/aspose.slides.export/pdfoptions/compliance/) dalam [opsi ekspor PDF](https://reference.aspose.com/slides/id/python-net/aspose.slides.export/pdfoptions/).

**Apakah perpustakaan mendukung penyematan font saat mengekspor ke PDF, dengan kontrol detail atas apa yang disematkan?**

Ya. Anda dapat mengatur apakah font disematkan sepenuhnya atau hanya subset (karakter yang digunakan), menentukan cara penanganan font sistem umum, dan mengonfigurasi perilaku teks ASCII melalui [opsi ekspor PDF](https://reference.aspose.com/slides/id/python-net/aspose.slides.export/pdfoptions/).

**Bisakah saya mendeteksi apakah sebuah file dilindungi kata sandi sebelum memuatnya?**

Ya. Menggunakan [API inspeksi berbasis factory](https://reference.aspose.com/slides/id/python-net/aspose.slides/presentationfactory/), Anda dapat menanyakan sebuah file presentasi untuk menentukan apakah ia dilindungi kata sandi tanpa harus membukanya sepenuhnya.

**Apakah ada mekanisme font fallback dan dukungan untuk font khusus?**

Ya. Perpustakaan mendukung [memuat](/slides/id/python-net/custom-font/) dan [menyematkan](/slides/id/python-net/embedded-font/) font khusus serta menyediakan [aturan fallback font](/slides/id/python-net/fallback-font/) untuk mencegah hilangnya glyph selama rendering dan konversi.

**Apakah saya dapat mengekspor slide ke XPS, dan apakah ada opsi untuk menyesuaikan output XPS?**

Ya. [Ekspor ke XPS](/slides/id/python-net/convert-powerpoint-to-xps/) didukung, dan Anda dapat menyesuaikan [opsi penyimpanan](https://reference.aspose.com/slides/id/python-net/aspose.slides.export/xpsoptions/) yang relevan untuk mengontrol kualitas output serta isi dokumen XPS.
