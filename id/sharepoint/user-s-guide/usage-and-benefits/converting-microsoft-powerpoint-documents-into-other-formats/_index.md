## **Mengonversi Presentasi**
Untuk mengonversi dokumen Microsoft PowerPoint dari pustaka dokumen SharePoint: 

1. Pilih dokumen Microsoft PowerPoint di pustaka dokumen.
2. Klik panah ke bawah untuk menampilkan menu dan klik **Convert via Aspose.Slides.SharePoint**. 

   **Menu file Presentation 2 yang menampilkan opsi Convert via Aspose.Slides** 

![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_2.png)




3. Pilih format output yang diinginkan dari formulir. Jika ingin, ubah nama file output dan folder tujuan.
4. Klik **Convert** untuk mengonversi file. 

   **Formulir konversi memungkinkan Anda memilih format file output, nama, dan tujuan** 

![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_3.png)




5. Setelah konversi selesai, pesan keberhasilan ditampilkan. 

   **Konversi berhasil** 

![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_4.png)




6. Klik **Source Library** (untuk pergi ke direktori sumber) atau **Destination Library** (untuk pergi ke direktori tempat file disimpan). 

   Dokumen yang dikonversi muncul di pustaka dokumen. 

   **Dokumen yang dikonversi ditampilkan di pustaka tempat disimpan** 

![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_5.png)
