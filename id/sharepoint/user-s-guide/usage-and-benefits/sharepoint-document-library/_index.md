Perpustakaan adalah struktur konten yang berisi file. Perpustakaan dapat berisi hanya satu jenis file atau banyak jenis yang berbeda, termasuk dokumen, lembar kerja, dan presentasi. Perpustakaan SharePoint membuat satu lokasi di mana orang dapat mengunggah dan mengakses dokumen. Dokumen disimpan dalam format biner di basis data SQL Server yang digunakan oleh situs SharePoint. Dokumen tersedia bagi pengguna SharePoint lainnya sebagai daftar tautan hiper.
