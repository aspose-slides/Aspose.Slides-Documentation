## **Persyaratan Server**
{{% alert color="primary" %}} 

Untuk menginstal Aspose.Slides untuk SharePoint, diperlukan sistem yang menjalankan salah satu dari berikut:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft Office SharePoint Server 2010
- Microsoft Office SharePoint Server 2013
- Microsoft Office SharePoint Server 2016
- Microsoft Office SharePoint Server 2019

{{% /alert %}} {{% alert color="primary" %}} 

Menginstal Service Pack 2 atau yang lebih baru untuk produk-produk ini disarankan.

{{% /alert %}} 
## **Persyaratan Klien**
{{% alert color="primary" %}} 

Situs SharePoint diakses melalui peramban web. Aspose.Slides untuk SharePoint telah diuji dengan peramban berikut:

- Microsoft Internet Explorer 6, 7, 8
- Mozilla FireFox 3.5

{{% /alert %}}
