{{% alert color="primary" %}} 

Anda dapat dengan mudah mengunduh Aspose.Slides untuk evaluasi. Unduhan evaluasi sama dengan unduhan yang dibeli. Versi evaluasi cukup menjadi berlisensi ketika Anda menambahkan beberapa baris kode untuk menerapkan lisensi.

Versi evaluasi Aspose.Slides (tanpa lisensi yang ditentukan) menyediakan fungsionalitas produk penuh, namun menambahkan watermark evaluasi pada setiap slide file PDF, TIFF, dan XPS yang dihasilkan.
