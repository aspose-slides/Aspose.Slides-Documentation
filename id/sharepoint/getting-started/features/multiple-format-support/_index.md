{{% alert color="primary" %}} 

Dengan Aspose.Slides for SharePoint, Anda dapat mengonversi dokumen antara banyak format dokumen perkantoran populer dari dalam pustaka dokumen SharePoint. Konversi dilakukan dengan fidelitas tinggi dan presisi. 

{{% /alert %}} 
## **Format Input yang Didukung**
Aspose.Slides for SharePoint mendukung format input berikut: 

- PPT – Microsoft PowerPoint Presentation 97 - 2003
- PPS – Microsoft PowerPoint SlideShow 97 - 2003
- POT – Microsoft PowerPoint Template 97 - 2003
- PPTX – Office Open XML Presentation
- PPSX – Office Open XML SlideShow
- POTX – Office Open XML Template

{{% alert color="primary" %}} 

Untuk menghasilkan dokumen, Aspose.Slides for SharePoint bergantung pada versi bawaan [Aspose.Slides for .NET](http://www.aspose.com/categories/.net-components/aspose.slides-for-.net/default.aspx), satu-satunya komponen pemrosesan dokumen PowerPoint milik Aspose. 

{{% /alert %}}
## **Format Output yang Didukung**
Format output berikut didukung oleh Aspose.Slides for SharePoint: 

- PDF – Portable Document Format
- TIFF – Pictures Package
- XPS – XML Paper Specification
