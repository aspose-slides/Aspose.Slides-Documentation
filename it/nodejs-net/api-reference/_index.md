{{% alert color="primary" %}} 

Aspose.Slides for Node.js via .NET è una libreria di classi che consente alle applicazioni Node.js di leggere e scrivere documenti di presentazione senza utilizzare Microsoft PowerPoint® o altri. Aspose.Slides for Node.js via .NET ti permette di gestire testo, forme, tabelle e animazioni, aggiungere audio e video alle diapositive, visualizzare in anteprima le diapositive, esportare le diapositive in formato SVG, PDF, e molto altro.

Aspose.Slides for Node.js via .NET utilizza la [stessa referenza API di Aspose.Slides .NET](https://reference.aspose.com/slides/it/nodejs-net/).
