## **Piattaforme supportate**
Aspose.Slides per Node.js via .NET è un'API indipendente dalla piattaforma. Può essere utilizzata su qualsiasi piattaforma (Windows, Linux, macOS, ecc.) dove è installato Node.js 7 o versioni successive. Oracle JDK 8 o versioni successive è un ulteriore requisito.

## **Formati di file e conversioni**
Aspose.Slides per Node.js via .NET supporta la maggior parte dei formati di documento PowerPoint. Consente anche di esportarli nei formati di file più popolari che le organizzazioni utilizzano e condividono. Vedi i dettagli:

|**Feature**|**Description**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides per Node.js via .NET fornisce l'elaborazione più veloce per questo formato di documento di presentazione.|
|Conversione da PPT a PPTX|Aspose.Slides per Node.js via .NET supporta la conversione da PPT a PPTX|
|Portable Document Format (PDF)|È possibile esportare tutti i formati di file supportati in documenti Adobe Portable Document Format (PDF) con un unico metodo.|
|XML Parser Specification (XPS)|È possibile esportare tutti i formati di file supportati in documenti XML Parser Specification (XPS) con un unico metodo.|
|Tagged Image File Format (TIFF)|È possibile esportare tutti i formati di file di presentazione supportati in Tagged Image File Format (TIFF).|
|Conversione da PPTX a HTML|Aspose.Slides per Node.js via .NET supporta la conversione di PresentationEx in formato HTML.|

## **Rendering e stampa**
Aspose.Slides per Node.js via .NET supporta il rendering ad alta fedeltà delle diapositive di presentazione in diversi formati grafici. Vedi i dettagli:

|**Feature**|**Description**|
| :- | :- |
|Formati immagine supportati da .NET|Utilizzando Aspose.Slides per Node.js via .NET, è possibile renderizzare le diapositive di presentazione e le immagini sulle diapositive in tutti i formati grafici supportati da .NET, come TIFF, PNG, BMP, JPEG, GIF.|
|Formato SVG|Aspose.Slides per Node.js via .NET fornisce inoltre metodi integrati che consentono di esportare le diapositive di presentazione in formati Scalable Vector Graphics (SVG).|
|Stampa della presentazione|Le versioni più recenti di Aspose.Slides per Node.js via .NET forniscono metodi di stampa integrati con diverse opzioni.|

## **Funzionalità del contenuto**
Aspose.Slides per Node.js via .NET consente di accedere, modificare o creare quasi tutti gli elementi o i contenuti dei documenti di presentazione. Vedi i dettagli:

|**Feature**|**Description**|
| :- | :- |
|Diapositive master|Le diapositive master definiscono il layout delle diapositive normali. Aspose.Slides per Node.js via .NET consente di accedere e modificare le diapositive master dei documenti di presentazione|
|Diapositive normali|Con Aspose.Slides per Node.js via .NET, è possibile creare nuove diapositive di diversi tipi; è inoltre possibile accedere e modificare le diapositive esistenti nelle presentazioni|
|Clonazione / Copia di diapositive|Sono disponibili metodi integrati forniti da Aspose.Slides per Node.js via .NET che consentono di clonare o copiare le diapositive esistenti all'interno di una presentazione. È inoltre possibile utilizzare le diapositive copiate e clonate da una presentazione all'altra. Poiché una diapositiva eredita il suo layout dalla diapositiva master, i metodi di clonazione integrati copiano automaticamente il master durante la clonazione|
|Gestione delle sezioni delle diapositive|Metodi per organizzare le diapositive in diverse sezioni all'interno di una presentazione|
|Segnaposti e segnaposti di testo|È possibile accedere ai segnaposti e ai segnaposti di testo in una diapositiva. Inoltre, è possibile creare una diapositiva con segnaposti di testo da zero utilizzando il metodo appropriato|
|Intestazioni e piè di pagina|Aspose.Slides per Node.js via .NET agevola la gestione di intestazioni/piè di pagina nelle diapositive|
|Note nelle diapositive|Con Aspose.Slides per Node.js via .NET, è possibile accedere e modificare le note associate a una diapositiva e anche aggiungere nuove note|
|Ricerca di una forma|È inoltre possibile trovare una forma specifica in una diapositiva utilizzando il testo alternativo associato alla forma|
|Sfondi|Aspose.Slides per Node.js via .NET consente di lavorare con gli sfondi associati a una diapositiva master o normale in una presentazione|
|Caselle di testo|Le caselle di testo possono essere create da zero. È possibile accedere alle caselle di testo esistenti. È inoltre possibile modificare i loro testi senza perdere il formato originale del testo|
|Forme rettangolari|È possibile creare o modificare forme rettangolari con Aspose.Slides per Node.js via .NET|
|Forme polilinea|È possibile creare o modificare forme polilinea con Aspose.Slides per Node.js via .NET|
|Forme ellittiche|È possibile creare o modificare forme ellittiche con Aspose.Slides per Node.js via .NET|
|Forme raggruppate|Aspose.Slides per Node.js via .NET supporta le forme raggruppate|
|Forme automatiche|Aspose.Slides per Node.js via .NET supporta le forme automatiche|
|SmartArt|Aspose.Slides per Node.js via .NET fornisce supporto per le forme SmartArt in MS PowerPoint|
|Charts|Aspose.Slides per Node.js via .NET fornisce supporto per i grafici MSO in PowerPoint|
|Serializzazione delle forme|Aspose.Slides per Node.js via .NET supporta un gran numero di forme. Quando Aspose.Slides per Node.js via .NET non supporta una forma, è possibile utilizzare un metodo di serializzazione attraverso il quale è possibile serializzare quella forma da una diapositiva esistente. In questo modo, è possibile utilizzare ulteriormente la forma secondo le proprie esigenze|
|Cornici immagine|È possibile gestire le immagini nelle cornici immagine con Aspose.Slides per Node.js via .NET|
|Cornici audio|È possibile collegare o incorporare file audio nelle cornici audio sulle diapositive con Aspose.Slides per Node.js via .NET|
|Cornici video|È possibile gestire i file video nelle cornici video. Aspose.Slides per Node.js via .NET fornisce inoltre supporto per video collegati e incorporati|
|Cornice OLE|È possibile gestire oggetti OLE nelle cornici OLE con Aspose.Slides per Node.js via .NET|
|Tables|Aspose.Slides per Node.js via .NET supporta tabelle nelle diapositive|
|ActiveX Controls|Supporto per controlli ActiveX|
|VBA Macros|Supporto per la gestione di macro VBA nelle presentazioni.|
|Cornice di testo|È possibile accedere al testo di qualsiasi forma tramite la cornice di testo associata a quella forma|
|Scansione del testo|È possibile scansionare il testo in una presentazione a livello di presentazione o di diapositiva tramite metodi di scansione integrati.|
|Animations|È possibile applicare animazioni su forme|
|Slide Shows|Aspose.Slides per Node.js via .NET supporta presentazioni e transizioni di diapositiva|

## **Funzionalità di formattazione**
Con Aspose.Slides per Node.js via .NET, è possibile formattare testi e forme sulle diapositive delle presentazioni. Vedi i dettagli:

|**Feature**|**Description**|
| :- | :- |
|Text Formatting|<p>In Aspose.Slides per Node.js via .NET, è possibile gestire i testi tramite le cornici di testo associate alle forme. Pertanto, è possibile formattare i testi usando i paragrafi e le porzioni associate alle cornici di testo. Questi elementi di testo possono essere formattati tramite Aspose.Slides per Node.js via .NET.</p><p>- Tipo di carattere</p><p>- Dimensione del carattere</p><p>- Colore del carattere</p><p>- Sfumature del carattere</p><p>- Allineamento del paragrafo</p><p>- Elenco puntato del paragrafo</p><p>- Orientamento del paragrafo</p>|
|Shape Formatting|<p>In Aspose.Slides per Node.js via .NET, l'elemento base di una diapositiva è una forma. È possibile formattare questi elementi forma con Aspose.Slides per Node.js via .NET:</p><p>- Posizione</p><p>- Dimensione</p><p>- Linea</p><p>- Riempimento (inclusi modello, gradiente, solido)</p><p>- Testo</p><p>- Immagine</p>|
