## **Installazione**
È possibile installare Aspose.Slides for Reporting Services tramite un installer MSI. 

{{% alert title="Note" color="warning" %}} 

**Aspose.Slides for Reporting Services** richiede l'installazione di **.NET Framework 3.5** sulla macchina host. 

{{% /alert %}}

Esegui ***Aspose.Slides.ReportingServices.msi*** e segui i passaggi offerti dall'installer. 

L'installer copierà l'assembly e gli altri file nella directory specificata e installerà il prodotto sull'istanza predefinita di Reporting Services. Non è necessario copiare o modificare manualmente alcun file, a meno che non si desideri aggiungere parametri di configurazione speciali. 

L'installazione tramite l'installer MSI è l'opzione migliore nella maggior parte dei casi. Tuttavia, potresti voler installare il prodotto manualmente in alcune situazioni: 

- L'installazione automatica fallisce a causa di problemi di sicurezza o altri motivi. 
- Il prodotto deve essere installato su un'istanza denominata (non predefinita) di Reporting Services o su più istanze. 
- Dopo l'aggiornamento all'ultima versione, desideri semplicemente sostituire l'assembly invece di disinstallare la vecchia versione e installare quella nuova usando l'installer MSI. **Nota** che potresti ritrovare altri file in questo caso.
