## **Supporto licenza**

Puoi scaricare una versione di valutazione di **Aspose.Slides for Reporting Services** dalla sua [pagina dei Rilasci](https://releases.aspose.com/slides/it/reportingservices/). La versione di valutazione offre le stesse funzionalità della versione con licenza del prodotto. Il pacchetto di valutazione è identico a quello acquistato. La versione di valutazione diventa semplicemente con licenza dopo aver aggiunto alcune righe di codice (per applicare la licenza).

Una volta soddisfatto della valutazione del prodotto, puoi [acquistare una licenza](https://purchase.aspose.com/buy). Ti consigliamo di esaminare i diversi tipi di abbonamento. Se hai domande, contatta il team vendite di Aspose.

## **Licenza in Aspose.Slides for Reporting Service**

* Una versione di valutazione diventa con licenza dopo aver acquistato una licenza e aggiunto un paio di righe di codice (per applicare la licenza).
* La licenza è un file XML di testo semplice che contiene dettagli come il nome del prodotto, il numero di sviluppatori a cui è concessa la licenza, la data di scadenza dell'abbonamento e così via. 
* Il file di licenza è firmato digitalmente, quindi non devi modificarlo. Anche l'aggiunta involontaria di un'interruzione di riga extra al contenuto del file lo invaliderà.
* Per evitare le limitazioni associate alla versione di valutazione, è necessario impostare una licenza prima di utilizzare Aspose.Slides for Reporting Service. 

Scarica la licenza sul tuo computer e copiala nella cartella **C:\Program Files\Microsoft SQL Server\<Instance>\Reporting Services\ReportServer\bin**, dove è installato **Aspose.Slides.ReportingServices.dll**. 

Per confermare che la licenza sia stata installata correttamente, esporta qualsiasi report come una presentazione Microsoft PowerPoint. Se il documento non contiene una filigrana, la licenza è stata attivata con successo. 

![todo:image_alt_text](license-aspose-slides-for-reporting-services_1.png)

Quando un file Aspose.Slides.ReportingServices.lic valido è presente nella cartella *ReportServer\bin*, non viene visualizzata alcuna filigrana di valutazione. 

**Modalità di valutazione**

Aspose.Slides for Reporting Services inserisce una filigrana quando si lavora in modalità di valutazione (senza licenza)
