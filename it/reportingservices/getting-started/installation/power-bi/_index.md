## **Power BI**
È possibile installare Aspose.Slides for Reporting Services su un server Power BI tramite gli stessi processi di installazione:

Vedi 

* [Installa con MSI Installer](https://docs.aspose.com/slides/it/reportingservices/install-with-msi-installer/#installation). Il programma di installazione rileva automaticamente il Power BI installato sul server, installa il prodotto e quindi lo configura per il tuo ambiente di lavoro.
* [Installa manualmente](https://docs.aspose.com/slides/it/reportingservices/install-manually/).

{{% alert title="Note" color="warning" %}} 
**Aspose.Slides for Reporting Services** richiede l'installazione di **.NET Framework 3.5** sulla macchina host. 
{{% /alert %}}

Aspose.Slides for SSRS supporta i report paginati (cdl).
