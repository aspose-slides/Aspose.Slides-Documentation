## Tipi di licenza 
{{% alert color="primary" %}}Sono attualmente offerte due tipologie di licenza: Server Subscription Site Subscription{{% /alert %}}
Ogni licenza include 1 anno di aggiornamenti gratuiti a qualsiasi nuova versione del prodotto rilasciata e 1 anno di supporto tecnico illimitato gratuito. 

Le versioni di valutazione e a pagamento di Aspose.Slides for Reporting Services sono lo stesso download. Basta scaricare Aspose.Slides for Reporting Services e installarlo sul server; funzionerà in modalità di valutazione. 


Assicurati di sfruttare la valutazione di Aspose.Slides for Reporting Services poiché non ha un limite di tempo e viene fornito supporto tecnico gratuito per gli utenti in valutazione. 


La modalità di valutazione inserisce filigrane nei report esportati. Quando [acquisti una licenza](https://purchase.aspose.com/buy), copia semplicemente la licenza nella cartella appropriata e Aspose.Slides for Reporting Services inizierà a funzionare in modalità licenziata. 
## **Abbonamento Server**
L'abbonamento server fornisce le seguenti funzionalità:

- È necessaria una licenza per ogni server fisico o virtuale.
- Numero illimitato di processori o core.
- Numero illimitato di istanze Microsoft SQL Server.
- Numero illimitato di sviluppatori.
- Rilasci nuovi e correzioni rapide gratuiti per 1 anno.
- Supporto tecnico gratuito per 1 anno.
- Lo sconto si applica a più acquisti.
## **Abbonamento Sito**
L'abbonamento sito fornisce le seguenti funzionalità:

- È necessaria una licenza per ogni sito fisico (campus).
- Numero illimitato di server fisici o virtuali.
- Numero illimitato di processori o core.
- Numero illimitato di istanze Microsoft SQL Server.
- Numero illimitato di sviluppatori.
- Rilasci nuovi e correzioni rapide gratuiti per 1 anno.
- Supporto tecnico gratuito per 1 anno.
- Lo sconto si applica a più acquisti.
