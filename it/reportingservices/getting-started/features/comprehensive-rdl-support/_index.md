{{% alert color="primary" %}} 

Aspose.Slides for Reporting Services supporta completamente tutte le specifiche RDL. Questo comporta due grandi vantaggi: 

- Nessuna necessità di ridisegnare i report esistenti. Puoi esportare qualsiasi report RDL esistente come una presentazione Microsoft PowerPoint e apparirà esattamente secondo il design RDL.
- Nessuna necessità di utilizzare un designer di report specifico. Puoi usare qualsiasi designer di report RDL e il report verrà esportato esattamente come lo hai progettato.

{{% /alert %}} 

Aspose.Slides for Reporting Services supporta i seguenti elementi RDL: 

- Pagina, intestazioni, piè di pagina
- Caselle di testo
- Immagini
- Sub‑report
- Grafici
- Elenchi
- Tabelle
- Matrici
- Stili
- Rettangoli
