{{% alert color="primary" %}} 

Microsoft SQL Server Reporting Services non dispone di funzionalità integrate per esportare i report come presentazioni Microsoft PowerPoint, ma dopo aver installato Aspose.Slides for Reporting Services sul server, otterrai l'accesso a due formati di esportazione aggiuntivi: 

- Microsoft PowerPoint Presentation (PPT)
- Microsoft PowerPoint SlideShow (PPS)
- Microsoft PowerPoint 2007 Presentation (PPTX)
- Microsoft PowerPoint 2007 SlideShow (PPSX)

{{% /alert %}} 

Per creare documenti in questi formati, Aspose.Slides for Reporting Services si basa su una versione integrata di [Aspose.Slides for .NET](http://www.aspose.com/Products/Aspose.Slides/), il componente leader di mercato per l'elaborazione di presentazioni di Aspose. Microsoft PowerPoint non viene utilizzato per generare i documenti. 


Le copie PPT e PPS dello stesso report appariranno identiche, grazie all'elevato livello di supporto per i formati di file sottostanti fornito da Aspose.Slides.
