{{% alert color="primary" %}} 

Aspose.Slides for Reporting Services ha implementato il supporto per Power BI. 

{{% /alert %}} 

I report paginati sono report formattati per adattarsi bene a una pagina. Tali report sono ideali per operazioni di esportazione e stampa. I report paginati sono progettati in modo da fornire tutte le pagine necessarie per stampare o esportare ogni riga di dati nella tua tabella. 

Ad esempio, se desideri creare un report in modo che i proprietari di quel report possano visualizzare la quantità di prodotto, il prezzo di vendita e il profitto per ogni articolo venduto negli ultimi 12 mesi, allora un report paginato è ciò che ti serve. 

Per consentirti di creare report paginati, Microsoft fornisce Power BI Report Builder in Power BI. Dopo aver creato un report paginato e averlo salvato sul tuo computer, il file risultante avrà l'estensione .rdl.
