## **Piattaforme supportate**
Aspose.Slides for Python via Java è un'API indipendente dalla piattaforma. Può essere utilizzata su qualsiasi piattaforma (Windows, Linux, macOS, ecc.) dove è installato Python 3.7 o versioni successive. JDK 8 o versioni successive è un altro requisito.

## **Formati file e conversioni**
Aspose.Slides for Python via Java supporta la maggior parte dei formati di documento PowerPoint. Consente inoltre di esportarli nei formati di file più popolari che le organizzazioni usano e condividono tra loro. Vedi i dettagli seguenti:

|**Funzione**|**Descrizione**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Python via Java offre l'elaborazione più veloce per questo formato di documento di presentazione.|
|PPT to PPTX conversion|Aspose.Slides for Python via Java supporta la conversione da PPT a PPTX.|
|Portable Document Format (PDF)|È possibile esportare tutti i formati file supportati in documenti Adobe Portable Document Format (PDF) con un unico metodo.|
|XML Parser Specification (XPS)|È possibile esportare tutti i formati file supportati in documenti XML Parser Specification (XPS) con un unico metodo.|
|Tagged Image File Format (TIFF)|È possibile esportare tutti i formati file di presentazione supportati in Tagged Image File Format (TIFF).|
|PPTX To HTML Conversion|Aspose.Slides for Python via Java supporta la conversione di PresentationEx in formato HTML.|

## **Rendering e stampa**
Aspose.Slides for Python via Java supporta il rendering ad alta fedeltà delle diapositive di una presentazione in diversi formati grafici. Vedi i dettagli seguenti:

|**Funzione**|**Descrizione**|
| :- | :- |
|Java Supported Image Formats|Utilizzando Aspose.Slides for Python via Java, è possibile renderizzare le diapositive della presentazione e le immagini sulle diapositive in tutti i formati grafici supportati da Java, come TIFF, PNG, BMP, JPEG, GIF.|
|SVG Format|Aspose.Slides for Python via Java fornisce inoltre metodi integrati che consentono di esportare le diapositive della presentazione in formati Scalable Vector Graphics (SVG).|
|Presentation Printing|Le versioni più recenti di Aspose.Slides for Python via Java offrono metodi di stampa integrati con diverse opzioni.|

## **Funzionalità del contenuto**
Aspose.Slides for Python via Java consente di accedere, modificare o creare quasi tutti gli elementi o contenuti dei documenti di presentazione. Vedi i dettagli seguenti:

|**Funzione**|**Descrizione**|
| :- | :- |
|Master Slides|Le diapositive master definiscono il layout delle diapositive normali. Aspose.Slides for Python via Java consente di accedere e modificare le diapositive master dei documenti di presentazione.|
|Normal Slides|Con Aspose.Slides for Python via Java, è possibile creare nuove diapositive di diversi tipi; è anche possibile accedere e modificare le diapositive esistenti nelle presentazioni.|
|Cloning / Copying Slides|Sono disponibili metodi integrati forniti da Aspose.Slides for Python via Java che consentono di clonare o copiare diapositive esistenti all'interno di una presentazione. È inoltre possibile utilizzare diapositive copiate e clonate da una presentazione all'altra. Poiché una diapositiva eredita il layout dalla diapositiva master, i metodi di clonazione integrati copiano automaticamente il master durante la clonazione.|
|Managing Slides sections|Metodi per organizzare le diapositive in diverse sezioni all'interno di una presentazione.|
|Place Holders and Text Holders|È possibile accedere ai segnaposto e ai segnaposto di testo in una diapositiva. Inoltre, è possibile creare una diapositiva con segnaposto di testo da zero utilizzando il metodo appropriato.|
|Header and Footers|Aspose.Slides for Python via Java facilita la gestione di intestazioni e piè di pagina nelle diapositive.|
|Notes in Slides|Con Aspose.Slides for Python via Java, è possibile accedere e modificare le note associate a una diapositiva e anche aggiungere nuove note.|
|Finding a Shape|È inoltre possibile trovare una forma particolare in una diapositiva utilizzando il testo alternativo associato alla forma.|
|Backgrounds|Aspose.Slides for Python via Java consente di lavorare con gli sfondi associati a una diapositiva master o normale in una presentazione.|
|Text Boxes|Le caselle di testo possono essere create da zero. È possibile accedere alle caselle di testo esistenti. È anche possibile modificare i loro testi senza perdere il formato originale del testo.|
|Rectangle Shapes|È possibile creare o modificare forme rettangolari con Aspose.Slides for Python via Java.|
|Poly Line Shapes|È possibile creare o modificare forme a linea poligonale con Aspose.Slides for Python via Java.|
|Ellipse Shapes|È possibile creare o modificare forme ellittiche con Aspose.Slides for Python via Java.|
|Group Shapes|Aspose.Slides for Python via Java supporta le forme raggruppate.|
|Auto Shapes|Aspose.Slides for Python via Java supporta le forme automatiche.|
|SmartArt|Aspose.Slides for Python via Java fornisce supporto per le forme SmartArt in MS PowerPoint.|
|Charts|Aspose.Slides for Python via Java fornisce supporto per i grafici MSO in PowerPoint.|
|Shapes Serialization|Aspose.Slides for Python via Java supporta un gran numero di forme. Quando Aspose.Slides for Python via Java non supporta una forma, è possibile utilizzare un metodo di serializzazione attraverso il quale si può serializzare quella forma da una diapositiva esistente. In questo modo, è possibile utilizzare la forma successivamente secondo le proprie esigenze.|
|Picture Frames|È possibile gestire le immagini nei riquadri immagine con Aspose.Slides for Python via Java.|
|Audio Frames|È possibile collegare o incorporare file audio nei riquadri audio sulle diapositive con Aspose.Slides for Python via Java.|
|Video Frames|È possibile gestire i file video nei riquadri video. Aspose.Slides for Python via Java fornisce inoltre supporto per i video collegati e incorporati.|
|OLE Frame|È possibile gestire gli oggetti OLE nei riquadri OLE con Aspose.Slides for Python via Java.|
|Tables|Aspose.Slides for Python via Java supporta le tabelle nelle diapositive.|
|ActiveX Controls|Supporto per i controlli ActiveX.|
|VBA Macros|Supporto per la gestione delle macro VBA all'interno delle presentazioni.|
|Text Frame|È possibile accedere al testo di qualsiasi forma tramite il riquadro di testo associato a quella forma.|
|Text Scanning|È possibile scansionare il testo in una presentazione a livello di presentazione o diapositiva tramite metodi di scansione integrati.|
|Animations|È possibile applicare animazioni alle forme.|
|Slide Shows|Aspose.Slides for Python via Java supporta le presentazioni e le transizioni delle diapositive.|

## **Funzionalità di formattazione**
Con Aspose.Slides for Python via Java, è possibile formattare testi e forme sulle diapositive delle presentazioni. Vedi i dettagli seguenti:

|**Funzione**|**Descrizione**|
| :- | :- |
|Text Formatting|<p>In Aspose.Slides for Python via Java, è possibile gestire i testi tramite i riquadri di testo associati alle forme. Pertanto, è possibile formattare i testi usando i paragrafi e le parti associate ai riquadri di testo. Questi elementi di testo possono essere formattati tramite Aspose.Slides for Python via Java.</p><p>- Tipo di carattere</p><p>- Dimensione del carattere</p><p>- Colore del carattere</p><p>- Tonalità del carattere</p><p>- Allineamento del paragrafo</p><p>- Elenco puntato del paragrafo</p><p>- Orientamento del paragrafo</p>|
|Shape Formatting|<p>In Aspose.Slides for Python via Java, l'elemento base di una diapositiva è una forma. È possibile formattare questi elementi di forma con Aspose.Slides for Python via Java:</p><p>- Posizione</p><p>- Dimensione</p><p>- Linea</p><p>- Riempimento (inclusi Pattern, Gradient, Solid)</p><p>- Testo</p><p>- Immagine</p>|
