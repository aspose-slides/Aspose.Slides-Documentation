Una libreria è una struttura di contenuto che contiene file. Una libreria può contenere solo un tipo di file o molti tipi diversi, inclusi documenti, fogli di calcolo e presentazioni. Una libreria SharePoint crea un'unica posizione in cui le persone possono caricare e accedere ai documenti. I documenti sono memorizzati in formato binario nel database SQL Server utilizzato dal sito SharePoint. I documenti sono disponibili per gli altri utenti di SharePoint come un elenco di collegamenti ipertestuali.
