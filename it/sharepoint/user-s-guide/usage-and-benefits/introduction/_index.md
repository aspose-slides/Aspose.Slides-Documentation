I siti SharePoint sono pensati per la collaborazione, la comunicazione e l'archiviazione dei contenuti al fine di soddisfare esigenze aziendali specifiche. Consentono a un gruppo di persone di raggiungere un obiettivo comune facilitando la condivisione delle informazioni e la comunicazione. Le informazioni sono spesso disponibili come sito web e sono accessibili tramite un browser. 
{{% /alert %}} 
## **Cos'è SharePoint e Aspose.Slides?**
Aspose.Slides per SharePoint è una soluzione flessibile per la conversione di documenti Microsoft PowerPoint in altri formati come PPT, POT, PPS, PPTX, PPSX e POTX. Aspose.Slides per SharePoint consente di leggere e convertire file PowerPoint in un'applicazione SharePoint senza utilizzare Microsoft PowerPoint.
