## **Requisiti del server**
{{% alert color="primary" %}} 

Per installare Aspose.Slides per SharePoint, è necessario un sistema che esegua una delle seguenti versioni:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft Office SharePoint Server 2010
- Microsoft Office SharePoint Server 2013
- Microsoft Office SharePoint Server 2016
- Microsoft Office SharePoint Server 2019

{{% /alert %}} {{% alert color="primary" %}} 

Si raccomanda di installare Service Pack 2 o versioni successive per questi prodotti.

{{% /alert %}} 
## **Requisiti del client**
{{% alert color="primary" %}} 

I siti SharePoint sono accessibili tramite un browser web. Aspose.Slides per SharePoint è stato testato con i seguenti browser:

- Microsoft Internet Explorer 6, 7, 8
- Mozilla FireFox 3.5

{{% /alert %}}
