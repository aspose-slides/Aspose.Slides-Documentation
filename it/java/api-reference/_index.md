{{% alert color="primary" %}} 

Aspose.Slides for Java è una libreria di classi che consente alle applicazioni Java di leggere e scrivere documenti di presentazione senza utilizzare Microsoft PowerPoint® o altri. Aspose.Slides for Java consente di gestire testo, forme, tabelle e animazioni, aggiungere audio e video alle diapositive, visualizzare in anteprima le diapositive, esportare le diapositive in formato SVG, PDF e altro.

L'ultima API Reference è disponibile su [Aspose.Slides for Java API Reference](https://reference.aspose.com/slides/it/java).
