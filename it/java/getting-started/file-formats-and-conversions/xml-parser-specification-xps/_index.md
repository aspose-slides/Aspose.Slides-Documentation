La [XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) è un linguaggio di descrizione di pagina e un formato di documento fisso originariamente sviluppato da Microsoft. Come PDF, XPS è progettato per preservare la fedeltà del documento e fornire un aspetto del documento indipendente dal dispositivo. 

{{% /alert %}} 

## **XPS in Aspose.Slides per Java**
Qualsiasi documento di presentazione che può essere caricato da Aspose.Slides per Java può essere convertito nel formato XPS. Aspose.Slides per Java utilizza il motore di layout di pagina e rendering ad alta fedeltà per produrre output nel formato di documento XPS a layout fisso.
Puoi apprendere come esportare i documenti di presentazione in documenti XPS tramite Aspose.Slides per Java in [Converting to XPS](https://docs.aspose.com/slides/it/java/convert-powerpoint-to-xps/).
