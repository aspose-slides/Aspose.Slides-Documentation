## **Panoramica**

Questo articolo descrive un bug di licenza riscontrato in Aspose.Slides per Java versioni 23.2, 23.3, 23.4, 23.5, 23.6 e 23.7. Questo bug ha provocato che le licenze perpetue diventassero temporaneamente inutilizzabili dopo la scadenza della loro sottoscrizione.

## **Sintomi**

Dopo la scadenza della sottoscrizione della licenza perpetua, Aspose.Slides per Java versioni 23.2 - 23.7 può segnalare la licenza come scaduta, passando tutte le funzioni a modalità di valutazione. Questo comportamento è errato e non riguarda le versioni precedenti alla 23.2 e successive alla 23.7.

## **Risoluzione**

Aspose.Slides per Java ha risolto questo problema e ha rilasciato versioni aggiornate (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) con la correzione. 

Se utilizzi una delle versioni interessate di Aspose.Slides per Java nel tuo progetto, passa alla versione corretta.

## **Prodotti e versioni interessati**

Nota che questo bug interessa solo le versioni di Aspose.Slides per Java dalla 23.2 alla 23.7. **Altri prodotti Aspose e altre versioni di Aspose.Slides non sono interessati**.
