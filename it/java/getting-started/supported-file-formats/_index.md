## **Panoramica**

Aspose.Slides supporta file di presentazione da Microsoft PowerPoint 97 fino a Office 365, inclusi Microsoft PowerPoint per Mac. Questo articolo elenca le versioni di PowerPoint supportate dalla libreria e fornisce una tabella dei formati di file che possono essere caricati, salvati o entrambi.

L'articolo risponde anche a domande comuni riguardo la conformità PDF, l'incorporamento dei caratteri, i file protetti da password, i caratteri personalizzati, il fallback dei caratteri e le opzioni di esportazione XPS.

## **FAQ**

**Posso salvare le presentazioni in PDF che soddisfano gli standard di archiviazione e accessibilità (PDF/A e PDF/UA)?**

Sì. Aspose.Slides supporta l'esportazione in PDF con livelli di conformità come PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b, nonché PDF/UA tramite l'impostazione [conformità](https://reference.aspose.com/slides/it/java/com.aspose.slides/pdfoptions/#setCompliance-int-) in [opzioni di esportazione PDF](https://reference.aspose.com/slides/it/java/com.aspose.slides/pdfoptions/).

**La libreria supporta l'incorporamento dei font durante l'esportazione in PDF, con un controllo dettagliato su ciò che viene incorporato?**

Sì. È possibile controllare se i font sono completamente incorporati o sottoselezionati (solo i glifi usati), specificare come vengono trattati i font di sistema comuni e configurare il comportamento per il testo ASCII tramite [opzioni di esportazione PDF](https://reference.aspose.com/slides/it/java/com.aspose.slides/pdfoptions/).

**Posso rilevare se un file è protetto da password prima di caricarlo effettivamente?**

Sì. Utilizzando l'[API di ispezione basata su factory](https://reference.aspose.com/slides/it/java/com.aspose.slides/presentationfactory/), è possibile interrogare un file di presentazione per determinare se è protetto da password senza aprirlo completamente.

**Esistono meccanismi di fallback dei font e supporto per font personalizzati?**

Sì. La libreria supporta il [caricamento](/slides/it/java/custom-font/) e l'[incorporamento](/slides/it/java/embedded-font/) di font personalizzati e fornisce [regole di fallback dei caratteri](/slides/it/java/fallback-font/) per evitare glifi mancanti durante il rendering e la conversione.

**Posso esportare le diapositive in XPS e ci sono opzioni per regolare l'output XPS?**

Sì. [Esporta in XPS](/slides/it/java/convert-powerpoint-to-xps/) è supportato, e è possibile regolare le relative [opzioni di salvataggio](https://reference.aspose.com/slides/it/java/com.aspose.slides/xpsoptions/) per controllare la qualità e il contenuto del documento XPS.
