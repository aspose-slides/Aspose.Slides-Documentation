Aspose.Slides for Java 14.3.0 (14.4.0) fornisce un'implementazione completamente nuova dell'elaborazione PPT. Ci sono molti miglioramenti, conversione parziale da PPTX a PPT. Tuttavia, alcune funzionalità non sono ancora implementate:

- Alcune forme hanno geometria errata nei documenti PPT serializzati (Call outs)
- Non tutte le funzionalità di formattazione testo di PPTX sono supportate nella serializzazione PPT
- Le informazioni sulla lingua del testo e le impostazioni di ortografia non sono presenti nei documenti PPT serializzati
- Non tutte le funzionalità dei temi PPTX sono supportate nella serializzazione PPT

**Ci sono alcune differenze rispetto a Aspose.Slides for Java 8.6.0:**

- Sono noti problemi nella serializzazione PPT da OLE/ActiveX a PPT

**Ci sono alcune differenze rispetto a Aspose.Slides for .NET 14.3.0:**

- Il supporto alla stampa delle presentazioni al momento non è disponibile in Aspose.Slides for Java
