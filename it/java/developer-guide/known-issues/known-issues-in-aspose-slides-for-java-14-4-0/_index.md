{{% alert color="primary" %}} 

Aspose.Slides for Java 14.4.0 fornisce una nuova decisione per l'elaborazione dei documenti PowerPoint. Ci sono alcune limitazioni e problemi noti, che saranno rimossi nelle prossime release:

- Alcune forme hanno geometria errata nei documenti PPT serializzati (arco, freccia circolare, didascalie).
- Non tutte le funzionalità di formattazione del testo PPTX sono supportate nella serializzazione PPT (tabulazione, rientro e limitazioni nella formattazione dei paragrafi).
- Le informazioni sulla lingua del testo e le impostazioni di ortografia non sono presenti nei documenti PPT serializzati.
- Non tutte le funzionalità del tema PPTX sono supportate nella serializzazione PPT (solo la serializzazione di formati di riempimento, formati di linea e carattere).
- Sono noti problemi nella serializzazione OLE/ActiveX da PPT a PPT.
- La serializzazione e il rendering di WordArt non sono supportati.
