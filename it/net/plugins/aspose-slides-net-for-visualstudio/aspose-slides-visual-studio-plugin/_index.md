## **Introduzione**
Microsoft Visual Studio è un ambiente di sviluppo integrato (IDE) di Microsoft. Viene utilizzato per sviluppare programmi per Microsoft Windows, nonché siti web, applicazioni web e servizi web. Visual Studio utilizza le piattaforme di sviluppo software Microsoft come Windows API, Windows Forms, Windows Presentation Foundation, Windows Store e Microsoft Silverlight. Può produrre sia codice nativo che codice gestito.

### **Aspose.Slides for .NET**
Aspose.Slides for .NET è un’unica API di gestione di PowerPoint® che consente alle applicazioni .NET di leggere, scrivere e manipolare documenti PowerPoint senza utilizzare Microsoft PowerPoint.

Aspose.Slides for .NET è la prima e unica API .NET per PowerPoint che offre funzionalità per gestire documenti PowerPoint all’interno delle proprie applicazioni. Come tutti i componenti Aspose .NET, Aspose.Slides è scritto in C# gestito, con prezzi estremamente competitivi e prestazioni fulminee.

Per ulteriori dettagli su [Aspose.Slides for .NET API](http://www.aspose.com/.net/powerpoint-component.aspx)

### **Esplora gli esempi di Aspose.Slides for .NET con Visual Studio**
Aspose Visual Studio Plugin - **Explore Aspose.Slides for .NET Examples** è uno strumento eccellente per scaricare rapidamente ed esplorare gli esempi dell’API Aspose.Slides for .NET. Consente di risparmiare tempo e sforzi fornendo un’opzione molto semplice per selezionare, scaricare e aprire gli ultimi progetti di esempio senza dover uscire da Visual Studio.

Questa versione offre le seguenti funzionalità:

- Supporta Visual Studio 2010, Visual Studio 2012 e Visual Studio 2013.  
- Facile da avviare dal menu **File > Aspose** in Visual Studio  
- Risparmia tempo e riduce la curva di apprendimento.  
- Consente di selezionare e aprire gli ultimi esempi dell’API Aspose.Slides for .NET ospitati su Github.com.  
- Consente di aprire progetti di esempio **C#** o **VB.NET**  
- Un progetto selezionato viene aperto automaticamente in Visual Studio con tutti i riferimenti necessari impostati, fornendo un ambiente pronto per l’esecuzione e l’esplorazione.  
- Scarica automaticamente le librerie API più recenti da **NuGet** e gli esempi da **GitHub**.

Per installare e utilizzare Aspose Visual Studio Plugin è necessario disporre di una delle seguenti versioni di Visual Studio installate

Sentitevi liberi di contattarci se riscontrate problemi nell’installazione o nell’uso di questo plugin.

## **Installazione**
L’installazione di Aspose Visual Studio Plugin è molto semplice, seguite questi passaggi

- Fate doppio clic sul file msi scaricato  
- Seguite i passaggi, potete semplicemente premere **Next** su ogni schermata per installarlo con le impostazioni predefinite.

Verrà visualizzata una finestra di dialogo di conferma al termine dell’installazione.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_2)

**Nota:** Assicuratevi di riavviare Visual Studio affinché le modifiche abbiano effetto.

## **Utilizzo**
Una volta installato potete avviare e usare il plugin interattivo come descritto di seguito:
**1.** Aprite il plugin selezionando **File > Aspose** seguito da **New Aspose.Slides Example Project** in Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_3)

**2.** La prima volta che aprite il plugin vedrete la finestra di dialogo di selezione dell’API come mostrato sotto. Ci vorranno alcuni minuti per scaricare le ultime API e le relative librerie da **NuGet** e il codice sorgente degli ultimi esempi da **GitHub**.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_4)

**3.** Nella finestra di dialogo *Select an API to View Examples*, selezionate o modificate il percorso in cui verranno creati i progetti di esempio e scegliete o cambiate il tipo di progetto di esempio tra **C#** o **VB.NET**, quindi fate clic su **Continue** per aprire il progetto di esempio in Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_1)

**4.** Nuovi progetti di esempio in base alla selezione del tipo di progetto (**C#** o **VB.NET**) in Visual Studio.

#### **Codice sorgente**
Potete ottenere l’ultima versione del codice sorgente da una delle seguenti posizioni

- [Github](https://github.com/aspose-slides/Aspose.Slides-for-.NET)

#### **Come configurare il codice sorgente**
È necessario avere installato quanto segue per aprire ed estendere il codice sorgente

- Visual Studio 2010

Seguite questi semplici passaggi per iniziare

1. Scaricate/Clonate il codice sorgente.  
1. Aprite Visual Studio 2010 e scegliete **File > Open Project**  
1. Navigate fino al codice sorgente più recente scaricato e aprite **AsposeVisualStudioPlugin.sln**

#### **Panoramica del codice sorgente**
La soluzione contiene due progetti

- **AsposeVisualStudioPlugin** – Contiene l’integrazione con Visual Studio, la selezione e il download automatico delle API Aspose e degli esempi.  
- **Installer** – Progetto Wix per generare il file di installazione msi del plugin.
