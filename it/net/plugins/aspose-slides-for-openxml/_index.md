## **Che cos'è OpenXML?**
OpenXML SDK è un SDK .NET di Microsoft per elaborare i formati di file di Microsoft Office 2007 e versioni successive. Questi formati di file sono fondamentalmente pacchetti zip e l'Open XML SDK per Office semplifica il compito di manipolare i pacchetti Open XML e gli elementi dello schema Open XML sottostanti all'interno di un pacchetto. Le classi nell'Open XML SDK incapsulano molte attività comuni che gli sviluppatori eseguono sui pacchetti Open XML, in modo da poter eseguire operazioni complesse con poche righe di codice.

Una volta installato l'Open XML SDK 2.5, apri il tuo progetto o applicazione esistente in Visual Studio, o crea un nuovo progetto o applicazione. Quindi, nel tuo progetto o applicazione, aggiungi riferimenti ai seguenti componenti:

- DocumentFormat.OpenXml
- WindowsBase

## **Aspose.Slides per .NET**
Aspose.Slides per .NET è un componente unico per la gestione di PowerPoint® che consente alle applicazioni .NET di leggere, scrivere e manipolare documenti PowerPoint senza utilizzare Microsoft PowerPoint.

Aspose.Slides per .NET è il primo e unico componente .NET che fornisce funzionalità per gestire documenti PowerPoint nelle proprie applicazioni. Come tutti i componenti Aspose .NET, Aspose.Slides è scritto in managed C#, ha un prezzo incredibilmente conveniente ed è rapidissimo.

## **Introduzione**
Gli sviluppatori alla ricerca di librerie per formati di file potrebbero dover esaminare le funzionalità offerte dalle librerie disponibili sul mercato. Principalmente, due fattori sono considerati nella scelta di un'opzione:

- Caratteristiche
- Sforzo richiesto per utilizzare la libreria

Se sei uno sviluppatore .NET alla ricerca di una libreria per formati di file più veloce, semplice e leggera per l'elaborazione dei file, potresti dover confrontare VSTO (Visual Studio tool per Office) con Aspose. In questo caso, questo progetto ti aiuterà a confrontare le funzionalità e il codice di entrambe le librerie.

## **Requisiti di sistema e piattaforme supportate**
### **Requisiti di sistema**
Una o più delle seguenti edizioni di Visual Studio Tools for Office:

- Microsoft Visual Studio 2005 Tools for the Microsoft Office System
- Microsoft Visual Studio 2005 Tools for the 2007 Microsoft Office System
- Visual Studio 2008 Professional Edition
- Visual Studio 2008 Team Suite Edition
- Una versione di Microsoft Office:
  - Microsoft Office Professional 2003 SP1
  - 2007 Microsoft Office system
### **Sistemi operativi supportati**
- Windows 2000; Windows Server 2003; Windows Vista; Windows XP
### **Versioni .NET Framework supportate**
- solo .NET framework 2.0 o versioni successive.
