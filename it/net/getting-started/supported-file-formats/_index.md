## **Panoramica**

Aspose.Slides supporta i file di presentazione da Microsoft PowerPoint 97 fino a Office 365, inclusi Microsoft PowerPoint per Mac. Questo articolo elenca le versioni di PowerPoint supportate dalla libreria e fornisce una tabella dei formati di file che possono essere caricati, salvati o entrambi.

L'articolo risponde anche alle domande più comuni su conformità PDF, incorporamento dei caratteri, file protetti da password, caratteri personalizzati, fallback dei caratteri e opzioni di esportazione XPS.

## **Versioni Microsoft PowerPoint supportate**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint for MAC
- Office 365

## **FAQ**

**Posso salvare le presentazioni in PDF che soddisfano gli standard di archiviazione e accessibilità (PDF/A e PDF/UA)?**

Sì. Aspose.Slides supporta l'esportazione in PDF con livelli di conformità come PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b, nonché PDF/UA tramite l'impostazione [conformità](https://reference.aspose.com/slides/it/net/aspose.slides.export/pdfoptions/compliance/) nelle [opzioni di esportazione PDF](https://reference.aspose.com/slides/it/net/aspose.slides.export/pdfoptions/).

**La libreria supporta l'incorporamento dei caratteri durante l'esportazione in PDF, con controllo dettagliato su cosa viene incorporato?**

Sì. Puoi controllare se i caratteri sono completamente incorporati o sottoposti a subset (solo glyph usati), specificare come vengono trattati i caratteri di sistema comuni e configurare il comportamento per il testo ASCII tramite le [opzioni di esportazione PDF](https://reference.aspose.com/slides/it/net/aspose.slides.export/pdfoptions/).

**Posso rilevare se un file è protetto da password prima di caricarlo effettivamente?**

Sì. Utilizzando l'[API di ispezione basata su factory](https://reference.aspose.com/slides/it/net/aspose.slides/presentationfactory/), è possibile verificare se un file di presentazione è protetto da password senza aprirlo completamente.

**Esistono meccanismi di fallback dei caratteri e supporto per caratteri personalizzati?**

Sì. La libreria supporta il [caricamento](/slides/it/net/custom-font/) e l'[incorporamento](/slides/it/net/embedded-font/) di caratteri personalizzati e fornisce regole di [fallback dei caratteri](/slides/it/net/fallback-font/) per evitare glyph mancanti durante il rendering e la conversione.

**Posso esportare le diapositive in XPS e sono disponibili opzioni per regolare l'output XPS?**

Sì. L'[esportazione in XPS](/slides/it/net/convert-powerpoint-to-xps/) è supportata, e è possibile regolare le relative [opzioni di salvataggio](https://reference.aspose.com/slides/it/net/aspose.slides.export/xpsoptions/) per controllare la qualità e il contenuto del documento XPS.
