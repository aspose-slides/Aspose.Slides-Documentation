{{% alert color="primary" %}} 

Aspose.Slides per PHP via Java è una libreria di classi che consente alle applicazioni PHP di leggere e scrivere documenti di presentazione senza utilizzare Microsoft PowerPoint® o altri. Aspose.Slides per PHP via Java permette di gestire testo, forme, tabelle e animazioni, aggiungere audio e video alle diapositive, visualizzare le diapositive in anteprima, esportare le diapositive in formato SVG, PDF e altro.

L'ultima documentazione di riferimento dell'API è disponibile su [Aspose.Slides for PHP via Java API Reference](https://reference.aspose.com/slides/it/php-java/).
