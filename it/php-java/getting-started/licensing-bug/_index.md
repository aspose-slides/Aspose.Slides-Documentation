## **Panoramica**

Questo articolo descrive un bug di licenza riscontrato in Aspose.Slides for PHP via Java versioni 23.2, 23.3, 23.4, 23.5, 23.6 e 23.7. Il bug ha reso le licenze perpetue temporaneamente inutilizzabili dopo la scadenza della loro sottoscrizione.

## **Sintomi**

Dopo la scadenza della sottoscrizione della licenza perpetua, Aspose.Slides for PHP via Java versioni 23.2 - 23.7 può segnalare la licenza come scaduta, passando tutte le funzioni in modalità di valutazione.  
Questo comportamento è errato e non riguarda le versioni precedenti alla 23.2 né quelle successive alla 23.7.

## **Risoluzione**

Aspose.Slides for PHP via Java ha risolto il problema e ha rilasciato versioni aggiornate (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) con la correzione.

Se utilizzi una delle versioni interessate di Aspose.Slides for PHP via Java nel tuo progetto, passa alla versione corretta.

## **Prodotti e versioni interessati**

Si noti che questo bug interessa solo le versioni di Aspose.Slides for PHP via Java dalla 23.2 alla 23.7. **Altri prodotti Aspose e altre versioni di Aspose.Slides non sono interessati**.
