## **Supporto tecnico**
{{% alert color="primary" %}}

Aspose fornisce supporto tecnico illimitato e gratuito per tutti i suoi prodotti. Il supporto è disponibile per tutti gli utenti (inclusi gli utenti con pacchetti di valutazione).

{{% /alert %}} 

Quando hai bisogno di assistenza per Aspose.Slides for PHP via Java, considera quanto segue:

- Il canale principale di supporto è il [Aspose.Forums](https://forum.aspose.com/c/slides/it/11). Pubblica la tua domanda nel forum Aspose.Slides—verrà risposta entro poche ore. Il team di supporto di Aspose.Slides risponde direttamente alle domande pubblicate sul forum.
- Si prega di notare che Aspose non offre supporto tecnico telefonico. Il supporto telefonico è disponibile solo per domande relative a vendite e acquisti.
- Quando ti aspetti una risposta nei forum, sii paziente e tieni conto delle differenze di fuso orario.

Quando riscontri un problema con Aspose.Slides for PHP via Java, segui queste linee guida per risolverlo nel modo più efficiente:

- Verifica e conferma di stare usando l'ultima versione di Aspose.Slides for PHP via Java prima di segnalare il problema. È possibile che il problema che hai riscontrato sia già stato risolto in un aggiornamento.

  Consulta [Aspose.Slides download](https://downloads.aspose.com/slides/it/php-java) per ottenere informazioni sull'ultima versione.

- Prima di segnalare un problema, potresti voler consultare i nostri forum, questa documentazione e il Riferimento API. Forse la risposta alla tua domanda è già stata fornita.

- Quando segnali un problema, includi il documento originale e (se possibile) un frammento del tuo codice relativo al problema. Se devi allegare più file, mettili in un pacchetto ZIP. Puoi allegare in sicurezza i tuoi documenti su Aspose.Forums perché **solo** tu e gli sviluppatori di Aspose avete accesso ai file allegati.

- Crea e utilizza un unico thread per un problema specifico. Se riscontri un altro problema, devi creare un altro thread per segnalarlo.
