## **Panoramica**

Aspose.Slides supporta i file di presentazione da Microsoft PowerPoint 97 fino a Office 365, inclusi Microsoft PowerPoint per Mac. Questo articolo elenca le versioni di PowerPoint supportate dalla libreria e fornisce una tabella dei formati di file che possono essere caricati, salvati o entrambi.

L'articolo risponde anche a domande comuni su conformità PDF, incorporamento dei caratteri, file protetti da password, caratteri personalizzati, fallback dei caratteri e opzioni di esportazione XPS.

## **Versioni Microsoft PowerPoint supportate**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint per MAC
- Office 365

## **Domande frequenti**

**Posso salvare le presentazioni in PDF che soddisfano gli standard di archivio e accessibilità (PDF/A e PDF/UA)?**

Sì. Aspose.Slides supporta l'esportazione in PDF con livelli di conformità come PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b, nonché PDF/UA tramite l'impostazione [compliance](https://reference.aspose.com/slides/it/php-java/aspose.slides/pdfoptions/setcompliance/) nelle [PDF export options](https://reference.aspose.com/slides/it/php-java/aspose.slides/pdfoptions/).

**La libreria supporta l'incorporamento dei caratteri durante l'esportazione in PDF, con controllo dettagliato su ciò che viene incorporato?**

Sì. È possibile controllare se i caratteri sono incorporati completamente o in sottoinsieme (solo glifi utilizzati), specificare come vengono trattati i caratteri di sistema comuni e configurare il comportamento per il testo ASCII tramite le [PDF export options](https://reference.aspose.com/slides/it/php-java/aspose.slides/pdfoptions/).

**Posso rilevare se un file è protetto da password prima di caricarlo effettivamente?**

Sì. Utilizzando l'[API di ispezione basata su factory](https://reference.aspose.com/slides/it/php-java/aspose.slides/presentationfactory/), è possibile interrogare un file di presentazione per determinare se è protetto da password senza aprirlo completamente.

**Esistono meccanismi di fallback dei caratteri e supporto per caratteri personalizzati?**

Sì. La libreria supporta il [caricamento](/slides/it/php-java/custom-font/) e l'[incorporamento](/slides/it/php-java/embedded-font/) di caratteri personalizzati e fornisce le [regole di fallback dei caratteri](/slides/it/php-java/fallback-font/) per prevenire glifi mancanti durante il rendering e la conversione.

**Posso esportare le diapositive in XPS e ci sono opzioni per regolare l'output XPS?**

Sì. L'[Export to XPS](/slides/it/php-java/convert-powerpoint-to-xps/) è supportato e è possibile regolare le relative [save options](https://reference.aspose.com/slides/it/php-java/aspose.slides/xpsoptions/) per controllare la qualità e il contenuto del documento XPS.
