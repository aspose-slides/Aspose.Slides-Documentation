## **Informazioni su TIFF**

Il [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) noto per la memorizzazione di più immagini in un unico documento è stato originariamente creato da Aldus. Questo formato è ampiamente supportato da scanner, fax e altre applicazioni di manipolazione delle immagini.

## **TIFF in Aspose.Slides per PHP via Java**

Qualsiasi documento che può essere caricato in Aspose.Slide per Java può anche essere convertito direttamente in documento TIFF da Aspose.Slides per PHP via Java, eliminando la necessità di componenti di terze parti. 
Inoltre, è possibile definire opzionalmente le dimensioni delle immagini nel documento TIFF risultante. È possibile trovare informazioni sull'esportazione dei documenti di presentazione in documenti TIFF tramite Aspose.Slides per PHP via Java in [questa pagina](/slides/it/php-java/convert-powerpoint-to-tiff/).
