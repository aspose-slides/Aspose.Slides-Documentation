Il [Portable Document Format](https://en.wikipedia.org/wiki/PDF) è un formato di file creato da Adobe Systems per lo scambio di documenti tra organizzazioni. Lo scopo del formato era mantenere il contenuto e il layout identici, indipendentemente dalla piattaforma su cui viene visualizzato. Aspose.Slides for PHP via Java consente di convertire i file di presentazione in PDF.

{{% /alert %}} 

## **PDF in Aspose.Slides for PHP via Java**
Qualsiasi presentazione che può essere caricata in Aspose.Slides for PHP via Java può essere convertita in un PDF conforme a [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) o [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA) a seconda della tua scelta. Aspose.Slides for PHP via Java esporta le presentazioni in PDF e, nella maggior parte dei casi, il PDF risultante è identico alla presentazione originale.

Aspose.Slides supporta le seguenti funzionalità della presentazione durante la conversione in PDF:

- Immagini, caselle di testo e altre forme.
- Testo e formattazione.
- Paragrafi e formattazione.
- Collegamenti ipertestuali.
- Intestazioni e piè di pagina.
- Elenco puntato.
- Tabelle.

Puoi esportare le presentazioni in PDF direttamente usando Aspose.Slides for PHP via Java: non è necessario alcun altro componente. Inoltre, puoi personalizzare l'esportazione da presentazione a PDF con varie opzioni come spiegato in [Conversione in PDF](/slides/it/php-java/convert-powerpoint-to-pdf/).
