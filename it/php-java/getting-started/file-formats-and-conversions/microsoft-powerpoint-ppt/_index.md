{{% alert color="primary" %}} 

[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) è il formato di file di documento di presentazione che può essere creato, letto, manipolato e scritto da diverse versioni di Microsoft PowerPoint. Questo è il formato binario per i documenti di presentazione sviluppato da Microsoft.

{{% /alert %}} 

## **PPT in Aspose.Slides for PHP via Java**
Aspose.Slides for PHP via Java può leggere i file PPT creati dal software elencato di seguito.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Allo stesso modo, i file PPT creati da Aspose.Slides for PHP via Java possono essere letti dal suddetto insieme di software.

## **Supporto completo per PPT**
Aspose.Slides for PHP via Java fornisce supporto per quasi tutte le funzionalità supportate dal formato di file del documento PPT. Non solo copre le funzionalità di base e avanzate fornite dalle diverse versioni di Microsoft PowerPoint per la manipolazione dei documenti PPT, ma anche funzionalità che non sono nemmeno supportate da Microsoft PowerPoint. Il principale vantaggio dell'utilizzo della libreria API Aspose.Slides for PHP via Java è la facilità d'uso nella gestione di tali funzionalità.

Oltre alle attività di base relative alla creazione, lettura e scrittura di file di documento PPT, ci sono diverse funzionalità fornite da Aspose.Slides for PHP via Java:

- Importare altri formati di file Microsoft Office come [oggetti OLE nei documenti PPT](/slides/it/php-java/manage-ole/).
- [Esportare documenti PPT in PDF](/slides/it/php-java/convert-powerpoint-to-pdf/).
- Esportare le diapositive nei documenti PPT in formati SVG.
- Renderizzare le diapositive in qualsiasi formato immagine supportato dal Java Framework.
- Impostare le dimensioni delle diapositive nei documenti PPT.
- Gestire le animazioni sulle forme.
- Gestire le presentazioni.
- [Formattare il testo sulle diapositive](/slides/it/php-java/text-formatting/).
- Estrarre il testo dai documenti PPT.
- [Gestire le tabelle sulle diapositive](/slides/it/php-java/powerpoint-table/).
- Copiare automaticamente i master usando [la funzione di clonazione](/slides/it/php-java/clone-slides/).
