## **Panoramica**

Aspose.Slides supporta file di presentazione da Microsoft PowerPoint 97 fino a Office 365, inclusi Microsoft PowerPoint per Mac. Questo articolo elenca le versioni di PowerPoint supportate dalla libreria e fornisce una tabella dei formati di file che è possibile caricare, salvare o entrambi.

L'articolo fornisce anche risposte alle domande comuni su conformità PDF, incorporamento dei font, file protetti da password, font personalizzati, fallback dei font e opzioni di esportazione XPS.

## **Versioni Microsoft PowerPoint supportate**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint per Mac
- Office 365

## **FAQ**

**Posso salvare le presentazioni in PDF che soddisfano gli standard di archiviazione e accessibilità (PDF/A e PDF/UA)?**

Sì. Aspose.Slides per Python via .NET supporta l'esportazione in PDF con livelli di conformità quali PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b, e PDF/UA tramite l'impostazione [compliance](https://reference.aspose.com/slides/it/python-net/aspose.slides.export/pdfoptions/compliance/) nelle [PDF export options](https://reference.aspose.com/slides/it/python-net/aspose.slides.export/pdfoptions/).

**La libreria supporta l'incorporazione dei font durante l'esportazione in PDF, con controllo dettagliato su ciò che viene incorporato?**

Sì. È possibile controllare se i font sono incorporati completamente o in forma di sottoinsieme (solo i glifi utilizzati), specificare come vengono trattati i font di sistema comuni e configurare il comportamento per il testo ASCII tramite le [PDF export options](https://reference.aspose.com/slides/it/python-net/aspose.slides.export/pdfoptions/).

**Posso rilevare se un file è protetto da password prima di caricarlo effettivamente?**

Sì. Utilizzando l'[API di ispezione basata su factory](https://reference.aspose.com/slides/it/python-net/aspose.slides/presentationfactory/), è possibile interrogare un file di presentazione per determinare se è protetto da password senza aprirlo completamente.

**Esistono meccanismi di fallback dei font e supporto per font personalizzati?**

Sì. La libreria supporta il [caricamento](/slides/it/python-net/custom-font/) e l'[incorporamento](/slides/it/python-net/embedded-font/) di font personalizzati e fornisce [regole di fallback dei font](/slides/it/python-net/fallback-font/) per evitare glifi mancanti durante il rendering e la conversione.

**Posso esportare le diapositive in XPS e sono disponibili opzioni per affinare l'output XPS?**

Sì. L'[esportazione in XPS](/slides/it/python-net/convert-powerpoint-to-xps/) è supportata e si possono regolare le pertinenti [save options](https://reference.aspose.com/slides/it/python-net/aspose.slides.export/xpsoptions/) per controllare la qualità dell'output e il contenuto del documento XPS.
