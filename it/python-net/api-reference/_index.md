{{% alert color="primary" %}} 

Aspose.Slides for Python via .NET è una libreria di classi che consente alle applicazioni Python di leggere e scrivere documenti di presentazione senza utilizzare Microsoft PowerPoint® o altri software. Aspose.Slides for Python via .NET ti permette di gestire testo, forme, tabelle e animazioni; aggiungere audio e video alle diapositive; visualizzare in anteprima le diapositive; ed esportare le diapositive in formato SVG, PDF e altro.

L'ultima referenza API è disponibile su [Aspose.Slides for Python via .NET API Reference](https://reference.aspose.com/slides/it/python-net/).
