{{% alert color="primary" %}} 
Aspose.Slides for Node.js via Java è una libreria di classi che consente alle applicazioni Node.js di leggere e scrivere documenti di presentazione senza utilizzare Microsoft PowerPoint® o altri. Aspose.Slides for Node.js via Java consente di gestire testo, forme, tabelle e animazioni, aggiungere audio e video alle diapositive, visualizzare l'anteprima delle diapositive, esportare le diapositive in formato SVG o PDF e molto altro.

L'ultima documentazione API è disponibile su [Riferimento API di Aspose.Slides per Node.js via Java](https://reference.aspose.com/slides/it/nodejs-java/).
