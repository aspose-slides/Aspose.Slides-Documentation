{{% alert color="primary" %}} 

Tutti i componenti Aspose C++ richiedono l'impostazione di autorizzazione Full Trust. Il motivo è che i componenti Aspose C++ devono accedere alle impostazioni del registro, ai file di sistema oltre alla directory virtuale per alcune operazioni, come l'analisi dei font, ecc. Inoltre, i componenti Aspose C++ si basano su classi di sistema C++ di base che anch'esse richiedono l'impostazione di autorizzazione Full Trust in molti casi. 

{{% /alert %}}
