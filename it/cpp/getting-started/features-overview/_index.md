## **Formati di file e conversioni**
Aspose.Slides per C++ supporta la maggior parte dei formati di documento PowerPoint e li esporta anche in formati popolari ampiamente utilizzati per lo scambio tra organizzazioni. La tabella seguente elenca questi formati:

|**Funzionalità**|**Descrizione**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/it/cpp/microsoft-powerpoint-ppt/)|Aspose.Slides per C++ offre l'elaborazione più veloce per questo formato di documento di presentazione.|
|[Portable Document Format (PDF)](/slides/it/cpp/portable-document-format-pdf/)|I formati di file supportati possono essere esportati in documenti Adobe Portable Document Format (PDF) con un unico metodo.|
|[XML Parser Specification (XPS)](/slides/it/cpp/xml-parser-specification-xps/)|Tutti i formati di file supportati possono essere esportati in documenti XML Parser Specification (XPS) con un unico metodo.|
|[PresentationML (PPTX, XML)](/slides/it/cpp/presentationml-pptx-xml/)|Aspose.Slides per C++ supporta l'elaborazione del formato di presentazione OOXML noto anche come PresentationML o PPTX.|
|[PPTX To HTML Conversion](https://docs.aspose.com/slides/it/cpp/convert-powerpoint-to-html/)|Aspose.Slides per C++ supporta la conversione di presentazioni in formato HTML.|

## **Rendering**
Aspose.Slides per C++ supporta il rendering ad alta fedeltà delle diapositive nei documenti di presentazione in vari formati grafici come elencato di seguito:

|**Funzionalità**|**Descrizione**|
| :- | :- |
|C++ Supported Image Formats|Utilizzando Aspose.Slides per C++, potrai renderizzare non solo le diapositive della presentazione, ma anche le immagini nelle diapositive in tutti i formati grafici supportati da C++ come PNG, BMP e JPEG.|
|SVG Format|Aspose.Slides per C++ fornisce anche un metodo integrato per esportare le diapositive della presentazione in formati Scalable Vector Graphics (SVG).|

## **Funzionalità del contenuto**
Aspose.Slides per C++ consente di accedere, modificare o creare quasi tutti i contenuti possibili dei documenti di presentazione.

|**Funzionalità**|**Descrizione**|
| :- | :- |
|Diapositive master|Le diapositive master definiscono il layout delle diapositive normali. Aspose.Slides per C++ consente di accedere e modificare le diapositive master dei documenti di presentazione.|
|Diapositive normali|Utilizzando Aspose.Slides per C++, è possibile non solo creare nuove diapositive di diversi tipi, ma anche accedere e modificare le diapositive esistenti nei documenti di presentazione.|
|Clonazione / Copia diapositive|Aspose.Slides per C++ offre metodi integrati per clonare o copiare diapositive esistenti non solo all'interno di una presentazione, ma anche da una presentazione all'altra. Poiché una diapositiva eredita il suo layout dalla diapositiva master, i metodi di clonazione integrati copiano automaticamente anche il master durante la clonazione.|
|Segnaposto e segnaposto di testo|È inoltre possibile accedere ai segnaposto e ai segnaposto di testo in una diapositiva. Inoltre, è possibile creare una diapositiva con segnaposto di testo da zero utilizzando il metodo appropriato.|
|Intestazioni e piè di pagina|Aspose.Slides per C++ facilita la gestione di intestazioni/ piè di pagina nelle diapositive.|
|Note nelle diapositive|Con Aspose.Slides per C++, è possibile non solo accedere e modificare le note associate a una diapositiva, ma anche aggiungere note.|
|Ricerca di una forma|È possibile trovare una forma specifica in una diapositiva utilizzando il testo alternativo associato alla forma.|
|Sfondi|Aspose.Slides per C++ consente di lavorare con lo sfondo associato a una diapositiva master o normale della presentazione.|
|Caselle di testo|Le caselle di testo possono essere create da zero. Le caselle di testo esistenti possono essere accessate e il loro testo può essere modificato senza perdere il formato originale del testo.|
|Forme rettangolari|Le forme rettangolari possono essere create o modificate da Aspose.Slides per C++.|
|Forme a linea poligonale|Le forme a linea poligonale possono anche essere create o modificate da Aspose.Slides per C++.|
|Forme ellittiche|Le forme ellittiche sono anch'esse create o modificate da Aspose.Slides per C++.|
|Forme raggruppate|Aspose.Slides per C++ supporta anche le forme raggruppate.|
|Forme automatiche|Le forme automatiche sono supportate da Aspose.Slides per C++.|
|Serializzazione delle forme|Sebbene Aspose.Slides per C++ supporti un gran numero di forme, se una forma non è supportata, è disponibile un metodo di serializzazione fornito da Aspose.Slides per C++ attraverso il quale è possibile serializzare quella forma da una diapositiva esistente per ulteriore utilizzo da parte di Aspose.Slides per C++ secondo necessità.|
|Cornici immagine|Le immagini sono gestite in cornici immagine utilizzando Aspose.Slides per C++.|
|Cornici audio|I file audio possono essere collegati o incorporati nelle diapositive in cornici audio da Aspose.Slides per C++.|

|Cornice OLE|Gli oggetti OLE sono gestiti in cornici OLE da Aspose.Slides per C++.|
|Tabelle|Le tabelle nelle diapositive sono supportate da Aspose.Slides per C++.|
|Cornice di testo|Il testo associato a qualsiasi forma può essere accessato attraverso la cornice di testo associata a quella forma.|

## **FAQ**

**Devo installare Microsoft PowerPoint sul server/PC affinché la libreria funzioni?**

No. PowerPoint non è necessario; Aspose.Slides è un motore autonomo per creare, modificare, convertire e rendere presentazioni.

**Come funziona il multithreading? È possibile parallelizzare l'elaborazione?**

È sicuro elaborare documenti diversi in thread diversi; lo stesso [presentazione](https://reference.aspose.com/slides/it/cpp/aspose.slides/presentation/) object non deve essere usato da [thread multipli](/slides/it/cpp/multithreading/) contemporaneamente.

**Sono supportate le password dei file e la crittografia?**

Sì. [È possibile](/slides/it/cpp/password-protected-presentation/) aprire presentazioni crittografate, impostare o rimuovere una password di apertura e scrittura e verificare lo stato di protezione.

**Devo preoccuparmi dei pacchetti di font nei container Linux?**

Sì. Si consiglia di installare pacchetti di font comuni e/o specificare esplicitamente le [directory dei font](/slides/it/cpp/custom-font/) nella tua applicazione per evitare sostituzioni inaspettate.

**Ci sono limitazioni nella versione di valutazione?**

In [modalità di valutazione](/slides/it/cpp/licensing/), viene aggiunta una filigrana all'output e si applicano alcune limitazioni; è disponibile una [licenza temporanea di 30 giorni](https://purchase.aspose.com/temporary-license/) per testare tutte le funzionalità.

**È supportata l'importazione di formati esterni in una presentazione (PDF/HTML → PPTX)?**

Sì. È possibile aggiungere [pagine PDF e contenuti HTML](/slides/it/cpp/import-presentation/) a una presentazione, trasformandoli in diapositive.
