## **Requisiti software**
Assicurati di soddisfare i seguenti requisiti prima di scaricare ed eseguire gli esempi.

1. Visual Studio 2015 o versioni successive.
## **Scarica Aspose.Slides da GitHub**
Tutti gli esempi di Aspose.Slides per C++ sono ospitati su [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-C).

- È possibile clonare il repository usando il client GitHub preferito oppure scaricare il file ZIP da [qui](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/refs/heads/master.zip).
- Estrai il contenuto del file ZIP in una qualsiasi cartella del tuo computer. Tutti gli esempi si trovano nella cartella **Examples**.
- È presente un file di soluzione Visual Studio per C++.
- I progetti sono stati creati in Visual Studio 2015, ma i file di soluzione sono compatibili con Visual Studio 2015 e versioni successive.
- Apri il file di soluzione in Visual Studio e compila il progetto.
- Scarica l'ultima versione di Aspose.Slides per C++ dalla [Sezione Download](https://downloads.aspose.com/slides/it/cpp).
- Estrai il pacchetto e copia le directory **Include** e **Bin** nella cartella **Examples**.
- Apri il file **main.cpp**, tutti gli esempi vengono richiamati da qui.
- Decommenta gli esempi che desideri eseguire all'interno del progetto.
- Compila e costruisci la soluzione

Sentiti libero di contattarci tramite i nostri Forum se hai problemi nell'installazione o nell'esecuzione degli esempi.
## **Contribuisci**
Se desideri aggiungere o migliorare un esempio, ti invitiamo a contribuire al progetto. Tutti gli esempi e i progetti dimostrativi in questo repository sono open source e possono essere usati liberamente nelle tue applicazioni.

Per contribuire, puoi fare fork del repository, modificare il codice sorgente e creare una pull request. Revisioneremo le modifiche e le includeremo nel repository se ritenute utili.
