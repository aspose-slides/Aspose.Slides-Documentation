## **Informazioni su XPS**
La [Specificazione del parser XML](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) è un linguaggio di descrizione di pagina e un formato di documento fisso originariamente sviluppato da Microsoft. Come il PDF, XPS è un formato di documento a layout fisso progettato per preservare la fedeltà del documento e fornire un aspetto indipendente dal dispositivo.
## **XPS in Aspose.Slides per C++**
Qualsiasi documento di presentazione che può essere caricato da Aspose.Slides per C++ può essere convertito nel formato XPS. Aspose.Slides per C++ utilizza il motore di layout e rendering ad alta fedeltà per produrre output nel formato di documento XPS a layout fisso.
Vale la pena menzionare che Aspose.Slides per C++ genera direttamente XPS senza dipendere dalle classi Windows Presentation Foundation (WPF) incluse nel Framework C++ 3.5, consentendo così ad Aspose.Slides per C++ di produrre documenti XPS su macchine che eseguono versioni del Framework C++ precedenti alla 3.5. È possibile apprendere come esportare i documenti di presentazione in documenti XPS tramite Aspose.Slides per C++ in [questo argomento](/slides/it/net/).
