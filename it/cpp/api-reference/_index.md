{{% alert color="primary" %}} 

Aspose.Slides for C++ è una libreria di classi che consente alle applicazioni C++ di leggere e scrivere documenti di presentazione senza utilizzare Microsoft PowerPoint® o altri. Aspose.Slides for C++ permette di gestire testo, forme, tabelle e animazioni, aggiungere audio e video alle diapositive, visualizzare l'anteprima delle diapositive, esportare le diapositive in formato SVG, PDF e altro.

L'ultima documentazione API è disponibile su [Aspose.Slides for C++ API Reference](https://reference.aspose.com/slides/it/cpp/).
