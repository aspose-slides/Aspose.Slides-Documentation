{{% alert color="primary" %}} 

JasperReports e JasperServer non dispongono di funzionalità integrate per esportare i report come presentazioni Microsoft PowerPoint, ma con Aspose.Slides per JasperReports, ottieni l'accesso a formati di esportazione aggiuntivi:

- Presentazione Microsoft PowerPoint (PPT)
- Presentazione Microsoft PowerPoint (PPTX)
- HTML
- PDF

{{% /alert %}} 

Per creare documenti in questi formati, Aspose.Slides per JasperReports si basa su una versione incorporata di [Aspose.Slides for Java](https://products.aspose.com/slides/it/java/), la libreria di elaborazione di presentazioni leader di mercato di Aspose. Microsoft PowerPoint non viene utilizzato per generare i documenti.
