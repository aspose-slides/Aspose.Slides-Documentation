Aspose.Slides for JasperReports funziona con:

**Aspose.Slides for JasperReports 17.5 e versioni precedenti:**

- Java 2 Platform Standard Edition 1.4 o superiore.
- JasperReports 2.0, 3.0 o superiore.

**Aspose.Slides for JasperReports 17.6 e versioni successive:**

- Java 2 Platform Standard Edition 6.0 o superiore.
- JasperReports 3.7.2 o superiore.
