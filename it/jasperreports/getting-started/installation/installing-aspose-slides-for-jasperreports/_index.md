Per utilizzare Aspose.Slides per JasperReports dalla tua applicazione, copia aspose.slides.jasperreports.library-xx.x.jar dalla cartella \lib\JasperReports X.X.X - X.X.X del file aspose-slides-xx.x-jasperreports.zip nella directory JasperReports\lib o in una cartella di libreria della tua applicazione. Dopo di ciò, puoi accedere agli esportatori programmaticamente.
