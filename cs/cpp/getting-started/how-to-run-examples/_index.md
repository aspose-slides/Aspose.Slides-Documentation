## **Požadavky na software**
Ujistěte se, že splňujete následující požadavky před stažením a spuštěním příkladů.

1. Visual Studio 2015 nebo novější.
## **Stáhněte Aspose.Slides z GitHubu**
Všechny příklady Aspose.Slides pro C++ jsou hostovány na [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-C).

- Můžete buď naklonovat repozitář pomocí vašeho oblíbeného klienta GitHub, nebo stáhnout ZIP soubor z [zde](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/refs/heads/master.zip).
- Rozbalte obsah ZIP souboru do libovolné složky ve vašem počítači. Všechny příklady jsou umístěny ve složce **Examples**.
- K dispozici je soubor řešení C++ pro Visual Studio.
- Projekty byly vytvořeny ve Visual Studio 2015, ale soubory řešení jsou kompatibilní s Visual Studio 2015 a novějšími.
- Otevřete soubor řešení ve Visual Studio a sestavte projekt.
- Stáhněte si nejnovější verzi Aspose.Slides pro C++ z [Stahovací sekce](https://downloads.aspose.com/slides/cs/cpp).
- Rozbalte vydání a zkopírujte adresáře **Include** a **Bin** do složky **Examples**.
- Otevřete soubor **main.cpp**, odkud jsou volány všechny příklady.
- Odekomentujte příklady, které chcete spustit v rámci projektu.
- Sestavte a zkompilujte řešení

Neváhejte se obrátit na naše fóra, pokud máte jakékoli potíže s nastavením nebo spuštěním příkladů.
## **Přispět**
Pokud chcete přidat nebo vylepšit příklad, povzbuzujeme vás k přispění do projektu. Všechny příklady a ukázkové projekty v tomto repozitáři jsou open source a mohou být volně použity ve vašich vlastních aplikacích.

Pro přispění můžete forkovat repozitář, upravit zdrojový kód a vytvořit pull request. Změny přezkoumáme a pokud budou užitečné, zahrneme je do repozitáře.
