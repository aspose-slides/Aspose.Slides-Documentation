## **Formáty souborů a konverze**
Aspose.Slides pro C++ podporuje většinu formátů dokumentů PowerPoint a také je exportuje do populárních formátů široce používaných pro výměnu mezi organizacemi. Následující tabulka uvádí tyto formáty:

|**Funkce**|**Popis**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/cs/cpp/microsoft-powerpoint-ppt/)|Aspose.Slides pro C++ poskytuje nejrychlejší zpracování tohoto formátu prezentačního dokumentu.|
|[Portable Document Format (PDF)](/slides/cs/cpp/portable-document-format-pdf/)|Podporované souborové formáty lze jedním metodou exportovat do dokumentů Adobe Portable Document Format (PDF).|
|[XML Parser Specification (XPS)](/slides/cs/cpp/xml-parser-specification-xps/)|Všechny podporované souborové formáty lze jedním metodou exportovat do dokumentů XML Parser Specification (XPS).|
|[PresentationML (PPTX, XML)](/slides/cs/cpp/presentationml-pptx-xml/)|Aspose.Slides pro C++ podporuje zpracování formátu OOXML prezentací, také známého jako PresentationML nebo PPTX.|
|[PPTX To HTML Conversion](https://docs.aspose.com/slides/cs/cpp/convert-powerpoint-to-html/)|Aspose.Slides pro C++ podporuje konverzi prezentace do formátu HTML.|

## **Renderování**
Aspose.Slides pro C++ podporuje vysoce věrné vykreslování snímků v prezentačních dokumentech do různých grafických formátů, jak je uvedeno níže:

|**Funkce**|**Popis**|
| :- | :- |
|C++ podporované formáty obrázků|Pomocí Aspose.Slides pro C++ budete moci vykreslovat nejen prezentační snímky, ale také obrázky na snímcích do všech grafických formátů podporovaných C++, jako jsou PNG, BMP a JPEG.|
|SVG formát|Aspose.Slides pro C++ také poskytuje vestavěnou metodu pro export prezentačních snímků do formátů Scalable Vector Graphics (SVG).|

## **Často kladené otázky**

**Musím mít nainstalovaný Microsoft PowerPoint na serveru/PC, aby knihovna fungovala?**

Ne. PowerPoint není vyžadován; Aspose.Slides je samostatný engine pro vytváření, úpravu, konverzi a vykreslování prezentací.

**Jak funguje vícevláknové zpracování? Lze zpracování paralelizovat?**

Je bezpečné zpracovávat různé dokumenty v různých vláknech; stejný [prezentaci](https://reference.aspose.com/slides/cs/cpp/aspose.slides/presentation/) objekt nesmí být používán [více vlákny](/slides/cs/cpp/multithreading/) současně.

**Jsou podporována hesla souborů a šifrování?**

Ano. [Můžete](/slides/cs/cpp/password-protected-presentation/) otevřít šifrované prezentace, nastavit nebo odebrat heslo pro otevření a zápis a zkontrolovat stav ochrany.

**Musím se starat o balíčky fontů v Linux kontejnerech?**

Ano. Doporučuje se nainstalovat běžné balíčky fontů a/nebo explicitně [specifikovat adresáře fontů](/slides/cs/cpp/custom-font/) ve vaší aplikaci, aby se předešlo neočekávaným náhradám.

**Existují omezení ve zkušební verzi?**

V [režimu hodnocení](/slides/cs/cpp/licensing/) je do výstupu přidáno vodoznak a platí určitá omezení; [30denní dočasná licence](https://purchase.aspose.com/temporary-license/) je k dispozici pro testování všech funkcí.

**Je podporován import externích formátů do prezentace (PDF/HTML → PPTX)?**

Ano. Můžete přidat [PDF stránky a HTML obsah](/slides/cs/cpp/import-presentation/) do prezentace, čímž je změníte na snímky.
