## **O PPT**
[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) je formát souboru prezentačního dokumentu, který může být vytvářen, čten, upravován a zapisován různými verzemi Microsoft PowerPoint. Jedná se o binární formát prezentačních dokumentů vyvinutý společností Microsoft. 
## **PPT v Aspose.Slides pro C++**
Aspose.Slides pro C++ dokáže číst soubory PPT vytvořené následujícím softwarem. 

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Obdobně mohou být soubory PPT vytvořené pomocí Aspose.Slides pro C++ čteny výše uvedeným softwarem. 
## **Komplexní podpora pro PPT**
Aspose.Slides pro C++ poskytuje podporu téměř pro všechny funkce související s formátem souboru PPT. Nezahrnuje pouze základní a pokročilé funkce poskytované různými verzemi Microsoft PowerPoint pro manipulaci s PPT dokumenty, ale také některé funkce, které Microsoft PowerPoint ani nepodporuje. Hlavní výhodou použití knihovny Aspose.Slides pro C++ API je snadnost použití při práci s těmito funkcemi. 

Kromě základních úkolů souvisejících s vytvářením, čtením a zápisem souborů PPT existuje několik funkcí, které poskytuje Aspose.Slides pro C++, například: 

- Importovat jiné formáty souborů MS Office jako OLE objekty do PPT dokumentů
- Exportovat PPT dokumenty do formátů PDF, TIFF, XPS
- Exportovat snímky v PPT dokumentech do formátu SVG
- Vykreslit snímek do libovolného formátu obrázku podporovaného C++ frameworkem
- Nastavit velikost snímků v PPT dokumentu
- Spravovat animace tvarů
- Spravovat prezentace
- Formátovat text na snímcích
- Skenovat text z PPT dokumentů
- Zpracovávat tabulky na snímcích
- Automatické kopírování masterů pomocí funkce klonování
