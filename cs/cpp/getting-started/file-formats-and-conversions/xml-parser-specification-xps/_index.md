## **O XPS**
Specifikace XML parseru ([Specifikace XML parseru](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification)) je jazyk pro popis stránek a formát pevných dokumentů původně vyvinutý společností Microsoft. Podobně jako PDF, XPS je formát dokumentu s pevnou stránkou navržený tak, aby zachoval věrnost dokumentu a poskytoval nezávislý vzhled na zařízení.
## **XPS v Aspose.Slides for C++**
Jakýkoli prezentační dokument, který může být načten pomocí Aspose.Slides for C++, může být převeden do formátu XPS. Aspose.Slides for C++ používá vysoce věrný rozvržení stránek a vykreslovací engine k vytvoření výstupu ve formátu pevných dokumentů XPS.
Stojí za zmínku, že Aspose.Slides for C++ přímo generuje XPS bez závislosti na třídách Windows Presentation Foundation (WPF), které jsou součástí .NET Frameworku 3.5, což umožňuje Aspose.Slides for C++ vytvářet XPS dokumenty na počítačích běžících na verzích .NET Frameworku starších než 3.5. Více o exportu prezentačních dokumentů do XPS dokumentů pomocí Aspose.Slides for C++ se dozvíte v [tomto tématu](/slides/cs/net/).
