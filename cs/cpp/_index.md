**Vítejte v Aspose.Slides for C++**

Aspose.Slides for C++ je API pro správu Microsoft PowerPoint® umožňující aplikacím v C++ číst a zapisovat dokumenty PowerPoint® bez použití Microsoft PowerPoint®. Aspose.Slides for C++ je první a jediné komponenta, která poskytuje funkčnost pro správu dokumentů PowerPoint®. Poskytuje mnoho klíčových funkcí, jako je správa textu, tvarů, tabulek a animací, přidávání zvuku a videa do snímků, náhled snímků, export snímků do formátů SVG, PDF a další.
