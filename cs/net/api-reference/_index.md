{{% alert color="primary" %}} 

Aspose.Slides for .NET je knihovna tříd, která umožňuje aplikacím .NET číst a zapisovat prezentační dokumenty bez použití Microsoft PowerPoint® nebo jiných. Aspose.Slides for .NET umožňuje spravovat text, tvary, tabulky a animace, přidávat audio a video do snímků, prohlížet snímky, exportovat snímky do formátů SVG, PDF a další.

Nejnovější referenci API najdete na [Aspose.Slides for .NET API Reference](https://reference.aspose.com/slides/cs/net).

{{% /alert %}}
