## **Úvod**
Microsoft Visual Studio je integrované vývojové prostředí (IDE) od společnosti Microsoft. Používá se k vývoji počítačových programů pro Microsoft Windows, stejně jako webových stránek, webových aplikací a webových služeb. Visual Studio využívá vývojové platformy Microsoftu, jako jsou Windows API, Windows Forms, Windows Presentation Foundation, Windows Store a Microsoft Silverlight. Umí vytvářet jak nativní kód, tak řízený kód.

### **Aspose.Slides pro .NET**
Aspose.Slides pro .NET je jedinečné API pro správu souborů PowerPoint®, které umožňuje .NET aplikacím číst, zapisovat a manipulovat s dokumenty PowerPoint bez použití Microsoft PowerPoint.

Aspose.Slides pro .NET je první a jediný .NET PowerPoint API, který poskytuje funkce pro správu dokumentů PowerPoint ve vašich vlastních aplikacích. Stejně jako všechny komponenty Aspose .NET je Aspose.Slides napsáno v řízeném C#, má neuvěřitelně výhodnou cenu a je bleskově rychlé.

Pro více informací o [Aspose.Slides pro .NET API](http://www.aspose.com/.net/powerpoint-component.aspx)

### **Prozkoumejte příklady Aspose.Slides pro .NET pomocí Visual Studio**
Plugin Aspose Visual Studio – **Explore Aspose.Slides for .NET Examples** je skvělý nástroj pro rychlé stažení a prozkoumání příkladů API Aspose.Slides pro .NET. Šetří spoustu času a úsilí tím, že poskytuje velmi jednoduchou možnost plynule vybrat, stáhnout a otevřít nejnovější ukázkové projekty, aniž byste opustili Visual Studio.

This version provides the following features:

- Podporuje Visual Studio 2010, Visual Studio 2012 a Visual Studio 2013.
- Snadno spustitelné z nabídky **File > Aspose** ve Visual Studio.
- Šetří čas a snižuje křivku učení.
- Umožňuje vybrat a otevřít nejnovější příklady API Aspose.Slides pro .NET hostované na Github.com.
- Umožňuje otevřít projekty s příklady v **C#** nebo **VB.NET**.
- Vybraný projekt je automaticky otevřen ve Visual Studio se všemi potřebnými odkazy nastavenými tak, aby vám poskytl připravené prostředí pro spuštění a prozkoumání.
- Automaticky stahuje nejnovější knihovny API z **NuGet** a příklady z **GitHub**.

## **Instalace**
Instalace pluginu Aspose Visual Studio je velmi jednoduchá, postupujte podle následujících kroků

- Dvakrát klikněte na stažený soubor msi
- Postupujte podle kroků, můžete jednoduše stisknout Next na každé obrazovce a nainstalovat pomocí standardních nastavení.

Po dokončení instalace se zobrazí dialog s potvrzením úspěchu.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_2)

**Poznámka:** Ujistěte se, že jste restartovali Visual Studio, aby se změny projevily.

## **Použití**
Po instalaci můžete snadno spustit a používat interaktivní plugin podle níže uvedeného postupu:

**1.** Otevřete plugin výběrem **File > Aspose** a následně **New Aspose.Slides Example Project** ve Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_3)

**2.** Při prvním otevření pluginu se zobrazí dialog pro výběr API, jak je vidět níže. Stáhnutí nejnovějšího API a jejich knihoven z **NuGet** a nejnovějšího zdrojového kódu příkladů z **GitHub** bude trvat několik minut.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_4)

**3.** V dialogu Vyberte API pro zobrazení příkladů vyberte nebo změňte umístění, kde budou vytvořeny projekty s příklady, a také vyberte nebo změňte typ projektu příkladů na **C#** nebo **VB.NET** a klikněte na **Continue**, aby se otevřel projekt s příklady ve Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_1)

**4.** Nové projekty s příklady podle výběru typu projektu **C#** nebo **VB.NET** ve Visual Studio.

#### **Jak nakonfigurovat zdrojový kód**
Pro otevření a rozšíření zdrojového kódu musíte mít nainstalováno následující

- Visual Studio 2010

Postupujte podle těchto jednoduchých kroků, abyste začali

1. Stáhněte/klonujte zdrojový kód.
1. Otevřete Visual Studio 2010 a vyberte **File** > **Open Project**
1. Procházejte k nejnovějšímu staženému zdrojovému kódu a otevřete **AsposeVisualStudioPlugin.sln**

#### **Přehled zdrojového kódu**
Řešení obsahuje dva projekty

- **AsposeVisualStudioPlugin** – Obsahuje integraci s Visual Studio, výběr a automatické stažení API Aspose a příkladů.
- **Installer** – Projekt instalátoru Wix pro vytvoření souboru msi instalátoru pluginu.
