## **Co je VSTO?**
VSTO, neboli Visual Studio Tools for Office, nebo Microsoft Visual Studio Tools, je součástí sady nástrojů Microsoft Visual Studio .NET a podporuje Microsoft .NET Framework. Používají jej vývojáři k psaní kódu, který propojuje aplikace Microsoft Office Word, Excel a Outlook. Vývojáři používající VSTO mohou využívat jazyky C# nebo Visual Basic. Jedná se o alternativu k Visual Basic for Applications, tedy VBA.
## **Aspose.Slides pro .NET**
Aspose.Slides pro .NET je jedinečná komponenta pro správu PowerPoint® umožňující .NET aplikacím číst, zapisovat a manipulovat s dokumenty PowerPoint bez použití Microsoft PowerPoint.

Aspose.Slides pro .NET je první a jediná .NET komponenta, která poskytuje funkce pro správu dokumentů PowerPoint ve vašich vlastních aplikacích. Stejně jako všechny komponenty Aspose .NET je Aspose.Slides napsán v řízeném C#, je neuvěřitelně cenově výhodný a bleskově rychlý.
## **Aspose.Slides pro VSTO**
## **Úvod**
Vývojáři hledající knihovny pro práci se souborovými formáty mohou potřebovat projít funkce, které poskytují knihovny dostupné na trhu. Hlavně jsou při výběru zvažovány dvě faktory:

- Funkce
- Námaha potřebná k použití knihovny

Pokud jste .NET vývojář a hledáte rychlejší, snadnou a nenáročnou knihovnu pro zpracování souborů, můžete potřebovat porovnat VSTO (Visual Studio Tools for Office) s Aspose. V takovém případě vám tento projekt pomůže porovnat funkce a kód v obou knihovnách.
## **Systémové požadavky a podporované platformy**
### **Systémové požadavky**
One or more of the following editions of Visual Studio Tools for Office:

- Microsoft Visual Studio 2005 Tools for the Microsoft Office System
- Microsoft Visual Studio 2005 Tools for the 2007 Microsoft Office System
- Visual Studio 2008 Professional Edition
- Visual Studio 2008 Team Suite Edition
- Jedna verze Microsoft Office:
- Microsoft Office Professional 2003 SP1
- Microsoft Office 2007
### **Podporované operační systémy**
Windows 2000; Windows Server 2003; Windows Vista; Windows XP
### **Podporované verze .NET Framework**
pouze .NET Framework 2.0 nebo vyšší.
