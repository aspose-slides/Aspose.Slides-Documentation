## **Co je OpenXML?**
OpenXML SDK je .NET SDK od společnosti Microsoft určené k zpracování souborových formátů Microsoft Office 2007 a novějších. Tyto souborové formáty jsou v podstatě zipované balíčky a Open XML SDK pro Office zjednodušuje úlohu manipulace s Open XML balíčky a podkladovými schématy Open XML v balíčku. Třídy v Open XML SDK zapouzdřují mnoho běžných úkolů, které vývojáři provádějí s Open XML balíčky, takže můžete provádět složité operace pomocí několika řádků kódu.

Po instalaci Open XML SDK 2.5 otevřete svůj existující projekt nebo aplikaci ve Visual Studiu, nebo vytvořte nový projekt či aplikaci. Poté ve svém projektu nebo aplikaci přidejte odkazy na následující komponenty:

- DocumentFormat.OpenXml
- WindowsBase
## **Aspose.Slides pro .NET**
Aspose.Slides pro .NET je jedinečná komponenta pro správu PowerPoint® umožňující .NET aplikacím číst, zapisovat a manipulovat s PowerPoint dokumenty bez použití Microsoft PowerPoint.

Aspose.Slides pro .NET je první a jediné .NET komponenta, která poskytuje funkčnost pro správu PowerPoint dokumentů ve vašich vlastních aplikacích. Stejně jako všechny Aspose .NET komponenty, Aspose.Slides je napsána v managed C#, je neuvěřitelně cenově výhodná a bleskově rychlá.
## **Aspose.Slides pro OpenXML**
## **Úvod**
Vývojáři hledající knihovny pro formáty souborů mohou potřebovat projít funkce nabízené knihovnami dostupnými na trhu. Hlavně se při výběru možnosti zvažují dva faktory:

- Funkce
- Náročnost použití knihovny

Pokud jste .NET vývojář hledající rychlejší, snadnou a lehkou knihovnu pro práci s formáty souborů, můžete potřebovat porovnat VSTO (Visual Studio tool for Office) s Aspose. V tomto případě vám tento projekt pomůže porovnat funkce a kód v obou knihovnách.
