{{% alert color="primary" %}}

Aspose.Slides for Node.js via Java je knihovna tříd, která umožňuje aplikacím Node.js číst a zapisovat prezentační dokumenty bez použití Microsoft PowerPoint® nebo jiných. Aspose.Slides for Node.js via Java vám umožňuje spravovat text, tvary, tabulky a animace, přidávat audio a video do snímků, náhled snímků, exportovat snímky do formátu SVG, PDF a mnoho dalšího.

Nejnovější referenční API najdete na [Aspose.Slides for Node.js via Java API Reference](https://reference.aspose.com/slides/cs/nodejs-java/).
