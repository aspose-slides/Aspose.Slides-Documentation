## **Podporované platformy**
Aspose.Slides for Node.js via Java je platformově nezávislé API. Lze jej použít na jakékoli platformě (Windows, Linux, macOS atd.), kde je nainstalováno Node.js 7 nebo vyšší. Dalším požadavkem je Oracle JDK 8 nebo vyšší.

## **FAQ**

**Potřebuji nainstalovat Microsoft PowerPoint na server/PC, aby knihovna fungovala?**

Ne. PowerPoint není vyžadován; Aspose.Slides je samostatný engine pro vytváření, úpravu, konverzi a renderování prezentací.

**Jak funguje multithreading? Lze zpracování paralelizovat?**

Je bezpečné zpracovávat různé dokumenty v různých vláknech; stejný [prezentace](https://reference.aspose.com/slides/cs/nodejs-java/aspose.slides/presentation/) objekt nesmí být používán [více vláknech](/slides/cs/nodejs-java/multithreading/) současně.

**Jsou podporována hesla souborů a šifrování?**

Ano. [Můžete](/slides/cs/nodejs-java/password-protected-presentation/) otevřít šifrované prezentace, nastavit nebo odstranit heslo pro otevření a zápis a zkontrolovat stav ochrany.

**Musím se starat o balíčky fontů v Linuxových kontejnerech?**

Ano. Doporučuje se nainstalovat běžné balíčky fontů a/nebo výslovně [specifikovat adresáře fontů](/slides/cs/nodejs-java/custom-font/) ve vaší aplikaci, aby se předešlo neočekávaným náhradám.

**Existují omezení ve zkušební verzi?**

V [zkušebním režimu](/slides/cs/nodejs-java/licensing/) je do výstupu přidán vodoznak a platí určitá omezení; [30denní dočasná licence](https://purchase.aspose.com/temporary-license/) je k dispozici pro testování všech funkcí.

**Je podporováno importování externích formátů do prezentace (PDF/HTML → PPTX)?**

Ano. Můžete přidat [PDF stránky a HTML obsah](/slides/cs/nodejs-java/import-presentation/) do prezentace, čímž je převedete na snímky.
