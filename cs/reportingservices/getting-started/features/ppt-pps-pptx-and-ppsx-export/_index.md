{{% alert color="primary" %}} 

Microsoft SQL Server Reporting Services nemá vestavěné možnosti exportovat zprávy jako prezentace Microsoft PowerPoint, ale po instalaci Aspose.Slides for Reporting Services na vašem serveru získáte přístup ke dvěma dalším exportním formátům: 

- Microsoft PowerPoint Presentation (PPT)
- Microsoft PowerPoint SlideShow (PPS)
- Microsoft PowerPoint 2007 Presentation (PPTX)
- Microsoft PowerPoint 2007 SlideShow (PPSX)

{{% /alert %}} 

Pro vytváření dokumentů v těchto formátech Aspose.Slides for Reporting Services používá vestavěnou verzi [Aspose.Slides for .NET](http://www.aspose.com/Products/Aspose.Slides/), předního komponentu pro zpracování prezentací na trhu od společnosti Aspose. Microsoft PowerPoint není při generování dokumentů používán. 

Kopie PPT a PPS stejné zprávy budou vypadat identicky, díky vysoké úrovni podpory podkladových formátů souborů poskytované společností Aspose.Slides.
