## **Podpora licencí**
{{% alert color="primary" %}} 

Můžete si stáhnout zkušební verzi **Aspose.Slides for Reporting Services** ze [stránky vydání](https://releases.aspose.com/slides/cs/reportingservices/). Zkušební verze poskytuje stejné funkce jako licencovaná verze produktu. Zkušební balíček je stejný jako zakoupený balíček. Zkušební verze se jednoduše stane licencovanou po přidání několika řádků kódu (pro aplikaci licence).

Jakmile budete s vyzkoušením produktu spokojeni, můžete [zakoupit licenci](https://purchase.aspose.com/buy). Doporučujeme projít různé typy předplatného. Pokud máte otázky, kontaktujte prodejní tým Aspose.

{{% /alert %}} 

## **Licencování v Aspose.Slides for Reporting Service**

* Zkušební verze se stane licencovanou po zakoupení licence a přidání několika řádků kódu (pro aplikaci licence).
* Licence je soubor XML v prostém textu, který obsahuje údaje jako název produktu, počet vývojářů, pro které je licence určena, datum vypršení předplatného a podobně. 
* Soubor licence je digitálně podepsaný, takže jej nesmíte měnit. I neúmyslné přidání dalšího řádku do obsahu souboru jej zneplatní.
* Aby se předešlo omezením spojeným se zkušební verzí, musíte nastavit licenci před použitím Aspose.Slides for Reporting Service. 

Stáhněte licenci do počítače a zkopírujte ji do složky **C:\Program Files\Microsoft SQL Server\<Instance>\Reporting Services\ReportServer\bin**, kde je nainstalován **Aspose.Slides.ReportingServices.dll**. 

Pro ověření, že byla licence nainstalována správně, exportujte libovolnou sestavu jako prezentaci Microsoft PowerPoint. Pokud dokument neobsahuje vodoznak, licence byla úspěšně aktivována.
