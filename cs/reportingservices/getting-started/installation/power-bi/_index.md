## **Power BI**
Aspose.Slides for Reporting Services můžete nainstalovat na server Power BI stejnými instalačními postupy:

Viz

* [Install with MSI Installer](https://docs.aspose.com/slides/cs/reportingservices/install-with-msi-installer/#installation). Instalační program automaticky zjistí nainstalovaný Power BI na serveru, nainstaluje produkt a následně jej nakonfiguruje pro vaše pracovní prostředí. 
* [Install Manually](https://docs.aspose.com/slides/cs/reportingservices/install-manually/).

{{% alert title="Note" color="warning" %}} 

**Aspose.Slides for Reporting Services** vyžaduje instalaci **.NET Framework 3.5** na hostitelském počítači. 

{{% /alert %}}

Aspose.Slides for SSRS podporuje paginované zprávy (cdl).
