## **Popis produktu**

Aspose.Slides for Python via Java je multiplatformní knihovna tříd, která umožňuje vašim aplikacím provádět širokou škálu úkolů zpracování dokumentů bez použití Microsoft PowerPoint®.

### **Proč byste měli používat Aspose.Slides for Python via Java**

- Aspose.Slides for Python via Java je první a jedinou komponentou, která poskytuje funkce potřebné pro správu dokumentů PowerPoint®.
- Aspose.Slides for Python via Java nabízí mnoho klíčových funkcí, jako je správa textů, tvarů, tabulek a animací, přidávání zvuku a videa do snímků, náhled snímků, export snímků do SVG, PDF a dalších formátů a mnoho dalších možností.
- Aspose.Slides for Python via Java vám nejen umožní otevřít soubory PowerPoint z různých zdrojů, ale také vám umožní ukládat prezentace různými způsoby. Například můžete uložit své prezentace ve formátu PPT; své snímky můžete také uložit jako obrázky.
- Aspose.Slides for Python via Java poskytuje téměř všechny funkce, které můžete (ne)naleznout v Microsoft PowerPoint. Díky všestrannosti Aspose.Slides for Python via Java můžete kromě tabulek a různých typů tvarů přidávat do snímků různé typy rámců – textové, zvukové a video rámce.

## **Funkce Aspose.Slides**

Funkce Aspose.Slides for Python via Java lze rozdělit do následujících skupin:

- Prezentační funkce.
- Funkce snímků.
- Grafické a multimediální funkce.
- Funkce tabulek.
- Funkce textu a odstavců.
- Pokročilé funkce

### **Prezentační funkce**

- Otevírání prezentací Microsoft PowerPoint z různých zdrojů.
- Ukládání prezentací v různých formátech Microsoft PowerPoint.
- Vytváření prezentací od začátku.
- Převod prezentací do PDF dokumentů bez použití jiných komponent.
- Čtení nebo úprava vlastností dokumentu prezentace.
- Aplikace ochrany na prezentaci.
- Export prezentace do PDF.
- Export prezentace do TIFF.
- Export prezentace do XPS.
- Export prezentace do SWF.
- Export poznámek prezentace do PDF/Tiff.
- Export prezentace do HTML.

### **Funkce snímků**

- Vytváření nových snímků od nuly a jejich přidávání do prezentací.
- Přístup k libovolnému snímku podle jeho pozice v prezentaci.
- Kopírování nebo klonování snímků do stejné nebo jiné prezentace.
- Úprava a mazání snímků.
- Automatické kopírování hlavních snímků (master) spolu se snímky.
- Náhled snímků.
- Získávání nebo nastavení názvu snímku.
- Ukládání snímků jako obrázky.
- Export snímků do formátu SVG.
- Práce s více hlavními snímky v prezentacích.
- Vyplnění pozadí snímků libovolným podporovaným stylem.
- Přidávání nových nebo správa poznámek ke snímkům.
- Dynamické nastavení nových hlavních snímků a stylů.
- Přidávání nebo úprava přechodů prezentace.
- Přidávání Morph přechodů

### **Grafické a multimediální funkce**

- Vytváření nebo správa tvarů, jako jsou čáry, obdélníky, elipsy a polylinie.
- Správa stylů čar v tvarech.
- Získání alternativního textu tvaru.
- Rotace a převracení tvarů.
- Vyplňování tvarů různými styly – například plná barva, gradient, vzor, textura a obrázek.
- Použití některého ze 48 předdefinovaných vzorů a 24 textur.
- Přiřazení odkazů k tvarům.
- Čtení, úprava a přidávání nových animačních efektů ke snímkům a tvarům.
- Přidávání nových obrázků do prezentací a správa existujících.
- Přidávání nových obrázkových odrážek do prezentací a správa existujících.
- Přidávání nových audio rámců do prezentací a správa existujících.
- Přidávání nových video rámců do prezentací a správa existujících.

### **Funkce tabulek**

- Vytváření tabulek od nuly a jejich přidávání do snímků.
- Změna velikosti a přesouvání tabulek.
- Přístup k libovolné buňce v tabulce.
- Změna formátu a ohraničení libovolné buňky.
- Změna velikosti sloupců a řádků.
- Přidávání a mazání sloupců a řádků.
- Nastavení šířky a barvy okraje celé tabulky.

### **Funkce textu a odstavců**

- Získávání a nastavení textů v textových zástupcích a textových rámcích.
- Získání typu zástupce.
- Změna formátování textu v textovém rámci.
- Přidávání nových fontů nebo správa již použitého fontu v prezentaci.
- Změna textů a formátování textu.
- Vytváření nebo správa odkazů v textu.
- Podpora tabulátorů a efektivních tabulátorů.
- Přidávání nových textových rámců téměř k jakémukoli typu tvaru.
- Přístup a úprava okrajů a nastavení zalamování textu v textových rámcích.
- Přístup a úprava odsazení odstavců.

### **Pokročilé funkce**

- Přístup k OLE objektům vloženým v prezentacích.
- Zamčení prezentací.

## **Technická podpora**

Aspose poskytuje neomezenou bezplatnou technickou podporu pro všechny své produkty. Podpora je dostupná všem uživatelům (včetně uživatelů s hodnotícími balíčky). Pokud potřebujete pomoc s Aspose.Slides for Python via Java, zvažte následující:

- Hlavním kanálem podpory je [Aspose Forums](https://forum.aspose.com/). Zeptejte se ve [Aspose.Slides forum](https://forum.aspose.com/c/slides/cs/11)—a na vaši otázku bude odpovězeno během několika hodin. Tým podpory Aspose.Slides přímo odpovídá na otázky zveřejněné ve fóru.
- Vezměte prosím na vědomí, že Aspose neposkytuje technickou podporu po telefonu. Telefonická podpora je k dispozici pouze pro otázky týkající se prodeje a nákupu.
- Při čekání na odpověď ve fórech buďte prosím trpěliví a zohledněte časové rozdíly.

Pokud máte problém s Aspose.Slides for Python via Java, postupujte podle těchto pokynů, abyste jej vyřešili co nejefektivněji:

  Viz [Aspose.Slides download](https://releases.aspose.com/slides/cs/python-java/) pro získání informací o nejnovější verzi produktu.

- Před nahlášením problému můžete projít naše fóra, tuto dokumentaci a referenci API. Možná, že odpověď na vaši otázku již byla poskytnuta.

  Svůj dokument můžete bezpečně připojit v Aspose.Forums, protože k přiloženým souborům mají přístup pouze vy a vývojáři Aspose.

- Vytvořte a používejte jedno vlákno pro konkrétní problém. Pokud se setkáte s dalším problémem, musíte vytvořit další vlákno pro jeho hlášení.
