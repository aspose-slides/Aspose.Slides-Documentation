## **Požadavky na server**
{{% alert color="primary" %}} 

Pro instalaci Aspose.Slides pro SharePoint je vyžadován systém, který používá jedno z následujících:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft Office SharePoint Server 2010
- Microsoft Office SharePoint Server 2013
- Microsoft Office SharePoint Server 2016
- Microsoft Office SharePoint Server 2019

{{% /alert %}} {{% alert color="primary" %}} 

Instalace Service Pack 2 nebo novější verze pro tyto produkty se doporučuje.

{{% /alert %}} 
## **Požadavky na klienta**
{{% alert color="primary" %}} 

Webové stránky SharePoint jsou přístupné prostřednictvím webového prohlížeče. Aspose.Slides pro SharePoint byla testována s následujícími prohlížeči:

- Microsoft Internet Explorer 6, 7, 8
- Mozilla FireFox 3.5

{{% /alert %}}
