{{% alert color="primary" %}} 

S Aspose.Slides for SharePoint můžete převádět dokumenty mezi mnoha populárními formáty kancelářských dokumentů přímo v knihovně dokumentů SharePoint. Převody jsou prováděny s vysokou věrností a přesností. 

{{% /alert %}} 
## **Podporované vstupní formáty**
Aspose.Slides for SharePoint podporuje následující vstupní formáty: 

- PPT – Microsoft PowerPoint prezentace 97 - 2003
- PPS – Microsoft PowerPoint prezentace 97 - 2003
- POT – Microsoft PowerPoint šablona 97 - 2003
- PPTX – Office Open XML prezentace
- PPSX – Office Open XML prezentace
- POTX – Office Open XML šablona

{{% alert color="primary" %}} 

Pro generování dokumentů Aspose.Slides for SharePoint využívá vestavěnou verzi [Aspose.Slides for .NET](http://www.aspose.com/categories/.net-components/aspose.slides-for-.net/default.aspx), jedinou komponentu Aspose pro zpracování dokumentů PowerPoint. 

{{% /alert %}}
## **Podporované výstupní formáty**
Následující výstupní formáty jsou podporovány Aspose.Slides for SharePoint: 

- PDF – Portable Document Format
- TIFF – Balíček obrázků
- XPS – XML Paper Specification
