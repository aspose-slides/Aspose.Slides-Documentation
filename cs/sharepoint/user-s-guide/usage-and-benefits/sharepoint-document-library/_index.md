Knihovna je struktura obsahu, která obsahuje soubory. Knihovna může obsahovat jen jeden typ souboru nebo mnoho různých typů, včetně dokumentů, tabulek a prezentací. Knihovna SharePoint vytváří jediné místo, kde mohou lidé nahrávat a přistupovat k dokumentům. Dokumenty jsou uloženy v binárním formátu v databázi SQL Server, která je používána webem SharePoint. Dokumenty jsou ostatním uživatelům SharePointu k dispozici jako seznam hypertextových odkazů.
