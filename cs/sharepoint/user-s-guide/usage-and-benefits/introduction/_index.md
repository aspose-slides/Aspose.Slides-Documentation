Weby SharePoint jsou určeny pro spolupráci, komunikaci a ukládání obsahu s cílem naplnit konkrétní obchodní potřeby. Umožňují skupině lidí dosáhnout společného cíle tím, že usnadňují sdílení informací a komunikaci. Informace jsou nejčastěji dostupné jako webové stránky a jsou přístupné prostřednictvím webového prohlížeče. 

{{% /alert %}} 
## **Co je SharePoint a Aspose.Slides?**
Aspose.Slides pro SharePoint je flexibilní řešení pro převod dokumentů Microsoft PowerPoint do jiných formátů, jako jsou PPT, POT, PPS, PPTX, PPSX a POTX. Aspose.Slides pro SharePoint vám umožňuje číst a převádět soubory PowerPoint v aplikaci SharePoint bez použití Microsoft PowerPoint.
