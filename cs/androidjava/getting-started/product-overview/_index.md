## **Vítejte v Aspose.Slides pro Android přes Java!**

Aspose.Slides pro Android přes Java je API pro správu Microsoft PowerPoint® umožňující Java aplikacím číst a zapisovat dokumenty PowerPoint® bez Microsoft PowerPoint®.

Aspose.Slides pro Android přes Java je první a jediné komponenta, která poskytuje funkce potřebné ke správě dokumentů PowerPoint®.

Aspose.Slides pro Android přes Java nabízí řadu klíčových funkcí, jako je správa textů, tvarů, tabulek a animací, přidávání audia a videa do snímků, náhled snímků, export snímků do SVG, PDF a dalších formátů.

## **Popis produktu**

Aspose.Slides pro Android přes Java vám nejen umožní otevírat soubory PowerPoint z různých zdrojů, ale také vám umožní prezentace uložit různými způsoby. Například můžete prezentace uložit ve formátu PPT; můžete také uložit snímky jako obrázky.

Aspose.Slides pro Android přes Java vám poskytuje téměř všechny funkce, které můžete najít v Microsoft PowerPoint. Vzhledem k univerzálnosti Aspose.Slides pro Android přes Java můžete kromě tabulek a různých druhů tvarů také přidávat různé typy rámců — textové, audio a video rámce — do vašich snímků.

Aspose.Slides pro Android přes Java také poskytuje plnohodnotné ukázky a funkční příklady pro vývojáře, aby získali lepší představu o API.

## **Často kladené otázky**

**Podporuje úrovně souladu PDF pro archivaci a přístupnost (PDF/A a PDF/UA)?**

Ano. Můžete uložit do PDF s PDF/A-2a/2b/2u, PDF/A-3a/3b, stejně jako PDF/UA konfigurací [PDF export options](https://reference.aspose.com/slides/cs/androidjava/com.aspose.slides/pdfoptions/).

**Existuje mechanismus pro náhradu fontů a podpora vlastních fontů pro zajištění správného vykreslování?**

Ano. Knihovna vám umožňuje [load custom fonts](/slides/cs/androidjava/custom-font/) a [define fallback rules](/slides/cs/androidjava/fallback-font/), takže chybějící glyfy jsou spolehlivě nahrazeny během konverze a vykreslování.

**Mohu zjistit, zda je soubor chráněn heslem, aniž bych ho plně otevřel?**

Ano. Můžete [inspect a presentation](/slides/cs/androidjava/examine-presentation/), abyste zjistili, zda vyžaduje heslo před načtením celého dokumentu.

**Je pro zpracování a konverze nutný Microsoft PowerPoint?**

Ne. Aspose.Slides je samostatný engine; PowerPoint není potřeba na serveru ani na pracovním stanici.

**Je bezpečné zpracovávat prezentace z více vláken?**

Ano, můžete zpracovávat různé dokumenty paralelně v samostatných vláknech; jen se vyhněte současnému používání stejné instance prezentace [across threads](/slides/cs/androidjava/multithreading/).

**Jsou makra zachována a mohu spravovat VBA v souborech PPTM/PPSM?**

Ano. Prezentace s makry [are supported](/slides/cs/androidjava/presentation-via-vba/), a můžete [inspect and manage VBA projects](https://reference.aspose.com/slides/cs/androidjava/com.aspose.slides/presentation/#getVbaProject--) v těchto souborech.

**Mohu převést PDF nebo HTML zpět do snímků PowerPoint?**

Ano. Můžete [import PDF pages or HTML content](/slides/cs/androidjava/import-presentation/), abyste vytvořili nebo naplnili snímky v prezentaci.

**Je export do XPS podporován a mohu řídit kvalitu a obsah výstupu XPS?**

Ano. [Export to XPS](/slides/cs/androidjava/convert-powerpoint-to-xps/) je k dispozici a [save options](https://reference.aspose.com/slides/cs/androidjava/com.aspose.slides/xpsoptions/) vám umožní ladit kvalitu výstupu a zahrnutý obsah.

**Mohu převést snímky na obrázky a řídit kvalitu výstupu?**

Ano. Snímky mohou být [rendered to PNG, JPEG, GIF, BMP, TIFF](/slides/cs/androidjava/convert-powerpoint-to-png/) a další formáty s podrobnou kontrolou velikosti a kvality.

**Jsou podporovány matematické rovnice (MathML/MathText) a lze je exportovat?**

Ano. Můžete [create and edit mathematical text](/slides/cs/androidjava/powerpoint-math-equations/) a [export equations](/slides/cs/androidjava/exporting-math-equations/), včetně do MathML.

**Jak knihovna zachází s šifrováním a hesly (otevření, nastavení, odstranění)?**

[It supports](/slides/cs/androidjava/password-protected-presentation/) otevírání šifrovaných prezentací, nastavení nebo odstranění hesel pro otevření a úpravy a kontrolu stavu ochrany souboru.
