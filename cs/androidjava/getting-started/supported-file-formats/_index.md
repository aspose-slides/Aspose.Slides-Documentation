## **Přehled**

Aspose.Slides podporuje soubory prezentací z Microsoft PowerPoint 97 až po Office 365, včetně Microsoft PowerPoint pro Mac. Tento článek uvádí verze PowerPointu podporované knihovnou a poskytuje tabulku formátů souborů, které lze načíst, uložit nebo obojí.

Článek také odpovídá na běžné otázky týkající se shody PDF, vkládání písem, souborů chráněných heslem, vlastních písem, náhradních písem a možností exportu do XPS.

## **Podporované verze Microsoft PowerPoint**

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint pro MAC
- Office 365

## **Často kladené otázky**

**Mohu ukládat prezentace do PDF, které splňují archivní a přístupové standardy (PDF/A a PDF/UA)?**

Ano. Aspose.Slides podporuje export do PDF s úrovněmi shody, jako jsou PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b, a také PDF/UA pomocí nastavení [shoda](https://reference.aspose.com/slides/cs/androidjava/com.aspose.slides/pdfoptions/#setCompliance-int-) v [Možnosti exportu PDF](https://reference.aspose.com/slides/cs/androidjava/com.aspose.slides/pdfoptions/).

**Podporuje knihovna vkládání písem při exportu do PDF s podrobnou kontrolou toho, co je vkládáno?**

Ano. Můžete řídit, zda jsou písma plně vložena nebo podmnožena (pouze použité glyfy), určit, jak se zachází se společnými systémovými písmy, a konfigurovat chování pro ASCII text pomocí [Možnosti exportu PDF](https://reference.aspose.com/slides/cs/androidjava/com.aspose.slides/pdfoptions/).

**Mohu zjistit, jestli je soubor chráněn heslem, před tím, než jej skutečně načtu?**

Ano. Pomocí [API pro inspekci založenou na továrně](https://reference.aspose.com/slides/cs/androidjava/com.aspose.slides/presentationfactory/) můžete dotazovat soubor prezentace a zjistit, zda je chráněn heslem, aniž byste jej plně otevřeli.

**Existují mechanismy náhradního písma a podpora vlastních písem?**

Ano. Knihovna podporuje [načítání](/slides/cs/androidjava/custom-font/) a [vkládání](/slides/cs/androidjava/embedded-font/) vlastních písem a poskytuje [pravidla náhradního písma](/slides/cs/androidjava/fallback-font/), aby se zabránilo chybějícím glyfům během vykreslování a konverze.

**Mohu exportovat snímky do XPS a jsou k dispozici možnosti ladění výstupu XPS?**

Ano. [Export do XPS](/slides/cs/androidjava/convert-powerpoint-to-xps/) je podporován a můžete upravit příslušné [možnosti uložení](https://reference.aspose.com/slides/cs/androidjava/com.aspose.slides/xpsoptions/), abyste řídili kvalitu a obsah dokumentu XPS.
