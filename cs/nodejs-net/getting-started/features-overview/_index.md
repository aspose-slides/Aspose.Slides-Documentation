## **Podporované platformy**
Aspose.Slides for Node.js via .NET je platformně nezávislé API. Lze jej použít na jakékoli platformě (Windows, Linux, macOS atd.), kde je nainstalován Node.js verze 7 nebo novější. Dalším požadavkem je Oracle JDK verze 8 nebo novější.

## **Formáty souborů a konverze**
Aspose.Slides for Node.js via .NET podporuje většinu formátů dokumentů PowerPoint. Také vám umožňuje exportovat je do populárních formátů souborů, které organizace používají a sdílejí mezi sebou. Projděte si následující podrobnosti:

|**Funkce**|**Popis**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via .NET poskytuje nejrychlejší zpracování tohoto formátu prezentačního dokumentu.|
|PPT to PPTX conversion|Aspose.Slides for Node.js via .NET podporuje konverzi PPT na PPTX.|
|Portable Document Format (PDF)|Můžete exportovat všechny podporované formáty souborů do dokumentů Adobe Portable Document Format (PDF) pomocí jedné metody.|
|XML Parser Specification (XPS)|Můžete exportovat všechny podporované formáty souborů do dokumentů XML Parser Specification (XPS) pomocí jedné metody.|
|Tagged Image File Format (TIFF)|Můžete exportovat všechny podporované formáty prezentačních souborů do formátu Tagged Image File Format (TIFF).|
|PPTX To HTML Conversion|Aspose.Slides for Node.js via .NET podporuje konverzi PresentationEx do formátu HTML.|

|**Funkce**|**Popis**|

Aspose.Slides for Node.js via .NET vám umožňuje přistupovat, upravovat nebo vytvářet téměř všechny položky či obsah prezentačních dokumentů. Projděte si následující podrobnosti:

|**Funkce**|**Popis**|
| :- | :- |

|Normal Slides|S Aspose.Slides for Node.js via .NET můžete vytvářet nové snímky různých typů; také můžete přistupovat a upravovat existující snímky v prezentacích.|

Protože snímek dědí své rozvržení ze vzorového snímku, vestavěné metody klonování automaticky zkopírují vzor při klonování.|
|Managing Slides sections|Metody pro uspořádání snímků do různých sekcí v rámci prezentace.|
|Place Holders and Text Holders|Můžete přistupovat ke zástupcům místa a textovým zástupcům ve snímku. Navíc můžete vytvořit snímek s textovými zástupci od nuly pomocí příslušné metody.|
|Header and Footers|Aspose.Slides for Node.js via .NET usnadňuje práci se záhlavími a zápatími ve snímcích.|

|Text Boxes|Textová pole lze vytvořit od začátku. Můžete přistupovat k existujícím textovým polím. Také můžete upravovat jejich texty bez ztráty původního formátování textu.|
