{{% alert color="primary" %}} 

Aspose.Slides pro Node.js prostřednictvím .NET je knihovna tříd, která umožňuje aplikacím Node.js číst a zapisovat prezentační dokumenty bez použití Microsoft PowerPoint® nebo jiných. Aspose.Slides pro Node.js prostřednictvím .NET vám umožňuje spravovat text, tvary, tabulky a animace, přidávat audio a video do snímků, zobrazovat náhled snímků, exportovat snímky do formátu SVG, PDF a mnohé další.

Aspose.Slides pro Node.js prostřednictvím .NET používá [stejnou referenci API jako Aspose.Slides .NET](https://reference.aspose.com/slides/cs/nodejs-net/). 

{{% /alert %}}
