Tato sekce poskytuje podrobný popis funkcí Aspose.Slides pro JasperReports. Klikněte na odkazy v tabulce a zjistěte více o funkcích. 

|**Funkce** |**Popis** |
| :- | :- |
|[Export PPT, PPS, PPTX a PPSX](/slides/cs/jasperreports/ppt-pptx-pdf-and-html-export/)|Exportujte zprávy jako prezentace Microsoft PowerPoint z vaší aplikace, aniž byste používali Microsoft PowerPoint. |
|Vysoká věrnost návrhu zprávy|Zprávy PPT a PPS mají stejný rozvržení a vzhled jako PDF a další formáty. |
|Jednoduché a nenáročné nasazení|Aspose.Slides pro JasperReport je exportér pro JasperReport a JasperServer do formátu Microsoft PowerPoint. |
|[Jednoduché a cenově dostupné licencování](/slides/cs/jasperreports/licensing/)|Jedna licence na server nebo jedna licence pro všechny vaše servery. Neomezený počet vývojářů. |
|[Světová třída bezplatné technické podpory](/slides/cs/jasperreports/technical-support/)|Neomezená technická podpora je poskytována přímo vývojáři, kteří pracují na produktu.|
