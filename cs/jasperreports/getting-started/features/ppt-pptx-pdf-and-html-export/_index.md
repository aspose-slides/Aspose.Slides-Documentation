{{% alert color="primary" %}} 
JasperReports a JasperServer nemají vestavěné možnosti exportovat zprávy jako prezentace Microsoft PowerPoint, ale s Aspose.Slides for JasperReports získáte přístup k dalším exportním formátům:

- Prezentace Microsoft PowerPoint (PPT)
- Prezentace Microsoft PowerPoint (PPTX)
- HTML
- PDF

{{% /alert %}} 
Pro vytváření dokumentů v těchto formátech Aspose.Slides for JasperReports využívá vestavěnou verzi [Aspose.Slides for Java](https://products.aspose.com/slides/cs/java/), přední knihovny pro zpracování prezentací na trhu od společnosti Aspose. Microsoft PowerPoint se nepoužívá k vytváření dokumentů.
