Aspose.Slides for JasperReports funguje s:

**Aspose.Slides for JasperReports 17.5 a starší verze:**

- Java 2 Platform Standard Edition 1.4 nebo vyšší.
- JasperReports 2.0, 3.0 nebo vyšší.

**Aspose.Slides for JasperReports 17.6 a novější verze:**

- Java 2 Platform Standard Edition 6.0 nebo vyšší.
- JasperReports 3.7.2 nebo vyšší.
