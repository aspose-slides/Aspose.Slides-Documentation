**Vítejte v Aspose.Slides for JasperReports**

Aspose.Slides for JasperReports je knihovna speciálně navržena a vyvíjena pro vývojáře, kteří potřebují snadno exportovat zprávy z JasperReports do formátů Microsoft PowerPoint Presentation (PPT) a Microsoft PowerPoint Show (PPS) ve svých Java aplikacích. Všechny funkce zpráv jsou převedeny s nejvyšší přesností do prezentací Microsoft PowerPoint. Aspose.Slides for JasperReports zahrnuje podporu pro JasperReports 5+.
