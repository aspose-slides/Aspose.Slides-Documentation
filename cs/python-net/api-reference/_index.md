{{% alert color="primary" %}}

Aspose.Slides for Python via .NET je knihovna tříd, která umožňuje aplikacím v jazyce Python číst a zapisovat prezentační dokumenty bez použití Microsoft PowerPoint® nebo jiného softwaru. Aspose.Slides for Python via .NET vám umožňuje spravovat text, tvary, tabulky a animace; přidávat zvuk a video do snímků; prohlížet snímky; a exportovat snímky do formátů SVG, PDF a dalších.

Nejnovější referenční dokumentaci API najdete na [Aspose.Slides for Python via .NET API Reference](https://reference.aspose.com/slides/cs/python-net/).
