{{% alert color="primary" %}} 

Aspose.Slides for PHP via Java je knihovna tříd, která umožňuje aplikacím PHP číst a zapisovat prezentační dokumenty bez použití Microsoft PowerPoint® nebo jiných. Aspose.Slides for PHP via Java umožňuje spravovat text, tvary, tabulky a animace, přidávat zvuk a video do snímků, náhled snímků, exportovat snímky do formátu SVG, PDF a další.

Nejnovější referenční API lze najít na [Aspose.Slides for PHP via Java API Reference](https://reference.aspose.com/slides/cs/php-java/).
