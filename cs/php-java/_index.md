**Vítejte v Aspose.Slides for PHP via Java**

![Produktové logo Aspose.Slides for PHP via Java](aspose_slides-for-php-via-java.png)

Aspose.Slides for PHP via Java je knihovna tříd, která umožňuje vašim aplikacím číst a zapisovat dokumenty PowerPoint® bez použití Microsoft PowerPoint®.

Aspose.Slides for PHP via Java je první a jediné komponenta, která poskytuje funkčnost pro správu dokumentů PowerPoint®.

Aspose.Slides for PHP via Java poskytuje řadu klíčových funkcí, jako je správa textu, tvarů, tabulek a animací, přidávání audia a videa na snímky, náhled snímků, export snímků do formátu SVG, PDF a další.
