## **Přehled**

Tento článek vysvětluje, jak získat podporu pro Aspose.Slides a jak efektivně nahlásit problémy. Popisuje hlavní kanál podpory, poskytuje doporučení pro podání problému a odkazuje na užitečné technické zdroje, jako je stránka produktu, blog, ke stažení, fórum podpory, dokumentace a API Reference. Článek také odpovídá na běžné otázky týkající se placené podpory, eskalace problémů a podpory starších verzí knihovny.

## **Technická podpora**

{{% alert color="primary" %}} 

Aspose poskytuje neomezenou bezplatnou technickou podporu pro všechny své produkty. Podpora je dostupná všem uživatelům, včetně zkušební verze.

{{% /alert %}} 

If you need help with Aspose.Slides for PHP via Java, consider the following:

- Hlavním kanálem podpory jsou Aspose.Forums. Zeptejte se ve fóru Aspose.Slides a odpověď dostanete během několika hodin. Na otázky odpovídá přímo tým podpory Aspose.Slides.
- Všimněte si, že Aspose neposkytuje technickou podporu telefonicky. Telefonní podpora je k dispozici pouze pro otázky ohledně prodeje a nákupu.
- Při očekávání odpovědi na fórech počítejte s časovým rozdílem časových pásem.

If you have an issue with Aspose.Slides for PHP via Java, follow these simple steps to make sure it is resolved in the most efficient way:

- Ujistěte se, že používáte nejnovější verzi Aspose.Slides for PHP via Java před nahlášením problému; viz stažení Aspose.Slides pro informace o nejnovější verzi.
- Prohlédněte fórum, tuto dokumentaci a API Reference před nahlášením problému; možná již byla vaše otázka zodpovězena.
- Při hlášení problému přiložte původní dokument a případně úryvek kódu, který problém způsobuje. Pokud potřebujete připojit více souborů, zkomprimujte je do jednoho zipu. Je bezpečné přikládat dokumenty v Aspose.Forums, protože k souborům mají přístup pouze vy a vývojáři Aspose.
- Snažte se v každém vlákně nahlásit jen jeden problém. Pokud máte další problém, nahlaste jej v samostatném vlákně.

## **Často kladené otázky**

**Jaké úrovně placené podpory jsou k dispozici a jak se liší?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Liší se počtem zahrnutých incidentů, prioritou odpovědi a dostupnými komunikačními kanály.

**Mohu eskalovat chybu přímo vývojovému týmu?**

Ano. [Paid support](https://helpdesk.aspose.com/) umožňuje eskalaci vývojářům, pokud problém vyžaduje podrobnější vyšetřování.

**Jsou podporovány starší verze knihovny a vydáváte pro ně opravy?**

Starší verze můžete používat, ale opravy jsou poskytovány jen pro aktuální vydání; záplaty pro starší verze nejsou vydávány.
