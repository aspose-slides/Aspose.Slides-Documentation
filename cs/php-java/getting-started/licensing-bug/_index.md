## **Přehled**

Tento článek popisuje chybu licencování, která se vyskytla v Aspose.Slides for PHP via Java verzích 23.2, 23.3, 23.4, 23.5, 23.6 a 23.7. Tato chyba způsobila, že trvalé licence se po vypršení jejich předplatného staly dočasně nepoužitelnými.

## **Příznaky**

Po vypršení předplatného vaší trvalé licence může Aspose.Slides for PHP via Java verze 23.2 - 23.7 hlásit licenci jako prošlou a přepnout všechny funkce do evaluačního režimu. Toto chování je nesprávné a neovlivňuje verze před 23.2 a po 23.7.

## **Řešení**

Aspose.Slides for PHP via Java tuto záležitost řešil a vydal aktualizované verze (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) s opravou.

Pokud ve svém projektu používáte některou z postižených verzí Aspose.Slides for PHP via Java, přejděte na opravenou verzi.

- Zkontrolujte verzi, kterou používáte, a ujistěte se, že není postižena výše popsanou chybou.

## **Postižené produkty a verze**

Upozorňujeme, že tato chyba ovlivňuje pouze Aspose.Slides for PHP via Java verze 23.2 až 23.7. **Ostatní produkty Aspose a jiné verze Aspose.Slides nejsou postiženy**.
