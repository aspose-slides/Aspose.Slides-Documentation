{{% alert color="primary" %}} 

The [Specifikace XML parseru](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) je jazyk pro popis stránky a formát fixovaných dokumentů, který původně vyvinula společnost Microsoft. Stejně jako PDF je XPS navržen tak, aby zachoval věrnost dokumentu a zajistil nezávislý vzhled dokumentu na zařízení. 

{{% /alert %}} 

## **XPS v Aspose.Slides pro PHP přes Java**
Jakýkoli prezentační dokument, který lze načíst pomocí Aspose.Slides pro PHP přes Java, lze převést do formátu XPS. Aspose.Slides pro PHP přes Java používá vysoce věrný engine pro rozvržení stránky a vykreslování k vytváření výstupu ve formátu XPS s pevně daným rozvržením.
O exportu prezentačních dokumentů do XPS dokumentů pomocí Aspose.Slides pro PHP přes Java se můžete dozvědět v [Převod do XPS](https://docs.aspose.com/slides/cs/php-java/convert-powerpoint-to-xps/).
