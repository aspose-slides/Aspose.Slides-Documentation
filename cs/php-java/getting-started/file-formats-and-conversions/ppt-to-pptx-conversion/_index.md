Převod PPT na PPTX je podporován v Aspose.Slides pro PHP via Java. Většina funkcí prezentace – hlavní snímky, struktura a podobně – je zachována při převodu z jednoho formátu na druhý, ale existuje [několik omezení](/slides/cs/php-java/ppt-to-pptx-conversion/).

## **Funkce podporované při převodu**
Aspose.Slides pro PHP via Java poskytuje částečnou podporu převodu formátu souboru PPT na PPTX. Podpora převodu byla právě zavedena v Aspose.Slides pro PHP via Java, takže má několik omezení a nejlépe funguje pro jednoduché prezentace. Hlavní výhodou, kterou Aspose.Slides pro PHP via Java při převodu PPT na PPTX nabízí, je snadnost použití API. Pro zobrazení příkladů kódu si přečtěte o [Převodu PPT na PPTX](). Níže jsou uvedeny seznamy, které vysvětlují, které funkce jsou podporovány a které ne při převodu PPT na PPTX.

## **Podporované funkce**
Následující funkce jsou při převodu podporovány:

- Převod struktury hlavních snímků, rozvržení a snímků.
- Převod grafů.
- Skupinové tvary.
- Převod automatických tvarů včetně obdélníků a elips.
- Tvary s vlastní geometrií.
- Textury a výplňové styly obrázků pro automatické tvary.
- Převod zástupných symbolů.
- Převod čar a křivek.
- Formáty čar a výplní.
- Styly gradientní výplně.
- OLE rámy, tabulky, video a audio rámy atd.
- Vlastnosti animací a prezentací.
- Převod textu v textových rámečcích a textových držácích.
