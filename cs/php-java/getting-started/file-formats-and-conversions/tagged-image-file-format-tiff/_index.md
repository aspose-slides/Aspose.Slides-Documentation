## **O TIFF**

Formát značkovaných obrázkových souborů [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF), který je známý tím, že ukládá více obrázků v jednom dokumentu, byl původně vytvořen společností Aldus. Tento formát je široce podporován skenovacími, faxovacími a dalšími aplikacemi pro manipulaci s obrázky.

## **TIFF v Aspose.Slides pro PHP přes Java**

Jakýkoli dokument, který lze načíst v Aspose.Slides pro Java, může být také přímo převeden do TIFF dokumentu pomocí Aspose.Slides pro PHP přes Java, čímž se eliminuje potřeba jakékoli třetí strany. Dále můžete volitelně určit velikost obrázků v výsledném TIFF dokumentu. Informace o exportu prezentačních dokumentů do TIFF dokumentů pomocí Aspose.Slides pro PHP přes Java najdete v [této tématu](/slides/cs/php-java/convert-powerpoint-to-tiff/).
