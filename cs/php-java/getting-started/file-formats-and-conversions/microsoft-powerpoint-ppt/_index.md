{{% alert color="primary" %}} 

[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) je formát souboru prezentačního dokumentu, který může být vytvářen, čten, upravován a zapisován různými verzemi Microsoft PowerPoint. Jedná se o binární formát pro prezentační dokumenty vyvinutý společností Microsoft.

{{% /alert %}} 

## **PPT v Aspose.Slides pro PHP přes Java**
Aspose.Slides pro PHP přes Java může číst soubory PPT vytvořené níže uvedeným software.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Podobně mohou být soubory PPT vytvořené pomocí Aspose.Slides pro PHP přes Java čteny výše uvedeným softwarem.

## **Komplexní podpora pro PPT**
Aspose.Slides pro PHP přes Java poskytuje podporu téměř pro všechny funkce podporované formátem souboru PPT. Nejenže pokrývá základní i pokročilé funkce poskytované různými verzemi Microsoft PowerPoint pro manipulaci s dokumenty PPT, ale také funkce, které nejsou ani podporovány Microsoft PowerPoint. Hlavní výhodou použití knihovny Aspose.Slides pro PHP přes Java API je jednoduchost při práci s takovými funkcemi.

Kromě základních úkolů souvisejících s vytvářením, čtením a zápisem souborů PPT existuje několik funkcí, které poskytuje Aspose.Slides pro PHP přes Java:

- Importovat další formáty souborů Microsoft Office jako [OLE objekty do dokumentů PPT](/slides/cs/php-java/manage-ole/).
- [Exportovat dokumenty PPT do PDF](/slides/cs/php-java/convert-powerpoint-to-pdf/).
- Exportovat snímky v dokumentech PPT do formátů SVG.
- Vykreslit snímky do libovolného formátu obrázku podporovaného Java Frameworkem.
- Nastavit velikost snímků v dokumentech PPT.
- Spravovat animace na tvarech.
- Spravovat prezentace.
- [Formátovat text na snímcích](/slides/cs/php-java/text-formatting/).
- Extrahovat text z dokumentů PPT.
- [Zpracovávat tabulky na snímcích](/slides/cs/php-java/powerpoint-table/).
- Automaticky kopírovat master snímky pomocí [funkce klonování](/slides/cs/php-java/clone-slides/).
