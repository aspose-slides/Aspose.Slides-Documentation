{{% alert color="primary" %}} 

The [Portable Document Format](https://en.wikipedia.org/wiki/PDF) je formát souboru vytvořený společností Adobe Systems pro výměnu dokumentů mezi organizacemi. Účelem tohoto formátu bylo zachovat obsah a rozvržení beze změny, bez ohledu na platformu, na které je prohlížen. Aspose.Slides for PHP via Java vám umožňuje převádět soubory prezentací do PDF.

{{% /alert %}} 

## **PDF v Aspose.Slides for PHP via Java**
Jakákoli prezentace, kterou lze načíst do Aspose.Slides for PHP via Java, může být převedena do PDF, který splňuje PDF 1.5, PDF/A-1a, PDF/A-1b nebo PDF/UA podle vašeho výběru. Aspose.Slides for PHP via Java exportuje prezentace do PDF a ve většině případů vypadá výsledné PDF přesně jako původní prezentace.

Aspose.Slides podporuje následující funkce prezentace při převodu do PDF:

- Obrázky, textová pole a další tvary.
- Text a formátování.
- Odstavce a formátování.
- Hyperlinky.
- Záhlaví a zápatí.
- Odrážky.
- Tabulky.

Prezentace můžete exportovat do PDF přímo pomocí Aspose.Slides for PHP via Java: nepotřebujete žádnou další komponentu. Dále můžete přizpůsobit export prezentace do PDF pomocí různých možností, jak je vysvětleno v [Převod do PDF](/slides/cs/php-java/convert-powerpoint-to-pdf/).
