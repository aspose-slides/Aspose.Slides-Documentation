## **Přehled**

Tento článek vysvětluje, jak získat podporu pro Aspose.Slides a jak efektivně hlásit problémy. Popisuje hlavní kanál podpory, poskytuje doporučení pro podání problému a odkazuje na užitečné technické zdroje, jako je stránka produktu, blog, ke stažení, fórum podpory, dokumentace a API Reference. Článek také odpovídá na běžné otázky o placené podpoře, eskalaci problémů a podpoře starších verzí knihovny.

## **Technická podpora**

{{% alert color="primary" %}} 

Aspose poskytuje neomezenou bezplatnou technickou podporu pro všechny své produkty. Podpora je k dispozici všem uživatelům, včetně hodnocení.

{{% /alert %}} 

Pokud potřebujete pomoc s Aspose.Slides for Java, zvažte následující:

- Hlavním kanálem podpory jsou Aspose.Forums. Zveřejněte svůj dotaz ve fóru Aspose.Slides a bude zodpovězen během několika hodin. Na dotazy odpovídá přímo tým podpory Aspose.Slides.
- Všimněte si, že Aspose neposkytuje technickou podporu telefonicky. Telefonní podpora je k dispozici jen pro otázky související s prodejem a nákupem.
- Při očekávání odpovědi ve fórech počítejte s časovým rozdílem mezi časovými pásmy.

Pokud máte problém s Aspose.Slides for Java, postupujte podle těchto jednoduchých kroků, abyste zajistili co nejefektivnější řešení:

- Ujistěte se, že používáte nejnovější verzi Aspose.Slides for Java před nahlášením problému, viz Aspose.Slides download a zjistěte nejnovější verzi.
- Prohlédněte si fóra, tuto dokumentaci a API Reference před nahlášením problému; možná již byla vaše otázka zodpovězena.
- Při hlášení problému prosím přiložte původní dokument a případně úryvek kódu, který problém způsobuje. Pokud potřebujete připojit více souborů, zabalte je do jednoho zipu. Je bezpečné přikládat dokumenty v Aspose.Forums, protože k přiloženým souborům budou mít přístup jen vy a vývojáři Aspose.
- Snažte se hlásit jeden problém na vlákno. Pokud máte další problém, nahlaste jej v samostatném vláknu.

## **Často kladené otázky**

**Jaké úrovně placené podpory jsou dostupné a jak se liší?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Liší se počtem zahrnutých incidentů, prioritou odpovědi a dostupnými komunikačními kanály.

**Mohu přímo eskalovat chybu vývojovému týmu?**

Ano. [Paid support](https://helpdesk.aspose.com/) umožňuje eskalaci vývojářům, pokud problém vyžaduje podrobnější vyšetření.

**Jsou starší verze knihovny podporovány a vydáváte pro ně opravy?**

Můžete používat starší verze, ale opravy jsou poskytovány pouze pro aktuální vydání; opravy pro starší verze nejsou vydávány.
