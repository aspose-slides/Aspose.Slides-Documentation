{{% alert color="primary" %}} 

The [Portable Document Format](https://en.wikipedia.org/wiki/PDF) je formát souboru vytvořený společností Adobe Systems pro výměnu dokumentů mezi organizacemi. Účelem formátu bylo zachovat obsah i rozvržení beze změny, bez ohledu na platformu, na které je zobrazován. Aspose.Slides for Java vám umožňuje převádět soubory prezentací do PDF.

{{% /alert %}} 

## **PDF v Aspose.Slides for Java**
Jakákoli prezentace, která může být načtena do Aspose.Slides for Java, může být převedena do PDF, který odpovídá standardu [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) nebo [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA) podle vašeho výběru. Aspose.Slides for Java exportuje prezentace do PDF a ve většině případů výstupní PDF vypadá přesně jako původní prezentace.

Aspose.Slides podporuje při převodu do PDF následující funkce prezentací:

- Obrázky, textová pole a další tvary.
- Text a formátování.
- Odstavce a formátování.
- Hypertextové odkazy.
- Záhlaví a zápatí.
- Odrážky.
- Tabulky.

Prezentace můžete exportovat do PDF přímo pomocí Aspose.Slides for Java: nepotřebujete žádnou další komponentu. Dále můžete přizpůsobit export prezentace do PDF pomocí různých možností, jak je vysvětleno v [Převod do PDF](/slides/cs/java/converting-a-presentation/).
