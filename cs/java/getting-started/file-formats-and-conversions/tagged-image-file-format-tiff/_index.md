## **O TIFF**

Formát [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF), který je známý tím, že ukládá více obrázků v jednom dokumentu, byl původně vytvořen společností Aldus. Tento formát je široce podporován skenovacími, faxovacími a dalšími aplikacemi pro manipulaci s obrázky.

## **TIFF v Aspose.Slides pro Java**

Jakýkoli dokument, který lze načíst v Aspose.Slides pro Java, může být také přímo převeden do TIFF dokumentu pomocí Aspose.Slides pro Java, čímž se eliminuje potřeba jakékoli třetí strany. Dále můžete volitelně definovat velikost obrázků ve výsledném TIFF dokumentu. Informace o exportu prezentačních dokumentů do TIFF dokumentů pomocí Aspose.Slides pro Java najdete v [toto téma](/slides/cs/java/converting-a-presentation/).
