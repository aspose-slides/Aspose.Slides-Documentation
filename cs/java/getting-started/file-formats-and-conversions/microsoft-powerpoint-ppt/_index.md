{{% alert color="primary" %}} 

[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) je formát souboru prezentace, který může být vytvářen, čten, upravován a zapisován různými verzemi Microsoft PowerPoint. Jedná se o binární formát pro prezentační dokumenty vyvinutý společností Microsoft.

{{% /alert %}} 

## **PPT v Aspose.Slides pro Java**
Aspose.Slides for Java dokáže číst soubory PPT vytvořené následujícím software.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Podobně soubory PPT vytvořené pomocí Aspose.Slides for Java mohou číst výše uvedené programy.

## **Komplexní podpora pro PPT**
Aspose.Slides for Java poskytuje podporu pro téměř všechny funkce podporované formátem souboru PPT. Nejenže pokrývá základní a pokročilé funkce různých verzí Microsoft PowerPoint pro manipulaci s PPT dokumenty, ale také funkce, které Microsoft PowerPoint vůbec nepodporuje. Hlavní výhodou použití knihovny Aspose.Slides for Java API je snadnost práce s takovými funkcemi.

Kromě základních úkolů souvisejících s vytvářením, čtením a zápisem souborů PPT existuje několik funkcí, které poskytuje Aspose.Slides for Java:

- Importovat jiné formáty souborů Microsoft Office jako [OLE objekty do PPT dokumentů]().
- [Exportovat PPT dokumenty do PDF](/slides/cs/java/convert-powerpoint-ppt-and-pptx-to-pdf/).
- Exportovat snímky v PPT dokumentech do formátů SVG.
- Vykreslit snímky do libovolného formátu obrazu podporovaného Java Frameworkem.
- Nastavit velikost snímků v PPT dokumentech.
- Spravovat animace na tvarech.
- Spravovat prezentace.
- [Formátovat text na snímcích]().
- Extrahovat text z PPT dokumentů.
- [Zpracovávat tabulky na snímcích]().
- Automaticky kopírovat master snímky pomocí [funkce klonování]().
