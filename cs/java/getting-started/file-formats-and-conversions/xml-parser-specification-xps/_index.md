{{% alert color="primary" %}} 

Specifikace XML parseru[https://en.wikipedia.org/wiki/Open_XML_Paper_Specification] je jazyk pro popis stránky a formát pevného dokumentu původně vyvinutý společností Microsoft. Stejně jako PDF, XPS je navržen tak, aby zachovával věrnost dokumentu a poskytoval nezávislý vzhled dokumentu na zařízení. 

{{% /alert %}} 

## **XPS v Aspose.Slides pro Java**
Jakýkoli prezentační dokument, který lze načíst pomocí Aspose.Slides pro Java, lze převést do formátu XPS. Aspose.Slides pro Java používá vysoce věrný engine rozvržení stránky a vykreslování k vytvoření výstupu ve formátu XPS s pevným rozvržením.
Můžete se dozvědět o exportu prezentačních dokumentů do XPS pomocí Aspose.Slides pro Java v [Převod do XPS](https://docs.aspose.com/slides/cs/java/convert-powerpoint-to-xps/).
