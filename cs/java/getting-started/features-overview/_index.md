## **Podporované platformy**
Aspose.Slides for Java podporuje nejpopulárnější vývojové a nasazovací platformy.

|**Funkce**|**Popis**|
| :- | :- |
|Desktopové aplikace|Aspose.Slides for Java lze použít k vývoji aplikací Windows Forms|
|Podnikové webové aplikace|Použití Aspose.Slides for Java pomáhá vytvářet webové aplikace. Je také poskytována podpora pro použití Aspose.Slides for Java s PHP.|

## **Často kladené otázky**

**Potřebuji nainstalovat Microsoft PowerPoint na server/PC, aby knihovna fungovala?**

Ne. PowerPoint není vyžadován; Aspose.Slides je samostatný engine pro vytváření, úpravu, konverzi a vykreslování prezentací.

**Jak funguje vícevláknové zpracování? Lze zpracování paralelizovat?**

Je bezpečné zpracovávat různé dokumenty v různých vláknech; stejný [prezentace](https://reference.aspose.com/slides/cs/java/com.aspose.slides/presentation/) objekt nesmí být používán [více vláken](/slides/cs/java/multithreading/) současně.

**Jsou podporována hesla souborů a šifrování?**

Ano. [Můžete](/slides/cs/java/password-protected-presentation/) otevřít šifrované prezentace, nastavit nebo odstranit heslo pro otevření a zápis a zkontrolovat stav ochrany.

**Musím se starat o balíčky písem v Linux kontejnerech?**

Ano. Doporučuje se nainstalovat běžné balíčky písem a/nebo výslovně [specifikovat adresáře písem](/slides/cs/java/custom-font/) ve vaší aplikaci, aby se předešlo neočekávaným nahrazením.

**Existují omezení ve zkušební verzi?**

V [zkušební režim](/slides/cs/java/licensing/) je do výstupu přidán vodoznak a platí určitá omezení; je k dispozici [30denní dočasná licence](https://purchase.aspose.com/temporary-license/) pro testování všech funkcí.

**Je podporováno importování externích formátů do prezentace (PDF/HTML → PPTX)?**

Ano. Můžete přidat [PDF stránky a HTML obsah](/slides/cs/java/import-presentation/) do prezentace, přeměnit je na snímky.
