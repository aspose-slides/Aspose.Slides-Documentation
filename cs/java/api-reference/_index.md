{{% alert color="primary" %}} 

Aspose.Slides for Java je knihovna tříd, která umožňuje aplikacím Java číst a zapisovat prezentační dokumenty bez použití Microsoft PowerPoint® nebo jiných. Aspose.Slides for Java umožňuje spravovat text, tvary, tabulky a animace, přidávat zvuk a video do snímků, prohlížet snímky, exportovat snímky do formátu SVG, PDF a další.

Nejnovější referenci API najdete na [Aspose.Slides for Java API Reference](https://reference.aspose.com/slides/cs/java).

{{% /alert %}}
