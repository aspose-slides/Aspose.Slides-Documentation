Bylo vydáno nové [Aspose.Slides for Java API](https://docs.aspose.com/slides/cs/java/) a nyní tento jedinečný produkt podporuje možnost generovat PowerPoint dokumenty od začátku i upravovat existující.

## **Podpora pro starší kód**
Aby bylo možné použít starý kód vyvinutý s Aspose.Slides for Java ve verzích před 13.x, je třeba provést některé drobné úpravy ve vašem kódu a kód bude fungovat jako dříve. Všechny třídy, které byly v starém Aspose.Slides for Java v prostorách názvů Aspose.Slide a Aspose.Slides.Pptx, jsou nyní sloučeny do jediné prostoru názvů Aspose.Slides. Podívejte se na následující jednoduchý úryvek kódu pro přidání záhlaví a zápatí do prezentace ve starší Aspose.Slides API a postupujte podle kroků popisujících, jak migrovat na nové sloučené API.
