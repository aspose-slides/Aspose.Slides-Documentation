Běžný úkol, který vývojáři často řeší, je dynamické přidávání textu do snímků. Tento článek ukazuje příklady kódu pro dynamické přidávání textu pomocí [VSTO](/slides/cs/java/adding-text-dynamically-using-vsto-and-aspose-slides-for-java/) a [Aspose.Slides for Java](/slides/cs/java/adding-text-dynamically-using-vsto-and-aspose-slides-for-java/).

{{% /alert %}} 
## **Dynamické přidávání textu**
Obě metody postupují podle následujících kroků:

1. Vytvořte prezentaci.
1. Přidejte prázdný snímek.
1. Přidejte textové pole.
1. Nastavte text.
1. Uložte prezentaci.
