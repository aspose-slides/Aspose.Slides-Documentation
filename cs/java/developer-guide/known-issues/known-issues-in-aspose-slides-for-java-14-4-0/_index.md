{{% alert color="primary" %}} 

Aspose.Slides pro Java 14.4.0 poskytuje nové řešení pro zpracování dokumentů PowerPoint. Existují některá omezení a známé problémy, které budou v nadcházejících verzích odstraněny:

- Některé tvary mají ve serializovaných PPT dokumentech nesprávnou geometrii (oblouk, kruhová šipka, popisky).
- Ne všechny funkce formátování textu v PPTX jsou podporovány při serializaci do PPT (omezena podpora tabulátorů, odsazení a formátování odstavců).
- Informace o jazyce textu a nastaveních pravopisu nejsou ve serializovaných PPT dokumentech přítomny.
- Ne všechny funkce témat PPTX jsou podporovány při serializaci do PPT (pouze serializace formátů výplní, čar a fontů).
- Jsou známy problémy při serializaci OLE/ActiveX PPT do PPT.
- Serializace a vykreslování WordArtu nejsou podporovány.
