## Aspose.Slides for .NET

{{% alert color="primary" %}}

![Aspose.Slides for .NET Product Logo](home_1.png)

Aspose.Slides for .NET je Microsoft PowerPoint® spravovací API, které umožňuje aplikacím .NET číst a zapisovat PowerPoint® dokumenty bez použití Microsoft PowerPoint®. Aspose.Slides for .NET je první a jediná komponenta, která poskytuje funkčnost pro správu PowerPoint® dokumentů. Aspose.Slides for .NET nabízí řadu klíčových funkcí, jako je správa textu, tvarů, tabulek a animací, přidávání audia a videa do snímků, náhled snímků, export snímků do SVG, PDF formátu a další.
