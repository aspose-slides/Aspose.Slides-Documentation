# Aspose.Slides Documentation

This repository contains the Aspose.Slides product family documentation.

## Documentation Structure

Documentation is organized by language first:

```text
/{language}/{product}/{section}/_index.md
```

For example, English documentation for Aspose.Slides for .NET starts at:

```text
/en/net/_index.md
```

Language root pages use this structure:

```text
/{language}/_index.md
```

Product root pages use this structure:

```text
/{language}/{product}/_index.md
```
