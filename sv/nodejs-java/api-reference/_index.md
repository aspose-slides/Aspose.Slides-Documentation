{{% alert color="primary" %}} 

Aspose.Slides for Node.js via Java är ett klassbibliotek som gör att Node.js-applikationer kan läsa och skriva presentationsdokument utan att använda Microsoft PowerPoint® eller andra. Aspose.Slides for Node.js via Java låter dig hantera text, former, tabeller och animationer, lägga till ljud och video på bilder, förhandsvisa bilder, exportera bilder till SVG- och PDF-format och mycket mer.

Den senaste API-referensen finns på [Aspose.Slides for Node.js via Java API Reference](https://reference.aspose.com/slides/sv/nodejs-java/).
