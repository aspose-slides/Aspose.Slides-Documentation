**Välkommen till Aspose.Slides för Node.js via Java**

![Aspose.Slides för Node.js via Java produktlogotyp](aspose_slides-for-nodejs-via-java.png)

Aspose.Slides för Node.js via Java är ett klassbibliotek som gör det möjligt för dina applikationer att läsa och skriva PowerPoint®-dokument utan att använda Microsoft PowerPoint®.

Aspose.Slides för Node.js via Java är den första och enda komponenten som erbjuder funktionalitet för att hantera PowerPoint®-dokument.

Aspose.Slides för Node.js via Java tillhandahåller många viktiga funktioner som att hantera text, former, tabeller och animationer, lägga till ljud och video i bilder, förhandsgranska bilder, exportera bilder till SVG, PDF-format och mer.
