Aspose.Slides för JasperReports fungerar med:

**Aspose.Slides för JasperReports 17.5 och tidigare versioner:**

- Java 2 Platform Standard Edition 1.4 eller högre.
- JasperReports 2.0, 3.0 eller högre.

**Aspose.Slides för JasperReports 17.6 och nyare versioner:**

- Java 2 Platform Standard Edition 6.0 eller högre.
- JasperReports 3.7.2 eller högre.
