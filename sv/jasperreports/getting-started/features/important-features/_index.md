Det här avsnittet ger en detaljerad beskrivning av Aspose.Slides for JasperReports funktioner. Klicka på länkarna i tabellen för att lära dig mer om funktionerna. 

|**Funktion** |**Beskrivning** |
| :- | :- |
|[PPT, PPS, PPTX och PPSX-export](/slides/sv/jasperreports/ppt-pptx-pdf-and-html-export/)|Exportera rapporter som Microsoft PowerPoint-presentationer från din applikation utan att använda Microsoft PowerPoint.|
|High Fidelity to the Report Design|PPT- och PPS-rapporter har samma layout och utseende som PDF och andra format.|
|Easy and Lightweight Deployment|Aspose.Slides for JasperReport är en JasperReport- och JasperServer-exportör till Microsoft PowerPoint-format.|
|[Enkel och prisvärd licensiering](/slides/sv/jasperreports/licensing/)|En licens per server eller en licens för alla dina servrar. Obegränsat antal utvecklare.|
|[Världsledande gratis teknisk support](/slides/sv/jasperreports/technical-support/)|Obegränsad teknisk support tillhandahålls direkt av de utvecklare som arbetar med produkten.|
