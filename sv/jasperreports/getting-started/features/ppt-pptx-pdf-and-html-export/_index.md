{{% alert color="primary" %}} 

JasperReports och JasperServer har inte inbyggda funktioner för att exportera rapporter som Microsoft PowerPoint-presentationer, men med Aspose.Slides för JasperReports får du tillgång till ytterligare exportformat:

- Microsoft PowerPoint Presentation (PPT)
- Microsoft PowerPoint Presentation (PPTX)
- HTML
- PDF

{{% /alert %}} 

För att skapa dokument i dessa format förlitar sig Aspose.Slides för JasperReports på en inbyggd version av [Aspose.Slides for Java](https://products.aspose.com/slides/sv/java/), det marknadsledande presentationsbearbetningsbiblioteket från Aspose. Microsoft PowerPoint används inte för att generera dokumenten.
