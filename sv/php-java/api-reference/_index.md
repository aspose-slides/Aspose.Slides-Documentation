{{% alert color="primary" %}} 

Aspose.Slides for PHP via Java är ett klassbibliotek som möjliggör för PHP-applikationer att läsa och skriva presentationsdokument utan att använda Microsoft PowerPoint® eller andra. Aspose.Slides for PHP via Java gör det möjligt att hantera text, former, tabeller och animationer, lägga till ljud och video på bilder, förhandsgranska bilder, exportera bilder till SVG, PDF-format och mer.

Den senaste API-referensen finns på [Aspose.Slides for PHP via Java API-referens](https://reference.aspose.com/slides/sv/php-java/).
