## **Stödda plattformar**
Aspose.Slides for PHP via Java är ett plattformsoberoende API. Det kan användas på vilken plattform som helst (Windows, Linux, macOS osv.) där PHP 7 eller högre är installerat. Oracle JDK 7 eller högre är ett annat krav.

## **Filformat och konverteringar**
Aspose.Slides for PHP via Java stöder de flesta PowerPoint‑dokumentformat. Det låter dig också exportera dem till de populära filformat som organisationer använder och delar med varandra. Läs mer nedan:

|**Funktion**|**Beskrivning**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for PHP via Java erbjuder den snabbaste bearbetningen för detta presentationsdokumentformat.|
|PPT till PPTX‑konvertering|Aspose.Slides for PHP via Java stödjer PPT‑till‑PPTX‑konvertering.|
|Portable Document Format (PDF)|Du kan exportera alla stödda filformat till Adobe Portable Document Format (PDF)‑dokument med en enda metod.|
|XML Parser Specification (XPS)|Du kan exportera alla stödda filformat till XML Parser Specification (XPS)‑dokument med en enda metod.|
|Tagged Image File Format (TIFF)|Du kan exportera alla stödda presentationsfilformat till Tagged Image File Format (TIFF).|
|PPTX‑till‑HTML‑konvertering|Aspose.Slides for PHP via Java stödjer konvertering av PresentationEx till HTML‑format.|

## **Vanliga frågor**

**Behöver jag installera Microsoft PowerPoint på servern/PC:n för att biblioteket ska fungera?**

Nej. PowerPoint krävs inte; Aspose.Slides är en fristående motor för att skapa, redigera, konvertera och rendera presentationer.

**Hur fungerar multitrådning? Kan bearbetning parallelliseras?**

Det är säkert att bearbeta olika dokument i olika trådar; samma [presentation](https://reference.aspose.com/slides/sv/php-java/aspose.slides/presentation/)‑objekt får inte användas av [flera trådar](/slides/sv/php-java/multithreading/) samtidigt.

**Stöds fillösenord och kryptering?**

Ja. [Du kan](/slides/sv/php-java/password-protected-presentation/) öppna krypterade presentationer, ange eller ta bort ett öppnings‑ och skrivlösenord samt kontrollera skyddsstatusen.

**Behöver jag tänka på teckensnittspaket i Linux‑behållare?**

Ja. Det rekommenderas att installera vanliga teckensnittspaket och/eller uttryckligen [ange teckensnittskataloger](/slides/sv/php-java/custom-font/) i din applikation för att undvika oväntade ersättningar.

**Finns det begränsningar i evalueringsversionen?**

I [utvärderingsläge](/slides/sv/php-java/licensing/) läggs ett vattenstämpel till i utdata och vissa begränsningar gäller; en [30‑dagars tillfällig licens](https://purchase.aspose.com/temporary-license/) är tillgänglig för fullständig funktionstestning.

**Stöds import av externa format till en presentation (PDF/HTML → PPTX)?**

Ja. Du kan lägga till [PDF‑sidor och HTML‑innehåll](/slides/sv/php-java/import-presentation/) i en presentation, vilket omvandlar dem till bilder.
