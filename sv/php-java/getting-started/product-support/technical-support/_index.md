## **Översikt**

Denna artikel förklarar hur du får support för Aspose.Slides och hur du rapporterar problem på ett effektivt sätt. Den beskriver huvudsupportkanalen, ger rekommendationer för hur du lämnar in ett problem och pekar på användbara tekniska resurser såsom produktsidan, bloggen, nedladdningar, supportforum, dokumentation och API‑referens. Artikeln svarar också på vanliga frågor om betald support, eskalering av ärenden och support för äldre versioner av biblioteket.

## **Teknisk support**

{{% alert color="primary" %}} 

Aspose erbjuder obegränsad gratis teknisk support för alla sina produkter. Supporten är tillgänglig för alla användare, inklusive utvärdering.

{{% /alert %}} 

Om du behöver hjälp med Aspose.Slides for PHP via Java, överväg följande:

- Huvudkanalen för support är Aspose.Forums. Posta din fråga i Aspose.Slides‑forumet så besvaras den inom några timmar. Frågorna besvaras direkt av Aspose.Slides‑supportteamet.
- Observera att Aspose inte erbjuder teknisk support via telefon. Telefonsupport är endast tillgänglig för försäljnings‑ och inköpsfrågor.
- När du väntar på ett svar i forumet, ta hänsyn till tidszonsskillnader.

Om du har ett problem med Aspose.Slides for PHP via Java, följ dessa enkla steg för att säkerställa att det löses på det mest effektiva sättet:

- Se till att du använder den senaste versionen av Aspose.Slides for PHP via Java innan du rapporterar problemet, se Aspose.Slides‑nedladdning för att ta reda på den senaste versionen.
- Titta igenom forumet, denna dokumentation och API‑Reference innan du rapporterar problemet; kanske har din fråga redan besvarats.
- När du rapporterar ett problem, inkludera det ursprungliga dokumentet och eventuellt ett kodfragment som orsakar problemet. Om du behöver bifoga flera filer, zip‑a dem till en. Det är säkert att bifoga dina dokument i Aspose.Forums eftersom endast du och Aspose‑utvecklare har åtkomst till de bifogade filerna.
- Försök att rapportera ett problem per tråd. Om du har ett annat problem, rapportera det i en separat tråd.

## **Vanliga frågor**

**Vilka betalda supportnivåer finns tillgängliga och hur skiljer de sig?**

[Utvecklare, Företag och Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). De skiljer sig åt genom antalet incidenter som ingår, svarsprioritet och tillgängliga kommunikationskanaler.

**Kan jag eskalera en bugg direkt till ingenjörsteamet?**

Ja. [Betald support](https://helpdesk.aspose.com/) möjliggör eskalering till utvecklarna när ett problem kräver djupare undersökning.

**Stöds äldre versioner av biblioteket, och släpper ni korrigeringar för dem?**

Du kan använda äldre versioner, men korrigeringar tillhandahålls för den aktuella releasen; patchar för legacy‑versioner utfärdas inte.
