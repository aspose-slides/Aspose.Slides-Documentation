## **Översikt**

Aspose.Slides stöder presentationsfiler från Microsoft PowerPoint 97 till Office 365, inklusive Microsoft PowerPoint för Mac. Denna artikel listar de PowerPoint‑versioner som biblioteket stödjer och ger en tabell över filformat som kan läsas in, sparas eller både och.

Artikeln svarar också på vanliga frågor om PDF‑efterlevnad, inbäddning av teckensnitt, lösenordsskyddade filer, anpassade teckensnitt, reservteckensnitt och XPS‑exportalternativ.

- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007

- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019

- Office 365

## **Vanliga frågor**

**Kan jag spara presentationer till PDF som uppfyller arkiverings- och tillgänglighetsstandarder (PDF/A och PDF/UA)?**

Ja. Aspose.Slides stöder export till PDF med efterlevnadsnivåer såsom PDF/A‑2a, PDF/A‑2b, PDF/A‑2u, PDF/A‑3a, PDF/A‑3b samt PDF/UA genom inställningen [efterlevnad](https://reference.aspose.com/slides/sv/php-java/aspose.slides/pdfoptions/setcompliance/) i [PDF‑exportalternativ](https://reference.aspose.com/slides/sv/php-java/aspose.slides/pdfoptions/).

**Stöder biblioteket inbäddning av teckensnitt vid export till PDF, med fin kontroll över vad som bäddas in?**

Ja. Du kan styra om teckensnitt ska bäddas in helt eller delvis (endast använda tecken), specificera hur vanliga systemteckensnitt hanteras och konfigurera beteendet för ASCII‑text via [PDF‑exportalternativ](https://reference.aspose.com/slides/sv/php-java/aspose.slides/pdfoptions/).

**Kan jag upptäcka om en fil är lösenordsskyddad innan jag faktiskt läser in den?**

Ja. Med hjälp av det [fabrikbaserade inspektions‑API:t](https://reference.aspose.com/slides/sv/php-java/aspose.slides/presentationfactory/) kan du undersöka en presentationsfil för att avgöra om den är lösenordsskyddad utan att öppna den helt.

**Finns det reservteckensnittsmekanismer och stöd för anpassade teckensnitt?**

Ja. Biblioteket stöder [laddning](/slides/sv/php-java/custom-font/) och [inbäddning](/slides/sv/php-java/embedded-font/) av anpassade teckensnitt samt erbjuder teckensnittets [reservregler](/slides/sv/php-java/fallback-font/) för att förhindra saknade tecken under rendering och konvertering.

**Kan jag exportera bilder till XPS, och finns det alternativ för att finjustera XPS‑utdata?**

Ja. [Export till XPS](/slides/sv/php-java/convert-powerpoint-to-xps/) stöds, och du kan justera relevanta [spara‑alternativ](https://reference.aspose.com/slides/sv/php-java/aspose.slides/xpsoptions/) för att styra kvaliteten och innehållet i XPS‑dokumentet.
