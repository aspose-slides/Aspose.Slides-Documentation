## **Funktioner som stöds i konvertering**
Aspose.Slides för PHP via Java erbjuder partiellt stöd för att konvertera PPT‑filformatet till PPTX. Stödet för konvertering har just introducerats i Aspose.Slides för PHP via Java, så det har några begränsningar och fungerar bäst för enkla presentationer. Den största fördelen som Aspose.Slides för PHP via Java ger vid konvertering från PPT till PPTX är hur enkel API:et är att använda. För att se exempel på koden, läs om [Konvertera PPT till PPTX](). Nedan förklarar listorna vilka funktioner som stöds och vilka som inte gör det för PPT‑till‑PPTX‑konvertering.

## **Stödda funktioner**
Följande funktioner stöds för konvertering:

- Konvertera strukturen för masterbilder, layouter och bilder.
- Konvertera diagram.
- Gruppformer.
- Konvertera automatiska former inklusive rektanglar och ellipser.
- Former med anpassad geometri.
- Texturer och bildfyllningsstilar för automatiska former.
- Konvertera platshållare.
- Konvertera linjer och polylinjer.
- Linje- och fyllningsformat.
- Gradientfyllningsstilar.
- OLE‑ramar, tabeller, video‑ och audio‑ramar osv.
- Animations‑ och bildspels‑egenskaper.
- Konvertera text i textramar och texthållare.
