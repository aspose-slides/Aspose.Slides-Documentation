## **Om TIFF**

The [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) which is known for storing multiple images in one document was originally created by Aldus. This format is widely supported by scanning, faxing and other image manipulation applications.

## **TIFF i Aspose.Slides för PHP via Java**

Any document that can be loaded in Aspose.Slide for Java can also be converted to TIFF document directly by Aspose.Slides for PHP via Java eliminating requirement of any third party component. Further, you can optionally define the size of the images in the resulting TIFF document. You can find information about exporting the presentation documents to TIFF documents through Aspose.Slides for PHP via Java in [det här avsnittet](/slides/sv/php-java/convert-powerpoint-to-tiff/).
