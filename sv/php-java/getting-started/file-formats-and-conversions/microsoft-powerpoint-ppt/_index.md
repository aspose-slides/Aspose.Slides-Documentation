{{% alert color="primary" %}} 

PPT är presentationsdokumentfilformatet som kan skapas, läsas, manipuleras och skrivas av olika versioner av Microsoft PowerPoint. Detta är det binära formatet för presentationsdokument som utvecklats av Microsoft.

{{% /alert %}} 

## **PPT i Aspose.Slides för PHP via Java**
Aspose.Slides för PHP via Java kan läsa PPT-filer skapade av mjukvaran som listas nedan.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

På samma sätt kan PPT-filer som skapats av Aspose.Slides för PHP via Java läsas av ovanstående programvara.

## **Omfattande stöd för PPT**
Aspose.Slides för PHP via Java erbjuder stöd för nästan alla funktioner som stöds av PPT-dokumentfilformatet. Det täcker inte bara de grundläggande och avancerade funktionerna som tillhandahålls av olika versioner av Microsoft PowerPoint för PPT-dokumentmanipulering, utan även funktioner som inte ens stöds av Microsoft PowerPoint. Den största fördelen med att använda Aspose.Slides för PHP via Java API-biblioteket är hur enkelt det är att hantera sådana funktioner.

Förutom de grundläggande uppgifterna för att skapa, läsa och skriva PPT-dokumentfiler finns det flera funktioner som tillhandahålls av Aspose.Slides för PHP via Java:

- Importera andra Microsoft Office‑filformat som [OLE‑objekt i PPT-dokument](/slides/sv/php-java/manage-ole/).
- [Exportera PPT-dokument till PDF](/slides/sv/php-java/convert-powerpoint-to-pdf/).
- Exportera bilder i PPT-dokumenten till SVG-format.
- Rendera bilder till vilket bildformat som helst som stöds av Java‑ramverket.
- Ställ in storleken på bilder i PPT-dokument.
- Hantera animationer på former.
- Hantera bildspel.
- [Formatera text på bilder](/slides/sv/php-java/text-formatting/).
- Extrahera text från PPT-dokument.
- [Hantera tabeller på bilder](/slides/sv/php-java/powerpoint-table/).
- Kopiera automatiskt masterbilder med hjälp av [kloningsfunktionen](/slides/sv/php-java/clone-slides/).
