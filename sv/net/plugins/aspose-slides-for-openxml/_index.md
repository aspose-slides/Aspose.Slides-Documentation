## **Vad är OpenXML?**
OpenXML SDK är ett .NET‑SDK från Microsoft för att bearbeta Microsoft Office 2007 och senare filformat. Dessa filformat är i princip zip‑paket och Open XML SDK för Office förenklar uppgiften att manipulera Open XML‑paket och de underliggande Open XML‑schematelementen i ett paket. Klasserna i Open XML SDK kapslar in många vanliga uppgifter som utvecklare utför på Open XML‑paket, så att du kan utföra komplexa operationer med några kodrader.

När du har installerat Open XML SDK 2.5, öppna ditt befintliga projekt eller program i Visual Studio, eller skapa ett nytt projekt eller program. Lägg sedan till referenser till följande komponenter i ditt projekt eller program.

- DocumentFormat.OpenXml
- WindowsBase
## **Aspose.Slides för .NET**
Aspose.Slides för .NET är en unik PowerPoint®‑hanteringskomponent som gör det möjligt för .NET‑applikationer att läsa, skriva och manipulera PowerPoint‑dokument utan att använda Microsoft PowerPoint.

Aspose.Slides för .NET är den första och enda .NET‑komponenten som erbjuder funktionalitet för att hantera PowerPoint‑dokument i dina egna applikationer. Precis som alla Aspose .NET‑komponenter är Aspose.Slides skrivet i hanterad C#, otroligt prisvärt och blixtsnabbt.

## **Introduktion**
Utvecklare som söker filformatbibliotek kan behöva gå igenom funktionerna som erbjuds av de bibliotek som finns på marknaden. Vanligtvis beaktas två faktorer när ett alternativ väljs:

- Funktioner
- Ansträngning som krävs för att använda biblioteket

Om du är en .NET‑utvecklare som söker ett snabbare, enkelt och lättviktigt filformatbibliotek för att bearbeta filer, kan du behöva jämföra VSTO (Visual Studio‑verktyg för Office) med Aspose. I det här fallet hjälper detta projekt dig att jämföra funktioner och kod i båda biblioteken.
