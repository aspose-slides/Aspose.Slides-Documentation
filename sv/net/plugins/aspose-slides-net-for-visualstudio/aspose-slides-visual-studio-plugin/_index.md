## **Introduktion**
Microsoft Visual Studio är en integrerad utvecklingsmiljö (IDE) från Microsoft. Den används för att utveckla program för Microsoft Windows samt webbplatser, webbapplikationer och webbtjänster. Visual Studio använder Microsofts mjukvaruutvecklingsplattformar såsom Windows API, Windows Forms, Windows Presentation Foundation, Windows Store och Microsoft Silverlight. Den kan producera både inhemsk kod och hanterad kod.
### **Aspose.Slides för .NET**
Aspose.Slides för .NET är ett unikt PowerPoint®‑hanterings‑API som möjliggör för .NET‑applikationer att läsa, skriva och manipulera PowerPoint‑dokument utan att använda Microsoft PowerPoint.

Aspose.Slides för .NET är det första och enda .NET PowerPoint‑API‑et som tillhandahåller funktionalitet för att hantera PowerPoint‑dokument i dina egna applikationer. Som med alla Aspose .NET‑komponenter är Aspose.Slides skrivet i hanterad C#, oerhört prisvärt och blixtsnabb.

För mer information om [Aspose.Slides för .NET API](http://www.aspose.com/.net/powerpoint-component.aspx)
### **Utforska exempel för Aspose.Slides för .NET med Visual Studio**
Aspose Visual Studio‑plugin – **Explore Aspose.Slides for .NET Examples** är ett utmärkt verktyg för att snabbt ladda ner och utforska Aspose.Slides för .NET API‑exempel. Det sparar mycket tid och ansträngning genom att erbjuda ett mycket enkelt alternativ för att sömlöst välja, ladda ner och öppna de senaste exempelprojekten utan att behöva lämna Visual Studio.

Denna version erbjuder följande funktioner:

- Stöder Visual Studio 2010, Visual Studio 2012 och Visual Studio 2013.
- Lätt att starta från **File > Aspose**‑menyn i Visual Studio
- Sparar tid och minskar inlärningskurvan.
- Gör det möjligt att välja och öppna de senaste Aspose.Slides för .NET API‑exemplen som finns på Github.com.
- Låter dig öppna exempelprojekt i **C#** eller **VB.NET**
- Ett valt projekt öppnas automatiskt i Visual Studio med alla nödvändiga referenser satta, så att du får en färdig miljö att köra och utforska.
- Laddar automatiskt ner de senaste API‑biblioteken från **NuGet** och exempel från **GitHub**.

## **Installation**
Det är väldigt enkelt att installera Aspose Visual Studio‑plugin, följ dessa enkla steg

- Dubbelklicka på den nedladdade msi‑filen
- Följ stegen, du kan helt enkelt trycka på Nästa på varje skärm för att installera med standardinställningar.

Du får en bekräftelseruta när installationen är klar.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_2)

**Note:** Please make sure to restart Visual Studio for the changes to take effect.
## **Användning**
När den är installerad kan du enkelt starta och använda det interaktiva pluginet enligt nedan:
**1.** Öppna pluginet genom att välja **File > Aspose** och därefter **New Aspose.Slides Example Project** i Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_3)

**2.** Första gången du öppnar pluginet visas dialogrutan för API‑val som visas nedan. Det tar några minuter att ladda ner det senaste API‑et och dess bibliotek från **NuGet** samt den senaste exempelkoden från **GitHub**.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_4)

**3.** I dialogrutan Välj ett API för att visa exempel, välj eller ändra platsen där exempelprojekten ska skapas samt välj eller ändra projekttypen som **C#** eller **VB.NET** och klicka på **Continue** för att öppna exempelprojektet i Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_1)

**4.** Nya exempelprojekt skapas enligt det valda projekttypen **C#** eller **VB.NET** i Visual Studio.

#### **Källkod**
Du kan hämta den senaste källkoden från någon av följande platser

- [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-.NET)
#### **Hur du konfigurerar källkoden**
Du måste ha följande installerat för att kunna öppna och utöka källkoden

- Visual Studio 2010

Följ dessa enkla steg för att komma igång

1. Ladda ner/Klona källkoden.
1. Öppna Visual Studio 2010 och välj **File** > **Open Project**
1. Bläddra till den senaste källkoden du har laddat ner och öppna **AsposeVisualStudioPlugin.sln**
#### **Översikt över källkoden**
Det finns två projekt i lösningen

- **AsposeVisualStudioPlugin** – Innehåller integrationen med Visual Studio, val och automatisk nedladdning av Aspose‑API:er och exempel.
- **Installer** – Wix‑installationsprojekt för att skapa msi‑installationsfilen för pluginet.
