## **Översikt**

Aspose.Slides stöder presentationsfiler från Microsoft PowerPoint 97 till Office 365, inklusive Microsoft PowerPoint för Mac. Denna artikel listar de PowerPoint‑versioner som biblioteket stödjer och ger en tabell över filformat som kan läsas in, sparas eller båda.

Artikeln svarar också på vanliga frågor om PDF‑efterlevnad, teckensnittsinbäddning, lösenordsskyddade filer, anpassade teckensnitt, teckensnittsfallback och XPS‑exportalternativ.

## **Versioner av Microsoft PowerPoint som stöds**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint för MAC
- Office 365

## **Vanliga frågor**

**Kan jag spara presentationer till PDF som uppfyller arkiverings- och tillgänglighetsstandarder (PDF/A och PDF/UA)?**

Ja. Aspose.Slides stöder export till PDF med efterlevnadsnivåer såsom PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b samt PDF/UA via inställningen [compliance](https://reference.aspose.com/slides/sv/net/aspose.slides.export/pdfoptions/compliance/) i [PDF export options](https://reference.aspose.com/slides/sv/net/aspose.slides.export/pdfoptions/).

**Stöder biblioteket teckensnittsinbäddning vid export till PDF, med fininställning av vad som inbäddas?**

Ja. Du kan kontrollera om teckensnitt är helt inbäddade eller delmängdsinbäddade (endast använda tecken), ange hur vanliga systemteckensnitt hanteras och konfigurera beteendet för ASCII‑text via [PDF export options](https://reference.aspose.com/slides/sv/net/aspose.slides.export/pdfoptions/).

**Kan jag upptäcka om en fil är lösenordsskyddad innan den faktiskt läses in?**

Ja. Med hjälp av det fabrikbaserade inspektions‑API:et kan du undersöka en presentationsfil för att avgöra om den är lösenordsskyddad utan att öppna den fullt ut.

**Finns det mekanismer för teckensnittsfallback och stöd för anpassade teckensnitt?**

Ja. Biblioteket stöder inläsning och inbäddning av anpassade teckensnitt och tillhandahåller regler för teckensnittsfallback för att förhindra saknade tecken under rendering och konvertering.

**Kan jag exportera bilder till XPS, och finns det alternativ för att finjustera XPS‑utdata?**

Ja. Export till XPS stöds, och du kan justera relevanta [save options](https://reference.aspose.com/slides/sv/net/aspose.slides.export/xpsoptions/) för att kontrollera utskriftskvaliteten och innehållet i XPS‑dokumentet.
