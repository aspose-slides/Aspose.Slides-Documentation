{{% alert color="primary" %}} 

Aspose.Slides for .NET är ett klassbibliotek som möjliggör för .NET‑applikationer att läsa och skriva presentationsdokument utan att använda Microsoft PowerPoint® eller andra. Aspose.Slides for .NET låter dig hantera text, former, tabeller och animationer, lägga till ljud och video i bilder, förhandsgranska bilder, exportera bilder till SVG, PDF‑format och mer.

Den senaste API‑referensen finns på [Aspose.Slides for .NET API Reference](https://reference.aspose.com/slides/sv/net).
