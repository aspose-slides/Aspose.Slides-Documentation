**Välkommen till Aspose.Slides för Node.js via .NET**

![Aspose.Slides för Node.js via .NET produktlogo](aspose_slides-for-nodejs-via-net.png)

Aspose.Slides för Node.js via .NET är ett klassbibliotek som låter dina applikationer läsa och skriva PowerPoint®-dokument utan att använda Microsoft PowerPoint®.

Aspose.Slides för Node.js via .NET är den första och enda komponenten som erbjuder funktionalitet för att hantera PowerPoint®-dokument.

Aspose.Slides för Node.js via .NET tillhandahåller många viktiga funktioner såsom hantering av text, former, tabeller och animationer, lägga till ljud och video på bilder, förhandsgranska bilder, exportera bilder till SVG, PDF-format och mer.
