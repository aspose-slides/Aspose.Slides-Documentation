Aspose.Slides för Node.js via .NET kräver inte någon tredjepartsprodukt såsom Microsoft PowerPoint för att vara installerad. Aspose.Slides är i sig själva en motor för att skapa, modifiera, konvertera och rendera dokument i olika format, inklusive Microsoft PowerPoint-presentationformat.

## Stödda operativsystem

Aspose.Slides för .NET stödjer alla 32-bitars eller 64-bitars operativsystem som kör .NET-runtime, inklusive men inte begränsat till:

### **Windows**
- Microsoft Windows 2003 Server ( x64, x86)
- Microsoft Windows 2008 Server ( x64, x86)
- Microsoft Windows 2012 Server ( x64, x86)
- Microsoft Windows 2012 R2 Server ( x64, x86)
- Microsoft Windows 2016 Server ( x64, x86)
- Microsoft Windows 2019 Server ( x64, x86)
- Microsoft Windows Vista ( x64, x86)
- Microsoft Windows XP ( x64, x86)
- Microsoft Windows 7 ( x64, x86)
- Microsoft Windows 8, 8.1 ( x64, x86)
- Microsoft Windows 10 ( x64, x86)
- Microsoft Windows 11 ( x64, x86)

### **Linux**
- Linux (Ubuntu, CentOS and others)

### **Mac**
- Mac OS X
