## **Stödda plattformar**
Aspose.Slides for Node.js via .NET är ett plattformsoberoende API. Det kan användas på vilken plattform som helst (Windows, Linux, macOS osv.) där Node.js 7 eller högre är installerat. Oracle JDK 8 eller högre är ett annat krav.

## **Filformat och konverteringar**
Aspose.Slides for Node.js via .NET stödjer de flesta PowerPoint-dokumentformat. Det låter dig även exportera dem till de vanliga filformat som organisationer använder och delar sinsemellan. Gå igenom detaljerna:

|**Funktion**|**Beskrivning**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via .NET ger den snabbaste behandlingen för detta presentationsdokumentformat.|
|PPT to PPTX conversion|Aspose.Slides for Node.js via .NET stöder konvertering från PPT till PPTX|
|Portable Document Format (PDF)|Du kan exportera alla stödda filformat till Adobe Portable Document Format (PDF)-dokument med en enda metod.|
|XML Parser Specification (XPS)|Du kan exportera alla stödda filformat till XML Parser Specification (XPS)-dokument med en enda metod.|
|Tagged Image File Format (TIFF)|Du kan exportera alla stödda presentationsfilformat till Tagged Image File Format (TIFF).|
|PPTX To HTML Conversion|Aspose.Slides for Node.js via .NET stöder konvertering av PresentationEx till HTML-format.|

## **Rendering och utskrift**
Aspose.Slides for Node.js via .NET stödjer högkvalitativ rendering av bilder i en presentation till olika grafikformat. Gå igenom detaljerna:

|**Funktion**|**Beskrivning**|
| :- | :- |
|.NET Supported Image Formats|Med Aspose.Slides för Node.js via .NET kan du rendera presentationsbilder och bilder på bilder till alla .NET‑stödda grafikformat såsom TIFF, PNG, BMP, JPEG, GIF.|
|SVG Format|Aspose.Slides för Node.js via .NET tillhandahåller även inbyggda metoder som låter dig exportera presentationsbilder till Scalable Vector Graphics (SVG)-format.|
|Presentation Printing|De senaste versionerna av Aspose.Slides för Node.js via .NET erbjuder inbyggda utskriftsmetoder med olika alternativ.|
