## **Teknisk support**
{{% alert color="primary" %}}

Aspose erbjuder obegränsad gratis teknisk support för alla sina produkter. Supporten är tillgänglig för alla användare (inklusive användare med utvärderingspaket).

{{% /alert %}} 

När du behöver hjälp med Aspose.Slides för Node.js via .NET, överväg följande:

- Den främsta vägen för support är [Aspose.Forums](https://forum.aspose.com/c/slides/sv/11). Posta din fråga i Aspose.Slides‑forumet—den besvaras inom några timmar. Aspose.Slides supportteam svarar direkt på frågor som postas i forumet.
- Observera att Aspose inte erbjuder teknisk support via telefon. Telefonsupport är endast tillgänglig för försäljnings- och köpfrågor.
- När du väntar på ett svar i forumet, var tålmodig och ta hänsyn till tidszonskillnader.

När du upplever ett problem med Aspose.Slides för Node.js via .NET, följ dessa riktlinjer för att lösa det på det mest effektiva sättet:

- Kontrollera och bekräfta att du använder den senaste versionen av Aspose.Slides för Node.js via .NET innan du rapporterar problemet. Kanske var problemet du upplevde redan löst i en uppdatering. 

  Se [Aspose.Slides download](https://releases.aspose.com/slides/sv/nodejs-net/) för att få information om den senaste versionen.

- Innan du rapporterar ett problem kan du vilja gå igenom våra forum, den här dokumentationen och API‑referensen. Kanske har svaret på din fråga redan getts. 

- När du rapporterar ett problem, inkludera det ursprungliga dokumentet och (om möjligt) ett kodfragment som rör problemet. Om du behöver bifoga flera filer, lägg dem i ett ZIP‑paket. Du kan säkert bifoga dina dokument i Aspose.Forums eftersom **endast** du och Aspose‑utvecklare har åtkomst till de bifogade filerna.

- Skapa och använd ett tråd för ett specifikt problem. Om du upplever ett annat problem måste du skapa en ny tråd för att rapportera det.
