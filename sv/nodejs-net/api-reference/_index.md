Aspose.Slides for Node.js via .NET är ett klassbibliotek som möjliggör för Node.js‑applikationer att läsa och skriva presentationsdokument utan att använda Microsoft PowerPoint® eller andra. Aspose.Slides for Node.js via .NET låter dig hantera text, former, tabeller och animationer, lägga till ljud och video i bilder, förhandsgranska bilder, exportera bilder till SVG, PDF‑format och göra mycket mer.

Aspose.Slides for Node.js via .NET använder [samma API‑referens som Aspose.Slides .NET](https://reference.aspose.com/slides/sv/nodejs-net/). 
{{% /alert %}}
