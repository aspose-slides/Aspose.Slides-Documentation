## **Programvarukrav**
Se till att du uppfyller följande krav innan du laddar ner och kör exemplen.

1. Visual Studio 2015 eller högre.
## **Ladda ner Aspose.Slides från GitHub**
Alla exempel på Aspose.Slides för C++ finns på [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-C).

- Du kan antingen klona förvaret med din föredragna GitHub-klient eller ladda ner ZIP-filen från [here](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/refs/heads/master.zip).
- Extrahera innehållet i ZIP-filen till någon mapp på din dator. Alla exempel finns i mappen **Examples** .
- Det finns en C++ Visual Studio‑lösningsfil.
- Projekten skapades i Visual Studio 2015, men lösningsfilerna är kompatibla med Visual Studio 2015 och senare versioner.
- Öppna lösningsfilen i Visual Studio och bygg projektet.
- Ladda ner den senaste versionen av Aspose.Slides för C++ från [Download Section](https://downloads.aspose.com/slides/sv/cpp).
- Extrahera utgåvan och kopiera katalogerna **Include** och **Bin** till mappen **Examples**.
- Öppna **main.cpp**‑filen, alla exempel anropas därifrån.
- Avkommentera de exempel du vill köra i projektet.
- Bygg och kompilera lösningen

Tveka inte att kontakta oss via våra forum om du har problem med att konfigurera eller köra exemplen.
## **Bidra**
Om du vill lägga till eller förbättra ett exempel uppmuntrar vi dig att bidra till projektet. Alla exempel och showcase‑projekt i detta förvar är öppen källkod och kan fritt användas i dina egna applikationer.

För att bidra kan du fork:a förvaret, redigera källkoden och skapa en pull‑request. Vi kommer att granska ändringarna och inkludera dem i förvaret om de är hjälpsamma.
