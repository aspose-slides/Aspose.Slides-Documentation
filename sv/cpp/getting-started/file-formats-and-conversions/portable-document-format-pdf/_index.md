## **Om PDF**
Det [Portable Document Format](https://en.wikipedia.org/wiki/PDF) är ett filformat som skapades av Adobe System för utbyte av dokument mellan olika organisationer. Syftet med detta format var att möjliggöra att dokumentens innehåll kan representeras på ett sätt så att deras visuella utseende inte är beroende av den plattform som de visas på.

## **PDF i Aspose.Slides för C++**
Alla presentationsdokument som kan läsas in i Aspose.Slides for C++ kan konverteras till PDF‑dokument som kan följa [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A) eller [PDF /A-1b](https://en.wikipedia.org/wiki/PDF/A) beroende på ditt val. Aspose.Slides for C++ exporterar presentationsdokumenten till PDF på ett sätt så att de exporterade PDF‑dokumenten oftast ser nästan likadana ut som originalpresentationen.  
Aspose‑lösningen stöder följande funktioner i presentationsdokumenten vid konvertering till PDF‑dokument:

- Bilder, textrutor och andra former
- Text och formatering
- Stycken och formatering
- Hyperlänkar
- Sidhuvuden och sidfötter
- Punkter
- Tabeller  

Du kan exportera presentationsdokumenten till PDF‑dokument direkt med endast Aspose.Slides for C++‑komponenten. Det betyder att du inte behöver någon annan tredjeparts‑ eller Aspose.Pdf‑komponent för detta ändamål. Dessutom kan du anpassa exporten från presentation till PDF med olika alternativ som förklaras i [detta ämne](http://docs.aspose.com/display/slidesnet/Converting+to+PDF+File).
