## **Om XPS**
XML Parser Specification är ett sidbeskrivningsspråk och ett fast dokumentformat som ursprungligen utvecklades av Microsoft. Likt PDF är XPS ett fast layout-dokumentformat utformat för att bevara dokumentets äkthet och erbjuda en enhetsoberoende dokumentutseende.
## **XPS i Aspose.Slides för C++**
Alla presentationsdokument som kan läsas in av Aspose.Slides för C++ kan konverteras till XPS-format. Aspose.Slides för C++ använder den högupplösta sidlayouten och renderingsmotorn för att producera utdata i fast layout XPS-dokumentformat.
Det är värt att nämna att Aspose.Slides för C++ genererar XPS direkt utan att vara beroende av Windows Presentation Foundation (WPF)-klasserna som ingår i C++ Framework 3.5, vilket gör att Aspose.Slides för C++ kan skapa XPS-dokument på maskiner som kör C++ Framework-versioner tidigare än version 3.5. Du kan lära dig mer om hur du exporterar presentationsdokument till XPS-dokument via Aspose.Slides för C++ i [detta ämne](/slides/sv/net/).
