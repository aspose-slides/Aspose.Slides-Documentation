{{% alert color="primary" %}} 

Aspose.Slides for C++ är ett klassbibliotek som gör det möjligt för C++-applikationer att läsa och skriva presentationsdokument utan att använda Microsoft PowerPoint® eller andra. Aspose.Slides for C++ gör det möjligt att hantera text, former, tabeller och animationer, lägga till ljud och video i bilder, förhandsgranska bilder, exportera bilder till SVG, PDF-format och mer.

Den senaste API-referensen finns på [Aspose.Slides for C++ API Reference](https://reference.aspose.com/slides/sv/cpp/).
