**Välkommen till Aspose.Slides för C++**

Aspose.Slides för C++ är ett Microsoft PowerPoint®‑hanterings‑API som gör det möjligt för C++‑applikationer att läsa och skriva PowerPoint®‑dokument utan att använda Microsoft PowerPoint®. Aspose.Slides för C++ är den första och enda komponenten som erbjuder funktionalitet för att hantera PowerPoint®‑dokument. Aspose.Slides för C++ erbjuder många viktiga funktioner, såsom hantering av text, former, tabeller och animationer, att lägga till ljud och video i bilder, förhandsgranskning av bilder, export av bilder till SVG, PDF‑format och mer.
