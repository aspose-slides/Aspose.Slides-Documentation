## **Välkommen till Aspose.Slides för Reporting Services!**

Aspose.Slides för Reporting Services är den enda lösningen på marknaden som gör det möjligt att generera riktiga PPT‑ och PPS‑rapporter i Microsoft SQL Server 2005, 2008, 2012, 2016 och 2017 Reporting Services (32‑bit och 64‑bit). Alla RDL‑rapportfunktioner, inklusive tabeller, matriser, diagram och bilder, konverteras med högsta precision till Microsoft PowerPoint‑presentationer.

## **Produktöversikt**

Microsoft SQL Server Reporting Services har inga inbyggda möjligheter att exportera rapporter som Microsoft PowerPoint‑presentationer, men efter att ha installerat Aspose.Slides för Reporting Services på din server får du tillgång till ytterligare exportformat:

- PPT – PowerPoint‑presentation via Aspose.Slides
- PPS – PowerPoint‑diasshow via Aspose.Slides
- PPTX – PowerPoint 2007‑presentation via Aspose.Slides
- PPSX – PowerPoint 2007‑diasshow via Aspose.Slides

Aspose.Slides för Reporting Services skapar presentationer på servern utan att använda Microsoft PowerPoint. Aspose.Slides för Reporting Services använder internt Aspose.Slides för .NET – den världsledande komponenten för serverbaserad presentationbearbetning.

**Aspose.Slides för Reporting Services gör det möjligt att exportera vilken rapport som helst i PPT‑, PPS‑, PPTX‑ eller PPSX‑format.**
