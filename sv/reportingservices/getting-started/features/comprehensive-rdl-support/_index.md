{{% alert color="primary" %}} 
Aspose.Slides för Reporting Services stöder fullt ut alla RDL‑specifikationer. Det innebär två stora fördelar: 

- Ingen anledning att redesigna befintliga rapporter. Du kan exportera vilken befintlig RDL‑rapport som helst som en Microsoft PowerPoint‑presentation och den kommer att visas exakt enligt RDL‑designen.
- Ingen anledning att använda en specifik rapportdesigner. Du kan använda vilken RDL‑rapportdesigner som helst och rapporten exporteras exakt på det sätt du designade den.
{{% /alert %}} 

Aspose.Slides för Reporting Services stöder följande RDL‑element: 

- Sida, sidhuvuden, sidfötter
- Textrutor
- Bilder
- Delrapporter
- Diagram
- Listor
- Tabeller
- Matriser
- Stilar
- Rektanglar
