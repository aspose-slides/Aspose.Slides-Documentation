{{% alert color="primary" %}} 

Microsoft SQL Server Reporting Services har inte inbyggda funktioner för att exportera rapporter som Microsoft PowerPoint‑presentationer, men efter att du har installerat Aspose.Slides for Reporting Services på din server får du tillgång till två ytterligare exportformat: 

- Microsoft PowerPoint Presentation (PPT)
- Microsoft PowerPoint SlideShow (PPS)
- Microsoft PowerPoint 2007 Presentation (PPTX)
- Microsoft PowerPoint 2007 SlideShow (PPSX)

{{% /alert %}} 

För att skapa dokument i dessa format förlitar sig Aspose.Slides for Reporting Services på en inbyggd version av [Aspose.Slides for .NET](http://www.aspose.com/Products/Aspose.Slides/), den marknadsledande komponenten för presentationsbearbetning från Aspose. Microsoft PowerPoint används inte för att generera dokumenten. 


PPT‑ och PPS‑kopior av samma rapport kommer att se identiska ut, tack vare den höga graden av stöd för de underliggande filformaten som tillhandahålls av Aspose.Slides.
