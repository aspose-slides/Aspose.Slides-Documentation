En ny [Aspose.Slides for Java API](https://docs.aspose.com/slides/sv/java/) har släppts och nu stödjer denna enda produkt möjligheten att skapa PowerPoint-dokument från grunden och redigera befintliga.

## **Stöd för äldre kod**
För att använda den äldre koden som utvecklats med Aspose.Slides for Java versioner innan 13.x måste du göra några mindre ändringar i din kod så att den fungerar som tidigare. Alla klasser som fanns i den gamla Aspose.Slides for Java under namnutrymmena Aspose.Slide och Aspose.Slides.Pptx är nu sammanslagna i ett enda Aspose.Slides‑namnutrymme. Titta på följande enkla kodexempel för att lägga till sidhuvud och sidfot i en presentation i den äldre Aspose.Slides‑API:n och följ stegen som beskriver hur du migrerar till den nya sammanslagna API:n.
