En ny [Aspose.Slides for Java API](https://docs.aspose.com/slides/sv/java/) har släppts och nu stödjer detta enda produkt möjligheten att generera PowerPoint-dokument från början och redigera befintliga.
{{% /alert %}} 
## **Stöd för äldre kod**
För att använda den äldre koden som utvecklats med Aspose.Slides for Java-versioner före 14.x.x måste du göra några mindre ändringar i din kod så att den fungerar som tidigare. Alla klasser som fanns i den gamla Aspose.Slides for Java under namnutrymmena com.aspose.slides och com.aspose.slides.pptx är nu sammanslagna till ett enda namnutrymme com.aspose.slides. Se det följande enkla kodexemplet för att skapa ett vanligt diagram från början i en presentation med den äldre Aspose.Slides API:n och följ stegen som beskriver hur du migrerar till det nya sammanslagna API:t.
