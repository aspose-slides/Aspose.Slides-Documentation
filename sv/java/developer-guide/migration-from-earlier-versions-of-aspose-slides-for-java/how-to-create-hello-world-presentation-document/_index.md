En ny [Aspose.Slides for Java API](/slides/sv/java/) har släppts och nu stödjer denna enda produkt möjligheten att skapa PowerPoint‑dokument från grunden och redigera befintliga.
{{% /alert %}} 
## **Stöd för äldre kod**
För att kunna använda den äldre koden som utvecklats med Aspose.Slides for Java versioner tidigare än 13.x måste du göra några små ändringar i din kod så att den fungerar som tidigare. Alla klasser som fanns i den gamla Aspose.Slides for Java under namnrymderna Aspose.Slide och Aspose.Slides.Pptx har nu slagits ihop i en enda Aspose.Slides‑namnrymd. Titta på följande enkla kodexempel för att skapa ett Hello World Presentation‑dokument i det äldre Aspose.Slides‑API‑et och följ stegen som beskriver hur du migrerar till det nya sammanslagna API‑et.
