{{% alert color="primary" %}} 

Aspose.Slides for Java är ett klassbibliotek som gör det möjligt för Java-applikationer att läsa och skriva presentationsdokument utan att använda Microsoft PowerPoint® eller andra. Aspose.Slides for Java möjliggör hantering av text, former, tabeller och animationer, lägga till ljud och video i bilder, förhandsgranska bilder, exportera bilder till SVG, PDF-format och mer.

Den senaste API-referensen finns på [Aspose.Slides för Java API-referens](https://reference.aspose.com/slides/sv/java).
