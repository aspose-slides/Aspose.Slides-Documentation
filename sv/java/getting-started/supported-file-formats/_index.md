## **Översikt**

Aspose.Slides stöder presentationsfiler från Microsoft PowerPoint 97 till Office 365, inklusive Microsoft PowerPoint för Mac. Denna artikel listar de PowerPoint‑versioner som biblioteket stödjer och innehåller en tabell med filformat som kan läsas, sparas eller både och.

Artikeln besvarar också vanliga frågor om PDF‑efterlevnad, inbäddning av typsnitt, lösenordsskyddade filer, egna typsnitt, typsnittsreserv och XPS‑exportalternativ.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint for MAC
- Office 365

## **Vanliga frågor**

**Kan jag spara presentationer till PDF som uppfyller arkiverings- och tillgänglighetsstandarder (PDF/A och PDF/UA)?**

Ja. Aspose.Slides stöder export till PDF med efterlevnadsnivåer såsom PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b samt PDF/UA via inställningen [compliance](https://reference.aspose.com/slides/sv/java/com.aspose.slides/pdfoptions/#setCompliance-int-) i [PDF export options](https://reference.aspose.com/slides/sv/java/com.aspose.slides/pdfoptions/).

**Stöder biblioteket inbäddning av typsnitt vid export till PDF, med fininställd kontroll över vad som inbäddas?**

Ja. Du kan kontrollera om typsnitt ska inbäddas helt eller delvis (endast använda glyfer), ange hur vanliga systemtypsnitt behandlas och konfigurera beteende för ASCII‑text via [PDF export options](https://reference.aspose.com/slides/sv/java/com.aspose.slides/pdfoptions/).

**Kan jag upptäcka om en fil är lösenordsskyddad innan den faktiskt laddas?**

Ja. Med hjälp av det fabrikbaserade inspektions‑API:t kan du fråga efter en presentationsfil för att avgöra om den är lösenordsskyddad utan att öppna den helt via [factory-based inspection API](https://reference.aspose.com/slides/sv/java/com.aspose.slides/presentationfactory/).

**Finns det mekanismer för typsnittsreserv och stöd för egna typsnitt?**

Ja. Biblioteket stöder [loading](/slides/sv/java/custom-font/) och [embedding](/slides/sv/java/embedded-font/) av egna typsnitt samt tillhandahåller [fallback rules](/slides/sv/java/fallback-font/) för att förhindra saknade glyfer under rendering och konvertering.

**Kan jag exportera bilder till XPS, och finns det alternativ för att justera XPS‑utdata?**

Ja. [Export to XPS](/slides/sv/java/convert-powerpoint-to-xps/) stöds, och du kan justera relevanta [save options](https://reference.aspose.com/slides/sv/java/com.aspose.slides/xpsoptions/) för att styra utskriftskvaliteten och innehållet i XPS‑dokumentet.
