## **Översikt**

Denna artikel beskriver ett licensproblem som uppstod i Aspose.Slides for Java versionerna 23.2, 23.3, 23.4, 23.5, 23.6 och 23.7. Detta fel gjorde att eviga licenser blev tillfälligt oanvändbara efter att deras prenumeration gått ut.

## **Symptom**

När din eviga licensprenumeration löper ut kan Aspose.Slides for Java versionerna 23.2‑23.7 rapportera licensen som utgången och sätta alla funktioner i ett evalueringsläge. Detta beteende är felaktigt och påverkar inte versioner före 23.2 och efter 23.7.

## **Lösning**

Aspose.Slides for Java har åtgärdat problemet och släppt uppdaterade versioner (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) med korrigeringen.  

Om du använder någon av de drabbade versionerna av Aspose.Slides for Java i ditt projekt, byt till den korrigerade versionen.

Om du upplever några licensproblem med Aspose.Slides for Java:

- Kontrollera vilken version du använder och säkerställ att den inte påverkas av felet som beskrivits ovan.
- Se [Aspose-dokumentationen](https://docs.aspose.com/slides/sv/java/getting-started/) för felsökningssteg och kända licensproblem.
- Kontakta [Aspose support](https://forum.aspose.com/) för ytterligare hjälp.

## **Påverkade produkter och versioner**

Observera att detta fel endast påverkar Aspose.Slides for Java versionerna 23.2 till 23.7. **Andra Aspose‑produkter och andra versioner av Aspose.Slides påverkas inte**.
