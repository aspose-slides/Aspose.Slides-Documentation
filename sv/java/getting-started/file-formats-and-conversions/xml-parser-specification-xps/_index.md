Den [XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) är ett sidbeskrivningsspråk och ett fast‑dokumentformat som ursprungligen utvecklades av Microsoft. Likt PDF är XPS utformat för att bevara dokumentets kvalitet och erbjuda en enhetsoberoende visning av dokumentet.

{{% /alert %}}

## **XPS i Aspose.Slides för Java**
Alla presentationsdokument som kan läsas in av Aspose.Slides för Java kan konverteras till XPS‑format. Aspose.Slides för Java använder den högupplösta sidlayouten och renderingsmotorn för att producera output i fast‑layout XPS‑dokumentformat.
Du kan läsa om export av presentationsdokument till XPS‑dokument via Aspose.Slides för Java i [Konvertera till XPS](https://docs.aspose.com/slides/sv/java/convert-powerpoint-to-xps/).
