## **Om TIFF**

Den [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) som är känd för att lagra flera bilder i ett dokument skapades ursprungligen av Aldus. Detta format stöds brett av skanning, faxning och andra bildmanipuleringsapplikationer.

## **TIFF i Aspose.Slides för Java**

Alla dokument som kan läsas in i Aspose.Slides för Java kan också konverteras direkt till TIFF-dokument av Aspose.Slides för Java, vilket eliminerar behovet av någon tredjepartskomponent. Dessutom kan du valfritt ange bildstorleken i det resulterande TIFF-dokumentet. Du kan hitta information om att exportera presentationsdokument till TIFF-dokument via Aspose.Slides för Java i [detta ämne](/slides/sv/java/converting-a-presentation/).
