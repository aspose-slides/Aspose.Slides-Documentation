PPT till PPTX‑konvertering stöds av Aspose.Slides för Java. De flesta presentationsfunktioner – masterbilder, struktur och så vidare – behålls vid konvertering från ett format till det andra, men det finns [några begränsningar](/slides/sv/java/ppt-to-pptx-conversion/).

## **Funktioner som stöds i konvertering**
Aspose.Slides för Java erbjuder partiellt stöd för att konvertera PPT‑filformatet till PPTX. Stödet för konvertering har precis introducerats i Aspose.Slides för Java, så det har några begränsningar och fungerar bäst för enkla presentationer. Den största fördelen som Aspose.Slides för Java ger vid konvertering från PPT till PPTX är hur enkelt API:et är att använda. För att se kodexempel, läs om [Konvertera PPT till PPTX](). Nedan förklarar listorna vilka funktioner som stöds och vilka som inte gör det för PPT‑till‑PPTX‑konvertering.

## **Stödda funktioner**
Följande funktioner stöds för konvertering:

- Konvertera strukturen för masterbilder, layouter och bilder.
- Konvertera diagram.
- Grupperade former.
- Konvertera autogjorda former inklusive rektanglar och ellipser. 
- Former med anpassad geometri.
- Textur- och bildfyllningsstilar för autogjorda former.
- Konvertera platshållare.
- Konvertera linjer och polylinjer.
- Linje- och fyllningsformat.
- Gradientfyllningsstilar.
- OLE‑ramar, tabeller, video‑ och audio‑ramar osv.
- Animations‑ och bildspelsinställningar.
- Konvertera text i textramar och textbehållare.
