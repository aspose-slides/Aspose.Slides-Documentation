{{% alert color="primary" %}} 

[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) är presentationsdokumentfilformatet som kan skapas, läsas, manipuleras och skrivas av olika versioner av Microsoft PowerPoint. Detta är det binära formatet för presentationsdokument som utvecklats av Microsoft.

{{% /alert %}} 

## **PPT i Aspose.Slides för Java**
Aspose.Slides for Java kan läsa PPT‑filer som skapats av programvaran som listas nedan.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

På samma sätt kan PPT‑filer som skapats av Aspose.Slides for Java läsas av programvarorna ovan.

## **Omfattande stöd för PPT**
Aspose.Slides for Java erbjuder stöd för nästan alla funktioner som PPT-dokumentfilformatet stödjer. Det täcker inte bara de grundläggande och avancerade funktionerna som tillhandahålls av olika Microsoft PowerPoint‑versioner för manipulation av PPT‑dokument, utan även funktioner som inte ens stöds av Microsoft PowerPoint. Den största fördelen med att använda Aspose.Slides for Java API‑biblioteket är dess användarvänlighet när det gäller hantering av sådana funktioner.

Förutom de grundläggande uppgifterna för att skapa, läsa och skriva PPT-dokumentfiler finns flera funktioner som tillhandahålls av Aspose.Slides for Java:

- Importera andra Microsoft Office‑filformat som [OLE-objekt i PPT‑dokument]().
- [Exportera PPT‑dokument till PDF](/slides/sv/java/convert-powerpoint-ppt-and-pptx-to-pdf/).
- Exportera bilder i PPT‑dokument till SVG‑format.
- Rendera bilder till vilket bildformat som helst som stöds av Java‑ramverket.
- Ange storlek på bilder i PPT‑dokument.
- Hantera animationer på former.
- Hantera bildspel.
- [Formatera text på bilder]().
- Extrahera text från PPT‑dokument.
- [Hantera tabeller på bilder]().
- Kopiera automatiskt master‑bilder med [kloningsfunktionen]().
