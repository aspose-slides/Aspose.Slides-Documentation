[Portable Document Format](https://en.wikipedia.org/wiki/PDF) är ett filformat som skapades av Adobe Systems för att utbyta dokument mellan organisationer. Syftet med formatet var att behålla innehåll och layout lika, oavsett på vilken plattform det visades. Aspose.Slides for Java låter dig konvertera presentationsfiler till PDF.

{{% /alert %}} 

## **PDF i Aspose.Slides for Java**
Alla presentationer som kan läsas in i Aspose.Slides for Java kan konverteras till en PDF som följer [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) eller [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA) beroende på ditt val. Aspose.Slides for Java exporterar presentationer till PDF och i de flesta fall ser den resulterande PDF:n exakt likadan ut som den ursprungliga presentationen.

Aspose.Slides stödjer följande presentationsfunktioner vid konvertering till PDF:

- Bilder, textrutor och andra former.
- Text och formatering.
- Stycken och formatering.
- Hyperlänkar.
- Sidhuvuden och sidfötter.
- Punkter.
- Tabeller.

Du kan exportera presentationer till PDF direkt med Aspose.Slides for Java: du behöver ingen annan komponent. Dessutom kan du anpassa exporten av presentation till PDF med olika alternativ som förklaras i [Konvertera till PDF](/slides/sv/java/converting-a-presentation/).
