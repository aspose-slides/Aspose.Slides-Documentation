## Aspose.Slides for .NET

{{% alert color="primary" %}}

![Aspose.Slides for .NET Product Logo](home_1.png)

Aspose.Slides for .NET är ett Microsoft PowerPoint®‑hanterings‑API som gör det möjligt för .NET‑applikationer att läsa och skriva PowerPoint®‑dokument utan att använda Microsoft PowerPoint®. Aspose.Slides for .NET är den första och enda komponenten som tillhandahåller funktionalitet för att hantera PowerPoint®‑dokument. Aspose.Slides for .NET erbjuder många viktiga funktioner såsom hantering av text, former, tabeller och animationer, lägga till ljud och video på bilder, förhandsgranska bilder, exportera bilder till SVG, PDF‑format och mer.
