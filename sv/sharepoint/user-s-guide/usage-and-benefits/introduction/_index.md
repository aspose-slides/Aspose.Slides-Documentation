SharePoint‑webbplatser är avsedda för samarbete, kommunikation och lagring av innehåll för att uppfylla specifika affärsbehov. De möjliggör att en grupp individer kan nå ett gemensamt mål genom att underlätta informationsdelning och kommunikation. Informationen finns oftast som en webbplats och nås via en webbläsare. 

{{% /alert %}} 
## **Vad är SharePoint och Aspose.Slides?**
Aspose.Slides för SharePoint är en flexibel lösning för att konvertera Microsoft PowerPoint‑dokument till andra format som PPT, POT, PPS, PPTX, PPSX och POTX. Aspose.Slides för SharePoint låter dig läsa och konvertera PowerPoint‑filer i en SharePoint‑applikation utan att använda Microsoft PowerPoint.
