För att avinstallera Aspose.Slides för SharePoint:

1. Kör installationsprogrammet.
   Om Aspose.Slides för SharePoint redan är installerat föreslår installationsprogrammet att ta bort eller reparera det.
1. Välj **Remove** för att avinstallera Aspose.Slides för SharePoint.
