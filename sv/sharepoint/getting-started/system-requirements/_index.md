## **Serverkrav**
{{% alert color="primary" %}} 

För att installera Aspose.Slides för SharePoint krävs ett system som kör någon av följande:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft Office SharePoint Server 2010
- Microsoft Office SharePoint Server 2013
- Microsoft Office SharePoint Server 2016
- Microsoft Office SharePoint Server 2019

{{% /alert %}} {{% alert color="primary" %}} 

Det rekommenderas att installera Service Pack 2 eller senare för dessa produkter.

{{% /alert %}} 
## **Klientkrav**
{{% alert color="primary" %}} 

SharePoint-webbplatser nås via en webbläsare. Aspose.Slides för SharePoint har testats med följande webbläsare:

- Microsoft Internet Explorer 6, 7, 8
- Mozilla FireFox 3.5

{{% /alert %}}
