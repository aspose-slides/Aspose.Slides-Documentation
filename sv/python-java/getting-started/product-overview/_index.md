## **Produktbeskrivning**

Aspose.Slides for Python via Java är ett plattformsoberoende klassbibliotek som låter dina applikationer utföra en mängd olika dokumentbehandlingsuppgifter utan att använda Microsoft PowerPoint®.

### **Varför du bör använda Aspose.Slides för Python via Java**

- Aspose.Slides for Python via Java är den första och enda komponenten som erbjuder de funktioner som krävs för att hantera PowerPoint®‑dokument. 
- Aspose.Slides for Python via Java erbjuder många viktiga funktioner såsom hantering av text, former, tabeller och animationer, samt lägga till ljud och video på bilder, förhandsgranska bilder, exportera bilder till SVG, PDF och andra format, och många fler möjligheter. 
- Aspose.Slides for Python via Java låter dig inte bara öppna PowerPoint‑filer från olika källor utan också spara dina presentationer på olika sätt. Till exempel kan du spara presentationerna i PPT; du kan också spara bilderna som bilder. 
- Aspose.Slides for Python via Java ger dig nästan alla funktioner som du eventuellt saknar i Microsoft PowerPoint. Med tanke på Aspose.Slides for Python via Javas mångsidighet får du, förutom tabeller och olika typer av former, möjlighet att lägga till olika typer av ramar – text‑, ljud‑ och videoramar – i dina bilder. 

## **Aspose.Slides-funktioner**

Aspose.Slides for Python via Java‑funktioner kan delas in i dessa grupper:

- Presentationsfunktioner.
- Bildfunktioner.
- Grafik‑ och multimediefunktioner.
- Tabellfunktioner.
- Text‑ och stycke‑funktioner
- Avancerade funktioner

### **Presentationsfunktioner**

- Öppna Microsoft PowerPoint‑presentationer från olika källor.
- Spara presentationer i olika Microsoft PowerPoint‑format.
- Skapa presentationer från grunden.
- Konvertera presentationer till PDF‑dokument utan att använda någon annan komponent
- Läsa eller ändra en presentations dokumentegenskaper
- Applicera skydd på presentationen
- Exportera presentation till PDF
- Exportera presentation till TIFF
- Exportera presentation till XPS
- Exportera presentation till SWF
- Exportera presentation till PDF‑anteckningar / Tiff‑anteckningar
- Exportera presentation till HTML

### **Bildfunktioner**

- Skapa nya bilder från grunden och lägga till dem i presentationer.
- Åtkomst till vilken bild som helst via dess position i en presentation.
- Kopiera eller klona bilder till samma eller en annan presentation.
- Redigera och radera bilder.
- Kopiera masterbilder med bilder automatiskt.
- Förhandsgranska bilder.
- Hämta eller ange en bilds namn.
- Spara bilder som bilder.
- Exportera bilder till SVG‑format.
- Arbeta med många masterbilder i presentationer.
- Fyll bildbakgrund med någon av de stödjade stilarna.
- Lägga till eller hantera bildanteckningar.
- Ställa in nya masterbilder och stilar dynamiskt.
- Lägga till eller ändra bildspelsövergång.
- Lägga till Morph‑övergångar

### **Grafik‑ & multimediefunktioner**

- Skapa eller hantera former såsom linjer, rektanglar, ellipser och polylinjer.
- Hantera linjestilar i former.
- Hämta en formes alternativa text.
- Rotera och vända former.
- Fyll former i olika stilar – till exempel solid, gradient, mönster, textur och bild.
- Använda någon av 48 fördefinierade mönster och 24 texturer.
- Tilldela länkar till former.
- Läsa, ändra och lägga till nya animeringseffekter till bilder och former.
- Lägga till nya bilder i presentationer och hantera befintliga.
- Lägga till nya bildpunkter i presentationer och hantera befintliga.
- Lägga till nya ljudramar i presentationer och hantera befintliga.
- Lägga till nya videoramar i presentationer och hantera befintliga.

### **Tabellfunktioner**

- Skapa tabeller från grunden och lägga till dem i bilder.
- Ändra storlek och flytta tabeller.
- Åtkomst till vilken cell som helst i en tabell.
- Ändra format och kanter för vilken cell som helst.
- Ändra storlek på kolumner och rader.
- Lägga till och radera kolumner och rader.
- Ställa in kantbredd och färg för hela tabellen.

### **Text‑ och stycke‑funktioner**

- Hämta och ange texter i textplatshållare och textramar.
- Hämta en platshållares typ.
- Ändra textformatering i en textram.
- Lägga till nya teckensnitt eller hantera teckensnitt som redan används i en presentation.
- Ändra texter och textformatering.
- Skapa eller hantera länkar i texten.
- Stöd för flikar och effektiva flikar.
- Lägga till nya textramnar i nästan vilken form som helst.
- Åtkomst till och ändra marginaler och textomslag för textramnar.
- Åtkomst till och ändra styckeindrag.

### **Avancerade funktioner**

- Åtkomst till OLE‑objekt inbäddade i presentationer.
- Låsa presentationer.

## **Teknisk support**

Aspose tillhandahåller obegränsad gratis teknisk support för alla sina produkter. Supporten är tillgänglig för alla användare (inklusive användare med utvärderingspaket). Om du behöver hjälp med Aspose.Slides for Python via Java, överväg följande:

- Huvudvägen för support är [Aspose Forums](https://forum.aspose.com/). Publicera din fråga i [Aspose.Slides‑forumet](https://forum.aspose.com/c/slides/sv/11)—så svaras den inom några timmar. Aspose.Slides‑supportteamet svarar direkt på frågor som postas i forumet.
- Observera att Aspose inte ger teknisk support via telefon. Telefonsupport är endast tillgänglig för försäljnings‑ och köpsfrågor.
- När du väntar på svar i forumet, var tålmodig och ta hänsyn till tidsskillnader.

Om du har ett problem med Aspose.Slides for Python via Java, följ dessa riktlinjer för att lösa det på det mest effektiva sättet:

- Kontrollera och bekräfta att du använder den senaste versionen av Aspose.Slides for Python via Java innan du rapporterar problemet. Kanske har problemet du upplevde redan lösts i en uppdatering.  

  Se [Aspose.Slides‑nedladdning](https://releases.aspose.com/slides/sv/python-java/) för information om den senaste produktversionen.

- Innan du rapporterar ett problem, kan du gå igenom våra forum, den här dokumentationen och API‑referensen. Kanske har svaret på din fråga redan givits. 

- När du rapporterar ett problem, inkludera originaldokumentet och (om möjligt) ett kodfragment som rör problemet. Om du behöver bifoga flera filer, lägg dem i ett ZIP‑paket. 

  Du kan säkert bifoga dina dokument i Aspose.Forums eftersom endast du och Aspose‑utvecklare har åtkomst till de bifogade filerna.

- Skapa och använd ett tråd för ett specifikt problem. Om du upplever ett annat problem, måste du skapa ett nytt tråd för att rapportera det.
