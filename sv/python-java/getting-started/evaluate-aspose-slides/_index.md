Du kan enkelt ladda ner Aspose.Slides för utvärdering. Utvärderingspaketet är detsamma som det köpta paketet. Utvärderingsversionen blir enkelt licensierad efter att du har lagt till några kodrader för att tillämpa licensen. 

Utvärderingsversionen av Aspose.Slides (utan en specificerad licens) ger full produktfunktionalitet, men den lägger in ett utvärderingsvattenmärke högst upp i dokumentet vid öppning och sparande. Du är också begränsad till en bild när du extraherar text från presentationsbilder.
