## **Översikt**

Denna artikel förklarar hur du får support för Aspose.Slides och hur du rapporterar problem på ett effektivt sätt. Den beskriver huvudstödkanelen, ger rekommendationer för att skicka in ett problem och pekar på användbara tekniska resurser såsom produktsidan, bloggen, nedladdningar, supportforum, dokumentation och API-referens. Artikeln svarar också på vanliga frågor om betald support, ärendeeskalering och support för äldre biblioteksversioner.

## **Teknisk support**

{{% alert color="primary" %}} 

Aspose erbjuder obegränsad gratis teknisk support för alla sina produkter. Supporten är tillgänglig för alla användare (inklusive användare med evalueringspaket).

{{% /alert %}} 

Om du behöver hjälp med Aspose.Slides för Android via Java, överväg följande:

- Huvudkanalen för support är Aspose.Forums. Posta din fråga i Aspose.Slides-forumet - den besvaras inom några timmar. Aspose.Slides supportteam svarar direkt på frågor som publiceras i forumet.
- Observera att Aspose inte erbjuder teknisk support via telefon. Telefonsupport är endast tillgänglig för försäljnings- och köpsfrågor.
- När du väntar på svar i forumet, var vänlig ha tålamod och ta hänsyn till tidszonsskillnader.

När du stöter på ett problem med Aspose.Slides för Android via Java, följ dessa riktlinjer för att lösa det på det mest effektiva sättet:

- Kontrollera och bekräfta att du använder den senaste versionen av Aspose.Slides för Android via Java innan du rapporterar problemet. Kanske var problemet du upplevde redan löst i en uppdatering. 
  Se Aspose.Slides-nedladdning för att få information om den senaste versionen.
- Innan du rapporterar ett problem kan du gå igenom våra forum, den här dokumentationen och API-referensen. Kanske har svaret på din fråga redan givits. 
- När du rapporterar ett problem, inkludera det ursprungliga dokumentet och (om möjligt) ett kodfragment som gäller problemet. Om du behöver bifoga flera filer, lägg dem i ett ZIP-paket. 
  Du kan säkert bifoga dina dokument i Aspose.Forums eftersom bara du och Aspose-utvecklare har åtkomst till de bifogade filerna.
- Skapa och använd en tråd för ett specifikt problem. Om du stöter på ett annat problem måste du skapa en ny tråd för att rapportera det. 

## **FAQ**

**Vilka betalda supportnivåer finns tillgängliga och hur skiljer de sig?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). De skiljer sig åt med avseende på antalet incidenter som ingår, svarsprioritet och tillgängliga kommunikationskanaler.

**Kan jag eskalera ett fel direkt till ingenjörsteamet?**

Ja. [Paid support](https://helpdesk.aspose.com/) möjliggör eskalering till utvecklarna när ett problem kräver djupare undersökning.

**Stöds äldre biblioteksversioner, och släpper ni korrigeringar för dem?**

Du kan använda äldre versioner, men korrigeringar tillhandahålls för den aktuella releasen; patchar för äldre versioner utfärdas inte.
