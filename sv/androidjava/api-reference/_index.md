{{% alert color="primary" %}} 

Aspose.Slides for Android via Java är ett klassbibliotek som gör det möjligt för Java‑applikationer för Android att läsa och skriva presentationsdokument utan att använda Microsoft PowerPoint® eller annan mjukvara. Aspose.Slides for Android via Java låter dig hantera text, former, tabeller och animationer; lägga till ljud och video i bilder; förhandsgranska bilder; och exportera bilder till SVG, PDF‑format och mer.

Den senaste API‑referensen finns på [Aspose.Slides for Android via Java API-referens](https://reference.aspose.com/slides/sv/androidjava/).
