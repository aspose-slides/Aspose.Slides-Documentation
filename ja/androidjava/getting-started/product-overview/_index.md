**Aspose.Slides for Android via Javaへようこそ！**

Aspose.Slides for Android via Java は、Microsoft PowerPoint® の管理 API であり、Microsoft PowerPoint® を使用せずに Java アプリケーションが PowerPoint® ドキュメントを読み書きできるようにします。

Aspose.Slides for Android via Java は、PowerPoint® ドキュメントの管理に必要な機能を提供する唯一のコンポーネントです。

Aspose.Slides for Android via Java は、テキスト、シェイプ、テーブル、アニメーションの管理、スライドへのオーディオ・ビデオの追加、スライドのプレビュー、スライドの SVG、PDF などへのエクスポートといった多数の主要機能を提供します。


## **製品の説明**
Aspose.Slides for Android via Java は、さまざまなソースから PowerPoint ファイルを開くだけでなく、プレゼンテーションをさまざまな方法で保存できます。たとえば、プレゼンテーションを PPT 形式で保存したり、スライドを画像として保存したりできます。

Aspose.Slides for Android via Java は、Microsoft PowerPoint で利用できる機能のほとんどを提供します。Aspose.Slides for Android via Java の汎用性により、テーブルやさまざまなシェイプに加えて、テキスト、オーディオ、ビデオフレームなどのさまざまなフレームをスライドに追加できます。

Aspose.Slides for Android via Java では、開発者が API をよりよく理解できるように、完全なデモと実用的なサンプルも提供しています。
### **プレゼンテーション機能**
- さまざまなソースから Microsoft PowerPoint プレゼンテーションを開く。
- さまざまな Microsoft PowerPoint 形式でプレゼンテーションを保存。
- ゼロからプレゼンテーションを作成。
- 他のコンポーネントを使用せずにプレゼンテーションを PDF 文書に変換。
- Open Document Presentation (ODP) を読み取る機能。
- PPT と PPTX の相互変換機能。
- プレゼンテーションの文書プロパティの読み取りまたは変更。
- プレゼンテーションを PDF にエクスポート。
- プレゼンテーションを TIFF にエクスポート。
- プレゼンテーションを XPS にエクスポート。
- プレゼンテーションを SWF にエクスポート。
- プレゼンテーションを PDF ノート / Tiff ノートにエクスポート。
- プレゼンテーションを HTML にエクスポート。
### **スライド機能**
- ゼロから新しいスライドを作成し、プレゼンテーションに追加。
- プレゼンテーション内の位置で任意のスライドにアクセス。
- 同一または別のプレゼンテーションにスライドをコピーまたはクローン。
- スライドの編集と削除。
- スライドとマスタを自動的にコピー。
- スライドのプレビュー。
- スライド名の取得または設定。
- スライドを画像として保存。
- スライドを SVG 形式にエクスポート。
- プレゼンテーション内の多数のマスタを操作。
- 任意のサポートスタイルでスライドの背景を塗りつぶし。
- 新しいスライドノートの追加または管理。
- 動的に新しいスライドマスタとスタイルを設定。
- スライドショーのトランジションの追加または変更。
- モーフィック トランジションの追加。
### **グラフィックスとマルチメディア機能**

- シェイプにリンクを割り当て。
- スライドやシェイプに新しいアニメーション効果を読み取り、変更、追加。
- プレゼンテーションに新しい画像を追加し、既存の画像を管理。
- プレゼンテーションに新しい画像バレットを追加し、既存のものを管理。
- プレゼンテーションに新しいオーディオフレームを追加し、既存のものを管理。
- プレゼンテーションに新しいビデオフレームを追加し、既存のものを管理。
### **テーブル機能**

- テーブルにプリセットスタイルを適用。
### **テキストと段落機能**
- テキストプレースホルダーとテキストフレーム内のテキストの取得と設定。
- プレースホルダーのタイプ取得。
- テキストフレーム内のテキスト書式設定の変更。
- プレゼンテーションで使用されているフォントの追加または管理。

## **FAQ**

**PDF/A および PDF/UA など、アーカイブやアクセシビリティ用の PDF 準拠レベルに対応していますか？**

はい。PDF エクスポート オプションを設定することで、PDF/A-2a/2b/2u、PDF/A-3a/3b、PDF/UA で PDF を保存できます。[PDF エクスポート オプション](https://reference.aspose.com/slides/androidjava/com.aspose.slides/pdfoptions/) を参照してください。

**フォント置換メカニズムやカスタム フォントのサポートはありますか？正しくレンダリングするための機能ですか？**

はい。ライブラリはカスタム フォントの[読み込み](/slides/ja/androidjava/custom-font/)と[フォールバック ルールの定義](/slides/ja/androidjava/fallback-font/)を提供し、変換やレンダリング時に欠損グリフを確実に置換します。

**ファイルがパスワードで保護されているかどうかを、完全に開かずに検出できますか？**

はい。プレゼンテーションを[検査](/slides/ja/androidjava/examine-presentation/)して、ロード前にパスワードが必要かどうかを判断できます。

**処理や変換に Microsoft PowerPoint は必要ですか？**

いいえ。Aspose.Slides は単体エンジンであり、サーバーやワークステーションに PowerPoint は不要です。

**複数スレッドからプレゼンテーションを安全に処理できますか？**

はい。異なるドキュメントを別スレッドで並行処理できますが、同一のプレゼンテーション インスタンスを同時に使用しないようにしてください。[スレッド間の使用例](/slides/ja/androidjava/multithreading/) を参照してください。

**マクロは保持されますか？PPTM/PPSM ファイルで VBA を管理できますか？**

はい。マクロ付きプレゼンテーションは[サポート](/slides/ja/androidjava/presentation-via-vba/)されており、[VBA プロジェクトの検査と管理]((https://reference.aspose.com/slides/androidjava/com.aspose.slides/presentation/#getVbaProject--)) が可能です。

**PDF や HTML を PowerPoint スライドに変換できますか？**

はい。[PDF ページや HTML コンテンツのインポート](/slides/ja/androidjava/import-presentation/)により、プレゼンテーション内にスライドを作成または追加できます。

**XPS エクスポートはサポートされていますか？また、品質や内容を制御できますか？**

はい。[XPS へのエクスポート](/slides/ja/androidjava/convert-powerpoint-to-xps/)が利用可能で、[保存オプション]((https://reference.aspose.com/slides/androidjava/com.aspose.slides/xpsoptions/))で出力品質や含めるコンテンツを調整できます。

**スライドを画像に変換し、出力品質を制御できますか？**

はい。スライドは PNG、JPEG、GIF、BMP、TIFF などに[レンダリング](/slides/ja/androidjava/convert-powerpoint-to-png/)でき、サイズや品質を細かく設定できます。

**数式 (MathML/MathText) はサポートされていますか？エクスポートも可能ですか？**

はい。数式の[作成と編集](/slides/ja/androidjava/powerpoint-math-equations/)と、[エクスポート](/slides/ja/androidjava/exporting-math-equations/)（MathML への出力を含む）が可能です。

**ライブラリは暗号化やパスワード (開く、設定、削除) をどのように扱いますか？**

はい。[パスワード保護されたプレゼンテーション](/slides/ja/androidjava/password-protected-presentation/)の開封、パスワード設定や削除、保護状態の確認に対応しています。
