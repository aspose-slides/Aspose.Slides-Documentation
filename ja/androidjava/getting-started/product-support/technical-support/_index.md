## **テクニカルサポート**

{{% alert color="primary" %}} 

Aspose はすべての製品に対して無制限の無料テクニカルサポートを提供しています。評価パッケージのユーザーを含むすべてのユーザーがサポートを利用できます。

{{% /alert %}} 

Aspose.Slides for Android via Java のサポートが必要な場合、以下をご検討ください：

- サポートの主な窓口は Aspose.Forums です。Aspose.Slides フォーラムに質問を投稿すれば、数時間以内に回答が得られます。Aspose.Slides サポートチームがフォーラムに投稿された質問に直接回答します。
- Aspose は電話でのテクニカルサポートを提供していないことにご注意ください。電話サポートは販売や購入に関する質問のみ利用可能です。
- フォーラムでの返信を期待する際は、時間帯の違いを考慮し、忍耐強く待ってください。

Aspose.Slides for Android via Java で問題が発生した場合、以下のガイドラインに従って効率的に解決してください：

- 問題を報告する前に、最新の Aspose.Slides for Android via Java バージョンを使用しているか確認してください。おそらく、経験した問題はすでにアップデートで解決されている可能性があります。

  最新バージョンの情報は Aspose.Slides ダウンロードをご覧ください。

- 問題を報告する前に、フォーラム、当ドキュメント、API リファレンスをご確認ください。質問の答えがすでに提供されている可能性があります。

- 問題を報告する際は、元のドキュメントと（可能であれば）問題に関するコードの一部を含めてください。複数のファイルを添付する必要がある場合は、ZIP パッケージにまとめてください。

  Aspose.Forums にドキュメントを安全に添付できます。添付ファイルにアクセスできるのは、あなたと Aspose の開発者だけです。

- 特定の問題ごとに1つのスレッドを作成して使用してください。別の問題が発生した場合は、別のスレッドを作成して報告してください。

## **よくある質問**

**利用可能な有料サポート層は何ですか、またそれらはどのように異なりますか？**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services)。含まれるインシデント数、応答優先度、利用可能な通信チャネルにより異なります。

**バグを直接エンジニアリングチームにエスカレーションできますか？**

はい。[Paid support](https://helpdesk.aspose.com/) は、問題が詳細な調査を必要とする場合に開発者へのエスカレーションを可能にします。

**古いライブラリバージョンはサポートされていますか、またそれらに対して修正はリリースされますか？**

古いバージョンも使用できますが、修正は現在のリリースに対して提供されます。レガシーバージョン用のパッチは提供されません。
