PowerPoint 2019では、Microsoftはユーザーがプレゼンテーションを保護するために使用できるオプションの一つとして、**常に読み取り専用で開く**設定を導入しました。この読み取り専用設定を使用してプレゼンテーションを保護したい場合があります。

-  accidentalな編集を防ぎ、プレゼンテーションの内容を安全に保ちたいとき。  
-  提供したプレゼンテーションが最終版であることをアラートしたいとき。  

プレゼンテーションに対して**常に読み取り専用で開く**オプションを選択すると、ユーザーがプレゼンテーションを開いたときに、**読み取り専用**の推奨が表示され、次のようなメッセージが表示されるかもしれません：*偶発的な変更を防ぐために、作成者はこのファイルを読み取り専用で開くように設定しました。*

読み取り専用の推奨は、編集を思いとどまらせるシンプルで効果的な抑止力であり、ユーザーはプレゼンテーションを編集する前にこれを解除するための作業を行わなければなりません。ユーザーにプレゼンテーションへの変更をさせたくなく、それについて丁寧に知らせたいのであれば、読み取り専用の推奨は良い選択肢となるかもしれません。

> **読み取り専用**保護がかけられたプレゼンテーションが、最近導入された機能をサポートしていない古いMicrosoft PowerPointアプリケーションで開かれると、**読み取り専用**の推奨は無視され（プレゼンテーションは通常通りに開かれます）。

Aspose.Slides for Android via Javaは、プレゼンテーションを**読み取り専用**に設定することを可能にします。これは、ユーザーがプレゼンテーションを開いた後に**読み取り専用**の推奨を確認することを意味します。以下のサンプルコードは、Aspose.Slidesを使用してJavaでプレゼンテーションを**読み取り専用**に設定する方法を示しています：

```java
Presentation pres = new Presentation();
try {
    pres.getProtectionManager().setReadOnlyRecommended(true);
    pres.save("ReadOnlyPresentation.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```

{{% alert color="primary" %}} 

**注意**：**読み取り専用**の推奨は、単に編集を思いとどまらせたり、ユーザーがPowerPointプレゼンテーションに偶発的な変更を加えるのを防ぐことを目的としています。もし、何をしているかが分かっている意欲的な人があなたのプレゼンテーションを編集することを決めた場合、彼らは容易に読み取り専用設定を解除することができます。無断での編集を真剣に防ぐ必要がある場合は、[暗号化やパスワードを伴うより厳格な保護を使用する方が良い](https://docs.aspose.com/slides/androidjava/password-protected-presentation/)です。
