## **動的にテキストを追加する**
両方の方法は次の手順に従います：

1. プレゼンテーションを作成します。
1. 空白のスライドを追加します。
1. テキストボックスを追加します。
1. テキストを設定します。
1. プレゼンテーションを書き込みます。
