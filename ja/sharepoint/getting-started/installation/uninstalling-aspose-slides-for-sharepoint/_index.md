Aspose.Slides for SharePointをアンインストールするには: 

1. セットアッププログラムを実行します。
   Aspose.Slides for SharePointがすでにインストールされている場合、セットアッププログラムはそれを削除または修復することを提案します。 
2. **削除**を選択してAspose.Slides for SharePointをアンインストールします。
