## **テクニカルサポート**
{{% alert color="primary" %}}
Aspose はすべての製品に対して無制限の無料テクニカルサポートを提供しています。サポートはすべてのユーザー（評価版パッケージのユーザーを含む）が利用可能です。
{{% /alert %}} 

Aspose.Slides for Node.js via Java のサポートが必要な場合は、次の点を検討してください。

- サポートの主要な窓口は [Aspose.Forums](https://forum.aspose.com/c/slides/11) です。Aspose.Slides フォーラムに質問を投稿すると、数時間以内に回答が得られます。Aspose.Slides サポートチームが直接フォーラム上の質問に回答します。
- Aspose は電話でのテクニカルサポートを提供していないことにご注意ください。電話サポートは販売および購入に関する質問にのみ利用可能です。
- フォーラムでの返信を期待する際は、時差を考慮し、忍耐強くお待ちください。

Aspose.Slides for Node.js via Java で問題が発生した場合は、最も効率的に解決するために以下のガイドラインに従ってください。

- 問題を報告する前に、使用している Aspose.Slides for Node.js via Java のバージョンが最新であることを確認してください。もしかすると、経験した問題はすでにアップデートで解決されているかもしれません。  
  詳細は [Aspose.Slides download](https://releases.aspose.com/slides/nodejs-java/) で最新バージョン情報をご確認ください。

- 問題を報告する前に、フォーラム、本文書、API リファレンスを確認してください。質問への回答がすでに提供されている可能性があります。

- 問題を報告する際は、元のドキュメントと（可能であれば）問題に関係するコードの一部を添付してください。複数のファイルを添付する必要がある場合は、ZIP パッケージにまとめます。Aspose.Forums にドキュメントを安全に添付できるのは、**only** あなたと Aspose の開発者だけが添付ファイルにアクセスできるためです。

- 特定の問題につき、1 つのスレッドを作成して使用してください。別の問題が発生した場合は、別のスレッドを作成して報告してください。

## **FAQ**

**利用可能な有料サポートの階層は何ですか、またそれらはどのように異なりますか？**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services)。インシデント数、応答優先度、利用可能なコミュニケーションチャネルに違いがあります。

**バグを直接エンジニアリングチームにエスカレーションできますか？**

はい。[有料サポート](https://helpdesk.aspose.com/) により、問題の深刻な調査が必要な場合に開発者へエスカレーションできます。

**古いライブラリバージョンはサポートされますか、またそれら向けの修正はリリースされますか？**

古いバージョンも使用可能ですが、修正は現在のリリースに対して提供されます。レガシーバージョン向けのパッチは提供されません。
