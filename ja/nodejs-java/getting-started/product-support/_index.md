## **テクニカルサポート**
{{% alert color="primary" %}}

Aspose はすべての製品に対して無制限の無料テクニカルサポートを提供しています。評価パッケージを含むすべてのユーザーがサポートを利用できます。

{{% /alert %}} 

Aspose.Slides for Node.js via Java のサポートが必要な場合は、以下をご確認ください。

- サポートの主要な手段は [Aspose.Forums](https://forum.aspose.com/c/slides/11) です。Aspose.Slides フォーラムに質問を投稿すると、数時間以内に回答が得られます。Aspose.Slides サポートチームがフォーラムに投稿された質問に直接回答します。
- Aspose は電話でのテクニカルサポートを提供していないことにご注意ください。電話サポートは販売や購入に関する質問にのみ利用可能です。
- フォーラムでの返信を期待する際は、時差を考慮してご辛抱ください。

Aspose.Slides for Node.js via Java で問題が発生した場合は、以下のガイドラインに従って効率的に解決してください。

- 問題を報告する前に、最新の Aspose.Slides for Node.js via Java バージョンを使用しているか確認してください。問題はすでにアップデートで解決されている可能性があります。  
  最新バージョンの情報は [Aspose.Slides download](https://releases.aspose.com/slides/nodejs-java/) で確認できます。
- 問題を報告する前に、フォーラム、本文書、API リファレンスを確認してください。質問への回答が既に提供されている可能性があります。
- 問題を報告する際は、元のドキュメントと（可能であれば）問題に関連するコードの断片を添付してください。複数のファイルを添付する必要がある場合は、ZIP にまとめてください。Aspose.Forums にドキュメントを安全に添付できます。**あなたと Aspose の開発者のみが添付ファイルにアクセスできる**ためです。
- 特定の問題ごとに 1 つのスレッドを作成して使用してください。別の問題が発生した場合は、別のスレッドを作成して報告してください。

## **FAQ**

**利用可能な有料サポート層は何ですか、そしてそれらはどのように異なりますか？**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services)。インシデント数、対応の優先度、利用可能なコミュニケーションチャネルにより違いがあります。

**バグをエンジニアリングチームに直接エスカレーションできますか？**

はい。[Paid support](https://helpdesk.aspose.com/) により、問題が深い調査を要する場合は開発者へのエスカレーションが可能です。

**古いライブラリバージョンはサポートされますか、またそれらに対して修正がリリースされますか？**

古いバージョンを使用することは可能ですが、修正は現在のリリースに対して提供されます。レガシーバージョン向けのパッチは提供されません。
