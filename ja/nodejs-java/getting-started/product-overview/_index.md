## **製品説明**
Aspose.Slides for Node.js via Java は、Microsoft PowerPoint® を使用せずに、さまざまな文書処理タスクを実行できるクロスプラットフォームのクラスライブラリです。

### **Aspose.Slides for Node.js via Java を使用すべき理由**

- Aspose.Slides for Node.js via Java は、PowerPoint® ドキュメントの管理に必要な機能を提供する唯一のコンポーネントです。  
- Aspose.Slides for Node.js via Java は、テキスト、シェイプ、テーブル＆アニメーションの管理、スライドへの音声・ビデオの追加、スライドのプレビュー、SVG、PDF などへのエクスポートなど、多数の主要機能を提供します。  
- Aspose.Slides for Node.js via Java は、さまざまなソースから PowerPoint ファイルを開くだけでなく、プレゼンテーションを多様な方法で保存できます。たとえば、PPT 形式で保存したり、スライドを画像として保存したりできます。  
- Aspose.Slides for Node.js via Java は、Microsoft PowerPoint にあるほぼすべての機能を提供します。テーブルやさまざまなシェイプに加えて、テキスト、音声、ビデオ フレームといったさまざまなフレームをスライドに追加して使用できます。  

## **Aspose.Slides の機能**

Aspose.Slides for Node.js via Java の機能は、次のグループに分けられます。

- プレゼンテーション機能
- スライド機能
- グラフィックスおよびマルチメディア機能
- テーブル機能
- テキスト＆段落機能
- 高度な機能

### **プレゼンテーション機能**

- 異なるソースから Microsoft PowerPoint プレゼンテーションを開く  
- 異なる Microsoft PowerPoint 形式でプレゼンテーションを保存  
- ゼロからプレゼンテーションを作成  
- 他のコンポーネントを使用せずにプレゼンテーションを PDF に変換  
- プレゼンテーションの文書プロパティを読み取り・変更  
- プレゼンテーションに保護を適用  
- プレゼンテーションを PDF にエクスポート  
- プレゼンテーションを TIFF にエクスポート  
- プレゼンテーションを XPS にエクスポート  
- プレゼンテーションを SWF にエクスポート  
- プレゼンテーションを PDF ノート / TIFF ノートにエクスポート  
- プレゼンテーションを HTML にエクスポート  

### **スライド機能**

- ゼロから新しいスライドを作成し、プレゼンテーションに追加  
- プレゼンテーション内のスライド位置で任意のスライドにアクセス  
- 同じプレゼンテーションまたは別のプレゼンテーションへスライドをコピーまたはクローン  
- スライドの編集と削除  
- スライドと同時にマスタをコピー  
- スライドのプレビュー  
- スライド名の取得・設定  
- スライドを画像として保存  
- スライドを SVG 形式にエクスポート  
- プレゼンテーション内で多数のマスタを操作  
- 任意のサポートスタイルでスライド背景を塗りつぶし  
- 新しいスライドノートの追加・管理  
- 動的に新しいスライドマスタとスタイルを設定  
- スライドショーのトランジションの追加または変更  
- モーフ トランジションの追加  

### **グラフィックス＆マルチメディア機能**

- 線、長方形、円、ポリラインなどのシェイプの作成・管理  
- シェイプの線スタイルの管理  
- シェイプの代替テキストの取得  
- シェイプの回転・反転  
- ソリッド、グラデーション、パターン、テクスチャ、画像など、さまざまなスタイルでシェイプを塗りつぶし  
- 48 種類の事前定義パターンと 24 種類のテクスチャを使用可能  
- シェイプにリンクを割り当て  
- スライドやシェイプに対して新しいアニメーション効果を読み取り、変更、追加  
- プレゼンテーションに新しい画像を追加し、既存画像を管理  
- プレゼンテーションに新しい画像バレットを追加し、既存バレットを管理  
- プレゼンテーションに新しい音声フレームを追加し、既存フレームを管理  
- プレゼンテーションに新しいビデオフレームを追加し、既存フレームを管理  

### **テーブル機能**

- ゼロからテーブルを作成し、スライドに追加  
- テーブルのサイズ変更と移動  
- テーブル内の任意のセルにアクセス  
- 任意のセルの書式と罫線を変更  
- 列と行のサイズ変更  
- 列と行の追加・削除  
- テーブル全体の罫線幅と色を設定  

### **テキスト＆段落機能**

- テキストプレースホルダーやテキストフレーム内のテキストの取得・設定  
- プレースホルダーのタイプ取得  
- テキストフレーム内のテキスト書式の変更  
- 新しいフォントの追加や、プレゼンテーションで使用中のフォントの管理  
- テキストとテキスト書式の変更  
- テキストへのリンク作成・管理  
- タブと有効タブのサポート  
- ほぼすべてのシェイプタイプに新しいテキストフレームを追加  
- テキストフレームの余白とテキスト折り返し設定の取得・変更  
- 段落インデントの取得・変更  

### **高度な機能**

- プレゼンテーションに埋め込まれた OLE オブジェクトへのアクセス  
- プレゼンテーションのロック  

## **テクニカルサポート**

Aspose はすべての製品に対して無制限の無料テクニカルサポートを提供しています。評価版を含むすべてのユーザーが利用可能です。Aspose.Slides for Node.js via Java に関してサポートが必要な場合は、次の方法をご検討ください。

- 主なサポート窓口は [Aspose フォーラム](https://forum.aspose.com/) です。質問は [Aspose.Slides フォーラム](https://forum.aspose.com/c/slides/11) に投稿してください — 数時間以内に回答が得られます。Aspose.Slides サポートチームが直接フォーラムの質問に回答します。  
- Aspose は電話によるテクニカルサポートは提供していません。電話サポートは販売・購入に関する質問のみ利用可能です。  
- フォーラムでの返信を待つ際は、タイムゾーンの違いを考慮して忍耐強くお待ちください。  

Aspose.Slides for Node.js via Java に問題がある場合は、以下の手順で効率的に解決してください。

- 報告前に、まず最新バージョンを使用しているか確認してください。問題は既にアップデートで解決されている可能性があります。  

  最新バージョン情報は [Aspose.Slides ダウンロード](https://releases.aspose.com/slides/nodejs-java/) をご覧ください。  

- 問題を報告する前に、フォーラム、ドキュメント、API リファレンスを確認してください。質問への回答がすでに掲載されていることがあります。  

- 問題を報告する際は、元のドキュメントと（可能であれば）問題に関するコードの断片を添付してください。複数ファイルを添付する場合は ZIP パッケージにまとめてください。  

  Aspose.Forums では、添付ファイルは投稿者と Aspose 開発者のみがアクセスできるため安全です。  

- 特定の問題ごとにスレッドを作成し、別の問題が発生した場合は新しいスレッドで報告してください。

## **FAQ**

**PDF/A および PDF/UA といったアーカイブ・アクセシビリティ用の PDF 準拠レベルはサポートされていますか？**

はい。PDF/A-2a/2b/2u、PDF/A-3a/3b、PDF/UA を、[PDF エクスポート オプション](https://reference.aspose.com/slides/nodejs-java/aspose.slides/pdfoptions/) の設定で保存できます。

**フォント置換メカニズムやカスタムフォントのサポートはありますか？正しくレンダリングするために必要です。**

はい。ライブラリでは [カスタムフォントのロード](/slides/ja/nodejs-java/custom-font/) と [フォールバック ルールの定義](/slides/ja/nodejs-java/fallback-font/) が可能で、欠落したグリフは変換・レンダリング時に確実に置換されます。

**ファイルがパスワード保護されているかどうかを、全文を開かずに検出できますか？**

はい。プレゼンテーションを [検査](/slides/ja/nodejs-java/examine-presentation/) して、ロード前にパスワードが必要かどうかを判断できます。

**Microsoft PowerPoint がインストールされていなくても処理・変換は可能ですか？**

いいえ。Aspose.Slides は単体エンジンであり、サーバーやワークステーションに PowerPoint は不要です。

**複数スレッドから同時にプレゼンテーションを処理しても安全ですか？**

はい。別々のスレッドで異なるドキュメントを並行処理できます。同一プレゼンテーション インスタンスをスレッド間で同時に使用しないようにしてください。詳細は [マルチスレッド対応](/slides/ja/nodejs-java/multithreading/) を参照。

**マクロは保持されますか？PPTM/PPSM ファイルの VBA を管理できますか？**

はい。マクロ付きプレゼンテーションは [サポート](/slides/ja/nodejs-java/presentation-via-vba/) されており、VBA プロジェクトの検査・管理が可能です。

**PDF や HTML を PowerPoint スライドに変換できますか？**

はい。PDF ページや HTML コンテンツを [インポート](/slides/ja/nodejs-java/import-presentation/) して、プレゼンテーション内にスライドを作成・追加できます。

**XPS エクスポートはサポートされていますか？出力品質や内容を制御できますか？**

はい。[XPS へのエクスポート](/slides/ja/nodejs-java/convert-powerpoint-to-xps/) が可能で、[保存オプション](https://reference.aspose.com/slides/nodejs-java/aspose.slides/xpsoptions/) により品質や含めるコンテンツを調整できます。

**スライドを画像に変換し、出力品質を制御できますか？**

はい。スライドは [PNG、JPEG、GIF、BMP、TIFF](/slides/ja/nodejs-java/convert-powerpoint-to-png/) などの形式でレンダリングでき、サイズや品質を細かく制御できます。

**数式（MathML／MathText）はサポートされ、エクスポートできますか？**

はい。数式の作成・編集は [こちら](/slides/ja/nodejs-java/powerpoint-math-equations/) で、[エクスポート](/slides/ja/nodejs-java/exporting-math-equations/) も可能で、MathML 形式への出力もサポートしています。

**暗号化やパスワード操作（開く、設定、削除）はどのように行いますか？**

[こちら](/slides/ja/nodejs-java/password-protected-presentation/) で、暗号化プレゼンテーションの開封、開閉・編集用のパスワード設定・削除、ファイルの保護状態確認がサポートされています。
