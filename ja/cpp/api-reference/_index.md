{{% alert color="primary" %}} 

Aspose.Slides for C++ は、Microsoft PowerPoint® などを使用せずに、C++ アプリケーションがプレゼンテーション ドキュメントを読み書きできるクラス ライブラリです。Aspose.Slides for C++ を使用すると、テキスト、図形、テーブルおよびアニメーションを管理し、スライドに音声や動画を追加し、スライドをプレビューし、スライドを SVG、PDF 形式などにエクスポートできます。

最新の API リファレンスは [Aspose.Slides for C++ API Reference](https://reference.aspose.com/slides/ja/cpp/) にあります。
