## **製品概要**

{{% alert color="primary" %}} 

![todo:image_alt_text](product-overview_1)

**Aspose.Slides for C++へようこそ**

Aspose.Slides for C++ は Microsoft PowerPoint® を使用せずに PowerPoint® ドキュメントの読み書きを可能にする Microsoft PowerPoint® 管理 API です。Aspose.Slides for C++ は PowerPoint® ドキュメントの管理機能を提供する唯一のコンポーネントです。テキスト、図形、テーブル・アニメーションの管理、スライドへの音声・動画の追加、スライドのプレビュー、SVG や PDF 形式へのエクスポートなど、多くの主要機能を備えています。

{{% /alert %}} 
## **製品説明**
Aspose.Slides for C++ はネイティブ C++ で開発されています。現在は Windows x64 環境向けにコンパイルされていますが、Linux 環境向けもコンパイル予定です。

Aspose.Slides for C++ はさまざまなソースから PowerPoint ファイルを開くだけでなく、プレゼンテーションを多彩な形式で保存できます。たとえば、PPT だけでなく PDF 形式でも保存できます。さらに、スライドを画像として保存することも可能です。Aspose.Slides for C++ は Microsoft PowerPoint が提供するほぼすべての機能を提供します。テーブルやさまざまな図形の追加だけでなく、テキスト、音声、動画フレームといったさまざまなフレームも追加できます。

## **豊富な機能セット**
Aspose.Slides for C++ の機能は次の領域に分類されます。

1. プレゼンテーション機能
1. スライド機能
1. グラフィックス・マルチメディア機能
1. テーブル機能
1. テキスト・段落機能
1. 高度機能

### **プレゼンテーション機能**
- 様々なソースから Microsoft PowerPoint プレゼンテーションを開く
- 異なる Microsoft PowerPoint 形式でプレゼンテーションを保存
- ゼロからプレゼンテーションを作成
- 追加コンポーネントを使用せずに PDF に変換
- プレゼンテーションの文書プロパティを読み書き
- プレゼンテーションに保護を適用
- プレゼンテーションを PDF にエクスポート
- プレゼンテーションを XPS にエクスポート
- プレゼンテーションを SWF にエクスポート
- プレゼンテーションのノートを PDF にエクスポート
- プレゼンテーションを HTML にエクスポート

### **スライド機能**
- ゼロから新しいスライドを作成し、プレゼンテーションに追加
- プレゼンテーション内の位置で任意のスライドにアクセス
- 同一または別のプレゼンテーションへスライドをコピーまたはクローン
- スライドの編集と削除
- マスタをスライドと共に自動コピー
- スライドをプレビュー
- スライド名の取得または設定
- スライドを画像として保存
- スライドを SVG 形式にエクスポート
- 複数のマスタと連携
- 任意のサポートスタイルでスライド背景を塗りつぶし
- スライドノートの追加または管理
- 新しいスライドマスタとスタイルを動的に設定
- スライドショーの遷移を追加または変更
- モーフ遷移を追加

### **グラフィックス・マルチメディア機能**
- 線、矩形、楕円、ポリラインなどの図形を作成または管理
- 図形の線スタイルを管理
- 図形の代替テキストを取得
- 図形を回転または反転
- 塗りつぶしスタイル（単色、グラデーション、パターン、テクスチャ、画像）を設定
- 48 種類の事前定義パターンと 24 種類のテクスチャを使用
- 図形にハイパーリンクを割り当て
- スライドと図形にアニメーション効果を読み取り、変更、または新規追加
- プレゼンテーションに画像を新規追加または既存画像を管理
- 画像バレットを新規追加または既存バレットを管理
- オーディオフレームを新規追加または既存フレームを管理
- ビデオフレームを新規追加または既存フレームを管理

### **テーブル機能**
- テーブルをゼロから作成し、スライドに追加
- テーブルのサイズ変更と移動
- 任意のセルにアクセス
- 任意のセルの書式と罫線を変更
- 列と行のサイズ変更
- 列と行の追加・削除
- テーブル全体の罫線幅と色を設定

### **テキスト・段落機能**
- テキストプレースホルダーとテキストフレームのテキスト取得・設定
- プレースホルダーのタイプ取得
- テキストフレーム内のテキスト書式を変更
- プレゼンテーションで使用されるフォントの新規追加または既存管理
- テキストとテキスト書式を変更
- テキストへのリンクを作成または管理
- タブと有効タブをサポート
- ほぼすべての図形タイプに新規テキストフレームを追加
- テキストフレームの余白とテキスト折り返し設定を取得・変更
- 段落インデントを取得・変更

### **高度機能**
- プレゼンテーションに埋め込まれた OLE オブジェクトにアクセス
- プレゼンテーションをロック

## **サポートプラットフォーム**

## **FAQ**

**PDF/A および PDF/UA など、アーカイブやアクセシビリティ向けの PDF 準拠レベルをサポートしていますか？**

はい。PDF/A-2a/2b/2u、PDF/A-3a/3b、PDF/UA を [PDF エクスポート オプション](https://reference.aspose.com/slides/cpp/aspose.slides.export/pdfoptions/)で設定して保存できます。

**フォント置換機構やカスタムフォントのサポートはありますか？**

はい。ライブラリは [/slides/cpp/custom-font/](/slides/ja/cpp/custom-font/) でカスタムフォントを読み込み、[/slides/cpp/fallback-font/](/slides/ja/cpp/fallback-font/) でフォールバック規則を定義でき、欠損グリフを確実に置換します。

**ファイルがパスワード保護されているか、完全に開かずに検出できますか？**

はい。[/slides/cpp/examine-presentation/](/slides/ja/cpp/examine-presentation/) でプレゼンテーションを検査し、パスワードが必要かどうかを判断できます。

**Microsoft PowerPoint は必要ですか？**

いいえ。Aspose.Slides は単体エンジンであり、サーバやワークステーションに PowerPoint は不要です。

**マルチスレッドで安全に処理できますか？**

はい。別スレッドで異なるドキュメントを並行処理可能ですが、同一プレゼンテーションインスタンスを跨いで同時に使用しないでください。[/slides/cpp/multithreading/](/slides/ja/cpp/multithreading/) を参照。

**マクロは保持され、PPTM/PPSM ファイルの VBA を管理できますか？**

はい。マクロ付きプレゼンテーションは [/slides/cpp/presentation-via-vba/](/slides/ja/cpp/presentation-via-vba/) でサポートされ、[/reference.aspose.com/slides/cpp/aspose.slides.vba/](/reference.aspose.com/slides/cpp/aspose.slides.vba/) で VBA プロジェクトを検査・管理できます。

**PDF や HTML から PowerPoint スライドへ変換できますか？**

はい。[/slides/cpp/import-presentation/](/slides/ja/cpp/import-presentation/) で PDF ページや HTML コンテンツをインポートしてスライドを作成または補充できます。

**XPS エクスポートはサポートされ、品質や内容を制御できますか？**

はい。[/slides/cpp/convert-powerpoint-to-xps/](/slides/ja/cpp/convert-powerpoint-to-xps/) でエクスポートでき、[/reference.aspose.com/slides/cpp/aspose.slides.export/xpsoptions/](/reference.aspose.com/slides/cpp/aspose.slides.export/xpsoptions/) で出力品質や含める内容を調整できます。

**スライドを画像に変換し、出力品質を制御できますか？**

はい。スライドは [/slides/cpp/convert-powerpoint-to-png/](/slides/ja/cpp/convert-powerpoint-to-png/) で PNG、JPEG、GIF、BMP、TIFF などにレンダリングでき、サイズと品質を細かく制御できます。

**数式（MathML/MathText）はサポートされ、エクスポートできますか？**

はい。[/slides/cpp/powerpoint-math-equations/](/slides/ja/cpp/powerpoint-math-equations/) で数式テキストの作成・編集ができ、[/slides/cpp/exporting-math-equations/](/slides/ja/cpp/exporting-math-equations/) でエクスポートできます（MathML へのエクスポート含む）。

**暗号化やパスワード（開く、設定、削除）はどのように扱われますか？**

[/slides/cpp/password-protected-presentation/](/slides/ja/cpp/password-protected-presentation/) で暗号化されたプレゼンテーションの開封、パスワード設定・削除、保護状態の確認がサポートされています。
