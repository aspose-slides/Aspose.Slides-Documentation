## **ファイル形式と変換**
Aspose.Slides for C++はほとんどのPowerPoint文書形式をサポートし、また、組織間で広く使用されている人気のある形式にエクスポートすることもできます。以下の表にこれらの形式を列挙します：

|**機能**|**説明**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/ja/cpp/microsoft-powerpoint-ppt/)|Aspose.Slides for C++はこのプレゼンテーション文書形式に対して最も高速な処理を提供します。|
|[Portable Document Format (PDF)](/slides/ja/cpp/portable-document-format-pdf/)|サポートされているファイル形式は、単一のメソッドでAdobe Portable Document Format (PDF) ドキュメントにエクスポートできます。|
|[XML Parser Specification (XPS)](/slides/ja/cpp/xml-parser-specification-xps/)|サポートされているすべてのファイル形式は、単一のメソッドでXML Parser Specification (XPS) ドキュメントにエクスポートできます。|
|[PresentationML (PPTX, XML)](/slides/ja/cpp/presentationml-pptx-xml/)|Aspose.Slides for C++は、PresentationMLまたはPPTXとして知られるOOXMLプレゼンテーション形式の処理をサポートしています。|
|[PPTX To HTML Conversion](https://docs.aspose.com/slides/cpp/convert-powerpoint-to-html/)|Aspose.Slides for C++は、プレゼンテーションをHTML形式に変換することをサポートしています。|
## **レンダリング**
Aspose.Slides for C++は、プレゼンテーション文書内のスライドをさまざまなグラフィックス形式に高忠実度でレンダリングすることをサポートしています。以下にリストを示します：

|**機能**|**説明**|
| :- | :- |
|C++サポートの画像形式|Aspose.Slides for C++を使用すると、プレゼンテーションスライドだけでなく、スライド上の画像をPNG、BMP、JPEGなどのC++サポートのグラフィックス形式にレンダリングすることができます。|
|SVG形式|Aspose.Slides for C++は、プレゼンテーションスライドをスケーラブルベクターグラフィックス（SVG）形式にエクスポートするための組み込みメソッドも提供します。|
## **コンテンツ機能**
Aspose.Slides for C++は、プレゼンテーション文書のほぼすべての可能なコンテンツにアクセス、変更、または作成することを可能にします。

|**機能**|**説明**|
| :- | :- |
|マスタースライド|マスタースライドは通常のスライドのレイアウトを定義します。Aspose.Slides for C++は、プレゼンテーション文書のマスタースライドにアクセスして変更することを可能にします。|
|通常のスライド|Aspose.Slides for C++を使用すると、さまざまなタイプの新しいスライドを作成するだけでなく、既存のスライドにアクセスして変更することもできます。|
|スライドのクローン/コピー|Aspose.Slides for C++は、プレゼンテーション内だけでなく、別のプレゼンテーションから既存のスライドをクローンまたはコピーするための組み込みメソッドを提供します。スライドはマスタースライドからレイアウトを継承するため、組み込みのクローンメソッドは自動的にマスターもコピーします。|
|プレースホルダーとテキストホルダー|スライド内のプレースホルダーとテキストホルダーにもアクセスできます。さらに、適切なメソッドを使用してテキストホルダーを持つスライドをゼロから作成することもできます。|
|ヘッダーとフッター|Aspose.Slides for C++はスライド内のヘッダー/フッターの処理もサポートします。|
|スライド内のノート|Aspose.Slides for C++を使用すると、スライドに関連付けられたノートにアクセスして変更するだけでなく、ノートを追加することもできます。|
|形状の検索|形状に関連付けられた代替テキストを使用して、スライドから特定の形状を見つけることもできます。|
|背景|Aspose.Slides for C++は、プレゼンテーションのマスターまたは通常のスライドに関連付けられた背景との作業を可能にします。|
|テキストボックス|テキストボックスはゼロから作成できます。既存のテキストボックスにもアクセスでき、そのテキストを元のテキストフォーマットを失うことなく変更できます。|
|長方形の形状|Aspose.Slides for C++を使用して長方形の形状を作成または変更できます。|
|ポリラインの形状|Aspose.Slides for C++を使用してポリラインの形状を作成または変更できます。|
|楕円形の形状|楕円形の形状もAspose.Slides for C++を使用して作成または変更できます。|
|グループ形状|Aspose.Slides for C++はグループ形状もサポートしています。|
|オートシェイプ|オートシェイプもAspose.Slides for C++でサポートされています。|
|形状のシリアル化|Aspose.Slides for C++は多数の形状をサポートしています。ただし、Aspose.Slides for C++でサポートされていない形状がある場合は、その形状を既存のスライドからシリアル化してAspose.Slides for C++でのさらなる使用のために提供されるシリアル化メソッドがあります。|
|画像フレーム|画像はAspose.Slides for C++を使用して画像フレーム内で管理されます。|
|オーディオフレーム|オーディオファイルは、Aspose.Slides for C++を使用してスライド内のオーディオフレームにリンクまたは埋め込むことができます。|
|ビデオフレーム|ビデオファイルはAspose.Slides for C++を使用してビデオフレーム内で処理されます。|
|OLEフレーム|OLEオブジェクトはAspose.Slides for C++を使用してOLEフレーム内で管理されます。|
|テーブル|スライド上のテーブルもAspose.Slides for C++でサポートされています。|
|テキストフレーム|任意の形状に関連したテキストは、その形状に関連したテキストフレームを通じてアクセスできます。|
|テキストスキャン|プレゼンテーション内のテキストは、組み込みのスキャンメソッドを使用してプレゼンテーションレベルまたはスライドレベルでスキャンできます。|
|アニメーション|形状にアニメーションを適用できます。|
|スライドショー|スライドショー、スライド切り替え、およびモーフィングトランジションもサポートされています。|
## **フォーマット機能**
Aspose.Slides for C++を使用して、プレゼンテーション文書内のスライド上のテキストと形状をフォーマットすることができます。

|**機能**|**説明**|
| :- | :- |
|テキストフォーマット|<p>Aspose.Slides for C++では、テキストは形状に関連付けられたテキストフレームを通じて管理されます。したがって、テキストはテキストフレームに関連付けられた段落や部分を使用してフォーマットできます。以下のテキスト要素がAspose.Slides for C++を通じてフォーマットできます。</p><p>- フォントタイプ</p><p>- フォントサイズ</p><p>- フォントカラー</p><p>- フォントの影</p><p>- 段落の整列</p><p>- 段落の箇条書き</p><p>- 段落の向き</p>|
|形状フォーマット|<p>Aspose.Slides for C++では、スライドの基本要素は形状です。以下の形状要素はAspose.Slides for C++を使用してフォーマットできます：</p><p>- 位置</p><p>- サイズ</p><p>- 線</p><p>- 塗りつぶし（パターン、グラデーション、ソリッドを含む）</p><p>- テキスト</p><p>- 画像</p>|
