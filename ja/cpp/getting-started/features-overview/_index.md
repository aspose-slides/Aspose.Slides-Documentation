## **ファイル形式と変換**
Aspose.Slides for C++ は、ほとんどの PowerPoint ドキュメント形式をサポートし、組織間で広く利用されている一般的な形式へエクスポートすることもできます。以下の表にこれらの形式を示します。

|**機能**|**説明**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/ja/cpp/microsoft-powerpoint-ppt/)|Aspose.Slides for C++ はこのプレゼンテーションドキュメント形式の最速処理を提供します。|
|[ポータブルドキュメント形式 (PDF)](/slides/ja/cpp/portable-document-format-pdf/)|サポートされているファイル形式は、単一のメソッドで Adobe Portable Document Format (PDF) ドキュメントへエクスポートできます。|
|[XML パーサー仕様 (XPS)](/slides/ja/cpp/xml-parser-specification-xps/)|サポートされているすべてのファイル形式は、単一のメソッドで XML Parser Specification (XPS) ドキュメントへエクスポートできます。|
|[PresentationML (PPTX, XML)](/slides/ja/cpp/presentationml-pptx-xml/)|Aspose.Slides for C++ は、PresentationML または PPTX とも呼ばれる OOXML プレゼンテーション形式の処理をサポートします。|
|[PPTX から HTML への変換](https://docs.aspose.com/slides/cpp/convert-powerpoint-to-html/)|Aspose.Slides for C++ はプレゼンテーションを HTML 形式に変換する機能を提供します。|

## **レンダリング**
Aspose.Slides for C++ は、プレゼンテーションドキュメント内のスライドを以下に示すさまざまなグラフィック形式に高忠実度でレンダリングできます。

|**機能**|**説明**|
| :- | :- |
|C++ がサポートする画像形式|Aspose.Slides for C++ を使用すると、プレゼンテーションスライドだけでなく、スライド上の画像も PNG、BMP、JPEG などの C++ がサポートするすべてのグラフィック形式にレンダリングできます。|
|SVG 形式|Aspose.Slides for C++ は、スケーラブルベクターグラフィックス (SVG) 形式へプレゼンテーションスライドをエクスポートする組み込みメソッドも提供します。|

## **コンテンツ機能**
Aspose.Slides for C++ は、プレゼンテーションドキュメントのほぼすべてのコンテンツにアクセス、変更、作成が可能です。

|**機能**|**説明**|
| :- | :- |
|マスタースライド|マスタースライドは通常スライドのレイアウトを定義します。Aspose.Slides for C++ はプレゼンテーションドキュメントのマスタースライドにアクセスし、変更できるようにします。|
|通常スライド|Aspose.Slides for C++ を使用すると、さまざまなタイプの新規スライドを作成できるだけでなく、既存のスライドにアクセスして変更できます。|
|スライドのクローン／コピー|Aspose.Slides for C++ には、スライドを同一プレゼンテーション内だけでなく別のプレゼンテーション間でもクローンまたはコピーする組み込みメソッドがあります。スライドはマスタースライドからレイアウトを継承するため、クローン作成時にマスターも自動的にコピーされます。|
|プレースホルダーとテキストホルダー|スライド内のプレースホルダーとテキストホルダーにもアクセスできます。また、適切なメソッドを使用してテキストホルダーだけのスライドをゼロから作成できます。|
|ヘッダーとフッター|Aspose.Slides for C++ はスライドのヘッダー／フッターの操作もサポートします。|
|スライドノート|Aspose.Slides for C++ を使用すると、スライドに関連付けられたノートにアクセス・変更できるだけでなく、ノートを追加することもできます。|
|シェイプの検索|シェイプに割り当てられた代替テキストを使用して、スライド内の特定のシェイプを検索できます。|
|背景|プレゼンテーションのマスターまたは通常スライドに関連付けられた背景の操作が可能です。|
|テキストボックス|テキストボックスはゼロから作成できます。既存のテキストボックスにアクセスし、元の書式を保持したままテキストを変更することもできます。|
|長方形シェイプ|長方形シェイプを作成または変更できます。|
|ポリラインシェイプ|ポリラインシェイプも作成または変更できます。|
|楕円シェイプ|楕円シェイプも作成または変更できます。|
|グループシェイプ|Aspose.Slides for C++ はグループシェイプをサポートします。|
|オートシェイプ|オートシェイプもサポートされています。|
|シェイプのシリアライズ|Aspose.Slides for C++ は多数のシェイプをサポートしていますが、サポート外のシェイプについては、既存スライドからシリアライズし、後で再利用できるメソッドが提供されています。|
|画像フレーム|画像は Aspose.Slides for C++ を使用して画像フレーム内で管理されます。|
|オーディオフレーム|オーディオファイルは、オーディオフレームにリンクまたは埋め込みで配置できます。|
|ビデオフレーム|ビデオファイルはビデオフレームを介して処理されます。|
|OLE フレーム|OLE オブジェクトは OLE フレームで管理されます。|
|テーブル|スライド上のテーブルもサポートされています。|
|テキストフレーム|任意のシェイプに関連付けられたテキストは、そのシェイプのテキストフレームを通じてアクセスできます。|
|テキストスキャン|プレゼンテーション全体またはスライド単位でテキストをスキャンできる組み込みメソッドがあります。|
|アニメーション|シェイプにアニメーションを適用できます。|
|スライドショー|スライドショー、スライド遷移、モーフ遷移もサポートされています。|

## **書式設定機能**
Aspose.Slides for C++ を使用すると、プレゼンテーションドキュメント内のスライド上のテキストとシェイプの書式設定が可能です。

|**機能**|**説明**|
| :- | :- |
|テキスト書式設定|<p>Aspose.Slides for C++ では、テキストはシェイプに関連付けられたテキストフレームを通じて管理されます。そのため、段落やテキストフレーム内の各部分を使用してテキストをフォーマットできます。以下のテキスト要素が書式設定可能です。</p><p>- フォントの種類</p><p>- フォントサイズ</p><p>- フォントカラー</p><p>- フォントの濃淡</p><p>- 段落の配置</p><p>- 段落の箇条書き</p><p>- 段落の方向</p>|
|シェイプ書式設定|<p>Aspose.Slides for C++ では、スライドの基本要素はシェイプです。以下のシェイプ要素が書式設定可能です。</p><p>- 位置</p><p>- サイズ</p><p>- 線種</p><p>- 塗りつぶし（パターン、グラデーション、単色を含む）</p><p>- テキスト</p><p>- 画像</p>|

## **FAQ**

**サーバーや PC に Microsoft PowerPoint をインストールする必要がありますか？**

いいえ。PowerPoint は必要ありません。Aspose.Slides はプレゼンテーションの作成、編集、変換、レンダリングを行うスタンドアロンエンジンです。

**マルチスレッドはどのように機能しますか？処理の並列化は可能ですか？**

異なるスレッドで別々のドキュメントを処理することは安全です。ただし、同じ [presentation](https://reference.aspose.com/slides/cpp/aspose.slides/presentation/) オブジェクトを [複数のスレッド](/slides/ja/cpp/multithreading/) が同時に使用してはいけません。

**ファイルのパスワードや暗号化はサポートされていますか？**

はい。[暗号化されたプレゼンテーション](/slides/ja/cpp/password-protected-presentation/) を開いたり、開くパスワードや書き込みパスワードを設定・削除したり、保護状態を確認したりできます。

**Linux コンテナでフォントパッケージに注意する必要がありますか？**

はい。一般的なフォントパッケージをインストールするか、アプリケーションで [フォントディレクトリを明示的に指定](/slides/ja/cpp/custom-font/) して、予期しない置き換えを防止することを推奨します。

**評価版に制限はありますか？**

[評価モード](/slides/ja/cpp/licensing/) では出力に透かしが追加され、いくつかの制限が適用されます。フル機能テスト用に [30 日間の一時ライセンス](https://purchase.aspose.com/temporary-license/) が利用可能です。

**外部形式（PDF/HTML → PPTX）をプレゼンテーションにインポートできますか？**

はい。[PDF ページや HTML コンテンツ](/slides/ja/cpp/import-presentation/) をプレゼンテーションに追加し、スライドに変換できます。
