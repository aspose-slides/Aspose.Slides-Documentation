## **プレゼンテーションとシェイプのテキストの言語を変更する**
- [Presentation](https://reference.aspose.com/slides/net/aspose.slides/presentation) クラスのインスタンスを作成します。
- インデックスを使用してスライドの参照を取得します。
- スライドに長方形型のオートシェイプを追加します。
- テキストフレームにテキストを追加します。
- テキストに言語 ID を設定します。
- プレゼンテーションを PPTX ファイルとして保存します。
