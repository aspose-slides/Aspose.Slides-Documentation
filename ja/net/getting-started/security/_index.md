## **Aspose.Slides のセキュリティ**

Aspose は製品開発時にベストプラクティスを適用しています。

* Aspose.Slides for .NET は、他の形式のファイルでプレゼンテーションを操作および変換するために使用されます。プレゼンテーション内のスクリプトは実行しません。Aspose.Slides は単にプレゼンテーションの構造を解析し、エンドユーザーのコードがオブジェクトモデルを便利に操作できるようにします。 
* Aspose.Slides は、リモートコードの実行なしにドキュメントを解析・解釈するライブラリとして機能します。すべての Aspose 製品はお使いのマシン上で実行され、Aspose にデータを送信しません。唯一の例外は、[従量課金ライセンス](https://purchase.aspose.com/faqs/licensing/metered) が関与している場合で、この場合でも（従量課金ライセンスを使用している場合）処理されるのは API の使用情報だけです。

* Aspose コンポーネントは通常のアプリケーションと同じユーザー コンテキストで実行されます。したがって、Aspose コンポーネントが重要なシステム リソースにリスクをもたらすことはありません。さらに、Aspose コンポーネントでドキュメントが開かれた場合、マクロは自動的に実行されません。 

* Microsoft Office パッケージに固有または関連するリスクは Aspose コンポーネントには適用されないため、Aspose 製品は非常に安全です。

## **FAQ**

**Aspose のコードに対する脆弱性を監視するために使用しているシステムは何ですか？**

すべての Aspose.Slides リリースに対して静的コード分析を実行しています。Aspose.Slides のコードが OWASP Top 10 を通過していることを証明するセキュリティレポートを提供可能です。 

**外部パッケージの脆弱性を監視するために使用しているシステムはありますか？**

Aspose.Slides は外部パッケージを含んだり使用したりしないため、外部パッケージの脆弱性を確認するシステムは導入していません。Aspose.Slides は .NET フレームワーク自体のみに依存しています。
