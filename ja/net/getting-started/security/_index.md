## **Aspose.Slides のセキュリティ**

Aspose は製品開発時にベストプラクティスを適用しています。

* Aspose.Slides for .NET は、プレゼンテーションを他の形式のファイルで操作および変換するために使用されます。プレゼンテーション内のスクリプトは実行しません。Aspose.Slides はプレゼンテーション構造を単に解析し、エンドユーザーのコードがオブジェクトモデルを便利に操作できるようにします。
* Aspose.Slides は、ドキュメントを解析・解釈するライブラリとして機能し、リモートコードの実行は行いません。すべての Aspose 製品はお客様のマシン上で実行され、Aspose にデータを送信することはありません。唯一の例外は、[metered license](https://purchase.aspose.com/faqs/licensing/metered) を使用する場合であり、その場合でも（メータライセンスを使用している場合）API 使用情報のみが処理されます。
* Aspose コンポーネントは、通常のアプリケーションと同じユーザーコンテキストで実行されます。したがって、Aspose コンポーネントが重要なシステムリソースにリスクをもたらすことはありません。また、Aspose コンポーネントでドキュメントが開かれた場合、マクロは自動的に実行されません。
* Microsoft Office パッケージに固有または関連するリスクは Aspose コンポーネントには適用されないため、Aspose 製品は非常に安全です。

## 質問と回答

> Aspose のコードにおける脆弱性を監視するために使用されているシステムは何ですか？

Aspose.Slides の各リリースに対して静的コード解析を実施しています。Aspose.Slides のコードが OWASP Top 10 を満たしていることを証明するセキュリティレポートを提供できます。

> 外部パッケージの脆弱性を監視するために、どのようなシステムを導入していますか？

Aspose.Slides は外部パッケージを含んだり使用したりしないため、外部パッケージの脆弱性をチェックするシステムは導入していません。Aspose.Slides は .NET フレームワーク自体のみを依存しています。
