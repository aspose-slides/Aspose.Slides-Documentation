## **Product Description**
Aspose.Slides for .NET は Microsoft PowerPoint® 管理 API で、.NET アプリケーションが Microsoft PowerPoint® を使用せずに PowerPoint® ドキュメントの読み書きを可能にします。

### **Why You Should Use Aspose.Slides for .NET**

- Aspose.Slides for .NET は PowerPoint® ドキュメントの管理に必要な機能を提供する、唯一かつ最初のコンポーネントです。  
- Aspose.Slides for .NET はテキスト、図形、表、アニメーションの管理、スライドへの音声・動画の追加、スライドのプレビュー、SVG・PDF などへのエクスポートなど、数多くの重要機能を提供します。  
- Aspose.Slides for .NET はさまざまなソースから PowerPoint ファイルを開くだけでなく、プレゼンテーションを多様な方法で保存できます。たとえば、PPT 形式で保存したり、スライドを画像として保存したりできます。  

- Aspose.Slides for .NET は Microsoft PowerPoint にあるほぼすべての機能を提供します。テーブルやさまざまな形状に加えて、テキスト、音声、動画フレームなどの異なるフレームをスライドに追加して利用できます。  
- Aspose.Slides for .NET は Managed C# で開発されています。C#、VB.NET、J# など、任意の .NET 言語で使用できます。また、Web アプリケーションでも Windows アプリケーションでも利用できます。  
- Aspose.Slides for .NET は、開発者が API をよりよく理解できるように、完全なデモと実用的なサンプルも提供しています。

## **Aspose.Slides Features**
Aspose.Slides for .NET の機能は以下のグループに分けられます。

1. Presentation Features.  
1. Slides Features.  
1. Graphics and Multimedia Features.  
1. Table Features.  
1. Text & Paragraph Features  
1. Advanced Features  

### **Presentation Features**
- さまざまなソースから Microsoft PowerPoint プレゼンテーションを開く。  
- 異なる Microsoft PowerPoint 形式でプレゼンテーションを保存。  
- ゼロからプレゼンテーションを作成。  
- 他のコンポーネントを使用せずに PDF ドキュメントに変換。  
- プレゼンテーションのドキュメントプロパティを読み取りまたは変更。  
- プレゼンテーションに保護を適用。  
- プレゼンテーションを PDF にエクスポート。  
- プレゼンテーションを TIFF にエクスポート。  
- プレゼンテーションを XPS にエクスポート。  
- プレゼンテーションを SWF にエクスポート。  
- プレゼンテーションを PDF Note / Tiff Notes にエクスポート。  
- プレゼンテーションを HTML にエクスポート。

### **Slides Features**
- ゼロから新しいスライドを作成し、プレゼンテーションに追加。  
- プレゼンテーション内の位置で任意のスライドにアクセス。  
- 同一または別のプレゼンテーションにスライドをコピーまたはクローン。  
- スライドの編集と削除。  
- マスタとスライドを自動的にコピー。  
- スライドのプレビュー。  
- スライド名の取得または設定。  
- スライドを画像として保存。  
- スライドを SVG 形式にエクスポート。  
- プレゼンテーション内の多数のマスタを操作。  
- 任意のサポートされたスタイルでスライド背景を塗りつぶし。  
- 新しいスライドノートの追加または管理。  
- 新しいスライドマスタとスタイルを動的に設定。  
- スライドショーのトランジションの追加または変更。  
- モーフィング トランジションの追加。

### **Graphics & Multimedia Features**
- 線、長方形、楕円、多角線などの図形を作成または管理。  
- 図形の線スタイルを管理。  
- 図形の代替テキストを取得。  
- 図形を回転・反転。  
- 図形をさまざまなスタイルで塗りつぶし（単色、グラデーション、パターン、テクスチャ、画像など）。  
- 48 種類の事前定義パターンと 24 種類のテクスチャを使用。  
- 図形にハイパーリンクを割り当て。  
- スライドや図形に新しいアニメーション効果を読み取り、変更、追加。  
- プレゼンテーションに新しい画像を追加し、既存画像を管理。  
- プレゼンテーションに新しい画像バレットを追加し、既存バレットを管理。  
- プレゼンテーションに新しい音声フレームを追加し、既存フレームを管理。  
- プレゼンテーションに新しい動画フレームを追加し、既存フレームを管理。

### **Table Features**
- ゼロから表を作成し、スライドに追加。  
- 表のサイズ変更と移動。  
- 表の任意のセルにアクセス。  
- 任意のセルの書式と罫線を変更。  
- 列と行のサイズ変更。  
- 列と行の追加・削除。  
- 表全体の罫線幅と色を設定。

### **Text & Paragraph Features**
- テキストプレースホルダーとテキストフレームのテキスト取得・設定。  
- プレースホルダーのタイプ取得。  
- テキストフレーム内のテキスト書式変更。  
- プレゼンテーションで使用されているフォントの追加または管理。  
- テキストとテキスト書式の変更。  
- テキストへのリンク作成または管理。  
- タブと有効タブのサポート。  
- ほぼすべての形状に新しいテキストフレームを追加。  
- テキストフレームの余白とテキスト折り返し設定の取得・変更。  
- 段落の字下げ取得・変更。

### **Advanced Features**
- プレゼンテーションに埋め込まれた OLE オブジェクトにアクセス。  
- プレゼンテーションのロック。

## **Declaration**
すべての Aspose .NET コンポーネントは Full Trust 権限セットが必要です。理由は、フォントの解析や同様の処理のためにレジストリ設定、システムファイル、仮想ディレクトリへの高度な権限が求められるからです。また、Aspose .NET コンポーネントはコア .NET システムクラスに基づいており、多くの場合 Full Trust が必要です。

複数企業のアプリケーションをホストする ISP は通常 Medium Trust セキュリティレベルを適用します。NET 2.0 の場合、次の制限が課されます:

- OleDbPermission が利用できません。これにより ADO.NET 管理 OLE DB データプロバイダーを使用したデータベースアクセスが不可となります。  
- EventLogPermission が利用できません。Windows イベントログへのアクセスが不可です。  
- ReflectionPermission が利用できません。リフレクションが使用できません。  
- RegistryPermission が利用できません。レジストリへのアクセスが不可です。  
- WebPermission が制限されます。<trust> 要素で定義したアドレスまたはアドレス範囲にしか通信できません。  
- FileIOPermission が制限されます。アプリケーションの仮想ディレクトリ階層内のファイルしかアクセスできません。

上記の理由により、Aspose .NET コンポーネントは Full Trust 権限セットを付与されたサーバーでのみ使用可能です。

## **FAQ**

**Does it support PDF conformance levels for archiving and accessibility (PDF/A and PDF/UA)?**

Yes. You can save to PDF with PDF/A-2a/2b/2u, PDF/A-3a/3b, as well as PDF/UA by configuring [PDF export options](https://reference.aspose.com/slides/net/aspose.slides.export/pdfoptions/).

**Is there a font substitution mechanism and support for custom fonts to ensure correct rendering?**

Yes. The library lets you [load custom fonts](/slides/ja/net/custom-font/) and [define fallback rules](/slides/ja/net/fallback-font/) so missing glyphs are substituted reliably during conversion and rendering.

**Can I detect whether a file is password-protected without fully opening it?**

Yes. You can [inspect a presentation](/slides/ja/net/examine-presentation/) to determine if it requires a password before loading the full document.

**Is Microsoft PowerPoint required for processing and conversions?**

No. Aspose.Slides is a standalone engine; PowerPoint is not needed on the server or workstation.

**Is it safe to process presentations from multiple threads?**

Yes, you can process different documents in parallel in separate threads; just avoid using the same presentation instance concurrently [across threads](/slides/ja/net/multithreading/).

**Are macros preserved and can I manage VBA in PPTM/PPSM files?**

Yes. Presentations with macros [are supported](/slides/ja/net/presentation-via-vba/), and you can [inspect and manage VBA projects](https://reference.aspose.com/slides/net/aspose.slides.vba/) in those files.

**Can I convert PDF or HTML back into PowerPoint slides?**

Yes. You can [import PDF pages or HTML content](/slides/ja/net/import-presentation/) to create or populate slides in a presentation.

**Is XPS export supported, and can I control the quality and content of the XPS output?**

Yes. [Export to XPS](/slides/ja/net/convert-powerpoint-to-xps/) is available, and [save options](https://reference.aspose.com/slides/net/aspose.slides.export/xpsoptions/) allow you to tune output quality and included content.

**Can I convert slides to images and control the output quality?**

Yes. Slides can be [rendered to PNG, JPEG, GIF, BMP, TIFF](/slides/ja/net/convert-powerpoint-to-png/) and other formats with fine-grained control over size and quality.

**Are mathematical formulas supported (MathML/MathText) and can they be exported?**

Yes. You can [create and edit mathematical text](/slides/ja/net/powerpoint-math-equations/) and [export equations](/slides/ja/net/exporting-math-equations/), including to MathML.

**How does the library handle encryption and passwords (open, set, remove)?**

[It supports](/slides/ja/net/password-protected-presentation/) opening encrypted presentations, setting or removing passwords for opening and modifying, and checking a file’s protection status.
