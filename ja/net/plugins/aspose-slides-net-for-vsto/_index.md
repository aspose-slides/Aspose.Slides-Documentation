## **VSTOとは？**
VSTO（Visual Studio Tools for Office）または Microsoft Visual Studio Tools は、Microsoft の Visual Studio .NET ツールスイートの一部で、Microsoft .NET Framework をサポートしています。開発者はこれを使用して Microsoft Office の Word、Excel、Outlook アプリケーションと連携するコードを記述します。VSTO を使用する開発者は C# または Visual Basic 言語を利用できます。これは Visual Basic for Applications（VBA）の代替手段です。

## **Aspose.Slides for .NET**
Aspose.Slides for .NET は、Microsoft PowerPoint を使用せずに .NET アプリケーションが PowerPoint ドキュメントの読み取り、書き込み、操作を行えるユニークな PowerPoint® 管理コンポーネントです。

Aspose.Slides for .NET は、独自のアプリケーション内で PowerPoint ドキュメントを管理する機能を提供する、唯一かつ初の .NET コンポーネントです。すべての Aspose .NET コンポーネントと同様に、Aspose.Slides はマネージド C# で記述され、非常に高価値かつ高速です。

## **Aspose.Slides for VSTO**
## **概要**
ファイル形式ライブラリを探している開発者は、市場に出回っているライブラリが提供する機能を比較検討する必要があります。選択時に主に考慮されるのは次の 2 つの要素です。

- 機能
- ライブラリ使用に必要な工数

より高速で、簡単かつ軽量なファイル形式ライブラリを探している .NET 開発者は、VSTO（Visual Studio Tools for Office）と Aspose を比較したくなるでしょう。このプロジェクトは、両ライブラリの機能とコードを比較する手助けをします。

## **システム要件およびサポートプラットフォーム**
### **システム要件**
以下のいずれかの Visual Studio Tools for Office エディション：

- Microsoft Visual Studio 2005 Tools for the Microsoft Office System
- Microsoft Visual Studio 2005 Tools for the 2007 Microsoft Office System
- Visual Studio 2008 Professional Edition
- Visual Studio 2008 Team Suite Edition
- Microsoft Office のいずれかのバージョン：
  - Microsoft Office Professional 2003 SP1
  - 2007 Microsoft Office System

### **サポート対象 OS**
Windows 2000、Windows Server 2003、Windows Vista、Windows XP

### **サポート対象 .NET Framework バージョン**
.NET Framework 2.0 以降
