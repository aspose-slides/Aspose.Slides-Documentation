## **導入**
Microsoft Visual Studio は Microsoft が提供する統合開発環境（IDE）です。Microsoft Windows 用のコンピュータプログラムだけでなく、Web サイト、Web アプリケーション、Web サービスの開発にも使用されます。Visual Studio は Windows API、Windows Forms、Windows Presentation Foundation、Windows Store、Microsoft Silverlight などの Microsoft ソフトウェア開発プラットフォームを利用します。ネイティブコードとマネージドコードの両方を生成できます。
### **Aspose.Slides for .NET**
Aspose.Slides for .NET は、.NET アプリケーションが Microsoft PowerPoint を使用せずに PowerPoint 文書を読み書き・操作できるユニークな PowerPoint® 管理 API です。

Aspose.Slides for .NET は、.NET 用 PowerPoint API として初めて、そして唯一、独自のアプリケーション内で PowerPoint 文書を管理する機能を提供します。すべての Aspose .NET コンポーネントと同様に、Aspose.Slides はマネージド C# で記述され、非常にコストパフォーマンスが高く、驚異的な高速性を誇ります。

詳細は [Aspose.Slides for .NET API](http://www.aspose.com/.net/powerpoint-component.aspx) をご参照ください。
### **Visual Studio を使用した Aspose.Slides for .NET のサンプルの探求**
Aspose Visual Studio Plugin - **Explore Aspose.Slides for .NET Examples** は、Aspose.Slides for .NET API のサンプルをすばやくダウンロードして探索できる便利なツールです。Visual Studio を離れることなく、最新のサンプルプロジェクトをシームレスに選択、ダウンロード、オープンできるシンプルなオプションを提供し、時間と労力を大幅に節約します。

このバージョンの主な機能は以下のとおりです。

- Visual Studio 2010、Visual Studio 2012、Visual Studio 2013 をサポート
- Visual Studio の **File > Aspose** メニューから簡単に起動
- 時間の節約と学習曲線の短縮
- Github.com にホストされている最新の Aspose.Slides for .NET API サンプルを選択して開くことが可能
- **C#** または **VB.NET** のサンプルプロジェクトを開くことができる
- 選択したプロジェクトは、必要な参照が自動的に設定された状態で Visual Studio に開かれ、すぐに実行・探索できる環境が整います
- **NuGet** から最新の API ライブラリを、**GitHub** からサンプルを自動的にダウンロード

## **システム要件および対応プラットフォーム**
Aspose Visual Studio Plugin をインストールして使用するには、以下のいずれかの Visual Studio バージョンがインストールされている必要があります。

- Visual Studio 2010、2012、2013

インストールや使用中に問題が発生した場合はお気軽にお問い合わせください。

## **インストール**
Aspose Visual Studio Plugin のインストールは非常にシンプルです。以下の手順に従ってください。

- ダウンロードした msi ファイルをダブルクリック
- 各画面で **Next** を押すだけの標準設定でインストールを進めます

インストールが完了すると成功ダイアログが表示されます。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_2)

**Note:** 変更を有効にするため、Visual Studio を再起動してください。
## **使用方法**
インストール後は、以下の手順でインタラクティブなプラグインを簡単に起動して使用できます。

**1.** Visual Studio で **File > Aspose** を選択し、続いて **New Aspose.Slides Example Project** をクリックしてプラグインを開きます。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_3)

**2.** プラグインを初めて開くと、以下のように API 選択ダイアログが表示されます。**NuGet** から最新の API とライブラリ、**GitHub** から最新のサンプル ソースコードをダウンロードするのに数分かかります。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_4)

**3.** 「Select an API to View Examples」ダイアログで、サンプルプロジェクトの作成場所を選択または変更し、サンプルプロジェクトの種類（**C#** または **VB.NET**）を選択して **Continue** をクリックすると、Visual Studio でサンプルプロジェクトが開きます。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_1)

**4.** **C#** または **VB.NET** のプロジェクトタイプに応じた新しいサンプルプロジェクトが Visual Studio に作成されます。

### **拡張と貢献**
**Explore Aspose.Slides for .NET Examples** for Visual Studio はオープンソースであり、ソースコードは以下の主要なソーシャルコーディングサイトで入手可能です。開発者はソースコードをダウンロードし、独自の要件に合わせて機能を拡張することが奨励されています。

#### **ソースコード**
最新のソースコードは次のいずれかから取得できます。

- [Github](https://github.com/aspose-slides/Aspose.Slides-for-.NET)

#### **ソースコードの設定方法**
ソースコードを開いて拡張するには、以下がインストールされている必要があります。

- Visual Studio 2010

以下の手順に従って開始してください。

1. ソースコードをダウンロードまたはクローンします。
1. Visual Studio 2010 を起動し、**File > Open Project** を選択します。
1. ダウンロードした最新のソースコードに移動し、**AsposeVisualStudioPlugin.sln** を開きます。

#### **ソースコードの概要**
ソリューションには 2 つのプロジェクトが含まれます。

- **AsposeVisualStudioPlugin** – Visual Studio との統合、Aspose API およびサンプルの選択と自動ダウンロードを行うプロジェクト
- **Installer** – プラグイン用の MSI インストーラーファイルを生成する Wix インストーラープロジェクト
