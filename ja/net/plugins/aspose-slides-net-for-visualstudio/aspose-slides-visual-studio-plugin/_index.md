## **紹介**
Microsoft Visual Studio は、Microsoft が提供する統合開発環境 (IDE) です。Windows 用のコンピュータプログラムだけでなく、ウェブサイト、ウェブアプリケーション、ウェブサービスの開発にも使用されます。Visual Studio は、Windows API、Windows Forms、Windows Presentation Foundation、Windows Store、Microsoft Silverlight などの Microsoft ソフトウェア開発プラットフォームを利用します。ネイティブコードとマネージドコードの両方を生成できます。

### **Aspose.Slides for .NET**
Aspose.Slides for .NET は、Microsoft PowerPoint を使用せずに .NET アプリケーションが PowerPoint 文書を読み取り、書き込み、操作できるユニークな PowerPoint® 管理 API です。

Aspose.Slides for .NET は、独自のアプリケーション内で PowerPoint 文書を管理する機能を提供する、唯一かつ初の .NET PowerPoint API です。すべての Aspose .NET コンポーネントと同様に、Aspose.Slides はマネージド C# で記述されており、価格も驚くほど手頃で、極めて高速です。

Aspose.Slides for .NET API の詳細については、[Aspose.Slides for .NET API](http://www.aspose.com/.net/powerpoint-component.aspx)

### **Visual Studio を使用した Aspose.Slides for .NET のサンプルの探索**
Aspose Visual Studio Plugin - **Explore Aspose.Slides for .NET Examples** は、Aspose.Slides for .NET API のサンプルをすばやくダウンロードして探索できる優れたツールです。Visual Studio を離れることなく、最新のサンプルプロジェクトをシームレスに選択、ダウンロード、開くという非常にシンプルなオプションを提供し、時間と労力を大幅に節約します。

このバージョンは次の機能を提供します：

- Visual Studio 2010、Visual Studio 2012、Visual Studio 2013 をサポート。
- Visual Studio の **File > Aspose** メニューから簡単に起動。
- 時間を節約し、学習コストを低減。
- Github.com にホストされている最新の Aspose.Slides for .NET API サンプルを選択して開くことが可能。
- **C#** または **VB.NET** のサンプルプロジェクトを開くことができる。
- 選択されたプロジェクトは、すべての必須参照が自動的に設定された状態で Visual Studio で開かれ、すぐに実行および探索できる環境が整います。
- **NuGet** から最新の API ライブラリを自動的にダウンロードし、**GitHub** からサンプルを取得。

## **インストール**
Aspose Visual Studio Plugin のインストールは非常にシンプルです。以下の手順に従ってください。

- ダウンロードした msi ファイルをダブルクリック。
- 画面の指示に従い、各画面で「Next」を押すだけで標準設定でインストールできます。

インストールが完了すると、成功ダイアログが表示されます。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_2)

**注:** 変更を有効にするため、Visual Studio を再起動してください。

## **使用方法**
インストール後は、以下の手順でインタラクティブなプラグインを起動して使用できます。

**1.** Visual Studio で **File > Aspose** を選択し、続いて **New Aspose.Slides Example Project** を選びます。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_3)

**2.** プラグインを初めて開くと、以下のように API 選択ダイアログが表示されます。**NuGet** から最新の API とライブラリ、**GitHub** から最新のサンプル ソースコードをダウンロードするのに数分かかります。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_4)

**3.** 「Select an API to View Examples」ダイアログで、サンプルプロジェクトの作成場所とプロジェクトの種類 (**C#** または **VB.NET**) を選択または変更し、**Continue** をクリックして Visual Studio でサンプルプロジェクトを開きます。

![todo:image_alt_text](aspose-slides-visual-studio-plugin_1)

**4.** プロジェクトタイプ選択に応じた新しいサンプルプロジェクトが Visual Studio に作成されます。

#### **ソースコードの設定方法**
ソースコードを開いて拡張するには、次のものがインストールされている必要があります。

- Visual Studio 2010

以下の簡単な手順に従って開始してください。

1. ソースコードをダウンロードまたはクローンします。
1. Visual Studio 2010 を開き、**File** > **Open Project** を選択します。
1. ダウンロードした最新のソースコードに移動し、**AsposeVisualStudioPlugin.sln** を開きます。

#### **ソースコード概要**
ソリューションには 2 つのプロジェクトが含まれています。

- **AsposeVisualStudioPlugin** – Visual Studio との統合、Aspose API とサンプルの選択および自動ダウンロードを行うプロジェクト。
- **Installer** – プラグイン用の MSI インストーラ ファイルを生成する Wix インストーラ プロジェクト。
