## **OpenXML とは何ですか？**
OpenXML SDK は、Microsoft が提供する .NET 用 SDK で、Microsoft Office 2007 以降のファイル形式を処理するために使用されます。これらのファイル形式は基本的に zip 圧縮されたパッケージであり、Open XML SDK for Office は、パッケージ内の Open XML パッケージおよび基礎となる Open XML スキーマ要素の操作を簡素化します。Open XML SDK のクラスは、開発者が Open XML パッケージで頻繁に行う多くの共通タスクをカプセル化しているため、数行のコードで複雑な操作を実行できます。

Open XML SDK 2.5 をインストールしたら、Visual Studio で既存のプロジェクトまたはアプリケーションを開くか、新規にプロジェクトまたはアプリケーションを作成します。その後、プロジェクトまたはアプリケーションに次のコンポーネントへの参照を追加します。

- DocumentFormat.OpenXml
- WindowsBase
## **Aspose.Slides for .NET**
Aspose.Slides for .NET は、PowerPoint® を使用せずに .NET アプリケーションで PowerPoint ドキュメントの読み取り、書き込み、操作を可能にするユニークな PowerPoint 管理コンポーネントです。

Aspose.Slides for .NET は、PowerPoint ドキュメントを自アプリケーション内で管理する機能を提供する、唯一の .NET コンポーネントです。すべての Aspose .NET コンポーネントと同様に、Aspose.Slides はマネージド C# で記述され、非常にコストパフォーマンスが高く、超高速です。

## **Introduction**
ファイル形式ライブラリを探している開発者は、市場に出回っているライブラリの機能を比較検討する必要があります。主に、次の二つの要素が選択時の判断基準となります。

- 機能
- ライブラリを使用するために必要な工数

もし .NET 開発者で、ファイル処理において高速・簡単・軽量なライブラリを求めているなら、VSTO（Visual Studio Tools for Office）と Aspose を比較検討する必要があります。本プロジェクトは、両ライブラリの機能とコードを比較する際の手助けとなります。
## **System Requirements and Supported Platforms**
### **System Requirements**
以下のいずれかの Visual Studio Tools for Office エディションが必要です。

- Microsoft Visual Studio 2005 Tools for the Microsoft Office System
- Microsoft Visual Studio 2005 Tools for the 2007 Microsoft Office System
- Visual Studio 2008 Professional Edition
- Visual Studio 2008 Team Suite Edition
- 次のいずれかの Microsoft Office バージョン
  - Microsoft Office Professional 2003 SP1
  - 2007 Microsoft Office system
### **Supported Operating Systems**
- Windows 2000、Windows Server 2003、Windows Vista、Windows XP
### **.NET Framework Versions Supported**
- .NET Framework 2.0 以降のみ
