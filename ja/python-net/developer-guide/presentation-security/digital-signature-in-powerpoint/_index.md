## **はじめに**

**デジタル証明書** は、特定の組織または個人が作成したことを示す、パスワードで保護された PowerPoint プレゼンテーションを作成するために使用されます。デジタル証明書は、認可された組織（証明機関）に問い合わせることで取得できます。システムにデジタル証明書をインストールした後、File -> Info -> Protect Presentation からプレゼンテーションにデジタル署名を追加できます。
