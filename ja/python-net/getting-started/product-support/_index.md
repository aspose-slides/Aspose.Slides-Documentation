## **技術サポート**
{{% alert color="primary" %}}

Aspose はすべての製品に対して無制限の無料テクニカルサポートを提供しています。このサポートはすべてのユーザー（評価パッケージのユーザーを含む）に利用可能です。

{{% /alert %}} 

Aspose.Slides for Python via .NET のサポートが必要な場合は、次の点をご検討ください。

- サポートの主な手段は [Aspose.Forums](https://forum.aspose.com/c/slides/11) です。Aspose.Slides フォーラムに質問を投稿すると、数時間以内に回答が得られます。Aspose.Slides サポートチームはフォーラムに投稿された質問に直接回答します。
- Aspose は電話によるテクニカルサポートは提供していないことにご注意ください。電話サポートは販売および購入に関する質問のみ利用可能です。
- フォーラムでの返信を期待する場合は、時間帯の違いを考慮し、辛抱強くお待ちください。

Aspose.Slides for Python via .NET で問題が発生した場合は、次のガイドラインに従ってできるだけ効率的に解決してください。

- 問題を報告する前に、使用している Aspose.Slides for Python via .NET が最新バージョンであることを確認してください。おそらく、発生した問題は更新により既に解決されている可能性があります。

  最新バージョンの情報は Aspose.Slides ダウンロードで確認できます。

- 問題を報告する前に、フォーラム、このドキュメント、API リファレンスを確認してください。質問の答えはすでに提供されている可能性があります。

- 問題を報告する際は、元のドキュメントと（可能であれば）問題に関連するコードの一部を必ず添付してください。複数のファイルを添付する必要がある場合は、ZIP パッケージにまとめてください。

  Aspose.Forums にドキュメントを安全に添付できます。**only** あなたと Aspose の開発者だけが添付ファイルにアクセスできます。

- 特定の問題ごとに1つのスレッドを作成して使用してください。別の問題が発生した場合は、別のスレッドを作成して報告する必要があります。

## **FAQ**

**利用可能な有料サポートの層は何ですか、そしてそれらはどのように異なりますか？**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services)。含まれるインシデント数、応答優先度、利用可能な通信チャネルにより異なります。

**バグを直接エンジニアリングチームにエスカレーションできますか？**

はい。[Paid support](https://helpdesk.aspose.com/) は、問題の詳細な調査が必要な場合に開発者へのエスカレーションを可能にします。

**古いライブラリバージョンはサポートされていますか、またそれらに対して修正はリリースされますか？**

古いバージョンを使用することは可能ですが、修正は現在のリリース向けに提供されます。レガシーバージョン向けのパッチは提供されません。
