## **製品説明**
Aspose.Slides for Python via .NET は、Microsoft PowerPoint® を使用せずにさまざまなドキュメント処理タスクを実行できるクロスプラットフォーム クラス ライブラリです。

### Aspose.Slides for Python via .NET を使用すべき理由

- Aspose.Slides for Python via .NET は、PowerPoint® ドキュメントの管理に必要な機能を提供する唯一のコンポーネントです。  
- Aspose.Slides for Python via .NET は、テキスト、シェイプ、テーブル、アニメーションの管理、スライドへの音声・動画の追加、スライドのプレビュー、SVG や PDF などへのエクスポートなど、多数の主要機能を提供します。  
- Aspose.Slides for Python via .NET は、さまざまなソースから PowerPoint ファイルを開くだけでなく、プレゼンテーションを多様な方法で保存できます。たとえば、プレゼンテーションを PPT 形式で保存したり、スライドを画像として保存したりできます。  

- Aspose.Slides for Python via .NET は、Microsoft PowerPoint で利用できるほぼすべての機能を提供します。テーブルやさまざまなシェイプに加えて、テキスト、音声、動画フレームといった多種多様なフレームをスライドに追加できます。  
- Aspose.Slides for Python via .NET は Managed Python で開発されており、Python、VB.NET、J# などの任意の .NET 言語で使用できます。また、Web アプリケーションでも Windows アプリケーションでも利用可能です。  
- Aspose.Slides for Python via .NET には、開発者が API をよりよく理解できるように、豊富なデモや実用的なサンプルが同梱されています。

## **Aspose.Slides の機能**
Aspose.Slides for Python via .NET の機能は以下のグループに分けられます。

1. プレゼンテーション機能  
1. スライド機能  
1. グラフィックおよびマルチメディア機能  
1. テーブル機能  
1. テキストと段落機能  
1. 高度な機能  

### **プレゼンテーション機能**
- さまざまなソースから Microsoft PowerPoint プレゼンテーションを開く  
- プレゼンテーションをさまざまな PowerPoint 形式で保存  
- ゼロからプレゼンテーションを作成  
- 他のコンポーネントを使用せずに PDF に変換  
- プレゼンテーションの文書プロパティを読み取り・変更  
- プレゼンテーションに保護を適用  
- プレゼンテーションを PDF にエクスポート  
- プレゼンテーションを TIFF にエクスポート  
- プレゼンテーションを XPS にエクスポート  
- プレゼンテーションを SWF にエクスポート  
- プレゼンテーションを PDF ノート / Tiff ノートにエクスポート  
- プレゼンテーションを HTML にエクスポート  

### **スライド機能**
- ゼロから新しいスライドを作成し、プレゼンテーションに追加  
- プレゼンテーション内の位置で任意のスライドにアクセス  
- 同一または別のプレゼンテーションにスライドをコピーまたはクローン  
- スライドを編集・削除  
- スライドと一緒にマスターを自動的にコピー  
- スライドをプレビュー  
- スライド名の取得または設定  
- スライドを画像として保存  
- スライドを SVG 形式にエクスポート  
- プレゼンテーション内で多数のマスターを操作  
- 任意のサポートスタイルでスライド背景を塗りつぶし  
- スライドノートの追加または管理  
- 新しいスライドマスターとスタイルを動的に設定  
- スライドショー遷移の追加または変更  
- モーフ遷移の追加  

### **グラフィック & マルチメディア機能**
- 線、長方形、楕円、ポリラインなどのシェイプの作成または管理  
- シェイプの線スタイルの管理  
- シェイプの代替テキスト取得  
- シェイプの回転・反転  
- ソリッド、グラデーション、パターン、テクスチャ、画像などのさまざまなスタイルでシェイプを塗りつぶし  
- 48 種類の定義済みパターンと 24 種類のテクスチャを使用  
- シェイプにリンクを割り当て  
- スライドやシェイプに新しいアニメーション効果を読み取り、変更、追加  
- プレゼンテーションに新しい画像を追加し、既存画像を管理  
- プレゼンテーションに新しい画像バレットを追加し、既存バレットを管理  
- プレゼンテーションに新しい音声フレームを追加し、既存フレームを管理  
- プレゼンテーションに新しい動画フレームを追加し、既存フレームを管理  

### **テーブル機能**
- ゼロからテーブルを作成し、スライドに追加  
- テーブルのサイズ変更と移動  
- テーブル内の任意のセルにアクセス  
- 任意のセルの書式や罫線を変更  
- 列と行のサイズ変更  
- 列と行の追加・削除  
- テーブル全体の罫線幅と色を設定  

### **テキスト & 段落機能**
- テキストプレースホルダーやテキストフレーム内のテキスト取得・設定  
- プレースホルダーのタイプ取得  
- テキストフレーム内のテキスト書式変更  
- 新しいフォントの追加またはプレゼンテーションで使用中のフォント管理  
- テキストとテキスト書式の変更  
- テキストへのリンク作成または管理  
- タブと有効タブのサポート  
- ほぼすべてのシェイプタイプに新しいテキストフレームを追加  
- テキストフレームの余白と改行設定の取得・変更  
- 段落のインデント取得・変更  

### **高度な機能**
- プレゼンテーションに埋め込まれた OLE オブジェクトへのアクセス  
- プレゼンテーションのロック  

## **テクニカルサポート**
Aspose はすべての製品に対して無制限の無料テクニカルサポートを提供しています。評価版を含むすべてのユーザーが利用できます。Aspose.Slides for Python via .NET のサポートが必要な場合は、以下をご確認ください。

- 主なサポート窓口は[Aspose Forums](https://forum.aspose.com/)です。[Aspose.Slides フォーラム](https://forum.aspose.com/c/slides/11)に質問を投稿すれば、数時間以内に回答が得られます。Aspose.Slides サポートチームが直接フォーラムの質問に回答します。  
- Aspose は電話でのテクニカルサポートは提供していません。電話サポートは販売・購入に関する質問のみ利用可能です。  
- フォーラムでの返信を待つ際は、時差を考慮して忍耐強くお待ちください。  

Aspose.Slides for Python via .NET に問題がある場合は、以下の手順で効率的に解決してください。

- 問題を報告する前に、まず使用している Aspose.Slides for Python via .NET が最新バージョンか確認してください。既に更新で解決されている可能性があります。  

  最新バージョンは[Aspose.Slides ダウンロード](https://pypi.org/project/aspose.slides/)で確認できます。  

- 問題を報告する前に、フォーラムや本ドキュメント、API リファレンスを確認してください。すでに回答が掲載されていることがあります。  

- 問題を報告する際は、対象のドキュメントと（可能であれば）問題に関係するコードの断片を添付してください。複数ファイルを添付する場合は ZIP パッケージにまとめてください。  

  Aspose.Forums では、添付ファイルはあなたと Aspose の開発者だけがアクセスできるため、安全に使用できます。  

- 特定の問題ごとにスレッドを 1 つ作成してください。別の問題がある場合は新しいスレッドを作成して報告してください。

## **FAQ**

**PDF/A や PDF/UA などのアーカイブ・アクセシビリティ向け PDF 準拠レベルをサポートしていますか？**  
はい。PDF エクスポート オプションを設定することで、PDF/A-2a/2b/2u、PDF/A-3a/3b、PDF/UA での保存が可能です。[PDF エクスポート オプション](https://reference.aspose.com/slides/python-net/aspose.slides.export/pdfoptions/)をご参照ください。

**フォント置換機構やカスタムフォントのサポートはありますか？**  
はい。ライブラリは[カスタムフォントのロード](/slides/ja/python-net/custom-font/)と[フォールバック規則の定義](/slides/ja/python-net/fallback-font/)を提供し、欠損グリフを確実に代替できます。

**ファイルがパスワード保護されているか、完全に開かずに検出できますか？**  
はい。[プレゼンテーションの検査](/slides/ja/python-net/examine-presentation/)でパスワードが必要かどうかを判別できます。

**Microsoft PowerPoint は処理や変換に必須ですか？**  
いいえ。Aspose.Slides は単体エンジンであり、サーバーやワークステーションに PowerPoint は不要です。

**マルチスレッドでプレゼンテーションを処理しても安全ですか？**  
はい。別スレッドで異なるドキュメントを並行処理できますが、同一プレゼンテーション インスタンスをスレッド間で同時に使用しないでください。[マルチスレッド処理](/slides/ja/python-net/multithreading/)をご参照ください。

**マクロは保持されますか？PPTM/PPSM ファイルの VBA を管理できますか？**  
はい。マクロ付きプレゼンテーションは[サポート](/slides/ja/python-net/presentation-via-vba/)されており、[VBA プロジェクトの検査と管理](https://reference.aspose.com/slides/python-net/aspose.slides.vba/)が可能です。

**PDF や HTML を PowerPoint スライドに変換できますか？**  
はい。[PDF ページや HTML コンテンツのインポート](/slides/ja/python-net/import-presentation/)でスライドを作成または追加できます。

**XPS エクスポートはサポートされていますか？出力品質や内容の制御は可能ですか？**  
はい。[XPS へのエクスポート](/slides/ja/python-net/convert-powerpoint-to-xps/)が利用でき、[保存オプション](https://reference.aspose.com/slides/python-net/aspose.slides.export/xpsoptions/)で品質や含める内容を調整できます。

**スライドを画像に変換し、出力品質を制御できますか？**  
はい。スライドは[PNG、JPEG、GIF、BMP、TIFF](/slides/ja/python-net/convert-powerpoint-to-png/)などにレンダリングでき、サイズや品質を細かく制御できます。

**数式 (MathML/MathText) はサポートされ、エクスポートできますか？**  
はい。[数式テキストの作成と編集](/slides/ja/python-net/powerpoint-math-equations/)および[数式のエクスポート](/slides/ja/python-net/exporting-math-equations/)が可能で、MathML 形式にもエクスポートできます。

**暗号化やパスワード (開く、設定、削除) の取り扱いはどうなりますか？**  
[パスワード保護されたプレゼンテーションの開封](/slides/ja/python-net/password-protected-presentation/)、パスワードの設定・解除、保護状態の確認がサポートされています。
