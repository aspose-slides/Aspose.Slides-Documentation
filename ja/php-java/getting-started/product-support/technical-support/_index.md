## **テクニカルサポート**

{{% alert color="primary" %}} 
Asposeはすべての製品に対して無制限の無料テクニカルサポートを提供しています。評価版を含むすべてのユーザーが利用できます。
{{% /alert %}} 

Aspose.Slides for PHP via Java のサポートが必要な場合は、次の点をご検討ください。

- 主なサポート窓口は Aspose.Forums です。Aspose.Slides フォーラムに質問を投稿すると、数時間以内に回答が得られます。質問は Aspose.Slides サポートチームが直接回答します。
- Aspose は電話でのテクニカルサポートは提供していません。電話サポートは販売や購入に関する質問のみ利用可能です。
- フォーラムでの返信を期待する際は、時差を考慮してください。

Aspose.Slides for PHP via Java に問題がある場合は、以下の手順に従って最も効率的に解決してください。

- 問題を報告する前に、最新の Aspose.Slides for PHP via Java バージョンを使用していることを確認してください。最新バージョンは Aspose.Slides のダウンロードページで確認できます。
- フォーラム、本文書、および API リファレンスを確認してください。質問はすでに回答されている可能性があります。
- 問題を報告する際は、元のドキュメントと問題を引き起こすコードの断片を添付してください。複数ファイルを添付する場合は、1 つの ZIP にまとめてください。Aspose.Forums に添付されたファイルは、あなたと Aspose の開発者だけがアクセスできます。
- 1 スレッドにつき 1 件の問題を報告してください。別の問題がある場合は、別のスレッドで報告してください。

## **よくある質問**

**利用可能な有料サポート層は何ですか？それらはどのように異なりますか？**

[開発者、ビジネス、エンタープライズ](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services)。インシデント数、応答優先度、利用可能な通信チャネルに違いがあります。

**バグをエンジニアリングチームに直接エスカレーションできますか？**

はい。[有料サポート](https://helpdesk.aspose.com/) を利用すれば、問題が深い調査を必要とする場合に開発者へエスカレーションできます。

**古いライブラリ バージョンはサポートされますか？それらに対して修正はリリースされますか？**

古いバージョンを使用できますが、修正は現在のリリースに対してのみ提供されます。レガシー バージョン用のパッチは 発行されません。
