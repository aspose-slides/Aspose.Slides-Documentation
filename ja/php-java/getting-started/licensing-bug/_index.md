この記事では、Aspose.Slides for PHP via Java バージョン 23.2、23.3、23.4、23.5、23.6、23.7 で発生したライセンスバグについて説明します。このバグにより、サブスクリプションが期限切れになった永続ライセンスが一時的に使用できなくなることがあります。

## **症状**

永続ライセンスのサブスクリプションが期限切れになると、Aspose.Slides for PHP via Java バージョン 23.2〜23.7 はライセンスが期限切れであると報告し、すべての機能が評価モードに切り替わります。この動作は正しくなく、23.2 より前および 23.7 より後のバージョンには影響しません。

## **解決策**

Aspose.Slides for PHP via Java はこの問題に対処し、修正を含む更新バージョン (23.2.1、23.3.1、23.4.1、23.5.1、23.6.1、23.7.1) をリリースしました。

プロジェクトで影響を受けるバージョンの Aspose.Slides for PHP via Java を使用している場合は、修正済みバージョンに切り替えてください。

Aspose.Slides for PHP via Java でライセンスに関する問題が発生した場合は:

- 使用しているバージョンを確認し、上記のバグの影響を受けていないことを確認してください。
- [Aspose ドキュメント](https://docs.aspose.com/slides/php-java/getting-started/) を参照して、トラブルシューティング手順や既知のライセンス問題をご確認ください。
- [Aspose サポート](https://forum.aspose.com/) に連絡して、さらに支援を受けてください。

## **影響を受ける製品とバージョン**

このバグは Aspose.Slides for PHP via Java バージョン 23.2 から 23.7 のみが対象です。**他の Aspose 製品および Aspose.Slides の他のバージョンは影響を受けません**。
