この記事では、Aspose.Slides for PHP via Java バージョン 23.2、23.3、23.4、23.5、23.6、および 23.7で発生したライセンスバグについて説明します。このバグの影響で、 perpetual licenses がサブスクリプションの期限切れ後に一時的に使用できなくなりました。

## 症状 ##

永続ライセンスのサブスクリプションが期限切れになると、Aspose.Slides for PHP via Java バージョン 23.2 - 23.7 はライセンスが期限切れであると報告し、すべての機能が評価モードに移行する場合があります。この動作は誤りであり、23.2以前および23.7以降のバージョンには影響しません。

## 解決策 ##

Aspose.Slides for PHP via Java はこの問題に対処し、修正を含む更新版 (23.2.1、23.3.1、23.4.1、23.5.1、23.6.1、23.7.1) をリリースしました。

プロジェクトで影響を受けたバージョンの Aspose.Slides for PHP via Java を使用している場合は、修正版に切り替えてください。

Aspose.Slides for PHP via Java でライセンスの問題が発生した場合は：

- 使用しているバージョンを確認し、上記のバグに影響を受けていないことを確認してください。
- トラブルシューティング手順や既知のライセンス問題については、[Aspose ドキュメント](https://docs.aspose.com/slides/php-java/getting-started/)を参照してください。
- 更なる支援が必要な場合は、[Aspose サポート](https://forum.aspose.com/)までお問い合わせください。

## 影響を受ける製品とバージョン ##

このバグは、Aspose.Slides for PHP via Java バージョン 23.2 から 23.7 にのみ影響を与えます。**他の Aspose 製品や他のバージョンの Aspose.Slides には影響を与えません**。
