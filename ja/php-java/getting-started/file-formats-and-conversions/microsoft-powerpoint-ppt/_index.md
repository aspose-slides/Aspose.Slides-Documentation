{{% alert color="primary" %}} 

[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) は、さまざまなバージョンの Microsoft PowerPoint で作成、読み取り、操作、書き込みができるプレゼンテーション ドキュメント ファイル形式です。これは、Microsoft が開発したプレゼンテーション ドキュメント用のバイナリ形式です。

{{% /alert %}} 

## **PHP via Java 用 Aspose.Slides の PPT**
Aspose.Slides for PHP via Java は、以下に示すソフトウェアで作成された PPT ファイルを読み取ることができます。

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

同様に、Aspose.Slides for PHP via Java が作成した PPT ファイルは、上記のソフトウェアで読み取ることができます。

## **PPT の包括的サポート**
Aspose.Slides for PHP via Java は、PPT ドキュメント ファイル形式でサポートされているほぼすべての機能を提供します。これは、さまざまな Microsoft PowerPoint バージョンが提供する基本的および高度な機能だけでなく、Microsoft PowerPoint でもサポートされていない機能もカバーしています。Aspose.Slides for PHP via Java API ライブラリを使用する主な利点は、これらの機能を簡単に扱えることです。

基本的な作成、読み取り、書き込みタスクに加えて、Aspose.Slides for PHP via Java が提供するいくつかの機能は次のとおりです。

- 他の Microsoft Office ファイル形式を、[PPT ドキュメントへの OLE オブジェクト](/slides/ja/php-java/manage-ole/) としてインポートします。
- [PPT ドキュメントを PDF にエクスポート](/slides/ja/php-java/convert-powerpoint-to-pdf/)。
- PPT ドキュメント内のスライドを SVG 形式にエクスポートします。
- Java フレームワークがサポートする任意の画像形式にスライドをレンダリングします。
- PPT ドキュメント内のスライドのサイズを設定します。
- シェイプのアニメーションを管理します。
- スライドショーを管理します。
- [スライド上のテキストをフォーマット](/slides/ja/php-java/text-formatting/)。
- PPT ドキュメントからテキストを抽出します。
- [スライド上のテーブルを操作](/slides/ja/php-java/powerpoint-table/)。
- クローン機能を使用してマスターを自動的にコピーします。[クローン機能](/slides/ja/php-java/clone-slides/)。
