## **TIFF の概要**

[Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) は、1つの文書に複数の画像を格納できることで知られ、もともと Aldus によって作成されました。この形式は、スキャン、FAX、その他の画像操作アプリケーションで広くサポートされています。

## **PHP 用 Aspose.Slides（Java 経由）での TIFF**

Java 用 Aspose.Slide で読み込める任意の文書は、サードパーティ コンポーネントを必要とせずに、PHP 用 Aspose.Slides（Java 経由）で直接 TIFF 文書に変換できます。さらに、生成された TIFF 文書内の画像サイズを任意で指定することも可能です。PHP 用 Aspose.Slides（Java 経由）でプレゼンテーション文書を TIFF 文書にエクスポートする方法については、[this topic](/slides/ja/php-java/convert-powerpoint-to-tiff/) に情報があります。
