{{% alert color="primary" %}} 

Portable Document Format（[PDF](https://en.wikipedia.org/wiki/PDF)）は、Adobe Systems が組織間で文書を交換するために作成したファイル形式です。この形式の目的は、表示プラットフォームに関係なく、コンテンツとレイアウトを同一に保つことです。Aspose.Slides for PHP via Java を使用すると、プレゼンテーション ファイルを PDF に変換できます。

{{% /alert %}} 

## **Aspose.Slides for PHP via Java の PDF**
Aspose.Slides for PHP via Java に読み込むことができる任意のプレゼンテーションは、[PDF 1.5](https://en.wikipedia.org/wiki/PDF/A)、[PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A)、[PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) または [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA) に準拠した PDF に変換できます。Aspose.Slides for PHP via Java はプレゼンテーションを PDF にエクスポートし、ほとんどの場合、出力 PDF は元のプレゼンテーションとまったく同じ外観になります。

Aspose.Slides が PDF へ変換する際にサポートするプレゼンテーション機能：

- 画像、テキスト ボックス、その他の図形。
- テキストと書式設定。
- 段落と書式設定。
- ハイパーリンク。
- ヘッダーとフッター。
- 箇条書き。
- 表。

Aspose.Slides for PHP via Java を使用すれば、他のコンポーネントを必要とせずにプレゼンテーションを直接 PDF にエクスポートできます。さらに、[Converting to PDF](/slides/ja/php-java/convert-powerpoint-to-pdf/) で説明されているさまざまなオプションを使用して、プレゼンテーションから PDF へのエクスポートをカスタマイズできます。
