## **動的テキストの追加**
両方の方法は以下のステップに従います：

1. プレゼンテーションを作成します。
1. 空白のスライドを追加します。
1. テキストボックスを追加します。
1. テキストを設定します。
1. プレゼンテーションを書き込みます。
