{{% alert color="primary" %}}

Aspose.Slides for PHP via Java は、Microsoft PowerPoint® などを使用せずに、PHP アプリケーションがプレゼンテーション文書を読み書きできるクラス ライブラリです。Aspose.Slides for PHP via Java を使用すると、テキスト、図形、テーブルおよびアニメーションの管理、スライドへの音声・動画の追加、スライドのプレビュー、SVG や PDF 形式へのエクスポートなどが可能です。

最新の API リファレンスは、[Aspose.Slides for PHP via Java APIリファレンス](https://reference.aspose.com/slides/ja/php-java/)で確認できます。

{{% /alert %}}
