{{% alert color="primary" %}} 

Aspose.Slides for Java 14.4.0は、PowerPoint文書処理の新しい決定を提供します。いくつかの制限と既知の問題があり、今後のリリースで解消される予定です：

- 一部の図形がシリアル化されたPPT文書で誤ったジオメトリを持っています（弧、円形矢印、コールアウト）。
- PPTシリアル化ではすべてのPPTXテキストフォーマット機能がサポートされているわけではありません（タブ、インデント、および段落フォーマットの制限）。
- シリアル化されたPPT文書にはテキスト言語およびスペル設定に関する情報がありません。
- PPTシリアル化ではすべてのPPTXテーマ機能がサポートされているわけではありません（塗りつぶしフォーマット、線フォーマット、フォントのシリアル化のみ）。
- OLE/ActiveX PPTシリアル化における既知の問題があります。
- WordArtのシリアル化およびレンダリングはサポートされていません。

{{% /alert %}}
