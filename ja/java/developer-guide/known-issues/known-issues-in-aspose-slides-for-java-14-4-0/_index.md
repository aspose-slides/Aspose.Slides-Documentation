{{% alert color="primary" %}}

Aspose.Slides for Java 14.4.0 は、PowerPoint ドキュメントの処理に関する新しい決定事項を提供します。いくつかの制限と既知の問題があり、これらは今後のリリースで削除される予定です:

- シリアライズされた PPT ドキュメント内で、一部のシェイプのジオメトリが正しくありません（円弧、円形矢印、吹き出し）。
- PPT のシリアライズでは、すべての PPTX テキスト書式設定機能がサポートされているわけではありません（タブ設定、インデント、段落書式設定の制限）。
- テキストの言語情報やスペル設定がシリアライズされた PPT ドキュメントに含まれていません。
- PPT のシリアライズでは、すべての PPTX テーマ機能がサポートされているわけではありません（塗りつぶし形式、線形式、フォントのシリアライズのみ）。
- PPT への OLE/ActiveX のシリアライズに既知の問題があります。
- WordArt のシリアライズおよびレンダリングはサポートされていません。

{{% /alert %}}
