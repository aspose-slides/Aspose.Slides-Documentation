Aspose.Slides for Java 14.3.0 (14.4.0) は、PPT 処理の完全に新しい実装を提供します。多くの改善点があり、PPTX から PPT への部分的変換が可能です。ただし、未実装の機能もいくつかあります。

- シリアライズされた PPT ドキュメント内の一部のシェイプ（吹き出しなど）のジオメトリが正しくありません
- PPT シリアライズ時に PPTX のすべてのテキスト書式設定機能がサポートされていません
- シリアライズされた PPT ドキュメントにテキストの言語情報やスペル設定が含まれていません
- PPT シリアライズ時に PPTX テーマのすべての機能がサポートされていません

**Aspose.Slides for Java 8.6.0 と比較したいくつかの違い:**

- OLE/ActiveX の PPT へのシリアライズに既知の問題があります

**Aspose.Slides for .NET 14.3.0 と比較したいくつかの違い:**

- 現在、Aspose.Slides for Java ではプレゼンテーションの印刷サポートが利用できません
