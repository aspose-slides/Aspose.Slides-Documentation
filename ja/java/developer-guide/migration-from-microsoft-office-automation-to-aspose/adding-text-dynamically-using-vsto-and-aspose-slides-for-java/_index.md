## **動的なテキストの追加**
両方の方法は次の手順に従います：

1. プレゼンテーションを作成します。
1. 空のスライドを追加します。
1. テキストボックスを追加します。
1. いくつかのテキストを設定します。
1. プレゼンテーションを書き出します。
