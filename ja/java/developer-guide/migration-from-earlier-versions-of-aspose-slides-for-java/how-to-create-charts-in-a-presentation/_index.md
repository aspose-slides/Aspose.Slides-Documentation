## **レガシーコードのサポート**
Aspose.Slides for Javaの14.x.x以前のバージョンで開発されたレガシーコードを使用するには、コードにいくつかの小さい変更を加える必要があります。そうすれば以前と同じように動作します。古いAspose.Slides for Javaのcom.aspose.slidesおよびcom.aspose.slides.pptx名前空間に存在したすべてのクラスは、単一のcom.aspose.slides名前空間に統合されました。レガシーAspose.Slides APIを使用してプレゼンテーションで通常のチャートを最初から作成するための以下の簡単なコードスニペットを見て、新しい統合APIへの移行方法を示す手順に従ってください。
