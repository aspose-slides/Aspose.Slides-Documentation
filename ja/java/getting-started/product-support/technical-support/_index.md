## **テクニカルサポート**

{{% alert color="primary" %}} 

Aspose はすべての製品に対して無制限の無料テクニカルサポートを提供しています。評価版を含むすべてのユーザーがサポートをご利用いただけます。

{{% /alert %}} 

Aspose.Slides for Java のサポートが必要な場合は、以下をご覧ください。

- サポートの主要な窓口は Aspose.Forums です。Aspose.Slides フォーラムに質問を投稿すれば、数時間以内に回答が得られます。質問には Aspose.Slides サポートチームが直接回答します。
- Aspose は電話でのテクニカルサポートを提供していないことにご留意ください。電話サポートは販売・購入に関する質問のみ利用可能です。
- フォーラムでの返信を期待する場合、時差を考慮してください。


Aspose.Slides for Java に問題がある場合は、最も効率的に解決できるように次の簡単な手順に従ってください。

- 問題を報告する前に、最新の Aspose.Slides for Java バージョンを使用していることを確認してください。最新バージョンは Aspose.Slides ダウンロードで確認できます。
- フォーラム、この記事、および API リファレンスを確認してから問題を報告してください。すでに質問が回答されている可能性があります。
- 問題を報告する際は、元のドキュメントと問題を引き起こすコードの断片を添付してください。複数ファイルを添付する必要がある場合は、1つの zip にまとめてください。Aspose.Forums に添付したドキュメントは、あなたと Aspose の開発者だけがアクセスできるため安全です。
- 1 スレッドにつき 1 件の問題を報告するようにしてください。別の問題がある場合は、別スレッドで報告してください。

## **FAQ**

**利用可能な有料サポートプランは何ですか、またそれらはどのように違いますか？**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). インシデント数、回答優先度、利用可能なコミュニケーションチャネルがプランごとに異なります。

**バグを直接エンジニアリングチームにエスカレーションできますか？**

はい。[Paid support](https://helpdesk.aspose.com/) を利用すると、問題の詳細な調査が必要な場合に開発者へエスカレーションできます。

**古いライブラリバージョンはサポートされますか、また修正は提供されますか？**

古いバージョンも使用可能ですが、修正は現在のリリース向けに提供されます。レガシーバージョン向けのパッチは配布されません。
