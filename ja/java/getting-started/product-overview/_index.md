**Aspose.Slides for Javaへようこそ!**

Aspose.Slides for Javaは、Microsoft PowerPoint® を使用せずに、Java アプリケーションが PowerPoint® ドキュメントを読み書きできる Microsoft PowerPoint® 管理 API です。

Aspose.Slides for Javaは、PowerPoint® ドキュメントの管理に必要な機能を提供する唯一のコンポーネントです。

Aspose.Slides for Javaは、テキスト、シェイプ、テーブル、アニメーションの管理、スライドへのオーディオ・ビデオの追加、スライドのプレビュー、SVG、PDF などへのエクスポートなど、多くの主要機能を提供します。

## **製品の説明**
Aspose.Slides for Java は、さまざまなソースから PowerPoint ファイルを開くだけでなく、プレゼンテーションをさまざまな形式で保存できます。たとえば、プレゼンテーションを PPT 形式で保存したり、スライドを画像として保存したりできます。

Aspose.Slides for Java は、Microsoft PowerPoint にあるほぼすべての機能を提供します。Aspose.Slides for Java の柔軟性により、テーブルやさまざまなシェイプに加えて、テキスト、オーディオ、ビデオ フレームなどのさまざまなフレームをスライドに追加できます。

Aspose.Slides for Java には、開発者が API をよりよく理解できるように、完全なデモと実用的なサンプルも用意されています。

### **プレゼンテーション機能**

- さまざまなソースから Microsoft PowerPoint プレゼンテーションを開く。
- 異なる Microsoft PowerPoint フォーマットでプレゼンテーションを保存。
- ゼロからプレゼンテーションを作成。
- 他のコンポーネントを使用せずにプレゼンテーションを PDF 文書に変換。
- Open Document Presentation (ODP) を読み取る機能。
- PPT から PPTX、またはその逆への変換機能。
- プレゼンテーションのドキュメント プロパティを読み取りまたは変更。
- プレゼンテーションを PDF にエクスポート。
- プレゼンテーションを TIFF にエクスポート。
- プレゼンテーションを XPS にエクスポート。
- プレゼンテーションを SWF にエクスポート。
- プレゼンテーションを PDF ノート / Tiff ノートにエクスポート。
- プレゼンテーションを HTML にエクスポート。

### **スライド機能**

- ゼロから新しいスライドを作成し、プレゼンテーションに追加。
- プレゼンテーション内の位置で任意のスライドにアクセス。
- 同じプレゼンテーションまたは別のプレゼンテーションにスライドをコピーまたはクローン。
- スライドの編集と削除。
- マスタースライドを自動的にコピー。
- スライドのプレビュー。
- スライド名の取得または設定。
- スライドを画像として保存。
- スライドを SVG 形式にエクスポート。
- プレゼンテーション内で多数のマスターを操作。
- 任意のサポートされたスタイルでスライドの背景を塗りつぶし。
- 新しいスライドノートの追加または管理。
- 新しいスライドマスターとスタイルを動的に設定。
- スライドショーの遷移を追加または変更。
- モーフ遷移の追加

### **グラフィックとマルチメディア機能**

- 線、長方形、楕円、ポリラインなどのシェイプの作成または管理。

- シェイプの代替テキストの取得。
- シェイプの回転や反転。

- シェイプにリンクを割り当て。
- スライドやシェイプに新しいアニメーション効果を読み取り、変更、追加。
- プレゼンテーションに新しい画像を追加し、既存の画像を管理。
- プレゼンテーションに新しい画像バレットを追加し、既存のバレットを管理。
- プレゼンテーションに新しいオーディオフレームを追加し、既存のフレームを管理。
- プレゼンテーションに新しいビデオフレームを追加し、既存のフレームを管理。

### **テーブル機能**

- ゼロからテーブルを作成し、スライドに追加。

- テーブル内の任意のセルにアクセス。
- 任意のセルの書式と罫線を変更。

- 列と行の追加および削除。
- テーブル全体の罫線幅と色の設定。
- テーブルに事前設定のスタイルを適用

### **テキストと段落機能**

- テキストプレースホルダーとテキストフレーム内のテキストの取得と設定。
- プレースホルダーのタイプ取得。
- テキストフレーム内のテキスト書式の変更。
- プレゼンテーションで使用されているフォントの追加または管理。
- テキストとテキスト書式の変更。
- テキストへのリンクの作成または管理。

- 段落インデントの取得および変更。

### **高度な機能**

## **FAQ**

**PDF/A および PDF/UA など、アーカイブとアクセシビリティ用の PDF 準拠レベルをサポートしていますか？**

はい。PDF/A-2a/2b/2u、PDF/A-3a/3b、PDF/UA での保存は、[PDFエクスポート オプション](https://reference.aspose.com/slides/java/com.aspose.slides/pdfoptions/)を設定することで可能です。

**フォント置換機構やカスタムフォントのサポートはありますか？正しいレンダリングのために必要ですか？**

はい。ライブラリは[カスタムフォントのロード](/slides/ja/java/custom-font/)と[フォールバック ルールの定義](/slides/ja/java/fallback-font/)をサポートしており、変換およびレンダリング時に欠落したグリフを確実に置換できます。

**ファイルがパスワードで保護されているかどうかを、完全に開かずに検出できますか？**

はい。[プレゼンテーションを検査](/slides/ja/java/examine-presentation/)して、ロード前にパスワードが必要かどうかを判断できます。

**Microsoft PowerPoint は処理や変換に必須ですか？**

いいえ。Aspose.Slides は単体エンジンであり、サーバーやワークステーションに PowerPoint は必要ありません。

**複数スレッドからプレゼンテーションを安全に処理できますか？**

はい。異なるドキュメントを別々のスレッドで並行処理できますが、同じプレゼンテーション インスタンスを同時に使用しないようにしてください。[スレッド間での使用](/slides/ja/java/multithreading/)は避けましょう。

**マクロは保持され、PPTM/PPSM ファイルで VBA を管理できますか？**

はい。マクロ付きプレゼンテーションは[サポート](/slides/ja/java/presentation-via-vba/)されており、[VBA プロジェクトの検査と管理](https://reference.aspose.com/slides/java/com.aspose.slides/presentation/#getVbaProject--) が可能です。

**PDF や HTML を PowerPoint スライドに変換できますか？**

はい。[PDF ページまたは HTML コンテンツのインポート](/slides/ja/java/import-presentation/)で、プレゼンテーションにスライドを作成または追加できます。

**XPS エクスポートはサポートされており、出力品質や内容を制御できますか？**

はい。[XPS へのエクスポート](/slides/ja/java/convert-powerpoint-to-xps/)が利用可能で、[保存オプション](https://reference.aspose.com/slides/java/com.aspose.slides/xpsoptions/)で品質や含めるコンテンツを調整できます。

**スライドを画像に変換し、出力品質を制御できますか？**

はい。スライドは[PNG、JPEG、GIF、BMP、TIFF](/slides/ja/java/convert-powerpoint-to-png/)などにレンダリングでき、サイズや品質を細かく制御できます。

**数式 (MathML/MathText) のサポートとエクスポートはありますか？**

はい。[数式テキストの作成と編集](/slides/ja/java/powerpoint-math-equations/)および[数式のエクスポート](/slides/ja/java/exporting-math-equations/)が可能で、MathML 形式にもエクスポートできます。

**ライブラリは暗号化とパスワード (開く、設定、削除) をどのように扱いますか？**

[暗号化されたプレゼンテーションのオープン](/slides/ja/java/password-protected-presentation/)や、開くためと変更するためのパスワード設定・削除、保護状態の確認がサポートされています。
