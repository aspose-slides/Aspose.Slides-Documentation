## **サポートされているプラットフォーム**
Aspose.Slides for Java は最も一般的な開発およびデプロイ プラットフォームをサポートします。

|**機能**|**説明**|
| :- | :- |
|Desktop Applications|Aspose.Slides for Java は Windows Forms アプリケーションの開発に使用できます|
|Enterprise Web Applications|Aspose.Slides for Java を使用すると、Web アプリケーションの構築に役立ちます。PHP での Aspose.Slides for Java の使用もサポートされています。|
|Linux/Unix|Aspose.Slides for Java はプラットフォームに依存しない API で、Linux および Unix 環境で動作します。|

## **ファイル形式と変換**
Aspose.Slides for Java は Microsoft PowerPoint のほとんどのドキュメント形式をサポートし、組織で広く使用されている一般的な形式へエクスポートできます。

|**機能**|**説明**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/ja/java/microsoft-powerpoint-ppt/)|Aspose.Slides for Java はこのプレゼンテーション ドキュメント形式の最速処理を提供します。|
|[PresentationML (PPTX, XML)](/slides/ja/java/presentationml-pptx-xml/)|Aspose.Slides for Java は OOXML プレゼンテーション形式 (PresentationML または PPTX とも呼ばれる) の処理をサポートします。|
|[PPT to PPTX conversion](/slides/ja/java/ppt-to-pptx-conversion/)|Aspose.Slides for Java は PPT から PPTX への変換をサポートします。|
|[Portable Document Format (PDF)](/slides/ja/java/developer-guide/)|サポートされているファイル形式は、単一のメソッドで Adobe Portable Document Format (PDF) ドキュメントにエクスポートできます。|
|[XML Parser Specification (XPS)](/slides/ja/java/xml-parser-specification-xps/)|サポートされているすべてのファイル形式は、単一のメソッドで XML Parser Specification (XPS) ドキュメントにエクスポートできます。|
|[Tagged Image File Format (TIFF)](https://docs.aspose.com/slides/java/convert-powerpoint-to-tiff/)|Aspose.Slides for Java がサポートするプレゼンテーション ファイル形式は、Tagged Image File Format (TIFF) にもエクスポートできます。|
|[ODP to PPTX Conversion](https://docs.aspose.com/slides/java/convert-odp-to-pptx/)|Aspose.Slides for Java は OpenDocument Presentation (ODP) の読み込みと PPTX への変換をサポートします。|
|[Presentation To HTML Conversion](https://docs.aspose.com/slides/java/convert-powerpoint-to-html/)|Aspose.Slides for Java は PresentationEx を HTML 形式に変換する機能を提供します。|

## **プレゼンテーションのレンダリング**
Aspose.Slides for Java はプレゼンテーション内のスライドをさまざまなグラフィックス形式に高忠実度でレンダリングします。

|**機能**|**説明**|
| :- | :- |
|サポートされている画像形式|Aspose.Slides for Java を使用すると、プレゼンテーション スライドだけでなく、スライド上の画像も TIFF、PNG、BMP、JPEG、GIF、メタファイルなどの一般的なグラフィックス形式にレンダリングできます。|
|SVG 形式|Aspose.Slides for Java はスケーラブル ベクター グラフィックス (SVG) 形式へのエクスポート用の組み込みメソッドを提供します。|

|スライド セクションの管理|プレゼンテーション内でスライドを異なるセクションに整理する機能を提供します。|

## **書式設定機能**
Aspose.Slides for Java を使用して、プレゼンテーション ドキュメント内のスライド上のテキストやシェイプの書式設定が可能です。

|**機能**|**説明**|
| :- | :- |
|テキスト書式設定|<p>Aspose.Slides for Java では、テキストはシェイプに関連付けられたテキスト フレームを通じて管理されます。そのため、テキストはテキスト フレームに紐付く段落と部分を使用して書式設定されます。以下のテキスト要素を設定できます。</p><p>- フォントの種類。</p><p>- フォントサイズ。</p><p>- フォントの色。</p><p>- フォントの濃淡。</p><p>- 段落の配置。</p><p>- 段落の箇条書き。</p><p>- 段落の方向。</p>|
|シェイプ書式設定|<p>Aspose.Slides for Java では、スライドの基本要素はシェイプです。以下のシェイプ要素を Aspose.Slides for Java で書式設定できます。</p><p>- 位置</p><p>- サイズ</p><p>- 線</p><p>- 塗りつぶし（パターン、グラデーション、単色を含む）</p><p>- テキスト</p><p>- 画像</p>|

## **FAQ**

**サーバー/PC に Microsoft PowerPoint をインストールする必要がありますか？**

いいえ。PowerPoint は不要です。Aspose.Slides はプレゼンテーションの作成、編集、変換、レンダリングを行うスタンドアロン エンジンです。

**マルチスレッドはどのように機能しますか？処理を並列化できますか？**

異なるスレッドで別々のドキュメントを処理することは安全です。同じ [presentation](https://reference.aspose.com/slides/java/com.aspose.slides/presentation/) オブジェクトを [multiple threads](/slides/ja/java/multithreading/) で同時に使用してはいけません。

**ファイルのパスワードと暗号化はサポートされていますか？**

はい。[You can](/slides/ja/java/password-protected-presentation/) 暗号化されたプレゼンテーションを開くことができ、開くパスワードと書き込みパスワードの設定・削除、および保護状態の確認が可能です。

**Linux コンテナでフォント パッケージに注意する必要がありますか？**

はい。一般的なフォント パッケージをインストールするか、アプリケーションで明示的に [specify font directories](/slides/ja/java/custom-font/) を指定して、予期しない置換を防ぐことが推奨されます。

**評価版に制限はありますか？**

[評価モード](/slides/ja/java/licensing/) では、出力に透かしが追加され、いくつかの制限が適用されます。フル機能テスト用に [30 日間の一時ライセンス](https://purchase.aspose.com/temporary-license/) が利用可能です。

**外部形式 (PDF/HTML → PPTX) のプレゼンテーションへのインポートはサポートされていますか？**

はい。PDF ページや HTML コンテンツをプレゼンテーションに追加し、スライドに変換することができます。
