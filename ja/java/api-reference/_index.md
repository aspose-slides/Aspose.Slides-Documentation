{{% alert color="primary" %}} 
Aspose.Slides for Java は、Microsoft PowerPoint® などを使用せずに、Java アプリケーションがプレゼンテーション ドキュメントを読み書きできるクラス ライブラリです。Aspose.Slides for Java は、テキスト、シェイプ、テーブルとアニメーションの管理、スライドへのオーディオおよびビデオの追加、スライドのプレビュー、SVG や PDF 形式へのエクスポートなどを可能にします。

最新の API リファレンスは以下で確認できます
[Aspose.Slides for Java API リファレンス.](https://reference.aspose.com/slides/java)
{{% /alert %}}
