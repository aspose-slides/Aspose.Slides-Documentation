{{% alert color="primary" %}} 

Aspose.Slides for Java は、Microsoft PowerPoint® などを使用せずに、Java アプリケーションがプレゼンテーション ドキュメントの読み取りおよび書き込みを行えるクラス ライブラリです。Aspose.Slides for Java は、テキスト、図形、テーブルとアニメーションの管理、スライドへのオーディオおよびビデオの追加、スライドのプレビュー、SVG や PDF 形式へのエクスポートなどを可能にします。

最新の API リファレンスは [Aspose.Slides for Java API Reference](https://reference.aspose.com/slides/ja/java) で確認できます。

{{% /alert %}}
