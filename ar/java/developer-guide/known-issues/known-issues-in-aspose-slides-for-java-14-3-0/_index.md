Aspose.Slides for Java 14.3.0 (14.4.0) يوفر تنفيذًا جديدًا تمامًا لمعالجة ملفات PPT. هناك الكثير من التحسينات، وتحويل جزئي من PPTX إلى PPT. ولكن هناك بعض الميزات غير المنفذة:

- بعض الأشكال لها هندسة خاطئة في مستندات PPT المتسلسلة (التعليقات التوضيحية)
- ليست جميع ميزات تنسيق النص في PPTX مدعومة في تسلسل PPT
- لا توجد معلومات حول لغة النص وإعدادات التدقيق الإملائي في مستندات PPT المتسلسلة
- ليست جميع ميزات سمات PPTX مدعومة في تسلسل PPT

**هناك بعض الاختلافات مقارنةً بـ Aspose.Slides for Java 8.6.0:**

- هناك مشكلات معروفة في تسلسل OLE/ActiveX من PPT إلى PPT

**هناك بعض الاختلافات مقارنةً بـ Aspose.Slides for .NET 14.3.0:**

- دعم طباعة العروض التقديمية غير متوفر حاليًا في Aspose.Slides for Java
