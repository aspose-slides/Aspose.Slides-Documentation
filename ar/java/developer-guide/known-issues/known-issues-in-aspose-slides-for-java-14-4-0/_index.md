{{% alert color="primary" %}} 

توفر Aspose.Slides for Java الإصدار 14.4.0 قرارًا جديدًا لمعالجة مستندات PowerPoint. هناك بعض القيود والمشكلات المعروفة، والتي سيتم إزالتها في الإصدارات القادمة:

- بعض الأشكال لديها هندسة غير صحيحة في مستندات PPT المتسلسلة (القوس، السهم الدائري، التعليقات التوضيحية).
- ليست جميع ميزات تنسيق النص في PPTX مدعومة في تسلسل PPT (الجدولة، المسافات البادئة ومحدوديات تنسيق الفقرات).
- معلومات عن لغة النص وإعدادات التدقيق الإملائي غير موجودة في مستندات PPT المتسلسلة.
- ليست جميع ميزات سمة PPTX مدعومة في تسلسل PPT (فقط تسلسل تنسيقات الملء، تنسيقات الخط والخط).
- هناك مشكلات معروفة في تسلسل OLE/ActiveX إلى PPT.
- تسلسل وعرض WordArt غير مدعوم.

{{% /alert %}}
