{{% alert color="primary" %}} 

في شرائح بوربوينت، فإن استخدام الرسوم البيانية القابلة للتعديل لعرض البيانات بشكل بياني هو نشاط شائع. توفر Aspose دعمًا لإنشاء مخططات إكسل باستخدام Aspose.Cells for Java، وبدورها يمكن إدماج هذه الرسوم البيانية ككائن OLE في شريحة بوربوينت من خلال Aspose.Slides for Java. تغطي هذه المقالة الخطوات المطلوبة جنبًا إلى جنب مع التنفيذ في Java لإنشاء وإدماج مخطط MS إكسل ككائن OLE في عرض بوربوينت باستخدام Aspose.Cells for Java وAspose.Slides for Java.

{{% /alert %}} 
## **الخطوات المطلوبة**
تتبع الخطوات التالية لإنشاء وإدماج مخطط إكسل ككائن OLE في شريحة بوربوينت:# إنشاء مخطط إكسل باستخدام Aspose.Cells for Java.# تحديد حجم OLE لمخطط إكسل باستخدام Aspose.Cells for Java.# الحصول على صورة لمخطط إكسل باستخدام Aspose.Cells for Java.# إدماج مخطط إكسل ككائن OLE داخل عرض PPTX باستخدام Aspose.Slides for Java.# استبدال الصورة المعدلة للكائن بالصورة التي تم الحصول عليها في الخطوة 3 لمعالجة مشكلة تغير الكائن# حفظ العرض الناتج على القرص بتنسيق PPTX

## **الخلاصة**
{{% alert color="primary" %}} 

باستخدام Aspose.Cells for Java جنبًا إلى جنب مع Aspose.Slides for Java، يمكننا إنشاء أي من مخططات إكسل المدعومة بواسطة Aspose.Cells for Java وإدماج المخطط الذي تم إنشاؤه ككائن OLE في شريحة بوربوينت. يمكن أيضًا تحديد حجم OLE لمخطط إكسل. يمكن للمستخدمين النهائيين تعديل مخطط إكسل مثل أي كائن OLE آخر.
