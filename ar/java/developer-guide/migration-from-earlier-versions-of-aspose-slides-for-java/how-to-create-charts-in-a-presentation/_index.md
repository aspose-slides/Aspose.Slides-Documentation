تم إصدار Aspose.Slides for Java API جديد الآن، ويُتيح هذا المنتج الموحد القدرة على إنشاء مستندات PowerPoint من الصفر وتعديل المستندات الموجودة.

{{% /alert %}} 
## **دعم الشيفرة القديمة**
للاستخدام الشيفرة القديمة التي تم تطويرها باستخدام إصدارات Aspose.Slides for Java السابقة للنسخة 14.x.x، تحتاج إلى إجراء بعض التعديلات البسيطة في الشيفرة وستعمل كما كانت من قبل. جميع الفئات التي كانت موجودة في Aspose.Slides for Java القديمة ضمن مساحات الأسماء com.aspose.slides و com.aspose.slides.pptx تم دمجها الآن في مساحة اسم واحدة com.aspose.slides. يرجى إلقاء نظرة على المقتطف البرمجي البسيط التالي لإنشاء مخطط عادي من الصفر في العرض التقديمي باستخدام واجهة Aspose.Slides القديمة واتبع الخطوات التي تصف كيفية الترحيل إلى الواجهة المدمجة الجديدة.
