تم إصدار [Aspose.Slides for Java API](https://docs.aspose.com/slides/java/) جديد والآن هذا المنتج الفردي يدعم إمكانية إنشاء مستندات PowerPoint من الصفر وتعديل المستندات الحالية.

{{% /alert %}}
## **دعم الشيفرة القديمة**
لاستخدام الشيفرة القديمة التي تم تطويرها باستخدام Aspose.Slides for Java بالإصدارات التي تسبق 14.x.x، تحتاج إلى إجراء بعض التغييرات الطفيفة في الشيفرة الخاصة بك وستعمل الشيفرة كما كانت في السابق. جميع الفئات التي كانت موجودة في Aspose.Slides for Java القديمة تحت المساحتي com.aspose.slides و com.aspose.slides.pptx تم دمجها الآن في مساحة أسماء واحدة com.aspose.slides. يرجى إلقاء نظرة على مقتطف الشيفرة البسيط التالي لإنشاء رسم بياني عادي من الصفر في عرض تقديمي باستخدام واجهة برمجة التطبيقات القديمة لـ Aspose.Slides واتباع الخطوات التي تصف كيفية الانتقال إلى واجهة برمجة التطبيقات الجديدة المدمجة.
