## **إضافة نص ديناميكي**
تتبع كلتا الطريقتين الخطوات التالية:

1. إنشاء عرض تقديمي.
1. إضافة شريحة فارغة.
1. إضافة مربع نص.
1. تعيين بعض النصوص.
1. كتابة العرض التقديمي.
