{{% alert color="primary" %}} 

تم إصدار [Aspose.Slides for Java API](https://docs.aspose.com/slides/androidjava/) جديدة والآن يدعم هذا المنتج الوحيد القدرة على إنشاء مستندات PowerPoint من الصفر وتحرير المستندات الموجودة.

{{% /alert %}} 
## **الدعم للكود القديم**
لاستخدام الكود القديم الذي تم تطويره باستخدام Aspose.Slides for Java بالإصدارات السابقة لـ 13.x، تحتاج إلى إجراء بعض التعديلات الطفيفة على كودك وسيعمل الكود كما كان من قبل. جميع الفئات التي كانت موجودة في Aspose.Slides for Java القديمة تحت أسماء النطاق Aspose.Slide و Aspose.Slides.Pptx قد تم دمجها الآن في اسم النطاق الواحد Aspose.Slides. يرجى إلقاء نظرة على مقتطف كود بسيط التالي لإضافة الرأس والتذييل في العرض التقديمي في API Aspose.Slides القديم واتبع الخطوات التي تصف كيفية الانتقال إلى API المدمجة الجديدة.
