{{% alert color="primary" %}} 

تم إصدار [Aspose.Slides for Java API](https://docs.aspose.com/slides/androidjava/) جديدة والآن يدعم هذا المنتج الواحد القدرة على إنشاء مستندات PowerPoint من الصفر وتحرير المستندات الموجودة.

{{% /alert %}} 
## **دعم الكود القديم**
لاستخدام الكود القديم الذي تم تطويره مع إصدارات Aspose.Slides for Java السابقة لـ 14.x.x، تحتاج إلى إجراء بعض التغييرات الطفيفة في الكود الخاص بك وسيعمل كما في السابق. جميع الفئات التي كانت موجودة في Aspose.Slides for Java القديمة تحت أسماء المساحات com.aspose.slides و com.aspose.slides.pptx قد تم دمجها الآن في مساحة اسم واحدة com.aspose.slides. يرجى إلقاء نظرة على الكود البسيط التالي لإنشاء رسم بياني عادي من الصفر في عرض تقديمي باستخدام Aspose.Slides API القديم واتباع الخطوات التي تصف كيفية الانتقال إلى واجهة برمجة التطبيقات المدمجة الجديدة.
