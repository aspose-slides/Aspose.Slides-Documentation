## **الدعم للكود القديم**
لاستخدام الكود القديم الذي تم تطويره باستخدام إصدارات Aspose.Slides for Java قبل 13.x، تحتاج إلى إجراء بعض التغييرات الطفيفة في كودك وسيعمل الكود كما كان من قبل. جميع الفئات التي كانت موجودة في Aspose.Slides for Java القديمة تحت مساحات أسماء Aspose.Slide و Aspose.Slides.Pptx تم دمجها الآن في مساحة اسم Aspose.Slides واحدة. يرجى إلقاء نظرة على جزء الكود البسيط التالي لإنشاء وثيقة عرض تقديمي Hello World في API Aspose.Slides القديم واتباع الخطوات التي تصف كيفية الانتقال إلى API المدمجة الجديدة.
