## **نظرة عامة**

يمكن لـ Aspose.Slides for Android عبر Java تحميل عروض PowerPoint وOpenDocument وحفظها أو عرضها بصيغ متعددة دون الحاجة إلى Microsoft PowerPoint أو OpenOffice أو LibreOffice. يمكنك تحويل ملفات PPT القديمة إلى PPTX الحديثة، وتصدير العروض إلى مستندات ذات تخطيط ثابت مثل PDF وXPS، ونشر الشرائح كـ HTML، أو عرض الشرائح كملفات صور للمعاينات، والصور المصغرة، والأرشفة.

غالبًا ما يستخدم تحويل المستندات نفس سير العمل العام: تحميل الملف المصدر، اختيار صيغة الإخراج المطلوبة، وتطبيق الخيارات الخاصة بالصيغ عند الحاجة. بالنسبة لصيغ الصور، يتم عرض كل شريحة بشكل منفصل ثم حفظها كصورة نقطية أو متجهة. المقالات المخصصة في الروابط أدناه توفر تفاصيل التنفيذ لكل حالة.

## **تحويل PowerPoint وOpenDocument**

يدعم Aspose.Slides for Android عبر Java التحويل من صيغ العروض الشائعة مثل PPT وPPTX وPPS وPPSX وPOT وPOTX وODP. تُستخدم نفس واجهة برمجة التطبيقات للتحويل لملفات PowerPoint وOpenDocument، لذا يمكن عادةً تطبيق سير عمل يحفظ ملف PPTX ك PDF على ملف ODP بمجرد تغيير ملف الإدخال فقط.

عند تحويل ملفات ODP، تذكر أن تطبيقات PowerPoint وOpenDocument لا تدعم كل ميزات التخطيط والتنسيق بنفس الطريقة تمامًا. إذا تم إنشاء ملف ODP في LibreOffice أو OpenOffice Impress، راجع النتيجة واستخدم الخيارات الموضحة في [تحويل عروض OpenDocument](/slides/ar/androidjava/convert-openoffice-odp/) عندما تحتاج إلى إرشادات خاصة بالصيغ.

## **تحويل PPT إلى PPTX**

PPT هو تنسيق PowerPoint الثنائي الأقدم، بينما PPTX هو تنسيق Office Open XML الحديث. يدعم Aspose.Slides for Android عبر Java تحويل PPT إلى PPTX بدقة عالية مع الحفاظ على هياكل العرض المعقدة مثل القوالب الرئيسية، التخطيطات، الشرائح، المخططات، الأشكال المجمعة، العناصر النائبة، إطارات النص، القوام، وتعبئة الصور.

للتفاصيل، راجع [تحويل PPT إلى PPTX](/slides/ar/androidjava/convert-ppt-to-pptx/) و[PPT vs PPTX](/slides/ar/androidjava/ppt-vs-pptx/).

## **تصدير بتخطيط ثابت**

تعد صيغ PDF وXPS وTIFF مفيدة عندما يجب أن يبدو الإخراج متطابقًا عبر الأجهزة ولا ينبغي تحريره كعرض تقديمي. توضح المقالات المخصصة لPDF وXPS وTIFF كيفية التحكم في الامتثال، الشرائح المخفية، الملاحظات، جودة الصورة، الضغط، تنسيق البكسل، وحجم الإخراج.

## **تصدير HTML والصور**

يعد تصدير HTML وHTML5 مفيدًا للعرض عبر المتصفح، النشر على الويب، والمشاركة الخفيفة. تصدير الصور مفيد عندما يجب أن تصبح كل شريحة معاينة منفصلة أو صورة مصغرة أو أصل نقطي. استخدم مقالات PNG وJPG وSVG للحصول على إرشادات العرض الخاصة بكل صيغة.

## **الأسئلة الشائعة**

**هل أحتاج إلى Microsoft PowerPoint لتحويل العروض؟**

لا. Aspose.Slides for Android عبر Java هو مكتبة مستقلة ولا يتطلب Microsoft PowerPoint أو أتمتة Office.

**هل يمكنني تحويل العديد من العروض دفعة واحدة؟**

نعم. قم بتحميل كل عرض، احفظه بالتنسيق المطلوب، وتخلص من كائن العرض بعد المعالجة. للمعالجة المتوازية، استخدم نسخًا منفصلة من العروض واتبع إرشادات [multithreading](/slides/ar/androidjava/multithreading/).

**هل يمكنني تصدير شرائح مختارة فقط؟**

نعم. تسمح العديد من طرق التصدير بتمرير مؤشرات الشرائح أو عرض شرائح منفردة، حسب تنسيق الإخراج. راجع المقالة المخصصة للتنسيق المستهدف.

**هل يمكنني تضمين الشرائح المخفية عند التصدير إلى PDF أو XPS؟**

نعم. استخدم إعدادات تصدير الشرائح المخفية الموضحة في مقالات التحويل لـ [PDF](/slides/ar/androidjava/convert-powerpoint-to-pdf/) و[XPS](/slides/ar/androidjava/convert-powerpoint-to-xps/).

**هل يمكنني إنشاء مخرجات PDF/A؟**

نعم. تتوفر إعدادات الامتثال لـ PDF عند تصدير PDF. راجع [تحويل PowerPoint إلى PDF](/slides/ar/androidjava/convert-powerpoint-to-pdf/) للمزيد من التفاصيل.

**كيف يتم التعامل مع الخطوط أثناء التحويل؟**

يمكن لـ Aspose.Slides استخدام الخطوط المضمَّنة، والبدائل الخطية، وإعدادات استبدال الخطوط. راجع [Embedded Font](/slides/ar/androidjava/embedded-font/)، [Fallback Font](/slides/ar/androidjava/fallback-font/)، و[Font Substitution](/slides/ar/androidjava/font-substitution/).
