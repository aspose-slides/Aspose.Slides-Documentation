## **مرحبًا بكم في Aspose.Slides لنظام Android عبر Java!**

Aspose.Slides لنظام Android عبر Java هو واجهة برمجة تطبيقات لإدارة Microsoft PowerPoint® تمكّن تطبيقات Java من قراءة وكتابة مستندات PowerPoint® دون الحاجة إلى Microsoft PowerPoint®.

Aspose.Slides لنظام Android عبر Java هو المكوّن الأول والوحيد الذي يوفر الوظائف المطلوبة لإدارة مستندات PowerPoint®.

Aspose.Slides لنظام Android عبر Java يقدم العديد من الميزات الرئيسية مثل إدارة النصوص والأشكال والجداول والرسوم المتحركة، إضافة الصوت والفيديو إلى الشرائح، معاينة الشرائح، تصدير الشرائح إلى SVG وPDF وغيرها من الصيغ.

## **وصف المنتج**
Aspose.Slides لنظام Android عبر Java لا يتيح لك فقط فتح ملفات PowerPoint من مصادر مختلفة، بل يسمح لك أيضًا بحفظ عروضك بطرق متعددة. على سبيل المثال، يمكنك حفظ عروضك بصيغة PPT؛ ويمكنك أيضًا حفظ الشرائح كصور.

Aspose.Slides لنظام Android عبر Java يقدم لك تقريبًا جميع الميزات التي قد تجدها أو لا تجدها في Microsoft PowerPoint. بالنظر إلى تنوع Aspose.Slides لنظام Android عبر Java، بالإضافة إلى الجداول وأنواع مختلفة من الأشكال، يمكنك أيضًا إضافة أنواع مختلفة من الإطارات—إطارات نصية، صوتية، وفيديوية—إلى شرائحك.

Aspose.Slides لنظام Android عبر Java يوفر أيضًا نماذج تجريبية كاملة وأمثلة عملية للمطورين للحصول على فهم أعمق للواجهة البرمجية.

### **ميزات العرض التقديمي**
- فتح عروض Microsoft PowerPoint من مصادر مختلفة.
- حفظ العروض بصيغ Microsoft PowerPoint المختلفة.
- إنشاء عروض من الصفر.
- تحويل العروض إلى مستندات PDF دون استخدام أي مكوّن آخر.
- القدرة على قراءة Open Document Presentation (ODP).
- القدرة على تحويل PPT إلى PPTX أو العكس.
- قراءة أو تعديل خصائص مستند العرض.
- تصدير العروض إلى PDF
- تصدير العروض إلى TIFF
- تصدير العروض إلى XPS
- تصدير العروض إلى SWF
- تصدير العروض إلى ملاحظات PDF / TIFF
- تصدير العروض إلى HTML

### **ميزات الشريحة**
- إنشاء شرائح جديدة من الصفر وإضافتها إلى العروض.
- الوصول إلى أي شريحة عبر موقعها في العرض.
- نسخ أو استنساخ الشرائح إلى نفس العرض أو إلى عرض آخر.
- تحرير وحذف الشرائح.
- نسخ الماسترز مع الشرائح تلقائيًا.
- معاينة الشرائح.
- الحصول على اسم الشريحة أو تعيينه.
- حفظ الشرائح كصور.
- تصدير الشرائح إلى صيغة SVG.
- العمل مع عدة ماسترز في العروض.
- ملء خلفية الشرائح بأي نمط مدعوم.
- إضافة ملاحظات جديدة إلى الشريحة أو إدارة الملاحظات الموجودة.
- تعيين ماسترز وأنماط جديدة للشرائح بشكل ديناميكي.
- إضافة أو تعديل انتقالات عرض الشرائح.
- إضافة انتقالات Morph

### **ميزات الرسومات والوسائط المتعددة**
- إنشاء أو إدارة الأشكال مثل الخطوط، المستطيلات، الإهليلجات، والبوليلينات.
- إدارة أنماط الخطوط في الأشكال.
- الحصول على النص البديل لشكل.
- تدوير أو عكس الأشكال.
- ملء الأشكال بأنماط مختلفة، مثل الصلب، التدرج، النمط، النسيج والصورة.
- استخدام أي من 48 نمطًا معرفًا مسبقًا و24 نسيجًا.
- إرفاق روابط بالأشكال.
- قراءة، تعديل، وإضافة تأثيرات حركة جديدة إلى الشرائح والأشكال.
- إضافة صور جديدة إلى العروض وإدارة الصور الموجودة.
- إضافة نقاط صورة جديدة إلى العروض وإدارة النقاط الموجودة.
- إضافة إطارات صوتية جديدة إلى العروض وإدارة الإطارات الموجودة.
- إضافة إطارات فيديو جديدة إلى العروض وإدارة الإطارات الموجودة.
