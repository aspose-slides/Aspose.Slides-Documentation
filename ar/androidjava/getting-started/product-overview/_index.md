**مرحبًا بكم في Aspose.Slides لنظام Android عبر Java!**

Aspose.Slides لنظام Android عبر Java هو واجهة برمجة تطبيقات لإدارة ملفات Microsoft PowerPoint® تتيح لتطبيقات Java قراءة وكتابة مستندات PowerPoint® بدون الحاجة إلى Microsoft PowerPoint®.

Aspose.Slides لنظام Android عبر Java هو المكوّن الأول والوحيد الذي يوفر الوظائف المطلوبة لإدارة مستندات PowerPoint®.

Aspose.Slides لنظام Android عبر Java يقدم العديد من الميزات الأساسية مثل إدارة النصوص والأشكال والجدول والرسوم المتحركة، إضافة الصوت والفيديو إلى الشرائح، معاينة الشرائح، وتصدير الشرائح إلى SVG وPDF وصيغ أخرى.

## **وصف المنتج**
Aspose.Slides لنظام Android عبر Java لا يتيح لك فقط فتح ملفات PowerPoint من مصادر مختلفة، بل يسمح لك أيضًا بحفظ عروضك التقديمية بطرق متعددة. على سبيل المثال، يمكنك حفظ عروضك بصيغة PPT؛ كما يمكنك حفظ الشرائح كصور.

Aspose.Slides لنظام Android عبر Java يقدم لك تقريبًا جميع الميزات التي قد تجدها أو لا تجدها في Microsoft PowerPoint. نظرًا لتعدد استخدامات Aspose.Slides لنظام Android عبر Java، إلى جانب الجداول وأنواع الأشكال المختلفة، يمكنك إضافة إطارات متنوعة—نصية، صوتية، وفيديوية—إلى الشرائح.

Aspose.Slides لنظام Android عبر Java يوفر أيضًا عروض توضيحية كاملة وأمثلة عملية للمطورين لفهم API بشكل أفضل.

### **ميزات العرض التقديمي**
- فتح عروض Microsoft PowerPoint من مصادر مختلفة.
- حفظ العروض بصيغ Microsoft PowerPoint المختلفة.
- إنشاء عروض تقديمية من الصفر.
- تحويل العروض إلى مستندات PDF دون الحاجة إلى أي مكوّن آخر.
- القدرة على قراءة Open Document Presentation (ODP).
- القدرة على تحويل PPT إلى PPTX أو العكس.
- قراءة أو تعديل خصائص وثيقة العرض.
- تصدير العروض إلى PDF
- تصدير العروض إلى TIFF
- تصدير العروض إلى XPS
- تصدير العروض إلى SWF
- تصدير العروض إلى ملاحظات PDF / Tiff
- تصدير العروض إلى HTML

### **ميزات الشريحة**
- إنشاء شرائح جديدة من الصفر وإضافتها إلى العروض.
- الوصول إلى أي شريحة عبر موقعها في العرض.
- نسخ أو استنساخ الشرائح إلى نفس العرض أو عرض آخر.
- تحرير وحذف الشرائح.
- نسخ القوالب مع الشرائح تلقائيًا.
- معاينة الشرائح.
- الحصول على اسم الشريحة أو تعيينه.
- حفظ الشرائح كصور.
- تصدير الشرائح إلى صيغة SVG.
- العمل مع عدة قوالب في العروض.
- تعبئة خلفية الشرائح بأي نمط مدعوم.
- إضافة ملاحظات جديدة للشرائح أو إدارة الملاحظات الموجودة.
- تعيين قوالب شرائح وأنماط جديدة ديناميكيًا.
- إضافة أو تعديل انتقال عرض الشرائح.
- إضافة انتقالات Morph

### **ميزات الرسومات والوسائط المتعددة**
- إنشاء أو إدارة أشكال مثل الخطوط والمستطيلات والدوائر والمتعدد الخطوط.
- إدارة أنماط الخط في الأشكال.
- الحصول على النص البديل للشكل.
- دوران أو عكس الأشكال.
- تعبئة الأشكال بأنماط مختلفة، مثل صلبة، تدرج لوني، نمط، نسيج وصورة.
- استخدام أي من 48 نمطًا مسبقًا و24 نسيجًا.
- ربط روابط بالأشكال.
- قراءة وتعديل وإضافة تأثيرات رسوم متحركة جديدة إلى الشرائح والأشكال.
- إضافة صور جديدة إلى العروض وإدارة الصور الموجودة.
- إضافة نقاط صورة جديدة إلى العروض وإدارة النقاط الموجودة.
- إضافة إطارات صوتية جديدة إلى العروض وإدارة الإطارات الموجودة.
- إضافة إطارات فيديو جديدة إلى العروض وإدارة الإطارات الموجودة.

### **ميزات الجدول**
- إنشاء جداول من الصفر وإضافتها إلى الشرائح.
- تغيير حجم الجداول وتحريكها.
- الوصول إلى أي خلية في الجدول.
- تغيير تنسيق وحدود أي خلية.
- تعديل عرض الأعمدة والصفوف.
- إضافة وحذف الأعمدة والصفوف.
- تعيين عرض الحدود ولونها للجدول بأكمله.
- تطبيق أنماط مسبقة على الجداول

### **ميزات النص والفقرة**
- الحصول على النصوص وتعيينها في عناصر النص وإطارات النص.
- معرفة نوع العنصر النائب.
- تغيير تنسيق النص في إطار النص.
- إضافة خطوط جديدة أو إدارة الخطوط المستخدمة بالفعل في العرض.
- تغيير النصوص وتنسيق النص.
- إنشاء أو إدارة روابط إلى النص.
- دعم علامات التبويب والـ tabs الفعّالة.
- إضافة إطارات نصية جديدة إلى أي نوع من الأشكال تقريبًا.
- الوصول إلى هوامش إعدادات التفاف النص في إطارات النص وتعديلها.
- الوصول إلى إزاحة الفقرات وتعديلها.

### **ميزات متقدمة**
- الوصول إلى كائنات OLE المدمجة في العروض.
- قفل العروض.
- إضافة علامات مائية إلى العروض.
- تطبيق رسوم متحركة على الأشكال داخل الشرائح.

## **الأسئلة المتكررة**

**هل يدعم مستويات التوافق مع PDF لأغراض الأرشفة وإمكانية الوصول (PDF/A و PDF/UA)؟**

نعم. يمكنك الحفظ إلى PDF بصيغ PDF/A-2a/2b/2u و PDF/A-3a/3b، بالإضافة إلى PDF/UA عن طريق تكوين [خيارات تصدير PDF](https://reference.aspose.com/slides/androidjava/com.aspose.slides/pdfoptions/).

**هل يوجد آلية لاستبدال الخطوط ودعم الخطوط المخصصة لضمان العرض الصحيح؟**

نعم. المكتبة تسمح لك [بتحميل خطوط مخصصة](/slides/ar/androidjava/custom-font/) و[بتعريف قواعد احتياطية](/slides/ar/androidjava/fallback-font/) بحيث يتم استبدال الحروف المفقودة بثقة أثناء التحويل والعرض.

**هل يمكنني اكتشاف ما إذا كان الملف محميًا بكلمة مرور دون فتحه بالكامل؟**

نعم. يمكنك [فحص عرض تقديمي](/slides/ar/androidjava/examine-presentation/) لتحديد ما إذا كان يتطلب كلمة مرور قبل تحميل المستند بالكامل.

**هل يلزم وجود Microsoft PowerPoint للمعالجة والتحويلات؟**

لا. Aspose.Slides هو محرك مستقل؛ لا يلزم وجود PowerPoint على الخادم أو محطة العمل.

**هل من الآمن معالجة العروض من عدة خيوط (threads)؟**

نعم، يمكنك معالجة مستندات مختلفة بالتوازي في خيوط منفصلة؛ فقط تجنب استخدام نفس كائن العرض التقديمي في الوقت نفسه عبر الخيوط [عبر الخيوط](/slides/ar/androidjava/multithreading/).

**هل يتم الحفاظ على الماكروات ويمكنني إدارة VBA في ملفات PPTM/PPSM؟**

نعم. العروض التي تحتوي على ماكروات [مدعومة](/slides/ar/androidjava/presentation-via-vba/)، ويمكنك [فحص وإدارة مشاريع VBA](https://reference.aspose.com/slides/androidjava/com.aspose.slides/presentation/#getVbaProject--) في تلك الملفات.

**هل يمكنني تحويل PDF أو HTML مرة أخرى إلى شرائح PowerPoint؟**

نعم. يمكنك [استيراد صفحات PDF أو محتوى HTML](/slides/ar/androidjava/import-presentation/) لإنشاء أو تعبئة شرائح في عرض تقديمي.

**هل تدعم عملية تصدير XPS، وهل يمكنني التحكم في جودة ومحتوى مخرجات XPS؟**

نعم. [التصدير إلى XPS](/slides/ar/androidjava/convert-powerpoint-to-xps/) متاح، و[خيارات الحفظ](https://reference.aspose.com/slides/androidjava/com.aspose.slides/xpsoptions/) تتيح لك ضبط جودة المخرج والمحتوى المتضمن.

**هل يمكنني تحويل الشرائح إلى صور والتحكم في جودة الإخراج؟**

نعم. يمكن تصيير الشرائح إلى PNG أو JPEG أو GIF أو BMP أو TIFF [/slides/androidjava/convert-powerpoint-to-png/] مع تحكم دقيق في الحجم والجودة.

**هل تدعم الصيغ الرياضية (MathML/MathText) ويمكن تصديرها؟**

نعم. يمكنك [إنشاء وتحرير نص رياضي](/slides/ar/androidjava/powerpoint-math-equations/) و[تصدير المعادلات](/slides/ar/androidjava/exporting-math-equations/)، بما في ذلك إلى MathML.

**كيف تتعامل المكتبة مع التشفير وكلمات المرور (فتح، تعيين، إزالة)؟**

[تدعم](/slides/ar/androidjava/password-protected-presentation/) فتح العروض المشفّرة، تعيين أو إزالة كلمات المرور للفتح والتعديل، وفحص حالة حماية الملف.
