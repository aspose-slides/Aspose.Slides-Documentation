{{% alert color="primary" %}} 

Aspose.Slides for C++ هي مكتبة فئات تمكّن تطبيقات C++ من قراءة وكتابة مستندات العروض التقديمية دون الحاجة إلى استخدام Microsoft PowerPoint® أو غيره. تتيح Aspose.Slides for C++ إدارة النصوص، الأشكال، الجداول والرسوم المتحركة، إضافة الصوت والفيديو إلى الشرائح، معاينة الشرائح، تصدير الشرائح إلى صيغة SVG وPDF والمزيد.

يمكن العثور على أحدث مرجع برمجي على [دليل API ل Aspose.Slides for C++](https://reference.aspose.com/slides/ar/cpp/).
