## **تحويل PPT إلى PPTX**

لتحويل PPT إلى PPTX، ما عليك سوى تمرير اسم الملف وصيغة الحفظ إلى طريقة **Save** لفئة **Presentation**. العينة البرمجية بلغة Python أدناه تحول عرضًا من PPT إلى PPTX باستخدام الخيارات الافتراضية.

```python
import aspose.slides as slides

# Instantiate a Presentation object that represents a PPT file
pres = slides.Presentation("PPTtoPPTX.ppt")

# Save the presentation in PPTX format
pres.save("PPTtoPPTX_out.pptx", slides.export.SaveFormat.PPTX)
```

اقرأ المزيد حول صيغ العرض **PPT مقابل PPTX** وكيف يدعم **Aspose.Slides تحويل PPT إلى PPTX**.
