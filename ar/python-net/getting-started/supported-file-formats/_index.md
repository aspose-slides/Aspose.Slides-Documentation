## **الأسئلة الشائعة**

**هل يمكنني حفظ العروض التقديمية إلى PDF تت符合 معايير الأرشفة وإمكانية الوصول (PDF/A و PDF/UA)؟**

نعم. يدعم Aspose.Slides for Python via .NET تصدير إلى PDF مع مستويات التوافق مثل PDF/A-2a و PDF/A-2b و PDF/A-2u و PDF/A-3a و PDF/A-3b، بالإضافة إلى PDF/UA من خلال إعداد [compliance]((https://reference.aspose.com/slides/python-net/aspose.slides.export/pdfoptions/compliance/)) في [PDF export options]((https://reference.aspose.com/slides/python-net/aspose.slides.export/pdfoptions/)).

**هل يدعم المكتبة تضمين الخطوط عند تصدير إلى PDF، مع تحكم دقيق في ما يتم تضمينه؟**

نعم. يمكنك التحكم في ما إذا كانت الخطوط مُضمَّنة بالكامل أو كجزء جزئي (الأحرف المستخدمة فقط)، وتحديد كيفية معالجة الخطوط النظامية الشائعة، وتكوين سلوك النص ASCII عبر [PDF export options]((https://reference.aspose.com/slides/python-net/aspose.slides.export/pdfoptions/)).

**هل يمكنني اكتشاف ما إذا كان الملف محميًا بكلمة مرور قبل تحميله فعليًا؟**

نعم. باستخدام [factory-based inspection API]((https://reference.aspose.com/slides/python-net/aspose.slides/presentationfactory/))، يمكنك الاستعلام عن ملف العرض لتحديد ما إذا كان محميًا بكلمة مرور دون فتحه بالكامل.

**هل توجد آليات احتياطي للخطوط ودعم للخطوط المخصصة؟**

نعم. تدعم المكتبة [loading](/slides/ar/python-net/custom-font/) و [embedding](/slides/ar/python-net/embedded-font/) الخطوط المخصصة وتوفر قواعد [fallback](/slides/ar/python-net/fallback-font/) للخطوط لتجنب فقد الأحرف أثناء العرض والتحويل.

**هل يمكنني تصدير الشرائح إلى XPS، وهل توجد خيارات لضبط مخرج XPS؟**

نعم. يدعم [Export to XPS](/slides/ar/python-net/convert-powerpoint-to-xps/) ويمكنك تعديل [save options]((https://reference.aspose.com/slides/python-net/aspose.slides.export/xpsoptions/)) ذات الصلة للتحكم في جودة ومحتوى مستند XPS.
