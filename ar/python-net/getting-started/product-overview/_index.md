## **مرحبًا بكم في Aspose.Slides لـ Python عبر .NET!**

Aspose.Slides هي واجهة برمجة تطبيقات لإدارة ملفات Microsoft PowerPoint® تسمح للتطبيقات بقراءة وكتابة وإنشاء وتعديل عروض PowerPoint بدون استخدام Microsoft PowerPoint®. تدعم المكتبة العمل مع العروض، الشرائح، الأشكال، الجداول، النصوص، الرسوم المتحركة، الصوت، الفيديو، وعناصر العرض الأخرى.

باستخدام Aspose.Slides، يمكن للمطورين فتح العروض من مصادر مختلفة، حفظها بصيغ PowerPoint المتنوعة، تصدير العروض إلى صيغ مثل PDF و TIFF و XPS و SWF و HTML و SVG، وحفظ الشرائح كصور. كما توفر الواجهة ميزات لإدارة خصائص المستند، تطبيق الحماية، معاينة الشرائح، تحرير محتوى الشريحة، التعامل مع ملاحظات الشريحة، ومعالجة عناصر عرض متقدمة مثل كائنات OLE المضمنة.

## **لماذا يجب عليك استخدام Aspose.Slides لـ Python عبر .NET**

- Aspose.Slides لـ Python عبر .NET هو المكوّن الأول والوحيد الذي يوفر الوظائف المطلوبة لإدارة مستندات PowerPoint®.
- Aspose.Slides لـ Python عبر .NET يقدم العديد من الميزات الرئيسية مثل إدارة النصوص، الأشكال، الجداول والرسوم المتحركة، إضافة الصوت والفيديو إلى الشرائح، معاينة الشرائح، تصدير الشرائح إلى SVG و PDF وغيرها من الصيغ، والعديد من القدرات الأخرى.
- Aspose.Slides لـ Python عبر .NET لا يتيح لك فقط فتح ملفات PowerPoint من مصادر مختلفة بل يسمح لك أيضًا بحفظ عروضك بطرق متعددة. على سبيل المثال، يمكنك حفظ العروض بصيغة PPT؛ كما يمكنك حفظ الشرائح كصور.
- Aspose.Slides لـ Python عبر .NET يوفر لك تقريبًا جميع الميزات التي قد تجدها أو لا تجدها في Microsoft PowerPoint. بفضل مرونة Aspose.Slides لـ Python عبر .NET، بالإضافة إلى الجداول وأنواع الأشكال المختلفة، يمكنك إضافة أنواع مختلفة من الإطارات—نصية، صوتية، وفيديوية—إلى الشرائح.
- تم تطوير Aspose.Slides لـ Python عبر .NET باستخدام Managed Python. يمكنك استخدام Aspose.Slides لـ Python عبر .NET مع أي لغة .NET مثل Python و VB.NET و J# وغيرها. يمكن أيضًا استخدامه مع أي نوع من التطبيقات—ويب أو تطبيقات Windows.
- Aspose.Slides لـ Python عبر .NET يوفر نماذج تجريبية كاملة وأمثلة عملية لمساعدة المطورين على فهم الواجهة بشكل أفضل.

### **ميزات العرض**
- فتح عروض Microsoft PowerPoint من مصادر مختلفة.
- حفظ العروض بصيغ Microsoft PowerPoint المختلفة.
- إنشاء عروض من الصفر.
- تحويل العروض إلى مستندات PDF دون الحاجة إلى أي مكوّن آخر.
- قراءة أو تعديل خصائص مستند العرض.
- تطبيق الحماية على العرض.
- تصدير العرض إلى PDF.
- تصدير العرض إلى TIFF.
- تصدير العرض إلى XPS.
- تصدير العرض إلى SWF.
- تصدير ملاحظات العرض إلى PDF / TIFF.
- تصدير العرض إلى HTML.

### **ميزات الشرائح**
- إنشاء شرائح جديدة من الصفر وإضافتها إلى العروض.
- الوصول إلى أي شريحة من خلال موقعها في العرض.
- نسخ أو استنساخ الشرائح إلى نفس العرض أو إلى عرض آخر.
- تحرير وحذف الشرائح.
- نسخ القوالب مع الشرائح تلقائيًا.
- معاينة الشرائح.
- الحصول على اسم الشريحة أو تعيينه.
- حفظ الشرائح كصور.
- تصدير الشرائح إلى صيغ SVG.
- العمل مع عدة قوالب في العروض.
- ملء خلفية الشرائح بأي نمط مدعوم.
- إضافة أو إدارة ملاحظات الشرائح.
- تعيين قوالب شرائح جديدة وأنماطها بشكل ديناميكي.
- إضافة أو تعديل انتقالات عرض الشرائح.
- إضافة انتقالات Morph.

### **ميزات الرسوميات والوسائط المتعددة**
- إنشاء أو إدارة الأشكال مثل الخطوط، المستطيلات، القطوع، والمتعددات.
- إدارة أنماط الخطوط في الأشكال.
- الحصول على النص البديل للشكل.
- تدوير أو عكس الأشكال.
- ملء الأشكال بأنماط مختلفة—مثل الصلب، التدرج، النمط، الملمس، والصورة.
- استخدام أي من 48 نمطًا محددًا مسبقًا و24 ملمسًا.
- ربط الأشكال بروابط.
- قراءة، تعديل، وإضافة تأثيرات رسوم متحركة جديدة إلى الشرائح والأشكال.
- إضافة صور جديدة إلى العروض وإدارة الصور الموجودة.
- إضافة نقاط صور جديدة إلى العروض وإدارة النقاط الموجودة.
- إضافة إطارات صوتية جديدة إلى العروض وإدارة الإطارات الموجودة.
- إضافة إطارات فيديو جديدة إلى العروض وإدارة الإطارات الموجودة.

### **ميزات الجداول**
- إنشاء جداول من الصفر وإضافتها إلى الشرائح.
- تغيير حجم الجداول وتحريكها.
- الوصول إلى أي خلية في الجدول.
- تغيير تنسيق وحدود أي خلية.
- تغيير حجم الأعمدة والصفوف.
- إضافة وحذف الأعمدة والصفوف.
- تعيين عرض الحدود ولونها للجدول بالكامل.

### **ميزات النص والفقرات**
- الحصول على النصوص وتعيينها في عناصر النص وإطارات النص.
- الحصول على نوع العنصر النائب.
- تغيير تنسيق النص في إطار النص.
- إضافة خطوط جديدة أو إدارة الخطوط المستخدمة بالفعل في العرض.
- تغيير النصوص وتنسيق النص.
- إنشاء أو إدارة الروابط داخل النص.
- دعم علامات التبويب والعلامات الفعّالة.
- إضافة إطارات نص جديدة إلى تقريبًا أي نوع من الأشكال.
- الوصول إلى تعديل الهوامش وإعدادات التفاف النص في إطارات النص.
- الوصول إلى تعديل إزاحة الفقرات.

### **ميزات متقدمة**
- الوصول إلى كائنات OLE المضمنة في العروض.
- قفل العروض.

- القناة الرئيسية للدعم هي [منتديات Aspose](https://forum.aspose.com/). انشر سؤالك في [منتدى Aspose.Slides](https://forum.aspose.com/c/slides/ar/11)—وسيتم الرد عليه خلال بضع ساعات. فريق دعم Aspose.Slides يجيب مباشرة على الأسئلة المنشورة في المنتدى.

- عند انتظار الرد في المنتديات، يرجى التحلي بالصبر وإجراء تقييمات لاختلافات المناطق الزمنية.
