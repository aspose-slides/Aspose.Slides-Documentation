{{% alert color="primary" %}} 

Aspose.Slides لـ Node.js عبر .NET هي مكتبة فصول تمكّن تطبيقات Node.js من قراءة وكتابة مستندات العروض التقديمية دون استخدام Microsoft PowerPoint® أو غيره. تتيح لك Aspose.Slides لـ Node.js عبر .NET إدارة النصوص والأشكال والجداول والرسوم المتحركة، وإضافة الصوت والفيديو إلى الشرائح، ومعاينة الشرائح، وتصدير الشرائح إلى تنسيق SVG و PDF، والكثير غير ذلك.

تستخدم Aspose.Slides لـ Node.js عبر .NET [نفس مرجع واجهة برمجة التطبيقات كما في Aspose.Slides .NET](https://reference.aspose.com/slides/nodejs-net/).
