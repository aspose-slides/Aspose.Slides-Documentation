Aspose.Slides لـ Node.js عبر .NET لا تتطلب أي منتج طرف ثالث مثل Microsoft PowerPoint ليتم تثبيته. Aspose.Slides نفسها هي محرك لإنشاء وتعديل وتحويل وعرض المستندات بتنسيقات مختلفة، بما في ذلك تنسيقات عرض Microsoft PowerPoint.

## أنظمة التشغيل المدعومة

Aspose.Slides لـ .NET تدعم أي نظام تشغيل بنظام 32 بت أو 64 بت يعمل على تشغيل وقت تشغيل .NET بما في ذلك، ولكن لا تقتصر على:

### **ويندوز**
- خادم Microsoft Windows 2003 ( x64, x86)
- خادم Microsoft Windows 2008 ( x64, x86)
- خادم Microsoft Windows 2012 ( x64, x86)
- خادم Microsoft Windows 2012 R2 ( x64, x86)
- خادم Microsoft Windows 2016 ( x64, x86)
- خادم Microsoft Windows 2019 ( x64, x86)
- Microsoft Windows Vista ( x64, x86)
- Microsoft Windows XP ( x64, x86)
- Microsoft Windows 7 ( x64, x86)
- Microsoft Windows 8, 8.1 ( x64, x86)
- Microsoft Windows 10 ( x64, x86)
- Microsoft Windows 11 ( x64, x86)

### **لينوكس**
- لينوكس (أوبونتو، سنتوس وغيرها)

### **ماك**
