## **حول TIFF**

تنسيق [تنسيق ملف الصورة الموسوم](https://en.wikipedia.org/wiki/TIFF) المعروف بتخزين صور متعددة في مستند واحد تم إنشاؤه أصلاً بواسطة Aldus. يُدعم هذا التنسيق على نطاق واسع من قبل تطبيقات المسح والفاكس وتطبيقات معالجة الصور الأخرى.

## **TIFF في Aspose.Slides for PHP عبر Java**

يمكن تحويل أي مستند يمكن تحميله في Aspose.Slides for Java إلى مستند TIFF مباشرةً بواسطة Aspose.Slides for PHP عبر Java، مما يلغي الحاجة إلى أي مكوّن طرف ثالث. بالإضافة إلى ذلك، يمكنك اختيارياً تحديد حجم الصور في مستند TIFF الناتج. يمكنك العثور على معلومات حول تصدير مستندات العروض التقديمية إلى مستندات TIFF عبر Aspose.Slides for PHP عبر Java في [هذا الموضوع](/slides/ar/php-java/convert-powerpoint-to-tiff/).
