{{% alert color="primary" %}} 

[تنسيق المستند المحمول](https://en.wikipedia.org/wiki/PDF) هو تنسيق ملف أنشأته Adobe Systems لتبادل المستندات بين المؤسسات. كان هدف التنسيق هو الحفاظ على المحتوى والتخطيط كما هو، بغض النظر عن المنصة التي يُعرض عليها. يتيح Aspose.Slides for PHP via Java تحويل ملفات العروض التقديمية إلى PDF.

{{% /alert %}} 

## **PDF في Aspose.Slides for PHP via Java**
يمكن لأي عرض تقديمي يمكن تحميله في Aspose.Slides for PHP via Java أن يتحول إلى PDF يتوافق مع [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A)، [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A)، [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) أو [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA) بحسب اختيارك. تقوم Aspose.Slides for PHP via Java بتصدير العروض التقديمية إلى PDF وفي معظم الحالات، يبدو ملف PDF الناتج مطابِقًا تمامًا للعرض الأصلي.

تدعم Aspose.Slides الميزات التالية في العرض التقديمي عند التحويل إلى PDF:

- الصور، مربعات النص وغيرها من الأشكال.
- النص والتنسيق.
- الفقرات والتنسيق.
- الروابط التشعبية.
- الرؤوس وتذييلات الصفحات.
- القوائم النقطية.
- الجداول.

يمكنك تصدير العروض التقديمية إلى ملفات PDF مباشرةً باستخدام Aspose.Slides for PHP via Java: لا تحتاج إلى أي مكوّن آخر. علاوةً على ذلك، يمكنك تخصيص تصدير العرض إلى PDF باستخدام خيارات مختلفة كما هو موضح في [تحويل إلى PDF](/slides/ar/php-java/convert-powerpoint-to-pdf/).
