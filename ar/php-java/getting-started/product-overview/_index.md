## **وصف المنتج**
Aspose.Slides لـ PHP عبر Java هي مكتبة فئات متعددة المنصات تمكن تطبيقاتك من أداء مجموعة واسعة من مهام معالجة الوثائق بدون استخدام Microsoft PowerPoint®.

### لماذا يجب عليك استخدام Aspose.Slides لـ PHP عبر Java

- Aspose.Slides لـ PHP عبر Java هو المكون الأول والوحيد الذي يوفر الوظائف المطلوبة لإدارة مستندات PowerPoint®.
- Aspose.Slides لـ PHP عبر Java يقدم العديد من الميزات الرئيسية مثل إدارة النصوص، والأشكال، والجداول والرسوم المتحركة، وإضافة الصوت والفيديو إلى الشرائح، والمعاينة، وتصدير الشرائح إلى SVG وPDF وصيغ أخرى، والعديد من القدرات الأخرى.
- Aspose.Slides لـ PHP عبر Java لا يتيح لك فقط فتح ملفات PowerPoint من مصادر مختلفة، بل يتيح أيضًا حفظ العروض التقديمية بطرق متنوعة. على سبيل المثال، يمكنك حفظ عروضك التقديمية بصيغة PPT؛ يمكنك أيضًا حفظ الشرائح كصور.
- Aspose.Slides لـ PHP عبر Java يمنحك تقريبًا جميع الميزات التي قد تجدها أو لا تجدها في Microsoft PowerPoint. بالنظر إلى تعدد استخدامات Aspose.Slides لـ PHP عبر Java، بجانب الجداول وأشكال مختلفة، يمكنك استخدامه لإضافة أنواع مختلفة من الإطارات—الإطارات النصية، وإطارات الصوت، وإطارات الفيديو—إلى شرائحك.

## **ميزات Aspose.Slides**

يمكن تقسيم ميزات Aspose.Slides لـ PHP عبر Java إلى هذه المجموعات:

- ميزات العروض التقديمية.
- ميزات الشرائح.
- ميزات الرسوميات والوسائط المتعددة.
- ميزات الجداول.
- ميزات النص والفقرات.
- ميزات متقدمة.

### **ميزات العروض التقديمية**

- فتح عروض Microsoft PowerPoint من مصادر مختلفة.
- حفظ العروض التقديمية بصيغ Microsoft PowerPoint المختلفة.
- إنشاء عروض تقديمية من الصفر.
- تحويل العروض التقديمية إلى مستندات PDF بدون استخدام أي مكون آخر.
- قراءة أو تعديل خصائص مستند العرض التقديمي.
- تطبيق الحماية على العرض التقديمي.
- تصدير العرض التقديمي إلى PDF.
- تصدير العرض التقديمي إلى TIFF.
- تصدير العرض التقديمي إلى XPS.
- تصدير العرض التقديمي إلى SWF.
- تصدير العرض التقديمي إلى ملاحظات PDF / ملاحظات TIFF.
- تصدير العرض التقديمي إلى HTML.

### **ميزات الشرائح**

- إنشاء شرائح جديدة من الصفر وإضافتها إلى العروض التقديمية.
- الوصول إلى أي شريحة من خلال موضعها في العرض التقديمي.
- نسخ أو استنساخ الشرائح إلى نفس أو عرض تقديمي آخر.
- تحرير وحذف الشرائح.
- نسخ الماسترز مع الشرائح تلقائيًا.
- معاينة الشرائح.
- الحصول على اسم الشريحة أو تعيينه.
- حفظ الشرائح كصور.
- تصدير الشرائح إلى صيغة SVG.
- العمل مع العديد من الماسترز في العروض التقديمية.
- ملء خلفية الشرائح بأي نمط مدعوم.
- إضافة ملاحظات جديدة للشرائح أو إدارتها.
- تعيين الماسترز والأساليب الجديدة للشرائح بشكل ديناميكي.
- إضافة أو تعديل انتقال العرض التقديمي.
- إضافة انتقالات Morph.

### **ميزات الرسوميات والوسائط المتعددة**

- إنشاء أو إدارة أشكال مثل الخطوط والمربعات والدوائر والخطوط المنحنية.
- إدارة أنماط الخطوط في الأشكال.
- الحصول على النص البديل لشكل معين.
- تدوير الأشكال المقلوبة.
- ملء الأشكال بأساليب مختلفة—على سبيل المثال، صلبة، تدرج، نمط، قوام، وصورة.
- استخدام أي من 48 نمطًا متعريفًا مسبقًا و24 قوامًا.
- تعيين روابط للأشكال.
- قراءة وتعديل وإضافة تأثيرات الحركة الجديدة إلى الشرائح والأشكال.
- إضافة صور جديدة إلى العروض التقديمية وإدارة الموجودة بالفعل.
- إضافة نقاط صورة جديدة إلى العروض التقديمية وإدارة الموجودة بالفعل.
- إضافة إطارات صوت جديدة إلى العروض التقديمية وإدارة الموجودة بالفعل.
- إضافة إطارات فيديو جديدة إلى العروض التقديمية وإدارة الموجودة بالفعل.

### **ميزات الجداول**

- إنشاء جداول من الصفر وإضافتها إلى الشرائح.
- تغيير حجم الجداول ونقلها.
- الوصول إلى أي خلية في جدول.
- تغيير التنسيق والحدود لأي خلية.
- تغيير حجم الأعمدة والصفوف.
- إضافة وحذف الأعمدة والصفوف.
- تعيين عرض ولون الحدود للجدول بالكامل.

### **ميزات النص والفقرات**

- الحصول على النصوص وتعيينها في عناصر النص وإطارات النص.
- الحصول على نوع العنصر النائب.
- تغيير تنسيق النص في إطار النص.
- إضافة خطوط جديدة أو إدارة الخطوط المستخدمة بالفعل في العرض التقديمي.
- تغيير النصوص وتنسيق النص.
- إنشاء أو إدارة الروابط للنص.
- دعم علامات التبويب وعلامات التبويب الفعّالة.
- إضافة إطارات نص جديدة إلى تقريبًا أي نوع من الأشكال.
- الوصول إلى وتعديل إعدادات الهوامش والتفاف النص لإطارات النص.
- الوصول إلى وتعديل مسافة فقرات.

### **ميزات متقدمة**

- الوصول إلى كائنات OLE المدمجة في العروض التقديمية.
- قفل العروض التقديمية.

## **الدعم الفني**

تقدم Aspose دعمًا فنيًا مجانيًا غير محدود لجميع منتجاتها. الدعم متاح لجميع المستخدمين (بما في ذلك المستخدمين مع حزم التقييم). إذا كنت بحاجة إلى مساعدة بشأن Aspose.Slides لـ PHP عبر Java، انظر ما يلي:

- الطريق الرئيسي للدعم هو [منتديات Aspose](https://forum.aspose.com/). قدم سؤالك في [منتدى Aspose.Slides](https://forum.aspose.com/c/slides/11)—وسيتم الرد عليه خلال بضع ساعات. فريق دعم Aspose.Slides يجيب مباشرة على الأسئلة المنشورة في المنتدى.
- يرجى ملاحظة أن Aspose لا تقدم دعمًا فنيًا عبر الهاتف. دعم الهاتف متاح فقط لأسئلة المبيعات والشراء.
- عند توقع رد في المنتديات، يرجى التحلي بالصبر وإجراء تقييمات للفروق الزمنية.

إذا كانت لديك مشكلة مع Aspose.Slides لـ PHP عبر Java، اتبع هذه الإرشادات لحلها بأكثر الطرق كفاءة:

- تحقق وتأكد من أنك تستخدم أحدث إصدار من Aspose.Slides لـ PHP عبر Java أولاً قبل الإبلاغ عن المشكلة. ربما، تم حل المشكلة التي واجهتها بالفعل في تحديث.

  انظر [تحميل Aspose.Slides](https://downloads.aspose.com/slides/php-java) للحصول على معلومات حول أحدث إصدار من المنتج.

- قبل الإبلاغ عن مشكلة، قد ترغب في تصفح منتدياتنا، وهذه الوثائق، ومرجع API. ربما، تم تقديم إجابة على سؤالك بالفعل.

- عند الإبلاغ عن مشكلة، يرجى تضمين المستند الأصلي و(إذا أمكن) جزء من الشيفرة المتعلقة بالمشكلة. إذا كنت بحاجة إلى إرفاق ملفات متعددة، ضمها في حزمة ZIP.

  يمكنك بأمان إرفاق مستنداتك في Aspose.Forums لأنك فقط وأعضاء تطوير Aspose لديهم وصول إلى الملفات المرفقة.

- أنشئ واستخدم سلسلة واحدة لمشكلة معينة. إذا واجهت مشكلة أخرى، يجب عليك إنشاء سلسلة أخرى للإبلاغ عنها.
