## **مرحبًا بكم في Aspose.Slides للـ PHP عبر Java!**

Aspose.Slides هي واجهة برمجة تطبيقات لإدارة Microsoft PowerPoint® تتيح للتطبيقات قراءة وعرض وإنشاء وتعديل عروض PowerPoint دون الحاجة إلى Microsoft PowerPoint®. تدعم المكتبة العمل مع العروض، الشرائح، الأشكال، الجداول، النصوص، الرسوم المتحركة، الصوت، الفيديو، وعناصر العرض الأخرى.

مع Aspose.Slides، يمكن للمطورين فتح عروض من مصادر مختلفة، حفظها بصيغ PowerPoint متنوعة، تصدير العروض إلى صيغ مثل PDF و TIFF و XPS و SWF و HTML و SVG، وحفظ الشرائح كصور. توفر الواجهة أيضًا ميزات لإدارة خصائص المستند، تطبيق الحماية، معاينة الشرائح، تحرير محتوى الشريحة، العمل مع ملاحظات الشرائح، ومعالجة العناصر المتقدمة مثل كائنات OLE المضمَّنة.

### **لماذا ينبغي عليك استخدام Aspose.Slides للـ PHP عبر Java**

- Aspose.Slides للـ PHP عبر Java هو المكوّن الأول والوحيد الذي يوفّر الوظائف المطلوبة لإدارة مستندات PowerPoint®.  
- يقدم Aspose.Slides للـ PHP عبر Java العديد من الميزات الأساسية مثل إدارة النصوص، الأشكال، الجداول والرسوم المتحركة، إضافة الصوت والفيديو إلى الشرائح، معاينة الشرائح، تصدير الشرائح إلى SVG و PDF وصيغ أخرى، وغيرها الكثير.  
- لا يقتصر Aspose.Slides للـ PHP عبر Java على فتح ملفات PowerPoint من مصادر مختلفة فقط، بل يتيح لك أيضًا حفظ عروضك بطرق متعددة. على سبيل المثال، يمكنك حفظ العروض بصيغة PPT؛ كما يمكنك حفظ الشرائح كصور.  
- يقدم Aspose.Slides للـ PHP عبر Java تقريبًا جميع الميزات التي قد تجدها أو لا تجدها في Microsoft PowerPoint. وبفضل مرونة Aspose.Slides للـ PHP عبر Java، بخلاف الجداول وأنواع الأشكال المختلفة، يمكنك استخدامه لإضافة إطارات متنوعة—نصية، صوتية، وفيديوية—إلى الشرائح.  

## **ميزات Aspose.Slides**

يمكن تقسيم ميزات Aspose.Slides للـ PHP عبر Java إلى المجموعات التالية:

- ميزات العرض.
- ميزات الشرائح.
- ميزات الرسوميات والوسائط المتعددة.
- ميزات الجداول.
- ميزات النص والفقرات.
- ميزات متقدمة.

### **ميزات العرض**

- فتح عروض Microsoft PowerPoint من مصادر مختلفة.  
- حفظ العروض بصيغ Microsoft PowerPoint المختلفة.  
- إنشاء عروض من الصفر.  
- تحويل العروض إلى مستندات PDF بدون استخدام أي مكوّن آخر.  
- قراءة أو تعديل خصائص مستند العرض.  
- تطبيق الحماية على العرض.  
- تصدير العرض إلى PDF.  
- تصدير العرض إلى TIFF.  
- تصدير العرض إلى XPS.  
- تصدير العرض إلى SWF.  
- تصدير ملاحظات العرض إلى PDF / TIFF.  
- تصدير العرض إلى HTML.  

### **ميزات الشريحة**

- إنشاء شرائح جديدة من الصفر وإضافتها إلى العروض.  
- الوصول إلى أي شريحة عبر موقعها في العرض.  
- نسخ أو استنساخ الشرائح إلى نفس العرض أو عرض آخر.  
- تحرير وحذف الشرائح.  
- نسخ القوالب مع الشرائح تلقائيًا.  
- معاينة الشرائح.  
- الحصول على اسم الشريحة أو تعيينه.  
- حفظ الشرائح كصور.  
- تصدير الشرائح إلى صيغة SVG.  
- العمل مع عدة قوالب في العروض.  
- تعبئة خلفية الشرائح بأي نمط مدعوم.  
- إضافة أو إدارة ملاحظات الشرائح.  
- تعيين قوالب وأنماط جديدة للشرائح بشكل ديناميكي.  
- إضافة أو تعديل انتقالات عرض الشرائح.  
- إضافة انتقالات Morph.  

### **ميزات الرسوميات والوسائط المتعددة**

- إنشاء أو إدارة أشكال مثل الخطوط، المستطيلات، القطوع الإهليلجية، والبوليلينات.  
- إدارة أنماط الخط في الأشكال.  
- الحصول على النص البديل للشكل.  
- تدوير أو عكس الأشكال.  
- تعبئة الأشكال بأنماط مختلفة—مثل صلبة، تدرجية، نمطية، نسيجية، وصورية.  
- استخدام أي من 48 نمطًا محددًا مسبقًا و24 نسيجًا.  
- إسناد روابط إلى الأشكال.  
- قراءة وتعديل وإضافة تأثيرات رسوم متحركة جديدة إلى الشرائح والأشكال.  
- إضافة صور جديدة إلى العروض وإدارة الصور الموجودة.  
- إضافة نقاط صورة جديدة إلى العروض وإدارة النقاط الموجودة.  
- إضافة إطارات صوت جديدة إلى العروض وإدارة الإطارات الموجودة.  
- إضافة إطارات فيديو جديدة إلى العروض وإدارة الإطارات الموجودة.  

### **ميزات الجداول**

- إنشاء جداول من الصفر وإضافتها إلى الشرائح.  
- تعديل حجم الجداول وتحريكها.  
- الوصول إلى أي خلية في الجدول.  
- تغيير تنسيق وحدود أي خلية.  
- تعديل حجم الأعمدة والصفوف.  
- إضافة وحذف الأعمدة والصفوف.  
- تعيين عرض ولون الحدود للجدول بأكمله.  

### **ميزات النص والفقرات**

- الحصول على النصوص وتعيينها في العناصر النائبة للنص وإطارات النص.  
- الحصول على نوع العنصر النائب.  
- تغيير تنسيق النص في إطار النص.  
- إضافة خطوط جديدة أو إدارة الخطوط المستخدمة بالفعل في العرض.  
- تعديل النصوص وتنسيق النص.  
- إنشاء أو إدارة روابط النص.  
- دعم الفواصل والجدولات الفعالة.  
- إضافة إطارات نص جديدة إلى تقريبًا أي نوع من الأشكال.  
- الوصول إلى الهوامش وإعدادات لف النص في إطارات النص وتعديلها.  
- الوصول إلى إزاحة الفقرات وتعديلها.  

### **ميزات متقدمة**

- الوصول إلى كائنات OLE المضمنة في العروض.  
- قفل العروض.  

## **الدعم الفني**

توفر Aspose دعمًا فنيًا مجانيًا غير محدود لجميع منتجاتها. الدعم متاح لجميع المستخدمين (بما في ذلك المستخدمين الذين لديهم حزم تقييم). إذا كنت تحتاج إلى مساعدة بخصوص Aspose.Slides للـ PHP عبر Java، يرجى مراعاة ما يلي:

- القناة الرئيسية للدعم هي [منتديات Aspose](https://forum.aspose.com/). انشر سؤالك في [منتدى Aspose.Slides](https://forum.aspose.com/c/slides/ar/11)—وسيتم الرد خلال بضع ساعات. يجيب فريق دعم Aspose.Slides مباشرة على الأسئلة المطروحة في المنتدى.  
- يرجى ملاحظة أن Aspose لا توفر دعمًا فنيًا عبر الهاتف. الدعم الهاتفي متاح فقط لأسئلة المبيعات والشراء.
