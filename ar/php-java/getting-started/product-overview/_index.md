## **وصف المنتج**
Aspose.Slides for PHP via Java هي مكتبة فئات متعددة المنصات تمكن تطبيقاتك من تنفيذ مجموعة واسعة من مهام معالجة المستندات دون الحاجة إلى استخدام Microsoft PowerPoint®.

### **لماذا يجب عليك استخدام Aspose.Slides for PHP via Java**

- Aspose.Slides for PHP via Java هو أول مكون وحيد يوفّر الوظائف المطلوبة لإدارة مستندات PowerPoint®.  
- Aspose.Slides for PHP via Java يقدم العديد من الميزات الأساسية مثل إدارة النصوص والأشكال والجداول والرسوم المتحركة، إضافة الصوت والفيديو إلى الشرائح، معاينة الشرائح، تصدير الشرائح إلى SVG وPDF وغيرها من الصيغ، والعديد من القدرات الأخرى.  
- Aspose.Slides for PHP via Java لا يفتح ملفات PowerPoint من مصادر مختلفة فحسب، بل يتيح لك أيضًا حفظ عروضك التقديمية بطرق متنوعة. على سبيل المثال، يمكنك حفظ العروض بصيغة PPT؛ ويمكنك أيضًا حفظ الشرائح كصور.  
- Aspose.Slides for PHP via Java يمنحك تقريبًا جميع الميزات التي قد تجدها أو لا تجدها في Microsoft PowerPoint. بفضل مرونة Aspose.Slides for PHP via Java، إلى جانب الجداول وأنواع الأشكال المختلفة، يمكنك استخدامه لإضافة أنواع مختلفة من الإطارات—نصية، صوتية، وفيديوية—إلى الشرائح.  

## **ميزات Aspose.Slides**

يمكن تقسيم ميزات Aspose.Slides for PHP via Java إلى المجموعات التالية:

- ميزات العرض التقديمي.  
- ميزات الشرائح.  
- ميزات الرسومات والوسائط المتعددة.  
- ميزات الجداول.  
- ميزات النص والفقرات  
- ميزات متقدمة  

### **ميزات العرض التقديمي**

- فتح عروض Microsoft PowerPoint من مصادر مختلفة.  
- حفظ العروض بصيغ Microsoft PowerPoint المختلفة.  
- إنشاء عروض من الصفر.  
- تحويل العروض إلى مستندات PDF دون استخدام أي مكوّن آخر  
- قراءة أو تعديل خصائص مستند العرض  
- تطبيق الحماية على العرض  
- تصدير العرض إلى PDF  
- تصدير العرض إلى TIFF  
- تصدير العرض إلى XPS  
- تصدير العرض إلى SWF  
- تصدير ملاحظات العرض إلى PDF / TIFF  
- تصدير العرض إلى HTML  

### **ميزات الشريحة**

- إنشاء شرائح جديدة من الصفر وإضافتها إلى العروض.  
- الوصول إلى أي شريحة عبر موقعها في العرض.  
- نسخ أو استنساخ الشرائح إلى نفس العرض أو عرض آخر.  
- تحرير وحذف الشرائح.  
- نسخ القوالب مع الشرائح تلقائيًا.  
- معاينة الشرائح.  
- الحصول على اسم الشريحة أو تعيينه.  
- حفظ الشرائح كصور.  
- تصدير الشرائح إلى صيغة SVG.  
- العمل مع عدة قوالب في العروض.  
- ملء خلفية الشرائح بأي نمط مدعوم.  
- إضافة ملاحظات جديدة أو إدارة ملاحظات الشرائح.  
- تعيين قوالب الشرائح والأنماط الجديدة ديناميكيًا.  
- إضافة أو تعديل انتقال عرض الشرائح.  
- إضافة انتقالات Morph  

### **ميزات الرسومات والوسائط المتعددة**

- إنشاء أو إدارة الأشكال مثل الخطوط، المستطيلات، الإهليلجات، والمتعدد الخطوط.  
- إدارة أنماط الخطوط في الأشكال.  
- الحصول على النص البديل للشكل.  
- تدوير أو قلب الأشكال.  
- ملء الأشكال بأنماط مختلفة—مثل اللون الصلب، التدرج، النمط، الملمس، والصورة.  
- استخدام أي من الأنماط المحددة مسبقًا (48 نمطًا) و24 نسيجًا.  
- إسناد روابط إلى الأشكال.  
- قراءة وتعديل وإضافة تأثيرات تحريك جديدة إلى الشرائح والأشكال.  
- إضافة صور جديدة إلى العروض وإدارة الصور الحالية.  
- إضافة نقاط صورة جديدة إلى العروض وإدارة النقاط الحالية.  
- إضافة إطارات صوتية جديدة إلى العروض وإدارة الإطارات الحالية.  
- إضافة إطارات فيديو جديدة إلى العروض وإدارة الإطارات الحالية.  

### **ميزات الجدول**

- إنشاء جداول من الصفر وإضافتها إلى الشرائح.  
- تغيير حجم الجداول وتحريكها.  
- الوصول إلى أي خلية في الجدول.  
- تغيير تنسيق وحدود أي خلية.  
- تغيير حجم الأعمدة والصفوف.  
- إضافة وحذف الأعمدة والصفوف.  
- تعيين عرض ولون الحدود للجدول بالكامل.  

### **ميزات النص والفقرات**

- الحصول على النصوص وتعيينها في العناصر النائبة للنص وإطارات النص.  
- الحصول على نوع العنصر النائب.  
- تغيير تنسيق النص في إطار النص.  
- إضافة خطوط جديدة أو إدارة الخطوط المستخدمة بالفعل في العرض.  
- تغيير النصوص وتنسيق النص.  
- إنشاء أو إدارة الروابط داخل النص.  
- دعم علامات التبويب والتابات الفعّالة.  
- إضافة إطارات نصية جديدة إلى تقريبًا أي نوع من الأشكال.  
- الوصول إلى هوامش إعدادات النص وإعدادات التفاف النص في إطارات النص وتعديلها.  
- الوصول إلى إزاحة الفقرة وتعديلها.  

### **ميزات متقدمة**

- الوصول إلى كائنات OLE المضمنة في العروض.  
- قفل العروض.  

## **الدعم الفني**

تقدم Aspose دعمًا فنيًا غير محدود ومجانيًا لجميع منتجاتها. الدعم متاح لجميع المستخدمين (بما في ذلك المستخدمين الذين لديهم حزم تقييم). إذا كنت بحاجة إلى مساعدة بخصوص Aspose.Slides for PHP via Java، فكر في ما يلي:

- المسار الرئيسي للدعم هو [منتدى Aspose](https://forum.aspose.com/). انشر سؤالك في [منتدى Aspose.Slides](https://forum.aspose.com/c/slides/11)—وسيتم الرد خلال بضع ساعات. فريق دعم Aspose.Slides يجيب مباشرة على الأسئلة المنشورة في المنتدى.  
- يرجى ملاحظة أن Aspose لا تقدم دعمًا فنيًا عبر الهاتف. الدعم الهاتفي متاح فقط لأسئلة المبيعات والشراء.  
- عند انتظار رد في المنتديات، يرجى التحلي بالصبر وأخذ الفروقات الزمنية في الاعتبار.  

إذا كان لديك مشكلة مع Aspose.Slides for PHP عبر Java، اتبع هذه الإرشادات لحلها بأكثر طريقة كفاءة:

- تحقق وتأكد من أنك تستخدم أحدث إصدار من Aspose.Slides for PHP عبر Java قبل الإبلاغ عن المشكلة. ربما تم حل المشكلة التي واجهتها بالفعل في تحديث.  
  راجع [تحميل Aspose.Slides](https://downloads.aspose.com/slides/php-java) للحصول على معلومات حول أحدث نسخة من المنتج.  
- قبل الإبلاغ عن مشكلة، قد ترغب في مراجعة منتدانا، هذه الوثائق، ومرجع API. ربما تم توفير إجابة سؤالك بالفعل.  
- عند الإبلاغ عن مشكلة، يرجى إرفاق المستند الأصلي و(إن أمكن) جزء من الكود المتعلق بالمشكلة. إذا كنت بحاجة لإرفاق عدة ملفات، ضعها في حزمة ZIP.  
  يمكنك إرفاق مستنداتك بأمان في Aspose.Forums لأنك والمطورين في Aspose فقط من يمكنه الوصول إلى الملفات المرفقة.  
- أنشئ واستخدم موضوعًا واحدًا لمشكلة محددة. إذا واجهت مشكلة أخرى، يجب إنشاء موضوع آخر للإبلاغ عنها.

## **الأسئلة المتكررة**

**هل يدعم مستويات توافق PDF للأرشفة وإمكانية الوصول (PDF/A و PDF/UA)؟**

نعم. يمكنك الحفظ إلى PDF مع PDF/A-2a/2b/2u، PDF/A-3a/3b، وكذلك PDF/UA عن طريق تكوين [خيارات تصدير PDF](https://reference.aspose.com/slides/php-java/aspose.slides/pdfoptions/).

**هل هناك آلية استبدال الخطوط ودعم الخطوط المخصصة لضمان العرض الصحيح؟**

نعم. تسمح المكتبة لك بـ [تحميل الخطوط المخصصة](/slides/ar/php-java/custom-font/) و[تعريف قواعد الاحتياطي](/slides/ar/php-java/fallback-font/) بحيث يتم استبدال الأحرف المفقودة بشكل موثوق أثناء التحويل والعرض.

**هل يمكنني اكتشاف ما إذا كان الملف محميًا بكلمة مرور دون فتحه بالكامل؟**

نعم. يمكنك [فحص العرض](/slides/ar/php-java/examine-presentation/) لتحديد ما إذا كان يتطلب كلمة مرور قبل تحميل المستند بالكامل.

**هل يلزم وجود Microsoft PowerPoint للمعالجة والتحويلات؟**

لا. Aspose.Slides هو محرك مستقل؛ لا يلزم وجود PowerPoint على الخادم أو محطة العمل.

**هل من الآمن معالجة العروض من عدة خيوط (threads)؟**

نعم، يمكنك معالجة مستندات مختلفة بشكل متوازي في خيوط منفصلة؛ فقط تجنب استخدام نفس كائن العرض في وقت واحد [عبر الخيوط](/slides/ar/php-java/multithreading/).

**هل يتم الحفاظ على الماكروهات وهل يمكنني إدارة VBA في ملفات PPTM/PPSM؟**

نعم. العروض التي تحتوي على ماكروهات [مدعومة](/slides/ar/php-java/presentation-via-vba/)، ويمكنك [فحص وإدارة مشاريع VBA](https://reference.aspose.com/slides/php-java/aspose.slides/presentation/getvbaproject/) في تلك الملفات.

**هل يمكنني تحويل PDF أو HTML مرة أخرى إلى شرائح PowerPoint؟**

نعم. يمكنك [استيراد صفحات PDF أو محتوى HTML](/slides/ar/php-java/import-presentation/) لإنشاء أو ملء الشرائح في عرض تقديمي.

**هل يتم دعم تصدير XPS، وهل يمكنني التحكم في جودة ومحتوى مخرجات XPS؟**

نعم. [التصدير إلى XPS](/slides/ar/php-java/convert-powerpoint-to-xps/) متاح، و[خيارات الحفظ](https://reference.aspose.com/slides/php-java/aspose.slides/xpsoptions/) تتيح لك ضبط جودة المخرج والمحتوى المتضمن.

**هل يمكنني تحويل الشرائح إلى صور والتحكم في جودة المخرج؟**

نعم. يمكن تحويل الشرائح إلى [PNG، JPEG، GIF، BMP، TIFF](/slides/ar/php-java/convert-powerpoint-to-png/) وغيرها من الصيغ مع تحكم دقيق في الحجم والجودة.

**هل يتم دعم الصيغ الرياضية (MathML/MathText) وهل يمكن تصديرها؟**

نعم. يمكنك [إنشاء وتحرير النص الرياضي](/slides/ar/php-java/powerpoint-math-equations/) و[تصدير المعادلات](/slides/ar/php-java/exporting-math-equations/)، بما في ذلك إلى MathML.

**كيف يتعامل المكتبة مع التشفير وكلمات المرور (فتح، تعيين، إزالة)؟**

[تدعم](/slides/ar/php-java/password-protected-presentation/) فتح العروض المشفرة، تعيين أو إزالة كلمات المرور للفتح والتعديل، والتحقق من حالة حماية الملف.
