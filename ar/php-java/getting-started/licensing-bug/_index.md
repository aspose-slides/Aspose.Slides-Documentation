تصف هذه المقالة خطأً في الترخيص تم مواجهته في إصدارات Aspose.Slides for PHP via Java 23.2 و 23.3 و 23.4 و 23.5 و 23.6 و 23.7. جعل هذا الخطأ التراخيص الدائمة غير صالحة مؤقتًا بعد انتهاء فترة الاشتراك الخاصة بها.

## **الأعراض**

بعد انتهاء اشتراك الترخيص الدائم، قد تقرأ إصدارات Aspose.Slides for PHP via Java 23.2‑23.7 الترخيص كمنتهي، مما ينتقل بجميع الوظائف إلى وضع التقييم. هذا السلوك غير صحيح ولا يؤثر على الإصدارات السابقة للـ 23.2 وما بعد 23.7.

## **الحل**

قامت Aspose.Slides for PHP via Java بمعالجة هذه المشكلة وأصدرت إصدارات محدثة (23.2.1، 23.3.1، 23.4.1، 23.5.1، 23.6.1، 23.7.1) تحتوي على الإصلاح.

إذا كنت تستخدم أحد الإصدارات المتأثرة من Aspose.Slides for PHP via Java في مشروعك، يرجى الانتقال إلى الإصدار المصحح.

إذا واجهت أي مشكلات ترخيص مع Aspose.Slides for PHP via Java:

- تحقق من الإصدار الذي تستخدمه وتأكد من أنه غير متأثر بالخطأ المذكور أعلاه.
- ارجع إلى [توثيق Aspose](https://docs.aspose.com/slides/php-java/getting-started/) للحصول على خطوات استكشاف الأخطاء وإصلاحها ومشكلات الترخيص المعروفة.
- تواصل مع [دعم Aspose](https://forum.aspose.com/) للحصول على مساعدة إضافية.

## **المنتجات والإصدارات المتأثرة**

يرجى ملاحظة أن هذا الخطأ يؤثر فقط على إصدارات Aspose.Slides for PHP via Java من 23.2 إلى 23.7. **المنتجات الأخرى من Aspose والإصدارات الأخرى من Aspose.Slides غير متأثرة**.
