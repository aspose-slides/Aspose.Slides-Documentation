## **المنصات المدعومة**
Aspose.Slides for PHP عبر Java هي واجهة برمجة تطبيقات مستقلة عن المنصة. يمكن استخدامها على أي منصة (Windows, Linux, MacOS, إلخ) حيث يتم تثبيت PHP 7 أو أحدث. Oracle JDK 7 أو أحدث هو متطلب آخر.

## **تنسيقات الملفات والتحويلات**
Aspose.Slides for PHP عبر Java يدعم معظم تنسيقات مستندات PowerPoint. كما يسمح لك بتصديرها إلى تنسيقات الملفات الشائعة التي تستخدمها المنظمات وتشاركها فيما بينها. اطلع على التفاصيل التالية:

|**الميزة**|**الوصف**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for PHP عبر Java يوفر أسرع معالجة لهذا تنسيق مستند العرض.|
|تحويل PPT إلى PPTX|Aspose.Slides for PHP عبر Java يدعم تحويل PPT إلى PPTX|
|Portable Document Format (PDF)|يمكنك تصدير جميع تنسيقات الملفات المدعومة إلى مستندات Adobe Portable Document Format (PDF) باستخدام طريقة واحدة.|
|XML Parser Specification (XPS)|يمكنك تصدير جميع تنسيقات الملفات المدعومة إلى مستندات XML Parser Specification (XPS) باستخدام طريقة واحدة.|
|Tagged Image File Format (TIFF)|يمكنك تصدير جميع تنسيقات ملفات العرض المدعومة إلى Tagged Image File Format (TIFF).|
|تحويل PPTX إلى HTML|Aspose.Slides for PHP عبر Java يدعم تحويل PresentationEx إلى تنسيق HTML.|

## **عروض الشرائح**
Aspose.Slides for PHP عبر Java يدعم عرض عالي الدقة للشرائح في العرض إلى تنسيقات رسومية مختلفة. اطلع على التفاصيل التالية:

|**الميزة**|**الوصف**|
| :- | :- |
|تنسيقات الصور المدعومة في Java|باستخدام Aspose.Slides for PHP عبر Java، يمكنك عرض شرائح العرض والصور على الشرائح إلى جميع تنسيقات الرسومات المدعومة في Java مثل TIFF, PNG, BMP, JPEG, GIF.|
|تنسيق SVG|Aspose.Slides for PHP عبر Java يوفر أيضًا طرقًا مدمجة تسمح لك بتصدير شرائح العرض إلى تنسيقات Scalable Vector Graphics (SVG).|
|طباعة العرض|الإصدارات الأخيرة من Aspose.Slides for PHP عبر Java توفر طرق طباعة مدمجة مع خيارات مختلفة.|

## **ميزات التنسيق**
باستخدام Aspose.Slides for PHP عبر Java، يمكنك تنسيق النصوص والأشكال على الشرائح في العروض. اطلع على التفاصيل التالية:

|**الميزة**|**الوصف**|
| :- | :- |
|تنسيق النص|<p>في Aspose.Slides for PHP عبر Java، يمكنك إدارة النصوص عبر إطارات النص المرتبطة بالأشكال. وبالتالي، يمكنك تنسيق النصوص باستخدام الفقرات والأجزاء المرتبطة بإطارات النص. يمكن تنسيق هذه العناصر النصية عبر Aspose.Slides for PHP عبر Java.</p><p>- نوع الخط</p><p>- حجم الخط</p><p>- لون الخط</p><p>- تدرجات اللون</p><p>- محاذاة الفقرة</p><p>- تعداد الفقرة</p><p>- اتجاه الفقرة</p>|
|تنسيق الشكل|<p>في Aspose.Slides for PHP عبر Java، العنصر الأساسي للشريحة هو الشكل. يمكنك تنسيق هذه العناصر الشكلية عبر Aspose.Slides for PHP عبر Java:</p><p>- الموضع</p><p>- الحجم</p><p>- الخط</p><p>- التعبئة (بما في ذلك النمط، التدرج، الصلب)</p><p>- النص</p><p>- الصورة</p>|

## **الأسئلة المتكررة**

**هل أحتاج إلى تثبيت Microsoft PowerPoint على الخادم/الكمبيوتر لتعمل المكتبة؟**

لا. لا يلزم PowerPoint؛ Aspose.Slides محرك مستقل لإنشاء وتحرير وتحويل وعرض العروض.

**كيف يعمل تعدد الخيوط؟ هل يمكن إجراء المعالجة بالتوازي؟**

من الآمن معالجة مستندات مختلفة في خيوط مختلفة؛ لا يجب استخدام نفس [العرض](https://reference.aspose.com/slides/php-java/aspose.slides/presentation/) من قبل [خيوط متعددة](/slides/ar/php-java/multithreading/) في نفس الوقت.

**هل تدعم كلمات مرور الملفات والتشفير؟**

نعم. [يمكنك](/slides/ar/php-java/password-protected-presentation/) فتح العروض المشفرة، تعيين أو إزالة كلمة مرور الفتح والكتابة، والتحقق من حالة الحماية.

**هل يجب الاهتمام بحزم الخطوط في حاويات Linux؟**

نعم. يُنصح بتثبيت حزم الخطوط الشائعة و/أو [تحديد دلائل الخطوط](/slides/ar/php-java/custom-font/) صراحةً في تطبيقك لتجنب الاستبدالات غير المتوقعة.

**هل هناك قيود في نسخة التقييم؟**

في [وضع التقييم](/slides/ar/php-java/licensing/)، تُضاف علامة مائية إلى الناتج وتطبق قيود معينة؛ تتوفر [رخصة مؤقتة لمدة 30 يومًا](https://purchase.aspose.com/temporary-license/) لاختبار جميع الميزات.

**هل يتم دعم استيراد تنسيقات خارجية إلى عرض تقديمي (PDF/HTML → PPTX)؟**

نعم. يمكنك إضافة [صفحات PDF ومحتوى HTML](/slides/ar/php-java/import-presentation/) إلى عرض تقديمي، وتحويلها إلى شرائح.
