{{% alert color="primary" %}} 

Aspose.Slides لـ PHP عبر Java هو مكتبة فصول تتيح لتطبيقات PHP قراءة وكتابة وثائق العروض التقديمية دون الحاجة إلى استخدام Microsoft PowerPoint® أو غيره. Aspose.Slides لـ PHP عبر Java يتيح إدارة النصوص والأشكال والجداول والرسوم المتحركة، وإضافة الصوت والفيديو إلى الشرائح، ومعاينة الشرائح، وتصدير الشرائح إلى SVG، وPDF والمزيد.

يمكن العثور على أحدث مرجع واجهة برمجة التطبيقات في [مرجع Aspose.Slides لـ Java.](https://reference.aspose.com/slides/php-java/)
