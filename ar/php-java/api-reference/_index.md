{{% alert color="primary" %}} 
Aspose.Slides for PHP عبر Java هو مكتبة فئات تمكّن تطبيقات PHP من قراءة وكتابة مستندات العروض التقديمية دون الحاجة إلى استخدام Microsoft PowerPoint® أو غيره. يسمح Aspose.Slides for PHP عبر Java بإدارة النصوص والأشكال والجداول والرسوم المتحركة، وإضافة الصوت والفيديو إلى الشرائح، ومعاينة الشرائح، وتصدير الشرائح إلى تنسيقات SVG وPDF والمزيد.

المرجع API الأحدث يمكن العثور عليه في [مرجع API لـ Aspose.Slides for PHP عبر Java](https://reference.aspose.com/slides/ar/php-java/).

{{% /alert %}}
