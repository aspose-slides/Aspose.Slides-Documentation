## **إضافة نص ديناميكياً**
تتبع كلا الطريقتين هذه الخطوات:

1. إنشاء عرض تقديمي.
1. إضافة شريحة فارغة.
1. إضافة مربع نص.
1. تعيين نص.
1. كتابة العرض التقديمي.
