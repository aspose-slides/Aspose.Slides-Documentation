## **استبدال الصورة داخل مجموعة صور العرض التقديمي**
توفر Aspose.Slides لـ PHP عبر Java طرق API بسيطة لاستبدال الصور داخل مجموعة صور العرض التقديمي. يرجى اتباع الخطوات أدناه:

1. قم بتحميل ملف العرض التقديمي الذي يحتوي على الصورة باستخدام فئة Presentation.
1. قم بتحميل صورة من ملف في مصفوفة بايت.
1. استبدل الصورة المستهدفة بصورة جديدة في مصفوفة البايت.
1. في الطريقة الثانية، قم بتحميل الصورة في كائن Image واستبدل الصورة المستهدفة بالصورة المعبأة.
1. في الطريقة الثالثة، استبدل الصورة بصورة تمت إضافتها مسبقًا في مجموعة صور العرض التقديمي.
1. قم بكتابة العرض التقديمي المعدل كملف PPTX.
