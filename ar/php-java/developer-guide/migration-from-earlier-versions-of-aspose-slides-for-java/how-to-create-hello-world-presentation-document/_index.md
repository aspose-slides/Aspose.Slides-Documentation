تم إصدار [Aspose.Slides for PHP via Java API](/slides/php-java/) جديدة والآن هذا المنتج الواحد يدعم القدرة على إنشاء وثائق PowerPoint من الصفر وتحرير الوثائق الموجودة.

{{% /alert %}} 
## **الدعم للشفرة القديمة**
لاستخدام الشفرة القديمة المطورة مع Aspose.Slides for PHP via Java بالإصدارات السابقة لـ 13.x، تحتاج إلى إجراء بعض التغييرات البسيطة في الشفرة الخاصة بك وستعمل الشفرة كما في السابق. جميع الفئات التي كانت موجودة في Aspose.Slides for PHP via Java القديمة تحت أسماء نطاق Aspose.Slide و Aspose.Slides.Pptx قد تم دمجها الآن في اسم نطاق Aspose.Slides واحد. يرجى إلقاء نظرة على مقتطف الشفرة البسيط التالي لإنشاء وثيقة عرض تقديمي مرحبا بالعالم في واجهة برمجة التطبيقات القديمة لـ Aspose.Slides واتباع الخطوات الموضحة حول كيفية الانتقال إلى واجهة برمجة التطبيقات المدمجة الجديدة.
