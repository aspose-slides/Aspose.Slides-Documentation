تم إصدار [Aspose.Slides for PHP via Java API](https://docs.aspose.com/slides/php-java/) جديد والآن يدعم هذا المنتج الوحيد القدرة على إنشاء مستندات PowerPoint من الصفر وتحرير المستندات الموجودة.

{{% /alert %}} 
## **الدعم للكود القديم**
لاستخدام الكود القديم الذي تم تطويره باستخدام Aspose.Slides for PHP via Java بالإصدارات السابقة لـ 14.x.x، تحتاج إلى إجراء بعض التغييرات الطفيفة في الكود الخاص بك وسيدعم الكود كما كان من قبل. جميع الفئات التي كانت موجودة في Aspose.Slides for PHP via Java القديمة ضمن مساحات الأسماء com.aspose.slides و com.aspose.slides.pptx قد تم دمجها الآن في مساحة اسم واحدة com.aspose.slides. يرجى الاطلاع على مقتطف الكود البسيط التالي لإنشاء مخطط عادي من الصفر في العرض التقديمي باستخدام واجهة برمجة التطبيقات القديمة لـ Aspose.Slides واتباع الخطوات التي تصف كيفية الترحيل إلى واجهة برمجة التطبيقات الجديدة المدمجة.
