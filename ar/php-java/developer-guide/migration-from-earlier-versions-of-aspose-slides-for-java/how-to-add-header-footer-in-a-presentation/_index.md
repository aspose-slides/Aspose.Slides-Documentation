تم إصدار [Aspose.Slides for PHP via Java API](https://docs.aspose.com/slides/php-java/) جديدة والآن يدعم هذا المنتج الفردي القدرة على إنشاء مستندات PowerPoint من البداية وتحرير المستندات الموجودة.

{{% /alert %}} 
## **الدعم للكود القديم**
لاستخدام الكود القديم الذي تم تطويره بواسطة Aspose.Slides for PHP via Java بالإصدارات السابقة لـ 13.x، تحتاج إلى إجراء بعض التعديلات الطفيفة على الكود الخاص بك وسيعمل الكود كما كان سابقًا. جميع الفئات التي كانت موجودة في Aspose.Slides for PHP via Java القديمة تحت مسارات Aspose.Slide و Aspose.Slides.Pptx قد تم دمجها الآن في مساحة أسماء Aspose.Slides واحدة. يرجى الاطلاع على لقطة الكود البسيطة التالية لإضافة رأس وتذييل في العرض التقديمي في واجهة برمجة التطبيقات القديمة لـ Aspose.Slides واتباع الخطوات التي تصف كيفية الانتقال إلى واجهة برمجة التطبيقات المدمجة الجديدة.
