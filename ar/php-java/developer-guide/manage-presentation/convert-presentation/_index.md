## **نظرة عامة**

يمكن لـ Aspose.Slides for PHP via Java تحميل عروض PowerPoint وOpenDocument وحفظها أو تصييرها إلى العديد من الصيغ الأخرى دون الحاجة إلى Microsoft PowerPoint أو OpenOffice أو LibreOffice. يمكنك تحويل ملفات PPT القديمة إلى PPTX الحديثة، وتصدير العروض إلى مستندات ذات تخطيط ثابت مثل PDF وXPS، ونشر الشرائح كـ HTML، أو تصيير الشرائح كملفات صورة للمعاينات، والضحائح المصغرة، والأرشفة.

تستخدم أغلب عمليات تحويل المستندات نفس سير العمل العام: تحميل ملف المصدر، اختيار صيغة الإخراج المطلوبة، وتطبيق الخيارات الخاصة بالصِيغة عند الحاجة. بالنسبة لصيغ الصور، يتم تصيير كل شريحة على حدة ثم حفظها كصورة نقطية أو متجهة. المقالات المخصصة المرتبطة أدناه توفر تفاصيل التنفيذ لكل حالة.

## **تحويل PowerPoint وOpenDocument**

يدعم Aspose.Slides for PHP via Java التحويل من صيغ العروض الشائعة مثل PPT وPPTX وPPS وPPSX وPOT وPOTX وODP. يتم استخدام نفس واجهة برمجة التحويل لملفات PowerPoint وOpenDocument، لذا يمكن عادة تطبيق سير عمل يحفظ ملف PPTX إلى PDF على ملف ODP بتغيير ملف الإدخال فقط.

عند تحويل ملفات ODP، تذكر أن تطبيقات PowerPoint وOpenDocument لا تدعم كل تخطيط وميزة تنسيق بنفس الطريقة تمامًا. إذا تم إنشاء ملف ODP في LibreOffice أو OpenOffice Impress، راجع النتيجة واستخدم الخيارات المذكورة في [Convert OpenDocument Presentations](/slides/ar/php-java/convert-openoffice-odp/) عندما تحتاج إلى إرشاد خاص بالصِيغة.

## **تحويل PPT إلى PPTX**

PPT هو تنسيق PowerPoint الثنائي القديم، بينما PPTX هو تنسيق Office Open XML الحديث. يدعم Aspose.Slides for PHP via Java تحويل PPT إلى PPTX بدقة عالية مع الحفاظ على هياكل العرض المعقدة مثل القوالب الرئيسية، التخطيطات، الشرائح، المخططات، الأشكال المجمعة، العناصر النائبة، إطارات النص، القوام، وتعبئات الصورة.

للتفاصيل، راجع [Convert PPT to PPTX](/slides/ar/php-java/convert-ppt-to-pptx/) و[PPT vs PPTX](/slides/ar/php-java/ppt-vs-pptx/).

## **تصدير تخطيط ثابت**

PDF وXPS وTIFF مفيدة عندما يجب أن يبدو الناتج نفسه على جميع الأجهزة ولا ينبغي تحريره كعرض تقديمي. تشرح المقالات المخصصة لـ PDF وXPS وTIFF كيفية التحكم في الالتزام، الشرائح المخفية، الملاحظات، جودة الصورة، الضغط، تنسيق البكسل، وحجم الإخراج.

## **تصدير HTML والصور**

تصدير HTML وHTML5 مفيد للعرض في المتصفح، النشر على الويب، والمشاركة الخفيفة. تصدير الصور مفيد عندما يجب أن تتحول كل شريحة إلى معاينة منفصلة أو صورة مصغرة أو أصل نقطي. استخدم مقالات PNG وJPG وSVG للحصول على إرشادات تصيير خاصة بالصِيغة.

## **الأسئلة الشائعة**

**هل أحتاج إلى Microsoft PowerPoint لتحويل العروض؟**

لا. Aspose.Slides for PHP via Java مكتبة مستقلة ولا تتطلب Microsoft PowerPoint أو أتمتة Office.

**هل يمكنني تحويل العديد من العروض دفعة واحدة؟**

نعم. قم بتحميل كل عرض، حفظه بالتنسيق المطلوب، وتحرير كائن العرض بعد المعالجة. للمعالجة المتوازية، استخدم نسخًا منفصلة من كائن العرض واتبع إرشادات [multithreading](/slides/ar/php-java/multithreading/).

**هل يمكنني تصدير شرائح مختارة فقط؟**

نعم. تسمح عدة طرق تصدير بتمرير مؤشرات الشرائح أو تصيير الشرائح الفردية، حسب تنسيق الإخراج. راجع المقالة المخصصة للتنسيق المستهدف.

**هل يمكنني تضمين الشرائح المخفية عند التصدير إلى PDF أو XPS؟**

نعم. استخدم إعدادات تصدير الشرائح المخفية الموصوفة في مقالات التحويل [PDF](/slides/ar/php-java/convert-powerpoint-to-pdf/) و[XPS](/slides/ar/php-java/convert-powerpoint-to-xps/).

**هل يمكنني إنشاء مخرجات PDF/A؟**

نعم. تتوفر إعدادات الالتزام بـ PDF لتصدير PDF. راجع [Convert PowerPoint to PDF](/slides/ar/php-java/convert-powerpoint-to-pdf/) للتفاصيل.

**كيف يتم التعامل مع الخطوط أثناء التحويل؟**

يمكن لـ Aspose.Slides استخدام الخطوط المضمنة، والبدائل، وإعدادات استبدال الخطوط. راجع [Embedded Font](/slides/ar/php-java/embedded-font/)، [Fallback Font](/slides/ar/php-java/fallback-font/)، و[Font Substitution](/slides/ar/php-java/font-substitution/).
