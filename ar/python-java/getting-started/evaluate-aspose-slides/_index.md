توفر نسخة التقييم من Aspose.Slides (بدون ترخيص محدد) كامل وظائف المنتج، ولكنها تُدخل علامة مائية للتقييم في أعلى المستند عند الفتح والحفظ. كما أنك محدود بشريحة واحدة عند استخراج النصوص من شرائح العرض.
