## **المنصات المدعومة**
Aspose.Slides for Python via Java هو واجهة برمجة تطبيقات مستقلة عن النظام الأساسي. يمكن استخدامه على أي منصة (Windows، Linux، MacOS، إلخ) حيث تم تثبيت Python 3.7 أو أعلى. JDK 8 أو أعلى هو شرط آخر.

## **تنسيقات الملفات والتحويلات**
يدعم Aspose.Slides for Python via Java معظم تنسيقات مستندات PowerPoint. كما يتيح لك تصديرها إلى التنسيقات الشائعة التي تستخدمها المؤسسات وتشاركها فيما بينها. اطلع على هذه التفاصيل:

|**الميزة**|**الوصف**|
| :- | :- |
|Microsoft PowerPoint (PPT)|يوفر Aspose.Slides for Python via Java أسرع معالجة لهذا التنسيق من مستندات العروض التقديمية.|
|تحويل PPT إلى PPTX|يدعم Aspose.Slides for Python via Java تحويل PPT إلى PPTX|
|تنسيق المستندات المحمولة (PDF)|يمكنك تصدير جميع تنسيقات الملفات المدعومة إلى مستندات تنسيق المستندات المحمولة من Adobe (PDF) بطريقة واحدة.|
|مواصفات محلل XML (XPS)|يمكنك تصدير جميع تنسيقات الملفات المدعومة إلى مستندات مواصفات محلل XML (XPS) بطريقة واحدة.|
|تنسيق ملف الصورة المميزة (TIFF)|يمكنك تصدير جميع تنسيقات ملفات العروض المدعومة إلى تنسيق ملف الصورة المميزة (TIFF).|
|تحويل PPTX إلى HTML|يدعم Aspose.Slides for Python via Java تحويل PresentationEx إلى تنسيق HTML.|

|**الميزة**|**الوصف**|
| :- | :- |
|تنسيقات الصور المدعومة من Java|باستخدام Aspose.Slides for Python via Java، يمكنك عرض شرائح العروض والصور في الشرائح إلى جميع تنسيقات الرسوم المدعومة من Java مثل TIFF وPNG وBMP وJPEG وGIF.|
|تنسيق SVG|يوفر Aspose.Slides for Python via Java أيضًا طرقًا مدمجة تتيح لك تصدير شرائح العروض إلى تنسيقات الرسوم البيانية القابلة للتوسيع (SVG).|

## **ميزات التنسيق**

|**الميزة**|**الوصف**|
| :- | :- |
|تنسيق النص|<p>في Aspose.Slides for Python via Java، يمكنك إدارة النصوص من خلال إطارات النص المرتبطة بالأشكال. وبالتالي، يمكنك تنسيق النصوص باستخدام الفقرات والأجزاء المرتبطة بإطارات النص. يمكن تنسيق هذه العناصر النصية من خلال Aspose.Slides for Python via Java.</p><p>- نوع الخط</p><p>- حجم الخط</p><p>- لون الخط</p><p>- ظلال الخط</p><p>- محاذاة الفقرة</p><p>- تعداد الفقرة</p><p>- اتجاه الفقرة</p>|
|تنسيق الشكل|<p>في Aspose.Slides for Python via Java، العنصر الأساسي في الشريحة هو الشكل. يمكنك تنسيق هذه العناصر الشكلية باستخدام Aspose.Slides for Python via Java:</p><p>- الموقع</p><p>- الحجم</p><p>- الخط</p><p>- التعبئة (بما في ذلك النمط، التدرج، الصلب)</p><p>- النص</p><p>- الصورة</p>|
