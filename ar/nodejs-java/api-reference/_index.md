{{% alert color="primary" %}} 

Aspose.Slides لـ Node.js عبر Java هي مكتبة فئات تمكن تطبيقات Node.js من قراءة وكتابة مستندات العروض التقديمية دون استخدام Microsoft PowerPoint® أو غيره. تتيح لك Aspose.Slides لـ Node.js عبر Java إدارة النصوص والأشكال والجداول والحركات، وإضافة الصوت والفيديو إلى الشرائح، ومعاينة الشرائح، وتصدير الشرائح إلى SVG، PDF، والقيام بالكثير غير ذلك.

تستخدم Aspose.Slides لـ Node.js عبر Java [نفس مرجع API مثل Aspose.Slides لـ Java](https://reference.aspose.com/slides/nodejs-java/).
