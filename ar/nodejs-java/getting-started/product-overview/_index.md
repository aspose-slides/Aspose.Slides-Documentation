## **مرحبًا بك في Aspose.Slides لـ Node.js عبر Java!**

Aspose.Slides هو API لإدارة Microsoft PowerPoint® يتيح للتطبيقات قراءة وكتابة وإنشاء وتعديل عروض PowerPoint دون الحاجة إلى استخدام Microsoft PowerPoint®. تدعم المكتبة العمل مع العروض، الشرائح، الأشكال، الجداول، النص، الرسوم المتحركة، الصوت، الفيديو، والعناصر الأخرى للعرض.

مع Aspose.Slides، يمكن للمطورين فتح العروض من مصادر مختلفة، حفظها بتنسيقات PowerPoint المتعددة، تصدير العروض إلى صيغ مثل PDF و TIFF و XPS و SWF و HTML و SVG، وحفظ الشرائح كصور. كما يوفر API ميزات لإدارة خصائص المستند، تطبيق الحماية، عرض معاينات الشرائح، تحرير محتوى الشريحة، العمل مع ملاحظات الشرائح، ومعالجة عناصر العرض المتقدمة مثل كائنات OLE المدمجة.

## **لماذا يجب عليك استخدام Aspose.Slides لـ Node.js عبر Java**

- Aspose.Slides لـ Node.js عبر Java هو المكوّن الأول والوحيد الذي يوفر الوظائف المطلوبة لإدارة مستندات PowerPoint®.  
- يقدم Aspose.Slides لـ Node.js عبر Java الكثير من الميزات الأساسية مثل إدارة النصوص، الأشكال، الجداول والرسوم المتحركة، إضافة الصوت والفيديو إلى الشرائح، عرض معاينات الشرائح، تصدير الشرائح إلى SVG و PDF وغيرها من الصيغ، والعديد من القدرات الأخرى.  
- لا يقتصر Aspose.Slides لـ Node.js عبر Java على فتح ملفات PowerPoint من مصادر مختلفة فحسب، بل يسمح أيضًا بحفظ عروضك بطرق متعددة. على سبيل المثال، يمكنك حفظ عروضك بصيغة PPT؛ يمكنك أيضًا حفظ الشرائح كصور.  
- يمنحك Aspose.Slides لـ Node.js عبر Java تقريبًا جميع الميزات التي قد تجدها أو لا تجدها في Microsoft PowerPoint. بفضل مرونة Aspose.Slides لـ Node.js عبر Java، إلى جانب الجداول وأنواع الأشكال المختلفة، يمكنك إضافة إطارات مختلفة—نصية، صوتية، وفيديوية—إلى الشرائح.

## **ميزات Aspose.Slides**

يمكن تقسيم ميزات Aspose.Slides لـ Node.js عبر Java إلى المجموعات التالية:

- ميزات العرض.
- ميزات الشرائح.
- ميزات الرسومات والوسائط المتعددة.
- ميزات الجداول.
- ميزات النص والفقرات.
- الميزات المتقدمة.

### **ميزات العرض**

- فتح عروض Microsoft PowerPoint من مصادر مختلفة.  
- حفظ العروض بصيغ Microsoft PowerPoint المتعددة.  
- إنشاء عروض من الصفر.  
- تحويل العروض إلى مستندات PDF دون الحاجة إلى مكوّن آخر.  
- قراءة أو تعديل خصائص مستند العرض.  
- تطبيق الحماية على العرض.  
- تصدير العرض إلى PDF.  
- تصدير العرض إلى TIFF.  
- تصدير العرض إلى XPS.  
- تصدير العرض إلى SWF.  
- تصدير ملاحظات العرض إلى PDF / TIFF.  
- تصدير العرض إلى HTML.

### **ميزات الشرائح**

- إنشاء شرائح جديدة من الصفر وإضافتها إلى العروض.  
- الوصول إلى أي شريحة عبر موقعها في العرض.  
- نسخ أو استنساخ الشرائح إلى نفس العرض أو إلى عرض آخر.  
- تحرير وحذف الشرائح.  
- نسخ القوالب مع الشرائح تلقائيًا.  
- عرض معاينات للشرائح.  
- الحصول على اسم الشريحة أو تعيينه.  
- حفظ الشرائح كصور.  
- تصدير الشرائح إلى صيغة SVG.  
- العمل مع عدة قوالب في العروض.  
- ملء خلفية الشرائح بأي نمط مدعوم.  
- إضافة أو إدارة ملاحظات الشرائح.  
- تعيين قوالب وأنماط شريحة جديدة بشكل ديناميكي.  
- إضافة أو تعديل انتقالات عرض الشرائح.  
- إضافة انتقالات Morph.

### **ميزات الرسومات والوسائط المتعددة**

- إنشاء أو إدارة الأشكال مثل الخطوط، المستطيلات، الأقراص، والمتعددات.  
- إدارة أنماط الخطوط في الأشكال.  
- الحصول على النص البديل للشكل.  
- تدوير أو عكس الأشكال.  
- ملء الأشكال بأنماط مختلفة—مثل صلب، تدرج، نمط، نسيج، وصورة.  
- استخدام أيٍ من 48 نمطًا مسبقًا و 24 نسيجًا.  
- إضافة روابط إلى الأشكال.  
- قراءة، تعديل، وإضافة تأثيرات رسوم متحركة جديدة إلى الشرائح والأشكال.  
- إضافة صور جديدة إلى العروض وإدارة الصور الموجودة.  
- إضافة نقاط صور جديدة إلى العروض وإدارة النقاط الموجودة.  
- إضافة إطارات صوتية جديدة إلى العروض وإدارة الإطارات الموجودة.  
- إضافة إطارات فيديو جديدة إلى العروض وإدارة الإطارات الموجودة.

### **ميزات الجداول**

- إنشاء جداول من الصفر وإضافتها إلى الشرائح.  
- تغيير حجم الجداول وتحريكها.  
- الوصول إلى أي خلية في الجدول.  
- تغيير تنسيق وحدود أي خلية.  
- تعديل حجم الأعمدة والصفوف.  
- إضافة وحذف الأعمدة والصفوف.  
- تعيين عرض الحدود ولونها للجدول بالكامل.

### **ميزات النص والفقرات**

- الحصول على النصوص أو تعيينها في عناصر النص والإطارات النصية.  
- الحصول على نوع العنصر النائب.  
- تغيير تنسيق النص داخل الإطار النصي.  
- إضافة خطوط جديدة أو إدارة الخطوط المستخدمة بالفعل في العرض.  
- تغيير النصوص وتنسيق النص.  
- إنشاء أو إدارة الروابط إلى النص.  
- دعم الفواصل والمسافات الفعالة.  
- إضافة إطارات نصية جديدة إلى تقريبًا أي نوع من الأشكال.  
- الوصول إلى تعديل الهوامش وإعدادات إلتفاف النص في الإطارات النصية.  
- الوصول إلى تعديل مسافة الفقرة.

### **الميزات المتقدمة**

- الوصول إلى كائنات OLE المدمجة في العروض.  
- قفل العروض.
