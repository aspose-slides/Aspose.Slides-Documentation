## **تقييم Aspose.Slides**
يمكنك بسهولة تنزيل Aspose.Slides للتقييم. حزمة التقييم هي نفس حزمة الشراء. تصبح نسخة التقييم مرخصة بمجرد إضافة بضع أسطر من الشيفرة لتطبيق الترخيص.

## **قيود نسخة التقييم**
توفر نسخة التقييم من Aspose.Slides (بدون ترخيص محدد) جميع وظائف المنتج، لكن يتم إضافة علامة مائية للتقييم في أعلى المستند عند الفتح والحفظ. كما تُقصر استخراج النصوص من شرائح العرض على شريحة واحدة فقط.

{{% alert color="primary" %}} 
إذا رغبت في اختبار Aspose.Slides دون قيود نسخة التقييم، يمكنك طلب **رخصة مؤقتة لمدة 30 يومًا**. يرجى الرجوع إلى [كيفية الحصول على رخصة مؤقتة؟](https://purchase.aspose.com/temporary-license) لمزيد من المعلومات.
{{% /alert %}} 

## **حول الترخيص**
يمكنك بسهولة تنزيل نسخة تقييم Aspose.Slides لـ Node.js عبر Java من صفحة تنزيله [هنا](https://releases.aspose.com/slides/nodejs-java/). تقدم نسخة التقييم نفس القدرات تمامًا كما النسخة المرخصة من Aspose.Slides. علاوة على ذلك، تصبح نسخة التقييم مرخصة بمجرد شراء ترخيص وإضافة بضع أسطر من الشيفرة لتطبيق الترخيص.

الترخيص هو ملف XML نصي بسيط يحتوي على تفاصيل مثل اسم المنتج، عدد المطورين المرخص لهم، تاريخ انتهاء الاشتراك، وما إلى ذلك. الملف موقّع رقمياً، لذا لا تقم بتعديل الملف. حتى إضافة سطر فارغ غير مقصودة إلى محتويات الملف سيجعل الترخيص غير صالح.

لتجنب القيود المرتبطة بنسخة التقييم، تحتاج إلى تعيين الترخيص قبل استخدام **Aspose.Slides**. يُطلب منك تعيين الترخيص مرة واحدة فقط لكل تطبيق أو عملية.

قد ترغب في الاطلاع على [ترخيص المقياس](https://docs.aspose.com/slides/nodejs-java/metered-licensing/).

### **إعداد الترخيص في Aspose.Slides لـ Node.js عبر Java**

يمكن تطبيق الترخيص من هذه المواقع:

* مسار صريح
* تدفق
* كرخصة مقياس – آلية ترخيص جديدة

{{% alert color="primary" %}}
استخدم طريقة **setLicense** لترخيص مكوّن.
على الرغم من أن استدعاءات متعددة لـ **setLicense** ليست ضارة، إلا أنها مضيعة للموارد (المعالج).
{{% /alert %}}

{{% alert color="warning" %}}
يمكن للتراخيص الجديدة تفعيل Aspose.Slides فقط مع الإصدار 21.4 أو أعلاه. الإصدارات السابقة تستخدم نظام ترخيص مختلف ولن تتعرف على هذه التراخيص.
{{% /alert %}}

#### **تطبيق الترخيص باستخدام ملف**

يُستخدم هذا المقتطف لضبط ملف الترخيص:

**Node.js**

```javascript
var aspose = aspose || {};

aspose.slides = require("aspose.slides.via.java");

var license = new aspose.slides.License();
license.setLicense("Aspose.Slides.lic");
```

عند استدعاء طريقة `setLicense`، يجب أن يكون اسم الترخيص هو نفسه اسم ملف الترخيص الخاص بك. على سبيل المثال، يمكنك تغيير اسم ملف الترخيص إلى "Aspose.Slides.lic.xml". ثم، في الشيفرة، عليك تمرير الاسم الجديد للترخيص (Aspose.Slides.lic.xml) إلى طريقة `setLicense`.

#### **تطبيق الترخيص من تدفق**

يُستخدم هذا المقتطف لتطبيق الترخيص من تدفق:

**Node.js**

```javascript
var aspose = aspose || {};

aspose.slides = require("aspose.slides.via.java");

var license = new aspose.slides.License();

var fs = require("fs");

var readStream = fs.createReadStream("Aspose.Slides.lic");

license.setLicense(readStream, function(err, list) {
    if(err) { 
        console.error(err); return; 
    }});
```
