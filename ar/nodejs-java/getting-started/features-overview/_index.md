## **المنصات المدعومة**
Aspose.Slides for Node.js via Java هو API مستقل عن المنصة. يمكن استخدامه على أي منصة (Windows, Linux, MacOS, إلخ) حيث يتم تثبيت Node.js 7 أو أعلى. يتطلب أيضًا Oracle JDK 8 أو أعلى.

## **تنسيقات الملفات والتحويلات**
Aspose.Slides for Node.js via Java يدعم معظم تنسيقات مستندات PowerPoint. كما يتيح لك تصديرها إلى تنسيقات الملفات الشائعة التي تستخدمها المؤسسات وتشاركها فيما بينها. اطلع على التفاصيل التالية:

|**الميزة**|**الوصف**|
| :- | :- |
|Microsoft PowerPoint (PPT)|توفر Aspose.Slides for Node.js via Java أسرع معالجة لهذا تنسيق مستند العرض التقديمي.|
|PPT to PPTX conversion|يدعم Aspose.Slides for Node.js via Java تحويل PPT إلى PPTX.|
|Portable Document Format (PDF)|يمكنك تصدير جميع تنسيقات الملفات المدعومة إلى مستندات Adobe Portable Document Format (PDF) باستخدام طريقة واحدة.|
|XML Parser Specification (XPS)|يمكنك تصدير جميع تنسيقات الملفات المدعومة إلى مستندات XML Parser Specification (XPS) باستخدام طريقة واحدة.|
|Tagged Image File Format (TIFF)|يمكنك تصدير جميع تنسيقات ملفات العروض التقديمية المدعومة إلى Tagged Image File Format (TIFF).|
|PPTX To HTML Conversion|يدعم Aspose.Slides for Node.js via Java تحويل PresentationEx إلى تنسيق HTML.|

## **التصيير والطباعة**
يدعم Aspose.Slides for Node.js via Java تصيير عالي الدقة للشرائح في العرض التقديمي إلى تنسيقات رسومية مختلفة. اطلع على التفاصيل التالية:

|**الميزة**|**الوصف**|
| :- | :- |
|Java Supported Image Formats|باستخدام Aspose.Slides for Node.js via Java، يمكنك تصيير شرائح العروض التقديمية والصور على الشرائح إلى جميع تنسيقات الرسومات المدعومة في Java مثل TIFF, PNG, BMP, JPEG, GIF.|
|SVG Format|يوفر Aspose.Slides for Node.js via Java أيضًا طرقًا مدمجة تسمح لك بتصدير شرائح العرض إلى تنسيقات Scalable Vector Graphics (SVG).|
|Presentation Printing|الإصدارات الأخيرة من Aspose.Slides for Node.js via Java توفر طرق طباعة مدمجة مع خيارات مختلفة.|

|**الميزة**|**الوصف**|
| :- | :- |
|Master Slides|تحدد الشرائح الأساسية تخطيط الشرائح العادية. يسمح Aspose.Slides for Node.js via Java لك بالوصول إلى الشرائح الأساسية وتعديلها في مستندات العرض.|
|Normal Slides|مع Aspose.Slides for Node.js via Java، يمكنك إنشاء شرائح جديدة من أنواع مختلفة؛ كما يمكنك الوصول إلى الشرائح الموجودة وتعديلها في العروض.|
|Cloning / Copying Slides|توفر Aspose.Slides for Node.js via Java طرقًا مدمجة تسمح لك بنسخ أو استنساخ الشرائح الحالية داخل عرض تقديمي. يمكنك أيضًا استخدام الشرائح المنسوخة أو المستنسخة من عرض إلى آخر. نظرًا لأن الشريحة ترث تخطيطها من الشريحة الأساسية، فإن طرق الاستنساخ المدمجة تنسخ الشريحة الأساسية تلقائيًا.|
|Managing Slides sections|طرق لتنظيم الشرائح في أقسام مختلفة داخل العرض التقديمي.|
|Place Holders and Text Holders|يمكنك الوصول إلى حاملي الأماكن وحاملي النص في الشريحة. علاوةً على ذلك، يمكنك إنشاء شريحة بحاملات نص من الصفر باستخدام الطريقة المناسبة.|
|Header and Footers|يسهل Aspose.Slides for Node.js via Java التعامل مع رؤوس وتذييلات الشرائح.|
|Notes in Slides|مع Aspose.Slides for Node.js via Java، يمكنك الوصول إلى الملاحظات المرتبطة بشريحة وتعديلها، وكذلك إضافة ملاحظات جديدة.|
|Finding a Shape|يمكنك أيضًا العثور على شكل معين في شريحة باستخدام النص البديل المرتبط بالشكل.|

|Text Boxes|يمكن إنشاء مربعات نص من الصفر. يمكنك أيضًا الوصول إلى مربعات النص الموجودة وتعديل نصوصها دون فقدان تنسيق النص الأصلي.|
|Rectangle Shapes|يمكنك إنشاء أو تعديل أشكال المستطيل باستخدام Aspose.Slides for Node.js via Java.|
|Poly Line Shapes|يمكنك إنشاء أو تعديل أشكال الخط المتعدد باستخدام Aspose.Slides for Node.js via Java.|

|Auto Shapes|يدعم Aspose.Slides for Node.js via Java الأشكال التلقائية.|

## **الأسئلة المتكررة**

**هل أحتاج إلى تثبيت Microsoft PowerPoint على الخادم/الكمبيوتر لتعمل المكتبة؟**

لا. لا يلزم PowerPoint؛ Aspose.Slides هو محرك مستقل لإنشاء وتعديل وتحويل وعرض العروض التقديمية.

**كيف يعمل تعدد الخيوط؟ هل يمكن تنفيذ المعالجة بصورة متوازية؟**

من الآمن معالجة مستندات مختلفة في خيوط مختلفة؛ يجب ألا يتم استخدام نفس [العرض التقديمي](https://reference.aspose.com/slides/nodejs-java/aspose.slides/presentation/) بواسطة [خيوط متعددة](/slides/ar/nodejs-java/multithreading/) في نفس الوقت.

**هل تدعم كلمات مرور الملفات والتشفير؟**

نعم. يمكنك [فتح](/slides/ar/nodejs-java/password-protected-presentation/) عروض تقديمية مشفرة، وضع أو إزالة كلمة مرور للفتح والكتابة، والتحقق من حالة الحماية.

**هل يجب أن أهتم بحزم الخطوط في حاويات Linux؟**

نعم. يُنصح بتثبيت حزم الخطوط الشائعة و/أو تحديد أدلة الخطوط صراحةً [/slides/nodejs-java/custom-font/](/slides/ar/nodejs-java/custom-font/) في تطبيقك لتجنب الاستبدالات غير المتوقعة.

**هل هناك قيود في نسخة التقييم؟**

في [وضع التقييم](/slides/ar/nodejs-java/licensing/)، يتم إضافة علامة مائية إلى الناتج وتطبق بعض القيود؛ يتوفر [رخصة مؤقتة لمدة 30 يومًا](https://purchase.aspose.com/temporary-license/) لاختبار جميع الميزات.

**هل يدعم استيراد الصيغ الخارجية إلى عرض تقديمي (PDF/HTML → PPTX)؟**

نعم. يمكنك إضافة [صفحات PDF ومحتوى HTML](/slides/ar/nodejs-java/import-presentation/) إلى عرض تقديمي، وتحويلها إلى شرائح.
