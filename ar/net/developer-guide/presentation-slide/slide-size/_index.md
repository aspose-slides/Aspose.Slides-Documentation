## **تخصيص أحجام الشرائح ونسب العرض إلى الارتفاع في PowerPoint**

توفر Aspose.Slides for .NET أدوات شاملة لضبط حجم الشريحة ونسبة العرض إلى الارتفاع في عروض PowerPoint التقديمية، وهو أمر حاسم للطباعة والعرض على الشاشة.

### **الأحجام والنسب الشائعة للشرائح**

- **قياسي (نسبة 4:3)**: مثالية للشاشات والأجهزة القديمة.  
- **شاشة واسعة (نسبة 16:9)**: يوصى بها لأجهزة العرض والشاشات الحديثة.

تأكد من الاتساق عبر كامل العرض التقديمي حيث يتم تطبيق حجم شريحة واحد ونسبة عرض إلى ارتفاع واحدة على جميع الشرائح. للحصول على نتائج مثالية، قم بتعيين أبعاد الشريحة في بداية عملية إنشاء العرض التقديمي لتجنب التعقيدات.

{{% alert color="primary" %}} 
بشكل افتراضي، تستخدم العروض التقديمية التي تم إنشاؤها باستخدام Aspose.Slides نسبة 4:3 القياسية.
{{% /alert %}}

## **كيفية تغيير حجم الشريحة في PowerPoint**

يوضح هذا المثال كيفية تغيير حجم شريحة العرض التقديمي باستخدام Aspose.Slides في C#:
```csharp
using (Presentation pres = new Presentation("presentation-4x3.pptx"))
{
    pres.SlideSize.SetSize(SlideSizeType.OnScreen16x9, SlideSizeScaleType.DoNotScale);
    pres.Save("presentation-16x9.pptx", SaveFormat.Pptx);
}
```


## **تحديد أحجام شرائح مخصصة**

تخصيص حجم الشريحة وفقًا لاحتياجاتك الخاصة، مثل تخطيطات الورق الفريدة أو مواصفات الشاشة، يمكن أن يكون مفيدًا. إليك طريقة ضبط حجم شريحة مخصص باستخدام Aspose.Slides for .NET:
```csharp
using (Presentation pres = new Presentation("presentation.pptx"))
{
    pres.SlideSize.SetSize(780, 540, SlideSizeScaleType.DoNotScale); // حجم ورق A4
    pres.Save("presentation-a4.pptx", SaveFormat.Pptx);
}
```


## **معالجة محتوى الشريحة بعد تغيير الحجم**

بعد تغيير الحجم، قد يتشوه محتوى الشرائح. يمكنك التحكم في كيفية إدارة Aspose.Slides لهذا التغيير:

- **`DoNotScale`**: حافظ على الكائنات بحجمها الأصلي لتجنب التحجيم.
- **`EnsureFit`**: قم بتحجيم الكائنات لتناسب الشرائح الأصغر، مما يمنع فقدان المحتوى.
- **`Maximize`**: قم بتوسيع الكائنات لتتناسب مع الشرائح الأكبر لضمان اتساق جمالي.

مثال على استخدام الإعداد `Maximize` لضبط حجم الشريحة:
```csharp
using (Presentation pres = new Presentation("presentation.pptx"))
{
   pres.SlideSize.SetSize(SlideSizeType.Ledger, SlideSizeScaleType.Maximize);
}
```


## **الأسئلة الشائعة**

**هل يمكنني ضبط حجم شريحة مخصص باستخدام وحدات غير البوصة (مثل النقاط أو المليمترات)؟**

نعم. تستخدم Aspose.Slides النقاط داخليًا، حيث يساوي 1 نقطة 1/72 من البوصة. يمكنك تحويل أي وحدة (مثل المليمترات أو السنتيمترات) إلى نقاط واستخدام القيم المحولة لتحديد عرض وارتفاع الشريحة.

**هل سيؤثر حجم شريحة مخصص كبير جدًا على الأداء واستهلاك الذاكرة أثناء التصيير؟**

نعم. تؤدي أبعاد الشريحة الكبيرة (بالنقاط) إلى جانب مقياس التصيير العالي إلى زيادة استهلاك الذاكرة وزيادة أوقات المعالجة. استهدف حجم شريحة عملي وعدّل مقياس التصيير فقط حسب الحاجة لتحقيق جودة الإخراج المطلوبة.

**هل يمكنني تعريف حجم شريحة غير قياسي ثم دمج الشرائح من عروض تقديمية ذات أحجام مختلفة؟**

لا يمكنك [دمج العروض التقديمية](/slides/ar/net/merge-presentation/) عندما تكون لها أحجام شرائح مختلفة — يجب أولاً تغيير حجم أحد العروض ليطابق الآخر. عند تغيير حجم الشريحة، يمكنك اختيار كيفية معالجة المحتوى الموجود عبر خيار [SlideSizeScaleType](https://reference.aspose.com/slides/net/aspose.slides/slidesizescaletype/). بعد موابقة الأحجام، يمكنك دمج الشرائح مع الحفاظ على التنسيق.

**هل يمكنني إنشاء صور مصغرة للأشكال الفردية أو مناطق محددة من الشريحة، وهل ستحترم حجم الشريحة الجديد؟**

نعم. يمكن لـ Aspose.Slides إنشاء صور مصغرة لـ [الشرائح بالكامل](https://reference.aspose.com/slides/net/aspose.slides/slide/getimage/) وكذلك لـ [الأشكال المحددة](https://reference.aspose.com/slides/net/aspose.slides/shape/getimage/). تعكس الصور الناتجة حجم الشريحة الحالي ونسبة العرض إلى الارتفاع، ما يضمن تأطيرًا وتكوينًا متسقًا.
