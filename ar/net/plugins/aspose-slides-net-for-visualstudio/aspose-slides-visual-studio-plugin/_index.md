## **مقدمة**
Microsoft Visual Studio هو بيئة تطوير متكاملة (IDE) من Microsoft. يتم استخدامه لتطوير برامج الكمبيوتر لنظام التشغيل Microsoft Windows، بالإضافة إلى مواقع الويب، وتطبيقات الويب، والخدمات السحابية. يستخدم Visual Studio منصات تطوير البرمجيات من Microsoft مثل Windows API وWindows Forms وWindows Presentation Foundation وWindows Store وMicrosoft Silverlight. يمكن أن ينتج كل من الشيفرة الأصلية والشيفرة المدارية.
### **Aspose.Slides لــ .NET**
Aspose.Slides لــ .NET هو واجهة برمجة تطبيقات إدارة PowerPoint® فريدة تمكّن تطبيقات .NET من قراءة وكتابة والتلاعب بمستندات PowerPoint دون الحاجة إلى استخدام Microsoft PowerPoint.

Aspose.Slides لــ .NET هو أول وأوحد واجهة برمجة تطبيقات PowerPoint لــ .NET التي توفر وظيفة إدارة مستندات PowerPoint ضمن تطبيقاتك الخاصة. كما هو الحال مع جميع مكونات Aspose .NET، تم كتابة Aspose.Slides بلغة C# المدارية، بسعر معقول وسرعة مذهلة.

لمزيد من التفاصيل حول [واجهة برمجة تطبيقات Aspose.Slides لــ .NET](http://www.aspose.com/.net/powerpoint-component.aspx)
### **استكشاف أمثلة Aspose.Slides لــ .NET باستخدام Visual Studio**
ملحق Aspose لــ Visual Studio - **استكشاف أمثلة Aspose.Slides لــ .NET** هو أداة رائعة لتنزيل واستكشاف أمثلة واجهة برمجة تطبيقات Aspose.Slides لــ .NET بسرعة. يوفر الكثير من الوقت والجهود من خلال توفير خيار بسيط للغاية لاختيار وتنزيل وفتح أحدث مشاريع الأمثلة بسهولة دون الحاجة إلى مغادرة Visual Studio الخاص بك.

تقدم هذه النسخة الميزات التالية:

- تدعم Visual Studio 2010 وVisual Studio 2012 وVisual Studio 2013.
- من السهل بدء تشغيله من قائمة **ملف > Aspose** في Visual Studio.
- يوفر الوقت ويقلل من منحنى التعلم.
- يمكّنك من اختيار وفتح أحدث أمثلة واجهة برمجة تطبيقات Aspose.Slides لــ .NET المستضافة على GitHub.com.
- يتيح لك فتح مشروع أمثلة **C#** أو **VB.NET**.
- يتم فتح المشروع المحدد تلقائيًا في Visual Studio مع جميع المراجع المطلوبة لتوفير بيئة جاهزة للتشغيل والاستكشاف.
- يقوم تلقائيًا بتنزيل أحدث المكتبات من واجهة برمجة التطبيقات من **NuGet** وأمثلة من **GitHub**.

يمكنك الحصول على أحدث كود مصدر من أحد المواقع التالية:

- [CodePlex ](https://asposeSlidesvs.codeplex.com/SourceControl/latest)

#### **كيف تر.configureكود المصدر**
تحتاج إلى تثبيت ما يلي من أجل فتح وتوسيع كود المصدر:

- Visual Studio 2010

يرجى اتباع هذه الخطوات البسيطة للبدء:

1. قم بتنزيل/استنساخ كود المصدر.
1. افتح Visual Studio 2010 واختر **ملف** > **فتح مشروع**.
1. تصفح إلى أحدث كود مصدر قم بتنزيله وافتح **AsposeVisualStudioPlugin.sln**.
#### **نظرة عامة على كود المصدر**
هناك مشروعان في الحل:

- **AsposeVisualStudioPlugin** - يحتوي على التكامل مع Visual Studio، واختيار وتنزيل تلقائي لواجهات برمجة تطبيقات Aspose والأمثلة.
- **Installer** - مشروع تثبيت Wix لإنشاء ملف مثبت msi للملحق.
