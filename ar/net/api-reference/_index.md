{{% alert color="primary" %}}

Aspose.Slides for .NET هو مكتبة فئة تمكّن تطبيقات .NET من قراءة وكتابة مستندات العروض التقديمية دون الحاجة إلى Microsoft PowerPoint® أو غيره. يتيح Aspose.Slides for .NET إدارة النصوص، الأشكال، الجداول والرسوم المتحركة، إضافة الصوت والفيديو إلى الشرائح، معاينة الشرائح، وتصدير الشرائح إلى صيغة SVG أو PDF وأكثر.

يمكن العثور على أحدث مرجع API في [مرجع API لـ Aspose.Slides for .NET.](https://reference.aspose.com/slides/net)

{{% /alert %}}
