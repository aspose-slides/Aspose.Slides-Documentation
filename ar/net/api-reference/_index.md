{{% alert color="primary" %}}

Aspose.Slides for .NET هي مكتبة فئات تمكّن تطبيقات .NET من قراءة وكتابة مستندات العروض التقديمية دون الحاجة إلى Microsoft PowerPoint® أو غيرها. تتيح Aspose.Slides for .NET إدارة النصوص والأشكال والجداول والرسوم المتحركة، إضافة الصوت والفيديو إلى الشرائح، معاينة الشرائح، تصدير الشرائح إلى صيغة SVG و PDF وغيرها.

يمكن العثور على أحدث مرجع API في [Aspose.Slides for .NET API Reference.](https://reference.aspose.com/slides/net)

{{% /alert %}}
