{{% alert color="primary" %}} 

Aspose.Slides for .NET هي مكتبة فئات تمكّن تطبيقات .NET من قراءة وكتابة مستندات العروض التقديمية دون الحاجة إلى Microsoft PowerPoint® أو غيره. تسمح Aspose.Slides for .NET بإدارة النصوص والأشكال والجداول والرسوم المتحركة، وإضافة الصوت والفيديو إلى الشرائح، ومعاينة الشرائح، وتصدير الشرائح إلى صيغة SVG أو PDF والمزيد.

يمكن العثور على أحدث مرجع API في [مرجع API لـ Aspose.Slides for .NET.](https://reference.aspose.com/slides/net)

{{% /alert %}}
