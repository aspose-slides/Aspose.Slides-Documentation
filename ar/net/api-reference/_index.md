{{% alert color="primary" %}} 

Aspose.Slides for .NET هي مكتبة فئة تمكّن تطبيقات .NET من قراءة وكتابة مستندات العروض التقديمية دون الحاجة إلى استخدام Microsoft PowerPoint® أو غيره. يسمح Aspose.Slides for .NET بإدارة النصوص والأشكال والجداول والرسوم المتحركة، وإضافة الصوت والفيديو إلى الشرائح، ومعاينة الشرائح، وتصدير الشرائح إلى تنسيق SVG وPDF والمزيد.

يمكن العثور على أحدث مرجع API في [مرجع Aspose.Slides for .NET API](https://reference.aspose.com/slides/ar/net).

{{% /alert %}}
