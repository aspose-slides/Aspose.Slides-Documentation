## **الترخيص في Aspose.Slides**
* تصبح نسخة التقييم مرخصة بعد شراء ترخيص وإضافة بعض الأسطر من الشيفرة لتطبيق الترخيص.
* الترخيص هو ملف XML نصي بسيط يحتوي على تفاصيل مثل اسم المنتج، عدد المطورين المسموح لهم باستخدامه، تاريخ انتهاء الاشتراك، وما إلى ذلك.
* ملف الترخيص موقع رقمياً، لذا لا يجب تعديل الملف. حتى إضافة غير مقصودة لسطر جديد إلى محتويات الملف سيجعله غير صالح.
* عادةً ما يحاول Aspose.Slides for .NET العثور على الترخيص في المواقع التالية:
  * مسار صريح
  * المجلد الذي يحتوي على dll الخاص بالمكوّن (مضمن في Aspose.Slides)
  * المجلد الذي يحتوي على التجميع الذي استدعى dll الخاص بالمكوّن (مضمن في Aspose.Slides)
  * المجلد الذي يحتوي على تجميع الدخول (ملف .exe الخاص بك)
  * مورد مضمن في التجميع الذي استدعى dll الخاص بالمكوّن (مضمن في Aspose.Slides).
* لتجنب القيود المرتبطة بنسخة التقييم، يجب ضبط الترخيص قبل استخدام Aspose.Slides. تحتاج إلى ضبط الترخيص مرة واحدة فقط لكل تطبيق أو عملية.

{{% alert color="primary" %}} 

قد ترغب في الاطلاع على [الترخيص القائم على العداد](https://docs.aspose.com/slides/net/metered-licensing/).

{{% /alert %}} 

## **تطبيق الترخيص**
يمكن تحميل الترخيص من **ملف**، **دفق**، أو **مورد مضمن**. 

{{% alert color="primary" %}}

توفر Aspose.Slides الفئة [License](https://reference.aspose.com/slides/net/aspose.slides/license) لإجراءات الترخيص.

{{% /alert %}} 

{{% alert color="warning" %}} 

يمكن للتراخيص الجديدة تفعيل Aspose.Slides فقط مع الإصدار 21.4 أو أحدث. الإصدارات السابقة تستخدم نظام ترخيص مختلف ولن تتعرف على هذه التراخيص.

{{% /alert %}}

### **ملف**
أسهل طريقة لضبط الترخيص تتطلب وضع ملف الترخيص في نفس المجلد الذي يحتوي على DLL الخاص بالمكوّن (مضمن في Aspose.Slides) وتحديد اسم الملف فقط دون مساره.

هذا كود C# يوضح لك كيفية ضبط ملف ترخيص:
``` csharp
// إنشاء كائن من الفئة License 
Aspose.Slides.License license = new Aspose.Slides.License();

// يضبط مسار ملف الترخيص
license.SetLicense("Aspose.Slides.lic");
```


{{% alert color="warning" %}} 

إذا وضعت ملف الترخيص في دليل مختلف، عند استدعاء طريقة [SetLicense](https://reference.aspose.com/slides/net/aspose.slides/license/setlicense/#setlicense_1)، يجب أن يكون اسم ملف الترخيص في النهاية من المسار الصريح المحدد هو نفسه اسم ملف الترخيص الخاص بك.

على سبيل المثال، يمكنك تغيير اسم ملف الترخيص إلى *Aspose.Slides.lic.xml*. ثم، في كودك، يجب تمرير مسار الملف (المنتهي بـ *Aspose.Slides.lic.xml*) إلى طريقة [SetLicense](https://reference.aspose.com/slides/net/aspose.slides/license/setlicense/#setlicense_1).

{{% /alert %}}

### **دفق**
يمكنك تحميل ترخيص من تدفق. هذا كود C# يوضح لك كيفية تطبيق الترخيص من تدفق:
``` csharp
// إنشاء كائن من الفئة License
Aspose.Slides.License license = new Aspose.Slides.License();

// يضبط الترخيص عبر تدفق
license.SetLicense(myStream);
```


### **مورد مضمن**
يمكنك تضمين الترخيص مع تطبيقك (لتجنب فقدانه) عن طريق إضافة الترخيص كمورد مضمن إلى أحد التجميعات التي تستدعي DLL الخاص بالمكوّن (مضمن في Aspose.Slides).

هذا هو طريقة إضافة ملف الترخيص كمورد مضمن:

1. في Visual Studio، أضف ملف الترخيص (.lic) إلى المشروع بهذه الطريقة: انتقل عبر **File** > **Add Existing Item** > **Add**.
2. حدد الملف في **Solution Explorer**.
3. في نافذة **Properties**، اضبط **Build Action** إلى **Embedded Resource**.
4. للوصول إلى الترخيص المضمن في التجميع، أضف ملف الترخيص كمورد مضمن إلى المشروع، ثم مرّر اسم ملف الترخيص إلى طريقة `SetLicense`.


فئة `License` تجد ملف الترخيص تلقائيًا في الموارد المضمنة. لا تحتاج إلى استدعاء طريقتي `GetExecutingAssembly` و `GetManifestResourceStream` من الفئة `System.Reflection.Assembly` في إطار عمل Microsoft .NET.

هذا كود C# يوضح لك كيفية ضبط ترخيص كمورد مضمن:
``` csharp
// ينشئ كائن من فئة License
Aspose.Slides.License license = new Aspose.Slides.License();

// يمرر اسم ملف الترخيص المضمّن في التجميع
license.SetLicense("Aspose.Slides.lic");
```


## **التحقق من الترخيص**

للتحقق مما إذا تم ضبط الترخيص بشكل صحيح، يمكنك التحقق منه. هذا كود C# يوضح لك كيفية التحقق من الترخيص:
```c#
Aspose.Slides.License license = new Aspose.Slides.License();

license.SetLicense("Aspose.Slides.lic");

if (license.IsLicensed())
{
    Console.WriteLine("License is good!");
    Console.Read();
}
```
