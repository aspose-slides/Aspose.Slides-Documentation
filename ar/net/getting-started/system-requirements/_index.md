## **نظرة عامة**
لا يتطلب Aspose.Slides لـ .NET تثبيت Microsoft PowerPoint لأن Aspose.Slides هو محرك مستقل لإنشاء مستندات Microsoft PowerPoint وتحويلها وتخطيط الصفحات وعرضها.
## **أنظمة التشغيل المدعومة**
يدعم Aspose.Slides لـ .NET أي نظام تشغيل 32 بت أو 64 بت حيث تم تثبيت إطار عمل .NET أو Mono بما في ذلك (ولكن ليس مقتصراً على):
### **ويندوز**
- خادم Microsoft Windows 2000 (x64، x86)
- خادم Microsoft Windows 2003 (x64، x86)
- خادم Microsoft Windows 2022
- Microsoft Windows Vista (x64، x86)
- Microsoft Windows XP (x64، x86)
- Microsoft Windows 7 (x64، x86)
- Microsoft Windows 8، 8.1 (x64، x86)
- Microsoft Windows 10 (x64، x86)
- Microsoft Azure

### **لينكس**
- لينكس (أوبونتو، OpenSUSE، CentOS، وغيرها)

{{%  alert  title="ملاحظات"  color="primary"  %}} 

1. إصدار GLIBC في CentOS 7 هو 2.14، ولكن Aspose.Slides لـ .NET 6 و .NET 7 يتطلب Linux x86_x64 مع GLIBC 2.23 وما فوق. ينطبق هذا الشرط على Aspose.Slides عبر المنصة، والذي يمكن استخدامه فقط لـ .NET 6 و .NET 7 أو أعلى.

2. إذا كان نظامك لا يفي بالمتطلبات في النقطة 1، استخدم Aspose Slides لـ .NETStandard.

{{% /alert %}} 

### **ماك**
- نظام ماك OS X
