## IViewProperties class

Presentation wide view properties.

The IViewProperties type exposes the following members:
## Properties
| Name | Description |
| :- | :- |
|last_view|Specifies the view mode that was used when the presentation document was last saved.<br/>            Read/write [ViewType](/slides/python-net/api-reference/aspose.slides/viewtype/).|
|show_comments|Specifies whether the slide comments should be shown.<br/>            Read/write [NullableBool](/slides/python-net/api-reference/aspose.slides/nullablebool/).|
|slide_view_properties|Specifies common view properties associated with the slide view mode.<br/>            Read-only [ICommonSlideViewProperties](/slides/python-net/api-reference/aspose.slides/icommonslideviewproperties/).|
|notes_view_properties|Specifies common view properties associated with the notes view mode.<br/>            Read-only [ICommonSlideViewProperties](/slides/python-net/api-reference/aspose.slides/icommonslideviewproperties/).|
|normal_view_properties|Represents normal view properties. The normal view consists of<br/>            three content regions: the slide itself, a side content region, and a bottom content region.<br/>            Read-only [INormalViewProperties](/slides/python-net/api-reference/aspose.slides/inormalviewproperties/).|
