## Licenc típusok
{{% alert color="primary" %}} Jelenleg két licenc típus érhető el: Server Subscription Site Subscription {{% /alert %}}
Minden licenc 1 évnyi ingyenes frissítést tartalmaz az új termékverziókra, valamint 1 év korlátlan ingyenes technikai támogatást.

Az Aspose.Slides for Reporting Services értékelő és fizetett verziója ugyanaz a letöltés. Egyszerűen töltse le az Aspose.Slides for Reporting Services-t, telepítse a szerveren, és értékelő módban fog működni.

Győződjön meg arról, hogy kihasználja az Aspose.Slides for Reporting Services értékelő verzióját, mivel annak nincs időkorlátja, és az értékelő felhasználók ingyenes technikai támogatást kapnak.

Az értékelő mód vízjeleket helyez el az exportált jelentésekbe. Amikor [licencet vásárol](https://purchase.aspose.com/buy), egyszerűen másolja a licencet a megfelelő mappába, és az Aspose.Slides for Reporting Services licencelt módban fog működni.
## **Server Subscription**
A Server Subscription a következő funkciókat biztosítja:

- Minden fizikai vagy virtuális szerverhez egy licenc szükséges.
- Korlátlan számú processzor vagy mag.
- Korlátlan számú Microsoft SQL Server példány.
- Korlátlan számú fejlesztő.
- Ingyenes új kiadások és hibajavítások 1 évig.
- Ingyenes technikai támogatás 1 évig.
- Kedvezmény érvényes többszörös vásárlás esetén.

## **Site Subscription**
A Site Subscription a következő funkciókat biztosítja:

- Minden egyetlen fizikai helyszínhez (campus) egy licenc szükséges.
- Korlátlan számú fizikai vagy virtuális szerver.
- Korlátlan számú processzor vagy mag.
- Korlátlan számú Microsoft SQL Server példány.
- Korlátlan számú fejlesztő.
- Ingyenes új kiadások és hibajavítások 1 évig.
- Ingyenes technikai támogatás 1 évig.
- Kedvezmény érvényes többszörös vásárlás esetén.
