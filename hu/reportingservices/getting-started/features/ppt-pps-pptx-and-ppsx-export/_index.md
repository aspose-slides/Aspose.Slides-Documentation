{{% alert color="primary" %}} 

A Microsoft SQL Server Reporting Services nem rendelkezik beépített képességekkel a jelentések Microsoft PowerPoint előadássá exportálására, de ha a szerverére telepíti az Aspose.Slides for Reporting Services-t, hozzáférést kap két további exportformátumhoz: 

- Microsoft PowerPoint bemutató (PPT)
- Microsoft PowerPoint diavetítés (PPS)
- Microsoft PowerPoint 2007 bemutató (PPTX)
- Microsoft PowerPoint 2007 diavetítés (PPSX)

{{% /alert %}} 

Az ebben a formátumban történő dokumentumok létrehozásához az Aspose.Slides for Reporting Services egy beépített változatára támaszkodik az [Aspose.Slides for .NET](http://www.aspose.com/Products/Aspose.Slides/), az Aspose piacvezető prezentációfeldolgozó komponensére. A dokumentumok generálásához nem a Microsoft PowerPointet használja. 


A PPT és PPS másolatai ugyanannak a jelentésnek egymással identikusak lesznek, köszönhetően az Aspose.Slides által biztosított alapul szolgáló fájlformátumok magas fokú támogatásának.
