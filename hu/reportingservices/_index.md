**Üdvözöljük az Aspose.Slides for Reporting Services-ban**

Az Aspose.Slides for Reporting Services az egyetlen megoldás a piacon, amely lehetővé teszi valódi PPT és PPS jelentések generálását a Microsoft SQL Server 2005, 2008, 2012, 2016 és 2017 Reporting Services (32-bit és 64-bit) környezetben. Az összes RDL jelentésfunkció, beleértve a táblázatokat, mátrixokat, diagramokat és képeket, a legmagasabb pontossággal konvertálódik Microsoft PowerPoint prezentációkká.
