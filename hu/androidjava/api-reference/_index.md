{{% alert color="primary" %}} 

Az Aspose.Slides for Android via Java egy osztálykönyvtár, amely lehetővé teszi az Androidra írt Java‑alkalmazások számára, hogy Microsoft PowerPoint® vagy más szoftver használata nélkül olvassanak és írjanak prezentációs dokumentumokat. Az Aspose.Slides for Android via Java lehetővé teszi a szöveg, alakzatok, táblázatok és animációk kezelését; hang és videó hozzáadását a diákhoz; a diák előnézetét; valamint a diák exportálását SVG, PDF formátumba és egyebekbe.

A legújabb API-referencia elérhető a [Aspose.Slides for Android via Java API Reference](https://reference.aspose.com/slides/hu/androidjava/).
