## **Áttekintés**

Ez a cikk leírja a licencelési hibát, amelyet az Aspose.Slides for PHP via Java 23.2, 23.3, 23.4, 23.5, 23.6 és 23.7 verziókban tapasztaltak. Ez a hiba azt eredményezte, hogy a perpetuális licencek átmenetileg használhatatlanná váltak, miután előfizetésük lejárt.

## **Tünetek**

Miután a perpetuális licenc előfizetése lejár, az Aspose.Slides for PHP via Java 23.2 - 23.7 verziók a licencet lejártként jelenthetik, és az összes funkció értékelési módba kerül. Ez a viselkedés helytelen, és a 23.2 előtti és a 23.7 utáni verziókat nem érinti.

## **Megoldás**

Az Aspose.Slides for PHP via Java orvosolta ezt a problémát, és kiadta a frissített verziókat (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) a javítással.

Ha a projektjében az érintett Aspose.Slides for PHP via Java verziók valamelyikét használja, kérjük, váltsa át a javított verzióra.

Ha licencelési problémákat tapasztal az Aspose.Slides for PHP via Java esetén:

- Ellenőrizze, hogy melyik verziót használja, és győződjön meg róla, hogy nem érinti a fent leírt hiba.
- Tekintse meg az [Aspose dokumentációt](https://docs.aspose.com/slides/hu/php-java/getting-started/) a hibaelhárítási lépésekért és a ismert licencelési problémákért.
- Lépjen kapcsolatba az [Aspose támogatással](https://forum.aspose.com/) további segítségért.

## **Érintett termékek és verziók**

Kérjük, vegye figyelembe, hogy ez a hiba csak az Aspose.Slides for PHP via Java 23.2 és 23.7 közötti verziókat érinti. **Más Aspose termékek és az Aspose.Slides más verziói nem érintettek**.
