## **Technical support**
{{% alert color="primary" %}}

Az Aspose korlátlan, ingyenes technikai támogatást nyújt minden termékéhez. A támogatás minden felhasználó számára elérhető (beleértve a kiértékelő csomagok felhasználóit is).

{{% /alert %}} 

Ha segítségre van szüksége az Aspose.Slides for PHP via Java használatához, vegye figyelembe a következőket:

- A fő támogatási csatorna az [Aspose.Forums](https://forum.aspose.com/c/slides/hu/11). Tegye fel kérdését az Aspose.Slides fórumon—és néhány órán belül választ kap. Az Aspose.Slides támogatási csapata közvetlenül válaszol a fórumon közzétett kérdésekre.
- Felhívjuk figyelmét, hogy az Aspose nem nyújt telefonos technikai támogatást. A telefonos támogatás csak értékesítési és vásárlási kérdésekre érhető el.
- A fórumon történő válaszra várva legyen türelmes, és vegye figyelembe az időzóna‑különbségeket.

Ha problémát tapasztal az Aspose.Slides for PHP via Java használata során, kövesse az alábbi irányelveket a leghatékonyabb megoldás érdekében:

- Ellenőrizze és erősítse meg, hogy a legújabb Aspose.Slides for PHP via Java verziót használja a hiba bejelentése előtt. Lehetséges, hogy a tapasztalt problémát már egy frissítés megoldotta.  

  Lásd az [Aspose.Slides download](https://downloads.aspose.com/slides/hu/php-java) oldalt a legújabb verzió információiért.

- A hiba bejelentése előtt tekintse át fórumainkat, ezt a dokumentációt és az API‑referenciát. Lehet, hogy a kérdésére már korábban válasz érkezett. 

- Ha hibát jelent be, kérjük, csatolja az eredeti dokumentumot és (ha lehetséges) a problémához kapcsolódó kódrészletet. Ha több fájlt kell csatolnia, tegye őket egy ZIP csomagba. Biztonságosan csatolhatja dokumentumait az Aspose.Forums‑on, mivel **csak** Ön és az Aspose fejlesztői férnek hozzá a csatolt fájlokhoz.

- Hozzon létre és használjon egy szálat egy adott problémához. Ha másik problémát tapasztal, egy új szálat kell létrehoznia a jelentéshez.
