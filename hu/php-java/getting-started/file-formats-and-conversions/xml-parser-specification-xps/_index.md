{{% alert color="primary" %}} 

Az [XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) egy oldal leíró nyelv és egy fix dokumentumformátum, amelyet eredetileg a Microsoft fejlesztett ki. A PDF-hez hasonlóan az XPS a dokumentum hűségét kívánja megőrizni, és eszközfüggetlen dokumentummegjelenést biztosít. 

{{% /alert %}} 

## **XPS az Aspose.Slides for PHP via Java-ban**
Bármely, az Aspose.Slides for PHP via Java által betölthető bemutató dokumentum átalakítható XPS formátumba. Az Aspose.Slides for PHP via Java a nagy pontosságú oldalelrendezési és renderelőmotorját használja, hogy fix elrendezésű XPS dokumentumformátumban állítson elő kimenetet.
A prezentációs dokumentumok XPS dokumentumokká exportálásáról az Aspose.Slides for PHP via Java segítségével a [Átalakítás XPS-re](https://docs.aspose.com/slides/hu/php-java/convert-powerpoint-to-xps/) cikkben olvashat.
