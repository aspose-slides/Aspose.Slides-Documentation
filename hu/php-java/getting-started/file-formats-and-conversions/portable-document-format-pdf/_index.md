{{% alert color="primary" %}} 

A [Hordozható Dokumentum Formátum](https://en.wikipedia.org/wiki/PDF) egy Adobe Systems által létrehozott fájlformátum a dokumentumok szervezetek közötti cseréjére. A formátum célja, hogy a tartalom és a megjelenés azonos maradjon, függetlenül attól, hogy melyik platformon tekintik meg. Az Aspose.Slides for PHP via Java lehetővé teszi, hogy prezentációs fájlokat PDF‑be konvertáljon.

{{% /alert %}} 

## **PDF az Aspose.Slides for PHP via Java segítségével**
Bármely prezentáció, amely betölthető az Aspose.Slides for PHP via Java‑ba, konvertálható olyan PDF‑be, amely megfelel a [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) vagy [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA) szabványnak, az Ön választása szerint. Az Aspose.Slides for PHP via Java exportálja a prezentációkat PDF‑be, és a legtöbb esetben a kimeneti PDF pontosan úgy néz ki, mint az eredeti prezentáció.

Az Aspose.Slides a következő prezentációs elemeket támogatja a PDF‑be konvertálás során:

- Képek, szövegdobozok és egyéb alakzatok.
- Szöveg és formázás.
- Bekezdések és formázás.
- Hiperhivatkozások.
- Fejlécek és láblécek.
- Listaelemek.
- Táblázatok.

A prezentációk közvetlenül exportálhatók PDF‑be az Aspose.Slides for PHP via Java segítségével: nincs szükség másik komponensre. Továbbá a prezentáció PDF‑exportálását testreszabhatja különböző beállításokkal, ahogyan azt a [Converting to PDF](/slides/hu/php-java/convert-powerpoint-to-pdf/) című dokumentációban leírják.
