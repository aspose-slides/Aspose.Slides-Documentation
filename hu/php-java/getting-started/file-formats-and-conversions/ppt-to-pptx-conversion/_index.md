A PPT‑ről PPTX‑re konvertálást támogatja az Aspose.Slides for PHP via Java. A legtöbb prezentációs funkció – mester diák, felépítés és így tovább – megmarad a formátumok közti átalakításnál, de vannak [néhány korlátozás](/slides/hu/php-java/ppt-to-pptx-conversion/).

{{% /alert %}} 
## **A konverzió támogatott funkciói**
Aspose.Slides for PHP via Java részleges támogatást nyújt a PPT fájlformátum PPTX‑re történő konvertálásához. A konverzió támogatása csak most került bevezetésre az Aspose.Slides for PHP via Java‑ban, ezért vannak néhány korlátozása, és leginkább egyszerű prezentációk esetén működik a legjobban. A fő előnye az Aspose.Slides for PHP via Java‑nak a PPT‑ről PPTX‑re történő konvertálás során, hogy az API rendkívül könnyen használható. A kódpéldák megtekintéséhez olvassa el a [PPT konvertálása PPTX‑re](). Az alábbi listák leírják, mely funkciók támogatottak és melyek nem a PPT‑ről PPTX‑re konverziónál.

## **Támogatott funkciók**
A következő funkciók támogatottak a konverzióhoz:

- A mesterek, elrendezések és diák struktúrájának konvertálása.
- Diagramok konvertálása.
- Csoport alakzatok.
- Automatikus alakzatok (közülük téglalapok és ellipszisek) konvertálása.
- Egyedi geometriájú alakzatok.
- Textúrák és képek kitöltési stílusa az automatikus alakzatokhoz.
- Helyfoglalók konvertálása.
- Vonalak és polivonalak konvertálása.
- Vonal és kitöltés formátumok.
- Színátmenetes kitöltési stílusok.
- OLE keretek, táblázatok, videó- és hangsáv keretek stb.
- Animáció és diavetítés tulajdonságok.
- Szöveg konvertálása szövegkeretekben és szöveghelyeken.
