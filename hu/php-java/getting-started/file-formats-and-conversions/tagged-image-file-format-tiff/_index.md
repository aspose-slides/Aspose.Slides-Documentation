## **A TIFF-ről**

A [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) amely több képet képes tárolni egy dokumentumban, eredetileg az Aldus fejlesztette. Ez a formátum széles körben támogatott a szkennelés, faxolás és egyéb képmanipulációs alkalmazások által.

## **TIFF az Aspose.Slides for PHP via Java-ban**

Bármely dokumentum, amely betölthető az Aspose.Slides for Java-ban, közvetlenül konvertálható TIFF-dokumentummá az Aspose.Slides for PHP via Java segítségével, ezzel kiküszöbölve bármely harmadik fél komponensre vonatkozó követelményt. Továbbá, opcionálisan meghatározhatja a képek méretét a létrejövő TIFF-dokumentumban. Az Aspose.Slides for PHP via Java használatával történő prezentációs dokumentumok TIFF-dokumentummá exportálásával kapcsolatban információt találhat [ebben a témában](/slides/hu/php-java/convert-powerpoint-to-tiff/).
