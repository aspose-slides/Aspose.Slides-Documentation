{{% alert color="primary" %}} 

[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) a prezentációs dokumentumfájl formátuma, amelyet a Microsoft PowerPoint különböző verziói hozhatnak létre, olvashatnak, módosíthatnak és írhatnak. Ez a Microsoft által fejlesztett bináris formátum a prezentációs dokumentumokhoz.

{{% /alert %}} 

## **PPT az Aspose.Slides for PHP via Java-ban**
Az Aspose.Slides for PHP via Java képes olvasni az alább felsorolt szoftverek által létrehozott PPT fájlokat.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Hasonló módon az Aspose.Slides for PHP via Java által létrehozott PPT fájlokat is olvashatja a fenti szoftverek valamelyike.

## **Átfogó támogatás a PPT-hez**
Az Aspose.Slides for PHP via Java támogatja a PPT dokumentumformátum szinte összes funkcióját. Nemcsak a különböző Microsoft PowerPoint verziók által nyújtott alapvető és fejlett funkciókat fedi le a PPT dokumentumok manipulálásához, hanem olyan funkciókat is, amelyeket a Microsoft PowerPoint egyáltalán nem támogat. Az Aspose.Slides for PHP via Java API könyvtár használatának fő előnye a könnyű kezelhetőség ezeknek a funkcióknak a felhasználásakor.

Az alapvető PPT dokumentumok létrehozásával, olvasásával és írásával kapcsolatos feladatokon túl az Aspose.Slides for PHP via Java a következő funkciókat is nyújtja:

- Más Microsoft Office fájlformátumok importálása [OLE objektumként a PPT dokumentumokba](/slides/hu/php-java/manage-ole/).
- [PPT dokumentumok exportálása PDF-be](/slides/hu/php-java/convert-powerpoint-to-pdf/).
- Diák exportálása SVG formátumba a PPT dokumentumokban.
- Diák renderelése bármely, a Java keretrendszer által támogatott képformátumba.
- Diák méretének beállítása a PPT dokumentumokban.
- Animációk kezelése alakzatokon.
- Diavetítések kezelése.
- [Szöveg formázása a diákon](/slides/hu/php-java/text-formatting/).
- Szöveg kinyerése PPT dokumentumokból.
- [Táblázatok kezelése a diákon](/slides/hu/php-java/powerpoint-table/).
- Masterek automatikus másolása a [klónozási funkció](/slides/hu/php-java/clone-slides/) segítségével.
