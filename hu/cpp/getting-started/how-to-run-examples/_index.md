## **Szoftverkövetelmények**
Kérjük, győződjön meg arról, hogy megfelel az alábbi követelményeknek a példák letöltése és futtatása előtt.

1. Visual Studio 2015 vagy újabb.
## **Aspose.Slides letöltése a GitHubról**
Az Aspose.Slides for C++ összes példája a [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-C) oldalon érhető el.

- Klónozhatja a tárolót a kedvenc GitHub kliensével, vagy letöltheti a ZIP-fájlt innen: [itt](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/refs/heads/master.zip).
- Csomagolja ki a ZIP-fájl tartalmát egy tetszőleges mappába a számítógépén. Az összes példa a **Examples** mappában található.
- Van egy C++ Visual Studio megoldásfájl.
- A projektek a Visual Studio 2015-ben lettek létrehozva, de a megoldásfájlok kompatibilisek a Visual Studio 2015-tel és újabb verziókkal.
- Nyissa meg a megoldásfájlt a Visual Studio-ban, és építse fel a projektet.
- Töltse le az Aspose.Slides for C++ legújabb kiadását a [Letöltési szekció](https://downloads.aspose.com/slides/hu/cpp) oldalról.
- Csomagolja ki a kiadást, és másolja a **Include** és **Bin** könyvtárakat a **Examples** mappába.
- Nyissa meg a **main.cpp** fájlt, innen hívódnak meg az összes példa.
- Törölje a kommenteket azokból a példákból, amelyeket futtatni szeretne a projektben.
- Építse és fordítsa le a megoldást.

Ha bármilyen problémája van a példák beállításával vagy futtatásával kapcsolatban, kérjük, forduljon fórumainkhoz.
## **Közreműködés**
Ha szeretne példát hozzáadni vagy javítani, ösztönözzük, hogy közreműködjön a projektben. Az ebben a tárolóban található összes példa és kiállítási projekt nyílt forráskódú, és szabadon felhasználható saját alkalmazásaiban.

A közreműködéshez leágazhatja a tárolót, szerkesztheti a forráskódot, és pull requestet hozhat létre. Átnézzük a módosításokat, és ha hasznosnak találjuk, beépítjük a tárolóba.
