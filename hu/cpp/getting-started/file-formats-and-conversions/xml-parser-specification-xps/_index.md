## **Az XPS-ről**
Az [XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) egy oldal leíró nyelv és egy rögzített dokumentumformátum, amelyet eredetileg a Microsoft fejlesztett ki. A PDF-hez hasonlóan az XPS egy rögzített elrendezésű dokumentumformátum, amelyet a dokumentum hűségének megőrzésére és az eszközfüggetlen megjelenés biztosítására terveztek.

## **Az XPS az Aspose.Slides for C++-ban**
Bármely prezentációs dokumentum, amelyet az Aspose.Slides for C++ be tud tölteni, átalakítható XPS formátumba. Az Aspose.Slides for C++ a magas hűségű oldalelrendezési és renderelő motorját használja, hogy rögzített elrendezésű XPS dokumentumformátumban állítson elő kimenetet.
Érdemes megemlíteni, hogy az Aspose.Slides for C++ közvetlenül generál XPS-t anélkül, hogy a C++ Framework 3.5-vel szállított Windows Presentation Foundation (WPF) osztályokra támaszkodna, így az Aspose.Slides for C++ képes XPS dokumentumokat előállítani olyan gépeken is, ahol a C++ Framework verziója korábbi, mint a 3.5. A prezentációs dokumentumok XPS formátumba történő exportálásáról az Aspose.Slides for C++ segítségével a [ez a téma](/slides/hu/net/) című témában olvashat.
