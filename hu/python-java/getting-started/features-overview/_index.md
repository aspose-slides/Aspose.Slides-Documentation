## **Támogatott platformok**
Az Aspose.Slides for Python via Java egy platformfüggetlen API. Bármilyen platformon (Windows, Linux, macOS stb.) használható, ahol a Python 3.7 vagy frissebb verziója telepítve van. A JDK 8 vagy újabb verziója szintén szükséges.

## **Fájlformátumok és konvertálások**
Az Aspose.Slides for Python via Java a legtöbb PowerPoint dokumentumformátumot támogatja. Lehetővé teszi, hogy azokat a szervezetek által használt és megosztott népszerű fájlformátumokba exportálja. Tekintse meg az alábbi részleteket:

|**Funkció**|**Leírás**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Az Aspose.Slides for Python via Java biztosítja a leggyorsabb feldolgozást ehhez a prezentációs dokumentumformátumhoz.|
|PPT → PPTX konvertálás|Az Aspose.Slides for Python via Java támogatja a PPT‑t PPTX‑re konvertálást.|
|Portable Document Format (PDF)|Egyetlen metódussal exportálhatja az összes támogatott fájlformátumot az Adobe Portable Document Format (PDF) dokumentumokba.|
|XML Parser Specification (XPS)|Egyetlen metódussal exportálhatja az összes támogatott fájlformátumot az XML Parser Specification (XPS) dokumentumokba.|
|Tagged Image File Format (TIFF)|Exportálhatja az összes támogatott prezentációs fájlformátumot a Tagged Image File Format (TIFF) formátumba.|
|PPTX HTML konvertálás|Az Aspose.Slides for Python via Java támogatja a PresentationEx HTML formátumba konvertálását.|

## **Renderelés és nyomtatás**
Az Aspose.Slides for Python via Java magas hűségű renderelést biztosít a prezentációs diák különböző grafikus formátumokba. Tekintse meg az alábbi részleteket:

|**Funkció**|**Leírás**|
| :- | :- |
|Java által támogatott képfájlformátumok|Az Aspose.Slides for Python via Java segítségével renderelheti a prezentációs diákat és a diákon lévő képeket minden Java által támogatott grafikus formátumba, például TIFF, PNG, BMP, JPEG, GIF.|
|SVG formátum|Az Aspose.Slides for Python via Java beépített metódusokat is biztosít, amelyek lehetővé teszik a prezentációs diák exportálását Scalable Vector Graphics (SVG) formátumokba.|
|Prezentáció nyomtatása|Az Aspose.Slides for Python via Java legújabb verziói beépített nyomtatási metódusokat kínálnak különféle beállításokkal.|

## **Tartalom funkciók**
Az Aspose.Slides for Python via Java lehetővé teszi a prezentációs dokumentumok szinte minden elemének vagy tartalmának elérését, módosítását vagy létrehozását. Tekintse meg az alábbi részleteket:

|**Funkció**|**Leírás**|
| :- | :- |
|Mester diák|A mester diák határozzák meg a normál diák elrendezését. Az Aspose.Slides for Python via Java lehetővé teszi a prezentációs dokumentumok mester diáinak elérését és módosítását.|
|Normál diák|Az Aspose.Slides for Python via Java segítségével különböző típusú új diák létrehozhatók; emellett elérheti és módosíthatja a meglévő diákot a prezentációkban.|
|Diák klónozása / másolása|Az Aspose.Slides for Python via Java beépített metódusai lehetővé teszik egy prezentáción belül a meglévő diák klónozását vagy másolását. A másolt és klónozott diákot egy prezentációból egy másikba is felhasználhatja. Mivel egy dia a mester diáktól örökli az elrendezést, a beépített klónozási metódusok automatikusan másolják a mestert a klónozás során.|
|Diák szekcióinak kezelése|Metódusok a diák különböző szekciókba rendezéséhez a prezentáción belül.|
|Helyőrzők és szöveghelyőrzők|Elérheti a diák helyőrzőit és szöveghelyőrzőit. Ezen felül egy diát szöveghelyőrzőkkel teljesen az elejétől létrehozhat a megfelelő metódussal.|
|Élőfejek és élőlábak|Az Aspose.Slides for Python via Java megkönnyíti az élőfejek/élőlábak kezelését a diákon.|
|Jegyzetek a diákon|Az Aspose.Slides for Python via Java segítségével elérheti és módosíthatja a diákhoz tartozó jegyzeteket, valamint új jegyzeteket is hozzáadhat.|
|Alakzat keresése|Egy adott alakzatot a dián a hozzá tartozó alternatív szöveg alapján is megtalálhatja.|
|Háttérképek|Az Aspose.Slides for Python via Java lehetővé teszi a mester vagy normál diához tartozó háttérképek kezelését a prezentációban.|
|Szövegdobozok|Szövegdobozok teljesen az elejétől létrehozhatók. Elérheti a már meglévő szövegdobozokat. A szövegek módosítása közben az eredeti formázás megmarad.|
|Téglalap alakzatok|Az Aspose.Slides for Python via Java segítségével téglalap alakzatokat hozhat létre vagy módosíthat.|
|Polivonal alakzatok|Az Aspose.Slides for Python via Java segítségével polivonal alakzatokat hozhat létre vagy módosíthat.|
|Ellipszis alakzatok|Az Aspose.Slides for Python via Java segítségével ellipszis alakzatokat hozhat létre vagy módosíthat.|
|Csoportosított alakzatok|Az Aspose.Slides for Python via Java támogatja a csoportosított alakzatokat.|
|Auto alakzatok|Az Aspose.Slides for Python via Java támogatja az auto alakzatokat.|
|SmartArt|Az Aspose.Slides for Python via Java támogatja a SmartArt alakzatokat az MS PowerPointban.|
|Diagramok|Az Aspose.Slides for Python via Java támogatja az MSO diagramokat a PowerPointban.|
|Alakzatok sorosítása|Az Aspose.Slides for Python via Java számos alakzatot támogat. Ha egy alakzat nincs támogatva, egy sorosítási módszerrel sorosíthatja azt egy meglévő diáról. Így a alakzatot a továbbiakban igényei szerint felhasználhatja.|
|Képkeretek|Az Aspose.Slides for Python via Java segítségével képeket kezelhet képkeretekben.|
|Audio keretek|Az Aspose.Slides for Python via Java segítségével audio fájlokat linkelhet vagy beágyazhat audio keretekbe a diákon.|
|Video keretek|Kezelheti a videó fájlokat a video keretekben. Az Aspose.Slides for Python via Java támogatja a hivatkozott és beágyazott videókat is.|
|OLE keret|Az Aspose.Slides for Python via Java segítségével OLE objektumokat kezelhet OLE keretekben.|
|Táblázatok|Az Aspose.Slides for Python via Java táblázatokat támogat a diákon.|

|VBA makrók|VBA makrók kezelése a prezentációkban.|
|Szövegkeret|A szöveget bármely alakzatról elérheti az adott alakzathoz tartozó szövegkereten keresztül.|
|Szöveg beolvasása|A beépített beolvasási metódusokkal prezentáció vagy diaszinten szövegeket olvashat be.|
|Animációk|Alakzatokra animációkat alkalmazhat.|
|Diavetítések|Az Aspose.Slides for Python via Java támogatja a diavetítéseket és diák áttűnéseit.|

## **Formázási funkciók**
Az Aspose.Slides for Python via Java segítségével formázhat szövegeket és alakzatokat a prezentációk diáin. Tekintse meg az alábbi részleteket:

|**Funkció**|**Leírás**|
| :- | :- |
|Szöveg formázása|<p>Az Aspose.Slides for Python via Java segítségével a szövegeket az alakzatokhoz tartozó szövegkeretekben kezelheti. Így a szövegeket a szövegkeretekhez tartozó bekezdések és részek segítségével formázhatja. Ezeket a szövegelemeket az Aspose.Slides for Python via Java segítségével formázhatja.</p><p>- Betűtípus</p><p>- Betűméret</p><p>- Betűszín</p><p>- Betűárnyalatok</p><p>- Bekezdés igazítás</p><p>- Bekezdés felsorolás</p><p>- Bekezdés tájolás</p>|
|Alakzat formázása|<p>Az Aspose.Slides for Python via Java-ban egy dia alapvető eleme egy alakzat. Ezeket az alakzat elemeket formázhatja az Aspose.Slides for Python via Java segítségével:</p><p>- Pozíció</p><p>- Méret</p><p>- Vonal</p><p>- Kitöltés (beleértve a mintát, színátmenetet, egyenletes kitöltést)</p><p>- Szöveg</p><p>- Kép</p>|
