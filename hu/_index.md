## Aspose.Slides for .NET

{{% alert color="primary" %}}

![Aspose.Slides for .NET terméklogó](home_1.png)

Aspose.Slides for .NET egy Microsoft PowerPoint® kezelő API, amely lehetővé teszi a .NET alkalmazások számára PowerPoint® dokumentumok beolvasását és írását a Microsoft PowerPoint® használata nélkül. Az Aspose.Slides for .NET az első és egyetlen komponens, amely a PowerPoint® dokumentumok kezelésére szolgáló funkciókat biztosítja. Az Aspose.Slides for .NET számos kulcsfontosságú funkciót kínál, például szöveg, alakzatok, táblázatok és animációk kezelését, audio és videó hozzáadását a diához, diák előnézetét, SVG‑ és PDF‑formátumba történő exportálást, és még sok mást.
