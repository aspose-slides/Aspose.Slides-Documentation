Megjelent egy új [Aspose.Slides for Java API](https://docs.aspose.com/slides/hu/java/) , és most ez a termék támogatja a PowerPoint dokumentumok előállítását nulláról, valamint a meglévő dokumentumok szerkesztését.

{{% /alert %}} 
## **Örökségi kód támogatása**
Hogy az Aspose.Slides for Java 14.x.x előtti verziókkal készült örökségi kódot használhassa, néhány kisebb módosítást kell végeznie a kódban, és a kód úgy fog működni, mint korábban. Az összes osztály, amely korábban az Aspose.Slides for Java régi változatában a com.aspose.slides és a com.aspose.slides.pptx névtérben volt, most egyetlen com.aspose.slides névtérbe van összevonva. Tekintse meg az alábbi egyszerű kódrészletet, amely egy normál diagramot hoz létre a prezentációban nulláról az örökségi Aspose.Slides API használatával, és kövesse a lépéseket, amelyek leírják, hogyan költözzön az új összevont API-ra.
