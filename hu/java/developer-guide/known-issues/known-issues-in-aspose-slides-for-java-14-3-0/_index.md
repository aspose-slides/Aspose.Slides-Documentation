Az Aspose.Slides for Java 14.3.0 (14.4.0) teljesen új PPT feldolgozási megvalósítást biztosít. Számos fejlesztés történt, részleges PPTX‑ről PPT‑re konverzió is elérhető. Azonban néhány meg nem valósított funkció van:

- Néhány alakzat helytelen geometriával rendelkezik a sorosított PPT dokumentumokban (feliratok)
- Nem minden PPTX szövegformázási funkció támogatott a PPT szerializálásban
- A szöveg nyelvéről és helyesírási beállításairól szóló információ nem található a sorosított PPT dokumentumokban
- Nem minden PPTX téma funkció támogatott a PPT szerializálásban

**Néhány különbség van az Aspose.Slides for Java 8.6.0-hoz képest:**

- Ismert problémák vannak az OLE/ActiveX PPT‑k szerializálásában PPT‑be

**Néhány különbség van az Aspose.Slides for .NET 14.3.0-hoz képest:**

- A prezentáció nyomtatási támogatása jelenleg nem érhető el az Aspose.Slides for Java számára
