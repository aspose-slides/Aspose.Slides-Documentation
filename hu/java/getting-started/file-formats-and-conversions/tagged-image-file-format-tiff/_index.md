## **A TIFF-ről**

A Tagged Image File Format, amely több kép tárolására képes egy dokumentumban, eredetileg az Aldus fejlesztette. Ezt a formátumot széles körben támogatják a szkennelő, faxoló és egyéb képfeldolgozó alkalmazások.

## **A TIFF az Aspose.Slides for Java-ban**

Bármely dokumentum, amely betölthető az Aspose.Slides for Java-ban, közvetlenül konvertálható TIFF dokumentummá az Aspose.Slides for Java segítségével, anélkül, hogy harmadik fél komponensére lenne szükség. Továbbá, opcionálisan meghatározhatja a kép méretét a létrejövő TIFF dokumentumban. Az Aspose.Slides for Java segítségével történő prezentációs dokumentumok TIFF formátumba exportálásáról információt a [ebben a témában](/slides/hu/java/converting-a-presentation/) talál.
