{{% alert color="primary" %}} 

Az [XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) egy oldalleíró nyelv és egy rögzített-dokumentum formátum, amelyet eredetileg a Microsoft fejlesztett ki. A PDF-hez hasonlóan az XPS is arra tervezték, hogy megőrizze a dokumentum hűségét és eszközfüggetlen megjelenést biztosítson. 

{{% /alert %}} 

## **XPS az Aspose.Slides for Java-ban**
Bármely, az Aspose.Slides for Java által betölthető prezentációs dokumentum átalakítható XPS formátumba. Az Aspose.Slides for Java a magas hűségű oldalelrendezési és renderelő motorját használja, hogy rögzített elrendezésű XPS dokumentumformátumban állítsa elő a kimenetet.
Az XPS formátumba történő prezentációk exportálásáról bővebben a [Converting to XPS](https://docs.aspose.com/slides/hu/java/convert-powerpoint-to-xps/) oldalon olvashat az Aspose.Slides for Java dokumentációban.
