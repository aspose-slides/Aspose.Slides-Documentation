{{% alert color="primary" %}} 

A [Portable Document Format](https://en.wikipedia.org/wiki/PDF) egy az Adobe Systems által létrehozott fájlformátum a szervezetek közötti dokumentumcseréhez. A formátum célja az volt, hogy a tartalom és a megjelenés azonos maradjon, függetlenül attól, hogy melyik platformon tekintik meg. Az Aspose.Slides for Java lehetővé teszi, hogy a prezentációs fájlokat PDF-be konvertálja.

{{% /alert %}} 

## **PDF az Aspose.Slides for Java-ban**
Az Aspose.Slides for Java-ba betölthető minden prezentáció PDF formátumba konvertálható, amely megfelel a [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A), a [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A), a [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) vagy a [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA) szabványnak, a választásától függően. Az Aspose.Slides for Java exportálja a prezentációkat PDF-be, és a legtöbb esetben a kimeneti PDF pontosan úgy néz ki, mint az eredeti prezentáció.

Az Aspose.Slides a következő prezentációs funkciókat támogatja PDF konvertálásakor:

- Képek, szövegdobozok és egyéb alakzatok.
- Szöveg és formázás.
- Bekezdések és formázás.
- Hiperhivatkozások.
- Fejlécek és láblécek.
- Felsorolások.
- Táblázatok.

Az Aspose.Slides for Java segítségével közvetlenül exportálhatja a prezentációkat PDF‑ekbe: nem szükséges más komponens. Továbbá a prezentáció PDF‑exportálását testreszabhatja különféle beállításokkal, ahogyan a [PDF konvertálás](/slides/hu/java/converting-a-presentation/) részben le van írva.
