## **Támogatott platformok**
Aspose.Slides for Java támogatja a legnépszerűbb fejlesztési és telepítési platformokat.

|**Funkció**|**Leírás**|
| :- | :- |
|Asztali alkalmazások|Aspose.Slides for Java használható Windows Forms alkalmazások fejlesztésére|
|Vállalati webalkalmazások|Az Aspose.Slides for Java segít webalkalmazások építésében. Támogatás biztosított az Aspose.Slides for Java PHP-vel való használatához is.|
|Linux/Unix|Az Aspose.Slides for Java platformfüggetlen API, amely Linux és Unix környezetben is működik.|

## **Fájlformátumok és konverziók**
Aspose.Slides for Java támogatja a Microsoft PowerPoint dokumentumformátumok többségét, és exportálja őket a szervezetek által széles körben használt formátumokba.

|**Funkció**|**Leírás**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/hu/java/microsoft-powerpoint-ppt/)|Az Aspose.Slides for Java a leggyorsabb feldolgozást biztosít ehhez a prezentációs dokumentumformátumhoz.|
|[PresentationML (PPTX, XML)](/slides/hu/java/presentationml-pptx-xml/)|Az Aspose.Slides for Java támogatja az OOXML prezentációs formátum feldolgozását (más néven PresentationML vagy PPTX).|
|[PPT → PPTX konverzió](/slides/hu/java/ppt-to-pptx-conversion/)|Az Aspose.Slides for Java támogatja a PPT-ről PPTX-re konvertálást.|
|[Portable Document Format (PDF)](/slides/hu/java/developer-guide/)|A támogatott fájlformátumok egyetlen metódussal exportálhatók az Adobe Portable Document Format (PDF) dokumentumokba.|
|[XML Parser Specification (XPS)](/slides/hu/java/xml-parser-specification-xps/)|Az összes támogatott fájlformátum egyetlen metódussal exportálható XML Parser Specification (XPS) dokumentumokba.|
|[Tagged Image File Format (TIFF)](https://docs.aspose.com/slides/hu/java/convert-powerpoint-to-tiff/)|Az Aspose.Slides for Java által támogatott prezentációs fájlformátumok szintén exportálhatók Tagged Image File Format (TIFF) formátumba.|
|[ODP → PPTX konverzió](https://docs.aspose.com/slides/hu/java/convert-odp-to-pptx/)|Az Aspose.Slides for Java támogatja az OpenDocument Presentation (ODP) betöltését és konvertálását PPTX-be.|
|[Prezentáció HTML konvertálása](https://docs.aspose.com/slides/hu/java/convert-powerpoint-to-html/)|Az Aspose.Slides for Java támogatja a PresentationEx HTML formátumba történő konvertálását.|

## **Prezentáció renderelése**
Aspose.Slides for Java támogatja a diák magas hűségű renderelését a prezentációkban különböző grafikus formátumokba:

|**Funkció**|**Leírás**|
| :- | :- |
|Támogatott képformátumok|Az Aspose.Slides for Java segítségével nem csak a prezentációs diák, hanem a diákon lévő képek is renderelhetők a népszerű, támogatott grafikus formátumokba, például TIFF, PNG, BMP, JPEG, GIF és metafájlok.|
|SVG formátum|Az Aspose.Slides for Java beépített metódust biztosít a prezentációs diák Scalable Vector Graphics (SVG) formátumba exportálásához.|

## **GYIK**

**Szükséges-e a Microsoft PowerPoint telepítése a szerveren/PC-n a könyvtár működéséhez?**

Nem. A PowerPoint nem szükséges; az Aspose.Slides egy önálló motor a prezentációk létrehozásához, szerkesztéséhez, konvertálásához és rendereléséhez.

**Hogyan működik a több szálas feldolgozás? Párhuzamosítható a feldolgozás?**

Biztonságos különböző dokumentumok feldolgozása különálló szálakon; ugyanazt a [prezentáció](https://reference.aspose.com/slides/hu/java/com.aspose.slides/presentation/) objektumot nem szabad egyszerre [több szál](/slides/hu/java/multithreading/) használja.

**Támogatottak-e a fájl jelszavak és a titkosítás?**

Igen. [Megnyithatja](/slides/hu/java/password-protected-presentation/) a titkosított prezentációkat, beállíthat vagy eltávolíthat megnyitási és írási jelszót, valamint ellenőrizheti a védettségi státuszt.

**Gondoskodnom kell-e a betűcsomagokról Linux konténerekben?**

Igen. Ajánlott közös betűcsomagokat telepíteni és/vagy kifejezetten [adja meg a betűkészlet könyvtárakat](/slides/hu/java/custom-font/) az alkalmazásában, hogy elkerülje a váratlan helyettesítéseket.

**Vannak-e korlátozások az értékelő verzióban?**

Az [értékelő mód](/slides/hu/java/licensing/) esetén vízjel kerül a kimenetre, és bizonyos korlátozások érvényesek; egy [30 napos ideiglenes licenc](https://purchase.aspose.com/temporary-license/) érhető el a teljes funkcionalitás teszteléséhez.

**Támogatott-e külső formátumok importálása egy prezentációba (PDF/HTML → PPTX)?**

Igen. Hozzáadhatja a [PDF oldalak és HTML tartalom](/slides/hu/java/import-presentation/) egy prezentációhoz, ezáltal diaokká alakítva.
