## **Támogatott platformok**
Az Aspose.Slides for Node.js via Java egy platformfüggetlen API. Bármilyen platformon (Windows, Linux, macOS stb.) használható, ahol a Node.js 7 vagy újabb verziója telepítve van. Az Oracle JDK 8 vagy újabb szintén kötelező.

## **FAQ**

**Szükséges-e a Microsoft PowerPoint telepítése a szerveren/PC-n a könyvtár működéséhez?**

Nem. A PowerPoint nem szükséges; az Aspose.Slides egy önálló motor a prezentációk létrehozásához, szerkesztéséhez, konvertálásához és rendereléséhez.

**Hogyan működik a több szálú feldolgozás? Párhuzamosítható a feldolgozás?**

Biztonságosan feldolgozhat különböző dokumentumokat külön szálakon; ugyanazt a [presentation](https://reference.aspose.com/slides/hu/nodejs-java/aspose.slides/presentation/) objektumot nem szabad [multiple threads](/slides/hu/nodejs-java/multithreading/) egyszerre használni.

**Támogatottak-e a fájl jelszavak és a titkosítás?**

Igen. [Megnyithat](/slides/hu/nodejs-java/password-protected-presentation/) titkosított prezentációkat, beállíthat vagy eltávolíthat nyitási és írási jelszót, valamint ellenőrizheti a védelem állapotát.

**Szükséges-e a betűkészletek kezelése Linux konténerekben?**

Igen. Ajánlott a közös betűkészletek telepítése és/vagy a [betűkészlet könyvtárak megadása](/slides/hu/nodejs-java/custom-font/) kifejezett megadása az alkalmazásban a váratlan helyettesítések elkerülése érdekében.

**Vannak-e korlátozások az értékelő verzióban?**

Az [értékelő mód](/slides/hu/nodejs-java/licensing/) esetén vízjel kerül a kimenetre, és bizonyos korlátozások léteznek; egy [30‑napos ideiglenes licenc](https://purchase.aspose.com/temporary-license/) elérhető a teljes funkcionalitású teszteléshez.

**Támogatott-e külső formátumok importálása egy prezentációba (PDF/HTML → PPTX)?**

Igen. Hozzáadhat [PDF oldalak és HTML tartalom](/slides/hu/nodejs-java/import-presentation/) egy prezentációhoz, átalakítva őket diákra.
