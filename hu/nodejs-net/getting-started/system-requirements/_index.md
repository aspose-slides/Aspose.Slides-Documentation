Az Aspose.Slides for Node.js a .NET-en keresztül nem igényel semmilyen harmadik féltől származó terméket, például a Microsoft PowerPoint telepítését. Maga az Aspose.Slides egy motor a dokumentumok létrehozására, módosítására, konvertálására és megjelenítésére különböző formátumokban, beleértve a Microsoft PowerPoint bemutatóformátumokat.

## Támogatott operációs rendszerek

Az Aspose.Slides .NET számára támogatja a .NET futtatókörnyezetet futtató bármely 32 vagy 64 bites operációs rendszert, többek között, de nem kizárólag:

### **Windows**
- Microsoft Windows 2003 Server ( x64, x86)
- Microsoft Windows 2008 Server ( x64, x86)
- Microsoft Windows 2012 Server ( x64, x86)
- Microsoft Windows 2012 R2 Server ( x64, x86)
- Microsoft Windows 2016 Server ( x64, x86)
- Microsoft Windows 2019 Server ( x64, x86)
- Microsoft Windows Vista ( x64, x86)
- Microsoft Windows XP ( x64, x86)
- Microsoft Windows 7 ( x64, x86)
- Microsoft Windows 8, 8.1 ( x64, x86)
- Microsoft Windows 10 ( x64, x86)
- Microsoft Windows 11 ( x64, x86)

### **Linux**
- Linux (Ubuntu, CentOS és egyebek)

### **Mac**
- Mac OS X
