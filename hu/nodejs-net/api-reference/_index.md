{{% alert color="primary" %}} 

Az Aspose.Slides for Node.js via .NET egy osztálykönyvtár, amely lehetővé teszi a Node.js alkalmazások számára, hogy Microsoft PowerPoint® vagy egyéb szoftverek használata nélkül olvassanak és írjanak prezentációs dokumentumokat. Az Aspose.Slides for Node.js via .NET lehetővé teszi a szöveg, alakzatok, táblázatok és animációk kezelését, hang- és videófájlok hozzáadását a diákhoz, a diák előnézetét, a diák exportálását SVG vagy PDF formátumba, és még sok mást.

Az Aspose.Slides for Node.js via .NET a [azonos API hivatkozás, mint az Aspose.Slides .NET](https://reference.aspose.com/slides/hu/nodejs-net/) használja.

{{% /alert %}}
