Az Aspose.Slides for JasperReports a következőkkel működik:

**Aspose.Slides for JasperReports 17.5 és korábbi verziók:**

- Java 2 Platform Standard Edition 1.4 vagy újabb.
- JasperReports 2.0, 3.0 vagy újabb.

**Aspose.Slides for JasperReports 17.6 és újabb verziók:**

- Java 2 Platform Standard Edition 6.0 vagy újabb.
- JasperReports 3.7.2 vagy újabb.
