Az Aspose.Slides for JasperReports alkalmazásból történő használatához másolja az aspose.slides.jasperreports.library-xx.x.jar fájlt a \lib\JasperReports X.X.X - X.X.X mappából, amely az aspose-slides-xx.x-jasperreports.zip-ben található, a JasperReports\lib könyvtárba vagy az alkalmazás könyvtárába. Ezután programozott módon hozzáférhet az exporterekhez.
