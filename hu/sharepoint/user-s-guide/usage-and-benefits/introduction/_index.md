{{% alert color="primary" %}} 
SharePoint webhelyek az együttműködésre, kommunikációra és tartalomtárolásra szolgálnak, hogy bizonyos üzleti igényeket kielégítsenek. Lehetővé teszik, hogy egy csoport egyének közös célt érjen el az információmegosztás és a kommunikáció elősegítésével. Az információ a leggyakrabban weboldalként érhető el, és webböngészőn keresztül nyitható meg. 
{{% /alert %}} 
## **Mi a SharePoint és az Aspose.Slides?**
Az Aspose.Slides for SharePoint egy rugalmas megoldás a Microsoft PowerPoint dokumentumok más formátumokra, például PPT, POT, PPS, PPTX, PPSX és POTX konvertálására. Az Aspose.Slides for SharePoint lehetővé teszi PowerPoint fájlok olvasását és konvertálását egy SharePoint alkalmazásban a Microsoft PowerPoint használata nélkül.
