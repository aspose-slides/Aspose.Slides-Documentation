Egy könyvtár egy olyan tartalomszerkezet, amely fájlokat tartalmaz. Egy könyvtár csak egy típusú fájlt vagy sok különböző típust tartalmazhat, többek között dokumentumokat, táblázatokat és prezentációkat. A SharePoint könyvtár egyetlen helyet hoz létre, ahol az emberek feltölthetik és elérhetik a dokumentumokat. A dokumentumok bináris formátumban tárolódnak a SharePoint webhely által használt SQL Server adatbázisban. A dokumentumok a többi SharePoint felhasználó számára hiperhivatkozások listájaként érhetők el.
