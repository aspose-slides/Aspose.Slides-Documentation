## **Szerverkövetelmények**
{{% alert color="primary" %}} 

Az Aspose.Slides for SharePoint telepítéséhez a következő egyik változatát futtató rendszer szükséges:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft Office SharePoint Server 2010
- Microsoft Office SharePoint Server 2013
- Microsoft Office SharePoint Server 2016
- Microsoft Office SharePoint Server 2019

{{% /alert %}} {{% alert color="primary" %}} 

A termékekhez javasolt a Service Pack 2 vagy későbbi verzió telepítése.

{{% /alert %}} 
## **Klienskövetelmények**
{{% alert color="primary" %}} 

A SharePoint webhelyek web böngészőn keresztül érhetők el. Az Aspose.Slides for SharePoint a következő böngészőkkel került tesztelésre:

- Microsoft Internet Explorer 6, 7, 8
- Mozilla Firefox 3.5

{{% /alert %}}
