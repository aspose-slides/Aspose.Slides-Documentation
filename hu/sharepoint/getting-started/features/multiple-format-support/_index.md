{{% alert color="primary" %}} 

Az Aspose.Slides for SharePoint‑el számos népszerű irodai dokumentumformátum között konvertálhat dokumentumokat egy SharePoint dokumentumtárból. A konverziók magas hűséggel és pontossággal történnek. 

{{% /alert %}} 
## **Támogatott bemeneti formátumok**
Az Aspose.Slides for SharePoint a következő bemeneti formátumokat támogatja: 

- PPT – Microsoft PowerPoint prezentáció 97 - 2003
- PPS – Microsoft PowerPoint diavetítés 97 - 2003
- POT – Microsoft PowerPoint sablon 97 - 2003
- PPTX – Office Open XML prezentáció
- PPSX – Office Open XML diavetítés
- POTX – Office Open XML sablon

{{% alert color="primary" %}} 

A dokumentumok előállításához az Aspose.Slides for SharePoint egy beépített verzióra támaszkodik az [Aspose.Slides for .NET](http://www.aspose.com/categories/.net-components/aspose.slides-for-.net/default.aspx) komponensből, amely az Aspose egyetlen PowerPoint dokumentum feldolgozó komponense.

{{% /alert %}}
## **Támogatott kimeneti formátumok**
Az Aspose.Slides for SharePoint a következő kimeneti formátumokat támogatja: 

- PDF – Portable Document Format
- TIFF – Képek csomag
- XPS – XML Paper Specification

**A kimeneti formátum kiválasztása a Konverziós beállítások képernyőn**
