**Welcome to Aspose.Slides for PHP via Java!**

Aspose.Slides for PHP via Java is a Microsoft PowerPoint® management API that enables Java applications to read and write PowerPoint® documents without Microsoft PowerPoint®. Aspose.Slides for PHP via Java is the first and only component that provides the functionality to manage PowerPoint® documents. Aspose.Slides for PHP via Java provides a lot of key features such as managing text, shapes, tables & animations, adding audio & video to slides, previewing slides, exporting slides to SVG, PDF and other formats.
