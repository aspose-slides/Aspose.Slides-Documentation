## **Αξιολόγηση Aspose.Slides**
Μπορείτε εύκολα να κατεβάσετε το Aspose.Slides για αξιολόγηση. Το πακέτο αξιολόγησης είναι το ίδιο με το αγορασμένο πακέτο. Η έκδοση αξιολόγησης απλώς ενεργοποιείται ως αδειοδοτημένη μετά την προσθήκη μερικών γραμμών κώδικα για την εφαρμογή της άδειας. 

## **Περιορισμός Έκδοσης Αξιολόγησης**
Η έκδοση αξιολόγησης του Aspose.Slides (χωρίς καθορισμένη άδεια) παρέχει πλήρη λειτουργικότητα του προϊόντος, αλλά εισάγει ένα υδατογράφημα αξιολόγησης στην κορυφή του εγγράφου κατά το άνοιγμα και την αποθήκευση. Επίσης περιορίζεστε σε μία διαφάνεια όταν εξάγετε κείμενα από τις διαφάνειες παρουσίασης.

{{% alert color="primary" %}} 
Εάν θέλετε να δοκιμάσετε το Aspose.Slides χωρίς τους περιορισμούς της έκδοσης αξιολόγησης, μπορείτε να ζητήσετε μια **30‑ημερη Προσωρινή Άδεια**. Παρακαλούμε ανατρέξτε στο [Πώς να λάβετε μια Προσωρινή Άδεια?](https://purchase.aspose.com/temporary-license) για περισσότερες πληροφορίες.

## **Σχετικά με την Άδεια**
Μπορείτε εύκολα να κατεβάσετε μια έκδοση αξιολόγησης του Aspose.Slides για Node.js μέσω .NET από τη [σελίδα λήψης](https://releases.aspose.com/slides/el/nodejs-net/). Η έκδοση αξιολόγησης προσφέρει απολύτως **τις ίδιες δυνατότητες** με την αδειοδοτημένη έκδοση του Aspose.Slides. Επιπλέον, η έκδοση αξιολόγησης απλώς ενεργοποιείται ως αδειοδοτημένη μετά την αγορά άδειας και την προσθήκη μερικών γραμμών κώδικα για την εφαρμογή της άδειας.

Η άδεια είναι ένα αρχείο XML απλού κειμένου που περιέχει λεπτομέρειες όπως το όνομα προϊόντος, ο αριθμός των προγραμματιστών στα οποία αδειοδοτείται, η ημερομηνία λήξης της συνδρομής κ.λπ. Το αρχείο είναι ψηφιακά υπογεγραμμένο, επομένως μην το τροποποιήσετε. Ακόμη και μια ακούσια προσθήκη μιας επιπλέον αλλαγής γραμμής στο περιεχόμενο του αρχείου θα το ακυρώσει.

Για να αποφύγετε τους περιορισμούς που σχετίζονται με την έκδοση αξιολόγησης, πρέπει να ορίσετε άδεια πριν χρησιμοποιήσετε το **Aspose.Slides**. Απαιτείται να ορίσετε άδεια μόνο μία φορά ανά εφαρμογή ή διαδικασία.

### **Ορισμός Άδειας στο Aspose.Slides για Node.js μέσω .NET**

Οι άδειες μπορούν να εφαρμοστούν από τις ακόλουθες τοποθεσίες:

* Ένδειξη διαδρομής
* Ροή
* Ως Μετρημένη Άδεια – ένας νέος μηχανισμός αδειοδότησης

{{% alert color="primary" %}}
Χρησιμοποιήστε τη μέθοδο **setLicense** για να αδειοδοτήσετε ένα στοιχείο.

Ενώ πολλαπλές κλήσεις στη **setLicense** δεν είναι επιβλαβείς, είναι σπατάλη πόρων (επεξεργαστή).

{{% alert color="warning" %}}
Οι νέες άδειες μπορούν να ενεργοποιήσουν το Aspose.Slides μόνο με την έκδοση 21.4 ή νεότερη. Οι παλαιότερες εκδόσεις χρησιμοποιούν διαφορετικό σύστημα αδειοδότησης και δεν θα αναγνωρίσουν αυτές τις άδειες.
{{% /alert %}}

#### **Εφαρμογή Άδειας με Αρχείο**

Αυτό το απόσπασμα κώδικα χρησιμοποιείται για να ορίσετε ένα αρχείο άδειας:

**Node.js**

```javascript
// Εισαγωγή του module Aspose.Slides για διαχείριση αρχείων PowerPoint
const asposeSlides = require('aspose.slides.via.net');

// Αυτή η συνάρτηση ρυθμίζει τη βιβλιοθήκη Aspose.Slides με άδεια
function setupAsposeSlidesLicense() {
	
    // Αρχικοποίηση της κλάσης License από το module Aspose.Slides
    var license = new asposeSlides.License();
    
    // Εφαρμογή της άδειας από αρχείο
    // Αντικαταστήστε το "your_license_file.lic" με τη διαδρομή προς το πραγματικό αρχείο άδειας σας
    license.setLicense("your_license_file.lic");
}

// Εκτέλεση της συνάρτησης για να ρυθμίσετε την άδεια για το Aspose.Slides
setupAsposeSlidesLicense();
```
{{% alert color="primary" %}}
Κατά την κλήση της μεθόδου setLicense, το όνομα της άδειας πρέπει να είναι το ίδιο με το όνομα του αρχείου άδειας σας. Για παράδειγμα, μπορείτε να αλλάξετε το όνομα του αρχείου άδειας σε "Aspose.Slides.lic.xml". Στη συνέχεια, στον κώδικά σας, πρέπει να μεταβιβάσετε το νέο όνομα άδειας (Aspose.Slides.lic.xml) στη μέθοδο setLicense.
{{% /alert %}}
