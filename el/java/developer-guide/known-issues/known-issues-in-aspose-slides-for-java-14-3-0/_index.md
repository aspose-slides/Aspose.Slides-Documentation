Aspose.Slides for Java 14.3.0 (14.4.0) παρέχει εντελώς νέα υλοποίηση επεξεργασίας PPT. Υπάρχουν πολλές βελτιώσεις, μερική μετατροπή PPTX σε PPT. Ωστόσο υπάρχουν ορισμένες μη υλοποιημένες λειτουργίες:

- Κάποιες μορφές έχουν λανθασμένη γεωμετρία στα σειριοποιημένα έγγραφα PPT (Call outs)
- Δεν υποστηρίζονται όλες οι δυνατότητες μορφοποίησης κειμένου PPTX στην σειριοποίηση PPT
- Πληροφορίες για τη γλώσσα του κειμένου και τις ρυθμίσεις ορθογραφίας δεν περιλαμβάνονται στα σειριοποιημένα έγγραφα PPT
- Δεν υποστηρίζονται όλες οι δυνατότητες θεμάτων PPTX στην σειριοποίηση PPT

**Υπάρχουν ορισμένες διαφορές σε σύγκριση με το Aspose.Slides for Java 8.6.0:**

- Υπάρχουν γνωστά προβλήματα στην σειριοποίηση OLE/ActiveX PPT σε PPT

**Υπάρχουν ορισμένες διαφορές σε σύγκριση με το Aspose.Slides for .NET 14.3.0:**

- Η υποστήριξη εκτύπωσης παρουσίασης δεν είναι αυτή τη στιγμή διαθέσιμη στο Aspose.Slides for Java
