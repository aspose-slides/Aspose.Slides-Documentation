{{% alert color="primary" %}} 

Aspose.Slides for Java 14.4.0 παρέχει νέα απόφαση για την επεξεργασία εγγράφων PowerPoint. Υπάρχουν ορισμένοι περιορισμοί και γνωστά ζητήματα, τα οποία θα αφαιρεθούν στις επόμενες κυκλοφορίες:

- Κάποιες σχήματα έχουν λανθασμένη γεωμετρία σε σειριακοποιημένα έγγραφα PPT (τόξο, κυκλικό βέλος, επεξηγήσεις).
- Δεν υποστηρίζονται όλες οι δυνατότητες μορφοποίησης κειμένου PPTX στη σειριακοποίηση PPT (περιορισμοί στη στηλοθέτηση, την εσοχή και τη μορφοποίηση παραγράφου).
- Πληροφορίες για τη γλώσσα κειμένου και τις ρυθμίσεις ορθογραφίας δεν εμφανίζονται στα σειριακοποιημένα έγγραφα PPT.
- Δεν υποστηρίζονται όλες οι δυνατότητες θέματος PPTX στη σειριακοποίηση PPT (μόνο η σειριακοποίηση μορφών γεμίσματος, μορφών γραμμής και γραμματοσειράς).
- Υπάρχουν γνωστά ζητήματα στην σειριακοποίηση OLE/ActiveX PPT σε PPT.
- Η σειριακοποίηση και η απόδοση του WordArt δεν υποστηρίζονται.
