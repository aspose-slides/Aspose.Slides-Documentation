## **Εισαγωγή**

Στο PowerPoint 2019, η Microsoft εισήγαγε τη ρύθμιση **Always Open Read-Only** ως μια από τις επιλογές που μπορούν να χρησιμοποιήσουν οι χρήστες για να προστατεύουν τις παρουσιάσεις τους. Μπορεί να θέλετε να χρησιμοποιήσετε αυτή τη ρύθμιση Read-Only για να προστατεύσετε μια παρουσίαση όταν

- Θέλετε να αποτρέψετε τυχαίες επεκδόσεις και να διατηρήσετε το περιεχόμενο της παρουσίασής σας ασφαλές. 
- Θέλετε να ενημερώσετε τους ανθρώπους ότι η παρουσίαση που παρείχατε είναι η τελική έκδοση. 

Αφού επιλέξετε την επιλογή **Always Open Read-Only** για μια παρουσίαση, όταν οι χρήστες ανοίγουν την παρουσίαση, βλέπουν τη σύσταση **Read-Only** και μπορεί να εμφανιστεί ένα μήνυμα στην εξής μορφή: *Για να αποτραπούν τυχαίες αλλαγές, ο δημιουργός έχει ορίσει αυτό το αρχείο να ανοίγει ως μόνο για ανάγνωση.*

Η σύσταση Read-Only είναι μια απλή αλλά αποτελεσματική αποθάρρυνση που αποτρέπει την επεξεργασία, καθώς οι χρήστες πρέπει να εκτελέσουν μια ενέργεια για να την αφαιρέσουν πριν τους επιτραπεί η επεξεργασία μιας παρουσίασης. Εάν δεν θέλετε οι χρήστες να κάνουν αλλαγές σε μια παρουσίαση και θέλετε να το τους πείτε με ευγενικό τρόπο, τότε η σύσταση Read-Only μπορεί να είναι μια καλή επιλογή για εσάς. 

> Εάν μια παρουσίαση με την προστασία **Read-Only** ανοίξει σε παλαιότερη εφαρμογή Microsoft PowerPoint—που δεν υποστηρίζει τη πρόσφατα εισαχθείσα λειτουργία—η σύσταση **Read-Only** αγνοείται (η παρουσίαση ανοίγει κανονικά).

## **Εφαρμογή Λειτουργίας Read-Only**

Το Aspose.Slides for Java σας επιτρέπει να ορίσετε μια παρουσίαση σε **Read-Only**, πράγμα που σημαίνει ότι οι χρήστες (αφού ανοίξουν την παρουσίαση) βλέπουν τη σύσταση **Read-Only**. Αυτό το παράδειγμα κώδικα δείχνει πώς να ορίσετε μια παρουσίαση σε **Read-Only** στη Java χρησιμοποιώντας το Aspose.Slides:

```java
Presentation pres = new Presentation();
try {
    pres.getProtectionManager().setReadOnlyRecommended(true);
    pres.save("ReadOnlyPresentation.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```

{{% alert color="primary" %}} 

**Σημείωση**: Η σύσταση **Read-Only** προορίζεται απλώς να αποθαρρύνει την επεξεργασία ή να εμποδίσει τους χρήστες από τυχαίες αλλαγές σε μια παρουσίαση PowerPoint. Εάν ένα κίνητρο άτομο—που ξέρει τι κάνει—αποφασίσει να επεξεργαστεί την παρουσίασή σας, μπορεί εύκολα να αφαιρέσει τη ρύθμιση Read-Only. Εάν χρειάζεστε σοβαρά να αποτρέψετε μη εξουσιοδοτημένη επεξεργασία, είναι καλύτερο να χρησιμοποιήσετε [πιο αυστηρές προστασίες που περιλαμβάνουν κρυπτογραφήσεις και κωδικούς πρόσβασης](https://docs.aspose.com/slides/el/java/password-protected-presentation/).

## **FAQ**

**Πώς διαφέρει το 'Read-Only recommended' από πλήρη προστασία κωδικού;**

'Read-Only recommended' εμφανίζει μόνο μια πρόταση να ανοίξετε το αρχείο σε λειτουργία μόνο για ανάγνωση και είναι εύκολο να παρακαμφθεί. [Password protection](/slides/el/java/password-protected-presentation/) περιορίζει στην πραγματικότητα το άνοιγμα ή την επεξεργασία και είναι κατάλληλο όταν χρειάζεστε πραγματικούς ελέγχους ασφαλείας.

**Μπορεί το 'Read-Only recommended' να συνδυαστεί με υδατογραφήματα για να αποθαρρύνει περαιτέρω τις επεξεργασίες;**

Ναι. Η σύσταση μπορεί να συνδυαστεί με [watermarks](/slides/el/java/watermark/) ως οπτική αποθάρρυνση· είναι ξεχωριστοί μηχανισμοί και λειτουργούν καλά μαζί.

**Μπορεί ένα μακρό ή εξωτερικό εργαλείο να τροποποιήσει ακόμη το αρχείο όταν η σύσταση είναι ενεργοποιημένη;**

Ναι. Η σύσταση δεν περιορίζει τις προγραμματισμένες αλλαγές. Για να αποτρέψετε αυτοματοποιημένες επεξεργασίες, χρησιμοποιήστε [passwords and encryption](/slides/el/java/password-protected-presentation/).

**Πώς σχετίζεται το 'Read-Only recommended' με τις μεθόδους 'isEncrypted' και 'isWriteProtected';**

Είναι διαφορετικά σήματα. Το 'Read-Only recommended' είναι μια ήπια, προαιρετική προτροπή· [isWriteProtected](https://reference.aspose.com/slides/el/java/com.aspose.slides/protectionmanager/#isWriteProtected--) και [isEncrypted](https://reference.aspose.com/slides/el/java/com.aspose.slides/protectionmanager/#isEncrypted--) υποδεικνύουν πραγματικούς περιορισμούς εγγραφής ή ανάγνωσης που εξαρτώνται από κωδικούς ή κρυπτογράφηση.
