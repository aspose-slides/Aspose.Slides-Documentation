## **Επισκόπηση**

Αυτό το άρθρο εξηγεί πώς να αποδίδεται διαφάνειες παρουσίασης ως εικόνες SVG χρησιμοποιώντας το Aspose.Slides. Περιγράφει τη μορφή SVG και τα πλεονεκτήματά της, συμπεριλαμβανομένης της κλιμασιμότητας, της προσβασιμότητας και της καταλληλότητας για ανάπτυξη ιστού.

Θα μάθετε πώς να φορτώσετε ένα αρχείο παρουσίασης, να επαναλάβετε τις διαφάνειες του και να αποθηκεύσετε κάθε διαφάνεια ως ξεχωριστό αρχείο SVG. Το άρθρο καλύπτει τις μορφές παρουσίασης PowerPoint και OpenDocument, συμπεριλαμβανομένων των PPT, PPTX, ODP και PPS, και δείχνει πώς να πραγματοποιήσετε τη μετατροπή προγραμματικά με την κλάση `Presentation` και τη μέθοδο `writeAsSvg`.

## **Μορφή SVG**

Το SVG—ακρωνύμιο του Scalable Vector Graphics—είναι ένας τυπικός τύπος ή μορφή γραφικών που χρησιμοποιείται για την απόδοση δισδιάστατων εικόνων. Το SVG αποθηκεύει τις εικόνες ως διανύσματα σε XML με λεπτομέρειες που ορίζουν τη συμπεριφορά ή την εμφάνισή τους.

Το SVG είναι μία από τις λίγες μορφές εικόνων που ικανοποιούν πολύ υψηλά πρότυπα σε αυτούς τους όρους: κλιμασιμότητα, διαδραστικότητα, απόδοση, προσβασιμότητα, προγραμματισιμότητα και άλλα. Για τους λόγους αυτούς, χρησιμοποιείται συνήθως στην ανάπτυξη ιστού.

Μπορεί να θέλετε να χρησιμοποιήσετε αρχεία SVG όταν χρειάζεται να

- **εκτυπώσετε την παρουσίασή σας σε *πολύ μεγάλο φορμάτ*.** Οι εικόνες SVG μπορούν να κλιμακωθούν σε οποιαδήποτε ανάλυση ή επίπεδο. Μπορείτε να αλλάζετε το μέγεθος των εικόνων SVG όσες φορές χρειαστεί χωρίς να θυσιάζετε την ποιότητα.
- **χρησιμοποιήσετε διαγράμματα και γραφήματα από τις διαφάνειές σας σε *διαφορετικά μέσα ή πλατφόρμες*.** Οι περισσότεροι προγράμματα ανάγνωσης μπορούν να ερμηνεύσουν αρχεία SVG.
- **χρησιμοποιήσετε το *μικρότερο δυνατό μέγεθος εικόνων*.** Τα αρχεία SVG είναι γενικά μικρότερα από τα ισοδύναμα υψηλής ανάλυσης σε άλλες μορφές, ειδικά σε μορφές που βασίζονται σε bitmap (JPEG ή PNG).

## **Απόδοση μιας Διαφάνειας ως Εικόνα SVG**

Το Aspose.Slides for Java σας επιτρέπει να εξάγετε τις διαφάνειες στις παρουσιάσεις σας ως εικόνες SVG. Ακολουθήστε αυτά τα βήματα για να δημιουργήσετε εικόνες SVG:

1. Δημιουργήστε μια παρουσία της κλάσης Presentation.
2. Περιηγηθείτε σε όλες τις διαφάνειες της παρουσίασης.
3. Γράψτε κάθε διαφάνεια στο δικό της αρχείο SVG μέσω FileOutputStream.

{{% alert color="primary" %}} 
Μπορείτε να δοκιμάσετε την [δωρεάν διαδικτυακή εφαρμογή](https://products.aspose.app/slides/el/conversion/ppt-to-svg) στην οποία υλοποιήσαμε τη λειτουργία μετατροπής PPT σε SVG από το Aspose.Slides for Java.
{{% /alert %}} 

Αυτό το δείγμα κώδικα σε Java δείχνει πώς να μετατρέψετε PPT σε SVG χρησιμοποιώντας το Aspose.Slides:

``` java
Presentation pres = new Presentation("pres.pptx");
try {
    for (int index = 0; index < pres.getSlides().size(); index++)
    {
        ISlide slide = pres.getSlides().get_Item(index);

        FileOutputStream fileStream = new FileOutputStream("slide-" + index + ".svg");
        try {
            slide.writeAsSvg(fileStream);
        } finally {
            fileStream.close();
        }
    }
} catch(IOException e) {
} finally {
    if (pres != null) pres.dispose();
}
```

## **Συχνές Ερωτήσεις**

**Γιατί το προκύπτον SVG μπορεί να φαίνεται διαφορετικό σε διαφορετικά προγράμματα περιήγησης;**

Η υποστήριξη συγκεκριμένων λειτουργιών SVG υλοποιείται διαφορετικά από τις μηχανές των προγραμμάτων περιήγησης. Οι παράμετροι [SVGOptions](https://reference.aspose.com/slides/el/java/com.aspose.slides/svgoptions/) βοηθούν στην εξομάλυνση των ασυμβατοτήτων.

**Μπορεί να εξαχθούν όχι μόνο οι διαφάνειες αλλά και μεμονωμένα σχήματα σε SVG;**

Ναι. Κάθε [σχήμα μπορεί να αποθηκευτεί ως ξεχωριστό SVG](https://reference.aspose.com/slides/el/java/com.aspose.slides/shape/#writeAsSvg-java.io.OutputStream-com.aspose.slides.ISVGOptions-) είναι βολικό για εικονίδια, εικονογράμματα και επαναχρησιμοποίηση γραφικών.

**Μπορούν πολλές διαφάνειες να συνδυαστούν σε ένα ενιαίο SVG (strip/document);**

Το τυπικό σενάριο είναι μία διαφάνεια → ένα SVG. Ο συνδυασμός πολλών διαφανειών σε ένα ενιαίο καμβά SVG αποτελεί βήμα μετα-επεξεργασίας που εκτελείται σε επίπεδο εφαρμογής.
