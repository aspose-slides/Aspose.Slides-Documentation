Έχει κυκλοφορήσει ένα νέο [Aspose.Slides for Java API](/slides/el/java/) και πλέον αυτό το μοναδικό προϊόν υποστηρίζει τη δυνατότητα δημιουργίας εγγράφων PowerPoint από την αρχή και επεξεργασίας των υπαρχόντων.

## **Υποστήριξη για Παλιό Κώδικα**
Για να χρησιμοποιήσετε τον κώδικα κληρονομίας που αναπτύχθηκε με εκδόσεις του Aspose.Slides for Java προγενέστερες της 13.x, πρέπει να κάνετε ορισμένες μικρές αλλαγές στον κώδικά σας και ο κώδικας θα λειτουργεί όπως πριν. Όλες οι κλάσεις που υπήρχαν στο παλιό Aspose.Slides for Java στα namespaces Aspose.Slide και Aspose.Slides.Pptx έχουν πλέον συγχωνευτεί σε ένα μοναδικό namespace Aspose.Slides. Παρακαλούμε ρίξτε μια ματιά στο παρακάτω απλό τμήμα κώδικα για τη δημιουργία ενός εγγράφου Hello World Presentation στην παλαιά Aspose.Slides API και ακολουθήστε τα βήματα που περιγράφουν πώς να μεταβείτε στη νέα συγχωνευμένη API.
