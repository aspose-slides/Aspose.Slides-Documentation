Μία νέα [Aspose.Slides for Java API](https://docs.aspose.com/slides/el/java/) έχει κυκλοφορήσει και τώρα αυτό το μοναδικό προϊόν υποστηρίζει τη δυνατότητα δημιουργίας εγγράφων PowerPoint από το μηδέν και την επεξεργασία των υπαρχόντων.

## **Υποστήριξη Παλαιού Κώδικα**
Για να χρησιμοποιήσετε τον κώδικα κληρονομικό που αναπτύχθηκε με τις εκδόσεις του Aspose.Slides for Java προγενέστερες των 14.x.x, πρέπει να κάνετε κάποιες μικρές αλλαγές στον κώδικά σας ώστε να λειτουργεί όπως παλιότερα. Όλες οι κλάσεις που υπήρχαν στην παλαιά έκδοση του Aspose.Slides for Java κάτω από τα ονοματοχώρους com.aspose.slides και com.aspose.slides.pptx έχουν πλέον συγχωνευτεί σε έναν μόνο ονοματοχώρο com.aspose.slides. Παρακαλούμε ρίξτε μια ματιά στο παρακάτω απλό κομμάτι κώδικα για τη δημιουργία ενός κανονικού διαγράμματος από το μηδέν σε μια παρουσίαση χρησιμοποιώντας το κληρονομικό API του Aspose.Slides και ακολουθήστε τα βήματα που περιγράφουν πώς να μεταβείτε στο νέο συγχωνευμένο API.
