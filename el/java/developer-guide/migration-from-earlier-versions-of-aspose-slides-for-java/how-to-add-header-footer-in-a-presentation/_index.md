## **Υποστήριξη Παλαιού Κώδικα**
Για να χρησιμοποιήσετε τον κώδικα κληρονομίας που αναπτύχθηκε με εκδόσεις του Aspose.Slides for Java προγενέστερες της 13.x, πρέπει να κάνετε ορισμένες μικρές αλλαγές στον κώδικά σας ώστε να λειτουργεί όπως παλαιότερα. Όλες οι κλάσεις που υπήρχαν στην παλιά έκδοση του Aspose.Slides for Java στα ονόματα χώρου Aspose.Slide και Aspose.Slides.Pptx έχουν πλέον συγχωνευθεί σε ένα μοναδικό όνομα χώρου Aspose.Slides. Παρακαλούμε ρίξτε μια ματιά στο παρακάτω απλό απόσπασμα κώδικα για την προσθήκη κεφαλίδας/υποσέλιδου στην παρουσίαση στην κληρονομική Aspose.Slides API και ακολουθήστε τα βήματα που περιγράφουν πώς να μεταβείτε στη νέα συγχωνευμένη API.
