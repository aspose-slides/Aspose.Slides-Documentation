Κάποιες φορές, χρειάζεται να μορφοποιήσετε το κείμενο στις διαφάνειες προγραμματιστικά. Αυτό το άρθρο δείχνει πώς να διαβάσετε μια δείγμα παρουσίαση με κάποιο κείμενο στην πρώτη διαφάνεια χρησιμοποιώντας είτε [VSTO](/slides/el/java/format-text-using-vsto-and-aspose-slides-for-java/) και [Aspose.Slides for Java](/slides/el/java/format-text-using-vsto-and-aspose-slides-for-java/). Ο κώδικας μορφοποιεί το κείμενο στο τρίτο πλαίσιο κειμένου της διαφάνειας ώστε να μοιάζει με το κείμενο στο τελευταίο πλαίσιο κειμένου.

{{% /alert %}} 
## **Μορφοποίηση Κειμένου**
Και οι μέθοδοι VSTO και Aspose.Slides ακολουθούν τα παρακάτω βήματα:

1. Ανοίξτε την πηγαία παρουσίαση.
1. Αποκτήστε πρόσβαση στην πρώτη διαφάνεια.
1. Αποκτήστε πρόσβαση στο τρίτο πλαίσιο κειμένου.
1. Αλλάξτε τη μορφοποίηση του κειμένου στο τρίτο πλαίσιο κειμένου.
1. Αποθηκεύστε την παρουσίαση στο δίσκο.

Τα στιγμιότυπα οθόνης παρακάτω δείχνουν τη διαφάνεια δείγματος πριν και μετά την εκτέλεση του κώδικα VSTO και Aspose.Slides for Java.
