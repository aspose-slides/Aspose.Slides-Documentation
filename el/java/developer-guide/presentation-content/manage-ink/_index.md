## **Εισαγωγή**

Το PowerPoint παρέχει τη λειτουργία μελάνης ώστε να μπορείτε να σχεδιάζετε μη‑τυπικά σχήματα, τα οποία μπορούν να χρησιμοποιηθούν για να επισημάνουν άλλα αντικείμενα, να δείξουν συνδέσεις και διαδικασίες και να τραβήξουν την προσοχή σε συγκεκριμένα στοιχεία σε μια διαφάνεια. 

Το Aspose.Slides παρέχει όλους τους τύπους Ink (π.χ. κλάση [Ink](https://reference.aspose.com/slides/el/java/com.aspose.slides/ink/)) που χρειάζεστε για τη δημιουργία και διαχείριση αντικειμένων μελάνης. 

## **Διαφορές μεταξύ Κανονικών Αντικειμένων και Αντικειμένων Μελάνης**

Τα αντικείμενα σε μια διαφάνεια PowerPoint αντιπροσωπεύονται συνήθως από αντικείμενα σχήματος. Ένα αντικείμενο σχήματος, στην πιο απλή του μορφή, είναι ένα δοχείο που ορίζει την περιοχή του ίδιου του αντικειμένου (το πλαίσιο του) μαζί με τις ιδιότητές του. Οι τελευταίες περιλαμβάνουν το μέγεθος της περιοχής του δοχείου, το σχήμα του δοχείου, το φόντο του δοχείου κ.λπ. Για περισσότερες πληροφορίες, δείτε το [Μορφοποίηση Διατάξεων Σχήματος](https://docs.aspose.com/slides/el/java/shape-manipulations/#access-layout-formats-for-shape).

Ωστόσο, όταν το PowerPoint αντιμετωπίζει ένα αντικείμενο μελάνης, αγνοεί όλες τις ιδιότητες του πλαισίου του αντικειμένου (του δοχείου) εκτός από το μέγεθός του. Το μέγεθος της περιοχής του δοχείου καθορίζεται από τις τυπικές τιμές `width` και `height`:

## **Ιχνηλάσεις Σχήματος Μελάνης**

Η ίχνος είναι ένα βασικό στοιχείο ή πρότυπο που χρησιμοποιείται για την καταγραφή της τροχιάς μιας πένας καθώς ο χρήστης γράφει ψηφιακή μελάνη. Τα ίχνη είναι εγγραφές που περιγράφουν ακολουθίες συνδεδεμένων σημείων. 

Η πιο απλή μορφή κωδικοποίησης καθορίζει τις συντεταγμένες X και Y κάθε σημείου δείγματος. Όταν όλα τα συνδεδεμένα σημεία αποδοθούν, δημιουργούν μια εικόνα όπως αυτή:

## **Ιδιότητες Πινέλου για Σχεδίαση**

Μπορείτε να χρησιμοποιήσετε ένα πινέλο για να σχεδιάσετε γραμμές που συνδέουν τα σημεία των στοιχείων ίχνους. Το πινέλο έχει το δικό του χρώμα και μέγεθος, τα οποία αντιστοιχούν στις ιδιότητες `Brush.Color` και `Brush.Size`. 

### **Ορισμός Χρώματος Πινέλου Μελάνης**

Αυτός ο κώδικας Java σας δείχνει πώς να ορίσετε το χρώμα για ένα πινέλο:

```java
Presentation pres = new Presentation("pres.pptx");
try {
    IInk ink = (IInk)pres.getSlides().get_Item(0).getShapes().get_Item(0);
    IInkTrace[] traces = ink.getTraces();
    IInkBrush brush = traces[0].getBrush();
    Color brushColor = brush.getColor();
    brush.setColor(Color.RED);
} finally {
    if (pres != null) pres.dispose();
}
```

### **Ορισμός Μεγέθους Πινέλου Μελάνης** 

Αυτός ο κώδικας Java σας δείχνει πώς να ορίσετε το μέγεθος για ένα πινέλο:

```java
Presentation pres = new Presentation("pres.pptx");
try {
    IInk ink = (IInk)pres.getSlides().get_Item(0).getShapes().get_Item(0);
    IInkTrace[] traces = ink.getTraces();
    IInkBrush brush = traces[0].getBrush();
    Dimension2D brushSize = brush.getSize();
    brush.setSize(new Dimension(5, 10));
} finally {
    if (pres != null) pres.dispose();
}
```

Γενικά, το πλάτος και το ύψος ενός πινέλου δεν ταιριάζουν, έτσι το PowerPoint δεν εμφανίζει το μέγεθος του πινέλου (η ενότητα δεδομένων είναι σε γκρι). Αλλά όταν το πλάτος και το ύψος του πινέλου ταιριάζουν, το PowerPoint εμφανίζει το μέγεθός του ως εξής:

![ink_powerpoint3](ink_powerpoint3.png)

Για σαφήνεια, ας αυξήσουμε το ύψος του αντικειμένου μελάνης και ας εξετάσουμε τις σημαντικές διαστάσεις: 

![ink_powerpoint4](ink_powerpoint4.png)

Το δοχείο (πλαίσιο) δεν λαμβάνει υπόψη το μέγεθος των πινέλων—υποθέτει πάντα ότι το πάχος της γραμμής είναι μηδέν (δείτε την τελευταία εικόνα). 

Συνεπώς, για να προσδιορίσουμε την ορατή περιοχή ολόκληρου του αντικειμένου μελάνης, πρέπει να λάβουμε υπόψη το μέγεθος του πινέλου των αντικειμένων ίχνους. Εδώ, το αντικείμενο-στόχος (το αντικείμενο ίχνους χειρόγραφου κειμένου) έχει κλιμακωθεί στο μέγεθος του δοχείου (πλαισίου). Όταν το μέγεθος του δοχείου (πλαισίου) αλλάζει, το μέγεθος του πινέλου παραμένει σταθερό και αντίστροφα. 

![ink_powerpoint5](ink_powerpoint5.png)

Το PowerPoint παρουσιάζει την ίδια συμπεριφορά όταν χειρίζεται κείμενα:
