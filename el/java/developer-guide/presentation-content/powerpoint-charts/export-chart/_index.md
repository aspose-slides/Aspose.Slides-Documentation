## **Επισκόπηση**

Aspose.Slides σας επιτρέπει να εξάγετε ένα γράφημα από μια παρουσίαση ως εικόνα. Αυτό το άρθρο δείχνει πώς να λάβετε μια εικόνα από ένα γράφημα και να την αποθηκεύσετε, χρήσιμο όταν χρειάζεται να επαναχρησιμοποιήσετε τα γραφικά του γραφήματος εκτός μιας παρουσίασης PowerPoint.

Εκτός της βασικής ροής εξαγωγής εικόνας, το άρθρο καλύπτει επίσης κοινές ερωτήσεις σχετικές με την εξαγωγή, όπως η αποθήκευση του περιεχομένου του γραφήματος σε SVG, ο έλεγχος του μεγέθους εξόδου μέσω επιλογών απόδοσης, η φόρτωση γραμματοσειρών για διατήρηση της εμφάνιση ετικετών και υπομνήματος, και η διατήρηση της αρχικής μορφοποίησης της παρουσίασης όπως θέματα, στυλ, γεμίσματα και εφέ κατά την απόδοση.

## **Απόκτηση εικόνας γραφήματος**
Aspose.Slides for Java παρέχει υποστήριξη για την εξαγωγή εικόνας συγκεκριμένου γραφήματος. Παρακάτω δίνεται ένα παράδειγμα.

```java
Presentation pres = new Presentation();
try {
    IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(ChartType.ClusteredColumn, 50, 50, 600, 400);

    IImage slideImage = chart.getImage();

    try {
          slideImage.save("image.jpg", ImageFormat.Jpeg);
    } finally {
         if (slideImage != null) slideImage.dispose();
    }
} finally {
    if (pres != null) pres.dispose();
}
```

## **Συχνές ερωτήσεις**

**Μπορώ να εξάγω ένα γράφημα ως διάνυσμα (SVG) αντί για ραστερ εικόνα;**

Ναι. Ένα γράφημα είναι σχήμα, και το περιεχόμενό του μπορεί να αποθηκευτεί σε SVG χρησιμοποιώντας τη [μέθοδο αποθήκευσης shape-to-SVG](https://reference.aspose.com/slides/el/java/com.aspose.slides/shape/#writeAsSvg-java.io.OutputStream-com.aspose.slides.ISVGOptions-).

**Πώς μπορώ να ορίσω το ακριβές μέγεθος του εξαγόμενου γραφήματος σε εικονοστοιχεία;**

Χρησιμοποιήστε τις υπερφορτώσεις απόδοσης εικόνας που επιτρέπουν τον καθορισμό του μεγέθους ή της κλίμακας — η βιβλιοθήκη υποστηρίζει την απόδοση αντικειμένων με τις δοσμένες διαστάσεις/κλίμακα.

**Τι πρέπει να κάνω εάν οι γραμματοσειρές στις ετικέτες και το υπόμνημα εμφανίζονται λανθασμένα μετά την εξαγωγή;**

[Φορτώστε τις απαιτούμενες γραμματοσειρές](/slides/el/java/custom-font/) μέσω [FontsLoader](https://reference.aspose.com/slides/el/java/com.aspose.slides/fontsloader/) ώστε η απόδοση του γραφήματος να διατηρεί τις μετρικές και την εμφάνιση του κειμένου.

**Σεβεται η εξαγωγή το θέμα, τα στυλ και τα εφέ του PowerPoint;**

Ναι. Ο renderizer του Aspose.Slides ακολουθεί τη μορφοποίηση της παρουσίασης (θέματα, στυλ, γεμίσματα, εφέ), οπότε η εμφάνιση του γραφήματος διατηρείται.

**Πού μπορώ να βρω τις διαθέσιμες δυνατότητες απόδοσης/εξαγωγής εκτός των εικόνων γραφημάτων;**

Δείτε το [API](https://reference.aspose.com/slides/el/java/com.aspose.slides/)/[τεκμηρίωση](/slides/el/java/convert-powerpoint/) για τους προορισμούς εξόδου ([PDF](/slides/el/java/convert-powerpoint-to-pdf/), [SVG](/slides/el/java/render-a-slide-as-an-svg-image/), [XPS](/slides/el/java/convert-powerpoint-to-xps/), [HTML](/slides/el/java/convert-powerpoint-to-html/), κ.ά.) και τις σχετικές επιλογές απόδοσης.
