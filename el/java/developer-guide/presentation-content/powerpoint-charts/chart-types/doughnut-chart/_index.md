## **Επισκόπηση**

Αυτό το άρθρο δείχνει πώς να εργαστείτε με ένα διάγραμμα ντόνατ στο Aspose.Slides προσθέτοντας το διάγραμμα σε μια διαφάνεια, ορίζοντας το μέγεθος της κεντρικής τρύπας και αποθηκεύοντας την παρουσίαση. Επικεντρώνεται στη μέθοδο `setDoughnutHoleSize` και παρουσιάζει τα βασικά βήματα που απαιτούνται για την προσαρμογή αυτού του τύπου διαγράμματος με κώδικα.

Περιλαμβάνει επίσης μια σύντομη ενότητα Συχνών Ερωτήσεων που καλύπτει σχετικές περιπτώσεις διαγραμμάτων ντόνατ, όπως η χρήση πολλαπλών σειρών για τη δημιουργία πολλαπλών δαχτυλιδιών, η εργασία με εκραγμένα διαγράμματα ντόνατ και η εξαγωγή ενός διαγράμματος ως εικόνα raster ή SVG.

## **Καθορισμός του κεντρικού χάσματος σε διάγραμμα ντόνατ**
{{% alert color="primary" %}} 

Το Aspose.Slides για Java υποστηρίζει πλέον τον καθορισμό του μεγέθους της τρύπας σε διάγραμμα ντόνατ. Σε αυτό το θέμα, θα δούμε με παράδειγμα πώς να καθορίσετε το μέγεθος της τρύπας σε ένα διάγραμμα ντόνατ.

{{% /alert %}} 

Για να καθορίσετε το μέγεθος της τρύπας σε ένα διάγραμμα ντόνατ, ακολουθήστε τα παρακάτω βήματα:

1. Δημιουργήστε ένα αντικείμενο [Presentation](https://reference.aspose.com/slides/el/java/com.aspose.slides/presentation).
1. Προσθέστε διάγραμμα ντόνατ στη διαφάνεια.
1. Καθορίστε το μέγεθος της τρύπας σε ένα διάγραμμα ντόνατ.
1. Γράψτε την παρουσίαση στο δίσκο.

Στο παρακάτω παράδειγμα, έχουμε ορίσει το μέγεθος της τρύπας σε ένα διάγραμμα ντόνατ.

```java
// Δημιουργήστε μια παρουσίαση της κλάσης Presentation
Presentation pres = new Presentation();
try {
    IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(ChartType.Doughnut, 50, 50, 400, 400);
    
    chart.getChartData().getSeriesGroups().get_Item(0).setDoughnutHoleSize((byte)90);

    // Αποθηκεύστε την παρουσίαση στον δίσκο
    pres.save("DoughnutHoleSize_out.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```

## **Συχνές ερωτήσεις**

**Μπορώ να δημιουργήσω ντόνατ πολλαπλών επιπέδων με πολλαπλά δαχτυλίδια;**

Ναι. Προσθέστε πολλαπλές σειρές σε ένα ενιαίο διάγραμμα ντόνατ — κάθε σειρά γίνεται ένα ξεχωριστό δαχτυλίδι. Η σειρά των δαχτυλιδιών καθορίζεται από τη σειρά των σειρών στη συλλογή.

**Υποστηρίζεται ντόνατ «εκραγμένο» (ξεχωριστές φέτες);**

Ναι. Υπάρχει τύπος διαγράμματος Exploded Doughnut [chart type](https://reference.aspose.com/slides/el/java/com.aspose.slides/charttype/) και μια ιδιότητα έκρηξης στα σημεία δεδομένων· μπορείτε να χωρίσετε μεμονωμένες φέτες.

**Πώς μπορώ να λάβω μια εικόνα ενός διαγράμματος ντόνατ (PNG/SVG) για μια αναφορά;**

Ένα διάγραμμα είναι σχήμα· μπορείτε να το αποδώσετε ως [raster image](https://reference.aspose.com/slides/el/java/com.aspose.slides/shape/#getImage-int-float-float-) ή να εξάγετε το διάγραμμα σε μια [SVG image](https://reference.aspose.com/slides/el/java/com.aspose.slides/shape/#writeAsSvg-java.io.OutputStream-com.aspose.slides.ISVGOptions-).
