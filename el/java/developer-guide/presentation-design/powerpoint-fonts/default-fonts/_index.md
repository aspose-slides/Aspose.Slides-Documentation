## **Επισκόπηση**

Το Aspose.Slides σας επιτρέπει να καθορίζετε τις προεπιλεγμένες γραμματοσειρές που χρησιμοποιούνται όταν αποδίδεται μια παρουσίαση. Αυτό είναι χρήσιμο κατά τη δημιουργία μικρογραφιών διαφανειών ή κατά την εξαγωγή μιας παρουσίασης σε μορφές όπως PDF και XPS. Οι προεπιλεγμένες γραμματοσειρές ρυθμίζονται μέσω του `LoadOptions` πριν φορτωθεί η παρουσίαση.

Η μέθοδος `setDefaultRegularFont` ορίζει τη προεπιλεγμένη γραμματοσειρά για κανονικό κείμενο, ενώ η `setDefaultAsianFont` ορίζει τη προεπιλεγμένη γραμματοσειρά για ασιατικό κείμενο. Αφού οριστούν αυτές οι επιλογές, η παρουσίαση μπορεί να φορτωθεί και να αποδοθεί χρησιμοποιώντας τις καθορισμένες γραμματοσειρές.

## **Χρήση Προεπιλεγμένων Γραμματοσειρών για Απόδοση Παρουσίασης**
Το Aspose.Slides σας επιτρέπει να ορίσετε τη προεπιλεγμένη γραμματοσειρά για την απόδοση της παρουσίασης σε PDF, XPS ή μικρογραφίες. Αυτό το άρθρο δείχνει πώς να ορίσετε τις DefaultRegular Font και DefaultAsian Font για χρήση ως προεπιλεγμένες γραμματοσειρές. Παρακαλούμε ακολουθήστε τα παρακάτω βήματα για τη φόρτωση γραμματοσειρών από εξωτερικούς φακέλους χρησιμοποιώντας το Aspose.Slides for Java API:

1. Δημιουργήστε μια παρουσία του [LoadOptions](https://reference.aspose.com/slides/el/java/com.aspose.slides/LoadOptions).
1. Ορίστε το [DefaultRegularFont](https://reference.aspose.com/slides/el/java/com.aspose.slides/LoadOptions#setDefaultRegularFont-java.lang.String-) στην επιθυμητή γραμματοσειρά σας. Στο παρακάτω παράδειγμα, χρησιμοποίησα Wingdings.
1. Ορίστε το [DefaultAsianFont](https://reference.aspose.com/slides/el/java/com.aspose.slides/LoadOptions#setDefaultAsianFont-java.lang.String-) στην επιθυμητή γραμματοσειρά σας. Χρησιμοποίησα Wingdings στο παρακάτω παράδειγμα.
1. Φορτώστε την παρουσίαση χρησιμοποιώντας το Presentation και ορίζοντας τις επιλογές φόρτωσης.
1. Τώρα, δημιουργήστε τη μικρογραφία της διαφάνειας, το PDF και το XPS για να επαληθεύσετε τα αποτελέσματα.

Η υλοποίηση του παραπάνω δίνεται παρακάτω.

```java
// Χρησιμοποιήστε επιλογές φόρτωσης για να ορίσετε τις προεπιλεγμένες κανονικές και ασιατικές γραμματοσειρές
LoadOptions loadOptions = new LoadOptions(LoadFormat.Auto);
loadOptions.setDefaultRegularFont("Wingdings");
loadOptions.setDefaultAsianFont("Wingdings");

// Φορτώστε την παρουσίαση
Presentation pres = new Presentation("DefaultFonts.pptx", loadOptions);
try {
    // Δημιουργήστε μικρογραφία διαφάνειας
    IImage slideImage = pres.getSlides().get_Item(0).getImage(1, 1);
    try {
         // αποθηκεύστε την εικόνα στο δίσκο.
          slideImage.save("output.png", ImageFormat.Png);
    } finally {
         if (slideImage != null) slideImage.dispose();
    }

    // Δημιουργήστε PDF
    pres.save("output_out.pdf", SaveFormat.Pdf);

    // Δημιουργήστε XPS
    pres.save("output_out.xps", SaveFormat.Xps);
} catch (IOException e) {
} finally {
    if (pres != null) pres.dispose();
}
```

## **Συχνές Ερωτήσεις**

**Τι ακριβώς επηρεάζουν οι DefaultRegularFont και DefaultAsianFont — μόνο την εξαγωγή, ή και τις μικρογραφίες, PDF, XPS, HTML και SVG;**

Συμμετέχουν στη διαδικασία απόδοσης για όλες τις υποστηριζόμενες εξόδους. Αυτό περιλαμβάνει μικρογραφίες διαφανειών, [PDF](/slides/el/java/convert-powerpoint-to-pdf/), [XPS](/slides/el/java/convert-powerpoint-to-xps/), [εικόνες raster](/slides/el/java/convert-powerpoint-to-png/), [HTML](/slides/el/java/convert-powerpoint-to-html/), και [SVG](/slides/el/java/render-a-slide-as-an-svg-image/), επειδή το Aspose.Slides χρησιμοποιεί την ίδια λογική διάταξης και επίλυσης γλυφών για όλους αυτούς τους προορισμούς.

**Εφαρμόζονται οι προεπιλεγμένες γραμματοσειρές όταν απλώς διαβάζετε και αποθηκεύετε ένα PPTX χωρίς καμία απόδοση;**

Όχι. Οι προεπιλεγμένες γραμματοσειρές έχουν σημασία όταν πρέπει να μετρηθεί και να σχεδιαστεί το κείμενο. Μια απλή ενέργεια ανοίγματος‑αποθήκευσης μιας παρουσίασης δεν αλλάζει τις αποθηκευμένες ακολουθίες γραμματοσειρών ή τη δομή του αρχείου. Οι προεπιλεγμένες γραμματοσειρές παρεμβαίνουν κατά την εκτέλεση λειτουργιών που αποδίδουν ή επαναδιατάσσουν το κείμενο.

**Αν προσθέσω τους δικούς μου φακέλους γραμματοσειρών ή παρέχω γραμματοσειρές από μνήμη, θα ληφθούν υπόψη κατά την επιλογή των προεπιλεγμένων γραμματοσειρών;**

Ναι. Οι [Προσαρμοσμένες πηγές γραμματοσειρών](/slides/el/java/custom-font/) επεκτείνουν τον κατάλογο των διαθέσιμων οικογενειών και γλυφών που μπορεί να χρησιμοποιήσει η μηχανή. Οι προεπιλεγμένες γραμματοσειρές και οποιοσδήποτε [κανόνες εναλλακτικών](/slides/el/java/fallback-font/) θα επιλύονται πρώτα με αυτές τις πηγές, παρέχοντας πιο αξιόπιστη κάλυψη σε διακομιστές και σε κοντέινερ.

**Θα επηρεάσουν οι προεπιλεγμένες γραμματοσειρές τις μετρικές του κειμένου (απόσταση γραμμάτων, προχωρήσεις) και συνεπώς τις αλλαγές γραμμής και την αναδίπλωση;**

Ναι. Η αλλαγή της γραμματοσειράς αλλάζει τις μετρικές των γλυφών και μπορεί να τροποποιήσει τις αλλαγές γραμμής, την αναδίπλωση και την σελιδοποίηση κατά την απόδοση. Για σταθερότητα διάταξης, [ενσωματώστε τις αρχικές γραμματοσειρές](/slides/el/java/embedded-font/) ή επιλέξτε προεπιλεγμένες και εναλλακτικές οικογένειες συμβατές μετρικά.

**Υπάρχει κάποιο όφελος στο να ορίζονται προεπιλεγμένες γραμματοσειρές εάν όλες οι γραμματοσειρές που χρησιμοποιούνται στην παρουσίαση είναι ενσωματωμένες;**

Συχνά δεν είναι απαραίτητο, επειδή οι [ενσωματωμένες γραμματοσειρές](/slides/el/java/embedded-font/) ήδη διασφαλίζουν συνεπή εμφάνιση. Οι προεπιλεγμένες γραμματοσειρές εξακολουθούν να βοηθούν ως δίχτυ ασφαλείας για χαρακτήρες που δεν καλύπτονται από το ενσωματωμένο υποσύνολο ή όταν ένα αρχείο συνδυάζει ενσωματωμένο και μη ενσωματωμένο κείμενο.
