## **Εισαγωγή**

Οι εφεδρικές γραμματοσειρές χρησιμοποιούνται όταν η γραμματοσειρά που καθορίζεται για το κείμενο είναι διαθέσιμη στο σύστημα αλλά δεν περιέχει το απαιτούμενο σύμβολο. Σε αυτήν την περίπτωση, το Aspose.Slides μπορεί να χρησιμοποιήσει μία από τις καθορισμένες εφεδρικές γραμματοσειρές για να αντικαταστήσει το έλλειμμα σύμβολο.

## **Εφεδρική Γραμματοσειρά**

Το Aspose.Slides επιτρέπει τη δημιουργία εφεδρικών γραμματοσειρών, την προσθήκη τους στη συλλογή εφεδρικών γραμματοσειρών, τον καθορισμό της συλλογής εφεδρικής γραμματοσειράς για μία συγκεκριμένη παρουσίαση, την αφαίρεση εφεδρικών γραμματοσειρών από την παρουσίαση, τον ορισμό των κανόνων εφαρμογής εφεδρικών γραμματοσειρών και άλλα.

Για να εξοικειωθείτε με αυτές τις δυνατότητες, χρησιμοποιήστε τους παρακάτω συνδέσμους:

- [Δημιουργία Συλλογής Εφεδρικών Γραμματοσειρών](/slides/el/java/create-fallback-fonts-collection)

## **Συχνές Ερωτήσεις**

**Πώς διαφέρουν οι εφεδρικές γραμματοσειρές από την αντικατάσταση γραμματοσειρών;**

Η εφεδρική γραμματοσειρά εφαρμόζεται ανά χαρακτήρα ή ανά περιοχή Unicode όταν η κύρια γραμματοσειρά δεν διαθέτει συγκεκριμένα σύμβολα· καλύπτει μόνο τους χαρακτήρες που λείπουν. [Αντικατάσταση](/slides/el/java/font-substitution/) αντικαθιστά μια ελλειπούσα ή μη διαθέσιμη γραμματοσειρά για ολόκληρο τμήμα ή τμήμα κειμένου με μια άλλη γραμματοσειρά. Μπορούν να συνδυαστούν, αλλά το πεδίο εφαρμογής και η λογική επιλογής διαφέρουν.

**Αποθηκεύονται οι ρυθμίσεις εφεδρικής γραμματοσειράς μέσα στο αρχείο παρουσίασης;**

Όχι. Η διαμόρφωση εφεδρικής γραμματοσειράς υπάρχει μόνο κατά την επεξεργασία/απόδοση στη βιβλιοθήκη και δεν σειριοποιείται στο PPTX. Η παρουσίαση δεν αποθηκεύει τους κανόνες εφεδρικής γραμματοσειράς.

**Επηρεάζει η εφεδρική γραμματοσειρά στοιχεία που δημιουργούνται από αντικείμενα του PowerPoint (SmartArt, διαγράμματα, WordArt);**

Ναι. Το κείμενο μέσα σε αυτά τα αντικείμενα περνάει μέσω της ίδιας διαδικασίας απόδοσης, επομένως οι ίδιες κανόνες εφεδρικής γραμματοσειράς εφαρμόζονται σε αυτό όπως και στο κανονικό κείμενο.
