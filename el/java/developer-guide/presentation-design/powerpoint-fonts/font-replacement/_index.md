## **Επισκόπηση**

Το Aspose.Slides σάς επιτρέπει να αντικαταστήσετε μία γραμματοσειρά με άλλη σε όλη την παρουσίαση. Όταν μια γραμματοσειρά αντικαθίσταται, όλες οι εμφανίσεις της αρχικής γραμματοσειράς αλλάζουν στη νέα γραμματοσειρά.

Για να εκτελέσετε αντικατάσταση γραμματοσειράς, φορτώστε την παρουσίαση, ορίστε τη γραμματοσειρά πηγής και τη γραμματοσειρά αντικατάστασης, καλέστε τη μέθοδο αντικατάστασης γραμματοσειράς και αποθηκεύστε την τροποποιημένη παρουσίαση ως αρχείο PPTX. Αυτή η προσέγγιση είναι χρήσιμη όταν θέλετε σκόπιμα να μεταβείτε από μια οικογένεια γραμματοσειρών σε άλλη σε όλη την παρουσίαση.

## **Αντικατάσταση Γραμματοσειρών**

Αν αλλάξετε γνώμη σχετικά με τη χρήση μιας γραμματοσειράς, μπορείτε να την αντικαταστήσετε με άλλη γραμματοσειρά. Όλες οι εμφανίσεις της παλιάς γραμματοσειράς θα αντικατασταθούν από τη νέα γραμματοσειρά.

Το Aspose.Slides σάς επιτρέπει να αντικαταστήσετε μια γραμματοσειρά με αυτόν τον τρόπο:

1. Φορτώστε τη σχετική παρουσίαση. 
2. Φορτώστε τη γραμματοσειρά που θα αντικατασταθεί. 
3. Φορτώστε τη νέα γραμματοσειρά. 
4. Αντικαταστήστε τη γραμματοσειρά. 
5. Αποθηκεύστε την τροποποιημένη παρουσίαση ως αρχείο PPTX.

Αυτός ο κώδικας Java δείχνει την αντικατάσταση γραμματοσειράς:

```java
// Φορτώνει μια παρουσίαση
Presentation pres = new Presentation("Fonts.pptx");
try {
    // Φορτώνει τη γραμματοσειρά προέλευσης που θα αντικατασταθεί
    IFontData sourceFont = new FontData("Arial");
    
    // Φορτώνει τη νέα γραμματοσειρά
    IFontData destFont = new FontData("Times New Roman");
    
    // Αντικαθιστά τις γραμματοσειρές
    pres.getFontsManager().replaceFont(sourceFont, destFont);
    
    // Αποθηκεύει την παρουσίαση
    pres.save("UpdatedFont_out.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```

{{% alert title="Note" color="warning" %}} 

Για να ορίσετε κανόνες που καθορίζουν τι συμβαίνει σε ορισμένες συνθήκες (π.χ. αν δεν είναι εφικτή η πρόσβαση σε μια γραμματοσειρά), δείτε [**Υποκατάσταση Γραμματοσειράς**](/slides/el/java/font-substitution/). 

{{% /alert %}}

## **Συχνές Ερωτήσεις**

**Ποια είναι η διαφορά μεταξύ «αντικατάστασης γραμματοσειράς», «υποκατάστασης γραμματοσειράς» και «εφεδρικών γραμματοσειρών»;**

Η αντικατάσταση είναι μια σκόπιμη μετάβαση από μία οικογένεια γραμματοσειρών σε άλλη σε ολόκληρο το έγγραφο. Η [Υποκατάσταση](/slides/el/java/font-substitution/) είναι ένας κανόνας όπως «αν η γραμματοσειρά δεν είναι διαθέσιμη, χρησιμοποιήστε X». Η [Εφεδρική γραμματοσειρά](/slides/el/java/fallback-font/) εφαρμόζεται στοχευμένα για μεμονωμένα λείποντα γλύφους όταν η βασική γραμματοσειρά είναι εγκατεστημένη αλλά δεν περιέχει τους απαιτούμενους χαρακτήρες.

**Η αντικατάσταση εφαρμόζεται σε master διαφάνειες, διατάξεις, σημειώσεις και σχόλια;**

Ναι. Η αντικατάσταση επηρεάζει όλα τα αντικείμενα παρουσίασης που χρησιμοποιούν την αρχική γραμματοσειρά, συμπεριλαμβανομένων των master διαφανειών και των σημειώσεων· τα σχόλια επίσης αποτελούν μέρος του εγγράφου και λαμβάνονται υπόψη από τη μηχανή γραμματοσειρών.

**Θα αλλάξει η γραμματοσειρά μέσα σε ενσωματωμένα αντικείμενα OLE (π.χ. Excel);**

Όχι. Το [περιεχόμενο OLE](/slides/el/java/manage-ole/) ελέγχεται από τη δική του εφαρμογή. Η αντικατάσταση στην παρουσίαση δεν επαναμορφοποιεί τα εσωτερικά δεδομένα OLE· μπορεί να εμφανιστεί ως εικόνα ή ως εξωτερικά επεξεργάσιμο περιεχόμενο.

**Μπορώ να αντικαταστήσω μια γραμματοσειρά μόνο σε μέρος της παρουσίασης (ανά διαφάνειες ή περιοχές);**

Στοχευμένη αντικατάσταση είναι δυνατή εάν αλλάξετε τη γραμματοσειρά στο επίπεδο των απαιτούμενων αντικειμένων/εύρους αντί να εφαρμόσετε καθολική αντικατάσταση σε ολόκληρο το έγγραφο. Η συνολική λογική επιλογής γραμματοσειράς κατά το rendering παραμένει η ίδια.

**Πώς μπορώ να καθορίσω εκ των προτέρων ποιες γραμματοσειρές χρησιμοποιεί η παρουσίαση;**

Χρησιμοποιήστε τον [διαχειριστή γραμματοσειρών] της παρουσίασης (https://reference.aspose.com/slides/el/java/com.aspose.slides/fontsmanager/): παρέχει λίστα με τις [οικογένειες σε χρήση](https://reference.aspose.com/slides/el/java/com.aspose.slides/fontsmanager/#getFonts--) και πληροφορίες για τις [υποκαταστάσεις/«άγνωστες» γραμματοσειρές](https://reference.aspose.com/slides/el/java/com.aspose.slides/fontsmanager/#getSubstitutions--), κάτι που βοηθά στον προγραμματισμό της αντικατάστασης.

**Λειτουργεί η αντικατάσταση γραμματοσειράς κατά τη μετατροπή σε PDF/ εικόνες;**

Ναι. Κατά την εξαγωγή, το Aspose.Slides εφαρμόζει την ίδια [ακολουθία επιλογής/υποκατάστασης γραμματοσειράς](/slides/el/java/font-selection-sequence/), έτσι ώστε μια αντικατάσταση που εκτελέστηκε εκ των προτέρων να τηρηθεί κατά τη μετατροπή.

**Πρέπει να εγκαταστήσω τη στοχευμένη γραμματοσειρά στο σύστημα ή μπορώ να συνημψώ έναν φάκελο γραμματοσειρών;**

Η εγκατάσταση δεν απαιτείται: η βιβλιοθήκη επιτρέπει τη [φόρτωση εξωτερικών γραμματοσειρών](/slides/el/java/custom-font/) από φακέλους χρήστη για χρήση κατά το [rendering και την εξαγωγή](/slides/el/java/convert-powerpoint/).

**Θα διορθώσει η αντικατάσταση το «tofu» (τετράγωνα) αντί για χαρακτήρες;**

Μόνο εφόσον η στοχευμένη γραμματοσειρά περιέχει πραγματικά τα απαιτούμενα γλύφους. Αν όχι, [ρυθμίστε την εφεδρική γραμματοσειρά](/slides/el/java/fallback-font/) για να καλύψετε τους λείποντες χαρακτήρες.
