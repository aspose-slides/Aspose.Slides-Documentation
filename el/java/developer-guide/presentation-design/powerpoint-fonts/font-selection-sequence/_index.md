## **Επισκόπηση**

Όταν μια παρουσίαση φορτώνεται, αποδίδεται ή μετατρέπεται σε άλλη μορφή, το Aspose.Slides ελέγχει αν οι γραμματοσειρές που χρησιμοποιούνται στην παρουσίαση είναι διαθέσιμες στο λειτουργικό σύστημα. Εάν λείψει μια απαιτούμενη γραμματοσειρά, το Aspose.Slides επιλέγει μια εναλλακτική γραμματοσειρά που είναι όσο το δυνατόν πιο κοντά σε αυτή που θα χρησιμοποιούσε το PowerPoint.

Το Aspose.Slides πρώτα αναζητά τη επιλεγμένη γραμματοσειρά στο λειτουργικό σύστημα. Εάν η γραμματοσειρά βρεθεί, χρησιμοποιείται. Εάν δεν βρεθεί, εφαρμόζεται μια κατάλληλη εναλλακτική. Όταν οι κανόνες αντικατάστασης γραμματοσειρών ορίζονται μέσω `FontSubstRule`, λαμβάνονται επίσης υπόψη.

Μπορείτε επίσης να προσθέσετε γραμματοσειρές κατά τη διάρκεια εκτέλεσης της εφαρμογής, να χρησιμοποιήσετε ενσωματωμένες γραμματοσειρές από μια παρουσίαση ή να φορτώσετε εξωτερικές γραμματοσειρές για έγγραφα εξόδου όπως αρχεία PDF.

## **Επιλογή Γραμματοσειράς**

Ορισμένοι κανόνες ισχύουν για τις γραμματοσειρές σε μια παρουσίαση όταν η παρουσίαση φορτώνεται, αποδίδεται ή μετατρέπεται σε άλλη μορφή. Για παράδειγμα, όταν προσπαθείτε να μετατρέψετε μια παρουσίαση (τις διαφάνειές της) σε εικόνες, ελέγχονται οι γραμματοσειρές της παρουσίασης για να επιβεβαιωθεί ότι οι επιλεγμένες γραμματοσειρές είναι διαθέσιμες στο λειτουργικό σύστημα. Εάν οι γραμματοσειρές διαπιστωθεί ότι λείπουν, αντικαθίστανται — δείτε [**Αντικατάσταση Γραμματοσειράς**](https://docs.aspose.com/slides/el/java/font-replacement/) και [**Αντικατάσταση Γραμματοσειράς**](https://docs.aspose.com/slides/el/java/font-substitution/).

Αυτή είναι η διαδικασία που ακολουθεί το Aspose.Slides όταν χειρίζεται γραμματοσειρές:

1. Το Aspose.Slides αναζητά γραμματοσειρές στο λειτουργικό σύστημα για να βρει τη γραμματοσειρά που ταιριάζει στη γραμματοσειρά που επέλεξε η παρουσίαση. 
2. Εάν η επιλεγμένη γραμματοσειρά βρεθεί, το Aspose.Slides τη χρησιμοποιεί. Διαφορετικά, το Aspose.Slides χρησιμοποιεί μια εναλλακτική γραμματοσειρά που είναι όσο το δυνατόν πιο κοντά σε αυτή που θα χρησιμοποιούσε το PowerPoint.
3. Εάν έχουν οριστεί κανόνες αντικατάστασης γραμματοσειράς μέσω [FontSubstRule](https://reference.aspose.com/slides/el/java/com.aspose.slides/fontsubstrule/), εφαρμόζονται. 

Το Aspose.Slides σας επιτρέπει να προσθέσετε γραμματοσειρές στο χρόνο εκτέλεσης της εφαρμογής και στη συνέχεια να τις χρησιμοποιήσετε. Δείτε [**Προσαρμοσμένες γραμματοσειρές**](https://docs.aspose.com/slides/el/java/custom-font/). 

Όταν επιπλέον γραμματοσειρές τοποθετούνται μέσα σε μια παρουσίαση, ονομάζονται [**Ενσωματωμένες γραμματοσειρές**](https://docs.aspose.com/slides/el/java/embedded-font/).

Το Aspose.Slides σας επιτρέπει να προσθέσετε γραμματοσειρές που εφαρμόζονται μόνο στα έγγραφα εξόδου. Για παράδειγμα, εάν μια παρουσίαση που θέλετε να μετατρέψετε σε PDF περιέχει γραμματοσειρές που λείπουν από το σύστημά σας και τις ενσωματωμένες γραμματοσειρές, μπορείτε να προσθέσετε ή να φορτώσετε τις απαιτούμενες γραμματοσειρές ως **εξωτερικές γραμματοσειρές**. 

{{% alert title="Note" color="primary" %}} 
Δεν διανέμουμε καμία γραμματοσειρά, είτε επί πληρωμή είτε δωρεάν. Το API μας σας επιτρέπει να φορτώνετε εξωτερικές γραμματοσειρές και να τις ενσωματώνετε σε έγγραφα, αλλά το κάνετε με τις γραμματοσειρές με δική σας διακριτική ευχέρεια και ευθύνη.
{{% /alert %}}

## **Συχνές Ερωτήσεις**

**Πώς μπορώ να προσδιορίσω ποιες γραμματοσειρές χρησιμοποιούνται πραγματικά σε μια παρουσίαση πριν από τη μετατροπή;**

Το Aspose.Slides σας επιτρέπει να εξετάσετε τις γραμματοσειρές μέσω του [διαχειριστή γραμματοσειρών](https://reference.aspose.com/slides/el/java/com.aspose.slides/fontsmanager/), ώστε να αποφασίσετε αν θα [ενσωματώσετε](/slides/el/java/embedded-font/), [αντικαταστήσετε](/slides/el/java/font-replacement/) ή προσθέσετε [εξωτερικές πηγές](/slides/el/java/custom-font/). Αυτό βοηθά να αποτρέψετε ανεπιθύμητες αντικαταστάσεις κατά την απόδοση και την εξαγωγή.

**Μπορώ να προσθέσω επιπλέον φακέλους γραμματοσειρών χωρίς να τους εγκαταστήσω στο λειτουργικό σύστημα;**

Ναι. Μπορείτε να καταχωρίσετε [εξωτερικές πηγές γραμματοσειρών](/slides/el/java/custom-font/) όπως φακέλους ή ροές σε μνήμη για απόδοση και εξαγωγή. Αυτό αφαιρεί την εξάρτηση από τις γραμματοσειρές του κεντρικού συστήματος και διατηρεί την διάταξη προβλέψιμη.

**Πώς μπορώ να αποτρέψω μια σιωπηρή επιστροφή σε ακατάλληλη γραμματοσειρά όταν λείπει κάποιο γλύφο;**

Ορίστε ρητές [αντικαταστάσεις γραμματοσειρών](/slides/el/java/font-replacement/) και κανόνες [εναλλακτικής γραμματοσειράς](/slides/el/java/fallback-font/) εκ των προτέρων. Αναλύοντας τις χρησιμοποιούμενες γραμματοσειρές και θέτοντας ελεγχόμενη προτεραιότητα για τους υποκατάστατες, διασφαλίζετε συνεπή τυπογραφία και αποφεύγετε ανεπιθύμητα αποτελέσματα.
