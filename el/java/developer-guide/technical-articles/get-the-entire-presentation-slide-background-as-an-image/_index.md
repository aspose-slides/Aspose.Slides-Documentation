## **Επισκόπηση**

Σε παρουσιάσεις PowerPoint, το παρασκήνιο μιας διαφάνειας μπορεί να αποτελείται από πολλά στοιχεία, συμπεριλαμβανομένης της εικόνας φόντου της διαφάνειας, του θέματος παρουσίασης, του χρωματικού σχήματος και των αντικειμένων που τοποθετούνται στην κύρια διαφάνεια ή στη διαφάνεια διάταξης.

Αυτό το άρθρο δείχνει πώς να εξάγετε ολόκληρο το φόντο μιας διαφάνειας ως εικόνα χρησιμοποιώντας το Aspose.Slides για .NET. Δεδομένου ότι δεν υπάρχει μία ενιαία μέθοδος για αυτήν την εργασία, η προσέγγιση περιλαμβάνει την κλωνοποίηση της επιλεγμένης διαφάνειας σε μια προσωρινή παρουσίαση, την αφαίρεση των σχημάτων της διαφάνειας και στη συνέχεια τη μετατροπή του προκύποντος φόντου διαφάνειας σε εικόνα.

## **Απόκτηση Ολόκληρου Φόντου Διαφάνειας**

Το Aspose.Slides for Java δεν παρέχει απλή μέθοδο για την εξαγωγή ολόκληρου του φόντου μιας διαφάνειας παρουσίασης ως εικόνα, αλλά μπορείτε να ακολουθήσετε τα παρακάτω βήματα για να το κάνετε:
1. Φορτώστε την παρουσίαση χρησιμοποιώντας την κλάση [Presentation](https://reference.aspose.com/slides/el/java/com.aspose.slides/presentation/).
1. Αποκτήστε το μέγεθος της διαφάνειας από την παρουσίαση.
1. Επιλέξτε μια διαφάνεια.
1. Δημιουργήστε μια προσωρινή παρουσίαση.
1. Ορίστε το ίδιο μέγεθος διαφάνειας στην προσωρινή παρουσίαση.
1. Κλωνοποιήστε τη επιλεγμένη διαφάνεια στην προσωρινή παρουσίαση.
1. Διαγράψτε τα σχήματα από τη κλωνοποιημένη διαφάνεια.
1. Μετατρέψτε τη κλωνοποιημένη διαφάνεια σε εικόνα.

Το παρακάτω παράδειγμα κώδικα εξάγει ολόκληρο το φόντο μιας διαφάνειας παρουσίασης ως εικόνα.
```java
var slideIndex = 0;
var imageScale = 1;

var presentation = new Presentation("sample.pptx");

var slideSize = presentation.getSlideSize().getSize();
var slide = presentation.getSlides().get_Item(slideIndex);

var tempPresentation = new Presentation();

var slideWidth = (float)slideSize.getWidth();
var slideHeight = (float)slideSize.getHeight();
tempPresentation.getSlideSize().setSize(slideWidth, slideHeight, SlideSizeScaleType.DoNotScale);

var clonedSlide = tempPresentation.getSlides().addClone(slide);
clonedSlide.getShapes().clear();

var background = clonedSlide.getImage(imageScale, imageScale);
background.save("output.png", ImageFormat.Png);

tempPresentation.dispose();
presentation.dispose();
```

## **Συχνές Ερωτήσεις**

**Θα διατηρηθούν πολύπλοκα διαβαθμισμένα χρώματα, υφές ή γεμίσματα εικόνας από μια κύρια διαφάνεια στην τελική εικόνα φόντου;**

Ναι. Το Aspose.Slides αποδίδει γεμίσματα διαβαθμίσεων, εικόνων και υφών που ορίζονται στη διαφάνεια, τη διάταξη ή την κύρια διαφάνεια. Εάν χρειάζεται να απομονώσετε την εμφάνιση από κληρονομημένες κύριες διαφάνειες, [ορίστε ένα δικό σας φόντο](/slides/el/java/presentation-background/) στην τρέχουσα διαφάνεια πριν την εξαγωγή.

**Μπορώ να προσθέσω υδατογράφημα στην τελική εικόνα φόντου πριν την αποθηκεύσω;**

Ναι. Μπορείτε να [προσθέσετε ένα υδατογράφημα](/slides/el/java/watermark/) σχήμα ή εικόνα σε ένα ενεργό [αντίγραφο της διαφάνειας](/slides/el/java/clone-slides/) (τοποθετημένο πίσω από το υπόλοιπο περιεχόμενο) και στη συνέχεια να εξάγετε. Αυτό σας επιτρέπει να δημιουργήσετε μια εικόνα φόντου με το υδατογράφημα ενσωματωμένο.

**Μπορώ να λάβω το φόντο για μια συγκεκριμένη διάταξη ή κύρια διαφάνεια χωρίς να το συνδέσω με μια υπάρχουσα διαφάνεια;**

Ναι. Έχετε πρόσβαση στην επιθυμητή κύρια διαφάνεια ή διάταξη, εφαρμόστε την σε μια [προσωρινή διαφάνεια](/slides/el/java/clone-slides/) με το απαιτούμενο μέγεθος και εξάγετε αυτή τη διαφάνεια για να λάβετε το φόντο που προέρχεται από τη συγκεκριμένη διάταξη ή κύρια διαφάνεια.

**Υπάρχουν περιορισμοί αδειοδότησης που επηρεάζουν την εξαγωγή εικόνας;**
