## **Επισκόπηση**

Το Aspose.Slides μπορεί να χρησιμοποιηθεί σε λειτουργία αξιολόγησης ή με έγκυρη άδεια. Η έκδοση αξιολόγησης παρέχει την ίδια λειτουργικότητα με την άδεια έκδοση, αλλά προσθέτει υδατογράφημα αξιολόγησης όταν ανοίγονται ή αποθηκεύονται οι παρουσιάσεις και περιορίζει την εξαγωγή κειμένου σε μία διαφάνεια.

Αυτό το άρθρο εξηγεί πώς λειτουργεί η αδειοδότηση στο Aspose.Slides και πώς να εφαρμόσετε άδεια πριν χρησιμοποιήσετε τη βιβλιοθήκη. Μια άδεια μπορεί να φορτωθεί από αρχείο, ροή ή ενσωματωμένο πόρο χρησιμοποιώντας την κλάση `License`. Το άρθρο δείχνει επίσης πώς να επικυρώσετε εάν μια άδεια έχει εφαρμοστεί σωστά.

## **Αδειοδότηση στο Aspose.Slides**

* Μια έκδοση αξιολόγησης γίνεται άδεια αφού αγοράσετε μια άδεια και προσθέσετε μερικές γραμμές κώδικα (για την εφαρμογή της άδειας).
* Η άδεια είναι ένα αρχείο XML απλού κειμένου που περιέχει λεπτομέρειες όπως το όνομα προϊόντος, ο αριθμός των προγραμματιστών για τους οποίους αδειοδοτείται, η ημερομηνία λήξης της συνδρομής κ.λπ. 
* Το αρχείο άδειας είναι ψηφιακά υπογεγραμμένο, επομένως δεν πρέπει να τροποποιηθεί. Ακόμη και μια τυχαία προσθήκη μιας νέας γραμμής στο περιεχόμενο του αρχείου θα το ακυρώσει.
* Το Aspose.Slides for Java συνήθως προσπαθεί να εντοπίσει την άδεια στις ακόλουθες τοποθεσίες:
  * Ένας ρητός διαδρομή
  * Ο φάκελος που περιέχει το Aspose.Slides.jar
* Για να αποφύγετε τους περιορισμούς της έκδοσης αξιολόγησης, πρέπει να ορίσετε μία άδεια πριν χρησιμοποιήσετε το **Aspose.Slides**. Απαιτείται μόνο μία φορά ανά εφαρμογή ή διαδικασία.

## **Εφαρμογή Άδειας**

Μια άδεια μπορεί να φορτωθεί από **αρχείο** ή **ροή**.

{{% alert color="primary" %}}

Το Aspose.Slides παρέχει την κλάση [License](https://reference.aspose.com/slides/el/java/com.aspose.slides/License) για λειτουργίες αδειοδότησης.

{{% /alert %}} 

{{% alert color="warning" %}}

Νέες άδειες μπορούν να ενεργοποιήσουν το Aspose.Slides μόνο με την έκδοση 21.4 ή νεότερη. Παλαιότερες εκδόσεις χρησιμοποιούν διαφορετικό σύστημα αδειοδότησης και δεν θα αναγνωρίσουν αυτές τις άδειες.

{{% /alert %}}

### **Αρχείο**

Η πιο εύκολη μέθοδος ορισμού άδειας απαιτεί να τοποθετήσετε το αρχείο άδειας στον φάκελο που περιέχει το Aspose.Slides.jar ή το jar της εφαρμογής σας.

Αυτός ο κώδικας Java δείχνει πώς να ορίσετε ένα αρχείο άδειας:

``` java
// Δημιουργεί μια παρουσία της κλάσης License
com.aspose.slides.License license = new com.aspose.slides.License();

// Ορίζει τη διαδρομή του αρχείου άδειας
license.setLicense("Aspose.Slides.Java.lic");
```

{{% alert color="warning" %}} 

Αν τοποθετήσετε το αρχείο άδειας σε διαφορετικό κατάλογο, όταν καλέσετε τη μέθοδο [SetLicense](https://reference.aspose.com/slides/el/java/com.aspose.slides/License#setLicense-java.lang.String-) το όνομα του αρχείου άδειας στο τέλος του καθορισμένου ρητού πρέπει να είναι το ίδιο με το όνομα του αρχείου άδειας σας.

Για παράδειγμα, μπορείτε να αλλάξετε το όνομα του αρχείου άδειας σε *Aspose.Slides.Java.lic.xml*. Στη συνέχεια, στον κώδικά σας, πρέπει να περάσετε τη διαδρομή προς το αρχείο (ολοκληρώνοντας με *Aspose.Slides.Java.lic.xml*) στη μέθοδο [SetLicense](https://reference.aspose.com/slides/el/java/com.aspose.slides/License#setLicense-java.lang.String-).

{{% /alert %}}

### **Ροή**

Μπορείτε να φορτώσετε μια άδεια από ροή. Αυτός ο κώδικας Java δείχνει πώς να εφαρμόσετε μια άδεια από ροή:

``` java
// Δημιουργεί μια παρουσία της κλάσης License
com.aspose.slides.License license = new com.aspose.slides.License();

// Ορίζει την άδεια μέσω ροής
license.setLicense(new java.io.FileInputStream("Aspose.Slides.Java.lic"));
```

### **PHP/Java Bridge**

Αν χρησιμοποιείτε το Aspose.Slides για PHP μέσω Java, μπορείτε να ορίσετε άδεια μέσω μιας γέφυρας PHP/Java. Αυτή η γέφυρα σας επιτρέπει να χρησιμοποιείτε κλάσεις Java σε σύνταξη PHP. Για περισσότερες πληροφορίες, δείτε το [License in PHP](/slides/el/php-java/licensing/).

## **Επικύρωση Άδειας**

Για να ελέγξετε εάν μια άδεια έχει οριστεί σωστά, μπορείτε να την επικυρώσετε. Αυτός ο κώδικας Java δείχνει πώς να επικυρώσετε μια άδεια:

```java
License license = new License();
license.setLicense("Aspose.Slides.Java.lic");

if (License.isLicensed()) 
{
    System.out.println("License is good!");
}
```

## **Ασφάλεια Νήματος**

{{% alert title="Note" color="warning" %}} 

Η μέθοδος [SetLicense](https://reference.aspose.com/slides/el/java/com.aspose.slides/License#setLicense-java.io.InputStream-) δεν είναι ασφαλής για χρήση από πολλαπλά νήματα. Αν αυτή η μέθοδος πρέπει να κληθεί ταυτόχρονα από πολλά νήματα, ίσως θέλετε να χρησιμοποιήσετε μηχανισμούς συγχρονισμού (όπως κλείδωμα) για να αποφύγετε προβλήματα. 

{{% /alert %}}
