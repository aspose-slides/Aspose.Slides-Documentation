## **Αξιολόγηση Aspose.Slides**

Μπορείτε εύκολα να κατεβάσετε το Aspose.Slides για αξιολόγηση. Η λήψη αξιολόγησης είναι η ίδια με την αγορασμένη λήψη. Η έκδοση αξιολόγησης γίνεται απλώς αδειοδοτημένη όταν προσθέσετε μερικές γραμμές κώδικα για την εφαρμογή της άδειας.

Η έκδοση αξιολόγησης του Aspose.Slides (χωρίς καθορισμένη άδεια) παρέχει πλήρη λειτουργικότητα του προϊόντος, αλλά προσθέτει ένα υδατογράφημα αξιολόγησης στην κορυφή του εγγράφου κατά το άνοιγμα και την αποθήκευση, και περιορίζει σε μία διαφάνεια όταν εξάγετε το κείμενο από τις διαφάνειες της παρουσίασης.

## **Συχνές Ερωτήσεις**

**Μπορώ να δοκιμάσω πολλαπλές παρουσιάσεις ταυτόχρονα σε διαφορετικά νήματα σε λειτουργία αξιολόγησης;**

Ναι. Μπορείτε να επεξεργάζεστε διαφορετικά έγγραφα παράλληλα· δεν πρέπει να μοιράζεστε το ίδιο αντικείμενο παρουσίασης [μέσα από νήματα](/slides/el/java/multithreading/). Η λειτουργία αξιολόγησης δεν το επηρεάζει.

**Χρειάζεται να εγκαταστήσω το Microsoft PowerPoint για να αξιολογήσω τη βιβλιοθήκη σε διακομιστή ή σε CI;**

Όχι. Το Aspose.Slides είναι μια ανεξάρτητη μηχανή και δεν απαιτείται το PowerPoint εγκατεστημένο ούτε για αξιολόγηση ούτε για παραγωγή.

**Μπορώ να δοκιμάσω πλήρως τη μετατροπή του PPT/PPTX σε PDF και εικόνες σε λειτουργία αξιολόγησης;**

Ναι. Οι [μετατροπείς](/slides/el/java/convert-presentation/) λειτουργούν· η έξοδος θα περιλαμβάνει υδατογράφημα.

**Μπορώ να χρησιμοποιήσω προσωρινή άδεια για δοκιμές φορτίου χωρίς υδατογράφημα;**

Ναι. Μια προσωρινή άδεια 30 ημερών αφαιρεί τους περιορισμούς της λειτουργίας αξιολόγησης και επιτρέπει δοκιμές χωρίς υδατογράφημα.
