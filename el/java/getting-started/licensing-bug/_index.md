## **Επισκόπηση**

Αυτό το άρθρο περιγράφει ένα σφάλμα αδειοδότησης που εμφανίστηκε στις εκδόσεις Aspose.Slides for Java 23.2, 23.3, 23.4, 23.5, 23.6 και 23.7. Αυτό το σφάλμα προκάλεσε τις διαρκείς άδειες να γίνουν προσωρινά μη χρησιμοποιήσιμες μετά τη λήξη της συνδρομής τους.

## **Συμπτώματα**

Μετά τη λήξη της συνδρομής της διαρκούς άδειας, οι εκδόσεις Aspose.Slides for Java 23.2 - 23.7 ενδέχεται να αναφέρουν την άδεια ως ληγμένη, μεταβαίνοντας όλες οι λειτουργίες σε λειτουργία αξιολόγησης. Αυτή η συμπεριφορά είναι λανθασμένη και δεν επηρεάζει τις εκδόσεις πριν από την 23.2 και μετά την 23.7.

## **Επίλυση**

Η Aspose.Slides for Java αντιμετώπισε αυτό το ζήτημα και κυκλοφόρησε ενημερωμένες εκδόσεις (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) με τη διόρθωση.

Εάν χρησιμοποιείτε μία από τις επηρεαζόμενες εκδόσεις της Aspose.Slides for Java στο έργο σας, παρακαλούμε μεταβείτε στη διορθωμένη έκδοση.

Εάν αντιμετωπίσετε προβλήματα αδειοδότησης με την Aspose.Slides for Java:

- Ελέγξτε την έκδοση που χρησιμοποιείτε και βεβαιωθείτε ότι δεν επηρεάζεται από το παραπάνω σφάλμα.
- Ανατρέξτε στην [τεκμηρίωση Aspose](https://docs.aspose.com/slides/el/java/getting-started/) για βήματα αντιμετώπισης προβλημάτων και γνωστά ζητήματα αδειοδότησης.
- Επικοινωνήστε με την [υποστήριξη Aspose](https://forum.aspose.com/) για περαιτέρω βοήθεια.

## **Προϊόντα και εκδόσεις που επηρεάζονται**

Παρακαλούμε σημειώστε ότι αυτό το σφάλμα επηρεάζει μόνο τις εκδόσεις Aspose.Slides for Java 23.2 έως 23.7. **Άλλα προϊόντα Aspose και άλλες εκδόσεις της Aspose.Slides δεν επηρεάζονται**.
