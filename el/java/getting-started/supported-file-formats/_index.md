## **Επισκόπηση**

Το Aspose.Slides υποστηρίζει αρχεία παρουσίασης από το Microsoft PowerPoint 97 έως το Office 365, συμπεριλαμβανομένου του Microsoft PowerPoint για Mac. Αυτό το άρθρο απαριθμεί τις εκδόσεις PowerPoint που υποστηρίζονται από τη βιβλιοθήκη και παρέχει έναν πίνακα με μορφές αρχείων που μπορούν να φορτωθούν, να αποθηκευτούν ή και τα δύο.

Το άρθρο επίσης απαντά σε κοινές ερωτήσεις σχετικά με τη συμμόρφωση PDF, την ενσωμάτωση γραμματοσειρών, αρχεία προστατευμένα με κωδικό, προσαρμοσμένες γραμματοσειρές, εναλλακτική γραμματοσειρά και τις επιλογές εξαγωγής XPS.

## **Υποστηριζόμενες Εκδόσεις Microsoft PowerPoint**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint για Mac
- Office 365

## **Συχνές Ερωτήσεις**

**Μπορώ να αποθηκεύσω παρουσιάσεις σε PDF που πληρούν τα πρότυπα αρχειοθέτησης και προσιτότητας (PDF/A και PDF/UA);**

Ναι. Το Aspose.Slides υποστηρίζει εξαγωγή σε PDF με επίπεδα συμμόρφωσης όπως PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b, καθώς και PDF/UA μέσω της ρύθμισης [compliance](https://reference.aspose.com/slides/el/java/com.aspose.slides/pdfoptions/#setCompliance-int-) στις [επιλογές εξαγωγής PDF](https://reference.aspose.com/slides/el/java/com.aspose.slides/pdfoptions/).

**Η βιβλιοθήκη υποστηρίζει ενσωμάτωση γραμματοσειρών κατά την εξαγωγή σε PDF, με λεπτομερή έλεγχο του τι ενσωματώνεται;**

Ναι. Μπορείτε να ελέγξετε αν οι γραμματοσειρές θα ενσωματωθούν πλήρως ή με υποσύνολο (μόνο τα χρησιμοποιούμενα γλύφοι), να καθορίσετε πώς αντιμετωπίζονται οι κοινές γραμματοσειρές συστήματος και να ρυθμίσετε τη συμπεριφορά για κείμενο ASCII μέσω των [επιλογών εξαγωγής PDF](https://reference.aspose.com/slides/el/java/com.aspose.slides/pdfoptions/).

**Μπορώ να ανιχνεύσω εάν ένα αρχείο είναι προστατευμένο με κωδικό πριν το φορτώσω;**

Ναι. Χρησιμοποιώντας το [factory-based inspection API](https://reference.aspose.com/slides/el/java/com.aspose.slides/presentationfactory/), μπορείτε να ερωτήσετε ένα αρχείο παρουσίασης για να προσδιορίσετε εάν είναι προστατευμένο με κωδικό χωρίς να το ανοίξετε πλήρως.

**Υπάρχουν μηχανισμοί εναλλακτικής γραμματοσειράς και υποστήριξη προσαρμοσμένων γραμματοσειρών;**

Ναι. Η βιβλιοθήκη υποστηρίζει [φόρτωση](/slides/el/java/custom-font/) και [ενσωμάτωση](/slides/el/java/embedded-font/) προσαρμοσμένων γραμματοσειρών και παρέχει [κανόνες εναλλακτικής γραμματοσειράς](/slides/el/java/fallback-font/) για την αποφυγή ελλιπών γλυφών κατά τη σχεδίαση και τη μετατροπή.

**Μπορώ να εξάγω διαφάνειες σε XPS και υπάρχουν επιλογές για τη ρύθμιση του εξόδου XPS;**

Ναι. Η [εξαγωγή σε XPS](/slides/el/java/convert-powerpoint-to-xps/) υποστηρίζεται, και μπορείτε να προσαρμόσετε τις σχετικές [επιλογές αποθήκευσης](https://reference.aspose.com/slides/el/java/com.aspose.slides/xpsoptions/) για να ελέγξετε την ποιότητα εξόδου και το περιεχόμενο του εγγράφου XPS.
