|**Δυνατότητα**|**Περιγραφή**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/el/java/microsoft-powerpoint-ppt/)|Το Aspose.Slides for Java παρέχει την πιο γρήγορη επεξεργασία για αυτή τη μορφή παρουσίασης.|
|[PresentationML (PPTX, XML)](/slides/el/java/presentationml-pptx-xml/)|Το Aspose.Slides for Java υποστηρίζει την επεξεργασία της μορφής παρουσίασης OOXML (γνωστή και ως PresentationML ή PPTX).|
|[PPT to PPTX conversion](/slides/el/java/ppt-to-pptx-conversion/)|Το Aspose.Slides for Java υποστηρίζει τη μετατροπή από PPT σε PPTX.|
|[Portable Document Format (PDF)](/slides/el/java/developer-guide/)|Οι υποστηριζόμενες μορφές αρχείων μπορούν να εξαχθούν σε έγγραφα Adobe Portable Document Format (PDF) με μία μέθοδο.|
|[XML Parser Specification (XPS)](/slides/el/java/xml-parser-specification-xps/)|Όλες οι υποστηριζόμενες μορφές αρχείων μπορούν να εξαχθούν σε έγγραφα XML Parser Specification (XPS) με μία μέθοδο.|
|[Tagged Image File Format (TIFF)](https://docs.aspose.com/slides/el/java/convert-powerpoint-to-tiff/)|Οι μορφές αρχείων παρουσίασης που υποστηρίζονται από το Aspose.Slides for Java μπορούν επίσης να εξαχθούν σε Tagged Image File Format (TIFF).|
|[ODP to PPTX Conversion](https://docs.aspose.com/slides/el/java/convert-odp-to-pptx/)|Το Aspose.Slides for Java υποστηρίζει τη φόρτωση OpenDocument Presentation (ODP) και τη μετατροπή του σε PPTX.|

|**Δυνατότητα**|**Περιγραφή**|
| :- | :- |

## **FAQ**

**Χρειάζεται να εγκαταστήσω το Microsoft PowerPoint στον διακομιστή/PC για να λειτουργήσει η βιβλιοθήκη;**

Όχι. Δεν απαιτείται το PowerPoint· το Aspose.Slides είναι μια αυτόνομη μηχανή για δημιουργία, επεξεργασία, μετατροπή και απόδοση παρουσιάσεων.

**Πώς λειτουργεί ο πολυνηματισμός; Μπορεί η επεξεργασία να παραλληλοποιηθεί;**

Είναι ασφαλές να επεξεργάζεστε διαφορετικά έγγραφα σε διαφορετικά νήματα· το ίδιο αντικείμενο [presentation]https://reference.aspose.com/slides/el/java/com.aspose.slides/presentation/ δεν πρέπει να χρησιμοποιείται από [multiple threads](/slides/el/java/multithreading/) ταυτόχρονα.

**Υποστηρίζονται κωδικοί πρόσβασης αρχείων και κρυπτογράφηση;**

Ναι. [You can](/slides/el/java/password-protected-presentation/) να ανοίξετε κρυπτογραφημένες παρουσιάσεις, να ορίσετε ή να αφαιρέσετε κωδικό ανοίγματος και εγγραφής, και να ελέγξετε την κατάσταση προστασίας.

**Πρέπει να ανησυχείω για πακέτα γραμματοσειρών σε κοντέινερ Linux;**

Ναι. Συνίσταται η εγκατάσταση κοινών πακέτων γραμματοσειρών και/ή η ρητή [specify font directories](/slides/el/java/custom-font/) στην εφαρμογή σας για να αποφευχθούν μη αναμενόμενες αντικαταστάσεις.

**Υπάρχουν περιορισμοί στην έκδοση αξιολόγησης;**

Σε [evaluation mode](/slides/el/java/licensing/), προστίθεται υδατογράφημα στο αποτέλεσμα και εφαρμόζονται ορισμένοι περιορισμοί· διαθέσιμος είναι ένας [30‑day temporary license]https://purchase.aspose.com/temporary-license/ για πλήρη δοκιμή όλων των λειτουργιών.

**Υποστηρίζεται η εισαγωγή εξωτερικών μορφών σε παρουσίαση (PDF/HTML → PPTX);**

Ναι. Μπορείτε να προσθέσετε [PDF pages and HTML content](/slides/el/java/import-presentation/) σε παρουσίαση, μετατρέποντάς τα σε διαφάνειες.
