## **Σχετικά με TIFF**

Το [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) το οποίο είναι γνωστό για την αποθήκευση πολλαπλών εικόνων σε ένα έγγραφο δημιουργήθηκε αρχικά από την Aldus. Αυτή η μορφή υποστηρίζεται ευρέως από εφαρμογές σάρωσης, φαξ και άλλες εφαρμογές επεξεργασίας εικόνας.

## **TIFF στο Aspose.Slides for Java**

Οποιοδήποτε έγγραφο μπορεί να φορτωθεί στο Aspose.Slides for Java μπορεί επίσης να μετατραπεί απευθείας σε έγγραφο TIFF από το Aspose.Slides for Java, εξαλείφοντας την ανάγκη για οποιοδήποτε εξωτερικό στοιχείο. Επιπλέον, μπορείτε προαιρετικά να ορίσετε το μέγεθος των εικόνων στο παραγόμενο έγγραφο TIFF. Μπορείτε να βρείτε πληροφορίες για την εξαγωγή των εγγράφων παρουσίασης σε έγγραφα TIFF μέσω του Aspose.Slides for Java στο [αυτό το θέμα](/slides/el/java/converting-a-presentation/).
