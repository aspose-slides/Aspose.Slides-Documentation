Η μετατροπή PPT σε PPTX υποστηρίζεται από το Aspose.Slides for Java. Οι περισσότερες λειτουργίες παρουσίασης - master slides, structure and so on - διατηρούνται όταν μετατρέπονται από τη μία μορφή στην άλλη, αλλά υπάρχουν [μερικοί περιορισμοί](/slides/el/java/ppt-to-pptx-conversion/).
{{% /alert %}} 
## **Χαρακτηριστικά που υποστηρίζονται στη μετατροπή**
Το Aspose.Slides for Java παρέχει μερική υποστήριξη για τη μετατροπή του μορφότυπου αρχείου PPT σε PPTX. Η υποστήριξη για τη μετατροπή μόλις εισήχθη στο Aspose.Slides for Java, επομένως έχει ορισμένους περιορισμούς και λειτουργεί καλύτερα για απλές παρουσιάσεις. Το κύριο πλεονέκτημα που προσφέρει το Aspose.Slides for Java κατά τη μετατροπή PPT σε PPTX είναι πόσο εύκολο είναι το API στη χρήση. Για να δείτε παραδείγματα κώδικα, διαβάστε για [Μετατροπή PPT σε PPTX](). Παρακάτω, οι λίστες εξηγούν ποια χαρακτηριστικά υποστηρίζονται και ποια όχι στη μετατροπή PPT σε PPTX.

## **Υποστηριζόμενα Χαρακτηριστικά**
Τα ακόλουθα χαρακτηριστικά υποστηρίζονται για τη μετατροπή:

- Μετατροπή της δομής των masters, layouts και slides.
- Μετατροπή διαγραμμάτων.
- Ομαδοποίηση σχημάτων.
- Μετατροπή αυτόματων σχημάτων, συμπεριλαμβανομένων των ορθογωνίων και ελλείων.
- Σχήματα με προσαρμοσμένη γεωμετρία.
- Στυλ γεμίσματος με υφές και εικόνες για αυτόματα σχήματα.
- Μετατροπή placeholders.
- Μετατροπή γραμμών και πολυγραμμών.
- Μορφές γραμμής και γεμίσματος.
- Στυλ γεμίσματος διαβάθμισης.
- Πλαίσια OLE, πίνακες, πλαίσια βίντεο και ήχου κ.α.
- Ιδιότητες animation και παρουσίασης.
- Μετατροπή κειμένου σε πλαίσια κειμένου και κουτιά κειμένου.
