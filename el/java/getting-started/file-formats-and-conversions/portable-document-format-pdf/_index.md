{{% alert color="primary" %}} 

Το [Portable Document Format](https://en.wikipedia.org/wiki/PDF) είναι ένας τύπος αρχείου που δημιουργήθηκε από την Adobe Systems για την ανταλλαγή εγγράφων μεταξύ οργανισμών. Ο σκοπός του τύπου ήταν να διατηρεί το περιεχόμενο και τη διάταξη αμετάβλητα, ανεξάρτητα από την πλατφόρμα στην οποία προβάλλεται. Το Aspose.Slides for Java σας επιτρέπει να μετατρέπετε αρχεία παρουσιάσεων σε PDF.

{{% /alert %}} 

## **PDF στο Aspose.Slides for Java**
Οποιαδήποτε παρουσίαση μπορεί να φορτωθεί στο Aspose.Slides for Java και να μετατραπεί σε PDF που συμμορφώνεται με [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) ή [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA) ανάλογα με την επιλογή σας. Το Aspose.Slides for Java εξάγει παρουσιάσεις σε PDF και στις περισσότερες περιπτώσεις το παραγόμενο PDF μοιάζει ακριβώς με την αρχική παρουσίαση.

Το Aspose.Slides υποστηρίζει τις παρακάτω δυνατότητες παρουσίασης κατά τη μετατροπή σε PDF:

- Εικόνες, πλαίσια κειμένου και άλλα σχήματα.
- Κείμενο και μορφοποίηση.
- Παραγράφους και μορφοποίηση.
- Υπερσυνδέσμους.
- Κεφαλίδες και υποσέλιδα.
- Κουκκίδες.
- Πίνακες.

Μπορείτε να εξάγετε παρουσιάσεις σε PDF απευθείας χρησιμοποιώντας το Aspose.Slides for Java: δεν χρειάζεστε κανένα άλλο στοιχείο. Επιπλέον, μπορείτε να προσαρμόσετε την εξαγωγή παρουσίασης σε PDF με διάφορες επιλογές όπως εξηγείται στη σελίδα [Converting to PDF](/slides/el/java/converting-a-presentation/).
