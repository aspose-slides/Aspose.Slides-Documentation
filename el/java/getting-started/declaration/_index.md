{{% alert color="primary" %}} 
Όλα τα στοιχεία Aspose Java απαιτούν σύνολο δικαιωμάτων Full Trust. Ο λόγος είναι ότι τα στοιχεία Aspose Java χρειάζεται να έχουν πρόσβαση σε ρυθμίσεις μητρώου, σε συστημικά αρχεία εκτός του εικονικού καταλόγου για ορισμένες λειτουργίες όπως η ανάλυση γραμματοσειρών κ.λπ. Επιπλέον, τα στοιχεία Aspose Java βασίζονται σε βασικές κλάσεις του συστήματος Java που επίσης απαιτούν σύνολο δικαιωμάτων Full Trust σε πολλές περιπτώσεις. 
{{% /alert %}} 

Οι Πάροχοι Υπηρεσιών Διαδικτύου που φιλοξενούν πολλαπλές εφαρμογές από διαφορετικές εταιρείες συνήθως επιβάλλουν επίπεδο ασφαλείας Medium Trust: 

- OleDbPermission δεν είναι διαθέσιμη. Αυτό σημαίνει ότι δεν μπορείτε να χρησιμοποιήσετε τον διαχειρισμένο πάροχο δεδομένων OLE DB της ADO.NET για πρόσβαση σε βάσεις δεδομένων.  
- EventLogPermission δεν είναι διαθέσιμη. Αυτό σημαίνει ότι δεν μπορείτε να έχετε πρόσβαση στο αρχείο καταγραφής γεγονότων των Windows.  
- ReflectionPermission δεν είναι διαθέσιμη. Αυτό σημαίνει ότι δεν μπορείτε να χρησιμοποιήσετε την αντανακλαστικότητα.  
- RegistryPermission δεν είναι διαθέσιμη. Αυτό σημαίνει ότι δεν μπορείτε να έχετε πρόσβαση στο μητρώο.  
- WebPermission είναι περιορισμένη. Αυτό σημαίνει ότι η εφαρμογή σας μπορεί να επικοινωνεί μόνο με μια διεύθυνση ή εύρος διευθύνσεων που ορίζετε στο στοιχείο <trust>.  
- FileIOPermission είναι περιορισμένη. Αυτό σημαίνει ότι μπορείτε να έχετε πρόσβαση μόνο σε αρχεία στην ιεραρχία του εικονικού καταλόγου της εφαρμογής σας.  

{{% alert color="primary" %}} 
Λόγω των παραπάνω λόγων, τα στοιχεία Aspose Java δεν μπορούν να χρησιμοποιηθούν σε διακομιστές που παρέχουν σύνολο δικαιωμάτων διαφορετικό από το Full Trust. 
{{% /alert %}}
