{{% alert color="primary" %}} 

Το Aspose.Slides for Java είναι μια βιβλιοθήκη κλάσεων που επιτρέπει στις εφαρμογές Java να διαβάζουν και να γράφουν έγγραφα παρουσιάσεων χωρίς τη χρήση του Microsoft PowerPoint® ή άλλων. Το Aspose.Slides for Java επιτρέπει τη διαχείριση κειμένου, σχημάτων, 
πινάκων & κινούμενων σχεδίων, την προσθήκη ήχου και βίντεο στις διαφάνειες, την προεπισκόπηση των διαφανειών, την εξαγωγή των διαφανειών σε μορφή SVG, PDF και άλλα.

Η πιο πρόσφατη Αναφορά API μπορεί να βρεθεί στο [Aspose.Slides for Java API Reference](https://reference.aspose.com/slides/el/java).

{{% /alert %}}
