Aspose.Slides for Node.js via Java είναι μια βιβλιοθήκη κλάσεων που επιτρέπει στις εφαρμογές Node.js να διαβάζουν και να γράφουν έγγραφα παρουσίασης χωρίς να χρησιμοποιούν το Microsoft PowerPoint® ή άλλα. Aspose.Slides for Node.js via Java σας επιτρέπει να διαχειρίζεστε κείμενο, σχήματα, πίνακες & κινούμενα σχέδια, να προσθέτετε ήχο και βίντεο στις διαφάνειες, να προεπισκοπείτε τις διαφάνειες, να εξάγετε τις διαφάνειες σε μορφές SVG, PDF και πολλά άλλα.

Η πιο πρόσφατη Αναφορά API μπορεί να βρεθεί στο [Αναφορά API του Aspose.Slides for Node.js via Java](https://reference.aspose.com/slides/el/nodejs-java/).
