## **Αξιολόγηση Aspose.Slides**
Μπορείτε εύκολα να κατεβάσετε το Aspose.Slides για αξιολόγηση. Το πακέτο αξιολόγησης είναι το ίδιο με το αγορασμένο πακέτο. Η έκδοση αξιολόγησης απλώς γίνεται αδειοδοτημένη αφού προσθέσετε μερικές γραμμές κώδικα για την εφαρμογή της άδειας. 

## **Περιορισμός Έκδοσης Αξιολόγησης**
Η έκδοση αξιολόγησης του Aspose.Slides (χωρίς καθορισμένη άδεια) παρέχει όλη τη λειτουργικότητα του προϊόντος, αλλά τοποθετεί ένα υδατογράφημα αξιολόγησης στην κορυφή του εγγράφου κατά το άνοιγμα και την αποθήκευση. Επίσης, περιορίζεστε σε μία διαφάνεια όταν εξάγετε κείμενα από διαφάνειες παρουσίασης.

## **Σχετικά με την Άδεια**
Μπορείτε εύκολα να κατεβάσετε μια έκδοση αξιολόγησης του Aspose.Slides για Node.js μέσω Java από τη [σελίδα λήψης](https://releases.aspose.com/slides/el/nodejs-java/). Η έκδοση αξιολόγησης παρέχει απολύτως **τις ίδιες δυνατότητες** με την αδειοδοτημένη έκδοση του Aspose.Slides. Επιπλέον, η έκδοση αξιολόγησης απλώς γίνεται αδειοδοτημένη αφού αγοράσετε μια άδεια και προσθέσετε μερικές γραμμές κώδικα για την εφαρμογή της άδειας.

Η άδεια είναι ένα αρχείο XML απλού κειμένου που περιέχει λεπτομέρειες όπως το όνομα του προϊόντος, ο αριθμός των προγραμματιστών στους οποίους εκχωρείται, η ημερομηνία λήξης της συνδρομής κ.λπ. Το αρχείο είναι ψηφιακά υπογεγραμμένο, επομένως μην το τροποποιήσετε. Ακόμη και η τυχαία προσθήκη μιας πρόσθετης διακοπής γραμμής στο περιεχόμενο του αρχείου θα την ακυρώσει.

Για να αποφύγετε τους περιορισμούς που σχετίζονται με την έκδοση αξιολόγησης, πρέπει να ορίσετε μια άδεια πριν χρησιμοποιήσετε το **Aspose.Slides**. Απαιτείται να ορίσετε άδεια μόνο μία φορά ανά εφαρμογή ή διαδικασία.

### **Ρύθμιση Άδειας στο Aspose.Slides για Node.js μέσω Java**

Άδειες μπορούν να εφαρμοστούν από τις παρακάτω τοποθεσίες:

* Συγκεκριμένη διαδρομή
* Ροή
* Ως Μετρημένη Άδεια – ένας νέος μηχανισμός αδειοδότησης

{{% alert color="primary" %}}
Χρησιμοποιήστε τη μέθοδο **setLicense** για να αδειοδοτήσετε ένα συστατικό.

Παρά το γεγονός ότι πολλαπλές κλήσεις στη **setLicense** δεν προκαλούν προβλήματα, είναι σπατάλη πόρων (επεξεργαστή).
{{% /alert %}}

{{% alert color="warning" %}}
Νέες άδειες μπορούν να ενεργοποιήσουν το Aspose.Slides μόνο με την έκδοση 21.4 ή νεότερη. Οι παλαιότερες εκδόσεις χρησιμοποιούν διαφορετικό σύστημα αδειοδότησης και δεν θα αναγνωρίσουν αυτές τις άδειες.
{{% /alert %}}

#### **Εφαρμογή Άδειας μέσω Αρχείου**

Αυτό το απόσπασμα κώδικα χρησιμοποιείται για να ορίσει ένα αρχείο άδειας:

**Node.js**

```javascript
var aspose = aspose || {};

aspose.slides = require("aspose.slides.via.java");

var license = new aspose.slides.License();
license.setLicense("Aspose.Slides.lic");
```

Κατά την κλήση της μεθόδου setLicense, το όνομα της άδειας πρέπει να είναι ίδιο με αυτό του αρχείου άδειας σας. Για παράδειγμα, μπορείτε να αλλάξετε το όνομα του αρχείου άδειας σε "Aspose.Slides.lic.xml". Στη συνέχεια, στον κώδικά σας, πρέπει να περάσετε το νέο όνομα άδειας (Aspose.Slides.lic.xml) στη μέθοδο setLicense.

#### **Εφαρμογή Άδειας από Ροή**

Αυτό το απόσπασμα κώδικα χρησιμοποιείται για την εφαρμογή άδειας από ροή:

**Node.js**

```javascript
var aspose = aspose || {};

aspose.slides = require("aspose.slides.via.java");

var license = new aspose.slides.License();

var fs = require("fs");

var readStream = fs.createReadStream("Aspose.Slides.lic");

license.setLicense(readStream, function(err, list) {
    if(err) { 
        console.error(err); return; 
    }});
```
