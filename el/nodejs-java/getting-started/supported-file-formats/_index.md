## **Επισκόπηση**

Aspose.Slides υποστηρίζει αρχεία παρουσίασης από το Microsoft PowerPoint 97 έως το Office 365, συμπεριλαμβανομένου του Microsoft PowerPoint για Mac. Αυτό το άρθρο παραθέτει τις εκδόσεις PowerPoint που υποστηρίζονται από τη βιβλιοθήκη και παρέχει έναν πίνακα με τις μορφές αρχείων που μπορούν να φορτωθούν, να αποθηκευτούν ή και τα δύο.

Το άρθρο επίσης απαντά σε συχνές ερωτήσεις σχετικά με τη συμμόρφωση PDF, την ενσωμάτωση γραμματοσειρών, τα αρχεία με κωδικό πρόσβασης, τις προσαρμοσμένες γραμματοσειρές, την εναλλακτική γραμματοσειρά και τις επιλογές εξαγωγής XPS.

## **Υποστηριζόμενες Εκδόσεις Microsoft PowerPoint**

- Office 365

## **Συχνές Ερωτήσεις**

**Μπορώ να αποθηκεύσω παρουσιάσεις σε PDF που πληρούν τα πρότυπα αρχειοθέτησης και προσβασιμότητας (PDF/A και PDF/UA);**

Ναι. Το Aspose.Slides υποστηρίζει εξαγωγή σε PDF με επίπεδα συμμόρφωσης όπως PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b, καθώς και PDF/UA μέσω της ρύθμισης [compliance](https://reference.aspose.com/slides/el/nodejs-java/aspose.slides/pdfoptions/setcompliance/) στα [PDF export options](https://reference.aspose.com/slides/el/nodejs-java/aspose.slides/pdfoptions/).

**Η βιβλιοθήκη υποστηρίζει ενσωμάτωση γραμματοσειρών κατά την εξαγωγή σε PDF, με λεπτομερή έλεγχο του τι ενσωματώνεται;**

Ναι. Μπορείτε να ελέγξετε αν οι γραμματοσειρές ενσωματώνονται πλήρως ή υποσύνολο (μόνο τα χρησιμοποιούμενα γλύφους), να καθορίσετε πώς αντιμετωπίζονται οι κοινές σύστημα γραμματοσειρές και να ρυθμίσετε τη συμπεριφορά για κείμενο ASCII μέσω των [PDF export options](https://reference.aspose.com/slides/el/nodejs-java/aspose.slides/pdfoptions/).

**Μπορώ να εντοπίσω αν ένα αρχείο είναι προστατευμένο με κωδικό πρόσβασης πριν το φορτώσω;**

Ναι. Χρησιμοποιώντας το [factory-based inspection API](https://reference.aspose.com/slides/el/nodejs-java/aspose.slides/presentationfactory/), μπορείτε να ερωτήσετε ένα αρχείο παρουσίασης για να προσδιορίσετε αν είναι προστατευμένο με κωδικό πρόσβασης χωρίς να το ανοίξετε πλήρως.

**Υπάρχουν μηχανισμοί εναλλακτικής γραμματοσειράς και υποστήριξη προσαρμοσμένων γραμματοσειρών;**

Ναι. Η βιβλιοθήκη υποστηρίζει [loading](/slides/el/nodejs-java/custom-font/) και [embedding](/slides/el/nodejs-java/embedded-font/) προσαρμοσμένων γραμματοσειρών και παρέχει [fallback rules](/slides/el/nodejs-java/fallback-font/) για να αποτρέπει την έλλειψη γλύφων κατά την απόδοση και τη μετατροπή.

**Μπορώ να εξάγω διαφάνειες σε XPS, και υπάρχουν επιλογές για ρύθμιση της εξόδου XPS;**

Ναι. Η [Export to XPS](/slides/el/nodejs-java/convert-powerpoint-to-xps/) υποστηρίζεται και μπορείτε να προσαρμόσετε τις σχετικές [save options](https://reference.aspose.com/slides/el/nodejs-java/aspose.slides/xpsoptions/) για να ελέγξετε την ποιότητα και το περιεχόμενο του εγγράφου XPS.
