## **Αξιολόγηση Aspose.Slides**

Μπορείτε εύκολα να κατεβάσετε το Aspose.Slides για αξιολόγηση. Το πακέτο αξιολόγησης είναι το ίδιο με το αγορασμένο πακέτο. Η έκδοση αξιολόγησης γίνεται απλώς αδειοδοτημένη αφού προσθέσετε μερικές γραμμές κώδικα για την εφαρμογή της άδειας. 

Η έκδοση αξιολόγησης του Aspose.Slides (χωρίς καθορισμένη άδεια) παρέχει πλήρη λειτουργικότητα του προϊόντος, αλλά εισάγει ένα υδατογράφημα αξιολόγησης στην κορυφή του εγγράφου κατά το άνοιγμα και την αποθήκευση. Επιπλέον, περιορίζεστε σε μία διαφάνεια όταν εξάγετε κείμενο από τις διαφάνειες της παρουσίασης.

## **ΣΥΧΝΑ ΕΡΩΤΗΜΑΤΑ**

**Μπορώ να δοκιμάσω πολλές παρουσιάσεις ταυτόχρονα σε διαφορετικά νήματα σε κατάσταση αξιολόγησης;**

Ναι. Μπορείτε να επεξεργαστείτε διαφορετικά έγγραφα παράλληλα· δεν πρέπει να μοιράζεστε το ίδιο αντικείμενο παρουσίασης [μεταξύ νήματος](/slides/el/net/multithreading/). Η κατάσταση αξιολόγησης δεν το επηρεάζει.

**Χρειάζεται να εγκαταστήσω το Microsoft PowerPoint για να αξιολογήσω τη βιβλιοθήκη σε διακομιστή ή στο CI;**

Όχι. Το Aspose.Slides είναι μια αυτόνομη μηχανή και δεν απαιτεί εγκατάσταση του PowerPoint, είτε για αξιολόγηση είτε για παραγωγή.

**Μπορώ να δοκιμάσω πλήρως τη μετατροπή PPT/PPTX σε PDF και εικόνες σε κατάσταση αξιολόγησης;**

Ναι. Οι [μετατροπείς](/slides/el/net/convert-presentation/) λειτουργούν· η έξοδος θα περιλαμβάνει ένα υδατογράφημα.

**Μπορώ να χρησιμοποιήσω προσωρινή άδεια για δοκιμή φορτίου χωρίς υδατογράφημα;**

Ναι. Μια 30ήμερη προσωρινή άδεια αφαιρεί τους περιορισμούς της λειτουργίας αξιολόγησης και επιτρέπει δοκιμές χωρίς υδατογράφημα.
