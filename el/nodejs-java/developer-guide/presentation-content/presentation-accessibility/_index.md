## **Επισκόπηση**

Η προσβασιμότητα των παρουσιάσεων διασφαλίζει ότι τα άτομα που χρησιμοποιούν βοηθητικές τεχνολογίες—όπως αναγνώστες οθόνης, οθόνες Braille ή πλοήγηση μόνο με πληκτρολόγιο—μπορούν να κατανοήσουν και να περιηγηθούν στις διαφάνειές σας εξίσου αποτελεσματικά με το κοινό που βλέπει και χρησιμοποιεί ποντίκι. Η καλή πρακτική εστιάζει σε σαφή σειρά ανάγνωσης, ουσιαστικό εναλλακτικό κείμενο για πληροφοριακά οπτικά στοιχεία, επαρκή αντίθεση χρωμάτων, ευανάγνωστη τυπογραφία, περιγραφικό κείμενο συνδέσμου και αποφυγή μεταφοράς νοήματος μόνο με χρώμα ή θέση. Όταν η προσβασιμότητα σχεδιάζεται από την αρχή, το αποτέλεσμα είναι μια πιο καθαρή δομή, πιο συνεπή οπτικά στοιχεία και περιεχόμενο που φθάνει σε κάθε θεατή χωρίς παρακάμψεις.

## **Σήμανση ως διακοσμητικό**

Η σημαία «Σήμανση ως διακοσμητικό» επισημαίνει καθαρά διακοσμητικά οπτικά στοιχεία ώστε οι αναγνώστες οθόνης να τα παραλείπουν, μειώνοντας το «θόρυβο» και διατηρώντας την εστίαση στο ουσιαστικό περιεχόμενο. Εφαρμόζεται σε φόντα, διακοσμητικά στοιχεία και διαχωριστικά—ποτέ σε διαγράμματα, εικονίδια ή εικόνες που μεταδίδουν πληροφορίες. Το Aspose.Slides εκθέτει αυτή τη σημαία για ανίχνευση και επικύρωση, επιτρέποντας αυτοματοποιημένους ελέγχους προσβασιμότητας και καθαρισμό.

![Σήμανση ως διακοσμητικό](mark_as_decorative.png)

Το παρακάτω δείγμα κώδικα δείχνει πώς να προσδιορίσετε εάν ένα σχήμα είναι σημειωμένο ως διακοσμητικό.

```js
let presentation = new aspose.slides.Presentation("sample.pptx");
try {
    let shape = presentation.getSlides().get_Item(0).getShapes().get_Item(0);
    console.log("Is shape decorative:", shape.isDecorative());
} finally {
    presentation.dispose();
}
```
