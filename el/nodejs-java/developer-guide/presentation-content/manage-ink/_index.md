## **Εισαγωγή**

Το PowerPoint παρέχει τη λειτουργία μελάνης που σας επιτρέπει να σχεδιάζετε μη τυπικές μορφές, οι οποίες μπορούν να χρησιμοποιηθούν για να επισημάνουν άλλα αντικείμενα, να δείξουν συνδέσεις και διαδικασίες, και να τραβήξουν την προσοχή σε συγκεκριμένα στοιχεία σε μια διαφάνεια. 

Το Aspose.Slides παρέχει όλους τους τύπους Ink (π.χ. κλάση [Ink](https://reference.aspose.com/slides/el/nodejs-java/aspose.slides/ink/) ) που χρειάζεστε για τη δημιουργία και διαχείριση αντικειμένων μελάνης.

## **Διαφορές μεταξύ Κανονικού Αντικειμένου και Αντικειμένων Μελάνης**

Τα αντικείμενα σε μια διαφάνεια PowerPoint αντιπροσωπεύονται συνήθως από αντικείμενα σχήματος. Ένα αντικείμενο σχήματος, στην πιο απλή του μορφή, είναι ένας κάτοχος που ορίζει την περιοχή του ίδιου του αντικειμένου (το πλαίσιο του) μαζί με τις ιδιότητές του. Αυτό περιλαμβάνει το μέγεθος της περιοχής του υποδοχέα, το σχήμα του υποδοχέα, το φόντο του υποδοχέα κ.λπ. Για περισσότερες πληροφορίες, δείτε το [Shape Layout Format](https://docs.aspose.com/slides/el/nodejs-java/shape-manipulations/#access-layout-formats-for-shape).

Ωστόσο, όταν το PowerPoint δουλεύει με ένα αντικείμενο μελάνης, αγνοεί όλες τις ιδιότητες του πλαισίου του αντικειμένου (υποδοχέα) εκτός από το μέγεθός του. Το μέγεθος της περιοχής του υποδοχέα καθορίζεται από τις τυπικές τιμές `width` και `height`:

## **Ανίχνευση Σχήματος Μελάνης**

Η ανίχνευση είναι ένα βασικό στοιχείο ή πρότυπο που χρησιμοποιείται για την καταγραφή της τροχιάς ενός στυλό καθώς ο χρήστης γράφει ψηφιακή μελάνη. Οι ανιχνεύσεις είναι καταγραφές που περιγράφουν ακολουθίες συνδεδεμένων σημείων. 

Η πιο απλή μορφή κωδικοποίησης καθορίζει τις συντεταγμένες X και Y κάθε σημείου δείγματος. Όταν όλα τα συνδεδεμένα σημεία αποδοθούν, παράγουν μια εικόνα όπως αυτή:

## **Ιδιότητες Βούρτσας για Σχεδίαση**

Μπορείτε να χρησιμοποιήσετε μια βούρτσα για να σχεδιάσετε γραμμές που συνδέουν τα σημεία των στοιχείων της ανίχνευσης. Η βούρτσα έχει το δικό της χρώμα και μέγεθος, τα οποία αντιστοιχούν στις μεθόδους `Brush.setColor` και `Brush.setSize`. 

### **Ορισμός Χρώματος Βούρτσας Μελάνης**

Αυτός ο κώδικας JavaScript δείχνει πώς να ορίσετε το χρώμα για μια βούρτσα:

```javascript
var pres = new aspose.slides.Presentation("pres.pptx");
try {
    var ink = pres.getSlides().get_Item(0).getShapes().get_Item(0);
    var traces = ink.getTraces();
    var brush = traces[0].getBrush();
    var brushColor = brush.getColor();
    brush.setColor(java.getStaticFieldValue("java.awt.Color", "RED"));
} finally {
    if (pres != null) {
        pres.dispose();
    }
}
```

### **Ορισμός Μεγέθους Βούρτσας Μελάνης**

Αυτός ο κώδικας JavaScript δείχνει πώς να ορίσετε το μέγεθος για μια βούρτσα:

```javascript
var pres = new aspose.slides.Presentation("pres.pptx");
try {
    var ink = pres.getSlides().get_Item(0).getShapes().get_Item(0);
    var traces = ink.getTraces();
    var brush = traces[0].getBrush();
    var brushSize = brush.getSize();
    brush.setSize(java.newInstanceSync("java.awt.Dimension", 5, 10));
} finally {
    if (pres != null) {
        pres.dispose();
    }
}
```

Γενικά, το πλάτος και το ύψος μιας βούρτσας δεν ταιριάζουν, επομένως το PowerPoint δεν εμφανίζει το μέγεθος της βούρτσας (το τμήμα δεδομένων είναι κυματισμένο). Όταν όμως το πλάτος και το ύψος της βούρτσας ταιριάζουν, το PowerPoint εμφανίζει το μέγεθός της με τον εξής τρόπο:

![ink_powerpoint3](ink_powerpoint3.png)

Για σαφήνεια, ας αυξήσουμε το ύψος του αντικειμένου μελάνης και ας εξετάσουμε τις σημαντικές διαστάσεις: 

![ink_powerpoint4](ink_powerpoint4.png)

Ο υποδοχέας (πλαίσιο) δεν λαμβάνει υπόψη το μέγεθος των βουρτσών—πρόκειται πάντα για μηδενικό πάχος της γραμμής (δείτε την τελευταία εικόνα). 

Ως εκ τούτου, για να προσδιορίσουμε την ορατή περιοχή ολόκληρου του αντικειμένου μελάνης, πρέπει να λάβουμε υπόψη το μέγεθος της βούρτσας των αντικειμένων ανίχνευσης. Εδώ, το αντικείμενο-στόχος (η ανίχνευση του χειρόγραφου κειμένου) έχει κλιμακωθεί στο μέγεθος του υποδοχέα (πλαισίου). Όταν το μέγεθος του υποδοχέα (πλαισίου) αλλάζει, το μέγεθος της βούρτσας παραμένει σταθερό και αντίστροφα. 

![ink_powerpoint5](ink_powerpoint5.png)

Το PowerPoint εμφανίζει την ίδια συμπεριφορά όταν ασχολείται με κείμενα:
