## **Επισκόπηση**

Αυτό το άρθρο δείχνει πώς να εργάζεστε με ένα διάγραμμα δακτυλίου στο Aspose.Slides προσθέτοντας το διάγραμμα σε μια διαφάνεια, ορίζοντας το μέγεθος του κεντρικού του κενού και αποθηκεύοντας την παρουσίαση. Επικεντρώνεται στη μέθοδο `setDoughnutHoleSize` και παρουσιάζει τα βασικά βήματα που απαιτούνται για την προσαρμογή αυτού του τύπου διαγράμματος στον κώδικα.

Περιλαμβάνει επίσης μια σύντομη ενότητα Συχνές ερωτήσεις που καλύπτει σχετικές περιπτώσεις διαγράμματος δακτυλίου, όπως η χρήση πολλαπλών σειρών για τη δημιουργία πολλαπλών δαχτυλιδιών, η εργασία με εκραγμένα διαγράμματα δακτυλίου και η εξαγωγή ενός διαγράμματος ως εικόνα raster ή SVG.

## **Αλλαγή του κεντρικού κενού σε διάγραμμα δακτυλίου**

Για να ορίσετε το μέγεθος του κενού σε ένα διάγραμμα δακτυλίου, ακολουθήστε τα παρακάτω βήματα:

1. Δημιουργήστε ένα αντικείμενο [Presentation](https://reference.aspose.com/slides/el/nodejs-java/aspose.slides/presentation).
1. Προσθέστε διάγραμμα δακτυλίου στη διαφάνεια.
1. Καθορίστε το μέγεθος του κενού σε ένα διάγραμμα δακτυλίου.
1. Αποθηκεύστε την παρουσίαση στο δίσκο.

Στο παρακάτω παράδειγμα, έχουμε ορίσει το μέγεθος του κενού σε ένα διάγραμμα δακτυλίου.

```javascript
// Δημιουργία ενός αντικειμένου κλάσης Presentation
var pres = new aspose.slides.Presentation();
try {
    var chart = pres.getSlides().get_Item(0).getShapes().addChart(aspose.slides.ChartType.Doughnut, 50, 50, 400, 400);
    chart.getChartData().getSeriesGroups().get_Item(0).setDoughnutHoleSize(90);
    // Αποθήκευση παρουσίασης στο δίσκο
    pres.save("DoughnutHoleSize_out.pptx", aspose.slides.SaveFormat.Pptx);
} finally {
    if (pres != null) {
        pres.dispose();
    }
}
```

## **Συχνές ερωτήσεις**

**Μπορώ να δημιουργήσω ένα πολυεπίπεδο διάγραμμα δακτυλίου με πολλαπλά δαχτυλίδια;**

Ναι. Προσθέστε πολλαπλές σειρές σε ένα ενιαίο διάγραμμα δακτυλίου—κάθε σειρά γίνεται ένα ξεχωριστό δαχτυλίδι. Η σειρά των δαχτυλιδιών καθορίζεται από τη σειρά των σειρών στη συλλογή.

**Υποστηρίζεται ένα «εκραγμένο» διάγραμμα δακτυλίου (διαχωρισμένα τμήματα);**

Ναι. Υπάρχει ο τύπος διαγράμματος [Exploded Doughnut](https://reference.aspose.com/slides/el/nodejs-java/aspose.slides/charttype/) και η ιδιότητα έκρηξης στα σημεία δεδομένων· μπορείτε να διαχωρίσετε μεμονωμένα τμήματα.

**Πώς μπορώ να λάβω μια εικόνα ενός διαγράμματος δακτυλίου (PNG/SVG) για μια αναφορά;**

Ένα διάγραμμα είναι σχήμα· μπορείτε να το αποδώσετε ως [raster image](https://reference.aspose.com/slides/el/nodejs-java/aspose.slides/shape/#getImage) ή να εξάγετε το διάγραμμα σε μία [SVG image](https://reference.aspose.com/slides/el/nodejs-java/aspose.slides/shape/writeassvg/).
