## **Εισαγωγή**

Στο PowerPoint 2019, η Microsoft εισήγαγε τη ρύθμιση **Always Open Read-Only** ως μία από τις επιλογές που μπορούν να χρησιμοποιήσουν οι χρήστες για να προστατεύσουν τις παρουσιάσεις τους. Μπορεί να θέλετε να χρησιμοποιήσετε αυτή τη ρύθμιση Read-Only για να προστατεύσετε μια παρουσίαση όταν

- Θέλετε να αποτρέψετε τυχαίες επεξεργασίες και να κρατήσετε το περιεχόμενο της παρουσίασής σας ασφαλές.  
- Θέλετε να ενημερώσετε τους παραλήπτες ότι η παρουσίαση που παρέχετε είναι η τελική έκδοση.  

Αφού επιλέξετε την επιλογή **Always Open Read-Only** για μια παρουσίαση, όταν οι χρήστες ανοίγουν την παρουσίαση, βλέπουν την πρόταση **Read-Only** και μπορεί να εμφανιστεί το εξής μήνυμα: *Για να αποφύγετε τυχαίες αλλαγές, ο δημιουργός όρισε αυτό το αρχείο να ανοίγει ως μόνο-ανάγνωση.*

Η πρόταση Read-Only είναι ένας απλός αλλά αποτελεσματικός αποτρεπτικός μηχανισμός που αποθαρρύνει την επεξεργασία, επειδή οι χρήστες πρέπει να εκτελέσουν μια ενέργεια για να την αφαιρέσουν προτού μπορέσουν να επεξεργαστούν την παρουσίαση. Εάν δεν θέλετε οι χρήστες να κάνουν αλλαγές σε μια παρουσίαση και θέλετε να το τονίσετε με ευγενικό τρόπο, τότε η πρόταση Read-Only μπορεί να είναι μια καλή επιλογή για εσάς.

> Εάν μια παρουσίαση με την προστασία **Read-Only** ανοίξει σε παλαιότερη έκδοση του Microsoft PowerPoint—που δεν υποστηρίζει τη νεοεισαχθείσα λειτουργία—η πρόταση **Read-Only** αγνοείται (η παρουσίαση ανοίγει κανονικά).

## **Εφαρμογή λειτουργίας Read-Only**

Aspose.Slides for Node.js via Java σας επιτρέπει να ορίσετε μια παρουσίαση ως **Read-Only**, πράγμα που σημαίνει ότι οι χρήστες (αφού ανοίξουν την παρουσίαση) βλέπουν την πρόταση **Read-Only**. Αυτό το παράδειγμα κώδικα δείχνει πώς να ορίσετε μια παρουσίαση ως **Read-Only** σε JavaScript χρησιμοποιώντας το Aspose.Slides:

```javascript
var pres = new aspose.slides.Presentation();
try {
    pres.getProtectionManager().setReadOnlyRecommended(true);
    pres.save("ReadOnlyPresentation.pptx", aspose.slides.SaveFormat.Pptx);
} finally {
    if (pres != null) {
        pres.dispose();
    }
}
```

{{% alert color="primary" %}} 

**Σημείωση**: Η πρόταση **Read-Only** προορίζεται απλώς να αποθαρρύνει την επεξεργασία ή να εμποδίσει τους χρήστες από τυχαίες αλλαγές σε μια παρουσίαση PowerPoint. Εάν ένα πρόσωπο με κίνητρο—που γνωρίζει τι κάνει—αποφασίσει να επεξεργαστεί την παρουσίασή σας, μπορεί εύκολα να αφαιρέσει τη ρύθμιση Read-Only. Εάν χρειάζεστε σοβαρά την πρόληψη μη εξουσιοδοτημένης επεξεργασίας, είναι καλύτερο να χρησιμοποιήσετε [πιο αυστηρές προστασίες που περιλαμβάνουν κρυπτογραφήσεις και κωδικούς πρόσβασης](https://docs.aspose.com/slides/el/nodejs-java/password-protected-presentation/).

## **Συχνές ερωτήσεις**

**Πώς διαφέρει το 'Read-Only recommended' από την πλήρη προστασία με κωδικό;**

Το 'Read-Only recommended' εμφανίζει μόνο μια πρόταση να ανοίξετε το αρχείο σε λειτουργία μόνο-ανάγνωση και είναι εύκολο να παρακαμφθεί. Η [πρόσβαση με κωδικό](/slides/el/nodejs-java/password-protected-presentation/) περιορίζει πραγματικά το άνοιγμα ή την επεξεργασία και είναι κατάλληλη όταν χρειάζεστε αληθινά μέτρα ασφαλείας.

**Μπορεί το 'Read-Only recommended' να συνδυαστεί με υδατογραφήματα ώστε να αποθαρρύνει περαιτέρω τις επεξεργασίες;**

Ναι. Η πρόταση μπορεί να συνδυαστεί με [υδατογραφήματα](/slides/el/nodejs-java/watermark/) ως οπτικό αποτρεπτικό μέσο· είναι ξεχωριστοί μηχανισμοί και λειτουργούν καλά μαζί.

**Μπορεί ένα μακροεντολή ή εξωτερικό εργαλείο να τροποποιήσει ακόμα το αρχείο όταν η πρόταση είναι ενεργοποιημένη;**

Ναι. Η πρόταση δεν εμποδίζει προγραμματιστικές αλλαγές. Για να αποτρέψετε αυτοματοποιημένες επεμβάσεις, χρησιμοποιήστε [κωδικούς πρόσβασης και κρυπτογράφηση](/slides/el/nodejs-java/password-protected-presentation/).

**Πώς σχετίζεται το 'Read-Only recommended' με τις σημαίες 'IsEncrypted' και 'IsWriteProtected';**

Είναι διαφορετικά σήματα. Το 'Read-Only recommended' είναι ένα ήπιο, προαιρετικό προτροπικό μήνυμα· το [isWriteProtected](https://reference.aspose.com/slides/el/nodejs-java/aspose.slides/protectionmanager/iswriteprotected/) και το [isEncrypted](https://reference.aspose.com/slides/el/nodejs-java/aspose.slides/protectionmanager/isencrypted/) υποδεικνύουν πραγματικούς περιορισμούς εγγραφής ή ανάγνωσης που εξαρτώνται από κωδικούς πρόσβασης ή κρυπτογράφηση.
