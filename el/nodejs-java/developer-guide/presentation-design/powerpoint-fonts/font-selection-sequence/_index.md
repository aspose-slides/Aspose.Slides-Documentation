## **Επισκόπηση**

Όταν μια παρουσίαση φορτώνεται, αποδίδεται ή μετατρέπεται σε άλλη μορφή, το Aspose.Slides ελέγχει εάν οι γραμματοσειρές που χρησιμοποιούνται στην παρουσίαση είναι διαθέσιμες στο λειτουργικό σύστημα. Εάν λείπει κάποια απαιτούμενη γραμματοσειρά, το Aspose.Slides επιλέγει μια εναλλακτική γραμματοσειρά που είναι όσο το δυνατόν πιο κοντά σε αυτή που θα χρησιμοποιούσε το PowerPoint.

Το Aspose.Slides πρώτα αναζητά τη επιλεγμένη γραμματοσειρά στο λειτουργικό σύστημα. Εάν η γραμματοσειρά βρεθεί, χρησιμοποιείται. Εάν δεν βρεθεί, εφαρμόζεται μια κατάλληλη εναλλακτική. Όταν οι κανόνες υποκατάστασης γραμματοσειρών ορίζονται μέσω `FontSubstRule`, λαμβάνονται επίσης υπόψη.

Μπορείτε επίσης να προσθέσετε γραμματοσειρές κατά το χρόνο εκτέλεσης της εφαρμογής, να χρησιμοποιήσετε ενσωματωμένες γραμματοσειρές από μια παρουσίαση ή να φορτώσετε εξωτερικές γραμματοσειρές για έγγραφα εξόδου όπως αρχεία PDF.

## **Επιλογή γραμματοσειράς**

Ορισμένοι κανόνες ισχύουν για τις γραμματοσειρές σε μια παρουσίαση όταν η παρουσίαση φορτώνεται, αποδίδεται ή μετατρέπεται σε άλλη μορφή. Για παράδειγμα, όταν προσπαθείτε να μετατρέψετε μια παρουσίαση (τις διαφάνειές της) σε εικόνες, οι γραμματοσειρές της παρουσίασης ελέγχονται για να επαληθευτεί ότι οι επιλεγμένες γραμματοσειρές είναι διαθέσιμες στο λειτουργικό σύστημα. Εάν οι γραμματοσειρές επιβεβαιωθούν ότι λείπουν, αντικαθίστανται — δείτε [**Αντικατάσταση γραμματοσειράς**](https://docs.aspose.com/slides/el/nodejs-java/font-replacement/) και [**Υποκατάσταση γραμματοσειράς**](https://docs.aspose.com/slides/el/nodejs-java/font-substitution/).

Αυτή είναι η διαδικασία που ακολουθεί το Aspose.Slides κατά την αντιμετώπιση των γραμματοσειρών:

1. Το Aspose.Slides αναζητά γραμματοσειρές στο λειτουργικό σύστημα για να βρει τη γραμματοσειρά που ταιριάζει με τη γραμματοσειρά που επιλέχθηκε στην παρουσίαση. 
2. Εάν η επιλεγμένη γραμματοσειρά βρεθεί, το Aspose.Slides τη χρησιμοποιεί. Διαφορετικά, το Aspose.Slides χρησιμοποιεί μια εναλλακτική γραμματοσειρά που είναι όσο το δυνατόν πιο κοντά σε αυτή που θα χρησιμοποιούσε το PowerPoint.
3. Εάν έχουν οριστεί κανόνες αντικατάστασης γραμματοσειρών μέσω [FontSubstRule](https://reference.aspose.com/slides/el/nodejs-java/aspose.slides/fontsubstrule/), εφαρμόζονται.

Το Aspose.Slides σας επιτρέπει να προσθέσετε γραμματοσειρές κατά το χρόνο εκτέλεσης της εφαρμογής και στη συνέχεια να τις χρησιμοποιήσετε. Δείτε [**Προσαρμοσμένες γραμματοσειρές**](https://docs.aspose.com/slides/el/nodejs-java/custom-font/).

Όταν επιπλέον γραμματοσειρές τοποθετούνται μέσα σε μια παρουσίαση, ονομάζονται [**Ενσωματωμένες γραμματοσειρές**](https://docs.aspose.com/slides/el/nodejs-java/embedded-font/).

Το Aspose.Slides σας επιτρέπει να προσθέσετε γραμματοσειρές που εφαρμόζονται *μόνο* στα έγγραφα εξόδου. Για παράδειγμα, εάν μια παρουσίαση που θέλετε να μετατρέψετε σε PDF περιέχει γραμματοσειρές που λείπουν από το σύστημά σας και ενσωματωμένες γραμματοσειρές, μπορείτε να προσθέσετε ή να φορτώσετε τις απαιτούμενες γραμματοσειρές ως **εξωτερικές γραμματοσειρές**. 

{{% alert title="Note" color="primary" %}} 
Δεν διανείμουμε καμία γραμματοσειρά, είτε επί πληρωμή είτε δωρεάν. Το API μας επιτρέπει να φορτώνετε εξωτερικές γραμματοσειρές και να τις ενσωματώνετε σε έγγραφα, αλλά το κάνετε με τις γραμματοσειρές με δική σας επιλογή και ευθύνη.
{{% /alert %}}

## **Συχνές ερωτήσεις**

**Πώς μπορώ να προσδιορίσω ποιες γραμματοσειρές χρησιμοποιούνται πραγματικά σε μια παρουσίαση πριν τη μετατροπή;**

Το Aspose.Slides σας επιτρέπει να εξετάσετε τις χρησιμοποιούμενες γραμματοσειρές μέσω του [διαχειριστή γραμματοσειρών](https://reference.aspose.com/slides/el/nodejs-java/aspose.slides/presentation/getfontsmanager/), ώστε να αποφασίσετε αν θα [ενσωματώσετε](/slides/el/nodejs-java/embedded-font/), [αντικαταστήσετε](/slides/el/nodejs-java/font-replacement/) ή προσθέσετε [εξωτερικές πηγές](/slides/el/nodejs-java/custom-font/). Αυτό σας βοηθά να αποτρέψετε ανεπιθύμητες υποκαταστάσεις κατά την απόδοση και την εξαγωγή.

**Μπορώ να προσθέσω επιπλέον καταλόγους γραμματοσειρών χωρίς να τα εγκαταστήσω στο λειτουργικό σύστημα;**

Ναι. Μπορείτε να καταχωρίσετε [εξωτερικές πηγές γραμματοσειρών](/slides/el/nodejs-java/custom-font/) όπως φακέλους ή ροές μνήμης για την απόδοση και την εξαγωγή. Αυτό αφαιρεί την εξάρτηση από τις γραμματοσειρές του συστήματος υποδοχής και διατηρεί την διάταξη προβλέψιμη.

**Πώς μπορώ να αποτρέψω μια σιωπηρή υποκατάσταση με μη κατάλληλη γραμματοσειρά όταν λείπει ένα σύμβολο;**

Καθορίστε ρητά [αντικατάσταση γραμματοσειράς](/slides/el/nodejs-java/font-replacement/) και κανόνες [fallback γραμματοσειράς](/slides/el/nodejs-java/fallback-font/) εκ των προτέρων. Αναλύοντας τις χρησιμοποιούμενες γραμματοσειρές και ορίζοντας προτεραιότητα ελεγχόμενη για τις εναλλακτικές, διασφαλίζετε συνεπή τυπογραφία και αποφεύγετε ανεπιθύμητα αποτελέσματα.
