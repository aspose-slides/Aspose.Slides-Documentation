## **Εισαγωγή**

Οι εφεδρικές γραμματοσειρές χρησιμοποιούνται όταν η γραμματοσειρά που έχει οριστεί για το κείμενο είναι διαθέσιμη στο σύστημα αλλά δεν περιέχει το απαιτούμενο glyph. Σε αυτή την περίπτωση, η Aspose.Slides μπορεί να χρησιμοποιήσει μια από τις καθορισμένες εφεδρικές γραμματοσειρές για να αντικαταστήσει το ελλιπές glyph.

## **Εφεδρική γραμματοσειρά**

Η Aspose.Slides επιτρέπει τη δημιουργία εφεδρικών γραμματοσειρών, την προσθήκη τους στη συλλογή εφεδρικών γραμματοσειρών, τον ορισμό συλλογής εφεδρικών γραμματοσειρών για μια συγκεκριμένη παρουσίαση, την αφαίρεση εφεδρικών γραμματοσειρών από την παρουσίαση, τον καθορισμό των κανόνων εφαρμογής εφεδρικών γραμματοσειρών και άλλα.

## **Συχνές ερωτήσεις**

**Πώς διαφέρουν οι εφεδρικές γραμματοσειρές από την αντικατάσταση γραμματοσειρών;**

Η εφεδρική γραμματοσειρά εφαρμόζεται ανά χαρακτήρα ή ανά εύρος Unicode όταν η κύρια γραμματοσειρά δεν περιέχει συγκεκριμένα glyphs· συμπληρώνει μόνο τους χαρακτήρες που λείπουν. [Αντικατάσταση](/slides/el/nodejs-java/font-substitution/) αντικαθιστά μια ελλείπουσα ή μη διαθέσιμη γραμματοσειρά για ολόκληρη τη σειρά ή το τμήμα κειμένου με μια άλλη γραμματοσειρά. Μπορούν να συνδυαστούν, αλλά το πεδίο εφαρμογής και η λογική επιλογής τους διαφέρουν.

**Αποθηκεύονται οι ρυθμίσεις εφεδρείας μέσα στο αρχείο παρουσίασης;**

Όχι. Η διαμόρφωση εφεδρείας υφίσταται μόνο κατά την επεξεργασία/απόδοση στη βιβλιοθήκη και δεν σειριοποιείται στο PPTX. Η παρουσίαση δεν αποθηκεύει τους κανόνες εφεδρείας.

**Επηρεάζει η εφεδρεία στοιχεία που δημιουργούνται από αντικείμενα PowerPoint (SmartArt, διαγράμματα, WordArt);**

Ναι. Το κείμενο μέσα σε αυτά τα αντικείμενα περνάει από την ίδια διαδικασία απόδοσης, επομένως οι ίδιες κανόνες εφεδρείας ισχύουν για αυτό όπως και για το κανονικό κείμενο.
