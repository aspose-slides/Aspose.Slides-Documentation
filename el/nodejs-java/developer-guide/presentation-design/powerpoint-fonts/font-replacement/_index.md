## **Επισκόπηση**

Το Aspose.Slides σάς επιτρέπει να αντικαταστήσετε μια γραμματοσειρά με άλλη σε όλη την παρουσίαση. Όταν αντικαθίσταται μια γραμματοσειρά, όλες οι εμφανίσεις της αρχικής γραμματοσειράς αλλάζουν στη νέα γραμματοσειρά.

Για να πραγματοποιήσετε αντικατάσταση γραμματοσειράς, φορτώστε την παρουσίαση, ορίστε τη γραμματοσειρά πηγής και τη γραμματοσειρά αντικατάστασης, καλέστε τη μέθοδο αντικατάστασης γραμματοσειράς και αποθηκεύστε την τροποποιημένη παρουσίαση ως αρχείο PPTX. Αυτή η προσέγγιση είναι χρήσιμη όταν επιθυμείτε σκόπιμα να μεταβείτε από μία οικογένεια γραμματοσειρών σε άλλη σε όλη την παρουσίαση.

## **Αντικατάσταση Γραμματοσειρών**

Εάν αλλάξετε γνώμη σχετικά με τη χρήση μιας γραμματοσειράς, μπορείτε να την αντικαταστήσετε με άλλη. Όλες οι εμφανίσεις της παλιάς γραμματοσειράς θα αντικατασταθούν από τη νέα.

Το Aspose.Slides σάς επιτρέπει να αντικαταστήσετε μια γραμματοσειρά με τον παρακάτω τρόπο:

1. Φορτώστε την σχετική παρουσίαση. 
2. Φορτώστε τη γραμματοσειρά που θα αντικατασταθεί. 
3. Φορτώστε τη νέα γραμματοσειρά. 
4. Αντικαταστήστε τη γραμματοσειρά. 
5. Αποθηκεύστε την τροποποιημένη παρουσίαση ως αρχείο PPTX.

Αυτός ο κώδικας JavaScript δείχνει την αντικατάσταση γραμματοσειράς:

```javascript
// Φορτώνει μια παρουσίαση
var pres = new aspose.slides.Presentation("Fonts.pptx");
try {
    // Φορτώνει τη γραμματοσειρά προέλευσης που θα αντικατασταθεί
    var sourceFont = new aspose.slides.FontData("Arial");
    // Φορτώνει τη νέα γραμματοσειρά
    var destFont = new aspose.slides.FontData("Times New Roman");
    // Αντικαθιστά τις γραμματοσειρές
    pres.getFontsManager().replaceFont(sourceFont, destFont);
    // Αποθηκεύει την παρουσίαση
    pres.save("UpdatedFont_out.pptx", aspose.slides.SaveFormat.Pptx);
} finally {
    if (pres != null) {
        pres.dispose();
    }
}
```

## **ΣΥΝΑΡΤΗΣΕΙΣ**

**Ποια είναι η διαφορά μεταξύ «αντικατάστασης γραμματοσειράς», «αντικατάστασης» και «εφεδρικών γραμματοσειρών»;**

Η αντικατάσταση είναι σκόπιμη αλλαγή από μία οικογένεια σε άλλη σε ολόκληρο το έγγραφο. Η [Αντικατάσταση](/slides/el/nodejs-java/font-substitution/) είναι ένας κανόνας όπως «αν η γραμματοσειρά δεν είναι διαθέσιμη, χρησιμοποιήστε το X». Το [Εφεδρικό](/slides/el/nodejs-java/fallback-font/) εφαρμόζεται επιλεκτικά για μεμονωμένα ελλειπή σύμβολα όταν η βασική γραμματοσειρά είναι εγκατεστημένη αλλά δεν περιέχει τους απαιτούμενους χαρακτήρες.

**Εφαρμόζεται η αντικατάσταση σε κύριες διαφάνειες, διατάξεις, σημειώσεις και σχόλια;**

Ναι. Η αντικατάσταση επηρεάζει όλα τα αντικείμενα της παρουσίασης που χρησιμοποιούν την αρχική γραμματοσειρά, συμπεριλαμβανομένων των κύριων διαφανειών και των σημειώσεων· τα σχόλια επίσης αποτελούν μέρος του εγγράφου και λογίζονται από τη μηχανή γραμματοσειρών.

**Θα αλλάξει η γραμματοσειρά μέσα σε ενσωματωμένα αντικείμενα OLE (π.χ. Excel);**

Όχι. Το [περιεχόμενο OLE](/slides/el/nodejs-java/manage-ole/) ελέγχεται από την αντίστοιχη εφαρμογή του. Η αντικατάσταση στην παρουσίαση δεν μεταμορφώνει τα εσωτερικά δεδομένα OLE· μπορεί να εμφανιστεί ως εικόνα ή ως εξωτερικά επεξεργάσιμο περιεχόμενο.

**Μπορώ να αντικαταστήσω μια γραμματοσειρά μόνο σε μέρος της παρουσίασης (ανά διαφάνεια ή περιοχή);**

Στοχευμένη αντικατάσταση είναι δυνατή εάν αλλάξετε τη γραμματοσειρά στο επίπεδο των απαιτούμενων αντικειμένων/περιοχών αντί να εφαρμόσετε καθολική αντικατάσταση σε ολόκληρο το έγγραφο. Η γενική λογική επιλογής γραμματοσειράς κατά την απόδοση παραμένει η ίδια.

**Πώς μπορώ να προσδιορίσω εκ των προτέρων ποιες γραμματοσειρές χρησιμοποιεί η παρουσίαση;**

Χρησιμοποιήστε τον [διαχειριστή γραμματοσειρών] της παρουσίασης (https://reference.aspose.com/slides/el/nodejs-java/aspose.slides/fontsmanager/): παρέχει λίστα με τις [οικογένειες που χρησιμοποιούνται](https://reference.aspose.com/slides/el/nodejs-java/aspose.slides/fontsmanager/getfonts/) και πληροφορίες για τις [αντικαταστάσεις/«άγνωστες» γραμματοσειρές](https://reference.aspose.com/slides/el/nodejs-java/aspose.slides/fontsmanager/getsubstitutions/), που βοηθά στον προγραμματισμό της αντικατάστασης.

**Λειτουργεί η αντικατάσταση γραμματοσειράς κατά τη μετατροπή σε PDF/εικόνες;**

Ναι. Κατά την εξαγωγή, το Aspose.Slides εφαρμόζει την ίδια [ακολουθία επιλογής/αντικατάστασης γραμματοσειρών](/slides/el/nodejs-java/font-selection-sequence/), έτσι ώστε μια προγενέστερη αντικατάσταση να ληφθεί υπόψη κατά τη μετατροπή.

**Πρέπει να εγκαταστήσω τη στοχευμένη γραμματοσειρά στο σύστημα ή μπορώ να επισυνάψω έναν φάκελο γραμματοσειρών;**

Η εγκατάσταση δεν είναι απαραίτητη: η βιβλιοθήκη επιτρέπει τη [φόρτωση εξωτερικών γραμματοσειρών](/slides/el/nodejs-java/custom-font/) από φακέλους χρήστη για χρήση κατά τη [απόδοση και εξαγωγή](/slides/el/nodejs-java/convert-powerpoint/).

**Θα διορθώσει η αντικατάσταση το «tofu» (τετράγωνα) αντί των χαρακτήρων;**

Μόνο εάν η στοχευμένη γραμματοσειρά περιέχει πραγματικά τα απαιτούμενα σύμβολα. Εάν όχι, [ρυθμίστε το εφεδρικό](/slides/el/nodejs-java/fallback-font/) για να καλύψετε τους ελλιπείς χαρακτήρες.
