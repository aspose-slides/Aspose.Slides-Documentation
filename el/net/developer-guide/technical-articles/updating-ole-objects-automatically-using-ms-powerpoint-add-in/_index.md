## **Εισαγωγή**

Μία από τις πιο συχνές ερωτήσεις που θέτουν οι πελάτες του Aspose.Slides για .NET είναι πώς να δημιουργήσουν ή να τροποποιήσουν επεξεργάσιμα γραφήματα (ή άλλα αντικείμενα OLE) έτσι ώστε να ενημερώνονται αυτόματα όταν ανοίγεται η παρουσίαση. Δυστυχώς, το PowerPoint δεν υποστηρίζει αυτόματες μακροεντολές με τον ίδιο τρόπο όπως το Excel και το Word. Οι μόνες διαθέσιμες μακροεντολές είναι `Auto_Open` και `Auto_Close`, και αυτές εκτελούνται αυτόματα μόνο από ένα πρόσθετο. Αυτή η σύντομη τεχνική συμβουλή δείχνει πώς να το επιτύχετε.

## **Αυτόματη Ενημέρωση Αντικειμένων OLE**

Αρχικά, είναι διαθέσιμα αρκετά δωρεάν πρόσθετα που προσθέτουν τη λειτουργία μακροεντολής Auto_Open στο PowerPoint, για παράδειγμα [AutoEvents Add-in](http://skp.mvps.org/autoevents.htm) και [Event Generator](https://www.officeoneonline.com/eventgen/eventgen.html).

Αφού εγκαταστήσετε ένα από αυτά τα πρόσθετα, απλώς προσθέστε τη μακροεντολή `Auto_Open()` (ή `OnPresentationOpen()` αν χρησιμοποιείτε το Event Generator) στην παρουσίαση προτύπου σας όπως φαίνεται παρακάτω:

```cs
public void Auto_Open()
{
    // Διέλευση κάθε διαφάνειας στην παρουσίαση.
    foreach (var oSlide in ActivePresentation.Slides)
    {
        // Διέλευση όλων των σχημάτων στην τρέχουσα διαφάνεια.
        foreach (var oShape in oSlide.Shapes)
        {
            // Έλεγχος εάν το σχήμα είναι αντικείμενο OLE.
            if (oShape.Type == msoEmbeddedOLEObject)
            {
                // Βρέθηκε αντικείμενο OLE. Λάβετε την αναφορά του αντικειμένου και στη συνέχεια ενημερώστε το.
                oObject = oShape.OLEFormat.Object;
                oObject.Application.Update();

                // Τώρα, τερματίστε το πρόγραμμα διακομιστή OLE.
                // Αυτό ελευθερώνει μνήμη και αποτρέπει τυχόν προβλήματα.
                // Επίσης, ορίστε το oObject σε Nothing για να ελευθερώσετε το αντικείμενο.
                oObject.Application.Quit();
                oObject = null;
            }
        }
    }
}
```

Οποιαδήποτε αλλαγή γίνει στα αντικείμενα OLE με το Aspose.Slides για .NET θα ενημερώνεται αυτόματα όταν το PowerPoint ανοίγει την παρουσίαση. Εάν έχετε πολλά αντικείμενα OLE και δεν θέλετε να τα ενημερώσετε όλα, απλώς προσθέστε μια προσαρμοσμένη ετικέτα στα σχήματα που πρέπει να επεξεργασθείτε και ελέγξτε την στη μακροεντολή.
