## **Εισαγωγή**

Το Aspose.Slides for .NET χρησιμοποιείται για δημιουργία αρχείων παρουσίασης όπου κάθε σελίδα είναι μια διαφάνεια. Αυτές οι διαφάνειες μπορούν να προβληθούν ανοίγοντας τα αρχεία παρουσίασης με το Microsoft PowerPoint. Μερικές φορές, όμως, οι προγραμματιστές μπορεί να χρειαστεί να δουν τις εικόνες των σχημάτων ξεχωριστά σε προβολέα εικόνων. Σε τέτοιες περιπτώσεις, το Aspose.Slides for .NET σας βοηθά να δημιουργήσετε μικρογραφίες των σχημάτων της διαφάνειας. Πώς να χρησιμοποιήσετε αυτή τη δυνατότητα περιγράφεται σε αυτό το άρθρο.  
Αυτό το άρθρο εξηγεί πώς να δημιουργήσετε μικρογραφίες διαφάνειας με διαφορετικούς τρόπους:

- Δημιουργία μικρογραφίας σχήματος μέσα σε διαφάνεια.
- Δημιουργία μικρογραφίας σχήματος για σχήμα διαφάνειας με διαστάσεις ορισμένες από τον χρήστη.
- Δημιουργία μικρογραφίας σχήματος στα όρια της εμφάνισης ενός σχήματος.

## **Δημιουργία Μικρογραφίας Σχήματος από Διαφάνεια**
Για να δημιουργήσετε μια μικρογραφία σχήματος από οποιαδήποτε διαφάνεια χρησιμοποιώντας Aspose.Slides for .NET:

1. Δημιουργήστε μια παρουσία της κλάσης [Presentation](https://reference.aspose.com/slides/el/net/aspose.slides/presentation).
1. Λάβετε την αναφορά σε οποιαδήποτε διαφάνεια χρησιμοποιώντας το ID ή το δείκτη της.
1. Αποκτήστε τη μικρογραφία σχήματος της αναφερόμενης διαφάνειας στην προεπιλεγμένη κλίμακα.
1. Αποθηκεύστε τη μικρογραφία σε οποιαδήποτε επιθυμητή μορφή εικόνας.

Το παρακάτω παράδειγμα δημιουργεί μικρογραφία σχήματος.

```c#
using (Presentation presentation = new Presentation("HelloWorld.pptx"))
{
    IShape shape = presentation.Slides[0].Shapes[0];
    using (IImage image = shape.GetImage())
    {
        image.Save("Shape_thumbnail_out.png", ImageFormat.Png);
    }
}
```

## **Δημιουργία Μικρογραφίας με Ορισμένο Παράγοντα Κλίμακας από τον Χρήστη**
Για να δημιουργήσετε τη μικρογραφία σχήματος ενός σχήματος διαφάνειας χρησιμοποιώντας Aspose.Slides for .NET:

1. Δημιουργήστε μια παρουσία της κλάσης `Presentation`.
1. Λάβετε την αναφορά σε οποιαδήποτε διαφάνεια χρησιμοποιώντας το ID ή το δείκτη της.
1. Αποκτήστε τη μικρογραφία της αναφερόμενης διαφάνειας με όρια σχήματος.
1. Αποθηκεύστε τη μικρογραφία σε οποιαδήποτε επιθυμητή μορφή εικόνας.

Το παρακάτω παράδειγμα παράγει μια μικρογραφία με ορισμένο από τον χρήστη παράγοντα κλίμακας.

```c#
ShapeThumbnailBounds bounds = ShapeThumbnailBounds.Shape;
float scale = 1; // Κλιμάκωση κατά τους άξονες X και Y.

using (Presentation presentation = new Presentation("HelloWorld.pptx"))
{
    IShape shape = presentation.Slides[0].Shapes[0];
    using (IImage image = shape.GetImage(bounds, scale, scale))
    {
        image.Save("Scaling Factor Thumbnail_out.png", ImageFormat.Png);
    }
}
```

## **Δημιουργία Μικρογραφίας Εμφάνισης Σχήματος βάσει Ορίων**
Αυτή η μέθοδος δημιουργίας μικρογραφιών σχημάτων επιτρέπει στους προγραμματιστές να παράγουν μια μικρογραφία στα όρια της εμφάνισης του σχήματος. Λαμβάνει υπόψη όλα τα εφέ του σχήματος. Η παραγόμενη μικρογραφία σχήματος περιορίζεται από τα όρια της διαφάνειας. Για να δημιουργήσετε μια μικρογραφία οποιουδήποτε σχήματος διαφάνειας στα όρια της εμφάνισής του, χρησιμοποιήστε τον παρακάτω κώδικα:

1. Δημιουργήστε μια παρουσία της κλάσης `Presentation`.
1. Λάβετε την αναφορά σε οποιαδήποτε διαφάνεια χρησιμοποιώντας το ID ή το δείκτη της.
1. Αποκτήστε τη μικρογραφία της αναφερόμενης διαφάνειας με όρια σχήματος ως εμφάνιση.
1. Αποθηκεύστε τη μικρογραφία σε οποιαδήποτε επιθυμητή μορφή εικόνας.

Το παρακάτω παράδειγμα δημιουργεί μια μικρογραφία με ορισμένο από τον χρήστη παράγοντα κλίμακας.

```c#
ShapeThumbnailBounds bounds = ShapeThumbnailBounds.Appearance;
float scale = 1; // Κλιμάκωση κατά τους άξονες X και Y.

using (Presentation presentation = new Presentation("HelloWorld.pptx"))
{
    IShape shape = presentation.Slides[0].Shapes[0];
    using (IImage image = shape.GetImage(bounds, scale, scale))
    {
        image.Save("Shape_thumbnail_Bound_Shape_out.png", ImageFormat.Png);
    }
}
```

## **Συχνές Ερωτήσεις**

**Ποια μορφές εικόνας μπορούν να χρησιμοποιηθούν κατά την αποθήκευση μικρογραφιών σχημάτων;**

[PNG, JPEG, BMP, GIF, TIFF](https://reference.aspose.com/slides/el/net/aspose.slides/imageformat/), και άλλες. Τα σχήματα μπορούν επίσης να [εξαχθούν ως διανυσματικό SVG](https://reference.aspose.com/slides/el/net/aspose.slides/shape/writeassvg/) αποθηκεύοντας το περιεχόμενο του σχήματος ως SVG.

**Ποια είναι η διαφορά μεταξύ ορίων Shape και Appearance κατά την απόδοση μιας μικρογραφίας;**

`Shape` χρησιμοποιεί τη γεωμετρία του σχήματος· `Appearance` λαμβάνει υπόψη τα [οπτικά εφέ](/slides/el/net/shape-effect/) (σκιές, λάμψεις κ.λπ.).

**Τι συμβαίνει αν ένα σχήμα χαρακτηριστεί ως κρυφό; Θα εξακολουθεί να αποδίδεται ως μικρογραφία;**

Ένα κρυφό σχήμα παραμένει μέρος του μοντέλου και μπορεί να αποδοθεί· η σημαία κρυφής εμφάνισης επηρεάζει μόνο την παρουσίαση της διαφάνειας, αλλά δεν εμποδίζει τη δημιουργία της εικόνας του σχήματος.

**Υποστηρίζονται τα ομαδικά σχήματα, τα γραφήματα, το SmartArt και άλλα σύνθετα αντικείμενα;**

Ναι. Οποιοδήποτε αντικείμενο που αναπαρίσταται ως [Shape](https://reference.aspose.com/slides/el/net/aspose.slides/shape/) (συμπεριλαμβανομένου του [GroupShape](https://reference.aspose.com/slides/el/net/aspose.slides/groupshape/), του [Chart](https://reference.aspose.com/slides/el/net/aspose.slides.charts/chart/), και του [SmartArt](https://reference.aspose.com/slides/el/net/aspose.slides.smartart/smartart/)) μπορεί να αποθηκευτεί ως μικρογραφία ή ως SVG.

**Επηρεάζουν τα προ-εγκατεστημένα σύστημα γραμματοσειράς την ποιότητα των μικρογραφιών για σχήματα κειμένου;**

Ναι. Θα πρέπει να [παρέχετε τις απαιτούμενες γραμματοσειρές](/slides/el/net/custom-font/) (ή να [ρυθμίσετε τις υποκαταστάσεις γραμματοσειρών](/slides/el/net/font-substitution/)) ώστε να αποφευχθούν ανεπιθύμητες εναλλακτικές και μεταβολές κειμένου.
