## **Επισκόπηση**

Αυτό το άρθρο δείχνει πώς να προσθέσετε σχήματα έλλειψης σε διαφάνειες PowerPoint χρησιμοποιώντας το Aspose.Slides. Καλύπτει τη δημιουργία μιας απλής έλλειψης, τη δημιουργία μιας μορφοποιημένης έλλειψης και την αποθήκευση της ενημερωμένης παρουσίασης ως αρχείο PPTX. Επίσης, αγγίζει σχετικές ερωτήσεις όπως η εργασία με τη θέση και το μέγεθος της έλλειψης, ο έλεγχος της σειράς στοίβαξης και η εφαρμογή εφέ κίνησης.

## **Δημιουργία Έλλειψης**
Για να προσθέσετε μια απλή έλλειψη σε μια επιλεγμένη διαφάνεια της παρουσίασης, ακολουθήστε τα παρακάτω βήματα:

1. Δημιουργήστε ένα στιγμιότυπο της [Παρουσίαση ](https://reference.aspose.com/slides/el/net/aspose.slides/presentation)class
1. Αποκτήστε την αναφορά μιας διαφάνειας χρησιμοποιώντας το Index της
1. Προσθέστε ένα AutoShape τύπου Έλλειψης χρησιμοποιώντας τη μέθοδο AddAutoShape που εκτίθεται από το αντικείμενο IShapes
1. Γράψτε την τροποποιημένη παρουσίαση ως αρχείο PPTX

Στο παρακάτω παράδειγμα, προσθέσαμε μια έλλειψη στην πρώτη διαφάνεια.

```c#
 // Δημιουργία αντικειμένου Presentation που αντιπροσωπεύει το PPTX
using (Presentation pres = new Presentation())
{
    // Λήψη της πρώτης διαφάνειας
    ISlide sld = pres.Slides[0];

    // Προσθήκη autoshape τύπου έλλειψης
    sld.Shapes.AddAutoShape(ShapeType.Ellipse, 50, 150, 150, 50);

    //Αποθήκευση του αρχείου PPTX στο δίσκο
    pres.Save("EllipseShp1_out.pptx", SaveFormat.Pptx);
}
```

## **Δημιουργία Μορφοποιημένης Έλλειψης**
Για να προσθέσετε μια καλύτερα μορφοποιημένη έλλειψη σε μια διαφάνεια, ακολουθήστε τα παρακάτω βήματα:

1. Δημιουργήστε ένα στιγμιότυπο της [Παρουσίαση ](https://reference.aspose.com/slides/el/net/aspose.slides/presentation)class.
1. Αποκτήστε την αναφορά μιας διαφάνειας χρησιμοποιώντας το Index της.
1. Προσθέστε ένα AutoShape τύπου Έλλειψης χρησιμοποιώντας τη μέθοδο AddAutoShape που εκτίθεται από το αντικείμενο IShapes.
1. Ορίστε τον Τύπο Γέμισης της Έλλειψης σε Στερεό.
1. Ορίστε το Χρώμα της Έλλειψης χρησιμοποιώντας την ιδιότητα SolidFillColor.Color που εκτίθεται από το αντικείμενο FillFormat που συσχετίζεται με το αντικείμενο IShape.
1. Ορίστε το Χρώμα των γραμμών της Έλλειψης.
1. Ορίστε το Πλάτος των γραμμών της Έλλειψης.
1. Γράψτε την τροποποιημένη παρουσίαση ως αρχείο PPTX.

Στο παρακάτω παράδειγμα, προσθέσαμε μια μορφοποιημένη έλλειψη στην πρώτη διαφάνεια της παρουσίασης.

```c#
 // Δημιουργία αντικειμένου Presentation που αντιπροσωπεύει το PPTX
using (Presentation pres = new Presentation())
{

    // Λήψη της πρώτης διαφάνειας
    ISlide sld = pres.Slides[0];

    // Προσθήκη autoshape τύπου έλλειψης
    IShape shp = sld.Shapes.AddAutoShape(ShapeType.Ellipse, 50, 150, 150, 50);

    // Εφαρμογή μορφοποίησης στο σχήμα έλλειψης
    shp.FillFormat.FillType = FillType.Solid;
    shp.FillFormat.SolidFillColor.Color = Color.Chocolate;

    // Εφαρμογή μορφοποίησης στη γραμμή της έλλειψης
    shp.LineFormat.FillFormat.FillType = FillType.Solid;
    shp.LineFormat.FillFormat.SolidFillColor.Color = Color.Black;
    shp.LineFormat.Width = 5;

    //Αποθήκευση του αρχείου PPTX στο δίσκο
    pres.Save("EllipseShp2_out.pptx", SaveFormat.Pptx);
}
```

## **Συχνές Ερωτήσεις**

**Πώς μπορώ να ορίσω την ακριβή θέση και μέγεθος μιας έλλειψης σε σχέση με τις μονάδες της διαφάνειας;**

Οι συντεταγμένες και οι διαστάσεις συνήθως ορίζονται **σε σημεία**. Για προβλέψιμα αποτελέσματα, βασίστε τους υπολογισμούς σας στο μέγεθος της διαφάνειας και μετατρέψτε τα απαιτούμενα χιλιοστά ή ίντσες σε σημεία πριν αντιστοιχίσετε τις τιμές.

**Πώς μπορώ να τοποθετήσω μια έλλειψη πάνω ή κάτω από άλλα αντικείμενα (έλεγχος σειράς στοίβαξης);**

Ρυθμίστε τη σειρά σχεδίασης του αντικειμένου φέρνοντάς το μπροστά ή στέλνοντάς το πίσω. Αυτό επιτρέπει στην έλλειψη να επικαλύπτει άλλα αντικείμενα ή να αποκαλύπτει αυτά που είναι κάτω από αυτή.

**Πώς μπορώ να δημιουργήσω κίνηση για την εμφάνιση ή τονισμό μιας έλλειψης;**

[Εφαρμογή](/slides/el/net/shape-animation/) εφέ εισόδου, τονισμού ή εξόδου στο σχήμα, και ρυθμίστε τα ερεθίσματα και το χρονοδιάγραμμα για να καθορίσετε πότε και πώς θα εκτελείται η κίνηση.
