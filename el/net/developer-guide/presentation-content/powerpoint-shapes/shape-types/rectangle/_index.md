## **Επισκόπηση**

Αυτό το άρθρο δείχνει πώς να προσθέσετε σχήματα ορθογωνίου στις διαφάνειες του PowerPoint χρησιμοποιώντας το Aspose.Slides. Καλύπτει τη δημιουργία ενός απλού ορθογωνίου, τη δημιουργία ενός μορφοποιημένου ορθογωνίου και την αποθήκευση της ενημερωμένης παρουσίασης ως αρχείο PPTX.

Θα δείτε επίσης πώς να εφαρμόσετε βασική μορφοποίηση ορθογωνίου, όπως χρώμα συμπαγούς γεμίσματος, χρώμα γραμμής και πάχος γραμμής. Επιπλέον, η ενότητα FAQ του άρθρου παραπέμπει σε σχετικές εργασίες με ορθογώνια, όπως στρογγυλεμένες γωνίες, γεμίσματα εικόνας, οπτικά εφέ, υπερσυνδέσμους, κλειδώματα σχήματος, επιλογές εξαγωγής και αποτελεσματικές ιδιότητες.

## **Δημιουργία απλού ορθογωνίου**
Όπως και στα προηγούμενα θέματα, αυτό αφορά επίσης την προσθήκη ενός σχήματος και αυτή τη φορά το σχήμα που θα συζητήσουμε είναι το Ορθογώνιο. Στο θέμα αυτό, περιγράψαμε πώς οι προγραμματιστές μπορούν να προσθέσουν απλά ή μορφοποιημένα ορθογώνια στις διαφάνειές τους χρησιμοποιώντας το Aspose.Slides για .NET. Για να προσθέσετε ένα απλό ορθογώνιο σε μια επιλεγμένη διαφάνεια της παρουσίασης, ακολουθήστε τα παρακάτω βήματα:

1. Δημιουργήστε ένα στιγμιότυπο της κλάσης [Presentation](https://reference.aspose.com/slides/el/net/aspose.slides/presentation).
2. Αποκτήστε την αναφορά μιας διαφάνειας χρησιμοποιώντας το Index της.
3. Προσθέστε ένα IAutoShape τύπου Rectangle χρησιμοποιώντας τη μέθοδο AddAutoShape που εκτίθεται από το αντικείμενο IShapes.
4. Γράψτε την τροποποιημένη παρουσίαση ως αρχείο PPTX.

Στο παρακάτω παράδειγμα, προσθέσαμε ένα απλό ορθογώνιο στην πρώτη διαφάνεια της παρουσίασης.

```c#
// Δημιουργία κλάσης Presentation που αντιπροσωπεύει το PPTX
using (Presentation pres = new Presentation())
{

    // Αποκτήστε την πρώτη διαφάνεια
    ISlide sld = pres.Slides[0];

    // Προσθέστε αυτόματο σχήμα τύπου ορθογωνίου
    sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 150, 50);

    //Write Γράψτε το αρχείο PPTX στο δίσκο
    pres.Save("RectShp1_out.pptx", SaveFormat.Pptx);
}
```

## **Δημιουργία μορφοποιημένου ορθογωνίου**
Για να προσθέσετε ένα μορφοποιημένο ορθογώνιο σε μια διαφάνεια, ακολουθήστε τα παρακάτω βήματα:

1. Δημιουργήστε ένα στιγμιότυπο της κλάσης [Presentation](https://reference.aspose.com/slides/el/net/aspose.slides/presentation).
2. Αποκτήστε την αναφορά μιας διαφάνειας χρησιμοποιώντας το Index της.
3. Προσθέστε ένα IAutoShape τύπου Rectangle χρησιμοποιώντας τη μέθοδο AddAutoShape που εκτίθεται από το αντικείμενο IShapes.
4. Ορίστε τον τύπο γεμίσματος του Ορθογωνίου σε Solid.
5. Ορίστε το χρώμα του Ορθογωνίου χρησιμοποιώντας την ιδιότητα SolidFillColor.Color που εκτίθεται από το αντικείμενο FillFormat που συσχετίζεται με το αντικείμενο IShape.
6. Ορίστε το χρώμα των γραμμών του Ορθογωνίου.
7. Ορίστε το πλάτος των γραμμών του Ορθογωνίου.
8. Γράψτε την τροποποιημένη παρουσίαση ως αρχείο PPTX.

Τα παραπάνω βήματα υλοποιούνται στο παρακάτω παράδειγμα.

```c#
 // Δημιουργία κλάσης Presentation που αντιπροσωπεύει το PPTX
using (Presentation pres = new Presentation())
{

    // Αποκτήστε την πρώτη διαφάνεια
    ISlide sld = pres.Slides[0];

    // Προσθήκη αυτόματου σχήματος τύπου ορθογωνίου
    IShape shp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 150, 50);

    // Εφαρμόστε μορφοποίηση στο σχήμα ορθογωνίου
    shp.FillFormat.FillType = FillType.Solid;
    shp.FillFormat.SolidFillColor.Color = Color.Chocolate;

    // Εφαρμόστε μορφοποίηση στη γραμμή του ορθογωνίου
    shp.LineFormat.FillFormat.FillType = FillType.Solid;
    shp.LineFormat.FillFormat.SolidFillColor.Color = Color.Black;
    shp.LineFormat.Width = 5;

    // Γράψτε το αρχείο PPTX στο δίσκο
    pres.Save("RectShp2_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

## **ΣΥΧΝΕΣ ΕΡΩΤΗΣΕΙΣ**

**Πώς μπορώ να προσθέσω ένα ορθογώνιο με στρογγυλεμένες γωνίες;**  
Χρησιμοποιήστε τον τύπο σχήματος με στρογγυλεμένες γωνίες [shape type](https://reference.aspose.com/slides/el/net/aspose.slides/shapetype/) και προσαρμόστε την ακτίνα των γωνιών στις ιδιότητες του σχήματος· η στρογγυλοποίηση μπορεί να εφαρμοστεί και ανά γωνία μέσω γεωμετρικών προσαρμογών.

**Πώς μπορώ να γεμίσω ένα ορθογώνιο με εικόνα (υφή);**  
Επιλέξτε τον τύπο γεμίσματος εικόνας [fill type](https://reference.aspose.com/slides/el/net/aspose.slides/filltype/), παράσχετε την πηγή της εικόνας και διαμορφώστε τις λειτουργίες [stretching/tiling modes](https://reference.aspose.com/slides/el/net/aspose.slides/picturefillmode/).

**Μπορεί ένα ορθογώνιο να έχει σκιά και λάμψη;**  
Ναι. Τα [Outer/inner shadow, glow, and soft edges](/slides/el/net/shape-effect/) είναι διαθέσιμα με ρυθμιζόμενες παραμέτρους.

**Μπορώ να μετατρέψω ένα ορθογώνιο σε κουμπί με υπερσύνδεσμο;**  
Ναι. [Assign a hyperlink](/slides/el/net/manage-hyperlinks/) στο κλικ του σχήματος (μετάβαση σε διαφάνεια, αρχείο, διεύθυνση ιστού ή email).

**Πώς μπορώ να προστατεύσω ένα ορθογώνιο από μετακίνηση και αλλαγές;**  
[Use shape locks](/slides/el/net/applying-protection-to-presentation/): μπορείτε να απαγορεύσετε τη μετακίνηση, την αλλαγή μεγέθους, την επιλογή ή την επεξεργασία κειμένου για να διατηρήσετε τη διάταξη.

**Μπορώ να μετατρέψω ένα ορθογώνιο σε ραστερική εικόνα ή SVG;**  
Ναι. Μπορείτε να [render the shape](http://reference.aspose.com/slides/el/net/aspose.slides/shape/getimage/) σε εικόνα με καθορισμένο μέγεθος/κλίμακα ή να το [export it as SVG](https://reference.aspose.com/slides/el/net/aspose.slides/shape/writeassvg/) για χρήση ως διάνυσμα.

**Πώς μπορώ γρήγορα να λάβω τις πραγματικές (αποτελεσματικές) ιδιότητες ενός ορθογωνίου λαμβάνοντας υπόψη το θέμα και την κληρονομικότητα;**  
[Use the shape’s effective properties](/slides/el/net/shape-effective-properties/): το API επιστρέφει υπολογισμένες τιμές που λαμβάνουν υπόψη τα στυλ θέματος, τη διάταξη και τις τοπικές ρυθμίσεις, διευκολύνοντας την ανάλυση μορφοποίησης.
