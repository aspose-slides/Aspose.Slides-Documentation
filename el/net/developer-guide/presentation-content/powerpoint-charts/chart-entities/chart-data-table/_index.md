## **Επισκόπηση**

Αυτό το άρθρο εξηγεί πώς να εργαστείτε με πίνακες δεδομένων γραφημάτων στο Aspose.Slides. Δείχνει πώς να εμφανίσετε έναν πίνακα δεδομένων για ένα γράφημα και να προσαρμόσετε τη μορφοποίηση κειμένου ορίζοντας ιδιότητες γραμματοσειράς όπως έντονη μορφή και ύψος γραμματοσειράς. Το παράδειγμα δείχνει τη φόρτωση μιας παρουσίασης, την προσθήκη γραφήματος, την ενεργοποίηση του πίνακα δεδομένων του γραφήματος, την εφαρμογή ρυθμίσεων γραμματοσειράς και την αποθήκευση της ενημερωμένης παρουσίασης.

Περιλαμβάνει επίσης σύντομες απαντήσεις σε συνηθισμένες ερωτήσεις σχετικά με την εμφάνιση κλειδιών υπόμνησης σε πίνακα δεδομένων γραφήματος, τη διατήρηση του πίνακα δεδομένων κατά την εξαγωγή, την εργασία με γραφήματα που φορτώνονται από υπάρχουσες παρουσιάσεις ή πρότυπα, και την αναγνώριση γραφημάτων στα οποία ο πίνακας δεδομένων είναι ενεργοποιημένος.

## **Ορισμός Ιδιοτήτων Γραμματοσειράς για Πίνακα Δεδομένων Γραφήματος**
Το Aspose.Slides for .NET παρέχει υποστήριξη για την αλλαγή του χρώματος των κατηγοριών σε χρώμα σειράς.

1. Δημιουργήστε αντικείμενο κλάσης [Presentation](https://reference.aspose.com/slides/el/net/aspose.slides/presentation).
1. Προσθέστε γράφημα στη διαφάνεια.
1. Ορίστε πίνακα γραφήματος.
1. Ορίστε το ύψος γραμματοσειράς.
1. Αποθηκεύστε την τροποποιημένη παρουσίαση.

Παρακάτω παρατίθεται παράδειγμα.

```c#
using (Presentation pres = new Presentation("test.pptx"))
{
	IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 600, 400);

	chart.HasDataTable = true;

	chart.ChartDataTable.TextFormat.PortionFormat.FontBold = NullableBool.True;
	chart.ChartDataTable.TextFormat.PortionFormat.FontHeight = 20;

	pres.Save("output.pptx", SaveFormat.Pptx);
}
```

## **Συχνές ερωτήσεις**

**Μπορώ να εμφανίσω μικρά κλειδιά υπόμνησης δίπλα στις τιμές στον πίνακα δεδομένων του γραφήματος;**

Ναι. Ο πίνακας δεδομένων υποστηρίζει [legend keys](https://reference.aspose.com/slides/el/net/aspose.slides.charts/datatable/showlegendkey/), και μπορείτε να τα ενεργοποιήσετε ή να τα απενεργοποιήσετε.

**Θα διατηρηθεί ο πίνακας δεδομένων κατά την εξαγωγή της παρουσίασης σε PDF, HTML ή εικόνες;**

Ναι. Το Aspose.Slides αποδίδει το γράφημα ως μέρος της διαφάνειας, έτσι η εξαγόμενη [PDF](/slides/el/net/convert-powerpoint-to-pdf/)/[HTML](/slides/el/net/convert-powerpoint-to-html/)/[image](/slides/el/net/convert-powerpoint-to-png/) περιλαμβάνει το γράφημα με τον πίνακα δεδομένων του.

**Υποστηρίζονται πίνακες δεδομένων για γραφήματα που προέρχονται από αρχείο προτύπου;**

Ναι. Για οποιοδήποτε γράφημα που φορτώνεται από υπάρχουσα παρουσίαση ή πρότυπο, μπορείτε να ελέγξετε και να αλλάξετε αν ένας πίνακας δεδομένων [εμφανίζεται](https://reference.aspose.com/slides/el/net/aspose.slides.charts/chart/hasdatatable/) χρησιμοποιώντας τις ιδιότητες του γραφήματος.

**Πώς μπορώ γρήγορα να βρω ποια γραφήματα σε ένα αρχείο έχουν ενεργοποιημένο τον πίνακα δεδομένων;**

Ελέγξτε την ιδιότητα κάθε γραφήματος που υποδεικνύει αν ο πίνακας δεδομένων [εμφανίζεται](https://reference.aspose.com/slides/el/net/aspose.slides.charts/chart/hasdatatable/), και διασχίστε τις διαφάνειες για να εντοπίσετε τα γραφήματα όπου είναι ενεργοποιημένος.
