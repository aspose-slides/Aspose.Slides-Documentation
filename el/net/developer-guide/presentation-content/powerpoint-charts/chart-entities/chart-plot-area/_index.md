## **Επισκόπηση**

Αυτό το άρθρο δείχνει πώς να εργάζεστε με την περιοχή σχεδίασης ενός γραφήματος στο Aspose.Slides. Εξηγεί πώς να αποκτήσετε τη πραγματική θέση και το μέγεθος της περιοχής σχεδίασης επαληθεύοντας τη διάταξη του γραφήματος και στη συνέχεια διαβάζοντας τις τιμές X, Y, πλάτους και ύψους.

Δείχνει επίσης πώς να διαμορφώσετε τη λειτουργία διάταξης της περιοχής σχεδίασης όταν η διάταξη ορίζεται χειροκίνητα, χρησιμοποιώντας το `LayoutTargetType` για να ορίσετε αν η περιοχή σχεδίασης υπολογίζεται από την εσωτερική της περιοχή ή από την εξωτερική περιοχή μαζί με τους άξονες και τις ετικέτες των αξόνων.

## **Λήψη Πλάτους και Ύψους της Περιοχής Σχεδίασης Γραφήματος**
Το Aspose.Slides για .NET παρέχει ένα απλό API για .

1. Δημιουργήστε μια παρουσία της κλάσης [Presentation](https://reference.aspose.com/slides/el/net/aspose.slides/presentation).
1. Προσπελάστε την πρώτη διαφάνεια.
1. Προσθέστε ένα γράφημα με προεπιλεγμένα δεδομένα.
1. Καλέστε τη μέθοδο IChart.ValidateChartLayout() πριν για να λάβετε τις πραγματικές τιμές.
1. Λαμβάνει την πραγματική θέση X (αριστερά) του στοιχείου γραφήματος σε σχέση με την αριστερή άνω γωνία του γραφήματος.
1. Λαμβάνει την πραγματική κορυφή του στοιχείου γραφήματος σε σχέση με την αριστερή άνω γωνία του γραφήματος.
1. Λαμβάνει το πραγματικό πλάτος του στοιχείου γραφήματος.
1. Λαμβάνει το πραγματικό ύψος του στοιχείου γραφήματος.

```c#
using (Presentation pres = new Presentation("test.Pptx"))
{
    Chart chart = (Chart)pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 100, 100, 500, 350);
    chart.ValidateChartLayout();

    double x = chart.PlotArea.ActualX;
    double y = chart.PlotArea.ActualY;
    double w = chart.PlotArea.ActualWidth;
    double h = chart.PlotArea.ActualHeight;
	
	// Αποθήκευση παρουσίασης με το γράφημα
	pres.Save("Chart_out.pptx", SaveFormat.Pptx);
}
```




## **Ορισμός Λειτουργίας Διάταξης της Περιοχής Σχεδίασης Γραφήματος**
Το Aspose.Slides για .NET παρέχει ένα απλό API για να ορίσετε τη λειτουργία διάταξης της περιοχής σχεδίασης του γραφήματος. Η ιδιότητα **LayoutTargetType** προστέθηκε στις κλάσεις **ChartPlotArea** και **IChartPlotArea**. Εάν η διάταξη της περιοχής σχεδίασης ορίζεται χειροκίνητα, αυτή η ιδιότητα καθορίζει αν η διάταξη θα γίνει με βάση το εσωτερικό της (χωρίς το άξονα και τις ετικέτες άξονα) ή το εξωτερικό (συμπεριλαμβανομένου του άξονα και των ετικετών άξονα). Υπάρχουν δύο δυνατές τιμές που ορίζονται στον enumerator **LayoutTargetType**.

- **LayoutTargetType.Inner** - καθορίζει ότι το μέγεθος της περιοχής σχεδίασης θα καθορίζει το μέγεθος της περιοχής, χωρίς τα σημεία σήμανσης και τις ετικέτες των αξόνων.
- **LayoutTargetType.Outer** - καθορίζει ότι το μέγεθος της περιοχής σχεδίασης θα καθορίζει το μέγεθος της περιοχής, τα σημεία σήμανσης και τις ετικέτες των αξόνων.

Δείγμα κώδικα παρατίθεται παρακάτω.

```c#
using (Presentation presentation = new Presentation())
{
    ISlide slide = presentation.Slides[0];
    IChart chart = slide.Shapes.AddChart(ChartType.ClusteredColumn, 20, 100, 600, 400);
    chart.PlotArea.AsILayoutable.X = 0.2f;
    chart.PlotArea.AsILayoutable.Y = 0.2f;
    chart.PlotArea.AsILayoutable.Width = 0.7f;
    chart.PlotArea.AsILayoutable.Height = 0.7f;
    chart.PlotArea.LayoutTargetType = LayoutTargetType.Inner;

    presentation.Save("SetLayoutMode_outer.pptx", SaveFormat.Pptx);
}
```

## **Συχνές Ερωτήσεις**

**Σε ποιες μονάδες επιστρέφονται τα ActualX, ActualY, ActualWidth και ActualHeight;**

Σε μονάδες σημείων· 1 ίντσα = 72 σημεία. Αυτές είναι οι μονάδες συντεταγμένων του Aspose.Slides.

**Πώς διαφέρει η Περιοχή Σχεδίασης από την Περιοχή Γραφήματος όσον αφορά το περιεχόμενο;**

Η Περιοχή Σχεδίασης είναι η περιοχή σχεδίασης δεδομένων (σειρές, γραμμές πλέγματος, γραμμές τάσης κ.λπ.); η Περιοχή Γραφήματος περιλαμβάνει τα περιβάλλοντα στοιχεία (τίτλος, υπομνήμα, κ.λπ.). Σε διαγράμματα 3D, η Περιοχή Σχεδίασης περιλαμβάνει επίσης τα τοιχώματα/το πάτωμα και τους άξονες.

**Πώς ερμηνεύονται τα X, Y, Πλάτος και Ύψος της Περιοχής Σχεδίασης όταν η διάταξη είναι χειροκίνητη;**

Αποτελούν κλάσματα (0–1) του συνολικού μεγέθους του γραφήματος· σε αυτή τη λειτουργία, η αυτόματη τοποθέτηση είναι απενεργοποιημένη και χρησιμοποιούνται τα καθορισμένα κλάσματα.

**Γιατί άλλαξε η θέση της Περιοχής Σχεδίασης μετά την προσθήκη/μετακίνηση του υπομνήματος;**

Το υπόμνημα βρίσκεται στην περιοχή γραφήματος έξω από την Περιοχή Σχεδίασης, αλλά επηρεάζει τη διάταξη και τον διαθέσιμο χώρο, έτσι η Περιοχή Σχεδίασης μπορεί να μετακινηθεί όταν είναι ενεργή η αυτόματη τοποθέτηση. (Αυτή είναι η τυπική συμπεριφορά των διαγραμμάτων PowerPoint.)
