## **Επισκόπηση**

Αυτό το άρθρο δείχνει πώς να εργάζεστε με διαγράμματα φούσκας στο Aspose.Slides. Καλύπτει δύο συγκεκριμένες επιλογές προσαρμογής: την κλιμάκωση του μεγέθους των φουσκών μέσω της ιδιότητας `BubbleSizeScale` και τον έλεγχο του τρόπου με τον οποίο αντιπροσωπεύονται οι τιμές μεγέθους των φουσκών μέσω της ιδιότητας `BubbleSizeRepresentation`.

Τα παραδείγματα δείχνουν πώς να δημιουργήσετε ένα διάγραμμα φούσκας, να προσαρμόσετε την κλιμάκωση του μεγέθους του και να αλλάξετε την αναπαράσταση του μεγέθους της φούσκας ώστε να χρησιμοποιεί το πλάτος. Το άρθρο περιλαμβάνει επίσης μια σύντομη ενότητα FAQ που διευκρινίζει την υποστήριξη για τον τύπο διαγράμματος «Bubble with 3-D», σημειώνει ότι τα πρακτικά όρια των διαγραμμάτων εξαρτώνται από την απόδοση και την έκδοση του PowerPoint-στόχο, και εξηγεί ότι η εξαγωγή διατηρεί την εμφάνιση του διαγράμματος μέσω της μηχανής απόδοσης του Aspose.Slides.

## **Κλιμάκωση Μεγέθους Διαγράμματος Φούσκας**
Το Aspose.Slides για .NET παρέχει υποστήριξη για την κλιμάκωση του μεγέθους των διαγραμμάτων φούσκας. Στο Aspose.Slides για .NET έχουν προστεθεί οι ιδιότητες **IChartSeries.BubbleSizeScale** και **IChartSeriesGroup.BubbleSizeScale**. Παρατίθεται παρακάτω παράδειγμα δείγματος.

```c#
using (Presentation pres = new Presentation())
{
	IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Bubble, 100, 100, 400, 300);
	chart.ChartData.SeriesGroups[0].BubbleSizeScale = 150;
	pres.Save("Result.pptx",Aspose.Slides.Export.SaveFormat.Pptx);
}
```

## **Αναπαράσταση Δεδομένων ως Μεγέθη Διαγράμματος Φούσκας**
Η ιδιότητα **BubbleSizeRepresentation** έχει προστεθεί στα interfaces IChartSeries, IChartSeriesGroup και στις σχετικές κλάσεις. Η **BubbleSizeRepresentation** καθορίζει τον τρόπο με τον οποίο αντιπροσωπεύονται οι τιμές μεγέθους φούσκας στο διάγραμμα φούσκας. Πιθανές τιμές είναι: **BubbleSizeRepresentationType.Area** και **BubbleSizeRepresentationType.Width**. Συνεπώς, το enum **BubbleSizeRepresentationType** έχει προστεθεί για να καθορίζει τους δυνατούς τρόπους αναπαράστασης των δεδομένων ως μεγέθη διαγράμματος φούσκας. Παράδειγμα κώδικα παρατίθεται παρακάτω.

```c#
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Bubble, 50, 50, 600, 400, true);
    chart.ChartData.SeriesGroups[0].BubbleSizeRepresentation = BubbleSizeRepresentationType.Width;
    pres.Save("Presentation_BubbleSizeRepresentation.pptx", SaveFormat.Pptx);
}
```

## **FAQ**

**Υποστηρίζεται ένα «διάγραμμα φούσκας με 3‑D εφέ» και πώς διαφέρει από ένα κανονικό;**

Ναι. Υπάρχει ξεχωριστός τύπος διαγράμματος, «Bubble with 3‑D». Εφαρμόζει τρισδιάστατη μορφοποίηση στις φούσκες, αλλά δεν προσθέτει επιπλέον άξονα· τα δεδομένα παραμένουν X‑Y‑S (μέγεθος). Ο τύπος είναι διαθέσιμος στην αρίθμηση [chart type](https://reference.aspose.com/slides/el/net/aspose.slides.charts/charttype/).

**Υπάρχει όριο στον αριθμό σειρών και σημείων σε ένα διάγραμμα φούσκας;**

Δεν υπάρχει σκληρό όριο σε επίπεδο API· οι περιορισμοί καθορίζονται από την απόδοση και την έκδοση του PowerPoint-στόχου. Συνιστάται να διατηρείται ο αριθμός των σημείων λογικός για την ευκρίνεια και την ταχύτητα απόδοσης.

**Πώς θα επηρεάσει η εξαγωγή την εμφάνιση ενός διαγράμματος φούσκας (PDF, εικόνες);**

Η εξαγωγή σε υποστηριζόμενες μορφές διατηρεί την εμφάνιση του διαγράμματος· η απόδοση εκτελείται από τη μηχανή Aspose.Slides. Για μορφές raster/vector ισχύουν γενικοί κανόνες απόδοσης γραφικών διαγράμματος (ανάλυση, anti‑aliasing), επομένως επιλέξτε επαρκή DPI για εκτύπωση.
