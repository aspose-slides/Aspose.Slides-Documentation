## **Επισκόπηση**

Αυτό το άρθρο δείχνει πώς να εργαστείτε με ένα γράφημα δακτυλίου σε Aspose.Slides προσθέτοντας το γράφημα σε μια διαφάνεια, ορίζοντας το μέγεθος της κεντρικής τρύπας και αποθηκεύοντας την παρουσίαση. Επικεντρώνεται στη ρύθμιση `DoughnutHoleSize` και παρουσιάζει τα βασικά βήματα που απαιτούνται για την προσαρμογή αυτού του τύπου γραφήματος με κώδικα.

Περιλαμβάνει επίσης μια σύντομη Συχνές Ερωτήσεις (FAQ) που καλύπτει σχετικές περιπτώσεις γραφήματος δακτυλίου, όπως η χρήση πολλαπλών σειρών για δημιουργία πολλαπλών δαχτυλιδιών, η εργασία με εκραγμένα (exploded) γραφήματα δακτυλίου, και η εξαγωγή ενός γραφήματος ως εικόνα raster ή SVG.

## **Καθορίστε το κεντρικό κενό σε γράφημα δακτυλίου**

Για να καθορίσετε το μέγεθος της τρύπας σε ένα γράφημα δακτυλίου, ακολουθήστε τα παρακάτω βήματα:

- Δημιουργήστε ένα αντικείμενο κλάσης [Presentation](https://reference.aspose.com/slides/el/net/aspose.slides/presentation).
- Προσθέστε γράφημα δακτυλίου στη διαφάνεια.
- Καθορίστε το μέγεθος της τρύπας σε ένα γράφημα δακτυλίου.
- Αποθηκεύστε την παρουσίαση στο δίσκο.

Στο παρακάτω παράδειγμα, έχουμε ορίσει το μέγεθος της τρύπας σε ένα γράφημα δακτυλίου.

```c#
// Δημιουργήστε ένα παράδειγμα της κλάσης Presentation
Presentation presentation = new Presentation();

IChart chart = presentation.Slides[0].Shapes.AddChart(ChartType.Doughnut, 50, 50, 400, 400);
chart.ChartData.SeriesGroups[0].DoughnutHoleSize = 90;

// Αποθηκεύστε την παρουσίαση στο δίσκο
presentation.Save("DoughnutHoleSize_out.pptx", SaveFormat.Pptx);
```

## **Συχνές Ερωτήσεις**

**Μπορώ να δημιουργήσω ένα πολυεπίπεδο γράφημα δακτυλίου με πολλαπλά δαχτυλίδια;**

Ναι. Προσθέστε πολλαπλές σειρές σε ένα μόνο γράφημα δακτυλίου—κάθε σειρά γίνεται ένα ξεχωριστό δαχτυλίδιο. Η σειρά των δαχτυλιδιών καθορίζεται από τη σειρά των σειρών στη συλλογή.

**Υποστηρίζεται ένα «exploded» γράφημα δακτυλίου (διαχωρισμένα τμήματα);**

Ναι. Υπάρχει τύπος γραφήματος Exploded Doughnut [chart type](https://reference.aspose.com/slides/el/net/aspose.slides.charts/charttype/) και ιδιότητα έκρηξης στα σημεία δεδομένων· μπορείτε να διαχωρίσετε μεμονωμένα τμήματα.

**Πώς μπορώ να λάβω μια εικόνα ενός γραφήματος δακτυλίου (PNG/SVG) για μια αναφορά;**

Ένα γράφημα είναι ένα σχήμα· μπορείτε να το αποδώσετε σε μια [raster image](https://reference.aspose.com/slides/el/net/aspose.slides/shape/getimage/) ή να εξάγετε το γράφημα σε μια [SVG image](https://reference.aspose.com/slides/el/net/aspose.slides/shape/writeassvg/).
