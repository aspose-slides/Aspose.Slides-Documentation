## **Μετατροπή PPTX σε PPT**
Για να μετατρέψετε ένα PPTX σε PPT, περάστε απλώς το όνομα αρχείου και τη μορφή αποθήκευσης στη μέθοδο [**Save**](https://reference.aspose.com/slides/el/net/aspose.slides/presentation/save/) της κλάσης [**Presentation**](https://reference.aspose.com/slides/el/net/aspose.slides/presentation/). Το παρακάτω δείγμα κώδικα C# μετατρέπει μια Παρουσίαση από PPTX σε PPT χρησιμοποιώντας τις προεπιλεγμένες επιλογές.

```c#
// Δημιουργήστε ένα αντικείμενο Presentation που αντιπροσωπεύει ένα αρχείο PPTX
Presentation pres = new Presentation("presentation.pptx");

// Αποθήκευση της παρουσίασης PPTX σε μορφή PPT
pres.Save("presentation.ppt", SaveFormat.Ppt);
```

## **Συχνές ερωτήσεις**

**Επιβιώνουν όλα τα εφέ και τα χαρακτηριστικά του PPTX όταν αποθηκεύονται στη κληρονομική μορφή PPT (97–2003);**

Δεν πάντα. Η μορφή PPT δεν διαθέτει ορισμένες νεότερες δυνατότητες (π.χ., συγκεκριμένα εφέ, αντικείμενα και συμπεριφορές), έτσι τα χαρακτηριστικά μπορεί να απλουστευτούν ή να ραστεροποιηθούν κατά τη μετατροπή.

**Μπορώ να μετατρέψω μόνο επιλεγμένες διαφάνειες σε PPT αντί για ολόκληρη την παρουσίαση;**

Η άμεση αποθήκευση στοχεύει ολόκληρη την παρουσίαση. Για να μετατρέψετε συγκεκριμένες διαφάνειες, δημιουργήστε μια νέα παρουσίαση μόνο με αυτές τις διαφάνειες και αποθηκεύστε την ως PPT· εναλλακτικά, χρησιμοποιήστε μια υπηρεσία/API που υποστηρίζει παραμέτρους μετατροπής ανά διαφάνεια.

**Υποστηρίζονται παρουσιάσεις με προστασία κωδικού πρόσβασης;**

Ναι. Μπορείτε να εντοπίσετε αν ένα αρχείο είναι προστατευμένο, να το ανοίξετε με κωδικό πρόσβασης και επίσης [ρυθμίστε τις ρυθμίσεις προστασίας/κρυπτογράφησης](/slides/el/net/password-protected-presentation/) για το αποθηκευμένο PPT.
