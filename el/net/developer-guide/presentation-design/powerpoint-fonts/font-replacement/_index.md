## **Επισκόπηση**

Το Aspose.Slides σάς επιτρέπει να αντικαταστήσετε μια γραμματοσειρά με άλλη σε ολόκληρη την παρουσίαση. Όταν αντικαθίσταται μια γραμματοσειρά, όλες οι εμφανίσεις της αρχικής γραμματοσειράς αλλάζουν στη νέα γραμματοσειρά.

Για να εκτελέσετε την αντικατάσταση γραμματοσειράς, φορτώστε την παρουσίαση, ορίστε τη γραμματοσειρά προέλευσης και τη γραμματοσειρά αντικατάστασης, καλέστε τη μέθοδο αντικατάστασης γραμματοσειράς και αποθηκεύστε την τροποποιημένη παρουσίαση ως αρχείο PPTX. Αυτή η προσέγγιση είναι χρήσιμη όταν θέλετε σκόπιμα να αλλάξετε από μία οικογένεια γραμματοσειρών σε άλλη σε όλη την παρουσίαση.

## **Αντικατάσταση Γραμματοσειρών**

Αν αλλάξετε γνώμη σχετικά με τη χρήση μιας γραμματοσειράς, μπορείτε να την αντικαταστήσετε με μια άλλη. Όλες οι εμφανίσεις της παλιάς γραμματοσειράς θα αντικατασταθούν από τη νέα.

Το Aspose.Slides σάς επιτρέπει να αντικαταστήσετε μια γραμματοσειρά με αυτόν τον τρόπο:

1. Φορτώστε τη σχετική παρουσίαση. 
2. Φορτώστε τη γραμματοσειρά που θα αντικατασταθεί. 
3. Φορτώστε τη νέα γραμματοσειρά. 
4. Αντικαταστήστε τη γραμματοσειρά. 
5. Αποθηκεύστε την τροποποιημένη παρουσίαση ως αρχείο PPTX.

Αυτός ο κώδικας C# παρουσιάζει την αντικατάσταση γραμματοσειράς:

```c#
// Φορτώνει μια παρουσίαση
Presentation presentation = new Presentation("Fonts.pptx");

// Φορτώνει τη γραμματοσειρά προέλευσης που θα αντικατασταθεί
IFontData sourceFont = new FontData("Arial");

// Φορτώνει τη νέα γραμματοσειρά
IFontData destFont = new FontData("Times New Roman");

// Αντικαθιστά τις γραμματοσειρές
presentation.FontsManager.ReplaceFont(sourceFont, destFont);

// Αποθηκεύει την παρουσίαση
presentation.Save("UpdatedFont_out.pptx", SaveFormat.Pptx);
```

{{% alert title="Note" color="warning" %}} 
Για να ορίσετε κανόνες που καθορίζουν τι συμβαίνει σε ορισμένες συνθήκες (π.χ. εάν μια γραμματοσειρά δεν είναι προσβάσιμη), δείτε [**Αντικατάσταση Γραμματοσειράς**](/slides/el/net/font-substitution/). 
{{% /alert %}}

## **Συχνές Ερωτήσεις**

**Ποια είναι η διαφορά μεταξύ «αντικατάστασης γραμματοσειράς», «υποκατάστασης γραμματοσειράς» και «εφεδρικών γραμματοσειρών»;**

Η αντικατάσταση είναι μια σκόπωση αλλαγή από μία οικογένεια γραμματοσειρών σε άλλη σε ολόκληρο το έγγραφο. Η [υποκατάσταση](/slides/el/net/font-substitution/) είναι ένας κανόνας όπως «εάν η γραμματοσειρά δεν είναι διαθέσιμη, χρησιμοποιήστε X». Η [εφεδρική γραμματοσειρά](/slides/el/net/fallback-font/) εφαρμόζεται στοχευμένα για μεμονωμένα ελλιπή γλύφους όταν η βασική γραμματοσειρά είναι εγκατεστημένη αλλά δεν περιέχει τους απαιτούμενους χαρακτήρες.

**Η αντικατάσταση εφαρμόζεται σε κύριες διαφάνειες, διατάξεις, σημειώσεις και σχόλια;**

Ναι. Η αντικατάσταση επηρεάζει όλα τα αντικείμενα της παρουσίασης που χρησιμοποιούν τη αρχική γραμματοσειρά, συμπεριλαμβανομένων των κύριων διαφανειών και των σημειώσεων· τα σχόλια επίσης αποτελούν μέρος του εγγράφου και λαμβάνονται υπόψη από τη μηχανή γραμματοσειρών.

**Θα αλλάξει η γραμματοσειρά μέσα σε ενσωματωμένα αντικείμενα OLE (π.χ., Excel);**

Όχι. Το [περιεχόμενο OLE](/slides/el/net/manage-ole/) ελέγχεται από την δική του εφαρμογή. Η αντικατάσταση στην παρουσίαση δεν επαναμορφοποιεί τα εσωτερικά δεδομένα OLE· μπορεί να εμφανίζεται ως εικόνα ή ως εξωτερικά επεξεργάσιμο περιεχόμενο.

**Μπορώ να αντικαταστήσω μια γραμματοσειρά μόνο σε μέρος της παρουσίασης (ανά διαφάνεια ή περιοχή);**

Η στοχευμένη αντικατάσταση είναι εφικτή εάν αλλάξετε τη γραμματοσειρά σε επίπεδο των απαιτούμενων αντικειμένων/περιοχών αντί να εφαρμόσετε καθολική αντικατάσταση σε ολόκληρο το έγγραφο. Η γενική λογική επιλογής γραμματοσειράς κατά το Rendering παραμένει η ίδια.

**Πώς μπορώ να προσδιορίσω εκ των προτέρων ποιες γραμματοσειρές χρησιμοποιεί η παρουσίαση;**

Χρησιμοποιήστε το [διαχειριστή γραμματοσειρών](https://reference.aspose.com/slides/el/net/aspose.slides/fontsmanager/): παρέχει λίστα με τις [οικογένειες σε χρήση](https://reference.aspose.com/slides/el/net/aspose.slides/fontsmanager/getfonts/) και πληροφορίες για τις [υποκαταστάσεις/«άγνωστες» γραμματοσειρές](https://reference.aspose.com/slides/el/net/aspose.slides/fontsmanager/getsubstitutions/), κάτι που βοηθά στον προγραμματισμό της αντικατάστασης.

**Λειτουργεί η αντικατάσταση γραμματοσειράς κατά τη μετατροπή σε PDF/εικόνες;**

Ναι. Κατά την εξαγωγή, το Aspose.Slides εφαρμόζει την ίδια [ακολουθία επιλογής/υποκατάστασης γραμματοσειράς](/slides/el/net/font-selection-sequence/), έτσι ώστε μια αντικατάσταση που εκτελέστηκε εκ των προτέρων να τηρείται κατά τη μετατροπή.

**Χρειάζεται να εγκαταστήσω τη γραμματοσειρά-στόχο στο σύστημα ή μπορώ να συνημψώ φάκελο γραμματοσειρών;**

Η εγκατάσταση δεν απαιτείται: η βιβλιοθήκη επιτρέπει τη [φόρτωση εξωτερικών γραμματοσειρών](/slides/el/net/custom-font/) από φακέλους χρήστη για χρήση κατά το [rendering και την εξαγωγή](/slides/el/net/convert-powerpoint/).

**Θα διορθώσει η αντικατάσταση τα «τοφού» (τετράγωνα) αντί για χαρακτήρες;**

Μόνο εάν η γραμματοσειρά-στόχος περιέχει πραγματικά τους απαιτούμενους γλύφους. Αν όχι, [ρυθμίστε την εφεδρική γραμματοσειρά](/slides/el/net/fallback-font/) για να καλύψετε τους ελλιπείς χαρακτήρες.
