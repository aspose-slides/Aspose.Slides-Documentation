## **Επισκόπηση**

Όταν μια παρουσίαση φορτώνεται, αποδίδεται ή μετατρέπεται σε άλλη μορφή, το Aspose.Slides ελέγχει εάν οι γραμματοσειρές που χρησιμοποιούνται στην παρουσίαση είναι διαθέσιμες στο λειτουργικό σύστημα. Εάν λείπει μια απαιτούμενη γραμματοσειρά, το Aspose.Slides επιλέγει μια εναλλακτική γραμματοσειρά που είναι όσο το δυνατόν πιο κοντά σε αυτήν που θα χρησιμοποιούσε το PowerPoint.

Το Aspose.Slides πρώτα ψάχνει για τη συγκεκριμένη γραμματοσειρά στο λειτουργικό σύστημα. Εάν η γραμματοσειρά βρεθεί, χρησιμοποιείται. Εάν δεν βρεθεί, εφαρμόζεται μια κατάλληλη εναλλακτική. Όταν οι κανόνες αντικατάστασης γραμματοσειρών ορίζονται μέσω `FontSubstRule`, λαμβάνονται επίσης υπόψη.

Μπορείτε επίσης να προσθέσετε γραμματοσειρές κατά την εκτέλεση της εφαρμογής, να χρησιμοποιήσετε ενσωματωμένες γραμματοσειρές από μια παρουσίαση ή να φορτώσετε εξωτερικές γραμματοσειρές για έγγραφα εξόδου όπως αρχεία PDF.

## **Επιλογή γραμματοσειράς**

Ορισμένοι κανόνες ισχύουν για τις γραμματοσειρές σε μια παρουσίαση όταν αυτή φορτώνεται, αποδίδεται ή μετατρέπεται σε άλλη μορφή. Για παράδειγμα, όταν προσπαθείτε να μετατρέψετε μια παρουσίαση (τις διαφάνειές της) σε εικόνες, ελέγχονται οι γραμματοσειρές της παρουσίασης για να επιβεβαιωθεί ότι οι επιλεγμένες γραμματοσειρές είναι διαθέσιμες στο λειτουργικό σύστημα. Εάν οι γραμματοσειρές διαπιστωθεί ότι λείπουν, αντικαθίστανται — δείτε [**Font Replacement**](https://docs.aspose.com/slides/el/net/font-replacement/) και [**Font Substitution**](https://docs.aspose.com/slides/el/net/font-substitution/).

Αυτή είναι η διαδικασία που ακολουθεί το Aspose.Slides όταν χειρίζεται γραμματοσειρές:

1. Το Aspose.Slides ψάχνει για γραμματοσειρές στο λειτουργικό σύστημα ώστε να βρει τη γραμματοσειρά που ταιριάζει με τη γραμματοσειρά που επέλεξε η παρουσίαση.  
2. Εάν η επιλεγμένη γραμματοσειρά βρεθεί, το Aspose.Slides τη χρησιμοποιεί. Διαφορετικά, το Aspose.Slides χρησιμοποιεί μια εναλλακτική γραμματοσειρά που είναι όσο το δυνατόν πιο κοντά σε αυτή που θα χρησιμοποιούσε το PowerPoint.  
3. Εάν έχουν οριστεί κανόνες αντικατάστασης γραμματοσειρών μέσω [FontSubstRule](https://reference.aspose.com/slides/el/net/aspose.slides/fontsubstrule/), εφαρμόζονται.

Το Aspose.Slides σας επιτρέπει να προσθέσετε γραμματοσειρές στην εκτέλεση της εφαρμογής και στη συνέχεια να τις χρησιμοποιήσετε. Δείτε [**Custom fonts**](https://docs.aspose.com/slides/el/net/custom-font/).

Όταν προστίθενται επιπλέον γραμματοσειρές σε μια παρουσίαση, αυτές ονομάζονται [**Embedded fonts**](https://docs.aspose.com/slides/el/net/embedded-font/).

Το Aspose.Slides σας επιτρέπει να προσθέσετε γραμματοσειρές που εφαρμόζονται *μόνο* σε έγγραφα εξόδου. Για παράδειγμα, εάν μια παρουσίαση που θέλετε να μετατρέψετε σε PDF περιέχει γραμματοσειρές που λείπουν από το σύστημά σας και ενσωματωμένες γραμματοσειρές, μπορείτε να προσθέσετε ή να φορτώσετε τις απαιτούμενες γραμματοσειρές ως **external fonts**.

{{% alert title="Σημείωση" color="primary" %}} 
Δεν διανέμουμε καμία γραμματοσειρά, είτε επί πληρωμή είτε δωρεάν. Το API μας σας επιτρέπει να φορτώσετε εξωτερικές γραμματοσειρές και να τις ενσωματώσετε σε έγγραφα, αλλά το κάνετε με τις γραμματοσειρές που εσείς επιλέγετε και με δική σας ευθύνη.
{{% /alert %}}

## **Συχνές ερωτήσεις**

**Πώς μπορώ να προσδιορίσω ποιες γραμματοσειρές χρησιμοποιούνται πραγματικά σε μια παρουσίαση πριν από τη μετατροπή;**

Το Aspose.Slides σας επιτρέπει να ελέγξετε τις χρησιμοποιούμενες γραμματοσειρές μέσω του [font manager](https://reference.aspose.com/slides/el/net/aspose.slides/presentation/fontsmanager/), ώστε να αποφασίσετε αν θα [ενσωματώσετε](/slides/el/net/embedded-font/), [αντικαταστήσετε](/slides/el/net/font-replacement/) ή προσθέσετε [εξωτερικές πηγές](/slides/el/net/custom-font/). Αυτό βοηθά στην αποφυγή ανεπιθύμητων αντικαταστάσεων κατά την απόδοση και την εξαγωγή.

**Μπορώ να προσθέσω επιπλέον φακέλους γραμματοσειρών χωρίς να τους εγκαταστήσω στο λειτουργικό σύστημα;**

Ναι. Μπορείτε να καταχωρίσετε [εξωτερικές πηγές γραμματοσειρών](/slides/el/net/custom-font/) όπως φακέλους ή ροές στη μνήμη για απόδοση και εξαγωγή. Αυτό αφαιρεί την εξάρτηση από τις γραμματοσειρές του συστήματος φιλοξενίας και διατηρεί την διάταξη προβλέψιμη.

**Πώς αποτρέπω μια σιωπηρή μετάβαση σε ακατάλληλη γραμματοσειρά όταν λείπει κάποιο γράμμα;**

Ορίστε ρητά [κανόνες αντικατάστασης γραμματοσειράς](/slides/el/net/font-replacement/) και κανόνες [fallBack](/slides/el/net/fallback-font/) εκ των προτέρων. Αναλύοντας τις χρησιμοποιούμενες γραμματοσειρές και ορίζοντας ελεγχόμενη προτεραιότητα για τις εναλλακτικές, εξασφαλίζετε συνεπή τυπογραφία και αποτρέπετε μη αναμενόμενα αποτελέσματα.
