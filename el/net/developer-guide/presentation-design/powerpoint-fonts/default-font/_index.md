## **Επισκόπηση**

Aspose.Slides επιτρέπει να ορίσετε προεπιλεγμένες γραμματοσειρές που χρησιμοποιούνται όταν μια παρουσίαση αποδίδεται. Αυτό είναι χρήσιμο όταν δημιουργούνται μικρογραφίες διαφανειών ή εξάγονται παρουσιάσεις σε μορφές όπως PDF και XPS. Οι προεπιλεγμένες γραμματοσειρές ρυθμίζονται μέσω `LoadOptions` πριν φορτωθεί η παρουσίαση.

Η ιδιότητα `DefaultRegularFont` ορίζει τη προεπιλεγμένη γραμματοσειρά για κανονικό κείμενο, ενώ η `DefaultAsianFont` ορίζει τη προεπιλεγμένη γραμματοσειρά για ασιατικό κείμενο. Αφού οριστούν αυτές οι επιλογές, η παρουσίαση μπορεί να φορτωθεί και να αποδοθεί χρησιμοποιώντας τις καθορισμένες γραμματοσειρές.

## **Χρήση προεπιλεγμένων γραμματοσειρών για απόδοση παρουσίασης**
Aspose.Slides σας επιτρέπει να ορίσετε τη προεπιλεγμένη γραμματοσειρά για την απόδοση της παρουσίασης σε PDF, XPS ή μικρογραφίες. Αυτό το άρθρο δείχνει πώς να ορίσετε DefaultRegularFont και DefaultAsianFont ως προεπιλεγμένες γραμματοσειρές. Ακολουθήστε τα παρακάτω βήματα για τη φόρτωση γραμματοσειρών από εξωτερικούς φακέλους χρησιμοποιώντας το Aspose.Slides for .NET API:

1. Δημιουργήστε ένα αντικείμενο LoadOptions.
1. Ορίστε την DefaultRegularFont στη γραμματοσειρά της επιλογής σας. Στο παρακάτω παράδειγμα, έχω χρησιμοποιήσει Wingdings.
1. Ορίστε την DefaultAsianFont στη γραμματοσειρά της επιλογής σας. Στο παρακάτω δείγμα έχω χρησιμοποιήσει Wingdings.
1. Φορτώστε την παρουσίαση χρησιμοποιώντας το Presentation και ορίζοντας τις επιλογές φόρτωσης.
1. Τώρα, δημιουργήστε τη μικρογραφία διαφάνειας, το PDF και το XPS για να επαληθεύσετε τα αποτελέσματα.

```c#
// Χρησιμοποιήστε τις επιλογές φόρτωσης για να καθορίσετε τις προεπιλεγμένες κανονικές και ασιατικές γραμματοσειρές
LoadOptions loadOptions = new LoadOptions(LoadFormat.Auto);
loadOptions.DefaultRegularFont = "Wingdings";
loadOptions.DefaultAsianFont = "Wingdings";

using (Presentation pptx = new Presentation("DefaultFonts.pptx", loadOptions))
{
    using (IImage image = pptx.Slides[0].GetImage(1, 1))
    {
        image.Save("DefaultFonts_out.png", ImageFormat.Png);
    }

    pptx.Save("DefaultFonts_out.pdf", SaveFormat.Pdf);
    pptx.Save("DefaultFonts_out.xps", SaveFormat.Xps);
}
```

## **ΣΥΧΝΕΣ ΕΡΩΤΗΣΕΙΣ**

**Τι ακριβώς επηρεάζουν οι DefaultRegularFont και DefaultAsianFont—μόνο η εξαγωγή, ή και τις μικρογραφίες, PDF, XPS, HTML και SVG;**

Συμμετέχουν στην αλυσίδα απόδοσης για όλα τα υποστηριζόμενα αποτελέσματα. Αυτό περιλαμβάνει μικρογραφίες διαφανειών, [PDF](/slides/el/net/convert-powerpoint-to-pdf/), [XPS](/slides/el/net/convert-powerpoint-to-xps/), [raster images](/slides/el/net/convert-powerpoint-to-png/), [HTML](/slides/el/net/convert-powerpoint-to-html/), και [SVG](/slides/el/net/render-a-slide-as-an-svg-image/), επειδή η Aspose.Slides χρησιμοποιεί την ίδια λογική διάταξης και επίλυσης γλυφών σε όλους τους στόχους.

**Εφαρμόζονται οι προεπιλεγμένες γραμματοσειρές όταν απλώς διαβάζετε και αποθηκεύετε ένα PPTX χωρίς καμία απόδοση;**

Όχι. Οι προεπιλεγμένες γραμματοσειρές έχουν σημασία όταν το κείμενο πρέπει να μετρηθεί και να σχεδιαστεί. Μια απλή λειτουργία ανοίγματος‑αποθήκευσης μιας παρουσίασης δεν αλλάζει τις αποθηκευμένες ακολουθίες γραμματοσειρών ή τη δομή του αρχείου. Οι προεπιλεγμένες γραμματοσειρές παρεμβαίνουν κατά τις λειτουργίες που αποδίδουν ή επαναφορούν κείμενο.

**Αν προσθέσω τους δικούς μου φακέλους γραμματοσειρών ή παρέχω γραμματοσειρές από τη μνήμη, θα ληφθούν υπόψη κατά την επιλογή των προεπιλεγμένων γραμματοσειρών;**

Ναι. Οι [Custom font sources](/slides/el/net/custom-font/) επεκτείνουν τον κατάλογο των διαθέσιμων οικογενειών και γλυφών που μπορεί να χρησιμοποιήσει η μηχανή. Οι προεπιλεγμένες γραμματοσειρές και οι [fallback rules](/slides/el/net/fallback-font/) θα ελέγχονται πρώτα σε αυτές τις πηγές, προσφέροντας πιο αξιόπιστη κάλυψη σε διακομιστές και κοντέινερ.

**Θα επηρεάσουν οι προεπιλεγμένες γραμματοσειρές τις μετρικές του κειμένου (kerning, προοδεύσεις) και συνεπώς τις αλλαγές γραμμής και τη συρραφή;**

Ναι. Η αλλαγή της γραμματοσειράς αλλάζει τις μετρικές των γλυφών και μπορεί να αλλάξει τις αλλαγές γραμμής, τη συρραφή και την σελιδοποίηση κατά την απόδοση. Για σταθερότητα διάταξης, [embed the original fonts](/slides/el/net/embedded-font/) ή επιλέξτε προεπιλεγμένες και εναλλακτικές οικογένειες συμβατές μετρικά.

**Υπάρχει κάποιο νόημα στην ορισμό προεπιλεγμένων γραμματοσειρών εάν όλες οι γραμματοσειρές που χρησιμοποιούνται στην παρουσίαση είναι ενσωματωμένες;**

Συχνά δεν είναι απαραίτητο, επειδή οι [embedded fonts](/slides/el/net/embedded-font/) εξασφαλίζουν ήδη συνεπή εμφάνιση. Οι προεπιλεγμένες γραμματοσειρές εξακολουθούν να βοηθούν ως δικτύωση ασφάλειας για χαρακτήρες που δεν καλύπτονται από το ενσωματωμένο υποσύνολο ή όταν ένα αρχείο συνδυάζει ενσωματωμένο και μη ενσωματωμένο κείμενο.
