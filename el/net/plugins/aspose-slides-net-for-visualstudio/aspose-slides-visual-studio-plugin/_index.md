## **Εισαγωγή**
Microsoft Visual Studio είναι ένα ολοκληρωμένο περιβάλλον ανάπτυξης (IDE) της Microsoft. Χρησιμοποιείται για την ανάπτυξη προγραμμάτων υπολογιστών για τα Windows της Microsoft, καθώς και για ιστοτόπους, web εφαρμογές και web υπηρεσίες. Το Visual Studio χρησιμοποιεί τις πλατφόρμες ανάπτυξης λογισμικού της Microsoft όπως Windows API, Windows Forms, Windows Presentation Foundation, Windows Store και Microsoft Silverlight. Μπορεί να παράγει τόσο κώδικα μη διαχειριζόμενου (native) όσο και διαχειριζόμενο κώδικα.

### **Aspose.Slides for .NET**
Aspose.Slides for .NET είναι ένα μοναδικό API διαχείρισης PowerPoint® που επιτρέπει σε εφαρμογές .NET να διαβάζουν, να γράφουν και να επεξεργάζονται έγγραφα PowerPoint χωρίς να χρησιμοποιούν το Microsoft PowerPoint.

Aspose.Slides for .NET είναι το πρώτο και μοναδικό .NET PowerPoint API που παρέχει λειτουργικότητα διαχείρισης εγγράφων PowerPoint μέσα στις δικές σας εφαρμογές. Όπως και όλα τα συστατικά Aspose .NET, το Aspose.Slides είναι γραμμένο σε C# διαχειριζόμενο, εξαιρετικά αποδοτικό και αστραπιαία γρήγορο.

Για περισσότερες λεπτομέρειες, δείτε [Aspose.Slides for .NET API](http://www.aspose.com/.net/powerpoint-component.aspx)

### **Εξερευνήστε Παραδείγματα Aspose.Slides for .NET Χρησιμοποιώντας το Visual Studio**
Aspose Visual Studio Plugin - **Explore Aspose.Slides for .NET Examples** είναι ένα εξαιρετικό εργαλείο για γρήγορη λήψη και εξερεύνηση παραδειγμάτων του Aspose.Slides for .NET API. Εξοικονομεί χρόνο και προσπάθεια παρέχοντας μια πολύ απλή επιλογή για επιλογή, λήψη και άνοιγμα των τελευταίων έργων παραδείγματος χωρίς να χρειάζεται να φύγετε από το Visual Studio.

Αυτή η έκδοση παρέχει τις ακόλουθες δυνατότητες:

- Υποστηρίζει Visual Studio 2010, Visual Studio 2012 και Visual Studio 2013.
- Εύκολο να εκκινηθεί από το μενού **File > Aspose** στο Visual Studio
- Εξοικονομεί χρόνο και μειώνει την καμπύλη εκμάθησης.
- Σας επιτρέπει να επιλέξετε και να ανοίξετε τα πιο πρόσφατα παραδείγματα Aspose.Slides for .NET API που φιλοξενούνται στο Github.com.
- Σας δίνει τη δυνατότητα να ανοίξετε παραδείγματα έργων **C#** ή **VB.NET**
- Το επιλεγμένο έργο ανοίγει αυτόματα στο Visual Studio με όλες τις απαιτούμενες αναφορές ρυθμισμένες, ώστε να έχετε ένα περιβάλλον έτοιμο για εκτέλεση και εξερεύνηση.
- Λαμβάνει αυτόματα τις τελευταίες βιβλιοθήκες API από το **NuGet** και τα παραδείγματα από το **GitHub**.

## **Απαιτήσεις Συστήματος και Υποστηριζόμενες Πλατφόρμες**
Για να εγκαταστήσετε και να χρησιμοποιήσετε το Aspose Visual Studio Plugin πρέπει να έχετε εγκατεστημένη μία από τις παρακάτω εκδόσεις του Visual Studio:

- Visual Studio 2010, 2012, 2013

Παρακαλούμε μη διστάσετε να επικοινωνήσετε μαζί μας εάν αντιμετωπίσετε προβλήματα κατά την εγκατάσταση ή τη χρήση αυτού του plugin.

## **Εγκατάσταση**
Η εγκατάσταση του Aspose Visual Studio Plugin είναι πολύ απλή· ακολουθήστε τα παρακάτω βήματα:

- Κάντε διπλό κλικ στο ληφθέν αρχείο msi
- Ακολουθήστε τις οδηγίες· μπορείτε απλώς να πατάτε Next σε κάθε οθόνη για να το εγκαταστήσετε με τις προεπιλεγμένες ρυθμίσεις.

Θα εμφανιστεί ένα παράθυρο διαλόγου επιτυχίας όταν ολοκληρωθεί η εγκατάσταση.

**Note:** Βεβαιωθείτε ότι κάνετε επανεκκίνηση του Visual Studio ώστε η αλλαγή να τεθεί σε ισχύ.

## **Χρήση**
Αφού εγκατασταθεί, μπορείτε εύκολα να εκκινήσετε και να χρησιμοποιήσετε το διαδραστικό plugin όπως περιγράφεται παρακάτω:

**1.** Ανοίξτε το plugin επιλέγοντας **File > Aspose** και στη συνέχεια **New Aspose.Slides Example Project** στο Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_3)

**2.** Την πρώτη φορά που ανοίγετε το plugin, θα δείτε το παράθυρο διαλόγου επιλογής API όπως φαίνεται παρακάτω. Θα χρειαστούν λίγα λεπτά για τη λήψη του τελευταίου API και των βιβλιοθηκών του από το **NuGet** καθώς και του κώδικα των τελευταίων παραδειγμάτων από το **GitHub**.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_4)

**3.** Στο παράθυρο διαλόγου **Select an API to View Examples**, επιλέξτε ή αλλάξτε τη θέση όπου θα δημιουργηθούν τα έργα παραδείγματος και επίσης επιλέξτε ή αλλάξτε τον τύπο του έργου παραδείγματος ως **C#** ή **VB.NET** και κάντε κλικ στο **Continue** για να ανοίξετε το έργο παραδείγματος στο Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_1)

**4.** Νέα έργα παραδείγματος ανάλογα με την επιλογή τύπου έργου **C#** ή **VB.NET** στο Visual Studio.

Οι προγραμματιστές ενθαρρύνονται να κατεβάσουν τον κώδικα και να επεκτείνουν τη λειτουργικότητα σύμφωνα με τις δικές τους ανάγκες.

#### **Πώς να Διαμορφώσετε τον Κώδικα Πηγής**
Πρέπει να έχετε εγκατεστημένο το παρακάτω για να ανοίξετε και να επεκτείνετε τον κώδικα:

- Visual Studio 2010

Ακολουθήστε τα παρακάτω απλά βήματα για να ξεκινήσετε:

1. Κατεβάστε/Κλωνοποιήστε τον κώδικα πηγής.
1. Ανοίξτε το Visual Studio 2010 και επιλέξτε **File > Open Project**
1. Περιηγηθείτε στον φάκελο του τελευταίου κώδικα που κατεβάσατε και ανοίξτε **AsposeVisualStudioPlugin.sln**

#### **Επισκόπηση Κώδικα Πηγής**
Η λύση περιλαμβάνει δύο έργα:

- **AsposeVisualStudioPlugin** – Περιέχει την ενσωμάτωση με το Visual Studio, την επιλογή και την αυτόματη λήψη των Aspose APIs και των Παραδειγμάτων.
- **Installer** – Έργο εγκατάστασης Wix για τη δημιουργία αρχείου msi του plugin.
