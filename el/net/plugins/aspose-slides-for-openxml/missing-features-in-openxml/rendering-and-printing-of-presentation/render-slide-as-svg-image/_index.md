SVG—ένα ακρωνύμιο για Scalable Vector Graphics—είναι ένας τυπικός τύπος ή φορμά γραφικών που χρησιμοποιείται για την απόδοση δισδιάστατων εικόνων. Το SVG αποθηκεύει τις εικόνες ως διανύσματα σε XML με λεπτομέρειες που ορίζουν τη συμπεριφορά ή την εμφάνισή τους.

Το SVG είναι ένα από τα λίγα φορμά εικόνας που πληροί πολύ υψηλά πρότυπα σε αυτούς τους τομείς: κλιμακωσιμότητα, διαδραστικότητα, απόδοση, προσβασιμότητα, προγραμματισιμότητα και άλλα. Για αυτούς τους λόγους, χρησιμοποιείται ευρέως στην ανάπτυξη ιστού.

Μπορεί να θέλετε να χρησιμοποιήσετε αρχεία SVG σε αυτές τις περιπτώσεις:

- όταν σκοπεύετε να εκτυπώσετε την παρουσίασή σας σε πολύ μεγάλο φορμά. Οι εικόνες SVG μπορούν να κλιμακώνονται σε οποιαδήποτε ανάλυση ή επίπεδο. Μπορείτε να αλλάζετε το μέγεθος των εικόνων SVG όσες φορές χρειαστεί χωρίς να θυσιάσετε την ποιότητα.
- όταν σκοπεύετε να χρησιμοποιήσετε διαγράμματα και γραφήματα από τις διαφάνειές σας σε διαφορετικά μέσα ή πλατφόρμες. Οι περισσότεροι αναγνώστες μπορούν να ερμηνεύσουν αρχεία SVG.
- όταν χρειάζεστε τις μικρότερες δυνατές διαστάσεις εικόνων. Τα αρχεία SVG είναι γενικά μικρότερα από τα υψηλής ανάλυσης ισοδύναμα τους σε άλλα φορμά, ιδιαίτερα σε φορμά που βασίζονται σε bitmap (JPEG ή PNG).

Το Aspose.Slides for .NET επιτρέπει την εξαγωγή των διαφανειών στις παρουσιάσεις σας ως εικόνες **SVG**. Για να δημιουργήσετε μια εικόνα SVG από οποιαδήποτε, κάντε τα εξής:

- Δημιουργήστε ένα στιγμιότυπο της κλάσης Presentation.
- Περιηγηθείτε σε όλες τις διαφάνειες της παρουσίασης.
- Γράψτε κάθε διαφάνεια σε ξεχωριστό αρχείο SVG μέσω FileStream.

{{% alert color="primary" %}} 
Μπορείτε να δοκιμάσετε την [δωρεάν διαδικτυακή εφαρμογή](https://products.aspose.app/slides/el/conversion/ppt-to-svg) μας, στην οποία υλοποιήσαμε τη λειτουργία μετατροπής PPT σε SVG από το Aspose.Slides for .NET.
{{% /alert %}} 

Αυτό το δείγμα κώδικα σε C# δείχνει πώς να μετατρέψετε PPT σε SVG χρησιμοποιώντας το Aspose.Slides:

``` csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    for (var index = 0; index < pres.Slides.Count; index++)
    {
        ISlide slide = pres.Slides[index];

        using (FileStream fileStream = new FileStream($"slide-{index}.svg", FileMode.Create, FileAccess.Write))
        {
            slide.WriteAsSvg(fileStream);   
        }
    }
}
```
