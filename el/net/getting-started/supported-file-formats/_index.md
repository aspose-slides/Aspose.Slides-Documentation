## **Επισκόπηση**

Η Aspose.Slides υποστηρίζει αρχεία παρουσιάσεων από το Microsoft PowerPoint 97 έως το Office 365, συμπεριλαμβανομένου του Microsoft PowerPoint για Mac. Αυτό το άρθρο παραθέτει τις εκδόσεις PowerPoint που υποστηρίζονται από τη βιβλιοθήκη και παρέχει έναν πίνακα με τις μορφές αρχείων που μπορούν να φορτωθούν, να αποθηκευτούν ή και τα δύο.

Το άρθρο επίσης απαντά σε συχνές ερωτήσεις σχετικά με τη συμμόρφωση PDF, την ενσωμάτωση γραμματοσειρών, αρχεία με κωδικό πρόσβασης, προσαρμοσμένες γραμματοσειρές, εναλλακτικές γραμματοσειρές και τις επιλογές εξαγωγής XPS.

## **Υποστηριζόμενες Εκδόσεις Microsoft PowerPoint**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint για MAC
- Office 365

## **Συχνές Ερωτήσεις**

**Μπορώ να αποθηκεύσω παρουσιάσεις σε PDF που πληρούν τα πρότυπα αρχειοποίησης και προσβασιμότητας (PDF/A και PDF/UA);**

Ναι. Η Aspose.Slides υποστηρίζει εξαγωγή σε PDF με επίπεδα συμμόρφωσης όπως PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b, καθώς και PDF/UA μέσω της ρύθμισης [συμμόρφωση](https://reference.aspose.com/slides/el/net/aspose.slides.export/pdfoptions/compliance/) στα [επιλογές εξαγωγής PDF](https://reference.aspose.com/slides/el/net/aspose.slides.export/pdfoptions/).

**Η βιβλιοθήκη υποστηρίζει την ενσωμάτωση γραμματοσειρών κατά την εξαγωγή σε PDF, με λεπτομερή έλεγχο του τι ενσωματώνεται;**

Ναι. Μπορείτε να ελέγξετε αν οι γραμματοσειρές ενσωματώνονται πλήρως ή με υποσύνολο (μόνο τα χρησιμοποιούμενα γλύφυρα), να ορίσετε πώς αντιμετωπίζονται οι κοινές συστημικές γραμματοσειρές και να ρυθμίσετε τη συμπεριφορά για το κείμενο ASCII μέσω των [επιλογών εξαγωγής PDF](https://reference.aspose.com/slides/el/net/aspose.slides.export/pdfoptions/).

**Μπορώ να ανιχνεύσω αν ένα αρχείο είναι προστατευμένο με κωδικό πριν το φορτώσω;**

Ναι. Χρησιμοποιώντας το [API επιθεώρησης βάσει εργοστασίου](https://reference.aspose.com/slides/el/net/aspose.slides/presentationfactory/), μπορείτε να ερωτήσετε ένα αρχείο παρουσίασης για το αν είναι προστατευμένο με κωδικό χωρίς να το ανοίξετε πλήρως.

**Υπάρχουν μηχανισμοί εναλλακτικών γραμματοσειρών και υποστήριξη προσαρμοσμένων γραμματοσειρών;**

Ναι. Η βιβλιοθήκη υποστηρίζει [φόρτωση](/slides/el/net/custom-font/) και [ενσωμάτωση](/slides/el/net/embedded-font/) προσαρμοσμένων γραμματοσειρών και παρέχει [κανόνες εναλλακτικής γραμματοσειράς](/slides/el/net/fallback-font/) για την αποφυγή ελλιπών γλυφών κατά τη δημιουργία και τη μετατροπή.

**Μπορώ να εξάγω διαφάνειες σε XPS και υπάρχουν επιλογές για προσαρμογή της εξόδου XPS;**

Ναι. Η [Εξαγωγή σε XPS](/slides/el/net/convert-powerpoint-to-xps/) υποστηρίζεται, και μπορείτε να ρυθμίσετε τις σχετικές [επιλογές αποθήκευσης](https://reference.aspose.com/slides/el/net/aspose.slides.export/xpsoptions/) για να ελέγξετε την ποιότητα και το περιεχόμενο του εγγράφου XPS.
