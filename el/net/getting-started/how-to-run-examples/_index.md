## **Απαιτήσεις Λογισμικού**
Πριν κατεβάσετε και εκτελέσετε τα παραδείγματα, παρακαλούμε ελέγξτε και επιβεβαιώστε ότι η ρύθμιση σας πληροί αυτές τις απαιτήσεις: 

- Visual Studio 2010 ή νεότερο.
- NuGet Package Manager εγκατεστημένος στο Visual Studio. Επαληθεύστε ότι η τελευταία έκδοση του NuGet API είναι εγκατεστημένη στο Visual Studio. 

Για οδηγίες σχετικά με την εγκατάσταση του NuGet package manager, μεταβείτε σε αυτή τη σελίδα: https://docs.microsoft.com/en-us/nuget/install-nuget-client-tools

1. Μεταβείτε σε **Tools** > **Options** > **NuGet Package Manager**.

1. Αναπτύξτε το **NuGet Package Manager** (κάνοντας διπλό κλικ επάνω του) και στη συνέχεια επιλέξτε **Package Sources**. 

1. Ελέγξτε και επιβεβαιώστε ότι η παράμετρος nuget.org είναι επιλεγμένη. 

   Το παράδειγμα έργου χρησιμοποιεί τη λειτουργία NuGet Automatic Package Restore, έτσι χρειάζεται να έχετε ενεργή σύνδεση στο διαδίκτυο. 

   Εάν δεν έχετε ενεργή σύνδεση στο διαδίκτυο στον υπολογιστή όπου προτίθεστε να εκτελέσετε τα παραδείγματα, παρακαλούμε ελέγξτε [Installation](https://docs.aspose.com/slides/el/net/installation/) και (χειροκίνητα) προσθέστε μια αναφορά στο Aspose.Slides.dll στο έργο του παραδείγματος.
## **Λήψη Aspose.Slides από το GitHub**
Όλα τα παραδείγματα Aspose.Slides για .NET φιλοξενούνται στο [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-.NET).

Μπορείτε είτε να κλωνοποιήσετε το αποθετήριο χρησιμοποιώντας τον αγαπημένο σας πελάτη GitHub ή να κατεβάσετε το αρχείο ZIP [εδώ](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/master.zip).

1. Εάν κατεβάσετε το αρχείο ZIP, πρέπει να εξάγετε τα περιεχόμενά του σε φάκελο στον υπολογιστή σας. 

Όλα τα παραδείγματα αποθηκεύονται στον φάκελο **Examples**.

Υπάρχει ένα αρχείο λύσης C# Visual Studio. Τα έργα δημιουργήθηκαν στο Visual Studio 2013, αλλά τα αρχεία λύσης είναι συμβατά με το Visual Studio 2010 SP1 και άνω.

2. Ανοίξτε το αρχείο λύσης στο Visual Studio και δημιουργήστε (build) το έργο.

   Κατά την πρώτη εκτέλεση, οι εξαρτήσεις κατεβάζονται αυτόματα μέσω NuGet.

Ο φάκελο **Data** στη ρίζα του φακέλου **Examples** περιέχει αρχεία εισόδου που χρησιμοποιούνται στα παραδείγματα C#. Πρέπει να κατεβάσετε τον φάκελο **Data** μαζί με το έργο των παραδειγμάτων.

3. Ανοίξτε το αρχείο RunExamples.cs. Όλα τα παραδείγματα καλούνται από εδώ.

4. Αποσχολιάστε τα παραδείγματα που θέλετε να εκτελέσετε εντός του έργου.

Μη διστάσετε να επικοινωνήσετε μέσω των φόρουμ μας εάν αντιμετωπίζετε προβλήματα με τη ρύθμιση ή την εκτέλεση των παραδειγμάτων.
## **Συνεισφορά**
Μπορείτε να συνεισφέρετε στο έργο προσθέτοντας ή βελτιώνοντας ένα παράδειγμα. Όλα τα παραδείγματα και τα έργα επίδειξης στο αποθετήριο είναι ανοιχτού κώδικα, ώστε εσείς (και άλλοι) να τα χρησιμοποιείτε ελεύθερα σε εφαρμογές.

Για να συνεισφέρετε, μπορείτε να κάνετε fork το αποθετήριο, να επεξεργαστείτε τον πηγαίο κώδικα και να δημιουργήσετε ένα pull request. Θα εξετάσουμε τις αλλαγές. Εάν τις βρούμε χρήσιμες, θα τις προσθέσουμε στο αποθετήριο.
