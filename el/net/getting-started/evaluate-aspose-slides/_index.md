## **Αξιολόγηση Aspose.Slides**

Μπορείτε εύκολα να κατεβάσετε το Aspose.Slides για αξιολόγηση. Το πακέτο αξιολόγησης είναι το ίδιο με το αγορασμένο πακέτο. Η έκδοση αξιολόγησης γίνεται απλώς άδεια μετά την προσθήκη μερικών γραμμών κώδικα για την εφαρμογή της άδειας. 

Η έκδοση αξιολόγησης του Aspose.Slides (χωρίς καθορισμένη άδεια) παρέχει πλήρη λειτουργικότητα του προϊόντος, αλλά εισάγει ένα υδατογράφημα αξιολόγησης στην κορυφή του εγγράφου κατά το άνοιγμα και την αποθήκευση. Επίσης, περιορίζεστε σε μία διαφάνεια όταν εξάγετε κείμενα από διαφάνειες παρουσίασης.

## **ΣΥΧΝΑ ΕΡΩΤΗΣΕΙΣ**

**Μπορώ να δοκιμάσω πολλαπλές παρουσιάσεις παράλληλα σε διαφορετικά νήματα σε λειτουργία αξιολόγησης;**

Ναι. Μπορείτε να επεξεργάζεστε διαφορετικά έγγραφα παράλληλα· δεν πρέπει να μοιράζεστε το ίδιο αντικείμενο παρουσίασης [μεταξύ νημάτων](/slides/el/net/multithreading/). Η λειτουργία αξιολόγησης δεν επηρεάζει αυτό.

**Χρειάζεται να εγκαταστήσω το Microsoft PowerPoint για να αξιολογήσω τη βιβλιοθήκη σε διακομιστή ή σε CI;**

Όχι. Το Aspose.Slides είναι μια αυτόνομη μηχανή και δεν απαιτεί την εγκατάσταση του PowerPoint τόσο για αξιολόγηση όσο και για παραγωγή.

**Μπορώ να δοκιμάσω πλήρως τη μετατροπή από PPT/PPTX σε PDF και εικόνες σε λειτουργία αξιολόγησης;**

Ναι. Οι [μετατροπείς](/slides/el/net/convert-presentation/) λειτουργούν· η έξοδος θα περιέχει υδατογράφημα.

**Μπορώ να χρησιμοποιήσω προσωρινή άδεια για δοκιμές φορτίου χωρίς υδατογράφημα;**

Ναι. Μια 30‑ημέρη προσωρινή άδεια αφαιρεί τους περιορισμούς της λειτουργίας αξιολόγησης και επιτρέπει δοκιμές χωρίς υδατογράφημα.
