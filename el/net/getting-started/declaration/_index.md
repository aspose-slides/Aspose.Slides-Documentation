{{% alert color="primary" %}} 

Όλα τα στοιχεία Aspose .NET απαιτούν το σύνολο αδειοδοτήσεων Full Trust επειδή μερικές φορές πρέπει να έχουν πρόσβαση σε ρυθμίσεις μητρώου, σύστημα αρχείων και αρχεία αποθηκευμένα σε άλλες τοποθεσίες (εκτός από τον εικονικό φάκελο) για ορισμένες λειτουργίες (π.χ. ανάλυση γραμματοσειρών). Επιπλέον, τα στοιχεία Aspose .NET βασίζονται σε βασικές κλάσεις συστήματος .NET, οι οποίες απαιτούν το σύνολο αδειοδοτήσεων Full Trust σε πολλές περιπτώσεις. 

{{% /alert %}} 

Οι πάροχοι υπηρεσιών διαδικτύου (Internet Service Providers), που φιλοξενούν πολλαπλές εφαρμογές από διαφορετικές εταιρείες, συνήθως επιβάλλουν το επίπεδο ασφαλείας Medium Trust. Σε περίπτωση .NET 2.0, ένα τέτοιο επίπεδο ασφαλείας εφαρμόζει τους ακόλουθους περιορισμούς: 

- OleDbPermission δεν είναι διαθέσιμο. Αυτό σημαίνει ότι δεν μπορείτε να χρησιμοποιήσετε τον διαχειριζόμενο πάροχο δεδομένων OLE DB της ADO.NET για πρόσβαση σε βάσεις δεδομένων.  
- EventLogPermission δεν είναι διαθέσιμο. Αυτό σημαίνει ότι δεν μπορείτε να έχετε πρόσβαση στο ημερολόγιο συμβάντων των Windows.  
- ReflectionPermission δεν είναι διαθέσιμο. Αυτό σημαίνει ότι δεν μπορείτε να χρησιμοποιήσετε reflection.  
- RegistryPermission δεν είναι διαθέσιμο. Αυτό σημαίνει ότι δεν μπορείτε να έχετε πρόσβαση στο μητρώο.  
- WebPermission είναι περιορισμένο. Αυτό σημαίνει ότι η εφαρμογή σας μπορεί να επικοινωνεί μόνο με μια διεύθυνση ή την περιοχή διευθύνσεων που έχετε ορίσει στο στοιχείο <trust>.  
- FileIOPermission είναι περιορισμένο. Αυτό σημαίνει ότι μπορείτε να έχετε πρόσβαση μόνο σε αρχεία στη ιεραρχία του εικονικού καταλόγου της εφαρμογής σας.  

{{% alert color="primary" %}} 

Λόγω των παραπάνω λόγων, τα στοιχεία Aspose .NET μπορούν να χρησιμοποιηθούν μόνο σε διακομιστές που παρέχουν το σύνολο αδειοδοτήσεων Full Trust. 

{{% /alert %}}
