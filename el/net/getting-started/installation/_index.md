## **Επισκόπηση**

Αυτό το άρθρο εξηγεί πώς να εγκαταστήσετε το Aspose.Slides για .NET στα Windows και macOS. Επικεντρώνεται στην εγκατάσταση μέσω NuGet και δείχνει πώς να προσθέσετε τη βιβλιοθήκη σε ένα έργο Visual Studio είτε μέσω του NuGet Package Manager είτε του Package Manager Console στα Windows. Περιγράφει επίσης πώς να ενημερώσετε το πακέτο και να εγκαταστήσετε εκδόσεις προ‑απελευθέρωσης όταν χρειάζεται.

## **Windows**
Το NuGet προσφέρει την πιο εύκολη διαδρομή για τη λήψη και εγκατάσταση των Aspose API για .NET σε υπολογιστές. 

### **Μέθοδος 1: Εγκατάσταση ή Ενημέρωση του Aspose.Slides από το NuGet Package Manager**

1. Ανοίξτε το Microsoft Visual Studio. 
2. Δημιουργήστε μια απλή εφαρμογή κονσόλας ή ανοίξτε ένα υπάρχον έργο. 
3. Μεταβείτε στο **Tools** > **NuGet package manager**.
4. Στην ενότητα **Browse**, αναζητήστε *Aspose Slides* στο πεδίο κειμένου. 
{{% image img="installation_1.png" alt="Εγκατάσταση Aspose.Slides από το NuGet Package Manager - 1" %}}
5. Κάντε κλικ στο **Aspose.Slides.NET** και στη συνέχεια κάντε κλικ στο **Install**. 
   * Εάν θέλετε να ενημερώσετε το Aspose.Slides—υποθέτοντας ότι το έχετε ήδη εγκαταστήσει—κάντε κλικ στο **Update** αντί αυτού. 

Το επιλεγμένο API κατεβάζεται και προστίθεται ως αναφορά στο έργο σας.

### **Μέθοδος 2: Εγκατάσταση ή Ενημέρωση του Aspose.Slides μέσω του Package Manager Console**

Αυτή είναι η διαδικασία για να αναφερθείτε στο [Aspose.Slides API](https://www.nuget.org/packages/Aspose.Slides.NET/) μέσω του package manager console:

1. Ανοίξτε το Microsoft Visual Studio. 
2. Δημιουργήστε μια απλή εφαρμογή κονσόλας ή ανοίξτε ένα υπάρχον έργο. 
3. Μεταβείτε στο **Tools** > **Library Package Manager** > **Package Manager Console**. 
![todo:image_alt_text](installation_2.png)
4. Εκτελέστε αυτήν την εντολή: `Install-Package Aspose.Slides.NET` 
![todo:image_alt_text](installation_3.png)
Η πιο πρόσφατη πλήρης έκδοση εγκαθίσταται στην εφαρμογή σας. 

* Εναλλακτικά, μπορείτε να προσθέσετε το επίθημα `-prerelease` στην εντολή για να υποδείξετε ότι η πιο πρόσφατη έκδοση (συμπεριλαμβανομένων των hotfix) πρέπει επίσης να εγκατασταθεί. 

Η συμβουλή **Installing Aspose.Slides.NET** εμφανίζεται κοντά στο κάτω μέρος του παραθύρου. 
![todo:image_alt_text](installation_4.png)

Μόλις ολοκληρωθεί η λήψη, θα πρέπει να δείτε κάποια μηνύματα επιβεβαίωσης. 

Εάν δεν γνωρίζετε το [Aspose EULA](https://about.aspose.com/legal/eula), ίσως θελήσετε να διαβάσετε την άδεια που αναφέρεται στη διεύθυνση URL. 
![todo:image_alt_text](installation_5.png)

Στην εφαρμογή σας, θα πρέπει να δείτε ότι το Aspose.Slides έχει προστεθεί και αναφερθεί επιτυχώς. 
![todo:image_alt_text](installation_6.png)

Στο Package Manager Console, μπορείτε να εκτελέσετε την εντολή `Update-Package Aspose.Slides.NET` για να ελέγξετε ενημερώσεις του πακέτου Aspose.Slides. Οι ενημερώσεις (αν βρεθούν) εγκαθίστανται αυτόματα. Μπορείτε επίσης να χρησιμοποιήσετε το επίθημα `-prerelease` για να ενημερώσετε στην τελευταία έκδοση.
#### **Σκέψεις όταν εκτελείται σε κοινό περιβάλλον διακομιστή**
Συνιστούμε ανεπιφύλακτα να εκτελείτε όλα τα στοιχεία Aspose .NET με το σύνολο δικαιωμάτων **Full Trust**, επειδή τα στοιχεία Aspose μερικές φορές χρειάζεται να έχουν πρόσβαση σε ρυθμίσεις μητρώου και αρχεία που βρίσκονται σε τοποθεσίες εκτός του εικονικού καταλόγου — για παράδειγμα, όταν τα στοιχεία Aspose πρέπει να διαβάσουν γραμματοσειρές. 

Επιπλέον, τα στοιχεία Aspose.NET βασίζονται στις κεντρικές κλάσεις συστήματος .NET — και ορισμένες από αυτές τις κλάσεις απαιτούν επίσης δικαίωμα Full Trust για ορισμένες λειτουργίες. 

Οι πάροχοι υπηρεσιών Internet (ISP), που φιλοξενούν πολλαπλές εφαρμογές από διαφορετικές εταιρείες, συνήθως επιβάλλουν το επίπεδο ασφαλείας Medium Trust. Στην περίπτωση του .NET 2.0, αυτό το επίπεδο ασφαλείας μπορεί να δημιουργήσει περιορισμούς που επηρεάζουν τις λειτουργίες του Aspose.Slides:

- **RegistryPermission** δεν είναι διαθέσιμο. Αυτό σημαίνει ότι δεν μπορείτε να έχετε πρόσβαση στο μητρώο, το οποίο απαιτείται για την καταγραφή των εγκατεστημένων γραμματοσειρών κατά τη δημιουργία εγγράφων.
- **FileIOPermission** είναι περιορισμένο. Αυτό σημαίνει ότι μπορείτε να έχετε πρόσβαση μόνο σε αρχεία στην ιεραρχία του εικονικού καταλόγου της εφαρμογής σας. Αυτό ενδέχεται επίσης να σημαίνει ότι οι γραμματοσειρές δεν μπορούν να διαβαστούν κατά τις εξαγωγικές λειτουργίες. 

Για τους παραπάνω λόγους, συνιστούμε ανεπιφύλακτα να εκτελείτε το Aspose.Slides με δικαιώματα **Full Trust**. Εάν χρησιμοποιείτε **Medium trust**, μπορεί να αντιμετωπίσετε ασυνέπειες — ορισμένες λειτουργίες της βιβλιοθήκης (π.χ. η απόδοση) ενδέχεται να μην λειτουργούν όταν εκτελείτε συγκεκριμένες εργασίες. 

## **macOS**

Το NuGet προσφέρει την πιο εύκολη διαδρομή για τη λήψη και εγκατάσταση του Aspose.Slides για .NET σε Mac. 

**Εγκατάσταση Προαπαιτημάτων**

Ο χώρος ονομάτων `System.Drawing` λειτουργεί διαφορετικά στο macOS, γι' αυτό πρέπει να εγκαταστήσετε το mono-libgdiplus. 

> Στο .NET 5 και σε προηγούμενες εκδόσεις, το πακέτο NuGet [System.Drawing.Common](https://www.nuget.org/packages/System.Drawing.Common/) λειτουργεί σε Windows, Linux και macOS. Ωστόσο, υπάρχουν κάποιες διαφορές πλατφόρμας. Σε Linux και macOS, η λειτουργικότητα GDI+ υλοποιείται από τη βιβλιοθήκη [libgdiplus)](https://www.mono-project.com/docs/gui/libgdiplus/). Αυτή η βιβλιοθήκη δεν εγκαθίσταται εξ' ορισμού στις περισσότερες διανομές Linux και δεν υποστηρίζει όλη τη λειτουργικότητα του GDI+ σε Windows και macOS. Υπάρχουν επίσης πλατφόρμες όπου το libgdiplus δεν είναι καθόλου διαθέσιμο. Για να χρησιμοποιήσετε τύπους από το πακέτο System.Drawing.Common σε Linux και macOS, πρέπει να εγκαταστήσετε ξεχωριστά το libgdiplus. Για περισσότερες πληροφορίες, δείτε [Install .NET on Linux](https://docs.microsoft.com/en-us/dotnet/core/install/linux) ή [Install .NET on macOS](https://docs.microsoft.com/en-us/dotnet/core/install/macos#libgdiplus).

Για να εγκαταστήσετε το mono-libgdiplus ξεχωριστά στο mac σας, δείτε [αυτό το άρθρο](https://docs.microsoft.com/en-us/dotnet/core/install/macos#libgdiplus) από την τεκμηρίωση .NET. 

### **Εγκατάσταση Aspose.Slides**

1. Ανοίξτε το Visual Studio. 
2. Δημιουργήστε μια απλή εφαρμογή κονσόλας ή ανοίξτε ένα υπάρχον έργο.
3. Μεταβείτε στο **Project** > **Manage NuGet Packages...**
   ![path-to-nuget-macos](path-to-nuget-macos.png)
4. Πληκτρολογήστε *Aspose.Slides* στο πεδίο κειμένου. 
5. Κάντε κλικ στο **Aspose.Slides for .NET** και στη συνέχεια κάντε κλικ στο **Add Package.** 
6. Προσθέστε ένα απλό απόσπασμα κώδικα.
   * Μπορείτε να αντιγράψετε τον κώδικα από τη [αυτή τη σελίδα](/slides/el/net/create-presentation/).
7. Εκτελέστε την εφαρμογή.
8. Ανοίξτε το *folder/bin/Debug/presentation_file_name* του έργου σας.
