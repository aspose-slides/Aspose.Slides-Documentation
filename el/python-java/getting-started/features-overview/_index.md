## **Υποστηριζόμενες Πλατφόρμες**
Aspose.Slides for Python via Java είναι ένα ανεξάρτητο από την πλατφόρμα API. Μπορεί να χρησιμοποιηθεί σε οποιαδήποτε πλατφόρμα (Windows, Linux, MacOS κ.λπ.) όπου είναι εγκατεστημένη η Python 3.7 ή νεότερη. Το JDK 8 ή νεότερο είναι μια ακόμη απαίτηση.

## **Μορφές Αρχείων και Μετατροπές**
Το Aspose.Slides for Python via Java υποστηρίζει τις περισσότερες μορφές εγγράφων PowerPoint. Επιτρέπει επίσης την εξαγωγή τους σε δημοφιλείς μορφές αρχείων που χρησιμοποιούν οι οργανισμοί και τις μοιράζονται μεταξύ τους. Δείτε αυτές τις λεπτομέρειες:

|**Λειτουργία**|**Περιγραφή**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Το Aspose.Slides for Python via Java παρέχει την ταχύτερη επεξεργασία για αυτή τη μορφή εγγράφου παρουσίασης.|
|PPT to PPTX conversion|Το Aspose.Slides for Python via Java υποστηρίζει τη μετατροπή PPT σε PPTX|
|Portable Document Format (PDF)|Μπορείτε να εξάγετε όλες τις υποστηριζόμενες μορφές αρχείων σε έγγραφα Adobe Portable Document Format (PDF) με μία μόνο μέθοδο.| 
|XML Parser Specification (XPS)|Μπορείτε να εξάγετε όλες τις υποστηριζόμενες μορφές αρχείων σε έγγραφα XML Parser Specification (XPS) με μία μόνο μέθοδο.| 
|Tagged Image File Format (TIFF)|Μπορείτε να εξάγετε όλες τις υποστηριζόμενες μορφές αρχείων παρουσίασης σε Tagged Image File Format (TIFF).|
|PPTX To HTML Conversion|Το Aspose.Slides for Python via Java υποστηρίζει τη μετατροπή του PresentationEx σε μορφή HTML.|

## **Απόδοση και Εκτύπωση**
Το Aspose.Slides for Python via Java υποστηρίζει αποτύπωση υψηλής πιστότητας των διαφανειών μιας παρουσίασης σε διαφορετικές μορφές γραφικών. Δείτε αυτές τις λεπτομέρειες:

|**Λειτουργία**|**Περιγραφή**|
| :- | :- |
|Java Supported Image Formats|Χρησιμοποιώντας το Aspose.Slides for Python via Java, μπορείτε να αποτυπώσετε τις διαφάνειες της παρουσίασης και τις εικόνες στις διαφάνειες σε όλες τις μορφές γραφικών που υποστηρίζει η Java, όπως TIFF, PNG, BMP, JPEG, GIF.| 
|SVG Format|Το Aspose.Slides for Python via Java επίσης προσφέρει ενσωματωμένες μεθόδους που επιτρέπουν την εξαγωγή των διαφανειών παρουσίασης σε μορφές Scalable Vector Graphics (SVG).|
|Presentation Printing|Οι τελευταίες εκδόσεις του Aspose.Slides for Python via Java παρέχουν ενσωματωμένες μεθόδους εκτύπωσης με διαφορετικές επιλογές.|

|Text Boxes|Μπορείτε να δημιουργήσετε πλαίσια κειμένου από το μηδέν. Μπορείτε να έχετε πρόσβαση σε υπάρχοντα πλαίσια κειμένου. Μπορείτε επίσης να τροποποιήσετε το κείμενό τους χωρίς να χάσετε την αρχική μορφοποίηση του κειμένου.|

|**Λειτουργία**|**Περιγραφή**|
| :- | :- |
|Text Formatting|<p>Στο Aspose.Slides for Python via Java, μπορείτε να διαχειριστείτε κείμενα μέσω των πλαισίων κειμένου που σχετίζονται με τα σχήματα. Έτσι, μπορείτε να μορφοποιήσετε κείμενα χρησιμοποιώντας τις παραγράφους και τα τμήματα που σχετίζονται με τα πλαίσια κειμένου. Τα στοιχεία κειμένου μπορούν να μορφοποιηθούν μέσω του Aspose.Slides for Python via Java.</p><p>- Τύπος γραμματοσειράς</p><p>- Μέγεθος γραμματοσειράς</p><p>- Χρώμα γραμματοσειράς</p><p>- Σκιαγραφίες γραμματοσειράς</p><p>- Στοίχιση παραγράφου</p><p>- Κουκίδες παραγράφου</p><p>- Προσανατολισμός παραγράφου</p>|
|Shape Formatting|<p>Στο Aspose.Slides for Python via Java, το βασικό στοιχείο μιας διαφάνειας είναι ένα σχήμα.
