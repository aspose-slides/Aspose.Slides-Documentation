Μπορείτε εύκολα να κατεβάσετε το Aspose.Slides για αξιολόγηση. Το πακέτο αξιολόγησης είναι το ίδιο με το αγορασμένο πακέτο. Η έκδοση αξιολόγησης γίνεται απλώς αδειοδοτημένη μετά την προσθήκη μερικών γραμμών κώδικα για την εφαρμογή της άδειας. 

Η έκδοση αξιολόγησης του Aspose.Slides (χωρίς ορισμό άδειας) παρέχει πλήρη λειτουργικότητα του προϊόντος, αλλά προσθέτει ένα υδατογράφημα αξιολόγησης στην κορυφή του εγγράφου κατά το άνοιγμα και την αποθήκευση. Επιπλέον, περιορίζεστε σε μία διαφάνεια όταν εξάγετε κείμενα από τις διαφάνειες της παρουσίασης.
