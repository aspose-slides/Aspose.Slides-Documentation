Για να αποφύγετε τους περιορισμούς που σχετίζονται με την έκδοση αξιολόγησης, πρέπει να ορίσετε μια άδεια πριν χρησιμοποιήσετε το **Aspose.Slides**. Απαιτείται να ορίσετε την άδεια μόνο μία φορά ανά εφαρμογή ή διαδικασία.

#### **Εφαρμογή Άδειας με Αρχείο**
Αυτό το απόσπασμα κώδικα χρησιμοποιείται για τον ορισμό ενός αρχείου άδειας:

**Python**

```python
import jpype
import asposeslides

jpype.startJVM()

from asposeslides.api import Presentation, License

license = License();
pres = Presentation()
license.setLicense("Aspose.Slides.lic");

jpype.shutdownJVM()
```

Κατά την κλήση της μεθόδου setLicense, το όνομα της άδειας πρέπει να είναι το ίδιο με το όνομα του αρχείου άδειας σας. Για παράδειγμα, μπορείτε να αλλάξετε το όνομα του αρχείου άδειας σε "Aspose.Slides.lic.xml". Στη συνέχεια, στον κώδικά σας, πρέπει να περάσετε το νέο όνομα άδειας (Aspose.Slides.lic.xml) στη μέθοδο setLicense.

#### **Εφαρμογή Άδειας από Bytes**
Αυτό το απόσπασμα κώδικα χρησιμοποιείται για την εφαρμογή άδειας από bytes:

**Python**

```python
import jpype
import asposeslides

jpype.startJVM()

from asposeslides.api import Presentation, License

license = License();
input = open("Aspose.Slides.lic", mode="rb")
data = input.read()
pres = Presentation()
license.setLicenseFromBytes(data);

jpype.shutdownJVM()
```

#### Εφαρμογή Μετρημένης Άδειας
Το Aspose.Slides επιτρέπει στους προγραμματιστές να εφαρμόσουν ένα μετρημένο κλειδί. Αυτός είναι ένας νέος μηχανισμός αδειοδότησης.

Ο νέος μηχανισμός αδειοδότησης θα χρησιμοποιείται μαζί με την υπάρχουσα μέθοδο αδειοδότησης. Οι πελάτες που θέλουν να χρεώνονται βάσει της χρήσης των λειτουργιών API μπορούν να χρησιμοποιήσουν τη Μετρημένη Αδειοδότηση.

Αφού ολοκληρώσετε όλα τα απαραίτητα βήματα για την απόκτηση αυτού του τύπου άδειας, θα λάβετε τα κλειδιά, όχι το αρχείο άδειας. Αυτό το μετρημένο κλειδί μπορεί να εφαρμοστεί χρησιμοποιώντας την κλάση **Metered** που έχει ειδικά εισαχθεί για αυτό το σκοπό.

Το παρακάτω παράδειγμα κώδικα δείχνει πώς να ορίσετε δημόσια και ιδιωτικά κλειδιά μετρημένης άδειας:

```python
import jpype
import asposeslides

jpype.startJVM()

from asposeslides.api import Presentation, Metered, SaveFormat

# Δημιουργήστε μια παρουσία της κλάσης CAD Metered
metered = Metered();

# Πρόσβαση στην ιδιότητα set_metered_key και πέρασμα δημόσιων και ιδιωτικών κλειδιών ως παραμέτρων
metered.setMeteredKey("*****", "*****");

# Λάβετε την ποσότητα δεδομένων μέτρησης πριν καλέσετε το API
amountbefore = Metered.getConsumptionQuantity()

# Εμφάνιση πληροφοριών
print("Amount Consumed Before: \" + amountbefore + \"" )

# Φόρτωση του εγγράφου από τον δίσκο.
pres = Presentation();

# Λάβετε τον αριθμό σελίδων του εγγράφου
print("Amount Consumed After: \" +  pres.getSlides().size()) + \"" )

# αποθήκευση ως PDF
pres.save("out_pdf.pdf", SaveFormat.Pdf);

# Λάβετε την ποσότητα δεδομένων μέτρησης μετά την κλήση του API
amountafter = Metered.getConsumptionQuantity()

# Εμφάνιση πληροφοριών
print("Amount Consumed After: \" + amountafter + \"" )

jpype.shutdownJVM()
```
