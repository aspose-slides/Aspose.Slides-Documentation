Όλες οι demo που παρέχονται με Aspose.Slides για JasperReports είναι τροποποιημένες τυπικές demo. Είναι καλύτερο να αντιγράψετε όλες τις demo στο φάκελο demo του JasperReports:
...\jasperreports-x.x.x\demo\samples\

Χρησιμοποιήστε την τυπική ακολουθία εντολών για τη δημιουργία και εξαγωγή των εκθέσεων:

- ant javac
- ant compile
- ant fill
- ant ppt

{{% alert color="primary" %}} 
Παρακαλούμε μην ξεχάσετε να εκτελέσετε το HSQLDB με τη δοκιμαστική βάση δεδομένων για να γεμίσετε τις εκθέσεις με δεδομένα και να αντιγράψετε το aspose.slides.jasperreports.library-xx.x.jar από τον φάκελο \lib\JasperReports X.X.X - X.X.X του αρχείου aspose-slides-xx.x-jasperreports.zip στο &#60;InstallDir&#62;\lib directory.
{{% /alert %}} 
Οι περισσότερες demo (εκτός από Charts) έχουν ήδη παραγόμενες παρουσιάσεις, οπότε μπορείτε να παραλείψετε όλα τα βήματα “ant” και να ελέγξετε τα αποτελέσματα αμέσως.
