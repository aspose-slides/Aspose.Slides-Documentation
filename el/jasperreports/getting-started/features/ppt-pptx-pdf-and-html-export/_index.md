{{% alert color="primary" %}} 

Το JasperReports και το JasperServer δεν διαθέτουν ενσωματωμένες δυνατότητες εξαγωγής αναφορών ως παρουσιάσεις Microsoft PowerPoint, αλλά με το Aspose.Slides for JasperReports έχετε πρόσβαση σε πρόσθετες μορφές εξαγωγής:

- Παρουσίαση Microsoft PowerPoint (PPT)
- Παρουσίαση Microsoft PowerPoint (PPTX)
- HTML
- PDF

{{% /alert %}} 

Για τη δημιουργία εγγράφων σε αυτές τις μορφές, το Aspose.Slides for JasperReports βασίζεται σε μια ενσωματωμένη έκδοση του [Aspose.Slides for Java](https://products.aspose.com/slides/el/java/), της κορυφαίας στην αγορά βιβλιοθήκης επεξεργασίας παρουσιάσεων από την Aspose. Το Microsoft PowerPoint δεν χρησιμοποιείται για τη δημιουργία των εγγράφων.
