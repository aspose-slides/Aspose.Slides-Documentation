## **Επισκόπηση**

Αυτό το άρθρο περιγράφει ένα σφάλμα αδειοδότησης που εμφανίστηκε στις εκδόσεις 23.2, 23.3, 23.4, 23.5, 23.6 και 23.7 του Aspose.Slides for PHP via Java. Αυτό το σφάλμα έκανε τις μονίμες άδειες να γίνονται προσωρινά μη χρησιμοποιήσιμες μετά τη λήξη της συνδρομής τους.

## **Συμπτώματα**

Αφού λήξει η συνδρομή της μονίμης άδειάς σας, οι εκδόσεις 23.2‑23.7 του Aspose.Slides for PHP via Java μπορεί να αναφέρουν ότι η άδεια έχει λήξει, μεταβαίνοντας όλες τις λειτουργίες σε λειτουργία αξιολόγησης. Αυτή η συμπεριφορά είναι λανθασμένη και δεν επηρεάζει τις εκδόσεις προ 23.2 και μετά 23.7.

## **Επίλυση**

Το Aspose.Slides for PHP via Java αντιμετώπισε αυτό το ζήτημα και κυκλοφόρησε ενημερωμένες εκδόσεις (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) με τη διόρθωση.

Αν χρησιμοποιείτε μία από τις επηρεαζόμενες εκδόσεις του Aspose.Slides for PHP via Java στο έργο σας, παρακαλούμε μεταβείτε στη διορθωμένη έκδοση.

Εάν αντιμετωπίσετε προβλήματα αδειοδότησης με το Aspose.Slides for PHP via Java:

- Ελέγξτε την έκδοση που χρησιμοποιείτε και βεβαιωθείτε ότι δεν επηρεάζεται από το παραπάνω σφάλμα.
- Ανατρέξτε στην [τεκμηρίωση Aspose](https://docs.aspose.com/slides/el/php-java/getting-started/) για βήματα αντιμετώπισης προβλημάτων και γνωστά ζητήματα αδειοδότησης.
- Επικοινωνήστε με την [υποστήριξη Aspose](https://forum.aspose.com/) για περαιτέρω βοήθεια.

## **Προϊόντα και Εκδόσεις που Επηρεάζονται**

Παρακαλούμε σημειώστε ότι αυτό το σφάλμα επηρεάζει μόνο τις εκδόσεις 23.2 έως 23.7 του Aspose.Slides for PHP via Java. **Άλλα προϊόντα Aspose και άλλες εκδόσεις του Aspose.Slides δεν επηρεάζονται**.
