## **Επισκόπηση**

Το Aspose.Slides υποστηρίζει αρχεία παρουσίασης από το Microsoft PowerPoint 97 έως το Office 365, συμπεριλαμβανομένου του Microsoft PowerPoint για Mac. Αυτό το άρθρο απαριθμεί τις εκδόσεις του PowerPoint που υποστηρίζονται από τη βιβλιοθήκη και παρέχει έναν πίνακα με τις μορφές αρχείων που μπορούν να φορτωθούν, να αποθηκευτούν ή και τα δύο.

Το άρθρο επίσης απαντά σε συχνές ερωτήσεις σχετικά με τη συμμόρφωση PDF, την ενσωμάτωση γραμματοσειρών, τα αρχεία με προστασία κωδικού, τις προσαρμοσμένες γραμματοσειρές, την εναλλακτική γραμματοσειρά (fallback) και τις επιλογές εξαγωγής XPS.

## **Υποστηριζόμενες Εκδόσεις Microsoft PowerPoint**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint για Mac
- Office 365

## **Συχνές Ερωτήσεις**

**Μπορώ να αποθηκεύσω παρουσιάσεις σε PDF που πληρούν τα πρότυπα αρχειοθέτησης και προσβασιμότητας (PDF/A και PDF/UA);**

Ναι. Το Aspose.Slides υποστηρίζει εξαγωγή σε PDF με επίπεδα συμμόρφωσης όπως PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b, καθώς και PDF/UA μέσω της ρύθμισης [συμμόρφωση](https://reference.aspose.com/slides/el/php-java/aspose.slides/pdfoptions/setcompliance/) στο [επιλογές εξαγωγής PDF](https://reference.aspose.com/slides/el/php-java/aspose.slides/pdfoptions/).

**Υποστηρίζει η βιβλιοθήκη ενσωμάτωση γραμματοσειρών κατά την εξαγωγή σε PDF, με λεπτομερή έλεγχο του τι ενσωματώνεται;**

Ναι. Μπορείτε να ελέγχετε αν οι γραμματοσειρές ενσωματώνονται πλήρως ή με υποσύνολο (μόνο τα χρησιμοποιημένα γλυφά), να ορίσετε τη μεταχείριση των κοινών γραμματοσειρών συστήματος και να διαμορφώσετε τη συμπεριφορά για κείμενο ASCII μέσω των [επιλογών εξαγωγής PDF](https://reference.aspose.com/slides/el/php-java/aspose.slides/pdfoptions/).

**Μπορώ να εντοπίσω εάν ένα αρχείο είναι προστατευμένο με κωδικό πριν το φορτώσω;**

Ναι. Χρησιμοποιώντας το [factory-based inspection API](https://reference.aspose.com/slides/el/php-java/aspose.slides/presentationfactory/), μπορείτε να ερωτήσετε ένα αρχείο παρουσίασης για να καθορίσετε αν είναι προστατευμένο με κωδικό χωρίς να το ανοίξετε πλήρως.

**Υπάρχουν μηχανισμοί εναλλακτικής γραμματοσειράς (fallback) και υποστήριξη προσαρμοσμένων γραμματοσειρών;**

Ναι. Η βιβλιοθήκη υποστηρίζει [φόρτωση](/slides/el/php-java/custom-font/) και [ενσωμάτωση](/slides/el/php-java/embedded-font/) προσαρμοσμένων γραμματοσειρών και παρέχει [κανόνες εναλλακτικής γραμματοσειράς](/slides/el/php-java/fallback-font/) για την αποφυγή ελλιπών γλυφών κατά την απόδοση και τη μετατροπή.

**Μπορώ να εξάγω διαφάνειες σε XPS και υπάρχουν επιλογές ρύθμισης του αποτελέσματος XPS;**

Ναι. Η [εξαγωγή σε XPS](/slides/el/php-java/convert-powerpoint-to-xps/) υποστηρίζεται και μπορείτε να προσαρμόσετε τις σχετικές [επιλογές αποθήκευσης](https://reference.aspose.com/slides/el/php-java/aspose.slides/xpsoptions/) για να ελέγξετε την ποιότητα και το περιεχόμενο του εγγράφου XPS.
