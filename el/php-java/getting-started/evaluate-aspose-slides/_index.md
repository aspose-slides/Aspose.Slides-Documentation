## **Αξιολόγηση Aspose.Slides**

Μπορείτε εύκολα να κατεβάσετε το Aspose.Slides για αξιολόγηση. Το πακέτο αξιολόγησης είναι το ίδιο με το αγορασμένο πακέτο. Η έκδοση αξιολόγησης γίνεται απλώς αδειοδοτημένη αφού προσθέσετε μερικές γραμμές κώδικα για την εφαρμογή της άδειας. 

Η έκδοση αξιολόγησης του Aspose.Slides (χωρίς καθορισμένη άδεια) παρέχει πλήρη λειτουργικότητα του προϊόντος, αλλά προσθέτει ένα υδατογράφημα αξιολόγησης στην κορυφή του εγγράφου κατά το άνοιγμα και την αποθήκευση. Επίσης περιορίζεστε σε μία διαφάνεια όταν εξάγετε κείμενα από τις διαφάνειες παρουσίασης.

## **FAQ**

**Μπορώ να δοκιμάσω πολλαπλές παρουσιάσεις παράλληλα σε διαφορετικά νήματα σε λειτουργία αξιολόγησης;**

Ναι. Μπορείτε να επεξεργαστείτε διαφορετικά έγγραφα παράλληλα· δεν πρέπει να μοιράζεστε το ίδιο αντικείμενο παρουσίασης [μεταξύ νημάτων](/slides/el/php-java/multithreading/). Η λειτουργία αξιολόγησης δεν το επηρεάζει.

**Χρειάζεται να εγκαταστήσω το Microsoft PowerPoint για την αξιολόγηση της βιβλιοθήκης σε διακομιστή ή σε CI;**

Όχι. Το Aspose.Slides είναι μια ανεξάρτητη μηχανή και δεν απαιτείται η εγκατάσταση του PowerPoint, είτε για αξιολόγηση είτε για παραγωγή.

**Μπορώ να δοκιμάσω πλήρως τη μετατροπή του PPT/PPTX σε PDF και εικόνες σε λειτουργία αξιολόγησης;**

Ναι. Οι [μετατροπείς](/slides/el/php-java/convert-presentation/) λειτουργούν· το αποτέλεσμα θα περιλαμβάνει υδατογράφημα.

**Μπορώ να χρησιμοποιήσω μια προσωρινή άδεια για δοκιμές φορτίου χωρίς υδατογράφημα;**

Ναι. Μια 30‑ήμερη προσωρινή άδεια αφαιρεί τους περιορισμούς της λειτουργίας αξιολόγησης και επιτρέπει δοκιμές χωρίς υδατογράφημα.
