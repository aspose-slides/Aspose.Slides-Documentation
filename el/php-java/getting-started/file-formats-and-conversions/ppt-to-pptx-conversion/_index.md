Η μετατροπή PPT σε PPTX υποστηρίζεται από το Aspose.Slides for PHP via Java. Τα περισσότερα χαρακτηριστικά παρουσίασης - master διαφάνειες, η δομή κ.ά. - διατηρούνται όταν μετατρέπονται από τη μία μορφή στην άλλη, αλλά υπάρχουν [μερικοί περιορισμοί](/slides/el/php-java/ppt-to-pptx-conversion/).

{{% /alert %}} 
## **Χαρακτηριστικά που Υποστηρίζονται στη Μετατροπή**
Το Aspose.Slides for PHP via Java παρέχει μερική υποστήριξη για τη μετατροπή της μορφής αρχείου PPT σε PPTX. Η υποστήριξη για τη μετατροπή μόλις έχει εισαχθεί στο Aspose.Slides for PHP via Java, έτσι έχει κάποιους περιορισμούς και λειτουργεί καλύτερα για απλές παρουσιάσεις. Το κύριο πλεονέκτημα που προσφέρει το Aspose.Slides for PHP via Java κατά τη μετατροπή PPT σε PPTX είναι πόσο εύκολο είναι το API στη χρήση. Για να δείτε παραδείγματα του κώδικα, διαβάστε για [Μετατροπή PPT σε PPTX](). Παρακάτω, οι λίστες εξηγούν ποια χαρακτηριστικά υποστηρίζονται και ποια όχι στη μετατροπή PPT σε PPTX.

## **Χαρακτηριστικά που Υποστηρίζονται**
Τα παρακάτω χαρακτηριστικά υποστηρίζονται για τη μετατροπή:

- Μετατροπή της δομής των master, διατάξεων και διαφανειών.  
- Μετατροπή διαγραμμάτων.  
- Ομαδοποιημένα σχήματα.  
- Μετατροπή αυτόματων σχημάτων συμπεριλαμβανομένων των ορθογωνίων και ελλείψεων.  
- Σχήματα με προσαρμοσμένη γεωμετρία.  
- Στυλ γεμίσματος με υφές και εικόνες για αυτόματα σχήματα.  
- Μετατροπή placeholders.  
- Μετατροπή γραμμών και πολυγραμμών.  
- Μορφές γραμμής και γεμίσματος.  
- Στυλ γεμίσματος με διαβάθμιση.  
- OLE πλαίσια, πίνακες, βίντεο και ηχητικά πλαίσια κ.λπ.  
- Ιδιότητες κίνησης και παρουσίασης.  
- Μετατροπή κειμένου σε πλαίσια κειμένου και κειμενικούς χώρους.
