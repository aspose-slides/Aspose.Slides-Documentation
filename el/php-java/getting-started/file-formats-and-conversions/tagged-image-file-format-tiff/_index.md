## **Σχετικά με το TIFF**

Ο [Μορφότυπος Επισήμανσης Εικόνας](https://en.wikipedia.org/wiki/TIFF) που είναι γνωστός για την αποθήκευση πολλαπλών εικόνων σε ένα έγγραφο δημιουργήθηκε αρχικά από την Aldus. Αυτό το φορμάτ υποστηρίζεται ευρέως από εφαρμογές σάρωσης, φαξ και άλλες εφαρμογές επεξεργασίας εικόνας.

## **TIFF στο Aspose.Slides for PHP μέσω Java**

Οποιοδήποτε έγγραφο μπορεί να φορτωθεί στο Aspose.Slides for Java μπορεί επίσης να μετατραπεί απευθείας σε έγγραφο TIFF από το Aspose.Slides for PHP μέσω Java, εξαλείφοντας την ανάγκη οποιουδήποτε τρίτου στοιχείου. Επιπλέον, μπορείτε προαιρετικά να ορίσετε το μέγεθος των εικόνων στο παραγόμενο έγγραφο TIFF. Μπορείτε να βρείτε πληροφορίες για την εξαγωγή των εγγράφων παρουσίασης σε έγγραφα TIFF μέσω Aspose.Slides for PHP μέσω Java στην [αυτή η ενότητα](/slides/el/php-java/convert-powerpoint-to-tiff/).
