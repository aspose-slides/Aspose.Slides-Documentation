{{% alert color="primary" %}} 
Όλα τα στοιχεία Aspose Java απαιτούν σύνολο δικαιωμάτων Full Trust. Ο λόγος είναι ότι τα στοιχεία Aspose Java πρέπει να έχουν πρόσβαση σε ρυθμίσεις μητρώου, συστημικά αρχεία εκτός του εικονικού καταλόγου για ορισμένες λειτουργίες όπως η ανάλυση γραμματοσειρών κ.λπ. Επιπλέον, τα στοιχεία Aspose Java βασίζονται σε κεντρικές κλάσεις του συστήματος Java που επίσης απαιτούν σύνολο δικαιωμάτων Full Trust σε πολλές περιπτώσεις. 
{{% /alert %}} 
Οι Πάροχοι Υπηρεσιών Διαδικτύου που φιλοξενούν πολλαπλές εφαρμογές από διαφορετικές εταιρείες κυρίως εφαρμόζουν επίπεδο ασφαλείας Medium Trust: 

- OleDbPermission δεν είναι διαθέσιμο. Αυτό σημαίνει ότι δεν μπορείτε να χρησιμοποιήσετε τον διαχειριζόμενο παροχέα δεδομένων ADO.NET OLE DB για πρόσβαση σε βάσεις δεδομένων. 
- EventLogPermission δεν είναι διαθέσιμο. Αυτό σημαίνει ότι δεν μπορείτε να αποκτήσετε πρόσβαση στο αρχείο καταγραφής συμβάντων των Windows. 
- ReflectionPermission δεν είναι διαθέσιμο. Αυτό σημαίνει ότι δεν μπορείτε να χρησιμοποιήσετε αντανακλάσεις. 
- RegistryPermission δεν είναι διαθέσιμο. Αυτό σημαίνει ότι δεν μπορείτε να αποκτήσετε πρόσβαση στο μητρώο. 
- WebPermission είναι περιορισμένο. Αυτό σημαίνει ότι η εφαρμογή σας μπορεί να επικοινωνεί μόνο με μια διεύθυνση ή ένα εύρος διευθύνσεων που ορίζετε στο στοιχείο <trust>. 
- FileIOPermission είναι περιορισμένο. Αυτό σημαίνει ότι μπορείτε να έχετε πρόσβαση μόνο σε αρχεία στην ιεραρχία του εικονικού καταλόγου της εφαρμογής σας. 

{{% alert color="primary" %}} 
Λόγω των παραπάνω λόγων, τα στοιχεία Aspose Java δεν μπορούν να χρησιμοποιηθούν σε διακομιστές που παρέχουν σύνολο δικαιωμάτων διαφορετικό από Full Trust. 
{{% /alert %}}
