## **Εισαγωγή**

Μερικές φορές, για τα βέλτιστα αποτελέσματα αξιολόγησης, μπορεί να απαιτείται μια πρακτική προσέγγιση. Για αυτόν τον λόγο, η Aspose.Slides προσφέρει διαφορετικά σχέδια αγοράς και επίσης παρέχει Δωρεάν Δοκιμή και 30‑ήμερη Προσωρινή Άδεια για αξιολόγηση.

## **Αξιολογήστε Aspose.Slides**
Μπορείτε εύκολα να κατεβάσετε το Aspose.Slides για αξιολόγηση. Το πακέτο αξιολόγησης είναι το ίδιο με το αγορασμένο πακέτο. Η έκδοση αξιολόγησης απλώς ενεργοποιείται με άδεια αφού προσθέσετε μερικές γραμμές κώδικα για την εφαρμογή της άδειας.

## **Περιορισμός Έκδοσης Αξιολόγησης**
Η έκδοση αξιολόγησης του Aspose.Slides (χωρίς καθορισμένη άδεια) παρέχει τη πλήρη λειτουργικότητα του προϊόντος, αλλά εισάγει ένα υδατογράφημα αξιολόγησης στην κορυφή του εγγράφου κατά το άνοιγμα και την αποθήκευση. Επίσης, περιορίζεστε σε μία διαφάνεια όταν εξάγετε κείμενο από τις διαφάνειες παρουσίασης.

{{% alert color="primary" %}} 
Εάν θέλετε να δοκιμάσετε το Aspose.Slides χωρίς τους περιορισμούς της έκδοσης αξιολόγησης, μπορείτε να ζητήσετε μια **30‑ήμερη Προσωρινή Άδεια**. Ανατρέξτε στο [Πώς να αποκτήσετε Προσωρινή Άδεια;](https://purchase.aspose.com/temporary-license) για περισσότερες πληροφορίες.
{{% /alert %}} 

## **Σχετικά με την Άδεια**
Μπορείτε εύκολα να κατεβάσετε μια έκδοση αξιολόγησης του Aspose.Slides για PHP μέσω Java από τη [σελίδα λήψης](https://packagist.org/packages/aspose/slides). Η έκδοση αξιολόγησης παρέχει απολύτως **τις ίδιες δυνατότητες** με την έκδοση με άδεια του Aspose.Slides. Επιπλέον, η έκδοση αξιολόγησης ενεργοποιείται με άδεια αφού αγοράσετε μια άδεια και προσθέσετε μερικές γραμμές κώδικα για την εφαρμογή της άδειας.

Η άδεια είναι ένα αρχείο XML απλού κειμένου που περιέχει λεπτομέρειες όπως το όνομα προϊόντος, ο αριθμός των προγραμματιστών στους οποίους αδειοδοτείται, η ημερομηνία λήξης της συνδρομής κ.λπ. Το αρχείο είναι ψηφιακά υπογεγραμμένο, επομένως μην το τροποποιήσετε. Ακόμη και η ακούσια προσθήκη μιας επιπλέον αλλαγής γραμμής στο περιεχόμενο του αρχείου θα την ακυρώσει.

Για να αποφύγετε τους περιορισμούς που σχετίζονται με την έκδοση αξιολόγησης, πρέπει να ορίσετε μια άδεια πριν χρησιμοποιήσετε το **Aspose.Slides**. Απαιτείται να ορίσετε την άδεια μία φορά ανά εφαρμογή ή διαδικασία.

{{% alert color="primary" %}}
Ίσως θελήσετε να δείτε το [Μετρημένο Μοντέλο Αδειοδότησης](https://docs.aspose.com/slides/el/php-java/metered-licensing/).
{{% /alert %}}

### **Ορίστε Άδεια στο Aspose.Slides για PHP μέσω Java**

Οι άδειες μπορούν να εφαρμοστούν από τις εξής τοποθεσίες:

* Σαφής διαδρομή
* Ροή
* Ως Μετρημένη Άδεια – ένας νέος μηχανισμός αδειοδότησης

{{% alert color="primary" %}}
Χρησιμοποιήστε τη μέθοδο **setLicense** για την αδειοδότηση ενός στοιχείου.

Παρόλο που πολλές κλήσεις στη **setLicense** δεν είναι επιβλαβείς, είναι σπατάλη πόρων (επεξεργαστή).
{{% /alert %}}

{{% alert color="warning" %}}
Οι νέες άδειες μπορούν να ενεργοποιήσουν το Aspose.Slides μόνο με την έκδοση 21.4 ή νεότερη. Παλαιότερες εκδόσεις χρησιμοποιούν διαφορετικό σύστημα αδειοδότησης και δεν θα αναγνωρίσουν αυτές τις άδειες.
{{% /alert %}}

#### **Εφαρμογή Άδειας Χρησιμοποιώντας Αρχείο**

Αυτό το απόσπασμα κώδικα χρησιμοποιείται για τον ορισμό ενός αρχείου άδειας:

**PHP**

```php
<?php
require_once("http://localhost:8080/JavaBridge/java/Java.inc");
require_once("lib/aspose.slides.php");

use aspose\slides\License;

$license = new License();
$license->setLicense("Aspose.Slides.lic");
?>
```

Κατά την κλήση της μεθόδου setLicense, το όνομα της άδειας πρέπει να είναι ίδιο με αυτό του αρχείου άδειας σας. Για παράδειγμα, μπορείτε να αλλάξετε το όνομα του αρχείου άδειας σε "Aspose.Slides.lic.xml". Στη συνέχεια, στον κώδικά σας, πρέπει να περάσετε το νέο όνομα (Aspose.Slides.lic.xml) στη μέθοδο setLicense.

#### **Εφαρμογή Άδειας από Ροή**

Αυτό το απόσπασμα κώδικα χρησιμοποιείται για την εφαρμογή άδειας από ροή:

```php
<?php
require_once("http://localhost:8080/JavaBridge/java/Java.inc");
```
