## **Επισκόπηση**

Το Aspose.Slides σάς επιτρέπει να καθορίσετε προεπιλεγμένες γραμματοσειρές που χρησιμοποιούνται όταν αποδίδεται μια παρουσίαση. Αυτό είναι χρήσιμο κατά τη δημιουργία μικρογραφιών διαφανειών ή την εξαγωγή μιας παρουσίασης σε μορφές όπως PDF και XPS. Οι προεπιλεγμένες γραμματοσειρές διαμορφώνονται μέσω του `LoadOptions` πριν φορτωθεί η παρουσίαση.

Η μέθοδος `setDefaultRegularFont` ορίζει την προεπιλεγμένη γραμματοσειρά για κανονικό κείμενο, ενώ η `setDefaultAsianFont` ορίζει την προεπιλεγμένη γραμματοσειρά για ασιατικό κείμενο. Αφού οριστούν αυτές οι επιλογές, η παρουσίαση μπορεί να φορτωθεί και να αποδοθεί χρησιμοποιώντας τις καθορισμένες γραμματοσειρές.

## **Χρήση προεπιλεγμένων γραμματοσειρών για την απόδοση μιας παρουσίασης**
Το Aspose.Slides σάς επιτρέπει να ορίσετε την προεπιλεγμένη γραμματοσειρά για την απόδοση της παρουσίασης σε PDF, XPS ή μικρογραφίες. Αυτό το άρθρο δείχνει πώς να ορίσετε το DefaultRegularFont και το DefaultAsianFont ως προεπιλεγμένες γραμματοσειρές. Ακολουθήστε τα παρακάτω βήματα για τη φόρτωση γραμματοσειρών από εξωτερικούς φακέλους χρησιμοποιώντας το Aspose.Slides for PHP μέσω Java API:

1. Δημιουργήστε μια παρουσία της [LoadOptions](https://reference.aspose.com/slides/el/php-java/aspose.slides/LoadOptions).
1. [Ορίστε το DefaultRegularFont](https://reference.aspose.com/slides/el/php-java/aspose.slides/LoadOptions#setDefaultRegularFont-java.lang.String-) στην επιθυμητή γραμματοσειρά σας. Στο παρακάτω παράδειγμα, χρησιμοποίησα Wingdings.
1. [Ορίστε το DefaultAsianFont](https://reference.aspose.com/slides/el/php-java/aspose.slides/LoadOptions#setDefaultAsianFont-java.lang.String-) στην επιθυμητή γραμματοσειρά σας. Χρησιμοποίησα Wingdings στο παρακάτω δείγμα.
1. Φορτώστε την παρουσίαση χρησιμοποιώντας το Presentation και ορίζοντας τις επιλογές φόρτωσης.
1. Τώρα, δημιουργήστε τη μικρογραφία της διαφάνειας, το PDF και το XPS για να επαληθεύσετε τα αποτελέσματα.

Η υλοποίηση του παραπάνω δίνεται παρακάτω.

```php
  # Χρησιμοποιήστε επιλογές φόρτωσης για να ορίσετε τις προεπιλεγμένες κανονικές και ασιατικές γραμματοσειρές
  $loadOptions = new LoadOptions(LoadFormat::Auto);
  $loadOptions->setDefaultRegularFont("Wingdings");
  $loadOptions->setDefaultAsianFont("Wingdings");
  # Φορτώστε την παρουσίαση
  $pres = new Presentation("DefaultFonts.pptx", $loadOptions);
  try {
    # Δημιουργήστε μικρογραφία διαφάνειας
    $slideImage = $pres->getSlides()->get_Item(0)->getImage(1, 1);
    try {
      # αποθηκεύστε την εικόνα στο δίσκο.
      $slideImage->save("output.png", ImageFormat::Png);
    } finally {
      if (!java_is_null($slideImage)) {
        $slideImage->dispose();
      }
    }
    # Δημιουργήστε PDF
    $pres->save("output_out.pdf", SaveFormat::Pdf);
    # Δημιουργήστε XPS
    $pres->save("output_out.xps", SaveFormat::Xps);
  } catch (JavaException $e) {
  } finally {
    if (!java_is_null($pres)) {
      $pres->dispose();
    }
  }
```

## **Συχνές ερωτήσεις**

**Τι ακριβώς επηρεάζουν τα DefaultRegularFont και DefaultAsianFont—μόνο την εξαγωγή ή επίσης τις μικρογραφίες, PDF, XPS, HTML και SVG;**

Συμμετέχουν στη διαδικασία απόδοσης για όλα τα υποστηριζόμενα αποτελέσματα. Αυτό περιλαμβάνει μικρογραφίες διαφανειών, [PDF](/slides/el/php-java/convert-powerpoint-to-pdf/), [XPS](/slides/el/php-java/convert-powerpoint-to-xps/), [raster images](/slides/el/php-java/convert-powerpoint-to-png/), [HTML](/slides/el/php-java/convert-powerpoint-to-html/), και [SVG](/slides/el/php-java/render-a-slide-as-an-svg-image/), επειδή το Aspose.Slides χρησιμοποιεί την ίδια λογική διάταξης και επίλυσης γλυφών σε αυτούς τους προορισμούς.

**Εφαρμόζονται οι προεπιλεγμένες γραμματοσειρές όταν απλώς διαβάζετε και αποθηκεύετε ένα PPTX χωρίς καμία απόδοση;**

Όχι. Οι προεπιλεγμένες γραμματοσειρές έχουν σημασία όταν πρέπει να μετρηθεί και να σχεδιαστεί το κείμενο. Μια απλή εντολή άνοιγμα‑αποθήκευση μιας παρουσίασης δεν αλλάζει τις αποθηκευμένες εκτάσεις γραμματοσειράς ή τη δομή του αρχείου. Οι προεπιλεγμένες γραμματοσειρές εμπλέκονται σε λειτουργίες που αποδίδουν ή επαναδιαμορφώνουν το κείμενο.

**Αν προσθέσω τους δικούς μου φακέλους γραμματοσειρών ή παρέχω γραμματοσειρές από τη μνήμη, θα ληφθούν υπόψη κατά την επιλογή των προεπιλεγμένων γραμματοσειρών;**

Ναι. Οι [Custom font sources](/slides/el/php-java/custom-font/) επεκτείνουν τον κατάλογο των διαθέσιμων οικογενειών και γλυφών που μπορεί να χρησιμοποιήσει η μηχανή. Οι προεπιλεγμένες γραμματοσειρές και τυχόν [fallback rules](/slides/el/php-java/fallback-font/) θα αναζητηθούν πρώτα σε αυτές τις πηγές, προσφέροντας μεγαλύτερη κάλυψη σε διακομιστές και κοντέινερ.

**Θα επηρεάσουν οι προεπιλεγμένες γραμματοσειρές τις μετρικές του κειμένου (kerning, advances) και επομένως τις αλλαγές γραμμής και τη συσπείρωση;**

Ναι. Η αλλαγή της γραμματοσειράς αλλάζει τις μετρικές των γλυφών και μπορεί να τροποποιήσει τις αλλαγές γραμμής, τη συσπείρωση και την τμηματοποίηση σελίδων κατά την απόδοση. Για σταθερότητα διάταξης, [ενσωματώστε τις αρχικές γραμματοσειρές](/slides/el/php-java/embedded-font/) ή επιλέξτε προεπιλεγμένες και εναλλακτικές οικογένειες με συμβατές μετρικές.

**Υπάρχει κάποιο όφελος στο να ορίζονται προεπιλεγμένες γραμματοσειρές αν όλες οι γραμματοσειρές στην παρουσίαση είναι ενσωματωμένες;**

Συχνά δεν είναι απαραίτητο, επειδή οι [embedded fonts](/slides/el/php-java/embedded-font/) εξασφαλίζουν ήδη συνεπή εμφάνιση. Οι προεπιλεγμένες γραμματοσειρές εξακολουθούν να λειτουργούν ως εφεδρική λύση για χαρακτήρες που δεν καλύπτονται από το ενσωματωμένο υποσύνολο ή όταν ένα αρχείο συνδυάζει ενσωματωμένο και μη ενσωματωμένο κείμενο.
