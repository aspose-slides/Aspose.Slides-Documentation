## **Επισκόπηση**

Το Aspose.Slides σάς επιτρέπει να αντικαταστήσετε μια γραμματοσειρά με άλλη σε ολόκληρη την παρουσίαση. Όταν μια γραμματοσειρά αντικαθίσταται, όλες οι εμφανίσεις της αρχικής γραμματοσειράς αλλάζουν στη νέα γραμματοσειρά.

Για να εκτελέσετε την αντικατάσταση γραμματοσειράς, φορτώστε την παρουσίαση, ορίστε τη γραμματοσειρά προέλευσης και τη γραμματοσειρά αντικατάστασης, καλέστε τη μέθοδο αντικατάστασης γραμματοσειράς και αποθηκεύστε την τροποποιημένη παρουσίαση ως αρχείο PPTX. Αυτή η προσέγγιση είναι χρήσιμη όταν θέλετε σκόπιμα να μεταβείτε από μια οικογένεια γραμματοσειρών σε άλλη σε όλη την παρουσίαση.

## **Αντικατάσταση Γραμματοσειρών**

Αν αλλάξετε γνώμη σχετικά με τη χρήση μιας γραμματοσειράς, μπορείτε να την αντικαταστήσετε με άλλη γραμματοσειρά. Όλες οι εμφανίσεις της παλιάς γραμματοσειράς θα αντικατασταθούν από τη νέα.

Το Aspose.Slides σάς επιτρέπει να αντικαταστήσετε μια γραμματοσειρά με αυτόν τον τρόπο:

1. Φορτώστε την σχετική παρουσίαση. 
2. Φορτώστε τη γραμματοσειρά που θα αντικατασταθεί. 
3. Φορτώστε τη νέα γραμματοσειρά. 
4. Αντικαταστήστε τη γραμματοσειρά. 
5. Αποθηκεύστε την τροποποιημένη παρουσίαση ως αρχείο PPTX.

Αυτός ο κώδικας PHP επιδεικνύει την αντικατάσταση γραμματοσειράς:

```php
  # Φορτώνει μια παρουσίαση
  $pres = new Presentation("Fonts.pptx");
  try {
    # Φορτώνει τη γραμματοσειρά προέλευσης που θα αντικατασταθεί
    $sourceFont = new FontData("Arial");
    # Φορτώνει τη νέα γραμματοσειρά
    $destFont = new FontData("Times New Roman");
    # Αντικαθιστά τις γραμματοσειρές
    $pres->getFontsManager()->replaceFont($sourceFont, $destFont);
    # Αποθηκεύει την παρουσίαση
    $pres->save("UpdatedFont_out.pptx", SaveFormat::Pptx);
  } finally {
    if (!java_is_null($pres)) {
      $pres->dispose();
    }
  }
```

{{% alert title="Note" color="warning" %}} 
Για να ορίσετε κανόνες που καθορίζουν τι συμβαίνει σε ορισμένες συνθήκες (π.χ. εάν η γραμματοσειρά δεν είναι προσβάσιμη), δείτε [**Αντικατάσταση Γραμματοσειράς**](/slides/el/php-java/font-substitution/).

## **Συχνές ερωτήσεις**

**Ποια είναι η διαφορά μεταξύ "αντικατάστασης γραμματοσειράς", "υποκατάστασης γραμματοσειράς" και "εφεδρικών γραμματοσειρών";**

Η αντικατάσταση είναι μια σκόπιμη μετάβαση από μια οικογένεια γραμματοσειρών σε άλλη σε όλο το έγγραφο. Η [Υποκατάσταση](/slides/el/php-java/font-substitution/) είναι ένας κανόνας όπως «αν η γραμματοσειρά δεν είναι διαθέσιμη, χρησιμοποιήστε το X». Η [Εφεδρική γραμματοσειρά](/slides/el/php-java/fallback-font/) εφαρμόζεται σε μεμονωμένα ελλείποντα γλύφματα όταν η βασική γραμματοσειρά είναι εγκατεστημένη αλλά δεν περιέχει τους απαιτούμενους χαρακτήρες.

**Ισχύει η αντικατάσταση για master slides, layouts, notes και comments;**

Ναι. Η αντικατάσταση επηρεάζει όλα τα αντικείμενα της παρουσίασης που χρησιμοποιούν την αρχική γραμματοσειρά, συμπεριλαμβανομένων των master slides και των notes· τα comments είναι επίσης μέρος του εγγράφου και λαμβάνονται υπόψη από τη μηχανή γραμματοσειρών.

**Θα αλλάξει η γραμματοσειρά μέσα σε ενσωματωμένα αντικείμενα OLE (π.χ. Excel);**

Όχι. Το [περιεχόμενο OLE](/slides/el/php-java/manage-ole/) ελέγχεται από την δική του εφαρμογή. Η αντικατάσταση στην παρουσίαση δεν επαναδιαμορφώνει τα εσωτερικά δεδομένα OLE· μπορεί να εμφανίζεται ως εικόνα ή ως εξωτερικά επεξεργάσιμο περιεχόμενο.

**Μπορώ να αντικαταστήσω μια γραμματοσειρά μόνο σε μέρος της παρουσίασης (ανά διαφάνεια ή περιοχή);**

Στόχευση αντικατάστασης είναι δυνατόν εφόσον αλλάξετε τη γραμματοσειρά σε επίπεδο των απαιτούμενων αντικειμένων/εμβαδών αντί να εφαρμόσετε καθολική αντικατάσταση σε όλο το έγγραφο. Η συνολική λογική επιλογής γραμματοσειράς κατά την απόδοση παραμένει η ίδια.

**Πώς μπορώ να προσδιορίσω εκ των προτέρων ποιες γραμματοσειρές χρησιμοποιεί η παρουσίαση;**

Χρησιμοποιήστε το [διαχειριστή γραμματοσειρών](https://reference.aspose.com/slides/el/php-java/aspose.slides/fontsmanager/): παρέχει λίστα των [οικογενειών σε χρήση](https://reference.aspose.com/slides/el/php-java/aspose.slides/fontsmanager/getfonts/) και πληροφορίες για τις [υποκαταστάσεις/"άγνωστες" γραμματοσειρές](https://reference.aspose.com/slides/el/php-java/aspose.slides/fontsmanager/getsubstitutions/), που βοηθά στον προγραμματισμό της αντικατάστασης.

**Λειτουργεί η αντικατάσταση γραμματοσειράς κατά τη μετατροπή σε PDF/εικόνες;**

Ναι. Κατά την εξαγωγή, το Aspose.Slides εφαρμόζει την ίδια [ακολουθία επιλογής/υποκατάστασης γραμματοσειρών](/slides/el/php-java/font-selection-sequence/), έτσι ώστε μια αντικατάσταση που εκτελέστηκε εκ των προτέρων να τηρείται κατά τη μετατροπή.

**Χρειάζεται να εγκαταστήσω τη γραμματοσειρά-στόχο στο σύστημα, ή μπορώ να προσθέσω έναν φάκελο γραμματοσειρών;**

Η εγκατάσταση δεν απαιτείται: η βιβλιοθήκη επιτρέπει τη [φόρτωση εξωτερικών γραμματοσειρών](/slides/el/php-java/custom-font/) από φακέλους χρήστη για χρήση κατά το [απόδοση και εξαγωγή](/slides/el/php-java/convert-powerpoint/).

**Θα διορθώσει η αντικατάσταση το πρόβλημα "tofu" (τετράγωνα) αντί για χαρακτήρες;**

Μόνο εάν η γραμματοσειρά-στόχος περιέχει πραγματικά τα απαιτούμενα γλύφοι. Εάν όχι, [ρυθμίστε την εφεδρική γραμματοσειρά](/slides/el/php-java/fallback-font/) για να καλύψετε τους ελλείποντες χαρακτήρες.
