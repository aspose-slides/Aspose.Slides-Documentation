## **Εισαγωγή**

Οι γραμματοσειρές εφεδρείας χρησιμοποιούνται όταν η γραμματοσειρά που έχει οριστεί για το κείμενο είναι διαθέσιμη στο σύστημα, αλλά δεν περιέχει το απαιτούμενο σύμβολο. Σε αυτήν την περίπτωση, το Aspose.Slides μπορεί να χρησιμοποιήσει μία από τις καθορισμένες γραμματοσειρές εφεδρείας για την αντικατάσταση του ελλείποντος συμβόλου.

## **Γραμματοσειρά Εφεδρείας**
Η γραμματοσειρά εφεδρείας χρησιμοποιείται όταν η γραμματοσειρά που έχει οριστεί για το κείμενο είναι διαθέσιμη στο σύστημα, αλλά αυτή η γραμματοσειρά δεν περιέχει το απαραίτητο σύμβολο. Σε αυτήν την περίπτωση, είναι δυνατόν να χρησιμοποιηθεί μία από τις καθορισμένες γραμματοσειρές εφεδρείας για την αντικατάσταση του συμβόλου.

Το Aspose.Slides επιτρέπει τη δημιουργία γραμματοσειρών εφεδρείας, την προσθήκη τους στη συλλογή γραμματοσειρών εφεδρείας, τον ορισμό συλλογής γραμματοσειρών εφεδρείας για μια συγκεκριμένη παρουσίαση, την αφαίρεση γραμματοσειρών εφεδρείας από την παρουσίαση, τον καθορισμό των κανόνων για την εφαρμογή των γραμματοσειρών εφεδρείας και άλλα.

Η εφεδρεία εφαρμόζεται ανά χαρακτήρα ή ανά περιοχή Unicode όταν η κύρια γραμματοσειρά δεν περιέχει συγκεκριμένα σύμβολα· συμπληρώνει μόνο τους χαρακτήρες που λείπουν. Μπορούν να συνδυαστούν, αλλά η εμβέλειά τους και η λογική επιλογής διαφέρουν.
