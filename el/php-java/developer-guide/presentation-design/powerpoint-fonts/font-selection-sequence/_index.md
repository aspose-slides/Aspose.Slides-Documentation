## **Επισκόπηση**

Όταν μια παρουσίαση φορτώνεται, αποδίδεται ή μετατρέπεται σε άλλη μορφή, το Aspose.Slides ελέγχει αν οι γραμματοσειρές που χρησιμοποιεί η παρουσίαση είναι διαθέσιμες στο λειτουργικό σύστημα. Εάν λείπει κάποια απαιτούμενη γραμματοσειρά, το Aspose.Slides επιλέγει μια αντικαταστατική γραμματοσειρά που είναι όσο το δυνατόν πιο κοντά σε αυτή που θα χρησιμοποιούσε το PowerPoint.

Το Aspose.Slides πρώτα ψάχνει για την επιλεγμένη γραμματοσειρά στο λειτουργικό σύστημα. Αν βρεθεί η γραμματοσειρά, χρησιμοποιείται. Αν δεν βρεθεί, εφαρμόζεται μια κατάλληλη αντικατάσταση. Όταν οι κανόνες αντικατάστασης γραμματοσειρών ορίζονται μέσω `FontSubstRule`, λαμβάνονται επίσης υπόψη.

Μπορείτε επίσης να προσθέσετε γραμματοσειρές κατά τη διάρκεια εκτέλεσης της εφαρμογής, να χρησιμοποιήσετε ενσωματωμένες γραμματοσειρές από μια παρουσίαση ή να φορτώσετε εξωτερικές γραμματοσειρές για έγγραφα εξόδου όπως αρχεία PDF.

## **Επιλογή γραμματοσειράς**

Ορισμένοι κανόνες ισχύουν για τις γραμματοσειρές σε μια παρουσίαση όταν η παρουσίαση φορτώνεται, αποδίδεται ή μετατρέπεται σε άλλη μορφή. Για παράδειγμα, όταν προσπαθείτε να μετατρέψετε μια παρουσίαση (τις διαφάνειές της) σε εικόνες, οι γραμματοσειρές της παρουσίασης ελέγχονται για να επαληθευτεί ότι οι επιλεγμένες γραμματοσειρές είναι διαθέσιμες στο λειτουργικό σύστημα. Αν επιβεβαιωθεί ότι οι γραμματοσειρές λείπουν, αντικαθίστανται — δείτε [**Αντικατάσταση γραμματοσειράς**](https://docs.aspose.com/slides/el/php-java/font-replacement/) και [**Αντικατάσταση γραμματοσειρών**](https://docs.aspose.com/slides/el/php-java/font-substitution/).

Αυτή είναι η διαδικασία που ακολουθεί το Aspose.Slides όταν διαχειρίζεται γραμματοσειρές:

1. Το Aspose.Slides αναζητά γραμματοσειρές στο λειτουργικό σύστημα για να βρει τη γραμματοσειρά που ταιριάζει με την επιλεγμένη γραμματοσειρά της παρουσίασης. 
2. Αν η επιλεγμένη γραμματοσειρά βρεθεί, το Aspose.Slides τη χρησιμοποιεί. Διαφορετικά, το Aspose.Slides χρησιμοποιεί μια αντικαταστάτική γραμματοσειρά που είναι όσο το δυνατόν πιο κοντά σε αυτή που θα χρησιμοποιούσε το PowerPoint. 
3. Αν έχουν οριστεί κανόνες αντικατάστασης γραμματοσειρών μέσω [FontSubstRule](https://reference.aspose.com/slides/el/php-java/aspose.slides/fontsubstrule/), εφαρμόζονται.

Το Aspose.Slides σας επιτρέπει να προσθέσετε γραμματοσειρές στο runtime του Aspose και στη συνέχεια να τις χρησιμοποιήσετε. Δείτε [**Προσαρμοσμένες γραμματοσειρές**](https://docs.aspose.com/slides/el/php-java/custom-font/).

Όταν προστίθενται επιπλέον γραμματοσειρές σε μια παρουσίαση, αυτές ονομάζονται [**Ενσωματωμένες γραμματοσειρές**](https://docs.aspose.com/slides/el/php-java/embedded-font/).

Το Aspose.Slides σας επιτρέπει να προσθέσετε γραμματοσειρές που εφαρμόζονται *μόνο* σε έγγραφα εξόδου. Για παράδειγμα, εάν μια παρουσίαση που θέλετε να μετατρέψετε σε PDF περιέχει γραμματοσειρές που λείπουν από το σύστημά σας και ενσωματωμένες γραμματοσειρές, μπορείτε να προσθέσετε ή να φορτώσετε τις απαιτούμενες γραμματοσειρές ως **Εξωτερικές γραμματοσειρές**.

## **Συχνές ερωτήσεις**

**Πώς μπορώ να προσδιορίσω ποιες γραμματοσειρές χρησιμοποιούνται πραγματικά σε μια παρουσίαση πριν από τη μετατροπή;**

Το Aspose.Slides σας επιτρέπει να ελέγξετε τις χρησιμοποιούμενες γραμματοσειρές μέσω του [διαχειριστή γραμματοσειρών](https://reference.aspose.com/slides/el/php-java/aspose.slides/fontsmanager/), ώστε να αποφασίσετε αν θα [ενσωματώσετε](/slides/el/php-java/embedded-font/), [αντικαταστήσετε](/slides/el/php-java/font-replacement/), ή προσθέσετε [εξωτερικές πηγές](/slides/el/php-java/custom-font/). Αυτό σας βοηθά να αποτρέψετε ανεπιθύμητες αντικαταστάσεις κατά την απόδοση και την εξαγωγή.

**Μπορώ να προσθέσω επιπλέον φακέλους γραμματοσειρών χωρίς να τους εγκαταστήσω στο λειτουργικό σύστημα;**

Ναι. Μπορείτε να καταγράψετε [εξωτερικές πηγές γραμματοσειρών](/slides/el/php-java/custom-font/) όπως φακέλους ή ροές στη μνήμη για την απόδοση και την εξαγωγή. Αυτό αφαιρεί την εξάρτηση από τις γραμματοσειρές του συστήματος-κεντρικού υπολογιστή και διατηρεί την διάταξη προβλέψιμη.

**Πώς μπορώ να αποτρέψω μια σιωπηρή εναλλακτική σε μια ακατάλληλη γραμματοσειρά όταν λείπει ένα γλύφιο;**

Ορίστε ρητές [αντικαταστάσεις γραμματοσειρών](/slides/el/php-java/font-replacement/) και [κανόνες εναλλακτικών γραμματοσειρών](/slides/el/php-java/fallback-font/) εκ των προτέρων. Αναλύοντας τις χρησιμοποιούμενες γραμματοσειρές και ορίζοντας ελεγχόμενη προτεραιότητα για τα υποκατάστατα, εξασφαλίζετε συνεπή τυπογραφία και αποφεύγετε μη αναμενόμενα αποτελέσματα.
