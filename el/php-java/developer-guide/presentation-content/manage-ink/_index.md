## **Εισαγωγή**

Το PowerPoint παρέχει τη λειτουργία μελάνης ώστε να μπορείτε να σχεδιάζετε μη τυπικά σχήματα, τα οποία μπορούν να χρησιμοποιηθούν για να τονίσουν άλλα αντικείμενα, να δείξουν συνδέσεις και διαδικασίες, και να τραβήξουν την προσοχή σε συγκεκριμένα στοιχεία σε μια διαφάνεια.

Το Aspose.Slides παρέχει όλους τους τύπους Ink (π.χ. κλάση [Ink](https://reference.aspose.com/slides/el/php-java/aspose.slides/ink/)) που χρειάζεστε για να δημιουργήσετε και να διαχειριστείτε αντικείμενα μελάνης.

## **Διαφορές μεταξύ Κανονικών Αντικειμένων και Αντικειμένων Μελάνης**

Τα αντικείμενα σε μια διαφάνεια του PowerPoint συνήθως αντιπροσωπεύονται από αντικείμενα σχήματος. Ένα αντικείμενο σχήματος, στην πιο απλή του μορφή, είναι ένα δοχείο που ορίζει την περιοχή του ίδιου του αντικειμένου (το πλαίσιο του) μαζί με τις ιδιότητές του. Αυτές περιλαμβάνουν το μέγεθος της περιοχής του δοχείου, το σχήμα του δοχείου, το φόντο του δοχείου κ.λπ. Για περισσότερες πληροφορίες, δείτε [Shape Layout Format](https://docs.aspose.com/slides/el/php-java/shape-manipulations/#access-layout-formats-for-shape).

Ωστόσο, όταν το PowerPoint χειρίζεται ένα αντικείμενο μελάνης, αγνοεί όλες τις ιδιότητες του πλαισίου του αντικειμένου (δοχείου) εκτός από το μέγεθός του. Το μέγεθος της περιοχής του δοχείου καθορίζεται από τις τυπικές τιμές `width` και `height`:

## **Ιχνογραφήσεις Μελάνης**

Η ίχνος είναι ένα βασικό στοιχείο ή πρότυπο που χρησιμοποιείται για την καταγραφή της τροχιάς μιας γραφής καθώς ο χρήστης γράφει ψηφιακή μελάνη. Τα ίχνη είναι εγγραφές που περιγράφουν ακολουθίες συνδεδεμένων σημείων.

Η πιο απλή μορφή κωδικοποίησης καθορίζει τις συντεταγμένες X και Y κάθε σημείου δείγματος. Όταν όλα τα συνδεδεμένα σημεία αποδοθούν, δημιουργούν μια εικόνα όπως αυτή:

## **Ιδιότητες Πινέλου για Σχέδιο**

Μπορείτε να χρησιμοποιήσετε ένα πινέλο για να σχεδιάσετε γραμμές που συνδέουν τα σημεία των στοιχείων ίχνους. Το πινέλο έχει το δικό του χρώμα και μέγεθος, που αντιστοιχούν στις ιδιότητες `Brush.Color` και `Brush.Size`.

### **Ορισμός Χρώματος Πινέλου Μελάνης**

Αυτός ο κώδικας PHP δείχνει πώς να ορίσετε το χρώμα για ένα πινέλο:

```php
  $pres = new Presentation("pres.pptx");
  try {
    $ink = $pres->getSlides()->get_Item(0)->getShapes()->get_Item(0);
    $traces = $ink->getTraces();
    $brush = $traces[0]->getBrush();
    $brushColor = $brush->getColor();
    $brush->setColor(java("java.awt.Color")->RED);
  } finally {
    if (!java_is_null($pres)) {
      $pres->dispose();
    }
  }
```

### **Ορισμός Μεγέθους Πινέλου Μελάνης**

Αυτός ο κώδικας PHP δείχνει πώς να ορίσετε το μέγεθος για ένα πινέλο:

```php
  $pres = new Presentation("pres.pptx");
  try {
    $ink = $pres->getSlides()->get_Item(0)->getShapes()->get_Item(0);
    $traces = $ink->getTraces();
    $brush = $traces[0]->getBrush();
    $brushSize = $brush->getSize();
    $brush->setSize(new Java("java.awt.Dimension", 5, 10));
  } finally {
    if (!java_is_null($pres)) {
      $pres->dispose();
    }
  }
```

Γενικά, το πλάτος και το ύψος ενός πινέλου δεν ταιριάζουν, επομένως το PowerPoint δεν εμφανίζει το μέγεθος του πινέλου (η ενότητα δεδομένων εμφανίζεται σε γκρι). Ωστόσο, όταν το πλάτος και το ύψος του πινέλου ταιριάζουν, το PowerPoint εμφανίζει το μέγεθός του με τον ακόλουθο τρόπο:

![ink_powerpoint3](ink_powerpoint3.png)

Για σαφήνεια, ας αυξήσουμε το ύψος του αντικειμένου μελάνης και να εξετάσουμε τις σημαντικές διαστάσεις:

![ink_powerpoint4](ink_powerpoint4.png)

Το δοχείο (πλαίσιο) δεν λαμβάνει υπόψη το μέγεθος των πινέλων — πάντα υποθέτει ότι το πάχος της γραμμής είναι μηδέν (δείτε την τελευταία εικόνα).

Επομένως, για να προσδιορίσουμε την ορατή περιοχή του συνολικού αντικειμένου μελάνης, πρέπει να λάβουμε υπόψη το μέγεθος του πινέλου των αντικειμένων ίχνους. Εδώ, το αντικείμενο-στόχος (το αντικείμενο ίχνους χειρόγραφου κειμένου) έχει κλιμακωθεί στο μέγεθος του δοχείου (πλαισίου). Όταν το μέγεθος του δοχείου (πλαισίου) αλλάζει, το μέγεθος του πινέλου παραμένει σταθερό και αντίστροφα.

![ink_powerpoint5](ink_powerpoint5.png)

Το PowerPoint παρουσιάζει την ίδια συμπεριφορά όταν εργάζεται με κείμενα:

![ink_powerpoint6](ink_powerpoint6.png)
