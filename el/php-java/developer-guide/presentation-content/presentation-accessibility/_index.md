## **Επισκόπηση**

Η προσβασιμότητα των παρουσιάσεων εξασφαλίζει ότι οι άνθρωποι που χρησιμοποιούν βοηθητικές τεχνολογίες — όπως αναγνώστες οθόνης, εμφάνισεις μπράιγ ή πλοήγηση μόνο με το πληκτρολόγιο — μπορούν να κατανοήσουν και να περιηγηθούν στις διαφάνειές σας με την ίδια αποτελεσματικότητα με το κοινό που βλέπει και χρησιμοποιεί ποντίκι. Οι καλές πρακτικές εστιάζουν σε σαφή σειρά ανάγνωσης, ουσιαστικό εναλλακτικό κείμενο για πληροφοριακά οπτικά στοιχεία, επαρκή αντίθεση χρωμάτων, ευανάγνωστη τυπογραφία, περιγραφικό κείμενο συνδέσμων και στην αποφυγή μετάδοσης νοήματος μόνο μέσω χρώματος ή θέσης. Όταν η προσβασιμότητα σχεδιάζεται από την αρχή, το αποτέλεσμα είναι μια πιο καθαρή δομή, πιο συνεπές οπτικό περιεχόμενο και περιεχόμενο που φθάνει σε κάθε θεατή χωρίς παρακάμψεις.

## **Σημειώστε ως Διακοσμητικό**

Η σημαία «Σημειώστε ως διακοσμητικό» σημειώνει καθαρά διακοσμητικά οπτικά στοιχεία ώστε οι αναγνώστες οθόνης να τα παραλείπουν, μειώνοντας τον θόρυβο και διατηρώντας την εστίαση στο ουσιώδες περιεχόμενο. Εφαρμόστε τη σε φόντο, διακοσμητικά στοιχεία και διαχωριστικά — ποτέ σε γραφήματα, εικονίδια ή εικόνες που μεταφέρουν πληροφορίες. Το Aspose.Slides εκθέτει αυτή τη σημαία για ανίχνευση και επικύρωση, επιτρέποντας αυτοματοποιημένους ελέγχους προσβασιμότητας και καθαρισμό.

![Σημειωμένο ως Διακοσμητικό](mark_as_decorative.png)

Το παρακάτω δείγμα κώδικα δείχνει πώς να καθορίσετε εάν ένα σχήμα είναι σημειωμένο ως διακοσμητικό.

```php
$presentation = new Presentation("sample.pptx");
try {
    $shape = $presentation->getSlides()->get_Item(0)->getShapes()->get_Item(0);
    echo "Is shape decorative: " . ($shape->isDecorative() ? "true" : "false") . "\n";
} finally {
    $presentation->dispose();
}
```
