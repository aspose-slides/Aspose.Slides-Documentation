## **Επισκόπηση**

Αυτό το άρθρο εξηγεί πώς να εργαστείτε με πίνακες δεδομένων διαγραμμάτων στο Aspose.Slides. Δείχνει πώς να εμφανίσετε έναν πίνακα δεδομένων για ένα διάγραμμα και να προσαρμόσετε τη μορφοποίηση του κειμένου ορίζοντας ιδιότητες γραμματοσειράς όπως έντονη γραφή και ύψος γραμματοσειράς. Το παράδειγμα παρουσιάζει τη φόρτωση μιας παρουσίασης, την προσθήκη διαγράμματος, την ενεργοποίηση του πίνακα δεδομένων του διαγράμματος, την εφαρμογή ρυθμίσεων γραμματοσειράς και την αποθήκευση της ενημερωμένης παρουσίασης.

Περιλαμβάνει επίσης σύντομες απαντήσεις σε συχνές ερωτήσεις σχετικά με την εμφάνιση κλειδιών υπομνήματος σε πίνακα δεδομένων διαγράμματος, τη διατήρηση του πίνακα κατά την εξαγωγή, την εργασία με διαγράμματα που φορτώνονται από υπάρχουσες παρουσιάσεις ή πρότυπα, και τον εντοπισμό διαγραμμάτων όπου ο πίνακας δεδομένων είναι ενεργοποιημένος.

## **Ορισμός Ιδιοτήτων Γραμματοσειράς για Πίνακα Δεδομένων Διαγράμματος**
Aspose.Slides for PHP via Java παρέχει υποστήριξη για αλλαγή χρώματος των κατηγοριών σε χρώμα σειράς.  

1. Δημιουργήστε αντικείμενο κλάσης [Presentation](https://reference.aspose.com/slides/el/php-java/aspose.slides/Presentation).
1. Προσθέστε διάγραμμα στη διαφάνεια.
1. Ορίστε τον πίνακα του διαγράμματος.
1. Ορίστε το ύψος της γραμματοσειράς.
1. Αποθηκεύστε την τροποποιημένη παρουσίαση.

Παρακάτω δίνεται παράδειγμα δείγματος.  

```php
  # Δημιουργία κενής παρουσίασης
  $pres = new Presentation();
  try {
    $chart = $pres->getSlides()->get_Item(0)->getShapes()->addChart(ChartType::ClusteredColumn, 50, 50, 600, 400);
    $chart->setDataTable(true);
    $chart->getChartDataTable()->getTextFormat()->getPortionFormat()->setFontBold(NullableBool::True);
    $chart->getChartDataTable()->getTextFormat()->getPortionFormat()->setFontHeight(20);
    $pres->save("output.pptx", SaveFormat::Pptx);
  } finally {
    if (!java_is_null($pres)) {
      $pres->dispose();
    }
  }
```

## **FAQ**

**Μπορώ να εμφανίσω μικρά κλειδιά υπομνήματος δίπλα στις τιμές στον πίνακα δεδομένων του διαγράμματος;**

Ναι. Ο πίνακας δεδομένων υποστηρίζει [κλειδιά υπομνήματος](https://reference.aspose.com/slides/el/php-java/aspose.slides/datatable/setshowlegendkey/), και μπορείτε να τα ενεργοποιήσετε ή να τα απενεργοποιήσετε.

**Θα διατηρηθεί ο πίνακας δεδομένων κατά την εξαγωγή της παρουσίασης σε PDF, HTML ή εικόνες;**

Ναι. Το Aspose.Slides αποδίδει το διάγραμμα ως μέρος της διαφάνειας, έτσι το εξαγόμενο [PDF](/slides/el/php-java/convert-powerpoint-to-pdf/)/[HTML](/slides/el/php-java/convert-powerpoint-to-html/)/[image](/slides/el/php-java/convert-powerpoint-to-png/) περιλαμβάνει το διάγραμμα μαζί με τον πίνακα δεδομένων του.

**Υποστηρίζονται πίνακες δεδομένων για διαγράμματα που προέρχονται από αρχείο προτύπου;**

Ναι. Για οποιοδήποτε διάγραμμα που έχει φορτωθεί από υπάρχουσα παρουσίαση ή πρότυπο, μπορείτε να ελέγξετε και να αλλάξετε εάν ένας πίνακας δεδομένων [είναι εμφανιζόμενος](https://reference.aspose.com/slides/el/php-java/aspose.slides/chart/hasdatatable/) χρησιμοποιώντας τις ιδιότητες του διαγράμματος.

**Πώς μπορώ γρήγορα να βρω ποια διαγράμματα σε ένα αρχείο έχουν ενεργοποιημένο τον πίνακα δεδομένων;**

Εξετάστε την ιδιότητα κάθε διαγράμματος που υποδεικνύει εάν ο πίνακας δεδομένων [είναι εμφανιζόμενος](https://reference.aspose.com/slides/el/php-java/aspose.slides/chart/hasdatatable/) και διατρέξτε τις διαφάνειες για να εντοπίσετε τα διαγράμματα όπου είναι ενεργοποιημένος.
