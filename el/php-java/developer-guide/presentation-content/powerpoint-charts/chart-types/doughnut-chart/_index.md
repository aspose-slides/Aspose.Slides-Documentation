## **Επισκόπηση**

Αυτό το άρθρο δείχνει πώς να εργαστείτε με ένα γράφημα τύπου δαχτύλιος στο Aspose.Slides προσθέτοντας το γράφημα σε μια διαφάνεια, ορίζοντας το μέγεθος του κεντρικού του τρύπας και αποθηκεύοντας την παρουσίαση. Επικεντρώνεται στη μέθοδο `setDoughnutHoleSize` και παρουσιάζει τα βασικά βήματα που απαιτούνται για την προσαρμογή αυτού του τύπου γραφήματος μέσω κώδικα.

Περιλαμβάνει επίσης ένα σύντομο FAQ που καλύπτει σχετικές περιπτώσεις γραφημάτων δαχτυλίας, όπως η χρήση πολλαπλών σειρών για τη δημιουργία πολλαπλών δαχτυλιδιών, η εργασία με «ξεσπασμένα» γραφήματα δαχτυλίας, και η εξαγωγή γραφήματος ως ραστερ εικόνα ή SVG.

## **Καθορισμός του κεντρικού κενού σε γράφημα δαχτυλίας**

Για να καθορίσετε το μέγεθος της τρύπας σε ένα γράφημα δαχτυλίας, ακολουθήστε τα παρακάτω βήματα:

1. Δημιουργήστε το αντικείμενο [Presentation](https://reference.aspose.com/slides/el/php-java/aspose.slides/presentation).
1. Προσθέστε γράφημα δαχτυλίας στη διαφάνεια.
1. Καθορίστε το μέγεθος της τρύπας στο γράφημα δαχτυλίας.
1. Αποθηκεύστε την παρουσίαση στο δίσκο.

Στο παρακάτω παράδειγμα, έχουμε ορίσει το μέγεθος της τρύπας σε ένα γράφημα δαχτυλίας.

```php
  # Δημιουργήστε ένα αντικείμενο της κλάσης Presentation
  $pres = new Presentation();
  try {
    $chart = $pres->getSlides()->get_Item(0)->getShapes()->addChart(ChartType::Doughnut, 50, 50, 400, 400);
    $chart->getChartData()->getSeriesGroups()->get_Item(0)->setDoughnutHoleSize(90);
    # Αποθηκεύστε την παρουσίαση στο δίσκο
    $pres->save("DoughnutHoleSize_out.pptx", SaveFormat::Pptx);
  } finally {
    if (!java_is_null($pres)) {
      $pres->dispose();
    }
  }
```

## **FAQ**

**Μπορώ να δημιουργήσω ένα πολυεπίπεδο γράφημα δαχτυλίας με πολλαπλά δαχτυλίδια;**

Ναι. Προσθέστε πολλαπλές σειρές σε ένα μόνο γράφημα δαχτυλίας — κάθε σειρά γίνεται ένα ξεχωριστό δαχτυλίδι. Η σειρά των δαχτυλιών καθορίζεται από τη σειρά των σειρών στη συλλογή.

**Υποστηρίζεται ένα «ξεσπασμένο» γράφημα δαχτυλίας (διαχωρισμένα τμήματα);**

Ναι. Υπάρχει ο τύπος γραφήματος Exploded Doughnut και η ιδιότητα έκρηξης στα σημεία δεδομένων· μπορείτε να διαχωρίσετε μεμονωμένα τμήματα.

**Πώς μπορώ να αποκτήσω μια εικόνα ενός γραφήματος δαχτυλίας (PNG/SVG) για μια αναφορά;**

Ένα γράφημα είναι σχήμα· μπορείτε να το αποδώσετε σε μια [raster image](https://reference.aspose.com/slides/el/php-java/aspose.slides/shape/#getImage) ή να εξάγετε το γράφημα σε μια [SVG image](https://reference.aspose.com/slides/el/php-java/aspose.slides/shape/#writeAsSvg).
