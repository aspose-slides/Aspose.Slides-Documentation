## **Εισαγωγή**

Στο PowerPoint 2019, η Microsoft εισήγαγε τη ρύθμιση **Always Open Read-Only** ως μία από τις επιλογές που μπορούν να χρησιμοποιήσουν οι χρήστες για να προστατεύσουν τις παρουσιάσεις τους. Μπορεί να θέλετε να χρησιμοποιήσετε αυτή τη ρύθμιση Μόνο για ανάγνωση για να προστατεύσετε μια παρουσίαση όταν

- Θέλετε να αποτρέψετε τυχαίες επεμβάσεις και να διασφαλίσετε το περιεχόμενο της παρουσίασής σας. 
- Θέλετε να ενημερώσετε τους ανθρώπους ότι η παρουσίαση που παρέχετε είναι η τελική έκδοση. 

Αφού επιλέξετε την επιλογή **Always Open Read-Only** για μια παρουσίαση, όταν οι χρήστες ανοίξουν την παρουσίαση, βλέπουν τη σύσταση **Read-Only** και μπορεί να εμφανιστεί ένα μήνυμα της μορφής: *Για να αποτραπούν τυχαίες αλλαγές, ο δημιουργός ορίσε το αρχείο να ανοίγει μόνο για ανάγνωση.*

Η σύσταση **Read-Only** είναι μια απλή αλλά αποτελεσματική αποθάρρυνση που αποτρέπει την επεξεργασία επειδή οι χρήστες πρέπει να εκτελέσουν μια ενέργεια για να την αφαιρέσουν προτού τους επιτραπεί η επεξεργασία μιας παρουσίασης. Εάν δεν θέλετε οι χρήστες να κάνουν αλλαγές στην παρουσίαση και θέλετε να το επικοινωνήσετε με ευγενικό τρόπο, η σύσταση **Read-Only** μπορεί να είναι μια καλή επιλογή για εσάς. 

> Εάν μια παρουσίαση με την προστασία **Read-Only** ανοίξει σε μια παλαιότερη έκδοση του Microsoft PowerPoint—που δεν υποστηρίζει τη νεοεισαχθείσα λειτουργία—η σύσταση **Read-Only** αγνοείται (η παρουσίαση ανοίγει κανονικά).

## **Εφαρμογή λειτουργίας Read-Only**

Το Aspose.Slides for PHP via Java σας επιτρέπει να ορίσετε μια παρουσίαση ως **Read-Only**, που σημαίνει ότι οι χρήστες (αφού ανοίξουν την παρουσίαση) βλέπουν τη σύσταση **Read-Only**. Αυτό το παράδειγμα κώδικα σας δείχνει πώς να ορίσετε μια παρουσίαση ως **Read-Only** χρησιμοποιώντας το Aspose.Slides:

```php
  $pres = new Presentation();
  try {
    $pres->getProtectionManager()->setReadOnlyRecommended(true);
    $pres->save("ReadOnlyPresentation.pptx", SaveFormat::Pptx);
  } finally {
    if (!java_is_null($pres)) {
      $pres->dispose();
    }
  }
```

{{% alert color="primary" %}} 

**Σημείωση**: Η σύσταση **Read-Only** προορίζεται απλώς να αποθαρρύνει την επεξεργασία ή να εμποδίσει τυχαίες αλλαγές σε μια παρουσίαση PowerPoint. Εάν ένα άτομο με πρόθεση—που γνωρίζει τι κάνει—αποφασίσει να επεξεργαστεί την παρουσίασή σας, μπορεί εύκολα να αφαιρέσει τη ρύθμιση Read-Only. Εάν χρειάζεται σοβαρά να αποτρέψετε μη εξουσιοδοτημένη επεξεργασία, είναι καλύτερο να χρησιμοποιήσετε [more stringent protections that involve encryptions and passwords](https://docs.aspose.com/slides/el/php-java/password-protected-presentation/).

{{% /alert %}} 

## **Συχνές ερωτήσεις**

**Πώς διαφέρει το «Read-Only recommended» από την πλήρη προστασία με κωδικό;**

Το «Read-Only recommended» εμφανίζει μόνο μια πρόταση για άνοιγμα του αρχείου σε λειτουργία μόνο για ανάγνωση και είναι εύκολο να παρακαμφθεί. [Password protection](/slides/el/php-java/password-protected-presentation/) περιορίζει πραγματικά το άνοιγμα ή την επεξεργασία και είναι κατάλληλο όταν χρειάζεστε πραγματικό έλεγχο ασφαλείας.

**Μπορεί το «Read-Only recommended» να συνδυαστεί με υδατογραφήματα για περαιτέρω αποθάρρυνση των επεξεργασιών;**

Ναι. Η σύσταση μπορεί να συνδυαστεί με [watermarks](/slides/el/php-java/watermark/) ως οπτική αποθάρρυνση· είναι ξεχωριστοί μηχανισμοί και συνεργάζονται καλά.

**Μπορεί ένα μακρό ή εξωτερικό εργαλείο να τροποποιήσει ακόμα το αρχείο όταν η σύσταση είναι ενεργή;**

Ναι. Η σύσταση δεν εμποδίζει προγραμματιστικές αλλαγές. Για να αποτρέψετε αυτοματοποιημένες επεμβάσεις, χρησιμοποιήστε [passwords and encryption](/slides/el/php-java/password-protected-presentation/).

**Πώς σχετίζεται το «Read-Only recommended» με τις μεθόδους «isEncrypted» και «isWriteProtected»;**

Είναι διαφορετικά σήματα. Το «Read-Only recommended» είναι μια ήπια, προαιρετική προτροπή· [isWriteProtected](https://reference.aspose.com/slides/el/php-java/aspose.slides/protectionmanager/iswriteprotected/) και [isEncrypted](https://reference.aspose.com/slides/el/php-java/aspose.slides/protectionmanager/isencrypted/) υποδεικνύουν πραγματικούς περιορισμούς εγγραφής ή ανάγνωσης που εξαρτώνται από κωδικούς ή κρυπτογράφηση.
