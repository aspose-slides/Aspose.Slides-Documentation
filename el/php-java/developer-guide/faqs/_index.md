## **Επισκόπηση**

Αυτή η Συχνές Ερωτήσεις παρέχει απαντήσεις σε κοινές ερωτήσεις σχετικά με το Aspose.Slides. Καλύπτει τις υποστηριζόμενες μορφές αρχείων, τη διαχείριση εξαιρέσεων κατά την εργασία με μεγάλες παρουσιάσεις, την αλλαγή μεγέθους των διαφανειών, την προεπισκόπηση των διαφανειών, την ανάκτηση κειμένου από παρουσιάσεις, τη μορφοποίηση των περιγραμμάτων πινάκων, την τοποθέτηση εικόνων και την επίλυση προβλημάτων που σχετίζονται με γραμματοσειρές κατά τη μετατροπή των παρουσιάσεων σε PDF ή εικόνες.

## **Υποστηριζόμενες μορφές αρχείων**

**Q**: Ποιες μορφές αρχείων υποστηρίζει το Aspose.Slides για PHP μέσω Java;

**A**: Το Aspose.Slides για PHP μέσω Java υποστηρίζει τις μορφές αρχείων που περιγράφονται στην [Supported File Formats](/slides/el/php-java/supported-file-formats/).

## **Εξαιρέσεις**

**Q**: Λαμβάνω εξαίρεση «έλλειμμα μνήμης» κατά τη φόρτωση ενός μεγάλου αρχείου PPT με εικόνες. Υπάρχει περιορισμός στο Aspose.Slides σχετικά με το μέγεθος του αρχείου;

**A**: Δεν υπάρχει συγκεκριμένος τύπος για τον υπολογισμό του μεγέθους παρουσίασης που υποστηρίζεται από το Aspose.Slides. Πρέπει να υπάρχει επαρκής χώρος για τη διατήρηση ολόκληρης της δομής της παρουσίασης και των εικόνων στη μνήμη. Συνήθως, οι εικόνες στη μνήμη καταλαμβάνουν περισσότερο χώρο από τον σκληρό δίσκο, ειδικά όταν οι εικόνες έχουν πρόσθετα εφέ.

Γενικά, το Aspose.Slides για PHP μέσω Java μπορεί εύκολα να διαχειριστεί αρχεία παρουσίασης περίπου 300 MB σε ένα διακομιστή με 4 GB RAM.

## **Εργασία με διαφάνειες**

**Q**: Μπορώ να αλλάξω το μέγεθος των διαφανειών σε μια παρουσίαση;

**A**: Μπορείτε να χρησιμοποιήσετε τη μέθοδο `getSlideSize` που παρέχεται από την κλάση [Presentation](https://reference.aspose.com/slides/el/php-java/aspose.slides/presentation/) για να ορίσετε το μέγεθος των διαφανειών σε μια παρουσίαση.

**Q**: Υπάρχει τρόπος να οριστούν διαφάνειες διαφορετικού μεγέθους σε μια παρουσίαση;

**A**: Καθώς το μέγεθος των διαφανειών ορίζεται σε επίπεδο παρουσίασης στα έγγραφα Microsoft PowerPoint, δεν υπάρχει τρόπος να γίνει αυτό.

**Q**: Υποστηρίζει το Aspose.Slides για PHP μέσω Java την προεπισκόπηση μιας διαφάνειας πριν από την αποθήκευση;

**A**: Μπορείτε να αποδώσετε τις διαφάνειες της παρουσίασης σε εικόνες και να χρησιμοποιήσετε αυτές τις εικόνες για την προεπισκόπηση των διαφανειών.

## **Εργασία με κείμενο**

**Q**: Είναι δυνατόν να ανακτήσετε όλο το κείμενο από μια παρουσίαση;

**A**: Το Aspose.Slides για PHP μέσω Java παρέχει την κλάση [SlideUtil](https://reference.aspose.com/slides/el/php-java/aspose.slides/slideutil/) η οποία προσφέρει διάφορες μεθόδους για την ανάκτηση ολόκληρου του κειμένου από τις παρουσιάσεις.

**Q**: Γιατί τα μεγέθη παραγράφων διαφέρουν στα λειτουργικά συστήματα Windows και Linux;

**A**: Ο υπολογισμός των μεγεθών της παραγράφου βασίζεται στον υπολογισμό του μεγέθους του κειμένου που αναπαριστά τη συγκεκριμένη παράγραφο. Ο υπολογισμός του μεγέθους του κειμένου βασίζεται στις μετρικές της γραμματοσειράς που ορίζεται στην παρουσίαση PowerPoint. Εάν η καθορισμένη γραμματοσειρά λείπει, αντικαθίσταται με τη πιο παρόμοια γραμματοσειρά, αλλά αυτή η γραμματοσειρά έχει μετρικές διαφορετικές από τις αρχικές. Ως αποτέλεσμα, ο υπολογισμός των μεγεθών παραγράφων σε διαφορετικά συστήματα θα οδηγήσει σε διαφορετικά αποτελέσματα ανάλογα με το σύνολο των εγκατεστημένων γραμματοσειρών. Για να επιτύχετε το ίδιο αποτέλεσμα σε διαφορετικά λειτουργικά συστήματα, πρέπει να εγκαταστήσετε τις ίδιες γραμματοσειρές στα συστήματα ή να τις φορτώσετε κατά την εκτέλεση ως [external fonts](/slides/el/php-java/custom-font/).

## **Μορφοποίηση και εικόνες**

**Q**: Πώς μπορώ να ορίσω το χρώμα του περιγράμματος ενός πίνακα;

**A**: Μπορείτε να αλλάξετε το χρώμα όλων των περιγραμμάτων του πίνακα ή μόνο του περιγράμματος γύρω από ολόκληρο τον πίνακα. Για την αλλαγή όλων των περιγραμμάτων, χρησιμοποιήστε τη μέθοδο `getCellFormat` από την κλάση [Cell](https://reference.aspose.com/slides/el/php-java/aspose.slides/cell/). Για το περίγραμμα ολόκληρου του πίνακα, θα πρέπει να περάσετε από τα κελιά και να αλλάξετε το χρώμα των εξωτερικών περιγραμμάτων.

**Q**: Ποια μονάδα μέτρησης χρησιμοποιεί το Aspose.Slides για PHP μέσω Java για την τοποθέτηση εικόνων;

**A**: Οι συντεταγμένες και τα μεγέθη όλων των σχημάτων στις διαφάνειες μετρώνται σε points (72 dpi).

## **Εργασία με γραμματοσειρές**

**Q**: Κατά τη μετατροπή PPT σε PDF ή εικόνες, γιατί διαφέρουν οι γραμματοσειρές στα έγγραφα εξόδου;

**A**: Αυτό το ζήτημα μπορεί να υποδεικνύει ότι οι γραμματοσειρές που χρησιμοποιούνται στην παρουσίαση λείπουν από το λειτουργικό σύστημα στο οποίο εκτελέστηκε ο κώδικας. Θα πρέπει να εγκαταστήσετε τις γραμματοσειρές στο λειτουργικό σύστημα ή να τις φορτώσετε ως εξωτερικές γραμματοσειρές χρησιμοποιώντας την κλάση [FontsLoader](https://reference.aspose.com/slides/el/php-java/aspose.slides/fontsloader/) όπως φαίνεται παρακάτω:
```php
$folders = ["path_to_a_folder_with_fonts"];
FontsLoader::loadExternalFonts($folders);
```
