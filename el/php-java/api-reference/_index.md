Το Aspose.Slides for PHP via Java είναι μια βιβλιοθήκη κλάσεων που επιτρέπει στις εφαρμογές PHP να διαβάζουν και να γράφουν έγγραφα παρουσίασης χωρίς τη χρήση του Microsoft PowerPoint® ή άλλων. Το Aspose.Slides for PHP via Java επιτρέπει τη διαχείριση κειμένου, σχημάτων, πινάκων & κινούμενων σχεδίων, την προσθήκη ήχου και βίντεο στις διαφάνειες, την προεπισκόπηση διαφανειών, την εξαγωγή διαφανειών σε μορφή SVG, PDF και περισσότερα.

Η πιο πρόσφατη αναφορά API μπορεί να βρεθεί στη διεύθυνση [Aspose.Slides for PHP via Java API Reference](https://reference.aspose.com/slides/el/php-java/).
