Μια βιβλιοθήκη είναι μια δομή περιεχομένου που περιέχει αρχεία. Μια βιβλιοθήκη μπορεί να περιέχει μόνο έναν τύπο αρχείου ή πολλούς διαφορετικούς τύπους, συμπεριλαμβανομένων εγγράφων, λογιστικών φύλλων και παρουσιάσεων. Μια βιβλιοθήκη SharePoint δημιουργεί μια ενιαία τοποθεσία όπου οι άνθρωποι μπορούν να ανεβάζουν και να έχουν πρόσβαση σε έγγραφα. Τα έγγραφα αποθηκεύονται σε δυαδική μορφή στη βάση δεδομένων SQL Server που χρησιμοποιείται από τον ιστότοπο SharePoint. Τα έγγραφα είναι διαθέσιμα σε άλλους χρήστες του SharePoint ως λίστα υπερσυνδέσμων.
