## **Μετατροπή μιας Παρουσίασης**
Για να μεταφέρετε ένα έγγραφο Microsoft PowerPoint από βιβλιοθήκη εγγράφων SharePoint: 

1. Επιλέξτε ένα έγγραφο Microsoft PowerPoint σε μια βιβλιοθήκη εγγράφων.
2. Κάντε κλικ στο κάτω βέλος για να εμφανιστεί ένα μενού και επιλέξτε **Convert via Aspose.Slides.SharePoint**. 

   **Το μενού του αρχείου Presentation 2 εμφανίζει την επιλογή Convert via Aspose.Slides** 

![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_2.png)




3. Επιλέξτε τη ζητούμενη μορφή εξόδου από τη φόρμα. Εάν θέλετε, αλλάξτε το όνομα του αρχείου εξόδου και τον φάκελο προορισμού.
4. Κάντε κλικ στο **Convert** για να μετατρέψετε το αρχείο. 

   **Η φόρμα μετατροπής σας επιτρέπει να επιλέξετε τη μορφή αρχείου εξόδου, το όνομα και τον προορισμό** 

![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_3.png)




5. Όταν ολοκληρωθεί η μετατροπή, εμφανίζεται ένα μήνυμα επιτυχίας. 

   **Η μετατροπή ήταν επιτυχής** 

![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_4.png)




6. Κάντε κλικ στο **Source Library** (για να μεταβείτε στον πηγαίο κατάλογο) ή στο **Destination Library** (για να μεταβείτε στον κατάλογο όπου αποθηκεύτηκε το αρχείο). 

   Το μετατρεπόμενο έγγραφο εμφανίζεται στη βιβλιοθήκη εγγράφων. 

   **Το μετατρεπόμενο έγγραφο εμφανίζεται στη βιβλιοθήκη στην οποία αποθηκεύτηκε** 

![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_5.png)
