{{% alert color="primary" %}} 

Το Aspose.Slides for SharePoint κατεβάζεται ως το αρχείο Aspose.Slides.SharePoint.zip. Το αρχείο περιέχει: 

- **Aspose.Slides.SharePoint.wsp**: Αρχείο λύσης SharePoint. Το Aspose.Slides for SharePoint συσκευάζεται ως λύση SharePoint για να διευκολύνει την ενεργοποίηση και απενεργοποίηση σε όλη τη φάρμα διακομιστών.
- **Aspose_LicenseAgreement.rtf**: Η σύμβαση άδειας χρήσης τελικού χρήστη.
- **Setup.exe**: Το πρόγραμμα εγκατάστασης.
- **Setup.exe.config**: Το αρχείο ρυθμίσεων της εγκατάστασης.

{{% /alert %}} 
## **Διαδικασία εγκατάστασης**
Πριν ξεκινήσει η εγκατάσταση, το πρόγραμμα εγκατάστασης ελέγχει ότι:

- WSS 3.0 ή MOSS 2007 είναι εγκατεστημένο.
- Ο χρήστης διαθέτει άδεια για εγκατάσταση λύσεων SharePoint.
- Η βάση δεδομένων του SharePoint είναι ενεργή.
- Η υπηρεσία WSS Administration έχει ξεκινήσει.
- Η υπηρεσία WSS Timer έχει ξεκινήσει.

Οι υπηρεσίες WSS Administration και Timer απαιτούνται επειδή κάποιες ενέργειες εγκατάστασης εξαρτώνται από μια εργασία χρονοδιακόπτη για να διαδοθούν σε όλους τους διακομιστές της φάρμας. 
### **Εκτέλεση της εγκατάστασης**
Για την εγκατάσταση του Aspose.Slides for SharePoint: 

1. Αποσυμπιέστε το αρχείο Aspose.Slides.SharePoint zip στον τοπικό δίσκο του διακομιστή MOSS 7.0 ή WSS 3.0.
2. Εκτελέστε το setup.exe και ακολουθήστε τις οδηγίες στην οθόνη.
   Το πρόγραμμα εγκατάστασης εκτελεί τις παρακάτω ενέργειες: 
   1. Ελέγχει τις προϋποθέσεις εγκατάστασης. Η εγκατάσταση δεν θα προχωρήσει εάν κάποιος έλεγχος αποτύχει. 

      **Εκτέλεση ελέγχου συστήματος** 

![todo:image_alt_text](installing-aspose-slides-for-sharepoint_1.png)




3. Προβάλλει τη Συμφωνία Άδειας Χρήσης Τελικού Χρήστη. Πρέπει να αποδεχτείτε τη συμφωνία για να προχωρήσετε. 

   **Η Συμφωνία Άδειας Χρήσης** 

![todo:image_alt_text](installing-aspose-slides-for-sharepoint_2.png)




4. Προβάλλει την επιλογή του στόχου ανάπτυξης. Επιλέγει τις εφαρμογές web και τις συλλογές ιστότοπων για τις οποίες θα ενεργοποιηθεί η λειτουργία. 

   **Επιλογή στόχων ανάπτυξης** 

![todo:image_alt_text](installing-aspose-slides-for-sharepoint_3.png)




5. Αναπτύσσει τη λειτουργία στη φάρμα διακομιστών. 

   **Γραμμή προόδου εγκατάστασης** 

![todo:image_alt_text](installing-aspose-slides-for-sharepoint_4.png)




6. Ενεργοποιεί το Aspose.Slides για τις επιλεγμένες συλλογές ιστότοπων και διαμορφώνει τις γονικές εφαρμογές web τους.
7. Προβάλλει λίστα με τις εφαρμογές web και τις συλλογές ιστότοπων για τις οποίες η λειτουργία έχει αναπτυχθεί και ενεργοποιηθεί. 

   **Επιτυχής εγκατάσταση**
