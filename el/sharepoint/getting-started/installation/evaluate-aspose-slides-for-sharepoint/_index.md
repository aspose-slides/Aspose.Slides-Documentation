{{% alert color="primary" %}}

Βεβαιωθείτε ότι εκμεταλλεύεστε τη δωρεάν αξιολόγηση του Aspose.Words για SharePoint: δεν έχει χρονικό όριο και παρέχεται με δωρεάν τεχνική υποστήριξη. Ενθαρρύνουμε τους χρήστες μας να αξιολογούν τα προϊόντα μας πριν από την αγορά, ώστε να γνωρίζετε ότι το προϊόν θα κάνει αυτό που θέλετε.

{{% /alert %}}

Είναι το ίδιο αρχείο λήψης για την αξιολόγηση και την επίσημη έκδοση του Aspose.Slides για SharePoint. Απλώς [κατεβάστε το Aspose.Slides για SharePoint](http://www.aspose.com/community/files/73/sharepoint-components/aspose.slides-for-sharepoint/default.aspx) από τη σελίδα λήψης, [εγκαταστήστε το]() και θα λειτουργεί από προεπιλογή σε λειτουργία αξιολόγησης.

Η λειτουργία αξιολόγησης προσθέτει ένα υδατογράφημα αξιολόγησης σε κάθε διαφάνεια του εξαγόμενου εγγράφου. Όταν έχετε αγοράσει άδεια, απλώς εγκαταστήστε τη λύση άδειας πάνω στην εγκατεστημένη αντίγραφο αξιολόγησης του Aspose.Slides για SharePoint και τότε θα λειτουργεί σε λειτουργία αδειοδοτημένης χρήσης.
