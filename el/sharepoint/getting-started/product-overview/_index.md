## **Καλώς ήρθατε στο Aspose.Slides for SharePoint!**

Aspose.Slides for SharePoint είναι μια ευέλικτη λύση που καθιστά δυνατή τη μετατροπή εγγράφων PowerPoint® μέσα σε ιστότοπους Microsoft SharePoint.

## **Επισκόπηση προϊόντος**

Aspose.Slides for SharePoint υποστηρίζει έναν αριθμό μορφών εγγράφων PowerPoint:

- PPT – Παρουσίαση Microsoft PowerPoint 97‑2003
- PPS – Προβολή διαφανειών Microsoft PowerPoint 97‑2003
- POT – Πρότυπο Microsoft PowerPoint 97‑2003
- PPTX – Παρουσίαση Office Open XML
- PPSX – Προβολή διαφανειών Office Open XML
- POTX – Πρότυπο Office Open XML

Aspose.Slides for SharePoint είναι σχεδιασμένο να λειτουργεί με τα παρακάτω προϊόντα:

- Windows SharePoint Services 3.0 (WSS)
- Microsoft Office SharePoint Server 2007 (MOSS) Standard
- Microsoft Office SharePoint Server 2007 (MOSS) Enterprise
- Microsoft Office SharePoint Server 2013
- Microsoft Office SharePoint Server 2019

Δεν υπάρχουν άλλες απαιτήσεις συστήματος εκτός από αυτές που υπάρχουν για τα παραπάνω προϊόντα.
