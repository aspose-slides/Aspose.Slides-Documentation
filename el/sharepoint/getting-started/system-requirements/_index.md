## **Απαιτήσεις διακομιστή**
{{% alert color="primary" %}} 

Για την εγκατάσταση του Aspose.Slides for SharePoint, απαιτείται σύστημα που τρέχει μία από τις παρακάτω εκδόσεις:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft Office SharePoint Server 2010
- Microsoft Office SharePoint Server 2013
- Microsoft Office SharePoint Server 2016
- Microsoft Office SharePoint Server 2019

{{% /alert %}} {{% alert color="primary" %}} 

Η εγκατάσταση του Service Pack 2 ή νεότερης έκδοσης για αυτά τα προϊόντα συνιστάται.

{{% /alert %}} 
## **Απαιτήσεις πελάτη**
{{% alert color="primary" %}} 

Οι ιστότοποι SharePoint προσπελάζονται μέσω ενός web browser. Το Aspose.Slides for SharePoint έχει δοκιμαστεί με τους παρακάτω browsers:

- Microsoft Internet Explorer 6, 7, 8
- Mozilla FireFox 3.5

{{% /alert %}}
