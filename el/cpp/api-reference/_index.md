{{% alert color="primary" %}} 

Το Aspose.Slides for C++ είναι μια βιβλιοθήκη κλάσεων που επιτρέπει στις εφαρμογές C++ να διαβάζουν και να γράφουν έγγραφα παρουσίασης χωρίς τη χρήση του Microsoft PowerPoint® ή άλλων. Το Aspose.Slides for C++ επιτρέπει τη διαχείριση κειμένου, σχημάτων, πινάκων & εφέ κίνησης, την προσθήκη ήχου και βίντεο στις διαφάνειες, την προεπισκόπηση των διαφανειών, την εξαγωγή των διαφανειών σε μορφή SVG, PDF και άλλα.

Η πιο πρόσφατη Αναφορά API βρίσκεται στο [Aspose.Slides for C++ API Reference](https://reference.aspose.com/slides/el/cpp/).

{{% /alert %}}
