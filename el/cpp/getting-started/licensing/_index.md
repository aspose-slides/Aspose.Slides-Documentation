## **Επισκόπηση**

Το Aspose.Slides μπορεί να χρησιμοποιηθεί σε λειτουργία αξιολόγησης ή με έγκυρη άδεια. Η έκδοση αξιολόγησης παρέχει την ίδια λειτουργικότητα με την έκδοση με άδεια, αλλά προσθέτει υδατογράφημα αξιολόγησης όταν οι παρουσιάσεις ανοίγουν ή αποθηκεύονται και περιορίζει την εξαγωγή κειμένου σε μία διαφάνεια.

Αυτό το άρθρο εξηγεί πώς λειτουργεί η αδειοδότηση στο Aspose.Slides και πώς να εφαρμόσετε μια άδεια πριν χρησιμοποιήσετε τη βιβλιοθήκη. Μια άδεια μπορεί να φορτωθεί από αρχείο, ροή ή ενσωματωμένο πόρο χρησιμοποιώντας την κλάση `License`. Το άρθρο επίσης δείχνει πώς να πιστοποιήσετε αν μια άδεια έχει εφαρμοστεί σωστά.

## **Αδειοδότηση στο Aspose.Slides**

* Μια έκδοση αξιολόγησης γίνεται με άδεια μετά την αγορά μιας άδειας και την εφαρμογή της προσθέτοντας μερικές γραμμές κώδικα.
* Η άδεια είναι ένα αρχείο XML απλού κειμένου που περιέχει λεπτομέρειες όπως το όνομα του προϊόντος, ο αριθμός των προγραμματιστών που καλύπτεται, η ημερομηνία λήξης της συνδρομής κ.ά.
* Το αρχείο άδειας υπογράφεται ψηφιακά, επομένως δεν πρέπει να τροποποιηθεί. Ακόμη και μια τυχαία αλλαγή—όπως η προσθήκη διακοπής γραμμής—θα ακυρώσει το αρχείο.
* Το Aspose.Slides for C++ συνήθως αναζητά το αρχείο άδειας στις ακόλουθες θέσεις:
  * Μια διαδρομή που ορίζεται ρητά στον κώδικά σας
  * Ο φάκελος που περιέχει το DLL του component (περιλαμβάνεται στο Aspose.Slides)
  * Ο φάκελος που περιέχει τη συναρμολόγηση (assembly) που καλεί το DLL του component
* Για να αποφύγετε τους περιορισμούς της έκδοσης αξιολόγησης, πρέπει να ορίσετε την άδεια πριν χρησιμοποιήσετε το Aspose.Slides. Μια άδεια χρειάζεται να οριστεί μόνο μία φορά ανά εφαρμογή ή διαδικασία.

## **Εφαρμογή Άδειας**

Μια άδεια μπορεί να φορτωθεί από **αρχείο**, **ροή** ή **ενσωματωμένο πόρο**.

{{% alert color="primary" %}}

Το Aspose.Slides παρέχει την κλάση [License](https://reference.aspose.com/slides/el/cpp/class/aspose.slides.license/) για λειτουργίες αδειοδότησης.

{{% /alert %}} 

{{% alert color="warning" %}}

Οι νέες άδειες μπορούν να ενεργοποιήσουν το Aspose.Slides μόνο με την έκδοση 21.4 ή νεότερη. Παλαιότερες εκδόσεις χρησιμοποιούν διαφορετικό σύστημα αδειοδότησης και δεν θα αναγνωρίσουν αυτές τις άδειες.

{{% /alert %}}

### **Αρχείο**

Ο πιο εύκολος τρόπος για να ορίσετε μια άδεια είναι να τοποθετήσετε το αρχείο άδειας στον ίδιο φάκελο με το DLL του component (περιλαμβάνεται στο Aspose.Slides) και να δηλώσετε μόνο το όνομα του αρχείου, χωρίς τη διαδρομή.

Ο παρακάτω κώδικας C++ δείχνει πώς να ορίσετε ένα αρχείο άδειας:

```c++
#include <Util/License.h>

using namespace Aspose::Slides;

int main()
{
    auto license = MakeObject<License>();
    license->SetLicense(u"Aspose.Slides.lic");

    return 0;
}
```

{{% alert color="warning" %}} 

Αν τοποθετήσετε το αρχείο άδειας σε διαφορετικό κατάλογο, τότε όταν καλέσετε τη μέθοδο [License::SetLicense](https://reference.aspose.com/slides/el/cpp/aspose.slides/license/setlicense/), το όνομα αρχείου στο τέλος της συγκεκριμένης διαδρομής πρέπει να ταιριάζει ακριβώς με το όνομα του αρχείου άδειας σας.

Για παράδειγμα, αν μετονομάσετε το αρχείο άδειας σας σε *Aspose.Slides.lic.xml*, πρέπει να περάσετε τη πλήρη διαδρομή που λήγει σε *Aspose.Slides.lic.xml* στη μέθοδο [License::SetLicense](https://reference.aspose.com/slides/el/cpp/aspose.slides/license/setlicense/) στον κώδικά σας.

{{% /alert %}}

### **Ροή**

Μπορείτε να φορτώσετε μια άδεια από ροή. Ο παρακάτω κώδικας C++ δείχνει πώς να εφαρμόσετε μια άδεια από ροή:

```c++
auto license = MakeObject<License>();

auto stream = File::OpenRead(u"Aspose.Slides.lic");

license->SetLicense(stream);
```

## **Επικύρωση Άδειας**

Για να ελέγξετε αν μια άδεια έχει οριστεί σωστά, μπορείτε να την επικυρώσετε. Ο παρακάτω κώδικας C++ δείχνει πώς να επικυρώσετε μια άδεια:

```c++
auto license = MakeObject<License>();

license->SetLicense(u"Aspose.Slides.lic");

if (license->IsLicensed())
{
    Console::WriteLine(u"License is good!");
    Console::ReadKey();
}
```

## **Ασφάλεια Νήματος**

{{% alert title="Note" color="warning" %}} 

Η μέθοδος [License::SetLicense](https://reference.aspose.com/slides/el/cpp/aspose.slides/license/setlicense/) δεν είναι **ασφαλής για χρήση από πολλαπλά νήματα**. Εάν χρειάζεται να καλέσετε αυτή τη μέθοδο από πολλαπλά νήματα ταυτόχρονα, συνιστάται να χρησιμοποιήσετε πρωτόκολλα συγχρονισμού (όπως ένα κλείδωμα) για την αποφυγή πιθανών προβλημάτων.

{{% /alert %}}

Ναι. Η επικύρωση της άδειας γίνεται τοπικά χρησιμοποιώντας το αρχείο άδειας· δεν απαιτείται σύνδεση στο διαδίκτυο.
