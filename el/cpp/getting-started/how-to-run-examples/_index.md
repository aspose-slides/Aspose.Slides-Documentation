## **Software Requirements**
Παρακαλούμε βεβαιωθείτε ότι πληροίτε τις παρακάτω απαιτήσεις πριν τη λήψη και την εκτέλεση των παραδειγμάτων.

1. Visual Studio 2015 ή νεότερο.
## **Download Aspose.Slides from GitHub**
Όλα τα παραδείγματα του Aspose.Slides για C++ φιλοξενούνται στο [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-C).

- Μπορείτε είτε να κλωνοποιήσετε το αποθετήριο χρησιμοποιώντας τον αγαπημένο σας πελάτη GitHub είτε να κατεβάσετε το αρχείο ZIP από [εδώ](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/refs/heads/master.zip).
- Αποσυμπιέστε τα περιεχόμενα του αρχείου ZIP σε οποιονδήποτε φάκελο στον υπολογιστή σας. Όλα τα παραδείγματα βρίσκονται στον φάκελο **Examples**.
- Υπάρχει ένα αρχείο λύσης C++ Visual Studio.
- Τα έργα δημιουργήθηκαν στο Visual Studio 2015, αλλά τα αρχεία λύσης είναι συμβατά με το Visual Studio 2015 και νεότερα.
- Ανοίξτε το αρχείο λύσης στο Visual Studio και κατασκευάστε το έργο.
- Κατεβάστε την πιο πρόσφατη έκδοση του Aspose.Slides για C++ από την [Ενότητα Λήψης](https://downloads.aspose.com/slides/el/cpp).
- Αποσυμπιέστε την έκδοση και αντιγράψτε τους καταλόγους **Include** και **Bin** μέσα στο φάκελο **Examples**.
- Ανοίξτε το αρχείο **main.cpp**, όλα τα παραδείγματα καλούνται από αυτό.
- Αποσχολιάστε τα παραδείγματα που θέλετε να εκτελέσετε μέσα στο έργο.
- Κατασκευάστε και μεταγλωττίστε τη λύση.

Μη διστάσετε να επικοινωνήσετε μέσω των Φόρουμ μας εάν αντιμετωπίζετε προβλήματα με τη ρύθμιση ή την εκτέλεση των παραδειγμάτων.
## **Contribute**
Αν θέλετε να προσθέσετε ή να βελτιώσετε ένα παράδειγμα, σας ενθαρρύνουμε να συνεισφέρετε στο έργο. Όλα τα παραδείγματα και τα έργα επίδειξης σε αυτό το αποθετήριο είναι ανοιχτού κώδικα και μπορούν να χρησιμοποιηθούν ελεύθερα στις δικές σας εφαρμογές.

Για να συνεισφέρετε, μπορείτε να δημιουργήσετε fork το αποθετήριο, να επεξεργαστείτε τον πηγαίο κώδικα και να δημιουργήσετε ένα pull request. Θα εξετάσουμε τις αλλαγές και θα τις ενσωματώσουμε στο αποθετήριο εάν είναι χρήσιμες.
