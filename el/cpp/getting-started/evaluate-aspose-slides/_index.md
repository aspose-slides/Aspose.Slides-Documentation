## **Αξιολόγηση Aspose.Slides**

Μπορείτε εύκολα να κατεβάσετε το Aspose.Slides για αξιολόγηση. Η λήψη αξιολόγησης είναι η ίδια με τη λήψη που αγοράσατε. Η έκδοση αξιολόγησης γίνεται απλώς αδειοδοτημένη όταν προσθέσετε μερικές γραμμές κώδικα για να εφαρμόσετε την άδεια.

Η έκδοση αξιολόγησης του Aspose.Slides (χωρίς καθορισμένη άδεια) παρέχει πλήρη λειτουργικότητα του προϊόντος, αλλά προσθέτει ένα υδατογράφημα αξιολόγησης στην κορυφή του εγγράφου κατά το άνοιγμα και την αποθήκευση, και περιορίζει σε μία διαφάνεια όταν εξάγετε το κείμενο από τις διαφάνειες παρουσίασης.

## **Συχνές ερωτήσεις**

**Μπορώ να δοκιμάσω πολλαπλές παρουσιάσεις παράλληλα σε διαφορετικά νήματα σε λειτουργία αξιολόγησης;**

Ναι. Μπορείτε να επεξεργαστείτε διαφορετικά έγγραφα παράλληλα· δεν θα πρέπει να μοιράζεστε το ίδιο αντικείμενο παρουσίασης [μεταξύ νημάτων](/slides/el/cpp/multithreading/). Η λειτουργία αξιολόγησης δεν το επηρεάζει.

**Χρειάζεται να εγκαταστήσω το Microsoft PowerPoint για να αξιολογήσω τη βιβλιοθήκη σε διακομιστή ή στο CI;**

Όχι. Το Aspose.Slides είναι μια αυτόνομη μηχανή και δεν απαιτεί εγκατεστημένο το PowerPoint, είτε για αξιολόγηση είτε για παραγωγή.

**Μπορώ να δοκιμάσω πλήρως τη μετατροπή PPT/PPTX σε PDF και εικόνες σε λειτουργία αξιολόγησης;**

Ναι. Οι [μετατροπείς](/slides/el/cpp/convert-presentation/) λειτουργούν· το αποτέλεσμα θα περιλαμβάνει ένα υδατογράφημα.

**Μπορώ να χρησιμοποιήσω μια προσωρινή άδεια για δοκιμές φορτίου χωρίς υδατογράφημα;**

Ναι. Μια 30-ήμερη προσωρινή άδεια αφαιρεί τους περιορισμούς της λειτουργίας αξιολόγησης και επιτρέπει δοκιμές χωρίς υδατογράφημα.
