## **Εισαγωγή**

Το Aspose.Slides δεν απαιτεί την εγκατάσταση του Microsoft PowerPoint, επειδή το Aspose.Slides είναι μια ανεξάρτητη μηχανή δημιουργίας εγγράφων Microsoft PowerPoint, μετατροπής, διάταξης σελίδων και απόδοσης.

## **Υποστηριζόμενα λειτουργικά συστήματα**
Το Aspose.Slides for C++ είναι μια εγγενής βιβλιοθήκη C++. Το Aspose.Slides for C++ υποστηρίζει τα ακόλουθα λειτουργικά συστήματα και πλατφόρμες 64-bit και 32-bit:

### **Windows**
- Microsoft Windows Server 2008 (x64, x86)
- Microsoft Windows Server 2012 (x64, x86)
- Microsoft Windows Server 2012 R2 (x64, x86)
- Microsoft Windows Server 2016 (x64, x86)
- Microsoft Windows Server 2019 (x64, x86)
- Microsoft Windows XP (x64, x86)
- Microsoft Windows 7 (x64, x86)
- Microsoft Windows 8, 8.1 (x64, x86)
- Microsoft Windows 10 (x64, x86)

### **Linux**
- Ubuntu 16.04 ή νεότερο.
- CentOS 8 ή νεότερο.
- Fedora 24 ή νεότερο.
- Και άλλα Linux x86_64 με glibc 2.23 ή νεότερο.

### **macOS**
- macOS Monterey 12.1 ή νεότερο.

## **Περιβάλλοντα ανάπτυξης**
Μπορείτε να χρησιμοποιήσετε το Aspose.Slides for C++ κατά την ανάπτυξη εφαρμογών για Windows, Linux ή macOS.

### **Windows**
- Microsoft Visual Studio 2017 ή νεότερη έκδοση.
- CMake 3.18 ή νεότερο.

### **Linux**
- Clang 3.9 ή νεότερο.
- GCC 6.1 ή νεότερο.
- CMake 3.18 ή νεότερο.

### **macOS**
- Xcode 13.4 ή νεότερο.

## **Συχνές ερωτήσεις**

**Χρειάζεται να είναι εγκατεστημένο το Microsoft PowerPoint για τις μετατροπές και την απόδοση;**

Όχι, το PowerPoint δεν είναι απαραίτητο· το Aspose.Slides είναι μια αυτόνομη μηχανή για [δημιουργία](/slides/el/cpp/create-presentation/), τροποποίηση, [μετατροπή](/slides/el/cpp/convert-presentation/), και [απόδοση](/slides/el/cpp/convert-powerpoint-to-png/) παρουσιάσεων.

**Ποια γραμματοσειρά απαιτείται για σωστή απόδοση;**

Στην πράξη, οι γραμματοσειρές που χρησιμοποιούνται στην παρουσίαση ή οι κατάλληλες [εναλλακτικές](/slides/el/cpp/font-substitution/) πρέπει να είναι διαθέσιμες. Για να εξασφαλιστεί συνεπής απόδοση σε Linux/macOS, συνιστάται η εγκατάσταση κοινών πακέτων γραμματοσειρών.

**Γιατί μια προσαρμοσμένη γραμματοσειρά εμφανίζεται ως εφεδρική ή κείμενο που λείπει σε Linux;**

Εάν το αρχείο γραμματοσειράς έχει ασυνεπείς ή κατεστραμμένες εγγραφές στον πίνακα ονομάτων, η στοίβα αντιστοίχισης γραμματοσειρών του Linux (FreeType/fontconfig) μπορεί να επιλέξει άκυρο αρχείο, προκαλώντας το πρόβλημα μη εύρεσης της γραμματοσειράς. Η χρήση μιας έκδοσης γραμματοσειράς με διορθωμένες εγγραφές στον πίνακα ονομάτων ή η εγκατάσταση μιας συνεπούς εναλλακτικής λύσης λύνει το πρόβλημα.
