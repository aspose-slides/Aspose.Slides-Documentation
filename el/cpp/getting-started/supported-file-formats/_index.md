## **Επισκόπηση**

Το Aspose.Slides υποστηρίζει αρχεία παρουσίασης από το Microsoft PowerPoint 97 έως το Office 365, συμπεριλαμβανομένου του Microsoft PowerPoint για Mac. Αυτό το άρθρο παραθέτει τις εκδόσεις PowerPoint που υποστηρίζονται από τη βιβλιοθήκη και παρέχει έναν πίνακα μορφών αρχείων που μπορούν να φορτωθούν, να αποθηκευτούν ή και τα δύο.

Το άρθρο απαντά επίσης σε συχνές ερωτήσεις σχετικά με τη συμμόρφωση PDF, την ενσωμάτωση γραμματοσειρών, αρχεία με προστασία κωδικού, προσαρμοσμένες γραμματοσειρές, εναλλακτικές γραμματοσειρές και επιλογές εξαγωγής XPS.

## **Υποστηριζόμενες Εκδόσεις Microsoft PowerPoint**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint for MAC
- Office 365

## **Συχνές Ερωτήσεις**

**Μπορώ να αποθηκεύσω παρουσιάσεις σε PDF που να πληρούν τα πρότυπα αρχειοθέτησης και προσβασιμότητας (PDF/A και PDF/UA);**

Ναι. Το Aspose.Slides υποστηρίζει εξαγωγή σε PDF με επίπεδα συμμόρφωσης όπως PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b, καθώς και PDF/UA μέσω της ρύθμισης [συμμόρφωση](https://reference.aspose.com/slides/el/cpp/aspose.slides.export/pdfoptions/set_compliance/) στις [επιλογές εξαγωγής PDF](https://reference.aspose.com/slides/el/cpp/aspose.slides.export/pdfoptions/).

**Υποστηρίζει η βιβλιοθήκη ενσωμάτωση γραμματοσειρών κατά την εξαγωγή σε PDF, με λεπτομερή έλεγχο του τι ενσωματώνεται;**

Ναι. Μπορείτε να ελέγξετε εάν οι γραμματοσειρές θα ενσωματωθούν πλήρως ή ως υποσύνολο (μόνο τα χρησιμοποιημένα γλυφά), να καθορίσετε πώς αντιμετωπίζονται οι κοινές συστημικές γραμματοσειρές και να ρυθμίσετε τη συμπεριφορά για κείμενο ASCII μέσω των [επιλογών εξαγωγής PDF](https://reference.aspose.com/slides/el/cpp/aspose.slides.export/pdfoptions/).

**Μπορώ να εντοπίσω αν ένα αρχείο είναι προστατευμένο με κωδικό πριν το φορτώσω;**

Ναι. Χρησιμοποιώντας το API ελέγχου βασισμένο σε εργοστασιακό (factory) μοντέλο [factory-based inspection API](https://reference.aspose.com/slides/el/cpp/aspose.slides/presentationfactory/), μπορείτε να ερωτήσετε ένα αρχείο παρουσίασης για να διαπιστώσετε αν είναι προστατευμένο με κωδικό χωρίς να το ανοίξετε πλήρως.

**Υπάρχουν μηχανισμοί εναλλακτικών γραμματοσειρών και υποστήριξη προσαρμοσμένων γραμματοσειρών;**

Ναι. Η βιβλιοθήκη υποστηρίζει [φόρτωση](/slides/el/cpp/custom-font/) και [ενσωμάτωση](/slides/el/cpp/embedded-font/) προσαρμοσμένων γραμματοσειρών και παρέχει [κανόνες εναλλακτικών γραμματοσειρών](/slides/el/cpp/fallback-font/) για να αποτραπεί η έλλειψη γλυφών κατά τη σχεδίαση και τη μετατροπή.

**Μπορώ να εξάγω διαφάνειες σε XPS και υπάρχουν επιλογές για ρύθμιση της εξόδου XPS;**

Ναι. Η [εξαγωγή σε XPS](/slides/el/cpp/convert-powerpoint-to-xps/) υποστηρίζεται, και μπορείτε να προσαρμόσετε τις σχετικές [επιλογές αποθήκευσης](https://reference.aspose.com/slides/el/cpp/aspose.slides.export/xpsoptions/) για να ελέγξετε την ποιότητα και το περιεχόμενο του εγγράφου XPS.
