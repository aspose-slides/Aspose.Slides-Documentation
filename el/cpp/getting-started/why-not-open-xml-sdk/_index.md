## **Επισκόπηση**

Αυτό το άρθρο εξηγεί πότε οι προγραμματιστές μπορεί να επιλέξουν το Open XML SDK ή το Aspose.Slides για εργασία με έγγραφα παρουσίασης. Περιγράφει το Open XML SDK ως μια βιβλιοθήκη για τη διαχείριση πακέτων OOXML και των υποκείμενων στοιχείων XML, ενώ το Aspose.Slides παρουσιάζεται ως μια βιβλιοθήκη επεξεργασίας παρουσιάσεων με υψηλού επιπέδου μοντέλο αντικειμένων και υποστήριξη για πολλές εργασίες σχετικές με το PowerPoint.

Το άρθρο συγκρίνει και τις δύο επιλογές κατά μορφές που υποστηρίζονται, μοντέλο προγραμματισμού, δυνατότητες απόδοσης και εκτύπωσης, υποστήριξη πλατφόρμας και κοινές περιπτώσεις χρήσης. Επισημαίνει επίσης ότι το Open XML SDK μπορεί να είναι κατάλληλο για βασικές λειτουργίες PPTX ή άμεση πρόσβαση σε στοιχεία OOXML, ενώ το Aspose.Slides είναι πιο κατάλληλο για σύνθετες εργασίες παρουσίασης όπως εργασία με πολλαπλές μορφές PowerPoint, αντιγραφή ή κλωνοποίηση σχημάτων, αντικατάσταση κειμένου, εφαρμογή animations και μετατροπή παρουσιάσεων σε PDF, TIFF ή XPS.

## **Τι είναι το Open XML SDK;**
Κάποιες φορές ακούμε την ερώτηση: Γιατί να χρησιμοποιούμε προϊόντα Aspose αντί για το δωρεάν Open XML SDK; Η απάντηση είναι απλή: χαρακτηριστικά και λειτουργίες. Σύμφωνα με το[Βιβλιοθήκη MSDN](https://docs.microsoft.com/en-us/office/open-xml/open-xml-sdk), το Open XML SDK ορίζεται ως: Το Open XML SDK 2.0 απλοποιεί την εργασία της διαχείρισης πακέτων Open XML και των υποκείμενων στοιχείων σχήματος Open XML μέσα σε ένα πακέτο. Το Open XML SDK 2.0 ενσωματώνει πολλές κοινές εργασίες που εκτελούν οι προγραμματιστές σε πακέτα Open XML, ώστε να μπορείτε να εκτελείτε σύνθετες λειτουργίες με λίγες μόνο γραμμές κώδικα. Τα έγγραφα OOXML είναι ουσιαστικά συμπιεσμένα αρχεία XML και το Open XML SDK είναι μια συλλογή κλάσεων που σας επιτρέπει να εργάζεστε με το περιεχόμενο των εγγράφων OOXML με ισχυρά τυποποιημένο τρόπο. Δηλαδή αντί να αποσυμπιέζετε ένα αρχείο για να εξάγετε XML, να φορτώνετε αυτό το XML σε δέντρο DOM και να εργάζεστε απευθείας με στοιχεία και γνωρίσματα XML, το Open XML SDK παρέχει κλάσεις για να το κάνετε αυτό.

## **Τι είναι το Aspose.Slides;**
Το Aspose.Slides είναι μια βιβλιοθήκη κλάσεων που επιτρέπει στην εφαρμογή σας να εκτελεί τις ακόλουθες εργασίες επεξεργασίας παρουσιάσεων:

- Προγραμματισμός με ένα **Presentation** μοντέλο αντικειμένων.
- Υψηλής ποιότητας μετατροπές μεταξύ όλων των δημοφιλών υποστηριζόμενων μορφών PowerPoint, συμπεριλαμβανομένης της μετατροπής σε PDF και XPS.
- Δυνατότητα δημιουργίας μικρογραφιών διαφάνειας σε γνωστές μορφές όπως PNG, JPEG και BMP, μαζί με εξαγωγή διαφάνειας σε SVG.
- Δυνατότητα δημιουργίας παρουσιάσεων από το μηδέν ή με συνδυασμό από ένα ή πολλαπλά έγγραφα.
- Υποστήριξη προσθήκης animations, Ole Frames, Πινάκων, δημιουργίας και διαχείρισης διαγραμμάτων.
- Διαθεσιμότητα εκτενούς ελέγχου για τη διαχείριση της μορφοποίησης κειμένου σε επίπεδα TextFrames, Paragraphs και Portions.
  Για περισσότερες λεπτομέρειες σχετικά με τις υποστηριζόμενες δυνατότητες, επισκεφθείτε [Δυνατότητες Aspose.Slides](/slides/el/cpp/product-overview/).

|Λεπτομερής προγραμματισμός με μοντέλο αντικειμένων εγγράφου, πρόσβαση σε ατομικά στοιχεία και μορφοποίηση όπως TextHolders, TextFrames, Paragraphs και Portions.|Yes|Yes|
|Χαμηλού επιπέδου άμεση και πλήρης πρόσβαση στα υποκείμενα στοιχεία XML και γνωρίσματα όπως αναγνωριστικά σχέσεων, αναγνωριστικά λιστών ενός εγγράφου OOXML.|Yes|No|

## **Συμπέρασμα**
Το Open XML SDK και το Aspose.Slides δεν ανταγωνίζονται άμεσα επειδή εξυπηρετούν διαφορετικές ανάγκες και κοινά. Το Open XML SDK είναι μια βιβλιοθήκη κλάσεων που παρέχει έναν ισχυρά τυποποιημένο τρόπο εργασίας με έγγραφα OOXML. Το Aspose.Slides είναι μια πολύ χρήσιμη βιβλιοθήκη επεξεργασίας παρουσιάσεων που προσφέρει εξαιρετική υποστήριξη για σχεδόν όλες τις μορφές αρχείων Microsoft PowerPoint. Αν το μόνο που χρειάζεστε είναι μια αρκετά βασική λειτουργία προγραμματισμού σε έγγραφο PPTX, τότε το Open XML SDK μπορεί να είναι η κατάλληλη επιλογή. Με το Open XML SDK, θα είστε άνετοι με απλές εργασίες όπως η δημιουργία ενός απλού εγγράφου PPTX ή η αφαίρεση σχολίων, κεφαλίδων/υποσέλιδων, η εξαγωγή εικόνων κ.λπ. Ορισμένες εργασίες μπορούν να πραγματοποιηθούν με το Open XML SDK, αλλά δεν μπορούν με το Aspose.Slides. Για παράδειγμα, αν χρειάζεται να έχετε άμεση πρόσβαση στα στοιχεία XML και γνωρίσματα ενός εγγράφου OOXML, τότε πρέπει να χρησιμοποιήσετε το Open XML SDK. Ωστόσο, αν χρειάζεται να εκτελέσετε σύνθετες λειτουργίες στα έγγραφα, όπως τα παρακάτω, η χρήση του Aspose.Slides είναι η καλύτερη επιλογή:

- Υποστήριξη παλαιότερων μορφών PowerPoint εκτός του PPTX.
- Αντιγραφή ή κλωνοποίηση σχημάτων μέσα στις διαφάνειες με τρόπο που συνδυάζει αντικείμενα, στυλ και άλλες μορφοποιήσεις κατάλληλα.
- Αντικατάσταση μορφοποιημένου ή αμορφοποιημένου κειμένου.
- Εφαρμογή Animations και χρήση συνδέσμων με τα σχήματα.
- Μετατροπή εγγράφου σε PDF ή XPS ώστε να εμφανίζεται ακριβώς όπως το Microsoft PowerPoint θα το είχε μετατρέψει.
- Ανάπτυξη εφαρμογής C++ σε περιβάλλοντα επιφάνειας εργασίας και κονσόλα.
