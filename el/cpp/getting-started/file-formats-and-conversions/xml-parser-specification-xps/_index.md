## **Σχετικά με το XPS**
Η [XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) είναι μια γλώσσα περιγραφής σελίδων και μια μορφή σταθερού εγγράφου που αναπτύχθηκε αρχικά από τη Microsoft. Όπως το PDF, το XPS είναι μια μορφή εγγράφου σταθερής διάταξης σχεδιασμένη να διατηρεί την πιστότητα του εγγράφου και να παρέχει εμφάνιση ανεξάρτητη από τη συσκευή.
## **XPS στο Aspose.Slides for C++**
Οποιοδήποτε έγγραφο παρουσίασης που μπορεί να φορτωθεί από το Aspose.Slides for C++ μπορεί να μετατραπεί σε μορφή XPS. Το Aspose.Slides for C++ χρησιμοποιεί τη μηχανή διάταξης σελίδων υψηλής πιστότητας και απόδοσης για να παραγάγει έξοδο σε μορφή εγγράφου XPS σταθερής διάταξης.
Αξίζει να αναφερθεί ότι το Aspose.Slides for C++ δημιουργεί άμεσα XPS χωρίς να εξαρτάται από τις κλάσεις του Windows Presentation Foundation (WPF) που περιλαμβάνονται στο C++ Framework 3.5, επιτρέποντας έτσι στο Aspose.Slides for C++ να παράγει έγγραφα XPS σε μηχανές που εκτελούν εκδόσεις του C++ Framework πριν από την έκδοση 3.5. Μπορείτε να μάθετε για την εξαγωγή των εγγράφων παρουσίασης σε έγγραφα XPS μέσω του Aspose.Slides for C++ στο [αυτό το θέμα](/slides/el/net/).
