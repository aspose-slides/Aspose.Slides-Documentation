## **Microsoft PowerPoint (PPT)**
### **Σχετικά με το PPT**
[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) είναι η μορφή αρχείου παρουσίασης που μπορεί να δημιουργηθεί, να διαβαστεί, να τροποποιηθεί και να γραφτεί από διαφορετικές εκδόσεις του Microsoft PowerPoint. Πρόκειται για τη δυαδική μορφή εγγράφων παρουσίασης που αναπτύχθηκε από τη Microsoft.
### **PPT στο Aspose.Slides for C++**
Το Aspose.Slides for C++ μπορεί να διαβάσει αρχεία PPT που δημιουργήθηκαν από το λογισμικό που αναφέρεται παρακάτω.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Αντίστοιχα, τα αρχεία PPT που δημιουργούνται από το Aspose.Slides for C++ μπορούν να διαβαστούν από τα παραπάνω λογισμικά.
### **Πλήρης Υποστήριξη για PPT**
Το Aspose.Slides for C++ παρέχει υποστήριξη για σχεδόν όλες τις δυνατότητες που σχετίζονται με τη μορφή αρχείου PPT. Δεν καλύπτει μόνο τις βασικές / προχωρημένες δυνατότητες που παρέχονται από διαφορετικές εκδόσεις του Microsoft PowerPoint για την επεξεργασία εγγράφων PPT, αλλά επίσης ορισμένες δυνατότητες που δεν υποστηρίζονται ούτε από το Microsoft PowerPoint. Το κύριο πλεονέκτημα της χρήσης της βιβλιοθήκης API Aspose.Slides for C++ είναι η ευκολία στη χρήση για τη διαχείριση τέτοιων λειτουργιών.

Εκτός από τις βασικές εργασίες σχετικές με τη δημιουργία, ανάγνωση και εγγραφή αρχείων PPT, υπάρχουν αρκετές δυνατότητες που παρέχονται από το Aspose.Slides for C++ όπως:

- Εισαγωγή άλλων μορφών αρχείων MS Office ως OLE αντικείμενα σε έγγραφα PPT.
- Εξαγωγή εγγράφων PPT σε μορφές PDF, TIFF, XPS.
- Εξαγωγή διαφανειών στα έγγραφα PPT σε μορφές SVG.
- Απόδοση διαφάνειας σε οποιαδήποτε μορφή εικόνας υποστηρίζεται από το C++ Framework.
- Ορισμός μεγέθους διαφανειών στο έγγραφο PPT.
- Διαχείριση κινήσεων σε σχήματα.
- Διαχείριση παρουσιάσεων διαφανειών.
- Μορφοποίηση κειμένου στις διαφάνειες.
- Σάρωση κειμένου από τα έγγραφα PPT.
- Διαχείριση πινάκων στις διαφάνειες.
- Αυτόματη αντιγραφή master διαφανειών χρησιμοποιώντας τη λειτουργία κλωνοποίησης.

## **PresentationML (PPTX, XML)**
### **Σχετικά με το PresentationML**
Το PresentationML είναι ένα όνομα για μια οικογένεια μορφών βασισμένων σε XML για έγγραφα παρουσίασης. Το Office OpenXML (OOXML) είναι η μορφή βασισμένη σε XML που εισήχθη στις εφαρμογές του Microsoft Office 2007. Το Office OpenXML είναι μια μορφή περιέκτη για διάφορες εξειδικευμένες γλώσσες σήμανσης βασισμένες σε XML. Το PresentationML είναι η γλώσσα σήμανσης που χρησιμοποιεί το Microsoft Office PowerPoint 2007 για την αποθήκευση των εγγράφων του.
### **PresentationML στο Aspose.Slides for C++**
Τα έγγραφα OOXML PresentationML έρχονται ως αρχεία PPTX που είναι πακεταρισμένα XML αρχεία σε μορφή zip ακολουθώντας τις προδιαγραφές [OOXML ECMA-376](https://www.ecma-international.org/publications-and-standards/standards/ecma-376/). Το Aspose.Slides for C++ υποστηρίζει εκτενώς τη δημιουργία, ανάγνωση, επεξεργασία και εγγραφή εγγράφων PresentationML. Επιπλέον, το Aspose.Slides for C++ είναι ικανό να εξάγει έγγραφα PresentationML σε διαφορετικές ευρέως χρησιμοποιούμενες μορφές εγγράφων όπως PDF, TIFF και XPS. Αυτό είναι δυνατό επειδή το Aspose.Slides for C++ σχεδιάστηκε με στόχο την πλήρη διαχείριση εγγράφων παρουσίασης και το PresentationML ουσιαστικά περιέχει την εσωτερική παρουσίαση των εγγράφων ως πακέτο XML σε μορφή zip.

### **Το PresentationML είναι Ανοιχτό, Γιατί να Χρησιμοποιήσετε το Aspose.Slides for C++**
Δεδομένου ότι το PresentationML βασίζεται σε XML, είναι απολύτως εφικτό να δημιουργηθούν εφαρμογές για την επεξεργασία και παραγωγή εγγράφων PresentationML χρησιμοποιώντας κλάσεις XML χωρίς να εξαρτώνται από βιβλιοθήκες τρίτων όπως το Aspose.Slides for C++. Ωστόσο, υπάρχουν αρκετά πλεονεκτήματα στη χρήση του Aspose.Slides for C++ αντί των κλάσεων XML κατά την εργασία με έγγραφα PresentationML.

Η προδιαγραφή OOXML είναι πολύ εκτενής, με χιλίους σελίδες. Αυτό σημαίνει ότι για να διαχειριστείτε σωστά τα έγγραφα PresentationML θα πρέπει να επενδύσετε πολύ χρόνο και προσπάθεια για την κατανόηση της μορφής των εγγράφων. Από την άλλη, χρησιμοποιώντας το Aspose.Slides for C++, χρειάζεται μόνο να χρησιμοποιήσετε τις σχετικές κλάσεις και τις αντίστοιχες μεθόδους / ιδιότητές τους για την εκτέλεση λειτουργιών που θα φαίνονταν πολύπλοκες αν γίνονταν μέσω κλάσεων XML.

Οι παρακάτω είναι μερικές από τις δυνατότητες που δεν είναι διαθέσιμες όταν εργάζεστε με έγγραφα PresentationML μέσω κλάσεων XML:

- Εξαγωγή εγγράφων PPT σε μορφές PDF, TIFF, XPS.
- Εξαγωγή διαφανειών στα έγγραφα PPT σε μορφές SVG.
- Απόδοση διαφάνειας σε οποιαδήποτε μορφή εικόνας υποστηρίζεται από το C++ Framework.
- Αυτόματη αντιγραφή master από τις πηγαίες παρουσιάσεις χρησιμοποιώντας τη λειτουργία κλωνοποίησης.
- Εφαρμογή προστασίας σε σχήματα.

Ας πάρουμε ένα παράδειγμα εγγράφου PresentationML που έχει μία διαφάνεια με ένα πλαίσιο κειμένου που περιέχει το κείμενο “Hello World”. Για να διαβάσετε το κείμενο μέσω κλάσεων XML, θα πρέπει να γράψετε ένα πρόγραμμα που μπορεί να αναλύσει αυτό το απλό κείμενο από το ακόλουθο απόσπασμα:
``` cpp

 <?xml version="1.0" encoding="UTF-8" standalone="yes"?>

<p:sld xmlns:a="http://schemas.openxmlformats.org/drawingml/2006/main" xmlns:r="http://schemas.openxmlformats.org/officeDocument/2006/relationships" xmlns:p="http://schemas.openxmlformats.org/presentationml/2006/main">

  <p:cSld>

    <p:spTree>

      <p:nvGrpSpPr>

        <p:cNvPr id="1" name=""/>

        <p:cNvGrpSpPr/>

        <p:nvPr/>

      </p:nvGrpSpPr>

      <p:grpSpPr>

        <a:xfrm>

          <a:off x="0" y="0"/>

          <a:ext cx="0" cy="0"/>

          <a:chOff x="0" y="0"/>

          <a:chExt cx="0" cy="0"/>

        </a:xfrm></p:grpSpPr><p:sp>

          <p:nvSpPr><p:cNvPr id="4" name="TextBox 3"/>

          <p:cNvSpPr txBox="1"/>

            <p:nvPr/>

          </p:nvSpPr>

          <p:spPr>

            <a:xfrm>

              <a:off x="2819400" y="2590800"/>

              <a:ext cx="1297086" cy="369332"/>

            </a:xfrm>

            <a:prstGeom prst="rect">

              <a:avLst/>

            </a:prstGeom>

            <a:noFill/>

          </p:spPr>

          <p:txBody>

            <a:bodyPr wrap="none" rtlCol="0">

              <a:spAutoFit/>

            </a:bodyPr>

            <a:lstStyle/>

            <a:p>

              <a:r>

                <a:rPr lang="en-US"/>

                <a:t>Hello World

                </a:t>

              </a:r>

              <a:endParaRPr lang="en-US"/>

            </a:p>

          </p:txBody>

        </p:sp>

    </p:spTree>

  </p:cSld>

  <p:clrMapOvr>

    <a:masterClrMapping/>

  </p:clrMapOvr>

</p:sld>

```
## **Μετατροπή PPT σε PPTX**
### **Σχετικά με τη Μετατροπή**
Το Aspose.Slides τώρα υποστηρίζει επίσης τη μετατροπή PPT σε PPTX.
### **Χαρακτηριστικά που Υποστηρίζονται στη Μετατροπή**
Το Aspose.Slides for C++ παρέχει μερική υποστήριξη για τη μετατροπή παρουσιάσεων σε μορφή αρχείου PPT σε παρουσιάσεις μορφής PPTX. Καθώς η υποστήριξη για τη συγκεκριμένη δυνατότητα μετατροπής παρουσιάσεων μόλις εισήχθη στο Aspose.Slides for C++, αυτή τη στιγμή έχει περιορισμένη δυνατότητα και λειτουργεί μόνο για απλές παρουσιάσεις. Το κύριο πλεονέκτημα που παρέχει η βιβλιοθήκη API Aspose.Slides for C++ για τη μετατροπή παρουσίασης PPT σε μορφή PPTX είναι η ευκολία χρήσης του API για την επίτευξη του επιθυμητού αποτελέσματος. Παρακαλούμε προχωρήστε στο this[link]() για την ενότητα αποσ fragment κώδικα για περισσότερες λεπτομέρειες. Η ακόλουθη ενότητα δείχνει σαφώς ποια χαρακτηριστικά υποστηρίζονται και ποια δεν υποστηρίζονται κατά τη μετατροπή παρουσιάσεων μορφής PPT σε μορφή PPTX.
### **Υποστηριζόμενα Χαρακτηριστικά**
Τα παρακάτω χαρακτηριστικά υποστηρίζονται κατά τη μετατροπή:

- Μετατροπή της δομής των master, προσαρμογών (layouts) και διαφανειών
- Μετατροπή της δομής των master, προσαρμογών (layouts) και διαφανειών
- Μετατροπή διαγραμμάτων
- Ομαδοποίηση σχημάτων
- Μετατροπή αυτόματων σχημάτων (Auto-shapes) συμπεριλαμβανομένων τετραγώνων και ελλείψεων. Ωστόσο, ενδέχεται τα Auto-shapes να έχουν λανθασμένες τιμές ρυθμίσεων.
- Σχήματα με προσαρμοσμένη γεωμετρία. Μερικές φορές ενδέχεται να μην μετατραπούν.
- Υφές και στυλ γεμίσματος εικόνων για Auto-shapes. Μερικές φορές ενδέχεται να μην μετατραπούν.
- Μετατροπή θέσεων κράτησης (Placeholders).
- Μετατροπή κειμένου σε πλαίσια κειμένου και υποδοχείς κειμένου. Ωστόσο, τα σημεία, η στοίχιση και οι εσοχές δεν έχουν πλήρως υλοποιηθεί.
### **Μη Υποστηριζόμενα Χαρακτηριστικά**
Τα παρακάτω χαρακτηριστικά δεν υποστηρίζονται κατά τη μετατροπή:

- Διαφάνεια με σημειώσεις, καθώς η ανάγνωση σημειώσεων δεν έχει υλοποιηθεί σε PPTX. Σε περίπτωση που το PPT τις έχει, δεν μπορεί ακόμη να αποθηκευτεί ως PPTX* Μετατροπή γραμμών και πολυγραμμών
- Μορφές γραμμής και γεμίσματος
- Στυλ διαβάθμισης γεμίσματος
- Πλαίσια OLE, Πίνακες, Βίντεο και Ηχητικά πλαίσια κλπ
- Η κίνηση και άλλες ιδιότητες παρουσίασης παραλείπονται

Νέα ή ελλείποντα χαρακτηριστικά θα προστεθούν σε μεταγενέστερες εκδόσεις του Aspose.Slides for C++.

## **Μορφή Φορητού Εγγράφου (PDF)**
### **Σχετικά με το PDF**
Η [Μορφή Φορητού Εγγράφου](https://en.wikipedia.org/wiki/PDF) είναι μια μορφή αρχείου που δημιουργήθηκε από την Adobe Systems για την ανταλλαγή εγγράφων μεταξύ διαφορετικών οργανώσεων. Σκοπός αυτής της μορφής είναι να επιτρέπει το περιεχόμενο των εγγράφων να εμφανίζεται με τον ίδιο τρόπο ανεξάρτητα από την πλατφόρμα προβολής.
### **PDF στο Aspose.Slides for C++**
Οποιοδήποτε έγγραφο παρουσίασης που μπορεί να φορτωθεί στο Aspose.Slides for C++ μπορεί να μετατραπεί σε έγγραφο PDF το οποίο μπορεί να συμμορφώνεται με το [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A) ή το [PDF /A-1b](https://en.wikipedia.org/wiki/PDF/A) ανάλογα με την επιλογή σας. Το Aspose.Slides for C++ εξάγει τα έγγραφα παρουσίασης σε PDF με τέτοιο τρόπο ώστε στις περισσότερες περιπτώσεις το εξαγόμενο PDF να μοιάζει σχεδόν με το αρχικό έγγραφο παρουσίασης. Η λύση Aspose υποστηρίζει τις ακόλουθες δυνατότητες των εγγράφων παρουσίασης κατά τη μετατροπή σε PDF:

- Εικόνες, Πλαίσια Κειμένου και άλλα Σχήματα
- Κείμενο και Μορφοποίηση
- Παράγραφοι και Μορφοποίηση
- Υπερσύνδεσμοι
- Κεφαλίδες και Υποσέλιδα
- Κουκίδες
- Πίνακες

Μπορείτε να εξάγετε τα έγγραφα παρουσίασης σε έγγραφα PDF απευθείας χρησιμοποιώντας το στοιχείο Aspose.Slides for C++. Δηλαδή, δεν χρειάζεται κανένα άλλο τρίτο ή στοιχείο Aspose.Pdf για αυτόν τον σκοπό. Επιπλέον, μπορείτε να προσαρμόσετε την εξαγωγή παρουσίασης σε PDF με διαφορετικές επιλογές όπως εξηγείται στο [this topic](/slides/el/cpp/convert-powerpoint-to-pdf/).

## **Προδιαγραφή Αναλυτή XML (XPS)**
### **Σχετικά με το XPS**
Η [Προδιαγραφή Αναλυτή XML](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) είναι μια γλώσσα περιγραφής σελίδας και μια μορφή σταθερού εγγράφου που αρχικά αναπτύχθηκε από τη Microsoft. Όπως το PDF, το XPS είναι μια μορφή εγγράφου σταθερής διάταξης σχεδιασμένη να διατηρεί την ακρίβεια του εγγράφου και να παρέχει εμφάνιση ανεξάρτητη από τη συσκευή.
### **XPS στο Aspose.Slides for C++**
Οποιοδήποτε έγγραφο παρουσίασης που μπορεί να φορτωθεί από το Aspose.Slides for C++ μπορεί να μετατραπεί σε μορφή XPS. Το Aspose.Slides for C++ χρησιμοποιεί την υψηλής πιστότητας μηχανή διάταξης σελίδας και απόδοσης για την παραγωγή εξόδου σε μορφή XPS σταθερής διάταξης. Αξίζει να σημειωθεί ότι το Aspose.Slides for C++ δημιουργεί άμεσα XPS χωρίς να εξαρτάται από τις κλάσεις Windows Presentation Foundation (WPF) που περιλαμβάνονται στο C++ Framework 3.5, επιτρέποντας έτσι στο Aspose.Slides for C++ να παράγει έγγραφα XPS σε μηχανήματα που τρέχουν εκδόσεις του C++ Framework παλιότερες της 3.5. Μπορείτε να μάθετε για την εξαγωγή εγγράφων παρουσίασης σε έγγραφα XPS μέσω Aspose.Slides for C++ στο [this topic](https://docs.aspose.com/slides/el/cpp/convert-powerpoint-to-xps/).
