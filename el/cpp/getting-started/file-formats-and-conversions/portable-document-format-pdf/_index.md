## **Σχετικά με το PDF**
Το [Portable Document Format](https://en.wikipedia.org/wiki/PDF) είναι μια μορφή αρχείου που δημιουργήθηκε από την Adobe System για ανταλλαγή εγγράφων μεταξύ διαφορετικών οργανισμών. Ο σκοπός αυτής της μορφής ήταν να καταστήσει δυνατή η αναπαράσταση του περιεχομένου των εγγράφων με τέτοιο τρόπο ώστε η οπτική τους εμφάνιση να μην εξαρτάται από την πλατφόρμα στην οποία προβολίζεται.

## **PDF στο Aspose.Slides για C++**
Οποιοδήποτε έγγραφο παρουσίασης που μπορεί να φορτωθεί στο Aspose.Slides για C++ μπορεί να μετατραπεί σε έγγραφο PDF το οποίο μπορεί να συμμορφώνεται με [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A) ή [PDF /A-1b](https://en.wikipedia.org/wiki/PDF/A) ανάλογα με την επιλογή σας. Το Aspose.Slides για C++ εξάγει τα έγγραφα παρουσίασης σε PDF με τέτοιο τρόπο ώστε, στις περισσότερες περιπτώσεις, το εξαχθέν έγγραφο PDF να μοιάζει σχεδόν με το αρχικό έγγραφο παρουσίασης.

Η λύση Aspose υποστηρίζει τις ακόλουθες δυνατότητες των εγγράφων παρουσίασης κατά τη μετατροπή σε έγγραφα PDF:

- Εικόνες, Πλαίσια Κειμένου και άλλα Σχήματα
- Κείμενο και Μορφοποίηση
- Παράγραφοι και Μορφοποίηση
- Υπερσυνδέσεις
- Κεφαλίδες και Υποσέλιδα
- Κουκκίδες
- Πίνακες

Μπορείτε να εξάγετε τα έγγραφα παρουσίασης σε έγγραφα PDF απευθείας χρησιμοποιώντας μόνο το στοιχείο Aspose.Slides για C++. Δηλαδή, δεν χρειάζεστε κανένα άλλο τρίτο μέρος ή το στοιχείο Aspose.Pdf για αυτό το σκοπό. Επιπλέον, μπορείτε να προσαρμόσετε την εξαγωγή παρουσίασης σε PDF με διαφορετικές επιλογές όπως εξηγείται σε [αυτό το θέμα](http://docs.aspose.com/display/slidesnet/Converting+to+PDF+File).
