## **Σκοπός**
Η απόδοση είναι συχνά ο πρώτος κρίσιμος παράγοντας κατά την επιλογή ενός στοιχείου. Αυτό το άρθρο μετρά την απόδοση του Aspose.Slides for C++ και του VSTO 2008. Οι απλές δοκιμές διεξάγονται σε παρόμοιο σύστημα λειτουργίας, υλικά στοιχεία και ρυθμίσεις.  

Αυτό το άρθρο παρουσιάζει μετρήσεις απόδοσης για προϊόντα που περιλαμβάνουν **Aspose.Slides for C++** και **VSTO 2008**. Οι εκτιμήσεις απόδοσης που παρουσιάζονται εδώ προορίζονται να σας βοηθήσουν να κατανοήσετε τι να περιμένετε από διαφορετικά στοιχεία σε κάποιες συχνά χρησιμοποιούμενες περιπτώσεις υπό παρόμοιες ρυθμίσεις σε κοινό υλικό που λειτουργεί σε ευρέως χρησιμοποιούμενα λειτουργικά συστήματα. Φυσικά, η απόδοση της εφαρμογής σας εξαρτάται από τα δεδομένα σας, τα πρότυπα πρόσβασης δεδομένων, το μέγεθος της κρυφής μνήμης, άλλες παραμέτρους ρύθμισης, το λειτουργικό σύστημα και το υλικό κ.λπ. Το benchmark στοχεύει να δείξει πώς τα στοιχεία λειτουργούν υπό ελάχιστες συνθήκες υλικού· όσο πιο γρήγορο είναι το υλικό, τόσο πιο γρήγορα θα επεξεργάζονται τα καθήκοντα τα στοιχεία. 
## **Δήλωση**
Αυτό το έγγραφο παρέχεται μόνο για ενημερωτικούς σκοπούς και το περιεχόμενό του ενδέχεται να αλλάξει χωρίς προειδοποίηση. Αυτό το έγγραφο δεν εγγυάται ότι είναι απαλλαγμένο από σφάλματα, ούτε υπόκειται σε καμία άλλη εγγύηση ή προϋπόθεση, είτε εκφρασμένη προφορικά είτε ενσωματωμένη στο νόμο, συμπεριλαμβανομένων των ενσωματωμένων εγγυήσεων και προϋποθέσεων εμπορευσιμότητας ή καταλληλότητας για συγκεκριμένο σκοπό. Αποποιούμαστε σαφώς οποιαδήποτε ευθύνη σε σχέση με αυτό το έγγραφο και δεν δημιουργούνται συμβατικές υποχρεώσεις ούτε άμεσα ούτε έμμεσα από αυτό το έγγραφο. Αυτό το έγγραφο δεν επιτρέπεται να αναπαραχθεί ή να μεταδοθεί με οποιονδήποτε τρόπο, ηλεκτρονικό ή μηχανογραφικό, για οποιονδήποτε σκοπό. 

{{% alert color="primary" %}} 
Τα benchmark παρέχουν κατευθύνσεις και βοηθούν στον καθορισμό βασικών λειτουργικών προσδοκιών. Το θέμα παρουσιάζει τα benchmark τεστ που έχουν διεξαχθεί εναντίον του Aspose.Slides for C++ και του VSTO 2008. Τα Performance Measures *{*} επιτρέπουν ακόμη και σε αρχάριους χρήστες να αξιολογήσουν την απόδοση του στοιχείου που χρησιμοποιούν. Τα τεστ θα *{*} σας επιτρέψουν να αξιολογήσετε αντικειμενικά ένα στοιχείο χρησιμοποιώντας μια ποικιλία διαφορετικών τεστ ταχύτητας. Όλες οι εργασίες είναι κοινές και επιλεγμένες προσεκτικά, εξερευνώντας τα σχετικά χαρακτηριστικά ώστε να διασφαλιστεί ότι και τα δύο στοιχεία μπορούν να ολοκληρώσουν τις εργασίες εύκολα. Επιπλέον, τα API για την πραγματοποίηση ενός τεστ από κάθε στοιχείο έχουν επιλεγεί προσεκτικά ώστε να επιτευχθούν τα καλύτερα δυνατά αποτελέσματα που μπορεί να παράγει ένα στοιχείο κατά την αξιολόγηση της απόδοσής του, και όλες οι εργασίες υλοποιήθηκαν δύο ή τρεις φορές για καλύτερη εκτίμηση των στοιχείων. 
{{% /alert %}} 
## **Μεθοδολογία Δοκιμών**
Όλες οι δοκιμές απόδοσης διεξήχθησαν σε κοινό συνδυασμό υλικού και λειτουργικού συστήματος, χωρίς προσαρμοσμένες ρυθμίσεις, βελτιστοποίηση ή οποιεσδήποτε άλλες τεχνικές βελτίωσης της απόδοσης. Όλα τα τεστ εκτελέστηκαν με τις εγκαταστάσεις των στοιχείων στο ίδιο σύστημα, που διαφορετικά παρέμενε αδρανές. Για να ληφθούν ακριβείς μετρήσεις, εκτελέσαμε όλες τις εργασίες δύο ή τρεις φορές ταυτόχρονα ώστε να αξιολογηθεί καλύτερα ένα στοιχείο και να ληφθούν ακριβείς μετρήσεις.
