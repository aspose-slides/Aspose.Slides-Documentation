## **Επισκόπηση**

Όταν μια παρουσίαση φορτώνεται, αποδίδεται ή μετατρέπεται σε άλλη μορφή, το Aspose.Slides ελέγχει αν οι γραμματοσειρές που χρησιμοποιούνται στην παρουσίαση είναι διαθέσιμες στο λειτουργικό σύστημα. Εάν λείπει μια απαιτούμενη γραμματοσειρά, το Aspose.Slides επιλέγει μια αντικατάσταση γραμματοσειράς που είναι όσο πιο κοντά γίνεται σε αυτή που θα χρησιμοποίησε το PowerPoint.

Το Aspose.Slides πρώτα αναζητά τη συγκεκριμένη γραμματοσειρά στο λειτουργικό σύστημα. Εάν η γραμματοσειρά βρεθεί, χρησιμοποιείται. Εάν δεν βρεθεί, εφαρμόζεται μια κατάλληλη αντικατάσταση. Όταν οι κανόνες αντικατάστασης γραμματοσειρών ορίζονται μέσω `FontSubstRule`, αυτοί οι κανόνες λαμβάνονται επίσης υπόψη.

Μπορείτε επίσης να προσθέσετε γραμματοσειρές κατά την εκτέλεση της εφαρμογής, να χρησιμοποιήσετε ενσωματωμένες γραμματοσειρές από μια παρουσίαση ή να φορτώσετε εξωτερικές γραμματοσειρές για έγγραφα εξόδου όπως αρχεία PDF.

## **Επιλογή Γραμματοσειράς**

Ορισμένοι κανόνες εφαρμόζονται στις γραμματοσειρές μιας παρουσίασης όταν η παρουσίαση φορτώνεται, αποδίδεται ή μετατρέπεται σε άλλη μορφή. Για παράδειγμα, όταν προσπαθείτε να μετατρέψετε μια παρουσίαση (τις διαφάνειές της) σε εικόνες, οι γραμματοσειρές της παρουσίασης ελέγχονται για να επαληθευτεί ότι οι επιλεγμένες γραμματοσειρές είναι διαθέσιμες στο λειτουργικό σύστημα. Εάν διαπιστωθεί ότι οι γραμματοσειρές λείπουν, αντικαθίστανται — δείτε [**Αντικατάσταση Γραμματοσειράς**](https://docs.aspose.com/slides/el/cpp/font-replacement/) και [**Υποκατάσταση Γραμματοσειράς**](https://docs.aspose.com/slides/el/cpp/font-substitution/).

Αυτή είναι η διαδικασία που ακολουθεί το Aspose.Slides κατά τη διαχείριση των γραμματοσειρών:

1. Το Aspose.Slides αναζητά γραμματοσειρές στο λειτουργικό σύστημα για να βρει τη γραμματοσειρά που ταιριάζει με τη γραμματοσειρά που επιλέχθηκε στην παρουσίαση. 
2. Εάν η επιλεγμένη γραμματοσειρά βρεθεί, το Aspose.Slides τη χρησιμοποιεί. Διαφορετικά, το Aspose.Slides χρησιμοποιεί μια γραμματοσειρά αντικατάστασης που είναι όσο πιο κοντά γίνεται σε αυτή που θα χρησιμοποιούσε το PowerPoint.
3. Εάν έχουν οριστεί κανόνες αντικατάστασης γραμματοσειρών μέσω [FontSubstRule](https://reference.aspose.com/slides/el/cpp/aspose.slides/fontsubstrule/), εφαρμόζονται. 

Το Aspose.Slides σας επιτρέπει να προσθέσετε γραμματοσειρές κατά την εκτέλεση της εφαρμογής και στη συνέχεια να τις χρησιμοποιήσετε. Δείτε [**Προσαρμοσμένες γραμματοσειρές**](https://docs.aspose.com/slides/el/cpp/custom-font/). 

Όταν προστεθούν επιπλέον γραμματοσειρές μέσα σε μια παρουσίαση, αυτές ονομάζονται [**Ενσωματωμένες γραμματοσειρές**](https://docs.aspose.com/slides/el/cpp/embedded-font/).

Το Aspose.Slides σας επιτρέπει να προσθέσετε γραμματοσειρές που εφαρμόζονται *μόνο* σε έγγραφα εξόδου. Για παράδειγμα, εάν μια παρουσίαση που θέλετε να μετατρέψετε σε PDF περιέχει γραμματοσειρές που λείπουν από το σύστημά σας και ενσωματωμένες γραμματοσειρές, μπορείτε να προσθέσετε ή να φορτώσετε τις απαραίτητες γραμματοσειρές ως **εξωτερικές γραμματοσειρές**. 

{{% alert title="Note" color="primary" %}} 
Δεν διανέμουμε καμία γραμματοσειρά, είτε πληρωμένη είτε δωρεάν. Το API μας επιτρέπει να φορτώνετε εξωτερικές γραμματοσειρές και να τις ενσωματώνετε σε έγγραφα, αλλά το κάνετε με τις γραμματοσειρές κατά τη δική σας κρίση και ευθύνη.
{{% /alert %}}

## **Συχνές ερωτήσεις**

**Πώς μπορώ να προσδιορίσω ποιες γραμματοσειρές χρησιμοποιούνται πραγματικά σε μια παρουσίαση πριν από τη μετατροπή;**

Το Aspose.Slides σας επιτρέπει να εξετάσετε τις χρησιμοποιούμενες γραμματοσειρές μέσω του [διαχειριστή γραμματοσειρών](https://reference.aspose.com/slides/el/cpp/aspose.slides/presentation/get_fontsmanager/), ώστε να μπορέσετε να αποφασίσετε αν θα [ενσωμάτωση](/slides/el/cpp/embedded-font/), [αντικατάσταση](/slides/el/cpp/font-replacement/) ή προσθέσετε [εξωτερικές πηγές](/slides/el/cpp/custom-font/). Αυτό σας βοηθά να αποτρέψετε ανεπιθύμητες αντικαταστάσεις κατά την απόδοση και την εξαγωγή.

**Μπορώ να προσθέσω επιπλέον καταλόγους γραμματοσειρών χωρίς να τους εγκαταστήσω στο λειτουργικό σύστημα;**

Ναι. Μπορείτε να καταχωρήσετε [εξωτερικές πηγές γραμματοσειρών](/slides/el/cpp/custom-font/) όπως φάκελοι ή ροές στη μνήμη για απόδοση και εξαγωγή. Αυτό αφαιρεί την εξάρτηση από τις γραμματοσειρές του συστήματος υποδοχής και διατηρεί τη διάταξη προβλέψιμη.

**Πώς μπορώ να αποτρέψω μια σιωπηρή εναλλακτική σε μια ακατάλληλη γραμματοσειρά όταν λείπει ένα γλύφιο;**

Ορίστε ρητά [αντικατάσταση γραμματοσειράς](/slides/el/cpp/font-replacement/) και κανόνες [εναλλακτικών γραμματοσειρών](/slides/el/cpp/fallback-font/) εκ των προτέρων. Αναλύοντας τις χρησιμοποιούμενες γραμματοσειρές και θέτοντας ελεγχόμενη προτεραιότητα για τις εναλλακτικές, εξασφαλίζετε συνεπή τυπογραφία και αποφεύγετε ανεπιθύμητα αποτελέσματα.
