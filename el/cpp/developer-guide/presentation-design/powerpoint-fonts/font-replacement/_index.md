## **Επισκόπηση**

Το Aspose.Slides σάς επιτρέπει να αντικαταστήσετε μια γραμματοσειρά με άλλη σε ολόκληρη την παρουσίαση. Όταν αντικαθίσταται μια γραμματοσειρά, όλες οι εμφανίσεις της αρχικής γραμματοσειράς αλλάζουν στη νέα γραμματοσειρά.

Για να εκτελέσετε αντικατάσταση γραμματοσειράς, φορτώστε την παρουσίαση, ορίστε τη γραμματοσειρά προέλευσης και τη γραμματοσειρά αντικατάστασης, καλέστε τη μέθοδο αντικατάστασης γραμματοσειράς και αποθηκεύστε την τροποποιημένη παρουσίαση ως αρχείο PPTX. Αυτή η προσέγγιση είναι χρήσιμη όταν θέλετε εσκεμμένα να μεταβείτε από μία οικογένεια γραμματοσειρών σε άλλη σε όλη την παρουσίαση.

## **Αντικατάσταση Γραμματοσειρών**

Εάν αλλάξετε γνώμη σχετικά με τη χρήση μιας γραμματοσειράς, μπορείτε να την αντικαταστήσετε με άλλη. Όλες οι εμφανίσεις της παλιάς γραμματοσειράς θα αντικατασταθούν από τη νέα.

Το Aspose.Slides σάς επιτρέπει να αντικαταστήσετε μια γραμματοσειρά με αυτόν τον τρόπο:

1. Φορτώστε τη σχετική παρουσίαση. 
2. Φορτώστε τη γραμματοσειρά που θα αντικατασταθεί. 
3. Φορτώστε τη νέα γραμματοσειρά. 
4. Αντικαταστήστε τη γραμματοσειρά. 
5. Αποθηκεύστε την τροποποιημένη παρουσίαση ως αρχείο PPTX.

Αυτός ο κώδικας C++ δείχνει την αντικατάσταση γραμματοσειράς:

``` cpp
// Φορτώνει μια παρουσίαση
auto presentation = System::MakeObject<Presentation>(u"Fonts.pptx");

// Φορτώνει τη γραμματοσειρά προέλευσης που θα αντικατασταθεί
auto sourceFont = System::MakeObject<FontData>(u"Arial");

// Φορτώνει τη νέα γραμματοσειρά
auto destFont = System::MakeObject<FontData>(u"Times New Roman");

// Αντικαθιστά τις γραμματοσειρές
presentation->get_FontsManager()->ReplaceFont(sourceFont, destFont);

// Αποθηκεύει την παρουσίαση
presentation->Save(u"UpdatedFont_out.pptx", SaveFormat::Pptx);
```

Για να ορίσετε κανόνες που καθορίζουν τι συμβαίνει σε ορισμένες καταστάσεις (π.χ. εάν μια γραμματοσειρά δεν είναι προσβάσιμη), δείτε [**Αντικατάσταση Γραμματοσειράς**](/slides/el/cpp/font-substitution/).

## **FAQ**

**Ποια είναι η διαφορά μεταξύ "αντικατάστασης γραμματοσειράς", "υποκατάστασης γραμματοσειράς" και "εφεδρικών γραμματοσειρών";**

Η αντικατάσταση είναι σκόπιμη εναλλαγή από μία οικογένεια γραμματοσειρών σε άλλη σε όλο το έγγραφο. Η [Υποκατάσταση](/slides/el/cpp/font-substitution/) είναι ένας κανόνας του τύπου «αν η γραμματοσειρά δεν είναι διαθέσιμη, χρησιμοποιήστε τη X». Η [Εφεδρική](/slides/el/cpp/fallback-font/) εφαρμόζεται επιλεκτικά για μεμονωμένα ελλειπτικά γλύφους όταν η βασική γραμματοσειρά είναι εγκατεστημένη αλλά δεν περιέχει τους απαιτούμενους χαρακτήρες.

**Ισχύει η αντικατάσταση για τις κύριες διαφάνειες, τις διατάξεις, τις σημειώσεις και τα σχόλια;**

Ναι. Η αντικατάσταση επηρεάζει όλα τα αντικείμενα της παρουσίασης που χρησιμοποιούν την αρχική γραμματοσειρά, συμπεριλαμβανομένων των κύριων διαφανειών και των σημειώσεων· τα σχόλια επίσης αποτελούν μέρος του εγγράφου και λαμβάνονται υπόψη από τη μηχανή γραμματοσειρών.

**Θα αλλάξει η γραμματοσειρά μέσα σε ενσωματωμένα αντικείμενα OLE (π.χ., Excel);**

Όχι. Το [περιεχόμενο OLE](/slides/el/cpp/manage-ole/) ελέγχεται από την δική του εφαρμογή. Η αντικατάσταση στην παρουσίαση δεν επανεπεξεργάζεται τα εσωτερικά δεδομένα OLE· ενδέχεται να εμφανίζεται ως εικόνα ή ως εξωτερικά επεξεργάσιμο περιεχόμενο.

**Μπορώ να αντικαταστήσω μια γραμματοσειρά μόνο σε μέρος της παρουσίασης (ανά διαφάνεια ή περιοχές);**

Η στοχευμένη αντικατάσταση είναι εφικτή εάν αλλάξετε τη γραμματοσειρά στο επίπεδο των απαιτούμενων αντικειμένων/εύρους αντί να εφαρμόσετε παγκόσμια αντικατάσταση σε ολόκληρο το έγγραφο. Η λογική επιλογής γραμματοσειράς κατά τη σχεδίαση παραμένει η ίδια.

**Πώς μπορώ να προσδιορίσω εκ των προτέρων ποιες γραμματοσειρές χρησιμοποιεί η παρουσίαση;**

Χρησιμοποιήστε το [διαχειριστής γραμματοσειρών](https://reference.aspose.com/slides/el/cpp/aspose.slides/fontsmanager/) της παρουσίασης: παρέχει μια λίστα των [οικογενειών σε χρήση](https://reference.aspose.com/slides/el/cpp/aspose.slides/fontsmanager/getfonts/) και πληροφορίες για τις [υποκαταστάσεις/«άγνωστες» γραμματοσειρές](https://reference.aspose.com/slides/el/cpp/aspose.slides/fontsmanager/getsubstitutions/), που βοηθούν στον προγραμματισμό της αντικατάστασης.

**Λειτουργεί η αντικατάσταση γραμματοσειράς κατά τη μετατροπή σε PDF/εικόνες;**

Ναι. Κατά την εξαγωγή, το Aspose.Slides εφαρμόζει την ίδια [ακολουθία επιλογής/υποκατάστασης γραμματοσειράς](/slides/el/cpp/font-selection-sequence/), έτσι ώστε μια προκαταβολική αντικατάσταση να τηρείται κατά τη μετατροπή.

**Πρέπει να εγκαταστήσω τη γραμματοσειρά-στόχο στο σύστημα ή μπορώ να συνδέσω έναν φάκελο γραμματοσειρών;**

Η εγκατάσταση δεν απαιτείται: η βιβλιοθήκη επιτρέπει τη [φόρτωση εξωτερικών γραμματοσειρών](/slides/el/cpp/custom-font/) από φακέλους χρήστη για χρήση κατά το [σχεδιασμό και την εξαγωγή](/slides/el/cpp/convert-powerpoint/).

**Θα διορθώσει η αντικατάσταση το πρόβλημα των «τοφού» (τετράγωνα) αντί για χαρακτήρες;**

Μόνο εάν η γραμματοσειρά-στόχος περιέχει πραγματικά τα απαραίτητα γλύφα. Εάν όχι, [ρυθμίστε την εφεδρική](/slides/el/cpp/fallback-font/) για να καλύψετε τους ελλειπούσες χαρακτήρες.
