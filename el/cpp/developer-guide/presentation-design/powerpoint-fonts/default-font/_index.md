## **Επισκόπηση**

Το Aspose.Slides σας επιτρέπει να καθορίσετε προεπιλεγμένη γραμματοσειρές που χρησιμοποιούνται όταν μια παρουσίαση αποδίδεται. Αυτό είναι χρήσιμο κατά τη δημιουργία μικρογραφιών διαφανειών ή την εξαγωγή μιας παρουσίασης σε μορφές όπως PDF και XPS. Οι προεπιλεγμένες γραμματοσειρές ρυθμίζονται μέσω `LoadOptions` πριν φορτωθεί η παρουσίαση.

Η μέθοδος `set_DefaultRegularFont` ορίζει τη προεπιλεγμένη γραμματοσειρά για κανονικό κείμενο, ενώ η `set_DefaultAsianFont` ορίζει τη προεπιλεγμένη γραμματοσειρά για ασιατικό κείμενο. Αφού οριστούν αυτές οι επιλογές, η παρουσίαση μπορεί να φορτωθεί και να αποδοθεί χρησιμοποιώντας τις καθορισμένες γραμματοσειρές.

## **Χρήση προεπιλεγμένων γραμματοσειρών για απόδοση παρουσίασης**
Το Aspose.Slides σας επιτρέπει να ορίσετε την προεπιλεγμένη γραμματοσειρά για την απόδοση της παρουσίασης σε PDF, XPS ή μικρογραφίες. Αυτό το άρθρο δείχνει πώς να ορίσετε το DefaultRegular Font και το DefaultAsian Font για χρήση ως προεπιλεγμένες γραμματοσειρές. Παρακαλώ ακολουθήστε τα παρακάτω βήματα για τη φόρτωση γραμματοσειρών από εξωτερικούς καταλόγους χρησιμοποιώντας το Aspose.Slides for C++ API:

1. Δημιουργήστε ένα στιγμιότυπο του LoadOptions.
1. Ορίστε το DefaultRegularFont στη γραμματοσειρά της επιλογής σας. Στο παρακάτω παράδειγμα, χρησιμοποίησα το Wingdings.
1. Ορίστε το DefaultAsianFont στη γραμματοσειρά της επιλογής σας. Χρησιμοποίησα το Wingdings στο παρακάτω παράδειγμα.
1. Φορτώστε την παρουσίαση χρησιμοποιώντας το Presentation και ορίζοντας τις επιλογές φόρτωσης.
1. Τώρα, δημιουργήστε τη μικρογραφία της διαφάνειας, το PDF και το XPS για να επαληθεύσετε τα αποτελέσματα.

```cpp
// Χρησιμοποιήστε τις επιλογές φόρτωσης για να καθορίσετε τις προεπιλεγμένες κανονικές και ασιατικές γραμματοσειρές
auto loadOptions = MakeObject<LoadOptions>(LoadFormat::Auto);
loadOptions->set_DefaultRegularFont(u"Wingdings");
loadOptions->set_DefaultAsianFont(u"Wingdings");

auto pptx = MakeObject<Presentation>(u"DefaultFonts.pptx", loadOptions);

auto image = pptx->get_Slide(0)->GetImage(1, 1);
image->Save(u"DefaultFonts_out.png", ImageFormat::Png);
image->Dispose();

pptx->Save(u"DefaultFonts_out.pdf", SaveFormat::Pdf);
pptx->Save(u"DefaultFonts_out.xps", SaveFormat::Xps);

pptx->Dispose();
```

## **Συχνές ερωτήσεις**

**Τι ακριβώς επηρεάζουν οι DefaultRegularFont και DefaultAsianFont—μόνο η εξαγωγή, ή επίσης μικρογραφίες, PDF, XPS, HTML και SVG;**

Συμμετέχουν στη διαδικασία απόδοσης για όλες τις υποστηριζόμενες εξόδους. Αυτό περιλαμβάνει μικρογραφίες διαφανειών, [PDF](/slides/el/cpp/convert-powerpoint-to-pdf/), [XPS](/slides/el/cpp/convert-powerpoint-to-xps/), [εικονογραφίες raster](/slides/el/cpp/convert-powerpoint-to-png/), [HTML](/slides/el/cpp/convert-powerpoint-to-html/), και [SVG](/slides/el/cpp/render-a-slide-as-an-svg-image/), επειδή το Aspose.Slides χρησιμοποιεί την ίδια λογική διάταξης και επίλυσης γλύφων σε όλους αυτούς τους στόχους.

**Εφαρμόζονται οι προεπιλεγμένες γραμματοσειρές όταν απλώς διαβάζετε και αποθηκεύετε ένα PPTX χωρίς καμία απόδοση;**

Όχι. Οι προεπιλεγμένες γραμματοσειρές έχουν σημασία όταν το κείμενο πρέπει να μετρηθεί και να σχεδιαστεί. Μια απλή ενέργεια ανοίγματος‑αποθήκευσης μιας παρουσίασης δεν αλλάζει τις αποθηκευμένες ακολουθίες γραμματοσειρών ή τη δομή του αρχείου. Οι προεπιλεγμένες γραμματοσειρές παρεμβάλλονται κατά τις λειτουργίες που αποδίδουν ή επανατοποθετούν το κείμενο.

**Αν προσθέσω δικούς μου φακέλους γραμματοσειρών ή παρέχω γραμματοσειρές από μνήμη, θα ληφθούν υπόψη κατά την επιλογή προεπιλεγμένων γραμματοσειρών;**

Ναι. [Custom font sources](/slides/el/cpp/custom-font/) επεκτείνουν τον κατάλογο των διαθέσιμων οικογενειών και γλύφων που μπορεί να χρησιμοποιήσει η μηχανή. Οι προεπιλεγμένες γραμματοσειρές και τυχόν [fallback rules](/slides/el/cpp/fallback-font/) θα επιλύονται με βάση αυτές τις πηγές πρώτα, προσφέροντας πιο αξιόπιστη κάλυψη σε διακομιστές και σε containers.

**Θα επηρεάσουν οι προεπιλεγμένες γραμματοσειρές τις μετρικές του κειμένου (kerning, advances) και κατά συνέπεια τις αλλαγές γραμμής και την αναδίπλωση;**

Ναι. Η αλλαγή της γραμματοσειράς αλλάζει τις μετρικές των γλύφων και μπορεί να τροποποιήσει τις αλλαγές γραμμής, την αναδίπλωση και την σελιδοποίηση κατά την απόδοση. Για σταθερότητα διάταξης, [embed the original fonts](/slides/el/cpp/embedded-font/) ή επιλέξτε προεπιλεγμένες και εναλλακτικές οικογένειες συμβατές μετρικά.

**Υπάρχει κάποιος λόγος για τον ορισμό προεπιλεγμένων γραμματοσειρών αν όλες οι γραμματοσειρές που χρησιμοποιούνται στην παρουσίαση είναι ενσωματωμένες;**

Συχνά δεν είναι απαραίτητο, επειδή [embedded fonts](/slides/el/cpp/embedded-font/) ήδη εξασφαλίζουν συνεπή εμφάνιση. Οι προεπιλεγμένες γραμματοσειρές εξακολουθούν να είναι χρήσιμες ως εφεδρικό μέτρο για χαρακτήρες που δεν καλύπτονται από το ενσωματωμένο υποσύνολο ή όταν ένα αρχείο συνδυάζει ενσωματωμένο και μη ενσωματωμένο κείμενο.
