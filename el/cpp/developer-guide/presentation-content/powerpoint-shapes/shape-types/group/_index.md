## **Επισκόπηση**

Αυτό το άρθρο εξηγεί πώς να εργάζεστε με ομαδικά σχήματα στο Aspose.Slides. Δείχνει πώς να προσθέσετε ένα ομαδικό σχήμα σε μια διαφάνεια, να τοποθετήσετε σχήματα μέσα του και να αποθηκεύσετε την ενημερωμένη παρουσίαση. Επίσης, παρουσιάζει πώς να αποκτήσετε πρόσβαση σε σχήματα που είναι αποθηκευμένα μέσα σε μια ομάδα και να διαβάσετε τις τιμές `AlternativeText` τους. Επιπλέον, το άρθρο καλύπτει σύντομα σχετικές δυνατότητες ομαδικών σχημάτων όπως ένθετες ομάδες, σειρά z και επιλογές κλειδώματος.

## **Προσθήκη Ομαδικού Σχήματος**
Το Aspose.Slides υποστηρίζει την εργασία με ομαδικά σχήματα σε διαφάνειες. Αυτή η δυνατότητα βοηθά τους προγραμματιστές να δημιουργούν πιο πλούσιες παρουσιάσεις. Το Aspose.Slides for C++ υποστηρίζει την προσθήκη ή πρόσβαση σε ομαδικά σχήματα. Είναι δυνατόν να προσθέσετε σχήματα σε ένα προστεθέν ομαδικό σχήμα για να το γεμίσετε ή να έχετε πρόσβαση σε οποιαδήποτε ιδιότητα του ομαδικού σχήματος. Για να προσθέσετε ένα ομαδικό σχήμα σε μια διαφάνεια χρησιμοποιώντας το Aspose.Slides for C++:

1. Δημιουργήστε ένα στιγμιότυπο της κλάσης[Presentation](https://reference.aspose.com/slides/el/cpp/aspose.slides/presentation/).
2. Αποκτήστε την αναφορά μιας διαφάνειας χρησιμοποιώντας το Index της.
3. Προσθέστε ένα ομαδικό σχήμα στη διαφάνεια.
4. Προσθέστε τα σχήματα στο προστεθέν ομαδικό σχήμα.
5. Αποθηκεύστε την τροποποιημένη παρουσίαση ως αρχείο PPTX.

## **Πρόσβαση στην Ιδιότητα AltText**
Αυτό το θέμα δείχνει απλά βήματα, με παραδείγματα κώδικα, για την προσθήκη ενός ομαδικού σχήματος και την πρόσβαση στην ιδιότητα AltText των ομαδικών σχημάτων σε διαφάνειες. Για να αποκτήσετε πρόσβαση στο AltText ενός ομαδικού σχήματος σε μια διαφάνεια χρησιμοποιώντας το Aspose.Slides for C++:

1. Δημιουργήστε μια παρουσία της κλάσης`Presentation` που αντιπροσωπεύει ένα αρχείο PPTX.
2. Αποκτήστε την αναφορά μιας διαφάνειας χρησιμοποιώντας το Index της.
3. Πρόσβαση στη συλλογή σχημάτων των διαφανειών.
4. Πρόσβαση στο ομαδικό σχήμα.
5. Πρόσβαση στην ιδιότητα AltText.

Το παρακάτω παράδειγμα αποκτά πρόσβαση στο εναλλακτικό κείμενο του ομαδικού σχήματος.

## **Συχνές Ερωτήσεις**

**Υποστηρίζεται η ένθετη ομαδοποίηση (μια ομάδα μέσα σε άλλη ομάδα);**

Ναι. Το[GroupShape](https://reference.aspose.com/slides/el/cpp/aspose.slides/groupshape/) διαθέτει τη μέθοδο[get_ParentGroup](https://reference.aspose.com/slides/el/cpp/aspose.slides/shape/get_parentgroup/), η οποία δείχνει άμεσα την υποστήριξη ιεραρχίας (μια ομάδα μπορεί να είναι παιδί κάποιας άλλης ομάδας).

**Πώς ελέγχω τη σειρά z της ομάδας σε σχέση με άλλα αντικείμενα στη διαφάνεια;**

Χρησιμοποιήστε τη[GroupShape](https://reference.aspose.com/slides/el/cpp/aspose.slides/groupshape/)·[Z-Order position](https://reference.aspose.com/slides/el/cpp/aspose.slides/shape/get_zorderposition/) για να ελέγξετε τη θέση της στην στοίβα απεικόνισης.

**Μπορώ να αποτρέψω τη μετακίνηση/επεξεργασία/αποομαδοποίηση;**

Ναι. Η ενότητα κλειδώματος της ομάδας εκτίθεται μέσω του[get_GroupShapeLock](https://reference.aspose.com/slides/el/cpp/aspose.slides/groupshape/get_groupshapelock/), που σας επιτρέπει να περιορίσετε τις ενέργειες στο αντικείμενο.
