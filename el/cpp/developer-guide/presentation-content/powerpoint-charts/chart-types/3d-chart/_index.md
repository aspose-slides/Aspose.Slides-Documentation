## **Επισκόπηση**

Αυτό το άρθρο εξηγεί πώς να προσαρμόσετε ένα 3Δ διάγραμμα στο Aspose.Slides ρυθμίζοντας τις ρυθμίσεις `Rotation3D` όπως `RotationX`, `RotationY`, `DepthPercents` και `RightAngleAxes`. Παρουσιάζει τη δημιουργία μιας παρουσίασης, την προσθήκη ενός 3Δ διαγράμματος με προεπιλεγμένα δεδομένα, την εφαρμογή των απαιτούμενων ρυθμίσεων προβολής 3Δ και την αποθήκευση της τροποποιημένης παρουσίασης ως αρχείο PPTX.

## **Ορίστε τις Ιδιότητες RotationX, RotationY και DepthPercents ενός 3Δ Διαγράμματος**
Το Aspose.Slides for C++ παρέχει ένα απλό API για τον ορισμό αυτών των ιδιοτήτων. Το παρακάτω άρθρο θα σας βοηθήσει να ορίσετε διαφορετικές ιδιότητες όπως X,Y Rotation, **DepthPercents** κλπ. Ο δείγματος κώδικας εφαρμόζει τις παραπάνω ιδιότητες.

1. Δημιουργήστε μια παρουσία της κλάσης [Presentation](https://reference.aspose.com/slides/el/cpp/aspose.slides/presentation/) .
1. Αποκτήστε πρόσβαση στην πρώτη διαφάνεια.
1. Προσθέστε διάγραμμα με προεπιλεγμένα δεδομένα.
1. Ορίστε τις ιδιότητες Rotation3D.
1. Αποθηκεύστε την τροποποιημένη παρουσίαση σε αρχείο PPTX.

## **Συχνές Ερωτήσεις**

**Ποιους τύπους διαγραμμάτων υποστηρίζει η λειτουργία 3Δ στο Aspose.Slides;**

Το Aspose.Slides υποστηρίζει 3Δ παραλλαγές διαγραμμάτων στήλης, όπως Column 3D, Clustered Column 3D, Stacked Column 3D και 100% Stacked Column 3D, καθώς και σχετικούς 3Δ τύπους που εκτίθενται μέσω της απαρίθμησης [ChartType](https://reference.aspose.com/slides/el/cpp/aspose.slides.charts/charttype/). Για ακριβή, ενημερωμένη λίστα, ελέγξτε τα μέλη του [ChartType](https://reference.aspose.com/slides/el/cpp/aspose.slides.charts/charttype/) στην αναφορά API της εγκατεστημένης έκδοσής σας.

**Μπορώ να λάβω μια raster εικόνα ενός 3Δ διαγράμματος για αναφορά ή το web;**

Ναι. Μπορείτε να εξάγετε ένα διάγραμμα σε εικόνα μέσω του API διαγράμματος ή να αποδώσετε ολόκληρη τη διαφάνεια σε μορφές όπως PNG ή JPEG. Αυτό είναι χρήσιμο όταν χρειάζεστε ακριβή προεπισκόπηση σε επίπεδο pixel ή θέλετε να ενσωματώσετε το διάγραμμα σε έγγραφα, πίνακες ελέγχου ή ιστοσελίδες χωρίς να απαιτείται το PowerPoint.

**Πόσο αποδοτική είναι η δημιουργία και η απόδοση μεγάλων 3Δ διαγραμμάτων;**

Η απόδοση εξαρτάται από τον όγκο των δεδομένων και την οπτική πολυπλοκότητα. Για βέλτιστα αποτελέσματα, διατηρήστε τις 3Δ επιδράσεις στο ελάχιστο, αποφύγετε βαριές υφές στα τείχη και τις περιοχές σχεδίασης, περιορίστε τον αριθμό των σημείων δεδομένων ανά σειρά όποτε είναι δυνατόν και αποδώστε σε έξοδο κατάλληλου μεγέθους (ανάλυση και διαστάσεις) ώστε να ταιριάζει με την επιθυμητή οθόνη ή τις ανάγκες εκτύπωσης.
