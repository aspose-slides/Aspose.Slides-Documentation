## **Επισκόπηση**

Αυτό το άρθρο εξηγεί πώς να εργάζεστε με πίνακες δεδομένων γραφημάτων στο Aspose.Slides. Δείχνει πώς να εμφανίζετε έναν πίνακα δεδομένων για ένα γράφημα και να προσαρμόζετε τη μορφοποίηση κειμένου ορίζοντας ιδιότητες γραμματοσειράς όπως έντονη γραφή και ύψος γραμματοσειράς. Το παράδειγμα δείχνει τη φόρτωση μιας παρουσίασης, την προσθήκη γραφήματος, την ενεργοποίηση του πίνακα δεδομένων του γραφήματος, την εφαρμογή ρυθμίσεων γραμματοσειράς και την αποθήκευση της ενημερωμένης παρουσίασης.

## **Ορίστε Ιδιότητες Γραμματοσειράς για Πίνακα Δεδομένων Γραφήματος**
Aspose.Slides για C++ επιτρέπει την αλλαγή των ιδιοτήτων γραμματοσειράς για έναν πίνακα δεδομένων γραφήματος.  

1. Δημιουργήστε αντικείμενο κλάσης [Παρουσίαση](https://reference.aspose.com/slides/el/cpp/class/aspose.slides.presentation).
1. Προσθέστε γράφημα στη διαφάνεια.
1. Ορίστε τον πίνακα του γραφήματος.
1. Ορίστε το ύψος γραμματοσειράς.
1. Αποθηκεύστε τη τροποποιημένη παρουσίαση.

Παρακάτω παρέχεται παράδειγμα δείγματος.  

``` cpp
auto pres = System::MakeObject<Presentation>(u"test.pptx");
    
auto chart = pres->get_Slides()->idx_get(0)->get_Shapes()->AddChart(ChartType::ClusteredColumn, 50.0f, 50.0f, 600.0f, 400.0f);

chart->set_HasDataTable(true);

chart->get_ChartDataTable()->get_TextFormat()->get_PortionFormat()->set_FontBold(NullableBool::True);
chart->get_ChartDataTable()->get_TextFormat()->get_PortionFormat()->set_FontHeight(20.0f);

pres->Save(u"output.pptx", SaveFormat::Pptx);
```

## **Συχνές Ερωτήσεις**

**Μπορώ να εμφανίσω μικρά κλειδιά υπομνήματος δίπλα στις τιμές στον πίνακα δεδομένων του γραφήματος;**

Ναι. Ο πίνακας δεδομένων υποστηρίζει [κλειδιά υπομνήματος](https://reference.aspose.com/slides/el/cpp/aspose.slides.charts/datatable/set_showlegendkey/), και μπορείτε να τα ενεργοποιήσετε ή να τα απενεργοποιήσετε.

**Θα διατηρηθεί ο πίνακας δεδομένων κατά την εξαγωγή της παρουσίασης σε PDF, HTML ή εικόνες;**

Ναι. Το Aspose.Slides αποδίδει το γράφημα ως μέρος της διαφάνειας, έτσι ώστε το εξαγόμενο [PDF](/slides/el/cpp/convert-powerpoint-to-pdf/)/[HTML](/slides/el/cpp/convert-powerpoint-to-html/)/[image](/slides/el/cpp/convert-powerpoint-to-png/) να περιλαμβάνει το γράφημα με τον πίνακα δεδομένων του.

**Υποστηρίζονται πίνακες δεδομένων για γραφήματα που προέρχονται από αρχείο προτύπου;**

Ναι. Για οποιοδήποτε γράφημα που φορτώνεται από υπάρχουσα παρουσίαση ή πρότυπο, μπορείτε να ελέγξετε και να αλλάξετε αν ένας πίνακας δεδομένων [εμφανίζεται](https://reference.aspose.com/slides/el/cpp/aspose.slides.charts/chart/set_hasdatatable/) χρησιμοποιώντας τις ιδιότητες του γραφήματος.

**Πώς μπορώ γρήγορα να βρω ποια γραφήματα σε ένα αρχείο έχουν ενεργοποιημένο τον πίνακα δεδομένων;**

Εξετάστε την ιδιότητα κάθε γραφήματος που υποδεικνύει αν ο πίνακας δεδομένων [εμφανίζεται](https://reference.aspose.com/slides/el/cpp/aspose.slides.charts/chart/get_hasdatatable/) και διατρέξτε τις διαφάνειες για να εντοπίσετε τα γραφήματα όπου είναι ενεργοποιημένος.
