## **Επισκόπηση**

Το Aspose.Slides σάς επιτρέπει να μετατρέπετε παρουσιάσεις PowerPoint σε αρχεία animated GIF με λίγες μόνο γραμμές κώδικα. Είναι χρήσιμο όταν χρειάζεται να μοιραστείτε το περιεχόμενο των διαφανειών σε ένα ελαφρύ, ευρέως υποστηριζόμενο μορφό GIF που μπορεί να ενσωματωθεί σε ιστοσελίδες, εφαρμογές μηνυμάτων ή τεκμηρίωση. Αυτό το άρθρο εξηγεί πώς να εξάγετε μια παρουσίαση σε GIF χρησιμοποιώντας τις προεπιλεγμένες ρυθμίσεις και πώς να προσαρμόσετε το αποτέλεσμα διαμορφώνοντας επιλογές όπως το μέγεθος του πλαισίου, η καθυστέρηση διαφάνειας και ο ρυθμός καρέ μετάβασης μέσω του [GifOptions](https://reference.aspose.com/slides/el/cpp/aspose.slides.export/gifoptions/).

## **Μετατροπή Παρουσιάσεων σε Animated GIF με Προεπιλεγμένες Ρυθμίσεις**

Αυτό το δείγμα κώδικα σε C++ δείχνει πώς να μετατρέψετε μια παρουσίαση σε animated GIF χρησιμοποιώντας τις προεπιλεγμένες ρυθμίσεις:

``` cpp
auto pres = System::MakeObject<Presentation>(u"pres.pptx");
pres->Save(u"pres.gif", SaveFormat::Gif);
```

Το animated GIF θα δημιουργηθεί με τις προεπιλεγμένες παραμέτρους.

{{%  alert  title="TIP"  color="primary"  %}} 
Εάν προτιμάτε να προσαρμόσετε τις παραμέτρους για το GIF, μπορείτε να χρησιμοποιήσετε την κλάση GifOptions. Δείτε το δείγμα κώδικα παρακάτω. 
{{% /alert %}} 

## **Μετατροπή Παρουσιάσεων σε Animated GIF με Προσαρμοσμένες Ρυθμίσεις**

Αυτό το δείγμα κώδικα δείχνει πώς να μετατρέψετε μια παρουσίαση σε animated GIF χρησιμοποιώντας προσαρμοσμένες ρυθμίσεις σε C++:

``` cpp
auto gifOptions = System::MakeObject<GifOptions>();
// το μέγεθος του παραγόμενου GIF 
gifOptions->set_FrameSize(Size(960, 720));
// πόσος χρόνος θα προβληθεί κάθε διαφάνεια μέχρι να αντικατασταθεί από την επόμενη
gifOptions->set_DefaultDelay(2000);
// αύξηση FPS για καλύτερη ποιότητα μετάβασης της κίνησης
gifOptions->set_TransitionFps(35);

auto pres = System::MakeObject<Presentation>(u"pres.pptx");
pres->Save(u"pres.gif", SaveFormat::Gif, gifOptions);
```

{{% alert title="Info" color="info" %}}
Μπορεί να θέλετε να δοκιμάσετε έναν ΔΩΡΕΑΝ μετατροπέα Text to GIF που αναπτύχθηκε από την Aspose. 
{{% /alert %}}

## **Συχνές Ερωτήσεις**

**Τι γίνεται αν οι γραμματοσειρές που χρησιμοποιούνται στην παρουσίαση δεν είναι εγκατεστημένες στο σύστημα;**

Εγκαταστήστε τις ελλείπουσες γραμματοσειρές ή [ρυθμίστε εναλλακτικές γραμματοσειρές](/slides/el/cpp/powerpoint-fonts/). Το Aspose.Slides θα αντικαταστήσει, αλλά η εμφάνιση μπορεί να διαφέρει. Για branding, βεβαιωθείτε πάντα ότι οι απαιτούμενες γραμματοσειρές είναι ρητά διαθέσιμες.

**Μπορώ να προσθέσω υδατογράφημα πάνω στα πλαίσια του GIF;**

Ναι. [Προσθέστε ένα ημιδιαφανές αντικείμενο/λογότυπο](/slides/el/cpp/watermark/) στη κύρια διαφάνεια ή σε μεμονωμένες διαφάνειες πριν την εξαγωγή — το υδατογράφημα θα εμφανίζεται σε κάθε πλαίσιο.
