## **Επισκόπηση**

Αυτό το άρθρο εξηγεί πώς να αποδίδετε διαφάνειες παρουσίασης ως εικόνες SVG χρησιμοποιώντας το Aspose.Slides. Περιγράφει τη μορφή SVG και τα πλεονεκτήματά της, συμπεριλαμβανομένης της επεκτασιμότητας, της προσβασιμότητας και της καταλληλότητας για ανάπτυξη διαδικτύου.

Θα μάθετε πώς να φορτώσετε ένα αρχείο παρουσίασης, να διασχίσετε τις διαφάνειες του και να αποθηκεύσετε κάθε διαφάνεια ως ξεχωριστό αρχείο SVG. Το άρθρο καλύπτει τις μορφές παρουσίασης PowerPoint και OpenDocument, συμπεριλαμβανομένων των PPT, PPTX, ODP και PPS, και δείχνει πώς να πραγματοποιήσετε τη μετατροπή προγραμματικά με την κλάση `Presentation` και τη μέθοδο `WriteAsSvg`.

## **Μορφή SVG**

Το SVG—συντομογραφία του Scalable Vector Graphics—είναι ένας πρότυπος τύπος ή μορφή γραφικών που χρησιμοποιείται για την απόδοση δισδιάστατων εικόνων. Το SVG αποθηκεύει τις εικόνες ως διανύσματα σε XML με λεπτομέρειες που ορίζουν τη συμπεριφορά ή την εμφάνισή τους.

Το SVG είναι μία από τις λίγες μορφές εικόνων που πληρούν πολύ υψηλά πρότυπα σε αυτούς τους όρους: επεκτασιμότητα, αλληλενέργεια, απόδοση, προσβασιμότητα, προγραμματισιμότητα και άλλα. Για αυτούς τους λόγους, χρησιμοποιείται ευρέως στην ανάπτυξη διαδικτύου.

Μπορείτε να χρησιμοποιήσετε αρχεία SVG όταν χρειάζεται να

- **εκτυπώσετε την παρουσίασή σας σε *πολύ μεγάλο φορμάτ*.** Οι εικόνες SVG μπορούν να κλιμακώνονται σε οποιαδήποτε ανάλυση ή επίπεδο. Μπορείτε να αλλάζετε το μέγεθος των εικόνων SVG όσες φορές χρειάζεται χωρίς να θυσιάσετε την ποιότητα.
- **χρησιμοποιήσετε διαγράμματα και γραφήματα από τις διαφάνειές σας σε *διαφορετικά μέσα ή πλατφόρμες***. Οι περισσότεροι αναγνώστες μπορούν να ερμηνεύσουν αρχεία SVG.
- **χρησιμοποιήσετε τα *μικρότερα δυνατά μεγέθη εικόνων***. Τα αρχεία SVG είναι γενικά μικρότερα από τα ισοδύναμα υψηλής ανάλυσης σε άλλες μορφές, ειδικά σε μορφές που βασίζονται σε bitmap (JPEG ή PNG).

## **Απόδοση διαφάνειας ως εικόνα SVG**

Το Aspose.Slides για C++ σας επιτρέπει να εξάγετε τις διαφάνειες στις παρουσιάσεις σας ως εικόνες SVG. Ακολουθήστε αυτά τα βήματα για να δημιουργήσετε εικόνες SVG:

1. Δημιουργήστε μια παρουσία της κλάσης Presentation.
2. Διασχίστε όλες τις διαφάνειες στην παρουσίαση.
3. Γράψτε κάθε διαφάνεια στο δικό της αρχείο SVG μέσω του FileStream.

{{% alert color="primary" %}} 

Μπορείτε να δοκιμάσετε τη [δωρεάν διαδικτυακή εφαρμογή](https://products.aspose.app/slides/el/conversion/ppt-to-svg) στην οποία υλοποιήσαμε τη λειτουργία μετατροπής PPT σε SVG από το Aspose.Slides για C++.

{{% /alert %}} 

Αυτό το δείγμα κώδικα σε C++ δείχνει πώς να μετατρέψετε PPT σε SVG χρησιμοποιώντας το Aspose.Slides:

``` cpp
auto pres = System::MakeObject<Presentation>(u"pres.pptx");
        
for (int32_t index = 0; index < pres->get_Slides()->get_Count(); index++)
{
    auto fileName = String::Format(u"slide-{0}.svg", index);
    auto fileStream = System::MakeObject<FileStream>(fileName, FileMode::Create, FileAccess::Write);

    auto slide = pres->get_Slides()->idx_get(index);
    slide->WriteAsSvg(fileStream);
}
```

## **Συχνές ερωτήσεις**

**Γιατί το παραγόμενο SVG μπορεί να φαίνεται διαφορετικό σε διαφορετικά προγράμματα περιήγησης;**

Η υποστήριξη συγκεκριμένων λειτουργιών SVG υλοποιείται διαφορετικά από τις μηχανές των προγραμμάτων περιήγησης. Οι παράμετροι [SVGOptions](https://reference.aspose.com/slides/el/cpp/aspose.slides.export/svgoptions/) βοηθούν στην εξομάλυνση των ασυμβατοτήτων.

**Μπορεί να εξαχθούν όχι μόνο διαφάνειες αλλά και μεμονωμένα σχήματα σε SVG;**

Ναι. Οποιοδήποτε [σχήμα μπορεί να αποθηκευτεί ως ξεχωριστό SVG](https://reference.aspose.com/slides/el/cpp/aspose.slides/shape/writeassvg/), κάτι που είναι βολικό για εικονίδια, εικονογράμματα και επαναχρησιμοποίηση γραφικών.

**Μπορούν πολλές διαφάνειες να συνδυαστούν σε ένα μόνο SVG (strip/document);**

Το τυπικό σενάριο είναι μία διαφάνεια → ένα SVG. Η συνένωση πολλαπλών διαφανειών σε έναν ενιαίο καμβά SVG είναι ένα βήμα μετα-επεξεργασίας που εκτελείται σε επίπεδο εφαρμογής.
