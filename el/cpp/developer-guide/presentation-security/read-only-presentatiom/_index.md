## **Εισαγωγή**

Στο PowerPoint 2019, η Microsoft παρουσίασε τη ρύθμιση **Always Open Read-Only** ως μία από τις επιλογές που μπορούν να χρησιμοποιήσουν οι χρήστες για την προστασία των παρουσιάσεων τους. Μπορεί να θέλετε να χρησιμοποιήσετε αυτήν τη ρύθμιση Ανάγνωσης‑Μόνο για να προστατεύσετε μια παρουσίαση όταν

- Θέλετε να αποτρέψετε τυχαίες επεμβάσεις και να διατηρήσετε το περιεχόμενο της παρουσίασής σας ασφαλές.  
- Θέλετε να ενημερώσετε τους ανθρώπους ότι η παρουσίαση που παραχωρήσατε είναι η τελική έκδοση.  

Αφού επιλέξετε την επιλογή **Always Open Read-Only** για μια παρουσίαση, όταν οι χρήστες ανοίξουν την παρουσίαση, βλέπουν τη σύσταση **Read-Only** και μπορεί να εμφανιστεί το παρακάτω μήνυμα: *To prevent accidental changes, the author has set this file to open as read-only.*

Η σύσταση **Read-Only** είναι ένας απλός αλλά αποτελεσματικός αποτρεπτικός μηχανισμός που αποθαρρύνει την επεξεργασία, επειδή οι χρήστες πρέπει να εκτελέσουν μια ενέργεια για να την αφαιρέσουν πριν τους επιτραπεί η επεξεργασία της παρουσίασης. Εάν δεν θέλετε οι χρήστες να κάνουν αλλαγές σε μια παρουσίαση και θέλετε να το τοποθετήσετε αυτό με ευγενικό τρόπο, η σύσταση **Read-Only** μπορεί να είναι μια καλή επιλογή για εσάς.

> Εάν μια παρουσίαση με προστασία **Read-Only** ανοίξει σε παλαιότερη εφαρμογή Microsoft PowerPoint—που δεν υποστηρίζει τη νεοεισαχθείσα λειτουργία—η σύσταση **Read-Only** αγνοείται (η παρουσίαση ανοίγει κανονικά).

## **Εφαρμογή Λειτουργίας Ανάγνωσης‑Μόνο**

Το Aspose.Slides for C++ σας επιτρέπει να ορίσετε μια παρουσίαση σε **Read-Only**, πράγμα που σημαίνει ότι οι χρήστες (αφού ανοίξουν την παρουσίαση) βλέπουν τη σύσταση **Read-Only**. Αυτό το δείγμα κώδικα δείχνει πώς να ορίσετε μια παρουσίαση σε **Read-Only** σε C++ χρησιμοποιώντας το Aspose.Slides:

``` cpp
auto pres = System::MakeObject<Presentation>();
pres->get_ProtectionManager()->set_ReadOnlyRecommended(true);
pres->Save(u"ReadOnlyPresentation.pptx", SaveFormat::Pptx);
```

{{% alert color="primary" %}} 

**Σημείωση**: Η σύσταση **Read-Only** προορίζεται απλώς για να αποθαρρύνει την επεξεργασία ή να σταματήσει τους χρήστες από τυχαίες αλλαγές σε μια παρουσίαση PowerPoint. Εάν ένα άτομο με κίνητρο—που γνωρίζει τι κάνει—αποφασίσει να επεξεργαστεί την παρουσίασή σας, μπορεί εύκολα να αφαιρέσει τη ρύθμιση Read‑Only. Εάν χρειάζεται σοβαρά να αποτρέψετε μη εξουσιοδοτημένη επεξεργασία, είναι προτιμότερο να χρησιμοποιήσετε [more stringent protections that involve encryptions and passwords](https://docs.aspose.com/slides/el/cpp/password-protected-presentation/). 

{{% /alert %}} 

## **Συχνές Ερωτήσεις**

**Πώς διαφέρει το «Read-Only recommended» από την πλήρη προστασία με κωδικό;**

Το «Read-Only recommended» εμφανίζει μόνο μια πρόταση να ανοίξετε το αρχείο σε λειτουργία ανάγνωσης‑μόνο και είναι εύκολο να παρακαμφθεί. [Password protection](/slides/el/cpp/password-protected-presentation/) περιορίζει πραγματικά το άνοιγμα ή την επεξεργασία και είναι κατάλληλη όταν χρειάζεστε πραγματικούς ελέγχους ασφαλείας.

**Μπορεί το «Read-Only recommended» να συνδυασθεί με υδατογραφήματα για ακόμη μεγαλύτερη αποθάρρυνση των επεξεργασιών;**

Ναι. Η σύσταση μπορεί να συνδυαστεί με [watermarks](/slides/el/cpp/watermark/) ως οπτικό αποτρεπτικό μέτρο· είναι ξεχωριστοί μηχανισμοί και λειτουργούν καλά μαζί.

**Μπορεί ένα μακροεντολή ή εξωτερικό εργαλείο να τροποποιήσει το αρχείο όταν είναι ενεργοποιημένη η σύσταση;**

Ναι. Η σύσταση δεν εμποδίζει προγραμματιστικές αλλαγές. Για να αποτρέψετε αυτοματοποιημένες επεμβάσεις, χρησιμοποιήστε [passwords and encryption](/slides/el/cpp/password-protected-presentation/).

**Πώς σχετίζεται το «Read-Only recommended» με τις σημαίες «is encrypted» και «is write protected»;**

Είναι διαφορετικά σήματα. Το «Read-Only recommended» είναι μια ήπια, προαιρετική προτροπή· [get_IsWriteProtected](https://reference.aspose.com/slides/el/cpp/aspose.slides/protectionmanager/get_iswriteprotected/) και [get_IsEncrypted](https://reference.aspose.com/slides/el/cpp/aspose.slides/protectionmanager/get_isencrypted/) υποδεικνύουν πραγματικούς περιορισμούς εγγραφής ή ανάγνωσης που εξαρτώνται από κωδικούς ή κρυπτογράφηση.
