## **Εγκατάσταση**
Μπορείτε να εγκαταστήσετε το Aspose.Slides for Reporting Services μέσω ενός εγκαταστάτη MSI. 

{{% alert title="Note" color="warning" %}} 

**Aspose.Slides for Reporting Services** απαιτεί την εγκατάσταση του **.NET Framework 3.5** στο κεντρικό μηχάνημα. 

{{% /alert %}}

Εκτελέστε ***Aspose.Slides.ReportingServices.msi*** και ακολουθήστε τα βήματα που προσφέρει ο εγκαταστάτης. 

Ο εγκαταστάτης θα αντιγράψει τη συναρμολόγηση και άλλα αρχεία στον καθορισμένο κατάλογο και θα εγκαταστήσει το προϊόν στην προεπιλεγμένη παρουσία Reporting Services. Δεν χρειάζεται να αντιγράψετε ή να τροποποιήσετε χειροκίνητα κανένα αρχείο, εκτός εάν θέλετε να προσθέσετε ειδικές παραμέτρους διαμόρφωση. 

Η εγκατάσταση με χρήση του εγκαταστάτη MSI είναι η καλύτερη επιλογή στα περισσότερα περιστατικά. Ωστόσο, ίσως θελήσετε να εγκαταστήσετε το προϊόν χειροκίνητα σε ορισμένες καταστάσεις: 

- Η αυτόματη εγκατάσταση αποτυγχάνει λόγω ζητημάτων ασφαλείας ή άλλων λόγων. 
- Το προϊόν πρέπει να εγκατασταθεί σε ονομασμένη (μη προεπιλεγμένη) παρουσία του Reporting Services ή σε πολλαπλές παρουσίες.
- Μετά την αναβάθμιση στην τελευταία έκδοση, μπορεί να θέλετε απλώς να αντικαταστήσετε τη συναρμολόγηση αντί να απεγκαταστήσετε την παλιά έκδοση και να εγκαταστήσετε τη νέα χρησιμοποιώντας τον εγκαταστάτη MSI. **Σημείωση** ότι μπορεί να καταλήξετε με άλλα αρχεία σε αυτήν την περίπτωση.
