{{% alert color="primary" %}} 

Μπορείτε εύκολα να κάνετε λήψη του Aspose.Slides για αξιολόγηση. Η λήψη αξιολόγησης είναι η ίδια με την αγορά. Η έκδοση αξιολόγησης γίνεται απλώς αδειοδοτημένη όταν προσθέσετε μερικές γραμμές κώδικα για την εφαρμογή της άδειας.

Η έκδοση αξιολόγησης του Aspose.Slides (χωρίς καθορισμένη άδεια) παρέχει πλήρη λειτουργικότητα του προϊόντος, αλλά εισάγει ένα υδατογράφημα αξιολόγησης σε κάθε διαφάνεια της παραγόμενης παρουσίασης.

{{% /alert %}} {{% alert color="primary" %}} 

Εάν θέλετε να δοκιμάσετε το Aspose.Slides χωρίς τους περιορισμούς της έκδοσης αξιολόγησης, μπορείτε επίσης να ζητήσετε μια 30‑ημερη προσωρινή άδεια. Παρακαλούμε ανατρέξτε στο [Πώς να αποκτήσετε προσωρινή άδεια?](https://purchase.aspose.com/temporary-license)
