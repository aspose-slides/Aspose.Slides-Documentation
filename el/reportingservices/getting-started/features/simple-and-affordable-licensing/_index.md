## **Server Subscription**
Η συνδρομή Server παρέχει τα ακόλουθα χαρακτηριστικά:

- Απαιτείται μία άδεια για κάθε φυσικό ή εικονικό διακομιστή.
- Απεριόριστος αριθμός επεξεργαστών ή πυρήνων.
- Απεριόριστος αριθμός παρουσιών Microsoft SQL Server.
- Απεριόριστος αριθμός προγραμματιστών.
- Δωρεάν νέες κυκλοφορίες και διορθώσεις για 1 χρόνο.
- Δωρεάν τεχνική υποστήριξη για 1 χρόνο.
- Έκπτωση ισχύει για πολλαπλές αγορές.
