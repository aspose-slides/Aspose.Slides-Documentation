{{% alert color="primary" %}}

Το Aspose.Slides for Reporting Services υποστηρίζει πλήρως όλες τις προδιαγραφές RDL. Αυτό σημαίνει δύο σπουδαία πράγματα:

- Δεν χρειάζεται να σχεδιάσετε ξανά τις υπάρχουσες αναφορές. Μπορείτε να εξάγετε οποιαδήποτε υπάρχουσα αναφορά RDL ως παρουσίαση Microsoft PowerPoint και θα εμφανιστεί ακριβώς σύμφωνα με το σχεδιασμό RDL.
- Δεν χρειάζεται να χρησιμοποιείτε συγκεκριμένο σχεδιαστή αναφορών. Μπορείτε να χρησιμοποιήσετε οποιονδήποτε σχεδιαστή αναφορών RDL και η αναφορά θα εξαχθεί ακριβώς όπως την σχεδιάσατε.

{{% /alert %}}

Το Aspose.Slides for Reporting Services υποστηρίζει τα παρακάτω στοιχεία RDL:

- Σελίδα, κεφαλίδες, υποσέλιδα
- Πλαίσια κειμένου
- Εικόνες
- Υποαναφορές
- Διαγράμματα
- Λίστες
- Πίνακες
- Μήτρες
- Στυλ
- Ορθογώνια

**Ένα παράδειγμα αναφοράς με κεφαλίδες, υποσέλιδα, εικόνες, υποαναφορές, πίνακες, πλαίσια κειμένου και ορθογώνια, εξαγόμενο ως παρουσίαση Microsoft PowerPoint (PPT).**
