{{% alert color="primary" %}} 

Η Aspose.Slides for Reporting Services υλοποίησε υποστήριξη για το Power BI. 

{{% /alert %}} 

Οι σελιδοποιημένες αναφορές είναι αναφορές που μορφοποιούνται ώστε να ταιριάζουν καλά σε μια σελίδα. Τέτοιες αναφορές είναι ιδανικές για λειτουργίες εξαγωγής και εκτύπωση. Οι σελιδοποιημένες αναφορές έχουν σχεδιαστεί έτσι ώστε να παρέχουν όσες σελίδες χρειάζεται για την εκτύπωση ή εξαγωγή κάθε γραμμής δεδομένων στον πίνακά σας. 

Για παράδειγμα, εάν θέλετε να δημιουργήσετε μια αναφορά ώστε οι ιδιοκτήτες της να μπορούν να δουν την ποσότητα του προϊόντος, την τιμή πώλησης και το κέρδος για κάθε αντικείμενο που πωλήθηκε τους τελευταίους 12 μήνες, τότε μια σελιδοποιημένη αναφορά είναι αυτό που χρειάζεστε. 

Για να σας επιτρέψει να δημιουργήσετε σελιδοποιημένες αναφορές, η Microsoft παρέχει το Power BI Report Builder στο Power BI. Αφού δημιουργήσετε μια σελιδοποιημένη αναφορά και την αποθηκεύσετε στο μηχάνημά σας, το προκύπτον αρχείο λαμβάνει την κατάληξη .rdl.
