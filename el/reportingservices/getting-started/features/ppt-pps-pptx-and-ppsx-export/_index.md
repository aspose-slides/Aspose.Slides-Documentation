{{% alert color="primary" %}} 

Το Microsoft SQL Server Reporting Services δεν διαθέτει ενσωματωμένες δυνατότητες εξαγωγής αναφορών ως παρουσιάσεις Microsoft PowerPoint, αλλά μετά την εγκατάσταση του Aspose.Slides for Reporting Services στον διακομιστή σας, αποκτάτε πρόσβαση σε δύο επιπλέον μορφές εξαγωγής: 

- Microsoft PowerPoint Presentation (PPT)
- Microsoft PowerPoint SlideShow (PPS)
- Microsoft PowerPoint 2007 Presentation (PPTX)
- Microsoft PowerPoint 2007 SlideShow (PPSX)

{{% /alert %}} 

Για τη δημιουργία εγγράφων σε αυτές τις μορφές, το Aspose.Slides for Reporting Services βασίζεται σε μια ενσωματωμένη έκδοση του [Aspose.Slides for .NET](http://www.aspose.com/Products/Aspose.Slides/), του κορυφαίου στην αγορά στοιχείου επεξεργασίας παρουσιάσεων από την Aspose. Το Microsoft PowerPoint δεν χρησιμοποιείται για τη δημιουργία των εγγράφων. 


Τα αντίγραφα PPT και PPS της ίδιας αναφοράς θα εμφανίζονται πανομοιότυπα, χάρη στο υψηλό επίπεδο υποστήριξης των υποκείμενων μορφών αρχείων που παρέχει το Aspose.Slides.
