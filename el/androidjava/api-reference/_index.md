{{% alert color="primary" %}}

Το Aspose.Slides for Android μέσω Java είναι μια βιβλιοθήκη κλάσεων που επιτρέπει στις εφαρμογές Java για Android να διαβάζουν και να γράφουν έγγραφα παρουσίασης χωρίς χρήση του Microsoft PowerPoint® ή άλλου λογισμικού. Το Aspose.Slides for Android μέσω Java σάς επιτρέπει να διαχειρίζεστε κείμενο, σχήματα, πίνακες και κινούμενα σχέδια· να προσθέτετε ήχο και βίντεο σε διαφάνειες· να προεπισκοπείτε διαφάνειες· και να εξάγετε διαφάνειες σε μορφή SVG, PDF και άλλα.

Η πιο πρόσφατη Αναφορά API βρίσκεται στο [Aspose.Slides for Android via Java API Reference](https://reference.aspose.com/slides/el/androidjava/).

{{% /alert %}}
