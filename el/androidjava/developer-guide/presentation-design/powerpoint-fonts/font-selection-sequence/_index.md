## **Επισκόπηση**

Όταν μια παρουσίαση φορτώνεται, αποδίδεται ή μετατρέπεται σε άλλη μορφή, το Aspose.Slides ελέγχει εάν οι γραμματοσειρές που χρησιμοποιούνται στην παρουσίαση είναι διαθέσιμες στο λειτουργικό σύστημα. Εάν λείπει μια απαιτούμενη γραμματοσειρά, το Aspose.Slides επιλέγει μια εναλλακτική γραμματοσειρά που είναι όσο το δυνατόν πιο κοντινή σε αυτήν που θα χρησιμοποίησε το PowerPoint.

Το Aspose.Slides πρώτα αναζητά την επιλεγμένη γραμματοσειρά στο λειτουργικό σύστημα. Εάν η γραμματοσειρά βρεθεί, χρησιμοποιείται. Εάν δεν βρεθεί, εφαρμόζεται μια κατάλληλη εναλλακτική. Όταν οι κανόνες αντικατάστασης γραμματοσειρών ορίζονται μέσω `FontSubstRule`, λαμβάνονται επίσης υπόψη.

Μπορείτε επίσης να προσθέσετε γραμματοσειρές κατά την εκτέλεση της εφαρμογής, να χρησιμοποιήσετε ενσωματωμένες γραμματοσειρές από μια παρουσίαση ή να φορτώσετε εξωτερικές γραμματοσειρές για έγγραφα εξόδου, όπως αρχεία PDF.

## **Επιλογή γραμματοσειράς**

Ορισμένοι κανόνες εφαρμόζονται στις γραμματοσειρές μιας παρουσίασης όταν η παρουσίαση φορτώνεται, αποδίδεται ή μετατρέπεται σε άλλη μορφή. Για παράδειγμα, όταν προσπαθείτε να μετατρέψετε μια παρουσίαση (τις διαφάνειές της) σε εικόνες, ελέγχονται οι γραμματοσειρές της παρουσίασης για να επαληθευτεί ότι οι επιλεγμένες γραμματοσειρές είναι διαθέσιμες στο λειτουργικό σύστημα. Εάν επιβεβαιωθεί ότι οι γραμματοσειρές λείπουν, αντικαθίστανται — δείτε [**Αντικατάσταση γραμματοσειράς**](https://docs.aspose.com/slides/el/androidjava/font-replacement/) και [**Αντικατάσταση γραμματοσειράς**](https://docs.aspose.com/slides/el/androidjava/font-substitution/).

Αυτή είναι η διαδικασία που ακολουθεί το Aspose.Slides κατά την αντιμετώπιση των γραμματοσειρών:

1. Το Aspose.Slides αναζητά γραμματοσειρές στο λειτουργικό σύστημα για να βρει τη γραμματοσειρά που ταιριάζει με την επιλεγμένη γραμματοσειρά της παρουσίασης. 
2. Εάν η επιλεγμένη γραμματοσειρά βρεθεί, το Aspose.Slides τη χρησιμοποιεί. Διαφορετικά, το Aspose.Slides χρησιμοποιεί μια εναλλακτική γραμματοσειρά που είναι όσο το δυνατόν πιο κοντά σε αυτήν που θα χρησιμοποίησε το PowerPoint.
3. Εάν έχουν οριστεί κανόνες αντικατάστασης γραμματοσειράς μέσω [FontSubstRule](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/fontsubstrule/), εφαρμόζονται.

Το Aspose.Slides σας επιτρέπει να προσθέσετε γραμματοσειρές κατά την εκτέλεση της εφαρμογής και στη συνέχεια να χρησιμοποιήσετε αυτές τις γραμματοσειρές. Δείτε [**Προσαρμοσμένες γραμματοσειρές**](https://docs.aspose.com/slides/el/androidjava/custom-font/).

Όταν επιπλέον γραμματοσειρές τοποθετούνται μέσα σε μια παρουσίαση, ονομάζονται [**Ενσωματωμένες γραμματοσειρές**](https://docs.aspose.com/slides/el/androidjava/embedded-font/).

Το Aspose.Slides σας επιτρέπει να προσθέσετε γραμματοσειρές που εφαρμόζονται *μόνο* σε έγγραφα εξόδου. Για παράδειγμα, εάν μια παρουσίαση που θέλετε να μετατρέψετε σε PDF περιέχει γραμματοσειρές που λείπουν από το σύστημά σας και ενσωματωμένες γραμματοσειρές, μπορείτε να προσθέσετε ή να φορτώσετε τις απαιτούμενες γραμματοσειρές ως **εξωτερικές γραμματοσειρές**. 

{{% alert title="Note" color="primary" %}} 
Δεν διανέμουμε καμία γραμματοσειρά, είτε επί πληρωμή είτε δωρεάν. Το API μας σας επιτρέπει να φορτώσετε εξωτερικές γραμματοσειρές και να τις ενσωματώσετε σε έγγραφα, αλλά το κάνετε αυτό με γραμματοσειρές με δική σας ευχέρεια και ευθύνη.
{{% /alert %}}

## **Συχνές ερωτήσεις**

**Πώς μπορώ να προσδιορίσω ποιες γραμματοσειρές χρησιμοποιούνται πραγματικά σε μια παρουσίαση πριν από τη μετατροπή;**

Το Aspose.Slides σας επιτρέπει να ελέγξετε τις χρησιμοποιούμενες γραμματοσειρές μέσω του [font manager](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/fontsmanager/), ώστε να αποφασίσετε αν θα [ενσωματώσετε](/slides/el/androidjava/embedded-font/), [αντικαταστήσετε](/slides/el/androidjava/font-replacement/) ή προσθέσετε [εξωτερικές πηγές](/slides/el/androidjava/custom-font/). Αυτό σας βοηθά να αποτρέψετε ανεπιθύμητες αντικαταστάσεις κατά την απόδοση και την εξαγωγή.

**Μπορώ να προσθέσω επιπλέον καταλόγους γραμματοσειρών χωρίς να τους εγκαταστήσω στο λειτουργικό σύστημα;**

Ναι. Μπορείτε να καταχωρήσετε [εξωτερικές πηγές γραμματοσειρών](/slides/el/androidjava/custom-font/) όπως φακέλους ή ροές στη μνήμη για απόδοση και εξαγωγή. Αυτό αφαιρεί την εξάρτηση από τις γραμματοσειρές του συστήματος κεντρικού υπολογιστή και διατηρεί τη διάταξη προβλέψιμη.

**Πώς μπορώ να αποτρέψω μια σιωπηλή εναλλαγή σε ακατάλληλη γραμματοσειρά όταν λείπει ένα γλύφη;**

Ορίστε ρητά [font replacement](/slides/el/androidjava/font-replacement/) και [fallback rules](/slides/el/androidjava/fallback-font/) για τις γραμματοσειρές εκ των προτέρων. Αναλύοντας τις χρησιμοποιούμενες γραμματοσειρές και ορίζοντας ελεγχόμενη προτεραιότητα για τους υποκατάστατες, εξασφαλίζετε συνεπή τυπογραφία και αποφεύγετε ανεπιθύμητα αποτελέσματα.
