## **Επισκόπηση**

Το Aspose.Slides σάς επιτρέπει να αντικαταστήσετε μια γραμματοσειρά με άλλη σε όλη την παρουσίαση. Όταν αντικαθίσταται μια γραμματοσειρά, όλες οι εμφανίσεις της αρχικής γραμματοσειράς αλλάζουν στη νέα γραμματοσειρά.

Για να εκτελέσετε αντικατάσταση γραμματοσειράς, φορτώστε την παρουσίαση, ορίστε τη γραμματοσειρά προέλευσης και τη γραμματοσειρά αντικατάστασης, καλέστε τη μέθοδο αντικατάστασης γραμματοσειράς και αποθηκεύστε την τροποποιημένη παρουσίαση ως αρχείο PPTX. Αυτή η προσέγγιση είναι χρήσιμη όταν θέλετε σκόπιμα να μεταβείτε από μία οικογένεια γραμματοσειρών σε άλλη σε όλη την παρουσίαση.

## **Αντικατάσταση Γραμματοσειρών**

Αν αλλάξετε γνώμη σχετικά με τη χρήση μιας γραμματοσειράς, μπορείτε να την αντικαταστήσετε με άλλη. Όλες οι εμφανίσεις της παλιάς γραμματοσειράς θα αντικατασταθούν από τη νέα.

Το Aspose.Slides σάς επιτρέπει να αντικαταστήσετε μια γραμματοσειρά ως εξής:

1. Φορτώστε τη σχετική παρουσίαση. 
2. Φορτώστε τη γραμματοσειρά που θα αντικατασταθεί. 
3. Φορτώστε τη νέα γραμματοσειρά. 
4. Αντικαταστήστε τη γραμματοσειρά. 
5. Γράψτε την τροποποιημένη παρουσίαση ως αρχείο PPTX.

Αυτός ο κώδικας Java δείχνει την αντικατάσταση γραμματοσειράς:

```java
// Φορτώνει μια παρουσίαση
Presentation pres = new Presentation("Fonts.pptx");
try {
    // Φορτώνει τη γραμματοσειρά προέλευσης που θα αντικατασταθεί
    IFontData sourceFont = new FontData("Arial");
    
    // Φορτώνει τη νέα γραμματοσειρά
    IFontData destFont = new FontData("Times New Roman");
    
    // Αντικαθιστά τις γραμματοσειρές
    pres.getFontsManager().replaceFont(sourceFont, destFont);
    
    // Αποθηκεύει την παρουσίαση
    pres.save("UpdatedFont_out.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```

## **Συχνές Ερωτήσεις**

**Ποια είναι η διαφορά μεταξύ «αντικατάστασης γραμματοσειράς», «αντικατάστασης γραμματοσειράς» και «εφεδρικών γραμματοσειρών»;**

Η αντικατάσταση είναι μια σκόπιμη μετάβαση από μία οικογένεια σε άλλη σε όλο το έγγραφο. Η [Substitution](/slides/el/androidjava/font-substitution/) είναι ένας κανόνας όπως «αν η γραμματοσειρά δεν είναι διαθέσιμη, χρησιμοποιήστε X». Η [Fallback](/slides/el/androidjava/fallback-font/) εφαρμόζεται επιλεκτικά για μεμονωμένα ελλείποντα γλυφά όταν η βασική γραμματοσειρά είναι εγκατεστημένη αλλά δεν περιέχει τους απαιτούμενους χαρακτήρες.

**Εφαρμόζεται η αντικατάσταση σε κύριες διαφάνειες, διατάξεις, σημειώσεις και σχόλια;**

Ναι. Η αντικατάσταση επηρεάζει όλα τα αντικείμενα παρουσίασης που χρησιμοποιούν την αρχική γραμματοσειρά, συμπεριλαμβανομένων των κύριων διαφανειών και σημειώσεων· τα σχόλια επίσης αποτελούν μέρος του εγγράφου και λαμβάνονται υπόψη από τη μηχανή γραμματοσειρών.

**Θα αλλάξει η γραμματοσειρά μέσα σε ενσωματωμένα αντικείμενα OLE (π.χ. Excel);**

Όχι. Το [OLE content](/slides/el/androidjava/manage-ole/) ελέγχεται από τη δική του εφαρμογή. Η αντικατάσταση στην παρουσίαση δεν μορφοποιεί ξανά τα εσωτερικά δεδομένα OLE· μπορεί να εμφανίζεται ως εικόνα ή ως εξωτερικά επεξεργάσιμο περιεχόμενο.

**Μπορώ να αντικαταστήσω μια γραμματοσειρά μόνο σε μέρος της παρουσίασης (ανά διαφάνεια ή περιοχή);**

Στοχευμένη αντικατάσταση είναι δυνατή εάν αλλάξετε τη γραμματοσειρά στο επίπεδο των απαιτούμενων αντικειμένων/περιοχών αντί να εφαρμόσετε καθολική αντικατάσταση σε ολόκληρο το έγγραφο. Η λογική επιλογής γραμματοσειράς κατά τη σχεδίαση παραμένει η ίδια.

**Πώς μπορώ να καθορίσω εκ των προτέρων ποιες γραμματοσειρές χρησιμοποιεί η παρουσίαση;**

Χρησιμοποιήστε το [font manager] του παρουσίασης (https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/fontsmanager/): παρέχει μια λίστα με τις [families in use] (https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/fontsmanager/#getFonts--) και πληροφορίες για [substitutions/"unknown" fonts] (https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/fontsmanager/#getSubstitutions--), που βοηθούν στον προγραμματισμό της αντικατάστασης.

**Λειτουργεί η αντικατάσταση γραμματοσειράς κατά τη μετατροπή σε PDF/εικόνες;**

Ναι. Κατά την εξαγωγή, το Aspose.Slides εφαρμόζει την ίδια [font selection/substitution sequence](/slides/el/androidjava/font-selection-sequence/), έτσι ώστε μια αντικατάσταση που έγινε εκ των προτέρων να ληφθεί υπόψη κατά τη μετατροπή.

**Πρέπει να εγκαταστήσω τη γραμματοσειρά-στόχο στο σύστημα ή μπορώ να συνδέσω φάκελο γραμματοσειρών;**

Η εγκατάσταση δεν απαιτείται: η βιβλιοθήκη επιτρέπει το [loading external fonts](/slides/el/androidjava/custom-font/) από φακέλους χρήστη για χρήση κατά τη [rendering and export](/slides/el/androidjava/convert-powerpoint/).

**Θα διορθώσει η αντικατάσταση το πρόβλημα «tofu» (τετράγωνα) αντί για χαρακτήρες;**

Μόνο εάν η γραμματοσειρά-στόχος περιέχει πραγματικά τα απαιτούμενα γλυφά. Εάν όχι, [configure fallback](/slides/el/androidjava/fallback-font/) για να καλύψετε τους ελλείποντες χαρακτήρες.
