## **Μετατροπή PPT σε PPTX**
Το Aspose.Slides για Android μέσω Java διευκολύνει τώρα τους developers να προσπελάσουν το PPT χρησιμοποιώντας την κλάση [Presentation](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/presentation) και να το μετατρέψουν στη σχετική μορφή [PPTX](https://docs.fileformat.com/presentation/pptx/). Προς το παρόν, υποστηρίζει μερική μετατροπή του [PPT](https://docs.fileformat.com/presentation/ppt/) σε PPTX.

Το Aspose.Slides για Android μέσω Java προσφέρει την κλάση [Presentation](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/presentation) η οποία αντιπροσωπεύει ένα αρχείο παρουσίασης **PPTX**. Η κλάση Presentation μπορεί πλέον επίσης να προσπελάσει **PPT** μέσω του Presentation όταν το αντικείμενο δημιουργείται. Το παρακάτω παράδειγμα δείχνει πώς να μετατρέψετε μια παρουσίαση PPT σε παρουσίαση PPTX.

```java
// Δημιουργήστε ένα αντικείμενο Presentation που αντιπροσωπεύει αρχείο PPTX
Presentation pres = new Presentation("Aspose.ppt");
try {
// Αποθήκευση της παρουσίασης PPTX στη μορφή PPTX
    pres.save("ConvertedAspose.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```

Τα αρχεία PPTX προσφέρουν καλύτερη απόδοση, μειωμένο μέγεθος αρχείου και βελτιωμένη δυνατότητα ανάκτησης δεδομένων.
