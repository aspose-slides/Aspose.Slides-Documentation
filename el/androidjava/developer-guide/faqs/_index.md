## **Επισκόπηση**

Αυτή η Συχνές Ερωτήσεις παρέχει απαντήσεις σε κοινές ερωτήσεις σχετικά με το Aspose.Slides. Καλύπτει τις υποστηριζόμενες μορφές αρχείων, τη διαχείριση εξαιρέσεων κατά την εργασία με μεγάλα παρουσιάσεις, την αλλαγή μεγέθους των διαφανειών, την προεπισκόπηση διαφανειών, την ανάκτηση κειμένου από παρουσιάσεις, τη διαμόρφωση περιγραμμάτων πινάκων, την τοποθέτηση εικόνων και την επίλυση προβλημάτων που σχετίζονται με τις γραμματοσειρές κατά τη μετατροπή παρουσιάσεων σε PDF ή εικόνες.

## **Υποστηριζόμενες μορφές αρχείων**

**Q:** Ποιες μορφές αρχείων υποστηρίζει το Aspose.Slides for Android via Java;

**A:** Το Aspose.Slides for Android via Java υποστηρίζει τις μορφές αρχείων που περιγράφονται στις [Υποστηριζόμενες μορφές αρχείων](/slides/el/androidjava/supported-file-formats/).

## **Εξαιρέσεις**

**Q:** Λαμβάνω εξαίρεση «έλλειψη μνήμης» κατά τη φόρτωση ενός μεγάλου αρχείου PPT με εικόνες. Υπάρχει περιορισμός στο Aspose.Slides σχετικά με το μέγεθος του αρχείου;

**A:** Δεν υπάρχει συγκεκριμένος τύπος για τον υπολογισμό του μεγέθους παρουσίασης που υποστηρίζεται από το Aspose.Slides. Πρέπει να υπάρχει επαρκής χώρος για την αποθήκευση ολόκληρης της δομής της παρουσίασης και των εικόνων στη μνήμη. Κανονικά, οι εικόνες στη μνήμη καταλαμβάνουν περισσότερο χώρο από τον σκληρό δίσκο, ιδιαίτερα όταν έχουν πρόσθετα εφέ.

Γενικά, το Aspose.Slides for Android via Java μπορεί εύκολα να διαχειριστεί αρχεία παρουσίασης περίπου 300 MB σε διακομιστή με 4 GB RAM.

## **Εργασία με διαφάνειες**

**Q:** Μπορώ να αλλάξω το μέγεθος των διαφανειών σε μια παρουσίαση;

**A:** Μπορείτε να χρησιμοποιήσετε τη μέθοδο `getSlideSize` που εκτίθεται από την κλάση [Presentation](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/presentation/) για να ορίσετε το μέγεθος των διαφανειών σε μια παρουσίαση.

**Q:** Υπάρχει τρόπος να ορίσετε διαφάνειες διαφορετικού μεγέθους σε μια παρουσίαση;

**A:** Καθώς το μέγεθος των διαφανειών ορίζεται σε επίπεδο παρουσίασης στα έγγραφα Microsoft PowerPoint, δεν υπάρχει τρόπος να το γίνει αυτό.

**Q:** Υποστηρίζει το Aspose.Slides for Android via Java την προεπισκόπηση μιας διαφάνειας πριν από την αποθήκευση;

**A:** Μπορείτε να αποδώσετε τις διαφάνειες της παρουσίασης σε εικόνες και να χρησιμοποιήσετε αυτές τις εικόνες για την προεπισκόπηση των διαφανειών.

## **Εργασία με κείμενο**

**Q:** Είναι δυνατόν να ανακτήσετε όλο το κείμενο από μια παρουσίαση;

**A:** Το Aspose.Slides for Android via Java παρέχει την κλάση [SlideUtil](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/slideutil/) που προσφέρει διάφορες μεθόδους για την ανάκτηση ολόκληρου του κειμένου από τις παρουσιάσεις.

**Q:** Γιατί τα μεγέθη παραγράφων διαφέρουν μεταξύ PC και Android;

**A:** Ο υπολογισμός του μεγέθους των παραγράφων βασίζεται στον υπολογισμό του μεγέθους του κειμένου που αντιπροσωπεύει τη δεδομένη παράγραφο. Ο υπολογισμός του μεγέθους του κειμένου στηρίζεται στις μετρήσεις της γραμματοσειράς που ορίζεται στην παρουσίαση PowerPoint. Εάν η καθορισμένη γραμματοσειρά λείπει, αντικαθίσταται με την πιο παρόμοια γραμματοσειρά, αλλά αυτή έχει μετρήσεις διαφορετικές από τις αρχικές. Συνεπώς, ο υπολογισμός του μεγέθους των παραγράφων σε διαφορετικά συστήματα θα οδηγήσει σε διαφορετικά αποτελέσματα ανάλογα με το σύνολο των εγκατεστημένων γραμματοσειρών. Για να επιτύχετε το ίδιο αποτέλεσμα σε διαφορετικά λειτουργικά συστήματα, πρέπει να εγκαταστήσετε τις ίδιες γραμματοσειρές στα συστήματα ή να τις φορτώσετε κατά το χρόνο εκτέλεσης ως [εξωτερικές γραμματοσειρές](/slides/el/androidjava/custom-font/).

## **Μορφοποίηση και εικόνες**

**Q:** Πώς μπορώ να ορίσω το χρώμα του περιγράμματος ενός πίνακα;

**A:** Μπορείτε να αλλάξετε το χρώμα όλων των περιγραμμάτων του πίνακα ή μόνο του περιγράμματος γύρω από ολόκληρο τον πίνακα. Για την αλλαγή όλων των περιγραμμάτων, χρησιμοποιήστε τη μέθοδο `getCellFormat` από τη διεπαφή [ICell](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/icell/). Για το περίγραμμα ολόκληρου του πίνακα, πρέπει να επαναλάβετε τις κελιά και να αλλάξετε το χρώμα των εξωτερικών περιγραμμάτων.

**Q:** Τι μέτρηση χρησιμοποιεί το Aspose.Slides for Android via Java για την τοποθέτηση εικόνων;

**A:** Οι συντεταγμένες και τα μεγέθη όλων των σχημάτων στις διαφάνειες μετρώνται σε points (72 dpi).

## **Εργασία με γραμματοσειρές**

**Q:** Κατά τη μετατροπή PPT σε PDF ή εικόνες, γιατί οι γραμματοσειρές διαφέρουν στα έγγραφα εξόδου;

**A:** Το πρόβλημα αυτό ενδέχεται να υποδεικνύει ότι οι γραμματοσειρές που χρησιμοποιούνται στην παρουσίαση λείπουν από το λειτουργικό σύστημα στο οποίο εκτελέστηκε ο κώδικας. Θα πρέπει να εγκαταστήσετε τις γραμματοσειρές στο λειτουργικό σύστημα ή να τις φορτώσετε ως εξωτερικές γραμματοσειρές χρησιμοποιώντας την κλάση [FontsLoader](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/fontsloader/) όπως φαίνεται παρακάτω:
```java
String[] folders = new String[] { "path_to_a_folder_with_fonts" };
FontsLoader.loadExternalFonts(folders);
```
