## **Επισκόπηση**

Αυτό το άρθρο δείχνει πώς να εργάζεστε με ένα διάγραμμα δακτυλίου στο Aspose.Slides προσθέτοντας το διάγραμμα σε μια διαφάνεια, ορίζοντας το μέγεθος της κεντρικής τρύπας και αποθηκεύοντας την παρουσίαση. Επικεντρώνεται στη μέθοδο `setDoughnutHoleSize` και επιδεικνύει τα βασικά βήματα που απαιτούνται για την προσαρμογή αυτού του τύπου διαγράμματος στον κώδικα.

Περιλαμβάνει επίσης μια σύντομη ενότητα **FAQ** που καλύπτει σχετικά σενάρια με διαγράμματα δακτυλίου, όπως η χρήση πολλαπλών σειρών για τη δημιουργία πολλαπλών δαχτυλιδιών, η εργασία με «εκραγμένα» διαγράμματα δακτυλίου και η εξαγωγή ενός διαγράμματος ως ραστερική εικόνα ή SVG.

## **Καθορισμός του κεντρικού κενού σε διάγραμμα δακτυλίου**
{{% alert color="primary" %}} 
Το Aspose.Slides για Android μέσω Java υποστηρίζει πλέον τον καθορισμό του μεγέθους της τρύπας σε ένα διάγραμμα δακτυλίου. Σε αυτό το θέμα, θα δούμε με παράδειγμα πώς να ορίσουμε το μέγεθος της τρύπας σε ένα διάγραμμα δακτυλίου.
{{% /alert %}} 

Για να καθορίσετε το μέγεθος της τρύπας σε ένα διάγραμμα δακτυλίου, ακολουθήστε τα παρακάτω βήματα:

1. Δημιουργήστε αντικείμενο [Presentation](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/presentation) .
2. Προσθέστε διάγραμμα δακτυλίου στη διαφάνεια.
3. Καθορίστε το μέγεθος της τρύπας σε ένα διάγραμμα δακτυλίου.
4. Αποθηκεύστε την παρουσίαση στον δίσκο.

Στο παρακάτω παράδειγμα, έχουμε ορίσει το μέγεθος της τρύπας σε ένα διάγραμμα δακτυλίου.

```java
// Δημιουργία ενός αντικειμένου της κλάσης Presentation
Presentation pres = new Presentation();
try {
    IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(ChartType.Doughnut, 50, 50, 400, 400);
    
    chart.getChartData().getSeriesGroups().get_Item(0).setDoughnutHoleSize((byte)90);

    // Αποθήκευση της παρουσίασης στον δίσκο
    pres.save("DoughnutHoleSize_out.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```

## **Συχνές Ερωτήσεις**

**Μπορώ να δημιουργήσω ένα πολλαπλού επιπέδου διάγραμμα δακτυλίου με πολλαπλά δαχτυλίδια;**

Ναι. Προσθέστε πολλαπλές σειρές σε ένα ενιαίο διάγραμμα δακτυλίου — κάθε σειρά γίνεται ένα ξεχωριστό δαχτυλίδι. Η σειρά των δαχτυλιδιών καθορίζεται από τη σειρά των σειρών στη συλλογή.

**Υποστηρίζεται ένα «εκραγμένο» διάγραμμα δακτυλίου (διαχωρισμένα κομμάτια);**

Ναι. Υπάρχει τύπος διαγράμματος Exploded Doughnut [chart type](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/charttype/) και μια ιδιότητα explosion στα σημεία δεδομένων· μπορείτε να διαχωρίσετε μεμονωμένα κομμάτια.

**Πώς μπορώ να λάβω μια εικόνα ενός διαγράμματος δακτυλίου (PNG/SVG) για μια αναφορά;**

Ένα διάγραμμα είναι σχήμα· μπορείτε να το αποδώσετε σε μια [raster image](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/shape/#getImage-int-float-float-) ή να εξάγετε το διάγραμμα σε μια [SVG image](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/shape/#writeAsSvg-java.io.OutputStream-com.aspose.slides.ISVGOptions-).
