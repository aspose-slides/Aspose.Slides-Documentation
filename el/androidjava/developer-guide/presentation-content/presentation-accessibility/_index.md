## **Επισκόπηση**

Η προσβασιμότητα παρουσιάσεων εξασφαλίζει ότι τα άτομα που χρησιμοποιούν βοηθητικές τεχνολογίες — όπως αναγνώστες οθόνης, οθόνες braille ή πλοήγηση μόνο με πληκτρολόγιο — μπορούν να κατανοήσουν και να περιηγηθούν στις διαφάνειές σας τόσο αποτελεσματικά όσο και οι θεατές με όραση και χρήση ποντικιού. Η καλή πρακτική εστιάζει σε καθαρή σειρά ανάγνωσης, ουσιαστικό εναλλακτικό κείμενο για πληροφοριακά γραφικά, επαρκή αντίθεση χρωμάτων, ευανάγνωστη τυπογραφία, περιγραφικό κείμενο συνδέσμου και αποφυγή μετάδοσης σημασίας μόνο μέσω χρώματος ή θέσης. Όταν η προσβασιμότητα προγραμματίζεται από την αρχή, το αποτέλεσμα είναι μια πιο καθαρή δομή, πιο συνεπή γραφικά και περιεχόμενο που φθάνει σε κάθε θεατή χωρίς εναλλακτικές λύσεις.

## **Σήμανση ως διακοσμητικό**

Η σήμανση ως διακοσμητικό σημαδέψει καθαρά διακοσμητικά γραφικά ώστε οι αναγνώστες οθόνης να τα παραλείπουν, μειώνοντας το θόρυβο και διατηρώντας την εστίαση στο ουσιαστικό περιεχόμενο. Εφαρμόστε το σε φόντα, διακοσμητικά στοιχεία και διαχωριστικά — ποτέ σε γραφήματα, εικονίδια ή εικόνες που μεταδίδουν πληροφορίες. Η Aspose.Slides εκθέτει αυτή τη σημαία για τον εντοπισμό και την επικύρωση, επιτρέποντας αυτοματοποιημένους ελέγχους προσβασιμότητας και καθαρισμό.

![Σήμανση ως διακοσμητικό](mark_as_decorative.png)

Το παρακάτω παράδειγμα κώδικα δείχνει πώς να προσδιορίσετε αν ένα σχήμα έχει σημειωθεί ως διακοσμητικό.

```java
Presentation presentation = new Presentation("sample.pptx");
try {
    IShape shape = presentation.getSlides().get_Item(0).getShapes().get_Item(0);
    System.out.println("Is shape decorative: " + shape.isDecorative());
} finally {
    presentation.dispose();
}
```
