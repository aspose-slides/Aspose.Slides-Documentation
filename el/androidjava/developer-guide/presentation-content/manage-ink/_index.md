## **Εισαγωγή**

Το PowerPoint παρέχει τη λειτουργία μελάνης ώστε να μπορείτε να σχεδιάζετε μη τυπικά σχήματα, τα οποία μπορούν να χρησιμοποιηθούν για να τονίσουν άλλα αντικείμενα, να δείξουν συνδέσεις και διαδικασίες, και να τραβήξουν την προσοχή σε συγκεκριμένα στοιχεία σε μια διαφάνεια.  

Το Aspose.Slides παρέχει όλους τους τύπους Ink (π.χ.[Ink](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/ink/) κλάση) που χρειάζεστε για να δημιουργήσετε και να διαχειριστείτε αντικείμενα μελάνης.

## **Διαφορές μεταξύ Κανονικών Αντικειμένων και Αντικειμένων Μελάνης**

Τα αντικείμενα σε μια διαφάνεια του PowerPoint συνήθως αναπαρίστανται από αντικείμενα σχήματος. Ένα αντικείμενο σχήματος, στην απλούστερη του μορφή, είναι ένας περιέκτης που ορίζει την περιοχή του ίδιου του αντικειμένου (το πλαίσιο του) μαζί με τις ιδιότητές του. Τα τελευταία περιλαμβάνουν το μέγεθος της περιοχής του περιέκτη, το σχήμα του περιέκτη, το φόντο του περιέκτη κ.λπ. Για περισσότερες πληροφορίες, δείτε [Shape Layout Format](https://docs.aspose.com/slides/el/androidjava/shape-manipulations/#access-layout-formats-for-shape).

Ωστόσο, όταν το PowerPoint διαχειρίζεται ένα αντικείμενο μελάνης, αγνοεί όλες τις ιδιότητες του πλαισίου του αντικειμένου (του περιέκτη) εκτός από το μέγεθός του. Το μέγεθος της περιοχής του περιέκτη καθορίζεται από τις τυπικές τιμές `width` και `height`:

## **Ίχνη Μελάνης**

Το ίχνος είναι ένα βασικό στοιχείο ή πρότυπο που χρησιμοποιείται για την καταγραφή της τροχιάς ενός στυλό όταν ο χρήστης γράφει ψηφιακή μελάνη. Τα ίχνη είναι εγγραφές που περιγράφουν ακολουθίες συνδεδεμένων σημείων.  

Η πιο απλή μορφή κωδικοποίησης καθορίζει τις συντεταγμένες X και Y κάθε σημείου δείγματος. Όταν όλα τα συνδεδεμένα σημεία αποδοθούν, παράγουν μια εικόνα όπως αυτή:

## **Ιδιότητες Πινέλου για Σχεδίαση**

Μπορείτε να χρησιμοποιήσετε ένα πινέλο για να σχεδιάσετε γραμμές που συνδέουν τα σημεία των στοιχείων του ίχνους. Το πινέλο έχει το δικό του χρώμα και μέγεθος, που αντιστοιχούν στις ιδιότητες `Brush.Color` και `Brush.Size`.

### **Ορισμός Χρώματος Πινέλου Μελάνης**

Αυτός ο κώδικας Java δείχνει πώς να ορίσετε το χρώμα για ένα πινέλο:

```java
Presentation pres = new Presentation("pres.pptx");
try {
    IInk ink = (IInk)pres.getSlides().get_Item(0).getShapes().get_Item(0);
    IInkTrace[] traces = ink.getTraces();
    IInkBrush brush = traces[0].getBrush();
    Color brushColor = brush.getColor();
    brush.setColor(Color.RED);
} finally {
    if (pres != null) pres.dispose();
}
```

### **Ορισμός Μεγέθους Πινέλου Μελάνης** 

Αυτός ο κώδικας Java δείχνει πώς να ορίσετε το μέγεθος για ένα πινέλο:

```java
Presentation pres = new Presentation("pres.pptx");
try {
    IInk ink = (IInk)pres.getSlides().get_Item(0).getShapes().get_Item(0);
    IInkTrace[] traces = ink.getTraces();
    IInkBrush brush = traces[0].getBrush();
    Dimension2D brushSize = brush.getSize();
    brush.setSize(new Dimension(5, 10));
} finally {
    if (pres != null) pres.dispose();
}
```

Γενικά, το πλάτος και το ύψος ενός πινέλου δεν ταιριάζουν, έτσι το PowerPoint δεν εμφανίζει το μέγεθος του πινέλου (η ενότητα δεδομένων είναι γκριζαρισμένη). Ωστόσο, όταν το πλάτος και το ύψος του πινέλου είναι ίσα, το PowerPoint εμφανίζει το μέγεθός του με αυτόν τον τρόπο:

![ink_powerpoint3](ink_powerpoint3.png)

Για σαφήνεια, ας αυξήσουμε το ύψος του αντικειμένου μελάνης και να εξετάσουμε τις σημαντικές διαστάσεις:

![ink_powerpoint4](ink_powerpoint4.png)

Ο περιέκτης (πλαίσιο) δεν λαμβάνει υπόψη το μέγεθος των πινέλων — πάντα υποθέτει ότι το πάχος της γραμμής είναι μηδέν (δείτε την τελευταία εικόνα).

Ως εκ τούτου, για να καθορίσουμε την ορατή περιοχή ολόκληρου του αντικειμένου μελάνης, πρέπει να λάβουμε υπόψη το μέγεθος του πινέλου των αντικειμένων ίχνους. Εδώ, το αντικείμενο‑στόχος (το αντικείμενο ίχνους χειρόγραφου κειμένου) έχει κλιμακωθεί στο μέγεθος του περιέκτη (πλαισίου). Όταν αλλάζει το μέγεθος του περιέκτη (πλαισίου), το μέγεθος του πινέλου παραμένει σταθερό και αντίστροφα.

![ink_powerpoint5](ink_powerpoint5.png)
