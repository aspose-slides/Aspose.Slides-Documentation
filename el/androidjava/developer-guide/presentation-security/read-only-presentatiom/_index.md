## **Εισαγωγή**

Στο PowerPoint 2019, η Microsoft παρουσίασε τη ρύθμιση **Always Open Read-Only** ως μία από τις επιλογές που μπορούν να χρησιμοποιήσουν οι χρήστες για να προστατεύσουν τις παρουσιάσεις τους. Μπορεί να θέλετε να χρησιμοποιήσετε αυτήν τη ρύθμιση Μόνο προς Ανάγνωση για να προστατεύσετε μια παρουσίαση όταν

- Θέλετε να αποτρέψετε τυχαίες επεμβάσεις και να διατηρήσετε το περιεχόμενο της παρουσίασής σας ασφαλές. 
- Θέλετε να ενημερώσετε τους ανθρώπους ότι η παρουσίαση που παρέχετε είναι η τελική έκδοση. 

Αφού επιλέξετε την επιλογή **Always Open Read-Only** για μια παρουσίαση, όταν οι χρήστες ανοίγουν την παρουσίαση, βλέπουν την πρόταση **Read-Only** και μπορεί να εμφανιστεί το ακόλουθο μήνυμα: *Για να αποτρέψετε τυχαίες αλλαγές, ο δημιουργός έχει ορίσει αυτό το αρχείο να ανοίγει μόνο για ανάγνωση.*

Η πρόταση **Read-Only** είναι μια απλή αλλά αποτελεσματική αποτροπή που αποθαρρύνει την επεξεργασία, επειδή οι χρήστες πρέπει να εκτελέσουν μια ενέργεια για να την αφαιρέσουν πριν τους επιτραπεί η επεξεργασία μιας παρουσίασης. Εάν δεν θέλετε οι χρήστες να κάνουν αλλαγές σε μια παρουσίαση και θέλετε να το γνωστοποιήσετε με ευγενικό τρόπο, τότε η πρόταση **Read-Only** μπορεί να είναι μια καλή επιλογή για εσάς. 

> Εάν μια παρουσίαση με την προστασία **Read-Only** ανοίξει σε παλαιότερη εφαρμογή Microsoft PowerPoint—που δεν υποστηρίζει τη πρόσφατα εισαγόμενη λειτουργία—η πρόταση **Read-Only** αγνοείται (η παρουσίαση ανοίγει κανονικά).

## **Εφαρμογή λειτουργίας Μόνο προς ανάγνωση**

Το Aspose.Slides for Android μέσω Java σας επιτρέπει να ορίσετε μια παρουσίαση σε **Read-Only**, πράγμα που σημαίνει ότι οι χρήστες (αφού ανοίξουν την παρουσίαση) βλέπουν την πρόταση **Read-Only**. Αυτό το παράδειγμα κώδικα σας δείχνει πώς να ορίσετε μια παρουσίαση σε **Read-Only** σε Java χρησιμοποιώντας το Aspose.Slides:

```java
Presentation pres = new Presentation();
try {
    pres.getProtectionManager().setReadOnlyRecommended(true);
    pres.save("ReadOnlyPresentation.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```

{{% alert color="primary" %}} 

**Σημείωση**: Η πρόταση **Read-Only** προορίζεται απλώς για να αποθαρρύνει την επεξεργασία ή να εμποδίσει τους χρήστες από τυχαίες αλλαγές σε μια παρουσίαση PowerPoint. Εάν ένα πρόθυμο άτομο—που ξέρει τι κάνει—αποφασίσει να επεξεργαστεί την παρουσίασή σας, μπορεί εύκολα να αφαιρέσει τη ρύθμιση Read-Only. Εάν χρειάζεστε σοβαρά την αποτροπή μη εξουσιοδοτημένης επεξεργασίας, είναι καλύτερο να χρησιμοποιήσετε [πιο αυστηρές προστασίες που περιλαμβάνουν κρυπτογραφήσεις και κωδικούς πρόσβασης](https://docs.aspose.com/slides/el/androidjava/password-protected-presentation/).

## **Συχνές ερωτήσεις**

**Πώς διαφέρει το 'Read-Only recommended' από την πλήρη προστασία με κωδικό;**

`Read-Only recommended` εμφανίζει μόνο μια πρόταση να ανοίξετε το αρχείο σε λειτουργία μόνο για ανάγνωση και είναι εύκολο να παρακαμφθεί. Η [προστασία με κωδικό](/slides/el/androidjava/password-protected-presentation/) περιορίζει πραγματικά το άνοιγμα ή την επεξεργασία και είναι κατάλληλη όταν χρειάζεστε πραγματικό έλεγχο ασφαλείας.

**Μπορεί το 'Read-Only recommended' να συνδυαστεί με υδατογραφήματα για να αποθαρρύνει περαιτέρω τις επεξεργασίες;**

Ναι. Η πρόταση μπορεί να συνδυαστεί με τα [υδατογραφήματα](/slides/el/androidjava/watermark/) ως οπτική αποτροπή· είναι ξεχωριστοί μηχανισμοί και λειτουργούν καλά μαζί.

**Μπορεί μια μακροεντολή ή εξωτερικό εργαλείο να τροποποιήσει ακόμη το αρχείο όταν η πρόταση είναι ενεργοποιημένη;**

Ναι. Η πρόταση δεν εμποδίζει τις προγραμματιστικές αλλαγές. Για να αποτρέψετε αυτόματες επεμβάσεις, χρησιμοποιήστε [κωδικούς πρόσβασης και κρυπτογράφηση](/slides/el/androidjava/password-protected-presentation/).

**Πώς σχετίζεται το 'Read-Only recommended' με τις μεθόδους 'isEncrypted' και 'isWriteProtected';**

Είναι διαφορετικά σήματα. Το 'Read-Only recommended' είναι μια ήπια, προαιρετική προτροπή· το [isWriteProtected](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/protectionmanager/#isWriteProtected--) και το [isEncrypted](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/protectionmanager/#isEncrypted--) υποδεικνύουν πραγματικούς περιορισμούς εγγραφής ή ανάγνωσης που εξαρτώνται από κωδικούς πρόσβασης ή κρυπτογράφηση.
