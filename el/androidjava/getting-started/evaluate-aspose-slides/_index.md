## **Aspose.Slides Αξιολόγηση**

Μπορείτε εύκολα να κατεβάσετε το Aspose.Slides για αξιολόγηση. Το πακέτο αξιολόγησης είναι το ίδιο με το αγορασμένο πακέτο. Η έκδοση αξιολόγησης γίνεται απλώς αδειοδοτημένη μετά την προσθήκη μερικών γραμμών κώδικα για την εφαρμογή της άδειας.

Η έκδοση αξιολόγησης του Aspose.Slides (χωρίς καθορισμένη άδεια) παρέχει πλήρη λειτουργικότητα του προϊόντος, αλλά τοποθετεί ένα υδατογράφημα αξιολόγησης στην κορυφή του εγγράφου κατά το άνοιγμα και την αποθήκευση. Επίσης περιορίζεστε σε μία διαφάνεια κατά την εξαγωγή κειμένων από διαφάνειες παρουσίασης.

## **Συχνές Ερωτήσεις**

**Μπορώ να δοκιμάσω πολλαπλές παρουσιάσεις παράλληλα σε διαφορετικά νήματα σε λειτουργία αξιολόγησης;**

Ναι. Μπορείτε να επεξεργαστείτε διαφορετικά έγγραφα παράλληλα· δεν πρέπει να μοιράζεστε το ίδιο αντικείμενο παρουσίασης [μέσω νημάτων](/slides/el/androidjava/multithreading/). Η λειτουργία αξιολόγησης δεν επηρεάζει αυτό.

**Χρειάζεται να εγκαταστήσω το Microsoft PowerPoint για να αξιολογήσω τη βιβλιοθήκη σε server ή σε CI;**

Όχι. Το Aspose.Slides είναι μια αυτόνομη μηχανή και δεν απαιτεί την εγκατάσταση του PowerPoint, είτε για αξιολόγηση είτε για παραγωγή.

**Μπορώ να δοκιμάσω πλήρως τη μετατροπή PPT/PPTX σε PDF και εικόνες σε λειτουργία αξιολόγησης;**

Ναι. Οι [μετατροπείς](/slides/el/androidjava/convert-presentation/) λειτουργούν· η έξοδος θα περιέχει υδατογράφημα.

**Μπορώ να χρησιμοποιήσω προσωρινή άδεια για δοκιμές φόρτισης χωρίς υδατογράφημα;**

Ναι. Μια 30‑ημερή προσωρινή άδεια αφαιρεί τους περιορισμούς της λειτουργίας αξιολόγησης και επιτρέπει δοκιμές χωρίς υδατογράφημα.
