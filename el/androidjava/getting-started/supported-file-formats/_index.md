## **Επισκόπηση**

Aspose.Slides supports presentation files from Microsoft PowerPoint 97 through Office 365, including Microsoft PowerPoint for Mac. This article lists the PowerPoint versions supported by the library and provides a table of file formats that can be loaded, saved, or both.

The article also answers common questions about PDF compliance, font embedding, password-protected files, custom fonts, font fallback, and XPS export options.

## **Υποστηριζόμενες Εκδόσεις του Microsoft PowerPoint**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint for MAC
- Office 365

## **Συχνές Ερωτήσεις**

**Μπορώ να αποθηκεύσω παρουσιάσεις σε PDF που πληρούν τα πρότυπα αρχειοθέτησης και προσβασιμότητας (PDF/A και PDF/UA);**

Ναι. Το Aspose.Slides υποστηρίζει εξαγωγή σε PDF με επίπεδα συμμόρφωσης όπως PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b, καθώς και PDF/UA μέσω της ρύθμισης [συμμόρφωση](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/pdfoptions/#setCompliance-int-) στις [επιλογές εξαγωγής PDF](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/pdfoptions/).

**Υποστηρίζει η βιβλιοθήκη ενσωμάτωση γραμματοσειρών κατά την εξαγωγή σε PDF, με λεπτομερή έλεγχο του τι ενσωματώνεται;**

Ναι. Μπορείτε να ελέγξετε αν οι γραμματοσειρές θα ενσωματωθούν πλήρως ή μόνο με υποσύνολο (μόνο τα χρησιμοποιημένα γλυφικά), να καθορίσετε πώς αντιμετωπίζονται οι κοινές γραμματοσειρές συστήματος και να ρυθμίσετε τη συμπεριφορά για κείμενο ASCII μέσω των [επιλογών εξαγωγής PDF](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/pdfoptions/).

**Μπορώ να ανιχνεύσω αν ένα αρχείο είναι κωδικοποημένο πριν το φορτώσω;**

Ναι. Χρησιμοποιώντας το [API ελέγχου βασισμένο σε εργοστάσιο](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/presentationfactory/), μπορείτε να ερωτήσετε ένα αρχείο παρουσίασης για να καθορίσετε αν είναι κωδικοποημένο χωρίς να το ανοίξετε πλήρως.

**Υπάρχουν μηχανισμοί εναλλακτικών γραμματοσειρών και υποστήριξη προσαρμοσμένων γραμματοσειρών;**

Ναι. Η βιβλιοθήκη υποστηρίζει την [φόρτωση](/slides/el/androidjava/custom-font/) και την [ενσωμάτωση](/slides/el/androidjava/embedded-font/) προσαρμοσμένων γραμματοσειρών και παρέχει [κανόνες εναλλακτικών γραμματοσειρών](/slides/el/androidjava/fallback-font/) για την αποφυγή ελλιπών γλυφικών κατά την απόδοση και τη μετατροπή.

**Μπορώ να εξάγω διαφάνειες σε XPS, και υπάρχουν επιλογές για τη ρύθμιση της εξόδου XPS;**

Ναι. Η [εξαγωγή σε XPS](/slides/el/androidjava/convert-powerpoint-to-xps/) υποστηρίζεται, και μπορείτε να προσαρμόσετε τις σχετικές [επιλογές αποθήκευσης](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/xpsoptions/) για να ελέγξετε την ποιότητα και το περιεχόμενο της εξόδου του εγγράφου XPS.
