## **Επισκόπηση**

Το Aspose.Slides μπορεί να χρησιμοποιηθεί σε λειτουργία αξιολόγησης ή με έγκυρη άδεια. Η εκδοχή αξιολόγησης παρέχει την ίδια λειτουργικότητα με την εκδοχή με άδεια, αλλά προσθέτει υδατογράφημα αξιολόγησης όταν οι παρουσιάσεις ανοίγονται ή αποθηκεύονται και περιορίζει την εξαγωγή κειμένου σε μία διαφάνεια.

Αυτό το άρθρο εξηγεί πώς λειτουργεί η αδειοδότηση στο Aspose.Slides και πώς να εφαρμόσετε μια άδεια πριν χρησιμοποιήσετε τη βιβλιοθήκη. Μια άδεια μπορεί να φορτωθεί από αρχείο, ρεύμα ή ενσωματωμένος πόρος χρησιμοποιώντας την κλάση `License`. Το άρθρο επίσης δείχνει πώς να επικυρώσετε εάν μια άδεια έχει εφαρμοστεί σωστά.

## **Αδειοδότηση στο Aspose.Slides**

* Μια έκδοση αξιολόγησης γίνεται αδειοδοτημένη αφού αγοράσετε μια άδεια και προσθέσετε μερικές γραμμές κώδικα (για να εφαρμόσετε την άδεια).
* Η άδεια είναι ένα αρχείο XML απλού κειμένου που περιέχει λεπτομέρειες όπως το όνομα προϊόντος, ο αριθμός των προγραμματιστών στους οποίους αδειοδοτείται, η ημερομηνία λήξης της συνδρομής κ.λπ.
* Το αρχείο άδειας είναι ψηφιακά υπογεγραμμένο, οπότε δεν πρέπει να το τροποποιήσετε. Ακόμη και η τυχαία προσθήκη μιας επιπλέον αλλαγής γραμμής στο περιεχόμενο του αρχείου θα το ακυρώσει.
* Το Aspose.Slides for Android via Java συνήθως προσπαθεί να βρει την άδεια σε αυτές τις τοποθεσίες:
  * Ένα ρητό μονοπάτι
  * Ο φάκελος που περιέχει το Aspose.Slides.jar
* Για να αποφύγετε τους περιορισμούς που σχετίζονται με την έκδοση αξιολόγησης, πρέπει να ορίσετε μια άδεια πριν χρησιμοποιήσετε το **Aspose.Slides**. Πρέπει να ορίσετε μια άδεια μόνο μία φορά ανά εφαρμογή ή διαδικασία.

## **Εφαρμογή άδειας**

Μια άδεια μπορεί να φορτωθεί από **αρχείο** ή **ρεύμα**.

{{% alert color="primary" %}}

Το Aspose.Slides παρέχει την κλάση [License](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/license/) για λειτουργίες αδειοδότησης.

{{% /alert %}} 

{{% alert color="warning" %}}

Νέες άδειες μπορούν να ενεργοποιήσουν το Aspose.Slides μόνο με την έκδοση 21.4 ή νεότερη. Παλαιότερες εκδόσεις χρησιμοποιούν διαφορετικό σύστημα αδειοδότησης και δεν θα αναγνωρίσουν αυτές τις άδειες.

{{% /alert %}}

### **Αρχείο**

Η πιο εύκολη μέθοδος ορισμού άδειας απαιτεί να τοποθετήσετε το αρχείο άδειας στον φάκελο που περιέχει το Aspose.Slides.jar ή το .jar της εφαρμογής σας.

Αυτός ο κώδικας Java δείχνει πώς να ορίσετε ένα αρχείο άδειας:

``` java
// Δημιουργεί την κλάση License
com.aspose.slides.License license = new com.aspose.slides.License();

// Ορίζει τη διαδρομή του αρχείου άδειας
license.setLicense("Aspose.Slides.Android.via.Java.lic");
```

{{% alert color="warning" %}} 

Αν τοποθετήσετε το αρχείο άδειας σε διαφορετικό φάκελο, όταν καλέσετε τη μέθοδο [SetLicense](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/license/#setLicense-java.lang.String-), το όνομα του αρχείου άδειας στο τέλος του συγκεκριμένου ρητού πρέπει να είναι το ίδιο με το αρχείο άδειας σας.

Για παράδειγμα, μπορείτε να αλλάξετε το όνομα του αρχείου άδειας σε *Aspose.Slides.Android.via.Java.lic.xml*. Έτσι, στον κώδικά σας, πρέπει να περάσετε τη διαδρομή προς το αρχείο (τελειώνοντας με *Aspose.Slides.Android.via.Java.lic.xml*) στη μέθοδο [SetLicense](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/license/#setLicense-java.lang.String-).

{{% /alert %}}

### **Ρεύμα**

Μπορείτε να φορτώσετε μια άδεια από ρεύμα. Αυτός ο κώδικας Java δείχνει πώς να εφαρμόσετε μια άδεια από ρεύμα:

``` java
// Αρχικοποιεί την κλάση License
com.aspose.slides.License license = new com.aspose.slides.License();

// Ορίζει την άδεια μέσω ροής
license.setLicense(new java.io.FileInputStream("Aspose.Slides.Android.via.Java.lic"));
```

## **Επικύρωση άδειας**

Για να ελέγξετε εάν η άδεια έχει οριστεί σωστά, μπορείτε να την επικυρώσετε. Αυτός ο κώδικας Java δείχνει πώς να επικυρώσετε μια άδεια:

```java
License license = new License();
license.setLicense("Aspose.Slides.Android.via.Java.lic");

if (License.isLicensed()) 
{
    System.out.println("License is good!");
}
```

## **Ασφάλεια νήματος**

{{% alert title="Note" color="warning" %}} 

Η μέθοδος [SetLicense](https://reference.aspose.com/slides/el/androidjava/com.aspose.slides/license/#setLicense-java.io.InputStream-) δεν είναι ασφαλής για νήματα. Εάν αυτή η μέθοδος πρέπει να κληθεί ταυτόχρονα από πολλά νήματα, ίσως θελήσετε να χρησιμοποιήσετε πρωτόκολλα συγχρονισμού (όπως κλείδα) για να αποφύγετε προβλήματα. 

{{% /alert %}}

## **FAQ**

**Μπορώ να εφαρμόσω την άδεια σε ένα απολύτως εκτός σύνδεσης περιβάλλον (χωρίς πρόσβαση στο διαδίκτυο);**

Ναι. Η επικύρωση της άδειας πραγματοποιείται τοπικά χρησιμοποιώντας το αρχείο άδειας· δεν απαιτείται σύνδεση στο διαδίκτυο.

**Τι συμβαίνει μετά τη λήξη της ετήσιας συνδρομής; Θα σταματήσει η βιβλιοθήκη να λειτουργεί;**

Όχι. Η άδεια είναι διαρκής: μπορείτε να συνεχίσετε να χρησιμοποιείτε εκδόσεις που κυκλοφόρησαν πριν από την ημερομηνία λήξης της συνδρομής σας· απλώς δεν θα μπορείτε να χρησιμοποιήσετε νεότερες εκδόσεις χωρίς ανανέωση.
