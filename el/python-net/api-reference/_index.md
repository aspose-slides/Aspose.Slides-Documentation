{{% alert color="primary" %}} 

Aspose.Slides for Python via .NET είναι μια βιβλιοθήκη κλάσεων που επιτρέπει σε εφαρμογές Python να διαβάζουν και να γράφουν αρχεία παρουσίασης χωρίς να χρησιμοποιούν το Microsoft PowerPoint® ή άλλο λογισμικό. Aspose.Slides for Python via .NET σας επιτρέπει να διαχειρίζεστε κείμενο, σχήματα, πίνακες και κινούμενα σχέδια· προσθέτετε ήχο και βίντεο σε διαφάνειες· προεπισκοπείτε διαφάνειες· και εξάγετε διαφάνειες σε SVG, PDF μορφή και άλλα.

Η πιο πρόσφατη Αναφορά API μπορεί να βρεθεί στο [Αναφορά API του Aspose.Slides για Python μέσω .NET](https://reference.aspose.com/slides/el/python-net/).
