## **Εισαγωγή**

Η χρεωστική άδεια είναι ένας μηχανισμός αδειοδότησης που μπορεί να χρησιμοποιηθεί παράλληλα με τις υπάρχουσες μεθόδους αδειοδότησης. Εάν θέλετε να χρεώεστε βάσει της χρήσης των λειτουργιών του Aspose.Slides API, επιλέγετε τη χρεωστική άδεια.

## **Εφαρμογή κλειδιών χρέωσης**

{{% alert color="primary" %}} 

Η χρεωστική άδεια είναι ένας νέος μηχανισμός αδειοδότησης που μπορεί να χρησιμοποιηθεί παράλληλα με τις υπάρχουσες μεθόδους αδειοδότησης. Εάν θέλετε να χρεώεστε βάσει της χρήσης των λειτουργιών του Aspose.Slides API, επιλέγετε τη χρεωστική άδεια.

Όταν αγοράζετε μια χρεωστική άδεια, λαμβάνετε κλειδιά (και όχι ένα αρχείο άδειας). Αυτό το κλειδί χρέωσης μπορεί να εφαρμοστεί χρησιμοποιώντας την κλάση [Metered](https://reference.aspose.com/slides/el/python-net/aspose.slides/metered/) που παρέχει η Aspose για λειτουργίες χρέωσης. Για περισσότερες λεπτομέρειες, δείτε το [Συχνές ερωτήσεις χρεωστικής άδειας](https://purchase.aspose.com/faqs/licensing/metered).

{{% /alert %}} 

1. Δημιουργήστε ένα στιγμιότυπο της κλάσης [Metered](https://reference.aspose.com/slides/el/python-net/aspose.slides/metered/).
2. Περνάτε τα δημόσια και ιδιωτικά κλειδιά σας στη μέθοδο [set_metered_key](https://reference.aspose.com/slides/el/python-net/aspose.slides/metered/set_metered_key/#str-str).
3. Εκτελέστε μερική επεξεργασία (εκτελέστε εργασίες).
4. Καλέστε τη μέθοδο [get_consumption_quantity](https://reference.aspose.com/slides/el/python-net/aspose.slides/metered/get_consumption_quantity/#) της κλάσης `Metered`.

Θα πρέπει να δείτε την ποσότητα/την αριθμητική τιμή των αιτημάτων API που έχετε καταναλώσει μέχρι τώρα.

Αυτός ο κώδικας δείγματος δείχνει πώς να χρησιμοποιήσετε τη χρεωστική άδεια:

```python
import aspose.slides as slides

# Δημιουργεί ένα στιγμιότυπο της κλάσης Metered
metered = slides.Metered()

# Περνά τα δημόσια και ιδιωτικά κλειδιά στο αντικείμενο Metered
metered.set_metered_key("<valid public key>", "<valid private key>")

# Λαμβάνει την τιμή της καταναλωμένης ποσότητας πριν από τις κλήσεις API
amount_before = slides.Metered.get_consumption_quantity()
print("Amount consumed before:", amount_before)

# Κάντε κάτι με το API του Aspose.Slides εδώ
# ...

# Λαμβάνει την τιμή της καταναλωμένης ποσότητας μετά από τις κλήσεις API
amount_after = slides.Metered.get_consumption_quantity()
print("Amount consumed after:", amount_after)
```

{{% alert color="warning" title="NOTE"  %}} 

Για να χρησιμοποιήσετε τη χρεωστική άδεια, χρειάζεστε μια σταθερή σύνδεση στο διαδίκτυο, επειδή ο μηχανισμός αδειοδότησης χρησιμοποιεί το διαδίκτυο για συνεχής αλληλεπίδραση με τις υπηρεσίες μας και για τη διενέργεια υπολογισμών.

{{% /alert %}} 

## **Συχνές ερωτήσεις**

**Μπορώ να χρησιμοποιήσω μια χρεωστική άδεια μαζί με μια κανονική (μόνιμη ή προσωρινή) στην ίδια εφαρμογή;**

Ναι. Η χρεωστική άδεια είναι ένας πρόσθετος μηχανισμός αδειοδότησης που μπορεί να χρησιμοποιηθεί παράλληλα με τις υπάρχουσες [μεθόδους αδειοδότησης](/slides/el/python-net/licensing/). Επιλέγετε ποιον μηχανισμό θα εφαρμόσετε όταν η εφαρμογή ξεκινά.

**Τι ακριβώς μετράται ως κατανάλωση σε μια χρεωστική άδεια: λειτουργίες ή αρχεία;**

Μετριέται η χρήση του API, δηλαδή ο αριθμός των αιτημάτων ή των λειτουργιών. Μπορείτε να λάβετε την τρέχουσα κατανάλωση μέσω των [μεθόδων παρακολούθησης κατανάλωσης](https://reference.aspose.com/slides/el/python-net/aspose.slides/metered/).

**Είναι η χρεωστική άδεια κατάλληλη για μικροϋπηρεσίες και περιβάλλοντα serverless όπου οι παρουσίες επανεκκινούνται συχνά;**

Ναι. Δεδομένου ότι η λογιστική γίνεται σε επίπεδο κλήσης API, σενάρια με συχνές κρύες εκκινήσεις είναι συμβατά, υπό την προϋπόθεση ότι υπάρχει σταθερή πρόσβαση στο δίκτυο για τους υπολογισμούς χρέωσης.

**Διαφέρει η λειτουργικότητα της βιβλιοθήκης όταν χρησιμοποιείται μια χρεωστική άδεια σε σύγκριση με μια μόνιμη άδεια;**

Όχι. Πρόκειται μόνο για τον μηχανισμό αδειοδότησης και χρέωσης· οι δυνατότητες του προϊόντος παραμένουν οι ίδιες.

**Πώς σχετίζεται η χρεωστική άδεια με την δοκιμαστική έκδοση και την προσωρινή άδεια;**

Η δοκιμαστική έκδοση έχει περιορισμούς και υδατογραφήματα, η [προσωρινή άδεια](https://purchase.aspose.com/temporary-license/) αφαιρεί περιορισμούς για 30 ημέρες, ενώ η χρεωστική άδεια αφαιρεί περιορισμούς και χρεώνει βάσει της πραγματικής χρήσης.

**Μπορώ να ελέγξω τον προϋπολογισμό αντιδρώντας αυτόματα όταν ξεπερνάται ένα όριο κατανάλωσης;**

Ναι. Μια κοινή πρακτική είναι να διαβάζετε περιοδικά την τρέχουσα κατανάλωση μέσω των [μεθόδων παρακολούθησης](https://reference.aspose.com/slides/el/python-net/aspose.slides/metered/) και να εφαρμόζετε τα δικά σας όρια ή ειδοποιήσεις σε επίπεδο εφαρμογής ή παρακολούθησης.
