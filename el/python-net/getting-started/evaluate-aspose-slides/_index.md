## **Αξιολόγηση Aspose.Slides**

Μπορείτε εύκολα να κατεβάσετε το Aspose.Slides για αξιολόγηση. Το πακέτο αξιολόγησης είναι το ίδιο με το αγορασμένο πακέτο. Η έκδοση αξιολόγησης γίνεται απλώς αδειοδοτημένη αφού προσθέσετε μερικές γραμμές κώδικα για την εφαρμογή της άδειας. 

Η έκδοση αξιολόγησης του Aspose.Slides (χωρίς καθορισμένη άδεια) παρέχει πλήρη λειτουργικότητα του προϊόντος, αλλά προσθέτει υδατογράφημα αξιολόγησης στην κορυφή του εγγράφου κατά το άνοιγμα και την αποθήκευση. Επίσης περιορίζεστε σε μία διαφάνεια όταν εξάγετε κείμενο από τις διαφάνειες παρουσίασης.

## **Συχνές ερωτήσεις**

**Μπορώ να δοκιμάσω πολλαπλές παρουσιάσεις ταυτόχρονα σε διαφορετικά νήματα σε λειτουργία αξιολόγησης;**

Ναι. Μπορείτε να επεξεργάζεστε διαφορετικά έγγραφα ταυτόχρονα· δεν πρέπει να μοιράζεστε το ίδιο αντικείμενο παρουσίασης [σε νήματα](/slides/el/python-net/multithreading/). Η λειτουργία αξιολόγησης δεν επηρεάζει αυτό.

**Χρειάζεται να εγκαταστήσω το Microsoft PowerPoint για να αξιολογήσω τη βιβλιοθήκη σε διακομιστή ή σε CI;**

Όχι. Το Aspose.Slides είναι μια αυτόνομη μηχανή και δεν απαιτεί εγκατεστημένο το PowerPoint, είτε για αξιολόγηση είτε για παραγωγή.

**Μπορώ να δοκιμάσω πλήρως τη μετατροπή PPT/PPTX σε PDF και εικόνες σε λειτουργία αξιολόγησης;**

Ναι. Οι [μετατροπείς](/slides/el/python-net/convert-presentation/) λειτουργούν· το αποτέλεσμα θα περιλαμβάνει υδατογράφημα.

Ναι. Μια 30-ήμερη προσωρινή άδεια αφαιρεί τους περιορισμούς της λειτουργίας αξιολόγησης και επιτρέπει τη δοκιμή χωρίς υδατογράφημα.
