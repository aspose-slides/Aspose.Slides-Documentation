## **Επισκόπηση**

Το Aspose.Slides υποστηρίζει αρχεία παρουσίασης από το Microsoft PowerPoint 97 έως το Office 365, συμπεριλαμβανομένου του Microsoft PowerPoint για Mac. Αυτό το άρθρο καταγράφει τις εκδόσεις PowerPoint που υποστηρίζονται από τη βιβλιοθήκη και παρέχει έναν πίνακα με μορφές αρχείων που μπορούν να φορτωθούν, να αποθηκευτούν ή και τα δύο.

Το άρθρο απαντά επίσης σε συχνές ερωτήσεις σχετικά με τη συμμόρφωση PDF, την ενσωμάτωση γραμματοσειρών, τα αρχεία με προστασία κωδικού, τις προσαρμοσμένες γραμματοσειρές, την εναλλακτική γραμματοσειρά και τις επιλογές εξαγωγής XPS.

## **Συχνές Ερωτήσεις**

**Μπορώ να αποθηκεύσω παρουσιάσεις σε PDF που πληρούν τα πρότυπα αρχειοθέτησης και προσβασιμότητας (PDF/A και PDF/UA);**

Ναι. Το Aspose.Slides for Python via .NET υποστηρίζει εξαγωγή σε PDF με επίπεδα συμμόρφωσης όπως PDF/A‑2a, PDF/A‑2b, PDF/A‑2u, PDF/A‑3a, PDF/A‑3b, καθώς και PDF/UA μέσω της ρύθμισης **συμμόρφωση** στο **επιλογές εξαγωγής PDF**.

**Η βιβλιοθήκη υποστηρίζει ενσωμάτωση γραμματοσειρών κατά την εξαγωγή σε PDF, με λεπτομερή έλεγχο του τι ενσωματώνεται;**

Ναι. Μπορείτε να ελέγξετε αν οι γραμματοσειρές θα ενσωματωθούν πλήρως ή υπό‑συνολ (μόνο τα χρησιμοποιηθέντα γλυφά), να ορίσετε πώς αντιμετωπίζονται οι κοινές σύστημα γραμματοσειρές και να ρυθμίσετε τη συμπεριφορά για κείμενο ASCII μέσω των **επιλογές εξαγωγής PDF**.

**Μπορώ να εντοπίσω αν ένα αρχείο είναι προστατευμένο με κωδικό πριν το φορτώσω;**

Ναι. Χρησιμοποιώντας το **API επιθεώρησης με βάση το factory** μπορείτε να ελέγξετε ένα αρχείο παρουσίασης για προστασία κωδικού χωρίς να το ανοίξετε πλήρως.

**Υπάρχουν μηχανισμοί εναλλακτικής γραμματοσειράς και υποστήριξη προσαρμοσμένων γραμματοσειρών;**

Ναι. Η βιβλιοθήκη υποστηρίζει **φόρτωση** και **ενσωμάτωση** προσαρμοσμένων γραμματοσειρών και παρέχει **κανόνες εναλλακτικής γραμματοσειράς** για την αποφυγή ελλειπούσων γλυφών κατά την απόδοση και τη μετατροπή.

**Μπορώ να εξάγω διαφάνειες σε XPS και υπάρχουν επιλογές για τη ρύθμιση της εξόδου XPS;**

Ναί. Η **Εξαγωγή σε XPS** υποστηρίζεται και μπορείτε να προσαρμόσετε τις σχετικές **επιλογές αποθήκευσης** για να ελέγξετε την ποιότητα και το περιεχόμενο του εγγράφου XPS.
