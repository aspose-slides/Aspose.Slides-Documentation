## **Υποστηριζόμενες Πλατφόρμες**
Οι πλατφόρμες Aspose.Slides for Python via .NET μπορούν να χρησιμοποιηθούν στα Windows x64 ή x86 και σε ευρύ φάσμα διανομών Linux με εγκατεστημένο Python 3.5 ή νεότερο. Υπάρχουν πρόσθετες απαιτήσεις για την στοχευόμενη πλατφόρμα Linux:
- Βιβλιοθήκες χρόνου εκτέλεσης GCC-6 (ή νεότερες)
- Εξαρτήσεις του .NET Core Runtime. Η εγκατάσταση του .NET Core Runtime δεν απαιτείται.
- Για Python 3.5‑3.7: Απαιτείται η έκδοση `pymalloc` του Python. Η επιλογή κατασκευής Python `--with-pymalloc` είναι ενεργοποιημένη εξ ορισμού. Συνήθως, η έκδοση `pymalloc` του Python σημειώνεται με το επίθημα `m` στο όνομα αρχείου.
- `libpython` κοινόχρηστη βιβλιοθήκη Python. Η επιλογή κατασκευής Python `--enable-shared` είναι απενεργοποιημένη εξ ορισμού· ορισμένες διανομές Python δεν περιέχουν τη κοινόχρηστη βιβλιοθήκη `libpython`. Για ορισμένες πλατφόρμες Linux, η κοινόχρηστη βιβλιοθήκη `libpython` μπορεί να εγκατασταθεί μέσω του διαχειριστή πακέτων, π.χ.: `sudo apt-get install libpython3.7`. Το κοινό πρόβλημα είναι ότι η βιβλιοθήκη `libpython` εγκαθίσταται σε διαφορετική τοποθεσία από την τυπική τοποθεσία του συστήματος για κοινόχρηστες βιβλιοθήκες. Το πρόβλημα μπορεί να λυθεί χρησιμοποιώντας τις επιλογές κατασκευής Python για ορισμό εναλλακτικών διαδρομών βιβλιοθήκης κατά τη μεταγλώττιση του Python, ή δημιουργώντας έναν συμβολικό σύνδεσμο προς το αρχείο βιβλιοθήκης `libpython` στην τυπική τοποθεσία του συστήματος για κοινόχρηστες βιβλιοθήκες. Συνήθως, το όνομα αρχείου της κοινόχρηστης βιβλιοθήκης `libpython` είναι `libpythonX.Ym.so.1.0` για Python 3.5‑3.7, ή `libpythonX.Y.so.1.0` για Python 3.8 ή νεότερο (π.χ.: `libpython3.7m.so.1.0`, `libpython3.9.so.1.0`).

Εάν χρειάζεστε υποστήριξη για περισσότερες πλατφόρμες, ψάξτε για τα προϊόντα «διδυμακά» Aspose.Slides for .NET ή Aspose.Slides for Java.

## **Συχνές Ερωτήσεις**

**Χρειάζεται να εγκαταστήσω το Microsoft PowerPoint στον διακομιστή/PC για να λειτουργήσει η βιβλιοθήκη;**

Όχι. Το PowerPoint δεν απαιτείται· το Aspose.Slides είναι μια αυτόνομη μηχανή για δημιουργία, επεξεργασία, μετατροπή και απόδοση παρουσιάσεων.

**Πώς λειτουργεί η πολυνηματική εκτέλεση; Μπορεί η επεξεργασία να παραλληλοποιηθεί;**

Είναι ασφαλές να επεξεργάζεστε διαφορετικά έγγραφα σε διαφορετικά νήματα· το ίδιο αντικείμενο [presentation](https://reference.aspose.com/slides/el/python-net/aspose.slides/presentation/) δεν πρέπει να χρησιμοποιείται από [multiple threads](/slides/el/python-net/multithreading/) ταυτόχρονα.

**Υποστηρίζονται κωδικοί πρόσβασης αρχείων και κρυπτογράφηση;**

Ναι. [Μπορείτε](/slides/el/python-net/password-protected-presentation/) να ανοίξετε κρυπτογραφημένες παρουσιάσεις, να ορίσετε ή να αφαιρέσετε κωδικό ανοίγματος ή εγγραφής, και να ελέγξετε την κατάσταση προστασίας.

**Πρέπει να μεριμνώ για τα πακέτα γραμματοσειρών σε κοντέινερ Linux;**

Ναι. Συνιστάται η εγκατάσταση κοινών πακέτων γραμματοσειρών και/ή η ρητή [καθορισμός καταλόγων γραμματοσειρών](/slides/el/python-net/custom-font/) στην εφαρμογή σας για να αποφύγετε απροσδόκητες αντικαταστάσεις.

**Υπάρχουν περιορισμοί στην εκδοχή αξιολόγησης;**

Στην [evaluation mode](/slides/el/python-net/licensing/), προστίθεται υδατογράφημα στην έξοδο και ισχύουν ορισμένοι περιορισμοί· ένα [30‑day temporary license](https://purchase.aspose.com/temporary-license/) είναι διαθέσιμο για πλήρη δοκιμή λειτουργιών.

Ναι. Μπορείτε να προσθέσετε [PDF pages and HTML content](/slides/el/python-net/import-presentation/) σε μια παρουσίαση, μετατρέποντάς τα σε διαφάνειες.
