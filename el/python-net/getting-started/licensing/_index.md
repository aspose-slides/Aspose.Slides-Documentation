## **Επισκόπηση**

Το Aspose.Slides μπορεί να χρησιμοποιηθεί σε λειτουργία αξιολόγησης ή με έγκυρη άδεια. Η έκδοση αξιολόγησης παρέχει την ίδια λειτουργικότητα με την άδεια έκδοση, αλλά προσθέτει ένα υδατογράφημα αξιολόγησης όταν οι παρουσιάσεις ανοίγουν ή αποθηκεύονται και περιορίζει την εξαγωγή κειμένου σε μία διαφάνεια.

### **Αρχείο**

Ο πιο εύκολος τρόπος για να ορίσετε μια άδεια είναι να τοποθετήσετε το αρχείο άδειας στον ίδιο φάκελο με το DLL του στοιχείου και να καθορίσετε μόνο το όνομα του αρχείου (χωρίς διαδρομή).

Ο παρακάτω κώδικας Python δείχνει πώς να ορίσετε το αρχείο άδειας:

```py
import aspose.slides as slides

# Δημιουργεί ένα αντικείμενο της κλάσης License. 
license = slides.License()

# Ορίζει τη διαδρομή του αρχείου άδειας.
license.set_license("Aspose.Slides.lic")
```

{{% alert color="warning" %}}
Αν τοποθετήσετε το αρχείο άδειας σε διαφορετικό κατάλογο, όταν καλέσετε την [License.set_license()](https://reference.aspose.com/slides/el/python-net/aspose.slides/license/set_license/#str), το όνομα αρχείου στο τέλος της ρητής διαδρομής πρέπει να ταιριάζει με το όνομα του αρχείου άδειας σας.

Για παράδειγμα, μπορείτε να μετονομάσετε το αρχείο άδειας σε *Aspose.Slides.lic.xml*. Στη συνέχεια, στον κώδικά σας, περάστε τη πλήρη διαδρομή σε αυτό το αρχείο (τελειώνοντας με Aspose.Slides.lic.xml) στην μέθοδο [License.set_license()](https://reference.aspose.com/slides/el/python-net/aspose.slides/license/set_license/#str).
{{% /alert %}}

### **Stream**

Μπορείτε να φορτώσετε μια άδεια από ένα stream. Ο παρακάτω παράδειγμα Python δείχνει πώς να εφαρμόσετε μια άδεια από ένα stream:

```py
import aspose.slides as slides

# Δημιουργεί ένα αντικείμενο της κλάσης License.
license = slides.License()

# Ορίζει την άδεια από ένα stream.
license.set_license(stream)
```

## **Επικύρωση Άδειας**

Για να επαληθεύσετε ότι η άδεια έχει εφαρμοστεί σωστά, μπορείτε να την επικυρώσετε. Ο παρακάτω κώδικας Python δείχνει πώς να επικυρώσετε μια άδεια:

```py
import aspose.slides as slides

license = slides.License()

license.set_license("Aspose.Slides.lic")

if license.is_licensed():
    print("License is good!")
```

## **Ασφάλεια Νημάτων**

{{% alert title="Note" color="warning" %}}
Οι μέθοδοι [License.set_license](https://reference.aspose.com/slides/el/python-net/aspose.slides/license/) δεν είναι ασφαλείς για χρήση από πολλαπλά νήματα. Εάν χρειάζεται να κληθούν ταυτόχρονα από πολλά νήματα, χρησιμοποιήστε πρώτες συγχρονισμού (π.χ., `threading.Lock`) για να αποφύγετε προβλήματα.
{{% /alert %}}
