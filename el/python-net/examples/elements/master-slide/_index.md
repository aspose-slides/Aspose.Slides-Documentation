Οι κύριες διαφάνειες αποτελούν το ανώτερο επίπεδο της ιεραρχίας κληρονομικότητας διαφανειών στο PowerPoint. Μια **κύρια διαφάνεια** ορίζει κοινά στοιχεία σχεδίου όπως φόντα, λογότυπα και μορφοποίηση κειμένου. **Διαφάνειες διάταξης** κληρονομούν από τις κύριες διαφάνειες, και οι **κανονικές διαφάνειες** κληρονομούν από τις διαφάνειες διάταξης.

Αυτό το άρθρο δείχνει πώς να δημιουργήσετε, να τροποποιήσετε και να διαχειριστείτε τις κύριες διαφάνειες χρησιμοποιώντας το Aspose.Slides για Python μέσω .NET.

## **Προσθήκη κύριας διαφάνειας**

Αυτό το παράδειγμα δείχνει πώς να δημιουργήσετε μια νέα κύρια διαφάνεια κλωνοποιώντας την προεπιλεγμένη.

```py
def add_master_slide():
    with slides.Presentation() as presentation:

        # Κλωνοποίηση της προεπιλεγμένης κύριας διαφάνειας.
        default_master_slide = presentation.masters[0]
        new_master = presentation.masters.add_clone(default_master_slide)

        presentation.save("master_slide.pptx", slides.export.SaveFormat.PPTX)
```

> 💡 **Tip 1:** Οι κύριες διαφάνειες παρέχουν τρόπο για να εφαρμόζετε συνεπές branding ή κοινά στοιχεία σχεδίου σε όλες τις διαφάνειες. Οποιεσδήποτε αλλαγές γίνουν στην κύρια διαφάνεια θα αντικατοπτριστούν αυτόματα στις εξαρτημένες διαφάνειες διάταξης και τις κανονικές διαφάνειες.  
> 
> 💡 **Tip 2:** Οποιεσδήποτε σχήματα ή μορφοποιήσεις προστίθενται σε μια κύρια διαφάνεια κληρονομούνται από τις διαφάνειες διάταξης και, με τη σειρά τους, από όλες τις κανονικές διαφάνειες που χρησιμοποιούν αυτές τις διατάξεις.  
> Η παρακάτω εικόνα δείχνει πώς ένα πλαίσιο κειμένου που προστέθηκε σε μια κύρια διαφάνεια αποδίδεται αυτόματα στην τελική διαφάνεια.

## **Πρόσβαση σε κύρια διαφάνεια**

Μπορείτε να αποκτήσετε πρόσβαση στις κύριες διαφάνειες χρησιμοποιώντας τη συλλογή `Presentation.masters`. Αυτή είναι η διαδικασία ανάκτησης και εργασίας με αυτές:

```py
def access_master_slide():
    with slides.Presentation("master_slide.pptx") as presentation:
        # Πρόσβαση στην πρώτη κύρια διαφάνεια.
        first_master_slide = presentation.masters[0]
```

## **Αφαίρεση κύριας διαφάνειας**

Οι κύριες διαφάνειες μπορούν να αφαιρεθούν είτε με δείκτη είτε με αναφορά.

```py
def remove_master_slide():
    with slides.Presentation("master_slide.pptx") as presentation:

        # Αφαίρεση με δείκτη.
        presentation.masters.remove_at(0)

        # Ή αφαίρεση με αναφορά.
        first_master_slide = presentation.masters[0]
        presentation.masters.remove(first_master_slide)

        presentation.save("master_slide_removed.pptx", slides.export.SaveFormat.PPTX)
```

## **Αφαίρεση αχρησιμοποίητων κύριων διαφανειών**

Ορισμένες παρουσιάσεις περιέχουν κύριες διαφάνειες που δεν χρησιμοποιούνται. Η αφαίρεση αυτών των διαφανειών μπορεί να βοηθήσει στη μείωση του μεγέθους του αρχείου.

```py
def remove_unused_master_slides():
    with slides.Presentation("master_slide.pptx") as presentation:

        # Αφαίρεση όλων των αχρησιμοποίητων κύριων διαφανειών (ακόμη και αυτών που έχουν σημειωθεί ως Preserve).
        presentation.masters.remove_unused(True)

        presentation.save("master_slides_removed.pptx", slides.export.SaveFormat.PPTX)
```

> ⚙️ **Συμβουλή:** Χρησιμοποιήστε το `remove_unused(True)` για να αφαιρέσετε τις αχρησιμοποίητες κύριες διαφάνειες και να ελαχιστοποιήσετε το μέγεθος της παρουσίασης.
