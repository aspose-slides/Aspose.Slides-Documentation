## **Επισκόπηση**

Αυτό το άρθρο δείχνει πώς να εργαστείτε με ένα διάγραμμα δακτυλίου στο Aspose.Slides προσθέτοντας το διάγραμμα σε μια διαφάνεια, ορίζοντας το μέγεθος της κεντρικής τρύπας του και αποθηκεύοντας την παρουσίαση. Επικεντρώνεται στη ρύθμιση `doughnut_hole_size` και παρουσιάζει τα βασικά βήματα που απαιτούνται για την προσαρμογή αυτού του τύπου διαγράμματος με κώδικα.

Περιλαμβάνει επίσης μια σύντομη ενότητα Συχνών Ερωτήσεων που καλύπτει σχετικές περιπτώσεις διαγραμμάτων δακτυλίου, όπως η χρήση πολλαπλών σειρών για δημιουργία πολλαπλών δαχτυλιδιών, η εργασία με εκραγμένα διαγράμματα δακτυλίου, και η εξαγωγή ενός διαγράμματος ως εικόνα raster ή SVG.

## **Καθορίστε το κεντρικό διάστημα σε διάγραμμα δακτυλίου**
Για να καθορίσετε το μέγεθος της τρύπας σε ένα διάγραμμα δακτυλίου, ακολουθήστε τα παρακάτω βήματα:

- Δημιουργήστε μια παρουσία της κλάσης [Presentation](https://reference.aspose.com/slides/el/python-net/aspose.slides/presentation/).
- Προσθέστε διάγραμμα δακτυλίου στη διαφάνεια.
- Καθορίστε το μέγεθος της τρύπας στο διάγραμμα δακτυλίου.
- Αποθηκεύστε την παρουσίαση στο δίσκο.

Στο παρακάτω παράδειγμα, ορίσαμε το μέγεθος της τρύπας στο διάγραμμα δακτυλίου.

```py
import aspose.slides.charts as charts
import aspose.slides as slides

# Δημιουργήστε μια παρουσία της κλάσης Presentation
with slides.Presentation() as presentation:

    chart = presentation.slides[0].shapes.add_chart(charts.ChartType.DOUGHNUT, 50, 50, 400, 400)
    chart.chart_data.series_groups[0].doughnut_hole_size = 90

    # Αποθηκεύστε την παρουσίαση στο δίσκο
    presentation.save("DoughnutHoleSize_out.pptx", slides.export.SaveFormat.PPTX)
```

## **Συχνές Ερωτήσεις**

**Μπορώ να δημιουργήσω ένα πολυεπίπεδο διάγραμμα δακτυλίου με πολλαπλά δαχτυλίδια;**

Ναι. Προσθέστε πολλαπλές σειρές σε ένα ενιαίο διάγραμμα δακτυλίου — κάθε σειρά γίνεται ένα ξεχωριστό δαχτυλίδι. Η σειρά των δαχτυλιδιών καθορίζεται από τη σειρά των σειρών στη συλλογή.

**Υποστηρίζεται ένα «εκραγμένο» διάγραμμα δακτυλίου (ξεχωριστά κομμάτια);**

Ναι. Υπάρχει ο τύπος διαγράμματος Exploded Doughnut [chart type](https://reference.aspose.com/slides/el/python-net/aspose.slides.charts/charttype/) και μία ιδιότητα έκρηξης στα σημεία δεδομένων· μπορείτε να ξεχωρίσετε μεμονωμένα κομμάτια.

**Πώς μπορώ να αποκτήσω μια εικόνα διαγράμματος δακτυλίου (PNG/SVG) για μια αναφορά;**

Ένα διάγραμμα είναι σχήμα· μπορείτε να το αποδώσετε σε μια [raster image](https://reference.aspose.com/slides/el/python-net/aspose.slides/shape/get_image/) ή να εξάγετε το διάγραμμα σε μια [SVG image](https://reference.aspose.com/slides/el/python-net/aspose.slides/shape/write_as_svg/).
