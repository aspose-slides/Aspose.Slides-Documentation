## **Επισκόπηση**

Πριν προσθέσετε διαφάνειες σε μία παρουσίαση, είναι χρήσιμο να καταλάβετε πώς το PowerPoint τις οργανώνει. Κάθε παρουσίαση περιέχει μια κύρια διαφάνεια, προαιρετικές διαφάνειες διάταξης και μία ή περισσότερες κανονικές διαφάνειες. Κάθε διαφάνεια διαθέτει ένα μοναδικό αναγνωριστικό και οι κανονικές διαφάνειες ταξινομούνται με βάση έναν δείκτη που ξεκινά από το μηδέν. Αυτό το άρθρο δείχνει πώς να χρησιμοποιήσετε το Aspose.Slides για Python ώστε να δημιουργήσετε διαφάνειες και να επιλέξετε τις κατάλληλες διατάξεις.

## **Προσθήκη Διαφανειών σε Παρουσιάσεις**

Το Aspose.Slides σας επιτρέπει να προσθέτετε νέες διαφάνειες βασισμένες σε υπάρχουσες διαφάνειες διάταξης. Στο παρακάτω παράδειγμα γίνεται επανάληψη σε κάθε διάταξη της παρουσίασης, προστίθεται μια διαφάνεια που χρησιμοποιεί αυτή τη διάταξη και, στη συνέχεια, αποθηκεύεται το αρχείο.

1. Δημιουργήστε μια παρουσία της κλάσης [Presentation](https://reference.aspose.com/slides/el/python-net/aspose.slides/presentation/) .
2. Προσπελάστε τη [SlideCollection](https://reference.aspose.com/slides/el/python-net/aspose.slides/slidecollection/) .
3. Για κάθε στοιχείο στο `presentation.layout_slides`, καλέστε τη `add_empty_slide` για να προσαρτήσετε μια διαφάνεια που χρησιμοποιεί αυτή τη διάταξη.
4. Προαιρετικά, τροποποιήστε τις νέες προστιθέμενες διαφάνειες.
5. Αποθηκεύστε την παρουσίαση ως αρχείο PPTX.

```py
import aspose.slides as slides

# Δημιουργία της κλάσης Presentation.
with slides.Presentation() as presentation:
    # Πρόσβαση στη συλλογή διαφανειών.
    slides = presentation.slides

    for layout_slide in presentation.layout_slides:
        # Προσθήκη κενής διαφάνειας στη συλλογή διαφανειών.
        slides.add_empty_slide(layout_slide)

    # Εκτελέστε κάποιες εργασίες στις πρόσφατα προστιθέμενες διαφάνειες.

    # Αποθήκευση της παρουσίασης στο δίσκο.
    presentation.save("empty_slides.pptx", slides.export.SaveFormat.PPTX)
```

## **Συχνές Ερωτήσεις**

**Μπορώ να εισάγω μια νέα διαφάνεια σε συγκεκριμένη θέση, όχι μόνο στο τέλος;**

Ναι. Η βιβλιοθήκη υποστηρίζει συλλογές διαφανειών και τις λειτουργίες [insert](https://reference.aspose.com/slides/el/python-net/aspose.slides/slidecollection/insert_empty_slide/)/[clone](https://reference.aspose.com/slides/el/python-net/aspose.slides/slidecollection/insert_clone/) , έτσι μπορείτε να προσθέσετε μια διαφάνεια στο απαιτούμενο ευρετήριο αντί μόνο στο τέλος.

**Διατηρούνται οι θεματικές/στυλ όταν προσθέτετε μια διαφάνεια βάσει μιας διάταξης;**

Ναι. Μια διάταξη κληρονομεί τη μορφοποίηση από το κύριο της, και η νέα διαφάνεια κληρονομεί από τη επιλεγμένη διάταξη και το αντίστοιχο κύριο.

**Ποια διαφάνεια υπάρχει σε μια νέα «κενή» παρουσίαση πριν προστεθούν διαφάνειες;**

Μια νεοδημιουργημένη παρουσίαση περιέχει ήδη μία κενή διαφάνεια με δείκτη μηδέν. Αυτό είναι σημαντικό να ληφθεί υπόψη όταν υπολογίζετε δείκτες εισαγωγής.

**Πώς μπορώ να επιλέξω τη «σωστή» διάταξη για μια νέα διαφάνεια εάν ο κύριος έχει πολλές επιλογές;**

Γενικά επιλέξτε τη [LayoutSlide](https://reference.aspose.com/slides/el/python-net/aspose.slides/layoutslide/) που ταιριάζει με την απαιτούμενη δομή ([Τίτλο και Περιεχόμενο, Δύο Περιεχόμενα, κ.λπ.](https://reference.aspose.com/slides/el/python-net/aspose.slides/slidelayouttype/)). Εάν λείπει μια τέτοια διάταξη, μπορείτε να [προσθέσετε το στο κύριο](/slides/el/python-net/slide-layout/) και μετά να τη χρησιμοποιήσετε.
