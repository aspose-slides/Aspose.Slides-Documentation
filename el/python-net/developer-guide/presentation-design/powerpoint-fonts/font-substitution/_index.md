## **Επισκόπηση**

Η αντικατάσταση γραμματοσειράς επιτρέπει στο Aspose.Slides να χρησιμοποιεί μια άλλη γραμματοσειρά όταν η αρχική γραμματοσειρά της παρουσίασης δεν είναι διαθέσιμη κατά τη διαδικασία απόδοσης ή μετατροπής. Μπορείτε να ελέγξετε ποιες γραμματοσειρές αντικαταστάθηκαν χρησιμοποιώντας τη μέθοδο `get_substitutions` της κλάσης `FontsManager`.

Το Aspose.Slides επιτρέπει επίσης τον ορισμό κανόνων αντικατάστασης γραμματοσειρών. Για παράδειγμα, μπορείτε να καθορίσετε ότι μια μη προσβάσιμη γραμματοσειρά θα πρέπει να αντικατασταθεί με μια άλλη διαθέσιμη γραμματοσειρά και στη συνέχεια να εφαρμόσετε αυτούς τους κανόνες μέσω του διαχειριστή γραμματοσειρών της παρουσίασης.

## **Ορισμός κανόνων αντικατάστασης**

Το Aspose.Slides σας επιτρέπει να ορίσετε κανόνες για τις γραμματοσειρές που καθορίζουν τι πρέπει να γίνει σε ορισμένες συνθήκες (π.χ. όταν μια γραμματοσειρά δεν είναι προσβάσιμη) ως εξής:

1. Φορτώστε την αντίστοιχη παρουσίαση.  
2. Φορτώστε τη γραμματοσειρά που θα αντικατασταθεί.  
3. Φορτώστε τη νέα γραμματοσειρά.  
4. Προσθέστε έναν κανόνα για την αντικατάσταση.  
5. Προσθέστε τον κανόνα στη συλλογή κανόνων αντικατάστασης γραμματοσειρών της παρουσίασης.  
6. Δημιουργήστε την εικόνα της διαφάνειας για να παρατηρήσετε το αποτέλεσμα.

Αυτός ο κώδικας Python δείχνει τη διαδικασία αντικατάστασης γραμματοσειράς:

```python
import aspose.slides as slides

# Φορτώνει μια παρουσίαση
with slides.Presentation(path + "Fonts.pptx") as presentation:
    # Φορτώνει τη γραμματοσειρά προέλευσης που θα αντικατασταθεί
    sourceFont = slides.FontData("SomeRareFont")

    # Φορτώνει τη νέα γραμματοσειρά
    destFont = slides.FontData("Arial")

    # Προσθέτει έναν κανόνα γραμματοσειράς για αντικατάσταση γραμματοσειράς
    fontSubstRule = slides.FontSubstRule(sourceFont, destFont, slides.FontSubstCondition.WHEN_INACCESSIBLE)

    # Προσθέτει τον κανόνα στη συλλογή κανόνων αντικατάστασης γραμματοσειρών
    fontSubstRuleCollection = slides.FontSubstRuleCollection()
    fontSubstRuleCollection.add(fontSubstRule)

    # Προσθέτει τη συλλογή κανόνων γραμματοσειράς στη λίστα κανόνων
    presentation.fonts_manager.font_subst_rule_list = fontSubstRuleCollection

    # Η γραμματοσειρά Arial θα χρησιμοποιηθεί αντί της SomeRareFont όταν αυτή είναι μη προσβάσιμη
    with presentation.slides[0].get_image(1, 1) as bmp:
        # Αποθηκεύει την εικόνα στο δίσκο σε μορφή JPEG
        bmp.save("Thumbnail_out.jpg", slides.ImageFormat.JPEG)
```

## **Περιορισμοί για τις γραμματοσειρές μαθηματικών εξισώσεων**

Οι κανόνες αντικατάστασης γραμματοσειρών συμμετέχουν στη στάνταρ διαδικασία επιλογής γραμματοσειράς που χρησιμοποιείται κατά την απόδοση και τη μετατροπή. Είναι κατάλληλοι για τυπικά σενάρια κειμένου όπου το Aspose.Slides μπορεί να αντικαταστήσει μια μη προσβάσιμη γραμματοσειρά με μια άλλη διαθέσιμη σύμφωνα με τον καθορισμένο κανόνα.

Ωστόσο, οι μαθηματικές εξισώσεις του Office έχουν έναν σημαντικό περιορισμό. Εάν μια εξίσωση δημιουργήθηκε με **Cambria Math**, το Aspose.Slides ενδέχεται ακόμη να απαιτεί την αρχική γραμματοσειρά **Cambria Math** για να υπολογίσει και να αποδώσει σωστά τη διάταξη της εξίσωσης. Για αυτόν τον λόγο, η αντικατάσταση της **Cambria Math** με μια άλλη μαθηματική γραμματοσειρά, όπως η **STIX Two Math**, δεν υποστηρίζεται για την απόδοση εξίσωσης και μπορεί ακόμα να προκαλέσει εξαίρεση που υποδεικνύει ότι απαιτείται η **Cambria Math**.

Για να μετατρέψετε τέτοιες παρουσιάσεις επιτυχώς, βεβαιωθείτε ότι η **Cambria Math** είναι διαθέσιμη στο Aspose.Slides κατά το χρόνο εκτέλεσης. Μπορείτε να εγκαταστήσετε τη γραμματοσειρά στο λειτουργικό σύστημα ή να τη παρέχετε ως [εξωτερική γραμματοσειρά](/slides/el/python-net/custom-font/) ώστε να συμμετέχει στη φυσιολογική διαδικασία επιλογής γραμματοσειράς κατά την απόδοση και τη μετατροπή.

Αυτός ο περιορισμός είναι ειδικός για την απόδοση εξίσωσης. Οι στάνταρ κανόνες αντικατάστασης γραμματοσειρών που περιγράφηκαν παραπάνω εξακολουθούν να ισχύουν για το κανονικό κείμενο της παρουσίασης όταν η αρχική γραμματοσειρά δεν είναι προσβάσιμη.
