## **Επισκόπηση**

Το Aspose.Slides σας επιτρέπει να αντικαταστήσετε μια γραμματοσειρά με άλλη σε ολόκληρη την παρουσίαση. Όταν γίνεται αντικατάσταση μιας γραμματοσειράς, όλες οι εμφανίσεις της αρχικής γραμματοσειράς αλλάζουν στη νέα γραμματοσειρά.

Για να πραγματοποιήσετε αντικατάσταση γραμματοσειράς, φορτώστε την παρουσίαση, ορίστε τη γραμματοσειρά προέλευσης και τη γραμματοσειρά αντικατάστασης, καλέστε τη μέθοδο αντικατάστασης γραμματοσειράς και αποθηκεύστε την τροποποιημένη παρουσίαση ως αρχείο PPTX. Αυτή η προσέγγιση είναι χρήσιμη όταν θέλετε σκόπιμα να μεταβείτε από μια οικογένεια γραμματοσειρών σε άλλη σε όλη την παρουσίαση.

## **Αντικατάσταση Γραμματοσειρών**

Αν αλλάξετε γνώμη σχετικά με τη χρήση μιας γραμματοσειράς, μπορείτε να την αντικαταστήσετε με άλλη. Όλες οι εμφανίσεις της παλιάς γραμματοσειράς θα αντικατασταθούν από τη νέα.

Το Aspose.Slides σας επιτρέπει να αντικαταστήσετε μια γραμματοσειρά με τον εξής τρόπο:

1. Φορτώστε τη σχετική παρουσίαση.  
2. Φορτώστε τη γραμματοσειρά που θα αντικατασταθεί.  
3. Φορτώστε τη νέα γραμματοσειρά.  
4. Αντικαταστήστε τη γραμματοσειρά.  
5. Γράψτε την τροποποιημένη παρουσίαση ως αρχείο PPTX.

Αυτός ο κώδικας Python δείχνει την αντικατάσταση γραμματοσειράς:

```py
import aspose.pydrawing as draw
import aspose.slides as slides

# Φορτώνει μια παρουσίαση
with slides.Presentation(path + "Fonts.pptx") as presentation:
    # Φορτώνει τη γραμματοσειρά προέλευσης που θα αντικατασταθεί
    sourceFont = slides.FontData("Arial")

    # Φορτώνει τη νέα γραμματοσειρά
    destFont = slides.FontData("Times New Roman")

    # Αντικαθιστά τις γραμματοσειρές
    presentation.fonts_manager.replace_font(sourceFont, destFont)

    # Αποθηκεύει την παρουσίαση
    presentation.save("UpdatedFont_out.pptx", slides.export.SaveFormat.PPTX)
```

## **Συχνές ερωτήσεις**

**Ποια είναι η διαφορά μεταξύ "αντικατάστασης γραμματοσειράς", "αντικατάστασης γραμματοσειράς" και "εφεδρικών γραμματοσειρών";**

Η αντικατάσταση είναι μια σκόπιμη αλλαγή από μια οικογένεια σε άλλη σε όλο το έγγραφο. [Αντικατάσταση](/slides/el/python-net/font-substitution/) είναι ένας κανόνας όπως «αν η γραμματοσειρά δεν είναι διαθέσιμη, χρησιμοποιήστε X». [Εφεδρική γραμματοσειρά](/slides/el/python-net/fallback-font/) εφαρμόζεται χειροπιαστά για μεμονωμένα λείψανα χαρακτήρων όταν η βασική γραμματοσειρά είναι εγκατεστημένη αλλά δεν περιέχει τους απαιτούμενους χαρακτήρες.

**Εφαρμόζεται η αντικατάσταση στα master slides, τα layouts, τα σημειώματα και τα σχόλια;**

Ναι. Η αντικατάσταση επηρεάζει όλα τα αντικείμενα της παρουσίασης που χρησιμοποιούν την αρχική γραμματοσειρά, συμπεριλαμβανομένων των master slides και των σημειώματων· τα σχόλια είναι επίσης μέρος του εγγράφου και λαμβάνονται υπόψη από τη μηχανή γραμματοσειρών.

**Θα αλλάξει η γραμματοσειρά μέσα σε ενσωματωμένα αντικείμενα OLE (π.χ., Excel);**

Όχι. Το [περιεχόμενο OLE](/slides/el/python-net/manage-ole/) ελέγχεται από τη δική του εφαρμογή. Η αντικατάσταση στην παρουσίαση δεν επαναδιαμορφώνει τα εσωτερικά δεδομένα OLE· μπορεί να εμφανιστεί ως εικόνα ή ως εξωτερικά επεξεργάσιμο περιεχόμενο.

**Μπορώ να αντικαταστήσω μια γραμματοσειρά μόνο σε μέρος της παρουσίασης (με διαφάνειες ή περιοχές);**

Η στοχευμένη αντικατάσταση είναι δυνατή εάν αλλάξετε τη γραμματοσειρά στο επίπεδο των απαιτούμενων αντικειμένων/εύρους αντί να εφαρμόσετε μια παγκόσμια αντικατάσταση σε ολόκληρο το έγγραφο. Η λογική επιλογής γραμματοσειράς κατά τη διαδικασία αποτύπωσης παραμένει η ίδια.

**Πώς μπορώ να προσδιορίσω εκ των προτέρων ποιες γραμματοσειρές χρησιμοποιεί η παρουσίαση;**

Χρησιμοποιήστε το [font manager] του παρουσίασης (https://reference.aspose.com/slides/el/python-net/aspose.slides/fontsmanager/): παρέχει μια λίστα των [οικογενειών σε χρήση](/slides/el/python-net/font-substitution/) και πληροφορίες για τις [αντικαταστάσεις/«άγνωστες» γραμματοσειρές](/slides/el/python-net/font-substitution/), που βοηθούν στον προγραμματισμό της αντικατάστασης.

**Λειτουργεί η αντικατάσταση γραμματοσειράς κατά τη μετατροπή σε PDF/εικόνες;**

Ναι. Κατά την εξαγωγή, το Aspose.Slides εφαρμόζει την ίδια [ακολουθία επιλογής/αντικατάστασης γραμματοσειράς](/slides/el/python-net/font-selection-sequence/), ώστε μια αντικατάσταση που έχει γίνει εκ των προτέρων να τηρηθεί κατά τη μετατροπή.

**Πρέπει να εγκαταστήσω τη στοχευόμενη γραμματοσειρά στο σύστημα ή μπορώ να επισυνάψω ένα φάκελο γραμματοσειρών;**

Η εγκατάσταση δεν απαιτείται: η βιβλιοθήκη επιτρέπει [φόρτωση εξωτερικών γραμματοσειρών](/slides/el/python-net/custom-font/) από φάκελους χρήστη για χρήση κατά την [απόδοση και εξαγωγή](/slides/el/python-net/convert-powerpoint/).

**Θα διορθώσει η αντικατάσταση το "tofu" (τετράγωνα) αντί για χαρακτήρες;**

Μόνο εάν η στοχευόμενη γραμματοσειρά περιέχει πραγματικά τα απαιτούμενα γλύφη. Αν όχι, [ρυθμίστε την εφεδρική γραμματοσειρά](/slides/el/python-net/fallback-font/) για να καλύψετε τους ελλιπείς χαρακτήρες.
