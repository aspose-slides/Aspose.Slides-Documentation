## **Επισκόπηση**

Όταν μια παρουσίαση φορτώνεται, αποδίδεται ή μετατρέπεται σε άλλη μορφή, το Aspose.Slides ελέγχει εάν οι γραμματοσειρές που χρησιμοποιούνται στην παρουσίαση είναι διαθέσιμες στο λειτουργικό σύστημα. Εάν λείπει μια απαιτούμενη γραμματοσειρά, το Aspose.Slides επιλέγει μια εναλλακτική γραμματοσειρά που είναι όσο το δυνατόν πιο κοντά σε αυτή που θα χρησιμοποιήσει το PowerPoint.

Το Aspose.Slides πρώτα αναζητά τη επιλεγμένη γραμματοσειρά στο λειτουργικό σύστημα. Εάν η γραμματοσειρά βρεθεί, χρησιμοποιείται. Εάν δεν βρεθεί, εφαρμόζεται μια κατάλληλη εναλλακτική. Όταν οι κανόνες αντικατάστασης γραμματοσειρών ορίζονται μέσω `FontSubstRule`, λαμβάνονται επίσης υπόψη.

Μπορείτε επίσης να προσθέσετε γραμματοσειρές κατά τη διάρκεια εκτέλεσης της εφαρμογής, να χρησιμοποιήσετε ενσωματωμένες γραμματοσειρές από μια παρουσίαση ή να φορτώσετε εξωτερικές γραμματοσειρές για έγγραφα εξόδου όπως αρχεία PDF.

## **Επιλογή γραμματοσειρών**

Ορισμένοι κανόνες εφαρμόζονται στις γραμματοσειρές μιας παρουσίασης όταν η παρουσίαση φορτώνεται, αποδίδεται ή μετατρέπεται σε άλλη μορφή. Για παράδειγμα, όταν προσπαθείτε να μετατρέψετε μια παρουσίαση (τις διαφάνειές της) σε εικόνες, οι γραμματοσειρές της παρουσίασης ελέγχονται για να διασφαλιστεί ότι οι επιλεγμένες γραμματοσειρές είναι διαθέσιμες στο λειτουργικό σύστημα. Εάν επιβεβαιωθεί ότι οι γραμματοσειρές λείπουν, αντικαθίστανται — δείτε [**Αντικατάσταση γραμματοσειρών**](https://docs.aspose.com/slides/el/python-net/font-replacement/) και [**Αντικατάσταση γραμματοσειρών**](https://docs.aspose.com/slides/el/python-net/font-substitution/).

Αυτή είναι η διαδικασία που ακολουθεί το Aspose.Slides όταν διαχειρίζεται γραμματοσειρές:

1. Το Aspose.Slides αναζητά γραμματοσειρές στο λειτουργικό σύστημα για να βρει τη γραμματοσειρά που ταιριάζει με τη γραμματοσειρά που έχει επιλέξει η παρουσίαση. 
2. Εάν η επιλεγμένη γραμματοσειρά βρεθεί, το Aspose.Slides τη χρησιμοποιεί. Διαφορετικά, το Aspose.Slides χρησιμοποιεί μια εναλλακτική γραμματοσειρά που είναι όσο το δυνατόν πιο κοντά σε αυτή που θα έλεγε το PowerPoint.
3. Εάν έχουν οριστεί κανόνες αντικατάστασης γραμματοσειρών μέσω [FontSubstRule](https://reference.aspose.com/slides/el/python-net/aspose.slides/fontsubstrule/), εφαρμόζονται. 

Το Aspose.Slides σας επιτρέπει να προσθέσετε γραμματοσειρές κατά το χρόνο εκτέλεσης της εφαρμογής και στη συνέχεια να τις χρησιμοποιήσετε. Δείτε [**Προσαρμοσμένες γραμματοσειρές**](https://docs.aspose.com/slides/el/python-net/custom-font/). 

Όταν επιπλέον γραμματοσειρές τοποθετούνται μέσα σε μια παρουσίαση, ονομάζονται [**Ενσωματωμένες γραμματοσειρές**](https://docs.aspose.com/slides/el/python-net/embedded-font/).

Το Aspose.Slides σας επιτρέπει να προσθέσετε γραμματοσειρές που εφαρμόζονται μόνο σε έγγραφα εξόδου. Για παράδειγμα, εάν μια παρουσίαση που θέλετε να μετατρέψετε σε PDF περιέχει γραμματοσειρές που λείπουν από το σύστημά σας και ενσωματωμένες γραμματοσειρές, μπορείτε να προσθέσετε ή να φορτώσετε τις απαιτούμενες γραμματοσειρές ως **εξωτερικές γραμματοσειρές**. 

{{% alert title="Note" color="primary" %}} 
Δεν διανέμουμε καμία γραμματοσειρά, είτε επαγγελματική είτε δωρεάν. Το API μας σας επιτρέπει να φορτώσετε εξωτερικές γραμματοσειρές και να τις ενσωματώσετε σε έγγραφα, αλλά το κάνετε με τις γραμματοσειρές σύμφωνα με τη διακριτική σας ευχέρεια και ευθύνη.
{{% /alert %}}

## **Συχνές ερωτήσεις**

**Πώς μπορώ να καθορίσω ποιες γραμματοσειρές χρησιμοποιούνται πραγματικά σε μια παρουσίαση πριν από τη μετατροπή;**

Το Aspose.Slides σας επιτρέπει να ελέγξετε τις χρησιμοποιούμενες γραμματοσειρές μέσω του [διαχειριστή γραμματοσειρών](https://reference.aspose.com/slides/el/python-net/aspose.slides/presentation/fonts_manager/), ώστε να αποφασίσετε αν θα [ενσωματώσετε](/slides/el/python-net/embedded-font/), [αντικαταστήσετε](/slides/el/python-net/font-replacement/) ή προσθέσετε [εξωτερικές πηγές](/slides/el/python-net/custom-font/). Αυτό σας βοηθά να αποτρέψετε ανεπιθύμητες αντικαταστάσεις κατά την απόδοση και την εξαγωγή.

**Μπορώ να προσθέσω επιπλέον καταλόγους γραμματοσειρών χωρίς να τους εγκαταστήσω στο λειτουργικό σύστημα;**

Ναι. Μπορείτε να καταχωρίσετε [εξωτερικές πηγές γραμματοσειρών](/slides/el/python-net/custom-font/) όπως φάκελοι ή ροές μνήμης για την απόδοση και την εξαγωγή. Αυτό αφαιρεί την εξάρτηση από τις γραμματοσειρές του συστήματος φιλοξενίας και διατηρεί τη διάταξη προβλέψιμη.

**Πώς μπορώ να αποτρέψω μια σιωπηλή εναλλακτική σε μη κατάλληλη γραμματοσειρά όταν ένα σύμβολο λείπει;**

Ορίστε ρητά [αντικατάσταση γραμματοσειρών](/slides/el/python-net/font-replacement/) και [κανόνες fallBack γραμματοσειρών](/slides/el/python-net/fallback-font/) εκ των προτέρων. Αναλύοντας τις χρησιμοποιούμενες γραμματοσειρές και ορίζοντας ελεγχόμενη προτεραιότητα για τις εναλλακτικές, διασφαλίζετε συνεπή τυπογραφία και αποφεύγετε ανεπιθύμητα αποτελέσματα.
