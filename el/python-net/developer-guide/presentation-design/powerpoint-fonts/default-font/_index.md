## **Επισκόπηση**

Το Aspose.Slides σας επιτρέπει να καθορίσετε τις προεπιλεγμένες γραμματοσειρές που χρησιμοποιούνται όταν αποδίδεται μια παρουσίαση. Αυτό είναι χρήσιμο κατά τη δημιουργία μικρογραφιών διαφανειών ή κατά την εξαγωγή μιας παρουσίασης σε μορφές όπως PDF και XPS. Οι προεπιλεγμένες γραμματοσειρές διαμορφώνονται μέσω `LoadOptions` πριν φορτωθεί η παρουσίαση.

Η ιδιότητα `default_regular_font` ορίζει τη προεπιλεγμένη γραμματοσειρά για κανονικό κείμενο, ενώ η `default_asian_font` ορίζει τη προεπιλεγμένη γραμματοσειρά για ασιατικό κείμενο. Αφού οριστούν αυτές οι επιλογές, η παρουσίαση μπορεί να φορτωθεί και να αποδοθεί χρησιμοποιώντας τις καθορισμένες γραμματοσειρές.

## **Χρήση προεπιλεγμένων γραμματοσειρών για την απόδοση της παρουσίασης**
Το Aspose.Slides σας επιτρέπει να ορίσετε την προεπιλεγμένη γραμματοσειρά για την απόδοση της παρουσίασης σε PDF, XPS ή μικρογραφίες. Αυτό το άρθρο δείχνει πώς να ορίσετε το DefaultRegular Font και το DefaultAsian Font για χρήση ως προεπιλεγμένες γραμματοσειρές. Παρακαλούμε ακολουθήστε τα παρακάτω βήματα για τη φόρτωση γραμματοσειρών από εξωτερικούς φακέλους χρησιμοποιώντας το Aspose.Slides for Python μέσω .NET API:

1. Δημιουργήστε ένα στιγμιότυπο του LoadOptions.
2. Ορίστε το DefaultRegularFont στην επιθυμητή γραμματοσειρά σας. Στο παρακάτω παράδειγμα, χρησιμοποίησα το Wingdings.
3. Ορίστε το DefaultAsianFont στην επιθυμητή γραμματοσειρά σας. Στο παρακάτω δείγμα χρησιμοποίησα το Wingdings.
4. Φορτώστε την παρουσίαση χρησιμοποιώντας το Presentation και ορίζοντας τις επιλογές φόρτωσης.
5. Τώρα, δημιουργήστε τη μικρογραφία διαφάνειας, το PDF και το XPS για να επαληθεύσετε τα αποτελέσματα.

Η υλοποίηση του παραπάνω δίνεται παρακάτω.

```py
import aspose.slides as slides

# Χρησιμοποιήστε τις επιλογές φόρτωσης για να ορίσετε τις προεπιλεγμένες κανονικές και ασιατικές γραμματοσειρές# Χρησιμοποιήστε τις επιλογές φόρτωσης για να ορίσετε τις προεπιλεγμένες κανονικές και ασιατικές γραμματοσειρές
loadOptions = slides.LoadOptions(slides.LoadFormat.AUTO)
loadOptions.default_regular_font = "Wingdings"
loadOptions.default_asian_font = "Wingdings"

# Φορτώστε την παρουσίαση
with slides.Presentation(path + "DefaultFonts.pptx", loadOptions) as pptx:
    # Δημιουργία μικρογραφίας διαφάνειας
    with pptx.slides[0].get_image(1, 1) as img:
        img.save("output_out.png", slides.ImageFormat.PNG)

    # Δημιουργία PDF
    pptx.save("output_out.pdf", slides.export.SaveFormat.PDF)

    # Δημιουργία XPS
    pptx.save("output_out.xps", slides.export.SaveFormat.XPS)
```
