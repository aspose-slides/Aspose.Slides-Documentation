## **Εισαγωγή**

Στο PowerPoint 2019, η Microsoft παρουσίασε τη ρύθμιση **Always Open Read-Only** ως μία από τις επιλογές που μπορούν να χρησιμοποιήσουν οι χρήστες για να προστατεύουν τις παρουσιάσεις τους. Μπορεί να θέλετε να χρησιμοποιήσετε αυτήν τη ρύθμιση μόνο για ανάγνωση ώστε να προστατεύσετε μία παρουσίαση όταν

- Θέλετε να αποτρέψετε τυχαίες επεξεργασίες και να διατηρήσετε το περιεχόμενο της παρουσίασής σας ασφαλές. 
- Θέλετε να ενημερώσετε τους ανθρώπους ότι η παρουσίαση που παρέχετε είναι η τελική έκδοση. 

Αφού επιλέξετε την επιλογή **Always Open Read-Only** για μια παρουσίαση, όταν οι χρήστες την ανοίξουν, βλέπουν τη σύσταση **Read-Only** και μπορεί να εμφανιστεί ένα μήνυμα με την εξής μορφή: *Για να αποτραπούν τυχαίες αλλαγές, ο δημιουργός έχει ορίσει αυτό το αρχείο να ανοίγει μόνο για ανάγνωση.*

Η σύσταση **Read-Only** είναι μια απλή αλλά αποτελεσματική αποθαρρυντική μέτρηση που αποτρέπει την επεξεργασία, καθώς οι χρήστες πρέπει να εκτελέσουν μια ενέργεια για να την αφαιρέσουν πριν τους επιτραπεί η επεξεργασία της παρουσίασης. Εάν δεν θέλετε χρήστες να κάνουν αλλαγές σε μια παρουσίαση και θέλετε να το τους κοινοποιήσετε με ευγενικό τρόπο, τότε η σύσταση **Read-Only** μπορεί να είναι μια καλή επιλογή για εσάς. 

> Εάν μια παρουσίαση με την προστασία **Read-Only** ανοίξει σε μια παλαιότερη έκδοση του Microsoft PowerPoint — η οποία δεν υποστηρίζει τη νεοεισαγόμενη λειτουργία — η σύσταση **Read-Only** παραβλέπεται (η παρουσίαση ανοίγει κανονικά).

## **Εφαρμογή λειτουργίας Read-Only**

Η Aspose.Slides for Python via .NET σάς επιτρέπει να ορίσετε μια παρουσίαση σε **Read-Only**, πράγμα που σημαίνει ότι οι χρήστες (αφού ανοίξουν την παρουσίαση) βλέπουν τη σύσταση **Read-Only**. Αυτό το δείγμα κώδικα δείχνει πώς να ορίσετε μια παρουσίαση σε **Read-Only** με Python χρησιμοποιώντας Aspose.Slides:

```py
import aspose.slides as slides

with slides.Presentation() as pres:
    pres.protection_manager.read_only_recommended = True
    pres.save("ReadOnlyPresentation.pptx", slides.export.SaveFormat.PPTX)
```

{{% alert color="primary" %}} 

**Σημείωση**: Η σύσταση **Read-Only** προορίζεται απλώς για αποθάρρυνση της επεξεργασίας ή για την αποτροπή τυχαίων αλλαγών σε μια παρουσίαση PowerPoint. Εάν ένα πρόσωπο με κίνητρο — που γνωρίζει τι κάνει — αποφασίσει να επεξεργαστεί την παρουσίασή σας, μπορεί εύκολα να καταργήσει τη ρύθμιση Read-Only. Εάν χρειάζεστε σοβαρή πρόληψη ανεξουσιοδοτημένης επεξεργασίας, είναι προτιμότερο να χρησιμοποιήσετε [πιο αυστηρές προστασίες που περιλαμβάνουν κρυπτογράφηση και κωδικούς πρόσβασης](https://docs.aspose.com/slides/el/python-net/password-protected-presentation/).

## **Συχνές ερωτήσεις**

**Πώς διαφέρει το 'Read-Only recommended' από πλήρη προστασία με κωδικό;**

'Read-Only recommended' εμφανίζει μόνο μια πρόταση να ανοίξετε το αρχείο σε λειτουργία μόνο για ανάγνωση και είναι εύκολο να παρακαμφθεί. [Προστασία με κωδικό](/slides/el/python-net/password-protected-presentation/) περιορίζει πραγματικά το άνοιγμα ή την επεξεργασία και είναι κατάλληλη όταν χρειάζεστε πραγματικό έλεγχο ασφαλείας.

**Μπορεί το 'Read-Only recommended' να συνδυαστεί με υδατογραφήματα για περαιτέρω αποθάρρυνση των επεξεργασιών;**

Ναι. Η σύσταση μπορεί να συνδυαστεί με [υδατογραφήματα](/slides/el/python-net/watermark/) ως οπτική αποθάρρυνση· είναι ξεχωριστοί μηχανισμοί και λειτουργούν καλά μαζί.

**Μπορεί ένα μακροεντολή ή εξωτερικό εργαλείο να τροποποιήσει ακόμα το αρχείο όταν η σύσταση είναι ενεργή;**

Ναι. Η σύσταση δεν εμποδίζει προγραμματιστικές αλλαγές. Για να αποτρέψετε αυτοματοποιημένες επεμβάσεις, χρησιμοποιήστε [κωδικούς πρόσβασης και κρυπτογράφηση](/slides/el/python-net/password-protected-presentation/).

**Πώς σχετίζεται το 'Read-Only recommended' με τις σημαίες 'is_encrypted' και 'is_write_protected';**

Είναι διαφορετικά σήματα. 'Read-Only recommended' είναι ένα ήπιο, προαιρετικό προτράγεμα· [is_write_protected](https://reference.aspose.com/slides/el/python-net/aspose.slides/protectionmanager/is_write_protected/) και [is_encrypted](https://reference.aspose.com/slides/el/python-net/aspose.slides/protectionmanager/is_encrypted/) υποδεικνύουν πραγματικούς περιορισμούς εγγραφής ή ανάγνωσης που εξαρτώνται από κωδικούς πρόσβασης ή κρυπτογράφηση.
