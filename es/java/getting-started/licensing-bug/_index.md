Este artículo describe un error de licenciamiento encontrado en las versiones 23.2, 23.3, 23.4, 23.5, 23.6 y 23.7 de Aspose.Slides para Java. Este error hizo que las licencias perpetuas se volvieran temporalmente inutilizables después de que su suscripción expirara.

## Síntomas ##

Después de que expira su suscripción de licencia perpetua, Aspose.Slides para Java versiones 23.2 - 23.7 puede reportar la licencia como expirada, trasladando todas las funciones a un modo de evaluación. Este comportamiento es incorrecto y no afecta las versiones anteriores a 23.2 ni posteriores a 23.7.

## Resolución ##

Aspose.Slides para Java abordó este problema y lanzó versiones actualizadas (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) con la solución.

Si está utilizando una de las versiones afectadas de Aspose.Slides para Java en su proyecto, por favor cambie a la versión corregida.

Tenga en cuenta que este error solo afecta a las versiones de Aspose.Slides para Java desde la 23.2 hasta la 23.7. **Otros productos de Aspose y otras versiones de Aspose.Slides no están afectados**.
