{{% alert color="primary" %}} 

Aspose.Slides for Java es una biblioteca de clases que permite a las aplicaciones Java leer y escribir documentos de presentación sin usar Microsoft PowerPoint® u otros. Aspose.Slides for Java permite gestionar texto, formas, tablas y animaciones, añadir audio y vídeo a las diapositivas, previsualizar diapositivas, exportar diapositivas a formato SVG, PDF y más.

La referencia de la API más reciente se encuentra en [Referencia de la API de Aspose.Slides for Java](https://reference.aspose.com/slides/es/java).
