## **Soporte para Código Legado**
Para usar el código heredado desarrollado con versiones de Aspose.Slides for Java anteriores a la 13.x, debe hacer algunos cambios menores en su código y éste funcionará como antes. Todas las clases que estaban presentes en el antiguo Aspose.Slides for Java bajo los espacios de nombres Aspose.Slide y Aspose.Slides.Pptx ahora se han fusionado en un único espacio de nombres Aspose.Slides.
