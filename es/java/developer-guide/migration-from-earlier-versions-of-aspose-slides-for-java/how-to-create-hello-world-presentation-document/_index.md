{{% /alert %}} 
## **Soporte para código heredado**
Para utilizar el código heredado desarrollado con versiones de Aspose.Slides para Java anteriores a 13.x, necesita hacer algunos cambios menores en su código y este funcionará como antes. Todas las clases que estaban presentes en Aspose.Slides para Java bajo los espacios de nombres Aspose.Slide y Aspose.Slides.Pptx ahora se han fusionado en un único espacio de nombres Aspose.Slides. Por favor, eche un vistazo al siguiente fragmento de código simple para crear un documento de presentación Hello World en la API heredada de Aspose.Slides y siga los pasos que describen cómo migrar a la nueva API fusionada.
