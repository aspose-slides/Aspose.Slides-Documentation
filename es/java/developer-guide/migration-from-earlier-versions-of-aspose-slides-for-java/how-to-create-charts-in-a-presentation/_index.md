{{% /alert %}} 
## **Soporte para código heredado**
Para utilizar el código heredado desarrollado con Aspose.Slides para Java en versiones anteriores a 14.x.x, necesitas realizar algunos cambios menores en tu código y este funcionará como antes. Todas las clases que estaban presentes en la antigua Aspose.Slides para Java bajo los namespaces com.aspose.slides y com.aspose.slides.pptx ahora se han fusionado en un único namespace com.aspose.slides.
