Se ha lanzado una nueva [Aspose.Slides for Java API](https://docs.aspose.com/slides/java/) y ahora este único producto admite la capacidad de generar documentos PowerPoint desde cero y editar los existentes.

## **Compatibilidad con Código Legado**
Para usar el código legado desarrollado con versiones de Aspose.Slides for Java anteriores a la 14.x.x, debe realizar algunos cambios menores en su código y éste seguirá funcionando como antes. Todas las clases que estaban presentes en el Aspose.Slides for Java antiguo bajo los espacios de nombres com.aspose.slides y com.aspose.slides.pptx ahora se han fusionado en un único espacio de nombres com.aspose.slides.
