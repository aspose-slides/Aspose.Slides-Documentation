Aspose.Slides for Java 14.3.0 (14.4.0) ofrece una implementación completamente nueva del procesamiento de PPT. Hay muchas mejoras, conversión parcial de PPTX a PPT. Pero existen algunas características no implementadas:

- Algunas formas tienen geometría incorrecta en los documentos PPT serializados (Call outs)
- No todas las características de formato de texto de PPTX son compatibles en la serialización a PPT
- La información sobre el idioma del texto y la configuración ortográfica no está presente en los documentos PPT serializados
- No todas las características de los temas de PPTX son compatibles en la serialización a PPT

**Hay algunas diferencias en comparación con Aspose.Slides for Java 8.6.0:**

- Existen problemas conocidos en la serialización de PPT OLE/ActiveX a PPT

**Hay algunas diferencias en comparación con Aspose.Slides for .NET 14.3.0:**

- El soporte de impresión de presentaciones no está disponible actualmente en Aspose.Slides for Java
