## **Introducción**
Microsoft Visual Studio es un entorno de desarrollo integrado (IDE) de Microsoft. Se utiliza para desarrollar programas informáticos para Microsoft Windows, así como sitios web, aplicaciones web y servicios web. Visual Studio emplea plataformas de desarrollo de software de Microsoft como Windows API, Windows Forms, Windows Presentation Foundation, Windows Store y Microsoft Silverlight. Puede generar tanto código nativo como código administrado.

### **Aspose.Slides for .NET**
Aspose.Slides for .NET es una API única de gestión de PowerPoint® que permite a las aplicaciones .NET leer, escribir y manipular documentos de PowerPoint sin usar Microsoft PowerPoint.

Aspose.Slides for .NET es la primera y única API de PowerPoint para .NET que proporciona funcionalidad para gestionar documentos de PowerPoint dentro de sus propias aplicaciones. Al igual que todos los componentes Aspose .NET, Aspose.Slides está escrita en C# administrado, tiene un precio increíblemente competitivo y es extremadamente rápido.

Para más detalles sobre [API de Aspose.Slides para .NET](http://www.aspose.com/.net/powerpoint-component.aspx)

### **Explorar ejemplos de Aspose.Slides para .NET usando Visual Studio**
Aspose Visual Studio Plugin - **Explorar ejemplos de Aspose.Slides para .NET** es una herramienta excelente para descargar y explorar rápidamente ejemplos de la API de Aspose.Slides para .NET. Ahorra mucho tiempo y esfuerzo al ofrecer una opción muy sencilla para seleccionar, descargar y abrir los últimos proyectos de ejemplo sin tener que salir de Visual Studio.

Esta versión ofrece las siguientes características:

- Compatibilidad con Visual Studio 2010, Visual Studio 2012 y Visual Studio 2013.
- Fácil de lanzar desde el menú **File > Aspose** en Visual Studio
- Ahorra tiempo y reduce la curva de aprendizaje.
- Permite seleccionar y abrir los últimos ejemplos de la API de Aspose.Slides para .NET alojados en Github.com.
- Le permite abrir proyectos de ejemplos **C#** o **VB.NET**
- El proyecto seleccionado se abre automáticamente en Visual Studio con todas las referencias necesarias configuradas, listo para ejecutar y explorar.
- Descarga automáticamente las últimas bibliotecas de la API desde **NuGet** y los ejemplos desde **GitHub**.

## **Requisitos del sistema y plataformas compatibles**
Para instalar y usar Aspose Visual Studio Plugin necesita tener instalada una de las siguientes versiones de Visual Studio:

- Visual Studio 2010, 2012, 2013

No dude en ponerse en contacto con nosotros si encuentra cualquier problema al instalar o usar este plugin.

## **Instalación**
Instalar Aspose Visual Studio Plugin es muy sencillo, siga estos pasos:

- Haga doble clic en el archivo MSI descargado
- Siga los pasos, simplemente presione **Next** en cada pantalla para instalarlo con la configuración estándar.

Obtendrá un cuadro de diálogo de éxito una vez completada la instalación.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_2)

**Nota:** Asegúrese de reiniciar Visual Studio para que los cambios tengan efecto.

## **Uso**
Una vez instalado, puede lanzar y usar el plugin interactivo fácilmente como se describe a continuación:
**1.** Abra el plugin seleccionando **File > Aspose** seguido de **New Aspose.Slides Example Project** en Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_3)

**2.** La primera vez que abra el plugin verá el cuadro de diálogo de selección de API como se muestra a continuación. Tomará unos minutos descargar la última API y sus bibliotecas desde **NuGet** y el código fuente de los últimos ejemplos desde **GitHub**.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_4)

**3.** En el cuadro de diálogo *Select an API to View Examples*, seleccione o cambie la ubicación donde se crearán los proyectos de ejemplo y también seleccione o cambie el tipo de proyecto de ejemplo como **C#** o **VB.NET** y haga clic en **Continue** para abrir el proyecto de ejemplo en Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_1)

**4.** Nuevos proyectos de ejemplo según la selección del tipo de proyecto **C#** o **VB.NET** en Visual Studio.

#### **Cómo configurar el código fuente**
Necesita tener lo siguiente instalado para abrir y ampliar el código fuente:

- Visual Studio 2010

Siga estos pasos simples para comenzar:

1. Descargue/clone el código fuente.
1. Abra Visual Studio 2010 y elija **File > Open Project**
1. Navegue hasta el código fuente más reciente que haya descargado y abra **AsposeVisualStudioPlugin.sln**

#### **Visión general del código fuente**
Hay dos proyectos en la solución:

- **AsposeVisualStudioPlugin** – Contiene la integración con Visual Studio, la selección y descarga automática de las API de Aspose y los ejemplos.
- **Installer** – Proyecto de instalador Wix para generar el archivo MSI del plugin.
