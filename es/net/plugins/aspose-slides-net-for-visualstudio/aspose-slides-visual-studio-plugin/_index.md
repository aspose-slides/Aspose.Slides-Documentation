## **Introducción**
Microsoft Visual Studio es un entorno de desarrollo integrado (IDE) de Microsoft. Se usa para desarrollar programas informáticos para Microsoft Windows, así como sitios web, aplicaciones web y servicios web. Visual Studio utiliza plataformas de desarrollo de software de Microsoft como Windows API, Windows Forms, Windows Presentation Foundation, Windows Store y Microsoft Silverlight. Puede producir tanto código nativo como código administrado.
### **Aspose.Slides for .NET**
Aspose.Slides for .NET es una API única de gestión de PowerPoint® que permite a las aplicaciones .NET leer, escribir y manipular documentos PowerPoint sin usar Microsoft PowerPoint.

Aspose.Slides for .NET es la primera y única API de PowerPoint para .NET que brinda funcionalidad para gestionar documentos PowerPoint dentro de sus propias aplicaciones. Como todos los componentes Aspose .NET, Aspose.Slides está escrito en C# administrado, tiene un precio increíble y es ultrarrápido.

Para más detalles sobre [Aspose.Slides for .NET API](http://www.aspose.com/.net/powerpoint-component.aspx)
### **Explore Aspose.Slides for .NET Examples Using Visual Studio**
Aspose Visual Studio Plugin - **Explore Aspose.Slides for .NET Examples** es una excelente herramienta para descargar y explorar rápidamente ejemplos de la API Aspose.Slides for .NET. Ahorra mucho tiempo y esfuerzo al proporcionar una opción muy simple para seleccionar, descargar y abrir los últimos proyectos de ejemplo sin tener que salir de Visual Studio.

Esta versión incluye las siguientes características:

- Compatible con Visual Studio 2010, Visual Studio 2012 y Visual Studio 2013.
- Fácil de iniciar desde el menú **File > Aspose** en Visual Studio
- Ahorra tiempo y reduce la curva de aprendizaje.
- Permite seleccionar y abrir los últimos ejemplos de la API Aspose.Slides for .NET alojados en Github.com.
- Le permite abrir proyectos de ejemplos **C#** o **VB.NET**
- Un proyecto seleccionado se abre automáticamente en Visual Studio con todas las referencias requeridas configuradas para ofrecerle un entorno listo para ejecutar y explorar.
- Descarga automáticamente las últimas bibliotecas de la API desde **NuGet** y los ejemplos desde **GitHub**.

Es muy sencillo instalar Aspose Visual Studio Plugin, siga estos pasos simples

- Haga doble clic en el archivo msi descargado
- Siga los pasos, simplemente presione **Next** en cada pantalla para instalarlo con la configuración estándar.

Obtendrá un cuadro de diálogo de éxito una vez que la instalación haya finalizado.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_2)

**Nota:** Asegúrese de reiniciar Visual Studio para que los cambios surtan efecto.
## **Uso**
Una vez instalado, puede lanzar y usar fácilmente el complemento interactivo como se describe a continuación:
**1.** Abra el complemento seleccionando **File > Aspose** seguido de **New Aspose.Slides Example Project** en Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_3)

**2.** La primera vez que abre el complemento ve el cuadro de diálogo de selección de API como se muestra a continuación. Tomará unos minutos descargar la última API y sus bibliotecas desde **NuGet** y el código fuente de los últimos ejemplos desde **GitHub**.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_4)

**3.** En el cuadro de diálogo **Select an API to View Examples**, seleccione o cambie la ubicación donde se crearán los proyectos de ejemplo y también seleccione o cambie el tipo de proyecto de ejemplo como **C#** o **VB.NET** y haga clic en **Continue** para abrir el proyecto de ejemplo en Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_1)

**4.** Nuevos proyectos de ejemplos según la selección del tipo de proyecto **C#** o **VB.NET** en Visual Studio.
