## **Introducción**
Microsoft Visual Studio es un entorno de desarrollo integrado (IDE) de Microsoft. Se utiliza para desarrollar programas informáticos para Microsoft Windows, así como sitios web, aplicaciones web y servicios web. Visual Studio utiliza plataformas de desarrollo de software de Microsoft como Windows API, Windows Forms, Windows Presentation Foundation, Windows Store y Microsoft Silverlight. Puede producir tanto código nativo como código gestionado.
### **Aspose.Slides para .NET**
Aspose.Slides para .NET es una API única de gestión de PowerPoint® que permite a las aplicaciones .NET leer, escribir y manipular documentos de PowerPoint sin utilizar Microsoft PowerPoint.

Aspose.Slides para .NET es la primera y única API de PowerPoint para .NET que proporciona funcionalidad para gestionar documentos de PowerPoint dentro de sus propias aplicaciones. Al igual que todos los componentes de Aspose .NET, Aspose.Slides está escrito en C# gestionado, tiene un precio increíble y es extremadamente rápido.

Para obtener más detalles sobre [Aspose.Slides para .NET API](http://www.aspose.com/.net/powerpoint-component.aspx)
### **Explorar Ejemplos de Aspose.Slides para .NET usando Visual Studio**
Plugin de Visual Studio de Aspose - **Explorar Ejemplos de Aspose.Slides para .NET** es una gran herramienta para descargar y explorar rápidamente ejemplos de la API de Aspose.Slides para .NET. Ahorra mucho tiempo y esfuerzo al proporcionar una opción muy simple para seleccionar, descargar y abrir sin problemas los últimos proyectos de ejemplo sin tener que abandonar Visual Studio.

Esta versión proporciona las siguientes características:

- Soporta Visual Studio 2010, Visual Studio 2012 y Visual Studio 2013.
- Fácil de lanzar desde el menú **Archivo > Aspose** en Visual Studio
- Ahorra tiempo y reduce la curva de aprendizaje.
- Le permite seleccionar y abrir los últimos ejemplos de la API de Aspose.Slides para .NET alojados en Github.com.
- Permite abrir proyectos de ejemplos en **C#** o **VB.NET**
- Un proyecto seleccionado se abre automáticamente en Visual Studio con todas las referencias necesarias configuradas para brindarle un entorno listo para ejecutar y explorar.
- Descarga automáticamente las últimas bibliotecas de API de **NuGet** y ejemplos de **GitHub**.

## **Uso**
Una vez instalado, puede iniciar y usar fácilmente el plugin interactivo como se describe a continuación:
**1.** Abra el plugin seleccionando **Archivo > Aspose** seguido de **Nuevo Proyecto de Ejemplo Aspose.Slides** en Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_3)

**2.** La primera vez que abra el plugin verá el diálogo de selección de API como se muestra a continuación. Tomará unos minutos descargar la última API y sus bibliotecas desde **NuGet** y el código fuente de los últimos ejemplos desde **GitHub**.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_4)

**3.** En el diálogo Seleccionar una API para Ver Ejemplos, seleccione o cambie la ubicación donde se crearán los proyectos de ejemplo y también seleccione o cambie el tipo de proyecto de ejemplos como **C#** o **VB.NET** y haga clic en **Continuar** para abrir el proyecto de ejemplo en Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_1)

**4.** Nuevos proyectos de Ejemplo según la selección del tipo de proyecto de **C#** o **VB.NET** en Visual Studio.
