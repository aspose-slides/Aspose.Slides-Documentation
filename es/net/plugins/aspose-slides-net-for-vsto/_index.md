## **¿Qué es VSTO?**
VSTO, o Visual Studio Tools for Office, o Microsoft Visual Studio Tools, es parte del conjunto de herramientas Visual Studio .NET de Microsoft y admite el Microsoft .NET Framework. Los desarrolladores lo utilizan para escribir código que conecta las aplicaciones Microsoft Office Word, Excel y Outlook. Los desarrolladores que usan VSTO pueden emplear los lenguajes C# o Visual Basic. Es una alternativa a Visual Basic for Applications, o VBA.
## **Aspose.Slides for .NET**
Aspose.Slides for .NET es un componente único de gestión de PowerPoint® que permite a las aplicaciones .NET leer, escribir y manipular documentos PowerPoint sin utilizar Microsoft PowerPoint.

Aspose.Slides for .NET es el primer y único componente .NET que ofrece funcionalidad para gestionar documentos PowerPoint dentro de sus propias aplicaciones. Al igual que todos los componentes Aspose .NET, Aspose.Slides está escrito en C# administrado, tiene un precio increíble y es extremadamente rápido.

## **Requisitos del Sistema y Plataformas Compatibles**
### **Requisitos del Sistema**
Una o más de las siguientes ediciones de Visual Studio Tools for Office:

- Microsoft Visual Studio 2005 Tools for the Microsoft Office System
- Microsoft Visual Studio 2005 Tools for the 2007 Microsoft Office System
- Visual Studio 2008 Professional Edition
- Visual Studio 2008 Team Suite Edition
- Una versión de Microsoft Office:
- Microsoft Office Professional 2003 SP1
- 2007 Microsoft Office system
### **Sistemas Operativos Compatibles**
Windows 2000; Windows Server 2003; Windows Vista; Windows XP
### **Versiones de .NET Framework compatibles**
solo .NET Framework 2.0 o superior.
