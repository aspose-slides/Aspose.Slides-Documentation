## **¿Qué es VSTO?**
VSTO, o Visual Studio Tools for Office, o Microsoft Visual Studio Tools es parte del conjunto de herramientas de Microsoft Visual Studio .NET y soporta el Microsoft .NET Framework. Se utiliza por los desarrolladores para escribir código que conecte las aplicaciones de Microsoft Office Word, Excel y Outlook. Los desarrolladores que utilizan VSTO pueden emplear los lenguajes C# o Visual Basic. Es una alternativa a Visual Basic for Applications, o VBA.
## **Aspose.Slides para .NET**
Aspose.Slides para .NET es un componente único de gestión de PowerPoint® que permite a las aplicaciones .NET leer, escribir y manipular documentos de PowerPoint sin usar Microsoft PowerPoint.

Aspose.Slides para .NET es el primer y único componente .NET que proporciona funcionalidad para gestionar documentos de PowerPoint dentro de tus propias aplicaciones. Al igual que todos los componentes de Aspose .NET, Aspose.Slides está escrito en C# gestionado, increíblemente asequible y rápido como un rayo.

## **Requisitos del sistema y plataformas soportadas**
### **Requisitos del sistema**
Una o más de las siguientes ediciones de Visual Studio Tools for Office:

- Microsoft Visual Studio 2005 Tools para el Sistema Microsoft Office
- Microsoft Visual Studio 2005 Tools para el Sistema Microsoft Office 2007
- Visual Studio 2008 Professional Edition
- Visual Studio 2008 Team Suite Edition
- Una versión de Microsoft Office:
- Microsoft Office Professional 2003 SP1
- sistema Microsoft Office 2007
### **Sistemas operativos soportados**
Windows 2000; Windows Server 2003; Windows Vista; Windows XP
### **Versiones de .NET Framework soportadas**
solo .NET framework 2.0 o superior.
