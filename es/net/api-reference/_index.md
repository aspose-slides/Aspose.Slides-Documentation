{{% alert color="primary" %}}

Aspose.Slides for .NET es una biblioteca de clases que permite a las aplicaciones .NET leer y escribir documentos de presentación sin utilizar Microsoft PowerPoint® u otros. Aspose.Slides for .NET permite gestionar texto, formas, tablas y animaciones, añadir audio y vídeo a las diapositivas, previsualizar diapositivas, exportar diapositivas a formato SVG, PDF y más.

La referencia más reciente de la API se encuentra en [Referencia de API de Aspose.Slides for .NET](https://reference.aspose.com/slides/es/net).
