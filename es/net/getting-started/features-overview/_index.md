## **Formatos de archivo y conversiones**
Aspose.Slides for .NET admite la mayoría de los formatos de documento de PowerPoint. También le permite exportarlos a los formatos populares que las organizaciones utilizan y comparten entre sí. Consulte estos detalles:

|**Funcionalidad**|**Descripción**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/es/net/ppt-vs-pptx/)|Aspose.Slides for .NET ofrece el procesamiento más rápido para este formato de documento de presentación.|
|[PPT to PPTX conversion](/slides/es/net/convert-ppt-to-pptx/)|Aspose.Slides for .NET admite la conversión de PPT a PPTX.|
|[Portable Document Format (PDF)](/slides/es/net/convert-powerpoint-ppt-and-pptx-to-pdf/)|Puede exportar todos los formatos de archivo compatibles a documentos Adobe Portable Document Format (PDF) con un único método.|
|[XML Parser Specification (XPS)](https://docs.aspose.com/slides/net/convert-powerpoint-to-xps/)|Puede exportar todos los formatos de archivo compatibles a documentos XML Parser Specification (XPS) con un único método.|
|[Tagged Image File Format (TIFF)](/slides/es/net/convert-powerpoint-to-tiff/)|Puede exportar todos los formatos de archivo de presentación compatibles a Tagged Image File Format (TIFF).|
|[PPTX To HTML Conversion](/slides/es/net/convert-powerpoint-ppt-and-pptx-to-html/)|Aspose.Slides for .NET admite la conversión de PresentationEx a formato HTML.|

## **Preguntas frecuentes**

**¿Necesito instalar Microsoft PowerPoint en el servidor/PC para que la biblioteca funcione?**

No. PowerPoint no es necesario; Aspose.Slides es un motor independiente para crear, editar, convertir y renderizar presentaciones.

**¿Cómo funciona el multihilo? ¿Puede procesarse en paralelo?**

It is safe to process different documents in different threads; the same [presentation](https://reference.aspose.com/slides/net/aspose.slides/presentation/) object must not be used by [multiple threads](/slides/es/net/multithreading/) at the same time.

**¿Se admiten contraseñas y cifrado de archivos?**

Yes. [You can](/slides/es/net/password-protected-presentation/) open encrypted presentations, set or remove an open and write password, and check the protection status.

**¿Debo preocuparme por los paquetes de fuentes en contenedores Linux?**

Yes. It is recommended to install common font packages and/or explicitly [specify font directories](/slides/es/net/custom-font/) in your application to avoid unexpected substitutions.

**¿Hay limitaciones en la versión de evaluación?**

In [evaluation mode](/slides/es/net/licensing/), a watermark is added to the output and certain limitations apply; a [30-day temporary license](https://purchase.aspose.com/temporary-license/) is available for full-feature testing.

**¿Se admite la importación de formatos externos a una presentación (PDF/HTML → PPTX)?**

Yes. You can add [PDF pages and HTML content](/slides/es/net/import-presentation/) to a presentation, turning them into slides.
