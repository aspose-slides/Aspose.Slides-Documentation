|**Característica**|**Descripción**|
| :- | :- |

## **Formatos de Archivo y Conversiones**
Aspose.Slides para .NET soporta la mayoría de los formatos de documentos de PowerPoint. También te permite exportarlos a los formatos populares que las organizaciones utilizan e intercambian entre sí. Consulta estos detalles:

|**Característica**|**Descripción**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/es/net/ppt-vs-pptx/)|Aspose.Slides para .NET proporciona el procesamiento más rápido para este formato de documento de presentación.|
|[Conversión de PPT a PPTX](/slides/es/net/convert-ppt-to-pptx/)|Aspose.Slides para .NET soporta la conversión de PPT a PPTX.|
|[Formato de Documento Portátil (PDF)](/slides/es/net/convert-powerpoint-ppt-and-pptx-to-pdf/)|Puedes exportar todos los formatos de archivo soportados a documentos en formato de Documento Portátil de Adobe (PDF) con un solo método.|
|[Especificación de Analizador XML (XPS)](https://docs.aspose.com/slides/net/convert-powerpoint-to-xps/)|Puedes exportar todos los formatos de archivo soportados a documentos en Especificación de Analizador XML (XPS) con un solo método.|
|[Formato de Archivo de Imagen Etiquetada (TIFF)](/slides/es/net/convert-powerpoint-to-tiff/)|Puedes exportar todos los formatos de archivo de presentación soportados a Formatos de Archivo de Imagen Etiquetada (TIFF).|
|[Conversión de PPTX a HTML](/slides/es/net/convert-powerpoint-ppt-and-pptx-to-html/)|Aspose.Slides para .NET soporta la conversión de PresentationEx a formato HTML.|
## **Renderización e Impresión**
Aspose.Slides para .NET soporta la renderización de alta fidelidad de las diapositivas en los documentos de presentación a varios formatos gráficos. Consulta estos detalles:

|**Característica**|**Descripción**|
| :- | :- |
|Formatos de Imagen Soportados por .NET|Con Aspose.Slides para .NET, puedes renderizar diapositivas de presentación e imágenes en las diapositivas a todos los formatos gráficos soportados por .NET como TIFF, PNG, BMP, JPEG, GIF y metafiles.|
|Formato SVG|Aspose.Slides para .NET también proporciona métodos integrados que te permiten exportar diapositivas de presentación a formatos de Gráficos Vectoriales Escalables (SVG).|
|Impresión de Presentaciones|Las últimas versiones de Aspose.Slides para .NET proporcionan métodos de impresión integrados con diferentes opciones.|
## **Características de Contenido**
Aspose.Slides para .NET te permite acceder, modificar o crear casi todos los elementos o contenidos de documentos de presentación. Consulta estos detalles:

|**Característica**|**Descripción**|
| :- | :- |
|Diapositivas Maestras|Las diapositivas maestras definen el diseño de las diapositivas normales. Aspose.Slides para .NET te permite acceder y modificar las diapositivas maestras de documentos de presentación|
|Diapositivas Normales|Con Aspose.Slides para .NET, puedes crear nuevas diapositivas de diferentes tipos; también puedes acceder y modificar diapositivas existentes en las presentaciones|
|Clonación / Copia de Diapositivas|Existen métodos integrados proporcionados por Aspose.Slides para .NET que te permiten clonar o copiar diapositivas existentes dentro de una presentación. También puedes usar diapositivas copiadas y clonadas de una presentación a otra. Dado que una diapositiva hereda su diseño de la diapositiva maestra, los métodos de clonación integrados copian automáticamente la maestra al clonar|
|Gestión de Secciones de Diapositivas|Métodos para organizar diapositivas en diferentes secciones dentro de una presentación|
|Marcadores de Posición y Marcadores de Texto|Puedes acceder a los marcadores de posición y marcadores de texto en una diapositiva. Además, puedes crear una diapositiva con marcadores de texto desde cero utilizando el método apropiado|
|Encabezados y Pies de Página|Aspose.Slides para .NET facilita el manejo de encabezados/pies de página en diapositivas|
|Notas en Diapositivas|Con Aspose.Slides para .NET, puedes acceder y modificar las notas asociadas con una diapositiva y también agregar nuevas notas|
|Encontrar una Forma|También puedes encontrar una forma particular de una diapositiva utilizando el texto alternativo asociado con la forma|
|Fondos|Aspose.Slides para .NET te permite trabajar con fondos asociados a una diapositiva maestra o normal en una presentación|
|Cajas de Texto|Las cajas de texto se pueden crear desde cero. Puedes acceder a las cajas de texto existentes. También puedes modificar sus textos sin perder el formato de texto original|
|Formas Rectangulares|Puedes crear o modificar formas rectangulares con Aspose.Slides para .NET|
|Formas de Polilínea|Puedes crear o modificar formas de polilínea con Aspose.Slides para .NET|
|Formas Elípticas|Puedes crear o modificar formas elípticas con Aspose.Slides para .NET|
|Formas Agrupadas| Aspose.Slides para .NET soporta formas agrupadas                 |
|Formas Automáticas|Aspose.Slides para .NET soporta formas automáticas|
|SmartArt|Aspose.Slides para .NET proporciona soporte para formas SmartArt en MS PowerPoint|
|Gráficos|Aspose.Slides para .NET proporciona soporte para gráficos MSO en PowerPoint|
|Serialización de Formas| Aspose.Slides para .NET soporta una gran cantidad de formas. Cuando Aspose.Slides para .NET carece de soporte para una forma, puedes usar un método de serialización a través del cual puedes serializar esa forma de una diapositiva existente. De esta manera, puedes usar la forma según tus requisitos |
|Marcos de Imágenes|Puedes gestionar imágenes en marcos de imágenes con Aspose.Slides para .NET|
|Marcos de Audio|Puedes vincular o incrustar archivos de audio en marcos de audio en las diapositivas con Aspose.Slides para .NET|
|Marcos de Video| Puedes manejar archivos de video en marcos de video. Aspose.Slides para .NET también proporciona soporte para videos vinculados e incrustados |
|Marco OLE|Puedes gestionar objetos OLE en marcos OLE con Aspose.Slides para .NET|
|Tablas|Aspose.Slides para .NET soporta tablas en diapositivas|
|Controles ActiveX|Soporte para controles ActiveX|
|Macros VBA|Soporte para gestionar macros VBA dentro de las presentaciones.|
|Marco de Texto|Puedes acceder al texto de cualquier forma a través del marco de texto asociado con esa forma|
|Escaneo de Texto|Puedes escanear texto en una presentación a nivel de presentación o de diapositiva a través de métodos de escaneo integrados.|
|Animaciones|Puedes aplicar animaciones a las formas|
|Presentaciones de Diapositivas|Aspose.Slides para .NET soporta presentaciones de diapositivas y transiciones entre diapositivas|
## **Características de Formato**
Con Aspose.Slides para .NET, puedes dar formato a textos y formas en diapositivas en las presentaciones. Consulta estos detalles:

|**Característica**|**Descripción**|
| :- | :- |
|Formato de Texto|<p>En Aspose.Slides para .NET, puedes gestionar textos a través de los marcos de texto asociados con las formas. Así, puedes dar formato a los textos utilizando los párrafos y partes asociadas con los marcos de texto. Estos elementos de texto pueden ser formateados a través de Aspose.Slides para .NET.</p><p>- Tipo de Fuente</p><p>- Tamaño de Fuente</p><p>- Color de Fuente</p><p>- Sombras de Fuente</p><p>- Alineación de Párrafo</p><p>- Viñetas de Párrafo</p><p>- Orientación de Párrafo</p>|
|Formato de Forma|<p>En Aspose.Slides para .NET, el elemento básico de una diapositiva es una forma. Puedes formatear estos elementos de forma con Aspose.Slides para .NET:</p><p>- Posición</p><p>- Tamaño</p><p>- Línea</p><p>- Relleno (incluyendo Patrón, Gradiente, Sólido)</p><p>- Texto</p><p>- Imagen</p>|
