## **Conversión de PPT a PPTX**
### **Acerca de la conversión de PPT a PPTX**
Aspose.Slides ahora admite la conversión de PPT a PPTX.

### **Convertir PPT a PPTX con Aspose.Slides**
Aspose.Slides for .NET proporciona soporte parcial para convertir presentaciones en formato de archivo PPT a presentaciones en formato de archivo PPTX. Como el soporte para la característica de conversión de presentaciones mencionada acaba de introducirse en Aspose.Slides for .NET, por el momento tiene capacidad limitada y funciona solo para la forma simple de presentaciones. La principal ventaja que la biblioteca de API de Aspose.Slides for .NET ofrece para convertir presentaciones PPT a presentaciones en formato PPTX es la facilidad de uso de la API para lograr el objetivo deseado. Por favor diríjase a este [enlace](/slides/es/net/convert-presentation/) a la sección de fragmentos de código para obtener más detalles. La siguiente sección ilustra claramente qué características son compatibles y cuáles no al convertir presentaciones en formato PPT a presentaciones en formato PPTX.

## **Conversión de presentación a PDF**
### **Acerca de PDF**
El [Portable Document Format](https://en.wikipedia.org/wiki/PDF) es un formato de archivo creado por Adobe System para el intercambio de documentos entre diferentes organizaciones. El propósito de este formato es permitir que el contenido de los documentos se represente de manera que su apariencia visual no dependa de la plataforma en la que se visualice.

### **PDF en Aspose.Slides para .NET**
El [PDF](https://docs.fileformat.com/pdf/) es un formato de archivo creado por Adobe System para el intercambio de documentos entre diferentes organizaciones. El propósito de este formato es permitir que la apariencia visual del contenido del documento no dependa de la plataforma en la que se visualice.

Cualquier documento de presentación que pueda cargarse en Aspose.Slides para .NET puede convertirse a documento PDF. Puede exportar los documentos de presentación a PDF directamente utilizando solo el componente Aspose.Slides para .NET. No necesita ninguna otra biblioteca de terceros ni el componente Aspose.PDF para este propósito.

## **Conversión de presentación a XPS**
### **Acerca de XPS**
El [Especificación del Analizador XML](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) es un lenguaje de descripción de página y un formato de documento fijo originalmente desarrollado por Microsoft. Al igual que PDF, XPS es un formato de documento de diseño fijo diseñado para preservar la fidelidad del documento y proporcionar una apariencia independiente del dispositivo.

### **Convertir presentación a XPS con Aspose.Slides**
Cualquier documento de presentación que pueda cargarse con Aspose.Slides para .NET puede convertirse al formato XPS. Aspose.Slides para .NET utiliza el motor de diseño de página y renderizado de alta fidelidad para generar salida en formato de documento XPS de diseño fijo. Cabe mencionar que Aspose.Slides para .NET genera XPS directamente sin depender de las clases de Windows Presentation Foundation (WPF) incluidas en .NET Framework 3.5, lo que permite producir documentos XPS en máquinas que ejecutan versiones de .NET Framework anteriores a la 3.5. Puede obtener información sobre la exportación de documentos de presentación a documentos XPS mediante Aspose.Slides para .NET en [este tema](/slides/es/net/convert-powerpoint-ppt-and-pptx-to-microsoft-xps-document/).

## **Conversión de presentación a TIFF**
### **Acerca de TIFF**
El [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) , conocido por almacenar múltiples imágenes en un solo documento, fue creado originalmente por Aldus. Este formato es ampliamente soportado por aplicaciones de escaneo, fax y otras de manipulación de imágenes.

### **Convertir presentación a TIFF con Aspose.Slides**
Cualquier documento que pueda cargarse en Aspose.Slide para .NET también puede convertirse directamente a documento TIFF mediante Aspose.Slides para .NET, eliminando la necesidad de componentes de terceros. Además, puede definir opcionalmente el tamaño de las imágenes en el documento TIFF resultante. Puede encontrar información sobre la exportación de documentos de presentación a documentos TIFF mediante Aspose.Slides para .NET en [este tema](/slides/es/net/convert-powerpoint-ppt-and-pptx-to-tiff/).

## **Preguntas frecuentes**

**¿Puedo incluir diapositivas ocultas al exportar a PDF/XPS?**

Sí. La exportación admite incluir diapositivas ocultas mediante la opción correspondiente en la configuración de [PDF](https://reference.aspose.com/slides/net/aspose.slides.export/pdfoptions/showhiddenslides/)/[XPS](https://reference.aspose.com/slides/net/aspose.slides.export/xpsoptions/showhiddenslides/).

**¿Se admite guardar en el formato PDF/A (para almacenamiento de archivo)?**

Sí, los niveles de cumplimiento PDF/A [están disponibles](https://reference.aspose.com/slides/net/aspose.slides.export/pdfcompliance/) (incluyendo A-2a/A-2b/A-2u y A-3a/A-3b) durante la exportación.

**¿Qué ocurre con las fuentes durante la conversión: se incrustan o sustituyen?**

Existen opciones flexibles: puede [incrustar todos los glifos o solo los subconjuntos utilizados](/slides/es/net/embedded-font/), especificar una [fuente de respaldo](/slides/es/net/fallback-font/), y [controlar el comportamiento](/slides/es/net/font-substitution/) cuando una fuente carece de ciertos estilos.

**¿Cómo puedo controlar la calidad y el tamaño del PDF resultante?**

Hay opciones disponibles para la [calidad JPEG](https://reference.aspose.com/slides/net/aspose.slides.export/pdfoptions/jpegquality/), la [compresión de texto](https://reference.aspose.com/slides/net/aspose.slides.export/pdfoptions/textcompression/), y un umbral de [resolución suficiente](https://reference.aspose.com/slides/net/aspose.slides.export/pdfoptions/sufficientresolution/) para imágenes, además de un modo que selecciona la [mejor compresión para imágenes](https://reference.aspose.com/slides/net/aspose.slides.export/pdfoptions/bestimagescompressionratio/).

**¿Puedo exportar solo un rango de diapositivas (por ejemplo, 5–12)?**

Sí, la exportación admite seleccionar un subconjunto de diapositivas.

**¿Se admite el procesamiento multinúcleo de varios archivos al mismo tiempo?**

Es aceptable procesar diferentes presentaciones en paralelo en procesos separados. Importante: el mismo objeto de [presentación](https://reference.aspose.com/slides/net/aspose.slides/presentation/) no debe cargarse ni guardarse desde [múltiples hilos](/slides/es/net/multithreading/).

**¿Existen riesgos al aplicar la licencia desde diferentes hilos?**

Sí, las llamadas de [configuración de licencia](/slides/es/net/licensing/) no son seguras para subprocesos y requieren sincronización.
