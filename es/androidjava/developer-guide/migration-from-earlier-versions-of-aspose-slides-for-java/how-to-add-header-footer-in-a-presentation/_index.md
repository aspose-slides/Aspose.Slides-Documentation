## **Soporte para código heredado**
Para utilizar el código heredado desarrollado con versiones de Aspose.Slides para Java anteriores a la 13.x, necesita realizar algunos cambios menores en su código y funcionará como antes. Todas las clases que estaban presentes en el antiguo Aspose.Slides para Java bajo los espacios de nombres Aspose.Slide y Aspose.Slides.Pptx ahora están fusionadas en el único espacio de nombres Aspose.Slides. Por favor, eche un vistazo al siguiente fragmento de código simple para agregar encabezado y pie de página en una presentación en la antigua API de Aspose.Slides y siga los pasos que describen cómo migrar a la nueva API fusionada.
