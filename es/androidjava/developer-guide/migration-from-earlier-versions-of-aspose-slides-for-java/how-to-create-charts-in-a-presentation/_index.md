Se ha lanzado una nueva [API de Aspose.Slides para Java](https://docs.aspose.com/slides/androidjava/) y ahora este único producto admite la capacidad de generar documentos de PowerPoint desde cero y editar los existentes.

{{% /alert %}} 
## **Soporte para código antiguo**
Para utilizar el código antiguo desarrollado con las versiones de Aspose.Slides para Java anteriores a 14.x.x, necesita hacer algunos cambios menores en su código y el código funcionará como antes. Todas las clases que estaban presentes en las antiguas versiones de Aspose.Slides para Java bajo los espacios de nombres com.aspose.slides y com.aspose.slides.pptx ahora están fusionadas en un único espacio de nombres com.aspose.slides.
