## **Preguntas frecuentes**

**¿Puedo guardar presentaciones en PDF que cumplan con los estándares de archivo y accesibilidad (PDF/A y PDF/UA)?**

Sí. Aspose.Slides admite la exportación a PDF con niveles de cumplimiento como PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b, así como PDF/UA mediante la configuración [compliance](https://reference.aspose.com/slides/cpp/aspose.slides.export/pdfoptions/set_compliance/) en [PDF export options](https://reference.aspose.com/slides/cpp/aspose.slides.export/pdfoptions/).

**¿La biblioteca admite la incrustación de fuentes al exportar a PDF, con control detallado sobre lo que se incrusta?**

Sí. Puede controlar si las fuentes se incrustan completamente o se subestablecen (solo los glifos usados), especificar cómo se tratan las fuentes del sistema comunes, y configurar el comportamiento para texto ASCII mediante [PDF export options](https://reference.aspose.com/slides/cpp/aspose.slides.export/pdfoptions/).

**¿Puedo detectar si un archivo está protegido con contraseña antes de cargarlo realmente?**

Sí. Utilizando la [factory-based inspection API](https://reference.aspose.com/slides/cpp/aspose.slides/presentationfactory/), puede consultar un archivo de presentación para determinar si está protegido con contraseña sin abrirlo completamente.

**¿Existen mecanismos de sustitución de fuentes y soporte para fuentes personalizadas?**

Sí. La biblioteca admite [loading](/slides/es/cpp/custom-font/) y [embedding](/slides/es/cpp/embedded-font/) fuentes personalizadas y proporciona reglas de [fallback](/slides/es/cpp/fallback-font/) para evitar glifos faltantes durante la renderización y conversión.

**¿Puedo exportar diapositivas a XPS, y existen opciones para ajustar la salida XPS?**

Sí. [Export to XPS](/slides/es/cpp/convert-powerpoint-to-xps/) está soportado, y puede ajustar las [save options](https://reference.aspose.com/slides/cpp/aspose.slides.export/xpsoptions/) relevantes para controlar la calidad y el contenido de la salida del documento XPS.
