## **Establecer propiedades RotationX, RotationY y DepthPercents del Gráfico 3D**
Aspose.Slides para C++ proporciona una API simple para establecer estas propiedades. Este artículo siguiente te ayudará a configurar diferentes propiedades como Rotación X, Y, **DepthPercents**, etc. El código de muestra aplica la configuración de las propiedades mencionadas anteriormente.

1. Crear una instancia de la clase [Presentation](https://reference.aspose.com/slides/net/aspose.slides/presentation).
1. Acceder a la primera diapositiva.
1. Agregar un gráfico con datos predeterminados.
1. Establecer las propiedades de Rotación 3D.
1. Escribir la presentación modificada en un archivo PPTX.
