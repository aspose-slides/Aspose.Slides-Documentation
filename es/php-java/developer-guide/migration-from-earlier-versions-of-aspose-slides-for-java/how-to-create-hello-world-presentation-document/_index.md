## **Soporte para código heredado**
Para utilizar el código heredado desarrollado con Aspose.Slides para PHP vía Java en versiones anteriores a 13.x, necesitas hacer algunos cambios menores en tu código y funcionará como antes. Todas las clases que estaban presentes en el antiguo Aspose.Slides para PHP vía Java bajo los espacios de nombres Aspose.Slide y Aspose.Slides.Pptx ahora se han fusionado en un único espacio de nombres Aspose.Slides.
