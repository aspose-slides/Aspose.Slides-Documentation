## **Compatibilidad con código legado**
Para utilizar el código legado desarrollado con Aspose.Slides para PHP vía Java versiones anteriores a 14.x.x, necesita realizar algunos cambios menores en su código y este funcionará como antes. Todas las clases que estaban presentes en el viejo Aspose.Slides para PHP vía Java bajo los espacios de nombres com.aspose.slides y com.aspose.slides.pptx ahora se han fusionado en un solo espacio de nombres com.aspose.slides.
