## **Soporte para código heredado**
Para usar el código heredado desarrollado con Aspose.Slides para PHP a través de versiones de Java anteriores a 13.x, necesitas hacer algunos cambios menores en tu código y este funcionará como antes. Todas las clases que estaban presentes en la antigua Aspose.Slides para PHP a través de Java bajo los espacios de nombres Aspose.Slide y Aspose.Slides.Pptx ahora se han fusionado en un único espacio de nombres Aspose.Slides. Por favor, echa un vistazo al siguiente simple fragmento de código para agregar encabezado y pie de página en la presentación en la API heredada de Aspose.Slides y sigue los pasos que describen cómo migrar a la nueva API fusionada.
