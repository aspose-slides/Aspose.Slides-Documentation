{{% alert color="primary" %}} 

El [Portable Document Format](https://en.wikipedia.org/wiki/PDF) es un formato de archivo creado por Adobe Systems para el intercambio de documentos entre organizaciones. El objetivo del formato era mantener el contenido y el diseño idénticos, independientemente de la plataforma en la que se visualizara. Aspose.Slides for PHP via Java le permite convertir archivos de presentación a PDF.

{{% /alert %}} 

## **PDF en Aspose.Slides for PHP via Java**
Cualquier presentación que pueda cargarse en Aspose.Slides for PHP via Java puede convertirse en un PDF que cumpla con [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) o [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA) según su elección. Aspose.Slides for PHP via Java exporta presentaciones a PDF y, en la mayoría de los casos, el PDF de salida se ve exactamente como la presentación original.

Aspose.Slides admite las siguientes características de presentación al convertir a PDF:

- Imágenes, cuadros de texto y otras formas.
- Texto y formato.
- Párrafos y formato.
- Hipervínculos.
- Cabeceras y pies de página.
- Viñetas.
- Tablas.

Puede exportar presentaciones a PDF directamente con Aspose.Slides for PHP via Java: no necesita ningún otro componente. Además, puede personalizar la exportación de presentación a PDF con varias opciones, como se explica en [Converting to PDF](/slides/es/php-java/convert-powerpoint-to-pdf/).
