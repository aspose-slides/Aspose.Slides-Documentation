## **Acerca de TIFF**

El [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) que se conoce por almacenar varias imágenes en un documento fue creado originalmente por Aldus. Este formato es ampliamente compatible con aplicaciones de escaneo, fax y otras de manipulación de imágenes.

## **TIFF en Aspose.Slides para PHP mediante Java**

Cualquier documento que pueda cargarse en Aspose.Slides para Java también puede convertirse directamente en un documento TIFF mediante Aspose.Slides para PHP mediante Java, eliminando la necesidad de cualquier componente de terceros. Además, puede definir opcionalmente el tamaño de las imágenes en el documento TIFF resultante. Puede encontrar información sobre la exportación de documentos de presentación a documentos TIFF a través de Aspose.Slides para PHP mediante Java en [este tema](/slides/es/php-java/convert-powerpoint-to-tiff/).
