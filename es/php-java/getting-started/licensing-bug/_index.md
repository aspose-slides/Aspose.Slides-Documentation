Este artículo describe un error de licenciamiento encontrado en Aspose.Slides for PHP via Java versiones 23.2, 23.3, 23.4, 23.5, 23.6 y 23.7. Este error provocó que las licencias perpetuas quedaran temporalmente inutilizables después de que su suscripción expirara.

## **Síntomas**

Después de que la suscripción de su licencia perpetua expire, Aspose.Slides for PHP via Java versiones 23.2 - 23.7 puede informar que la licencia está caducada, pasando todas las funciones a un modo de evaluación.  
Este comportamiento es incorrecto y no afecta a versiones anteriores a la 23.2 ni posteriores a la 23.7.

## **Resolución**

Aspose.Slides for PHP via Java abordó este problema y lanzó versiones actualizadas (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) con la corrección.  

Si está utilizando una de las versiones afectadas de Aspose.Slides for PHP via Java en su proyecto, cambie a la versión corregida.

## **Productos y versiones afectados**

Tenga en cuenta que este error solo afecta a Aspose.Slides for PHP via Java versiones 23.2 a 23.7. **Otros productos de Aspose y otras versiones de Aspose.Slides no se ven afectados**.
