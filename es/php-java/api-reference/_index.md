{{% alert color="primary" %}} 

Aspose.Slides for PHP via Java es una biblioteca de clases que permite a las aplicaciones PHP leer y escribir documentos de presentaciones sin utilizar Microsoft PowerPoint® ni otros. Aspose.Slides for PHP via Java permite gestionar texto, formas, tablas y animaciones, añadir audio y vídeo a las diapositivas, previsualizar diapositivas, exportar diapositivas a formato SVG, PDF y más.

La última referencia de API se puede encontrar en [Referencia de API de Aspose.Slides para PHP vía Java](https://reference.aspose.com/slides/es/php-java/).
