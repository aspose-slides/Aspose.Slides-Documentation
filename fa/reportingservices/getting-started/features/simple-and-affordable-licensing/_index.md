## انواع مجوزها
{{% alert color="primary" %}} دو نوع مجوز در حال حاضر ارائه می‌شود: Server Subscription Site Subscription {{% /alert %}}
هر مجوز شامل یک سال ارتقاء رایگان به هر نسخه جدید محصول و یک سال پشتیبانی فنی نامحدود رایگان است.

نسخه ارزیابی و نسخه‌های پرداختی Aspose.Slides for Reporting Services یک فایل دانلود مشترک است. به سادگی Aspose.Slides for Reporting Services را دانلود کنید و بر روی سرور نصب کنید؛ در حالت ارزیابی کار خواهد کرد.

از ارزیابی Aspose.Slides for Reporting Services استفاده کنید زیرا محدودیت زمانی ندارد و پشتیبانی فنی رایگان برای کاربران ارزیابی ارائه می‌شود.

حالت ارزیابی نشانه‌های آبکی را به گزارش‌های خروجی اضافه می‌کند. هنگامی که یک [مجوز خریداری می‌کنید](https://purchase.aspose.com/buy)، کافی است مجوز را در پوشه مناسب کپی کنید و Aspose.Slides for Reporting Services به حالت دارای مجوز تبدیل خواهد شد.
## **اشتراک سرور**
اشتراک سرور ویژگی‌های زیر را ارائه می‌دهد:

- برای هر سرور فیزیکی یا مجرد یک مجوز لازم است.
- تعداد نامحدود پردازنده یا هسته.
- تعداد نامحدود نمونه‌های Microsoft SQL Server.
- تعداد نامحدود توسعه‌دهندگان.
- نسخه‌های جدید و اصلاحات رایگان به‌مدت ۱ سال.
- پشتیبانی فنی رایگان به‌مدت ۱ سال.
- تخفیف برای خریدهای متعدد.
