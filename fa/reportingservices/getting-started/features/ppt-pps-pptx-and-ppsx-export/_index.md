{{% alert color="primary" %}} 

سرویس گزارش‌دهی Microsoft SQL Server قابلیت‌های داخلی برای خروجی گرفتن گزارش‌ها به صورت ارائه‌های Microsoft PowerPoint ندارد، اما پس از نصب Aspose.Slides for Reporting Services بر روی سرور خود، به دو قالب خروجی اضافی دسترسی خواهید یافت:

- Microsoft PowerPoint Presentation (PPT)
- Microsoft PowerPoint SlideShow (PPS)
- Microsoft PowerPoint 2007 Presentation (PPTX)
- Microsoft PowerPoint 2007 SlideShow (PPSX)

{{% /alert %}} 

برای ایجاد اسناد در این قالب‌ها، Aspose.Slides for Reporting Services به نسخه‌ای داخلی از [Aspose.Slides for .NET](http://www.aspose.com/Products/Aspose.Slides/) متکی است که پیشرو در بازار به‌عنوان مؤلفه پردازش ارائه‌ها از Aspose محسوب می‌شود. برای تولید این اسناد از Microsoft PowerPoint استفاده نمی‌شود. 

نسخه‌های PPT و PPS یک گزارش واحد، به دلیل پشتیبانی گسترده‌ای که Aspose.Slides از فرمت‌های فایل پایه ارائه می‌دهد، به‌نظر می‌رسد که کاملاً یکسان هستند.
