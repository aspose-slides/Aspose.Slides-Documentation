{{% alert color="primary" %}} 

Aspose.Slides برای Reporting Services به طور کامل تمام مشخصات RDL را پشتیبانی می‌کند. این دو مزیت بزرگ دارد: 

- نیازی به طراحی مجدد گزارش‌های موجود نیست. می‌توانید هر گزارش RDL موجود را به عنوان یک ارائه Microsoft PowerPoint استخراج کنید و دقیقاً مطابق طراحی RDL نمایش داده خواهد شد.
- نیازی به استفاده از یک طراح گزارش خاص نیست. می‌توانید از هر طراح گزارش RDL استفاده کنید و گزارش دقیقاً به همان شکلی که طراحی کرده‌اید استخراج خواهد شد.

{{% /alert %}} 

Aspose.Slides برای Reporting Services از عناصر زیر RDL پشتیبانی می‌کند: 

- صفحه، سرصفحه‌ها، پانوشت‌ها
- جعبه‌های متن
- تصاویر
- گزارش‌های زیرمجموعه
- نمودارها
- فهرست‌ها
- جدول‌ها
- ماتریس‌ها
- استایل‌ها
- مستطیل‌ها

**یک مثال از گزارش شامل سرصفحه‌ها، پانوشت‌ها، تصاویر، گزارش‌های زیرمجموعه، جدول‌ها، جعبه‌های متن و مستطیل‌ها که به عنوان یک ارائه Microsoft PowerPoint (PPT) استخراج شده است.**
