## **خوش آمدید به Aspose.Slides for Reporting Services!**

Aspose.Slides for Reporting Services تنها راه‌حل موجود در بازار است که امکان تولید گزارش‌های واقعی PPT و PPS را در Microsoft SQL Server 2005، 2008، 2012، 2016 و 2017 Reporting Services (32‑bit و 64‑bit) فراهم می‌کند. تمام ویژگی‌های گزارش RDL، از جمله جداول، ماتریس‌ها، نمودارها و تصاویر با بالاترین دقت به ارائه‌های Microsoft PowerPoint تبدیل می‌شوند.

## **نمای کلی محصول**

Microsoft SQL Server Reporting Services قابلیت‌های داخلی برای استخراج گزارش‌ها به صورت ارائه‌های Microsoft PowerPoint ندارد، اما پس از نصب Aspose.Slides for Reporting Services بر روی سرور خود، به قالب‌های خروجی اضافی دسترسی خواهید یافت:

- PPT - ارائه PowerPoint از طریق Aspose.Slides
- PPS - اسلایدشو PowerPoint از طریق Aspose.Slides
- PPTX - ارائه PowerPoint 2007 از طریق Aspose.Slides
- PPSX - اسلایدشو PowerPoint 2007 از طریق Aspose.Slides

Aspose.Slides for Reporting Services ارائه‌ها را در سرور ایجاد می‌کند بدون استفاده از Microsoft PowerPoint. Aspose.Slides for Reporting Services به‌صورت داخلی از Aspose.Slides برای .NET استفاده می‌کند – مؤلفه‌ای سطح‌یک جهانی برای پردازش ارائه‌های سمت سرور.

**Aspose.Slides for Reporting Services امکان استخراج هر گزارشی را در قالب‌های PPT، PPS، PPTX یا PPSX فراهم می‌کند.**
