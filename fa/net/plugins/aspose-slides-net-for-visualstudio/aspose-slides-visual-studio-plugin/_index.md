## **معرفی**
Microsoft Visual Studio یک محیط توسعه یکپارچه (IDE) از Microsoft است. از آن برای توسعه برنامه‌های کامپیوتری برای Microsoft Windows، همچنین وب‌سایت‌ها، برنامه‌های وب و سرویس‌های وب استفاده می‌شود. Visual Studio از پلتفرم‌های توسعه نرم‌افزاری Microsoft مانند Windows API، Windows Forms، Windows Presentation Foundation، Windows Store و Microsoft Silverlight استفاده می‌کند. این ابزار می‌تواند هر دو نوع کد بومی و کد مدیریت‌شده را تولید کند.

### **Aspose.Slides برای .NET**
Aspose.Slides برای .NET یک API منحصر به فرد برای مدیریت PowerPoint® است که به برنامه‌های .NET امکان می‌دهد مستندات PowerPoint را بدون استفاده از Microsoft PowerPoint بخوانند، بنویسند و دستکاری کنند.

Aspose.Slides برای .NET اولین و تنها API PowerPoint برای .NET است که قابلیت مدیریت مستندات PowerPoint را در برنامه‌های خود شما فراهم می‌کند. همانند همهٔ اجزای Aspose .NET، Aspose.Slides به زبان C# مدیریت‌شده نوشته شده، قیمت‌گذاری فوق‌العاده‌ای دارد و به‌سرعت نور می‌دوشد.

برای جزئیات بیشتر درباره [اطلاعات بیشتر در مورد Aspose.Slides برای .NET API](http://www.aspose.com/.net/powerpoint-component.aspx)

### **کاوش مثال‌های Aspose.Slides برای .NET با استفاده از Visual Studio**
Aspose Visual Studio Plugin - **Explore Aspose.Slides for .NET Examples** ابزاری عالی برای دانلود و کاوش سریع مثال‌های API Aspose.Slides برای .NET است. این ابزار با ارائه گزینه‌ای بسیار ساده برای انتخاب، دانلود و باز کردن آخرین پروژه‌های مثال بدون نیاز به ترک Visual Studio، زمان و تلاش زیادی را ذخیره می‌کند.

این نسخه ویژگی‌های زیر را ارائه می‌دهد:

- از Visual Studio 2010 و Visual Studio 2012 و Visual Studio 2013 پشتیبانی می‌کند.
- به‌راحتی از منوی **File > Aspose** در Visual Studio اجرا می‌شود.
- زمان را صرفه‌جویی کرده و منحنی یادگیری را کاهش می‌دهد.
- امکان انتخاب و باز کردن آخرین مثال‌های API Aspose.Slides برای .NET میزبانی‌شده در Github.com را می‌دهد.
- امکان باز کردن پروژه‌های مثال **C#** یا **VB.NET** را فراهم می‌کند.
- یک پروژهٔ انتخاب‌شده به‌طور خودکار در Visual Studio باز می‌شود و تمام مراجع مورد نیاز تنظیم می‌شوند تا محیطی آماده برای اجرا و کاوش در اختیار شما قرار گیرد.
- کتابخانه‌های جدید API را به‌صورت خودکار از **NuGet** و مثال‌ها را از **GitHub** دانلود می‌کند.

## **نیازمندی‌های سیستم و پلتفرم‌های پشتیبانی‌شده**
برای نصب و استفاده از Aspose Visual Studio Plugin باید یکی از نسخه‌های زیر Visual Studio را نصب کرده باشید

- Visual Studio 2010, 2012, 2013

اگر در نصب یا استفاده از این افزونه مشکلی داشتید، لطفاً با ما تماس بگیرید.

## **نصب**
نصب Aspose Visual Studio Plugin بسیار ساده و آسان است؛ لطفاً مراحل زیر را دنبال کنید

- روی فایل msi دانلودشده دو بار کلیک کنید
- مراحل را دنبال کنید؛ می‌توانید به‌سادگی در هر صفحه گزینه Next را فشار دهید تا با تنظیمات استاندارد نصب انجام شود.

پس از اتمام نصب یک دیالوگ موفقیت‌آمیز دریافت خواهید کرد.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_2)

**Note:** لطفاً برای اعمال تغییرات Visual Studio را دوباره راه‌اندازی کنید.

## **استفاده**
پس از نصب می‌توانید به‌راحتی افزونه تعاملی را همان‌طور که در زیر توضیح داده شده است، اجرا و استفاده کنید:
**1.** افزونه را با انتخاب **File > Aspose** و سپس **New Aspose.Slides Example Project** در Visual Studio باز کنید.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_3)

**2.** اولین باری که افزونه را باز می‌کنید، دیالوگ انتخاب API همان‌طور که در تصویر زیر نشان داده شده است، نمایش داده می‌شود. چند دقیقه طول می‌کشد تا آخرین API و کتابخانه‌های آن از **NuGet** و کد منبع آخرین مثال‌ها از **GitHub** دانلود شوند.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_4)

**3.** در دیالوگ **Select an API to View Examples** مکان ایجاد پروژه‌های مثال را انتخاب یا تغییر دهید و همچنین نوع پروژه مثال را به **C#** یا **VB.NET** تغییر کنید و بر روی **Continue** کلیک کنید تا پروژهٔ مثال در Visual Studio باز شود.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_1)

**4.** پروژه‌های مثال جدید بر اساس انتخاب نوع پروژه **C#** یا **VB.NET** در Visual Studio ایجاد می‌شوند.

## **پشتیبانی، گسترش و مشارکت**
### **پشتیبانی**
از همان روزهای اول Aspose می‌دانستیم که ارائهٔ محصولات خوب کافی نیست؛ همچنین باید خدمات عالی ارائه دهیم. ما خودمان توسعه‌دهنده‌ایم و می‌دانیم که چقدر ناامیدکننده است وقتی یک مشکل فنی یا نقص نرم‌افزاری مانع انجام کاری می‌شود. ما اینجا هستیم تا مشکلات را حل کنیم، نه ایجادشان.

به همین دلیل پشتیبانی رایگان ارائه می‌دهیم. هر کسی که از محصول ما استفاده می‌کند، چه خریداری کرده باشد و چه در حالت ارزیابی باشد، شایستهٔ تمام توجه و احترام ما است.

می‌توانید هر گونه مشکل یا پیشنهادی مرتبط با Aspose Visual Studio Plugin را با استفاده از یکی از پلتفرم‌های زیر گزارش کنید

- [Github](https://github.com/asposeSlides/Aspose.Slides-for-.NET/issues)

### **گسترش و مشارکت**
**Explore Aspose.Slides برای .NET Examples** برای Visual Studio منبع باز است و کد منبع آن در وب‌سایت‌های مهم کدنویسی اجتماعی که در زیر فهرست شده‌اند، در دسترس است. توسعه‌دهندگان تشویق می‌شوند کد منبع را دانلود کرده و عملکرد آن را مطابق نیازهای خود گسترش دهند.

#### **کد منبع**
می‌توانید تازه‌ترین کد منبع را از یکی از مکان‌های زیر دریافت کنید

- [Github](https://github.com/aspose-slides/Aspose.Slides-for-.NET)

#### **چگونگی پیکربندی کد منبع**
برای باز کردن و گسترش کد منبع باید موارد زیر نصب شده باشد

- Visual Studio 2010

لطفاً این مراحل ساده را برای شروع دنبال کنید

1. کد منبع را دانلود/کلون کنید.
2. Visual Studio 2010 را باز کنید و **File** > **Open Project** را انتخاب کنید.
3. به مسیر کد منبعی که دانلود کرده‌اید بروید و **AsposeVisualStudioPlugin.sln** را باز کنید.

#### **مرور کلی کد منبع**
در این حل‌مسئله دو پروژه وجود دارد

- **AsposeVisualStudioPlugin** - شامل یکپارچه‌سازی با Visual Studio، انتخاب و دانلود خودکار APIهای Aspose و مثال‌ها.
- **Installer** - پروژهٔ نصب‌کننده Wix برای تولید فایل msi نصب افزونه.
