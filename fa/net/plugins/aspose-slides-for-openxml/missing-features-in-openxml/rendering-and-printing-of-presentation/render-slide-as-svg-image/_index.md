SVG—یک مخفف برای Scalable Vector Graphics—یک نوع یا قالب گرافیکی استاندارد است که برای رندر تصاویر دو بعدی استفاده می‌شود. SVG تصاویر را به صورت بردارها در XML ذخیره می‌کند و جزئیاتی که رفتار یا ظاهر آن‌ها را تعریف می‌کند، شامل می‌شود.

SVG یکی از تعداد کمی از قالب‌های تصویری است که در این موارد استانداردهای بسیار بالایی را برآورده می‌کند: مقیاس‌پذیری، تعاملی بودن، کارایی، دسترس‌پذیری، قابلیت برنامه‌نویسی و دیگر موارد. به همین دلایل، به طور گسترده‌ای در توسعه وب به کار می‌رود.

ممکن است بخواهید در این سناریوها از فایل‌های SVG استفاده کنید:

- زمانی که قصد دارید ارائه خود را در قالب بسیار بزرگی چاپ کنید. تصاویر SVG می‌توانند به هر وضوح یا سطحی مقیاس‌پذیر شوند. می‌توانید تصاویر SVG را هر تعداد که لازم است بدون از دست دادن کیفیت، تغییر اندازه دهید.
- زمانی که می‌خواهید نمودارها و گراف‌ها را از اسلایدهای خود در رسانه‌ها یا پلتفرم‌های مختلف استفاده کنید. اکثر مرورگرها می‌توانند فایل‌های SVG را تفسیر کنند.
- زمانی که نیاز به کوچک‌ترین حجم ممکن برای تصاویر دارید. فایل‌های SVG عموماً از معادل‌های با وضوح بالا در قالب‌های دیگر، به‌ویژه قالب‌های مبتنی بر بیت‌مپ (JPEG یا PNG) کوچک‌تر هستند.

Aspose.Slides for .NET به شما امکان می‌دهد اسلایدهای ارائه‌تان را به صورت تصاویر **SVG** صادر کنید. برای تولید یک تصویر SVG از هر اسلاید، این کار را انجام دهید:

- یک نمونه از کلاس Presentation ایجاد کنید.
- بر تمام اسلایدهای موجود در ارائه iteration کنید.
- هر اسلاید را از طریق FileStream به فایل SVG خود بنویسید.

{{% /alert %}} 

این نمونه کد به زبان C# نشان می‌دهد چگونه PPT را به SVG تبدیل کنید با استفاده از Aspose.Slides:

``` csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    for (var index = 0; index < pres.Slides.Count; index++)
    {
        ISlide slide = pres.Slides[index];

        using (FileStream fileStream = new FileStream($"slide-{index}.svg", FileMode.Create, FileAccess.Write))
        {
            slide.WriteAsSvg(fileStream);   
        }
    }
}
```
