## **OpenXML چیست؟**
OpenXML SDK یک SDK .NET از مایکروسافت است که برای پردازش قالب‌های فایل‌های Microsoft Office 2007 و بالاتر استفاده می‌شود. این قالب‌های فایل در واقع بسته‌های فشرده‌ (ZIP) هستند و Open XML SDK برای Office کار را برای دستکاری بسته‌های Open XML و عناصر طرح‌واره (schema) زیرین درون یک بسته ساده می‌کند. کلاس‌های موجود در Open XML SDK بسیاری از کارهای رایج را که توسعه‌دهندگان روی بسته‌های Open XML انجام می‌دهند، در بر می‌گیرد، به‌طوری که بتوانید عملیات پیچیده را با چند خط کد انجام دهید.

بعد از نصب Open XML SDK 2.5، پروژه یا برنامه موجود خود را در Visual Studio باز کنید یا یک پروژه یا برنامه جدید ایجاد کنید. سپس در پروژه یا برنامه‌تان، مراجع (references) به کامپوننت‌های زیر را اضافه کنید:

- DocumentFormat.OpenXml
- WindowsBase
## **Aspose.Slides for .NET**
Aspose.Slides for .NET یک کامپوننت مدیریت PowerPoint® منحصربه‌فرد است که به برنامه‌های .NET امکان خواندن، نوشتن و دستکاری اسناد PowerPoint را بدون استفاده از Microsoft PowerPoint می‌دهد.

Aspose.Slides for .NET اولین و تنها کامپوننت .NET است که قابلیت مدیریت اسناد PowerPoint را در برنامه‌های خود فراهم می‌کند. همانند تمام کامپوننت‌های Aspose .NET، Aspose.Slides به زبان C# مدیریت‌شده نوشته شده، قیمت‌گذاری فوق‌العاده مناسب دارد و سرعت فوق‌العاده‌ای دارد.

## **معرفی**
توسعه‌دهندگانی که به دنبال کتابخانه‌های قالب فایل هستند، ممکن است نیاز داشته باشند ویژگی‌های ارائه‌شده توسط کتابخانه‌های موجود در بازار را بررسی کنند. عموماً دو مؤلفه هنگام انتخاب یک گزینه در نظر گرفته می‌شود:

- ویژگی‌ها
- تلاش مورد نیاز برای استفاده از کتابخانه

اگر یک توسعه‌دهنده .NET هستید و به دنبال کتابخانه‌ای سریع، آسان و سبک برای پردازش فایل‌ها می‌گردید، ممکن است نیاز داشته باشید تا VSTO (Visual Studio Tools for Office) را با Aspose مقایسه کنید. در این حالت، این پروژه به شما کمک می‌کند تا ویژگی‌ها و کدهای هر دو کتابخانه را مقایسه کنید.

- Microsoft Office Professional 2003 SP1

- فقط .NET Framework 2.0 یا بالاتر.
