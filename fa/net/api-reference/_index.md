{{% alert color="primary" %}} 

Aspose.Slides for .NET یک کتابخانهٔ کلاسی است که به برنامه‌های .NET امکان خواندن و نوشتن اسناد ارائه را بدون استفاده از Microsoft PowerPoint® یا سایر ابزارها می‌دهد. Aspose.Slides for .NET امکان مدیریت متن، اشکال، جدول‌ها و انیمیشن‌ها، افزودن صدا و ویدیو به اسلایدها، پیش‌نمایش اسلایدها، خروجی اسلایدها به فرمت SVG، PDF و موارد دیگر را فراهم می‌کند.

جدیدترین مرجع API را می‌توانید در [Aspose.Slides for .NET API Reference](https://reference.aspose.com/slides/fa/net) پیدا کنید.
