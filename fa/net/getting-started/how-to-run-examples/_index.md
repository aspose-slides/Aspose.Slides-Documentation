## **الزامات نرم‌افزاری**
قبل از اینکه نمونه‌ها را دانلود و اجرا کنید، لطفاً بررسی کنید و تأیید کنید که تنظیمات شما این الزامات را برآورده می‌کند: 

- Visual Studio 2010 یا بالاتر.
- NuGet Package Manager نصب شده در Visual Studio. اطمینان حاصل کنید که آخرین نسخه API NuGet در Visual Studio نصب شده است. 

برای دستورالعمل‌های نصب NuGet package manager، به این صفحه مراجعه کنید: https://docs.microsoft.com/en-us/nuget/install-nuget-client-tools

1. به **Tools** > **Options** > **NuGet Package Manager** بروید.

1. **NuGet Package Manager** را گسترش دهید (با دوبار کلیک روی آن) و سپس **Package Sources** را انتخاب کنید. 

1. بررسی کنید و تأیید کنید که پارامتر nuget.org انتخاب شده است. 

   پروژه مثال از ویژگی NuGet Automatic Package Restore استفاده می‌کند، بنابراین نیاز به اتصال اینترنت فعال دارید. 

   اگر در ماشینی که قصد اجرای مثال‌ها را دارید، اتصال اینترنت فعال ندارید، لطفاً [Installation](https://docs.aspose.com/slides/fa/net/installation/) را بررسی کنید و به‌صورت دستی یک مرجع به Aspose.Slides.dll در پروژه مثال اضافه کنید.
## **دانلود Aspose.Slides از GitHub**
تمام مثال‌های Aspose.Slides برای .NET در [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-.NET) میزبانی می‌شوند.

می‌توانید مخزن را با استفاده از کلاینت مورد علاقهٔ GitHub خود کلون کنید یا فایل ZIP را [اینجا](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/master.zip) دانلود کنید.

1. اگر فایل ZIP را دانلود کنید، باید محتوای آن را در یک پوشه روی رایانه‌تان استخراج کنید. 

تمام مثال‌ها در پوشه **Examples** ذخیره شده‌اند.

یک فایل راه‌حل Visual Studio برای C# وجود دارد. پروژه‌ها در Visual Studio 2013 ایجاد شده‌اند، اما فایل‌های راه‌حل با Visual Studio 2010 SP1 و بالاتر سازگار هستند.

2. فایل راه‌حل را در Visual Studio باز کنید و پروژه را بسازید.

   در اولین اجرا، وابستگی‌ها به‌صورت خودکار از طریق NuGet دانلود می‌شوند.

پوشه **Data** در ریشهٔ پوشه **Examples** شامل فایل‌های ورودی استفاده‌شده در مثال‌های C# است. شما باید پوشه **Data** را همراه با پروژهٔ مثال‌ها دانلود کنید.

3. فایل RunExamples.cs را باز کنید. تمام مثال‌ها از اینجا فراخوانی می‌شوند.

4. مثال‌هایی که می‌خواهید اجرا کنید را در پروژه از حالت کامنت خارج کنید.

در صورت داشتن هرگونه مشکل در تنظیمات یا اجرای مثال‌ها، لطفاً با استفاده از انجمن‌های ما تماس بگیرید.
## **مشارکت**
می‌توانید با افزودن یا بهبود یک مثال به پروژه مشارکت کنید. تمام مثال‌ها و پروژه‌های نمایش در مخزن متن باز هستند، بنابراین شما (و دیگران) می‌توانید آن‌ها را به‌صورت آزاد در برنامه‌ها استفاده کنید.

برای مشارکت، می‌توانید مخزن را فورک کنید، کد منبع را ویرایش کنید و یک pull request ایجاد کنید. ما تغییرات را بررسی خواهیم کرد. اگر مفید باشند، آن‌ها را به مخزن اضافه می‌کنیم.
