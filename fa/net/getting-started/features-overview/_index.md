## **پلتفرم‌های پشتیبانی‌شده**
Aspose.Slides for .NET از محبوب‌ترین پلتفرم‌های توسعه و استقرار پشتیبانی می‌کند. جزئیات زیر را مرور کنید:

|**قابلیت**|**توضیح**|
| :- | :- |
|ASP.NET Web Applications|از Aspose.Slides for .NET برای ساخت برنامه‌های وب ASP.NET هدف‌گیری نسخه‌های .NET Framework 2.0 تا 4.6.2 استفاده کنید|
|Web Services|از Aspose.Slides for .NET برای استقرار وب سرویس‌ها استفاده کنید|
|WinForms Applications|Aspose.Slides for .NET می‌تواند برای توسعه برنامه‌های Windows Forms نیز به کار رود|

## **فرمت‌های فایل و تبدیل‌ها**
Aspose.Slides for .NET از اکثر فرمت‌های سند PowerPoint پشتیبانی می‌کند. همچنین امکان صادر کردن آن‌ها به فرمت‌های محبوبی که سازمان‌ها به‌طور گسترده استفاده و تبادل می‌کنند را فراهم می‌آورد. جزئیات زیر را مرور کنید:

|**قابلیت**|**توضیح**|
| :- | :- |
|[مایکروسافت پاورپوینت (PPT)](/slides/fa/net/ppt-vs-pptx/)|Aspose.Slides for .NET سریع‌ترین پردازش را برای این فرمت سند ارائه می‌دهد.|
|[تبدیل PPT به PPTX](/slides/fa/net/convert-ppt-to-pptx/)|Aspose.Slides for .NET از تبدیل PPT به PPTX پشتیبانی می‌کند.|
|[قالب سند قابل حمل (PDF)](/slides/fa/net/convert-powerpoint-ppt-and-pptx-to-pdf/)|می‌توانید تمام فرمت‌های فایل پشتیبانی‌شده را با یک متد به اسناد Adobe Portable Document Format (PDF) صادر کنید.|
|[مشخصات تجزیه‌گر XML (XPS)](https://docs.aspose.com/slides/fa/net/convert-powerpoint-to-xps/)|می‌توانید تمام فرمت‌های فایل پشتیبانی‌شده را با یک متد به اسناد XML Parser Specification (XPS) صادر کنید.|
|[فرمت فایل تصویری تگ‌شده (TIFF)](/slides/fa/net/convert-powerpoint-to-tiff/)|می‌توانید تمام فرمت‌های فایل ارائه پشتیبانی‌شده را به Tagged Image File Format (TIFF) صادر کنید.|
|[تبدیل PPTX به HTML](/slides/fa/net/convert-powerpoint-ppt-and-pptx-to-html/)|Aspose.Slides for .NET از تبدیل PresentationEx به فرمت HTML پشتیبانی می‌کند.|

## **رندرینگ و چاپ**
Aspose.Slides for .NET رندرینگ با دقت بالا از اسلایدهای اسناد ارائه به فرمت‌های گرافیکی مختلف را پشتیبانی می‌کند. جزئیات زیر را مرور کنید:

|**قابلیت**|**توضیح**|
| :- | :- |
|.NET Supported Image Formats|با Aspose.Slides for .NET می‌توانید اسلایدهای ارائه و تصاویر روی اسلایدها را به تمام فرمت‌های گرافیکی پشتیبانی‌شده توسط .NET مانند TIFF، PNG، BMP، JPEG، GIF و متافایل‌ها رندر کنید.|
|SVG Format|Aspose.Slides for .NET همچنین روش‌های داخلی برای صادر کردن اسلایدهای ارائه به فرمت Scalable Vector Graphics (SVG) فراهم می‌کند.|
|Presentation Printing|نسخه‌های جدید Aspose.Slides for .NET متدهای چاپ داخلی با گزینه‌های مختلف ارائه می‌دهند.|

## **ویژگی‌های محتوا**
Aspose.Slides for .NET به شما اجازه می‌دهد به تقریباً تمام آیتم‌ها یا محتوای اسناد ارائه دسترسی، اصلاح یا ایجاد کنید. جزئیات زیر را مرور کنید:

|**قابلیت**|**توضیح**|
| :- | :- |
|Master Slides|اسلایدهای مستر طرح‌بندی اسلایدهای عادی را تعریف می‌کنند. Aspose.Slides for .NET به شما اجازه می‌دهد به اسلایدهای مستر اسناد ارائه دسترسی و آن‌ها را اصلاح کنید.|
|Normal Slides|با Aspose.Slides for .NET می‌توانید اسلایدهای جدید با انواع مختلف ایجاد کنید؛ همچنین می‌توانید به اسلایدهای موجود در ارائه‌ها دسترسی و آن‌ها را اصلاح کنید.|
|Cloning / Copying Slides|متدهای داخلی Aspose.Slides for .NET به شما امکان می‌دهند اسلایدهای موجود را در یک ارائه کلون یا کپی کنید. می‌توانید اسلایدهای کپی‌شده و کلون‌شده را از یک ارائه به ارائه دیگر منتقل کنید. چون یک اسلاید طرح‌بندی خود را از اسلاید مستر به ارث می‌برد، متدهای کلونینگ داخلی به‌صورت خودکار مستر را هنگام کلون کردن کپی می‌کنند.|
|Managing Slides sections|متدهایی برای سازماندهی اسلایدها در بخش‌های مختلف داخل یک ارائه.|
|Place Holders and Text Holders|می‌توانید به جای‌نگهدارها و نگهدارهای متن در یک اسلاید دسترسی پیدا کنید. علاوه بر این، می‌توانید با استفاده از متد مناسب از ابتدا اسلایدی با نگهدارهای متن ایجاد کنید.|
|Header and Footers|Aspose.Slides for .NET مدیریت سرصفحه‌ها/پاصفحه‌ها در اسلایدها را تسهیل می‌کند.|
|Notes in Slides|با Aspose.Slides for .NET می‌توانید به یادداشت‌های مرتبط با یک اسلاید دسترسی و آن‌ها را اصلاح کنید و همچنین یادداشت‌های جدید اضافه کنید.|
|Finding a Shape|همچنین می‌توانید با استفاده از متن جایگزین مرتبط با شکل، شکل خاصی را از اسلاید پیدا کنید.|
|Backgrounds|Aspose.Slides for .NET به شما امکان کار با پس‌زمینه‌های مرتبط با یک اسلاید مستر یا عادی در یک ارائه را می‌دهد.|
|Text Boxes|قالب‌های متن می‌توانند از ابتدا ساخته شوند. می‌توانید به قالب‌های متن موجود دسترسی داشته باشید. همچنین می‌توانید متن‌های آن‌ها را بدون از دست دادن فرمت اصلی اصلاح کنید.|
|Rectangle Shapes|می‌توانید شکل‌های مستطیلی را با Aspose.Slides for .NET ایجاد یا اصلاح کنید.|
|Poly Line Shapes|می‌توانید شکل‌های خط چندضلعی را با Aspose.Slides for .NET ایجاد یا اصلاح کنید.|
|Ellipse Shapes|می‌توانید شکل‌های بیضی را با Aspose.Slides for .NET ایجاد یا اصلاح کنید.|
|Group Shapes|Aspose.Slides for .NET از شکل‌های گروهی پشتیبانی می‌کند|
|Auto Shapes|Aspose.Slides for .NET از شکل‌های خودکار پشتیبانی می‌کند|
|SmartArt|Aspose.Slides for .NET پشتیبانی از شکل‌های SmartArt در MS PowerPoint را فراهم می‌کند|
|Charts|Aspose.Slides for .NET پشتیبانی از نمودارهای MSO در PowerPoint را فراهم می‌کند|
|Shapes Serialization|Aspose.Slides for .NET از تعداد زیادی از شکل‌ها پشتیبانی می‌کند. وقتی Aspose.Slides for .NET از یک شکل پشتیبانی نمی‌کند، می‌توانید از یک متد سریال‌سازی استفاده کنید تا آن شکل را از یک اسلاید موجود سریال‌سازی کنید. به این ترتیب می‌توانید شکل را بر حسب نیازهای خود استفاده کنید.|
|Picture Frames|می‌توانید تصاویر را در فریم‌های تصویری با Aspose.Slides for .NET مدیریت کنید.|
|Audio Frames|می‌توانید فایل‌های صوتی را در فریم‌های صوتی بر روی اسلایدها با Aspose.Slides for .NET لینک یا جاسازی کنید.|
|Video Frames|می‌توانید فایل‌های ویدئویی را در فریم‌های ویدئویی مدیریت کنید. Aspose.Slides for .NET همچنین پشتیبانی از ویدئوهای لینک ‌شده و جاسازی‌شده را فراهم می‌کند|
|OLE Frame|می‌توانید اشیاء OLE را در فریم‌های OLE با Aspose.Slides for .NET مدیریت کنید.|
|Tables|Aspose.Slides for .NET از جدول‌ها در اسلایدها پشتیبانی می‌کند|
|ActiveX Controls|پشتیبانی از کنترل‌های ActiveX|
|VBA Macros|پشتیبانی از مدیریت ماکروهای VBA داخل ارائه‌ها.|
|Text Frame|می‌توانید متن را از هر شکل از طریق فریم متنی مرتبط با آن شکل دسترسی پیدا کنید|
|Text Scanning|می‌توانید متن را در یک ارائه در سطح ارائه یا اسلاید از طریق متدهای اسکن داخلی جستجو کنید.|
|Animations|می‌توانید انیمیشن‌ها را بر روی شکل‌ها اعمال کنید|
|Slide Shows|Aspose.Slides for .NET از نمایش اسلاید و انتقالات اسلاید پشتیبانی می‌کند|

## **سوالات متداول**

**آیا برای کارکرد کتابخانه نیاز به نصب Microsoft PowerPoint روی سرور/کامپیوتر دارم؟**

خیر. PowerPoint نیازی نیست؛ Aspose.Slides یک موتور مستقل برای ایجاد، ویرایش، تبدیل و رندرینگ ارائه‌ها است.

**چطور چندنخی کاری می‌کند؟ آیا پردازش می‌تواند موازی شود؟**

ایمن است که اسناد مختلف را در نخ‌های متفاوت پردازش کنید؛ شیء [presentation](https://reference.aspose.com/slides/fa/net/aspose.slides/presentation/) یکبار نباید توسط [multiple threads](/slides/fa/net/multithreading/) هم‌زمان استفاده شود.

**آیا پسوردهای فایل و رمزنگاری پشتیبانی می‌شود؟**

بله. می‌توانید [پرزنتیشن‌های رمزنگاری‌شده](/slides/fa/net/password-protected-presentation/) را باز کنید، پسورد باز و نوشتن را تنظیم یا حذف کنید و وضعیت حفاظت را بررسی کنید.

**آیا در کانتینرهای لینوکس باید به بسته‌های قلم توجه داشته باشم؟**

بله. توصیه می‌شود بسته‌های قلم عمومی را نصب کنید و/یا به‌طور صریح [دایرکتوری‌های قلم](/slides/fa/net/custom-font/) را در برنامه خود مشخص کنید تا از جایگزینی‌های غیرمنتظره جلوگیری شود.

**آیا در نسخه ارزیابی محدودیت‌هایی وجود دارد؟**

در [حالت ارزیابی](/slides/fa/net/licensing/)، یک واترمارک به خروجی اضافه می‌شود و محدودیت‌های خاصی اعمال می‌شود؛ یک [مجوز موقت 30 روزه](https://purchase.aspose.com/temporary-license/) برای تست کامل ویژگی‌ها موجود است.

**آیا وارد کردن فرمت‌های خارجی به یک ارائه (PDF/HTML → PPTX) پشتیبانی می‌شود؟**

بله. می‌توانید [صفحات PDF و محتوای HTML](/slides/fa/net/import-presentation/) را به یک ارائه اضافه کنید و آن‌ها را به اسلاید تبدیل کنید.
