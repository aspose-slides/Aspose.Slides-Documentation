## **نمای کلی**

Aspose.Slides از فایل‌های ارائه‌ای Microsoft PowerPoint 97 تا Office 365 را پشتیبانی می‌کند، از جمله Microsoft PowerPoint برای مک. این مقاله نسخه‌های PowerPoint پشتیبانی‌شده توسط کتابخانه را فهرست می‌کند و جدولی از فرمت‌های فایلی که می‌توان آنها را بارگیری، ذخیره یا هر دو انجام داد، ارائه می‌دهد.

این مقاله همچنین به سؤالات رایج درباره سازگاری PDF، جاسازی قلم‌ها، فایل‌های محافظت‌شده با رمز عبور، قلم‌های سفارشی، مکانیزم بازگشت قلم و گزینه‌های خروجی XPS پاسخ می‌دهد.

## **نسخه‌های پشتیبانی‌شده Microsoft PowerPoint**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint برای مک
- Office 365

## **سوالات متداول**

**آیا می‌توانم ارائه‌ها را به PDF ذخیره کنم که با استانداردهای بایگانی و دسترس‌پذیری (PDF/A و PDF/UA) مطابقت داشته باشد؟**

بله. Aspose.Slides از خروجی به PDF با سطوح سازگاری مانند PDF/A-2a، PDF/A-2b، PDF/A-2u، PDF/A-3a، PDF/A-3b و همچنین PDF/UA از طریق تنظیمات [compliance](https://reference.aspose.com/slides/fa/net/aspose.slides.export/pdfoptions/compliance/) در [PDF export options](https://reference.aspose.com/slides/fa/net/aspose.slides.export/pdfoptions/) پشتیبانی می‌کند.

**آیا کتابخانه از جاسازی قلم هنگام خروجی به PDF پشتیبانی می‌کند و امکان کنترل دقیق بر روی موارد جاسازی‌شده را فراهم می‌آورد؟**

بله. می‌توانید کنترل کنید که آیا قلم‌ها به‌صورت کامل یا به‌صورت زیرمجموعه (فقط گلیف‌های استفاده‌شده) جاسازی شوند، نحوه رفتار با قلم‌های رایج سیستم را مشخص کنید و رفتار متن ASCII را از طریق [PDF export options](https://reference.aspose.com/slides/fa/net/aspose.slides.export/pdfoptions/) پیکربندی کنید.

**آیا می‌توانم قبل از بارگذاری واقعی فایل، تشخیص دهم که آیا فایل رمز عبور دارد یا نه؟**

بله. با استفاده از [API بررسی مبتنی بر کارخانه](https://reference.aspose.com/slides/fa/net/aspose.slides/presentationfactory/)، می‌توانید بدون باز کردن کامل فایل، تعیین کنید که آیا فایل با رمز عبور محافظت شده است یا خیر.

**آیا مکانیزم‌های بازگشت قلم و پشتیبانی از قلم‌های سفارشی وجود دارد؟**

بله. کتابخانه از [بارگذاری](/slides/fa/net/custom-font/) و [جاسازی](/slides/fa/net/embedded-font/) قلم‌های سفارشی پشتیبانی می‌کند و قوانین [بازگشت قلم](/slides/fa/net/fallback-font/) را برای جلوگیری از عدم وجود گلیف‌ها در هنگام رندر و تبدیل فراهم می‌آورد.

**آیا می‌توانم اسلایدها را به XPS خروجی بگیرم و آیا گزینه‌هایی برای تنظیم خروجی XPS وجود دارد؟**

بله. خروجی به XPS (/slides/fa/net/convert-powerpoint-to-xps/) پشتیبانی می‌شود و می‌توانید گزینه‌های مربوط به [ذخیره]https://reference.aspose.com/slides/fa/net/aspose.slides.export/xpsoptions/) را تنظیم کنید تا کیفیت و محتوای سند XPS را کنترل نمایید.
