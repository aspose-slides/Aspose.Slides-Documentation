## **تغییرات API عمومی**
#### **CommonSlideViewProperties Class and ICommonSlideViewProperties Interface Have Been Added**
کلاس CommonSlideViewProperties و رابط ICommonSlideViewProperties اضافه شده‌اند.
#### **IAxis.LabelOffset Property Has Been Added**
خاصیت IAxis.LabelOffset اضافه شده است. این خاصیت فاصله برچسب‌ها از محور را مشخص می‌کند. برای محور دسته‌بندی یا تاریخ اعمال می‌شود.
#### **IChartTextBlockFormat.AutofitType Property Has Been Added**
خاصیت IChartTextBlockFormat.AutofitType اضافه شده است. تغییر این خاصیت می‌تواند تأثیر خاصی فقط بر روی این اجزای نمودار داشته باشد: DataLabel و DataLabelFormat (پشتیبانی کامل در PowerPoint 2013؛ در PowerPoint 2007 اثری برای رندر ندارند).
#### **IChartTextBlockFormat.WrapText Property Has Been Added**
خاصیت IChartTextBlockFormat.WrapText اضافه شده است. تغییر این خاصیت می‌تواند تأثیر خاصی فقط بر روی این اجزای نمودار داشته باشد: DataLabel و DataLabelFormat (پشتیبانی کامل در PowerPoint 2007/2013).
#### **Margin Properties Have Been Added to IChartTextBlockFormat**
ویژگی‌های Margin به IChartTextBlockFormat اضافه شده‌اند. تغییر این ویژگی‌ها می‌تواند تأثیر خاصی فقط بر روی این اجزای نمودار داشته باشد: DataLabel و DataLabelFormat (پشتیبانی کامل در PowerPoint 2013؛ در PowerPoint 2007 اثری برای رندر ندارند).
#### **ViewProperties.NotesViewProperties Property Has Been Added**
خاصیت ViewProperties.NotesViewProperties اضافه شده است. خاصیت Aspose.Slides.ViewProperties.NotesViewProperties اضافه شده است. این خاصیت ویژگی‌های نمای عمومی مرتبط با حالت نمای یادداشت‌ها را مشخص می‌کند.
#### **ViewProperties.SlideViewProperties Property Has Been Added**
خاصیت ViewProperties.SlideViewProperties اضافه شده است. خاصیت Aspose.Slides.ViewProperties.SlideViewProperties اضافه شده است. این خاصیت ویژگی‌های نمای عمومی مرتبط با حالت نمای اسلاید را مشخص می‌کند.
