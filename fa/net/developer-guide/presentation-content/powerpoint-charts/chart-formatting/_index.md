## **Щ…ШұЩҲШұЫҢ Ъ©Щ„ЫҢ**

Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮ ЩҶШӯЩҲЩҮ ЩӮШ§Щ„ШЁвҖҢШЁЩҶШҜЫҢ ЩҶЩ…ЩҲШҜШ§ШұЩҮШ§ ШҜШұ Ш§ШұШ§ШҰЩҮвҖҢЩҮШ§ЫҢ PowerPoint ШұШ§ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Aspose.Slides ШӘЩҲШ¶ЫҢШӯ Щ…ЫҢвҖҢШҜЩҮШҜ. ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ ЪҶЪҜЩҲЩҶЩҮ Ш№ЩҶШ§ШөШұ Ъ©Щ„ЫҢШҜЫҢ ЩҶЩ…ЩҲШҜШ§Шұ Щ…Ш§ЩҶЩҶШҜ Щ…ШӯЩҲШұЩҮШ§ШҢ Ш®Ш·ЩҲШ· ШҙШЁЪ©ЩҮШҢ Ш№ЩҶШ§ЩҲЫҢЩҶШҢ Ш§ЩҒШіШ§ЩҶЩҮвҖҢЩҮШ§ШҢ ЩҶШ§ШӯЫҢЩҮвҖҢЩҶЩӮШҙЩҮ ЩҲ ЩҫШұ Ъ©ШұШҜЩҶ ШҜЫҢЩҲШ§ШұЩҮШ§ ШұШ§ ШҙШ®ШөЫҢвҖҢШіШ§ШІЫҢ Ъ©ЩҶЫҢШҜ ШӘШ§ ШёШ§ЩҮШұ ЩҲ Ш®ЩҲШ§ЩҶШ§ЫҢЫҢ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ЩҶЩ…ЩҲШҜШ§Шұ ШЁЩҮШЁЩҲШҜ ЫҢШ§ШЁШҜ.

ЩҮЩ…ЪҶЩҶЫҢЩҶ ЩҶШӯЩҲЩҮ ШӘЩҶШёЫҢЩ… ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ ЩӮЩ„Щ… ШЁШұШ§ЫҢ Щ…ШӘЩҶ ЩҶЩ…ЩҲШҜШ§ШұШҢ Ш§Ш№Щ…Ш§Щ„ ЩӮШ§Щ„ШЁвҖҢЩҮШ§ЫҢ Ш№ШҜШҜЫҢ ЩҫЫҢШҙвҖҢШӘЩҶШёЫҢЩ…вҖҢШҙШҜЩҮ ЩҲ ШіЩҒШ§ШұШҙЫҢ ШЁЩҮ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ЩҶЩ…ЩҲШҜШ§ШұШҢ ЩҲ ЩҒШ№Ш§Щ„вҖҢШіШ§ШІЫҢ ЪҜЩҲШҙЩҮвҖҢЩҮШ§ЫҢ ЪҜШұШҜ ШЁШұШ§ЫҢ ЩҶШ§ШӯЫҢЩҮ ЩҶЩ…ЩҲШҜШ§Шұ ШұШ§ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ. Ш§ЫҢЩҶ Щ…Ш«Ш§Щ„вҖҢЩҮШ§ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ ЪҶЪҜЩҲЩҶЩҮ ЩҮЩ… ШіШЁЪ© ШЁШөШұЫҢ ЩҲ ЩҮЩ… Ш§ШұШ§ШҰЩҮ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ЩҶЩ…ЩҲШҜШ§Шұ ШұШ§ ШҜШұ ЫҢЪ© Ш§ШұШ§ШҰЩҮ Ъ©ЩҶШӘШұЩ„ Ъ©ЩҶЫҢШҜ.

## **ЩӮШ§Щ„ШЁвҖҢШЁЩҶШҜЫҢ Щ…ЩҲШ¬ЩҲШҜЫҢШӘвҖҢЩҮШ§ЫҢ ЩҶЩ…ЩҲШҜШ§Шұ**
Aspose.Slides ШЁШұШ§ЫҢ .NET ШЁЩҮ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ Ш§Щ…Ъ©Ш§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ ЩҶЩ…ЩҲШҜШ§ШұЩҮШ§ЫҢ ШіЩҒШ§ШұШҙЫҢ ШұШ§ Ш§ШІ Ш§ШЁШӘШҜШ§ ШЁЩҮ Ш§ШіЩ„Ш§ЫҢШҜЩҮШ§ЫҢ Ш®ЩҲШҜ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶЩҶШҜ. Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮ ЩҶШӯЩҲЩҮ ЩӮШ§Щ„ШЁвҖҢШЁЩҶШҜЫҢ Щ…ЩҲШ¬ЩҲШҜЫҢШӘвҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ ЩҶЩ…ЩҲШҜШ§Шұ Ш§ШІ Ш¬Щ…Щ„ЩҮ Щ…ШӯЩҲШұ ШҜШіШӘЩҮвҖҢШЁЩҶШҜЫҢ ЩҲ Щ…ШӯЩҲШұ Щ…ЩӮШҜШ§Шұ ШұШ§ ШӘЩҲШ¶ЫҢШӯ Щ…ЫҢвҖҢШҜЩҮШҜ.

Aspose.Slides ШЁШұШ§ЫҢ .NET ЫҢЪ© API ШіШ§ШҜЩҮ ШЁШұШ§ЫҢ Щ…ШҜЫҢШұЫҢШӘ Щ…ЩҲШ¬ЩҲШҜЫҢШӘвҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ ЩҶЩ…ЩҲШҜШ§Шұ ЩҲ ЩӮШ§Щ„ШЁвҖҢШЁЩҶШҜЫҢ ШўЩҶЩҮШ§ ШЁШ§ Щ…ЩӮШ§ШҜЫҢШұ ШіЩҒШ§ШұШҙЫҢ ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ:

1. ЫҢЪ© ЩҶЩ…ЩҲЩҶЩҮ Ш§ШІ Ъ©Щ„Ш§Ші **Presentation** Ш§ЫҢШ¬Ш§ШҜ Ъ©ЩҶЫҢШҜ.  
1. Щ…ШұШ¬Ш№ Ш§ШіЩ„Ш§ЫҢШҜ ШұШ§ ШЁШұ Ш§ШіШ§Ші ШҙШ§Ш®Шө ШўЩҶ ШҜШұЫҢШ§ЩҒШӘ Ъ©ЩҶЫҢШҜ.  
1. ЫҢЪ© ЩҶЩ…ЩҲШҜШ§Шұ ШЁШ§ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ЩҫЫҢШҙвҖҢЩҒШұШ¶ ШЁЩҮ ЩҮЩ…ШұШ§ЩҮ ЩҮШұ ЩҶЩҲШ№ ШҜЩ„Ш®ЩҲШ§ЩҮ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶЫҢШҜ (ШҜШұ Ш§ЫҢЩҶ Щ…Ш«Ш§Щ„ Ш§ШІ **ChartType.LineWithMarkers** Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЫҢЩ…).  
1. ШЁЩҮ **Value Axis** ЩҶЩ…ЩҲШҜШ§Шұ ШҜШіШӘШұШіЫҢ ЩҫЫҢШҜШ§ Ъ©ЩҶЫҢШҜ ЩҲ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ ШІЫҢШұ ШұШ§ ШӘЩҶШёЫҢЩ… Ъ©ЩҶЫҢШҜ:  
   1. ШӘЩҶШёЫҢЩ… **Line format** ШЁШұШ§ЫҢ Ш®Ш·ЩҲШ· Щ…ШҙШЁЪ© ШЁШІШұЪҜ (Major) Value Axis  
   1. ШӘЩҶШёЫҢЩ… **Line format** ШЁШұШ§ЫҢ Ш®Ш·ЩҲШ· Щ…ШҙШЁЪ© Ъ©ЩҲЪҶЪ© (Minor) Value Axis  
   1. ШӘЩҶШёЫҢЩ… **Number Format** ШЁШұШ§ЫҢ Value Axis  
   1. ШӘЩҶШёЫҢЩ… **Min, Max, Major and Minor units** ШЁШұШ§ЫҢ Value Axis  
   1. ШӘЩҶШёЫҢЩ… **Text Properties** ШЁШұШ§ЫҢ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ Value Axis  
   1. ШӘЩҶШёЫҢЩ… **Title** ШЁШұШ§ЫҢ Value Axis  
   1. ШӘЩҶШёЫҢЩ… **Line Format** ШЁШұШ§ЫҢ Value Axis  
1. ШЁЩҮ **Category Axis** ЩҶЩ…ЩҲШҜШ§Шұ ШҜШіШӘШұШіЫҢ ЩҫЫҢШҜШ§ Ъ©ЩҶЫҢШҜ ЩҲ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ ШІЫҢШұ ШұШ§ ШӘЩҶШёЫҢЩ… Ъ©ЩҶЫҢШҜ:  
   1. ШӘЩҶШёЫҢЩ… **Line format** ШЁШұШ§ЫҢ Ш®Ш·ЩҲШ· Щ…ШҙШЁЪ© ШЁШІШұЪҜ (Major) Category Axis  
   1. ШӘЩҶШёЫҢЩ… **Line format** ШЁШұШ§ЫҢ Ш®Ш·ЩҲШ· Щ…ШҙШЁЪ© Ъ©ЩҲЪҶЪ© (Minor) Category Axis  
   1. ШӘЩҶШёЫҢЩ… **Text Properties** ШЁШұШ§ЫҢ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ Category Axis  
   1. ШӘЩҶШёЫҢЩ… **Title** ШЁШұШ§ЫҢ Category Axis  
   1. ШӘЩҶШёЫҢЩ… **Label Positioning** ШЁШұШ§ЫҢ Category Axis  
   1. ШӘЩҶШёЫҢЩ… **Rotation Angle** ШЁШұШ§ЫҢ ШЁШұЪҶШіШЁвҖҢЩҮШ§ЫҢ Category Axis  
1. ШЁЩҮ **Legend** ЩҶЩ…ЩҲШҜШ§Шұ ШҜШіШӘШұШіЫҢ ЩҫЫҢШҜШ§ Ъ©ЩҶЫҢШҜ ЩҲ **Text Properties** ШўЩҶ ШұШ§ ШӘЩҶШёЫҢЩ… Ъ©ЩҶЫҢШҜ  
1. ЩҶЩ…Ш§ЫҢШҙ Ш§ЩҒШіШ§ЩҶЩҮвҖҢЩҮШ§ЫҢ ЩҶЩ…ЩҲШҜШ§Шұ ШұШ§ ШЁШҜЩҲЩҶ ЩҮЩ…ЩҫЩҲШҙШ§ЩҶЫҢ ШЁШ§ ЩҶЩ…ЩҲШҜШ§Шұ ШӘЩҶШёЫҢЩ… Ъ©ЩҶЫҢШҜ  
1. ШЁЩҮ **Secondary Value Axis** ЩҶЩ…ЩҲШҜШ§Шұ ШҜШіШӘШұШіЫҢ ЩҫЫҢШҜШ§ Ъ©ЩҶЫҢШҜ ЩҲ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ ШІЫҢШұ ШұШ§ ШӘЩҶШёЫҢЩ… Ъ©ЩҶЫҢШҜ:  
   1. ЩҒШ№Ш§Щ„вҖҢШіШ§ШІЫҢ **Value Axis** Ш«Ш§ЩҶЩҲЫҢЩҮ  
   1. ШӘЩҶШёЫҢЩ… **Line Format** ШЁШұШ§ЫҢ Secondary Value Axis  
   1. ШӘЩҶШёЫҢЩ… **Number Format** ШЁШұШ§ЫҢ Secondary Value Axis  
   1. ШӘЩҶШёЫҢЩ… **Min, Max, Major and Minor units** ШЁШұШ§ЫҢ Secondary Value Axis  
1. Ш§Ъ©ЩҶЩҲЩҶ ШіШұЫҢ ЩҶЩ…ЩҲШҜШ§Шұ Ш§ЩҲЩ„ ШұШ§ ШұЩҲЫҢ Secondary Value Axis ШұШіЩ… Ъ©ЩҶЫҢШҜ  
1. ШұЩҶЪҜ ЩҫШұ Ъ©ШұШҜЩҶ ШҜЫҢЩҲШ§Шұ ЩҫШҙШӘ ЩҶЩ…ЩҲШҜШ§Шұ ШұШ§ ШӘЩҶШёЫҢЩ… Ъ©ЩҶЫҢШҜ  
1. ШұЩҶЪҜ ЩҫШұ Ъ©ШұШҜЩҶ ЩҶШ§ШӯЫҢЩҮвҖҢЩҶЩӮШҙЩҮ ЩҶЩ…ЩҲШҜШ§Шұ ШұШ§ ШӘЩҶШёЫҢЩ… Ъ©ЩҶЫҢШҜ  
1. Ш§ШұШ§ШҰЩҮ Ш§ШөЩ„Ш§ШӯвҖҢШҙШҜЩҮ ШұШ§ ШЁЩҮ ЫҢЪ© ЩҒШ§ЫҢЩ„ PPTX ШЁЩҶЩҲЫҢШіЫҢШҜ  

```c#
// ШҜШұ ШӯШ§Щ„ Ш§ЫҢШ¬Ш§ШҜ Ш§ШұШ§ШҰЩҮ// ШҜШұ ШӯШ§Щ„ Ш§ЫҢШ¬Ш§ШҜ Ш§ШұШ§ШҰЩҮ
Presentation pres = new Presentation();

// ШҜШіШӘШұШіЫҢ ШЁЩҮ Ш§ШіЩ„Ш§ЫҢШҜ Ш§ЩҲЩ„
ISlide slide = pres.Slides[0];

// Ш§ЩҒШІЩҲШҜЩҶ ЩҶЩ…ЩҲШҜШ§Шұ ЩҶЩ…ЩҲЩҶЩҮ
IChart chart = slide.Shapes.AddChart(ChartType.LineWithMarkers, 50, 50, 500, 400);

// ШӘЩҶШёЫҢЩ… Ш№ЩҶЩҲШ§ЩҶ ЩҶЩ…ЩҲШҜШ§Шұ
chart.HasTitle = true;
chart.ChartTitle.AddTextFrameForOverriding("");
IPortion chartTitle = chart.ChartTitle.TextFrameForOverriding.Paragraphs[0].Portions[0];
chartTitle.Text = "Sample Chart";
chartTitle.PortionFormat.FillFormat.FillType = FillType.Solid;
chartTitle.PortionFormat.FillFormat.SolidFillColor.Color = Color.Gray;
chartTitle.PortionFormat.FontHeight = 20;
chartTitle.PortionFormat.FontBold = NullableBool.True;
chartTitle.PortionFormat.FontItalic = NullableBool.True;

// ШӘЩҶШёЫҢЩ… ЩӮШ§Щ„ШЁ Ш®Ш·ЩҲШ· ШҙШЁЪ©ЩҮ ШЁШІШұЪҜ ШЁШұШ§ЫҢ Щ…ШӯЩҲШұ Щ…ЩӮШҜШ§Шұ
chart.Axes.VerticalAxis.MajorGridLinesFormat.Line.FillFormat.FillType = FillType.Solid;
chart.Axes.VerticalAxis.MajorGridLinesFormat.Line.FillFormat.SolidFillColor.Color = Color.Blue;
chart.Axes.VerticalAxis.MajorGridLinesFormat.Line.Width = 5;
chart.Axes.VerticalAxis.MajorGridLinesFormat.Line.DashStyle = LineDashStyle.DashDot;

// ШӘЩҶШёЫҢЩ… ЩӮШ§Щ„ШЁ Ш®Ш·ЩҲШ· ШҙШЁЪ©ЩҮ Ъ©ЩҲЪҶЪ© ШЁШұШ§ЫҢ Щ…ШӯЩҲШұ Щ…ЩӮШҜШ§Шұ
chart.Axes.VerticalAxis.MinorGridLinesFormat.Line.FillFormat.FillType = FillType.Solid;
chart.Axes.VerticalAxis.MinorGridLinesFormat.Line.FillFormat.SolidFillColor.Color = Color.Red;
chart.Axes.VerticalAxis.MinorGridLinesFormat.Line.Width = 3;

// ШӘЩҶШёЫҢЩ… ЩӮШ§Щ„ШЁ Ш№ШҜШҜЫҢ Щ…ШӯЩҲШұ Щ…ЩӮШҜШ§Шұ
chart.Axes.VerticalAxis.IsNumberFormatLinkedToSource = false;
chart.Axes.VerticalAxis.DisplayUnit = DisplayUnitType.Thousands;
chart.Axes.VerticalAxis.NumberFormat = "0.0%";

// ШӘЩҶШёЫҢЩ… Щ…ЩӮШ§ШҜЫҢШұ ШӯШҜШ§Ъ©Ш«Шұ ЩҲ ШӯШҜШ§ЩӮЩ„ ЩҶЩ…ЩҲШҜШ§Шұ
chart.Axes.VerticalAxis.IsAutomaticMajorUnit = false;
chart.Axes.VerticalAxis.IsAutomaticMaxValue = false;
chart.Axes.VerticalAxis.IsAutomaticMinorUnit = false;
chart.Axes.VerticalAxis.IsAutomaticMinValue = false;

chart.Axes.VerticalAxis.MaxValue = 15f;
chart.Axes.VerticalAxis.MinValue = -2f;
```

## **ШӘЩҶШёЫҢЩ… ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ ЩӮЩ„Щ… ШЁШұШ§ЫҢ ЫҢЪ© ЩҶЩ…ЩҲШҜШ§Шұ**
Aspose.Slides ШЁШұШ§ЫҢ .NET ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ ШӘЩҶШёЫҢЩ… ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ Щ…ШұШӘШЁШ· ШЁШ§ ЩӮЩ„Щ… ШЁШұШ§ЫҢ ЩҶЩ…ЩҲШҜШ§Шұ ШұШ§ ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ. Щ„Ш·ЩҒШ§ЩӢ ШЁШұШ§ЫҢ ШӘЩҶШёЫҢЩ… ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ ЩӮЩ„Щ… ШЁШұШ§ЫҢ ЩҶЩ…ЩҲШҜШ§Шұ Щ…ШұШ§ШӯЩ„ ШІЫҢШұ ШұШ§ ШҜЩҶШЁШ§Щ„ Ъ©ЩҶЫҢШҜ.

- ШҙЫҢШЎ Ъ©Щ„Ш§Ші **Presentation** ШұШ§ ЩҶЩ…ЩҲЩҶЩҮвҖҢШіШ§ШІЫҢ Ъ©ЩҶЫҢШҜ.  
- ЫҢЪ© ЩҶЩ…ЩҲШҜШ§Шұ ШұШ§ ШЁЩҮ Ш§ШіЩ„Ш§ЫҢШҜ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶЫҢШҜ.  
- Ш§ШұШӘЩҒШ§Ш№ ЩӮЩ„Щ… ШұШ§ ШӘЩҶШёЫҢЩ… Ъ©ЩҶЫҢШҜ.  
- Ш§ШұШ§ШҰЩҮ Ш§ШөЩ„Ш§ШӯвҖҢШҙШҜЩҮ ШұШ§ Ш°Ш®ЫҢШұЩҮ Ъ©ЩҶЫҢШҜ.

ЫҢЪ© Щ…Ш«Ш§Щ„ ЩҶЩ…ЩҲЩҶЩҮ ШҜШұ ШІЫҢШұ ШўЩҲШұШҜЩҮ ШҙШҜЩҮ Ш§ШіШӘ.

```c#
using (Presentation pres = new Presentation())
{               
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 100, 100, 500, 400);
    chart.TextFormat.PortionFormat.FontHeight = 20;
```

## **ШӘЩҶШёЫҢЩ… ЩӮШ§Щ„ШЁ Ш№ШҜШҜЫҢ**
Aspose.Slides ШЁШұШ§ЫҢ .NET ЫҢЪ© API ШіШ§ШҜЩҮ ШЁШұШ§ЫҢ Щ…ШҜЫҢШұЫҢШӘ ЩӮШ§Щ„ШЁ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ЩҶЩ…ЩҲШҜШ§Шұ ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ:

1. ЫҢЪ© ЩҶЩ…ЩҲЩҶЩҮ Ш§ШІ Ъ©Щ„Ш§Ші [Presentation](https://reference.aspose.com/slides/fa/net/aspose.slides/presentation) Ш§ЫҢШ¬Ш§ШҜ Ъ©ЩҶЫҢШҜ.  
1. Щ…ШұШ¬Ш№ Ш§ШіЩ„Ш§ЫҢШҜ ШұШ§ ШЁШұ Ш§ШіШ§Ші ШҙШ§Ш®Шө ШўЩҶ ШҜШұЫҢШ§ЩҒШӘ Ъ©ЩҶЫҢШҜ.  
1. ЫҢЪ© ЩҶЩ…ЩҲШҜШ§Шұ ШЁШ§ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ЩҫЫҢШҙвҖҢЩҒШұШ¶ ШЁЩҮ ЩҮЩ…ШұШ§ЩҮ ЩҮШұ ЩҶЩҲШ№ ШҜЩ„Ш®ЩҲШ§ЩҮ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶЫҢШҜ (ШҜШұ Ш§ЫҢЩҶ Щ…Ш«Ш§Щ„ Ш§ШІ **ChartType.ClusteredColumn** Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ).  
1. ЩӮШ§Щ„ШЁ Ш№ШҜШҜЫҢ ЩҫЫҢШҙвҖҢШӘЩҶШёЫҢЩ…вҖҢШҙШҜЩҮ ШұШ§ Ш§ШІ Щ…ЩӮШ§ШҜЫҢШұ Щ…Щ…Ъ©ЩҶ Ш§ЩҶШӘШ®Ш§ШЁ Ъ©ЩҶЫҢШҜ.  
1. ШҜШұ ЩҮШұ ШіШұЫҢ Ш§ШІ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ЩҶЩ…ЩҲШҜШ§ШұШҢ ШіЩ„ЩҲЩ„вҖҢЩҮШ§ЫҢ ШҜШ§ШҜЩҮ ШұШ§ ЩҫЫҢЩ…Ш§ЫҢШҙ Ъ©ШұШҜЩҮ ЩҲ ЩӮШ§Щ„ШЁ Ш№ШҜШҜЫҢ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ЩҶЩ…ЩҲШҜШ§Шұ ШұШ§ ШӘЩҶШёЫҢЩ… Ъ©ЩҶЫҢШҜ.  
1. Ш§ШұШ§ШҰЩҮ ШұШ§ Ш°Ш®ЫҢШұЩҮ Ъ©ЩҶЫҢШҜ.  
1. ЩӮШ§Щ„ШЁ Ш№ШҜШҜЫҢ ШіЩҒШ§ШұШҙЫҢ ШұШ§ ШӘЩҶШёЫҢЩ… Ъ©ЩҶЫҢШҜ.  
1. ШҜШұ ЩҮШұ ШіШұЫҢ Ш§ШІ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ЩҶЩ…ЩҲШҜШ§ШұШҢ ШіЩ„ЩҲЩ„вҖҢЩҮШ§ЫҢ ШҜШ§ШҜЩҮ ШұШ§ ЩҫЫҢЩ…Ш§ЫҢШҙ Ъ©ШұШҜЩҮ ЩҲ ЩӮШ§Щ„ШЁ Ш№ШҜШҜЫҢ Щ…ШӘЩҒШ§ЩҲШӘЫҢ Ш§Ш№Щ…Ш§Щ„ Ъ©ЩҶЫҢШҜ.

Aspose.Slides ШЁШұШ§ЫҢ .NET ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ ШӘЩҶШёЫҢЩ… ЩҶШ§ШӯЫҢЩҮ ЩҶЩ…ЩҲШҜШ§Шұ ШұШ§ ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ. ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ **IChart.HasRoundedCorners** ЩҲ **Chart.HasRoundedCorners** ШҜШұ Aspose.Slides Ш§Ш¶Ш§ЩҒЩҮ ШҙШҜЩҮвҖҢШ§ЩҶШҜ.

1. ШҙЫҢШЎ Ъ©Щ„Ш§Ші `Presentation` ШұШ§ ЩҶЩ…ЩҲЩҶЩҮвҖҢШіШ§ШІЫҢ Ъ©ЩҶЫҢШҜ.  
1. ЫҢЪ© ЩҶЩ…ЩҲШҜШ§Шұ ШұШ§ ШЁЩҮ Ш§ШіЩ„Ш§ЫҢШҜ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶЫҢШҜ.  
1. ЩҶЩҲШ№ ЩҫШұ Ъ©ШұШҜЩҶ ЩҲ ШұЩҶЪҜ ЩҫШұ Ъ©ШұШҜЩҶ ЩҶЩ…ЩҲШҜШ§Шұ ШұШ§ ШӘЩҶШёЫҢЩ… Ъ©ЩҶЫҢШҜ.  
1. ЩҲЫҢЪҳЪҜЫҢ ЪҜЩҲШҙЩҮвҖҢЩҮШ§ЫҢ ЪҜШұШҜ ШұШ§ ШЁШұШ§ШЁШұ **True** ШӘЩҶШёЫҢЩ… Ъ©ЩҶЫҢШҜ.  
1. Ш§ШұШ§ШҰЩҮ Ш§ШөЩ„Ш§ШӯвҖҢШҙШҜЩҮ ШұШ§ Ш°Ш®ЫҢШұЩҮ Ъ©ЩҶЫҢШҜ.

ЫҢЪ© Щ…Ш«Ш§Щ„ ЩҶЩ…ЩҲЩҶЩҮ ШҜШұ ШІЫҢШұ ШўЩҲШұШҜЩҮ ШҙШҜЩҮ Ш§ШіШӘ.

```c#
using (Presentation presentation = new Presentation())
{
	ISlide slide = presentation.Slides[0];
	IChart chart = slide.Shapes.AddChart(ChartType.ClusteredColumn, 20, 100, 600, 400);
	chart.LineFormat.FillFormat.FillType = FillType.Solid;
	chart.LineFormat.Style = LineStyle.Single;
	chart.HasRoundedCorners = true;

	presentation.Save("out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

## **ШіЩҲШ§Щ„Ш§ШӘ Щ…ШӘШҜШ§ЩҲЩ„**

**ШўЫҢШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩ… ЩҫШұЪ©ЩҶЩҶШҜЩҮвҖҢЩҮШ§ЫҢ ЩҶЫҢЩ…ЩҮвҖҢШҙЩҒШ§ЩҒ ШЁШұШ§ЫҢ ШіШӘЩҲЩҶвҖҢЩҮШ§/ЩҶЩҲШ§ШӯЫҢ ШӘЩҶШёЫҢЩ… Ъ©ЩҶЩ… ШҜШұ ШӯШ§Щ„ЫҢ Ъ©ЩҮ ШӯШ§ШҙЫҢЩҮ ШҙЩҒШ§ЩҒ ШЁШ§ЩӮЫҢ Щ…ЫҢвҖҢЩ…Ш§ЩҶШҜШҹ**

ШЁЩ„ЩҮ. ШҙЩҒШ§ЩҒЫҢШӘ ЩҫШұЪ©ЩҶЩҶШҜЩҮ ЩҲ Ш®Ш·ЩҲШ· ШӯШ§ШҙЫҢЩҮ ШЁЩҮвҖҢШ·ЩҲШұ Ш¬ШҜШ§ЪҜШ§ЩҶЩҮ ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ. Ш§ЫҢЩҶ Ъ©Ш§Шұ ШЁШұШ§ЫҢ ШЁЩҮШЁЩҲШҜ Ш®ЩҲШ§ЩҶШ§ЫҢЫҢ ШҙШЁЪ©ЩҮ ЩҲ ШҜШ§ШҜЩҮвҖҢЩҮШ§ ШҜШұ ЩҶЩ…Ш§ЫҢЩҮвҖҢЩҮШ§ЫҢ Щ…ШӘШұШ§Ъ©Щ… Щ…ЩҒЫҢШҜ Ш§ШіШӘ.

**ЪҶЪҜЩҲЩҶЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩ… ШЁШ§ ШЁШұЪҶШіШЁвҖҢЩҮШ§ЫҢ ШҜШ§ШҜЩҮ Ъ©ЩҮ ЩҮЩ…ЩҫЩҲШҙШ§ЩҶЫҢ ШҜШ§ШұЩҶШҜ ШЁШұШ®ЩҲШұШҜ Ъ©ЩҶЩ…Шҹ**

Ш§ЩҶШҜШ§ШІЩҮ ЩӮЩ„Щ… ШұШ§ Ъ©Ш§ЩҮШҙ ШҜЩҮЫҢШҜШҢ Ш§Ш¬ШІШ§ЫҢ ШәЫҢШұШ¶ШұЩҲШұЫҢ ШЁШұЪҶШіШЁ (Щ…Ш§ЩҶЩҶШҜ ШҜШіШӘЩҮвҖҢЩҮШ§) ШұШ§ ШәЫҢШұЩҒШ№Ш§Щ„ Ъ©ЩҶЫҢШҜШҢ Ш¬Ш§ШЁШ¬Ш§ЫҢЫҢ/Щ…ЩҲЩӮШ№ЫҢШӘ ШЁШұЪҶШіШЁ ШұШ§ ШӘЩҶШёЫҢЩ… Ъ©ЩҶЫҢШҜШҢ ШҜШұ ШөЩҲШұШӘ Щ„ШІЩҲЩ… ЩҒЩӮШ· ШЁШұШ§ЫҢ ЩҶЩӮШ§Ш· Ш§ЩҶШӘШ®Ш§ШЁ ШҙШҜЩҮ ШЁШұЪҶШіШЁ ЩҶШҙШ§ЩҶ ШҜЩҮЫҢШҜ ЫҢШ§ ЩӮШ§Щ„ШЁ ШұШ§ ШЁЩҮ В«Щ…ЩӮШҜШ§Шұ + Ш§ЩҒШіШ§ЩҶЩҮВ» ШӘШәЫҢЫҢШұ ШҜЩҮЫҢШҜ.

**ШўЫҢШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩ… ЩҫШұЪ©ЩҶЩҶШҜЩҮвҖҢЩҮШ§ЫҢ ЪҜШұШ§ШҜЫҢШ§ЩҶ ЫҢШ§ Ш§Щ„ЪҜЩҲ ШұШ§ ШЁЩҮ ШіШұЫҢвҖҢЩҮШ§ Ш§Ш№Щ…Ш§Щ„ Ъ©ЩҶЩ…Шҹ**

ШЁЩ„ЩҮ. ЩҮШұ ШҜЩҲ ЩҶЩҲШ№ ЩҫШұЪ©ЩҶЩҶШҜЩҮЩ” Ш«Ш§ШЁШӘ ЩҲ ЪҜШұШ§ШҜЫҢШ§ЩҶ/Ш§Щ„ЪҜЩҲ Щ…Ш№Щ…ЩҲЩ„Ш§ЩӢ ШҜШұ ШҜШіШӘШұШі ЩҮШіШӘЩҶШҜ. ШҜШұ Ш№Щ…Щ„ШҢ Ш§ШІ ЪҜШұШ§ШҜЫҢШ§ЩҶвҖҢЩҮШ§ ШЁЩҮвҖҢШөЩҲШұШӘ Щ…ШӯШҜЩҲШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ ЩҲ ШӘШұЪ©ЫҢШЁвҖҢЩҮШ§ЫҢЫҢ Ъ©ЩҮ Ъ©ЩҶШӘШұШ§ШіШӘ ШұШ§ ШЁШ§ ШҙШЁЪ©ЩҮ ЩҲ Щ…ШӘЩҶ Ъ©Ш§ЩҮШҙ Щ…ЫҢвҖҢШҜЩҮЩҶШҜШҢ Ш§Ш¬ШӘЩҶШ§ШЁ Ъ©ЩҶЫҢШҜ.
