## **مقدمه**

دسترس‌پذیری ارائه‌ها اطمینان می‌دهد که افرادی که از فناوری‌های کمکی مانند صفحه‌خوان‌ها، نمایشگرهای بریل یا ناوبری فقط با صفحه‌کلید استفاده می‌کنند، می‌توانند اسلایدهای شما را به همان اندازه که مخاطبان بینایی‌دار و با ماوس می‌بینند، درک و مرور کنند. روش‌های خوب بر ترتیب خواندن واضح، متن جایگزین معنی‌دار برای تصاویر اطلاعاتی، کنتراست رنگ کافی، تایپوگرافی قابل‌خواندن، متن پیوند توصیفی و جلوگیری از انتقال معنا تنها از طریق رنگ یا موقعیت تمرکز می‌کند. وقتی دسترس‌پذیری از ابتدا برنامه‌ریزی شود، نتیجه ساختاری تمیزتر، تصاویری سازگارتر و محتوایی است که بدون راه‌حل‌های جانبی به هر بیننده‌ای می‌رسد.

## **علامت‌گذاری به‌عنوان تزئینی**

علامت‌گذاری به‌عنوان تزئینی برای عناصر صرفاً تزئینی استفاده می‌شود تا صفحه‌خوان‌ها از آن‌ها عبور کنند، باعث کاهش نویز و حفظ تمرکز بر محتوای معنی‌دار می‌شود. این را برای پس‌زمینه‌ها، تزئینات و فضای خالی به کار ببرید—هرگز برای نمودارها، آیکون‌ها یا تصاویری که اطلاعاتی منتقل می‌کنند. Aspose.Slides این پرچم را برای تشخیص و اعتبارسنجی در دسترس می‌گذارد و امکان انجام بررسی‌های خودکار دسترس‌پذیری و پاک‌سازی را فراهم می‌کند.

![علامت‌گذاری به‌عنوان تزئینی](mark_as_decorative.png)

نمونه کد زیر نشان می‌دهد که چگونه می‌توان تعیین کرد آیا یک شکل به‌عنوان تزئینی علامت‌گذاری شده است یا خیر.

```cs
using var presentation = new Presentation("sample.pptx");

var shape = presentation.Slides[0].Shapes[0];
Console.WriteLine($"Is shape decorative: {shape.IsDecorative}");
```
