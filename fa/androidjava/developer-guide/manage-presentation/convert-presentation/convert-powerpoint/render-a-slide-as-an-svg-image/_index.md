## **مرور کلی**

این مقاله توضیح می‌دهد که چگونه می‌توان اسلایدهای ارائه را به‌صورت تصاویر SVG با استفاده از Aspose.Slides رندر کرد. این مقاله فرمت SVG و مزایای آن از جمله مقیاس‌پذیری، دسترس‌پذیری و مناسبت برای توسعه وب را شرح می‌دهد.

شما می‌آموزید که چگونه یک فایل ارائه را بارگذاری کنید، اسلایدهای آن را مرور کنید و هر اسلاید را به‌صورت یک فایل SVG جداگانه ذخیره نمایید. مقاله فرمت‌های ارائه PowerPoint و OpenDocument شامل PPT، PPTX، ODP و PPS را پوشش می‌دهد و نشان می‌دهد چگونه تبدیل را به‌صورت برنامه‌نویسی با کلاس `Presentation` و متد `writeAsSvg` انجام دهید.

## **فرمت SVG**

SVG—مخفف Scalable Vector Graphics—یک نوع یا فرمت گرافیکی استاندارد برای رندر تصاویر دو‑بعدی است. SVG تصاویر را به‌صورت بردارها در XML با جزئیاتی که رفتار یا ظاهر آنها را تعریف می‌کند، ذخیره می‌کند.

SVG یکی از معدود فرمت‌هایی است که در این زمینه‌ها استانداردهای بسیار بالایی دارد: مقیاس‌پذیری، تعامل‌پذیری، عملکرد، دسترس‌پذیری، برنامه‌نویسی، و غیره. به همین دلایل، به‌طور گسترده‌ای در توسعه وب استفاده می‌شود.

ممکن است بخواهید از فایل‌های SVG استفاده کنید زمانی که نیاز دارید:

- **ارائه خود را در *فرمت بسیار بزرگ* چاپ کنید.** تصاویر SVG می‌توانند به هر وضوح یا سطحی مقیاس‌پذیر شوند. می‌توانید تصاویر SVG را هر تعداد که لازم است بدون از دست دادن کیفیت، تغییر اندازه دهید.
- **نمودارها و گراف‌ها را از اسلایدهای خود در *رسانه‌ها یا پلتفرم‌های مختلف* استفاده کنید.** اکثر مرورگرها قادر به تفسیر فایل‌های SVG هستند.
- **کوچک‌ترین حجم ممکن برای تصاویر را داشته باشید.** فایل‌های SVG عموماً کوچکتر از معادل‌های با وضوح بالا در فرمت‌های دیگر، به‌ویژه آنهایی که بر پایه بیت‌مپ (JPEG یا PNG) هستند، می‌باشند.

## **رندر یک اسلاید به‌عنوان تصویر SVG**

Aspose.Slides for Android via Java به شما امکان می‌دهد اسلایدهای ارائه خود را به‌صورت تصاویر SVG صادر کنید. برای تولید تصاویر SVG مراحل زیر را دنبال کنید:

1. یک نمونه از کلاس `Presentation` ایجاد کنید.
2. تمام اسلایدهای موجود در ارائه را مرور کنید.
3. هر اسلاید را از طریق `FileOutputStream` به فایل SVG جداگانه‌ای بنویسید.

{{% alert color="primary" %}} 

ممکن است بخواهید برنامه وب رایگان ما را امتحان کنید[برنامه وب رایگان](https://products.aspose.app/slides/fa/conversion/ppt-to-svg) که در آن عملکرد تبدیل PPT به SVG را با Aspose.Slides for Android via Java پیاده‌سازی کرده‌ایم.

{{% /alert %}} 

این کد نمونه در Java نشان می‌دهد چگونه می‌توان PPT را به SVG تبدیل کرد با استفاده از Aspose.Slides:

``` java
Presentation pres = new Presentation("pres.pptx");
try {
    for (int index = 0; index < pres.getSlides().size(); index++)
    {
        ISlide slide = pres.getSlides().get_Item(index);

        FileOutputStream fileStream = new FileOutputStream("slide-" + index + ".svg");
        try {
            slide.writeAsSvg(fileStream);
        } finally {
            fileStream.close();
        }
    }
} catch(IOException e) {
} finally {
    if (pres != null) pres.dispose();
}
```

## **سؤالات متداول**

**چرا ممکن است SVG تولید‌شده در مرورگرهای مختلف متفاوت به‌نظر برسد؟**

پشتیبانی از ویژگی‌های خاص SVG توسط موتورهای مرورگر به‌صورت متفاوت پیاده‌سازی می‌شود. پارامترهای [SVGOptions](https://reference.aspose.com/slides/fa/androidjava/com.aspose.slides/svgoptions/) به کاهش ناسازگاری‌ها کمک می‌کند.

**آیا امکان صادر کردن نه تنها اسلایدها بلکه شکل‌های فردی به SVG وجود دارد؟**

بله. هر [شکل می‌تواند به‌صورت SVG جداگانه ذخیره شود](https://reference.aspose.com/slides/fa/androidjava/com.aspose.slides/shape/#writeAsSvg-java.io.OutputStream-com.aspose.slides.ISVGOptions-) که برای آیکون‌ها، پیکتوگرام‌ها و استفاده مجدد از گرافیک‌ها مناسب است.

**آیا می‌توان چندین اسلاید را در یک SVG (نوار/سند) ترکیب کرد؟**

سناریوی استاندارد یک اسلاید → یک SVG است. ترکیب چندین اسلاید در یک بوم SVG یک مرحله پس‌پردازش است که در سطح برنامه انجام می‌شود.
