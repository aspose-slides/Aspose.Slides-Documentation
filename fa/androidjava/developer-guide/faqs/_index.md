## **مرور کلی**

این سؤالات متداول پاسخ‌های رایجی به پرسش‌های متداول درباره Aspose.Slides را ارائه می‌دهد. این مقاله به فرمت‌های فایل پشتیبانی‌شده، مدیریت استثناها هنگام کار با ارائه‌های بزرگ، تغییر اندازه اسلایدها, پیش‌نمایش اسلایدها, استخراج متن از ارائه‌ها, قالب‌بندی حاشیه‌های جدول, قرار دادن تصاویر و حل مشکلات مرتبط با قلم‌ها هنگام تبدیل ارائه‌ها به PDF یا تصویر می‌پردازد.

## **فرمت‌های فایل پشتیبانی‌شده**

**س: چه فرمت‌های فایلی را Aspose.Slides برای Android از طریق Java پشتیبانی می‌کند؟**

**پ**: Aspose.Slides برای Android از طریق Java از فرمت‌های فایلی که در [فرمت‌های فایل پشتیبانی‌شده](/slides/fa/androidjava/supported-file-formats/) توضیح داده شده‌اند، پشتیبانی می‌کند.

## **استثناها**

**س: در هنگام بارگیری یک فایل PPT بزرگ با تصاویر، خطای out of memory دریافت می‌کنم. آیا محدودیتی در Aspose.Slides برای اندازه فایل وجود دارد؟**

**پ**: هیچ فرمول خاصی برای محاسبه اندازه ارائه‌ای که توسط Aspose.Slides پشتیبانی می‌شود وجود ندارد. باید فضای کافی برای نگهداری کل ساختار ارائه و تصاویر در حافظه وجود داشته باشد. به طور معمول، تصاویر در حافظه بیشتر از فضای دیسک سخت اشغال می‌کنند، به‌ویژه هنگامی که تصاویر افکت‌های اضافی داشته باشند.

به‌طور کلی، Aspose.Slides برای Android از طریق Java می‌تواند به‌راحتی فایل‌های ارائه‌ای حدود 300 مگابایت را در سروری با 4 گیگابایت RAM مدیریت کند.

## **کار با اسلایدها**

**س: آیا می‌توانم اندازه اسلایدهای یک ارائه را تغییر دهم؟**

**پ**: می‌توانید از متد `getSlideSize` که توسط کلاس [Presentation](https://reference.aspose.com/slides/fa/androidjava/com.aspose.slides/presentation/) ارائه شده است، برای تعریف اندازه اسلایدها در یک ارائه استفاده کنید.

**س: آیا راهی برای تعریف اسلایدهایی با اندازه متفاوت در یک ارائه وجود دارد؟**

**پ**: از آنجا که اندازه اسلایدها در سطح ارائه در اسناد Microsoft PowerPoint تعریف می‌شود، چنین امکانی وجود ندارد.

**س: آیا Aspose.Slides برای Android از طریق Java پیش‌نمایش اسلاید قبل از ذخیره‌سازی را پشتیبانی می‌کند؟**

**پ**: می‌توانید اسلایدهای ارائه را به تصاویر رندر کنید و از این تصاویر برای پیش‌نمایش اسلایدها استفاده کنید.

## **کار با متن**

**س: آیا امکان استخراج تمام متن از یک ارائه وجود دارد؟**

**پ**: Aspose.Slides برای Android از طریق Java کلاس [SlideUtil](https://reference.aspose.com/slides/fa/androidjava/com.aspose.slides/slideutil/) را ارائه می‌دهد که روش‌های مختلفی برای استخراج کل متن از ارائه‌ها فراهم می‌کند.

**س: چرا اندازه‌های پاراگراف در PC و Android متفاوت است؟**

**پ**: محاسبه اندازه پاراگراف بر پایه محاسبه اندازه متن نمایانگر پاراگراف انجام می‌شود. محاسبه اندازه متن بر اساس معیارهای فونتی است که در ارائه PowerPoint مشخص شده است. اگر فونت مشخص‌شده موجود نباشد، با فونت مشابهی جایگزین می‌شود که معیارهای متفاوتی نسبت به فونت اصلی دارد. در نتیجه، محاسبه اندازه پاراگراف در سیستم‌های مختلف به دلیل مجموعه فونت‌های نصب‌شده متفاوت می‌شود. برای دستیابی به نتایج یکسان در سیستم‌عامل‌های مختلف، باید همان فونت‌ها را روی سیستم‌ها نصب کنید یا آن‌ها را در زمان اجرا به‌عنوان [فونت‌های خارجی](/slides/fa/androidjava/custom-font/) بارگذاری کنید.

## **قالب‌بندی و تصاویر**

**س: چگونه می‌توانم رنگ حاشیه جدول را تنظیم کنم؟**

**پ**: می‌توانید رنگ تمام حاشیه‌های جدول یا فقط حاشیه اطراف کل جدول را تغییر دهید. برای تغییر تمام حاشیه‌ها، لطفاً از متد `getCellFormat` در اینترفیس [ICell](https://reference.aspose.com/slides/fa/androidjava/com.aspose.slides/icell/) استفاده کنید. برای حاشیه کل جدول، باید در سلول‌ها تکرار کنید و رنگ حاشیه‌های خارجی را تغییر دهید.

**س: Aspose.Slides برای Android از طریق Java برای قرار دادن تصاویر از چه واحدی استفاده می‌کند؟**

**پ**: مختصات و اندازه‌های تمام اشکال بر روی اسلایدها بر حسب نقطه (72 dpi) اندازه‌گیری می‌شوند.

## **کار با قلم‌ها**

**س: هنگام تبدیل PPT به PDF یا تصاویر، چرا قلم‌ها در اسناد خروجی متفاوت هستند؟**

**پ**: این مشکل ممکن است نشان‌دهنده این باشد که قلم‌های استفاده‌شده در ارائه بر روی سیستم‌عامل‌ایی که کد اجرا شده است، موجود نیستند. باید قلم‌ها را بر روی سیستم‌عامل نصب کنید یا آن‌ها را به‌عنوان قلم‌های خارجی با استفاده از کلاس [FontsLoader](https://reference.aspose.com/slides/fa/androidjava/com.aspose.slides/fontsloader/) بارگذاری کنید، همان‌طور که در زیر نشان داده شده است:
```java
String[] folders = new String[] { "path_to_a_folder_with_fonts" };
FontsLoader.loadExternalFonts(folders);
```
