## **مروری کلی**

دسترس‌پذیری ارائه اطمینان می‌دهد که افرادی که از فناوری‌های کمکی—مانند خوانندگان صفحه‌خوان، نمایشگرهای بریل یا ناوبری فقط با صفحه‌کلید—استفاده می‌کنند، می‌توانند اسلایدهای شما را به‌طور مؤثر همانند مخاطبان بینایی‌دار و استفاده‌کننده از ماوس درک و مرور کنند. روش‌های مناسب بر ترتیب واضح خواندن، متن جایگزین معنادار برای تصاویر اطلاع‌رسانی، تضاد کافی رنگ، تایپوگرافی قابل خواندن، متن لینک توصیفی و جلوگیری از انتقال معنا تنها با رنگ یا موقعیت تمرکز می‌کنند. وقتی دسترس‌پذیری از ابتدا برنامه‌ریزی شود، ساختاری تمیزتر، بصری‌های سازگارتر و محتوایی که بدون راه‌حل‌های جایگزین به هر بیننده می‌رسد، حاصل می‌شود.

## **علامت‌گذاری به‌عنوان تزئینی**

پرچم «به‌عنوان تزئینی علامت‌گذاری» برای گرافیک‌های صرفاً تزئینی استفاده می‌شود تا خوانندگان صفحه‌خوان آن‌ها را نادیده بگیرند، که باعث کاهش نویز و حفظ تمرکز بر محتوای معنادار می‌شود. این پرچم را روی پس‌زمینه‌ها، تزیینات و فاصله‌گذارها اعمال کنید—هرگز روی نمودارها، نمادها یا تصاویری که اطلاعات منتقل می‌کنند. Aspose.Slides این پرچم را برای شناسایی و اعتبارسنجی در دسترس قرار می‌دهد و امکان انجام بررسی‌های خودکار دسترس‌پذیری و پاکسازی را فراهم می‌کند.

![علامت‌گذاری به‌عنوان تزئینی](mark_as_decorative.png)

```java
Presentation presentation = new Presentation("sample.pptx");
try {
    IShape shape = presentation.getSlides().get_Item(0).getShapes().get_Item(0);
    System.out.println("Is shape decorative: " + shape.isDecorative());
} finally {
    presentation.dispose();
}
```
