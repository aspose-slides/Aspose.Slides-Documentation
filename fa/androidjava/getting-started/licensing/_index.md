## **نمای کلی**

Aspose.Slides می‌تواند در حالت ارزیابی یا با لایسنس معتبر استفاده شود. نسخه ارزیابی همان عملکرد نسخه مجوزدار را دارد، اما هنگام باز یا ذخیره‌سازی ارائه‌ها، یک واترمارک ارزیابی اضافه می‌کند و استخراج متن را به یک اسلاید محدود می‌سازد.

این مقاله توضیح می‌دهد که مجوزدهی در Aspose.Slides چگونه کار می‌کند و چگونه قبل از استفاده از کتابخانه یک لایسنس اعمال کنید. یک لایسنس می‌تواند از فایل، استریم یا منبع جاسازی‌شده با استفاده از کلاس `License` بارگذاری شود. همچنین مقاله نشان می‌دهد چگونه صحت اعمال لایسنس را بررسی کنید.

## **ارزیابی Aspose.Slides**

{{% alert color="primary" %}} 

می‌توانید نسخه ارزیابی **Aspose.Slides for Android via Java** را از صفحهٔ [download page](https://releases.aspose.com/slides/fa/androidjava/) دریافت کنید. نسخه ارزیابی همان قابلیت‌های نسخه لایسنس‌دار محصول را فراهم می‌کند. بستهٔ ارزیابی همانند بستهٔ خریداری‌شده است. نسخهٔ ارزیابی به سادگی پس از افزودن چند خط کد برای اعمال لایسنس، به نسخهٔ لایسنس‌دار تبدیل می‌شود.

پس از اینکه از ارزیابی **Aspose.Slides** راضی شدید، می‌توانید [purchase a license](https://purchase.aspose.com/buy) کنید. توصیه می‌کنیم انواع مختلف اشتراک را بررسی کنید. اگر سؤال دارید، با تیم فروش Aspose تماس بگیرید.

هر لایسنس Aspose با یک اشتراک یک‌ساله برای ارتقاهای رایگان به نسخه‌های جدید یا اصلاحات منتشرشده در دورهٔ اشتراک همراه است. کاربران دارای محصولات لایسنس‌دار (حتی نسخه‌های ارزیابی) پشتیبانی فنی رایگان و نامحدود دریافت می‌کنند.

{{% /alert %}} 

**محدودیت‌های نسخه ارزیابی**

* نسخه ارزیابی Aspose.Slides (بدون لایسنس تعیین‌شده) تمام عملکرد محصول را فراهم می‌کند، اما یک واترمارک ارزیابی در بالای سند هنگام عملیات باز و ذخیره‌سازی درج می‌کند.

## **مجوزدهی در Aspose.Slides**

* یک نسخه ارزیابی پس از خرید لایسنس و افزودن چند خط کد برای اعمال لایسنس، تبدیل به نسخهٔ لایسنس‌دار می‌شود. 
* لایسنس یک فایل XML متنی ساده است که جزئیاتی مانند نام محصول، تعداد توسعه‌دهندگانی که به آن مجوز داده شده، تاریخ انقضای اشتراک و غیره را شامل می‌شود. 
* فایل لایسنس به صورت دیجیتالی امضا شده است، بنابراین نباید آن را ویرایش کنید. حتی افزودن ناخواسته یک خط جدید به محتوای فایل نیز اعتبار آن را از بین می‌برد. 
* Aspose.Slides for Android via Java معمولاً لایسنس را در این مکان‌ها جستجو می‌کند:
  * مسیر صریح
  * پوشهٔ حاوی Aspose.Slides.jar
* برای حذف محدودیت‌های نسخهٔ ارزیابی، باید قبل از استفاده از **Aspose.Slides** یک لایسنس تنظیم کنید. شما تنها یک بار در هر برنامه یا پردازش نیاز به تنظیم لایسنس دارید.

## **اعمال لایسنس**

یک لایسنس می‌تواند از **file** یا **stream** بارگذاری شود.

{{% alert color="primary" %}}

Aspose.Slides کلاس [License](https://reference.aspose.com/slides/fa/androidjava/com.aspose.slides/license/) را برای عملیات مربوط به لایسنس فراهم می‌کند.

{{% /alert %}} 

{{% alert color="warning" %}}

لایسنس‌های جدید می‌توانند Aspose.Slides را فقط از نسخهٔ 21.4 به بعد فعال کنند. نسخه‌های قبلی از یک سیستم مجوزدهی متفاوت استفاده می‌کنند و این لایسنس‌ها را تشخیص نمی‌دهند.

{{% /alert %}}

### **File**

ساده‌ترین روش برای تنظیم لایسنس این است که فایل لایسنس را در پوشهٔ حاوی Aspose.Slides.jar یا JAR برنامهٔ خود قرار دهید.

این کد Java نشان می‌دهد چگونه یک فایل لایسنس تنظیم کنید:

``` java
// یک نمونه از کلاس License ایجاد می‌کند
com.aspose.slides.License license = new com.aspose.slides.License();

// مسیر فایل لایسنس را تنظیم می‌کند
license.setLicense("Aspose.Slides.Android.via.Java.lic");
```

{{% alert color="warning" %}} 

اگر فایل لایسنس را در مسیری متفاوت قرار دهید، هنگام فراخوانی متد [SetLicense](https://reference.aspose.com/slides/fa/androidjava/com.aspose.slides/license/#setLicense-java.lang.String-)، نام فایل لایسنس در انتهای مسیر صریح باید همان نام فایل لایسنس شما باشد.

به عنوان مثال، می‌توانید نام فایل لایسنس را به *Aspose.Slides.Android.via.Java.lic.xml* تغییر دهید. سپس در کد خود باید مسیر فایل (که با *Aspose.Slides.Android.via.Java.lic.xml* به پایان می‌رسد) را به متد [SetLicense](https://reference.aspose.com/slides/fa/androidjava/com.aspose.slides/license/#setLicense-java.lang.String-) پاس دهید.

{{% /alert %}}

### **Stream**

می‌توانید یک لایسنس را از یک استریم بارگذاری کنید. این کد Java نشان می‌دهد چگونه یک لایسنس را از استریم اعمال کنید:

``` java
// یک نمونه از کلاس License ایجاد می‌کند
com.aspose.slides.License license = new com.aspose.slides.License();

// لایسنس را از طریق یک استریم تنظیم می‌کند
license.setLicense(new java.io.FileInputStream("Aspose.Slides.Android.via.Java.lic"));
```

## **اعتبارسنجی لایسنس**

برای بررسی اینکه آیا لایسنس به درستی تنظیم شده است یا نه، می‌توانید آن را اعتبارسنجی کنید. این کد Java نشان می‌دهد چگونه یک لایسنس را اعتبارسنجی کنید:

```java
License license = new License();
license.setLicense("Aspose.Slides.Android.via.Java.lic");

if (License.isLicensed()) 
{
    System.out.println("License is good!");
}
```

## **ایمنی در برابر چندنخی**

{{% alert title="Note" color="warning" %}} 

متد [SetLicense](https://reference.aspose.com/slides/fa/androidjava/com.aspose.slides/license/#setLicense-java.io.InputStream-) ایمن در مقابل چندنخی نیست. اگر این متد باید همزمان از چندین نخ فراخوانی شود، ممکن است بخواهید از سازوکارهای همگام‌سازی (مانند lock) برای جلوگیری از مشکلات استفاده کنید. 

{{% /alert %}}

## **سوالات متداول**

**آیا می‌توانم لایسنس را در محیط کاملاً آفلاین (بدون دسترسی به اینترنت) اعمال کنم؟**

بله. اعتبارسنجی لایسنس به صورت محلی با استفاده از فایل لایسنس انجام می‌شود؛ اتصال به اینترنت لازم نیست.

**بعد از پایان اشتراک یک‌ساله چه اتفاقی می‌افتد؟ آیا کتابخانه کارکرد خود را متوقف می‌کند؟**

خیر. لایسنس دائمی است: می‌توانید از نسخه‌های منتشرشده قبل از تاریخ پایان اشتراک خود استفاده کنید؛ فقط برای استفاده از نسخه‌های جدیدتر بدون تمدید اشتراک مجاز نخواهید بود.
