## **Overview**

این مقاله توضیح می‌دهد چگونه می‌توانید پشتیبانی برای Aspose.Slides را دریافت کنید و چگونه به‌صورت مؤثر مسائل را گزارش کنید. این مقاله کانال اصلی پشتیبانی را توصیف می‌کند، توصیه‌هایی برای ارسال مشکل ارائه می‌دهد و به منابع فنی مفیدی مانند صفحه محصول، وبلاگ، دانلودها، انجمن پشتیبانی، مستندات و API Reference اشاره می‌کند. همچنین به سؤالات رایج درباره پشتیبانی پولی، ارتقاء مشکل و پشتیبانی از نسخه‌های قدیمی کتابخانه پاسخ می‌دهد.

## **Technical support**

{{% alert color="primary" %}} 

Aspose پشتیبانی فنی نامحدود و رایگان برای تمام محصولات خود ارائه می‌دهد. این پشتیبانی برای همه کاربران (از جمله کاربران با بسته‌های ارزیابی) در دسترس است.

{{% /alert %}} 

هنگامی که به کمک برای Aspose.Slides for Android via Java نیاز دارید، موارد زیر را در نظر بگیرید:

- مسیر اصلی پشتیبانی، انجمن Aspose.Forums است. سؤال خود را در انجمن Aspose.Slides ارسال کنید — و ظرف چند ساعت پاسخ خواهد شد. تیم پشتیبانی Aspose.Slides به صورت مستقیم به سوالات ارسال شده در انجمن پاسخ می‌دهد.
- لطفاً توجه داشته باشید که Aspose پشتیبانی فنی از طریق تلفن ارائه نمی‌دهد. پشتیبانی تلفنی فقط برای سؤالات فروش و خرید در دسترس است.
- هنگام انتظار برای پاسخ در انجمن‌ها، صبور باشید و اختلافات ساعت‌های زمانی را در نظر بگیرید.

هنگامی که با مشکلی در Aspose.Slides for Android via Java مواجه می‌شوید، برای رفع آن به کارآمدترین شکل زیر را دنبال کنید:

- قبل از گزارش مشکل، بررسی و تأیید کنید که از آخرین نسخه Aspose.Slides for Android via Java استفاده می‌کنید. شاید مشکلی که تجربه کردید در به‌روزرسانی‌ای قبلی حل شده باشد.

  برای دریافت اطلاعات درباره آخرین نسخه، به صفحه دانلود Aspose.Slides مراجعه کنید.

- پیش از گزارش مشکل، ممکن است بخواهید انجمن‌ها، این مستندات و API Reference را مرور کنید. شاید پاسخ سؤال شما قبلاً ارائه شده باشد.

- هنگام گزارش مشکل، لطفاً سند اصلی و (در صورت امکان) بخشی از کد مربوط به مشکل را ضمیمه کنید. اگر نیاز به پیوست چندین فایل دارید، آنها را در یک بسته ZIP قرار دهید.

  می‌توانید به‌امنیت اسناد خود را در Aspose.Forums پیوست کنید زیرا فقط شما و توسعه‌دهندگان Aspose به فایل‌های پیوست‌شده دسترسی دارند.

- برای هر مشکل یک موضوع (thread) ایجاد کنید و استفاده کنید. اگر مشکل دیگری دارید، باید موضوع دیگری باز کنید تا آن را گزارش دهید.

## **FAQ**

**What paid support tiers are available and how do they differ?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). آنها بر اساس تعداد موارد گنجانده‌شده، اولویت پاسخ و کانال‌های ارتباطی موجود متفاوت هستند.

**Can I escalate a bug directly to the engineering team?**

بله. [Paid support](https://helpdesk.aspose.com/) امکان ارتقاء به توسعه‌دهندگان را زمانی که یک مسئله نیاز به بررسی عمیق‌تر دارد، فراهم می‌کند.

**Are older library versions supported, and do you release fixes for them?**

شما می‌توانید از نسخه‌های قدیمی استفاده کنید، اما اصلاحات فقط برای نسخه جاری ارائه می‌شوند؛ بسته‌های تعمیر برای نسخه‌های قدیمی منتشر نمی‌شوند.
