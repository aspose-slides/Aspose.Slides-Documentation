{{% alert color="primary" %}} 

Aspose.Slides for Android via Java یک کتابخانهٔ کلاس است که به برنامه‌های جاوا برای اندروید امکان می‌دهد اسناد ارائه را بدون استفاده از Microsoft PowerPoint® یا نرم‌افزارهای دیگر بخوانند و بنویسند. Aspose.Slides for Android via Java به شما اجازه می‌دهد متن، شکل‌ها، جدول‌ها و انیمیشن‌ها را مدیریت کنید؛ صدا و ویدئو را به اسلایدها اضافه کنید؛ پیش‌نمایش اسلایدها؛ و اسلایدها را به SVG، فرمت PDF و موارد دیگر صادر کنید.

آخرین مرجع API را می‌توانید در [Aspose.Slides for Android via Java API Reference](https://reference.aspose.com/slides/fa/androidjava/) پیدا کنید.
