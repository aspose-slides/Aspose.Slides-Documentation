Aspose.Slides for PHP via Java یک کتابخانهٔ کلاس است که به برنامه‌های PHP امکان خواندن و نوشتن اسناد ارائه‌ای را بدون استفاده از Microsoft PowerPoint® یا دیگران می‌دهد. Aspose.Slides for PHP via Java اجازه می‌دهد متن، اشکال، جدول‌ها و انیمیشن‌ها را مدیریت کنید، صدا و ویدئو به اسلایدها اضافه کنید، پیش‌نمایش اسلایدها را ببینید، اسلایدها را به فرمت SVG، PDF و موارد دیگر صادر کنید.

آخرین مرجع API می‌تواند در [Aspose.Slides for PHP via Java API Reference](https://reference.aspose.com/slides/fa/php-java/) یافت شود.
