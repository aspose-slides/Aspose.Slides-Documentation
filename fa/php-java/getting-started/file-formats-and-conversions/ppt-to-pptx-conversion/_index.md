## **ویژگی‌های پشتیبانی‌شده در تبدیل**
Aspose.Slides برای PHP از طریق Java پشتیبانی جزئی از تبدیل قالب فایل PPT به PPTX را فراهم می‌کند. پشتیبانی از تبدیل به‌تازگی به Aspose.Slides برای PHP از طریق Java اضافه شده است، بنابراین محدودیت‌هایی دارد و برای ارائه‌های ساده بهترین عملکرد را دارد. مزیت اصلی که Aspose.Slides برای PHP از طریق Java هنگام تبدیل PPT به PPTX ارائه می‌دهد، سهولت استفاده از API است. برای مشاهده مثال‌های کد، درباره [تبدیل PPT به PPTX]() بخوانید. در ادامه، فهرست‌ها نشان می‌دهند کدام ویژگی‌ها برای تبدیل PPT به PPTX پشتیبانی می‌شوند و کدام نیستند.

## **ویژگی‌های پشتیبانی‌شده**
ویژگی‌های زیر برای تبدیل پشتیبانی می‌شوند:

- تبدیل ساختار مسترها، طرح‌بندی‌ها و اسلایدها.
- تبدیل نمودارها.
- گروه‌اشکال.
- تبدیل شکل‌های خودکار شامل مستطیل‌ها و بیضی‌ها.
- اشکال با هندسه سفارشی.
- سبک‌های پر کردن با بافت و تصویر برای شکل‌های خودکار.
- تبدیل محل نگهداری‌ها.
- تبدیل خطوط و پلی‌لاین‌ها.
- فرمت‌های خط و پر کردن.
- سبک‌های پر کردن گرادیان.
- فریم‌های OLE، جدول‌ها، فریم‌های ویدئو و صدا و غیره.
- ویژگی‌های انیمیشن و اسلایدشو.
- تبدیل متن در فریم‌های متن و نگهدارنده‌های متن.
