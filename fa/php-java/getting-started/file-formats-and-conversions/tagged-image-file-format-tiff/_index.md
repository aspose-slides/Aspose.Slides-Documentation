## **درباره TIFF**

[فرمت فایل تصویری توگد](https://en.wikipedia.org/wiki/TIFF) که به خاطر امکان ذخیره‌سازی چندین تصویر در یک سند شناخته شده است، در اصل توسط Aldus ایجاد شد. این فرمت به‌طور گسترده‌ای توسط برنامه‌های اسکن، فاکس و سایر برنامه‌های دستکاری تصویر پشتیبانی می‌شود.

## **TIFF در Aspose.Slides برای PHP از طریق Java**

هر سندی که بتوان در Aspose.Slide برای Java بارگذاری کرد، می‌تواند به‌صورت مستقیم توسط Aspose.Slides برای PHP از طریق Java به سند TIFF تبدیل شود و نیازی به هیچ مؤلفه شخص ثالثی نیست. علاوه بر این، می‌توانید به‌طور اختیاری اندازهٔ تصاویر را در سند TIFF حاصل تعریف کنید. می‌توانید اطلاعات دربارهٔ خروجی‌گیری اسناد ارائه‌شده به سندهای TIFF از طریق Aspose.Slides برای PHP از طریق Java را در [این موضوع](/slides/fa/php-java/convert-powerpoint-to-tiff/) بیابید.
