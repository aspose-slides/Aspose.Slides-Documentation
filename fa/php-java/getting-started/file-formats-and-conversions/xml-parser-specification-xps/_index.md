{{% alert color="primary" %}} 

‏[XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) یک زبان توصیف صفحه و فرمت سند ثابت است که ابتدا توسط مایکروسافت توسعه یافت. همچون PDF، XPS برای حفظ صحت سند و ارائهٔ ظاهر سندی مستقل از دستگاه طراحی شده است. 

{{% /alert %}} 

## **XPS در Aspose.Slides برای PHP از طریق Java**
هر سند ارائه‌ای که بتوان توسط Aspose.Slides برای PHP از طریق Java بارگذاری شد می‌تواند به فرمت XPS تبدیل شود. Aspose.Slides برای PHP از طریق Java از موتور چیدمان صفحه با دقت بالا و رندرینگ استفاده می‌کند تا خروجی را در فرمت سند ثابت‑چیدمان XPS تولید کند.
می‌توانید دربارهٔ استخراج اسناد ارائه به اسناد XPS از طریق Aspose.Slides برای PHP از طریق Java در بخش [Converting to XPS](https://docs.aspose.com/slides/fa/php-java/convert-powerpoint-to-xps/) اطلاعات بیشتری کسب کنید.
