{{% alert color="primary" %}} 
[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) فرمت فایل سند ارائه است که می‌تواند توسط نسخه‌های مختلف مایکرروسافت پاورپوینت ایجاد، خوانده، دستکاری و نوشته شود. این فرمت باینری برای اسناد ارائه توسط مایکرروسافت توسعه یافته است.
{{% /alert %}} 

## **PPT در Aspose.Slides for PHP via Java**
Aspose.Slides for PHP via Java می‌تواند فایل‌های PPT ایجاد شده توسط نرم‌افزارهای زیر را بخواند.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

به‌طور مشابه، فایل‌های PPT ایجاد شده توسط Aspose.Slides for PHP via Java می‌توانند توسط مجموعه نرم‌افزارهای فوق خوانده شوند.

## **پشتیبانی جامع برای PPT**
Aspose.Slides for PHP via Java پشتیبانی تقریباً تمام ویژگی‌های پشتیبانی‌شده توسط فرمت فایل سند PPT را فراهم می‌کند. این کتابخانه نه تنها ویژگی‌های پایه و پیشرفته ارائه‌شده توسط نسخه‌های مختلف Microsoft PowerPoint برای دستکاری اسناد PPT را پوشش می‌دهد، بلکه ویژگی‌هایی را که حتی توسط Microsoft PowerPoint پشتیبانی نمی‌شوند، نیز شامل می‌شود. مزیت اصلی استفاده از کتابخانه API Aspose.Slides for PHP via Java، سهولت استفاده برای مدیریت این ویژگی‌هاست.

علاوه بر وظایف پایه مرتبط با ایجاد، خواندن و نوشتن فایل‌های سند PPT، چندین ویژگی توسط Aspose.Slides for PHP via Java فراهم شده است:

- Import other Microsoft Office file formats as [اشیاء OLE به اسناد PPT](/slides/fa/php-java/manage-ole/).
- [صدور اسناد PPT به PDF](/slides/fa/php-java/convert-powerpoint-to-pdf/).
- اسلایدهای موجود در اسناد PPT را به فرمت‌های SVG صادر کنید.
- اسلایدها را به هر فرمت تصویری که توسط چارچوب Java پشتیبانی می‌شود، رندر کنید.
- اندازهٔ اسلایدها در اسناد PPT را تنظیم کنید.
- انیمیشن‌های اشکال را مدیریت کنید.
- نمایش‌های اسلاید را مدیریت کنید.
- [قالب‌بندی متن در اسلایدها](/slides/fa/php-java/text-formatting/).
- متن را از اسناد PPT استخراج کنید.
- [مدیریت جدول‌ها در اسلایدها](/slides/fa/php-java/powerpoint-table/).
- به‌صورت خودکار مسترها را با استفاده از [ویژگی کلونینگ](/slides/fa/php-java/clone-slides/) کپی کنید.
