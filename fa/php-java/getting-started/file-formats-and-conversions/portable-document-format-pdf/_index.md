قالب سند قابل حمل ([قالب سند قابل حمل](https://en.wikipedia.org/wiki/PDF)) یک قالب فایل است که توسط Adobe Systems برای تبادل اسناد بین سازمان‌ها ایجاد شده است. هدف این قالب این بود که محتوا و چیدمان را بدون توجه به پلتفرمی که در آن مشاهده می‌شود، یکسان نگه دارد. Aspose.Slides for PHP via Java به شما امکان تبدیل فایل‌های ارائه به PDF را می‌دهد.

{{% /alert %}} 

## **PDF در Aspose.Slides for PHP via Java**
هر ارائه‌ای که می‌تواند در Aspose.Slides for PHP via Java بارگذاری شود، می‌تواند به PDFی تبدیل شود که با [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A)، [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A)، [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) یا [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA) مطابق باشد، بسته به انتخاب شما. Aspose.Slides for PHP via Java ارائه‌ها را به PDF صادر می‌کند و در اکثر موارد، PDF خروجی دقیقاً شبیه ارائه اصلی است.

Aspose.Slides ویژگی‌های ارائه زیر را هنگام تبدیل به PDF پشتیبانی می‌کند:

- تصاویر، جعبه‌های متن و اشکال دیگر.
- متن و قالب‌بندی.
- پاراگراف‌ها و قالب‌بندی.
- هایپرلینک‌ها.
- سرصفحه‌ها و پاصفحه‌ها.
- نشان‌های فهرست.
- جداول.

می‌توانید ارائه‌ها را مستقیماً به PDF صادر کنید با استفاده از Aspose.Slides for PHP via Java: نیازی به هیچ مؤلفه دیگری ندارید. علاوه بر این، می‌توانید صادرات ارائه به PDF را با گزینه‌های مختلف سفارشی کنید همان‌طور که در [تبدیل به PDF](/slides/fa/php-java/convert-powerpoint-to-pdf/) توضیح داده شده است.
