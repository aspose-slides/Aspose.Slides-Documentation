## **پلتفرم‌های پشتیبانی‌شده**
Aspose.Slides for PHP via Java یک API مستقل از پلتفرم است. می‌تواند بر روی هر پلتفرمی (Windows، Linux، macOS و غیره) که PHP 7 یا بالاتر نصب شده باشد، استفاده شود. Oracle JDK 7 یا بالاتر نیز یک پیش‌نیاز دیگر است.

## **قالب‌های فایل و تبدیل‌ها**
Aspose.Slides for PHP via Java اکثر قالب‌های سند PowerPoint را پشتیبانی می‌کند. همچنین امکان خروجی به قالب‌های پرکاربردی که سازمان‌ها برای به‌اشتراک‌گذاری استفاده می‌کنند را فراهم می‌آورد. جزئیات را مرور کنید:

|**ویژگی**|**توضیح**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for PHP via Java سریع‌ترین پردازش را برای این قالب ارائه می‌دهد.|
|تبدیل PPT به PPTX|Aspose.Slides for PHP via Java از تبدیل PPT به PPTX پشتیبانی می‌کند|
|Portable Document Format (PDF)|می‌توانید تمام قالب‌های فایل پشتیبانی‌شده را با یک متد به اسناد Adobe Portable Document Format (PDF) صادر کنید.|
|XML Parser Specification (XPS)|می‌توانید تمام قالب‌های فایل پشتیبانی‌شده را با یک متد به اسناد XML Parser Specification (XPS) صادر کنید.|
|Tagged Image File Format (TIFF)|می‌توانید تمام قالب‌های ارائه را به Tagged Image File Format (TIFF) صادر کنید.|
|تبدیل PPTX به HTML|Aspose.Slides for PHP via Java از تبدیل PresentationEx به قالب HTML پشتیبانی می‌کند.|

## **رندر ارائه**
Aspose.Slides for PHP via Java رندر با وفاداری بالا برای اسلایدهای ارائه به قالب‌های گرافیکی مختلف را پشتیبانی می‌کند. جزئیات را مرور کنید:

|**ویژگی**|**توضیح**|
| :- | :- |
|قالب‌های تصویر پشتیبانی‌شده توسط Java|با استفاده از Aspose.Slides for PHP via Java می‌توانید اسلایدها و تصاویر اسلایدها را به تمام قالب‌های گرافیکی پشتیبانی‌شده توسط Java مانند TIFF، PNG، BMP، JPEG، GIF رندر کنید.|
|قالب SVG|Aspose.Slides for PHP via Java همچنین متدهای داخلی برای خروجی اسلایدهای ارائه به قالب Scalable Vector Graphics (SVG) فراهم می‌آورد.|
|چاپ ارائه|نسخه‌های جدید Aspose.Slides for PHP via Java متدهای چاپ داخلی با گزینه‌های مختلف را ارائه می‌دهند.|

## **سوالات متداول**

**آیا برای کارکرد کتابخانه نیاز است که Microsoft PowerPoint را بر روی سرور/رایانه نصب کنم؟**

نه. PowerPoint لازم نیست؛ Aspose.Slides یک موتور مستقل برای ایجاد، ویرایش، تبدیل و رندر ارائه‌ها است.

**چندنخی (multithreading) چگونه کار می‌کند؟ آیا پردازش می‌تواند به صورت موازی باشد؟**

امن است که اسناد مختلف را در نخ‌های متفاوت پردازش کنید؛ همان شیء [presentation](https://reference.aspose.com/slides/fa/php-java/aspose.slides/presentation/) نباید توسط [multiple threads](/slides/fa/php-java/multithreading/) همزمان استفاده شود.

**آیا رمزهای عبور فایل و رمزنگاری پشتیبانی می‌شوند؟**

بله. شما می‌توانید ارائه‌های رمزنگاری‌شده را باز کنید، رمز ورود یا نوشتن را تنظیم یا حذف کنید و وضعیت حفاظت را بررسی کنید. ([You can](/slides/fa/php-java/password-protected-presentation/))

**آیا در کانتینرهای لینوکس باید به پکیج‌های قلم توجه داشته باشم؟**

بله. توصیه می‌شود پکیج‌های قلم رایج را نصب کنید یا به‌صورت صریح [specify font directories](/slides/fa/php-java/custom-font/) را در برنامه خود مشخص کنید تا از جایگزینی‌های غیرمنتظره جلوگیری شود.

**آیا در نسخه ارزیابی محدودیتی وجود دارد؟**

در [evaluation mode](/slides/fa/php-java/licensing/) یک واترمارک به خروجی اضافه می‌شود و محدودیت‌های خاصی اعمال می‌شود؛ یک [30-day temporary license](https://purchase.aspose.com/temporary-license/) برای تست کامل ویژگی‌ها در دسترس است.

**آیا وارد کردن قالب‌های خارجی به یک ارائه (PDF/HTML → PPTX) پشتیبانی می‌شود؟**

بله. می‌توانید [PDF pages and HTML content](/slides/fa/php-java/import-presentation/) را به یک ارائه اضافه کنید و آن‌ها را به اسلاید تبدیل نمایید.
