## **پشتیبانی فنی**
{{% alert color="primary" %}}

Aspose پشتیبانی فنی نامحدود و رایگان برای تمام محصولات خود ارائه می‌دهد. این پشتیبانی برای همه کاربران (از جمله کاربران دارای بسته‌های ارزیابی) در دسترس است.

{{% /alert %}} 

When you need help with Aspose.Slides for PHP via Java, consider the following:

- مسیر اصلی پشتیبانی، [Aspose.Forums](https://forum.aspose.com/c/slides/fa/11) است. سؤال خود را در انجمن Aspose.Slides بگذارید—و پاسخ آن ظرف چند ساعت خواهد آمد. تیم پشتیبانی Aspose.Slides به طور مستقیم به سؤالاتی که در انجمن ارسال می‌شوند، پاسخ می‌دهد.
- لطفاً توجه داشته باشید که Aspose پشتیبانی فنی از طریق تلفن ارائه نمی‌دهد. پشتیبانی تلفنی فقط برای سوالات فروش و خرید در دسترس است.
- هنگام انتظار برای پاسخ در انجمن‌ها، لطفاً صبور باشید و اختلافات منطقه زمانی را در نظر بگیرید.

When you experience an issue with Aspose.Slides for PHP via Java, follow these guidelines to resolve it in the most efficient manner:

- بررسی و تأیید کنید که قبل از گزارش مشکل، از آخرین نسخه Aspose.Slides for PHP via Java استفاده می‌کنید. شاید مشکلی که مواجه شده‌اید در به‌روزرسانی قبلی برطرف شده باشد.

  برای دریافت اطلاعات درباره آخرین نسخه، به [Aspose.Slides download](https://downloads.aspose.com/slides/fa/php-java) مراجعه کنید.

- قبل از گزارش مشکل، می‌توانید انجمن‌ها، این مستندات و مرجع API را مرور کنید. ممکن است پاسخ سؤال شما قبلاً ارائه شده باشد.

- هنگام گزارش مشکل، لطفاً سند اصلی و (در صورت امکان) قطعه کدی مرتبط با مشکل را ضمیمه کنید. اگر نیاز به پیوست چندین فایل دارید، آنها را در یک بسته ZIP بگذارید. می‌توانید اسناد خود را به‌صورت ایمن در Aspose.Forums ضمیمه کنید زیرا **only** شما و توسعه‌دهندگان Aspose به فایل‌های پیوست شده دسترسی دارند.

- برای هر مشکل یک موضوع ایجاد و استفاده کنید. اگر مشکل دیگری داشتید، باید یک موضوع دیگر برای گزارش آن ایجاد کنید.
