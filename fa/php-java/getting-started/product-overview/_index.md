## **به Aspose.Slides for PHP via Java خوش آمدید!**

Aspose.Slides یک API مدیریت Microsoft PowerPoint® است که به برنامه‌ها امکان خواندن، نوشتن، ایجاد و ویرایش ارائه‌های PowerPoint را بدون استفاده از Microsoft PowerPoint® می‌دهد. این کتابخانه از کار با ارائه‌ها، اسلایدها، اشکال، جداول، متن، انیمیشن‌ها، صدا، ویدیو و سایر عناصر ارائه پشتیبانی می‌کند.

با Aspose.Slides، توسعه‌دهندگان می‌توانند ارائه‌ها را از منابع مختلف باز کنند، آن‌ها را در قالب‌های مختلف PowerPoint ذخیره نمایند، ارائه‌ها را به فرمت‌هایی مانند PDF، TIFF، XPS، SWF، HTML و SVG صادر کنند و اسلایدها را به تصویر تبدیل کنند. این API همچنین ویژگی‌هایی برای مدیریت خصوصیات سند، اعمال محافظت، پیش‌نمایش اسلایدها، ویرایش محتوای اسلاید، کار با یادداشت‌های اسلاید و مدیریت عناصر پیشرفته مانند اشیای توکار OLE فراهم می‌کند.

### **چرا باید از Aspose.Slides for PHP via Java استفاده کنید**

- Aspose.Slides for PHP via Java اولین و تنها کامپوننتی است که عملکردهای لازم برای مدیریت اسناد PowerPoint® را فراهم می‌کند.  
- این محصول ویژگی‌های کلیدی بسیاری از جمله مدیریت متن‌ها، اشکال، جداول و انیمیشن‌ها، افزودن صدا و ویدیو به اسلایدها، پیش‌نمایش اسلایدها، صادر کردن اسلایدها به SVG، PDF و سایر فرمت‌ها و قابلیت‌های فراوان دیگری را ارائه می‌دهد.  
- Aspose.Slides for PHP via Java نه تنها امکان باز کردن فایل‌های PowerPoint از منابع مختلف را می‌دهد، بلکه به شما اجازه می‌دهد ارائه‌های خود را به روش‌های متنوعی ذخیره کنید. به عنوان مثال می‌توانید ارائه‌ها را در قالب PPT ذخیره کنید؛ همچنین می‌توانید اسلایدها را به تصویر تبدیل کنید.  
- این کتابخانه تقریباً تمام ویژگی‌هایی که ممکن است در Microsoft PowerPoint یافت کنید را فراهم می‌کند. به‌علاوه جداول و انواع مختلف اشکال، می‌توانید فریم‌های متنی، صوتی و تصویری مختلفی را به اسلایدهای خود اضافه کنید.  

## **ویژگی‌های Aspose.Slides**

ویژگی‌های Aspose.Slides for PHP via Java می‌توانند به گروه‌های زیر تقسیم شوند:

- ویژگی‌های ارائه  
- ویژگی‌های اسلاید  
- ویژگی‌های گرافیک و مولتی‌مدیا  
- ویژگی‌های جدول  
- ویژگی‌های متن و پاراگراف  
- ویژگی‌های پیشرفته  

### **ویژگی‌های ارائه**

- باز کردن ارائه‌های Microsoft PowerPoint از منابع مختلف  
- ذخیره ارائه‌ها در قالب‌های مختلف Microsoft PowerPoint  
- ایجاد ارائه‌های جدید از صفر  
- تبدیل ارائه‌ها به اسناد PDF بدون استفاده از کامپوننت دیگر  
- خواندن یا اصلاح خصوصیات سند ارائه  
- اعمال محافظت بر روی ارائه  
- صادرات ارائه به PDF  
- صادرات ارائه به TIFF  
- صادرات ارائه به XPS  
- صادرات ارائه به SWF  
- صادرات ارائه به PDF Note / Tiff Notes  
- صادرات ارائه به HTML  

### **ویژگی‌های اسلاید**

- ایجاد اسلایدهای جدید از صفر و افزودن آن‌ها به ارائه‌ها  
- دسترسی به هر اسلاید از طریق موقعیت آن در ارائه  
- کپی یا کلون کردن اسلایدها در همان ارائه یا ارائه دیگری  
- ویرایش و حذف اسلایدها  
- کپی خودکار اسلایدهای مستر همراه با اسلایدها  
- پیش‌نمایش اسلایدها  
- دریافت یا تنظیم نام اسلاید  
- ذخیره اسلایدها به عنوان تصویر  
- صادرات اسلایدها به فرمت SVG  
- کار با چندین مستر در ارائه‌ها  
- پر کردن پس‌زمینه اسلاید با هر سبک پشتیبانی‌شده  
- افزودن یا مدیریت یادداشت‌های اسلاید  
- تنظیم مسترهای جدید اسلاید و سبک‌ها به‌صورت پویا  
- افزودن یا اصلاح انتقال اسلایدشو  
- افزودن تغییرات Morph  

### **ویژگی‌های گرافیک و مولتی‌مدیا**

- ایجاد یا مدیریت اشکالی مانند خطوط، مستطیل‌ها، بیضی‌ها و پلی‌لاین‌ها  
- مدیریت سبک خطوط در اشکال  
- دریافت متن جایگزین یک شکل  
- چرخاندن یا وارونه‌کردن اشکال  
- پر کردن اشکال با سبک‌های مختلف—مثلاً ثابت، گرادیان، الگو، بافت یا تصویر  
- استفاده از 48 الگو پیش‌تعریف‌شده و 24 بافت  
- اختصاص لینک به اشکال  
- خواندن، اصلاح و افزودن افکت‌های انیمیشن جدید به اسلایدها و اشکال  
- افزودن تصویر جدید به ارائه‌ها و مدیریت تصاویر موجود  
- افزودن گلوله‌های تصویری جدید به ارائه‌ها و مدیریت گلوله‌های موجود  
- افزودن فریم‌های صوتی جدید به ارائه‌ها و مدیریت فریم‌های موجود  
- افزودن فریم‌های ویدئویی جدید به ارائه‌ها و مدیریت فریم‌های موجود  

### **ویژگی‌های جدول**

- ایجاد جداول از صفر و افزودن آن‌ها به اسلایدها  
- تغییر اندازه و جابه‌جایی جداول  
- دسترسی به هر سلول در جدول  
- تغییر قالب و حاشیه‌های هر سلول  
- تغییر اندازه ستون‌ها و ردیف‌ها  
- افزودن و حذف ستون‌ها و ردیف‌ها  
- تنظیم عرض و رنگ حاشیه برای کل جدول  

### **ویژگی‌های متن و پاراگراف**

- دریافت و تنظیم متن در جایگزین‌های متنی و فریم‌های متنی  
- دریافت نوع جایگزین  
- تغییر قالب‌بندی متن در یک فریم متنی  
- افزودن قلم‌های جدید یا مدیریت قلم‌های استفاده‌شده در یک ارائه  
- تغییر متن و قالب‌بندی متن  
- ایجاد یا مدیریت لینک‌های متنی  
- پشتیبانی از تب‌ها و تب‌های مؤثر  
- افزودن فریم‌های متنی جدید به تقریباً هر نوع شکل  
- دسترسی و اصلاح حاشیه‌ها و تنظیمات چیدمان متن فریم‌ها  
- دسترسی و اصلاح تورفتگی پاراگراف  

### **ویژگی‌های پیشرفته**

- دسترسی به اشیای OLE توکار در ارائه‌ها  
- قفل کردن ارائه‌ها  

## **پشتیبانی فنی**

Aspose پشتیبانی فنی نامحدود و رایگان برای همه محصولات خود فراهم می‌کند. این پشتیبانی برای تمام کاربران (از جمله کاربران با نسخه‌های ارزیابی) در دسترس است. اگر به کمک برای Aspose.Slides for PHP via Java نیاز دارید، موارد زیر را در نظر بگیرید:

- مسیر اصلی پشتیبانی، [Aspose Forums](https://forum.aspose.com/) است. سؤال خود را در [Aspose.Slides forum](https://forum.aspose.com/c/slides/fa/11) ارسال کنید—و پرسش شما ظرف چند ساعت پاسخ داده خواهد شد. تیم پشتیبانی Aspose.Slides به‌صورت مستقیم به سؤالات ارسال‌شده در فوروم پاسخ می‌دهد.  
- لطفاً توجه داشته باشید که Aspose پشتیبانی فنی از طریق تلفن ارائه نمی‌دهد. پشتیبانی تلفنی فقط برای سؤالات خرید و فروش در دسترس است.  
- هنگام انتظار برای پاسخ در فوروم، صبور باشید و به اختلاف ساعت‌های زمانی توجه کنید.  

اگر با Aspose.Slides for PHP via Java مشکلی دارید، برای حل مؤثر آن دستورالعمل‌های زیر را دنبال کنید:

- ابتدا بررسی کنید که از آخرین نسخه Aspose.Slides for PHP via Java استفاده می‌کنید. شاید مشکل شما پیش از این در به‌روزرسانی‌ای رفع شده باشد.  

  برای دریافت اطلاعات درباره آخرین نسخه محصول، به [Aspose.Slides download](https://downloads.aspose.com/slides/fa/php-java) مراجعه کنید.  

- پیش از گزارش مشکل، می‌توانید از فوروم‌ها، این مستندات و مرجع API بازدید کنید. شاید پاسخ سؤال شما قبلاً ارائه شده باشد.  

- هنگام گزارش مشکل، سند اصلی و (در صورت امکان) بخشی از کد مربوط به مشکل را ضمیمه کنید. اگر نیاز به پیوست فایل‌های متعدد دارید، آن‌ها را در یک بسته ZIP قرار دهید.  

  می‌توانید به‌راحتی اسناد خود را در Aspose.Forums پیوست کنید؛ زیرا فقط شما و توسعه‌دهندگان Aspose به فایل‌های ضمیمه دسترسی دارند.  

- برای هر مشکل خاص یک ردیف (thread) ایجاد کنید. اگر مشکل دیگری داشتید، باید ردیف جدیدی ایجاد کنید.

## **سوالات متداول**

**آیا سطوح تطبیق PDF برای آرشیو و دسترسی (PDF/A و PDF/UA) را پشتیبانی می‌کند؟**

بله. می‌توانید با پیکربندی [PDF export options](https://reference.aspose.com/slides/fa/php-java/aspose.slides/pdfoptions/) به PDF/A-2a/2b/2u، PDF/A-3a/3b و PDF/UA ذخیره کنید.

**آیا مکانیزم جایگزینی قلم و پشتیبانی از قلم‌های سفارشی برای رندر صحیح وجود دارد؟**

بله. کتابخانه به شما اجازه می‌دهد [قلم‌های سفارشی را بارگذاری](/slides/fa/php-java/custom-font/) کنید و [قوانین fallback را تعریف](/slides/fa/php-java/fallback-font/) کنید تا گلیف‌های گمشده به‌درستی در زمان تبدیل و رندر جایگزین شوند.

**آیا می‌توانم بدون باز کردن کامل فایل، تشخیص دهم که آیا رمز‌گذاری شده است؟**

بله. می‌توانید [یک ارائه را بررسی](/slides/fa/php-java/examine-presentation/) کنید تا متوجه شوید قبل از بارگذاری کامل نیاز به رمز عبور دارد یا نه.

**آیا برای پردازش و تبدیل‌ها به Microsoft PowerPoint نیاز است؟**

خیر. Aspose.Slides یک موتور مستقل است؛ PowerPoint روی سرور یا workstation لازم نیست.

**آیا پردازش ارائه‌ها از چندین رشته (thread) ایمن است؟**

بله، می‌توانید اسناد مختلف را به‌صورت موازی در رشته‌های جداگانه پردازش کنید؛ فقط از استفاده همزمان از یک نمونه ارائه در [غیرهمزمانی رشته‌ها](/slides/fa/php-java/multithreading/) خودداری کنید.

**آیا ماکروها حفظ می‌شوند و می‌توانم VBA را در فایل‌های PPTM/PPSM مدیریت کنم؟**

بله. ارائه‌های حاوی ماکرو [پشتیبانی می‌شوند](/slides/fa/php-java/presentation-via-vba/)، و می‌توانید [پروژه‌های VBA را بررسی و مدیریت کنید](https://reference.aspose.com/slides/fa/php-java/aspose.slides/presentation/getvbaproject/) در آن فایل‌ها.

**آیا می‌توانم PDF یا HTML را به اسلایدهای PowerPoint تبدیل کنم؟**

بله. می‌توانید [صفحات PDF یا محتوای HTML را وارد](/slides/fa/php-java/import-presentation/) کنید تا اسلایدها در یک ارائه ایجاد یا پر شوند.

**آیا صادرات XPS پشتیبانی می‌شود و می‌توانم کیفیت و محتوای خروجی XPS را کنترل کنم؟**

بله. [Export to XPS](/slides/fa/php-java/convert-powerpoint-to-xps/) در دسترس است و [گزینه‌های ذخیره]https://reference.aspose.com/slides/fa/php-java/aspose.slides/xpsoptions/ به شما امکان می‌دهد کیفیت خروجی و محتواهای گنجانده‌شده را تنظیم کنید.

**آیا می‌توانم اسلایدها را به تصویر تبدیل کنم و کیفیت خروجی را کنترل کنم؟**

بله. اسلایدها می‌توانند به [PNG, JPEG, GIF, BMP, TIFF](/slides/fa/php-java/convert-powerpoint-to-png/) و سایر فرمت‌ها رندر شوند و شما می‌توانید اندازه و کیفیت را به‌دقت تنظیم کنید.

**آیا فرمول‌های ریاضی (MathML/MathText) پشتیبانی می‌شوند و می‌توانند صادر شوند؟**

بله. می‌توانید [متن‌های ریاضی را ایجاد و ویرایش](/slides/fa/php-java/powerpoint-math-equations/) کنید و [معادلات را صادر](/slides/fa/php-java/exporting-math-equations/) کنید، از جمله به MathML.

**کتابخانه چطور رمزنگاری و رمزهای عبور (باز کردن، تنظیم، حذف) را مدیریت می‌کند؟**

[پشتیبانی می‌کند](/slides/fa/php-java/password-protected-presentation/) از باز کردن ارائه‌های رمزگذاری‌شده، تنظیم یا حذف رمز عبور برای باز کردن و ویرایش، و بررسی وضعیت محافظت یک فایل.
