## **مقدمه**

Aspose.Slides for PHP via Java نیازی به نصب هیچ محصول شخص ثالثی مانند Microsoft PowerPoint ندارد. خود Aspose.Slides یک موتور برای ایجاد، تغییر، تبدیل و رندرینگ اسناد در فرمت‌های مختلف، از جمله فرمت‌های ارائه Microsoft PowerPoint است.

## **سیستم‌عامل‌های پشتیبانی‌شده**

Aspose.Slides for Java از هر سیستم‌عامل 32‑bit یا 64‑bit که محیط اجرای Java را پشتیبانی می‌کند، پشتیبانی می‌کند، از جمله اما نه محدود به:

### **Windows**
- Microsoft Windows 2003 Server ( x64, x86)
- Microsoft Windows 2008 Server ( x64, x86)
- Microsoft Windows 2012 Server ( x64, x86)
- Microsoft Windows 2012 R2 Server ( x64, x86)
- Microsoft Windows 2016 Server ( x64, x86)
- Microsoft Windows 2019 Server ( x64, x86)
- Microsoft Windows Vista ( x64, x86)
- Microsoft Windows XP ( x64, x86)
- Microsoft Windows 7 ( x64, x86)
- Microsoft Windows 8, 8.1 ( x64, x86)
- Microsoft Windows 10 ( x64, x86)

### **Linux**
- Linux (Ubuntu, OpenSUSE, CentOS و دیگر توزیع‌ها)

### **Mac**
- Mac OS X

## **سوالات متداول**

**آیا برای تبدیل و رندرینگ نیاز به نصب Microsoft PowerPoint دارم؟**

نه، PowerPoint مورد نیاز نیست؛ Aspose.Slides یک موتور مستقل برای [ایجاد](/slides/fa/php-java/create-presentation/)، تغییر، [تبدیل](/slides/fa/php-java/convert-presentation/) و [نمایش](/slides/fa/php-java/convert-powerpoint-to-png/) ارائه‌ها است.

**برای رندرینگ صحیح به چه قلم‌هایی نیاز است؟**

در عمل، قلم‌های استفاده شده در ارائه یا [جایگزین‌های](/slides/fa/php-java/font-substitution/) مناسب باید در دسترس باشند. برای اطمینان از رندرینگ یکسان در Linux/macOS توصیه می‌شود بسته‌های قلم متداول نصب شوند.

**چرا یک قلم سفارشی در Linux به‌عنوان جایگزین یا متن گمشده نمایش داده می‌شود؟**

اگر فایل قلم دارای ورودی‌های جدول نام ناسازگار یا خراب باشد، پشتهٔ مطابقت قلم در Linux (FreeType/fontconfig) ممکن است رکورد نامعتبری را انتخاب کند و باعث شود قلم حل نشود. استفاده از نسخه‌ای از قلم با رکوردهای جدول نام اصلاح‌شده یا نصب یک جایگزین سازگار این مشکل را برطرف می‌کند.
