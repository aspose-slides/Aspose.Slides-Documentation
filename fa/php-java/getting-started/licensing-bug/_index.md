## **مرور کلی**

این مقاله یک مشکل مربوط به مجوز را که در نسخه‌های 23.2، 23.3، 23.4، 23.5، 23.6 و 23.7 Aspose.Slides for PHP via Java رخ داده است توصیف می‌کند. این مشکل باعث شد که مجوزهای دائمی پس از پایان دوره اشتراک موقتاً غیرقابل استفاده شوند.

## **علائم**

پس از پایان اعتبار اشتراک مجوز دائمی شما، نسخه‌های 23.2‑23.7 Aspose.Slides for PHP via Java ممکن است مجوز را به‌عنوان منقضی گزارش کنند و تمام عملکردها را به حالت ارزیابی منتقل کنند. این رفتار نادرست است و بر نسخه‌های قبل از 23.2 و پس از 23.7 تأثیری ندارد.

## **راه‌حل**

Aspose.Slides for PHP via Java این مشکل را برطرف کرد و نسخه‌های به‌روزرسانی‌شده (23.2.1، 23.3.1، 23.4.1، 23.5.1، 23.6.1، 23.7.1) را با اصلاح منتشر کرد.

اگر در پروژه خود از یکی از نسخه‌های تحت تأثیر Aspose.Slides for PHP via Java استفاده می‌کنید، لطفاً به نسخه‌ی اصلاح‌شده سوئیچ کنید.

اگر با هرگونه مشکل مجوزی در Aspose.Slides for PHP via Java مواجه شدید:

- نسخه‌ای که استفاده می‌کنید را بررسی کنید و اطمینان حاصل کنید که تحت تأثیر مشکل شرح‌داده‌شده در بالا نیست.
- برای مراحل رفع‌خطا و مشکلات شناخته‌شده مجوز به [مستندات Aspose](https://docs.aspose.com/slides/fa/php-java/getting-started/) مراجعه کنید.
- برای دریافت کمک بیشتر با [پشتیبانی Aspose](https://forum.aspose.com/) تماس بگیرید.

## **محصولات و نسخه‌های متاثر**

لطفاً توجه داشته باشید که این مشکل تنها بر نسخه‌های 23.2 تا 23.7 Aspose.Slides for PHP via Java تأثیر دارد. **سایر محصولات Aspose و سایر نسخه‌های Aspose.Slides تحت تأثیر نیستند**.
