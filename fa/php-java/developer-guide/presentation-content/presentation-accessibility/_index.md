## **نمای کلی**

دسترس‌پذیری ارائه‌ها اطمینان می‌دهد که افرادی که از فناوری‌های کمکی—مانند خوانندگان صفحه، نمایشگرهای بریل یا ناوبری فقط با صفحه‌کلید—استفاده می‌کنند، می‌توانند اسلایدهای شما را به همان اندازه که مخاطبان بینایی‌دار و با موس می‌بینند، درک و مرور کنند. شیوه‌های خوب بر ترتیب واضح خواندن، متن جایگزین معنادار برای نمودارهای اطلاعاتی، تضاد کافی رنگ، تایپوگرافی قابل خواندن، متن لینک توصیفی و جلوگیری از انتقال معنا تنها با رنگ یا موقعیت تمرکز دارند. وقتی دسترس‌پذیری از ابتدا برنامه‌ریزی می‌شود، نتیجه ساختار تمیزتر، گرافیک‌های سازگارتر و محتوایی است که بدون نیاز به راه‌حل‌های جایگزین به هر بیننده‌ای می‌رسد.

## **علامت‌گذاری به‌عنوان تزئینی**

علامت‌گذاری به‌عنوان تزئینی، تصاویر صرفاً تزئینی را پرچم‌گذاری می‌کند تا خوانندگان صفحه از آن‌ها عبور کنند، نویز را کاهش داده و تمرکز را بر محتوای معنادار حفظ می‌کند. این پرچم را به پس‌زمینه‌ها، تزئینات و فضا‌سازها اعمال کنید—هرگز به نمودارها، آیکون‌ها یا تصاویری که اطلاعات منتقل می‌کنند. Aspose.Slides این پرچم را برای تشخیص و اعتبارسنجی فراهم می‌کند و امکان بررسی‌های خودکار دسترس‌پذیری و پاک‌سازی را میسر می‌سازد.

![علامت‌گذاری به‌عنوان تزئینی](mark_as_decorative.png)

کد نمونه زیر نحوه تعیین اینکه آیا یک شکل به‌عنوان تزئینی علامت‌گذاری شده است را نشان می‌دهد.

```php
$presentation = new Presentation("sample.pptx");
try {
    $shape = $presentation->getSlides()->get_Item(0)->getShapes()->get_Item(0);
    echo "Is shape decorative: " . ($shape->isDecorative() ? "true" : "false") . "\n";
} finally {
    $presentation->dispose();
}
```
