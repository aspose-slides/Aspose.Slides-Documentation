## **تبدیل PPT به PPTX**

Aspose.Slides برای PHP از طریق Java اکنون به توسعه‌دهندگان امکان دسترسی به PPT را با استفاده از نمونهٔ کلاس [Presentation](https://reference.aspose.com/slides/fa/php-java/aspose.slides/presentation) و تبدیل آن به قالب مربوطهٔ [PPTX](https://docs.fileformat.com/presentation/pptx/) می‌دهد. در حال حاضر، تبدیل جزئی [PPT ](https://docs.fileformat.com/presentation/ppt/)به PPTX را پشتیبانی می‌کند. برای جزئیات بیشتر دربارهٔ ویژگی‌های پشتیبانی‌شده و پشتیبانی‌نشده در تبدیل PPT به PPTX، لطفاً به این مستندات [لینک](/slides/fa/php-java/ppt-to-pptx-conversion/) مراجعه کنید.

Aspose.Slides برای PHP از طریق Java کلاس [Presentation](https://reference.aspose.com/slides/fa/php-java/aspose.slides/presentation) را ارائه می‌دهد که نمایانگر یک فایل ارائهٔ **PPTX** است. کلاس Presentation اکنون می‌تواند همچنین به **PPT** از طریق Presentation دسترسی پیدا کند هنگامی که شیء نمونه‌سازی می‌شود. مثال زیر نشان می‌دهد چگونه یک ارائهٔ PPT را به ارائهٔ PPTX تبدیل کنیم.

```php
  # یک شی Presentation ایجاد کنید که نمایانگر یک فایل PPTX است
  $pres = new Presentation("Aspose.ppt");
  try {
    # ذخیره ارائهٔ PPTX در قالب PPTX
    $pres->save("ConvertedAspose.pptx", SaveFormat::Pptx);
  } finally {
    if (!java_is_null($pres)) {
      $pres->dispose();
    }
  }
```
