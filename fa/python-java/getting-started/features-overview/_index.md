## **پلتفرم‌های پشتیبانی‌شده**
Aspose.Slides for Python via Java یک API مستقل از پلتفرم است. می‌تواند بر روی هر پلتفرمی (Windows، Linux، MacOS و غیره) که Python 3.7 یا بالاتر نصب شده باشد، استفاده شود. JDK 8 یا بالاتر یک نیاز دیگر است.

## **قالب‌های فایل و تبدیل‌ها**
Aspose.Slides for Python via Java اکثر فرمت‌های سند PowerPoint را پشتیبانی می‌کند. همچنین به شما امکان می‌دهد آنها را به فرمت‌های پرطرفدار فایل که سازمان‌ها استفاده و بین خود به اشتراک می‌گذارند، صادر کنید. به جزئیات زیر مراجعه کنید:

|**Feature**|**Description**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Python via Java سریع‌ترین پردازش را برای این فرمت سند ارائه می‌دهد.|
|تبدیل PPT به PPTX|Aspose.Slides for Python via Java از تبدیل PPT به PPTX پشتیبانی می‌کند.|
|قالب سند قابل حمل (PDF)|می‌توانید تمام فرمت‌های فایل پشتیبانی‌شده را با یک متد به اسناد Adobe Portable Document Format (PDF) صادر کنید.|
|XML Parser Specification (XPS)|می‌توانید تمام فرمت‌های فایل پشتیبانی‌شده را با یک متد به اسناد XML Parser Specification (XPS) صادر کنید.|
|Tagged Image File Format (TIFF)|می‌توانید تمام فرمت‌های فایل ارائه پشتیبانی‌شده را به Tagged Image File Format (TIFF) صادر کنید.|
|تبدیل PPTX به HTML|Aspose.Slides for Python via Java از تبدیل PresentationEx به فرمت HTML پشتیبانی می‌کند.|

## **رندرینگ و چاپ**
Aspose.Slides for Python via Java رندرینگ با دقت بالا از اسلایدهای ارائه به فرمت‌های گرافیکی مختلف را پشتیبانی می‌کند. به جزئیات زیر مراجعه کنید:

|**Feature**|**Description**|
| :- | :- |
|فرمت‌های تصویر پشتیبانی‌شده توسط Java|با استفاده از Aspose.Slides for Python via Java، می‌توانید اسلایدهای ارائه و تصاویر روی اسلایدها را به تمام فرمت‌های گرافیکی پشتیبانی‌شده توسط Java مانند TIFF، PNG، BMP، JPEG و GIF رندر کنید.|
|فرمت SVG|Aspose.Slides for Python via Java همچنین روش‌های داخلی ارائه می‌دهد که به شما امکان صادر کردن اسلایدهای ارائه به فرمت Scalable Vector Graphics (SVG) را می‌دهد.|
|چاپ ارائه|نسخه‌های جدید Aspose.Slides for Python via Java روش‌های چاپ داخلی با گزینه‌های مختلف را فراهم می‌کنند.|
