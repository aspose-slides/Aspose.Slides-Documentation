## **شرح محصول**

Aspose.Slides for Python via Java یک کتابخانه‌ی کلاس قابل‌کاربرد بر روی چندین پلتفرم است که به برنامه‌های شما امکان می‌دهد تا انواع کارهای پردازش سند را بدون استفاده از Microsoft PowerPoint® انجام دهند.

### **چرا باید Aspose.Slides for Python via Java را استفاده کنید**

- Aspose.Slides for Python via Java اولین و تنها مؤلفه‌ای است که قابلیت‌های لازم برای مدیریت اسناد PowerPoint® را فراهم می‌کند.  
- Aspose.Slides for Python via Java ویژگی‌های کلیدی متعددی مانند مدیریت متن، اشکال، جداول و انیمیشن‌ها، افزودن صدا و ویدئو به اسلایدها، پیش‌نمایش اسلایدها، خروجی اسلایدها به SVG، PDF و سایر فرمت‌ها و بسیاری قابلیت‌های دیگر را ارائه می‌دهد.  
- Aspose.Slides for Python via Java نه تنها امکان باز کردن فایل‌های PowerPoint از منابع مختلف را می‌دهد، بلکه به شما اجازه می‌دهد ارائه‌های خود را به روش‌های گوناگون ذخیره کنید. به‌عنوان مثال می‌توانید ارائه‌ها را به فرمت PPT ذخیره کنید؛ همچنین می‌توانید اسلایدهای خود را به‌صورت تصویر ذخیره کنید.  
- Aspose.Slides for Python via Java تقریباً تمام ویژگی‌هایی را که ممکن است در Microsoft PowerPoint پیدا یا نیابید، در اختیار شما می‌گذارد. با توجه به انعطاف‌پذیری Aspose.Slides for Python via Java، علاوه بر جداول و انواع مختلف اشکال، می‌توانید فریم‌های متن، صدا و ویدئوی مختلفی را به اسلایدهای خود اضافه کنید.  

## **ویژگی‌های Aspose.Slides**

ویژگی‌های Aspose.Slides for Python via Java می‌توانند به این گروه‌ها تقسیم شوند:

- ویژگی‌های ارائه  
- ویژگی‌های اسلایدها  
- ویژگی‌های گرافیک و مولتی‌مدیا  
- ویژگی‌های جدول  
- ویژگی‌های متن و پاراگراف  
- ویژگی‌های پیشرفته  

### **ویژگی‌های ارائه**

- باز کردن ارائه‌های Microsoft PowerPoint از منابع مختلف  
- ذخیره ارائه‌ها در فرمت‌های مختلف Microsoft PowerPoint  
- ایجاد ارائه‌های جدید از صفر  
- تبدیل ارائه‌ها به اسناد PDF بدون استفاده از هیچ مؤلفه‌ی دیگری  
- خواندن یا تغییر ویژگی‌های سند ارائه  
- اعمال محافظت بر روی ارائه  
- خروجی ارائه به PDF  
- خروجی ارائه به TIFF  
- خروجی ارائه به XPS  
- خروجی ارائه به SWF  
- خروجی ارائه به PDF Note / Tiff Notes  
- خروجی ارائه به HTML  

### **ویژگی‌های اسلایدها**

- ایجاد اسلایدهای جدید از صفر و افزودن آن‌ها به ارائه‌ها  
- دسترسی به هر اسلاید بر اساس موقعیت آن در ارائه  
- کپی یا کلون کردن اسلایدها به همان ارائه یا به ارائه‌ای دیگر  
- ویرایش و حذف اسلایدها  
- کپی خودکار اسلایدهای اصلی همراه با اسلایدها  
- پیش‌نمایش اسلایدها  
- دریافت یا تنظیم نام یک اسلاید  
- ذخیره اسلایدها به‌صورت تصویر  
- خروجی اسلایدها به فرمت SVG  
- کار با تعداد زیادی از اسلایدهای اصلی در ارائه‌ها  
- پر کردن پس‌زمینه اسلایدها با هر سبک پشتیبانی‌شده  
- افزودن یا مدیریت یادداشت‌های اسلایدها  
- تنظیم پویا اسلایدهای اصلی و سبک‌ها  
- افزودن یا تغییر انتقال اسلاید쇼  
- افزودن انتقال Morph  

### **ویژگی‌های گرافیک و مولتی‌مدیا**

- ایجاد یا مدیریت اشکالی مانند خطوط، مستطیل‌ها، بیضی‌ها و پلی‌لاین‌ها  
- مدیریت سبک خطوط در اشکال  
- دریافت متن جایگزین یک شکل  
- چرخاندن یا وارونه‌کردن اشکال  
- پر کردن اشکال با سبک‌های مختلف — برای مثال، ثابت، گرادیان، الگو، بافت و تصویر  
- استفاده از هر یک از ۴۸ الگوی پیش‌تعریف‌شده و ۲۴ بافت  
- اختصاص لینک‌ها به اشکال  
- خواندن، تغییر و افزودن اثرات انیمیشن جدید به اسلایدها و اشکال  
- افزودن تصویرهای جدید به ارائه‌ها و مدیریت تصویرهای موجود  
- افزودن گلوله‌های تصویری جدید به ارائه‌ها و مدیریت گلوله‌های موجود  
- افزودن فریم‌های صوتی جدید به ارائه‌ها و مدیریت فریم‌های صوتی موجود  
- افزودن فریم‌های ویدئویی جدید به ارائه‌ها و مدیریت فریم‌های ویدئویی موجود  

### **ویژگی‌های جدول**

- ایجاد جداول از صفر و افزودن آن‌ها به اسلایدها  
- تغییر اندازه و جابجایی جداول  
- دسترسی به هر سلول در جدول  
- تغییر قالب و حاشیه‌های هر سلول  
- تغییر اندازه ستون‌ها و ردیف‌ها  
- افزودن و حذف ستون‌ها و ردیف‌ها  
- تنظیم عرض و رنگ حاشیه برای کل جدول  

### **ویژگی‌های متن و پاراگراف**

- دریافت و تنظیم متون در جای‌دارهای متن و فریم‌های متنی  
- دریافت نوع یک جای‌دار  
- تغییر قالب‌بندی متن در یک فریم متنی  
- افزودن قلم‌های جدید یا مدیریت قلم‌های پیشاپیش استفاده‌شده در ارائه  
- تغییر متون و قالب‌بندی متن  
- ایجاد یا مدیریت لینک‌ها به متن  
- پشتیبانی از تب‌ها و تب‌های مؤثر  
- افزودن فریم‌های متنی جدید به تقریباً هر نوع شکلی  
- دسترسی و تغییر حاشیه‌ها و تنظیمات پیچش متن در فریم‌های متنی  
- دسترسی و تغییر تو رفتگی پاراگراف  

### **ویژگی‌های پیشرفته**

- دسترسی به اشیاء OLE که در ارائه‌ها جاسازی شده‌اند  
- قفل کردن ارائه‌ها  

## **پشتیبانی فنی**

Aspose پشتیبانی فنی نامحدود و رایگان برای تمام محصولات خود ارائه می‌دهد. این پشتیبانی برای تمام کاربران (از جمله کاربران با بسته‌های ارزیابی) در دسترس است. اگر به کمک برای Aspose.Slides for Python via Java نیاز دارید، موارد زیر را در نظر بگیرید:

- مسیر اصلی برای پشتیبانی، [Aspose Forums](https://forum.aspose.com/) است. سؤال خود را در [Aspose.Slides forum](https://forum.aspose.com/c/slides/fa/11) مطرح کنید — و پاسخ آن ظرف چند ساعت خواهد بود. تیم پشتیبانی Aspose.Slides به‌صورت مستقیم به سؤال‌های ارسال‌شده در انجمن پاسخ می‌دهد.  
- لطفاً توجه داشته باشید که Aspose پشتیبانی فنی از طریق تلفن ارائه نمی‌دهد. پشتیبانی تلفنی فقط برای سوالات فروش و خرید در دسترس است.  
- هنگام انتظار برای پاسخ در انجمن‌ها، لطفاً صبور باشید و اختلافات منطقه زمانی را در نظر بگیرید.  

اگر با Aspose.Slides for Python via Java مشکلی دارید، برای رفع آن به کارآمدترین شکل ممکن این راهنمایی‌ها را دنبال کنید:

- ابتدا بررسی کنید و تأیید کنید که از آخرین نسخه Aspose.Slides for Python via Java استفاده می‌کنید قبل از گزارش مشکل. شاید مشکلی که تجربه کردید در به‌روزرسانی‌ای حل شده باشد.  

  برای دریافت اطلاعات درباره آخرین نسخه محصول، به [Aspose.Slides download](https://releases.aspose.com/slides/fa/python-java/) مراجعه کنید.  

- پیش از گزارش مشکل، ممکن است مایل باشید انجمن‌های ما، این مستندات و مرجع API را مرور کنید. شاید پاسخ سؤال شما قبلاً ارائه شده باشد.  

- هنگام گزارش مشکل، لطفاً سند اصلی و (در صورت امکان) بخشی از کد مربوط به مسئله را ضمیمه کنید. اگر نیاز به پیوست چندین فایل دارید، آن‌ها را در یک بسته ZIP قرار دهید.  

  می‌توانید اسناد خود را با اطمینان در Aspose.Forums پیوست کنید؛ زیرا فقط شما و توسعه‌دهندگان Aspose به فایل‌های پیوست‌شده دسترسی دارند.  

- برای یک مشکل خاص، یک رشته (thread) ایجاد و استفاده کنید. اگر مشکل دیگری پیش آمد، باید رشته‌ای دیگر برای گزارش آن ایجاد کنید.
