## **پشتیبانی فنی**
{{% alert color="primary" %}}

Aspose برای تمام محصولات خود پشتیبانی فنی رایگان و نامحدود ارائه می‌دهد. این پشتیبانی برای همه کاربران قابل دسترسی است (از جمله کاربرانی که بسته‌های ارزیابی دارند).

{{% /alert %}} 

زمانی که به کمک برای Aspose.Slides for Python via Java نیاز دارید، موارد زیر را در نظر بگیرید:

- مسیر اصلی پشتیبانی، [Aspose.Forums](https://forum.aspose.com/c/slides/fa/11) است. سؤال خود را در فروم Aspose.Slides ارسال کنید—و در عرض چند ساعت پاسخ دریافت خواهید کرد. تیم پشتیبانی Aspose.Slides به صورت مستقیم به سؤالاتی که در فروم ارسال می‌شوند، پاسخ می‌دهد.
- لطفاً توجه داشته باشید که Aspose پشتیبانی فنی از طریق تلفن ارائه نمی‌دهد. پشتیبانی تلفنی فقط برای سوالات فروش و خرید در دسترس است.
- هنگام انتظار برای پاسخ در فروم‌ها، لطفاً صبور باشید و اختلافات زمانی را در نظر بگیرید.

زمانی که با مشکلی در Aspose.Slides for Python via Java مواجه می‌شوید، برای حل آن به کارآمدترین شکل، این راهنما را دنبال کنید:

- پیش از گزارش مشکل، بررسی و تأیید کنید که از آخرین نسخه Aspose.Slides for Python via Java استفاده می‌کنید. ممکن است مشکلی که تجربه کردید، در یک به‌روزرسانی حل شده باشد.

  برای دریافت اطلاعات درباره آخرین نسخه، به [Aspose.Slides download](https://releases.aspose.com/slides/fa/python-java/) مراجعه کنید.

- پیش از گزارش مشکل، ممکن است بخواهید در فروم‌های ما، این مستندات، و مرجع API جستجو کنید. شاید پاسخ سؤال شما قبلاً ارائه شده باشد.

- هنگام گزارش مشکل، لطفاً سند اصلی و (در صورت امکان) بخشی از کد مربوط به مشکل را ضمیمه کنید. اگر نیاز به پیوست چندین فایل دارید، آن‌ها را در یک بسته ZIP قرار دهید. می‌توانید اسناد خود را به‌صورت ایمن در Aspose.Forums پیوست کنید، زیرا **فقط** شما و توسعه‌دهندگان Aspose به فایل‌های پیوست شده دسترسی دارند.

- برای هر مشکل یک موضوع (Thread) ایجاد کنید و از آن استفاده کنید. اگر مشکل دیگری دارید، باید یک موضوع دیگر ایجاد کنید تا آن را گزارش دهید.
