{{% alert color="primary" %}}

Aspose.Slides for Node.js via Java یک کتابخانه کلاسی است که به برنامه‌های Node.js امکان خواندن و نوشتن اسناد ارائه را بدون استفاده از Microsoft PowerPoint® یا سایر برنامه‌ها می‌دهد. Aspose.Slides for Node.js via Java به شما امکان مدیریت متن، اشکال، جدول‌ها و انیمیشن‌ها، افزودن صدا و ویدئو به اسلایدها، پیش‌نمایش اسلایدها، خروجی گرفتن اسلایدها به فرمت SVG، PDF و کارهای بسیار دیگر را می‌دهد.

آخرین مرجع API در [Aspose.Slides for Node.js via Java API Reference](https://reference.aspose.com/slides/fa/nodejs-java/) موجود است.
