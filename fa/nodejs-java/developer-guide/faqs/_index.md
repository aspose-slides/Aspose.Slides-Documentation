## **بررسی کلی**

این پرسش‌وپاسخ پاسخ‌های سوالات رایج درباره Aspose.Slides را ارائه می‌دهد. این مستند فرمت‌های فایل پشتیبانی‌شده، مدیریت استثناها هنگام کار با ارائه‌های بزرگ، تغییر اندازه اسلایدها، پیش‌نمایش اسلایدها، استخراج متن از ارائه‌ها، قالب‌بندی مرزهای جدول، قرار دادن تصاویر و رفع مشکلات مربوط به قلم‌ها هنگام تبدیل ارائه‌ها به PDF یا تصاویر را پوشش می‌دهد.

## **فرمت‌های فایل پشتیبانی‌شده**

**Q:** کدام فرمت‌های فایل توسط Aspose.Slides برای Node.js از طریق Java پشتیبانی می‌شود؟  

**A:** Aspose.Slides برای Node.js از طریق Java از فرمت‌های فایلی که در [Supported File Formats](/slides/fa/nodejs-java/supported-file-formats/) توضیح داده شده‌اند، پشتیبانی می‌کند.

## **استثناها**

**Q:** در هنگام بارگذاری یک فایل PPT بزرگ حاوی تصاویر، استثنای کمبود حافظه دریافت می‌کنم. آیا محدودیتی در Aspose.Slides نسبت به اندازه فایل وجود دارد؟  

**A:** فرمول خاصی برای محاسبهٔ اندازهٔ ارائه‌ای که توسط Aspose.Slides پشتیبانی می‌شود وجود ندارد. باید فضای کافی برای نگهداری کل ساختار ارائه و تصاویر در حافظه وجود داشته باشد. به طور معمول، تصاویر در حافظه فضای بیشتری نسبت به دیسک سخت اشغال می‌کنند، به‌ویژه وقتی که تصاویر دارای افکت‌های اضافی باشند.

به طور کلی، Aspose.Slides برای Node.js از طریق Java می‌تواند به راحتی فایل‌های ارائه حدود 300 مگابایت را روی سروری با 4 گیگابایت RAM مدیریت کند.

## **کار با اسلایدها**

**Q:** می‌توانم اندازهٔ اسلایدها در یک ارائه را تغییر دهم؟  

**A:** می‌توانید از متد `getSlideSize` که توسط کلاس [Presentation](https://reference.aspose.com/slides/fa/nodejs-java/aspose.slides/presentation/) ارائه شده است، برای تعریف اندازهٔ اسلایدها در یک ارائه استفاده کنید.

**Q:** آیا راهی برای تعریف اسلایدهای با اندازه‌های مختلف در یک ارائه وجود دارد؟  

**A:** از آنجا که اندازهٔ اسلایدها در سطح ارائه در اسناد Microsoft PowerPoint تعریف می‌شود، راهی برای انجام این کار وجود ندارد.

**Q:** آیا Aspose.Slides برای Node.js از طریق Java پیش‌نمایش یک اسلاید قبل از ذخیره‌سازی را پشتیبانی می‌کند؟  

**A:** می‌توانید اسلایدهای ارائه را به تصویر رندر کنید و از این تصاویر برای پیش‌نمایش اسلایدها استفاده کنید.

## **کار با متن**

**Q:** آیا امکان استخراج تمام متن از یک ارائه وجود دارد؟  

**A:** Aspose.Slides برای Node.js از طریق Java کلاس [SlideUtil](https://reference.aspose.com/slides/fa/nodejs-java/aspose.slides/slideutil/) را فراهم می‌کند که روش‌های مختلفی برای استخراج متن کامل از ارائه‌ها ارائه می‌دهد.

**Q:** چرا اندازه‌های پاراگراف در سیستم‌های عامل ویندوز و لینوکس متفاوت است؟  

**A:** محاسبهٔ اندازهٔ پاراگراف بر پایهٔ اندازهٔ متنی است که آن پاراگراف را نشان می‌دهد. اندازهٔ متن بر اساس معیارهای فونتی است که در ارائه PowerPoint مشخص شده است. اگر فونت مشخص شده موجود نباشد، با مشابه‌ترین فونت جایگزین می‌شود، اما این فونت معیارهای متفاوتی نسبت به فونت اصلی دارد. در نتیجه، محاسبهٔ اندازهٔ پاراگراف در سیستم‌های مختلف به نتایج متفاوت منجر می‌شود که بسته به مجموعهٔ فونت‌های نصب شده متفاوت است. برای دستیابی به همان نتیجه در سیستم‌عامل‌های مختلف، باید همان فونت‌ها را در سیستم‌ها نصب کنید یا آنها را در زمان اجرا به‌عنوان [external fonts](/slides/fa/nodejs-java/custom-font/) بارگذاری کنید.

## **قالب‌بندی و تصاویر**

**Q:** چگونه می‌توانم رنگ مرز جدول را تنظیم کنم؟  

**A:** می‌توانید رنگ تمام مرزهای جدول یا فقط مرز دور کل جدول را تغییر دهید. برای تغییر تمام مرزها، لطفاً از متد `getCellFormat` در کلاس [Cell](https://reference.aspose.com/slides/fa/nodejs-java/aspose.slides/cell/) استفاده کنید. برای مرز کل جدول، باید بر روی سلول‌ها تکرار کنید و رنگ مرزهای خارجی را تغییر دهید.

**Q:** Aspose.Slides برای Node.js از طریق Java از چه واحدی برای قرار دادن تصاویر استفاده می‌کند؟  

**A:** مختصات و ابعاد تمام اشکال روی اسلایدها به واحد نقطه (72 dpi) اندازه‌گیری می‌شوند.

## **کار با فونت‌ها**

**Q:** هنگام تبدیل PPT به PDF یا تصاویر، چرا فونت‌های مستندات خروجی متفاوت هستند؟  

**A:** این مشکل ممکن است نشان دهد که فونت‌های استفاده‌شده در ارائه در سیستم‌عامل اجرا کنندهٔ کد موجود نیستند. باید فونت‌ها را بر روی سیستم‌عامل نصب کنید یا آنها را به‌عنوان فونت‌های خارجی با استفاده از کلاس [FontsLoader](https://reference.aspose.com/slides/fa/nodejs-java/aspose.slides/fontsloader/) بارگذاری کنید همان‌طور که در زیر نشان داده شده است:
```javascript
var folders = java.newArray("java.lang.String", ["path_to_a_folder_with_fonts"]));
java.callStaticMethodSync("com.aspose.slides.FontsLoader", "loadExternalFonts", folders);
```
