## **بررسی کلی**

دسترس‌پذیری ارائه اطمینان می‌دهد که افرادی که از فناوری‌های کمکی—مانند خواننده‌های صفحه‌نمایش، نمایشگرهای بریل یا ناوبری فقط با صفحه‌کلید—استفاده می‌کنند، می‌توانند اسلایدهای شما را همانند مخاطبان بینایی‌دار و با ماوس به‌طور مؤثر درک و مرور کنند. روش‌های خوب بر روی ترتیب خواندن واضح، متن جایگزین معنادار برای تصاویر اطلاعاتی، تضاد رنگ کافی، تایپوگرافی خوانا، متن لینک توصیفی و جلوگیری از انتقال معنا فقط با رنگ یا موقعیت تمرکز می‌کند. وقتی دسترس‌پذیری از ابتدا برنامه‌ریزی شود، نتیجه ساختاری تمیزتر، تصاویری سازگارتر و محتوایی است که بدون راه‌حل‌های جانبی به هر بیننده‌ای می‌رسد.

## **علامت به عنوان تزئینی**

نشانه‌گذاری به عنوان تزئینی برای جلوه‌های صرفاً ornamental مورد استفاده قرار می‌گیرد تا خواننده‌های صفحه‌نمایش آن‌ها را نادیده بگیرند، باعث کاهش نویز و حفظ تمرکز بر محتوای معنادار می‌شود. این را می‌توان بر روی پس‌زمینه‌ها، تزئینات و فضاهای خالی اعمال کرد—هرگز برای نمودارها، نمادها یا تصاویری که اطلاعاتی منتقل می‌کنند. Aspose.Slides این نشانه را برای تشخیص و اعتبارسنجی فراهم می‌کند و امکان بررسی‌های خودکار دسترس‌پذیری و پاکسازی را می‌دهد.

![علامت به عنوان تزئینی](mark_as_decorative.png)

نمونه کد زیر نشان می‌دهد چگونه می‌توان تشخیص داد آیا یک شکل به عنوان تزئینی علامت‌گذاری شده است یا خیر.

```js
let presentation = new aspose.slides.Presentation("sample.pptx");
try {
    let shape = presentation.getSlides().get_Item(0).getShapes().get_Item(0);
    console.log("Is shape decorative:", shape.isDecorative());
} finally {
    presentation.dispose();
}
```
