## **تبدیل PPT به PPTX**
Aspose.Slides برای Node.js از طریق Java اکنون به توسعه‌دهندگان امکان دسترسی به PPT را با استفاده از کلاس [Presentation](https://reference.aspose.com/slides/fa/nodejs-java/aspose.slides/presentation) می‌دهد و این امکان را فراهم می‌کند که آن را به فرمت مربوطهٔ [PPTX](https://docs.fileformat.com/presentation/pptx/) تبدیل کنند. در حال حاضر، تبدیل جزئی [PPT ](https://docs.fileformat.com/presentation/ppt/)به PPTX را پشتیبانی می‌کند.

Aspose.Slides برای Node.js از طریق Java کلاس [Presentation](https://reference.aspose.com/slides/fa/nodejs-java/aspose.slides/presentation) را ارائه می‌دهد که نمایانگر یک فایل ارائه **PPTX** است. کلاس Presentation اکنون می‌تواند به **PPT** نیز از طریق Presentation دسترسی پیدا کند هنگامی که شیء ساخته می‌شود. مثال زیر نشان می‌دهد چگونه یک ارائه PPT را به ارائه PPTX تبدیل کنیم.

```javascript
// یک شی Presentation که نمایانگر یک فایل PPTX است را ایجاد می‌کند
var pres = new aspose.slides.Presentation("Aspose.ppt");
try {
    // ذخیره ارائه PPTX به فرمت PPTX
    pres.save("ConvertedAspose.pptx", aspose.slides.SaveFormat.Pptx);
} finally {
    if (pres != null) {
        pres.dispose();
    }
}
```
