## **بررسی کلی**

این مقاله توضیح می‌دهد چگونه اسلایدهای ارائه را به‌عنوان تصاویر SVG با استفاده از Aspose.Slides رندر کنید. این مقاله فرمت SVG و مزایای آن را توصیف می‌کند، از جمله مقیاس‌پذیری، دسترسی‌پذیری و مناسب بودن برای توسعه وب.

شما خواهید آموخت چگونه یک فایل ارائه را بارگذاری کنید، اسلایدهای آن را پیمایش کنید و هر اسلاید را به عنوان یک فایل SVG جداگانه ذخیره کنید. مقاله فرمت‌های ارائه PowerPoint و OpenDocument شامل PPT، PPTX، ODP و PPS را پوشش می‌دهد و نشان می‌دهد چگونه تبدیل را به‌صورت برنامه‌نویسی با کلاس `Presentation` و متد `writeAsSvg` انجام دهید.

## **قالب SVG**

SVG—مخفف Scalable Vector Graphics—یک نوع یا فرمت گرافیکی استاندارد برای رندر تصاویر دو‑بعدی است. SVG تصاویر را به‌صورت بردار در XML ذخیره می‌کند و جزئیاتی که رفتار یا ظاهر آنها را تعریف می‌کند، در بر دارد.

SVG یکی از معدود فرمت‌هایی است که در این موارد استانداردهای بسیار بالایی دارد: مقیاس‌پذیری، تعامل‌پذیری، کارایی، دسترسی‌پذیری، قابلیت برنامه‌نویسی و غیره. به همین دلایل، در توسعه وب به‌طور گسترده استفاده می‌شود.

شاید بخواهید از فایل‌های SVG زمانی استفاده کنید که نیاز به:

- **چاپ ارائه خود در *فرمت بسیار بزرگ***. تصاویر SVG می‌توانند به هر وضوح یا سطحی مقیاس شوند. می‌توانید تصاویر SVG را هر چند بار که لازم باشد بدون افت کیفیت تغییر اندازه دهید.
- **استفاده از نمودارها و گراف‌ها از اسلایدهای خود در *وسایل یا پلتفرم‌های مختلف***. بیشتر خوانندگان می‌توانند فایل‌های SVG را تفسیر کنند.
- **استفاده از *کوچک‌ترین اندازه ممکن برای تصاویر***. فایل‌های SVG معمولاً کوچکتر از معادل‌های با وضوح بالا در فرمت‌های دیگر، به‌خصوص فرمت‌های مبتنی بر بیت‌مپ (JPEG یا PNG) هستند.

## **رندر کردن اسلایدها به‌عنوان تصاویر SVG**

Aspose.Slides for Node.js via Java به شما امکان می‌دهد اسلایدهای ارائه خود را به‌عنوان تصاویر SVG صادر کنید. مراحل زیر را برای تولید تصاویر SVG دنبال کنید:

1. یک نمونه از کلاس `Presentation` ایجاد کنید.
2. تمام اسلایدهای موجود در ارائه را پیمایش کنید.
3. هر اسلاید را از طریق `FileOutputStream` به فایل SVG مخصوص خودش بنویسید.

{{% alert color="primary" %}} 
ممکن است بخواهید [برنامه وب رایگان ما](https://products.aspose.app/slides/fa/conversion/ppt-to-svg) را امتحان کنید که در آن عملکرد تبدیل PPT به SVG از Aspose.Slides برای Node.js via Java را پیاده‌سازی کرده‌ایم.
{{% /alert %}} 

این کد نمونه در JavaScript نشان می‌دهد چگونه با Aspose.Slides PPT را به SVG تبدیل کنید:

```javascript
var pres = new aspose.slides.Presentation("pres.pptx");
try {
    for (var index = 0; index < pres.getSlides().size(); index++) {
        var slide = pres.getSlides().get_Item(index);
        var fileStream = java.newInstanceSync("java.io.FileOutputStream", ("slide-" + index) + ".svg");
        try {
            slide.writeAsSvg(fileStream);
        } finally {
            fileStream.close();
        }
    }
} catch (e) {console.log(e);
} finally {
    if (pres != null) {
        pres.dispose();
    }
}
```

## **پرسش‌های متداول**

**چرا SVG حاصل می‌تواند در مرورگرهای مختلف متفاوت دیده شود؟**  
پشتیبانی از ویژگی‌های خاص SVG توسط موتورهای مرورگر به‌صورت متفاوتی پیاده‌سازی می‌شود. پارامترهای [SVGOptions](https://reference.aspose.com/slides/fa/nodejs-java/aspose.slides/svgoptions/) به رفع ناسازگاری‌ها کمک می‌کنند.

**آیا امکان صادرات نه تنها اسلایدها بلکه اشکال منفرد به SVG نیز وجود دارد؟**  
بله. هر [شکل می‌تواند به عنوان یک SVG جداگانه ذخیره شود](https://reference.aspose.com/slides/fa/nodejs-java/aspose.slides/shape/writeassvg/)، که برای آیکون‌ها، تصویرهای گرافیکی و استفاده مجدد از گرافیک‌ها مناسب است.

**آیا می‌توان چندین اسلاید را به یک SVG (نوار/سند) ترکیب کرد؟**  
سناریوی استاندارد یک اسلاید → یک SVG است. ترکیب چندین اسلاید در یک بوم SVG یکی از مراحل پس پردازشی است که در سطح برنامه انجام می‌شود.
