## **پلتفرم‌های پشتیبانی‌شده**
Aspose.Slides for Node.js via Java یک API مستقل از پلتفرم است. می‌تواند بر روی هر پلتفرمی (Windows، Linux، MacOS و غیره) که Node.js نسخه ۷ یا بالاتر نصب شده باشد، استفاده شود. Oracle JDK نسخه ۸ یا بالاتر نیز یک نیاز دیگر است.

## **قالب‌های فایل و تبدیل‌ها**
Aspose.Slides for Node.js via Java اکثر قالب‌های سند PowerPoint را پشتیبانی می‌کند. همچنین امکان صادرات آن‌ها به قالب‌های فایل محبوبی که سازمان‌ها استفاده و به اشتراک می‌گذارند را فراهم می‌کند. به جزئیات زیر نگاهی بیندازید:

|**ویژگی**|**توضیح**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via Java سریع‌ترین پردازش را برای این قالب سند ارائه می‌دهد.|
|تبدیل PPT به PPTX|Aspose.Slides for Node.js via Java تبدیل PPT به PPTX را پشتیبانی می‌کند.|
|قالب سند قابل حمل (PDF)|می‌توانید تمام قالب‌های فایل پشتیبانی‌شده را با یک روش به اسناد Adobe Portable Document Format (PDF) صادر کنید.|
|XML Parser Specification (XPS)|می‌توانید تمام قالب‌های فایل پشتیبانی‌شده را با یک روش به اسناد XML Parser Specification (XPS) صادر کنید.|
|Tagged Image File Format (TIFF)|می‌توانید تمام قالب‌های فایل ارائه پشتیبانی‌شده را به Tagged Image File Format (TIFF) صادر کنید.|
|تبدیل PPTX به HTML|Aspose.Slides for Node.js via Java تبدیل PresentationEx به قالب HTML را پشتیبانی می‌کند.|

## **رندرینگ و چاپ**
Aspose.Slides for Node.js via Java رندرینگ با دقت بالا از اسلایدهای ارائه به قالب‌های گرافیکی مختلف را پشتیبانی می‌کند. به جزئیات زیر نگاهی بیندازید:

|**ویژگی**|**توضیح**|
| :- | :- |
|قالب‌های تصویری پشتیبانی‌شده توسط Java|با استفاده از Aspose.Slides for Node.js via Java می‌توانید اسلایدهای ارائه و تصاویر موجود در اسلایدها را به تمام قالب‌های گرافیکی پشتیبانی‌شده توسط Java مانند TIFF، PNG، BMP، JPEG، GIF رندر کنید.|
|قالب SVG|Aspose.Slides for Node.js via Java همچنین روش‌های داخلی فراهم می‌کند که به شما اجازه می‌دهد اسلایدهای ارائه را به قالب Scalable Vector Graphics (SVG) صادر کنید.|
|چاپ ارائه|آخرین نسخه‌های Aspose.Slides for Node.js via Java روش‌های چاپ داخلی را با گزینه‌های مختلف فراهم می‌کنند.|

Aspose.Slides for Node.js via Java به شما امکان دسترسی، تغییر یا ایجاد تقریباً تمام آیتم‌ها یا محتویات اسناد ارائه را می‌دهد. به جزئیات زیر نگاهی بیندازید:

|**ویژگی**|**توضیح**|
| :- | :- |
|اسلایدهای مستر|اسلایدهای مستر چیدمان اسلایدهای عادی را تعریف می‌کنند. Aspose.Slides for Node.js via Java به شما اجازه دسترسی و اصلاح اسلایدهای مستر اسناد ارائه را می‌دهد.|
|اسلایدهای عادی|با Aspose.Slides for Node.js via Java می‌توانید اسلایدهای جدید از انواع مختلف ایجاد کنید؛ همچنین می‌توانید به اسلایدهای موجود در ارائه‌ها دسترسی پیدا کنید و آن‌ها را ویرایش کنید.|

|مدیریت بخش‌های اسلایدها|متدهایی برای سازماندهی اسلایدها در بخش‌های مختلف داخل یک ارائه.|

|یافتن یک شکل|می‌توانید یک شکل خاص را در یک اسلاید با استفاده از متن جایگزینی (alternative text) مرتبط با آن پیدا کنید.|

## **سوالات متداول**

**آیا برای کارکرد کتابخانه لازم است Microsoft PowerPoint را روی سرور/کامپیوتر نصب کنم؟**

خیر. PowerPoint نیازی نیست؛ Aspose.Slides یک موتور مستقل برای ایجاد، ویرایش، تبدیل و رندرینگ ارائه‌ها است.

**چگونه چندنخی (multithreading) کار می‌کند؟ آیا پردازش می‌تواند به صورت موازی انجام شود؟**

ایمن است که اسناد مختلف را در رشته‌های متفاوت پردازش کنید؛ شیء [ارائه](https://reference.aspose.com/slides/fa/nodejs-java/aspose.slides/presentation/) نباید همزمان توسط [چندین رشته](/slides/fa/nodejs-java/multithreading/) استفاده شود.

**آیا رمزهای عبور فایل و رمزنگاری پشتیبانی می‌شوند؟**

بله. [می‌توانید](/slides/fa/nodejs-java/password-protected-presentation/) ارائه‌های رمزگذاری‌شده را باز کنید، رمز عبور باز کردن و نوشتن را تنظیم یا حذف کنید، و وضعیت حفاظت را بررسی کنید.

**آیا نیاز است به بسته‌های قلم در کانتینرهای لینوکس توجه داشته باشم؟**

بله. توصیه می‌شود بسته‌های قلم عمومی را نصب کنید و/یا به‌طور صریح [دایرکتوری‌های قلم](/slides/fa/nodejs-java/custom-font/) را در برنامه خود مشخص کنید تا از تعویض‌های غیرمنتظره جلوگیری شود.

**آیا نسخه ارزیابی محدودیت‌هایی دارد؟**

در [حالت ارزیابی](/slides/fa/nodejs-java/licensing/) یک واترمارک به خروجی اضافه می‌شود و محدودیت‌های خاصی اعمال می‌شود؛ یک [مجوز موقت 30 روزه](https://purchase.aspose.com/temporary-license/) برای تست کامل ویژگی‌ها در دسترس است.

**آیا وارد کردن قالب‌های خارجی به یک ارائه (PDF/HTML → PPTX) پشتیبانی می‌شود؟**

بله. می‌توانید [صفحات PDF و محتوای HTML](/slides/fa/nodejs-java/import-presentation/) را به یک ارائه اضافه کنید و آن‌ها را به اسلایدها تبدیل کنید.
