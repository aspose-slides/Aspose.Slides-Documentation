## **ارزیابی Aspose.Slides**
می‌توانید به راحتی Aspose.Slides را برای ارزیابی بارگیری کنید. بسته ارزیابی همان بسته خریداری شده است. نسخه ارزیابی پس از افزودن چند خط کد برای اعمال مجوز به سادگی مجوزدار می‌شود.

## **محدودیت نسخه ارزیابی**
نسخه ارزیابی Aspose.Slides (بدون مشخص کردن مجوز) تمام قابلیت‌های محصول را ارائه می‌دهد، اما یک واترمارک ارزیابی را در بالای سند هنگام باز کردن و ذخیره‌سازی وارد می‌کند. همچنین هنگام استخراج متن‌ها از اسلایدهای ارائه، به یک اسلاید محدود می‌شوید.

{{% alert color="primary" %}} 
اگر می‌خواهید Aspose.Slides را بدون محدودیت‌های نسخه ارزیابی آزمایش کنید، می‌توانید **مجوز موقت 30 روزه** را درخواست کنید. لطفاً برای اطلاعات بیشتر به [چگونه یک مجوز موقت دریافت کنیم؟](https://purchase.aspose.com/temporary-license) مراجعه کنید.

## **درباره مجوز**
می‌توانید به راحتی نسخه ارزیابی Aspose.Slides برای Node.js از طریق Java را از [صفحه دانلود](https://releases.aspose.com/slides/fa/nodejs-java/) دریافت کنید. نسخه ارزیابی دقیقاً **همان قابلیت‌ها** را همانند نسخه دارای مجوز Aspose.Slides ارائه می‌دهد. علاوه بر این، پس از خرید یک مجوز و افزودن چند خط کد برای اعمال مجوز، نسخه ارزیابی به سادگی مجوزدار می‌شود.

مجوز یک فایل XML متنی ساده است که جزئیاتی مانند نام محصول، تعداد توسعه‌دهندگانی که مجوز به آنها داده شده است، تاریخ انقضای اشتراک و غیره را شامل می‌شود. این فایل به‌صورت دیجیتالی امضا شده است، بنابراین آن را ویرایش نکنید. حتی افزودن ناخواسته یک خط جدید به محتوای فایل باعث نامعتبر شدن آن می‌شود.

برای دور زدن محدودیت‌های مرتبط با نسخه ارزیابی، باید قبل از استفاده از **Aspose.Slides** یک مجوز تنظیم کنید. تنها یک بار در هر برنامه یا فرآیند نیاز به تنظیم مجوز دارید.

{{% alert color="primary" %}}
ممکن است بخواهید [مجوز متریک](https://docs.aspose.com/slides/fa/nodejs-java/metered-licensing/) را ببینید.
{{% /alert %}}

### **تنظیم مجوز در Aspose.Slides برای Node.js از طریق Java**
مجوزها می‌توانند از مکان‌های زیر اعمال شوند:
* مسیر صریح
* جریان
* به عنوان مجوز متره‌ای – مکانیزم جدید مجوزدهی

{{% alert color="primary" %}}
از متد **setLicense** برای صدور مجوز به یک مؤلفه استفاده کنید.

اگرچه چندین بار فراخوانی **setLicense** مضر نیست، اما باعث هدر رفتن منابع (پردازنده) می‌شود.
{{% /alert %}}

{{% alert color="warning" %}}
مجوزهای جدید می‌توانند Aspose.Slides را تنها از نسخه 21.4 به بعد فعال کنند. نسخه‌های قبلی از یک سیستم مجوزدهی متفاوت استفاده می‌کنند و این مجوزها را شناسایی نمی‌کنند.
{{% /alert %}}

#### **اعمال مجوز با استفاده از فایل**
این قطعه کد برای تنظیم فایل مجوز استفاده می‌شود:

**Node.js**

```javascript
var aspose = aspose || {};

aspose.slides = require("aspose.slides.via.java");

var license = new aspose.slides.License();
license.setLicense("Aspose.Slides.lic");
```

هنگام فراخوانی متد setLicense، نام مجوز باید همانند نام فایل مجوز شما باشد. به عنوان مثال، می‌توانید نام فایل مجوز را به "Aspose.Slides.lic.xml" تغییر دهید. سپس در کد خود، باید نام جدید مجوز (Aspose.Slides.lic.xml) را به متد setLicense پاس کنید.

#### **اعلام مجوز از یک جریان**
این قطعه کد برای اعمال مجوز از یک جریان استفاده می‌شود:

**Node.js**

```javascript
var aspose = aspose || {};

aspose.slides = require("aspose.slides.via.java");

var license = new aspose.slides.License();

var fs = require("fs");

var readStream = fs.createReadStream("Aspose.Slides.lic");

license.setLicense(readStream, function(err, list) {
    if(err) { 
        console.error(err); return; 
    }});
```

## **پرسش‌های متداول**

**آیا می‌توانم مجوز را در محیط کاملاً آفلاین (بدون دسترسی به اینترنت) اعمال کنم؟**  
بله. اعتبارسنجی مجوز به‌صورت محلی با استفاده از فایل مجوز انجام می‌شود؛ اتصال به اینترنت لازم نیست.

**بعد از پایان دوره یک‌ساله اشتراک چه اتفاقی می‌افتد؟ آیا کتابخانه متوقف می‌شود؟**  
نه. مجوز دائمی است: می‌توانید از نسخه‌های منتشر شده قبل از تاریخ پایان اشتراک خود استفاده کنید؛ فقط بدون تجدید، اجازه استفاده از نسخه‌های جدیدتر را ندارید.
