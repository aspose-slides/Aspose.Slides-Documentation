{{% alert color="primary" %}} 

تمام اجزای Aspose C++ به مجموعه‌مجوز Full Trust نیاز دارند. دلیل این است که اجزای Aspose C++ برای انجام برخی عملیات مانند تجزیه فونت‌ها و غیره، نیاز به دسترسی به تنظیمات رجیستری، فایل‌های سیستمی بجز دایرکتوری مجازی دارند. علاوه بر این، اجزای Aspose C++ بر پایه کلاس‌های اصلی سیستم C++ ساخته شده‌اند که در بسیاری از موارد نیز به مجموعه‌مجوز Full Trust نیاز دارند. 

{{% /alert %}}
