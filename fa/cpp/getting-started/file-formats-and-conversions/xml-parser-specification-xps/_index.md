## **درباره XPS**
[XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) یک زبان توصیف صفحه و یک قالب سند ثابت است که اصلاً توسط مایکروسافت توسعه یافته است. مشابه PDF، XPS یک قالب سند با چیدمان ثابت است که برای حفظ دقت سند و ارائه ظاهر دستگاه‑بی‌وابسته طراحی شده است.
## **XPS در Aspose.Slides for C++**
هر سند ارائه‌ای که بتواند توسط Aspose.Slides for C++ بارگذاری شود می‌تواند به قالب XPS تبدیل شود. Aspose.Slides for C++ از موتور چیدمان صفحه و رندرینگ با دقت بالا استفاده می‌کند تا خروجی را در قالب سند XPS با چیدمان ثابت تولید کند.
شایان ذکر است که Aspose.Slides for C++ به‌طور مستقیم XPS تولید می‌کند بدون آنکه به کلاس‌های Windows Presentation Foundation (WPF) که همراه با چارچوب C++ نسخه 3.5 بسته‌بندی شده‌اند وابسته باشد، و به این ترتیب به Aspose.Slides for C++ امکان می‌دهد تا اسناد XPS را بر روی ماشین‌هایی که نسخه‌های قدیمی‌تری از چارچوب C++ نسبت به 3.5 اجرا می‌کنند، تولید کند. می‌توانید در مورد صادرات اسناد ارائه به اسناد XPS از طریق Aspose.Slides for C++ در [این موضوع](/slides/fa/net/) بیشتر بخوانید.
