## **درباره PPT**
[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) فرمت فایل سند ارائه است که می‌تواند توسط نسخه‌های مختلف Microsoft PowerPoint ایجاد، خوانده، دستکاری و نوشته شود. این فرمت باینری برای اسناد ارائه است که توسط مایکروسافت توسعه یافته است. 
## **PPT در Aspose.Slides for C++**
Aspose.Slides for C++ می‌تواند فایل‌های PPT ایجاد شده توسط نرم‌افزارهای زیر را بخواند. 

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

به‌طور مشابه، فایل‌های PPT ایجاد شده توسط Aspose.Slides for C++ می‌تواند توسط مجموعه نرم‌افزارهای بالا خوانده شود. 
## **پشتیبانی جامع برای PPT**
Aspose.Slides for C++ پشتیبانی تقریباً تمام ویژگی‌های مرتبط با فرمت فایل سند PPT را فراهم می‌کند. این کتابخانه نه تنها ویژگی‌های پایه/پیشرفته ارائه‌شده توسط نسخه‌های مختلف Microsoft PowerPoint برای دستکاری اسناد PPT را پوشش می‌دهد، بلکه برخی ویژگی‌ها را که حتی توسط Microsoft PowerPoint پشتیبانی نمی‌شوند نیز شامل می‌شود. مزیت اصلی استفاده از کتابخانه API Aspose.Slides for C++ سادگی استفاده برای مدیریت چنین ویژگی‌هایی است. 

علاوه بر کارهای پایه مربوط به ایجاد، خواندن و نوشتن فایل‌های سند PPT، چندین ویژگی که توسط Aspose.Slides for C++ ارائه می‌شود عبارتند از: 

- وارد کردن سایر فرمت‌های فایل‌های MS Office به‌عنوان اشیای OLE در اسناد PPT
- صادرات اسناد PPT به فرمت‌های PDF، TIFF، XPS
- صادرات اسلایدهای موجود در اسناد PPT به فرمت SVG
- رندر اسلاید به هر فرمت تصویری که توسط فریم‌ورک C++ پشتیبانی می‌شود
- تنظیم اندازه اسلایدها در سند PPT
- مدیریت انیمیشن‌ها بر روی اشکال
- مدیریت نمایش اسلایدها
- قالب‌بندی متن در اسلایدها
- اسکن متن از اسناد PPT
- کار با جداول در اسلایدها
- کپی‌برداری خودکار از مسترها با استفاده از ویژگی کلونینگ
