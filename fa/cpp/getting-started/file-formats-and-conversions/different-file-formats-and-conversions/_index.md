## **مایکروسافت پاورپوینت (PPT)**
### **درباره PPT**
[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) یک فرمت فایل سند ارائه است که می‌تواند توسط نسخه‌های مختلف مایکروسافت پاورپوینت ایجاد، خوانده، دستکاری و نوشته شود. این یک فرمت باینری برای اسناد ارائه است که توسط مایکروسافت توسعه یافته است.
### **PPT در Aspose.Slides برای C++**
Aspose.Slides برای C++ می‌تواند فایل‌های PPT ایجاد شده توسط نرم‌افزارهای زیر را بخواند.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

به طور مشابه، فایل‌های PPT ایجاد شده توسط Aspose.Slides برای C++ می‌توانند توسط مجموعه نرم‌افزارهای بالا خوانده شوند.
### **پشتیبانی جامع برای PPT**
Aspose.Slides برای C++ پشتیبانی کاملی از تقریباً تمام ویژگی‌های مربوط به فرمت فایل سند PPT ارائه می‌دهد. این نه تنها ویژگی‌های پایه/پیشرفته ارائه شده توسط نسخه‌های مختلف مایکروسافت پاورپوینت برای دستکاری اسناد PPT را پوشش می‌دهد، بلکه برخی ویژگی‌ها را که حتی توسط مایکروسافت پاورپوینت پشتیبانی نمی‌شوند، شامل می‌شود. مزیت اصلی استفاده از کتابخانه API Aspose.Slides برای C++ سادگی استفاده برای مدیریت چنین ویژگی‌هایی است.

علاوه بر وظایف پایه مربوط به ایجاد، خواندن و نوشتن فایل‌های سند PPT، چندین ویژگی توسط Aspose.Slides برای C++ ارائه می‌شود مانند:

- وارد کردن فرمت‌های دیگر فایل‌های MS Office به عنوان اشیای OLE در اسناد PPT.
- صدور اسناد PPT به فرمت‌های PDF، TIFF، XPS.
- صدور اسلایدهای موجود در اسناد PPT به فرمت SVG.
- رندر اسلاید به هر فرمت تصویری که توسط چارچوب C++ پشتیبانی می‌شود.
- تنظیم اندازه اسلایدها در سند PPT.
- مدیریت انیمیشن‌ها روی اشکال.
- مدیریت نمایش اسلایدها.
- قالب‌بندی متن در اسلایدها.
- اسکن متن از اسناد PPT.
- دست‌کاری جداول در اسلایدها.
- کپی خودکار مسترها با استفاده از ویژگی کلونینگ.

## **PresentationML (PPTX، XML)**
### **درباره PresentationML**
PresentationML نامی است برای خانواده‌ای از فرمت‌های مبتنی بر XML برای اسناد ارائه. Office OpenXML (OOXML) فرمت مبتنی بر XML معرفی‌شده در برنامه‌های Microsoft Office 2007 است. Office OpenXML یک فرمت محفظه‌ای برای چندین زبان نشانه‌گذاری مبتنی بر XML تخصصی است. PresentationML زبان نشانه‌گذاری استفاده‌شده توسط Microsoft Office PowerPoint 2007 برای ذخیره اسناد خود است.
### **PresentationML در Aspose.Slides برای C++**
اسناد OOXML PresentationML به صورت فایل‌های PPTX آمده‌اند که بسته‌های XML فشرده‌شده هستند و مطابق با مشخصات [OOXML ECMA-376](https://www.ecma-international.org/publications-and-standards/standards/ecma-376/) هستند. Aspose.Slides برای C++ به طور گسترده‌ای از ایجاد، خواندن، دستکاری و نوشتن اسناد PresentationML پشتیبانی می‌کند. علاوه بر این، Aspose.Slides برای C++ قادر به صدور اسناد PresentationML به فرمت‌های اسناد پراستفاده مختلف مانند PDF، TIFF و XPS است. این امکان به این دلیل است که Aspose.Slides برای C++ با هدف مدیریت جامع اسناد ارائه طراحی شده و PresentationML اساساً نمایش داخلی اسناد را به صورت بسته XML فشرده نگه می‌دارد.

### **PresentationML باز است، چرا از Aspose.Slides برای C++ استفاده کنیم**
از آنجا که PresentationML مبتنی بر XML است، امکان ساخت برنامه‌هایی برای پردازش و تولید اسناد PresentationML با استفاده از کلاس‌های XML بدون تکیه بر کتابخانه‌های کلاس شخص ثالث مانند Aspose.Slides برای C++ وجود دارد. اما چندین مزیت استفاده از Aspose.Slides برای C++ نسبت به کلاس‌های XML در هنگام کار با اسناد PresentationML وجود دارد.

مشخصات OOXML بسیار طولانی است و به چندین هزار صفحه می‌رسد. این به این معنی است که برای مدیریت صحیح اسناد PresentationML باید زمان و تلاش زیادی صرف درک فرمت این اسناد کنید. از سوی دیگر، در هنگام استفاده از Aspose.Slides برای C++، تنها کافی است از کلاس‌های مرتبط و متدها/خواص مربوطه برای انجام عملیات‌هایی که اگر از طریق کلاس‌های XML انجام شوند بسیار پیچیده به‌نظر می‌رسند، استفاده کنید.

در ادامه برخی ویژگی‌ها که حتی هنگام کار با اسناد PresentationML از طریق کلاس‌های XML هم در دسترس نیستند، آورده شده است:

- صدور اسناد PPT به فرمت‌های PDF، TIFF، XPS
- صدور اسلایدهای موجود در اسناد PPT به فرمت SVG
- رندر اسلاید به هر فرمت تصویری که توسط چارچوب C++ پشتیبانی می‌شود
- کپی خودکار مسترها از ارائه‌های منبع با استفاده از ویژگی کلونینگ
- اعمال حفاظت بر اشکال

## **تبدیل PPT به PPTX**
### **درباره تبدیل**
Aspose.Slides اکنون همچنین از تبدیل PPT به PPTX پشتیبانی می‌کند.
### **ویژگی‌های پشتیبانی‌شده در تبدیل**
Aspose.Slides برای C++ پشتیبانی جزئی از تبدیل ارائه‌های فرمت فایل سند PPT به ارائه‌های فرمت فایل PPTX را فراهم می‌کند. از آنجا که ویژگی تبدیل ارائه‌ی ذکرشده به‌تازگی در Aspose.Slides برای C++ معرفی شده است، در حال حاضر توانایی محدود دارد و فقط برای انواع ساده ارائه‌ها کار می‌کند. مزیت اصلی که کتابخانه API Aspose.Slides برای C++ برای تبدیل ارائه PPT به فرمت PPTX فراهم می‌کند، سهولت استفاده از API برای دستیابی به هدف مطلوب است. لطفاً برای جزئیات بیشتر به this[link]() برای بخش نمونه کدها مراجعه کنید. بخش زیر به‌وضوح نشان می‌دهد کدام ویژگی‌ها هنگام تبدیل ارائه‌های فرمت PPT به ارائه‌های فرمت PPTX پشتیبانی می‌شوند و کدام‌ها پشتیبانی نمی‌شوند.
### **ویژگی‌های پشتیبانی‌شده**
ویژگی‌های پشتیبانی‌شده در حین تبدیل:

- تبدیل ساختار مسترها، چیدمان‌ها و اسلایدها
- تبدیل ساختار مسترها، چیدمان‌ها و اسلایدها
- تبدیل نمودارها
- گروه‌بندی اشکال
- تبدیل اشکال خودکار شامل مستطیل‌ها و بیضی‌ها. اما ممکن است مقادیر تنظیمات اشکال خودکار اشتباه باشد
- اشکال با هندسه سفارشی. گاهی ممکن است تبدیل نشوند
- بافت‌ها و سبک پر کردن تصویر برای اشکال خودکار. گاهی ممکن است تبدیل نشوند
- تبدیل جای‌گیرها
- تبدیل متن در فریم‌های متنی و نگهدارنده‌های متن. با این حال، بولت‌ها، تراز و تب‌ها به‌صورت کامل پیاده‌سازی نشده‌اند
### **ویژگی‌های پشتیبانی‌نشده**
ویژگی‌های پشتیبانی‌نشده در حین تبدیل:

- اسلاید با یادداشت‌ها؛ خواندن یادداشت‌ها در PPTX پیاده‌سازی نشده است. در صورتی که PPT دارای آن باشد، هنوز نمی‌تواند به PPTX ذخیره شود.
- تبدیل خطوط و پلی‌لاین‌ها
- قالب‌های خط و پرکن
- سبک‌های پرکن گرادیان
- فریم‌های OLE، جداول، فریم‌های ویدئو و صدا و غیره
- انیمیشن و سایر ویژگی‌های اسلایدشو نادیده گرفته می‌شوند.

ویژگی‌های جدید یا مفقودی به‌صورت پس‌آمدی در نسخه‌های آینده Aspose.Slides برای C++ افزوده خواهند شد.

## **فرمت سند قابل حمل (PDF)**
### **درباره PDF**
PDF (فرمت سند قابل حمل) فرمت فایلی است که توسط Adobe System برای تبادل اسناد بین سازمان‌های مختلف ایجاد شد. هدف از این فرمت این بود که محتویات اسناد به گونه‌ای نمایان شوند که ظاهر بصری آن‌ها به پلتفرمی که مشاهده می‌شود وابسته نباشد.
### **PDF در Aspose.Slides برای C++**
هر سند ارائه‌ای که می‌تواند در Aspose.Slides برای C++ بارگذاری شود، می‌تواند به سند PDF تبدیل شود که ممکن است با PDF 1.5 یا PDF/A‑1b مطابقت داشته باشد، بسته به انتخاب شما. Aspose.Slides برای C++ اسناد ارائه را به PDF به‌گونه‌ای صادر می‌کند که در اکثر موارد، سند PDF صادرشده تقریباً مشابه سند ارائه اصلی به نظر می‌رسد. راه‌حل Aspose ویژگی‌های زیر از اسناد ارائه را هنگام تبدیل به اسناد PDF پشتیبانی می‌کند:

- تصاویر، جعبه‌های متن و دیگر اشکال
- متن و قالب‌بندی
- پاراگراف‌ها و قالب‌بندی
- پیوندهای ابرمتنی
- سرصفحه‌ها و پاورقی‌ها
- نشان‌گرها
- جداول

می‌توانید اسناد ارائه را مستقیماً با استفاده فقط از مؤلفه Aspose.Slides برای C++ به اسناد PDF صادر کنید. یعنی نیازی به هیچ مؤلفه شخص ثالث یا Aspose.Pdf برای این منظور ندارید. علاوه بر این، می‌توانید صادرات ارائه به PDF را با گزینه‌های مختلف سفارشی‌سازی کنید همان‌طور که در [this topic](/slides/fa/cpp/convert-powerpoint-to-pdf/) توضیح داده شده است.

## **مشخصات پارسر XML (XPS)**
### **درباره XPS**
XML Parser Specification (مشخصات پارسر XML) یک زبان توصیف صفحه و فرمت سند ثابت است که در ابتدا توسط مایکروسافت توسعه یافته است. مشابه PDF، XPS یک فرمت سند با طرح‌بندی ثابت است که برای حفظ صحت سند و ارائه ظاهر دستگاه‑مستقل طراحی شده است.
### **XPS در Aspose.Slides برای C++**
هر سند ارائه‌ای که می‌تواند توسط Aspose.Slides برای C++ بارگذاری شود، می‌تواند به فرمت XPS تبدیل شود. Aspose.Slides برای C++ از موتور صفحه‌بندی و رندرینگ با دقت بالا برای تولید خروجی در فرمت سند XPS با طرح ثابت استفاده می‌کند. شایان ذکر است که Aspose.Slides برای C++ مستقیماً XPS تولید می‌کند بدون اینکه به کلاس‌های Windows Presentation Foundation (WPF) که با چارچوب C++ 3.5 بسته‌بندی شده‌اند وابسته باشد، بنابراین اجازه می‌دهد Aspose.Slides برای C++ اسناد XPS را در ماشین‌هایی که نسخه‌های قبلی چارچوب C++ را اجرا می‌کنند، تولید کند. می‌توانید درباره صدور اسناد ارائه به اسناد XPS از طریق Aspose.Slides برای C++ در [this topic](https://docs.aspose.com/slides/fa/cpp/convert-powerpoint-to-xps/) مطالعه کنید.
