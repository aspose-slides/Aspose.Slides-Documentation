## **الزامات نرم‌افزاری**
لطفاً قبل از بارگیری و اجرای مثال‌ها اطمینان حاصل کنید که الزامات زیر را برآورده می‌کنید.

1. Visual Studio 2015 یا بالاتر.

## **دریافت Aspose.Slides از GitHub**
تمام مثال‌های Aspose.Slides برای C++ در [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-C) میزبانی می‌شوند.

- می‌توانید مخزن را با استفاده از کلاینت GitHub مورد علاقه خود کلن کنید یا فایل ZIP را از [اینجا](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/refs/heads/master.zip) بارگیری کنید.
- محتویات فایل ZIP را به هر پوشه‌ای در رایانه خود استخراج کنید. تمام مثال‌ها در پوشه **Examples** قرار دارند.
- یک فایل راه‌حل Visual Studio برای C++ وجود دارد.
- پروژه‌ها در Visual Studio 2015 ساخته شده‌اند، اما فایل‌های راه‌حل با Visual Studio 2015 و بالاتر سازگار هستند.
- فایل راه‌حل را در Visual Studio باز کنید و پروژه را بیلد کنید.
- آخرین نسخه Aspose.Slides برای C++ را از [بخش دانلود](https://downloads.aspose.com/slides/fa/cpp) دانلود کنید.
- نسخه را استخراج کنید و پوشه‌های **Include** و **Bin** را داخل پوشه **Examples** کپی کنید.
- فایل **main.cpp** را باز کنید، تمام مثال‌ها از اینجا فراخوانی می‌شوند.
- مثال‌هایی که می‌خواهید اجرا کنید را در پروژه از حالت کامنت خارج کنید.
- راه‌حل را بیلد و کامپایل کنید

در صورت بروز هر گونه مشکل در تنظیم یا اجرای مثال‌ها، لطفاً از طریق انجمن‌های ما تماس بگیرید.
## **مشارکت**
اگر می‌خواهید یک مثال اضافه یا بهبود دهید، شما را تشویق می‌کنیم که به پروژه کمک کنید. تمام مثال‌ها و پروژه‌های نمایش در این مخزن منبع باز هستند و می‌توانید به‌راحتی در برنامه‌های خود استفاده کنید.

برای مشارکت، می‌توانید مخزن را Fork کنید، کد منبع را ویرایش کنید و یک Pull Request ایجاد کنید. ما تغییرات را بررسی کرده و در صورتی که مفید باشند، در مخزن گنجانده می‌شوند.
