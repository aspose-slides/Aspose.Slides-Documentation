## **فرمت‌های فایل و تبدیل‌ها**
Aspose.Slides forC++ از اکثر فرمت‌های سند پاورپوینت پشتیبانی می‌کند و همچنین آن‌ها را به فرمت‌های محبوب که به طور گسترده برای تبادل بین سازمان‌ها استفاده می‌شوند، صادر می‌کند. جدول زیر این فرمت‌ها را فهرست می‌کند:

|**ویژگی**|**توضیح**|
| :- | :- |
|[مایکروسافت پاورپوینت (PPT)](/slides/fa/cpp/microsoft-powerpoint-ppt/)|Aspose.Slides for C++ سریع‌ترین پردازش را برای این فرمت سند ارائه می‌دهد.|
|[فرمت سند قابل حمل (PDF)](/slides/fa/cpp/portable-document-format-pdf/)|فرمت‌های فایل پشتیبانی‌شده می‌توانند با یک متد به اسناد Adobe Portable Document Format (PDF) صادر شوند.|
|[مشخصات تجزیه‌گر XML (XPS)](/slides/fa/cpp/xml-parser-specification-xps/)|تمام فرمت‌های فایل پشتیبانی‌شده می‌توانند با یک متد به اسناد XML Parser Specification (XPS) صادر شوند.|
|[PresentationML (PPTX, XML)](/slides/fa/cpp/presentationml-pptx-xml/)|Aspose.Slides for C++ پردازش فرمت ارائه OOXML که همچنین به عنوان PresentationML یا PPTX شناخته می‌شود را پشتیبانی می‌کند.|
|[تبدیل PPTX به HTML](https://docs.aspose.com/slides/fa/cpp/convert-powerpoint-to-html/)|Aspose.Slides for C++ تبدیل ارائه به فرمت HTML را پشتیبانی می‌کند.|

## **رندرینگ**
Aspose.Slides for C++ رندرینگ با دقت بالا از اسلایدهای موجود در اسناد ارائه به فرمت‌های گرافیکی مختلف را همان‌طور که در زیر فهرست شده است، فراهم می‌کند:

|**ویژگی**|**توضیح**|
| :- | :- |
|فرمت‌های تصویر پشتیبانی‌شده در C++|با استفاده از Aspose.Slides for C++ می‌توانید نه تنها اسلایدهای ارائه، بلکه تصاویر روی اسلایدها را به تمام فرمت‌های گرافیکی پشتیبانی‌شده در C++ مانند PNG، BMP و JPEG رندر کنید.|
|فرمت SVG|Aspose.Slides for C++ همچنین متد داخلی برای صادر کردن اسلایدهای ارائه به فرمت Scalable Vector Graphics (SVG) فراهم می‌کند.|

## **قابلیت‌های محتوا**
Aspose.Slides forC++ به شما امکان دسترسی، تغییر یا ایجاد تقریباً تمام محتوای ممکن در اسناد ارائه را می‌دهد.

|**ویژگی**|**توضیح**|
| :- | :- |
|اسلایدهای اصلی|اسلایدهای اصلی چیدمان اسلایدهای عادی را تعریف می‌کنند. Aspose.Slides for C++ به شما امکان دسترسی و تغییر اسلایدهای اصلی اسناد ارائه را می‌دهد.|
|اسلایدهای عادی|با Aspose.Slides for C++ می‌توانید نه تنها اسلایدهای جدید از انواع مختلف ایجاد کنید، بلکه به اسلایدهای موجود در اسناد ارائه دسترسی داشته و آن‌ها را تغییر دهید.|
|کلون‌کردن / کپی‌کردن اسلایدها|متدهای داخلی توسط Aspose.Slides for C++ برای کلون یا کپی اسلایدهای موجود نه تنها درون یک ارائه، بلکه از یک ارائه به ارائه دیگر فراهم شده‌اند. از آنجا که یک اسلاید چیدمان خود را از اسلاید اصلی به ارث می‌برد، متدهای کلون داخلی به‌صورت خودکار اسلاید اصلی را نیز در هنگام کلون کپی می‌کنند.|
|محافظ‌محل و نگهدارنده‌های متن|شما می‌توانید به محافظ‌محل‌ها و نگهدارنده‌های متن داخل یک اسلاید دسترسی داشته باشید. علاوه بر این، می‌توانید اسلایدی با نگهدارنده‌های متن از ابتدا با استفاده از متد مناسب ایجاد کنید.|
|سربرگ و پابرگ|Aspose.Slides for C++ همچنین مدیریت سربرگ / پابرگ‌ها در اسلایدها را تسهیل می‌کند.|
|یادداشت‌ها در اسلایدها|با Aspose.Slides for C++ می‌توانید نه تنها به یادداشت‌های مرتبط با اسلاید دسترسی پیدا کنید و آن‌ها را تغییر دهید، بلکه یادداشت‌های جدید نیز اضافه کنید.|
|یافتن یک شکل|شما می‌توانید یک شکل خاص را از اسلاید با استفاده از متن جایگزین (alternative text) مرتبط با شکل پیدا کنید.|
|پس‌زمینه‌ها|Aspose.Slides for C++ امکان کار با پس‌زمینه مرتبط با اسلاید اصلی یا عادی ارائه را فراهم می‌کند.|
|جعبه‌های متن|جعبه‌های متن می‌توانند از ابتدا ایجاد شوند. جعبه‌های متن موجود می‌توانند دسترسی یافته و متن آن‌ها نیز بدون از دست رفتن قالب‌بندی اصلی تغییر یابد.|
|شکل‌های مستطیل|شکل‌های مستطیل می‌توانند توسط Aspose.Slides for C++ ایجاد یا تغییر یابند.|
|شکل‌های خط چندضلعی|شکل‌های خط چندضلعی نیز می‌توانند توسط Aspose.Slides for C++ ایجاد یا تغییر یابند.|
|شکل‌های بیضی|شکل‌های بیضی نیز توسط Aspose.Slides for C++ ایجاد یا تغییر می‌یابند.|
|شکل‌های گروهی|Aspose.Slides for C++ از شکل‌های گروهی نیز پشتیبانی می‌کند.|
|شکل‌های خودکار|شکل‌های خودکار نیز توسط Aspose.Slides for C++ پشتیبانی می‌شوند.|
|سریال‌سازی شکل‌ها|اگرچه Aspose.Slides for C++ از تعداد زیادی شکل پشتیبانی می‌کند، اما اگر شکلی توسط Aspose.Slides for C++ پشتیبانی نشود، متد سریال‌سازی توسط Aspose.Slides for C++ فراهم شده است که می‌توانید آن شکل را از یک اسلاید موجود سریال‌سازی کنید و برای استفاده‌های بعدی توسط Aspose.Slides for C++ بر حسب نیاز به کار ببرید.|
|قاب‌های تصویر|تصاویر در قاب‌های تصویر با استفاده از Aspose.Slides for C++ مدیریت می‌شوند.|
|قاب‌های صدا|فایل‌های صوتی می‌توانند به‌صورت لینک شده یا جاسازی‌شده در قاب‌های صوتی بر روی اسلایدها توسط Aspose.Slides for C++ قرار گیرند.|
|قاب‌های ویدئو|فایل‌های ویدئویی توسط Aspose.Slides for C++ در قاب‌های ویدئو مدیریت می‌شوند.|
|قاب OLE|اشیاء OLE توسط Aspose.Slides for C++ در قاب‌های OLE مدیریت می‌شوند.|
|جدول‌ها|جدول‌ها بر روی اسلایدها نیز توسط Aspose.Slides for C++ پشتیبانی می‌شوند.|
|قاب متن|متن مرتبط با هر شکل می‌تواند از طریق قاب متنی که به آن شکل تعلق دارد، دسترسی پیدا کند.|
|اسکن متن|متن در یک ارائه می‌تواند از سطح ارائه یا اسلاید با استفاده از متدهای داخلی اسکن شود.|
|انیمیشن‌ها|انیمیشن‌ها می‌توانند بر روی شکل‌ها اعمال شوند.|
|نمایش اسلاید|نمایش اسلایدها، انتقال اسلاید و انتقال مورف نیز پشتیبانی می‌شوند.|

## **FAQ**

**آیا نیاز است Microsoft PowerPoint را بر روی سرور/رایانه نصب کنم تا کتابخانه کار کند؟**

خیر. PowerPoint نیازی نیست؛ Aspose.Slides یک موتور مستقل برای ایجاد، ویرایش، تبدیل و رندرینگ ارائه‌ها است.

**چگونه_MULTITHREADING_کار می‌کند؟ آیا پردازش می‌تواند به صورت موازی انجام شود؟**

ایمن است که اسناد مختلف را در رشته‌های متفاوت پردازش کنید؛ شیء همان [presentation](https://reference.aspose.com/slides/fa/cpp/aspose.slides/presentation/) نباید توسط [multiple threads](/slides/fa/cpp/multithreading/) هم‌زمان استفاده شود.

**آیا رمز عبور فایل و رمزنگاری پشتیبانی می‌شوند؟**

بله. [You can](/slides/fa/cpp/password-protected-presentation/) ارائه‌های رمزگذاری‌شده را باز کنید، رمز عبور باز و نوشتن را تنظیم یا حذف کنید و وضعیت حفاظت را بررسی کنید.

**آیا باید درباره بسته‌های قلم در کانتینرهای لینوکس نگران باشم؟**

بله. توصیه می‌شود بسته‌های قلم عمومی را نصب کنید و/یا به‌صورت صریح [specify font directories](/slides/fa/cpp/custom-font/) را در برنامه خود مشخص کنید تا از جایگزینی‌های ناخواسته جلوگیری شود.

**آیا در نسخه ارزیابی محدودیتی وجود دارد؟**

در [evaluation mode](/slides/fa/cpp/licensing/)، یک واترمارک به خروجی اضافه می‌شود و برخی محدودیت‌ها اعمال می‌شود؛ یک [30-day temporary license](https://purchase.aspose.com/temporary-license/) برای تست کامل ویژگی‌ها در دسترس است.

**آیا وارد کردن فرمت‌های خارجی به یک ارائه (PDF/HTML → PPTX) پشتیبانی می‌شود؟**

بله. می‌توانید [PDF pages and HTML content](/slides/fa/cpp/import-presentation/) را به یک ارائه اضافه کنید و آن‌ها را به اسلاید تبدیل کنید.
