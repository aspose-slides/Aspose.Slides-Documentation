{{% alert color="primary" %}} 

Aspose.Slides for C++ یک کتابخانهٔ کلاس است که به برنامه‌های C++ امکان می‌دهد اسناد ارائه را بدون استفاده از Microsoft PowerPoint® یا سایر برنامه‌ها بخوانند و بنویسند. Aspose.Slides for C++ امکان مدیریت متن، اشکال، جداول و انیمیشن‌ها، افزودن صدا و ویدئو به اسلایدها، پیش‌نمایش اسلایدها، خروجی به SVG، فرمت PDF و موارد دیگر را فراهم می‌کند.

آخرین مرجع API می‌تواند در [Aspose.Slides for C++ API Reference](https://reference.aspose.com/slides/fa/cpp/) یافت شود.
