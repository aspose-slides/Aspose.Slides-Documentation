## **نمای کلی**

این مقاله نحوه رندر اسلایدهای ارائه به‌عنوان تصاویر SVG را با استفاده از Aspose.Slides توضیح می‌دهد. قالب SVG و مزایای آن را شامل مقیاس‌پذیری، دسترس‌پذیری و مناسب بودن برای توسعه وب شرح می‌دهد.

شما خواهید آموخت چگونه یک فایل ارائه را بارگذاری کنید، از اسلایدهای آن عبور کنید و هر اسلاید را به‌عنوان یک فایل SVG جداگانه ذخیره کنید. این مقاله فرمت‌های ارائه PowerPoint و OpenDocument شامل PPT، PPTX، ODP و PPS را پوشش می‌دهد و نشان می‌دهد چگونه تبدیل را به‌صورت برنامه‌نویسی با کلاس `Presentation` و متد `WriteAsSvg` انجام دهید.

## **قالب SVG**

SVG که مخفف Scalable Vector Graphics است، یک نوع گرافیک استاندارد یا قالب برای رندر تصاویر دو‑بعدی می‌باشد. SVG تصاویر را به‌صورت بردار در XML ذخیره می‌کند و جزئیاتی که رفتار یا ظاهر آن‌ها را تعریف می‌کند شامل می‌شود.

SVG یکی از معدود قالب‌های تصویری است که در این موارد استانداردهای بسیار بالایی را برآورده می‌کند: مقیاس‌پذیری، تعامل‌پذیری، عملکرد، دسترس‌پذیری، برنامه‌پذیری و غیره. به‌دلیل این دلایل، به‌طور گسترده‌ای در توسعه وب استفاده می‌شود.

ممکن است بخواهید از فایل‌های SVG زمانی که نیاز دارید:

- **چاپ ارائه خود در یک *فرمت بسیار بزرگ*.** تصاویر SVG می‌توانند به هر رزولوشن یا سطحی بزرگ شوند. می‌توانید تصاویر SVG را هر تعداد که لازم است بدون از دست دادن کیفیت تغییر اندازه دهید.
- **استفاده از نمودارها و گراف‌ها از اسلایدهای خود در *سایر رسانه‌ها یا پلتفرم‌ها*.** اکثر مرورگرها می‌توانند فایل‌های SVG را تفسیر کنند.
- **استفاده از *کوچک‌ترین اندازه ممکن برای تصاویر*.** فایل‌های SVG معمولاً کوچک‌تر از معادل‌های با وضوح بالا در قالب‌های دیگر هستند، به‌ویژه قالب‌های مبتنی بر بیت‌مپ (JPEG یا PNG).

## **رندر یک اسلاید به عنوان تصویر SVG**

Aspose.Slides for C++ به شما امکان می‌دهد اسلایدهای ارائه خود را به‌عنوان تصاویر SVG صادر کنید. با انجام این مراحل، تصاویر SVG را تولید کنید:

1. یک نمونه از کلاس `Presentation` ایجاد کنید.
2. از تمام اسلایدهای ارائه عبور کنید.
3. هر اسلاید را از طریق `FileStream` به فایل SVG اختصاصی خود بنویسید.

{{% alert color="primary" %}} 
ممکن است بخواهید برنامه وب رایگان ما را که در آن تابع تبدیل PPT به SVG از Aspose.Slides for C++ پیاده‌سازی شده است، امتحان کنید: [free web application](https://products.aspose.app/slides/fa/conversion/ppt-to-svg)
{{% /alert %}} 

این کد نمونه در C++ نشان می‌دهد چگونه PPT را با استفاده از Aspose.Slides به SVG تبدیل کنید:

``` cpp
auto pres = System::MakeObject<Presentation>(u"pres.pptx");
        
for (int32_t index = 0; index < pres->get_Slides()->get_Count(); index++)
{
    auto fileName = String::Format(u"slide-{0}.svg", index);
    auto fileStream = System::MakeObject<FileStream>(fileName, FileMode::Create, FileAccess::Write);

    auto slide = pres->get_Slides()->idx_get(index);
    slide->WriteAsSvg(fileStream);
}
```

## **پرسش‌های متداول**

**چرا ممکن است SVG حاصل در مرورگرهای مختلف متفاوت ظاهر شود؟**

پشتیبانی از ویژگی‌های خاص SVG به‌صورت متفاوتی توسط موتورهای مرورگرها پیاده‌سازی می‌شود. پارامترهای [SVGOptions](https://reference.aspose.com/slides/fa/cpp/aspose.slides.export/svgoptions/) به رفع ناسازگاری‌ها کمک می‌کنند.

**آیا می‌توان نه تنها اسلایدها بلکه شکل‌های منفرد را نیز به SVG صادر کرد؟**

بله. هر [shape can be saved as a separate SVG](https://reference.aspose.com/slides/fa/cpp/aspose.slides/shape/writeassvg/)، که برای آیکون‌ها، تصویرهای نمادین و استفاده مجدد از گرافیک‌ها مناسب است.

**آیا می‌توان چندین اسلاید را به یک SVG (strip/document) ترکیب کرد؟**

سناریوی استاندارد یک اسلاید → یک SVG است. ترکیب چندین اسلاید در یک بوم SVG یک مرحله پس‌پردازشی است که در سطح برنامه انجام می‌شود.
