## **معرفی**

Aspose.Slides ابزارهای کاملی برای تنظیم اندازه اسلاید و نسبت ابعاد در ارائه‌های PowerPoint فراهم می‌کند که برای چاپ و نمایش روی صفحه‌نمایش حیاتی است.

اندازه‌ها و نسبت‌های رایج اسلاید:

- **استاندارد (نسبت 4:3)**: برای صفحه‌نمایش‌ها و دستگاه‌های قدیمی ایده‌آل است.
- **پهن‌صفحه (نسبت 16:9)**: برای پروژکتورها و نمایشگرهای مدرن توصیه می‌شود.

سازگاری را در سراسر ارائه خود حفظ کنید؛ یک اندازه اسلاید و نسبت ابعاد برای همه اسلایدها اعمال می‌شود. برای دستیابی به نتایج بهینه، ابعاد اسلاید را در ابتدای فرآیند ایجاد ارائه تنظیم کنید تا از ایجاد پیچیدگی جلوگیری شود.

{{% alert color="primary" %}} 
به‌طور پیش‌فرض، ارائه‌های ایجاد شده با Aspose.Slides از نسبت ابعاد استاندارد 4:3 استفاده می‌کنند.
{{% /alert %}}

## **تغییر اندازه اسلاید در ارائه‌ها**

این کد نمونه نشان می‌دهد چگونه می‌توانید اندازه اسلاید را در یک ارائه با C++ و Aspose.Slides تغییر دهید:

``` cpp
auto pres = System::MakeObject<Presentation>(u"pres-4x3-aspect-ratio.pptx");
pres->get_SlideSize()->SetSize(SlideSizeType::OnScreen16x9, SlideSizeScaleType::DoNotScale);
pres->Save(u"pres-4x3-aspect-ratio.pptx", SaveFormat::Pptx);
```

## **مشخص کردن اندازه اسلاید سفارشی در ارائه‌ها**

اگر اندازه‌های رایج اسلاید (4:3 و 16:9) برای کار شما مناسب نیستند، می‌توانید از یک اندازه اسلاید خاص یا منحصر به‌فرد استفاده کنید. برای مثال، اگر قصد دارید اسلایدهای کامل‌سایز را روی یک چیدمان صفحه سفارشی چاپ کنید یا ارائه خود را بر روی انواع خاصی از صفحه‌نمایش‌ها نمایش دهید، احتمالاً بهره‌مند خواهید شد اگر اندازه سفارشی برای ارائه خود تنظیم کنید.

این کد نمونه نشان می‌دهد چگونه می‌توانید با Aspose.Slides برای C++ یک اندازه اسلاید سفارشی برای یک ارائه در C++ مشخص کنید:

``` cpp
auto pres = System::MakeObject<Presentation>(u"pres.pptx");
// اندازه کاغذ A4
pres->get_SlideSize()->SetSize(780.0f, 540.0f, SlideSizeScaleType::DoNotScale);
pres->Save(u"pres-a4-slide-size.pptx", SaveFormat::Pptx);
```

## **مدیریت محتوای اسلاید پس از تغییر اندازه**

پس از تغییر اندازه اسلاید برای یک ارائه، محتوای اسلایدها (مانند تصاویر یا اشیا) ممکن است دچار انحراف شود. به‌طور پیش‌فرض، اشیا به‌صورت خودکار برای متناسب شدن با اندازه جدید اسلاید تغییر اندازه می‌یابند. با این حال، هنگام تغییر اندازه اسلاید یک ارائه، می‌توانید تنظیمی را مشخص کنید که تعیین می‌کند Aspose.Slides چگونه با محتوای اسلایدها برخورد کند.

بسته به آنچه قصد دارید انجام دهید یا به دست آورید، می‌توانید از هر یک از این تنظیمات استفاده کنید:

- `DoNotScale`

  اگر نمی‌خواهید اشیا روی اسلایدها تغییر اندازه یابند، از این تنظیم استفاده کنید.

- `EnsureFit`

  اگر می‌خواهید به اندازه اسلاید کوچکتر مقیاس کنید و نیاز دارید Aspose.Slides اشیا را کوچکتر کند تا همه آنها در اسلایدها جا بگیرند (بدین ترتیب از از دست رفتن محتوا جلوگیری می‌شود)، از این تنظیم استفاده کنید.

- `Maximize`

  اگر می‌خواهید به اندازه اسلاید بزرگ‌تر مقیاس کنید و نیاز دارید Aspose.Slides اشیا را بزرگ‌تر کند تا نسبت به اندازه جدید اسلاید متناسب شوند، از این تنظیم استفاده کنید.

این کد نمونه نشان می‌دهد چگونه می‌توانید از تنظیم `Maximize` هنگام تغییر اندازه اسلاید یک ارائه استفاده کنید:

``` cpp
auto pres = System::MakeObject<Presentation>(u"pres.pptx");
pres->get_SlideSize()->SetSize(SlideSizeType::Ledger, SlideSizeScaleType::Maximize);
```

## **سؤالات متداول**

**آیا می‌توانم اندازه اسلاید سفارشی را با واحدهایی غیر از اینچ (مثلاً نقطه یا میلی‌متر) تنظیم کنم؟**

بله. Aspose.Slides به‌صورت داخلی از نقطه استفاده می‌کند که 1 نقطه برابر است با 1/72 اینچ. می‌توانید هر واحدی (مانند میلی‌متر یا سانتی‌متر) را به نقطه تبدیل کنید و از مقادیر تبدیل شده برای تعریف عرض و ارتفاع اسلاید استفاده کنید.

**آیا یک اندازه اسلاید سفارشی بسیار بزرگ می‌تواند بر عملکرد و مصرف حافظه هنگام رندرینگ تأثیر بگذارد؟**

بله. ابعاد بزرگ‌تر اسلاید (به نقطه) به همراه مقیاس رندرینگ بالاتر منجر به افزایش مصرف حافظه و زمان پردازش می‌شود. سعی کنید اندازه اسلایدی عملی انتخاب کنید و مقیاس رندرینگ را فقط به‌موقع که برای دستیابی به کیفیت خروجی موردنظر نیاز است، تنظیم کنید.

**آیا می‌توانم یک اندازه اسلاید غیراستاندارد تعریف کنم و سپس اسلایدها را از ارائه‌هایی که اندازه‌های متفاوت دارند ترکیب کنم؟**

نمی‌توانید [ادغام ارائه‌ها](/slides/fa/cpp/merge-presentation/) را هنگام داشتن اندازه‌های اسلاید متفاوت انجام دهید — ابتدا یک ارائه را به اندازه دیگری تغییر اندازه دهید. هنگام تغییر اندازه اسلاید، می‌توانید نحوه‌ٔ برخورد با محتویات موجود را از طریق گزینه [SlideSizeScaleType](https://reference.aspose.com/slides/fa/cpp/aspose.slides/slidesizescaletype/) انتخاب کنید. پس از همتراز کردن اندازه‌ها، می‌توانید اسلایدها را ترکیب کنید و قالب‌بندی را حفظ کنید.

**آیا می‌توانم تصویرهای بندانگشتی برای اشکال منفرد یا نواحی خاصی از یک اسلاید تولید کنم و آیا این تصویرها اندازه جدید اسلاید را در نظر می‌گیرند؟**

بله. Aspose.Slides می‌تواند تصویرهای بندانگشتی برای [کل اسلایدها](https://reference.aspose.com/slides/fa/cpp/aspose.slides/slide/getimage/) و همچنین برای [اشکال انتخابی](https://reference.aspose.com/slides/fa/cpp/aspose.slides/shape/getimage/) تولید کند. تصاویر حاصل اندازه و نسبت ابعاد فعلی اسلاید را بازتاب می‌دهند و تضمین می‌کنند که قاب‌بندی و هندسهٔ تصویر منطبق باشد.
