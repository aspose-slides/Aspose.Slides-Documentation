## **نمای کلی**

این مقاله توضیح می‌دهد که چگونه با استفاده از Aspose.Slides خطوط روند را به نمودارهای ارائه اضافه کنید. نحوه ایجاد یک نمودار، افزودن خطوط روند به سری‌های نمودار و کار با انواع مختلف خطوط روند از جمله نمایی، خطی، لگاریتمی، میانگین متحرک، چندجمله‌ای و توان را نشان می‌دهد.

همچنین نحوه اضافه کردن یک خط سفارشی به نمودار با وارد کردن یک شکل خطی را شرح می‌دهد و شامل یک سؤالات متداول کوتاه درباره مقادیر پیشروی و پسروی خط روند و اینکه آیا خطوط روند در زمان خروجی به PDF یا SVG و هنگام رندر نمودارها به عنوان تصویر حفظ می‌شوند یا خیر.

## **افزودن خط روند**
Aspose.Slides for C++ یک API ساده برای مدیریت خطوط روند مختلف نمودارها فراهم می‌کند:

1. یک نمونه از کلاس [Presentation](https://reference.aspose.com/slides/fa/cpp/aspose.slides/presentation/) ایجاد کنید.
2. مرجع یک اسلاید را بر اساس ایندکس آن بدست آورید.
3. یک نمودار با داده‌های پیش‌فرض به همراه هر نوع دلخواه اضافه کنید (در این مثال از ChartType.ClusteredColumn استفاده شده است).
4. افزودن خط روند نمایی برای سری نمودار 1.
5. افزودن خط روند خطی برای سری نمودار 1.
6. افزودن خط روند لگاریتمی برای سری نمودار 2.
7. افزودن خط روند میانگین متحرک برای سری نمودار 2.
8. افزودن خط روند چندجمله‌ای برای سری نمودار 3.
9. افزودن خط روند توان برای سری نمودار 3.
10. ارائه‌ی تغییر یافته را در یک فایل PPTX بنویسید.

## **افزودن خط سفارشی**
Aspose.Slides for C++ یک API ساده برای افزودن خطوط سفارشی در یک نمودار فراهم می‌کند. برای افزودن یک خط ساده به اسلاید انتخاب‌شده‌ی ارائه، مراحل زیر را دنبال کنید:

- یک نمونه از کلاس Presentation ایجاد کنید
- مرجع یک اسلاید را با استفاده از Index آن به دست آورید
- یک نمودار جدید با استفاده از متد AddChart که توسط شی Shapes ارائه می‌شود، ایجاد کنید
- یک AutoShape از نوع Line با استفاده از متد AddAutoShape که توسط شی Shapes ارائه می‌شود، اضافه کنید
- رنگ خطوط شکل را تنظیم کنید.
- ارائه‌ی تغییر یافته را به عنوان یک فایل PPTX بنویسید

## **سؤالات متداول**

**'forward' و 'backward' در یک خط روند به چه معناست؟**

آنها طول‌های خط روند هستند که به سمت جلو/عقب پیش‌بینی می‌شوند: برای نمودارهای پراکنده (XY) — برحسب واحدهای محور؛ برای نمودارهای غیرپراکنده — برحسب تعداد دسته‌ها. فقط مقادیر غیرمنفی مجاز هستند.

**آیا خط روند هنگام خروجی گرفتن ارائه به PDF یا SVG یا هنگام رندر اسلاید به یک تصویر حفظ می‌شود؟**

بله. Aspose.Slides ارائه‌ها را به [PDF](/slides/fa/cpp/convert-powerpoint-to-pdf/)/[SVG](/slides/fa/cpp/render-a-slide-as-an-svg-image/) تبدیل می‌کند و نمودارها را به تصویر رندر می‌نماید؛ خطوط روند به عنوان بخشی از نمودار در این عملیات‌ها حفظ می‌شوند. همچنین روشی برای [صادر کردن تصویر نمودار](/slides/fa/cpp/create-shape-thumbnails/) موجود است.
