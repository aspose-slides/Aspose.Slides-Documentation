## **مقدمه**

دسترس‌پذیری ارائه تضمین می‌کند که افرادی که از فناوری‌های کمکی—مانند خوانندگان صفحه‌نمایش، نمایشگرهای بریل یا ناوبری فقط با صفحه‌کلید—استفاده می‌کنند، بتوانند اسلایدهای شما را همانند مخاطبان بینایی‌دار و استفاده‌کننده از ماوس به‌طور مؤثر درک و پیمایش کنند. شیوه‌های خوب بر روی ترتیب خواندن واضح، متن جایگزین معنی‌دار برای تصاویر اطلاعاتی، تضاد رنگ کافی، تایپوگرافی قابل خواندن، متن توصیفی لینک و جلوگیری از انتقال معنا تنها از طریق رنگ یا موقعیت تمرکز می‌کند. وقتی دسترس‌پذیری از همان ابتدا برنامه‌ریزی شود، نتیجه ساختاری تمیزتر، تصاویر سازگارتر و محتوایی است که بدون راه‌حل‌های جایگزین به هر بیننده می‌رسد.

## **علامت‌گذاری به عنوان تزئینی**

علامت‌گذاری به عنوان تزئینی، تصاویر صرفاً تزیینی را مشخص می‌کند تا خوانندگان صفحه‌نمایش آن‌ها را نادیده بگیرند، باعث کاهش نویز و حفظ تمرکز بر محتوای معنادار می‌شود. این را بر روی پس‌زمینه‌ها، تزئینات و فواصل اعمال کنید—هرگز بر روی نمودارها، آیکون‌ها یا تصاویری که اطلاعات انتقال می‌دهند. Aspose.Slides این پرچم را برای تشخیص و اعتبارسنجی فراهم می‌کند و امکان انجام بررسی‌های خودکار دسترس‌پذیری و پاک‌سازی را می‌دهد.

![علامت‌گذاری به عنوان تزئینی](mark_as_decorative.png)

```java
Presentation presentation = new Presentation("sample.pptx");
try {
    IShape shape = presentation.getSlides().get_Item(0).getShapes().get_Item(0);
    System.out.println("Is shape decorative: " + shape.isDecorative());
} finally {
    presentation.dispose();
}
```
