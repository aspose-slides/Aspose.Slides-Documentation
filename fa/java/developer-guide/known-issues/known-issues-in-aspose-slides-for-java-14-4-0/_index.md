{{% alert color="primary" %}} 

Aspose.Slides برای Java نسخه 14.4.0 تصمیم جدیدی برای پردازش اسناد PowerPoint ارائه می‌دهد. برخی محدودیت‌ها و مسائل شناخته‌شده وجود دارد که در نسخه‌های آینده حذف خواهند شد:

- برخی شکل‌ها در اسناد PPT سریالیزه‌شده دارای هندسه نادرست هستند (قوس، پیکان دایره‌ای، حاشیه‌نویسی).
- تمام ویژگی‌های قالب‌بندی متن PPTX در سریالیزه‌سازی PPT پشتیبانی نمی‌شوند (محدودیت‌های تب‌بندی، تورفتگی و قالب‌بندی پاراگراف).
- اطلاعات مربوط به زبان متن و تنظیمات املایی در اسناد PPT سریالیزه‌شده موجود نیست.
- تمام ویژگی‌های تم PPTX در سریالیزه‌سازی PPT پشتیبانی نمی‌شوند (فقط سریالیزه‌سازی فرمت‌های پر کردن، خط و قلم).
- مشکلات شناخته‌شده‌ای در سریالیزه‌سازی OLE/ActiveX از PPT به PPT وجود دارد.
- سریالیزه‌سازی و رندرینگ WordArt پشتیبانی نمی‌شود.

{{% /alert %}}
