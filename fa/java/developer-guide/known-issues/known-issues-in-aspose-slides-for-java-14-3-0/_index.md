Aspose.Slides for Java 14.3.0 (14.4.0) یک پیاده‌سازی کاملاً جدید از پردازش PPT را ارائه می‌دهد. بهبودهای زیادی وجود دارد، تبدیل جزئی PPTX به PPT. اما برخی ویژگی‌های پیاده‌سازی نشده وجود دارند:

- برخی شکل‌ها در اسناد PPT سریال‌سازی‌شده هندسه اشتباهی دارند (Call outs)
- همه ویژگی‌های قالب‌بندی متن PPTX در سریال‌سازی PPT پشتیبانی نمی‌شوند
- اطلاعات درباره زبان متن و تنظیمات املایی در اسناد PPT سریال‌سازی‌شده موجود نیست
- همه ویژگی‌های تم‌های PPTX در سریال‌سازی PPT پشتیبانی نمی‌شوند

**در مقایسه با Aspose.Slides for Java 8.6.0 برخی تفاوت‌ها وجود دارد:** 

- مشکلات شناخته‌شده‌ای در سریال‌سازی OLE/ActiveX PPT به PPT وجود دارد

**در مقایسه با Aspose.Slides for .NET 14.3.0 برخی تفاوت‌ها وجود دارد:** 

- پشتیبانی از چاپ ارائه در حال حاضر در Aspose.Slides for Java در دسترس نیست
