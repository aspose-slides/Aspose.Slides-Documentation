## **مرور کلی**

این بخش پرسش‌های متداول پاسخ‌های رایجی درباره Aspose.Slides را ارائه می‌دهد. مواردی مانند قالب‌های فایل پشتیبانی‌شده، مدیریت استثناها هنگام کار با ارائه‌های بزرگ، تغییر اندازه اسلایدها، پیش‌نمایش اسلایدها، استخراج متن از ارائه‌ها، قالب‌بندی حاشیه جدول‌ها، قرار دادن تصاویر و حل مشکلات مربوط به فونت‌ها هنگام تبدیل ارائه‌ها به PDF یا تصویر را پوشش می‌دهد.

## **قالب‌های فایل پشتیبانی‌شده**

**س: چه قالب‌های فایلی را Aspose.Slides برای Java پشتیبانی می‌کند؟**

**ج**: Aspose.Slides برای Java قالب‌های فایل توصیف‌شده در [قالب‌های فایل پشتیبانی‌شده](/slides/fa/java/supported-file-formats/) را پشتیبانی می‌کند.

## **استثناها**

**س: هنگام بارگذاری یک فایل PPT بزرگ حاوی تصاویر، استثنای out of memory دریافت می‌کنم. آیا محدودیتی برای اندازه فایل در Aspose.Slides وجود دارد؟**

**ج**: هیچ فرمول خاصی برای محاسبه اندازه ارائه‌ای که توسط Aspose.Slides پشتیبانی می‌شود، وجود ندارد. باید فضای کافی برای نگهداری کل ساختار ارائه و تصاویر در حافظه وجود داشته باشد. معمولاً تصاویر در حافظه فضای بیشتری نسبت به دیسک سخت اشغال می‌کنند، به‌ویژه وقتی اثرات اضافی بر روی آنها اعمال شده باشد.

به‌طور کلی، Aspose.Slides برای Java می‌تواند به‌راحتی فایل‌های ارائه حدود 300 مگابایت را روی سروری با 4 گیگابایت RAM پردازش کند.

## **کار با اسلایدها**

**س: آیا می‌توانم اندازه اسلایدهای یک ارائه را تغییر دهم؟**

**ج**: می‌توانید از متد `getSlideSize` موجود در کلاس [Presentation](https://reference.aspose.com/slides/fa/java/com.aspose.slides/presentation/) برای تعیین اندازه اسلایدهای ارائه استفاده کنید.

**س: آیا راهی برای تعریف اسلایدهایی با اندازه‌های مختلف در یک ارائه وجود دارد؟**

**ج**: از آنجایی که اندازه اسلایدها در سطح ارائه در اسناد Microsoft PowerPoint تعریف می‌شود، امکان این کار وجود ندارد.

**س: آیا Aspose.Slides برای Java امکان پیش‌نمایش یک اسلاید قبل از ذخیره‌سازی را دارد؟**

**ج**: می‌توانید اسلایدهای ارائه را به تصویر رندر کنید و از این تصاویر برای پیش‌نمایش اسلایدها استفاده کنید.

## **کار با متن**

**س: آیا می‌توان تمام متن موجود در یک ارائه را استخراج کرد؟**

**ج**: Aspose.Slides برای Java کلاس [SlideUtil](https://reference.aspose.com/slides/fa/java/com.aspose.slides/slideutil/) را فراهم می‌کند که روش‌های مختلفی برای استخراج کل متن از ارائه‌ها دارد.

**س: چرا اندازه پاراگراف‌ها در سیستم‌عامل‌های ویندوز و لینوکس متفاوت است؟**

**ج**: محاسبه اندازه پاراگراف‌ها بر اساس محاسبه اندازه متن نمایانگر پاراگراف انجام می‌شود. این محاسبه براساس معیارهای فونت مشخص‌شده در ارائه PowerPoint است. اگر فونت مشخص‌شده موجود نباشد، با فونتی که بیشترین شباهت را دارد جایگزین می‌شود، اما معیارهای این فونت متفاوت از فونت اصلی هستند. در نتیجه، محاسبه اندازه پاراگراف‌ها در سیستم‌های مختلف نتایج متفاوتی می‌دهد که به مجموعه فونت‌های نصب‌شده بستگی دارد. برای دستیابی به همان نتایج در سیستم‌عامل‌های مختلف، باید فونت‌های یکسان را بر روی سیستم‌ها نصب کنید یا آنها را در زمان اجرا به‌صورت [فونت‌های خارجی](/slides/fa/java/custom-font/) بارگذاری کنید.

## **قالب‌بندی و تصاویر**

**س: چگونه می‌توان رنگ حاشیه جدول را تنظیم کرد؟**

**ج**: می‌توانید رنگ تمام حاشیه‌های جدول یا فقط حاشیه اطراف کل جدول را تغییر دهید. برای تغییر تمام حاشیه‌ها، از متد `getCellFormat` در اینترفیس [ICell](https://reference.aspose.com/slides/fa/java/com.aspose.slides/icell/) استفاده کنید. برای تغییر حاشیه اطراف کل جدول، باید روی سلول‌ها مرور کنید و رنگ حاشیه‌های خارجی را تغییر دهید.

**س: Aspose.Slides برای Java برای قرار دادن تصاویر از چه مقیاسی استفاده می‌کند؟**

**ج**: مختصات و اندازه‌های همه اشکال روی اسلایدها بر حسب نقطه (72 dpi) اندازه‌گیری می‌شود.

## **کار با فونت‌ها**

**س: هنگام تبدیل PPT به PDF یا تصاویر، چرا فونت‌های خروجی متفاوت هستند؟**

**ج**: این مشکل ممکن است نشان‌دهنده این باشد که فونت‌های استفاده‌شده در ارائه بر روی سیستم‌عامل اجرا کننده کد موجود نیستند. باید فونت‌ها را بر روی سیستم‌عامل نصب کنید یا آنها را به‌صورت فونت‌های خارجی با استفاده از کلاس [FontsLoader](https://reference.aspose.com/slides/fa/java/com.aspose.slides/fontsloader/) بارگذاری کنید همان‌طور که در زیر نشان داده شده است:
```cs
var folders = new String[] { "path_to_a_folder_with_fonts" };
FontsLoader.loadExternalFonts(folders);
```
