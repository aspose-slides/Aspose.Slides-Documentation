## **مروری**

این مقاله توضیح می‌دهد که چگونه اسلایدهای ارائه را به‌عنوان تصاویر SVG با استفاده از Aspose.Slides رندر کنید. این مقاله فرمت SVG و مزایای آن از جمله مقیاس‌پذیری، دسترسی‌پذیری و مناسب بودن برای توسعه وب را شرح می‌دهد.

شما خواهید آموخت که چگونه یک فایل ارائه را بارگذاری کنید، در اسلایدهای آن حلقه بزنید و هر اسلاید را به‌صورت یک فایل SVG جداگانه ذخیره کنید. این مقاله به فرمت‌های ارائه PowerPoint و OpenDocument شامل PPT، PPTX، ODP و PPS می‌پردازد و نشان می‌دهد که تبدیل را به‌صورت برنامه‌نویسی با کلاس `Presentation` و متد `writeAsSvg` چگونه انجام دهید.

## **فرمت SVG**

SVG—مخفف Scalable Vector Graphics—یک نوع یا فرمت گرافیکی استاندارد برای رندر تصاویر دوبعدی است. SVG تصاویر را به‌صورت بردار در XML ذخیره می‌کند و جزئیاتی که رفتار یا ظاهر آنها را تعریف می‌کند، شامل می‌شود.

SVG یکی از معدود فرمت‌هایی است که در این موارد استانداردهای بسیار بالایی دارد: مقیاس‌پذیری، تعامل‌پذیری، عملکرد، دسترسی‌پذیری، برنامه‌نویسی و غیره. به همین دلایل، SVG به‌طور گسترده‌ای در توسعه وب استفاده می‌شود.

ممکن است بخواهید از فایل‌های SVG زمانی استفاده کنید که نیاز دارید:

- **چاپ ارائه خود در *فرمت بسیار بزرگ***. تصاویر SVG می‌توانند به هر وضوح یا سطحی مقیاس شوند. می‌توانید تصاویر SVG را هر تعداد که لازم است بدون از دست دادن کیفیت تغییر اندازه دهید.
- **از نمودارها و گراف‌ها در اسلایدهای خود در *رسانه‌ها یا پلتفرم‌های مختلف* استفاده کنید**. اکثر خوانندگان می‌توانند فایل‌های SVG را تفسیر کنند.
- **از *کوچک‌ترین اندازه‌های ممکن برای تصاویر* استفاده کنید**. فایل‌های SVG عموماً نسبت به معادل‌های با وضوح بالا در فرمت‌های دیگر به‌خصوص فرمت‌های مبتنی بر بیت‌مپ (JPEG یا PNG) کوچکتر هستند.

## **رندر یک اسلاید به عنوان تصویر SVG**

Aspose.Slides for Java به شما امکان می‌دهد اسلایدهای ارائه خود را به‌عنوان تصاویر SVG صادر کنید. برای تولید تصاویر SVG این مراحل را طی کنید:

1. یک نمونه از کلاس Presentation ایجاد کنید.
2. در تمام اسلایدهای ارائه حلقه بزنید.
3. هر اسلاید را از طریق FileOutputStream به فایل SVG مخصوص خود بنویسید.

{{% alert color="primary" %}} 
ممکن است بخواهید برنامه وب [رایگان ما](https://products.aspose.app/slides/fa/conversion/ppt-to-svg) را امتحان کنید که در آن عملکرد تبدیل PPT به SVG را از Aspose.Slides for Java پیاده‌سازی کرده‌ایم.
{{% /alert %}} 

این کد نمونه در Java نشان می‌دهد چگونه با Aspose.Slides PPT را به SVG تبدیل کنید:

``` java
Presentation pres = new Presentation("pres.pptx");
try {
    for (int index = 0; index < pres.getSlides().size(); index++)
    {
        ISlide slide = pres.getSlides().get_Item(index);

        FileOutputStream fileStream = new FileOutputStream("slide-" + index + ".svg");
        try {
            slide.writeAsSvg(fileStream);
        } finally {
            fileStream.close();
        }
    }
} catch(IOException e) {
} finally {
    if (pres != null) pres.dispose();
}
```

## **سوالات متداول**

**چرا ممکن است SVG تولید شده در مرورگرهای مختلف متفاوت دیده شود؟**  
پشتیبانی از ویژگی‌های خاص SVG توسط موتورهای مرورگر به‌صورت متفاوتی پیاده‌سازی می‌شود. پارامترهای [SVGOptions](https://reference.aspose.com/slides/fa/java/com.aspose.slides/svgoptions/) به هم‌راه کردن ناسازگاری‌ها کمک می‌کنند.

**آیا می‌توان نه تنها اسلایدها بلکه شکل‌های تک‌تکه را نیز به SVG صادر کرد؟**  
بله. هر [شکل می‌تواند به‌عنوان یک SVG جداگانه ذخیره شود](https://reference.aspose.com/slides/fa/java/com.aspose.slides/shape/#writeAsSvg-java.io.OutputStream-com.aspose.slides.ISVGOptions-)، که برای آیکن‌ها، پیکتوگرام‌ها و استفاده مجدد از گرافیک‌ها مناسب است.

**آیا می‌توان چندین اسلاید را در یک SVG (نوار/سند) ترکیب کرد؟**  
سناریوی استاندارد یک اسلاید → یک SVG است. ترکیب چندین اسلاید در یک بوم SVG یک گام پس‌پردازشی است که در سطح برنامه انجام می‌شود.
