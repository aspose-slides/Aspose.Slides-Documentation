## **نمای کلی**

Aspose.Slides از فایل‌های ارائه از Microsoft PowerPoint 97 تا Office 365، از جمله Microsoft PowerPoint برای مک، پشتیبانی می‌کند. این مقاله نسخه‌های PowerPoint پشتیبانی‌شده توسط کتابخانه را فهرست می‌کند و جدول فرمت‌های فایلی که می‌توانند بارگذاری، ذخیره یا هر دو شوند را ارائه می‌دهد.

این مقاله همچنین به سوالات رایج درباره سازگاری PDF، جاسازی فونت، فایل‌های محافظت‌شده با گذرواژه، فونت‌های سفارشی، جانشینی فونت و گزینه‌های خروجی XPS پاسخ می‌دهد.

## **نسخه‌های پشتیبانی‌شده Microsoft PowerPoint**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint برای مک
- Office 365

## **سوالات متداول**

**آیا می‌توانم ارائه‌ها را به PDF ذخیره کنم که استانداردهای بایگانی و دسترسی (PDF/A و PDF/UA) را برآورده کنند؟**

بله. Aspose.Slides امکان خروجی به PDF را با سطوح سازگاری مانند PDF/A-2a، PDF/A-2b، PDF/A-2u، PDF/A-3a، PDF/A-3b و همچنین PDF/UA از طریق تنظیم [compliance](https://reference.aspose.com/slides/fa/java/com.aspose.slides/pdfoptions/#setCompliance-int-) در [PDF export options](https://reference.aspose.com/slides/fa/java/com.aspose.slides/pdfoptions/) پشتیبانی می‌کند.

**آیا کتابخانه از جاسازی فونت هنگام خروجی به PDF پشتیبانی می‌کند و امکان کنترل دقیق روی آنچه جاسازی می‌شود را دارد؟**

بله. می‌توانید کنترل کنید که آیا فونت‌ها به طور کامل جاسازی شوند یا به صورت زیرمجموعه (فقط گلیف‌های استفاده‌شده) باشد، نحوهٔ رفتار فونت‌های سیستم عمومی را مشخص کنید و رفتار برای متن ASCII را از طریق [PDF export options](https://reference.aspose.com/slides/fa/java/com.aspose.slides/pdfoptions/) پیکربندی کنید.

**آیا می‌توانم قبل از بارگذاری واقعی، تشخیص دهم که آیا فایل با گذرواژه محافظت شده است؟**

بله. با استفاده از [API بازرسی مبتنی بر کارخانه](https://reference.aspose.com/slides/fa/java/com.aspose.slides/presentationfactory/) می‌توانید یک فایل ارائه را بررسی کنید تا تعیین کنید آیا با گذرواژه محافظت شده است بدون اینکه آن را به‌طور کامل باز کنید.

**آیا مکانیزم‌های جایگزینی فونت و پشتیبانی از فونت‌های سفارشی وجود دارد؟**

بله. کتابخانه از [بارگذاری](/slides/fa/java/custom-font/) و [جاسازی](/slides/fa/java/embedded-font/) فونت‌های سفارشی پشتیبانی می‌کند و قوانین [fallback](/slides/fa/java/fallback-font/) فونت را برای جلوگیری از گلیف‌های گمشده در رندرینگ و تبدیل فراهم می‌آورد.

**آیا می‌توانم اسلایدها را به XPS خروجی بگیرم و گزینه‌هایی برای تنظیم خروجی XPS وجود دارد؟**

بله. [خروجی به XPS](/slides/fa/java/convert-powerpoint-to-xps/) پشتیبانی می‌شود و می‌توانید گزینه‌های مربوط به [save options](https://reference.aspose.com/slides/fa/java/com.aspose.slides/xpsoptions/) را تنظیم کنید تا کیفیت خروجی و محتوای سند XPS را کنترل کنید.
