## **بررسی کلی**

این مقاله یک خطای مجوز در Aspose.Slides for Java نسخه‌های 23.2، 23.3، 23.4، 23.5، 23.6 و 23.7 را توضیح می‌دهد. این خطا باعث می‌شد مجوزهای دائمی پس از انقضای دورهٔ اشتراک موقتاً غیرقابل استفاده شوند.

## **علائم**

پس از پایان دورهٔ اشتراک مجوز دائمی شما، Aspose.Slides for Java نسخه‌های 23.2 تا 23.7 ممکن است مجوز را منقضی گزارش کند و تمام عملکردها را به حالت ارزیابی منتقل کند. این رفتار نادرست است و بر نسخه‌های قبل از 23.2 و پس از 23.7 تأثیری ندارد.

## **راه‌حل**

Aspose.Slides for Java این مشکل را برطرف کرد و نسخه‌های به‌روز شده (23.2.1، 23.3.1، 23.4.1، 23.5.1، 23.6.1، 23.7.1) را با اصلاح منتشر کرد.  

اگر شما یکی از نسخه‌های تحت تأثیر Aspose.Slides for Java را در پروژهٔ خود استفاده می‌کنید، لطفاً به نسخهٔ اصلاح‌شده سوئیچ کنید.

اگر با هر مشکلی در زمینهٔ مجوز Aspose.Slides for Java مواجه شدید:

- نسخه‌ای که استفاده می‌کنید را بررسی کنید و مطمئن شوید تحت تأثیر باگی که در بالا توصیف شد نیست.
- به [مستندات Aspose](https://docs.aspose.com/slides/fa/java/getting-started/) برای مراحل عیب‌یابی و مشکلات شناخته‌شدهٔ مجوز مراجعه کنید.
- برای دریافت کمک بیشتر، با [پشتیبانی Aspose](https://forum.aspose.com/) تماس بگیرید.

## **محصولات و نسخه‌های تحت تأثیر**

لطفاً توجه داشته باشید که این خطا فقط بر نسخه‌های 23.2 تا 23.7 Aspose.Slides for Java تأثیر می‌گذارد. **سایر محصولات Aspose و نسخه‌های دیگر Aspose.Slides تحت تأثیر نیستند**.
