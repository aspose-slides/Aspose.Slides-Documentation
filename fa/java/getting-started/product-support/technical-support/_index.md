## **بررسی کلی**

این مقاله توضیح می‌دهد که چگونه می‌توانید از پشتیبانی Aspose.Slides استفاده کنید و مشکلات را به‌طور مؤثر گزارش دهید. این مقاله کانال اصلی پشتیبانی را توصیف می‌کند، توصیه‌هایی برای ارسال یک مشکل ارائه می‌دهد و به منابع فنی مفیدی مانند صفحه محصول، وبلاگ، دانلودها، انجمن پشتیبانی، مستندات و مرجع API اشاره می‌کند. همچنین سوالات متداول درباره پشتیبانی پولی، ارتقاء مشکل و پشتیبانی از نسخه‌های قدیمی کتابخانه را پاسخ می‌دهد.

## **پشتیبانی فنی**

{{% alert color="primary" %}} 

Aspose پشتیبانی فنی نامحدود و رایگان را برای تمام محصولات خود ارائه می‌دهد. این پشتیبانی برای تمام کاربران، از جمله کاربران ارزیابی، در دسترس است.

{{% /alert %}} 

اگر به کمک برای Aspose.Slides for Java نیاز دارید، موارد زیر را در نظر بگیرید:

- مسیر اصلی پشتیبانی، انجمن Aspose.Forums است. سؤال خود را در انجمن Aspose.Slides ارسال کنید و در عرض چند ساعت پاسخی دریافت خواهید کرد. سؤالات مستقیماً توسط تیم پشتیبانی Aspose.Slides پاسخ داده می‌شوند.
- لطفاً توجه داشته باشید که Aspose پشتیبانی فنی از طریق تلفن ارائه نمی‌دهد. پشتیبانی تلفنی فقط برای سوالات فروش و خرید در دسترس است.
- هنگام انتظار برای پاسخ در انجمن‌ها، لطفاً تفاوت‌های زمانی را در نظر بگیرید.

اگر مشکلی با Aspose.Slides for Java دارید، مراحل ساده زیر را برای اطمینان از حل کارآمد آن دنبال کنید:

- قبل از گزارش مشکل، مطمئن شوید که از آخرین نسخه Aspose.Slides for Java استفاده می‌کنید؛ برای اطلاعات درباره آخرین نسخه به دانلود Aspose.Slides مراجعه کنید.
- قبل از گزارش مشکل، به انجمن‌ها، این مستندات و مرجع API نگاه کنید؛ شاید سؤال شما قبلاً پاسخ داده شده باشد.
- هنگام گزارش یک مشکل، لطفاً سند اصلی و در صورت امکان بخشی از کد شما که باعث مشکل می‌شود را ضمیمه کنید. اگر نیاز به پیوست چندین فایل دارید، آن‌ها را در یک فایل zip فشرده کنید. پیوست اسناد در Aspose.Forums ایمن است زیرا فقط شما و توسعه‌دهندگان Aspose به فایل‌های پیوست‌شده دسترسی دارند.
- لطفاً سعی کنید یک مشکل را در هر موضوع (thread) گزارش کنید. اگر مشکل دیگری دارید، آن را در یک موضوع جداگانه گزارش کنید.

**سطوح پشتیبانی پولی موجود چه هستند و چگونه متفاوت‌اند؟**

**آیا می‌توانم یک باگ را به‌صورت مستقیم به تیم مهندسی ارجاع دهم؟**

بله. [Paid support](https://helpdesk.aspose.com/) امکان ارتقاء به توسعه‌دهندگان را زمانی که یک مشکل نیاز به بررسی عمیق‌تر دارد، فراهم می‌کند.

**آیا نسخه‌های قدیمی کتابخانه پشتیبانی می‌شوند و آیا برای آن‌ها اصلاحاتی منتشر می‌کنید؟**
