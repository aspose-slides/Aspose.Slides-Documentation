[مشخصات تجزیه‌گر XML](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) یک زبان توصیف صفحه و فرمت سند ثابت است که در اصل توسط مایکروسافت توسعه یافته است. مانند PDF، XPS برای حفظ صحت سند و ارائه ظاهر مستقل از دستگاه طراحی شده است. 

{{% /alert %}} 

## **XPS در Aspose.Slides for Java**
هر سند ارائه‌ای که بتواند توسط Aspose.Slides for Java بارگذاری شود می‌تواند به فرمت XPS تبدیل شود. Aspose.Slides for Java از موتور چینش صفحه و رندرینگ با دقت بالا استفاده می‌کند تا خروجی را در قالب سند XPS با چیدمان ثابت تولید کند.
می‌توانید درباره صادرات اسناد ارائه به اسناد XPS از طریق Aspose.Slides for Java در [تبدیل به XPS](https://docs.aspose.com/slides/fa/java/convert-powerpoint-to-xps/) بیاموزید.
