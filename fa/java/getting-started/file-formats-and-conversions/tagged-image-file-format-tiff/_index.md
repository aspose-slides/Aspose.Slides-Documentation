## **درباره TIFF**

The [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) که به‌خاطر ذخیره‌سازی چندین تصویر در یک سند شناخته می‌شود و در اصل توسط Aldus ایجاد شد. این قالب به‌طور گسترده‌ای توسط برنامه‌های اسکن، فکس و سایر برنامه‌های پردازش تصویر پشتیبانی می‌شود.

## **TIFF در Aspose.Slides برای جاوا**

هر سندی که می‌تواند در Aspose.Slide برای جاوا بارگذاری شود، می‌تواند مستقیماً توسط Aspose.Slides برای جاوا به سند TIFF تبدیل شود و نیازی به هیچ مؤلفه شخص ثالثی ندارد. علاوه بر این، می‌توانید به‌صورت اختیاری اندازه تصاویر را در سند TIFF حاصل تعریف کنید. می‌توانید اطلاعات درباره خروجی‌گیری اسناد ارائه به اسناد TIFF از طریق Aspose.Slides برای جاوا را در [this topic](/slides/fa/java/converting-a-presentation/) پیدا کنید.
