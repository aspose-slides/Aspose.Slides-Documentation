## **به Aspose.Slides for Java خوش آمدید!**

Aspose.Slides for Java یک API مدیریت Microsoft PowerPoint® است که به برنامه‌های Java امکان خواندن و نوشتن اسناد PowerPoint® را بدون نیاز به Microsoft PowerPoint® می‌دهد.

Aspose.Slides for Java اولین و تنها مؤلفه‌ای است که عملکردهای لازم برای مدیریت اسناد PowerPoint® را فراهم می‌کند.

Aspose.Slides for Java ویژگی‌های کلیدی زیادی مانند مدیریت متن، اشکال، جداول و انیمیشن‌ها، افزودن صدا و ویدیو به اسلایدها، پیش‌نمایش اسلایدها، خروجی اسلایدها به SVG، PDF و سایر فرمت‌ها را ارائه می‌دهد.

## **شرح محصول**

Aspose.Slides for Java نه تنها به شما اجازه می‌دهد فایل‌های PowerPoint را از منابع مختلف باز کنید، بلکه امکان ذخیره ارائه‌های خود را به روش‌های گوناگون نیز می‌دهد. به عنوان مثال می‌توانید ارائه‌های خود را به فرمت PPT ذخیره کنید؛ همچنین می‌توانید اسلایدهای خود را به تصویر تبدیل کنید.

Aspose.Slides for Java تقریباً تمام ویژگی‌های موجود در Microsoft PowerPoint را در اختیار شما می‌گذارد. با توجه به انعطاف‌پذیری Aspose.Slides for Java، علاوه بر جداول و انواع مختلف اشکال، می‌توانید فریم‌های مختلفی مانند متن، صدا و ویدیو را به اسلایدهای خود اضافه کنید.

Aspose.Slides for Java همچنین نمونه‌های کامل و دموی کاربردی برای توسعه‌دهندگان فراهم می‌کند تا درک بهتری از API به دست آورند.

### **ویژگی‌های ارائه**

- باز کردن ارائه‌های Microsoft PowerPoint از منابع مختلف.
- ذخیره ارائه‌ها در فرمت‌های مختلف Microsoft PowerPoint.
- ایجاد ارائه‌ها از ابتدا.
- تبدیل ارائه‌ها به اسناد PDF بدون استفاده از هیچ مؤلفه دیگری.
- توانایی خواندن Open Document Presentation (ODP).
- توانایی تبدیل PPT به PPTX یا بالعکس.
- خواندن یا اصلاح خصوصیات سند یک ارائه.
- خروجی ارائه‌ها به PDF
- خروجی ارائه‌ها به TIFF
- خروجی ارائه‌ها به XPS
- خروجی ارائه‌ها به SWF
- خروجی ارائه‌ها به PDF Note / Tiff Notes
- خروجی ارائه‌ها به HTML

### **ویژگی‌های اسلاید**

- ایجاد اسلایدهای جدید از ابتدا و افزودن آنها به ارائه‌ها.
- دسترسی به هر اسلاید از طریق موقعیت آن در ارائه.
- کپی یا کلون کردن اسلایدها به همان ارائه یا به ارائه دیگر.
- ویرایش و حذف اسلایدها.
- کپی خودکار اسلایدهای اصلی همراه با اسلایدها.
- پیش‌نمایش اسلایدها.
- دریافت یا تنظیم نام اسلاید.
- ذخیره اسلایدها به عنوان تصویر.
- خروجی اسلایدها به فرمت SVG.
- کار با چندین اسلاید اصلی در ارائه‌ها.
- پر کردن پس‌زمینه اسلایدها با هر سبک پشتیبانی‌شده‌ای.
- افزودن یا مدیریت یادداشت‌های اسلاید.
- تنظیم دینامیک اسلایدهای اصلی و سبک‌ها.
- افزودن یا اصلاح انتقال اسلایدشو.
- افزودن انتقال‌های مورف

### **ویژگی‌های گرافیک و چندرسانه‌ای**

- ایجاد یا مدیریت اشکالی مانند خطوط، مستطیل‌ها، بیضی‌ها و پلی‌لاین‌ها.
- مدیریت سبک‌های خط در اشکال.
- دریافت متن جایگزین یک شکل.
- چرخاندن یا وارونه کردن اشکال.
- پر کردن اشکال با سبک‌های مختلف، به‌عنوان مثال جامد، گرادیان، الگو، بافت و تصویر.
- استفاده از هر یک از ۴۸ الگوی پیش‌تعریف‌شده و ۲۴ بافت.
- اختصاص دادن لینک به اشکال.
- خواندن، اصلاح و افزودن اثرهای انیمیشنی جدید به اسلایدها و اشکال.
- افزودن تصاویر جدید به ارائه‌ها و مدیریت تصاویر موجود.
- افزودن گلوله‌های تصویری جدید به ارائه‌ها و مدیریت آنها.
- افزودن فریم‌های صوتی جدید به ارائه‌ها و مدیریت آنها.
- افزودن فریم‌های ویدیویی جدید به ارائه‌ها و مدیریت آنها.

### **ویژگی‌های جدول**

- ایجاد جداول از ابتدا و افزودن آنها به اسلایدها.
- تغییر اندازه و جابجایی جداول.
- دسترسی به هر سلول در جدول.
- تغییر قالب‌بندی و مرزهای هر سلول.
- تغییر اندازه ستون‌ها و ردیف‌ها.
- افزودن و حذف ستون‌ها و ردیف‌ها.
- تنظیم عرض و رنگ مرز برای کل جدول.
- اعمال استایل‌های پیش‌فرض به جداول

### **ویژگی‌های متن و پاراگراف**

- دریافت و تنظیم متن‌ها در مکان‌دارهای متنی و فریم‌های متنی.
- دریافت نوع یک مکان‌دار.
- تغییر قالب‌بندی متن در یک فریم متنی.
- افزودن فونت جدید یا مدیریت فونت‌های استفاده‌شده در یک ارائه.
- تغییر متن‌ها و قالب‌بندی آن‌ها.
- ایجاد یا مدیریت لینک‌ها به متن.
- پشتیبانی از تب‌ها و تب‌های مؤثر.
- افزودن فریم‌های متنی جدید به تقریباً هر نوع شکلی.
- دسترسی و اصلاح حاشیه‌ها و تنظیمات پیچش متن در فریم‌های متنی.
- دسترسی و اصلاح تو رفتگی پاراگراف.

### **ویژگی‌های پیشرفته**

- دسترسی به اشیاء OLE جاسازی‌شده در ارائه‌ها.
- قفل کردن ارائه‌ها.
- افزودن واترمارک به ارائه‌ها.
- اعمال انیمیشن به اشکال داخل اسلایدها.

## **پرسش‌های متداول**

**آیا از سطوح سازگاری PDF برای بایگانی و دسترس‌پذیری (PDF/A و PDF/UA) پشتیبانی می‌کند؟**

Yes. You can save to PDF with PDF/A-2a/2b/2u, PDF/A-3a/3b, as well as PDF/UA by configuring [PDF export options](https://reference.aspose.com/slides/fa/java/com.aspose.slides/pdfoptions/).

**آیا مکانیزم جایگزینی فونت و پشتیبانی از فونت‌های سفارشی برای رندر صحیح موجود است؟**

Yes. The library lets you [load custom fonts](/slides/fa/java/custom-font/) and [define fallback rules](/slides/fa/java/fallback-font/) so missing glyphs are substituted reliably during conversion and rendering.

**آیا می‌توانم بدون باز کردن کامل فایل، تشخیص دهم که آیا فایل رمزعبور دارد؟**

Yes. You can [inspect a presentation](/slides/fa/java/examine-presentation/) to determine if it requires a password before loading the full document.

**آیا برای پردازش و تبدیل‌ها نیاز به Microsoft PowerPoint است؟**

No. Aspose.Slides is a standalone engine; PowerPoint is not needed on the server or workstation.

**آیا پردازش ارائه‌ها از چندین رشته (thread) ایمن است؟**

Yes, you can process different documents in parallel in separate threads; just avoid using the same presentation instance concurrently [across threads](/slides/fa/java/multithreading/).

**آیا ماکروها حفظ می‌شوند و می‌توانم VBA را در فایل‌های PPTM/PPSM مدیریت کنم؟**

Yes. Presentations with macros [are supported](/slides/fa/java/presentation-via-vba/), and you can [inspect and manage VBA projects](https://reference.aspose.com/slides/fa/java/com.aspose.slides/presentation/#getVbaProject--) in those files.

**آیا می‌توانم PDF یا HTML را به اسلایدهای PowerPoint تبدیل کنم؟**

Yes. You can [import PDF pages or HTML content](/slides/fa/java/import-presentation/) to create or populate slides in a presentation.

**آیا خروجی XPS پشتیبانی می‌شود و می‌توانم کیفیت و محتویات خروجی XPS را کنترل کنم؟**

Yes. [Export to XPS](/slides/fa/java/convert-powerpoint-to-xps/) is available, and [save options](https://reference.aspose.com/slides/fa/java/com.aspose.slides/xpsoptions/) allow you to tune output quality and included content.

**آیا می‌توانم اسلایدها را به تصویر تبدیل کنم و کیفیت خروجی را کنترل کنم؟**

Yes. Slides can be [rendered to PNG, JPEG, GIF, BMP, TIFF](/slides/fa/java/convert-powerpoint-to-png/) and other formats with fine-grained control over size and quality.

**آیا فرمول‌های ریاضی (MathML/MathText) پشتیبانی می‌شوند و می‌توان آنها را خروجی داد؟**

Yes. You can [create and edit mathematical text](/slides/fa/java/powerpoint-math-equations/) and [export equations](/slides/fa/java/exporting-math-equations/), including to MathML.

**کتابخانه چگونه رمزنگاری و پسوردها (باز کردن، تنظیم، حذف) را مدیریت می‌کند؟**

[It supports](/slides/fa/java/password-protected-presentation/) opening encrypted presentations, setting or removing passwords for opening and modifying, and checking a file’s protection status.
