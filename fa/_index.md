## Aspose.Slides برای .NET

{{% alert color="primary" %}}

![لوگو محصول Aspose.Slides برای .NET](home_1.png)

Aspose.Slides for .NET یک API مدیریت Microsoft PowerPoint® است که به برنامه‌های .NET امکان خواندن و نوشتن اسناد PowerPoint® را بدون استفاده از Microsoft PowerPoint® می‌دهد. Aspose.Slides for .NET اولین و تنها مؤلفه‌ای است که قابلیت مدیریت اسناد PowerPoint® را فراهم می‌کند. Aspose.Slides for .NET ویژگی‌های کلیدی بسیاری مانند مدیریت متن، اشکال، جداول و انیمیشن‌ها، افزودن صدا و ویدئو به اسلایدها، پیش‌نمایش اسلایدها، استخراج اسلایدها به فرمت SVG، PDF و موارد دیگر را ارائه می‌دهد.
