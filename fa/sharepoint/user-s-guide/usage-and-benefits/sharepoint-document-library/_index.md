یک کتابخانه ساختار محتوا است که حاوی فایل‌ها می‌باشد. یک کتابخانه می‌تواند فقط یک نوع فایل یا انواع مختلفی از فایل‌ها، از جمله اسناد، صفحات‑گسترده و ارائه‌ها، را شامل شود. یک کتابخانهٔ SharePoint مکان واحدی ایجاد می‌کند که افراد می‌توانند اسناد را بارگذاری و دسترسی داشته باشند. اسناد به‌صورت باینری در پایگاه دادهٔ SQL Server که توسط سایت SharePoint استفاده می‌شود، ذخیره می‌شوند. این اسناد برای سایر کاربران SharePoint به‌صورت فهرستی از پیوندهای هیپرمتن در دسترس هستند.
