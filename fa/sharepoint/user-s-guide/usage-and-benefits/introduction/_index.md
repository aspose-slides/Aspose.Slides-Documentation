سایت‌های SharePoint برای همکاری، ارتباط و ذخیره‌سازی محتوا به منظور برآورده کردن نیازهای خاص تجاری طراحی شده‌اند. این پلتفرم به یک گروه از افراد اجازه می‌دهد تا با تسهیل به اشتراک‌گذاری اطلاعات و ارتباط، به هدف مشترک دست یابند. معمولاً این اطلاعات به صورت یک وب‌سایت در دسترس هستند و از طریق مرورگر وب قابل دسترسی‌اند. 

{{% /alert %}} 
## **SharePoint و Aspose.Slides چیست؟**
Aspose.Slides for SharePoint یک راه حل انعطاف‌پذیر برای تبدیل اسناد Microsoft PowerPoint به فرمت‌های دیگر مانند PPT، POT، PPS، PPTX، PPSX و POTX است. Aspose.Slides for SharePoint به شما امکان می‌دهد فایل‌های PowerPoint را در یک برنامه SharePoint بدون استفاده از Microsoft PowerPoint بخوانید و تبدیل کنید.
