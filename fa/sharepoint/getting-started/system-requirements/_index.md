## **نیازهای سرور**
{{% alert color="primary" %}} 

برای نصب Aspose.Slides برای SharePoint، سیستمی که یکی از موارد زیر را اجرا می‌کند لازم است:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft Office SharePoint Server 2010
- Microsoft Office SharePoint Server 2013
- Microsoft Office SharePoint Server 2016
- Microsoft Office SharePoint Server 2019

{{% /alert %}} {{% alert color="primary" %}} 

نصب Service Pack 2 یا بالاتر برای این محصولات توصیه می‌شود.

{{% /alert %}} 
## **نیازهای کلاینت**
{{% alert color="primary" %}} 

سایت‌های SharePoint از طریق مرورگر وب دسترسی می‌یابند. Aspose.Slides برای SharePoint با مرورگرهای زیر آزمایش شده است:

- Microsoft Internet Explorer 6, 7, 8
- Mozilla FireFox 3.5

{{% /alert %}}
