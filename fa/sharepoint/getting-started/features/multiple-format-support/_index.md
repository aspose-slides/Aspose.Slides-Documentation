{{% alert color="primary" %}} 

با Aspose.Slides for SharePoint، می‌توانید اسناد را بین بسیاری از فرمت‌های محبوب اسناد آفیس از داخل کتابخانه اسناد SharePoint تبدیل کنید. تبدیل‌ها با دقت و صحت بالا انجام می‌شوند. 

{{% /alert %}} 
## **قالب‌های ورودی پشتیبانی‌شده**
Aspose.Slides for SharePoint قالب‌های ورودی زیر را پشتیبانی می‌کند: 

- PPT – Microsoft PowerPoint Presentation 97 - 2003
- PPS – Microsoft PowerPoint SlideShow 97 - 2003
- POT – Microsoft PowerPoint Template 97 - 2003
- PPTX – Office Open XML Presentation
- PPSX – Office Open XML SlideShow
- POTX – Office Open XML Template

{{% alert color="primary" %}} 

برای تولید اسناد، Aspose.Slides for SharePoint بر نسخه داخلی [Aspose.Slides for .NET](http://www.aspose.com/categories/.net-components/aspose.slides-for-.net/default.aspx) تک‌مولفه پردازش اسناد PowerPoint شرکت Aspose تکیه دارد.

{{% /alert %}}
## **قالب‌های خروجی پشتیبانی‌شده**
قالب‌های خروجی زیر توسط Aspose.Slides for SharePoint پشتیبانی می‌شوند: 

- PDF – Portable Document Format
- TIFF – Pictures Package
- XPS – XML Paper Specification
