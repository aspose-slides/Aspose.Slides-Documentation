Aspose.Slides for Node.js via .NET یک کتابخانهٔ کلاسی است که به برنامه‌های Node.js امکان می‌دهد اسناد ارائه را بدون استفاده از Microsoft PowerPoint® یا سایر برنامه‌ها بخوانند و بنویسند. Aspose.Slides for Node.js via .NET به شما اجازه می‌دهد متن، اشکال، جداول و انیمیشن‌ها را مدیریت کنید، صدا و فیلم به اسلایدها اضافه کنید، پیش‌نمایش اسلایدها را ببینید، اسلایدها را به فرمت SVG و PDF صادر کنید و کارهای بسیار دیگری انجام دهید.

Aspose.Slides for Node.js via .NET از [همان مرجع API که Aspose.Slides .NET](https://reference.aspose.com/slides/fa/nodejs-net/) استفاده می‌کند.
