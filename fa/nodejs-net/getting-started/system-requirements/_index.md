Aspose.Slides برای Node.js از طریق .NET نیازی به نصب هیچ محصول شخص ثالثی مانند Microsoft PowerPoint ندارد. خود Aspose.Slides یک موتور برای ایجاد، ویرایش، تبدیل و رندر اسناد در فرمت‌های مختلف است که شامل فرمت‌های ارائه Microsoft PowerPoint نیز می‌شود.

## سیستم‌عامل‌های پشتیبانی‌شده

Aspose.Slides برای .NET هر سیستم‌عامل 32 بیتی یا 64 بیتی که Runtime .NET را اجرا می‌کند، از جمله اما نه محدود به، پشتیبانی می‌کند.
