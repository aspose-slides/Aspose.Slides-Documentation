## **پشتیبانی فنی**
{{% alert color="primary" %}}

Aspose پشتیبانی فنی رایگان و نامحدود برای تمام محصولات خود ارائه می‌دهد. این پشتیبانی برای همه کاربران (از جمله کاربرانی که بسته‌های ارزیابی دارند) در دسترس است.

{{% /alert %}} 

زمانی که به کمک برای Aspose.Slides for Node.js via .NET نیاز دارید، موارد زیر را در نظر بگیرید:

- اصلی‌ترین مسیر پشتیبانی، [Aspose.Forums](https://forum.aspose.com/c/slides/fa/11) است. سؤال خود را در انجمن Aspose.Slides ارسال کنید—و طی چند ساعت پاسخ دریافت خواهید کرد. تیم پشتیبانی Aspose.Slides به‌صورت مستقیم به سؤالات ارسال‌شده در انجمن پاسخ می‌دهد.
- لطفاً توجه داشته باشید که Aspose پشتیبانی فنی از طریق تلفن ارائه نمی‌دهد. پشتیبانی تلفنی فقط برای سؤالات مربوط به فروش و خرید در دسترس است.
- هنگامی که انتظار پاسخ در انجمن‌ها را دارید، لطفاً صبور باشید و اختلافات منطقه زمانی را در نظر بگیرید.

وقتی با مشکلی در Aspose.Slides for Node.js via .NET مواجه شدید، برای حل مؤثر آن، نکات زیر را دنبال کنید:

- قبل از گزارش مشکل، بررسی و تأیید کنید که از آخرین نسخه Aspose.Slides for Node.js via .NET استفاده می‌کنید. شاید مشکلی که تجربه کردید قبلاً در یک به‌روزرسانی رفع شده باشد.  

  برای دریافت اطلاعات درباره آخرین نسخه، به [Aspose.Slides download](https://releases.aspose.com/slides/fa/nodejs-net/) مراجعه کنید.

- قبل از گزارش مشکل، ممکن است بخواهید انجمن‌های ما، این مستندات و مرجع API را بررسی کنید. شاید پاسخ سؤال شما قبلاً ارائه شده باشد. 

- در هنگام گزارش مشکل، لطفاً سند اصلی و (در صورت امکان) بخشی از کد مرتبط با مشکل را شامل کنید. اگر نیاز به پیوست چندین فایل دارید، آن‌ها را در یک بسته ZIP قرار دهید. می‌توانید اسناد خود را به‌صورت ایمن در Aspose.Forums پیوست کنید زیرا **فقط** شما و توسعه‌دهندگان Aspose به فایل‌های پیوست‌شده دسترسی دارند.

- برای هر مشکل خاص یک رشته (thread) ایجاد و استفاده کنید. اگر مشکل دیگری دارید، باید رشتهٔ دیگری ایجاد کنید تا آن را گزارش دهید.
