## **پلتفرم‌های پشتیبانی‌شده**
Aspose.Slides for Node.js via .NET یک API مستقل از پلتفرم است. می‌توان آن را در هر پلتفرمی (Windows, Linux, macOS و غیره) که Node.js 7 یا بالاتر نصب شده باشد استفاده کرد. Oracle JDK 8 یا بالاتر نیز یک پیش‌نیاز دیگر است.

## **فرمت‌های فایل و تبدیل‌ها**
Aspose.Slides for Node.js via .NET بیشتر فرمت‌های سند PowerPoint را پشتیبانی می‌کند. همچنین امکان خروجی گرفتن به فرمت‌های محبوبی که سازمان‌ها برای به‌اشتراک‌گذاری استفاده می‌کنند را فراهم می‌آورد. جزئیات را بررسی کنید:

|**ویژگی**|**توضیح**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via .NET سریع‌ترین پردازش را برای این فرمت سند ارائه فراهم می‌کند.|
|تبدیل PPT به PPTX|Aspose.Slides for Node.js via .NET از تبدیل PPT به PPTX پشتیبانی می‌کند|
|Portable Document Format (PDF)|می‌توانید تمام فرمت‌های فایل پشتیبانی‌شده را با یک متد به اسناد Adobe Portable Document Format (PDF) صادر کنید.|
|XML Parser Specification (XPS)|می‌توانید تمام فرمت‌های فایل پشتیبانی‌شده را با یک متد به اسناد XML Parser Specification (XPS) صادر کنید.|
|Tagged Image File Format (TIFF)|می‌توانید تمام فرمت‌های ارائه پشتیبانی‌شده را به فرمت Tagged Image File Format (TIFF) صادر کنید.|
|تبدیل PPTX به HTML|Aspose.Slides for Node.js via .NET از تبدیل PresentationEx به فرمت HTML پشتیبانی می‌کند.|

## **رندرینگ و چاپ**
Aspose.Slides for Node.js via .NET رندر با وفاداری بالا برای اسلایدهای ارائه به قالب‌های گرافیکی مختلف را پشتیبانی می‌کند. جزئیات را بررسی کنید:

|**ویژگی**|**توضیح**|
| :- | :- |
|فرمت‌های تصویر پشتیبانی‌شده توسط .NET|با استفاده از Aspose.Slides for Node.js via .NET می‌توانید اسلایدهای ارائه و تصاویر روی اسلایدها را به تمام فرم‌ت‌های گرافیکی پشتیبانی‌شده توسط .NET مانند TIFF, PNG, BMP, JPEG, GIF رندر کنید.|
|فرمت SVG|Aspose.Slides for Node.js via .NET همچنین متدهای داخلی فراهم می‌کند که امکان خروجی گرفتن اسلایدهای ارائه به فرمت Scalable Vector Graphics (SVG) را می‌دهد.|
|چاپ ارائه|نسخه‌های جدید Aspose.Slides for Node.js via .NET متدهای چاپ داخلی با گزینه‌های مختلف را ارائه می‌دهند.|
