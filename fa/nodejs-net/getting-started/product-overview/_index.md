## **توضیح محصول**

Aspose.Slides for Node.js via .NET یک کتابخانه کلاسی چند‑پلتفرمی است که به برنامه‌های شما امکان انجام طیفی وسیع از وظایف پردازش اسناد را بدون استفاده از Microsoft PowerPoint® می‌دهد.

### **چرا باید از Aspose.Slides for Node.js via .NET استفاده کنید**

- Aspose.Slides for Node.js via .NET اولین و تنها مؤلفه‌ای است که عملکردهای مورد نیاز برای مدیریت اسناد PowerPoint® را فراهم می‌کند.  
- Aspose.Slides for Node.js via .NET ویژگی‌های کلیدی فراوانی از جمله مدیریت متن‌ها، اشکال، جداول و انیمیشن‌ها، افزودن صدا و ویدئو به اسلایدها، پیش‌نمایش اسلایدها، صادر کردن اسلایدها به SVG، PDF و سایر فرمت‌ها و قابلیت‌های بیشتر را ارائه می‌دهد.  
- Aspose.Slides for Node.js via .NET نه تنها به شما اجازه می‌دهد فایل‌های PowerPoint را از منابع مختلف باز کنید، بلکه امکان ذخیره ارائه‌های شما را به روش‌های متنوعی فراهم می‌کند. به عنوان مثال می‌توانید ارائه‌های خود را به فرمت PPT ذخیره کنید؛ همچنین می‌توانید اسلایدهای خود را به عنوان تصویر ذخیره کنید.  
- Aspose.Slides for Node.js via .NET تقریباً تمام ویژگی‌هایی را که ممکن است در Microsoft PowerPoint پیدا کنید، در اختیار شما می‌گذارد. با توجه به انعطاف‌پذیری Aspose.Slides for Node.js via .NET، علاوه بر جداول و انواع مختلف اشکال، می‌توانید فریم‌های مختلفی شامل متن، صدا و ویدئو را به اسلایدهای خود اضافه کنید.  

## **ویژگی‌های Aspose.Slides**

ویژگی‌های Aspose.Slides for Node.js via .NET می‌توانند به این گروه‌ها تقسیم شوند:

- ویژگی‌های ارائه
- ویژگی‌های اسلاید
- ویژگی‌های گرافیک و چندرسانه‌ای
- ویژگی‌های جدول
- ویژگی‌های متن و پاراگراف
- ویژگی‌های پیشرفته

### **ویژگی‌های ارائه**

- باز کردن ارائه‌های Microsoft PowerPoint از منابع مختلف
- ذخیره ارائه‌ها در فرمت‌های مختلف Microsoft PowerPoint
- ایجاد ارائه‌ها از صفر
- تبدیل ارائه‌ها به اسناد PDF بدون استفاده از مؤلفه دیگری
- خواندن یا اصلاح ویژگی‌های سند ارائه
- اعمال حفاظت بر روی ارائه
- صادرات ارائه به PDF
- صادرات ارائه به TIFF
- صادرات ارائه به XPS
- صادرات ارائه به SWF
- صادرات ارائه به PDF Note / Tiff Notes
- صادرات ارائه به HTML

### **ویژگی‌های اسلاید**

- ایجاد اسلایدهای جدید از صفر و افزودن آنها به ارائه‌ها
- دسترسی به هر اسلاید بر اساس موقعیت آن در ارائه
- کپی یا تکثیر اسلایدها به همان ارائه یا ارائه دیگری
- ویرایش و حذف اسلایدها
- کپی خودکار مسترهای اسلاید همراه با اسلایدها
- پیش‌نمایش اسلایدها
- دریافت یا تنظیم نام اسلاید
- ذخیره اسلایدها به عنوان تصویر
- صادرات اسلایدها به فرمت SVG
- کار با مسترهای متعدد در ارائه‌ها
- پر کردن پس‌زمینه اسلایدها با هر سبک پشتیبانی‌شده
- افزودن یا مدیریت یادداشت‌های اسلاید
- تنظیم دینامیک مسترها و سبک‌های جدید اسلاید
- افزودن یا اصلاح انتقال نمایش اسلاید
- افزودن انتقال‌های مورف

### **ویژگی‌های گرافیک و چندرسانه‌ای**

- ایجاد یا مدیریت اشکالی مانند خطوط، مستطیل‌ها، بیضی‌ها و چندخطی‌ها
- مدیریت سبک خطوط در اشکال
- دریافت متن جایگزین یک شکل
- چرخش و وارونگی اشکال
- پر کردن اشکال با سبک‌های مختلف – برای مثال ثابت، گرادیان، الگو، بافت و تصویر
- استفاده از هر یک از ۴۸ الگوی پیش‌تعریف‌شده و ۲۴ بافت
- اختصاص لینک‌ها به اشکال
- خواندن، اصلاح و افزودن افکت‌های انیمیشن جدید به اسلایدها و اشکال
- افزودن تصاویر جدید به ارائه‌ها و مدیریت تصاویر موجود
- افزودن گلوله‌های تصویری جدید به ارائه‌ها و مدیریت آنها
- افزودن فریم‌های صوتی جدید به ارائه‌ها و مدیریت آنها
- افزودن فریم‌های ویدئویی جدید به ارائه‌ها و مدیریت آنها

### **ویژگی‌های جدول**

- ایجاد جداول از صفر و افزودن آنها به اسلایدها
- تغییر اندازه و جابه‌جایی جداول
- دسترسی به هر سلول در جدول
- تغییر قالب‌بندی و حاشیه‌های هر سلول
- تغییر اندازه ستون‌ها و ردیف‌ها
- افزودن و حذف ستون‌ها و ردیف‌ها
- تنظیم عرض و رنگ حاشیه برای کل جدول

### **ویژگی‌های متن و پاراگراف**

- دریافت و تنظیم متن در جاگیرهای متن و فریم‌های متنی
- دریافت نوع یک جاگیر
- تغییر قالب‌بندی متن در یک فریم متنی
- افزودن قلم‌های جدید یا مدیریت قلم‌های مورد استفاده در ارائه
- تغییر متن و قالب‌بندی متنی
- ایجاد یا مدیریت لینک‌ها به متن
- پشتیبانی از تب‌ها و تب‌های مؤثر
- افزودن فریم‌های متنی جدید به تقریباً هر نوع شکلی
- دسترسی و اصلاح حاشیه‌ها و تنظیمات عبور متن در فریم‌های متنی
- دسترسی و اصلاح تورفتگی پاراگراف

### **ویژگی‌های پیشرفته**

- دسترسی به اشیاء OLE جاسازی‌شده در ارائه‌ها
- قفل کردن ارائه‌ها

## **پشتیبانی فنی**

Aspose پشتیبانی فنی نامحدود و رایگان برای تمام محصولات خود ارائه می‌دهد. این پشتیبانی برای همه کاربران (از جمله کاربران با بسته‌های ارزیابی) در دسترس است. اگر به کمک برای Aspose.Slides for Node.js via .NET نیاز دارید، موارد زیر را در نظر بگیرید:

- مسیر اصلی پشتیبانی، [انجمن‌های Aspose](https://forum.aspose.com/) است. سؤال خود را در [انجمن Aspose.Slides](https://forum.aspose.com/c/slides/fa/11) ارسال کنید—و در عرض چند ساعت پاسخ دریافت خواهید کرد. تیم پشتیبانی Aspose.Slides به‌طور مستقیم به سؤالات ارسال‌شده در انجمن پاسخ می‌دهد.  
- لطفاً توجه داشته باشید که Aspose پشتیبانی فنی از طریق تلفن ارائه نمی‌دهد. پشتیبانی تلفنی فقط برای سؤالات فروش و خرید در دسترس است.  
- هنگام انتظار برای پاسخ در انجمن‌ها، صبور باشید و تفاوت‌های زمانی را در نظر بگیرید.

اگر با Aspose.Slides for Node.js via .NET مشکلی دارید، برای حل آن به‌صورت مؤثر، دستورالعمل‌های زیر را دنبال کنید:

- ابتدا بررسی کنید و تأیید کنید که از آخرین نسخه Aspose.Slides for Node.js via .NET استفاده می‌کنید قبل از گزارش مشکل. شاید مشکلی که تجربه کرده‌اید در به‌روزرسانی‌ای قبلاً رفع شده باشد.  

  برای دریافت اطلاعات درباره آخرین نسخه محصول، به [بارگیری Aspose.Slides](https://releases.aspose.com/slides/fa/nodejs-net/) مراجعه کنید.

- قبل از گزارش مشکل، ممکن است بخواهید انجمن‌های ما، این مستندات و مرجع API را مرور کنید. شاید پاسخ سؤال شما قبلاً ارائه شده باشد.  

- هنگام گزارش مشکل، لطفاً سند اصلی و (در صورت امکان) بخشی از کد مرتبط با مشکل را ضمیمه کنید. اگر نیاز به پیوست چندین فایل دارید، آنها را در یک بسته ZIP قرار دهید.  

  می‌توانید به‌سرچانه اسناد خود را در Aspose.Forums پیوست کنید زیرا فقط شما و توسعه‌دهندگان Aspose به فایل‌های پیوست‌شده دسترسی دارند.

- برای یک مشکل خاص، یک موضوع (thread) ایجاد و استفاده کنید. اگر مشکل دیگری پیش آمد، باید موضوع دیگری برای گزارش آن ایجاد کنید.
