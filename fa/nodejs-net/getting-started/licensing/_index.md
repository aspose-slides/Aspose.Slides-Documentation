می‌توانید به راحتی Aspose.Slides را برای ارزیابی دانلود کنید. بسته ارزیابی همان بسته خریداری شده است. نسخه ارزیابی پس از افزودن چند خط کد برای اعمال مجوز به سادگی دارای مجوز می‌شود.

## **درباره مجوز**
می‌توانید به راحتی نسخه ارزیابی Aspose.Slides برای Node.js از طریق .NET را از [صفحه دانلود](https://releases.aspose.com/slides/fa/nodejs-net/) دریافت کنید. نسخه ارزیابی کاملاً **همان قابلیت‌ها** را مانند نسخه دارای مجوز Aspose.Slides ارائه می‌دهد. علاوه بر این، پس از خرید مجوز و افزودن چند خط کد برای اعمال مجوز، نسخه ارزیابی به سادگی دارای مجوز می‌شود.

مجوز یک فایل XML متنی ساده است که شامل جزئیاتی مانند نام محصول، تعداد توسعهگرانی که به آنها مجوز داده شده، تاریخ انقضای اشتراک و غیره می‌باشد. این فایل به صورت دیجیتالی امضا شده است، بنابراین آن را تغییر ندهید. حتی افزودن ناخواسته یک خط جدید به محتوای فایل، آن را نامعتبر می‌کند.

برای جلوگیری از محدودیت‌های مرتبط با نسخه ارزیابی، باید قبل از استفاده از **Aspose.Slides** یک مجوز تنظیم کنید. شما فقط یک بار در هر برنامه یا فرآیند نیاز به تنظیم مجوز دارید.

### **تنظیم مجوز در Aspose.Slides برای Node.js از طریق .NET**

مجوزها می‌توانند از این مکان‌ها اعمال شوند:

* مسیر صریح
* جریان
* به عنوان مجوز متری – یک مکانیزم مجوز جدید

{{% alert color="primary" %}}
از متد **setLicense** برای اعطای مجوز به یک کامپوننت استفاده کنید.

اگرچه فراخوانی‌های متعدد **setLicense** مضر نیستند، اما منبع (پردازنده) را هدر می‌دهند.
{{% /alert %}}

{{% alert color="warning" %}}
مجوزهای جدید تنها با نسخه 21.4 یا بالاتر می‌توانند Aspose.Slides را فعال کنند. نسخه‌های قبلی از یک سیستم مجوز متفاوت استفاده می‌کنند و این مجوزها را شناسایی نمی‌کنند.
{{% /alert %}}

#### **اعمال مجوز با استفاده از فایل**

این قطعه کد برای تنظیم یک فایل مجوز استفاده می‌شود:

**Node.js**

```javascript
// ماژول Aspose.Slides را برای دستکاری فایل‌های PowerPoint وارد کنید
const asposeSlides = require('aspose.slides.via.net');

// این تابع کتابخانه Aspose.Slides را با یک لایسنس تنظیم می‌کند
function setupAsposeSlidesLicense() {
	
    // کلاس License را از ماژول Aspose.Slides مقداردهی اولیه کنید
    var license = new asposeSlides.License();
    
    // لایسنس را از یک فایل اعمال کنید
    // "your_license_file.lic" را با مسیر فایل لایسنس واقعی خود جایگزین کنید
    license.setLicense("your_license_file.lic");
}

// تابع را برای تنظیم لایسنس Aspose.Slides اجرا کنید
setupAsposeSlidesLicense();
```
{{% alert color="primary" %}}
هنگام فراخوانی متد setLicense، نام مجوز باید همانند نام فایل مجوز شما باشد. به عنوان مثال، می‌توانید نام فایل مجوز را به "Aspose.Slides.lic.xml" تغییر دهید. سپس در کد خود باید نام جدید مجوز (Aspose.Slides.lic.xml) را به متد setLicense پاس بدهید.
{{% /alert %}}
