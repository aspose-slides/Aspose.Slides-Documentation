{{% alert color="primary" %}} 
JasperReports و JasperServer توانایی‌های داخلی برای استخراج گزارش‌ها به‌صورت ارائه‌های Microsoft PowerPoint ندارند، اما با Aspose.Slides for JasperReports می‌توانید به فرمت‌های صادراتی اضافی دسترسی پیدا کنید:

- Microsoft PowerPoint Presentation (PPT)
- Microsoft PowerPoint Presentation (PPTX)
- HTML
- PDF
{{% /alert %}} 

برای ایجاد اسناد در این فرمت‌ها، Aspose.Slides for JasperReports بر نسخه‌ای داخلی از [Aspose.Slides for Java](https://products.aspose.com/slides/fa/java/) متکی است، که کتابخانه‌ی پیشرو در پردازش ارائه‌ها در بازار از Aspose می‌باشد. برای تولید اسناد از Microsoft PowerPoint استفاده نمی‌شود.
