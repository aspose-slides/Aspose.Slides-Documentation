این بخش توصیف مفصل ویژگی‌های Aspose.Slides برای JasperReports را ارائه می‌دهد. برای آشنایی بیشتر با ویژگی‌ها، روی پیوندهای موجود در جدول کلیک کنید.

|**ویژگی** |**توضیح** |
| :- | :- |
|[خروجی PPT، PPS، PPTX و PPSX](/slides/fa/jasperreports/ppt-pptx-pdf-and-html-export/)|صادر کردن گزارش‌ها به صورت ارائه‌های Microsoft PowerPoint از برنامه شما بدون نیاز به استفاده از Microsoft PowerPoint.|
|دقت بالا نسبت به طرح گزارش|گزارش‌های PPT و PPS دارای همان چینش و ظاهر گزارش‌های PDF و سایر قالب‌ها هستند.|
|استقرار آسان و سبک|Aspose.Slides for JasperReport یک صادرکننده JasperReport و JasperServer به فرمت Microsoft PowerPoint است.|
|[مجوز ساده و مقرون به صرفه](/slides/fa/jasperreports/licensing/)|یک مجوز برای هر سرور یا یک مجوز برای همه سرورهای شما. توسعه‌دهندگان نامحدود.|
|[پشتیبانی فنی رایگان سطح جهانی](/slides/fa/jasperreports/technical-support/)|پشتیبانی فنی نامحدود مستقیماً توسط توسعه‌دهندگانی که بر روی محصول کار می‌کنند ارائه می‌شود.|
