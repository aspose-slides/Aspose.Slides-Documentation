**به Aspose.Slides for JasperReports خوش آمدید**

Aspose.Slides for JasperReports یک کتابخانه است که به‌ طور خاص برای توسعه‌دهندگانی طراحی و توسعه یافته است که نیاز دارند به راحتی گزارش‌ها را از JasperReports به فرمت‌های Microsoft PowerPoint Presentation (PPT) و Microsoft PowerPoint Show (PPS) در برنامه‌های Java خود صادر کنند. تمام ویژگی‌های گزارش با بالاترین دقت به ارائه‌های Microsoft PowerPoint تبدیل می‌شوند. Aspose.Slides for JasperReports از JasperReports 5+ پشتیبانی می‌کند.
