## **پلتفرم‌های پشتیبانی‌شده**
پلتفرم‌هایی که Aspose.Slides for Python via .NET می‌تواند بر روی آن‌ها استفاده شود شامل Windows x64 یا x86 و طیف وسیعی از توزیع‌های Linux با نصب Python 3.5 یا بالاتر می‌باشد. برای پلتفرم هدف Linux الزامات اضافی زیر وجود دارد:
- کتابخانه‌های زمان اجرای GCC-6 (یا بالاتر)
- وابستگی‌های .NET Core Runtime. نصب خود .NET Core Runtime **لازم نیست**
- برای Python 3.5‑3.7: نیاز به نسخه `pymalloc` پایتون است. گزینه ساخت `--with-pymalloc` به‌صورت پیش‌فرض فعال است. معمولاً نسخه `pymalloc` پایتون با پسوند `m` در نام فایل مشخص می‌شود.
- کتابخانهٔ مشترک `libpython`. گزینه ساخت `--enable-shared` به‌صورت پیش‌فرض غیرفعال است و برخی توزیع‌های پایتون این کتابخانه را شامل نمی‌شوند. برای برخی پلتفرم‌های لینوکس می‌توان این کتابخانه را با مدیر بسته نصب کرد، برای مثال: `sudo apt-get install libpython3.7`. مشکل رایج این است که کتابخانه `libpython` در مسیری متفاوت از مکان استاندارد سیستم برای کتابخانه‌های مشترک نصب می‌شود. می‌توان این مشکل را با تنظیم مسیرهای کتابخانهٔ جایگزین در گزینه‌های ساخت پایتون هنگام کامپایل یا با ایجاد یک لینک نمادین به فایل کتابخانهٔ `libpython` در مکان استاندارد سیستم رفع کرد. به‌طور معمول نام فایل کتابخانهٔ مشترک `libpython` به شکل `libpythonX.Ym.so.1.0` برای Python 3.5‑3.7 و یا `libpythonX.Y.so.1.0` برای Python 3.8 یا بالاتر است (مثال: `libpython3.7m.so.1.0`, `libpython3.9.so.1.0`).

اگر به پشتیبانی از پلتفرم‌های بیشتری نیاز دارید، به محصولات «برادر دوقلوی» Aspose.Slides for .NET یا Aspose.Slides for Java مراجعه کنید.

## **سؤالات متداول**

**آیا برای کار کردن کتابخانه نیاز به نصب Microsoft PowerPoint روی سرور/کامپیوتر دارم؟**

نه. PowerPoint مورد نیاز نیست؛ Aspose.Slides یک موتور مستقل برای ایجاد، ویرایش، تبدیل و رندر ارائه‌هاست.

**چرا مَردِک‌پردازی (multithreading) چگونه کار می‌کند؟ آیا می‌توان پردازش را موازی کرد؟**

ایمن است که اسناد مختلف را در رشته‌های متفاوت پردازش کنید؛ همان شیء [ارائه](https://reference.aspose.com/slides/fa/python-net/aspose.slides/presentation/) نباید همزمان توسط [چندین رشته](/slides/fa/python-net/multithreading/) استفاده شود.

**آیا پسوردهای فایل و رمزنگاری پشتیبانی می‌شوند؟**

بله. می‌توانید [ارائه‌های رمزنگاری‌شده](/slides/fa/python-net/password-protected-presentation/) را باز کنید، پسورد باز و نوشتن را تنظیم یا حذف کنید و وضعیت حفاظت را بررسی کنید.

**آیا باید به بسته‌های قلم در کانتینرهای لینوکس توجه داشته باشم؟**

بله. توصیه می‌شود بسته‌های قلم عمومی را نصب کنید و یا به‌صورت صریح [دایرکتوری‌های قلم](/slides/fa/python-net/custom-font/) را در برنامهٔ خود مشخص کنید تا از جایگزینی‌های ناخواسته جلوگیری شود.

**آیا در نسخهٔ ارزیابی محدودیت‌هایی وجود دارد؟**

در [حالت ارزیابی](/slides/fa/python-net/licensing/)، یک واترمارک به خروجی اضافه می‌شود و محدودیت‌هایی اعمال می‌گردد؛ یک [مجوز موقت 30 روزه](https://purchase.aspose.com/temporary-license/) برای تست کامل قابلیت‌ها در دسترس است.

**آیا وارد کردن قالب‌های خارجی به یک ارائه (PDF/HTML → PPTX) پشتیبانی می‌شود؟**

بله. می‌توانید [صفحات PDF و محتوای HTML](/slides/fa/python-net/import-presentation/) را به یک ارائه اضافه کنید و آن‌ها را به اسلاید تبدیل کنید.
