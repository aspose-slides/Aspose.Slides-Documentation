## **اعمال لایسنس**

یک لایسنس می‌تواند از **فایل**، **جریان** یا **منابع جاسازی‌شده** بارگذاری شود.

{{% alert color="primary" %}}
Aspose.Slides کلاس [License](https://reference.aspose.com/slides/fa/python-net/aspose.slides/license/) را برای مدیریت لایسنس‌ها فراهم می‌کند.
{{% /alert %}}

{{% alert color="warning" %}}
لایسنس‌های جدید می‌توانند Aspose.Slides را فقط با نسخه 21.4 یا بالاتر فعال کنند. نسخه‌های قدیمی‌تر از سیستم لایسنس متفاوتی استفاده می‌کنند و این لایسنس‌ها را تشخیص نمی‌دهند.
{{% /alert %}}

### **فایل**

ساده‌ترین روش برای تنظیم لایسنس این است که فایل لایسنس را در همان پوشه‌ای که DLL مؤلفه قرار دارد قرار دهید و فقط نام فایل (بدون مسیر) را مشخص کنید.

کد Python زیر نحوه تنظیم فایل لایسنس را نشان می‌دهد:

```py
import aspose.slides as slides

# نمونه‌سازی کلاس License.
license = slides.License()

# مسیر فایل مجوز را تنظیم می‌کند.
license.set_license("Aspose.Slides.lic")
```

{{% alert color="warning" %}}
اگر فایل لایسنس را در دایرکتوری متفاوتی قرار دهید، هنگام فراخوانی [License.set_license()](https://reference.aspose.com/slides/fa/python-net/aspose.slides/license/set_license/#str)، نام فایل در انتهای مسیر صریح باید با نام فایل لایسنس شما مطابقت داشته باشد.

به عنوان مثال، می‌توانید فایل لایسنس را به *Aspose.Slides.lic.xml* تغییر نام دهید. سپس در کد خود، مسیر کامل آن فایل (که به Aspose.Slides.lic.xml ختم می‌شود) را به متد [License.set_license()](https://reference.aspose.com/slides/fa/python-net/aspose.slides/license/set_license/#str) بدهید.
{{% /alert %}}

### **جریان**

می‌توانید لایسنس را از یک جریان بارگذاری کنید. مثال Python زیر نشان می‌دهد چگونه لایسنس را از یک جریان اعمال کنید:

```py
import aspose.slides as slides

# نمونه‌سازی کلاس License.
license = slides.License()

# تنظیم مجوز از یک جریان.
license.set_license(stream)
```

## **اعتبارسنجی لایسنس**

برای اطمینان از اینکه لایسنس به‌درستی اعمال شده است، می‌توانید آن را اعتبارسنجی کنید. کد Python زیر نحوه اعتبارسنجی لایسنس را نشان می‌دهد:

```py
import aspose.slides as slides

license = slides.License()

license.set_license("Aspose.Slides.lic")

if license.is_licensed():
    print("License is good!")
```

## **ایمنی ریسه**

{{% /alert %}}

## **FAQ**

**آیا می‌توانم لایسنس را در محیط کاملاً آفلاین (بدون دسترسی به اینترنت) اعمال کنم؟**

بله. اعتبارسنجی لایسنس به‌صورت محلی با استفاده از فایل لایسنس انجام می‌شود؛ اتصال به اینترنت نیازی نیست.

**پس از انقضای اشتراک یک‌ساله چه اتفاقی می‌افتد؟ آیا کتابخانه متوقف خواهد شد؟**

خیر. لایسنس دائم است: می‌توانید از نسخه‌های منتشرشده قبل از تاریخ پایان اشتراک خود استفاده کنید؛ فقط بدون تمدید، امکان استفاده از نسخه‌های جدیدتر وجود ندارد.
