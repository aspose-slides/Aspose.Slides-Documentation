## **بررسی کلی**

Aspose.Slides از فایل‌های ارائه‌ای از Microsoft PowerPoint 97 تا Office 365، از جمله Microsoft PowerPoint برای مک، پشتیبانی می‌کند. این مقاله نسخه‌های PowerPoint را که کتابخانه از آن‌ها پشتیبانی می‌کند فهرست می‌کند و جدولی از قالب‌های فایل را که می‌توانند بارگذاری، ذخیره یا هر دو شوند، ارائه می‌دهد.

این مقاله همچنین به سؤالات رایج درباره سازگاری PDF، جاسازی فونت، فایل‌های محافظت‌شده با رمز عبور، فونت‌های سفارشی، جایگزینی فونت و گزینه‌های خروجی XPS پاسخ می‌دهد.

## **نسخه‌های پشتیبانی‌شده Microsoft PowerPoint**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP

- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint برای مک
- Office 365

## **سؤالات متداول**

**آیا می‌توانم ارائه‌ها را به PDF ذخیره کنم که با استانداردهای بایگانی و دسترسی (PDF/A و PDF/UA) سازگار باشند؟**

بله. Aspose.Slides برای Python از طریق .NET از خروجی به PDF با سطوح سازگاری مانند PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b، همچنین PDF/UA از طریق تنظیمات [سازگاری](https://reference.aspose.com/slides/fa/python-net/aspose.slides.export/pdfoptions/compliance/) در [گزینه‌های خروجی PDF](https://reference.aspose.com/slides/fa/python-net/aspose.slides.export/pdfoptions/) پشتیبانی می‌کند.

**آیا کتابخانه از جاسازی فونت هنگام خروجی به PDF، با کنترل دقیق بر آنچه جاسازی می‌شود، پشتیبانی می‌کند؟**

بله. می‌توانید کنترل کنید که آیا فونت‌ها به‌طور کامل یا به‌صورت زیرمجموعه (تنها گلیف‌های استفاده‌شده) جاسازی شوند، نحوهٔ رفتار با فونت‌های عمومی سیستم را مشخص کنید و رفتار متن ASCII را از طریق [گزینه‌های خروجی PDF](https://reference.aspose.com/slides/fa/python-net/aspose.slides.export/pdfoptions/) پیکربندی کنید.

**آیا می‌توانم قبل از بارگذاری واقعی فایل، تشخیص دهم که آیا فایل با رمز عبور محافظت شده است؟**

بله. با استفاده از [API بازرسی مبتنی بر کارخانه](https://reference.aspose.com/slides/fa/python-net/aspose.slides/presentationfactory/) می‌توانید یک فایل ارائه را پرس‌و‌جو کنید تا مشخص کنید آیا با رمز عبور محافظت شده است یا خیر، بدون اینکه آن را به‌طور کامل باز کنید.

**آیا مکانیزم‌های جایگزینی فونت و پشتیبانی از فونت‌های سفارشی وجود دارد؟**

بله. کتابخانه از [بارگذاری](/slides/fa/python-net/custom-font/) و [جاسازی](/slides/fa/python-net/embedded-font/) فونت‌های سفارشی پشتیبانی می‌کند و قوانین [جایگزینی فونت](/slides/fa/python-net/fallback-font/) را برای جلوگیری از نداشتن گلیف‌ها در حین رندر و تبدیل فراهم می‌سازد.

**آیا می‌توانم اسلایدها را به XPS خروجی دهم و گزینه‌هایی برای تنظیم خروجی XPS وجود دارد؟**

بله. [خروجی به XPS](/slides/fa/python-net/convert-powerpoint-to-xps/) پشتیبانی می‌شود و می‌توانید [گزینه‌های ذخیره](https://reference.aspose.com/slides/fa/python-net/aspose.slides.export/xpsoptions/) مرتبط را تنظیم کنید تا کیفیت خروجی و محتوای سند XPS را کنترل کنید.
