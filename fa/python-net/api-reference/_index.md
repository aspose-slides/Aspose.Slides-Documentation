Aspose.Slides for Python via .NET یک کتابخانه کلاس است که به برنامه‌های Python امکان خواندن و نوشتن اسناد ارائه را بدون استفاده از Microsoft PowerPoint® یا نرم‌افزارهای دیگر می‌دهد. Aspose.Slides for Python via .NET به شما اجازه می‌دهد متن، اشکال، جدول‌ها و انیمیشن‌ها را مدیریت کنید؛ صوت و ویدئو را به اسلایدها اضافه کنید؛ پیش‌نمایش اسلایدها را ببینید؛ و اسلایدها را به فرمت‌های SVG، PDF و بیشتر صادر کنید.

آخرین مرجع API می‌تواند در [Aspose.Slides for Python via .NET API Reference](https://reference.aspose.com/slides/fa/python-net/) یافت شود.
