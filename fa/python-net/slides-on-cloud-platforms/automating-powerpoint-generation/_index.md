## **معرفی**

ایجاد ارائه‌های PowerPoint به‌صورت دستی می‌تواند کار زمان‌بر و تکراری باشد — به‌ویژه وقتی محتوای آن مبتنی بر داده‌های دینامیکی باشد که به‌طور مکرر تغییر می‌کنند. چه تولید گزارش‌های تجاری هفتگی، ترکیب مطالب آموزشی، یا تولید اسلایدهای فروش آماده برای مشتری باشد، اتوماسیون می‌تواند ساعت‌ها زمان را صرفه‌جویی کند و ثبات را در بین تیم‌ها تضمین کند.

برای توسعه‌دهندگان Python، خودکارسازی ایجاد ارائه‌های PowerPoint امکانات قدرتمندی فراهم می‌کند. می‌توانید تولید اسلایدها را در پورتال‌های وب، ابزارهای دسکتاپ، سرویس‌های بک‌اند یا پلتفرم‌های ابری یک‌پارچه کنید تا به‌صورت دینامیک داده‌ها را به ارائه‌های حرفه‌ای و برنددار تبدیل کنید — به‌صورت درخواست.

در این مقاله، موارد استفاده رایج برای تولید خودکار PowerPoint در برنامه‌های Python (از جمله استقرار بر روی پلتفرم‌های ابری) را بررسی می‌کنیم و دلیل تبدیل شدن آن به ویژگی اساسی در راه‌حل‌های مدرن را توضیح می‌دهیم. از استخراج داده‌های تجاری لحظه‌ای تا تبدیل متن یا تصاویر به اسلایدها، هدف تبدیل محتوای خام به قالب‌های ساختاریافته و بصری است که مخاطب به‌سرعت درک کند.

## **موارد استفاده رایج برای خودکارسازی PowerPoint در Python**

خودکارسازی تولید PowerPoint به‌ویژه در سناریوهایی که محتوای ارائه باید به‌صورت دینامیک ترکیب، شخصی‌سازی یا به‌طور مکرر به‌روز شود، مفید است. برخی از رایج‌ترین موارد استفاده در دنیای واقعی شامل موارد زیر هستند:

- **گزارش‌ها و داشبوردهای تجاری**
  خلاصه‌های فروش، KPIها یا گزارش‌های عملکرد مالی را با استخراج داده‌های زنده از پایگاه‌های داده یا APIها ایجاد کنید.

- **اسلایدهای فروش و بازاریابی شخصی‌سازی شده**
  به‌صورت خودکار اسلایدهای پیشنهادی مخصوص هر مشتری را با استفاده از داده‌های CRM یا فرم‌ها ایجاد کنید و تحویل سریع و ثبات برند را تضمین کنید.

- **محتوای آموزشی**
  مطالب آموزشی، پرسش‌نامه‌ها یا خلاصه‌های دوره را به اسلایدهای ساختاریافته برای پلتفرم‌های آموزش الکترونیک تبدیل کنید.

- **بینش‌های مبتنی بر داده و هوش مصنوعی**
  از پردازش زبان طبیعی یا موتورهای تحلیل برای تبدیل داده‌های خام یا متن‌های طولانی به ارائه‌های خلاصه‌شده استفاده کنید.

- **اسلایدهای مبتنی بر رسانه**
  ارائه‌ها را از تصاویر بارگذاری‌شده، اسکرین‌شات‌های حاشیه‌دار یا فریم‌های کلیدی ویدیو به همراه توضیحات تکمیلی ترکیب کنید.

- **تبدیل اسناد**
  به‌صورت خودکار اسناد Word، PDFها یا ورودی‌های فرم را به ارائه‌های بصری تبدیل کنید با کمترین تلاش دستی.

- **ابزارهای توسعه‌دهنده و فنی**
  دموی فنی، مرور مستندات یا فهرست تغییرات را به‌صورت اسلاید مستقیم از کد یا محتوای Markdown ایجاد کنید.

با خودکارسازی این جریان‌های کاری، سازمان‌ها می‌توانند تولید محتوا را گسترش دهند، ثبات را حفظ کنند و زمان بیشتری را برای کارهای استراتژیک آزاد کنند.

### **ایجاد اسلاید عنوان**

ما با ایجاد یک ارائه جدید و افزودن اسلاید عنوانی که شامل عنوان اصلی و زیرعنوان است، آغاز می‌کنیم.

```py
import aspose.slides as slides
import aspose.slides.charts as charts
import aspose.pydrawing as draw

with slides.Presentation() as presentation:

    slide_0 = presentation.slides[0]
    slide_0.layout_slide = presentation.layout_slides.get_by_type(slides.SlideLayoutType.TITLE)

    title_shape = slide_0.shapes[0]
    subtitle_shape = slide_0.shapes[1]

    title_shape.text_frame.text = "Quarterly Business Review – Q1 2025"
    subtitle_shape.text_frame.text = "Prepared for Executive Team"
```

### **افزودن اسلاید خلاصه با موارد بولت‌دار**

در نهایت، یک خلاصه و برنامه اقدام را با استفاده از لیست بولت ساده اضافه می‌کنیم.

```py
def create_bullet_paragraph(text):
    paragraph = slides.Paragraph()
    paragraph.paragraph_format.bullet.type = slides.BulletType.SYMBOL
    paragraph.paragraph_format.indent = 15
    paragraph.paragraph_format.default_portion_format.fill_format.fill_type = slides.FillType.SOLID
    paragraph.paragraph_format.default_portion_format.fill_format.solid_fill_color.color = draw.Color.black
    paragraph.text = text
    return paragraph
```
```py
layout_slide_3 = presentation.layout_slides.get_by_type(slides.SlideLayoutType.BLANK)
slide_3 = presentation.slides.add_empty_slide(layout_slide_3)

bullet_list = slide_3.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 100, 50, 600, 200)
bullet_list.fill_format.fill_type = slides.FillType.NO_FILL
bullet_list.line_format.fill_format.fill_type = slides.FillType.NO_FILL

bullet_list.text_frame.paragraphs.clear()
bullet_list.text_frame.paragraphs.add(create_bullet_paragraph("Strong performance in North America; growth opportunity in Asia Pacific"))
bullet_list.text_frame.paragraphs.add(create_bullet_paragraph("Improve marketing outreach in underperforming regions"))
bullet_list.text_frame.paragraphs.add(create_bullet_paragraph("Prepare new campaign strategy for Q2"))
bullet_list.text_frame.paragraphs.add(create_bullet_paragraph("Schedule follow-up review in early July"))
```
