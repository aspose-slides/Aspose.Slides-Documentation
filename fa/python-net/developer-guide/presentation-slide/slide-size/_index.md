## **مقدمه**

Aspose.Slides ابزارهای کاملی برای تنظیم اندازه اسلاید و نسبت ابعاد در ارائه‌های PowerPoint فراهم می‌کند که برای چاپ و نمایش روی صفحه‌نمایش اهمیت دارد.

اندازه‌ها و نسبت‌های رایج اسلاید:

- **استاندارد (نسبت 4:3)**: برای صفحه‌نمایش‌ها و دستگاه‌های قدیمی ایده‌آل است.
- **عریض (نسبت 16:9)**: برای پروژکتورها و نمایشگرهای مدرن توصیه می‌شود.

برای حفظ سازگاری در سراسر ارائه خود، یک اندازه اسلاید و نسبت ابعاد واحد برای تمام اسلایدها اعمال می‌شود. برای به‌دست آوردن نتایج بهینه، ابعاد اسلاید را در ابتدای فرایند ایجاد ارائه تنظیم کنید تا از بروز مشکلات جلوگیری شود.

{{% alert color="primary" %}} 
به طور پیش‌فرض، ارائه‌های ایجاد شده با Aspose.Slides از نسبت استاندارد 4:3 استفاده می‌کنند.
{{% /alert %}}

## **تغییر اندازه اسلاید در یک ارائه**

این مثال کد نشان می‌دهد که چگونه اندازه اسلاید را در یک ارائه با استفاده از Aspose.Slides در Python تغییر دهید:

```py
import aspose.slides as slides

with slides.Presentation(path + "AccessSlides.pptx") as pres:
    pres.slide_size.set_size(slides.SlideSizeType.ON_SCREEN16X9, slides.SlideSizeScaleType.DO_NOT_SCALE)
    pres.save("pres-4x3-aspect-ratio.pptx", slides.export.SaveFormat.PPTX)
```

## **مشخص کردن اندازه‌های سفارشی اسلاید**

اگر اندازه‌های رایج اسلاید (4:3 و 16:9) برای کار شما مناسب نیستند، ممکن است تصمیم بگیرید از یک اندازهٔ خاص یا منحصربه‌فرد اسلاید استفاده کنید. برای مثال، اگر قصد دارید اسلایدهای تمام‌اندازه را از ارائه خود بر روی یک قالب صفحهٔ سفارشی چاپ کنید یا اگر می‌خواهید ارائه را روی نوع خاصی از صفحه‌نمایش نمایش دهید، احتمالاً از تنظیم اندازهٔ سفارشی برای ارائه خود بهره‌مند خواهید شد.

این مثال کد نشان می‌دهد که چگونه از Aspose.Slides برای Python از طریق .NET برای مشخص کردن یک اندازهٔ سفارشی اسلاید برای یک ارائه در Python استفاده کنید:

```py
import aspose.slides as slides

with slides.Presentation(path + "AccessSlides.pptx") as pres:
    pres.slide_size.set_size(780, 540, slides.SlideSizeScaleType.DO_NOT_SCALE) # اندازه کاغذ A4
    pres.save("pres-a4-slide-size.pptx", slides.export.SaveFormat.PPTX)
```

## **مدیریت محتویات اسلاید پس از تغییر اندازه**

پس از تغییر اندازه اسلاید برای یک ارائه، محتویات اسلایدها (مانند تصاویر یا اشیاء) ممکن است دچار اعوجاج شوند. به طور پیش‌فرض، اشیاء به‌صورت خودکار برای مطابقت با اندازهٔ جدید اسلاید تغییر اندازه می‌یابند. اما هنگام تغییر اندازه اسلاید ارائه، می‌توانید تنظیمی را مشخص کنید که تعیین می‌کند Aspose.Slides چگونه با محتویات اسلایدها رفتار می‌کند.

بسته به آنچه قصد دارید انجام دهید یا به دست آورید، می‌توانید از هر یک از این تنظیمات استفاده کنید:

- `DO_NOT_SCALE`

  اگر نمی‌خواهید اشیاء روی اسلایدها تغییر اندازه یابند، از این تنظیم استفاده کنید.

- `ENSURE_FIT`

  اگر می‌خواهید به یک اندازهٔ اسلاید کوچکتر مقیاس‌بندی کنید و نیاز دارید Aspose.Slides اشیاء اسلایدها را کاهش دهد تا همهٔ آن‌ها در اسلایدها جا بگیرند (به این ترتیب از از دست رفتن محتوا جلوگیری می‌کنید)، از این تنظیم استفاده کنید.

- `MAXIMIZE`

  اگر می‌خواهید به یک اندازهٔ اسلاید بزرگتر مقیاس‌بندی کنید و نیاز دارید Aspose.Slides اشیاء اسلایدها را بزرگ‌تر کند تا نسبت به اندازهٔ جدید اسلاید متناسب شوند، از این تنظیم استفاده کنید.

این مثال کد نشان می‌دهد که چگونه هنگام تغییر اندازه اسلاید یک ارائه از تنظیم `MAXIMIZE` استفاده کنید:

```py
import aspose.slides as slides

with slides.Presentation(path + "AccessSlides.pptx") as pres:
   pres.slide_size.set_size(slides.SlideSizeType.LEDGER, slides.SlideSizeScaleType.MAXIMIZE)
```

## **پرسش‌های متداول**

**آیا می‌توانم اندازهٔ سفارشی اسلاید را با استفاده از واحدهایی غیر از اینچ (مثلاً پوینت یا میلی‌متر) تنظیم کنم؟**

بله. Aspose.Slides به‌صورت داخلی از پوینت استفاده می‌کند، به‌طوری که 1 پوینت برابر با 1/72 اینچ است. می‌توانید هر واحدی (مانند میلی‌متر یا سانتی‌متر) را به پوینت تبدیل کنید و از مقادیر تبدیل‌شده برای تعریف عرض و ارتفاع اسلاید استفاده کنید.

**آیا اندازهٔ سفارشی بسیار بزرگ اسلاید بر عملکرد و مصرف حافظه هنگام رندرینگ تأثیر می‌گذارد؟**

بله. ابعاد بزرگ‌تر اسلاید (به پوینت) همراه با مقیاس رندرینگ بالاتر منجر به مصرف حافظه بیشتر و زمان پردازش طولانی‌تر می‌شود. سعی کنید اندازهٔ اسلاید عملی را انتخاب کنید و مقیاس رندرینگ را فقط در صورت نیاز برای دستیابی به کیفیت خروجی مطلوب تنظیم کنید.

**آیا می‌توانم یک اندازهٔ اسلاید غیر استاندارد تعریف کنم و سپس اسلایدها را از ارائه‌هایی با اندازه‌های متفاوت ترکیب کنم؟**

نمی‌توانید [merge presentations](/slides/fa/python-net/merge-presentation/) را زمانی که اندازه‌های اسلاید متفاوت هستند انجام دهید — ابتدا یک ارائه را به اندازهٔ دیگر تغییر اندازه دهید. هنگام تغییر اندازه اسلاید، می‌توانید نحوهٔ مدیریت محتویات موجود را از طریق گزینهٔ [SlideSizeScaleType](https://reference.aspose.com/slides/fa/python-net/aspose.slides/slidesizescaletype/) انتخاب کنید. پس از هماهنگ‌سازی اندازه‌ها، می‌توانید اسلایدها را ترکیب کنید در حالی که قالب‌بندی حفظ می‌شود.

**آیا می‌توانم برای شکل‌های منفرد یا نواحی خاص یک اسلاید تصویر کوچک (thumbnail) تولید کنم و آیا این تصاویر به اندازهٔ جدید اسلاید احترام می‌گذارند؟**

بله. Aspose.Slides می‌تواند تصویرهای کوچک برای [entire slides](https://reference.aspose.com/slides/fa/python-net/aspose.slides/slide/get_image/) و همچنین برای [selected shapes](https://reference.aspose.com/slides/fa/python-net/aspose.slides/shape/get_image/) تولید کند. تصاویر حاصل اندازه و نسبت ابعاد فعلی اسلاید را منعکس می‌کنند و چارچوب و هندسهٔ سازگار را تضمین می‌نمایند.
