## **بررسی کلی**

این پرسش‌وپاسخ پاسخ‌های سوالات رایج درباره Aspose.Slides را ارائه می‌دهد. این مستند شامل فرمت‌های فایل پشتیبانی‌شده، مدیریت استثناها هنگام کار با ارائه‌های بزرگ، تغییر اندازه اسلایدها، پیش‌نمایش اسلایدها، استخراج متن از ارائه‌ها، قالب‌بندی حاشیه جدول‌ها، قرار دادن تصویرها، و رفع مشکلات مربوط به قلم‌ها هنگام تبدیل ارائه‌ها به PDF یا تصاویر است.

## **فرمت‌های فایل پشتیبانی‌شده**

**Q: چه فرمت‌های فایلی Aspose.Slides for Python via .NET پشتیبانی می‌کند؟**  
**A**: Aspose.Slides for Python via .NET فرمت‌های فایلی را که در [فرمت‌های فایل پشتیبانی‌شده](/slides/fa/python-net/supported-file-formats/) توضیح داده شده است، پشتیبانی می‌کند.

## **استثناها**

**Q: هنگام بارگذاری یک فایل PPT بزرگ با تصاویر، استثنای کمبود حافظه دریافت می‌کنم. آیا محدودیتی در Aspose.Slides درباره اندازه فایل وجود دارد؟**  
**A**: فرمول خاصی برای محاسبه اندازه ارائه‌ای که Aspose.Slides پشتیبانی می‌کند وجود ندارد. باید فضای کافی برای نگهداری کل ساختار ارائه و تصاویر در حافظه موجود باشد. به طور معمول، تصاویر در حافظه فضای بیشتری نسبت به دیسک سخت اشغال می‌کنند، به‌ویژه وقتی تصاویر افکت‌های اضافی دارند.

به‌طور کلی، Aspose.Slides for Python via .NET می‌تواند به راحتی فایل‌های ارائه حدود 300 مگابایت را روی سرور با 4 گیگابایت RAM مدیریت کند.

## **کار با اسلایدها**

**Q: می‌توانم اندازه اسلایدهای یک ارائه را تغییر دهم؟**  
**A**: می‌توانید از ویژگی `slide_size` که توسط کلاس [Presentation](https://reference.aspose.com/slides/fa/python-net/aspose.slides/presentation/) ارائه می‌شود، برای تعریف اندازه اسلایدها در یک ارائه استفاده کنید.

**Q: آیا راهی برای تعریف اسلایدهایی با اندازه متفاوت در یک ارائه وجود دارد؟**  
**A**: از آنجا که اندازه اسلایدها در سطح ارائه در اسناد Microsoft PowerPoint تعریف می‌شود، راهی برای انجام این کار وجود ندارد.

**Q: آیا Aspose.Slides for Python via .NET امکان پیش‌نمایش یک اسلاید قبل از ذخیره‌سازی را پشتیبانی می‌کند؟**  
**A**: می‌توانید اسلایدهای ارائه را به تصویر تبدیل کنید و از این تصاویر برای پیش‌نمایش اسلایدها استفاده کنید.

## **کار با متن**

**Q: آیا می‌توان تمام متن‌های یک ارائه را استخراج کرد؟**  
**A**: Aspose.Slides for Python via .NET کلاس [SlideUtil](https://reference.aspose.com/slides/fa/python-net/aspose.slides.util/slideutil/) را در فضای نام `aspose.slides.util` ارائه می‌دهد که متدهای مختلفی برای استخراج کل متن از ارائه‌ها فراهم می‌کند.

**Q: چرا اندازه پاراگراف‌ها در سیستم‌عامل‌های ویندوز و لینوکس متفاوت است؟**  
**A**: محاسبه اندازه پاراگراف‌ها بر پایه محاسبه اندازهٔ متن نمایانگر پاراگراف است. محاسبهٔ اندازهٔ متن براساس معیارهای قلم‌ مشخص‌شده در ارائه PowerPoint است. اگر قلم مشخص‌شده موجود نباشد، با شبیه‌ترین قلم جایگزین می‌شود، اما این قلم معیارهای متفاوتی نسبت به قلم اصلی دارد. در نتیجه، محاسبهٔ اندازهٔ پاراگراف‌ها در سیستم‌های مختلف منجر به نتایج متفاوتی می‌شود که به مجموعهٔ قلم‌های نصب‌شده بستگی دارد. برای دستیابی به همان نتیجه در سیستم‌عامل‌های مختلف، باید همان قلم‌ها را بر روی سیستم‌ها نصب کنید یا آنها را در زمان اجرا به‌عنوان [قلم‌های خارجی](/slides/fa/python-net/custom-font/) بارگذاری کنید.

## **قالب‌بندی و تصاویر**

**Q: چگونه می‌توانم رنگ حاشیه جدول را تنظیم کنم؟**  
**A**: می‌توانید رنگ تمام حاشیه‌های جدول یا فقط حاشیهٔ اطراف کل جدول را تغییر دهید. برای تغییر تمام حاشیه‌ها، لطفاً از ویژگی `cell_format` کلاس [Cell](https://reference.aspose.com/slides/fa/python-net/aspose.slides/cell/) استفاده کنید. برای حاشیهٔ کل جدول، باید سلول‌ها را پیمایش کنید و رنگ حاشیه‌های بیرونی را تغییر دهید.

**Q: Aspose.Slides for Python via .NET برای قرار دادن تصویرها از چه واحدی استفاده می‌کند؟**  
**A**: مختصات و اندازه‌های تمام اشکال روی اسلایدها بر حسب نقطه (72 dpi) اندازه‌گیری می‌شوند.

## **کار با قلم‌ها**

**Q: هنگام تبدیل PPT به PDF یا تصویرها، چرا قلم‌ها در اسناد خروجی متفاوت هستند؟**  
**A**: این مشکل ممکن است نشان‌دهنده این باشد که قلم‌های استفاده‌شده در ارائه در سیستم‌عامل اجرایی موجود نیستند. باید قلم‌ها را بر روی سیستم‌عامل نصب کنید یا آنها را به‌عنوان قلم‌های خارجی با استفاده از کلاس [FontsLoader](https://reference.aspose.com/slides/fa/python-net/aspose.slides/fontsloader/) بارگذاری کنید، همان‌طور که در زیر نشان داده شده است:
```cs
folders = [ "path_to_a_folder_with_fonts" ]
aspose.slides.FontsLoader.load_external_fonts(folders)
```
