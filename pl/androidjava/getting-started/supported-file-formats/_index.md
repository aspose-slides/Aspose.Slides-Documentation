## **Przegląd**

Aspose.Slides obsługuje pliki prezentacji od Microsoft PowerPoint 97 po Office 365, w tym Microsoft PowerPoint dla Mac. Ten artykuł wymienia wersje PowerPoint obsługiwane przez bibliotekę oraz zawiera tabelę formatów plików, które można wczytać, zapisać lub oba.

Artykuł odpowiada także na najczęstsze pytania dotyczące zgodności PDF, osadzania czcionek, plików chronionych hasłem, czcionek niestandardowych, awaryjnych czcionek oraz opcji eksportu XPS.

## **Obsługiwane wersje Microsoft PowerPoint**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint dla MAC
- Office 365

## **Najczęściej zadawane pytania**

**Czy mogę zapisywać prezentacje do formatu PDF spełniające standardy archiwizacji i dostępności (PDF/A i PDF/UA)?**

Tak. Aspose.Slides obsługuje eksport do PDF z poziomami zgodności takimi jak PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b, a także PDF/UA poprzez ustawienie [compliance](https://reference.aspose.com/slides/pl/androidjava/com.aspose.slides/pdfoptions/#setCompliance-int-) w [opcjach eksportu PDF](https://reference.aspose.com/slides/pl/androidjava/com.aspose.slides/pdfoptions/).

**Czy biblioteka obsługuje osadzanie czcionek przy eksporcie do PDF, z precyzyjną kontrolą tego, co jest osadzane?**

Tak. Możesz kontrolować, czy czcionki są w pełni osadzone czy podzbiorem (tylko użyte glify), określić sposób traktowania powszechnych czcionek systemowych oraz skonfigurować zachowanie dla tekstu ASCII za pomocą [opcji eksportu PDF](https://reference.aspose.com/slides/pl/androidjava/com.aspose.slides/pdfoptions/).

**Czy mogę wykryć, czy plik jest chroniony hasłem przed jego załadowaniem?**

Tak. Korzystając z [API inspekcji opartego na fabryce](https://reference.aspose.com/slides/pl/androidjava/com.aspose.slides/presentationfactory/), możesz zapytać plik prezentacji, aby określić, czy jest chroniony hasłem, bez pełnego otwierania go.

**Czy istnieją mechanizmy awaryjne czcionek i wsparcie dla czcionek niestandardowych?**

Tak. Biblioteka obsługuje [ładowanie](/slides/pl/androidjava/custom-font/) i [osadzanie](/slides/pl/androidjava/embedded-font/) czcionek niestandardowych oraz zapewnia [reguły awaryjne czcionek](/slides/pl/androidjava/fallback-font/), które zapobiegają brakującym glifom podczas renderowania i konwersji.

**Czy mogę eksportować slajdy do XPS i czy istnieją opcje dostosowywania wyjścia XPS?**

Tak. [Eksport do XPS](/slides/pl/androidjava/convert-powerpoint-to-xps/) jest obsługiwany, a odpowiednie [opcje zapisu](https://reference.aspose.com/slides/pl/androidjava/com.aspose.slides/xpsoptions/) pozwalają kontrolować jakość i zawartość dokumentu XPS.
