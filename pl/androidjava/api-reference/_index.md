{{% alert color="primary" %}} 

Aspose.Slides for Android via Java jest biblioteką klas, która umożliwia aplikacjom Java na Androida odczytywanie i zapisywanie dokumentów prezentacji bez użycia Microsoft PowerPoint® ani innego oprogramowania. Aspose.Slides for Android via Java pozwala zarządzać tekstem, kształtami, tabelami i animacjami; dodawać dźwięk i wideo do slajdów; przeglądać slajdy; oraz eksportować slajdy do formatu SVG, PDF i innych.

Najnowszą referencję API można znaleźć pod adresem [Aspose.Slides for Android via Java API Reference](https://reference.aspose.com/slides/pl/androidjava/).
