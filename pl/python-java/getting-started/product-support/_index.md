## **Wsparcie techniczne**
{{% alert color="primary" %}}

Aspose zapewnia nieograniczone bezpłatne wsparcie techniczne dla wszystkich swoich produktów. Wsparcie jest dostępne dla wszystkich użytkowników (w tym użytkowników z wersjami ewaluacyjnymi).

{{% /alert %}} 

Gdy potrzebujesz pomocy z Aspose.Slides for Python via Java, rozważ następujące opcje:

- Główną drogą do uzyskania wsparcia jest [Aspose.Forums](https://forum.aspose.com/c/slides/pl/11). Umieść swoje pytanie na forum Aspose.Slides — zostanie ono odpowiedziane w ciągu kilku godzin. Zespół wsparcia Aspose.Slides bezpośrednio odpowiada na pytania zamieszczone na forum.
- Należy pamiętać, że Aspose nie świadczy wsparcia technicznego telefonicznie. Telefoniczne wsparcie jest dostępne wyłącznie w przypadku pytań dotyczących sprzedaży i zakupu.
- Oczekując odpowiedzi na forum, bądź cierpliwy i weź pod uwagę różnice czasowe.

Gdy napotkasz problem z Aspose.Slides for Python via Java, postępuj zgodnie z poniższymi wytycznymi, aby rozwiązać go w najbardziej efektywny sposób:

- Sprawdź i potwierdź, że używasz najnowszej wersji Aspose.Slides for Python via Java przed zgłoszeniem problemu. Być może napotkany problem został już rozwiązany w aktualizacji. 

  Zobacz [Aspose.Slides download](https://releases.aspose.com/slides/pl/python-java/), aby uzyskać informacje o najnowszej wersji.

- Przed zgłoszeniem problemu warto przejrzeć nasze forum, tę dokumentację oraz referencję API. Być może odpowiedź na Twoje pytanie już została udostępniona. 

- Podczas zgłaszania problemu dołącz oryginalny dokument oraz (jeśli to możliwe) fragment kodu dotyczący problemu. Jeśli musisz dołączyć kilka plików, umieść je w archiwum ZIP. Możesz bezpiecznie dołączać dokumenty w Aspose.Forums, ponieważ **tylko** Ty i deweloperzy Aspose mają dostęp do załączonych plików.

- Utwórz i używaj jednego wątku dla konkretnego problemu. Jeśli napotkasz inny problem, musisz utworzyć nowy wątek, aby go zgłosić.
