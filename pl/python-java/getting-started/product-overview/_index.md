## **Opis produktu**

Aspose.Slides for Python via Java to wieloplatformowa biblioteka klas, która pozwala aplikacjom wykonywać szeroką gamę zadań przetwarzania dokumentów bez użycia Microsoft PowerPoint®.

### **Dlaczego warto używać Aspose.Slides for Python via Java**

- Aspose.Slides for Python via Java jest pierwszym i jedynym komponentem, który zapewnia funkcjonalności niezbędne do zarządzania dokumentami PowerPoint®.
- Aspose.Slides for Python via Java oferuje wiele kluczowych funkcji, takich jak zarządzanie tekstami, kształtami, tabelami i animacjami, dodawanie dźwięku i wideo do slajdów, podgląd slajdów, eksportowanie slajdów do SVG, PDF i innych formatów oraz wiele innych możliwości.
- Aspose.Slides for Python via Java nie tylko pozwala otwierać pliki PowerPoint z różnych źródeł, ale także umożliwia zapisywanie prezentacji na wiele sposobów. Na przykład możesz zapisać prezentację w formacie PPT; możesz także zapisać slajdy jako obrazy.
- Aspose.Slides for Python via Java zapewnia prawie wszystkie funkcje, które możesz (lub nie) znaleźć w Microsoft PowerPoint. Dzięki wszechstronności Aspose.Slides for Python via Java, oprócz tabel i różnych rodzajów kształtów, możesz używać go do dodawania różnych rodzajów ramek — tekstowych, audio i wideo — do swoich slajdów.

## **Funkcje Aspose.Slides**

Funkcje Aspose.Slides for Python via Java można podzielić na następujące grupy:

- Funkcje prezentacji.
- Funkcje slajdów.
- Funkcje grafiki i multimediów.
- Funkcje tabel.
- Funkcje tekstu i akapitu.
- Funkcje zaawansowane.

### **Funkcje prezentacji**

- Otwieranie prezentacji Microsoft PowerPoint z różnych źródeł.
- Zapisywanie prezentacji w różnych formatach Microsoft PowerPoint.
- Tworzenie prezentacji od podstaw.
- Konwertowanie prezentacji na dokumenty PDF bez użycia żadnego innego komponentu.
- Odczytywanie lub modyfikowanie właściwości dokumentu prezentacji.
- Stosowanie ochrony na prezentacji.
- Eksportowanie prezentacji do PDF.
- Eksportowanie prezentacji do TIFF.
- Eksportowanie prezentacji do XPS.
- Eksportowanie prezentacji do SWF.
- Eksportowanie prezentacji do notatek PDF / Tiff.
- Eksportowanie prezentacji do HTML.

### **Funkcje slajdów**

- Tworzenie nowych slajdów od podstaw i dodawanie ich do prezentacji.
- Dostęp do dowolnego slajdu przez jego pozycję w prezentacji.
- Kopiowanie lub klonowanie slajdów w tej samej lub innej prezentacji.
- Edytowanie i usuwanie slajdów.
- Kopiowanie masterów wraz ze slajdami automatycznie.
- Podgląd slajdów.
- Pobieranie lub ustawianie nazwy slajdu.
- Zapisywanie slajdów jako obrazy.
- Eksportowanie slajdów do formatu SVG.
- Praca z wieloma masterami w prezentacjach.
- Wypełnianie tła slajdu dowolnym obsługiwanym stylem.
- Dodawanie nowych lub zarządzanie notatkami slajdu.
- Ustawianie nowych masterów i stylów slajdów dynamicznie.
- Dodawanie lub modyfikowanie przejść pokazu slajdów.
- Dodawanie przejść Morph.

### **Funkcje grafiki i multimediów**

- Tworzenie lub zarządzanie kształtami takimi jak linie, prostokąty, elipsy i polilinie.
- Zarządzanie stylami linii w kształtach.
- Pobieranie alternatywnego tekstu kształtu.
- Obracanie i odbijanie kształtów.
- Wypełnianie kształtów różnymi stylami — na przykład jednolitym, gradientowym, wzorem, teksturą i obrazem.
- Korzystanie z dowolnego z 48 predefiniowanych wzorów i 24 tekstur.
- Przypisywanie linków do kształtów.
- Odczytywanie, modyfikowanie i dodawanie nowych efektów animacji do slajdów i kształtów.
- Dodawanie nowych obrazów do prezentacji i zarządzanie istniejącymi.
- Dodawanie nowych punktów graficznych do prezentacji i zarządzanie istniejącymi.
- Dodawanie nowych ramek audio do prezentacji i zarządzanie istniejącymi.
- Dodawanie nowych ramek wideo do prezentacji i zarządzanie istniejącymi.

### **Funkcje tabel**

- Tworzenie tabel od podstaw i dodawanie ich do slajdów.
- Zmiana rozmiaru i przesuwanie tabel.
- Dostęp do dowolnej komórki w tabeli.
- Zmiana formatu i obramowań dowolnej komórki.
- Zmiana rozmiaru kolumn i wierszy.
- Dodawanie i usuwanie kolumn oraz wierszy.
- Ustawianie szerokości i koloru obramowania dla całej tabeli.

### **Funkcje tekstu i akapitu**

- Pobieranie i ustawianie tekstów w placeholderach tekstowych i ramkach tekstowych.
- Pobieranie typu placeholdera.
- Zmiana formatowania tekstu w ramce tekstowej.
- Dodawanie nowych czcionek lub zarządzanie czcionkami już używanymi w prezentacji.
- Zmiana tekstów i formatowania tekstu.
- Tworzenie lub zarządzanie linkami w tekście.
- Obsługa tabulacji i efektywnych tabulacji.
- Dodawanie nowych ramek tekstowych do prawie każdego typu kształtu.
- Dostęp i modyfikacja marginesów oraz ustawień zawijania tekstu w ramkach tekstowych.
- Dostęp i modyfikacja wcięć akapitu.

### **Funkcje zaawansowane**

- Dostęp do obiektów OLE osadzonych w prezentacjach.
- Zablokowanie prezentacji.
