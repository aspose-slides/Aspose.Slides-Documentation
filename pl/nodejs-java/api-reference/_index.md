{{% alert color="primary" %}} 

Aspose.Slides for Node.js via Java jest biblioteką klas, która umożliwia aplikacjom Node.js odczytywanie i zapisywanie dokumentów prezentacji bez użycia Microsoft PowerPoint® ani innych programów. Aspose.Slides for Node.js via Java pozwala zarządzać tekstem, kształtami, tabelami i animacjami, dodawać dźwięk i wideo do slajdów, podglądać slajdy, eksportować slajdy do formatu SVG, PDF oraz robić wiele innych rzeczy.

Najnowszą dokumentację API można znaleźć pod adresem [Aspose.Slides for Node.js via Java API Reference](https://reference.aspose.com/slides/pl/nodejs-java/).
