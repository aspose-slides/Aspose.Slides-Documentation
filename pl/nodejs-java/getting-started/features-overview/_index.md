## **Obsługiwane platformy**
Aspose.Slides for Node.js via Java jest niezależnym od platformy API. Może być używany na dowolnej platformie (Windows, Linux, MacOS itp.), na której zainstalowano Node.js w wersji 7 lub wyższej. Oracle JDK w wersji 8 lub wyższej jest dodatkowym wymaganiem.

## **FAQ**

**Czy muszę zainstalować Microsoft PowerPoint na serwerze/komputerze, aby biblioteka działała?**

Nie. PowerPoint nie jest wymagany; Aspose.Slides jest samodzielnym silnikiem do tworzenia, edytowania, konwertowania i renderowania prezentacji.

**Jak działa wielowątkowość? Czy przetwarzanie może być równoległe?**

Można bezpiecznie przetwarzać różne dokumenty w różnych wątkach; ten sam [prezentacja](https://reference.aspose.com/slides/pl/nodejs-java/aspose.slides/presentation/) obiekt nie powinien być używany przez [wiele wątków](/slides/pl/nodejs-java/multithreading/) jednocześnie.

**Czy obsługiwane są hasła i szyfrowanie plików?**

Tak. [Możesz](/slides/pl/nodejs-java/password-protected-presentation/) otwierać zaszyfrowane prezentacje, ustawiać lub usuwać hasło otwarcia i zapisu oraz sprawdzać status ochrony.

**Czy muszę dbać o pakiety czcionek w kontenerach Linux?**

Tak. Zaleca się zainstalowanie popularnych pakietów czcionek i/lub explicite [określić katalogi czcionek](/slides/pl/nodejs-java/custom-font/) w aplikacji, aby uniknąć nieoczekiwanych zamian.

**Czy istnieją ograniczenia w wersji ewaluacyjnej?**

W [trybie ewaluacyjnym](/slides/pl/nodejs-java/licensing/) do wyjścia dodawany jest znak wodny i obowiązują pewne ograniczenia; dostępna jest [30-dniowa licencja tymczasowa](https://purchase.aspose.com/temporary-license/) umożliwiająca pełne testowanie funkcji.

**Czy obsługiwane jest importowanie zewnętrznych formatów do prezentacji (PDF/HTML → PPTX)?**

Tak. Możesz dodać [strony PDF i treść HTML](/slides/pl/nodejs-java/import-presentation/) do prezentacji, przekształcając je w slajdy.
