## **Formaty plików i konwersje**
Aspose.Slides for C++ obsługuje większość formatów dokumentów PowerPoint i umożliwia ich eksport do popularnych formatów szeroko wykorzystywanych w wymianie pomiędzy organizacjami. Poniższa tabela wymienia te formaty:

|**Funkcja**|**Opis**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/pl/cpp/microsoft-powerpoint-ppt/)|Aspose.Slides for C++ zapewnia najszybsze przetwarzanie tego formatu dokumentu prezentacji.|
|[Portable Document Format (PDF)](/slides/pl/cpp/portable-document-format-pdf/)|Obsługiwane formaty plików mogą być eksportowane do dokumentów Adobe Portable Document Format (PDF) przy użyciu jednej metody.|
|[XML Parser Specification (XPS)](/slides/pl/cpp/xml-parser-specification-xps/)|Wszystkie obsługiwane formaty plików mogą być eksportowane do dokumentów XML Parser Specification (XPS) za pomocą jednej metody.|
|[PresentationML (PPTX, XML)](/slides/pl/cpp/presentationml-pptx-xml/)|Aspose.Slides for C++ obsługuje przetwarzanie formatu prezentacji OOXML, znanego również jako PresentationML lub PPTX.|
|[PPTX To HTML Conversion](https://docs.aspose.com/slides/pl/cpp/convert-powerpoint-to-html/)|Aspose.Slides for C++ obsługuje konwersję prezentacji do formatu HTML.|

## **FAQ**

**Czy muszę zainstalować Microsoft PowerPoint na serwerze/komputerze, aby biblioteka działała?**

Nie. PowerPoint nie jest wymagany; Aspose.Slides jest samodzielnym silnikiem do tworzenia, edytowania, konwertowania i renderowania prezentacji.

**Jak działa wielowątkowość? Czy przetwarzanie może być równoległe?**

Bezpieczne jest przetwarzanie różnych dokumentów w oddzielnych wątkach; ten sam obiekt [prezentacji](https://reference.aspose.com/slides/pl/cpp/aspose.slides/presentation/) nie powinien być używany przez [wiele wątków](/slides/pl/cpp/multithreading/) jednocześnie.

**Czy obsługiwane są hasła i szyfrowanie plików?**

Tak. [Możesz](/slides/pl/cpp/password-protected-presentation/) otwierać zaszyfrowane prezentacje, ustawiać lub usuwać hasło otwierania i zapisu oraz sprawdzać status ochrony.

**Czy muszę dbać o pakiety czcionek w kontenerach Linux?**

Tak. Zaleca się zainstalowanie powszechnych pakietów czcionek i/lub jawne [określenie katalogów czcionek](/slides/pl/cpp/custom-font/) w aplikacji, aby uniknąć nieoczekiwanych zamian.

**Czy istnieją ograniczenia w wersji ewaluacyjnej?**

W [trybie ewaluacyjnym](/slides/pl/cpp/licensing/) do wyjścia dodawany jest znak wodny i obowiązują pewne ograniczenia; dostępna jest [30-dniowa tymczasowa licencja](https://purchase.aspose.com/temporary-license/) umożliwiająca pełne testowanie funkcji.

**Czy obsługiwane jest importowanie zewnętrznych formatów do prezentacji (PDF/HTML → PPTX)?**

Tak. Możesz dodać [strony PDF i treść HTML](/slides/pl/cpp/import-presentation/) do prezentacji, przekształcając je w slajdy.
