## **Wymagania oprogramowania**
Upewnij się, że spełniasz następujące wymagania przed pobraniem i uruchomieniem przykładów.

1. Visual Studio 2015 lub nowsze.
## **Pobierz Aspose.Slides z GitHub**
Wszystkie przykłady Aspose.Slides dla C++ są hostowane na [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-C).

- Możesz sklonować repozytorium używając ulubionego klienta GitHub lub pobrać plik ZIP z [tutaj](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/refs/heads/master.zip).
- Rozpakuj zawartość pliku ZIP do dowolnego folderu na komputerze. Wszystkie przykłady znajdują się w folderze **Examples**.
- W repozytorium znajduje się plik rozwiązania Visual Studio dla C++.
- Projekty zostały stworzone w Visual Studio 2015, ale pliki rozwiązania są kompatybilne z Visual Studio 2015 i nowszymi.
- Otwórz plik rozwiązania w Visual Studio i zbuduj projekt.
- Pobierz najnowszą wersję Aspose.Slides dla C++ z [Download Section](https://downloads.aspose.com/slides/pl/cpp).
- Rozpakuj wydanie i skopiuj katalogi **Include** i **Bin** do folderu **Examples**.
- Otwórz plik **main.cpp**, wszystkie przykłady są wywoływane stamtąd.
- Odkomentuj przykłady, które chcesz uruchomić w projekcie.
- Zbuduj i skompiluj rozwiązanie.

Jeśli napotkasz jakiekolwiek problemy przy konfigurowaniu lub uruchamianiu przykładów, skorzystaj z naszych forów.
## **Współtwórz**
Jeśli chcesz dodać lub ulepszyć przykład, zachęcamy Cię do wkładu w projekt. Wszystkie przykłady i projekty demonstracyjne w tym repozytorium są open source i mogą być swobodnie używane w Twoich własnych aplikacjach.

Aby wnieść wkład, możesz rozgałęzić repozytorium, edytować kod źródłowy i utworzyć pull request. Przejrzymy zmiany i, jeśli będą pomocne, włączymy je do repozytorium.
