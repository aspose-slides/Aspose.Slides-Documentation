## **O XPS**
Specyfikacja parsera XML[https://en.wikipedia.org/wiki/Open_XML_Paper_Specification] jest językiem opisu stron i formatem dokumentu o stałym układzie, pierwotnie opracowanym przez Microsoft. Podobnie jak PDF, XPS jest formatem dokumentu o stałym układzie, zaprojektowanym w celu zachowania wierności dokumentu i zapewnienia wyglądu niezależnego od urządzenia.

## **XPS w Aspose.Slides for C++**
Każdy dokument prezentacji, który może być załadowany przez Aspose.Slides for C++, może zostać przekonwertowany do formatu XPS. Aspose.Slides for C++ wykorzystuje silnik układu strony i renderowania o wysokiej wierności, aby generować wyjście w formacie dokumentu XPS o stałym układzie.

Warto wspomnieć, że Aspose.Slides for C++ generuje XPS bezpośrednio, nie zależąc od klas Windows Presentation Foundation (WPF) pakowanych z C++ Framework 3.5, co pozwala Aspose.Slides for C++ tworzyć dokumenty XPS na maszynach z wersjami C++ Framework starszymi niż wersja 3.5. Więcej informacji o eksportowaniu dokumentów prezentacji do dokumentów XPS za pośrednictwem Aspose.Slides for C++ znajdziesz w [tym temacie](/slides/pl/net/).
