{{% alert color="primary" %}}

Aspose.Slides for C++ jest biblioteką klas, która umożliwia aplikacjom C++ odczytywanie i zapisywanie dokumentów prezentacji bez użycia Microsoft PowerPoint® lub innych. Aspose.Slides for C++ pozwala zarządzać tekstem, kształtami, tabelami i animacjami, dodawać audio i wideo do slajdów, podglądać slajdy, eksportować je do formatu SVG, PDF i nie tylko.

Najnowszą dokumentację API można znaleźć pod adresem [Aspose.Slides for C++ API Reference](https://reference.aspose.com/slides/pl/cpp/).
