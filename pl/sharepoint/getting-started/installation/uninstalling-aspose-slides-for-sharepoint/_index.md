Aby odinstalować Aspose.Slides for SharePoint:

1. Uruchom program instalacyjny.
   Jeśli Aspose.Slides for SharePoint jest już zainstalowane, program instalacyjny proponuje jego usunięcie lub naprawę.
1. Wybierz **Remove**, aby odinstalować Aspose.Slides for SharePoint.
