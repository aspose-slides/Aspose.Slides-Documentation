## **Wymagania serwera**
{{% alert color="primary" %}} 

Aby zainstalować Aspose.Slides for SharePoint, wymagany jest system uruchomiony na jednej z następujących wersji:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft Office SharePoint Server 2010
- Microsoft Office SharePoint Server 2013
- Microsoft Office SharePoint Server 2016
- Microsoft Office SharePoint Server 2019

{{% /alert %}} {{% alert color="primary" %}} 

Zaleca się zainstalowanie Service Pack 2 lub nowszego dla tych produktów.

{{% /alert %}} 
## **Wymagania klienta**
{{% alert color="primary" %}} 

Strony SharePoint są dostępne przez przeglądarkę internetową. Aspose.Slides for SharePoint zostało przetestowane w następujących przeglądarkach:

- Microsoft Internet Explorer 6, 7, 8
- Mozilla FireFox 3.5

{{% /alert %}}
