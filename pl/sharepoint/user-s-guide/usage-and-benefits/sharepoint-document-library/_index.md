Biblioteka jest strukturą zawartości, która zawiera pliki. Biblioteka może zawierać tylko jeden typ pliku lub wiele różnych typów, w tym dokumenty, arkusze kalkulacyjne i prezentacje. Biblioteka SharePoint tworzy jedno miejsce, w którym użytkownicy mogą przesyłać i uzyskiwać dostęp do dokumentów. Dokumenty są przechowywane w formacie binarnym w bazie danych SQL Server używanej przez witrynę SharePoint. Dokumenty są dostępne dla innych użytkowników SharePoint jako lista hiperłączy.
