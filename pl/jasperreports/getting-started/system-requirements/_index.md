Aspose.Slides for JasperReports współpracuje z:

**Aspose.Slides for JasperReports 17.5 i wcześniejsze wersje:**

- Java 2 Platform Standard Edition 1.4 lub nowsza.
- JasperReports 2.0, 3.0 lub nowszy.

**Aspose.Slides for JasperReports 17.6 i nowsze wersje:**

- Java 2 Platform Standard Edition 6.0 lub nowsza.
- JasperReports 3.7.2 lub nowszy.
