Ta sekcja zawiera szczegółowy opis funkcji Aspose.Slides dla JasperReports. Kliknij linki w tabeli, aby dowiedzieć się więcej o funkcjach. 

|**Funkcja** |**Opis** |
| :- | :- |
|[Eksport PPT, PPS, PPTX i PPSX](/slides/pl/jasperreports/ppt-pptx-pdf-and-html-export/)|Eksportuj raporty jako prezentacje Microsoft PowerPoint z aplikacji bez użycia Microsoft PowerPoint. |

|Łatwa i lekka implementacja|Aspose.Slides for JasperReport jest eksporterem JasperReport i JasperServer do formatu Microsoft PowerPoint. |
