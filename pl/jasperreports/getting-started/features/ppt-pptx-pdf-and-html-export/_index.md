{{% alert color="primary" %}}

JasperReports i JasperServer nie mają wbudowanych możliwości eksportowania raportów jako prezentacji Microsoft PowerPoint, ale dzięki Aspose.Slides for JasperReports uzyskasz dostęp do dodatkowych formatów eksportu:

- Microsoft PowerPoint Presentation (PPT)
- Microsoft PowerPoint Presentation (PPTX)
- HTML
- PDF

{{% /alert %}}

Aby tworzyć dokumenty w tych formatach, Aspose.Slides for JasperReports opiera się na wbudowanej wersji [Aspose.Slides for Java](https://products.aspose.com/slides/pl/java/), wiodącej na rynku biblioteki do przetwarzania prezentacji firmy Aspose. Microsoft PowerPoint nie jest używany do generowania dokumentów.
