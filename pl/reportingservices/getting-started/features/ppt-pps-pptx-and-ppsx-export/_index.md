{{% alert color="primary" %}} 

Microsoft SQL Server Reporting Services nie posiada wbudowanych możliwości eksportowania raportów jako prezentacji Microsoft PowerPoint, ale po zainstalowaniu Aspose.Slides for Reporting Services na serwerze uzyskasz dostęp do dodatkowych formatów eksportu:

- Microsoft PowerPoint Presentation (PPT)
- Microsoft PowerPoint SlideShow (PPS)
- Microsoft PowerPoint 2007 Presentation (PPTX)
- Microsoft PowerPoint 2007 SlideShow (PPSX)

{{% /alert %}} 

Aby tworzyć dokumenty w tych formatach, Aspose.Slides for Reporting Services korzysta z wbudowanej wersji [Aspose.Slides for .NET](http://www.aspose.com/Products/Aspose.Slides/), wiodącego na rynku komponentu do przetwarzania prezentacji firmy Aspose. Microsoft PowerPoint nie jest używany do generowania dokumentów. 


Kopie PPT i PPS tego samego raportu będą wyglądać identycznie, dzięki wysokiemu stopniowi obsługi podstawowych formatów plików zapewnianemu przez Aspose.Slides.
