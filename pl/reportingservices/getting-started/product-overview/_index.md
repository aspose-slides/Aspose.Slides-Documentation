## **Witamy w Aspose.Slides for Reporting Services!**

Aspose.Slides for Reporting Services jest jedynym rozwiązaniem na rynku, które umożliwia generowanie prawdziwych raportów PPT i PPS w Microsoft SQL Server 2005, 2008, 2012, 2016 i 2017 Reporting Services (32-bit i 64-bit). Wszystkie funkcje raportu RDL, w tym tabele, macierze, wykresy i obrazy, są konwertowane z najwyższą precyzją na prezentacje Microsoft PowerPoint.

## **Przegląd produktu**

Microsoft SQL Server Reporting Services nie posiada wbudowanych funkcji eksportu raportów jako prezentacji Microsoft PowerPoint, ale po zainstalowaniu Aspose.Slides for Reporting Services na serwerze uzyskasz dostęp do dodatkowych formatów eksportu:

- PPT - Prezentacja PowerPoint za pomocą Aspose.Slides
- PPS - Pokaz slajdów PowerPoint za pomocą Aspose.Slides
- PPTX - Prezentacja PowerPoint 2007 za pomocą Aspose.Slides
- PPSX - Pokaz slajdów PowerPoint 2007 za pomocą Aspose.Slides

Aspose.Slides for Reporting Services tworzy prezentacje na serwerze bez użycia Microsoft PowerPoint. Aspose.Slides for Reporting Services wewnętrznie korzysta z Aspose.Slides for .NET – komponentu klasy światowej do przetwarzania prezentacji po stronie serwera.

**Aspose.Slides for Reporting Services umożliwia eksport dowolnego raportu w formacie PPT, PPS, PPTX lub PPSX.**
