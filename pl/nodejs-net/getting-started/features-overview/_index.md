## **Obsługiwane platformy**
Aspose.Slides for Node.js via .NET jest niezależnym od platformy API. Może być używany na dowolnej platformie (Windows, Linux, macOS itp.), na której zainstalowano Node.js w wersji 7 lub wyższej. Oracle JDK 8 lub wyższy jest kolejnym wymogiem.

## **Formaty plików i konwersje**
Aspose.Slides for Node.js via .NET obsługuje większość formatów dokumentów PowerPoint. Umożliwia również eksport do popularnych formatów plików używanych i udostępnianych przez organizacje. Zapoznaj się z poniższymi szczegółami:

|**Funkcja**|**Opis**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via .NET zapewnia najszybsze przetwarzanie tego formatu dokumentu prezentacji.|
|Konwersja PPT do PPTX|Aspose.Slides for Node.js via .NET obsługuje konwersję PPT do PPTX|
|Portable Document Format (PDF)|Możesz wyeksportować wszystkie obsługiwane formaty plików do dokumentów Adobe Portable Document Format (PDF) jedną metodą.|
|XML Parser Specification (XPS)|Możesz wyeksportować wszystkie obsługiwane formaty plików do dokumentów XML Parser Specification (XPS) jedną metodą.|
|Tagged Image File Format (TIFF)|Możesz wyeksportować wszystkie obsługiwane formaty prezentacji do Tagged Image File Format (TIFF).|
|Konwersja PPTX do HTML|Aspose.Slides for Node.js via .NET obsługuje konwersję PresentationEx do formatu HTML.|

## **Renderowanie i drukowanie**
Aspose.Slides for Node.js via .NET obsługuje renderowanie slajdów prezentacji w wysokiej jakości do różnych formatów graficznych. Zapoznaj się z poniższymi szczegółami:

|**Funkcja**|**Opis**|
| :- | :- |
|Obsługiwane formaty obrazów w .NET|Korzystając z Aspose.Slides for Node.js via .NET, możesz renderować slajdy prezentacji i obrazy na slajdach do wszystkich formatów graficznych obsługiwanych przez .NET, takich jak TIFF, PNG, BMP, JPEG, GIF.|
|Format SVG|Aspose.Slides for Node.js via .NET udostępnia także wbudowane metody umożliwiające eksport slajdów prezentacji do formatu Scalable Vector Graphics (SVG).|
|Drukowanie prezentacji|Najnowsze wersje Aspose.Slides for Node.js via .NET zapewniają wbudowane metody drukowania z różnymi opcjami.|

## **Funkcje zawartości**
Aspose.Slides for Node.js via .NET pozwala na dostęp, modyfikację lub tworzenie prawie wszystkich elementów lub treści dokumentów prezentacji. Zapoznaj się z poniższymi szczegółami:

|**Funkcja**|**Opis**|
| :- | :- |
|Slajdy główne|Slajdy główne definiują układ normalnych slajdów. Aspose.Slides for Node.js via .NET pozwala na dostęp i modyfikację slajdów głównych dokumentów prezentacji|
|Slajdy normalne|Za pomocą Aspose.Slides for Node.js via .NET możesz tworzyć nowe slajdy różnych typów; możesz także uzyskać dostęp i modyfikować istniejące slajdy w prezentacjach|
|Klonowanie / kopiowanie slajdów|Istnieją wbudowane metody dostarczane przez Aspose.Slides for Node.js via .NET, które pozwalają klonować lub kopiować istniejące slajdy w obrębie prezentacji. Możesz także używać skopiowanych i sklonowanych slajdów z jednej prezentacji do drugiej. Ponieważ slajd dziedziczy układ z slajdu głównego, wbudowane metody klonowania automatycznie kopiują master podczas klonowania|
|Zarządzanie sekcjami slajdów|Metody do organizowania slajdów w różnych sekcjach wewnątrz prezentacji|
|Place Holders i Text Holders|Możesz uzyskać dostęp do pól zastępczych i pól tekstowych w slajdzie. Ponadto możesz utworzyć slajd z polami tekstowymi od podstaw, używając odpowiedniej metody|
|Nagłówki i stopki|Aspose.Slides for Node.js via .NET ułatwia obsługę nagłówków/stopki w slajdach|
|Notatki w slajdach|Za pomocą Aspose.Slides for Node.js via .NET możesz uzyskać dostęp i modyfikować notatki powiązane ze slajdem oraz dodawać nowe notatki|
|Wyszukiwanie kształtu|Możesz także znaleźć konkretny kształt na slajdzie, używając alternatywnego tekstu powiązanego z tym kształtem|
|Tła|Aspose.Slides for Node.js via .NET pozwala pracować z tłami powiązanymi ze slajdem głównym lub normalnym w prezentacji|
|Pola tekstowe|Pola tekstowe mogą być tworzone od podstaw. Możesz uzyskać dostęp do istniejących pól tekstowych. Możesz także modyfikować ich teksty bez utraty pierwotnego formatowania|
|Kształty prostokątne|Możesz tworzyć lub modyfikować prostokątne kształty przy użyciu Aspose.Slides for Node.js via .NET|
|Kształty linii łamanej|Możesz tworzyć lub modyfikować kształty linii łamanej przy użyciu Aspose.Slides for Node.js via .NET|
|Kształty eliptyczne|Możesz tworzyć lub modyfikować eliptyczne kształty przy użyciu Aspose.Slides for Node.js via .NET|
|Kształty grupowe|Aspose.Slides for Node.js via .NET obsługuje kształty grupowe|
|Auto kształty|Aspose.Slides for Node.js via .NET obsługuje auto kształty|
|SmartArt|Aspose.Slides for Node.js via .NET zapewnia obsługę kształtów SmartArt w MS PowerPoint|
|Wykresy|Aspose.Slides for Node.js via .NET zapewnia obsługę wykresów MSO w PowerPoint|
|Serializacja kształtów|Aspose.Slides for Node.js via .NET obsługuje dużą liczbę kształtów. Gdy Aspose.Slides for Node.js via .NET nie obsługuje danego kształtu, możesz użyć metody serializacji, dzięki której możesz serializować ten kształt z istniejącego slajdu. W ten sposób możesz dalej wykorzystywać kształt zgodnie z wymaganiami|
|Ramki obrazu|Możesz zarządzać obrazami w ramach obrazu przy użyciu Aspose.Slides for Node.js via .NET|
|Ramki audio|Możesz linkować lub osadzać pliki audio w ramach audio na slajdach przy użyciu Aspose.Slides for Node.js via .NET|
|Ramki wideo|Możesz obsługiwać pliki wideo w ramach wideo. Aspose.Slides for Node.js via .NET zapewnia także obsługę wideo linkowanego i osadzonego|
|Ramka OLE|Możesz zarządzać obiektami OLE w ramkach OLE przy użyciu Aspose.Slides for Node.js via .NET|
|Tablice|Aspose.Slides for Node.js via .NET obsługuje tabele w slajdach|
|Kontrolki ActiveX|Obsługa kontrolek ActiveX|
|Makra VBA|Obsługa zarządzania makrami VBA w prezentacjach.|
|Ramka tekstowa|Możesz uzyskać dostęp do tekstu w dowolnym kształcie poprzez ramkę tekstową powiązaną z tym kształtem|
|Skanowanie tekstu|Możesz skanować tekst w prezentacji na poziomie prezentacji lub slajdu przy użyciu wbudowanych metod skanowania.|
|Animacje|Możesz stosować animacje na kształtach|
|Pokazy slajdów|Aspose.Slides for Node.js via .NET obsługuje pokazy slajdów i przejścia slajdów|

## **Funkcje formatowania**
Za pomocą Aspose.Slides for Node.js via .NET możesz formatować teksty i kształty na slajdach w prezentacjach. Zapoznaj się z poniższymi szczegółami:

|**Funkcja**|**Opis**|
| :- | :- |
|Formatowanie tekstu|<p>W Aspose.Slides for Node.js via .NET możesz zarządzać tekstami poprzez ramki tekstowe powiązane z kształtami. Dzięki temu możesz formatować teksty używając akapitów i fragmentów powiązanych z ramkami tekstowymi. Te elementy tekstowe mogą być formatowane przez Aspose.Slides for Node.js via .NET.</p><p>- Typ czcionki</p><p>- Rozmiar czcionki</p><p>- Kolor czcionki</p><p>- Odcienie czcionki</p><p>- Wyrównanie akapitu</p><p>- Wypunktowanie akapitu</p><p>- Orientacja akapitu</p>|
|Formatowanie kształtu|<p>W Aspose.Slides for Node.js via .NET podstawowym elementem slajdu jest kształt. Możesz formatować te elementy kształtów przy użyciu Aspose.Slides for Node.js via .NET:</p><p>- Pozycja</p><p>- Rozmiar</p><p>- Linia</p><p>- Wypełnienie (w tym Wzór, Gradient, Jednolity)</p><p>- Tekst</p><p>- Obraz</p>|
