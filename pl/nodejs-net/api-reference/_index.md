Aspose.Slides for Node.js via .NET to biblioteka klas, która umożliwia aplikacjom Node.js odczytywanie i zapisywanie dokumentów prezentacji bez użycia Microsoft PowerPoint® ani innych. Aspose.Slides for Node.js via .NET pozwala zarządzać tekstem, kształtami, tabelami i animacjami, dodawać dźwięk i wideo do slajdów, podglądać slajdy, eksportować slajdy do formatu SVG, PDF i robić wiele innych rzeczy.

Aspose.Slides for Node.js via .NET używa [tej samej referencji API co Aspose.Slides .NET](https://reference.aspose.com/slides/pl/nodejs-net/).
