**Witamy w Aspose.Slides for Node.js via .NET**

![Logo produktu Aspose.Slides for Node.js via .NET](aspose_slides-for-nodejs-via-net.png)

Aspose.Slides for Node.js via .NET jest biblioteką klas, która umożliwia aplikacjom odczytywanie i zapisywanie dokumentów PowerPoint® bez użycia Microsoft PowerPoint®.

Aspose.Slides for Node.js via .NET jest pierwszym i jedynym komponentem, który zapewnia funkcjonalność zarządzania dokumentami PowerPoint®.

Aspose.Slides for Node.js via .NET oferuje wiele kluczowych funkcji, takich jak zarządzanie tekstem, kształtami, tabelami i animacjami, dodawanie dźwięku i wideo do slajdów, podgląd slajdów, eksportowanie slajdów do formatu SVG, PDF i inne.
