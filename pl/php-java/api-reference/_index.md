Aspose.Slides for PHP via Java to biblioteka klas, która umożliwia aplikacjom PHP odczytywanie i zapisywanie dokumentów prezentacji bez użycia Microsoft PowerPoint® ani innych. Aspose.Slides for PHP via Java pozwala zarządzać tekstem, kształtami, tabelami i animacjami, dodawać dźwięk i wideo do slajdów, podglądać slajdy, eksportować je do formatu SVG, PDF i wiele innych.

Najnowszą dokumentację API można znaleźć pod adresem [Aspose.Slides for PHP via Java API Reference](https://reference.aspose.com/slides/pl/php-java/).
