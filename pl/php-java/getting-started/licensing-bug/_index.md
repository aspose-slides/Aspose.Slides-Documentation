## **Przegląd**

Ten artykuł opisuje błąd licencjonowania napotkany w Aspose.Slides for PHP via Java wersje 23.2, 23.3, 23.4, 23.5, 23.6 i 23.7. Ten błąd powodował, że licencje wieczyste stawały się tymczasowo nieużyteczne po wygaśnięciu subskrypcji.

## **Objawy**

Po wygaśnięciu subskrypcji licencji wieczystej, Aspose.Slides for PHP via Java wersje 23.2‑23.7 mogą zgłaszać licencję jako wygasłą, przełączając wszystkie funkcje w tryb ewaluacji. To zachowanie jest nieprawidłowe i nie dotyczy wersji wcześniejszych niż 23.2 oraz późniejszych niż 23.7.

## **Rozwiązanie**

Aspose.Slides for PHP via Java naprawiło ten problem i wydało zaktualizowane wersje (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) z poprawką.

Jeśli używasz jednej z dotkniętych wersji Aspose.Slides for PHP via Java w swoim projekcie, przejdź na poprawioną wersję.

Jeśli napotkasz jakiekolwiek problemy z licencjonowaniem Aspose.Slides for PHP via Java:

- Sprawdź wersję, której używasz, i upewnij się, że nie jest ona dotknięta opisanym powyżej błędem.
- Odwołaj się do [Aspose documentation](https://docs.aspose.com/slides/pl/php-java/getting-started/) w celu uzyskania kroków rozwiązywania problemów i znanych problemów z licencjami.
- Skontaktuj się z [Aspose support](https://forum.aspose.com/) w celu uzyskania dalszej pomocy.

## **Produkty i wersje objęte problemem**

Należy zauważyć, że błąd dotyczy wyłącznie Aspose.Slides for PHP via Java wersje 23.2‑23.7. **Inne produkty Aspose i inne wersje Aspose.Slides nie są objęte**.
