## **Wspierane platformy**
Aspose.Slides for PHP via Java jest niezależnym od platformy API. Może być używany na dowolnej platformie (Windows, Linux, macOS itp.), na której zainstalowano PHP 7 lub nowszy. Wymagany jest także Oracle JDK 7 lub nowszy.

## **FAQ**

**Czy muszę instalować Microsoft PowerPoint na serwerze/komputerze, aby biblioteka działała?**

Nie. PowerPoint nie jest wymagany; Aspose.Slides jest samodzielnym silnikiem do tworzenia, edytowania, konwertowania i renderowania prezentacji.

**Jak działa wielowątkowość? Czy przetwarzanie może być równoległe?**

Bezpieczne jest przetwarzanie różnych dokumentów w osobnych wątkach; ten sam [prezentacja](https://reference.aspose.com/slides/pl/php-java/aspose.slides/presentation/) nie powinien być używany przez [wiele wątków](/slides/pl/php-java/multithreading/) jednocześnie.

**Czy obsługiwane są hasła i szyfrowanie plików?**

Tak. [Możesz](/slides/pl/php-java/password-protected-presentation/) otwierać zaszyfrowane prezentacje, ustawiać lub usuwać hasło otwierające i zapisu oraz sprawdzać status ochrony.

**Czy muszę dbać o pakiety czcionek w kontenerach Linux?**

Tak. Zaleca się instalację popularnych pakietów czcionek i/lub explicite [określić katalogi czcionek](/slides/pl/php-java/custom-font/) w aplikacji, aby uniknąć nieoczekiwanych zamian.

**Czy istnieją ograniczenia w wersji ewaluacyjnej?**

W [tryb ewaluacji](/slides/pl/php-java/licensing/) dodawany jest znak wodny do wyniku i obowiązują pewne ograniczenia; dostępna jest [30-dniowa licencja tymczasowa](https://purchase.aspose.com/temporary-license/) umożliwiająca pełne testowanie funkcji.

**Czy importowanie zewnętrznych formatów do prezentacji (PDF/HTML → PPTX) jest wspierane?**

Tak. Możesz dodać [Strony PDF i treść HTML](/slides/pl/php-java/import-presentation/) do prezentacji, przekształcając je w slajdy.
