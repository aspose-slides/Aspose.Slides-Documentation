{{% alert color="primary" %}} 

The [Portable Document Format](https://en.wikipedia.org/wiki/PDF) jest formatem plików stworzonym przez Adobe Systems do wymiany dokumentów między organizacjami. Celem formatu było zachowanie treści i układu identycznym, niezależnie od platformy, na której był wyświetlany. Aspose.Slides for PHP via Java umożliwia konwertowanie plików prezentacji do PDF.

{{% /alert %}} 

## **PDF w Aspose.Slides for PHP via Java**
Każda prezentacja, którą można załadować do Aspose.Slides for PHP via Java, może zostać skonwertowana do PDF, który spełnia standardy [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) lub [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA), w zależności od wyboru. Aspose.Slides for PHP via Java eksportuje prezentacje do PDF i w większości przypadków wygenerowany PDF wygląda dokładnie tak jak oryginalna prezentacja.

Aspose.Slides obsługuje następujące elementy prezentacji podczas konwersji do PDF:

- Obrazy, pola tekstowe i inne kształty.
- Tekst i formatowanie.
- Akapity i formatowanie.
- Hiperdłącza.
- Nagłówki i stopki.
- Wypunktowanie.
- Tabele.

Możesz eksportować prezentacje do PDF bezpośrednio przy użyciu Aspose.Slides for PHP via Java: nie potrzebujesz żadnego innego komponentu. Ponadto możesz dostosować eksport prezentacji do PDF za pomocą różnych opcji, jak wyjaśniono w [Converting to PDF](/slides/pl/php-java/convert-powerpoint-to-pdf/).
