## **O formacie TIFF**

[Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) znany z przechowywania wielu obrazów w jednym dokumencie został pierwotnie stworzony przez firmę Aldus. Format ten jest szeroko wspierany przez aplikacje skanujące, faksujące i inne programy do manipulacji obrazami.

## **TIFF w Aspose.Slides for PHP via Java**

Każdy dokument, który może być wczytany w Aspose.Slides for Java, może również zostać bezpośrednio przekonwertowany na dokument TIFF przy użyciu Aspose.Slides for PHP via Java, eliminując potrzebę jakiegokolwiek komponentu zewnętrznego. Dodatkowo można opcjonalnie określić rozmiar obrazów w powstałym dokumencie TIFF. Informacje o eksportowaniu dokumentów prezentacji do dokumentów TIFF za pośrednictwem Aspose.Slides for PHP via Java można znaleźć w [this topic](/slides/pl/php-java/convert-powerpoint-to-tiff/).
