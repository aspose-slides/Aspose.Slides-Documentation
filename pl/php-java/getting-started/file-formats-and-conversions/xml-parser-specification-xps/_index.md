{{% alert color="primary" %}} 

The [XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) jest językiem opisu stron i formatem dokumentu stałego, pierwotnie opracowanym przez Microsoft. Podobnie jak PDF, XPS jest zaprojektowany tak, aby zachować wierność dokumentu i zapewnić wygląd niezależny od urządzenia. 

{{% /alert %}} 

## **XPS w Aspose.Slides for PHP via Java**
Każdy dokument prezentacji, który może zostać załadowany przez Aspose.Slides for PHP via Java, może zostać przekonwertowany do formatu XPS. Aspose.Slides for PHP via Java wykorzystuje wysokiej wierności silnik układu strony i renderowania, aby generować wyjście w formacie dokumentu XPS o stałym układzie.
Więcej informacji o eksportowaniu dokumentów prezentacji do dokumentów XPS przy użyciu Aspose.Slides for PHP via Java znajdziesz w [Converting to XPS](https://docs.aspose.com/slides/pl/php-java/convert-powerpoint-to-xps/).
