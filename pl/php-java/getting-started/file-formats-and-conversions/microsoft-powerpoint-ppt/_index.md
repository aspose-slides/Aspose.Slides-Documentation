{{% alert color="primary" %}} 

[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) jest formatem pliku dokumentu prezentacji, który może być tworzony, odczytywany, modyfikowany i zapisywany przez różne wersje Microsoft PowerPoint. Jest to binarny format dokumentów prezentacji opracowany przez firmę Microsoft.

{{% /alert %}} 

## **PPT w Aspose.Slides for PHP via Java**
Aspose.Slides for PHP via Java może odczytywać pliki PPT utworzone przy użyciu oprogramowania wymienionego poniżej.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Analogicznie, pliki PPT utworzone przez Aspose.Slides for PHP via Java mogą być odczytywane przez powyższy zestaw programów.

## **Kompleksowe wsparcie dla PPT**
Aspose.Slides for PHP via Java zapewnia wsparcie dla prawie wszystkich funkcji obsługiwanych przez format pliku PPT. Nie tylko obejmuje podstawowe i zaawansowane funkcje udostępniane przez różne wersje Microsoft PowerPoint do manipulacji dokumentami PPT, ale także funkcje, które nie są obsługiwane nawet przez Microsoft PowerPoint. Główną zaletą korzystania z biblioteki API Aspose.Slides for PHP via Java jest łatwość obsługi takich funkcji.

Oprócz podstawowych zadań związanych z tworzeniem, odczytywaniem i zapisywaniem plików PPT, Aspose.Slides for PHP via Java oferuje kilka dodatkowych funkcji:

- Importuj inne formaty plików Microsoft Office jako [obiekty OLE do dokumentów PPT](/slides/pl/php-java/manage-ole/).
- [Eksportuj dokumenty PPT do formatu PDF](/slides/pl/php-java/convert-powerpoint-to-pdf/).
- Eksportuj slajdy w dokumentach PPT do formatów SVG.
- Renderuj slajdy do dowolnego formatu obrazu obsługiwanego przez platformę Java.
- Ustaw rozmiar slajdów w dokumentach PPT.
- Zarządzaj animacjami kształtów.
- Zarządzaj pokazami slajdów.
- [Formatuj tekst na slajdach](/slides/pl/php-java/text-formatting/).
- Wyodrębnij tekst z dokumentów PPT.
- [Obsługuj tabele na slajdach](/slides/pl/php-java/powerpoint-table/).
- Automatycznie kopiuj mastery przy użyciu [funkcji klonowania](/slides/pl/php-java/clone-slides/).
