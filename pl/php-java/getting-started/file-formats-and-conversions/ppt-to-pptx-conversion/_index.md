{{% alert color="primary" %}} 
Konwersja PPT do PPTX jest obsługiwana w Aspose.Slides for PHP via Java. Większość funkcji prezentacji – slajdy master, struktura i inne – jest zachowana przy konwersji z jednego formatu na drugi, ale istnieją [kilka ograniczeń](/slides/pl/php-java/ppt-to-pptx-conversion/).
{{% /alert %}} 
## **Funkcje obsługiwane w konwersji**
Aspose.Slides for PHP via Java zapewnia częściowe wsparcie konwersji formatu pliku PPT do PPTX. Wsparcie dla konwersji zostało dopiero wprowadzone w Aspose.Slides for PHP via Java, więc ma kilka ograniczeń i najlepiej sprawdza się przy prostych prezentacjach. Główną zaletą, jaką oferuje Aspose.Slides for PHP via Java przy konwersji PPT do PPTX, jest łatwość użycia API. Aby zobaczyć przykłady kodu, przeczytaj o [Konwertowanie PPT na PPTX](). Poniżej listy wyjaśniają, które funkcje są obsługiwane, a które nie, w konwersji PPT do PPTX.

## **Obsługiwane funkcje**
Poniższe funkcje są obsługiwane przy konwersji:

- Konwersja struktury masterów, układów i slajdów.
- Konwersja wykresów.
- Kształty grupowe.
- Konwersja autokształtów, w tym prostokątów i elips.
- Kształty o niestandardowej geometrii.
- Tekstury i style wypełnienia obrazami dla autokształtów.
- Konwersja pól zastępczych.
- Konwersja linii i polilinii.
- Formaty linii i wypełnień.
- Style wypełnień gradientowych.
- Ramki OLE, tabele, ramki wideo i audio itp.
- Właściwości animacji i pokazu slajdów.
- Konwersja tekstu w ramkach tekstowych i uchwytach tekstowych.
