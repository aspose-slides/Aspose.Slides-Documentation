## **Wsparcie techniczne**
{{% alert color="primary" %}}

Aspose zapewnia nieograniczone bezpłatne wsparcie techniczne dla wszystkich swoich produktów. Wsparcie jest dostępne dla wszystkich użytkowników (w tym użytkowników z wersjami ewaluacyjnymi).

{{% /alert %}} 

Gdy potrzebujesz pomocy z Aspose.Slides for PHP via Java, rozważ poniższe opcje:

- Głównym kanałem wsparcia jest [Aspose.Forums](https://forum.aspose.com/c/slides/pl/11). Opublikuj swoje pytanie na forum Aspose.Slides — zostanie ono odpowiedziane w ciągu kilku godzin. Zespół wsparcia Aspose.Slides bezpośrednio odpowiada na pytania zamieszczane na forum.
- Należy pamiętać, że Aspose nie zapewnia wsparcia technicznego telefonicznie. Wsparcie telefoniczne jest dostępne wyłącznie w kwestiach sprzedaży i zakupu.
- Oczekując odpowiedzi na forum, prosimy o cierpliwość i uwzględnienie różnic czasowych.

Gdy napotkasz problem z Aspose.Slides for PHP via Java, postępuj zgodnie z poniższymi wytycznymi, aby rozwiązać go w najskuteczniejszy sposób:

- Sprawdź i potwierdź, że używasz najnowszej wersji Aspose.Slides for PHP via Java przed zgłoszeniem problemu. Być może problem, który napotkałeś, został już rozwiązany w aktualizacji.  

  Zobacz [Aspose.Slides download](https://downloads.aspose.com/slides/pl/php-java) aby uzyskać informacje o najnowszej wersji.

- Przed zgłoszeniem problemu warto przejrzeć nasze forum, tę dokumentację oraz odniesienie API. Być może odpowiedź na Twoje pytanie już tam znajduje się. 

- Zgłaszając problem, dołącz oryginalny dokument i (jeśli to możliwe) fragment swojego kodu dotyczący problemu. Jeśli musisz dołączyć wiele plików, umieść je w archiwum ZIP. Możesz bezpiecznie dołączać dokumenty na Aspose.Forums, ponieważ **tylko** Ty i deweloperzy Aspose mają dostęp do załączonych plików.

- Utwórz i używaj jednego wątku dla konkretnego problemu. Jeśli napotkasz inny problem, musisz utworzyć kolejny wątek, aby go zgłosić.
