## **Przegląd**

Ten artykuł wyjaśnia, jak uzyskać wsparcie dla Aspose.Slides oraz jak skutecznie zgłaszać problemy. Opisuje główny kanał wsparcia, zawiera zalecenia dotyczące zgłaszania problemu oraz wskazuje przydatne zasoby techniczne, takie jak strona produktu, blog, pobieranie, forum wsparcia, dokumentacja i odniesienie do API. Artykuł odpowiada również na często zadawane pytania dotyczące płatnego wsparcia, eskalacji problemów i wsparcia dla starszych wersji biblioteki.

## **Wsparcie techniczne**

{{% alert color="primary" %}} 

Aspose zapewnia nieograniczone bezpłatne wsparcie techniczne dla wszystkich swoich produktów. Wsparcie jest dostępne dla wszystkich użytkowników, w tym wersji ewaluacyjnych.

{{% /alert %}} 

If you need help with Aspose.Slides for PHP via Java, consider the following:

- Głównym kanałem wsparcia są Aspose.Forums. Zamieść swoje pytanie na forum Aspose.Slides, a zostanie ono odpowiedziane w ciągu kilku godzin. Pytania są odpowiadane bezpośrednio przez zespół wsparcia Aspose.Slides.
- Należy pamiętać, że Aspose nie zapewnia wsparcia technicznego telefonicznie. Telefoniczne wsparcie jest dostępne wyłącznie w kwestiach sprzedaży i zakupu.
- Podczas oczekiwania na odpowiedź na forum, prosimy uwzględnić różnice stref czasowych.

If you have an issue with Aspose.Slides for PHP via Java, follow these simple steps to make sure it is resolved in the most efficient way:

- Upewnij się, że używasz najnowszej wersji Aspose.Slides for PHP via Java przed zgłoszeniem problemu; zobacz pobieranie Aspose.Slides, aby dowiedzieć się o najnowszej wersji.
- Przejrzyj forum, tę dokumentację oraz Odniesienie do API przed zgłoszeniem problemu; być może Twoje pytanie zostało już odpowiedziane.
- Zgłaszając problem, prosimy dołączyć oryginalny dokument oraz ewentualnie fragment kodu, który powoduje problem. Jeśli musisz dołączyć wiele plików, spakuj je do jednego archiwum ZIP. Dołączanie dokumentów na Aspose.Forums jest bezpieczne, ponieważ dostęp do załączonych plików mają tylko Ty i programiści Aspose.
- Staraj się zgłaszać jeden problem na wątek. Jeśli masz kolejny problem, zgłoś go w osobnym wątku.

## **FAQ**

**Jakie dostępne są płatne poziomy wsparcia i jak się różnią?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Różnią się liczbą włączonych incydentów, priorytetem odpowiedzi oraz dostępnymi kanałami komunikacji.

**Czy mogę eskalować błąd bezpośrednio do zespołu inżynierskiego?**

Tak. [Paid support](https://helpdesk.aspose.com/) umożliwia eskalację do programistów, gdy problem wymaga głębszego dochodzenia.

**Czy starsze wersje biblioteki są wspierane i czy wypuszczacie dla nich poprawki?**

Możesz używać starszych wersji, ale poprawki są udostępniane tylko dla bieżącego wydania; łaty dla starszych wersji nie są wydawane.
