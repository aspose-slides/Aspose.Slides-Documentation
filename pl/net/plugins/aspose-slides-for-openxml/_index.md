## **Co to jest OpenXML?**
OpenXML SDK jest zestawem SDK .NET od firmy Microsoft służącym do przetwarzania formatów plików Microsoft Office 2007 i nowszych. Te formaty plików są zasadniczo spakowanymi pakietami, a Open XML SDK for Office upraszcza zadanie manipulacji pakietami Open XML oraz elementami schematu Open XML znajdującymi się w pakiecie. Klasy w Open XML SDK enkapsulują wiele typowych zadań, które programiści wykonują na pakietach Open XML, dzięki czemu możesz wykonywać złożone operacje przy użyciu kilku linii kodu.

Po zainstalowaniu Open XML SDK 2.5 otwórz istniejący projekt lub aplikację w Visual Studio lub utwórz nowy projekt lub aplikację. Następnie w swoim projekcie lub aplikacji dodaj odwołania do następujących komponentów:

- DocumentFormat.OpenXml
- WindowsBase
## **Aspose.Slides for .NET**
Aspose.Slides for .NET jest unikalnym komponentem zarządzania PowerPoint® pozwalającym aplikacjom .NET czytać, zapisywać i manipulować dokumentami PowerPoint bez użycia Microsoft PowerPoint.

Aspose.Slides for .NET jest pierwszym i jedynym komponentem .NET, który zapewnia funkcjonalność zarządzania dokumentami PowerPoint w Twoich własnych aplikacjach. Podobnie jak wszystkie komponenty Aspose .NET, Aspose.Slides jest napisany w zarządzanym C#, ma niewiarygodną cenę i działa błyskawicznie.

## **Wprowadzenie**
Następni deweloperzy poszukujący bibliotek formatów plików mogą potrzebować przejść przez funkcje oferowane przez dostępne na rynku biblioteki. Głównie przy wyborze opcji brane są pod uwagę dwa czynniki:

- Funkcje
- Wysiłek wymagany do użycia biblioteki

Jeśli jesteś programistą .NET poszukującym szybszej, łatwej i lekkiej biblioteki formatu plików do przetwarzania plików, możesz potrzebować porównać VSTO (Visual Studio tool for Office) z Aspose. W takim przypadku ten projekt pomoże Ci porównać funkcje i kod w obu bibliotekach.
