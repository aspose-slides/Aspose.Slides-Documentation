## **Czym jest VSTO?**
VSTO, czyli Visual Studio Tools for Office, lub Microsoft Visual Studio Tools, jest częścią zestawu narzędzi Microsoft Visual Studio .NET i obsługuje Microsoft .NET Framework. Służy programistom do pisania kodu łączącego aplikacje Microsoft Office Word, Excel i Outlook. Programiści korzystający z VSTO mogą używać języków C# lub Visual Basic. Jest to alternatywa dla Visual Basic for Applications, czyli VBA.

## **Aspose.Slides for .NET**
Aspose.Slides for .NET jest wyjątkowym komponentem zarządzania PowerPoint®, który umożliwia aplikacjom .NET odczyt, zapis i manipulację dokumentami PowerPoint bez użycia Microsoft PowerPoint.

Aspose.Slides for .NET jest pierwszym i jedynym komponentem .NET, który zapewnia funkcjonalność zarządzania dokumentami PowerPoint w Twoich własnych aplikacjach. Jak wszystkie komponenty Aspose .NET, Aspose.Slides jest napisany w zarządzanym C#, jest niezwykle przystępny cenowo i błyskawicznie szybki.

## **Wprowadzenie**
Programiści poszukujący bibliotek formatu plików mogą potrzebować przejrzeć funkcje oferowane przez dostępne na rynku biblioteki. Głównie przy wyborze rozważane są dwa czynniki:

- Funkcje
- Wysiłek wymagany do użycia biblioteki

Jeśli jesteś programistą .NET poszukującym szybkiej, łatwej i lekkiej biblioteki formatu plików do przetwarzania plików, możesz potrzebować porównać VSTO (Visual Studio Tools for Office) z Aspose. W tym przypadku projekt pomoże Ci porównać funkcje i kod w obu bibliotekach.

- Jedna wersja Microsoft Office:
- Microsoft Office Professional 2003 SP1
- 2007 Microsoft Office system

tylko .NET Framework 2.0 lub wyższy.
