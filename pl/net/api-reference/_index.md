{{% alert color="primary" %}} 

Aspose.Slides for .NET jest biblioteką klas, która umożliwia aplikacjom .NET odczytywanie i zapisywanie dokumentów prezentacji bez użycia Microsoft PowerPoint® ani innych. Aspose.Slides for .NET pozwala zarządzać tekstem, kształtami, tabelami i animacjami, dodawać dźwięk oraz wideo do slajdów, przeglądać slajdy, eksportować slajdy do formatu SVG, PDF i więcej.

Najnowszą dokumentację API można znaleźć pod adresem [Aspose.Slides for .NET API Reference](https://reference.aspose.com/slides/pl/net).

{{% /alert %}}
