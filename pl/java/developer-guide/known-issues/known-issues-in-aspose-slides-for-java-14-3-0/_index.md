Aspose.Slides for Java 14.3.0 (14.4.0) zapewnia całkowicie nową implementację przetwarzania plików PPT. Wprowadzono wiele usprawnień, częściową konwersję PPTX do PPT. Jednak istnieją niektóre niezaimplementowane funkcje:

- Niektóre kształty mają nieprawidłową geometrię w zserializowanych dokumentach PPT (dymki)
- Nie wszystkie funkcje formatowania tekstu PPTX są obsługiwane przy serializacji do PPT
- Informacje o języku tekstu i ustawieniach sprawdzania pisowni nie są obecne w zserializowanych dokumentach PPT
- Nie wszystkie funkcje motywów PPTX są obsługiwane przy serializacji do PPT

**Istnieją pewne różnice w porównaniu z Aspose.Slides for Java 8.6.0:**

- Istnieją znane problemy w serializacji OLE/ActiveX PPT do PPT

**Istnieją pewne różnice w porównaniu z Aspose.Slides for .NET 14.3.0:**

- Obsługa drukowania prezentacji jest obecnie niedostępna w Aspose.Slides for Java
