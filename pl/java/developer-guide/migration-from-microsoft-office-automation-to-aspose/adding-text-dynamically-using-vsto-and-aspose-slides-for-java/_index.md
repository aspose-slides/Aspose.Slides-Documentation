Typowym zadaniem, które programiści muszą wykonać, jest dynamiczne dodawanie tekstu do slajdów. Ten artykuł pokazuje przykłady kodu dodającego tekst dynamicznie przy użyciu [VSTO](/slides/pl/java/adding-text-dynamically-using-vsto-and-aspose-slides-for-java/) oraz [Aspose.Slides for Java](/slides/pl/java/adding-text-dynamically-using-vsto-and-aspose-slides-for-java/).

{{% /alert %}} 
## **Dodawanie Tekstu Dynamicznie**
Obie metody składają się z następujących kroków:

1. Utwórz prezentację.
1. Dodaj pusty slajd.
1. Dodaj pole tekstowe.
1. Ustaw tekst.
1. Zapisz prezentację.
