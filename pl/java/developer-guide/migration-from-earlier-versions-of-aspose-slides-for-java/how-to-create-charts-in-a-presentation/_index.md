## **Wsparcie dla kodu legacy**
Aby używać kodu legacy opracowanego w wersjach Aspose.Slides for Java wcześniejszych niż 14.x.x, należy wprowadzić niewielkie zmiany w kodzie, po których będzie on działał tak jak wcześniej. Wszystkie klasy, które znajdowały się w starszej wersji Aspose.Slides for Java w przestrzeniach nazw com.aspose.slides i com.aspose.slides.pptx, zostały teraz połączone w jedną przestrzeń nazw com.aspose.slides. Zapoznaj się z poniższym prostym fragmentem kodu tworzącym zwykły wykres od podstaw w prezentacji przy użyciu starszego API Aspose.Slides i postępuj zgodnie z krokami opisującymi, jak migrować do nowego, połączonego API.
