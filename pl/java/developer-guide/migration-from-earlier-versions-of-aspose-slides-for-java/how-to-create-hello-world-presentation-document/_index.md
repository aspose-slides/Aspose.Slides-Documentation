Nowy [API Aspose.Slides for Java](/slides/pl/java/) został wydany i teraz ten pojedynczy produkt obsługuje możliwość generowania dokumentów PowerPoint od podstaw oraz edycji istniejących.

## **Wsparcie dla starego kodu**
Aby używać starszego kodu opracowanego w wersjach Aspose.Slides for Java wcześniejszych niż 13.x, musisz wprowadzić kilka drobnych zmian w swoim kodzie, a kod będzie działał jak wcześniej. Wszystkie klasy, które znajdowały się w starych wersjach Aspose.Slides for Java w przestrzeniach nazw Aspose.Slide i Aspose.Slides.Pptx, są teraz połączone w jedną przestrzeń nazw Aspose.Slides. Proszę zapoznać się z poniższym prostym fragmentem kodu tworzącym dokument prezentacji Hello World w starszym API Aspose.Slides i postępować zgodnie z krokami opisującymi, jak migrować do nowego połączonego API.
