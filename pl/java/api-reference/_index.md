{{% alert color="primary" %}} 

Aspose.Slides for Java jest biblioteką klas, która umożliwia aplikacjom Java odczytywanie i zapisywanie dokumentów prezentacji bez użycia Microsoft PowerPoint® ani innych. Aspose.Slides for Java pozwala zarządzać tekstem, kształtami, tabelami i animacjami, dodawać dźwięk oraz wideo do slajdów, podglądać slajdy, eksportować slajdy do formatu SVG, PDF i wiele więcej.

Najnowsza dokumentacja API jest dostępna pod adresem [Aspose.Slides for Java API Reference](https://reference.aspose.com/slides/pl/java).
