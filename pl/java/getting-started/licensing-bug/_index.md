## **Przegląd**

Ten artykuł opisuje błąd licencyjny napotkany w Aspose.Slides for Java wersje 23.2, 23.3, 23.4, 23.5, 23.6 i 23.7. Błąd powodował, że wieczyste licencje stawały się tymczasowo nieużyteczne po wygaśnięciu ich subskrypcji.

## **Objawy**

Po wygaśnięciu subskrypcji wiecznej licencji, Aspose.Slides for Java wersje 23.2‑23.7 mogą zgłaszać, że licencja wygasła, przełączając wszystkie funkcje w tryb ewaluacyjny.  
To zachowanie jest nieprawidłowe i nie dotyczy wersji wcześniejszych niż 23.2 oraz nowszych niż 23.7.

## **Rozwiązanie**

Aspose.Slides for Java rozwiązało ten problem i wydało zaktualizowane wersje (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) z poprawką.  

Jeśli w swoim projekcie używasz jednej z dotkniętych wersji Aspose.Slides for Java, przejdź na wersję zawierającą poprawkę.

Jeśli napotkasz jakiekolwiek problemy licencyjne z Aspose.Slides for Java:

- Sprawdź wersję, której używasz i upewnij się, że nie jest ona dotknięta opisanym powyżej błędem.
- Odwołaj się do [dokumentacji Aspose](https://docs.aspose.com/slides/pl/java/getting-started/) w celu uzyskania instrukcji rozwiązywania problemów i znanych problemów licencyjnych.
- Skontaktuj się z [wsparciem Aspose](https://forum.aspose.com/) w celu uzyskania dalszej pomocy.

## **Produkty i wersje objęte problemem**

Należy zauważyć, że błąd dotyczy wyłącznie wersji Aspose.Slides for Java 23.2‑23.7. **Inne produkty Aspose oraz inne wersje Aspose.Slides nie są dotknięte**.
