## **Przegląd**

Ten artykuł wyjaśnia, jak uzyskać wsparcie dla Aspose.Slides oraz jak skutecznie zgłaszać problemy. Opisuje główny kanał wsparcia, zawiera zalecenia dotyczące zgłaszania problemu oraz wskazuje przydatne zasoby techniczne, takie jak strona produktu, blog, pobieranie, forum wsparcia, dokumentacja i referencja API. Artykuł odpowiada także na najczęstsze pytania dotyczące płatnego wsparcia, eskalacji problemów oraz wsparcia starszych wersji biblioteki.

## **Wsparcie techniczne**

{{% alert color="primary" %}} 
Aspose zapewnia nieograniczone bezpłatne wsparcie techniczne dla wszystkich swoich produktów. Wsparcie jest dostępne dla wszystkich użytkowników, w tym wersji ewaluacyjnych.
{{% /alert %}} 

Jeśli potrzebujesz pomocy z Aspose.Slides for Java, rozważ następujące:

- Głównym kanałem wsparcia są Aspose.Forums. Umieść swoje pytanie na forum Aspose.Slides, a zostanie ono odpowiedziane w ciągu kilku godzin. Pytania są odpowiadane bezpośrednio przez zespół wsparcia Aspose.Slides.
- Należy pamiętać, że Aspose nie zapewnia wsparcia technicznego telefonicznie. Pomoc telefoniczna jest dostępna wyłącznie w kwestiach sprzedaży i zakupu.
- Oczekując odpowiedzi na forum, prosimy uwzględnić różnice w strefach czasowych.

Jeśli masz problem z Aspose.Slides for Java, postępuj zgodnie z poniższymi prostymi krokami, aby zapewnić jego jak najskuteczniejsze rozwiązanie:

- Upewnij się, że używasz najnowszej wersji Aspose.Slides for Java przed zgłoszeniem problemu; zobacz sekcję pobierania Aspose.Slides, aby dowiedzieć się o najnowszej wersji.
- Przejrzyj forum, tę dokumentację oraz referencję API przed zgłoszeniem problemu; być może Twoje pytanie już zostało odpowiedziane.
- Zgłaszając problem, dołącz oryginalny dokument i ewentualnie fragment kodu powodującego problem. Jeśli musisz dołączyć wiele plików, spakuj je w jeden archiwum ZIP. Dołączanie dokumentów w Aspose.Forums jest bezpieczne, ponieważ dostęp do załączonych plików mają tylko Ty i programiści Aspose.
- Prosimy zgłaszać tylko jeden problem w wątku. Jeśli masz kolejny problem, zgłoś go w osobnym wątku.

## **FAQ**

**Jakie poziomy płatnego wsparcia są dostępne i jak się różnią?**

[Developer, Business i Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Różnią się liczbą objętych incydentów, priorytetem odpowiedzi oraz dostępnymi kanałami komunikacji.

**Czy mogę eskalować błąd bezpośrednio do zespołu inżynieryjnego?**

Tak. [Płatne wsparcie](https://helpdesk.aspose.com/) umożliwia eskalację do deweloperów, gdy problem wymaga głębszego dochodzenia.

**Czy starsze wersje biblioteki są wspierane i czy wydajecie dla nich poprawki?**

Można używać starszych wersji, ale poprawki są dostarczane dla bieżącej wersji; łatki dla wersji legacy nie są wydawane.
