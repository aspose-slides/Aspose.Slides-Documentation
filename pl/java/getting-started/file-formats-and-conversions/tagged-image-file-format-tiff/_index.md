## **O formacie TIFF**

[Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF), znany z przechowywania wielu obrazów w jednym dokumencie, został pierwotnie stworzony przez firmę Aldus. Ten format jest szeroko wspierany przez aplikacje do skanowania, faksowania oraz inne programy do manipulacji obrazami.

## **TIFF w Aspose.Slides for Java**

Każdy dokument, który może być załadowany w Aspose.Slides for Java, może również zostać bezpośrednio skonwertowany na dokument TIFF przez Aspose.Slides for Java, eliminując potrzebę używania jakichkolwiek komponentów firm trzecich. Dodatkowo możesz opcjonalnie określić rozmiar obrazów w wynikowym dokumencie TIFF. Informacje o eksportowaniu dokumentów prezentacji do dokumentów TIFF przy pomocy Aspose.Slides for Java znajdziesz w [tym temacie](/slides/pl/java/converting-a-presentation/).
