{{% alert color="primary" %}} 

[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) jest formatem pliku dokumentu prezentacji, który może być tworzony, odczytywany, modyfikowany i zapisywany przez różne wersje programu Microsoft PowerPoint. Jest to binarny format dokumentów prezentacji opracowany przez firmę Microsoft.

{{% /alert %}} 

## **PPT w Aspose.Slides for Java**
Aspose.Slides for Java może odczytywać pliki PPT utworzone przez oprogramowanie wymienione poniżej.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Podobnie, pliki PPT utworzone przez Aspose.Slides for Java mogą być odczytywane przez powyższy zestaw oprogramowania.

## **Kompleksowe wsparcie dla PPT**
Aspose.Slides for Java zapewnia wsparcie dla niemal wszystkich funkcji obsługiwanych przez format pliku dokumentu PPT. Obejmuje nie tylko podstawowe i zaawansowane funkcje dostarczane przez różne wersje Microsoft PowerPoint do manipulacji dokumentami PPT, ale także funkcje, które nie są wspierane nawet przez Microsoft PowerPoint. Główną zaletą korzystania z biblioteki API Aspose.Slides for Java jest łatwość obsługi takich funkcji.

Oprócz podstawowych zadań związanych z tworzeniem, odczytywaniem i zapisywaniem plików dokumentów PPT, istnieje kilka funkcji oferowanych przez Aspose.Slides for Java:

- Importuj inne formaty plików Microsoft Office jako [obiekty OLE w dokumentach PPT]().
- [Eksportuj dokumenty PPT do PDF](/slides/pl/java/convert-powerpoint-ppt-and-pptx-to-pdf/).
- Eksportuj slajdy z dokumentów PPT do formatów SVG.
- Renderuj slajdy do dowolnego formatu obrazu obsługiwanego przez platformę Java.
- Ustaw rozmiar slajdów w dokumentach PPT.
- Zarządzaj animacjami na kształtach.
- Zarządzaj pokazami slajdów.
- [Formatuj tekst na slajdach]().
- Wydobądź tekst z dokumentów PPT.
- [Obsługuj tabele na slajdach]().
- Automatycznie kopiuj wzorce przy użyciu [funkcji klonowania]().
