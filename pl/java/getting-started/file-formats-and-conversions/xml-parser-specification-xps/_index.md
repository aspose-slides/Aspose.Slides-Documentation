{{% alert color="primary" %}} 

Specyfikacja parsera XML jest językiem opisu stron i formatem dokumentu stałego, pierwotnie opracowanym przez Microsoft. Podobnie jak PDF, XPS został zaprojektowany, aby zachować wierność dokumentu i zapewnić wygląd niezależny od urządzenia. 

{{% /alert %}} 

## **XPS w Aspose.Slides for Java**
Każdy dokument prezentacji, który może zostać załadowany przez Aspose.Slides for Java, może być skonwertowany do formatu XPS. Aspose.Slides for Java wykorzystuje wysokiej wierności silnik układu strony i renderowania, aby generować wyjście w stałym układzie dokumentu XPS.
Więcej informacji o eksportowaniu dokumentów prezentacji do dokumentów XPS przy użyciu Aspose.Slides for Java znajdziesz w [Konwertowanie do XPS](https://docs.aspose.com/slides/pl/java/convert-powerpoint-to-xps/).
