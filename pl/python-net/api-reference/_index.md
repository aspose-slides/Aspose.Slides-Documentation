{{% alert color="primary" %}}

Aspose.Slides for Python via .NET jest biblioteką klas, która umożliwia aplikacjom Pythona odczyt i zapis dokumentów prezentacji bez użycia Microsoft PowerPoint® ani innego oprogramowania. Aspose.Slides for Python via .NET pozwala zarządzać tekstem, kształtami, tabelami i animacjami; dodawać dźwięk i wideo do slajdów; przeglądać slajdy; oraz eksportować slajdy do formatu SVG, PDF i innych.

Najnowsza dokumentacja API jest dostępna pod adresem [Aspose.Slides for Python via .NET API Reference](https://reference.aspose.com/slides/pl/python-net/).
