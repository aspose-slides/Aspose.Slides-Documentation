## **Yazılım Gereksinimleri**
Örnekleri indirmeden ve çalıştırmadan önce, kurulumunuzun bu gereksinimleri karşıladığından emin olun:

- Visual Studio 2010 veya daha yenisi.
- Visual Studio içinde yüklü NuGet Package Manager. En son NuGet API sürümünün Visual Studio'ya yüklü olduğunu doğrulayın.

NuGet paket yöneticisini kurma talimatları için bu sayfayı ziyaret edin: https://docs.microsoft.com/en-us/nuget/install-nuget-client-tools

1. **Tools** > **Options** > **NuGet Package Manager** sekmesinden geçin.

1. **NuGet Package Manager** öğesini (çift tıklayarak) genişletin ve ardından **Package Sources** seçeneğini seçin.

1. nuget.org parametresinin seçili olduğunu kontrol edin ve onaylayın.

   Örnek proje, NuGet Automatic Package Restore özelliğini kullanır; bu nedenle aktif bir internet bağlantınız olmalıdır.

   Örnekleri çalıştırmayı planladığınız makinede aktif bir internet bağlantısı yoksa, lütfen [Installation](https://docs.aspose.com/slides/tr/net/installation/) sayfasını inceleyin ve (elle) örnek projeye Aspose.Slides.dll referansı ekleyin.
## **Aspose.Slides'ı GitHub'dan İndirin**
Tüm Aspose.Slides for .NET örnekleri [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-.NET) üzerinde barındırılmaktadır.

Depoyu favori GitHub istemcinizle klonlayabilir veya ZIP dosyasını [here](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/master.zip) adresinden indirebilirsiniz.

1. ZIP dosyasını indirirseniz, içeriğini bilgisayarınızda bir klasöre çıkarmanız gerekir.

Tüm örnekler **Examples** klasöründe saklanır.

C# Visual Studio çözüm dosyası mevcuttur. Projeler Visual Studio 2013'te oluşturulmuş ancak çözüm dosyaları Visual Studio 2010 SP1 ve üzeriyle uyumludur.

2. Çözüm dosyasını Visual Studio'da açın ve projeyi derleyin.

   İlk çalıştırmada bağımlılıklar NuGet üzerinden otomatik olarak indirilir.

**Examples** klasörünün kök dizinindeki **Data** klasörü, C# örneklerinde kullanılan girdi dosyalarını içerir. **Data** klasörünü örnek projesiyle birlikte indirmeniz gerekir.

3. RunExamples.cs dosyasını açın. Tüm örnekler buradan çağrılır.

4. Projede çalıştırmak istediğiniz örneklerin yorumlarını kaldırın.

Kurulum veya örnekleri çalıştırma konusunda sorun yaşıyorsanız forumlarımız üzerinden bize ulaşabilirsiniz.
## **Katkıda Bulun**
Projeye bir örnek ekleyerek veya mevcut bir örneği iyileştirerek katkıda bulunabilirsiniz. Depodaki tüm örnekler ve gösterim projeleri açık kaynaklıdır; bu nedenle siz (ve diğer kişiler) bunları uygulamalarda özgürce kullanabilirsiniz.

Katkı sağlamak için depoyu fork'layabilir, kaynak kodu düzenleyebilir ve bir pull request oluşturabilirsiniz. Değişiklikleri inceleyeceğiz. Faydalı bulursak, depoya ekleyeceğiz.
