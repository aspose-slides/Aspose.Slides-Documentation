## **Genel Bakış**
Aspose.Slides for Java, Microsoft PowerPoint'in yüklü olmasını gerektirmez, çünkü Aspose.Slides kendisi bir Microsoft PowerPoint belge oluşturma, dönüştürme, sayfa düzeni ve render motorudur.

## **Desteklenen İşletim Sistemleri**
Aspose.Slides for Java, Java çalışma zamanını çalıştıran herhangi bir 32‑bit veya 64‑bit işletim sistemini destekler; ancak bunlarla sınırlı değildir:

### **Windows**
- Microsoft Windows 2003 Server ( x64, x86)
- Microsoft Windows 2008 Server ( x64, x86)
- Microsoft Windows 2012 Server ( x64, x86)
- Microsoft Windows 2012 R2 Server ( x64, x86)
- Microsoft Windows 2016 Server ( x64, x86)
- Microsoft Windows 2019 Server ( x64, x86)
- Microsoft Windows Vista ( x64, x86)
- Microsoft Windows XP ( x64, x86)
- Microsoft Windows 7 ( x64, x86)
- Microsoft Windows 8, 8.1 ( x64, x86)
- Microsoft Windows 10 ( x64, x86)


### **Linux**
- Linux (Ubuntu, OpenSUSE, CentOS ve diğerleri)

### **Mac**
- Mac OS X

## **Desteklenen Java Sürümleri**
Aspose.Slides for Java, J2SE 6.0 (Java 1.6) ve üzeri sürümleri destekler.

## **SSS**

**Dönüştürmeler ve render için Microsoft PowerPoint'in yüklü olması gerekiyor mu?**

Hayır, PowerPoint gerekli değildir; Aspose.Slides, sunumları [oluşturmak](/slides/tr/java/create-presentation/), değiştirmek, [dönüştürmek](/slides/tr/java/convert-presentation/) ve [render](/slides/tr/java/convert-powerpoint-to-png/) işlemleri için bağımsız bir motorudur.

**Doğru render için hangi yazı tiplerine ihtiyaç var?**

Pratikte, sunumda kullanılan yazı tipleri veya uygun [ikame](/slides/tr/java/font-substitution/) yazı tipleri mevcut olmalıdır. Linux/macOS üzerinde tutarlı render sağlamak için yaygın yazı tipi paketlerini yüklemek tavsiye edilir.

**Neden özel bir yazı tipi Linux'ta geri dönüşüm veya eksik metin olarak render oluyor?**

Yazı tipi dosyasının tutarsız veya bozuk name-table kayıtları varsa, Linux yazı tipi eşleştirme yığını (FreeType/fontconfig) geçersiz bir kaydı seçebilir ve bu da yazı tipinin çözülememesine neden olur. Düzeltildiği name-table kayıtlarına sahip bir yazı tipi sürümü kullanmak veya tutarlı bir yedek kurmak sorunu çözer.
