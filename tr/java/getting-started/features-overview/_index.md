## **Desteklenen Platformlar**
Aspose.Slides for Java en popüler geliştirme ve dağıtım platformlarını destekler.

|**Özellik**|**Açıklama**|
| :- | :- |
|Masaüstü Uygulamaları|Aspose.Slides for Java Windows Forms uygulamaları geliştirmek için kullanılabilir|
|Kurumsal Web Uygulamaları|Aspose.Slides for Java kullanarak hedeflenen Web uygulamaları oluşturabilirsiniz. PHP ile Aspose.Slides for Java kullanımına da destek sağlanmaktadır.|
|Linux/Unix|Aspose.Slides for Java platform bağımsız bir API'dir ve Linux ve Unix ortamlarında çalışabilir.|

## **Dosya Formatları ve Dönüşümler**
Aspose.Slides for Java, Microsoft PowerPoint belge formatlarının çoğunu destekler ve bunları kuruluşlar tarafından yaygın olarak kullanılan popüler formatlara dışa aktarır.

|**Özellik**|**Açıklama**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/tr/java/microsoft-powerpoint-ppt/)|Aspose.Slides for Java bu sunum belge formatı için en hızlı işleme performansını sunar.|
|[PresentationML (PPTX, XML)](/slides/tr/java/presentationml-pptx-xml/)|Aspose.Slides for Java, OOXML sunum formatını (PresentationML veya PPTX olarak da bilinir) işlemeyi destekler.|
|[PPT to PPTX conversion](/slides/tr/java/ppt-to-pptx-conversion/)|Aspose.Slides for Java, PPT'den PPTX'e dönüşümü destekler.|
|[Portable Document Format (PDF)](/slides/tr/java/developer-guide/)|Desteklenen dosya formatları tek bir yöntemle Adobe Portable Document Format (PDF) belgelerine dışa aktarılabilir.|
|[XML Parser Specification (XPS)](/slides/tr/java/xml-parser-specification-xps/)|Desteklenen tüm dosya formatları tek bir yöntemle XML Parser Specification (XPS) belgelerine dışa aktarılabilir.|
|[Tagged Image File Format (TIFF)](https://docs.aspose.com/slides/tr/java/convert-powerpoint-to-tiff/)|Aspose.Slides for Java tarafından desteklenen sunum dosya formatları aynı zamanda Tagged Image File Format (TIFF) olarak da dışa aktarılabilir.|
|[ODP to PPTX Conversion](https://docs.aspose.com/slides/tr/java/convert-odp-to-pptx/)|Aspose.Slides for Java, OpenDocument Presentation (ODP) dosyalarını yüklemeyi ve PPTX'e dönüştürmeyi destekler.|
|[Presentation To HTML Conversion](https://docs.aspose.com/slides/tr/java/convert-powerpoint-to-html/)|Aspose.Slides for Java, sunumları HTML formatına dönüştürmeyi destekler.|

## **Sunum Oluşturma**
Aspose.Slides for Java, sunumlardaki slaytların yüksek doğrulukta çeşitli grafik formatlarına render edilmesini destekler:

|**Özellik**|**Açıklama**|
| :- | :- |
|Desteklenen Resim Formatları|Aspose.Slides for Java ile sadece sunum slaytlarını değil, slaytlardaki resimleri de TIFF, PNG, BMP, JPEG, GIF ve metafile gibi popüler grafik formatlarına render edebilirsiniz.| 
|SVG Formatı|Aspose.Slides for Java, sunum slaytlarını Scalable Vector Graphics (SVG) formatına dışa aktarmak için yerleşik bir yöntem sağlar.|

|**Özellik**|**Açıklama**|
| :- | :- |
|Ana Slaytlar|Ana slaytlar, normal slaytların düzenini tanımlar. Aspose.Slides for Java, bir sunumun ana slaytlarına erişmenizi ve bunları değiştirmenizi sağlar.|
|Normal Slaytlar|Aspose.Slides for Java kullanarak yalnızca farklı tiplerde yeni slaytlar oluşturmakla kalmaz, aynı zamanda mevcut slaytlara da erişebilir ve bunları değiştirebilirsiniz.| 
|Slayt Çoğaltma / Kopyalama|Aspose.Slides for Java, mevcut slaytları aynı sunum içinde veya bir sunumdan diğerine kopyalamak/çoğaltmak için yöntemler sunar. Bir slayt, düzenini ana slayttan aldığı için yerleşik çoğaltma yöntemleri ana slaytı da otomatik olarak kopyalar.| 
|Slayt Bölümlerini Yönetme|Sunum içinde slaytları farklı bölümlerde düzenlemenize olanak tanır.| 
|Yer Tutucular ve Metin Tutucular|Bir slayttaki yer tutuculara ve metin tutuculara erişin. Ayrıca uygun yöntemle sıfırdan metin tutuculara sahip bir slayt oluşturabilirsiniz.| 
|Üst Bilgi ve Alt Bilgi|Aspose.Slides for Java, slaytlarda üst bilgi / alt bilgi yönetimini de kolaylaştırır.| 
|Slayt Notları|Aspose.Slides for Java ile bir slayta bağlı notlara erişebilir, bunları değiştirebilir ve yeni notlar ekleyebilirsiniz.| 
|Şekil Bulma|Bir şeklin alternatif metnini kullanarak belirli bir şekli slaytta bulabilirsiniz.|

|Metin Kutuları|Metin kutuları sıfırdan oluşturulabilir. Mevcut metin kutularına erişilip metinleri, orijinal biçimlendirmeyi kaybetmeden değiştirilebilir.| 
|Dikdörtgen Şekiller|Dikdörtgen şekiller Aspose.Slides for Java ile oluşturulabilir veya değiştirilebilir.|

|Grup Şekilleri|Aspose.Slides for Java grup şekilleri destekler.|

## **SSS**

**Sunucuda/PC'de Microsoft PowerPoint kurulu olmalı mı?**

Hayır. PowerPoint gerekli değildir; Aspose.Slides, sunumları oluşturmak, düzenlemek, dönüştürmek ve renderlamak için bağımsız bir motor sağlar.

**Çoklu iş parçacığı (multithreading) nasıl çalışır? İşlem paralel yapılabilir mi?**

Farklı belgeleri farklı iş parçacıklarında işlemek güvenlidir; aynı [sunum](https://reference.aspose.com/slides/tr/java/com.aspose.slides/presentation/) nesnesi aynı anda [birden fazla iş parçacığı](/slides/tr/java/multithreading/) tarafından kullanılmamalıdır.

**Dosya şifreleri ve şifreleme destekleniyor mu?**

Evet. [Şifreli sunumları](/slides/tr/java/password-protected-presentation/) açabilir, açma ve yazma şifresi ayarlayabilir veya kaldırabilir ve koruma durumunu kontrol edebilirsiniz.

**Linux konteynerlerinde yazı tipi paketlerine dikkat etmeli miyim?**

Evet. Beklenmedik yazı tipi ikamelerini önlemek için yaygın yazı tipi paketlerini kurmanız ve/veya uygulamanızda [yazı tipi dizinlerini açıkça belirtmeniz](/slides/tr/java/custom-font/) önerilir.

**Değerlendirme sürümünde sınırlamalar var mı?**

[Değerlendirme modunda](/slides/tr/java/licensing/) bir filigran çıkışa eklenir ve belirli sınırlamalar uygulanır; tam özellikli test için [30 günlük geçici lisans](https://purchase.aspose.com/temporary-license/) mevcuttur.

**Harici formatları (PDF/HTML → PPTX) bir sunuma içe aktarmak destekleniyor mu?**

Evet. [PDF sayfalarını ve HTML içeriğini](/slides/tr/java/import-presentation/) bir sunuma ekleyebilir, bunları slaytlara dönüştürebilirsiniz.
