## **TIFF Hakkında**

[Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) bir belgede birden çok resmi depolayabildiği bilinen format, orijinal olarak Aldus tarafından oluşturulmuştur. Bu format tarama, faks ve diğer görüntü işleme uygulamaları tarafından geniş ölçüde desteklenir.

## **Aspose.Slides for Java’da TIFF**

Aspose.Slides for Java’da yüklenebilen herhangi bir belge, üçüncü taraf bileşen gerektirmeden doğrudan TIFF belgesine dönüştürülebilir. Ayrıca, elde edilen TIFF belgesindeki görüntülerin boyutunu isteğe bağlı olarak belirleyebilirsiniz. Aspose.Slides for Java aracılığıyla sunum belgelerini TIFF belgelerine dışa aktarma hakkında bilgiyi [bu konuda](/slides/tr/java/converting-a-presentation/) bulabilirsiniz.
