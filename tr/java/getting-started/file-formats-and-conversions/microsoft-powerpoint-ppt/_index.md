{{% alert color="primary" %}} 

[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) Microsoft PowerPoint'in farklı sürümleri tarafından oluşturulabilen, okunabilen, işlenebilen ve yazılabilen bir sunum belge dosyası formatıdır. Bu, Microsoft tarafından geliştirilen sunum belgeleri için ikili bir formattır.

{{% /alert %}} 

## **PPT, Aspose.Slides for Java'da**
Aspose.Slides for Java, aşağıda listelenen yazılımlarla oluşturulmuş PPT dosyalarını okuyabilir.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Benzer şekilde, Aspose.Slides for Java tarafından oluşturulan PPT dosyaları, yukarıdaki yazılımlarla okunabilir.

## **PPT için Kapsamlı Destek**
Aspose.Slides for Java, PPT belge dosyası formatı tarafından desteklenen hemen hemen tüm özellikleri sağlar. Bu, farklı Microsoft PowerPoint sürümleri tarafından PPT belge manipülasyonları için sağlanan temel ve gelişmiş özelliklerin yanı sıra Microsoft PowerPoint tarafından bile desteklenmeyen özellikleri kapsar. Aspose.Slides for Java API kütüphanesini kullanmanın temel avantajı, bu tür özellikleri işleme kolaylığıdır.

PPT belge dosyalarını oluşturma, okuma ve yazma ile ilgili temel görevlerin yanı sıra, Aspose.Slides for Java tarafından sağlanan birkaç özellik daha vardır:

- Diğer Microsoft Office dosya formatlarını [OLE nesneleri olarak PPT belgelerine]().
- [PPT belgelerini PDF'ye dışa aktar](/slides/tr/java/convert-powerpoint-ppt-and-pptx-to-pdf/).
- PPT belgelerindeki slaytları SVG formatına dışa aktar.
- Slaytları Java Çerçevesi tarafından desteklenen herhangi bir görüntü formatına render et.
- PPT belgelerindeki slaytların boyutunu ayarla.
- Şekillerdeki animasyonları yönet.
- Slayt gösterilerini yönet.
- [Slaytlardaki metni biçimlendir]().
- PPT belgelerinden metin çıkar.
- [Slaytlardaki tabloları işle]().
- Klonlama özelliği kullanarak ana şablonları otomatik olarak kopyala.
