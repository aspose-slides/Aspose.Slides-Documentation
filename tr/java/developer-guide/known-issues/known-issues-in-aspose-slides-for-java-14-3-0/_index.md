Aspose.Slides for Java 14.3.0 (14.4.0), PPT işleme için tamamen yeni bir uygulama sunar. Birçok iyileştirme, PPTX'ten PPT'ye kısmi dönüştürme bulunmaktadır. Ancak bazı uygulanmamış özellikler vardır:

- Bazı şekiller, serileştirilmiş PPT belgelerinde (Call outs) yanlış geometriye sahiptir
- Tüm PPTX metin biçimlendirme özellikleri PPT serileştirmesinde desteklenmez
- Metin dili ve yazım denetimi ayarlarıyla ilgili bilgi serileştirilmiş PPT belgelerinde bulunmaz
- Tüm PPTX temalarındaki özellikler PPT serileştirmesinde desteklenmez

**Aspose.Slides for Java 8.6.0 ile karşılaştırıldığında bazı farklar vardır:**

- PPT'ye OLE/ActiveX PPT serileştirmesinde bilinen sorunlar vardır

**Aspose.Slides for .NET 14.3.0 ile karşılaştırıldığında bazı farklar vardır:**

- Sunum yazdırma desteği şu anda Aspose.Slides for Java'da mevcut değildir
