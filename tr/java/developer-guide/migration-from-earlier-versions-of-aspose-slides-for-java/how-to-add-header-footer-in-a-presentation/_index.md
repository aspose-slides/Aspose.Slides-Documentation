Yeni bir [Aspose.Slides for Java API](https://docs.aspose.com/slides/tr/java/) yayınlandı ve artık bu tek ürün, sıfırdan PowerPoint belgeleri oluşturma ve mevcut belgeleri düzenleme yeteneğini destekliyor.

{{% /alert %}} 
## **Eski Kod Desteği**
Aspose.Slides for Java 13.x öncesi sürümlerle geliştirilen eski kodu kullanmak için kodunuzda bazı küçük değişiklikler yapmanız gerekir ve kod daha önceki gibi çalışacaktır. Eski Aspose.Slides for Java'da Aspose.Slide ve Aspose.Slides.Pptx ad alanları altında bulunan tüm sınıflar artık tek bir Aspose.Slides ad alanında birleştirildi. Lütfen sunumda başlık ve altbilgi eklemek için eski Aspose.Slides API'sindeki aşağıdaki basit kod snippet'ine bir göz atın ve yeni birleştirilmiş API'ye nasıl geçileceğini açıklayan adımları izleyin.
