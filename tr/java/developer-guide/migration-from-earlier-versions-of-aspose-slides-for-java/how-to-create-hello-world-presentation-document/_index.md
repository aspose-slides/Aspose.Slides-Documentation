Yeni bir [Aspose.Slides for Java API](/slides/tr/java/) yayınlandı ve artık bu tek ürün, PowerPoint belgelerini sıfırdan oluşturma ve mevcut belgeleri düzenleme yeteneğini destekliyor.

## **Eski Kod Desteği**
13.x sürümlerinden önceki Aspose.Slides for Java sürümleriyle geliştirilen eski kodu kullanmak için kodunuzda bazı küçük değişiklikler yapmanız gerekir ve kod önceki gibi çalışacaktır. Eski Aspose.Slides for Java'da Aspose.Slide ve Aspose.Slides.Pptx ad alanları altında bulunan tüm sınıflar artık tek bir Aspose.Slides ad alanında birleştirildi. Lütfen aşağıdaki basit kod parçacığını, eski Aspose.Slides API'sinde bir Hello World Sunum belgesi oluşturmak için inceleyin ve yeni birleştirilmiş API'ye nasıl geçileceğini açıklayan adımları izleyin.
