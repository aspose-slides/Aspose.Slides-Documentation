## **Sunucu Gereksinimleri**
{{% alert color="primary" %}} 

Aspose.Slides for SharePoint'i kurmak için, aşağıdakilerden birini çalıştıran bir sistem gereklidir:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft Office SharePoint Server 2010
- Microsoft Office SharePoint Server 2013
- Microsoft Office SharePoint Server 2016
- Microsoft Office SharePoint Server 2019

{{% /alert %}} {{% alert color="primary" %}} 

Bu ürünler için Service Pack 2 veya daha yeni bir sürümün kurulması önerilir.

{{% /alert %}} 
## **İstemci Gereksinimleri**
{{% alert color="primary" %}} 

SharePoint sitelerine bir web tarayıcısı aracılığıyla erişilir. Aspose.Slides for SharePoint aşağıdaki tarayıcılarla test edilmiştir:

- Microsoft Internet Explorer 6, 7, 8
- Mozilla FireFox 3.5

{{% /alert %}}
