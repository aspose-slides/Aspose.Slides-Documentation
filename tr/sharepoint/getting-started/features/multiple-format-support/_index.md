{{% alert color="primary" %}} 
Aspose.Slides for SharePoint ile, bir SharePoint belge kitaplığının içinden birçok popüler ofis belge formatı arasında belgeleri dönüştürebilirsiniz. Dönüştürmeler yüksek doğruluk ve hassasiyetle gerçekleştirilir. 
{{% /alert %}} 
## **Desteklenen Giriş Formatları**
Aspose.Slides for SharePoint aşağıdaki giriş formatlarını destekler: 

- PPT – Microsoft PowerPoint Sunumu 97 - 2003
- PPS – Microsoft PowerPoint Slayt Gösterisi 97 - 2003
- POT – Microsoft PowerPoint Şablonu 97 - 2003
- PPTX – Office Open XML Sunumu
- PPSX – Office Open XML Slayt Gösterisi
- POTX – Office Open XML Şablonu

{{% alert color="primary" %}} 
Belge oluşturmak için, Aspose.Slides for SharePoint, [Aspose.Slides for .NET](http://www.aspose.com/categories/.net-components/aspose.slides-for-.net/default.aspx) adlı yerleşik sürümüne dayanır; bu, Aspose'un tek PowerPoint belge işleme bileşenidir.
{{% /alert %}}
## **Desteklenen Çıktı Formatları**
Aspose.Slides for SharePoint aşağıdaki çıktı formatlarını destekler: 

- PDF – Taşınabilir Belge Formatı
- TIFF – Resim Paketi
- XPS – XML Kağıt Spesifikasyonu
