---
Uygulamanızdan Aspose.Slides for JasperReports'ı kullanmak için, aspose.slides.jasperreports.library-xx.x.jar dosyasını aspose-slides-xx.x-jasperreports.zip dosyasının \lib\JasperReports X.X.X - X.X.X klasöründen JasperReports\lib dizinine veya uygulamanızın bir kütüphane klasörüne kopyalayın. Bundan sonra dışa aktarıcılara program aracılığıyla erişebilirsiniz.
