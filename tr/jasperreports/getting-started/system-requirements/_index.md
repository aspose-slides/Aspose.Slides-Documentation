Aspose.Slides for JasperReports aşağıdaki ile çalışır:

**Aspose.Slides for JasperReports 17.5 ve önceki sürümler:**

- Java 2 Platform Standard Edition 1.4 veya daha yeni.
- JasperReports 2.0, 3.0 veya daha yeni.

**Aspose.Slides for JasperReports 17.6 ve sonraki sürümler:**

- Java 2 Platform Standard Edition 6.0 veya daha yeni.
- JasperReports 3.7.2 veya daha yeni.
