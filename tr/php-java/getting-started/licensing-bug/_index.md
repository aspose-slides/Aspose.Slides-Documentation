## **Genel Bakış**

Bu makale, Aspose.Slides for PHP via Java 23.2, 23.3, 23.4, 23.5, 23.6 ve 23.7 sürümlerinde karşılaşılan bir lisanslama hatasını tanımlar. Bu hata, kalıcı lisansların abonelikleri sona erdiğinde geçici olarak kullanılamaz hale gelmesine sebep olmuştur.

## **Semptomlar**

Kalıcı lisans aboneliğiniz sona erdiğinde, Aspose.Slides for PHP via Java 23.2 - 23.7 sürümleri lisansı süresi dolmuş olarak raporlayabilir ve tüm işlevleri değerlendirme moduna geçirebilir.  
Bu davranış yanlıştır ve 23.2 öncesi ve 23.7 sonrası sürümleri etkilemez.

## **Çözüm**

Aspose.Slides for PHP via Java bu sorunu ele almış ve düzeltmeyi içeren güncellenmiş sürümleri (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) yayınlamıştır.
