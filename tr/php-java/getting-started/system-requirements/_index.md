## **Giriş**

Aspose.Slides for PHP via Java, Microsoft PowerPoint gibi üçüncü taraf bir ürünün kurulmasını gerektirmez. Aspose.Slides, Microsoft PowerPoint sunum formatları da dahil olmak üzere çeşitli biçimlerde belgeler oluşturma, değiştirme, dönüştürme ve renderleme motorudur.

## **Desteklenen İşletim Sistemleri**

Aspose.Slides for Java, Java çalışma zamanı çalıştıran herhangi bir 32‑bit veya 64‑bit işletim sistemini, aşağıdakilerle sınırlı olmaksızın destekler:

### **Windows**
- Microsoft Windows 2003 Server ( x64, x86)
- Microsoft Windows 2008 Server ( x64, x86)
- Microsoft Windows 2012 Server ( x64, x86)
- Microsoft Windows 2012 R2 Server ( x64, x86)
- Microsoft Windows 2016 Server ( x64, x86)
- Microsoft Windows 2019 Server ( x64, x86)
- Microsoft Windows Vista ( x64, x86)
- Microsoft Windows XP ( x64, x86)
- Microsoft Windows 7 ( x64, x86)
- Microsoft Windows 8, 8.1 ( x64, x86)
- Microsoft Windows 10 ( x64, x86)

### **Linux**
- Linux (Ubuntu, OpenSUSE, CentOS ve diğerleri)

### **Mac**
- Mac OS X

## **SSS**

**Dönüştürme ve renderleme için Microsoft PowerPoint'in yüklü olması gerekiyor mu?**

Hayır, PowerPoint gerekli değildir; Aspose.Slides, sunumları [oluşturmak](/slides/tr/php-java/create-presentation/), değiştirmek, [dönüştürmek](/slides/tr/php-java/convert-presentation/) ve [renderlemek](/slides/tr/php-java/convert-powerpoint-to-png/) için bağımsız bir motor sunar.

**Doğru renderleme için hangi fontlar gerekir?**

Pratikte, sunumda kullanılan fontlar veya uygun [alternatifler](/slides/tr/php-java/font-substitution/) mevcut olmalıdır. Linux/macOS üzerinde tutarlı renderleme sağlamak için yaygın font paketlerini kurmak tavsiye edilir.

**Neden özel bir font Linux'ta yedek veya eksik metin olarak görüntüleniyor?**

Font dosyasının tutarsız veya bozuk name-table girişleri varsa, Linux font eşleştirme yığını (FreeType/fontconfig) geçersiz bir kaydı seçebilir ve bu da fontun çözülmemesine neden olur. Düzeltilmiş name-table kayıtlarına sahip bir font sürümü kullanmak veya tutarlı bir yedek kurmak sorunu çözer.
