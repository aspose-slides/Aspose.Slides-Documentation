{{% alert color="primary" %}} 

[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) Microsoft PowerPoint'in farklı sürümleri tarafından oluşturulabilen, okunabilen, değiştirilebilen ve yazılabilen bir sunum belge dosya formatıdır. Bu, Microsoft tarafından geliştirilen sunum belgeleri için ikili bir formattır.

{{% /alert %}} 

## **Aspose.Slides for PHP via Java'da PPT**
Aspose.Slides for PHP via Java, aşağıda listelenen yazılımlar tarafından oluşturulan PPT dosyalarını okuyabilir.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Benzer şekilde, Aspose.Slides for PHP via Java ile oluşturulan PPT dosyaları da yukarıdaki yazılımlar tarafından okunabilir.

## **PPT için Kapsamlı Destek**
Aspose.Slides for PHP via Java, PPT belge dosya formatı tarafından desteklenen hemen hemen tüm özelliklere destek sağlar. Bu, farklı Microsoft PowerPoint sürümlerinin PPT belge manipülasyonları için sunduğu temel ve gelişmiş özellikleri kapsamakla kalmaz, aynı zamanda Microsoft PowerPoint tarafından bile desteklenmeyen özellikleri içerir. Aspose.Slides for PHP via Java API kitaplığını kullanmanın en önemli avantajı, bu özellikleri kolayca kullanabilme kolaylığıdır.

PPT belge dosyalarını oluşturma, okuma ve yazma ile ilgili temel görevlerin yanı sıra, Aspose.Slides for PHP via Java tarafından sağlanan çeşitli özellikler de bulunmaktadır:

- Diğer Microsoft Office dosya formatlarını [OLE nesneleri olarak PPT belgelerine](/slides/tr/php-java/manage-ole/) aktar.
- [PPT belgelerini PDF'e dışa aktar](/slides/tr/php-java/convert-powerpoint-to-pdf/).
- PPT belgelerindeki slaytları SVG formatına dışa aktar.
- Slaytları Java Framework'ün desteklediği herhangi bir görüntü formatına render et.
- PPT belgelerindeki slaytların boyutunu ayarla.
- Şekiller üzerindeki animasyonları yönet.
- Slayt gösterilerini yönet.
- [Slaytlardaki metni biçimlendir](/slides/tr/php-java/text-formatting/).
- PPT belgelerinden metni çıkar.
- [Slaytlardaki tabloları yönet](/slides/tr/php-java/powerpoint-table/).
- [Klonlama özelliğini](/slides/tr/php-java/clone-slides/) kullanarak ana slaytları otomatik olarak kopyala.
