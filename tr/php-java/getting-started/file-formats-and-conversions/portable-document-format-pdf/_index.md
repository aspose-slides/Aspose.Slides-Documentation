{{% alert color="primary" %}} 

[Taşınabilir Belge Biçimi](https://en.wikipedia.org/wiki/PDF) Adobe Systems tarafından organizasyonlar arasında belge değiş tokuşu için oluşturulan bir dosya formatıdır. Formatın amacı, içeriği ve düzeni, görüntülendiği platformdan bağımsız olarak aynı tutmaktı. Aspose.Slides for PHP via Java, sunum dosyalarını PDF'ye dönüştürmenizi sağlar.

{{% /alert %}} 

## **PDF, Aspose.Slides for PHP via Java'da**
Aspose.Slides for PHP via Java'ye yüklenebilen herhangi bir sunum, tercihinize bağlı olarak [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) veya [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA) standardına uyan bir PDF'ye dönüştürülebilir. Aspose.Slides for PHP via Java, sunumları PDF olarak dışa aktarır ve çoğu durumda, oluşan PDF orijinal sunuma tam olarak benzer.

Aspose.Slides, PDF'ye dönüştürürken aşağıdaki sunum özelliklerini destekler:

- Görseller, metin kutuları ve diğer şekiller.
- Metin ve biçimlendirme.
- Paragraflar ve biçimlendirme.
- Köprüler.
- Üstbilgiler ve altbilgiler.
- Madde işaretleri.
- Tablolar.

Aspose.Slides for PHP via Java kullanarak sunumları doğrudan PDF'ye dışa aktarabilirsiniz: başka bir bileşene ihtiyacınız yoktur. Ayrıca, [PDF'ye Dönüştürme](/slides/tr/php-java/convert-powerpoint-to-pdf/) bölümünde açıklandığı gibi çeşitli seçeneklerle sunumdan PDF'ye dışa aktarmayı özelleştirebilirsiniz.
