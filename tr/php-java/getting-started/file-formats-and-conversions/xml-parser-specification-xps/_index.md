{{% alert color="primary" %}} 

XML Ayrıştırıcı Spesifikasyonu ([https://en.wikipedia.org/wiki/Open_XML_Paper_Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification)) bir sayfa tanımlama dili ve Microsoft tarafından geliştirilen sabit belge formatıdır. PDF gibi, XPS belge sadakatini korumak ve cihazdan bağımsız belge görünümü sağlamak için tasarlanmıştır. 

{{% /alert %}} 

## **Aspose.Slides for PHP via Java içinde XPS**
Aspose.Slides for PHP via Java tarafından yüklenebilen herhangi bir sunum belgesi XPS formatına dönüştürülebilir. Aspose.Slides for PHP via Java, yüksek doğruluklu sayfa düzeni ve render motorunu kullanarak sabit‑düzen XPS belge formatında çıktı üretir.  
Aspose.Slides for PHP via Java ile sunum belgelerini XPS belgelerine dışa aktarmayı [XPS'ye Dönüştürme](https://docs.aspose.com/slides/tr/php-java/convert-powerpoint-to-xps/) bölümünde öğrenebilirsiniz.
