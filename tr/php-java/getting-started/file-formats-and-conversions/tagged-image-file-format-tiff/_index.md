## **TIFF Hakkında**

Birden fazla görüntüyü tek bir belgede saklamasıyla bilinen [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF), aslında Aldus tarafından oluşturulmuştur. Bu format tarama, faks ve diğer görüntü işleme uygulamaları tarafından yaygın olarak desteklenir.

## **Aspose.Slides for PHP via Java ile TIFF**

Aspose.Slides for Java içinde yüklenebilen herhangi bir belge, üçüncü taraf bileşen gereksinimini ortadan kaldırarak doğrudan Aspose.Slides for PHP via Java ile TIFF belgesine dönüştürülebilir. Ayrıca, ortaya çıkan TIFF belgesindeki görüntülerin boyutunu isteğe bağlı olarak belirtebilirsiniz. Aspose.Slides for PHP via Java aracılığıyla sunum belgelerini TIFF belgelerine dışa aktarma hakkında bilgiye [bu konuda](/slides/tr/php-java/convert-powerpoint-to-tiff/) ulaşabilirsiniz.
