{{% alert color="primary" %}} 

Aspose.Slides for Node.js via Java, Microsoft PowerPoint® veya diğer programları kullanmadan Node.js uygulamalarının sunum belgelerini okumasını ve yazmasını sağlayan bir sınıf kitaplığıdır. Aspose.Slides for Node.js via Java, metin, şekiller, tablolar ve animasyonları yönetmenize, slaytlara ses ve video eklemenize, slaytları önizlemenize, slaytları SVG, PDF formatına dışa aktarmanıza ve çok daha fazlasını yapmanıza olanak tanır.

En son API Referansı, [Aspose.Slides for Node.js via Java API Reference](https://reference.aspose.com/slides/tr/nodejs-java/) adresinde bulunabilir.
