## **Desteklenen Platformlar**
Aspose.Slides for Node.js via Java, platform bağımsız bir API'dir. Node.js 7 veya daha yüksek bir sürümü yüklü olduğu herhangi bir platformda (Windows, Linux, macOS vb.) kullanılabilir. Oracle JDK 8 veya daha yüksek bir sürüm de bir diğer gereksinimdir.

## **Dosya Formatları ve Dönüşümler**
Aspose.Slides for Node.js via Java, çoğu PowerPoint belge formatını destekler. Ayrıca bu belgeleri kuruluşların kullandığı ve birbirleriyle paylaştığı popüler dosya formatlarına dışa aktarmanıza olanak tanır. Ayrıntılara göz atın:

|**Özellik**|**Açıklama**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via Java, bu sunum belge formatı için en hızlı işleme sağlayıcısını sunar.|
|PPT'den PPTX'e dönüşüm|Aspose.Slides for Node.js via Java, PPT'den PPTX'e dönüşümü destekler.|
|Portable Document Format (PDF)|Desteklenen tüm dosya formatlarını tek bir yöntemle Adobe Taşınabilir Belge Formatı (PDF) belgelerine dışa aktarabilirsiniz.|
|XML Parser Specification (XPS)|Desteklenen tüm dosya formatlarını tek bir yöntemle XML Parser Specification (XPS) belgelerine dışa aktarabilirsiniz.|
|Tagged Image File Format (TIFF)|Desteklenen sunum dosya formatlarını Tagged Image File Format (TIFF) formatına dışa aktarabilirsiniz.|
|PPTX'den HTML Dönüşümü|Aspose.Slides for Node.js via Java, PresentationEx'in HTML formatına dönüştürülmesini destekler.|

|**Özellik**|**Açıklama**|
| :- | :- |

## **SSS**

**Kütüphanenin çalışması için sunucu/PC'ye Microsoft PowerPoint kurmam gerekiyor mu?**

Hayır. PowerPoint gerektirilmez; Aspose.Slides, sunumları oluşturmak, düzenlemek, dönüştürmek ve renderlemek için bağımsız bir motor sağlar.

**Çoklu iş parçacığı (multithreading) nasıl çalışır? İşlem paralelleştirilebilir mi?**

Farklı belgeleri farklı iş parçacıklarında işlemek güvenlidir; aynı [presentation](https://reference.aspose.com/slides/tr/nodejs-java/aspose.slides/presentation/) nesnesi aynı anda [multiple threads](/slides/tr/nodejs-java/multithreading/) tarafından kullanılmamalıdır.

**Dosya şifreleri ve şifreleme destekleniyor mu?**

Evet. [Şifreli sunumları açabilirsiniz](/slides/tr/nodejs-java/password-protected-presentation/), açma ve yazma şifresi ayarlayabilir veya kaldırabilir ve koruma durumunu kontrol edebilirsiniz.

**Linux konteynerlerinde font paketlerine dikkat etmem gerekiyor mu?**

Evet. Beklenmedik font değişimlerini önlemek için yaygın font paketlerini kurmanız ve/veya uygulamanızda font dizinlerini açıkça [font dizinlerini belirtmek](/slides/tr/nodejs-java/custom-font/) önerilir.

**Değerlendirme sürümünde sınırlamalar var mı?**

Değerlendirme modunda ([evaluation mode](/slides/tr/nodejs-java/licensing/)) çıktı üzerine bir filigran eklenir ve bazı sınırlamalar geçerli olur; tam özellikli test için bir [30 günlük geçici lisans](https://purchase.aspose.com/temporary-license/) mevcuttur.

**Harici formatların bir sunuma (PDF/HTML → PPTX) aktarılması destekleniyor mu?**

Evet. Sunuma [PDF sayfaları ve HTML içeriği](/slides/tr/nodejs-java/import-presentation/) ekleyebilir, bunları slaytlara dönüştürebilirsiniz.
