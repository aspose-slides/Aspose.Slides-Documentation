{{% alert color="primary" %}} 

Aspose.Slides for Python via .NET, Microsoft PowerPoint® veya diğer bir yazılım kullanmadan Python uygulamalarının sunum belgelerini okumasına ve yazmasına olanak tanıyan bir sınıf kitaplığıdır. Aspose.Slides for Python via .NET, metin, şekiller, tablolar ve animasyonları yönetmenizi; slaytlara ses ve video eklemenizi; slaytları önizlemenizi; ve slaytları SVG, PDF formatına ve daha fazlasına dışa aktarmanızı sağlar.

En son API Referansı, [Aspose.Slides for Python via .NET API Reference](https://reference.aspose.com/slides/tr/python-net/) adresinde bulunabilir.
