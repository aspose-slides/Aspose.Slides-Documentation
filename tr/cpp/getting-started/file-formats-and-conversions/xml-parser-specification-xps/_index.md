## **XPS Hakkında**
[XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) bir sayfa tanımlama dili ve Microsoft tarafından geliştirilen sabit belge formatıdır. PDF gibi, XPS belge bütünlüğünü korumak ve cihazdan bağımsız belge görünümü sağlamak için tasarlanmış sabit yerleşimli bir belge formatıdır.
## **Aspose.Slides for C++ içinde XPS**
Aspose.Slides for C++ tarafından yüklenebilen herhangi bir sunum belgesi XPS formatına dönüştürülebilir. Aspose.Slides for C++ yüksek doğruluklu sayfa yerleşimi ve renderleme motorunu kullanarak sabit yerleşimli XPS belge formatında çıktı üretir. Şunu belirtmek gerekir ki Aspose.Slides for C++ Windows Presentation Foundation (WPF) sınıflarına, C++ Framework 3.5 ile paketlenmiş olanlara bağımlı olmadan doğrudan XPS üretir; bu sayede Aspose.Slides for C++ C++ Framework 3.5'ten önceki sürümleri çalışan makinelerde XPS belgeleri oluşturabilir. Aspose.Slides for C++ aracılığıyla sunum belgelerini XPS belgelerine aktarmayı [bu konuda](/slides/tr/net/) öğrenebilirsiniz.
