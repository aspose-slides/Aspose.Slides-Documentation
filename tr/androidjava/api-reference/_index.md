{{% alert color="primary" %}}

Aspose.Slides for Android via Java, Android için Java uygulamalarının Microsoft PowerPoint® veya başka bir yazılım kullanmadan sunum belgelerini okumasını ve yazmasını sağlayan bir sınıf kitaplığıdır. Aspose.Slides for Android via Java, metin, şekil, tablo ve animasyonları yönetmenize; slaytlara ses ve video eklemenize; slaytları önizlemenize; ve slaytları SVG, PDF formatına ve daha fazlasına dışa aktarmanıza olanak tanır.

En son API Referansına şu adreste ulaşabilirsiniz: [Aspose.Slides for Android via Java API Referansı](https://reference.aspose.com/slides/tr/androidjava/).
