Aspose.Slides for Node.js via .NET, Microsoft PowerPoint gibi üçüncü taraf bir ürünün kurulmasını gerektirmez. Aspose.Slides, çeşitli formatlarda, Microsoft PowerPoint sunum formatları da dahil olmak üzere, belgeleri oluşturmak, değiştirmek, dönüştürmek ve işlemek için bir motor sağlar.

## Desteklenen İşletim Sistemleri

Aspose.Slides for .NET, .NET çalışma zamanını çalıştıran herhangi bir 32‑bit veya 64‑bit işletim sistemini, aşağıdakilerle sınırlı olmaksızın, destekler:

### **Windows**
- Microsoft Windows 2003 Server ( x64, x86)
- Microsoft Windows 2008 Server ( x64, x86)
- Microsoft Windows 2012 Server ( x64, x86)
- Microsoft Windows 2012 R2 Server ( x64, x86)
- Microsoft Windows 2016 Server ( x64, x86)
- Microsoft Windows 2019 Server ( x64, x86)
- Microsoft Windows Vista ( x64, x86)
- Microsoft Windows XP ( x64, x86)
- Microsoft Windows 7 ( x64, x86)
- Microsoft Windows 8, 8.1 ( x64, x86)
- Microsoft Windows 10 ( x64, x86)
- Microsoft Windows 11 ( x64, x86)

### **Linux**
- Linux (Ubuntu, CentOS ve diğerleri)

### **Mac**
- Mac OS X
