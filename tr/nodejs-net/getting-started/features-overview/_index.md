## **Desteklenen Platformlar**
Aspose.Slides for Node.js via .NET, platform bağımsız bir API'dir. Node.js 7 veya üzeri yüklü olan herhangi bir platformda (Windows, Linux, macOS vb.) kullanılabilir. Oracle JDK 8 veya üzeri bir diğer gereksinimdir.

## **Dosya Biçimleri ve Dönüşümler**
Aspose.Slides for Node.js via .NET, çoğu PowerPoint belge biçimini destekler. Ayrıca, bunları kuruluşların kullandığı ve paylaştığı popüler dosya biçimlerine dışa aktarabilirsiniz. Aşağıdaki detaylara göz atın:

|**Özellik**|**Açıklama**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via .NET, bu sunum belge formatı için en hızlı işleme sağlar.|
|PPT'den PPTX'e dönüşüm|Aspose.Slides for Node.js via .NET, PPT'den PPTX'e dönüşümü destekler|
|Taşınabilir Belge Biçimi (PDF)|Tek bir yöntemle tüm desteklenen dosya biçimlerini Adobe Portable Document Format (PDF) belgelerine dışa aktarabilirsiniz.|
|XML Parser Specification (XPS)|Tek bir yöntemle tüm desteklenen dosya biçimlerini XML Parser Specification (XPS) belgelerine dışa aktarabilirsiniz.|
|Tagged Image File Format (TIFF)|Tüm desteklenen sunum dosya biçimlerini Tagged Image File Format (TIFF) biçimine dışa aktarabilirsiniz.|
|PPTX'den HTML Dönüşümü|Aspose.Slides for Node.js via .NET, PresentationEx'in HTML formatına dönüştürülmesini destekler.|

## **Görselleştirme ve Yazdırma**
Aspose.Slides for Node.js via .NET, sunumdaki slaytların yüksek doğrulukta farklı grafik biçimlerine renderlanmasını destekler. Aşağıdaki detaylara göz atın:

|**Özellik**|**Açıklama**|
| :- | :- |
|.NET Destekli Resim Biçimleri|Aspose.Slides for Node.js via .NET kullanarak, sunum slaytlarını ve slaytlardaki görüntüleri TIFF, PNG, BMP, JPEG, GIF gibi .NET destekli tüm grafik biçimlerine renderlayabilirsiniz.|
|SVG Biçimi|Aspose.Slides for Node.js via .NET ayrıca sunum slaytlarını Scalable Vector Graphics (SVG) biçimlerine dışa aktarmanızı sağlayan yerleşik yöntemler sunar.|
|Sunum Yazdırma|Aspose.Slides for Node.js via .NET'in en son sürümleri, farklı seçeneklere sahip yerleşik yazdırma yöntemleri sunar.|

## **İçerik Özellikleri**
Aspose.Slides for Node.js via .NET, sunum belgelerinin neredeyse tüm öğelerine veya içeriklerine erişmenizi, değiştirmenizi veya oluşturmanızı sağlar. Aşağıdaki detaylara göz atın:

|**Özellik**|**Açıklama**|
| :- | :- |
|Master Slaytlar|Master slaytlar, normal slaytların düzenini tanımlar. Aspose.Slides for Node.js via .NET, sunum belgelerinin Master slaytlarına erişmenizi ve değiştirmenizi sağlar.|
|Normal Slaytlar|Aspose.Slides for Node.js via .NET ile farklı tiplerde yeni slaytlar oluşturabilir; ayrıca sunumlardaki mevcut slaytlara erişebilir ve bunları değiştirebilirsiniz.|
|Slayt Klonlama / Kopyalama|Aspose.Slides for Node.js via .NET, bir sunum içinde mevcut slaytları klonlamanızı veya kopyalamanızı sağlayan yerleşik yöntemler sunar. Kopyalanan ve klonlanan slaytları bir sunumdan diğerine de kullanabilirsiniz. Bir slayt, düzenini master slayttan miras aldığı için yerleşik klonlama yöntemleri, klonlama sırasında master'ı otomatik olarak kopyalar.|
|Slayt Bölümlerini Yönetme|Sunum içinde slaytları farklı bölümlerde düzenlemek için yöntemler.|
|Yer Tutucular ve Metin Tutucular|Bir slayttaki yer tutuculara ve metin tutuculara erişebilirsiniz. Ayrıca, uygun yöntemi kullanarak sıfırdan metin tutuculara sahip bir slayt oluşturabilirsiniz.|
|Üstbilgi ve Altbilgi|Aspose.Slides for Node.js via .NET, slaytlardaki üstbilgi/altbilgi yönetimini kolaylaştırır.|
|Slaytlardaki Notlar|Aspose.Slides for Node.js via .NET ile bir slaytla ilişkili notlara erişebilir, bunları değiştirebilir ve yeni notlar ekleyebilirsiniz.|
|Şekil Bulma|Şekilin alternatif metnini kullanarak bir slayttan belirli bir şekli bulabilirsiniz.|
|Arka Planlar|Aspose.Slides for Node.js via .NET, bir sunumda master veya normal slaytla ilişkili arka planlarla çalışmanıza olanak tanır.|
|Metin Kutuları|Metin kutuları sıfırdan oluşturulabilir. Mevcut metin kutularına erişebilirsiniz. Ayrıca, orijinal metin biçimini kaybetmeden metinlerini değiştirebilirsiniz.|
|Dikdörtgen Şekiller|Aspose.Slides for Node.js via .NET ile dikdörtgen şekiller oluşturabilir veya değiştirebilirsiniz.|
|Çoklu Çizgi Şekilleri|Aspose.Slides for Node.js via .NET ile çoklu çizgi şekilleri oluşturabilir veya değiştirebilirsiniz.|
|Elips Şekilleri|Aspose.Slides for Node.js via .NET ile elips şekilleri oluşturabilir veya değiştirebilirsiniz.|
|Grup Şekilleri|Aspose.Slides for Node.js via .NET, grup şekilleri destekler|
|Otomatik Şekiller|Aspose.Slides for Node.js via .NET, otomatik şekilleri destekler|
|SmartArt|Aspose.Slides for Node.js via .NET, MS PowerPoint'teki SmartArt şekilleri için destek sağlar|
|Grafikler|Aspose.Slides for Node.js via .NET, PowerPoint'teki MSO Grafikleri için destek sağlar|
|Şekil Serileştirme|Aspose.Slides for Node.js via .NET, çok sayıda şekli destekler. Aspose.Slides for Node.js via .NET bir şekli desteklemediğinde, mevcut bir slayttan o şekli serileştirebileceğiniz bir serileştirme yöntemi kullanabilirsiniz.
