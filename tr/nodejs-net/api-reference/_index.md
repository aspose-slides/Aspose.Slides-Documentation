Aspose.Slides for Node.js via .NET, Microsoft PowerPoint® veya diğer programları kullanmadan Node.js uygulamalarının sunum belgelerini okumasını ve yazmasını sağlayan bir sınıf kitaplığıdır. Aspose.Slides for Node.js via .NET, metin, şekiller, tablolar ve animasyonları yönetmenize, slaytlara ses ve video eklemenize, slaytları önizlemenize, slaytları SVG, PDF formatına dışa aktarmanıza ve çok daha fazlasını yapmanıza olanak tanır.

Aspose.Slides for Node.js via .NET, [Aspose.Slides .NET ile aynı API referansı](https://reference.aspose.com/slides/tr/nodejs-net/) kullanır.
