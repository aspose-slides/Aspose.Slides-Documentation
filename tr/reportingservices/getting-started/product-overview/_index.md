## **Aspose.Slides for Reporting Services'e Hoş Geldiniz!**

Aspose.Slides for Reporting Services, Microsoft SQL Server 2005, 2008, 2012, 2016 ve 2017 Reporting Services (32‑bit ve 64‑bit) içinde gerçek PPT ve PPS raporları oluşturmayı sağlayan piyasadaki tek çözümdür. Tablolar, matrisler, grafikler ve görseller dahil tüm RDL rapor özellikleri, en yüksek hassasiyetle Microsoft PowerPoint sunumlarına dönüştürülür.

## **Ürün Genel Bakış**

Microsoft SQL Server Reporting Services, raporları Microsoft PowerPoint sunumları olarak dışa aktarmak için yerleşik bir yeteneğe sahip değildir, ancak sunucunuza Aspose.Slides for Reporting Services yüklendikten sonra aşağıdaki ek dışa aktarma biçimlerine erişim elde edersiniz:

- PPT - Aspose.Slides aracılığıyla PowerPoint Sunumu
- PPS - Aspose.Slides aracılığıyla PowerPoint Slayt Gösterisi
- PPTX - Aspose.Slides aracılığıyla PowerPoint 2007 Sunumu
- PPSX - Aspose.Slides aracılığıyla PowerPoint 2007 Slayt Gösterisi

Aspose.Slides for Reporting Services, Microsoft PowerPoint kullanmadan sunucuda sunumlar oluşturur. Aspose.Slides for Reporting Services, içinde Aspose.Slides for .NET’i – sunucu tarafı sunum işleme için dünya çapında sınıf bir bileşeni – kullanır.

**Aspose.Slides for Reporting Services, herhangi bir raporu PPT, PPS, PPTX veya PPSX formatında dışa aktarmayı mümkün kılar.**
