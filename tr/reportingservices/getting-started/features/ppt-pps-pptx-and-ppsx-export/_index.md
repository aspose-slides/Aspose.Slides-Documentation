{{% alert color="primary" %}} 

Microsoft SQL Server Reporting Services, raporları Microsoft PowerPoint sunumları olarak dışa aktarmak için yerleşik yeteneklere sahip değildir, ancak sunucunuza Aspose.Slides for Reporting Services'i kurduktan sonra iki ek dışa aktarma formatına erişebileceksiniz: 

- Microsoft PowerPoint Presentation (PPT)
- Microsoft PowerPoint SlideShow (PPS)
- Microsoft PowerPoint 2007 Presentation (PPTX)
- Microsoft PowerPoint 2007 SlideShow (PPSX)

{{% /alert %}} 

Bu formatlarda belgeler oluşturmak için Aspose.Slides for Reporting Services, Aspose'tan pazara lider sunum işleme bileşeni olan [Aspose.Slides for .NET](http://www.aspose.com/Products/Aspose.Slides/) yerleşik bir sürümüne dayanır. Belgeleri oluşturmak için Microsoft PowerPoint kullanılmaz. 


PPT ve PPS kopyaları aynı raporun birbirine tamamen aynı görünecektir, Aspose.Slides tarafından sağlanan temel dosya formatları için yüksek düzeyde destek sayesinde.
