{{% alert color="primary" %}} 

Aspose.Slides pour Reporting Services prend entièrement en charge toutes les spécifications RDL. Cela signifie deux choses importantes : 

- Pas besoin de redessiner les rapports existants. Vous pouvez exporter n'importe quel rapport RDL existant en tant que présentation Microsoft PowerPoint et il apparaîtra exactement selon le design RDL.
- Pas besoin d'utiliser un concepteur de rapport spécifique. Vous pouvez utiliser n'importe quel concepteur de rapport RDL et le rapport sera exporté exactement comme vous l'avez conçu.

{{% /alert %}} 

Aspose.Slides pour Reporting Services prend en charge les éléments RDL suivants : 

- Page, en-têtes, pieds de page
- Zones de texte
- Images
- Sous-rapports
- Graphiques
- Listes
- Tables
- Matrices
- Styles
- Rectangles
