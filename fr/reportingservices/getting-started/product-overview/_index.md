## **Bienvenue sur Aspose.Slides for Reporting Services!**

Aspose.Slides for Reporting Services est la seule solution sur le marché qui permet de générer de véritables rapports PPT et PPS dans Microsoft SQL Server 2005, 2008, 2012, 2016 et 2017 Reporting Services (32 bits et 64 bits). Toutes les fonctionnalités des rapports RDL, y compris les tableaux, matrices, graphiques et images, sont converties avec le plus haut degré de précision en présentations Microsoft PowerPoint.

## **Aperçu du produit**

Microsoft SQL Server Reporting Services ne dispose pas de fonctionnalités intégrées pour exporter les rapports au format Microsoft PowerPoint, mais après avoir installé Aspose.Slides for Reporting Services sur votre serveur, vous avez accès à des formats d’exportation supplémentaires :

- PPT - Présentation PowerPoint via Aspose.Slides
- PPS - Diaporama PowerPoint via Aspose.Slides
- PPTX - Présentation PowerPoint 2007 via Aspose.Slides
- PPSX - Diaporama PowerPoint 2007 via Aspose.Slides

Aspose.Slides for Reporting Services crée des présentations sur le serveur sans utiliser Microsoft PowerPoint. Aspose.Slides for Reporting Services utilise en interne Aspose.Slides for .NET – le composant de classe mondiale pour le traitement de présentations côté serveur.

**Aspose.Slides for Reporting Services permet d’exporter n’importe quel rapport au format PPT, PPS, PPTX ou PPSX.**
