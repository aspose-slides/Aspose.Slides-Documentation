{{% alert color="primary" %}} 

Aspose.Slides for Java est une bibliothèque de classes qui permet aux applications Java de lire et d'écrire des documents de présentation sans utiliser Microsoft PowerPoint® ou d'autres. Aspose.Slides for Java permet de gérer le texte, les formes, les tableaux & les animations, d'ajouter de l'audio et de la vidéo aux diapositives, de prévisualiser les diapositives, d'exporter les diapositives au format SVG, PDF et plus encore.

La référence API la plus récente est disponible à [Aspose.Slides for Java API Reference](https://reference.aspose.com/slides/fr/java).
