Une tâche courante que les développeurs doivent accomplir est d'ajouter du texte aux diapositives de manière dynamique. 1. Créer une présentation.
1. Ajouter une diapositive vierge.
1. Ajouter une zone de texte.
1. Définir du texte.
1. Écrire la présentation.
