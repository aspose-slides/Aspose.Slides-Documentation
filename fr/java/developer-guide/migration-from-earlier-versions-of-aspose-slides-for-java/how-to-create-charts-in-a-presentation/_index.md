Une nouvelle [API Aspose.Slides pour Java](https://docs.aspose.com/slides/java/) a été publiée et maintenant, ce produit unique prend en charge la capacité de générer des documents PowerPoint à partir de zéro et de modifier ceux existants.

{{% /alert %}} 
## **Prise en charge du code hérité**
Pour utiliser le code hérité développé avec les versions d'Aspose.Slides pour Java antérieures à 14.x.x, vous devez apporter quelques modifications mineures à votre code et celui-ci fonctionnera comme avant. Toutes les classes présentes dans l'ancien Aspose.Slides pour Java sous les espaces de noms com.aspose.slides et com.aspose.slides.pptx sont désormais fusionnées dans un unique espace de noms com.aspose.slides. Veuillez jeter un œil à l'extrait de code suivant pour créer un graphique normal à partir de zéro dans une présentation en utilisant l'API Aspose.Slides héritée et suivre les étapes décrivant comment migrer vers la nouvelle API fusionnée.
