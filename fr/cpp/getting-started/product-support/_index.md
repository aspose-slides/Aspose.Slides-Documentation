## **Assistance technique**
Aspose fournit une assistance technique gratuite et illimitée pour tous ses produits. L'assistance est disponible pour tous les utilisateurs, y compris en version d'evaluation.
Si vous avez besoin d'aide avec Aspose.Slides for C++, considérez ce qui suit :

- Le principal canal d'assistance est les forums Aspose. Publiez votre question dans le forum Aspose.Slides et elle recevra une réponse en quelques heures. Les questions sont répondues directement par l'equipe d'assistance Aspose.Slides.
- Veuillez noter qu'Aspose ne fournit pas d'assistance technique par telephone. L'assistance telephone n'est disponible que pour les questions de vente et d'achat.
- Lorsque vous attendez une réponse sur les forums, veuillez tenir compte des differences de fuseau horaire.
  Si vous rencontrez un probleme avec Aspose.Slides for C++, suivez ces etapes simples pour vous assurer qu'il soit resolu de la maniere la plus efficace possible :
  - Assurez-vous d'utiliser la derniere version d'Aspose.Slides for C++ avant de signaler le probleme, consultez le telechargement Aspose.Slides pour connaitre la derniere version.
- Parcourez les forums, cette documentation et la reference API avant de signaler le problème ; il se peut que votre question ait deja ete repondu.
- Lorsque vous signalez un probleme, veuillez inclure le document original et éventuellement un extrait de votre code qui cause le probleme. Si vous devez joindre plusieurs fichiers, compressez-les en un seul zip. Il est sur de joindre vos documents dans Aspose.Forums car seuls vous et les developpeurs Aspose auront acces aux fichiers joints.
- Veuillez essayer de signaler un seul probleme par fil de discussion. Si vous avez un autre probleme, signalez-le dans un fil distinct.

## **FAQ**

**Quels niveaux de support payant sont disponibles et en quoi diffèrent-ils?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Ils diffèrent par le nombre d'incidents inclus, la priorite de reponse et les canaux de communication disponibles.

**Puis-je transmettre directement un bug a l'equipe d'ingénierie?**

Oui. [Support payant](https://helpdesk.aspose.com/) permet d'escalader aux developpeurs lorsqu'un probleme nécessite une enquête approfondie.

**Les versions plus anciennes de la bibliothèque sont-elles prises en charge, et publiez-vous des correctifs pour celles-ci?**

Vous pouvez utiliser des versions antérieures, mais les correctifs sont fournis pour la version actuelle ; les correctifs pour les versions heritees ne sont pas publishes.
