{{% alert color="primary" %}} 

Aspose.Slides for .NET est une bibliothèque de classes qui permet aux applications .NET de lire et écrire des documents de présentation sans utiliser Microsoft PowerPoint® ou d'autres. Aspose.Slides for .NET permet de gérer le texte, les formes, les tableaux & animations, d'ajouter de l'audio et de la vidéo aux diapositives, de prévisualiser les diapositives, d'exporter les diapositives au format SVG, PDF et plus encore.

La dernière référence API se trouve sur [Aspose.Slides for .NET API Reference.](https://reference.aspose.com/slides/net)
