## **Introduction**
Microsoft Visual Studio est un environnement de développement intégré (IDE) de Microsoft. Il est utilisé pour développer des programmes informatiques pour Microsoft Windows, ainsi que des sites web, des applications web et des services web. Visual Studio utilise les plates‑formes de développement Microsoft telles que Windows API, Windows Forms, Windows Presentation Foundation, Windows Store et Microsoft Silverlight. Il peut produire du code natif ainsi que du code géré.
### **Aspose.Slides for .NET**
Aspose.Slides for .NET est une API de gestion PowerPoint® unique qui permet aux applications .NET de lire, écrire et manipuler des documents PowerPoint sans utiliser Microsoft PowerPoint.

Aspose.Slides for .NET est la première et la seule API PowerPoint .NET qui offre des fonctionnalités de gestion des documents PowerPoint au sein de vos propres applications. Comme tous les composants Aspose .NET, Aspose.Slides est écrit en C# géré, proposé à un prix très compétitif et ultra‑rapide.

Pour plus de détails sur [API Aspose.Slides for .NET](http://www.aspose.com/.net/powerpoint-component.aspx)
### **Explorez les exemples Aspose.Slides for .NET avec Visual Studio**
Aspose Visual Studio Plugin – **Explorez les exemples Aspose.Slides for .NET** est un excellent outil pour télécharger rapidement et explorer les exemples de l’API Aspose.Slides for .NET. Il fait gagner beaucoup de temps et d’efforts en offrant une option très simple pour sélectionner, télécharger et ouvrir les derniers projets d’exemple sans quitter Visual Studio.

Cette version fournit les fonctionnalités suivantes :

- Prise en charge de Visual Studio 2010, Visual Studio 2012 et Visual Studio 2013.
- Lancement facile depuis le menu **File > Aspose** dans Visual Studio
- Économise du temps et réduit la courbe d’apprentissage.
- Vous permet de sélectionner et d’ouvrir les derniers exemples d’API Aspose.Slides for .NET hébergés sur Github.com.
- Vous donne la possibilité d’ouvrir des projets d’exemple **C#** ou **VB.NET**
- Le projet sélectionné s’ouvre automatiquement dans Visual Studio avec toutes les références requises, vous offrant un environnement prêt à l’emploi.
- Télécharge automatiquement les dernières bibliothèques d’API depuis **NuGet** et les exemples depuis **GitHub**.

## **Configuration système et plateformes prises en charge**
Pour installer et utiliser Aspose Visual Studio Plugin, vous devez disposer de l’une des versions suivantes de Visual Studio :

- Visual Studio 2010, 2012, 2013

N’hésitez pas à nous contacter si vous rencontrez des problèmes lors de l’installation ou de l’utilisation de ce plug‑in.

## **Installation**
L’installation d’Aspose Visual Studio Plugin est très simple, suivez ces étapes :

- Double‑cliquez sur le fichier msi téléchargé
- Suivez les étapes, vous pouvez simplement appuyer sur **Next** à chaque écran pour installer avec les paramètres standard.

Un dialogue de succès apparaît une fois l’installation terminée.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_2)

**Note :** Veuillez redémarrer Visual Studio pour que les modifications prennent effet.
## **Utilisation**
Une fois installé, vous pouvez lancer et utiliser facilement le plug‑in interactif comme décrit ci‑dessous :
**1.** Ouvrez le plug‑in en sélectionnant **File > Aspose** puis **New Aspose.Slides Example Project** dans Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_3)

**2.** La première fois que vous ouvrez le plug‑in, vous voyez la boîte de dialogue de sélection de l’API comme indiqué ci‑dessous. Le téléchargement des dernières API et de leurs bibliothèques depuis **NuGet** ainsi que du code source des exemples depuis **GitHub** prendra quelques minutes.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_4)

**3.** Dans la boîte de dialogue **Select an API to View Examples**, choisissez ou modifiez l’emplacement où les projets d’exemple seront créés et choisissez ou modifiez le type de projet d’exemple en **C#** ou **VB.NET**, puis cliquez sur **Continue** pour ouvrir le projet d’exemple dans Visual Studio.

![todo:image_alt_text](aspose-slides-visual-studio-plugin_1)

**4.** Nouveaux projets d’exemples selon le type de projet sélectionné (**C#** ou **VB.NET**) dans Visual Studio.
