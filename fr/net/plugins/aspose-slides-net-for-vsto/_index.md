## **Qu’est‑ce que VSTO ?**
VSTO, ou Visual Studio Tools for Office, ou Microsoft Visual Studio Tools fait partie de la suite d’outils Visual Studio .NET de Microsoft et prend en charge le Microsoft .NET Framework. Il est utilisé par les développeurs pour écrire du code qui se connecte aux applications Microsoft Office Word, Excel et Outlook. Les développeurs utilisant VSTO peuvent employer les langages C# ou Visual Basic. C’est une alternative à Visual Basic for Applications, ou VBA.
## **Aspose.Slides for .NET**
Aspose.Slides for .NET est un composant unique de gestion PowerPoint® qui permet aux applications .NET de lire, écrire et manipuler des documents PowerPoint sans utiliser Microsoft PowerPoint.

Aspose.Slides for .NET est le premier et le seul composant .NET qui fournit des fonctionnalités de gestion des documents PowerPoint au sein de vos propres applications. Comme tous les composants Aspose .NET, Aspose.Slides est écrit en C# géré, proposé à un prix très attractif et d’une rapidité fulgurante.
## **Aspose.Slides for VSTO**
## **Introduction**
Les développeurs à la recherche de bibliothèques de formats de fichiers peuvent devoir examiner les fonctionnalités offertes par les bibliothèques disponibles sur le marché. Principalement, deux facteurs sont pris en compte lors du choix d’une option :

- Fonctionnalités
- Efforts nécessaires pour utiliser la bibliothèque

Si vous êtes un développeur .NET à la recherche d’une bibliothèque de formats de fichiers plus rapide, simple et légère pour traiter des fichiers, vous pouvez comparer VSTO (Visual Studio Tools for Office) avec Aspose. Dans ce cas, ce projet vous aidera à comparer les fonctionnalités et le code des deux bibliothèques.
## **Exigences système et plates‑formes prises en charge**
### **Exigences système**
Une ou plusieurs des éditions suivantes de Visual Studio Tools for Office :

- Microsoft Visual Studio 2005 Tools for the Microsoft Office System
- Microsoft Visual Studio 2005 Tools for the 2007 Microsoft Office System
- Visual Studio 2008 Professional Edition
- Visual Studio 2008 Team Suite Edition
- Une version de Microsoft Office :
- Microsoft Office Professional 2003 SP1
- Système Microsoft Office 2007
### **Systèmes d’exploitation pris en charge**
Windows 2000 ; Windows Server 2003 ; Windows Vista ; Windows XP
### **Versions du .NET Framework prises en charge**
seulement le .NET Framework 2.0 ou supérieur.
