## **Assistance technique**
{{% alert color="primary" %}}

Aspose fournit une assistance technique gratuite et illimitée pour tous ses produits. Le support est disponible pour tous les utilisateurs (y compris les utilisateurs avec des versions d'évaluation).

{{% /alert %}} 

Lorsque vous avez besoin d'aide avec Aspose.Slides for .NET, considérez les éléments suivants :

- Le principal canal d'assistance est le [Aspose.Forums](https://forum.aspose.com/c/slides/11). Publiez votre question dans le forum Aspose.Slides — elle recevra une réponse en quelques heures. L'équipe de support d'Aspose.Slides répond directement aux questions publiées sur le forum.
- Veuillez noter qu'Aspose ne fournit pas d'assistance technique par téléphone. Le support téléphonique n'est disponible que pour les questions de vente et d'achat.
- Lorsque vous attendez une réponse sur les forums, soyez patient et tenez compte des différences de fuseau horaire.

Lorsque vous rencontrez un problème avec Aspose.Slides for .NET, suivez ces directives pour le résoudre de la manière la plus efficace :

- Vérifiez et confirmez que vous utilisez la dernière version d'Aspose.Slides for .NET avant de signaler le problème. Peut-être que le problème que vous avez rencontré a déjà été résolu dans une mise à jour.  

  Voir le téléchargement d'Aspose.Slides pour obtenir des informations sur la dernière version.

- Avant de signaler un problème, vous pouvez consulter nos forums, cette documentation et la Référence API. Peut-être que la réponse à votre question a déjà été fournie. 

- Lors du signalement d'un problème, veuillez inclure le document original et (si possible) un extrait de votre code lié au problème. Si vous devez joindre plusieurs fichiers, placez-les dans une archive ZIP.  

  Vous pouvez joindre en toute sécurité vos documents dans Aspose.Forums car **seuls** vous et les développeurs d'Aspose avez accès aux fichiers joints.

- Créez et utilisez un seul fil de discussion pour un problème spécifique. Si vous rencontrez un autre problème, vous devez créer un autre fil de discussion pour le signaler.

## **FAQ**

**Quels niveaux de support payant sont disponibles et comment diffèrent-ils ?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Ils diffèrent par le nombre d'incidents inclus, la priorité de réponse et les canaux de communication disponibles.

**Puis-je escalader un bug directement à l'équipe d'ingénierie ?**

Oui. Le [support payant](https://helpdesk.aspose.com/) permet d'escalader aux développeurs lorsqu'un problème nécessite une investigation plus approfondie.

**Les versions plus anciennes de la bibliothèque sont-elles prises en charge, et publiez‑vous des correctifs pour elles ?**

Vous pouvez utiliser les versions antérieures, mais les correctifs sont fournis pour la version actuelle ; les correctifs pour les versions legacy ne sont pas publiés.
