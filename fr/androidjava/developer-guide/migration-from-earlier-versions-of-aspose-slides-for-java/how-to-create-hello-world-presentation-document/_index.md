Une nouvelle [Aspose.Slides pour Java API](/slides/fr/androidjava/) a été publiée et maintenant ce produit unique prend en charge la capacité de générer des documents PowerPoint à partir de zéro et de modifier les documents existants.

{{% /alert %}} 
## **Support pour le code ancien**
Afin d'utiliser le code ancien développé avec Aspose.Slides pour Java versions antérieures à 13.x, vous devez apporter quelques modifications mineures dans votre code et celui-ci fonctionnera comme avant. Toutes les classes présentes dans l'ancien Aspose.Slides pour Java sous les espaces de noms Aspose.Slide et Aspose.Slides.Pptx sont désormais fusionnées dans un espace de noms unique Aspose.Slides. Veuillez consulter le code exemple suivant pour créer un document de présentation Hello World dans l'API Aspose.Slides ancienne et suivre les étapes décrivant comment migrer vers la nouvelle API fusionnée.
