## **Assistance technique**

{{% alert color="primary" %}} 

Aspose fournit une assistance technique illimitée gratuite pour tous ses produits. L'assistance est disponible pour tous les utilisateurs (y compris les utilisateurs avec des packages d'évaluation).

{{% /alert %}} 

Lorsque vous avez besoin d'aide avec Aspose.Slides for Android via Java, considérez ce qui suit :

- Le principal canal d'assistance est le Aspose.Forums. Publiez votre question dans le forum Aspose.Slides — elle recevra une réponse en quelques heures. L'équipe d'assistance Aspose.Slides répond directement aux questions postées sur le forum.
- Veuillez noter qu'Aspose ne fournit pas d'assistance technique par téléphone. L'assistance téléphonique est uniquement disponible pour les questions de vente et d'achat.
- Lorsque vous attendez une réponse sur les forums, soyez patient et tenez compte des différences de fuseau horaire.


Lorsque vous rencontrez un problème avec Aspose.Slides for Android via Java, suivez ces directives pour le résoudre de la manière la plus efficace :

- Vérifiez et confirmez que vous utilisez la dernière version d'Aspose.Slides for Android via Java avant de signaler le problème. Il se peut que le problème que vous avez rencontré ait déjà été résolu dans une mise à jour.  

  Consultez le téléchargement Aspose.Slides pour obtenir des informations sur la dernière version.

- Avant de signaler un problème, vous pouvez parcourir nos forums, cette documentation et la Référence API. Il se peut que la réponse à votre question ait déjà été fournie.  

- Lors du signalement d'un problème, veuillez inclure le document original et (si possible) un fragment de votre code lié au problème. Si vous devez joindre plusieurs fichiers, placez‑les dans une archive ZIP.  

  Vous pouvez joindre vos documents en toute sécurité sur Aspose.Forums car seuls vous et les développeurs d'Aspose avez accès aux fichiers joints.

- Créez et utilisez un fil de discussion unique pour un problème spécifique. Si vous rencontrez un autre problème, vous devez créer un nouveau fil pour le signaler.  

## **FAQ**

**Quels niveaux d'assistance payante sont disponibles et comment diffèrent‑ils ?**

[Développeur, Business et Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Ils diffèrent par le nombre d'incidents inclus, la priorité de réponse et les canaux de communication disponibles.

**Puis‑je escalader un bug directement à l'équipe d'ingénierie ?**

Oui. [Support payant](https://helpdesk.aspose.com/) permet d'escalader aux développeurs lorsqu'un problème nécessite une enquête approfondie.

**Les versions plus anciennes de la bibliothèque sont‑elles prises en charge, et publiez‑vous des correctifs pour elles ?**

Vous pouvez utiliser les versions plus anciennes, mais les correctifs sont fournis pour la version actuelle ; les correctifs pour les versions héritées ne sont pas publiés.
