## **À propos de TIFF**

Le [Format de fichier image balisé](https://en.wikipedia.org/wiki/TIFF) qui est connu pour stocker plusieurs images dans un même document a été créé à l'origine par Aldus. Ce format est largement pris en charge par les applications de numérisation, de fax et d'autres manipulations d'images.

## **TIFF dans Aspose.Slides for PHP via Java**

Tout document qui peut être chargé dans Aspose.Slide for Java peut également être converti directement en document TIFF par Aspose.Slides for PHP via Java, éliminant ainsi le besoin de tout composant tiers. De plus, vous pouvez éventuellement définir la taille des images dans le document TIFF résultant. Vous trouverez des informations sur l'exportation des documents de présentation vers des documents TIFF via Aspose.Slides for PHP via Java dans [cette rubrique](/slides/fr/php-java/convert-powerpoint-to-tiff/).
