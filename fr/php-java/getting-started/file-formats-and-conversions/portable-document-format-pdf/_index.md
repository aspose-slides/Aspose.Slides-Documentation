{{% alert color="primary" %}} 

Le [Portable Document Format](https://en.wikipedia.org/wiki/PDF) est un format de fichier créé par Adobe Systems pour échanger des documents entre organisations. Le but du format était de conserver le contenu et la mise en page identiques, quel que soit la plateforme sur laquelle il était visualisé. Aspose.Slides for PHP via Java vous permet de convertir des fichiers de présentation en PDF.

{{% /alert %}} 

## **PDF dans Aspose.Slides for PHP via Java**
Toute présentation pouvant être chargée dans Aspose.Slides for PHP via Java peut être convertie en PDF conforme à [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) ou [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA), selon votre choix. Aspose.Slides for PHP via Java exporte les présentations au format PDF et, dans la plupart des cas, le PDF généré ressemble exactement à la présentation originale.

Aspose.Slides prend en charge les fonctionnalités de présentation suivantes lors de la conversion en PDF :

- Images, zones de texte et autres formes.
- Texte et mise en forme.
- Paragraphes et mise en forme.
- Hyperliens.
- En‑têtes et pieds de page.
- Puces.
- Tableaux.

Vous pouvez exporter des présentations en PDF directement avec Aspose.Slides for PHP via Java : vous n'avez besoin d'aucun autre composant. De plus, vous pouvez personnaliser l'exportation de la présentation en PDF avec diverses options comme expliqué dans [Conversion en PDF](/slides/fr/php-java/convert-powerpoint-to-pdf/).
