Cet article décrit un bug de licence rencontré dans Aspose.Slides for PHP via Java versions 23.2, 23.3, 23.4, 23.5, 23.6 et 23.7. Ce bug rendait les licences perpétuelles temporairement inutilisables après l'expiration de leur abonnement.

## **Symptômes**

Après l'expiration de l'abonnement de votre licence perpétuelle, les versions 23.2 - 23.7 d'Aspose.Slides for PHP via Java peuvent signaler que la licence est expirée, passant toutes les fonctions en mode d'evaluation. Ce comportement est incorrect et n'affecte pas les versions antérieures à 23.2 ni postérieures à 23.7.

## **Solution**

Aspose.Slides for PHP via Java a résolu ce problème et a publié des versions mises à jour (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) contenant le correctif.

Si vous utilisez l'une des versions concernées d'Aspose.Slides for PHP via Java dans votre projet, veuillez passer à la version corrigée.

Si vous rencontrez des problèmes de licence avec Aspose.Slides for PHP via Java :

- Vérifiez la version que vous utilisez et assurez-vous qu'elle n'est pas affectée par le bug décrit ci-dessus.
- Consultez la [documentation Aspose](https://docs.aspose.com/slides/php-java/getting-started/) pour les étapes de dépannage et les problèmes de licence connus.
- Contactez le [support Aspose](https://forum.aspose.com/) pour une assistance supplémentaire.

## **Produits et versions affectés**

Veuillez noter que ce bug n'affecte que les versions 23.2 à 23.7 d'Aspose.Slides for PHP via Java. **Les autres produits Aspose et les autres versions d'Aspose.Slides ne sont pas concernés**.
