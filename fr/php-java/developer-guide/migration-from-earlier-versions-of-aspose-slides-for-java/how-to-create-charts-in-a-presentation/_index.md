## **Support pour le code hérité**
Pour utiliser le code hérité développé avec Aspose.Slides pour PHP via Java versions antérieures à 14.x.x, vous devez apporter quelques modifications mineures à votre code et le code fonctionnera comme auparavant. Toutes les classes qui étaient présentes dans l'ancienne Aspose.Slides pour PHP via Java sous les espaces de noms com.aspose.slides et com.aspose.slides.pptx sont maintenant fusionnées dans un espace de noms unique com.aspose.slides.
