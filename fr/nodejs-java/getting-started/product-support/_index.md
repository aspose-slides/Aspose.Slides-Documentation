## **Support technique**
{{% alert color="primary" %}}

Aspose fournit un support technique gratuit et illimité pour tous ses produits. Le support est disponible pour tous les utilisateurs (y compris les utilisateurs avec des packages d'évaluation).

{{% /alert %}}

Lorsque vous avez besoin d’aide avec Aspose.Slides for Node.js via Java, considérez les points suivants :

- Le principal moyen d’obtenir du support est le [Aspose.Forums](https://forum.aspose.com/c/slides/11). Publiez votre question dans le forum Aspose.Slides — elle sera répondue en quelques heures. L’équipe de support Aspose.Slides répond directement aux questions postées sur le forum.  
- Veuillez noter qu’Aspose ne fournit pas de support technique par téléphone. L’assistance téléphonique n’est disponible que pour les questions de vente et d’achat.  
- Lorsque vous attendez une réponse sur les forums, soyez patient et tenez compte des différences de fuseaux horaires.

Lorsque vous rencontrez un problème avec Aspose.Slides for Node.js via Java, suivez ces directives pour le résoudre de la manière la plus efficace :

- Vérifiez et confirmez que vous utilisez la dernière version d’Aspose.Slides for Node.js via Java avant de signaler le problème. Il est possible que le problème ait déjà été résolu dans une mise à jour.  

  Consultez le [téléchargement d’Aspose.Slides](https://releases.aspose.com/slides/nodejs-java/) pour obtenir les informations sur la version la plus récente.

- Avant de signaler un problème, parcourez nos forums, cette documentation et la référence API. Il se peut que la réponse à votre question y figure déjà.  

- Lors du signalement d’un problème, veuillez inclure le document original et (si possible) un fragment de votre code lié au problème. Si vous devez joindre plusieurs fichiers, regroupez‑les dans une archive ZIP. Vous pouvez joindre vos documents en toute sécurité sur Aspose.Forums car **seuls** vous et les développeurs d’Aspose ont accès aux fichiers joints.  

- Créez et utilisez un seul fil de discussion pour un problème précis. Si vous rencontrez un autre problème, créez un autre fil de discussion pour le signaler.

## **FAQ**

**Quels niveaux de support payant sont disponibles et en quoi diffèrent‑ils ?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). Ils diffèrent par le nombre d’incidents inclus, la priorité de réponse et les canaux de communication disponibles.

**Puis‑je escalader un bug directement à l’équipe d’ingénierie ?**

Oui. Le [support payant](https://helpdesk.aspose.com/) permet d’escalader les problèmes aux développeurs lorsqu’une investigation plus approfondie est nécessaire.

**Les versions plus anciennes de la bibliothèque sont‑elles prises en charge, et publiez‑vous des correctifs pour elles ?**

Vous pouvez utiliser les versions antérieures, mais les correctifs ne sont fournis que pour la version actuelle ; les correctifs pour les versions legacy ne sont pas publiés.
