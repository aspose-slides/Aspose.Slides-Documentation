## **Tổng quan**

Bài viết này mô tả lỗi cấp phép gặp phải trong Aspose.Slides for PHP via Java phiên bản 23.2, 23.3, 23.4, 23.5, 23.6 và 23.7. Lỗi này khiến các giấy phép vĩnh viễn trở nên không sử dụng được tạm thời sau khi thời gian đăng ký hết hạn.

## **Triệu chứng**

Sau khi thời gian đăng ký giấy phép vĩnh viễn của bạn hết hạn, Aspose.Slides for PHP via Java phiên bản 23.2 - 23.7 có thể báo cáo giấy phép là đã hết hạn, chuyển tất cả các chức năng sang chế độ đánh giá.
Hành vi này là không chính xác và không ảnh hưởng tới các phiên bản trước 23.2 và sau 23.7.

## **Cách khắc phục**

Aspose.Slides for PHP via Java đã khắc phục vấn đề này và phát hành các phiên bản cập nhật (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) với bản sửa lỗi.

Nếu bạn đang sử dụng một trong các phiên bản bị ảnh hưởng của Aspose.Slides for PHP via Java trong dự án, vui lòng chuyển sang sử dụng phiên bản đã được chỉnh sửa.

Nếu bạn gặp bất kỳ vấn đề cấp phép nào với Aspose.Slides for PHP via Java:

- Kiểm tra phiên bản bạn đang sử dụng và đảm bảo nó không bị ảnh hưởng bởi lỗi đã mô tả ở trên.
- Tham khảo [Aspose documentation](https://docs.aspose.com/slides/vi/php-java/getting-started/) để biết các bước khắc phục và các vấn đề cấp phép đã biết.
- Liên hệ [Aspose support](https://forum.aspose.com/) để được hỗ trợ thêm.

## **Sản phẩm và phiên bản bị ảnh hưởng**

Xin lưu ý rằng lỗi này chỉ ảnh hưởng tới Aspose.Slides for PHP via Java phiên bản 23.2 đến 23.7. **Các sản phẩm Aspose khác và các phiên bản khác của Aspose.Slides không bị ảnh hưởng**.
