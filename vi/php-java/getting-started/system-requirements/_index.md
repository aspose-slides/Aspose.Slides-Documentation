## **Giới thiệu**

Aspose.Slides for PHP via Java không yêu cầu cài đặt bất kỳ sản phẩm bên thứ ba nào như Microsoft PowerPoint. Chính Aspose.Slides là một công cụ để tạo, chỉnh sửa, chuyển đổi và hiển thị tài liệu ở nhiều định dạng, bao gồm các định dạng bản trình bày Microsoft PowerPoint.

## **Hệ điều hành được hỗ trợ**

Aspose.Slides for Java hỗ trợ bất kỳ hệ điều hành 32-bit hoặc 64-bit nào chạy môi trường Java, bao gồm nhưng không giới hạn:

### **Windows**
- Microsoft Windows 2003 Server ( x64, x86)
- Microsoft Windows 2008 Server ( x64, x86)
- Microsoft Windows 2012 Server ( x64, x86)
- Microsoft Windows 2012 R2 Server ( x64, x86)
- Microsoft Windows 2016 Server ( x64, x86)
- Microsoft Windows 2019 Server ( x64, x86)
- Microsoft Windows Vista ( x64, x86)
- Microsoft Windows XP ( x64, x86)
- Microsoft Windows 7 ( x64, x86)
- Microsoft Windows 8, 8.1 ( x64, x86)
- Microsoft Windows 10 ( x64, x86)

### **Linux**
- Linux (Ubuntu, OpenSUSE, CentOS và các nền tảng khác)

### **Mac**
- Mac OS X

## **Câu hỏi thường gặp**

**Tôi có cần cài đặt Microsoft PowerPoint để chuyển đổi và hiển thị không?**

Không, không cần PowerPoint; Aspose.Slides là một công cụ độc lập để [tạo](/slides/vi/php-java/create-presentation/), chỉnh sửa, [chuyển đổi](/slides/vi/php-java/convert-presentation/), và [hiển thị](/slides/vi/php-java/convert-powerpoint-to-png/) các bản trình bày.

**Cần những phông chữ nào để hiển thị đúng?**

Thực tế, các phông chữ được sử dụng trong bản trình bày hoặc các [phông chữ thay thế](/slides/vi/php-java/font-substitution/) phù hợp phải có sẵn. Để đảm bảo hiển thị nhất quán trên Linux/macOS, nên cài đặt các gói phông chữ phổ biến.

**Tại sao phông chữ tùy chỉnh lại hiển thị dưới dạng phông dự phòng hoặc văn bản bị thiếu trên Linux?**

Nếu tệp phông chữ có các mục trong bảng tên không nhất quán hoặc bị hỏng, hệ thống khớp phông chữ của Linux (FreeType/fontconfig) có thể lựa chọn một bản ghi không hợp lệ, dẫn đến việc không thể xác định phông chữ. Sử dụng phiên bản phông chữ với các mục bảng tên đã được sửa hoặc cài đặt một phông thay thế nhất quán sẽ giải quyết vấn đề.
