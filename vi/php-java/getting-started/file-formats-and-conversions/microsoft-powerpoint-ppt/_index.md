{{% alert color="primary" %}} 

[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) là định dạng tệp tài liệu thuyết trình có thể được tạo, đọc, chỉnh sửa và ghi bởi các phiên bản khác nhau của Microsoft PowerPoint. Đây là định dạng nhị phân cho tài liệu thuyết trình do Microsoft phát triển.

{{% /alert %}} 

## **PPT trong Aspose.Slides for PHP via Java**
Aspose.Slides for PHP via Java có thể đọc các tệp PPT được tạo bởi phần mềm liệt kê dưới đây.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Tương tự, các tệp PPT do Aspose.Slides for PHP via Java tạo ra cũng có thể được đọc bởi các phần mềm trên.

## **Hỗ trợ toàn diện cho PPT**
Aspose.Slides for PHP via Java cung cấp hỗ trợ cho hầu hết các tính năng của định dạng tệp tài liệu PPT. Nó không chỉ bao phủ các tính năng cơ bản và nâng cao do các phiên bản Microsoft PowerPoint khác nhau cung cấp cho việc thao tác tài liệu PPT, mà còn có những tính năng mà Microsoft PowerPoint thậm chí không hỗ trợ. Lợi thế chính khi sử dụng thư viện API Aspose.Slides for PHP via Java là dễ dàng xử lý những tính năng này.

Ngoài các tác vụ cơ bản liên quan đến tạo, đọc và ghi tệp tài liệu PPT, còn có một số tính năng được Aspose.Slides for PHP via Java cung cấp:

- Nhập các định dạng tệp Microsoft Office khác dưới dạng [đối tượng OLE vào tài liệu PPT](/slides/vi/php-java/manage-ole/).
- [Xuất tài liệu PPT sang PDF](/slides/vi/php-java/convert-powerpoint-to-pdf/).
- Xuất các slide trong tài liệu PPT sang định dạng SVG.
- Hiển thị các slide sang bất kỳ định dạng hình ảnh nào được Java Framework hỗ trợ.
- Đặt kích thước của các slide trong tài liệu PPT.
- Quản lý hoạt ảnh trên các hình dạng.
- Quản lý trình chiếu.
- [Định dạng văn bản trên các slide](/slides/vi/php-java/text-formatting/).
- Trích xuất văn bản từ tài liệu PPT.
- [Xử lý bảng trên các slide](/slides/vi/php-java/powerpoint-table/).
- Tự động sao chép master bằng [tính năng sao chép](/slides/vi/php-java/clone-slides/).
