## **Các tính năng được hỗ trợ trong chuyển đổi**
Aspose.Slides for PHP via Java cung cấp hỗ trợ một phần cho việc chuyển đổi định dạng tệp PPT sang PPTX. Hỗ trợ chuyển đổi mới được giới thiệu trong Aspose.Slides for PHP via Java, vì vậy nó có một vài hạn chế và hoạt động tốt nhất cho các bản trình chiếu đơn giản. Lợi thế chính mà Aspose.Slides for PHP via Java cung cấp khi chuyển đổi PPT sang PPTX là API rất dễ sử dụng. Để xem các ví dụ mã, đọc về [Chuyển đổi PPT sang PPTX](). Dưới đây, danh sách giải thích các tính năng được hỗ trợ và không được hỗ trợ cho việc chuyển đổi PPT sang PPTX.

## **Các tính năng được hỗ trợ**
The following features are supported for conversion:

- Chuyển đổi cấu trúc của master, bố cục và slide.
- Chuyển đổi biểu đồ.
- Nhóm hình dạng.
- Chuyển đổi autoshapes bao gồm hình chữ nhật và hình ellipse. 
- Hình dạng với hình học tùy chỉnh.
- Kết cấu và kiểu nền ảnh cho autoshapes.
- Chuyển đổi placeholder.
- Chuyển đổi đường thẳng và polyline.
- Định dạng đường viền và nền.
- Kiểu nền gradient.
- Khung OLE, bảng, khung video và âm thanh, v.v.
- Thuộc tính hoạt hình và trình chiếu.
- Chuyển đổi văn bản trong khung văn bản và holder văn bản.
