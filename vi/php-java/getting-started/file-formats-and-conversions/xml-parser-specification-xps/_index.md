{{% alert color="primary" %}} 

XML Parser Specification là một ngôn ngữ mô tả trang và một định dạng tài liệu cố định ban đầu được Microsoft phát triển. Giống như PDF, XPS được thiết kế để duy trì độ chính xác của tài liệu và cung cấp giao diện tài liệu không phụ thuộc vào thiết bị. 

{{% /alert %}} 

## **XPS trong Aspose.Slides cho PHP thông qua Java**
Bất kỳ tài liệu trình chiếu nào có thể được tải bởi Aspose.Slides cho PHP thông qua Java đều có thể chuyển đổi sang định dạng XPS. Aspose.Slides cho PHP thông qua Java sử dụng động cơ bố cục trang và render độ chính xác cao để tạo ra đầu ra ở định dạng tài liệu XPS cố định.
Bạn có thể tìm hiểu cách xuất tài liệu trình chiếu sang tài liệu XPS qua Aspose.Slides cho PHP thông qua Java trong [Chuyển đổi sang XPS](https://docs.aspose.com/slides/vi/php-java/convert-powerpoint-to-xps/).
