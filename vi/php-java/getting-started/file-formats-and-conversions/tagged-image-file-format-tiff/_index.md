## **Về TIFF**

Định dạng [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) được biết đến việc lưu trữ nhiều hình ảnh trong một tài liệu ban đầu được tạo ra bởi Aldus. Định dạng này được hỗ trợ rộng rãi bởi các ứng dụng quét, fax và các ứng dụng xử lý ảnh khác.

## **TIFF trong Aspose.Slides cho PHP qua Java**

Bất kỳ tài liệu nào có thể được tải trong Aspose.Slide cho Java cũng có thể được chuyển đổi trực tiếp sang tài liệu TIFF bởi Aspose.Slides cho PHP qua Java, loại bỏ yêu cầu về bất kỳ thành phần bên thứ ba nào. Thêm vào đó, bạn có thể tùy chọn định kích thước của các hình ảnh trong tài liệu TIFF kết quả. Bạn có thể tìm thông tin về việc xuất tài liệu trình chiếu sang tài liệu TIFF thông qua Aspose.Slides cho PHP qua Java trong [đề mục này](/slides/vi/php-java/convert-powerpoint-to-tiff/).
