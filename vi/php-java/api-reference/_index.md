{{% alert color="primary" %}} 

Aspose.Slides for PHP via Java là một thư viện lớp cho phép các ứng dụng PHP đọc và ghi tài liệu trình chiếu mà không cần sử dụng Microsoft PowerPoint® hoặc các phần mềm khác. Aspose.Slides for PHP via Java cho phép quản lý văn bản, hình dạng, bảng & hoạt hình, thêm âm thanh và video vào các slide, xem trước slide, xuất slide ra định dạng SVG, PDF và hơn nữa.

Có thể tìm thấy Tham chiếu API mới nhất tại [Aspose.Slides for PHP via Java API Reference](https://reference.aspose.com/slides/vi/php-java/).

{{% /alert %}}
