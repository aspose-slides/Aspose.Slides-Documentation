{{% alert color="primary" %}} 

Aspose.Slides for Android via Java là một thư viện lớp cho phép các ứng dụng Java trên Android đọc và ghi tài liệu trình chiếu mà không cần sử dụng Microsoft PowerPoint® hoặc phần mềm khác. Aspose.Slides for Android via Java cho phép bạn quản lý văn bản, hình dạng, bảng và hoạt ảnh; thêm âm thanh và video vào các slide; xem trước slide; và xuất slide sang định dạng SVG, PDF và hơn nữa.

Phiên bản Tham chiếu API mới nhất có thể được tìm thấy tại [Aspose.Slides for Android via Java API Reference](https://reference.aspose.com/slides/vi/androidjava/).

{{% /alert %}}
