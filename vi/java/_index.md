**Chào mừng đến với Aspose.Slides for Java!**

Aspose.Slides for Java là một API quản lý Microsoft PowerPoint® cho phép các ứng dụng Java đọc và ghi tài liệu PowerPoint® mà không cần Microsoft PowerPoint®. Aspose.Slides for Java là thành phần đầu tiên và duy nhất cung cấp chức năng quản lý tài liệu PowerPoint®. Aspose.Slides for Java cung cấp rất nhiều tính năng chính như quản lý văn bản, hình dạng, bảng & hoạt ảnh, thêm âm thanh & video vào slide, xem trước slide, xuất slide sang SVG, PDF và các định dạng khác.
