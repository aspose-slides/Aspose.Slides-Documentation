## **Hỗ trợ mã Legacy**
Để sử dụng mã legacy được phát triển với các phiên bản Aspose.Slides cho Java trước 14.x.x, bạn cần thực hiện một số thay đổi nhỏ trong mã của mình và mã sẽ hoạt động như trước. Tất cả các lớp đã có trong Aspose.Slides cho Java cũ dưới các namespace com.aspose.slides và com.aspose.slides.pptx hiện đã được hợp nhất thành một namespace duy nhất com.aspose.slides. Vui lòng xem đoạn mã mẫu đơn giản sau để tạo một biểu đồ thông thường từ đầu trong bản trình chiếu bằng API Aspose.Slides legacy và làm theo các bước mô tả cách di chuyển sang API hợp nhất mới.
