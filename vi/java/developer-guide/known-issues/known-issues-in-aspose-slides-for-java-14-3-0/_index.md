Aspose.Slides for Java 14.3.0 (14.4.0) cung cấp triển khai hoàn toàn mới cho việc xử lý PPT. Có rất nhiều cải tiến, chuyển đổi một phần từ PPTX sang PPT. Tuy nhiên vẫn còn một số tính năng chưa được triển khai:

- Một số hình dạng có hình học sai trong tài liệu PPT được tuần tự hoá (Call outs)
- Không phải tất cả các tính năng định dạng văn bản PPTX đều được hỗ trợ trong việc tuần tự hoá PPT
- Thông tin về ngôn ngữ văn bản và cài đặt chính tả không có trong tài liệu PPT được tuần tự hoá
- Không phải tất cả các tính năng của theme PPTX đều được hỗ trợ trong việc tuần tự hoá PPT

**Có một số khác biệt so với Aspose.Slides for Java 8.6.0:**

- Có các vấn đề đã biết trong việc tuần tự hoá OLE/ActiveX PPT sang PPT

**Có một số khác biệt so với Aspose.Slides for .NET 14.3.0:**

- Hỗ trợ in bản trình chiếu hiện tại chưa khả dụng trong Aspose.Slides for Java
