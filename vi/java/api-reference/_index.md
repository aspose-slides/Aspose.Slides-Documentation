{{% alert color="primary" %}} 

Aspose.Slides for Java là một thư viện lớp cho phép các ứng dụng Java đọc và ghi tài liệu trình chiếu mà không cần sử dụng Microsoft PowerPoint® hoặc các phần mềm khác. Aspose.Slides for Java cho phép quản lý văn bản, hình dạng, bảng & hoạt ảnh, thêm âm thanh và video vào slide, xem trước slide, xuất slide sang định dạng SVG, PDF và hơn nữa.

Tài liệu tham chiếu API mới nhất có thể được tìm thấy tại [Aspose.Slides for Java API Reference](https://reference.aspose.com/slides/vi/java).

{{% /alert %}}
