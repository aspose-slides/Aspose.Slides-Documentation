## **Tổng quan**
Aspose.Slides for Java không cần cài đặt Microsoft PowerPoint, vì Aspose.Slides tự nó là một công cụ tạo, chuyển đổi, bố trí trang và hiển thị tài liệu Microsoft PowerPoint.

## **Hệ điều hành được hỗ trợ**
Aspose.Slides for Java hỗ trợ bất kỳ hệ điều hành 32-bit hoặc 64-bit nào chạy môi trường Java, bao gồm, nhưng không giới hạn:

### **Windows**
- Microsoft Windows 2003 Server ( x64, x86)
- Microsoft Windows 2008 Server ( x64, x86)
- Microsoft Windows 2012 Server ( x64, x86)
- Microsoft Windows 2012 R2 Server ( x64, x86)
- Microsoft Windows 2016 Server ( x64, x86)
- Microsoft Windows 2019 Server ( x64, x86)
- Microsoft Windows Vista ( x64, x86)
- Microsoft Windows XP ( x64, x86)
- Microsoft Windows 7 ( x64, x86)
- Microsoft Windows 8, 8.1 ( x64, x86)
- Microsoft Windows 10 ( x64, x86)

### **Linux**
- Linux (Ubuntu, OpenSUSE, CentOS và các bản khác)

### **Mac**
- Mac OS X

## **Phiên bản Java được hỗ trợ**
Aspose.Slides for Java hỗ trợ J2SE 6.0 (Java 1.6) và các phiên bản cao hơn.

## **Câu hỏi thường gặp**

**Tôi có cần cài đặt Microsoft PowerPoint để chuyển đổi và hiển thị không?**

Không, PowerPoint không bắt buộc; Aspose.Slides là một engine độc lập cho việc [creating](/slides/vi/java/create-presentation/), modifying, [converting](/slides/vi/java/convert-presentation/), và [rendering](/slides/vi/java/convert-powerpoint-to-png/) presentations.

**Cần những phông chữ nào để hiển thị đúng?**

Trong thực tế, các phông chữ được sử dụng trong bản trình chiếu hoặc các [substitutes](/slides/vi/java/font-substitution/) phải có sẵn. Để đảm bảo hiển thị đồng nhất trên Linux/macOS, nên cài đặt các gói phông chữ thông dụng.

**Tại sao một phông chữ tùy chỉnh lại hiển thị dưới dạng dự phòng hoặc văn bản bị thiếu trên Linux?**

Nếu tệp phông chữ có các mục trong bảng tên không nhất quán hoặc bị hỏng, stack khớp phông chữ của Linux (FreeType/fontconfig) có thể chọn một bản ghi không hợp lệ, gây ra việc phông chữ không được giải quyết. Sử dụng phiên bản phông chữ có bảng tên đã được sửa chữa hoặc cài đặt một bản thay thế đồng nhất sẽ giải quyết vấn đề.
