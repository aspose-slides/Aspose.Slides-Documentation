{{% alert color="primary" %}} 

Bản đặc tả Trình phân tích XML là một ngôn ngữ mô tả trang và định dạng tài liệu cố định được Microsoft phát triển ban đầu. Giống như PDF, XPS được thiết kế để giữ nguyên độ trung thực của tài liệu và cung cấp giao diện tài liệu độc lập với thiết bị. 

{{% /alert %}} 

## **XPS trong Aspose.Slides cho Java**
Bất kỳ tài liệu trình chiếu nào có thể được Aspose.Slides cho Java tải lên đều có thể được chuyển đổi sang định dạng XPS. Aspose.Slides cho Java sử dụng công cụ bố trí trang và kết xuất độ trung thực cao để tạo ra đầu ra ở định dạng tài liệu XPS cố định.
Bạn có thể tìm hiểu về xuất các tài liệu trình chiếu sang tài liệu XPS thông qua Aspose.Slides cho Java trong [Chuyển đổi sang XPS](https://docs.aspose.com/slides/vi/java/convert-powerpoint-to-xps/).
