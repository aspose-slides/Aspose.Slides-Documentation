{{% alert color="primary" %}} 

[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) là định dạng tệp tài liệu trình chiếu có thể được tạo, đọc, thao tác và ghi bởi các phiên bản khác nhau của Microsoft PowerPoint. Đây là định dạng nhị phân cho tài liệu trình chiếu do Microsoft phát triển.

{{% /alert %}} 

## **PPT trong Aspose.Slides for Java**
Aspose.Slides for Java có thể đọc các tệp PPT được tạo bởi phần mềm liệt kê dưới đây.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

Tương tự, các tệp PPT được tạo bởi Aspose.Slides for Java cũng có thể được đọc bởi tập hợp phần mềm trên.

## **Hỗ trợ toàn diện cho PPT**
Aspose.Slides for Java cung cấp hỗ trợ cho hầu hết các tính năng của định dạng tệp tài liệu PPT. Nó không chỉ bao phủ các tính năng cơ bản và nâng cao do các phiên bản Microsoft PowerPoint khác nhau cung cấp cho việc xử lý tài liệu PPT, mà còn bao gồm các tính năng mà ngay cả Microsoft PowerPoint cũng không hỗ trợ. Ưu điểm chính của việc sử dụng thư viện API Aspose.Slides for Java là dễ dàng xử lý các tính năng này.

Ngoài các nhiệm vụ cơ bản liên quan đến tạo, đọc và ghi tệp tài liệu PPT, còn có một số tính năng được Aspose.Slides for Java cung cấp:

- Nhập các định dạng tệp Microsoft Office khác dưới dạng [đối tượng OLE vào tài liệu PPT]().
- [Xuất tài liệu PPT sang PDF](/slides/vi/java/convert-powerpoint-ppt-and-pptx-to-pdf/).
- Xuất các slide trong tài liệu PPT sang định dạng SVG.
- Kết xuất slide sang bất kỳ định dạng hình ảnh nào được Java Framework hỗ trợ.
- Đặt kích thước của các slide trong tài liệu PPT.
- Quản lý hoạt ảnh trên các hình dạng.
- Quản lý trình chiếu.
- [Định dạng văn bản trên slide]().
- Trích xuất văn bản từ tài liệu PPT.
- [Xử lý bảng trên slide]().
- Tự động sao chép master bằng [tính năng sao chép]().
