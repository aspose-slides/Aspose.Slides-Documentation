## **Giới thiệu về TIFF**

[Định dạng Tệp Hình ảnh Được Gắn Thẻ](https://en.wikipedia.org/wiki/TIFF) được biết đến với khả năng lưu trữ nhiều hình ảnh trong một tài liệu và ban đầu được tạo ra bởi Aldus. Định dạng này được hỗ trợ rộng rãi bởi các ứng dụng quét, fax và các ứng dụng xử lý hình ảnh khác.

## **TIFF trong Aspose.Slides for Java**

Bất kỳ tài liệu nào có thể được tải trong Aspose.Slides for Java cũng có thể được chuyển đổi trực tiếp sang tài liệu TIFF bằng Aspose.Slides for Java mà không cần thành phần của bên thứ ba. Ngoài ra, bạn có thể tùy chọn xác định kích thước của các hình ảnh trong tài liệu TIFF kết quả. Bạn có thể tìm thông tin về việc xuất các tài liệu trình chiếu sang tài liệu TIFF thông qua Aspose.Slides for Java trong [đề mục này](/slides/vi/java/converting-a-presentation/).
