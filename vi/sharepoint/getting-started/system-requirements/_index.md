## **Yêu cầu máy chủ**
{{% alert color="primary" %}} 

Để cài đặt Aspose.Slides for SharePoint, cần một hệ thống chạy một trong các phiên bản sau:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft Office SharePoint Server 2010
- Microsoft Office SharePoint Server 2013
- Microsoft Office SharePoint Server 2016
- Microsoft Office SharePoint Server 2019

{{% /alert %}} {{% alert color="primary" %}} 

Bạn nên cài đặt Service Pack 2 hoặc phiên bản mới hơn cho các sản phẩm này.

{{% /alert %}} 
## **Yêu cầu máy khách**
{{% alert color="primary" %}} 

Các trang SharePoint được truy cập thông qua trình duyệt web. Aspose.Slides for SharePoint đã được kiểm tra với các trình duyệt sau:

- Microsoft Internet Explorer 6, 7, 8
- Mozilla FireFox 3.5

{{% /alert %}}
