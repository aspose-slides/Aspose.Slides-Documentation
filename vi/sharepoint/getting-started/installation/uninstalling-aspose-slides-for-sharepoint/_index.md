Để gỡ cài đặt Aspose.Slides for SharePoint: 

1. Chạy chương trình cài đặt.  
   Nếu Aspose.Slides for SharePoint đã được cài đặt, chương trình cài đặt sẽ đề xuất gỡ bỏ hoặc sửa chữa.  
1. Chọn **Remove** để gỡ cài đặt Aspose.Slides for SharePoint.
