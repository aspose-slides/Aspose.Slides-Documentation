## **VSTO là gì?**
VSTO, hay Visual Studio Tools for Office, hay Microsoft Visual Studio Tools, là một phần của bộ công cụ Visual Studio .NET của Microsoft và hỗ trợ Microsoft .NET Framework. Nó được các nhà phát triển sử dụng để viết mã kết nối các ứng dụng Microsoft Office Word, Excel và Outlook. Các nhà phát triển sử dụng VSTO có thể dùng ngôn ngữ C# hoặc Visual Basic. Đây là một lựa chọn thay thế cho Visual Basic for Applications, hay VBA.
## **Aspose.Slides for .NET**
Aspose.Slides for .NET là một thành phần quản lý PowerPoint® duy nhất cho phép các ứng dụng .NET đọc, ghi và thao tác tài liệu PowerPoint mà không cần sử dụng Microsoft PowerPoint.

Aspose.Slides for .NET là thành phần .NET đầu tiên và duy nhất cung cấp chức năng quản lý tài liệu PowerPoint trong các ứng dụng của bạn. Như tất cả các thành phần Aspose .NET, Aspose.Slides được viết bằng C# quản lý, giá cả vô cùng hợp lý và tốc độ nhanh như chớp.

## **Giới thiệu**
Những nhà phát triển đang tìm kiếm thư viện định dạng tệp có thể cần xem xét các tính năng do các thư viện trên thị trường cung cấp. Chủ yếu, hai yếu tố được cân nhắc khi chọn một tùy chọn:

- Tính năng
- Nỗ lực cần thiết để sử dụng thư viện

Bạn là một nhà phát triển .NET đang tìm kiếm một thư viện định dạng tệp nhanh hơn, dễ sử dụng và nhẹ để xử lý các tệp, bạn có thể cần so sánh VSTO (Visual Studio tool for Office) với Aspose. Trong trường hợp này, dự án này sẽ giúp bạn so sánh các tính năng và mã trong cả hai thư viện.
## **Yêu cầu hệ thống và Nền tảng được hỗ trợ**
### **Yêu cầu hệ thống**
Ít nhất một trong các phiên bản sau của Visual Studio Tools for Office:

- Microsoft Visual Studio 2005 Tools for the Microsoft Office System
- Microsoft Visual Studio 2005 Tools for the 2007 Microsoft Office System
- Visual Studio 2008 Professional Edition
- Visual Studio 2008 Team Suite Edition
- Một phiên bản của Microsoft Office:
- Microsoft Office Professional 2003 SP1
- 2007 Microsoft Office system
### **Hệ điều hành được hỗ trợ**
Windows 2000; Windows Server 2003; Windows Vista; Windows XP
### **Các phiên bản .NET Framework được hỗ trợ**
chỉ .NET Framework 2.0 trở lên.

## **Hỗ trợ, Mở rộng và Đóng góp**
### **Hỗ trợ**
Chúng tôi cung cấp hỗ trợ miễn phí. Bất kỳ ai sử dụng sản phẩm của chúng tôi, dù đã mua hay đang dùng bản đánh giá, đều xứng đáng nhận được sự quan tâm và tôn trọng đầy đủ từ chúng tôi.

### **Mở rộng và Đóng góp**
Mã so sánh cho Aspose.Slides for .NET và mã nguồn VSTO có sẵn trên các trang web mã nguồn xã hội lớn được liệt kê dưới đây. Các nhà phát triển được khuyến khích tải mã nguồn để di chuyển từ VSTO sang Aspose.Slides theo yêu cầu của họ.

Bạn có thể lấy mã nguồn mới nhất từ một trong các địa điểm sau.

- [GitHub](https://goo.gl/JA8x5M)

- [Bitbucket](https://goo.gl/XBqAzx)
### **Cách cấu hình mã nguồn**
Vui lòng làm theo các bước đơn giản sau để bắt đầu

- Tải xuống/Clone mã nguồn.
- Mở Visual Studio 2010 và chọn File > Open Project.
- Duyệt tới mã nguồn mới nhất mà bạn đã tải và mở tệp .sln file.
