{{% alert color="primary" %}} 

Aspose.Slides for .NET là một thư viện lớp cho phép các ứng dụng .NET đọc và viết tài liệu trình chiếu mà không cần sử dụng Microsoft PowerPoint® hoặc các phần mềm khác. Aspose.Slides for .NET cho phép quản lý văn bản, hình dạng, bảng & hoạt hình, thêm audio và video vào các slide, xem trước slide, xuất slide sang định dạng SVG, PDF và hơn nữa.

Bạn có thể tìm thấy Tham chiếu API mới nhất tại [Aspose.Slides for .NET API Reference](https://reference.aspose.com/slides/vi/net).

{{% /alert %}}
