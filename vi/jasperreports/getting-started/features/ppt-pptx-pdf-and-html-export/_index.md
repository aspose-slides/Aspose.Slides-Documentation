{{% alert color="primary" %}} 

JasperReports và JasperServer không có khả năng tích hợp để xuất báo cáo dưới dạng bản trình bày Microsoft PowerPoint, nhưng với Aspose.Slides cho JasperReports, bạn sẽ được truy cập vào các định dạng xuất bổ sung:

- Microsoft PowerPoint Presentation (PPT)
- Microsoft PowerPoint Presentation (PPTX)
- HTML
- PDF

{{% /alert %}} 

Để tạo tài liệu ở các định dạng này, Aspose.Slides cho JasperReports dựa vào một phiên bản tích hợp của [Aspose.Slides for Java](https://products.aspose.com/slides/vi/java/), thư viện xử lý bản trình bày hàng đầu trên thị trường từ Aspose. Microsoft PowerPoint không được sử dụng để tạo ra các tài liệu.
