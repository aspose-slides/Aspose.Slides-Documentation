Aspose.Slides for JasperReports hoạt động với:

**Aspose.Slides for JasperReports 17.5 và các phiên bản trước:**

- Java 2 Platform Standard Edition 1.4 hoặc cao hơn.
- JasperReports 2.0, 3.0 hoặc cao hơn.

**Aspose.Slides for JasperReports 17.6 và các phiên bản mới hơn:**

- Java 2 Platform Standard Edition 6.0 hoặc cao hơn.
- JasperReports 3.7.2 hoặc cao hơn.
