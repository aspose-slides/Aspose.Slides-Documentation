{{% alert color="primary" %}} 

Aspose.Slides for Node.js via Java là một thư viện lớp cho phép các ứng dụng Node.js đọc và ghi tài liệu trình chiếu mà không cần sử dụng Microsoft PowerPoint® hoặc các phần mềm khác. Aspose.Slides for Node.js via Java cho phép bạn quản lý văn bản, hình dạng, bảng & hoạt ảnh, thêm âm thanh và video vào các slide, xem trước slide, xuất slide sang định dạng SVG, PDF và còn rất nhiều tính năng khác.

Bạn có thể tìm thấy Tham chiếu API mới nhất tại [Tham chiếu API Aspose.Slides cho Node.js thông qua Java](https://reference.aspose.com/slides/vi/nodejs-java/).
