## **Nền tảng được hỗ trợ**
Aspose.Slides cho Node.js qua Java là một API độc lập nền tảng. Nó có thể được sử dụng trên bất kỳ nền tảng nào (Windows, Linux, MacOS, v.v.) có Node.js 7 trở lên được cài đặt. Oracle JDK 8 hoặc cao hơn là yêu cầu khác.

## **Định dạng tệp và Chuyển đổi**
Aspose.Slides cho Node.js qua Java hỗ trợ hầu hết các định dạng tài liệu PowerPoint. Nó cũng cho phép bạn xuất chúng ra các định dạng tệp phổ biến mà các tổ chức sử dụng và chia sẻ với nhau. Xem chi tiết bên dưới:

|**Tính năng**|**Mô tả**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides cho Node.js qua Java cung cấp tốc độ xử lý nhanh nhất cho định dạng tài liệu trình chiếu này.|
|PPT to PPTX conversion|Aspose.Slides cho Node.js qua Java hỗ trợ chuyển đổi PPT sang PPTX|
|Portable Document Format (PDF)|Bạn có thể xuất tất cả các định dạng tệp được hỗ trợ sang tài liệu Adobe Portable Document Format (PDF) bằng một phương thức duy nhất.|
|XML Parser Specification (XPS)|Bạn có thể xuất tất cả các định dạng tệp được hỗ trợ sang tài liệu XML Parser Specification (XPS) bằng một phương thức duy nhất.|
|Tagged Image File Format (TIFF)|Bạn có thể xuất tất cả các định dạng tệp trình chiếu được hỗ trợ sang Tagged Image File Format (TIFF).|
|PPTX To HTML Conversion|Aspose.Slides cho Node.js qua Java hỗ trợ chuyển đổi PresentationEx sang định dạng HTML.|

## **Kết xuất và In ấn**
Aspose.Slides cho Node.js qua Java hỗ trợ việc kết xuất slide trong trình chiếu với độ trung thực cao sang các định dạng đồ họa khác nhau. Xem chi tiết bên dưới:

|**Tính năng**|**Mô tả**|
| :- | :- |
|Java Supported Image Formats|Sử dụng Aspose.Slides cho Node.js qua Java, bạn có thể kết xuất các slide trình chiếu và hình ảnh trên slide sang tất cả các định dạng đồ họa được Java hỗ trợ như TIFF, PNG, BMP, JPEG, GIF.|
|SVG Format|Aspose.Slides cho Node.js qua Java cũng cung cấp các phương thức tích hợp cho phép bạn xuất các slide trình chiếu sang định dạng Scalable Vector Graphics (SVG).|
|Presentation Printing|Các phiên bản mới nhất của Aspose.Slides cho Node.js qua Java cung cấp các phương thức in tích hợp với các tùy chọn khác nhau.|

## **FAQ**

**Có cần cài đặt Microsoft PowerPoint trên máy chủ/PC để thư viện hoạt động không?**

Không. PowerPoint không bắt buộc; Aspose.Slides là một công cụ độc lập để tạo, chỉnh sửa, chuyển đổi và kết xuất các trình chiếu.

**Đa luồng hoạt động như thế nào? Có thể xử lý song song không?**

Bạn có thể an toàn xử lý các tài liệu khác nhau trong các luồng khác nhau; cùng một đối tượng [presentation](https://reference.aspose.com/slides/vi/nodejs-java/aspose.slides/presentation/) không được sử dụng bởi [multiple threads](/slides/vi/nodejs-java/multithreading/) cùng lúc.

**Có hỗ trợ mật khẩu tệp và mã hoá không?**

Có. [You can](/slides/vi/nodejs-java/password-protected-presentation/) mở các trình chiếu được mã hoá, đặt hoặc xóa mật khẩu mở và ghi, và kiểm tra trạng thái bảo vệ.

**Có cần lưu ý gói phông chữ trong container Linux không?**

Có. Được khuyến nghị cài đặt các gói phông chữ phổ biến và/hoặc rõ ràng [specify font directories](/slides/vi/nodejs-java/custom-font/) trong ứng dụng của bạn để tránh việc thay thế không mong muốn.

**Phiên bản dùng thử có giới hạn không?**

Trong [evaluation mode](/slides/vi/nodejs-java/licensing/), một watermark được thêm vào đầu ra và một số hạn chế được áp dụng; một [30-day temporary license](https://purchase.aspose.com/temporary-license/) có sẵn để kiểm tra đầy đủ tính năng.

**Có hỗ trợ nhập các định dạng ngoài vào trình chiếu (PDF/HTML → PPTX) không?**

Có. Bạn có thể thêm [PDF pages and HTML content](/slides/vi/nodejs-java/import-presentation/) vào một trình chiếu, chuyển chúng thành các slide.
