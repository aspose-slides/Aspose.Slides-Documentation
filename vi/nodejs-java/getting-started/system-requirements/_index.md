## **Giới thiệu**

Aspose.Slides for Node.js via Java không yêu cầu bất kỳ sản phẩm bên thứ ba nào như Microsoft PowerPoint phải được cài đặt. Chính Aspose.Slides là một engine để tạo, sửa đổi, chuyển đổi và hiển thị tài liệu ở nhiều định dạng, bao gồm các định dạng trình chiếu Microsoft PowerPoint.

## **Hệ điều hành được hỗ trợ**

Aspose.Slides for Node.js via Java hỗ trợ bất kỳ hệ điều hành 32-bit hoặc 64-bit nào chạy môi trường Java, bao gồm nhưng không giới hạn ở:

### **Windows**
- Microsoft Windows 2003 Server ( x64, x86)
- Microsoft Windows 2008 Server ( x64, x86)
- Microsoft Windows 2012 Server ( x64, x86)
- Microsoft Windows 2012 R2 Server ( x64, x86)
- Microsoft Windows 2016 Server ( x64, x86)
- Microsoft Windows 2019 Server ( x64, x86)
- Microsoft Windows Vista ( x64, x86)
- Microsoft Windows XP ( x64, x86)
- Microsoft Windows 7 ( x64, x86)
- Microsoft Windows 8, 8.1 ( x64, x86)
- Microsoft Windows 10 ( x64, x86)

### **Linux**
- Linux (Ubuntu, CentOS và các bản khác)

### **Mac**
- Mac OS X

## **Câu hỏi thường gặp**

**Tôi có cần cài đặt Microsoft PowerPoint để chuyển đổi và hiển thị không?**

Không, PowerPoint không bắt buộc; Aspose.Slides là một engine độc lập để [tạo](/slides/vi/nodejs-java/create-presentation/), sửa đổi, [chuyển đổi](/slides/vi/nodejs-java/convert-presentation/), và [hiển thị](/slides/vi/nodejs-java/convert-powerpoint-to-png/) các bản trình chiếu.

**Cần những phông chữ nào để hiển thị đúng?**

Thực tế, các phông chữ được sử dụng trong bản trình chiếu hoặc các [phông thay thế](/slides/vi/nodejs-java/font-substitution/) thích hợp phải có sẵn. Để đảm bảo hiển thị nhất quán trên Linux/macOS, bạn nên cài đặt các gói phông chữ phổ biến.

**Tại sao một phông chữ tùy chỉnh lại hiển thị như văn bản thay thế hoặc mất trên Linux?**

Nếu tệp phông chữ có các mục trong bảng tên không nhất quán hoặc bị hỏng, ngăn xếp khớp phông chữ của Linux (FreeType/fontconfig) có thể chọn một bản ghi không hợp lệ, khiến phông chữ không được giải quyết. Sử dụng phiên bản phông chữ có các mục bảng tên đã được sửa hoặc cài đặt một bản thay thế đồng nhất sẽ giải quyết vấn đề.
