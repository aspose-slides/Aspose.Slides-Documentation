{{% alert color="primary" %}} 

Aspose.Slides for Python via .NET là một thư viện lớp cho phép các ứng dụng Python đọc và ghi tài liệu trình chiếu mà không cần sử dụng Microsoft PowerPoint® hoặc phần mềm khác. Aspose.Slides for Python via .NET cho phép bạn quản lý văn bản, hình dạng, bảng và hoạt hình; thêm âm thanh và video vào các slide; xem trước slide; và xuất slide sang định dạng SVG, PDF và hơn nữa.

Tham khảo API mới nhất có thể được tìm thấy tại [Aspose.Slides for Python via .NET API Reference](https://reference.aspose.com/slides/vi/python-net/).

{{% /alert %}}
