## **Nền tảng được hỗ trợ**
Aspose.Slides for Node.js via .NET là một API không phụ thuộc vào nền tảng. Nó có thể được sử dụng trên bất kỳ nền tảng nào (Windows, Linux, MacOS, v.v.) nơi đã cài đặt Node.js phiên bản 7 trở lên. Oracle JDK 8 trở lên là một yêu cầu khác.

## **Định dạng tệp và chuyển đổi**
Aspose.Slides for Node.js via .NET hỗ trợ hầu hết các định dạng tài liệu PowerPoint. Nó cũng cho phép bạn xuất chúng sang các định dạng tệp phổ biến mà các tổ chức sử dụng và chia sẻ với nhau. Xem chi tiết dưới đây:

|**Tính năng**|**Mô tả**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via .NET cung cấp tốc độ xử lý nhanh nhất cho định dạng tài liệu trình chiếu này.|
|PPT to PPTX conversion|Aspose.Slides for Node.js via .NET hỗ trợ chuyển đổi PPT sang PPTX|
|Portable Document Format (PDF)|Bạn có thể xuất tất cả các định dạng tệp được hỗ trợ sang tài liệu Adobe Portable Document Format (PDF) chỉ bằng một phương thức.|
|XML Parser Specification (XPS)|Bạn có thể xuất tất cả các định dạng tệp được hỗ trợ sang tài liệu XML Parser Specification (XPS) chỉ bằng một phương thức.|
|Tagged Image File Format (TIFF)|Bạn có thể xuất tất cả các định dạng tệp trình chiếu được hỗ trợ sang Tagged Image File Format (TIFF).|
|PPTX To HTML Conversion|Aspose.Slides for Node.js via .NET hỗ trợ chuyển đổi PresentationEx sang định dạng HTML.|

## **Kết xuất và In**
Aspose.Slides for Node.js via .NET hỗ trợ kết xuất độ trung thực cao của các slide trong bản trình chiếu sang các định dạng đồ họa khác nhau. Xem chi tiết dưới đây:

|**Tính năng**|**Mô tả**|
| :- | :- |
|.NET Supported Image Formats|Sử dụng Aspose.Slides for Node.js via .NET, bạn có thể kết xuất các slide trình chiếu và hình ảnh trên slide sang tất cả các định dạng đồ họa được .NET hỗ trợ như TIFF, PNG, BMP, JPEG, GIF.|
|SVG Format|Định dạng SVG: Aspose.Slides for Node.js via .NET cũng cung cấp các phương thức tích hợp cho phép bạn xuất các slide trình chiếu sang định dạng Scalable Vector Graphics (SVG).|
|Presentation Printing|In bản trình chiếu: Các phiên bản mới nhất của Aspose.Slides for Node.js via .NET cung cấp các phương thức in tích hợp với các tùy chọn khác nhau.|

## **Tính năng nội dung**
Aspose.Slides for Node.js via .NET cho phép bạn truy cập, sửa đổi hoặc tạo gần như tất cả các mục hoặc nội dung của tài liệu trình chiếu. Xem chi tiết dưới đây:

|**Tính năng**|**Mô tả**|
| :- | :- |
|Master Slides|Các Master Slides xác định bố cục của các slide thông thường. Aspose.Slides for Node.js via .NET cho phép bạn truy cập và sửa đổi các Master Slides của tài liệu trình chiếu|
|Normal Slides|Với Aspose.Slides for Node.js via .NET, bạn có thể tạo các slide mới thuộc các loại khác nhau; bạn cũng có thể truy cập và sửa đổi các slide hiện có trong bản trình chiếu|
|Cloning / Copying Slides|Có các phương thức tích hợp do Aspose.Slides for Node.js via .NET cung cấp cho phép bạn sao chép hoặc nhân bản các slide hiện có trong một bản trình chiếu. Bạn cũng có thể sử dụng các slide đã sao chép và nhân bản từ một bản trình chiếu sang bản khác. Vì một slide kế thừa bố cục từ master slide, các phương thức nhân bản tích hợp tự động sao chép master khi nhân bản|
|Managing Slides sections|Các phương thức để tổ chức các slide trong các phần khác nhau trong một bản trình chiếu|
|Place Holders and Text Holders|Bạn có thể truy cập các placeholder và text holder trong một slide. Hơn nữa, bạn có thể tạo một slide với các text holder từ đầu bằng cách sử dụng phương thức phù hợp|
|Header and Footers|Aspose.Slides for Node.js via .NET hỗ trợ việc xử lý header/footer trong các slide|
|Notes in Slides|Với Aspose.Slides for Node.js via .NET, bạn có thể truy cập và sửa đổi ghi chú liên kết với một slide và cũng có thể thêm ghi chú mới|
|Finding a Shape|Bạn cũng có thể tìm một hình dạng cụ thể trong slide bằng cách sử dụng văn bản thay thế liên kết với hình dạng đó|
|Backgrounds|Aspose.Slides for Node.js via .NET cho phép bạn làm việc với nền liên kết với master hoặc slide thông thường trong một bản trình chiếu|
|Text Boxes|Các hộp văn bản có thể được tạo từ đầu. Bạn có thể truy cập các hộp văn bản hiện có. Bạn cũng có thể sửa đổi nội dung của chúng mà không mất định dạng văn bản gốc|
|Rectangle Shapes|Bạn có thể tạo hoặc sửa đổi các hình chữ nhật bằng Aspose.Slides for Node.js via .NET|
|Poly Line Shapes|Bạn có thể tạo hoặc sửa đổi các hình poly line bằng Aspose.Slides for Node.js via .NET|
|Ellipse Shapes|Bạn có thể tạo hoặc sửa đổi các hình ellipse bằng Aspose.Slides for Node.js via .NET|
|Group Shapes|Aspose.Slides for Node.js via .NET hỗ trợ các nhóm hình dạng|
|Auto Shapes|Aspose.Slides for Node.js via .NET hỗ trợ các auto shape|
|SmartArt|Aspose.Slides for Node.js via .NET cung cấp hỗ trợ cho các hình SmartArt trong MS PowerPoint|
|Charts|Aspose.Slides for Node.js via .NET cung cấp hỗ trợ cho các biểu đồ MSO trong PowerPoint|
|Shapes Serialization|Aspose.Slides for Node.js via .NET hỗ trợ một số lượng lớn các hình dạng. Khi Aspose.Slides for Node.js via .NET không hỗ trợ một hình dạng nào đó, bạn có thể sử dụng phương thức serialization để tuần tự hoá hình dạng đó từ một slide hiện có. Bằng cách này, bạn có thể sử dụng hình dạng đó tiếp theo theo yêu cầu của mình|
|Picture Frames|Bạn có thể quản lý hình ảnh trong picture frames bằng Aspose.Slides for Node.js via .NET|
|Audio Frames|Bạn có thể liên kết hoặc nhúng các tệp âm thanh trong audio frames trên slide bằng Aspose.Slides for Node.js via .NET|
|Video Frames|Bạn có thể xử lý các tệp video trong video frames. Aspose.Slides for Node.js via .NET cũng cung cấp hỗ trợ cho video liên kết và nhúng|
|OLE Frame|Bạn có thể quản lý các đối tượng OLE trong OLE frames bằng Aspose.Slides for Node.js via .NET|
|Tables|Aspose.Slides for Node.js via .NET hỗ trợ các bảng trong slide|
|ActiveX Controls|Hỗ trợ các điều khiển ActiveX|
|VBA Macros|Hỗ trợ quản lý macro VBA trong các bản trình chiếu|
|Text Frame|Bạn có thể truy cập văn bản của bất kỳ hình dạng nào thông qua text frame được liên kết với hình dạng đó|
|Text Scanning|Bạn có thể quét văn bản trong một bản trình chiếu ở mức độ bản trình chiếu hoặc slide thông qua các phương thức quét tích hợp|
|Animations|Bạn có thể áp dụng các hoạt ảnh lên các hình dạng|
|Slide Shows|Aspose.Slides for Node.js via .NET hỗ trợ các slide show và chuyển đổi slide|

## **Tính năng định dạng**
Với Aspose.Slides for Node.js via .NET, bạn có thể định dạng văn bản và hình dạng trên các slide trong bản trình chiếu. Xem chi tiết dưới đây:

|**Tính năng**|**Mô tả**|
| :- | :- |
|Text Formatting|<p>Trong Aspose.Slides for Node.js via .NET, bạn có thể quản lý văn bản thông qua các text frame liên kết với các hình dạng. Do đó, bạn có thể định dạng văn bản bằng cách sử dụng các đoạn văn và phần liên quan đến các text frame. Các yếu tố văn bản này có thể được định dạng thông qua Aspose.Slides for Node.js via .NET.</p><p>- Kiểu phông chữ</p><p>- Cỡ phông chữ</p><p>- Màu phông chữ</p><p>- Tông màu phông chữ</p><p>- Căn đoạn</p><p>- Đánh dấu đoạn</p><p>- Hướng đoạn</p>|
|Shape Formatting|<p>Trong Aspose.Slides for Node.js via .NET, thành phần cơ bản của một slide là một shape. Bạn có thể định dạng các thành phần shape này bằng Aspose.Slides for Node.js via .NET:</p><p>- Vị trí</p><p>- Kích thước</p><p>- Đường viền</p><p>- Đổ màu (bao gồm Pattern, Gradient, Solid)</p><p>- Văn bản</p><p>- Hình ảnh</p>|
