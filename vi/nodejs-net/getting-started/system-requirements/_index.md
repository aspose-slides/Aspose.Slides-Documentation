Aspose.Slides cho Node.js qua .NET không yêu cầu bất kỳ sản phẩm bên thứ ba nào như Microsoft PowerPoint được cài đặt. Chính Aspose.Slides là một động cơ để tạo, chỉnh sửa, chuyển đổi và render tài liệu ở nhiều định dạng, bao gồm các định dạng bài thuyết trình của Microsoft PowerPoint.

## Hệ điều hành được hỗ trợ

Aspose.Slides cho .NET hỗ trợ bất kỳ hệ điều hành 32‑bit hoặc 64‑bit nào chạy môi trường .NET runtime, bao gồm nhưng không giới hạn ở:

### **Windows**
- Microsoft Windows 2003 Server ( x64, x86)
- Microsoft Windows 2008 Server ( x64, x86)
- Microsoft Windows 2012 Server ( x64, x86)
- Microsoft Windows 2012 R2 Server ( x64, x86)
- Microsoft Windows 2016 Server ( x64, x86)
- Microsoft Windows 2019 Server ( x64, x86)
- Microsoft Windows Vista ( x64, x86)
- Microsoft Windows XP ( x64, x86)
- Microsoft Windows 7 ( x64, x86)
- Microsoft Windows 8, 8.1 ( x64, x86)
- Microsoft Windows 10 ( x64, x86)
- Microsoft Windows 11 ( x64, x86)

### **Linux**
- Linux (Ubuntu, CentOS và các nền tảng khác)

### **Mac**
- Mac OS X
