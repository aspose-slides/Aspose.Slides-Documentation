{{% alert color="primary" %}} 

Aspose.Slides for Node.js via .NET là một thư viện lớp cho phép các ứng dụng Node.js đọc và ghi tài liệu trình chiếu mà không cần sử dụng Microsoft PowerPoint® hoặc các phần mềm khác. Aspose.Slides for Node.js via .NET cho phép bạn quản lý văn bản, hình dạng, bảng & hoạt ảnh, thêm âm thanh và video vào các slide, xem trước slide, xuất slide sang định dạng SVG, PDF và còn nhiều tính năng khác.

Aspose.Slides for Node.js via .NET sử dụng [cùng tài liệu API như Aspose.Slides .NET](https://reference.aspose.com/slides/vi/nodejs-net/). 

{{% /alert %}}
