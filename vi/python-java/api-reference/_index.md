{{% alert color="primary" %}} 

Aspose.Slides for Python via Java là một thư viện lớp cho phép các ứng dụng Python đọc và ghi tài liệu trình chiếu mà không cần sử dụng Microsoft PowerPoint® hoặc các phần mềm khác. Aspose.Slides for Python via Java cho phép bạn quản lý văn bản, hình dạng, bảng và hoạt ảnh, thêm âm thanh và video vào các slide, xem trước slide, xuất slide ra định dạng SVG, PDF và thực hiện nhiều thao tác khác.

Aspose.Slides for Python via Java sử dụng [cùng tài liệu tham chiếu API như Aspose.Slides for Java](https://reference.aspose.com/slides/vi/python-java/). 

{{% /alert %}}
