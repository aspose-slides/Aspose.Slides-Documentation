## **Power BI**
Bạn có thể cài đặt Aspose.Slides for Reporting Services trên máy chủ Power BI bằng các quy trình cài đặt tương tự:

Xem

* [Cài đặt bằng Trình cài đặt MSI](https://docs.aspose.com/slides/vi/reportingservices/install-with-msi-installer/#installation). Chương trình cài đặt tự động phát hiện Power BI đã cài trên máy chủ, cài đặt sản phẩm và sau đó cấu hình cho môi trường làm việc của bạn.
* [Cài đặt thủ công](https://docs.aspose.com/slides/vi/reportingservices/install-manually/).

{{% alert title="Note" color="warning" %}} 
**Aspose.Slides for Reporting Services** yêu cầu cài đặt **.NET Framework 3.5** trên máy chủ. 
{{% /alert %}}

Aspose.Slides for SSRS hỗ trợ báo cáo có phân trang (cdl).
