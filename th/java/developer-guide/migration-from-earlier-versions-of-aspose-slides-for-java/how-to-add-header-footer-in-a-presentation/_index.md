{{% alert color="primary" %}} 

มีการเปิดตัว [Aspose.Slides for Java API](https://docs.aspose.com/slides/th/java/) ใหม่และตอนนี้ผลิตภัณฑ์เดียวนี้รองรับความสามารถในการสร้างเอกสาร PowerPoint ตั้งแต่ต้นและแก้ไขเอกสารที่มีอยู่

{{% /alert %}} 
## **การสนับสนุนโค้ดรุ่นเก่า**
เพื่อใช้โค้ดรุ่นเก่าที่พัฒนาโดยใช้ Aspose.Slides for Java รุ่นก่อนหน้า 13.x คุณจำเป็นต้องทำการเปลี่ยนแปลงเล็กน้อยในโค้ดของคุณและโค้ดจะทำงานเช่นเดิม คลาสทั้งหมดที่เคยอยู่ใน Aspose.Slides for Java รุ่นเก่าภายใต้เนมสเปซ Aspose.Slide และ Aspose.Slides.Pptx ตอนนี้ได้ถูกรวมเข้าในเนมสเปซ Aspose.Slides เพียงหนึ่งเดียว กรุณาดูตัวอย่างโค้ดง่ายๆ ด้านล่างสำหรับการเพิ่มหัวกระดาษและเท้ากระดาษในงานนำเสนอด้วย Aspose.Slides API รุ่นเก่าและทำตามขั้นตอนที่อธิบายวิธีการย้ายไปยัง API ที่รวมใหม่
