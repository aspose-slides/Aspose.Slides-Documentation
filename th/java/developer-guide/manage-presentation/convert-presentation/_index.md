## **ภาพรวม**

Aspose.Slides for Java สามารถโหลดงานนำเสนอ PowerPoint และ OpenDocument และบันทึกหรือแสดงผลเป็นรูปแบบอื่น ๆ มากมายโดยไม่ต้องพึ่ง Microsoft PowerPoint, OpenOffice หรือ LibreOffice คุณสามารถแปลงไฟล์ PPT เก่าเป็น PPTX สมัยใหม่ ส่งออกงานนำเสนอเป็นเอกสารแบบจัดหน้าแน่นเช่น PDF และ XPS, เผยแพร่สไลด์เป็น HTML หรือแสดงสไลด์เป็นไฟล์รูปภาพสำหรับการพรีวิว, ภาพย่อและการเก็บถาวร

การแปลงเอกสารส่วนใหญ่ใช้กระบวนการทำงานทั่วไปเดียวกัน: โหลดไฟล์ต้นฉบับ, เลือกรูปแบบผลลัพธ์ที่ต้องการ, และใช้ตัวเลือกเฉพาะรูปแบบเมื่อจำเป็น สำหรับรูปแบบรูปภาพแต่ละสไลด์จะถูกเรนเดอร์แยกจากกันแล้วบันทึกเป็นภาพแรสเตอร์หรือเวกเตอร์ บทความเฉพาะด้านที่เชื่อมโยงด้านล่างให้รายละเอียดการใช้งานแต่ละกรณี

## **การแปลง PowerPoint และ OpenDocument**

Aspose.Slides for Java รองรับการแปลงจากรูปแบบงานนำเสนอที่ใช้บ่อย เช่น PPT, PPTX, PPS, PPSX, POT, POTX และ ODP API การแปลงเดียวกันใช้สำหรับไฟล์ PowerPoint และ OpenDocument ดังนั้นกระบวนการที่บันทึกไฟล์ PPTX เป็น PDF มักสามารถนำไปใช้กับไฟล์ ODP ได้โดยเปลี่ยนเฉพาะไฟล์อินพุต

เมื่อแปลงไฟล์ ODP, จำไว้ว่าแอปพลิเคชัน PowerPoint และ OpenDocument ไม่รองรับทุกการจัดเลย์เอาต์และฟีเจอร์การฟอร์แมตในแบบเดียวกัน หากไฟล์ ODP ถูกสร้างใน LibreOffice หรือ OpenOffice Impress ให้ตรวจสอบผลลัพธ์และใช้ตัวเลือกที่อธิบายไว้ใน [แปลงงานนำเสนอ OpenDocument](/slides/th/java/convert-openoffice-odp/) เมื่อคุณต้องการคำแนะนำเฉพาะรูปแบบ

## **การแปลง PPT เป็น PPTX**

 PPT เป็นรูปแบบ PowerPoint ไบนารีรุ่นเก่า ในขณะที่ PPTX เป็นรูปแบบ Office Open XML สมัยใหม่ Aspose.Slides for Java รองรับการแปลง PPT เป็น PPTX ด้วยความแม่นยำสูงโดยคงโครงสร้างงานนำเสนอที่ซับซ้อน เช่น มาสเตอร์, เลย์เอาต์, สไลด์, แผนภูมิ, รูปร่างที่จัดกลุ่ม, พื้นที่สำหรับวาง, กรอบข้อความ, เนื้อผิว, และการเติมรูปภาพ

สำหรับรายละเอียดเพิ่มเติม, ดูที่ [แปลง PPT เป็น PPTX](/slides/th/java/convert-ppt-to-pptx/) และ [PPT vs PPTX](/slides/th/java/ppt-vs-pptx/).

## **การส่งออกแบบจัดหน้าแน่น**

PDF, XPS, และ TIFF มีประโยชน์เมื่อผลลัพธ์ต้องแสดงเหมือนกันบนอุปกรณ์ต่าง ๆ และไม่ควรแก้ไขเป็นงานนำเสนอ บทความเฉพาะด้าน PDF, XPS, และ TIFF อธิบายวิธีควบคุมการปฏิบัติตาม, สไลด์ที่ซ่อน, บันทึกย่อ, คุณภาพภาพ, การบีบอัด, รูปแบบพิกเซล, และขนาดผลลัพธ์

## **การส่งออก HTML และรูปภาพ**

การส่งออก HTML และ HTML5 มีประโยชน์สำหรับการดูในเบราว์เซอร์, การเผยแพร่เว็บ, และการแชร์ที่เบา การส่งออกรูปภาพมีประโยชน์เมื่อแต่ละสไลด์ต้องเป็นพรีวิว, ภาพย่อ หรือแอสเซ็ทแรสเตอร์ที่แยกกัน ใช้บทความ PNG, JPG, และ SVG เพื่อคำแนะนำการเรนเดอร์เฉพาะรูปแบบ

## **คำถามที่พบบ่อย**

**ฉันต้องการ Microsoft PowerPoint เพื่อแปลงงานนำเสนอหรือไม่?**

ไม่ Aspose.Slides for Java เป็นไลบรารีแยกของตัวเองและไม่ต้องการ Microsoft PowerPoint หรือการทำงานอัตโนมัติของ Office

**ฉันสามารถแปลงงานนำเสนอหลายไฟล์เป็นกลุ่มได้หรือไม่?**

ได้ โหลดงานนำเสนอแต่ละไฟล์, บันทึกเป็นรูปแบบที่ต้องการ, แล้วทำลายอ็อบเจกต์งานนำเสนอหลังการประมวลผล สำหรับการประมวลผลแบบขนาน, ใช้อินสแตนซ์งานนำเสนอแยกกันและปฏิบัติตามคำแนะนำ [multithreading](/slides/th/java/multithreading/)

**ฉันสามารถส่งออกเฉพาะสไลด์ที่เลือกได้หรือไม่?**

ได้ วิธีการส่งออกหลายวิธีอนุญาตให้คุณส่งผ่านดัชนีสไลด์หรือเรนเดอร์สไลด์แต่ละอันตามรูปแบบผลลัพธ์ ดูบทความเฉพาะสำหรับรูปแบบเป้าหมาย

**ฉันสามารถรวมสไลด์ที่ซ่อนเมื่อส่งออกเป็น PDF หรือ XPS ได้หรือไม่?**

ได้ ใช้การตั้งค่าการส่งออกสไลด์ที่ซ่อนตามที่อธิบายในบทความการแปลง [PDF](/slides/th/java/convert-powerpoint-to-pdf/) และ [XPS](/slides/th/java/convert-powerpoint-to-xps/)

**ฉันสามารถสร้างผลลัพธ์ PDF/A ได้หรือไม่?**

ได้ มีการตั้งค่าการปฏิบัติตาม PDF สำหรับการส่งออก PDF ดูที่ [แปลง PowerPoint เป็น PDF](/slides/th/java/convert-powerpoint-to-pdf/) สำหรับรายละเอียด

**ฟอนต์จะถูกจัดการอย่างไรระหว่างการแปลง?**

Aspose.Slides สามารถใช้ฟอนต์แบบฝัง, การสำรองฟอนต์, และการตั้งค่าการแทนฟอนต์ ดูที่ [Embedded Font](/slides/th/java/embedded-font/), [Fallback Font](/slides/th/java/fallback-font/), และ [Font Substitution](/slides/th/java/font-substitution/).
