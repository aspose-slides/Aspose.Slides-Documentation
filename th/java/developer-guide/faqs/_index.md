## **ภาพรวม**

FAQนี้ให้คำตอบสำหรับคำถามทั่วไปเกี่ยวกับ Aspose.Slides ครอบคลุมรูปแบบไฟล์ที่รองรับ, การจัดการข้อยกเว้นเมื่อทำงานกับงานนำเสนอขนาดใหญ่, การเปลี่ยนขนาดสไลด์, การแสดงตัวอย่างสไลด์, การดึงข้อความจากงานนำเสนอ, การจัดรูปแบบขอบตาราง, การวางรูปภาพ, และการแก้ไขปัญหาเกี่ยวกับฟอนต์เมื่อแปลงงานนำเสนอเป็น PDF หรือภาพ

## **รูปแบบไฟล์ที่รองรับ**

**Q:** Aspose.Slides for Java รองรับรูปแบบไฟล์อะไรบ้าง?

**A**: Aspose.Slides for Java รองรับรูปแบบไฟล์ที่อธิบายไว้ใน [Supported File Formats](/slides/th/java/supported-file-formats/).

## **ข้อยกเว้น**

**Q:** ฉันได้รับข้อยกเว้น out of memory ขณะโหลดไฟล์ PPT ขนาดใหญ่ที่มีรูปภาพ มีข้อจำกัดใดใน Aspose.Slides เกี่ยวกับขนาดไฟล์หรือไม่?

**A**: ไม่มีสูตรเฉพาะสำหรับการคำนวณขนาดงานนำเสนอที่ Aspose.Slides รองรับ ควรมีพื้นที่เพียงพอเพื่อบรรจุโครงสร้างงานนำเสนอทั้งหมดและรูปภาพในหน่วยความจำ ปกติแล้วรูปภาพในหน่วยความจำจะใช้พื้นที่มากกว่าที่อยู่บนฮาร์ดดิสก์ โดยเฉพาะเมื่อรูปภาพมีเอฟเฟกต์เพิ่มเติม

โดยทั่วไป Aspose.Slides for Java สามารถจัดการไฟล์งานนำเสนอขนาดประมาณ 300 MB บนเซิร์ฟเวอร์ที่มี RAM 4 GB ได้อย่างง่ายดาย

## **การทำงานกับสไลด์**

**Q:** ฉันสามารถเปลี่ยนขนาดของสไลด์ในงานนำเสนอได้หรือไม่?

**A**: คุณสามารถใช้เมธอด `getSlideSize` ที่เปิดให้ใช้โดยคลาส [Presentation](https://reference.aspose.com/slides/th/java/com.aspose.slides/presentation/) เพื่อกำหนดขนาดของสไลด์ในงานนำเสนอ

**Q:** มีวิธีใดที่จะกำหนดสไลด์ที่มีขนาดต่างกันในงานนำเสนอหรือไม่?

**A**: เนื่องจากขนาดของสไลด์ถูกกำหนดในระดับงานนำเสนอในเอกสาร Microsoft PowerPoint จึงไม่มีวิธีทำเช่นนี้

**Q:** Aspose.Slides for Java รองรับการแสดงตัวอย่างสไลด์ก่อนบันทึกหรือไม่?

**A**: คุณสามารถแปลงสไลด์ของงานนำเสนอเป็นภาพและใช้ภาพเหล่านั้นสำหรับการแสดงตัวอย่างสไลด์

## **การทำงานกับข้อความ**

**Q:** สามารถดึงข้อความทั้งหมดจากงานนำเสนอได้หรือไม่?

**A**: Aspose.Slides for Java มีคลาส [SlideUtil](https://reference.aspose.com/slides/th/java/com.aspose.slides/slideutil/) ที่ให้เมธอดต่าง ๆ สำหรับดึงข้อความทั้งหมดจากงานนำเสนอ

**Q:** ทำไมขนาดย่อหน้าจึงต่างกันบนระบบปฏิบัติการ Windows และ Linux?

**A**: การคำนวนขนาดย่อหน้าขึ้นอยู่กับการคำนวนขนาดข้อความที่แสดงย่อหน้าที่กำหนด ขนาดข้อความคำนวนจากเมตริกของฟอนต์ที่ระบุในงานนำเสนอ PowerPoint หากฟอนต์ที่ระบุไม่มีอยู่ จะถูกแทนที่ด้วยฟอนต์ที่คล้ายที่สุด แต่ฟอนต์นั้นมีเมตริกที่ต่างจากฟอนต์ต้นฉบับ ดังนั้นการคำนวนขนาดย่อหน้าในระบบต่าง ๆ จะให้ผลลัพธ์ที่ต่างกันขึ้นอยู่กับชุดฟอนต์ที่ติดตั้ง เพื่อให้ได้ผลลัพธ์เดียวกันบนระบบปฏิบัติการต่าง ๆ คุณต้องติดตั้งฟอนต์เดียวกันบนระบบหรือโหลดฟอนต์ในระหว่างการทำงานเป็น [external fonts](/slides/th/java/custom-font/).

## **การจัดรูปแบบและรูปภาพ**

**Q:** วิธีตั้งค่าสีของขอบตารางได้อย่างไร?

**A**: คุณสามารถเปลี่ยนสีของขอบตารางทั้งหมดหรือเพียงขอบรอบตารางโดยใช้เมธอด `getCellFormat` จากอินเทอร์เฟซ [ICell](https://reference.aspose.com/slides/th/java/com.aspose.slides/icell/) สำหรับการเปลี่ยนสีขอบทั้งหมด, โปรดใช้เมธอด `getCellFormat`. สำหรับขอบของตารางทั้งหมด, คุณต้องวนลูปเซลล์และเปลี่ยนสีของขอบภายนอก

**Q:** Aspose.Slides for Java ใช้หน่วยวัดใดในการวางรูปภาพ?

**A**: พิกัดและขนาดของรูปร่างทั้งหมดบนสไลด์จะวัดเป็นหน่วย point (72 dpi).

## **การทำงานกับฟอนต์**

**Q:** เมื่อแปลง PPT เป็น PDF หรือภาพ ทำไมฟอนต์ถึงแตกต่างกันในเอกสารผลลัพธ์?

**A**: ปัญหานี้อาจบ่งชี้ว่าฟอนต์ที่ใช้ในงานนำมายังไม่มีอยู่ในระบบปฏิบัติการที่รันโค้ด คุณควรติดตั้งฟอนต์บนระบบปฏิบัติการหรือโหลดฟอนต์เป็นฟอนต์ภายนอกโดยใช้คลาส [FontsLoader](https://reference.aspose.com/slides/th/java/com.aspose.slides/fontsloader/) ดังที่แสดงด้านล่าง:
```cs
var folders = new String[] { "path_to_a_folder_with_fonts" };
FontsLoader.loadExternalFonts(folders);
```
