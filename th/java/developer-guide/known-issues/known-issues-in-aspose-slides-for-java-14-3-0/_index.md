Aspose.Slides for Java 14.3.0 (14.4.0) มีการนำเสนอการประมวลผล PPT ด้วยการนำไปใช้ใหม่อย่างสมบูรณ์ มีการปรับปรุงหลายประการ รวมถึงการแปลงบางส่วนจาก PPTX ไปเป็น PPT อย่างไรก็ตาม มีคุณลักษณะที่ยังไม่ได้ทำงาน:

- รูปทรงบางอย่างมีเรขาคณิตไม่ถูกต้องในเอกสาร PPT ที่ทำซีเรียลไลซ์ (Call outs)
- ไม่รองรับคุณลักษณะการจัดรูปแบบข้อความ PPTX ทั้งหมดในการทำซีเรียลไลซ์ PPT
- ข้อมูลเกี่ยวกับภาษาข้อความและการตั้งค่าการตรวจสอบการสะกดไม่มีในเอกสาร PPT ที่ทำซีเรียลไลซ์
- ไม่รองรับคุณลักษณะของธีม PPTX ทั้งหมดในการทำซีเรียลไลซ์ PPT

**มีความแตกต่างบางประการเมื่อเปรียบเทียบกับ Aspose.Slides for Java 8.6.0:**

- มีปัญหาที่ทราบอยู่ในการทำซีเรียลไลซ์ OLE/ActiveX PPT เป็น PPT

**มีความแตกต่างบางประการเมื่อเปรียบเทียบกับ Aspose.Slides for .NET 14.3.0:**

- การสนับสนุนการพิมพ์งานนำเสนอในขณะนี้ยังไม่พร้อมใช้งานใน Aspose.Slides for Java
