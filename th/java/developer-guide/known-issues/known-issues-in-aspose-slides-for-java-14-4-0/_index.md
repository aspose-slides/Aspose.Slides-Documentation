{{% alert color="primary" %}} 

Aspose.Slides for Java 14.4.0 มีการตัดสินใจใหม่สำหรับการประมวลผลเอกสาร PowerPoint มีข้อจำกัดและปัญหาที่ทราบอยู่บางประการ ซึ่งจะถูกลบออกในรุ่นต่อ ๆ ไป:

- บางรูปทรงมีรูปทรงเรขาคณิตที่ไม่ถูกต้องในเอกสาร PPT ที่ทำการจัดลำดับ (โค้ง, ลูกศรวงกลม, คำบรรยาย).
- คุณลักษณะการจัดรูปแบบข้อความของ PPTX ทั้งหมดไม่ได้รับการสนับสนุนในการจัดลำดับเป็น PPT (การจัดแท็บ, การเยื้องและข้อจำกัดการจัดรูปแบบย่อหน้า).
- ข้อมูลเกี่ยวกับภาษาของข้อความและการตั้งค่าการตรวจสอบการสะกดไม่ปรากฏในเอกสาร PPT ที่ทำการจัดลำดับ.
- คุณลักษณะธีมของ PPTX ทั้งหมดไม่ได้รับการสนับสนุนในการจัดลำดับเป็น PPT (มีการจัดลำดับเฉพาะรูปแบบการเติม, รูปแบบเส้นและแบบอักษร).
- มีปัญหาที่ทราบอยู่ในการจัดลำดับ OLE/ActiveX จาก PPT ไปเป็น PPT.
- การจัดลำดับและการเรนเดอร์ WordArt ไม่ได้รับการสนับสนุน.
