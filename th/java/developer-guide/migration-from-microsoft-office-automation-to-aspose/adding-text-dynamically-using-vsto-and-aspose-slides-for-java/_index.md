## **การเพิ่มข้อความแบบไดนามิก**
ทั้งสองวิธีทำตามขั้นตอนต่อไปนี้:

1. สร้างงานพรีเซนเทชัน.
1. เพิ่มสไลด์เปล่า.
1. เพิ่มกล่องข้อความ.
1. กำหนดข้อความบางส่วน.
1. บันทึกงานพรีเซนเทชัน.
