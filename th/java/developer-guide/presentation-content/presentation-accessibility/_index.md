## **บทนำ**

การทำให้การนำเสนอสามารถเข้าถึงได้ทำให้คนที่ใช้เทคโนโลยีช่วยเหลือ—เช่นโปรแกรมอ่านหน้าจอ, จอแสดงผลอักษรเบรลล์, หรือการนำทางด้วยแป้นพิมพ์—สามารถเข้าใจและเลื่อนดูสไลด์ของคุณได้อย่างมีประสิทธิภาพเท่ากับผู้ชมที่มองเห็นและใช้เมาส์ การปฏิบัติเ�ดีมุ่งเน้นที่ลำดับการอ่านที่ชัดเจน, ข้อความแทนที่มีความหมายสำหรับภาพที่ให้ข้อมูล, ความแตกต่างของสีที่เพียงพอ, การจัดพิมพ์ที่อ่านง่าย, ข้อความลิงก์ที่อธิบาย, และหลีกเลี่ยงการสื่อความหมายโดยใช้สีหรือตำแหน่งอย่างเดียว เมื่อการเข้าถึงได้ถูกวางแผนตั้งแต่เริ่มต้น ผลลัพธ์จะเป็นโครงสร้างที่สะอาดตา, ภาพที่สอดคล้องกันมากขึ้น, และเนื้อหาที่ถึงผู้ชมทุกคนโดยไม่ต้องมีการแก้ไขเบราว์เซอร์

## **ทำเครื่องหมายเป็นการตกแต่ง**

ทำเครื่องหมายเป็นการตกแต่งใช้เพื่อระบุภาพที่เป็นเพียงการตกแต่งเท่านั้น เพื่อให้โปรแกรมอ่านหน้าจอข้ามผ่าน ลดเสียงรบกวนและทำให้ความสนใจอยู่ที่เนื้อหาที่สำคัญ ใช้กับพื้นหลัง, ลวดลาย, และช่องว่าง—อย่าใช้กับแผนภูมิ, ไอคอน, หรือรูปภาพที่สื่อข้อมูล Aspose.Slides เปิดเผยฟlags นี้เพื่อการตรวจจับและการตรวจสอบความถูกต้อง, ทำให้การตรวจสอบการเข้าถึงและการทำความสะอาดอัตโนมัติเป็นไปได้

![ทำเครื่องหมายเป็นการตกแต่ง](mark_as_decorative.png)

ตัวอย่างโค้ดต่อไปนี้แสดงวิธีตรวจสอบว่ารูปร่างถูกทำเครื่องหมายเป็นการตกแต่งหรือไม่

```java
Presentation presentation = new Presentation("sample.pptx");
try {
    IShape shape = presentation.getSlides().get_Item(0).getShapes().get_Item(0);
    System.out.println("Is shape decorative: " + shape.isDecorative());
} finally {
    presentation.dispose();
}
```
