## **ยินดีต้อนรับสู่ Aspose.Slides for Java!**

Aspose.Slides for Java คือ API การจัดการ Microsoft PowerPoint® ที่ทำให้แอปพลิเคชัน Java สามารถอ่านและเขียนเอกสาร PowerPoint® ได้โดยไม่ต้องใช้ Microsoft PowerPoint®  

Aspose.Slides for Java เป็นคอมโพเนนต์ตัวแรกและตัวเดียวที่ให้ฟังก์ชันที่จำเป็นสำหรับการจัดการเอกสาร PowerPoint®  

Aspose.Slides for Java มีคุณลักษณะสำคัญหลายอย่าง เช่น การจัดการข้อความ รูปร่าง ตารางและแอนิเมชัน การเพิ่มเสียงและวิดีโอบนสไลด์ การดูตัวอย่างสไลด์ การส่งออกสไลด์เป็น SVG, PDF และรูปแบบอื่น ๆ  

## **รายละเอียดสินค้า**

Aspose.Slides for Java ไม่เพียงแต่ให้คุณเปิดไฟล์ PowerPoint จากแหล่งต่าง ๆ เท่านั้น แต่ยังอนุญาตให้คุณบันทึกงานนำเสนอของคุณในหลายรูปแบบ ตัวอย่างเช่น คุณสามารถบันทึกงานนำเสนอเป็นไฟล์ PPT หรือบันทึกสไลด์เป็นรูปภาพได้  

Aspose.Slides for Java ให้คุณสมบัติเกือบทั้งหมดที่คุณอาจพบหรือไม่พบใน Microsoft PowerPoint ด้วยความยืดหยุ่นของ Aspose.Slides for Java นอกจากตารางและรูปทรงหลายประเภทแล้ว คุณยังสามารถเพิ่มเฟรมประเภทต่าง ๆ — ข้อความ เสียง และวิดีโอ — ไปยังสไลด์ของคุณได้อีกด้วย  

Aspose.Slides for Java ยังจัดเตรียมตัวอย่างสาธิตและตัวอย่างการใช้งานที่ครบครันสำหรับนักพัฒนาเพื่อให้เข้าใจ API ได้ดียิ่งขึ้น  

### **คุณสมบัติการนำเสนอ**

- เปิดงานนำเสนอ Microsoft PowerPoint จากแหล่งต่าง ๆ  
- บันทึกงานนำเสนอในรูปแบบ Microsoft PowerPoint ต่าง ๆ  
- สร้างงานนำเสนอจากศูนย์  
- แปลงงานนำเสนอเป็นเอกสาร PDF โดยไม่ต้องใช้คอมโพเนนต์อื่น  
- รองรับการอ่าน Open Document Presentation (ODP)  
- แปลง PPT เป็น PPTX หรือกลับกันได้  
- อ่านหรือแก้ไขคุณสมบัติของเอกสารงานนำเสนอ  
- ส่งออกงานนำเสนอเป็น PDF  
- ส่งออกงานนำเสนอเป็น TIFF  
- ส่งออกงานนำเสนอเป็น XPS  
- ส่งออกงานนำเสนอเป็น SWF  
- ส่งออกงานนำเสนอเป็น PDF Note / Tiff Notes  
- ส่งออกงานนำเสนอเป็น HTML  

### **คุณสมบัติของสไลด์**

- สร้างสไลด์ใหม่จากศูนย์และเพิ่มลงในงานนำเสนอ  
- เข้าถึงสไลด์ใด ๆ ตามตำแหน่งในงานนำเสนอ  
- คัดลอกหรือโคลนสไลด์ไปยังงานนำเสนอเดียวกันหรืออื่น ๆ  
- แก้ไขและลบสไลด์  
- คัดลอกมาสเตอร์พร้อมสไลด์โดยอัตโนมัติ  
- ดูตัวอย่างสไลด์  
- รับหรือกำหนดชื่อสไลด์  
- บันทึกสไลด์เป็นรูปภาพ  
- ส่งออกสไลด์เป็นรูปแบบ SVG  
- ทำงานกับมาสเตอร์หลายตัวในงานนำเสนอ  
- เติมพื้นหลังสไลด์ด้วยสไตล์ที่รองรับใด ๆ  
- เพิ่มหรือจัดการโน้ตสไลด์  
- กำหนดมาสเตอร์และสไตล์สไลด์ใหม่แบบไดนามิก  
- เพิ่มหรือแก้ไขการเปลี่ยนภาพสไลด์โชว์  
- เพิ่มการเปลี่ยนภาพแบบ Morph  

### **คุณลักษณะกราฟิกและมัลติมีเดีย**

- สร้างหรือจัดการรูปร่าง เช่น เส้น, สี่เหลี่ยม, วงรี และโพลีไลน์  
- จัดการสไตล์เส้นในรูปร่าง  
- รับข้อความอธิบายทางเลือกของรูปร่าง  
- หมุนหรือกลับรูปร่าง  
- เติมรูปร่างด้วยสไตล์ต่าง ๆ เช่น สีทึบ, ไร่สี, ลวดลาย, เนื้อพื้นผิวและรูปภาพ  
- ใช้ลวดลายที่กำหนดล่วงหน้า 48 แบบและพื้นผิว 24 แบบ  
- กำหนดลิงก์ให้กับรูปร่าง  
- อ่าน, แก้ไข และเพิ่มเอฟเฟกต์แอนิเมชันใหม่ให้สไลด์และรูปร่าง  
- เพิ่มรูปภาพใหม่ลงในงานนำเสนอและจัดการรูปภาพที่มีอยู่  
- เพิ่มจุดสัญลักษณ์รูปภาพใหม่ลงในงานนำเสนอและจัดการจุดสัญลักษณ์ที่มีอยู่  
- เพิ่มเฟรมเสียงใหม่ลงในงานนำเสนอและจัดการเฟรมที่มีอยู่  
- เพิ่มเฟรมวิดีโอใหม่ลงในงานนำเสนอและจัดการเฟรมที่มีอยู่  

### **คุณสมบัติตาราง**

- สร้างตารางจากศูนย์และเพิ่มลงในสไลด์  
- ปรับขนาดและย้ายตาราง  
- เข้าถึงเซลล์ใด ๆ ในตาราง  
- เปลี่ยนรูปแบบและขอบของเซลล์ใด ๆ  
- ปรับขนาดคอลัมน์และแถว  
- เพิ่มและลบคอลัมน์และแถว  
- กำหนดความกว้างและสีของขอบตารางทั้งหมด  
- ใช้สไตล์ตารางที่กำหนดล่วงหน้า  

### **คุณสมบัติข้อความและย่อหน้า**

- รับและกำหนดข้อความในตัวแทนข้อความและเฟรมข้อความ  
- รับประเภทของตัวแทนข้อความ  
- เปลี่ยนการจัดรูปแบบข้อความในเฟรมข้อความ  
- เพิ่มฟอนต์ใหม่หรือจัดการฟอนต์ที่ใช้อยู่ในงานนำเสนอ  
- เปลี่ยนข้อความและการจัดรูปแบบข้อความ  
- สร้างหรือจัดการลิงก์ในข้อความ  
- รองรับแท็บและแท็บที่มีประสิทธิภาพ  
- เพิ่มเฟรมข้อความใหม่ไปยังรูปร่างเกือบทุกประเภท  
- เข้าถึงและแก้ไขระยะขอบและการตัดคำของเฟรมข้อความ  
- แก้ไขการเยื้องของย่อหน้า  

### **คุณสมบัติขั้นสูง**

- เข้าถึงวัตถุ OLE ที่ฝังอยู่ในงานนำเสนอ  
- ล็อกงานนำเสนอ  
- เพิ่มลายน้ำในงานนำเสนอ  
- ใช้แอนิเมชันกับรูปร่างภายในสไลด์  

## **คำถามที่พบบ่อย**

**รองรับระดับการปฏิบัติตามมาตรฐาน PDF สำหรับการจัดเก็บและการเข้าถึงหรือไม่ (PDF/A และ PDF/UA)?**  

ใช่ คุณสามารถบันทึกเป็น PDF ด้วย PDF/A-2a/2b/2u, PDF/A-3a/3b รวมถึง PDF/UA ได้โดยกำหนด [ตัวเลือกการส่งออก PDF](https://reference.aspose.com/slides/th/java/com.aspose.slides/pdfoptions/)  

**มีกลไกการทดแทนฟอนต์และการสนับสนุนฟอนต์ที่กำหนดเองเพื่อให้การแสดงผลถูกต้องหรือไม่?**  

ใช่ ไลบรารีให้คุณ [โหลดฟอนต์ที่กำหนดเอง](/slides/th/java/custom-font/) และ [กำหนดกฎการสำรอง](/slides/th/java/fallback-font/) เพื่อแทนที่ glyph ที่หายได้อย่างเชื่อถือได้ระหว่างการแปลงและการแสดงผล  

**ฉันสามารถตรวจจับไฟล์ที่มีการตั้งรหัสผ่านโดยไม่ต้องเปิดทั้งหมดได้หรือไม่?**  

ใช่ คุณสามารถ [ตรวจสอบงานนำเสนอ](/slides/th/java/examine-presentation/) เพื่อระบุว่าต้องใช้รหัสผ่านก่อนโหลดเอกสารเต็มหรือไม่  

**จำเป็นต้องใช้ Microsoft PowerPoint สำหรับการประมวลผลและการแปลงหรือไม่?**  

ไม่ Aspose.Slides ทำงานเป็นเอนจินอิสระ; ไม่ต้องการ PowerPoint บนเซิร์ฟเวอร์หรือเวิร์กสเตชั่น  

**ปลอดภัยหรือไม่ที่จะประมวลผลงานนำเสนอจากหลายเธรดพร้อมกัน?**  

ใช่ คุณสามารถประมวลผลเอกสารหลายไฟล์แบบขนานในเธรดแยกต่างหาก; เพียงหลีกเลี่ยงการใช้อินสแตนซ์งานนำเสนอเดียวกันพร้อมกัน [ข้ามเธรด](/slides/th/java/multithreading/)  

**แมโครจะถูกเก็บรักษาไว้และฉันสามารถจัดการ VBA ในไฟล์ PPTM/PPSM ได้หรือไม่?**  

ใช่ งานนำเสนอที่มีแมโคร [ได้รับการสนับสนุน](/slides/th/java/presentation-via-vba/) และคุณสามารถ [ตรวจสอบและจัดการโครงการ VBA]((https://reference.aspose.com/slides/th/java/com.aspose.slides/presentation/#getVbaProject--)) ในไฟล์เหล่านั้นได้  

**ฉันสามารถแปลง PDF หรือ HTML กลับเป็นสไลด์ PowerPoint ได้หรือไม่?**  

ใช่ คุณสามารถ [นำเข้าหน้า PDF หรือเนื้อหา HTML](/slides/th/java/import-presentation/) เพื่อสร้างหรือเติมสไลด์ในงานนำเสนอ  

**การส่งออกเป็น XPS รองรับหรือไม่และฉันสามารถควบคุมคุณภาพและเนื้อหาของผลลัพธ์ XPS ได้หรือไม่?**  

ใช่ มี [การส่งออกเป็น XPS](/slides/th/java/convert-powerpoint-to-xps/) และ [ตัวเลือกการบันทึก]((https://reference.aspose.com/slides/th/java/com.aspose.slides/xpsoptions/)) ที่ช่วยปรับคุณภาพและเนื้อหาที่รวมอยู่  

**ฉันสามารถแปลงสไลด์เป็นรูปภาพและควบคุมคุณภาพการส่งออกได้หรือไม่?**  

ใช่ สไลด์สามารถ [เรนเดอร์เป็น PNG, JPEG, GIF, BMP, TIFF](/slides/th/java/convert-powerpoint-to-png/) และรูปแบบอื่น ๆ พร้อมการควบคุมขนาดและคุณภาพที่ละเอียด  

**รองรับสูตรคณิตศาสตร์ (MathML/MathText) และสามารถส่งออกได้หรือไม่?**  

ใช่ คุณสามารถ [สร้างและแก้ไขข้อความคณิตศาสตร์](/slides/th/java/powerpoint-math-equations/) และ [ส่งออกสมการ](/slides/th/java/exporting-math-equations/) รวมถึงเป็น MathML  

**ไลบรารีจัดการการเข้ารหัสและรหัสผ่าน (เปิด, ตั้ง, ลบ) อย่างไร?**  

[รองรับ](/slides/th/java/password-protected-presentation/) การเปิดงานนำเสนอที่ถูกเข้ารหัส การตั้งหรือการลบรหัสผ่านสำหรับการเปิดและการแก้ไข และการตรวจสอบสถานะการป้องกันของไฟล์
