## **ภาพรวม**

บทความนี้อธิบายข้อบกพร่องด้านการให้ใบอนุญาตที่พบใน Aspose.Slides for Java เวอร์ชัน 23.2, 23.3, 23.4, 23.5, 23.6 และ 23.7 ข้อบกพร่องนี้ทำให้ใบอนุญาตถาวรกลายเป็นไม่สามารถใช้งานได้ชั่วคราวหลังจากการสมัครสมาชิกหมดอายุ

## **อาการ**

หลังจากการสมัครสมาชิกของใบอนุญาตถาวรของคุณหมดอายุ Aspose.Slides for Java เวอร์ชัน 23.2 - 23.7 อาจรายงานว่าใบอนุญาตหมดอายุและเปลี่ยนการทำงานทั้งหมดเป็นโหมดประเมินผล พฤติกรรมนี้ไม่ถูกต้องและไม่ได้ส่งผลต่อเวอร์ชันก่อนหน้า 23.2 หรือหลังจาก 23.7

## **การแก้ไข**

Aspose.Slides for Java ได้แก้ไขปัญหานี้และปล่อยเวอร์ชันที่อัปเดต (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) พร้อมการแก้ไข

หากคุณกำลังใช้หนึ่งในเวอร์ชันที่ได้รับผลกระทบของ Aspose.Slides for Java ในโครงการของคุณ โปรดเปลี่ยนไปใช้เวอร์ชันที่ได้รับการแก้ไข

หากคุณพบปัญหาด้านใบอนุญาตใด ๆ กับ Aspose.Slides for Java:

- ตรวจสอบเวอร์ชันที่คุณใช้และยืนยันว่าไม่ได้รับผลกระทบจากข้อบกพร่องที่อธิบายข้างต้น
- ดูเอกสาร [Aspose documentation](https://docs.aspose.com/slides/th/java/getting-started/) เพื่อรับขั้นตอนการแก้ไขปัญหาและปัญหาการให้ใบอนุญาตที่ทราบกัน
- ติดต่อ [Aspose support](https://forum.aspose.com/) เพื่อขอความช่วยเหลือเพิ่มเติม

## **ผลิตภัณฑ์และเวอร์ชันที่ได้รับผลกระทบ**

โปรดทราบว่าข้อบกพร่องนี้มีผลต่อเพียง Aspose.Slides for Java เวอร์ชัน 23.2 ถึง 23.7 เท่านั้น **ผลิตภัณฑ์ Aspose อื่น ๆ และเวอร์ชันอื่นของ Aspose.Slides ไม่ได้รับผลกระทบ**
