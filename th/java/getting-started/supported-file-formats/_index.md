## **ภาพรวม**

Aspose.Slides รองรับไฟล์งานนำเสนอจาก Microsoft PowerPoint 97 ถึง Office 365 รวมถึง Microsoft PowerPoint สำหรับ Mac บทความนี้แสดงรายการเวอร์ชัน PowerPoint ที่ไลบรารีสนับสนุนและให้ตารางรูปแบบไฟล์ที่สามารถโหลด บันทึก หรือทั้งสองอย่างได้

บทความนี้ยังตอบคำถามทั่วไปเกี่ยวกับการปฏิบัติตามมาตรฐาน PDF การฝังฟอนต์ ไฟล์ที่ป้องกันด้วยรหัสผ่าน ฟอนต์แบบกำหนดเอง การสำรองฟอนต์ และตัวเลือกการส่งออก XPS

## **เวอร์ชัน Microsoft PowerPoint ที่รองรับ**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint สำหรับ Mac
- Office 365

## **คำถามที่พบบ่อย**

**ฉันสามารถบันทึกงานนำเสนอเป็น PDF ที่ตรงตามมาตรฐานการเก็บรักษาและการเข้าถึง (PDF/A และ PDF/UA) ได้หรือไม่?**

ใช่ครับ Aspose.Slides รองรับการส่งออกเป็น PDF พร้อมระดับการปฏิบัติตามเช่น PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b รวมถึง PDF/UA ผ่านการตั้งค่า [compliance](https://reference.aspose.com/slides/th/java/com.aspose.slides/pdfoptions/#setCompliance-int-) ใน [PDF export options](https://reference.aspose.com/slides/th/java/com.aspose.slides/pdfoptions/)  

**ไลบรารีรองรับการฝังฟอนต์เมื่อส่งออกเป็น PDF พร้อมการควบคุมละเอียดว่าฟอนต์ใดจะถูกฝังหรือไม่?**

ใช่ครับ คุณสามารถควบคุมได้ว่าฟอนต์จะถูกฝังเต็มรูปแบบหรือเป็นแบบ subset (ฝังเฉพาะเกลฟที่ใช้) ระบุวิธีการจัดการฟอนต์ระบบทั่วไป และกำหนดพฤติกรรมของข้อความ ASCII ผ่าน [PDF export options](https://reference.aspose.com/slides/th/java/com.aspose.slides/pdfoptions/)  

**ฉันสามารถตรวจจับว่าไฟล์มีการป้องกันด้วยรหัสผ่านหรือไม่ ก่อนที่จะโหลดจริงได้หรือไม่?**

ใช่ครับ ด้วย [factory-based inspection API](https://reference.aspose.com/slides/th/java/com.aspose.slides/presentationfactory/) คุณสามารถสอบถามไฟล์งานนำเสนอเพื่อระบุว่ามีการป้องกันด้วยรหัสผ่านโดยไม่ต้องเปิดไฟล์เต็มรูปแบบ  

**มีกลไกการสำรองฟอนต์และการสนับสนุนฟอนต์แบบกำหนดเองหรือไม่?**

ใช่ครับ ไลบรารีสนับสนุนการ [loading](/slides/th/java/custom-font/) และ [embedding](/slides/th/java/embedded-font/) ฟอนต์แบบกำหนดเอง และให้กฎ [fallback](/slides/th/java/fallback-font/) เพื่อป้องกันการขาดเกลฟระหว่างการเรนเดอร์และการแปลง  

**ฉันสามารถส่งออกสไลด์เป็น XPS ได้หรือไม่ และมีตัวเลือกใดบ้างเพื่อปรับแต่งผลลัพธ์ XPS?**

ใช่ครับ การ [Export to XPS](/slides/th/java/convert-powerpoint-to-xps/) รองรับ และคุณสามารถปรับ [save options](https://reference.aspose.com/slides/th/java/com.aspose.slides/xpsoptions/) ที่เกี่ยวข้องเพื่อควบคุมคุณภาพและเนื้อหาของเอกสาร XPS.
