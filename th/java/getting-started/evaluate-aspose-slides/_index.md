## **Aspose.Slides การประเมิน**

คุณสามารถดาวน์โหลด Aspose.Slides เพื่อการประเมินได้อย่างง่ายดาย การดาวน์โหลดสำหรับการประเมินนั้นมีความเหมือนกับการดาวน์โหลดที่ซื้อไว้ เวอร์ชันการประเมินจะกลายเป็นลิขสิทธิ์เมื่อคุณเพิ่มโค้ดไม่กี่บรรทัดเพื่อใช้ลิขสิทธิ์

เวอร์ชันการประเมินของ Aspose.Slides (โดยไม่ได้ระบุลิขสิทธิ์) ให้ฟังก์ชันการทำงานของผลิตภัณฑ์เต็มรูปแบบ แต่จะใส่น้ำโลโก้การประเมินที่ด้านบนของเอกสารเมื่อเปิดหรือบันทึก และจำกัดไว้ที่หนึ่งสไลด์เมื่อดึงข้อความจากสไลด์พรีเซนเทชัน

## **คำถามที่พบบ่อย**

**ฉันสามารถทดสอบหลายพรีเซนเทชันพร้อมกันในหลายเธรดในโหมดการประเมินได้หรือไม่?**

ได้. คุณสามารถประมวลผลเอกสารต่างๆ ได้พร้อมกัน; คุณไม่ควรแชร์อ็อบเจ็กต์พรีเซนเทชันเดียวกัน [across threads](/slides/th/java/multithreading/). โหมดการประเมินไม่ได้มีผลต่อสิ่งนี้.

**ฉันจำเป็นต้องติดตั้ง Microsoft PowerPoint เพื่อประเมินไลบรารีบนเซิร์ฟเวอร์หรือใน CI หรือไม่?**

ไม่. Aspose.Slides เป็นเอนจินแบบสแตนด์อโลนและไม่ต้องการให้ติดตั้ง PowerPoint ไม่ว่าจะในโหมดการประเมินหรือการผลิต.

**ฉันสามารถทดสอบการแปลงจาก PPT/PPTX ไปเป็น PDF และรูปภาพอย่างเต็มที่ในโหมดการประเมินได้หรือไม่?**

ได้. [converters](/slides/th/java/convert-presentation/) ทำงาน; ผลลัพธ์จะมีน้ำโลโก้.

**ฉันสามารถใช้ลิขสิทธิ์ชั่วคราวสำหรับการทดสอบโหลดโดยไม่ต้องมีน้ำโลโก้ได้หรือไม่?**

ได้. ลิขสิทธิ์ชั่วคราวเป็นเวลา 30 วันจะลบข้อจำกัดของโหมดการประเมินและอนุญาตให้ทดสอบโดยไม่มีน้ำโลโก้.
