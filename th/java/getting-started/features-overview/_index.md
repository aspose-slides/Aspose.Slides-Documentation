## **แพลตฟอร์มที่รองรับ**
Aspose.Slides for Java รองรับแพลตฟอร์มการพัฒนาและการปรับใช้ที่เป็นที่นิยมมากที่สุด.

|**คุณลักษณะ**|**คำอธิบาย**|
| :- | :- |
|แอปพลิเคชันเดสก์ท็อป|Aspose.Slides for Java สามารถใช้พัฒนาแอปพลิเคชัน Windows Forms|
|แอปพลิเคชันเว็บระดับองค์กร|การใช้ Aspose.Slides for Java ช่วยสร้างแอปพลิเคชันเว็บที่มุ่งเป้า การสนับสนุนการใช้ Aspose.Slides for Java ร่วมกับ PHP ก็มีให้ด้วย|
|Linux/Unix|Aspose.Slides for Java เป็น API ที่ไม่ขึ้นกับแพลตฟอร์มและสามารถทำงานในสภาพแวดล้อม Linux และ Unix|

|**คุณลักษณะ**|**คำอธิบาย**|
| :- | :- |

|[PresentationML (PPTX, XML)](/slides/th/java/presentationml-pptx-xml/)|Aspose.Slides for Java รองรับการประมวลผลรูปแบบการนำเสนอ OOXML (หรือที่เรียกว่า PresentationML หรือ PPTX).|
|[PPT to PPTX conversion](/slides/th/java/ppt-to-pptx-conversion/)|Aspose.Slides for Java รองรับการแปลงจาก PPT เป็น PPTX.|
|[Portable Document Format (PDF)](/slides/th/java/developer-guide/)|รูปแบบไฟล์ที่รองรับสามารถส่งออกไปยังเอกสาร Adobe Portable Document Format (PDF) ด้วยวิธีเดียว.|

## **การเรนเดอร์การนำเสนอ**
Aspose .Slides for Java supports high fidelity rendering of slides in the presentations to various graphics formats:

|**คุณลักษณะ**|**คำอธิบาย**|
| :- | :- |
|รูปแบบภาพที่รองรับ|โดยใช้ Aspose.Slides for Java คุณจะสามารถเรนเดอร์ไม่เพียงแต่สไลด์การนำเสนอ แต่ยังรวมถึงภาพบนสไลด์ไปยังรูปแบบกราฟิกที่เป็นที่นิยม เช่น TIFF, PNG, BMP, JPEG, GIF และเมตาไฟล์.|
|รูปแบบ SVG|Aspose.Slides for Java มีวิธีการในตัวเพื่อส่งออกสไลด์การนำเสนอเป็นรูปแบบ Scalable Vector Graphics (SVG).|

## **FAQ**

**ฉันจำเป็นต้องติดตั้ง Microsoft PowerPoint บนเซิร์ฟเวอร์/PC เพื่อให้ไลบรารีทำงานหรือไม่?**

ไม่. PowerPoint ไม่จำเป็น; Aspose.Slides เป็นเครื่องยนต์อิสระสำหรับสร้าง, แก้ไข, แปลง, และเรนเดอร์การนำเสนอ.

**การทำงานของ multithreading เป็นอย่างไร? สามารถทำการประมวลผลแบบขนานได้หรือไม่?**

ปลอดภัยที่จะประมวลผลเอกสารต่าง ๆ ในเธรดที่แตกต่างกัน; วัตถุ [การนำเสนอ](https://reference.aspose.com/slides/th/java/com.aspose.slides/presentation/) เดียวกันต้องไม่ถูกใช้โดย [หลายเธรด](/slides/th/java/multithreading/) พร้อมกัน.

**รองรับรหัสผ่านไฟล์และการเข้ารหัสหรือไม่?**

ใช่. [คุณสามารถ](/slides/th/java/password-protected-presentation/) เปิดการนำเสนอที่เข้ารหัส, ตั้งหรือเอารหัสผ่านสำหรับการเปิดและการเขียนออก, และตรวจสอบสถานะการป้องกัน.

**ฉันต้องกังวลเกี่ยวกับแพ็กเกจฟอนต์ในคอนเทนเนอร์ Linux หรือไม่?**

ใช่. แนะนำให้ติดตั้งแพ็กเกจฟอนต์ทั่วไปและ/หรือระบุ [ไดเรกทอรีฟอนต์](/slides/th/java/custom-font/) อย่างชัดเจนในแอปพลิเคชันของคุณเพื่อหลีกเลี่ยงการแทนที่ที่ไม่คาดคิด.

**มีข้อจำกัดในรุ่นทดลองหรือไม่?**

ใน [โหมดประเมินผล](/slides/th/java/licensing/) มีการเพิ่มลายน้ำในผลลัพธ์และมีข้อจำกัดบางประการ; มี [ใบอนุญาตชั่วคราว 30‑วัน](https://purchase.aspose.com/temporary-license/) ให้ใช้สำหรับการทดสอบเต็มคุณลักษณะ.

**รองรับการนำเข้ารูปแบบภายนอกเข้าสู่การนำเสนอ (PDF/HTML → PPTX) หรือไม่?**

ใช่. คุณสามารถเพิ่ม [หน้า PDF และเนื้อหา HTML](/slides/th/java/import-presentation/) เข้าไปในการนำเสนอ, ทำให้เป็นสไลด์.
