## **เกี่ยวกับ TIFF**

รูปแบบไฟล์รูปที่ติดแท็ก([Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF)) ซึ่งเป็นที่รู้จักสำหรับการเก็บหลายภาพไว้ในเอกสารเดียว ได้รับการสร้างขึ้นโดย Aldus แรกเริ่ม รูปแบบนี้ได้รับการสนับสนุนอย่างกว้างขวางโดยแอปพลิเคชันการสแกน, แฟกซ์ และการจัดการภาพอื่น ๆ

## **TIFF ใน Aspose.Slides for Java**

เอกสารใด ๆ ที่สามารถโหลดใน Aspose.Slides for Java ก็สามารถแปลงเป็นเอกสาร TIFF ได้โดยตรงโดย Aspose.Slides for Java โดยไม่ต้องพึ่งพาองค์ประกอบของบุคคลที่สามเพิ่มเติม นอกจากนี้ คุณยังสามารถกำหนดขนาดของภาพในเอกสาร TIFF ที่ได้ตามต้องการ คุณสามารถค้นหาข้อมูลเกี่ยวกับการส่งออกเอกสารการนำเสนอเป็นเอกสาร TIFF ผ่าน Aspose.Slides for Java ใน [หัวข้อนี้](/slides/th/java/converting-a-presentation/).
