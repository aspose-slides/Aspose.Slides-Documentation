การแปลง PPT เป็น PPTX ได้รับการสนับสนุนจาก Aspose.Slides for Java คุณสมบัติการนำเสนอส่วนใหญ่ - สไลด์มาสเตอร์, โครงสร้างและอื่น ๆ - จะถูกเก็บไว้เมื่อแปลงจากรูปแบบหนึ่งเป็นอีกรูปแบบหนึ่ง แต่ยังมี [การจำกัดบางประการ](/slides/th/java/ppt-to-pptx-conversion/).

{{% /alert %}} 
## **คุณลักษณะที่รองรับในการแปลง**
Aspose.Slides for Java ให้การสนับสนุนบางส่วนสำหรับการแปลงรูปแบบไฟล์ PPT เป็น PPTX ความสนับสนุนการแปลงเพิ่งถูกนำเข้ามาใน Aspose.Slides for Java ดังนั้นจึงมีข้อจำกัดบางประการและเหมาะสมที่สุดสำหรับการนำเสนอแบบง่าย ข้อได้เปรียบหลักที่ Aspose.Slides for Java ให้เมื่อแปลง PPT ไปเป็น PPTX คือความง่ายในการใช้ API เพื่อดูตัวอย่างโค้ด อ่านเพิ่มเติมเกี่ยวกับ [การแปลง PPT เป็น PPTX](). ด้านล่างรายการอธิบายว่าคุณลักษณะใดบ้างที่รองรับและไม่รองรับสำหรับการแปลง PPT เป็น PPTX.

## **คุณลักษณะที่รองรับ**
คุณลักษณะที่ต่อไปนี้ได้รับการรองรับสำหรับการแปลง:

- การแปลงโครงสร้างของมาสเตอร์, เลย์เอาต์และสไลด์
- การแปลงแผนภูมิ
- รูปร่างกลุ่ม
- การแปลง autoshapes รวมถึงสี่เหลี่ยมผืนผ้าและวงรี
- รูปร่างที่มีเรขาคณิตที่กำหนดเอง
- พื้นผิวและสไตล์การเติมรูปภาพสำหรับ autoshapes
- การแปลง placeholder
- การแปลงเส้นและ polyline
- รูปแบบเส้นและการเติม
- สไตล์การเติมแบบไล่สี
- เฟรม OLE, ตาราง, เฟรมวิดีโอและเสียง ฯลฯ
- คุณสมบัติการเคลื่อนไหวและการแสดงสไลด์
- การแปลงข้อความในกรอบข้อความและตัวจัดเก็บข้อความ
