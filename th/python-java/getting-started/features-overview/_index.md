## **แพลตฟอร์มที่สนับสนุน**
Aspose.Slides for Python via Java เป็น API ที่ไม่มีการผูกมัดกับแพลตฟอร์มใด ๆ สามารถใช้ได้บนทุกแพลตฟอร์ม (Windows, Linux, macOS ฯลฯ) ที่ติดตั้ง Python 3.7 หรือสูงกว่า และต้องการ JDK 8 หรือสูงกว่า

## **รูปแบบไฟล์และการแปลง**
Aspose.Slides for Python via Java รองรับรูปแบบเอกสาร PowerPoint ส่วนใหญ่ และสามารถส่งออกเป็นรูปแบบไฟล์ที่นิยมใช้และแชร์กันในองค์กรได้ ดูรายละเอียดต่อไปนี้:

|**คุณลักษณะ**|**คำอธิบาย**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Python via Java ให้การประมวลผลที่เร็วที่สุดสำหรับรูปแบบเอกสารการนำเสนอนี้|
|PPT to PPTX conversion|Aspose.Slides for Python via Java รองรับการแปลงจาก PPT เป็น PPTX|
|Portable Document Format (PDF)|คุณสามารถส่งออกไฟล์รูปแบบที่รองรับทั้งหมดไปเป็นเอกสาร Adobe Portable Document Format (PDF) ด้วยวิธีเดียว|
|XML Parser Specification (XPS)|คุณสามารถส่งออกไฟล์รูปแบบที่รองรับทั้งหมดไปเป็นเอกสาร XML Parser Specification (XPS) ด้วยวิธีเดียว|
|Tagged Image File Format (TIFF)|คุณสามารถส่งออกไฟล์รูปแบบการนำเสนอที่รองรับทั้งหมดไปเป็น Tagged Image File Format (TIFF)|
|PPTX To HTML Conversion|Aspose.Slides for Python via Java รองรับการแปลง PresentationEx เป็นรูปแบบ HTML|

## **การเรนเดอร์และการพิมพ์**
Aspose.Slides for Python via Java รองรับการเรนเดอร์สไลด์ที่มีความแม่นยำสูงเป็นรูปแบบกราฟิกต่าง ๆ ดูรายละเอียดต่อไปนี้:

|**คุณลักษณะ**|**คำอธิบาย**|
| :- | :- |
|Java Supported Image Formats|ด้วย Aspose.Slides for Python via Java คุณสามารถเรนเดอร์สไลด์และภาพบนสไลด์เป็นรูปแบบกราฟิกที่ Java รองรับทั้งหมด เช่น TIFF, PNG, BMP, JPEG, GIF|
|SVG Format|Aspose.Slides for Python via Java ยังมีเมธอดในตัวที่ช่วยให้คุณส่งออกสไลด์เป็นรูปแบบ Scalable Vector Graphics (SVG)|
|Presentation Printing|เวอร์ชันล่าสุดของ Aspose.Slides for Python via Java มีเมธอดการพิมพ์ในตัวพร้อมตัวเลือกต่าง ๆ|

## **คุณลักษณะของเนื้อหา**
Aspose.Slides for Python via Java ให้คุณเข้าถึง แก้ไข หรือสร้างส่วนต่าง ๆ ของเอกสารการนำเสนอเกือบทั้งหมด ดูรายละเอียดต่อไปนี้:

|**คุณลักษณะ**|**คำอธิบาย**|
| :- | :- |
|Master Slides|Master Slides กำหนดเค้าโครงของสไลด์ปกติ Aspose.Slides for Python via Java ให้คุณเข้าถึงและแก้ไข Master Slides ของเอกสารการนำเสนอ|
|Normal Slides|ด้วย Aspose.Slides for Python via Java คุณสามารถสร้างสไลด์ใหม่หลายประเภท รวมทั้งเข้าถึงและแก้ไขสไลด์ที่มีอยู่ในงานนำเสนอ|
|Cloning / Copying Slides|เมธอดในตัวของ Aspose.Slides for Python via Java ช่วยให้คุณคัดลอกหรือทำสำเนาสไลด์ที่มีอยู่ภายในงานนำเสนอได้ รวมถึงการใช้สไลด์ที่คัดลอกหรือทำสำเนาไปยังงานนำเสนออื่น เนื่องจากสไลด์รับมรดกเค้าโครงจาก Master Slide เมธอดคัดลอกในตัวจะคัดลอก Master ให้อัตโนมัติ|
|Managing Slides sections|เมธอดสำหรับจัดสไลด์ในส่วนต่าง ๆ ภายในงานนำเสนอ|
|Place Holders and Text Holders|คุณสามารถเข้าถึงตัวครอบตำแหน่งและตัวครอบข้อความในสไลด์ได้ นอกจากนี้ยังสามารถสร้างสไลด์พร้อมตัวครอบข้อความตั้งแต่เริ่มต้นด้วยเมธอดที่เหมาะสม|
|Header and Footers|Aspose.Slides for Python via Java ช่วยจัดการส่วนหัว/ส่วนท้ายในสไลด์|
|Notes in Slides|ด้วย Aspose.Slides for Python via Java คุณสามารถเข้าถึงและแก้ไขโน้ตที่แนบกับสไลด์ รวมถึงเพิ่มโน้ตใหม่|
|Finding a Shape|คุณสามารถค้นหา Shape เฉพาะจากสไลด์โดยใช้ข้อความแทนที่เชื่อมโยงกับ Shape นั้น|
|Backgrounds|Aspose.Slides for Python via Java ให้คุณทำงานกับพื้นหลังที่แนบกับ Master หรือสไลด์ปกติในงานนำเสนอ|
|Text Boxes|สามารถสร้าง Text Box ตั้งแต่เริ่มต้น เข้าถึง Text Box ที่มีอยู่แล้ว และแก้ไขข้อความโดยไม่สูญเสียรูปแบบข้อความเดิม|
|Rectangle Shapes|คุณสามารถสร้างหรือแก้ไขรูปสี่เหลี่ยมผืนผ้าด้วย Aspose.Slides for Python via Java|
|Poly Line Shapes|คุณสามารถสร้างหรือแก้ไขรูปโพลีไลน์ด้วย Aspose.Slides for Python via Java|
|Ellipse Shapes|คุณสามารถสร้างหรือแก้ไขรูปวงรีด้วย Aspose.Slides for Python via Java|
|Group Shapes|Aspose.Slides for Python via Java รองรับ Group Shapes|
|Auto Shapes|Aspose.Slides for Python via Java รองรับ Auto Shapes|
|SmartArt|Aspose.Slides for Python via Java ให้การสนับสนุน SmartArt ใน MS PowerPoint|
|Charts|Aspose.Slides for Python via Java ให้การสนับสนุน Chart ของ MS Office ใน PowerPoint|
|Shapes Serialization|Aspose.Slides for Python via Java รองรับรูปร่างหลายประเภท หากไม่รองรับรูปแบบใดรูปแบบหนึ่ง คุณสามารถใช้วิธีการ serialization เพื่อทำให้สามารถบันทึกรูปแบบนั้นจากสไลด์ที่มีอยู่แล้ว แล้วนำไปใช้ต่อได้ตามต้องการ|
|Picture Frames|คุณสามารถจัดการรูปภาพใน Picture Frames ด้วย Aspose.Slides for Python via Java|
|Audio Frames|คุณสามารถลิงก์หรือฝังไฟล์เสียงใน Audio Frames บนสไลด์ด้วย Aspose.Slides for Python via Java|
|Video Frames|คุณสามารถจัดการไฟล์วิดีโอใน Video Frames ได้ Aspose.Slides for Python via Java ยังรองรับวิดีโอแบบลิงก์และฝัง|
|OLE Frame|คุณสามารถจัดการ OLE Objects ใน OLE Frames ด้วย Aspose.Slides for Python via Java|

|ActiveX Controls|รองรับการควบคุม ActiveX|

|Text Scanning|คุณสามารถสแกนข้อความในงานนำเสนอระดับสไลด์หรือระดับงานนำเสนอด้วยเมธอดสแกนในตัว|
|Animations|คุณสามารถใช้แอนิเมชันบนรูปร่างได้|

|**คุณลักษณะ**|**คำอธิบาย**|
| :- | :- |
|Text Formatting|<p>ใน Aspose.Slides for Python via Java คุณสามารถจัดการข้อความผ่าน Text Frame ที่แนบกับรูปร่าง ดังนั้นคุณสามารถจัดรูปแบบข้อความได้โดยใช้ Paragraph และ Portion ที่แนบกับ Text Frame เหล่านั้น ซึ่งองค์ประกอบข้อความเหล่านี้สามารถจัดรูปแบบได้ผ่าน Aspose.Slides for Python via Java</p><p>- ประเภทฟอนต์</p><p>- ขนาดฟอนต์</p><p>- สีฟอนต์</p><p>- โทนสีฟอนต์</p><p>- การจัดตำแหน่งย่อหน้า</p><p>- จุดบูลเล็ตย่อหน้า</p><p>- แนวตั้งย่อหน้า</p>|
|Shape Formatting|<p>ใน Aspose.Slides for Python via Java รูปแบบพื้นฐานของสไลด์คือ Shape คุณสามารถจัดรูปแบบองค์ประกอบ Shape เหล่านี้ด้วย Aspose.Slides for Python via Java:</p><p>- ตำแหน่ง</p><p>- ขนาด</p><p>- เส้น</p><p>- การเติม (รวมถึง Pattern, Gradient, Solid)</p><p>- ข้อความ</p><p>- ภาพ</p>|
