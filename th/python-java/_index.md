**ยินดีต้อนรับสู่ Aspose.Slides สำหรับ Python ผ่าน Java**

![โลโก้ผลิตภัณฑ์ Aspose.Slides สำหรับ Python ผ่าน Java](aspose_slides-for-python-via-java.png)

Aspose.Slides for Python via Java เป็นไลบรารีคลาสที่ทำให้แอปพลิเคชันของคุณสามารถอ่านและเขียนเอกสาร PowerPoint® ได้โดยไม่ต้องใช้ Microsoft PowerPoint®.

Aspose.Slides for Python via Java เป็นส่วนประกอบแรกและ唯一ที่ให้ฟังก์ชันการจัดการเอกสาร PowerPoint®.

Aspose.Slides for Python via Java มีคุณสมบัติหลักมากมาย เช่น การจัดการข้อความ รูปร่าง ตารางและแอนิเมชัน การเพิ่มเสียงและวิดีโอลงในสไลด์ การพรีวิวสไลด์ การส่งออกสไลด์เป็นรูปแบบ SVG, PDF และอื่น ๆ
