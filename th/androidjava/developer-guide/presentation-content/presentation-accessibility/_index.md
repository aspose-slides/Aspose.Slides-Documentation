## **Overview**

การเข้าถึงการนำเสนอทำให้ผู้ใช้เทคโนโลยีช่วยเหลือ—เช่นโปรแกรมอ่านหน้าจอ, หน้าจอเบรลล์, หรือการนำทางด้วยคีย์บอร์ดเท่านั้น—สามารถเข้าใจและนำทางสไลด์ของคุณได้อย่างมีประสิทธิภาพเทียบเท่ากับผู้ชมที่มองเห็นและใช้เมาส์ การปฏิบัติที่ดีเน้นลำดับการอ่านที่ชัดเจน, ข้อความแทนที่มีความหมายสำหรับภาพที่ให้ข้อมูล, ความคอนทราสต์สีที่เพียงพอ, ตัวอักษรที่อ่านง่าย, ข้อความลิงก์ที่อธิบายชัดเจน, และหลีกเลี่ยงการสื่อความหมายด้วยสีหรือตำแหน่งเพียงอย่างเดียว เมื่อการเข้าถึงถูกวางแผนตั้งแต่ต้น ผลลัพธ์จะเป็นโครงสร้างที่สะอาดขึ้น, ภาพที่สอดคล้องกันมากขึ้น, และเนื้อหาที่เข้าถึงผู้ชมทุกคนโดยไม่มีวิธีแก้ไขเพิ่มเติม

## **Mark as Decorative**

ทำเครื่องหมายว่าเป็นแบบตกแต่งใช้เพื่อระบุภาพที่เป็นเพียงองค์ประกอบตกแต่งเท่านั้น เพื่อให้โปรแกรมอ่านหน้าจอข้ามผ่าน ซึ่งลดความรบกวนและทำให้โฟกัสอยู่ที่เนื้อหาที่มีความหมาย ให้ใช้กับพื้นหลัง, ลวดลาย, และช่องว่าง—ไม่ใช้กับแผนภูมิ, ไอคอน, หรือภาพที่สื่อข้อมูล Aspose.Slides เปิดเผยแฟล็กนี้สำหรับการตรวจจับและตรวจสอบ ทำให้สามารถตรวจสอบการเข้าถึงโดยอัตโนมัติและทำความสะอาดได้

![Mark as Decorative](mark_as_decorative.png)

```java
Presentation presentation = new Presentation("sample.pptx");
try {
    IShape shape = presentation.getSlides().get_Item(0).getShapes().get_Item(0);
    System.out.println("Is shape decorative: " + shape.isDecorative());
} finally {
    presentation.dispose();
}
```
