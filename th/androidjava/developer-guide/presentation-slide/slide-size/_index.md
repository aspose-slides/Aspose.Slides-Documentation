## **บทนำ**

Aspose.Slides มีเครื่องมือครบถ้วนเพื่อปรับขนาดสไลด์และอัตราส่วนภาพในงานนำเสนอ PowerPoint ซึ่งสำคัญทั้งสำหรับการพิมพ์และการแสดงผลบนหน้าจอ  

ขนาดสไลด์และอัตราส่วนที่นิยม:

- **Standard (อัตราส่วน 4:3)**: เหมาะสำหรับหน้าจอและอุปกรณ์เก่า
- **Widescreen (อัตราส่วน 16:9)**: แนะนำสำหรับโปรเจคเตอร์และจอแสดงผลสมัยใหม่

ตรวจสอบให้สไลด์ทั้งหมดมีความสอดคล้องกันโดยใช้ขนาดสไลด์และอัตราส่วนเดียวกันสำหรับทุกสไลด์ เพื่อให้ได้ผลลัพธ์ที่ดีที่สุด ตั้งค่าขนาดสไลด์ตั้งแต่เริ่มกระบวนการสร้างงานนำเสนอเพื่อหลีกเลี่ยงปัญหา  

{{% alert color="primary" %}} 
โดยค่าเริ่มต้น งานนำเสนอที่สร้างด้วย Aspose.Slides จะใช้ค่าอัตราส่วน 4:3 มาตรฐาน
{{% /alert %}}

## **เปลี่ยนขนาดสไลด์ในงานนำเสนอ**

โค้ดตัวอย่างนี้แสดงวิธีการเปลี่ยนขนาดสไลด์ในงานนำเสนอด้วย Java โดยใช้ Aspose.Slides:

```java
Presentation pres = new Presentation("pres-4x3-aspect-ratio.pptx");
try {
    pres.getSlideSize().setSize(SlideSizeType.OnScreen16x9, SlideSizeScaleType.DoNotScale);
    pres.save("pres-4x3-aspect-ratio.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```

## **ระบุขนาดสไลด์ที่กำหนดเองในงานนำเสนอ**

หากคุณพบว่าขนาดสไลด์ทั่วไป (4:3 และ 16:9) ไม่เหมาะกับงานของคุณ คุณอาจเลือกใช้ขนาดสไลด์ที่กำหนดเฉพาะหรือต่างออกไป ตัวอย่างเช่น หากคุณต้องการพิมพ์สไลด์ขนาดเต็มจากงานนำเสนอบนรูปแบบหน้ากระดาษที่กำหนดเอง หรือหากคุณต้องการแสดงงานนำเสนอบนหน้าจอประเภทต่างๆ การตั้งค่าขนาดสไลด์แบบกำหนดเองจะเป็นประโยชน์สำหรับคุณ  

โค้ดตัวอย่างนี้แสดงวิธีใช้ Aspose.Slides สำหรับ Android ผ่าน Java เพื่อกำหนดขนาดสไลด์ที่กำหนดเองสำหรับงานนำเสนอใน Java:

```java
Presentation pres = new Presentation("pres.pptx");
try {
    pres.getSlideSize().setSize(780, 540, SlideSizeScaleType.DoNotScale); // ขนาดกระดาษ A4
    pres.save("pres-a4-slide-size.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```

## **จัดการเนื้อหาสไลด์หลังการปรับขนาด**

หลังจากที่คุณเปลี่ยนขนาดสไลด์ของงานนำเสนอ เนื้อหาบนสไลด์ (เช่น รูปภาพหรือวัตถุ) อาจเกิดความบิดเบี้ยวขึ้น โดยค่าเริ่มต้น วัตถุต่างๆ จะถูกปรับขนาดอัตโนมัติเพื่อให้พอดีกับขนาดสไลด์ใหม่ อย่างไรก็ตามเมื่อเปลี่ยนขนาดสไลด์ของงานนำเสนอ คุณสามารถกำหนดการตั้งค่าที่บ่งบอกว่า Aspose.Slides จะจัดการกับเนื้อหาในสไลด์อย่างไร  

ขึ้นอยู่กับสิ่งที่คุณต้องการทำหรือบรรลุ คุณสามารถใช้การตั้งค่าเหล่านี้ได้ตามต้องการ:

- `DoNotScale`

  หากคุณไม่ต้องการให้วัตถุบนสไลด์ถูกปรับขนาด ให้ใช้การตั้งค่านี้

- `EnsureFit`

  หากคุณต้องการปรับขนาดให้เล็กลงและต้องการให้ Aspose.Slides ปรับขนาดวัตถุบนสไลด์ให้พอดีกับสไลด์ขนาดเล็ก (เพื่อหลีกเลี่ยงการสูญเสียเนื้อหา) ให้ใช้การตั้งค่านี้

- `Maximize`

  หากคุณต้องการปรับขนาดให้ใหญ่ขึ้นและต้องการให้ Aspose.Slides ขยายวัตถุบนสไลด์ให้เป็นสัดส่วนกับขนาดสไลด์ใหม่ ให้ใช้การตั้งค่านี้

โค้ดตัวอย่างนี้แสดงวิธีใช้การตั้งค่า `Maximize` เมื่อต้องการเปลี่ยนขนาดสไลด์ของงานนำเสนอ:

```java
Presentation pres = new Presentation("pres.pptx");
try {
    pres.getSlideSize().setSize(SlideSizeType.Ledger, SlideSizeScaleType.Maximize);
} finally {
    if (pres != null) pres.dispose();
}
```

## **คำถามที่พบบ่อย**

**ฉันสามารถกำหนดขนาดสไลด์ที่กำหนดเองโดยใช้หน่วยอื่นนอกจากนิ้ว (เช่น พอยท์หรือมิลลิเมตร) ได้หรือไม่?**

ใช่ Aspose.Slides ใช้หน่วย points ภายใน โดย 1 point เท่ากับ 1/72 นิ้ว คุณสามารถแปลงหน่วยใดก็ได้ (เช่น มิลลิเมตรหรือเซนติเมตร) ไปเป็น points แล้วใช้ค่าที่แปลงแล้วเพื่อกำหนดความกว้างและความสูงของสไลด์  

**ขนาดสไลด์ที่กำหนดเองขนาดใหญ่มากจะส่งผลต่อประสิทธิภาพและการใช้หน่วยความจำระหว่างการเรนเดอร์หรือไม่?**

ใช่ ขนาดสไลด์ที่ใหญ่ขึ้น (เป็น points) ร่วมกับสเกลการเรนเดอร์ที่สูงกว่าจะทำให้ใช้หน่วยความจำเพิ่มขึ้นและเวลาในการประมวลผลยาวนานขึ้น ควรกำหนดขนาดสไลด์ให้เหมาะสมและปรับสเกลการเรนเดอร์ตามความจำเป็นเพื่อให้ได้คุณภาพผลลัพธ์ที่ต้องการ  

**ฉันสามารถกำหนดขนาดสไลด์ที่ไม่มาตรฐานหนึ่งขนาดและจากนั้นรวมสไลด์จากงานนำเสนอที่มีขนาดต่างกันได้หรือไม่?**

คุณไม่สามารถ [merge presentations](/slides/th/androidjava/merge-presentation/) ในขณะที่มีขนาดสไลด์ต่างกัน — ก่อนอื่นให้ปรับขนาดงานนำเสนอหนึ่งให้ตรงกับอีกงานหนึ่ง เมื่อเปลี่ยนขนาดสไลด์ คุณสามารถเลือกวิธีจัดการเนื้อหาที่มีอยู่ผ่านตัวเลือก [SlideSizeScaleType](https://reference.aspose.com/slides/th/androidjava/com.aspose.slides/slidesizescaletype/) หลังจากปรับขนาดให้ตรงกันแล้ว คุณสามารถรวมสไลด์โดยคงรูปแบบไว้ได้  

**ฉันสามารถสร้างรูปย่อสำหรับรูปทรงแต่ละอันหรือพื้นที่เฉพาะของสไลด์ได้หรือไม่ และรูปย่อจะคงขนาดสไลด์ใหม่หรือไม่?**

ใช่ Aspose.Slides สามารถเรนเดอร์รูปย่อสำหรับ [entire slides](https://reference.aspose.com/slides/th/androidjava/com.aspose.slides/slide/#getImage-com.aspose.slides.IRenderingOptions-float-float-) รวมถึงสำหรับ [selected shapes](https://reference.aspose.com/slides/th/androidjava/com.aspose.slides/shape/#getImage-int-float-float-) ผลลัพธ์ภาพสะท้อนขนาดสไลด์และอัตราส่วนภาพปัจจุบัน ทำให้ได้กรอบและรูปทรงที่สม่ำเสมอ.
