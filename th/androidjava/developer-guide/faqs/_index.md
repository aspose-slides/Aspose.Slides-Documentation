## **ภาพรวม**

FAQ นี้ให้คำตอบสำหรับคำถามทั่วไปเกี่ยวกับ Aspose.Slides รวมถึงรูปแบบไฟล์ที่รองรับ การจัดการข้อยกเว้นเมื่อทำงานกับงานนำเสนอที่มีขนาดใหญ่ การเปลี่ยนขนาดสไลด์ การแสดงตัวอย่างสไลด์ การดึงข้อความจากงานนำเสนอ การจัดรูปแบบเส้นขอบตาราง การวางรูปภาพ และการแก้ไขปัญหาเกี่ยวกับแบบอักษรเมื่อแปลงงานนำเสนอเป็น PDF หรือภาพ

## **รูปแบบไฟล์ที่รองรับ**

**Q: Aspose.Slides for Android via Java รองรับรูปแบบไฟล์อะไรบ้าง?**

**A**: Aspose.Slides for Android via Java รองรับรูปแบบไฟล์ที่อธิบายไว้ใน [รูปแบบไฟล์ที่รองรับ](/slides/th/androidjava/supported-file-formats/).

## **ข้อยกเว้น**

**Q: ฉันได้รับข้อยกเว้นหน่วยความจำไม่พอขณะโหลดไฟล์ PPT ขนาดใหญ่ที่มีรูปภาพ มีข้อจำกัดขนาดไฟล์ใน Aspose.Slides หรือไม่?**

**A**: ไม่มีสูตรเฉพาะสำหรับคำนวณขนาดงานนำเสนอที่ Aspose.Slides รองรับ ควรมีพื้นที่เพียงพอเพื่อเก็บโครงสร้างงานนำเสนอทั้งหมดและรูปภาพในหน่วยความจำ โดยปกติรูปภาพในหน่วยความจำจะใช้พื้นที่มากกว่าที่เก็บบนฮาร์ดดิสก์ โดยเฉพาะเมื่อติดผลกระทบเพิ่มเติม

โดยทั่วไป Aspose.Slides for Android via Java สามารถจัดการไฟล์งานนำเสนอขนาดประมาณ 300 MB บนเซิร์ฟเวอร์ที่มี RAM 4 GB ได้อย่างง่ายดาย

## **การทำงานกับสไลด์**

**Q: ฉันสามารถเปลี่ยนขนาดของสไลด์ในงานนำเสนอได้หรือไม่?**

**A**: คุณสามารถใช้เมธอด `getSlideSize` ที่เปิดให้ใช้โดยคลาส [Presentation](https://reference.aspose.com/slides/th/androidjava/com.aspose.slides/presentation/) เพื่อกำหนดขนาดของสไลด์ในงานนำเสนอได้

**Q: มีวิธีกำหนดสไลด์ที่มีขนาดต่างกันในงานนำเสนอหรือไม่?**

**A**: เนื่องจากขนาดของสไลด์ถูกกำหนดระดับงานนำเสนอในเอกสาร Microsoft PowerPoint จึงไม่มีวิธีทำเช่นนั้น

**Q: Aspose.Slides for Android via Java รองรับการแสดงตัวอย่างสไลด์ก่อนบันทึกหรือไม่?**

**A**: คุณสามารถแปลงสไลด์ของงานนำเสนอเป็นภาพและใช้ภาพเหล่านี้เพื่อแสดงตัวอย่างสไลด์ได้

## **การทำงานกับข้อความ**

**Q: สามารถดึงข้อความทั้งหมดจากงานนำเสนอได้หรือไม่?**

**A**: Aspose.Slides for Android via Java มีคลาส [SlideUtil](https://reference.aspose.com/slides/th/androidjava/com.aspose.slides/slideutil/) ที่ให้เมธอดต่าง ๆ สำหรับดึงข้อความทั้งหมดจากงานนำเสนอ

**Q: ทำไมขนาดย่อหน้าจึงแตกต่างกันระหว่าง PC และ Android?**

**A**: การคำนวณขนาดย่อหน้าพิจารณาจากการคำนวณขนาดข้อความที่แสดงย่อหน้านั้น ขนาดข้อความคำนวณจากเมตริกของแบบอักษรที่ระบุในงานนำเสนอ PowerPoint หากแบบอักษรที่ระบุไม่มีอยู่ จะถูกแทนที่ด้วยแบบอักษรที่ใกล้เคียงที่สุด แต่แบบอักษรนั้นมีเมตริกที่ต่างจากต้นฉบับ ดังนั้นการคำนวณขนาดย่อหน้าในระบบต่าง ๆ จะให้ผลลัพธ์แตกต่างกันขึ้นอยู่กับชุดแบบอักษรที่ติดตั้งไว้ เพื่อให้ได้ผลลัพธ์เดียวกันบนระบบปฏิบัติการต่าง ๆ คุณต้องติดตั้งแบบอักษรเดียวกันบนระบบหรือโหลดแบบอักษรในช่วงเวลาการทำงานเป็น [external fonts](/slides/th/androidjava/custom-font/)

## **การจัดรูปแบบและรูปภาพ**

**Q: ฉันจะตั้งค่าสีของเส้นขอบตารางได้อย่างไร?**

**A**: คุณสามารถเปลี่ยนสีของเส้นขอบตารางทั้งหมดหรือเพียงเส้นขอบรอบตารางโดยรวมได้ สำหรับการเปลี่ยนเส้นขอบทั้งหมด ให้ใช้เมธอด `getCellFormat` จากอินเตอร์เฟส [ICell](https://reference.aspose.com/slides/th/androidjava/com.aspose.slides/icell/) สำหรับเส้นขอบของตารางทั้งหมด ให้วนผ่านเซลล์และเปลี่ยนสีของเส้นขอบภายนอก

**Q: Aspose.Slides for Android via Java ใช้หน่วยวัดอะไรในการวางรูปภาพ?**

**A**: พิกัดและขนาดของรูปทรงทั้งหมดบนสไลด์วัดเป็นหน่วย points (72 dpi)

## **การทำงานกับแบบอักษร**

**Q: เมื่อแปลง PPT เป็น PDF หรือภาพ ทำไมแบบอักษรในเอกสารผลลัพธ์ถึงแตกต่างกัน?**

**A**: ปัญหานี้อาจบ่งชี้ว่าแบบอักษรที่ใช้ในงานนำมายังไม่มีอยู่ในระบบปฏิบัติการที่รันโค้ด คุณควรติดตั้งแบบอักษรบนระบบปฏิบัติการหรือโหลดแบบอักษรเป็นแบบอักษรภายนอกโดยใช้คลาส [FontsLoader](https://reference.aspose.com/slides/th/androidjava/com.aspose.slides/fontsloader/) ตามตัวอย่างด้านล่าง:
```java
String[] folders = new String[] { "path_to_a_folder_with_fonts" };
FontsLoader.loadExternalFonts(folders);
```
