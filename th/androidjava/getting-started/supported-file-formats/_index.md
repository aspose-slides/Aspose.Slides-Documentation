## **ภาพรวม**

Aspose.Slides รองรับไฟล์พรีเซนเทชันจาก Microsoft PowerPoint 97 ถึง Office 365 รวมถึง Microsoft PowerPoint สำหรับ Mac บทความนี้แสดงเวอร์ชัน PowerPoint ที่ไลบรารีรองรับและให้ตารางรูปแบบไฟล์ที่สามารถโหลด บันทึก หรือทั้งสองอย่างได้

บทความนี้ยังตอบคำถามทั่วไปเกี่ยวกับความสอดคล้องของ PDF การฝังฟอนต์ ไฟล์ที่มีการป้องกันด้วยรหัสผ่าน ฟอนต์แบบกำหนดเอง การสำรองฟอนต์ และตัวเลือกการส่งออก XPS

## **เวอร์ชัน Microsoft PowerPoint ที่รองรับ**

## **FAQ**

**ฉันสามารถบันทึกการนำเสนอเป็น PDF ที่ตรงตามมาตรฐานการจัดเก็บและความสามารถในการเข้าถึง (PDF/A และ PDF/UA) ได้หรือไม่?**

ใช่ Aspose.Slides รองรับการส่งออกเป็น PDF พร้อมระดับความสอดคล้องเช่น PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b และ PDF/UA ผ่านการตั้งค่า [compliance](https://reference.aspose.com/slides/th/androidjava/com.aspose.slides/pdfoptions/#setCompliance-int-) ใน [PDF export options](https://reference.aspose.com/slides/th/androidjava/com.aspose.slides/pdfoptions/)  

**ไลบรารีรองรับการฝังฟอนต์เมื่อส่งออกเป็น PDF และให้การควบคุมละเอียดว่าอะไรจะถูกฝังหรือไม่?**

ใช่ คุณสามารถควบคุมได้ว่าฟอนต์จะถูกฝังเต็มรูปแบบหรือเป็นซับเซ็ต (เฉพาะอักขระที่ใช้) ระบุวิธีการจัดการฟอนต์ระบบทั่วไปและกำหนดพฤติกรรมสำหรับข้อความ ASCII ผ่าน [PDF export options](https://reference.aspose.com/slides/th/androidjava/com.aspose.slides/pdfoptions/)  

**ฉันสามารถตรวจสอบว่าไฟล์มีการป้องกันด้วยรหัสผ่านก่อนที่จะโหลดจริงหรือไม่?**

ใช่ โดยใช้ [factory-based inspection API](https://reference.aspose.com/slides/th/androidjava/com.aspose.slides/presentationfactory/) คุณสามารถสอบถามไฟล์พรีเซนเทชันเพื่อระบุว่ามีการป้องกันด้วยรหัสผ่านโดยไม่ต้องเปิดไฟล์เต็มรูปแบบ  

**มีกลไกการสำรองฟอนต์และรองรับฟอนต์แบบกำหนดเองหรือไม่?**

ใช่ ไลบรารีรองรับการ [loading](/slides/th/androidjava/custom-font/) และ [embedding](/slides/th/androidjava/embedded-font/) ฟอนต์แบบกำหนดเองและมี [fallback rules](/slides/th/androidjava/fallback-font/) เพื่อป้องกันการขาดอักขระขณะเรนเดอร์และแปลง  

**ฉันสามารถส่งออกสไลด์เป็น XPS และมีตัวเลือกในการปรับแต่งผลลัพธ์ XPS หรือไม่?**

ใช่ [Export to XPS](/slides/th/androidjava/convert-powerpoint-to-xps/) ถูกสนับสนุน และคุณสามารถปรับ [save options](https://reference.aspose.com/slides/th/androidjava/com.aspose.slides/xpsoptions/) ที่เกี่ยวข้องเพื่อควบคุมคุณภาพและเนื้อหาของเอกสาร XPS
