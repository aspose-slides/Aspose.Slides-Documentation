## **Overview**

Aspose.Slides สามารถใช้ได้ในโหมดประเมินหรือด้วยใบอนุญาตที่ถูกต้อง เวอร์ชันประเมินให้ฟังก์ชันการทำงานเดียวกับเวอร์ชันที่มีใบอนุญาต แต่จะเพิ่มลายน้ำการประเมินเมื่อเปิดหรือบันทึกการนำเสนอและจำกัดการดึงข้อความไว้ที่หนึ่งสไลด์  

บทความนี้อธิบายว่าการให้ลิขสิทธิ์ทำงานอย่างไรใน Aspose.Slides และวิธีการใช้ใบอนุญาตก่อนใช้งานไลบรารี ใบอนุญาตสามารถโหลดจากไฟล์, สตรีม, หรือทรัพยากรฝังโดยใช้คลาส `License` บทความนี้ยังแสดงวิธีตรวจสอบว่าใบอนุญาตถูกนำไปใช้อย่างถูกต้องหรือไม่

## **Applying a License**

ใบอนุญาตสามารถโหลดจาก **ไฟล์** หรือ **สตรีม**  

{{% alert color="primary" %}}

Aspose.Slides มีคลาส [License](https://reference.aspose.com/slides/th/androidjava/com.aspose.slides/license/) สำหรับดำเนินการเกี่ยวกับลิขสิทธิ์  

{{% /alert %}} 

{{% alert color="warning" %}}

ใบอนุญาตใหม่สามารถเปิดใช้งาน Aspose.Slides ได้เฉพาะกับเวอร์ชัน 21.4 ขึ้นไป เวอร์ชันก่อนหน้านั้นใช้ระบบลิขสิทธิ์ต่างหากและจะไม่รับรู้ใบอนุญาตเหล่านี้  

{{% /alert %}}

### **File**

วิธีที่ง่ายที่สุดในการตั้งค่าใบอนุญาตคือวางไฟล์ใบอนุญาตในโฟลเดอร์ที่มี Aspose.Slides.jar หรือ jar ของแอปพลิเคชันของคุณ  

โค้ด Java นี้แสดงวิธีตั้งค่าไฟล์ใบอนุญาต:  

``` java
// สร้างอินสแตนซ์ของคลาส License
com.aspose.slides.License license = new com.aspose.slides.License();

// กำหนดเส้นทางไฟล์ใบอนุญาต
license.setLicense("Aspose.Slides.Android.via.Java.lic");
```

{{% alert color="warning" %}} 

หากคุณวางไฟล์ใบอนุญาตในไดเรกทอรีอื่น เมื่อเรียกใช้เมธอด[SetLicense](https://reference.aspose.com/slides/th/androidjava/com.aspose.slides/license/#setLicense-java.lang.String-) พารามิเตอร์พาธที่ระบุต้องลงท้ายด้วยชื่อไฟล์ใบอนุญาตที่ตรงกับไฟล์ของคุณ  

เช่น คุณอาจเปลี่ยนชื่อไฟล์ใบอนุญาตเป็น *Aspose.Slides.Android.via.Java.lic.xml* จากนั้นในโค้ดของคุณต้องส่งพาธไปยังไฟล์ (ลงท้ายด้วย *Aspose.Slides.Android.via.Java.lic.xml*) ไปยังเมธอด[SetLicense](https://reference.aspose.com/slides/th/androidjava/com.aspose.slides/license/#setLicense-java.lang.String-)  

{{% /alert %}}

### **Stream**

คุณสามารถโหลดใบอนุญาตจากสตรีมได้ โค้ด Java นี้แสดงวิธีใช้ใบอนุญาตจากสตรีม:  

``` java
// สร้างอินสแตนซ์ของคลาส License
com.aspose.slides.License license = new com.aspose.slides.License();

// กำหนดใบอนุญาตผ่านสตรีม
license.setLicense(new java.io.FileInputStream("Aspose.Slides.Android.via.Java.lic"));
```

## **Validating a License**

เพื่อตรวจสอบว่าได้ตั้งค่าใบอนุญาตอย่างถูกต้องหรือไม่ คุณสามารถตรวจสอบได้ โค้ด Java นี้แสดงวิธีตรวจสอบใบอนุญาต:  

```java
License license = new License();
license.setLicense("Aspose.Slides.Android.via.Java.lic");

if (License.isLicensed()) 
{
    System.out.println("License is good!");
}
```

## **FAQ**

**ฉันสามารถใช้ใบอนุญาตในสภาพแวดล้อมออฟไลน์โดยสมบูรณ์ (ไม่มีการเชื่อมต่ออินเทอร์เน็ต) ได้หรือไม่?**  

ได้ การตรวจสอบใบอนุญาตทำงานในเครื่องโดยใช้ไฟล์ใบอนุญาต; ไม่จำเป็นต้องเชื่อมต่ออินเทอร์เน็ต  

**หลังจากการสมัครสมาชิกหนึ่งปีหมดอายุ จะเกิดอะไรขึ้น? ไลบรารีจะหยุดทำงานหรือไม่?**  

ไม่ ใบอนุญาตเป็นแบบถาวร: คุณสามารถใช้เวอร์ชันที่เปิดตัวก่อนวันหมดอายุของการสมัครได้ต่อไป; แต่คุณจะไม่สามารถใช้เวอร์ชันใหม่กว่าได้หากไม่ได้ต่ออายุ
