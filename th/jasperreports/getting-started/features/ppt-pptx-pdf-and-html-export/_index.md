{{% alert color="primary" %}} 

JasperReports และ JasperServer ไม่มีความสามารถในตัวสำหรับการส่งออกรายงานเป็นการนำเสนอ Microsoft PowerPoint แต่ด้วย Aspose.Slides for JasperReports คุณจะได้เข้าถึงรูปแบบการส่งออกเพิ่มเติมดังนี้:

- การนำเสนอ Microsoft PowerPoint (PPT)
- การนำเสนอ Microsoft PowerPoint (PPTX)
- HTML
- PDF

{{% /alert %}} 

เพื่อสร้างเอกสารในรูปแบบเหล่านี้ Aspose.Slides for JasperReports พึ่งพาเวอร์ชันในตัวของ [Aspose.Slides for Java](https://products.aspose.com/slides/th/java/), ไลบรารีการประมวลผลการนำเสนอขั้นนำของตลาดจาก Aspose. Microsoft PowerPoint ไม่ได้ถูกใช้ในการสร้างเอกสารเหล่านี้.
