Aspose.Slides for JasperReports ทำงานร่วมกับ:

**Aspose.Slides for JasperReports 17.5 และเวอร์ชันก่อนหน้า:**

- Java 2 Platform Standard Edition 1.4 หรือสูงกว่า.
- JasperReports 2.0, 3.0 หรือสูงกว่า.

**Aspose.Slides for JasperReports 17.6 และเวอร์ชันใหม่กว่า:**

- Java 2 Platform Standard Edition 6.0 หรือสูงกว่า.
- JasperReports 3.7.2 หรือสูงกว่า.
