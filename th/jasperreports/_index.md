**ยินดีต้อนรับสู่ Aspose.Slides for JasperReports**

Aspose.Slides for JasperReports เป็นไลบรารีที่ออกแบบและพัฒนาโดยเฉพาะสำหรับนักพัฒนาที่ต้องการส่งออกรายงานจาก JasperReports ไปยังรูปแบบ Microsoft PowerPoint Presentation (PPT) และ Microsoft PowerPoint Show (PPS) อย่างง่ายดายในแอปพลิเคชัน Java ของพวกเขา  
คุณลักษณะของรายงานทั้งหมดจะถูกแปลงด้วยความแม่นยำสูงสุดเป็นการนำเสนอ Microsoft PowerPoint  
Aspose.Slides for JasperReports รองรับ JasperReports 5+
