## Aspose.Slides for .NET

{{% alert color="primary" %}}

![โลโก้ผลิตภัณฑ์ Aspose.Slides for .NET](home_1.png)

Aspose.Slides for .NET คือ API การจัดการ Microsoft PowerPoint® ที่ทำให้แอปพลิเคชัน .NET สามารถอ่านและเขียนไฟล์ PowerPoint® ได้โดยไม่ต้องใช้ Microsoft PowerPoint® Aspose.Slides for .NET เป็นคอมโพเนนต์แรกและ唯一ที่ให้ความสามารถในการจัดการไฟล์ PowerPoint® Aspose.Slides for .NET มีคุณลักษณะหลักหลายอย่าง เช่น การจัดการข้อความ, รูปร่าง, ตาราง & แอนิเมชัน, การเพิ่มเสียงและวิดีโอลงในสไลด์, การแสดงตัวอย่างสไลด์, การส่งออกสไลด์เป็น SVG, PDF และอื่น ๆ อีกมากมาย

## Aspose.Slides for JasperReports

{{% alert color="primary" %}}

![โลโก้ผลิตภัณฑ์ Aspose.Slides for JasperReports](home_7.png)

Aspose.Slides for JasperReports เป็นไลบรารีที่ออกแบบและพัฒนาโดยเฉพาะสำหรับนักพัฒนาที่ต้องการส่งออกรายงานจาก JasperReports ไปเป็นรูปแบบ Microsoft PowerPoint Presentation (PPT) และ Microsoft PowerPoint Show (PPS) ในแอปพลิเคชัน Java ทั้งหมดของรายงานจะถูกแปลงด้วยความแม่นยำสูงสุดเป็นงานนำเสนอ Microsoft PowerPoint Aspose.Slides for JasperReports รองรับ JasperReports 5+
