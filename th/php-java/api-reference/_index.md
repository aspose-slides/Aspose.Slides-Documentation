Aspose.Slides for PHP via Java คือไลบรารีคลาสที่ช่วยให้แอปพลิเคชัน PHP สามารถอ่านและเขียนเอกสารพรีเซนเทชันโดยไม่ต้องใช้ Microsoft PowerPoint® หรืออื่น ๆ. Aspose.Slides for PHP via Java อนุญาตให้จัดการข้อความ, รูปร่าง, ตาราง & แอนิเมชัน, เพิ่มเสียงและวิดีโอในสไลด์, ดูตัวอย่างสไลด์, ส่งออกสไลด์เป็นรูปแบบ SVG, PDF และอื่น ๆ อีกมาก.

สามารถค้นหาเอกสารอ้างอิง API ล่าสุดได้ที่ [Aspose.Slides for PHP via Java API Reference](https://reference.aspose.com/slides/th/php-java/).
