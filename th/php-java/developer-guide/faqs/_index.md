## **ภาพรวม**

FAQ นี้ให้คำตอบสำหรับคำถามทั่วไปเกี่ยวกับ Aspose.Slides ซึ่งครอบคลุมรูปแบบไฟล์ที่รองรับ, การจัดการข้อยกเว้นเมื่อทำงานกับการนำเสนอขนาดใหญ่, การเปลี่ยนขนาดสไลด์, การแสดงตัวอย่างสไลด์, การดึงข้อความจากการนำเสนอ, การจัดรูปแบบขอบตาราง, การใส่รูปภาพ, และการแก้ไขปัญหาเกี่ยวกับฟอนต์เมื่อแปลงการนำเสนอเป็น PDF หรือรูปภาพ

## **รูปแบบไฟล์ที่รองรับ**

**Q: Aspose.Slides for PHP via Java รองรับรูปแบบไฟล์อะไรบ้าง?**

**A**: Aspose.Slides for PHP via Java รองรับรูปแบบไฟล์ที่อธิบายไว้ใน [รูปแบบไฟล์ที่รองรับ](/slides/th/php-java/supported-file-formats/)

## **ข้อยกเว้น**

**Q: ฉันได้รับข้อยกเว้น out of memory ขณะโหลดไฟล์ PPT ขนาดใหญ่ที่มีรูปภาพ มีข้อจำกัดเรื่องขนาดไฟล์ใน Aspose.Slides หรือไม่?**

**A**: ไม่มีสูตรเฉพาะสำหรับคำนวนขนาดการนำเสนอที่รองรับโดย Aspose.Slides ควรมีพื้นที่เพียงพอให้โครงสร้างการนำเสนอและรูปภาพทั้งหมดอยู่ในหน่วยความจำ โดยทั่วไปรูปภาพในหน่วยความจำจะใช้พื้นที่มากกว่าบนฮาร์ดดิสก์ โดยเฉพาะเมื่อรูปภาพมีเอฟเฟกต์เพิ่มเติม

โดยทั่วไป Aspose.Slides for PHP via Java สามารถจัดการไฟล์การนำเสนอประมาณ 300 MB บนเซิร์ฟเวอร์ที่มี RAM 4 GB ได้อย่างง่ายดาย

## **ทำงานกับสไลด์**

**Q: ฉันสามารถเปลี่ยนขนาดสไลด์ในการนำเสนอได้หรือไม่?**

**A**: คุณสามารถใช้เมธอด `getSlideSize` ที่เปิดให้ใช้งานโดยคลาส [Presentation](https://reference.aspose.com/slides/th/php-java/aspose.slides/presentation/) เพื่อกำหนดขนาดสไลด์ในการนำเสนอ

**Q: มีวิธีกำหนดสไลด์ขนาดแตกต่างกันในการนำเสนอเดียวได้หรือไม่?**

**A**: เนื่องจากขนาดสไลด์ถูกกำหนดระดับการนำเสนอในไฟล์ Microsoft PowerPoint จึงไม่มีวิธีทำเช่นนั้น

**Q: Aspose.Slides for PHP via Java รองรับการแสดงตัวอย่างสไลด์ก่อนบันทึกหรือไม่?**

**A**: คุณสามารถเรนเดอร์สไลด์การนำเสนอเป็นรูปภาพและใช้รูปภาพเหล่านี้สำหรับการแสดงตัวอย่างสไลด์ได้

## **ทำงานกับข้อความ**

**Q: สามารถดึงข้อความทั้งหมดจากการนำเสนอได้หรือไม่?**

**A**: Aspose.Slides for PHP via Java มีคลาส [SlideUtil](https://reference.aspose.com/slides/th/php-java/aspose.slides/slideutil/) ที่ให้เมธอดต่าง ๆ สำหรับดึงข้อความทั้งหมดจากการนำเสนอ

**Q: ทำไมขนาดย่อหน้าถึงต่างกันบนระบบปฏิบัติการ Windows และ Linux?**

**A**: การคำนวนขนาดย่อหน้าขึ้นอยู่กับการคำนวนขนาดข้อความของย่อหน้านั้น ขนาดข้อความคำนวนจากเมตริกของฟอนต์ที่ระบุในไฟล์ PowerPoint หากฟอนต์ที่ระบุไม่มีอยู่ ระบบจะเปลี่ยนเป็นฟอนต์ที่คล้ายที่สุด ซึ่งฟอนต์นั้นอาจมีเมตริกต่างจากฟอนต์ต้นฉบับ ทำให้การคำนวนขนาดย่อหน้าในแต่ละระบบให้ผลต่างกัน ขึ้นอยู่กับชุดฟอนต์ที่ติดตั้งไว้ เพื่อให้ได้ผลลัพธ์เหมือนกันบนระบบปฏิบัติการต่าง ๆ คุณควรติดตั้งฟอนต์เดียวกันบนแต่ละระบบหรือโหลดฟอนต์ในขณะรันไทม์เป็น [ฟอนต์ภายนอก](/slides/th/php-java/custom-font/)

## **การจัดรูปแบบและรูปภาพ**

**Q: จะตั้งค่าสีของขอบตารางอย่างไร?**

**A**: คุณสามารถเปลี่ยนสีของขอบตารางทั้งหมดหรือเพียงขอบโดยรอบตารางทั้งหมดได้ สำหรับการเปลี่ยนสีทุกขอบ ให้ใช้เมธอด `getCellFormat` จากคลาส [Cell](https://reference.aspose.com/slides/th/php-java/aspose.slides/cell/) สำหรับขอบโดยรอบตารางทั้งหมดให้ทำการวนลูปผ่านเซลล์และเปลี่ยนสีของขอบด้านนอก

**Q: Aspose.Slides for PHP via Java ใช้หน่วยวัดใดในการวางรูปภาพ?**

**A**: พิกัดและขนาดของรูปทรงทั้งหมดบนสไลด์จะวัดเป็นจุด (72 dpi)

## **ทำงานกับฟอนต์**

**Q: เมื่อแปลง PPT เป็น PDF หรือรูปภาพ ทำไมฟอนต์ในเอกสารผลลัพธ์ถึงแตกต่างกัน?**

**A**: ปัญหานี้อาจบ่งบอกว่าฟอนต์ที่ใช้ในการนำเสนอไม่มีอยู่ในระบบปฏิบัติการที่รันโค้ด คุณควรติดตั้งฟอนต์บนระบบปฏิบัติการหรือโหลดฟอนต์เป็นฟอนต์ภายนอกโดยใช้คลาส [FontsLoader](https://reference.aspose.com/slides/th/php-java/aspose.slides/fontsloader/) ตามตัวอย่างด้านล่าง:
```php
$folders = ["path_to_a_folder_with_fonts"];
FontsLoader::loadExternalFonts($folders);
```
