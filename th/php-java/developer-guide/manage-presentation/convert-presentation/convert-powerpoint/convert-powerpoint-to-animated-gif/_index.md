## **ภาพรวม**

Aspose.Slides ให้คุณแปลงงานนำเสนอ PowerPoint เป็นไฟล์ GIF แบบเคลื่อนไหวได้ด้วยแค่ไม่กี่บรรทัดของโค้ด ซึ่งมีประโยชน์เมื่อคุณต้องการแชร์เนื้อหาในสไลด์ในรูปแบบที่มีน้ำหนักเบา รองรับอย่างกว้างขวางและสามารถฝังในหน้าเว็บ แชท หรือเอกสารได้ บทความนี้อธิบายวิธีส่งออกงานนำเสนอเป็น GIF ด้วยการตั้งค่าเริ่มต้นและวิธีการปรับแต่งผลลัพธ์โดยกำหนดตัวเลือกต่างๆ เช่น ขนาดเฟรม, ระยะเวลาหน่วงสไลด์, และอัตราเฟรมการเปลี่ยนผ่าน ผ่าน GifOptions.

## **แปลงงานนำเสนอเป็น GIF แบบเคลื่อนไหวโดยใช้การตั้งค่าเริ่มต้น**

โค้ดตัวอย่างนี้แสดงวิธีแปลงงานนำเสนอเป็น GIF แบบเคลื่อนไหวโดยใช้การตั้งค่ามาตรฐาน:

```php
  $pres = new Presentation("pres.pptx");
  try {
    $pres->save("pres.gif", SaveFormat::Gif);
  } finally {
    if (!java_is_null($pres)) {
      $pres->dispose();
    }
  }
```

ไฟล์ GIF ที่ได้จะถูกสร้างด้วยพารามิเตอร์เริ่มต้น  

{{%  alert  title="TIP"  color="primary"  %}} 
หากคุณต้องการปรับแต่งพารามิเตอร์ของ GIF คุณสามารถใช้คลาส GifOptions ได้ ดูโค้ดตัวอย่างด้านล่าง  

{{% /alert %}} 

## **แปลงงานนำเสนอเป็น GIF แบบเคลื่อนไหด้วยการตั้งค่าที่กำหนดเอง**
โค้ดตัวอย่างนี้แสดงวิธีแปลงงานนำเสนอเป็น GIF แบบเคลื่อนไหวโดยใช้การตั้งค่าที่กำหนดเอง :

```php
  $pres = new Presentation("pres.pptx");
  try {
    $gifOptions = new GifOptions();
    $gifOptions->setFrameSize(new Java("java.awt.Dimension", 960, 720));// ขนาดของ GIF ที่ได้

    $gifOptions->setDefaultDelay(2000);// ระยะเวลาที่แต่ละสไลด์จะแสดงจนกว่าจะเปลี่ยนไปสไลด์ถัดไป

    $gifOptions->setTransitionFps(35);// เพิ่ม FPS เพื่อคุณภาพการเคลื่อนไหวระหว่างการเปลี่ยนที่ดียิ่งขึ้น

    $pres->save("pres.gif", SaveFormat::Gif, $gifOptions);
  } finally {
    if (!java_is_null($pres)) {
      $pres->dispose();
    }
  }
```

## **คำถามที่พบบ่อย**

**ถ้าฟอนต์ที่ใช้ในงานนำเสนอไม่ได้ติดตั้งบนระบบจะทำอย่างไร?**  
ติดตั้งฟอนต์ที่ขาดหายไปหรือกำหนดฟอนต์สำรอง Aspose.Slides จะทำการแทนที่ แต่ลักษณะอาจแตกต่างกัน สำหรับการสร้างแบรนด์ ควรตรวจสอบให้ฟอนต์ที่ต้องการพร้อมใช้งานอย่างชัดเจนเสมอ  

**ฉันสามารถใส่น้ำโลโก้ทับบนเฟรมของ GIF ได้หรือไม่?**  
ได้ คุณสามารถเพิ่มวัตถุ/โลโก้ที่มีความโปร่งแสงครึ่งหนึ่งลงในสไลด์หลักหรือสไลด์แต่ละหน้า ก่อนทำการส่งออก — น้ำโลโก้จะปรากฏบนทุกเฟรม
