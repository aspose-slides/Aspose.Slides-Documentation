## **ภาพรวม**

การเข้าถึงงานพรีเซนเทชันทำให้ผู้ใช้เทคโนโลยีเสริม—เช่น โปรแกรมอ่านหน้าจอ, หน้าจอเบรลล์, หรือการนำทางด้วยคีย์บอร์ดเท่านั้น—สามารถเข้าใจและนำทางสไลด์ของคุณได้อย่างมีประสิทธิภาพเทียบเท่ากับผู้ชมที่มองเห็นและใช้เมาส์ การปฏิบัติที่ดีมุ่งเน้นที่ลำดับการอ่านที่ชัดเจน, คำอธิบายภาพแบบทางเลือกที่มีความหมายสำหรับภาพข้อมูล, ความแตกต่างของสีที่เพียงพอ, ตัวอักษรที่อ่านง่าย, ข้อความลิงก์ที่อธิบายได้, และการหลีกเลี่ยงการสื่อความหมายด้วยสีหรือตำแหน่งเพียงอย่างเดียว เมื่อการเข้าถึงถูกวางแผนตั้งแต่ต้น ผลลัพธ์จะเป็นโครงสร้างที่สะอาดขึ้น, ภาพที่สอดคล้องกันมากขึ้น, และเนื้อหาที่เข้าถึงผู้ชมทุกคนโดยไม่ต้องใช้วิธีแก้ไข

## **ทำเครื่องหมายว่าเป็นของตกแต่ง**

ทำเครื่องหมายว่าเป็นของตกแต่งจะระบุภาพที่เป็นเพียงการตกแต่งเพื่อให้โปรแกรมอ่านหน้าจอข้ามไป, ลดสัญญาณรบกวนและเน้นไปที่เนื้อหาที่มีความหมาย ใช้กับพื้นหลัง, ลวดลาย, และช่องว่าง—ไม่ใช้กับแผนภูมิ, ไอคอน, หรือรูปภาพที่สื่อข้อมูล Aspose.Slides เปิดเผยแฟล็กนี้สำหรับการตรวจจับและการตรวจสอบ, ทำให้สามารถตรวจสอบการเข้าถึงอัตโนมัติและทำความสะอาดได้

![Mark as Decorative](mark_as_decorative.png)

ตัวอย่างโค้ดต่อไปนี้แสดงวิธีตรวจสอบว่า shape ถูกทำเครื่องหมายว่าเป็นของตกแต่งหรือไม่

```php
$presentation = new Presentation("sample.pptx");
try {
    $shape = $presentation->getSlides()->get_Item(0)->getShapes()->get_Item(0);
    echo "Is shape decorative: " . ($shape->isDecorative() ? "true" : "false") . "\n";
} finally {
    $presentation->dispose();
}
```
