## **แบบอักษรสำรอง**
แบบอักษรสำรองจะถูกใช้เมื่อแบบอักษรที่ระบุสำหรับข้อความมีอยู่ในระบบ แต่แบบอักษรนั้นไม่มี glyph ที่จำเป็น ในกรณีนี้สามารถใช้แบบอักษรสำรองที่กำหนดไว้หนึ่งตัวเพื่อทำการแทนที่ glyph ได้

Aspose.Slides อนุญาตให้สร้างแบบอักษรสำรอง, เพิ่มเข้าไปในคอลเลกชันแบบอักษรสำรอง, ตั้งค่าคอลเลกชันแบบอักษรสำรองสำหรับการนำเสนอบางไฟล์, ลบแบบอักษรสำรองออกจากการนำเสนอ, ระบุกฎเพื่อใช้แบบอักษรสำรองและอื่น ๆ

## **FAQ**

**แบบอักษรสำรองแตกต่างจากการแทนที่แบบอักษรอย่างไร?**

แบบอักษรสำรองจะถูกนำไปใช้ต่อหนึ่งอักขระหรือช่วงของ Unicode เมื่อแบบอักษรหลักไม่มี glyph เฉพาะ; มันเติมเฉพาะอักขระที่หายไปเท่านั้น [การแทนที่](/slides/th/php-java/font-substitution/) จะเปลี่ยนแบบอักษรที่หายหรือไม่มีให้กับส่วนข้อความทั้งหมดหรือช่วงข้อความด้วยแบบอักษรอื่น สามารถใช้ร่วมกันได้ แต่ขอบเขตและตรรกะการเลือกต่างกัน

**การตั้งค่าแบบอักษรสำรองจะถูกบันทึกไว้ในไฟล์การนำเสนอหรือไม่?**

ไม่ การกำหนดค่าแบบอักษรสำรองอยู่ในช่วงการประมวลผล/เรนเดอร์ในไลบรารีและไม่ได้ถูกจัดเก็บเป็นส่วนหนึ่งของไฟล์ PPTX การนำเสนอไม่เก็บกฎแบบอักษรสำรองของคุณ

**แบบอักษรสำรองมีผลต่อองค์ประกอบที่สร้างโดยวัตถุ PowerPoint (SmartArt, แผนภูมิ, WordArt) หรือไม่?**

ใช่ ข้อความภายในวัตถุเหล่านี้จะผ่านไปยังขั้นตอนการเรนเดอร์เดียวกัน ดังนั้นกฎแบบอักษรสำรองที่กำหนดไว้จะนำไปใช้กับข้อความเหล่านั้นเช่นเดียวกับข้อความทั่วไป
