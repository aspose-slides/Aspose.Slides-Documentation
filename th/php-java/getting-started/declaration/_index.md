{{% alert color="primary" %}} 

ส่วนประกอบ Aspose Java ทั้งหมดต้องการชุดสิทธิ์ Full Trust. สาเหตุคือ ส่วนประกอบ Aspose Java จำเป็นต้องเข้าถึงการตั้งค่าจดหมายทะเบียน, ไฟล์ระบบที่ไม่ใช่ไดเรกทอรีเสมือนสำหรับการดำเนินการบางอย่าง เช่น การวิเคราะห์แบบอักษร เป็นต้น. นอกจากนี้ ส่วนประกอบ Aspose Java ยังอิงกับคลาสระบบ Java หลักที่หลายกรณีก็ต้องการชุดสิทธิ์ Full Trust.

{{% /alert %}} 

ผู้ให้บริการอินเทอร์เน็ต (ISP) ที่โฮสต์แอปพลิเคชันหลายๆ ตัวจากบริษัทต่างๆ ส่วนใหญ่บังคับระดับความปลอดภัย Medium Trust: 

- OleDbPermission ไม่มีในระบบ. หมายความว่าคุณไม่สามารถใช้ ADO.NET managed OLE DB data provider เพื่อเข้าถึงฐานข้อมูลได้.
- EventLogPermission ไม่มีในระบบ. หมายความว่าคุณไม่สามารถเข้าถึง Windows event log ได้.
- ReflectionPermission ไม่มีในระบบ. หมายความว่าคุณไม่สามารถใช้ reflection ได้.
- RegistryPermission ไม่มีในระบบ. หมายความว่าคุณไม่สามารถเข้าถึง registry ได้.
- WebPermission ถูกจำกัด. หมายความว่าแอปพลิเคชันของคุณสามารถสื่อสารได้เฉพาะกับที่อยู่หรือช่วงที่อยู่ที่คุณกำหนดในองค์ประกอบ <trust>.
- FileIOPermission ถูกจำกัด. หมายความว่าคุณสามารถเข้าถึงไฟล์ได้เฉพาะในไดเรกทอรีเสมือนของแอปพลิเคชันของคุณเท่านั้น.

{{% alert color="primary" %}} 

ด้วยเหตุผลที่ระบุด้านบน ส่วนประกอบ Aspose Java ไม่สามารถใช้บนเซิร์ฟเวอร์ที่มอบชุดสิทธิ์ที่ไม่ใช่ Full Trust ได้.
