## **ภาพรวม**

บทความนี้อธิบายข้อบกพร่องด้านการจัดการใบอนุญาตที่พบใน Aspose.Slides for PHP via Java รุ่น 23.2, 23.3, 23.4, 23.5, 23.6 และ 23.7 ข้อบกพร่องนี้ทำให้ใบอนุญาตถาวรไม่สามารถใช้งานได้ชั่วคราวหลังจากการสมัครใช้งานหมดอายุ

## **อาการ**

เมื่อการสมัครใช้งานใบอนุญาตถาวรของคุณหมดอายุ Aspose.Slides for PHP via Java รุ่น 23.2 - 23.7 อาจรายงานว่าใบอนุญาตหมดอายุและเปลี่ยนการทำงานทั้งหมดเป็นโหมดประเมินผล  
พฤติกรรมนี้ไม่ถูกต้องและไม่ได้ส่งผลต่อรุ่นก่อนหน้า 23.2 หรือหลังจาก 23.7

## **วิธีแก้ไข**

Aspose.Slides for PHP via Java ได้แก้ไขปัญหานี้และปล่อยเวอร์ชันที่อัปเดต (23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1) พร้อมการแก้ไข

หากคุณกำลังใช้หนึ่งในรุ่นที่ได้รับผลกระทบของ Aspose.Slides for PHP via Java ในโปรเจกต์ของคุณ โปรดเปลี่ยนไปใช้เวอร์ชันที่แก้ไขแล้ว

- ตรวจสอบเวอร์ชันที่คุณใช้งานและให้แน่ใจว่าไม่ได้รับผลกระทบจากข้อบกพร่องที่อธิบายข้างต้น

- ติดต่อ [Aspose support](https://forum.aspose.com/) เพื่อขอความช่วยเหลือต่อไป

## **ผลิตภัณฑ์และรุ่นที่ได้รับผลกระทบ**

โปรดทราบว่าข้อบกพร่องนี้ส่งผลต่อเพียง Aspose.Slides for PHP via Java รุ่น 23.2 ถึง 23.7 เท่านั้น **ผลิตภัณฑ์ Aspose อื่น ๆ และรุ่นอื่น ๆ ของ Aspose.Slides ไม่ได้รับผลกระทบ**
