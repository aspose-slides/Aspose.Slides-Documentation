## **ประเมิน Aspose.Slides**
คุณสามารถดาวน์โหลด Aspose.Slides เพื่อการประเมินได้อย่างง่ายดาย แพ็กเกจการประเมินมีลักษณะเดียวกับแพ็กเกจที่ซื้อ เวอร์ชันการประเมินจะกลายเป็นเวอร์ชันที่มีใบอนุญาตเมื่อคุณเพิ่มบรรทัดโค้ดบางบรรทัดเพื่อใช้ใบอนุญาต

## **ข้อจำกัดของเวอร์ชันการประเมิน**
เวอร์ชันการประเมินของ Aspose.Slides (ไม่มีการระบุใบอนุญาต) ให้ฟังก์ชันการทำงานทั้งหมดของผลิตภัณฑ์ แต่จะใส่น้ำลายน้ำการประเมินที่ด้านบนของเอกสารเมื่อเปิดและบันทึก คุณยังถูกจำกัดให้ดึงข้อความจากสไลด์ได้เพียงหนึ่งสไลด์เท่านั้น

## **เกี่ยวกับใบอนุญาต**
คุณสามารถดาวน์โหลดเวอร์ชันการประเมินของ Aspose.Slides for PHP via Java ได้จาก [download page](https://packagist.org/packages/aspose/slides) เวอร์ชันการประเมินให้ **ความสามารถเดียวกัน** กับเวอร์ชันที่มีใบอนุญาตของ Aspose.Slides นอกจากนี้เวอร์ชันการประเมินจะกลายเป็นเวอร์ชันที่มีใบอนุญาตเมื่อคุณซื้อใบอนุญาตและเพิ่มบรรทัดโค้ดสองสามบรรทัดเพื่อใช้ใบอนุญาต

ใบอนุญาตเป็นไฟล์ XML ที่เป็นข้อความธรรมดาซึ่งบรรจุรายละเอียดต่างๆ เช่น ชื่อผลิตภัณฑ์ จำนวนผู้พัฒนาที่ได้รับอนุญาต วันที่หมดอายุการสมัครสมาชิก เป็นต้น ไฟล์ได้รับการลงลายเซ็นดิจิทัล ดังนั้นห้ามแก้ไขไฟล์ แม้แต่การเพิ่มบรรทัดว่างโดยบังเอิญก็จะทำให้ไฟล์ใช้การไม่ได้

เพื่อหลีกเลี่ยงข้อจำกัดที่เกี่ยวกับเวอร์ชันการประเมิน คุณต้องตั้งค่าใบอนุญาตก่อนใช้งาน **Aspose.Slides** คุณต้องตั้งค่าใบอนุญาตเพียงครั้งเดียวต่อแอปพลิเคชันหรือกระบวนการ

### **ตั้งค่าใบอนุญาตใน Aspose.Slides for PHP via Java**

ใบอนุญาตสามารถนำไปใช้จากตำแหน่งต่อไปนี้:

* เส้นทางแบบชี้ชัด
* สตรีม
* เป็น Metered License – กลไกการออกใบอนุญาตแบบใหม่

{{% alert color="primary" %}}

ใช้เมธอด **setLicense** เพื่อให้คอมโพเนนต์มีใบอนุญาต

แม้ว่าการเรียกเมธอด **setLicense** หลายครั้งจะไม่ทำให้เกิดอันตราย แต่ก็เป็นการเสียทรัพยากร (processor) โดยไม่มีประโยชน์

{{% /alert %}}

{{% alert color="warning" %}}

ใบอนุญาตใหม่สามารถเปิดใช้งาน Aspose.Slides ได้เฉพาะกับรุ่น 21.4 หรือใหม่กว่า รุ่นก่อนใช้ระบบใบอนุญาตต่างกันและจะไม่รับรู้ใบอนุญาตเหล่านี้

{{% /alert %}}

#### **ใช้ใบอนุญาตจากไฟล์**

โค้ดตัวอย่างนี้ใช้เพื่อกำหนดไฟล์ใบอนุญาต:

**PHP**

```php
<?php
require_once("http://localhost:8080/JavaBridge/java/Java.inc");
require_once("lib/aspose.slides.php");

use aspose\slides\License;

$license = new License();
$license->setLicense("Aspose.Slides.lic");
?>
```

เมื่อเรียกเมธอด setLicense ชื่อใบอนุญาตต้องเหมือนกับชื่อไฟล์ใบอนุญาตของคุณ ตัวอย่างเช่นคุณสามารถเปลี่ยนชื่อไฟล์ใบอนุญาตเป็น "Aspose.Slides.lic.xml" จากนั้นในโค้ดของคุณต้องส่งชื่อใบอนุญาตใหม่ (Aspose.Slides.lic.xml) ไปยังเมธอด setLicense

#### **ใช้ใบอนุญาตจากสตรีม**

## **FAQ**

**Can I apply the license in a completely offline environment (no internet access)?**

Yes. License validation is performed locally using the license file; no internet connection is required.

**What happens after the one-year subscription expires? Will the library stop working?**

No. The license is perpetual: you can continue using versions released before your subscription end date; you just won’t be eligible to use newer releases without renewing.
