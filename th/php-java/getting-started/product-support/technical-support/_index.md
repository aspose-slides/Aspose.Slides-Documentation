## **ภาพรวม**

บทความนี้อธิบายวิธีรับการสนับสนุนสำหรับ Aspose.Slides และวิธีรายงานปัญหาอย่างมีประสิทธิภาพ มันอธิบายช่องทางการสนับสนุนหลัก ให้คำแนะนำในการส่งปัญหา และชี้ไปยังทรัพยากรเทคนิคที่เป็นประโยชน์ เช่น หน้าผลิตภัณฑ์ บล็อก การดาวน์โหลด ฟอรัมสนับสนุน เอกสารกำกับและ API Reference บทความนี้ยังตอบคำถามทั่วไปเกี่ยวกับการสนับสนุนแบบชำระเงิน การส่งเรื่องต่อระดับและการสนับสนุนเวอร์ชันไลบรารีเก่า

## **การสนับสนุนด้านเทคนิค**

{{% alert color="primary" %}} 

Aspose ให้การสนับสนุนด้านเทคนิคฟรีไม่จำกัดสำหรับผลิตภัณฑ์ทั้งหมดของบริษัท การสนับสนุนนี้พร้อมให้บริการแก่ผู้ใช้ทั้งหมดรวมถึงการประเมินผล

{{% /alert %}} 

หากคุณต้องการความช่วยเหลือเกี่ยวกับ Aspose.Slides for PHP via Java โปรดพิจารณาตัวเลือกต่อไปนี้:

- ช่องทางการสนับสนุนหลักคือ Aspose.Forums โพสต์คำถามของคุณในฟอรั่ม Aspose.Slides และคำถามจะได้รับการตอบภายในไม่กี่ชั่วโมง คำถามจะตอบโดยทีมสนับสนุน Aspose.Slides โดยตรง
- โปรดทราบว่า Aspose ไม่ให้การสนับสนุนด้านเทคนิคทางโทรศัพท์ การสนับสนุนทางโทรศัพท์มีให้เฉพาะคำถามเกี่ยวกับการขายและการซื้อเท่านั้น
- เมื่อคาดว่าจะได้รับการตอบกลับในฟอรั่ม โปรดคำนึงถึงความแตกต่างของเขตเวลา

หากคุณพบปัญหากับ Aspose.Slides for PHP via Java ให้ทำตามขั้นตอนง่าย ๆ ด้านล่างเพื่อให้มั่นใจว่าปัญหา​จะได้รับการแก้ไขอย่างมีประสิทธิภาพที่สุด:

- ตรวจสอบว่าคุณใช้เวอร์ชันล่าสุดของ Aspose.Slides for PHP via Java ก่อนรายงานปัญหา ดูที่หน้า Aspose.Slides download เพื่อทราบเวอร์ชันล่าสุด
- ตรวจสอบฟอรั่ม เอกสารนี้และ API Reference ก่อนทำการรายงานปัญหา; บางทีคำถามของคุณอาจได้รับการตอบแล้ว
- เมื่อรายงานปัญหา โปรดแนบเอกสารต้นฉบับและอาจรวมถึงส่วนของโค้ดที่ทำให้เกิดปัญหา หากต้องแนบหลายไฟล์ ให้บีบอัดเป็น zip ไฟล์เดียว การแนบเอกสารของคุณใน Aspose.Forums ปลอดภัย เนื่องจากจะเข้าถึงได้เฉพาะคุณและนักพัฒนา Aspose เท่านั้น
- กรุณาพยายามรายงานปัญหาแต่ละรายการต่อหนึ่งหัวข้อ หากคุณมีปัญหาอื่น ให้รายงานในหัวข้อแยกต่างหาก

## **คำถามที่พบบ่อย**

**ระดับการสนับสนุนแบบชำระเงินที่มีให้และความแตกต่างคืออะไร?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). พวกเขาต่างกันตามจำนวนเหตุการณ์ที่รวมอยู่ ความสำคัญของการตอบสนอง และช่องทางการสื่อสารที่มีให้

**ฉันสามารถส่งปัญหา bug ไปยังทีมวิศวกรรมโดยตรงได้หรือไม่?**

ใช่. [Paid support](https://helpdesk.aspose.com/) อนุญาตให้ส่งต่อไปยังนักพัฒนาเมื่อปัญหาต้องการการตรวจสอบเชิงลึก

**เวอร์ชันไลบรารีเก่าถูกสนับสนุนหรือไม่ และคุณปล่อยการแก้ไขสำหรับเวอร์ชันเหล่านั้นหรือไม่?**

คุณสามารถใช้เวอร์ชันเก่าได้ แต่การแก้ไขจะถูกจัดให้กับรุ่นปัจจุบันเท่านั้น; แพตช์สำหรับเวอร์ชันเก่าไม่ได้ออก
