## **ภาพรวม**

Aspose.Slides รองรับไฟล์งานนำเสนอจาก Microsoft PowerPoint 97 ถึง Office 365 รวมถึง Microsoft PowerPoint สำหรับ Mac บทความนี้แสดงรายการเวอร์ชันของ PowerPoint ที่ไลบรารีสนับสนุนและให้ตารางรูปแบบไฟล์ที่สามารถโหลด บันทึก หรือทั้งสองอย่างได้.

บทความนี้ยังตอบคำถามทั่วไปเกี่ยวกับการปฏิบัติตามมาตรฐาน PDF การฝังฟอนต์ ไฟล์ที่ป้องกันด้วยรหัสผ่าน ฟอนต์แบบกำหนดเอง การสำรองฟอนต์ และตัวเลือกการส่งออก XPS.

## **เวอร์ชัน Microsoft PowerPoint ที่รองรับ**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint สำหรับ Mac
- Office 365

## **คำถามที่พบบ่อย**

**ฉันสามารถบันทึกงานนำเสนอเป็น PDF ที่ตรงตามมาตรฐานการเก็บรักษาและการเข้าถึง (PDF/A และ PDF/UA) ได้หรือไม่?**

ใช่ Aspose.Slides รองรับการส่งออกเป็น PDF ด้วยระดับการปฏิบัติตามเช่น PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b รวมถึง PDF/UA ผ่านการตั้งค่า [compliance](https://reference.aspose.com/slides/th/php-java/aspose.slides/pdfoptions/setcompliance/) ใน [PDF export options](https://reference.aspose.com/slides/th/php-java/aspose.slides/pdfoptions/).

**ไลบรารีนี้รองรับการฝังฟอนต์เมื่อส่งออกเป็น PDF พร้อมการควบคุมละเอียดว่าอะไรจะถูกฝังบ้างหรือไม่?**

ใช่ คุณสามารถควบคุมได้ว่าฟอนต์จะถูกฝังเต็มรูปแบบหรือเป็นส่วนย่อย (เฉพาะ glyph ที่ใช้) ระบุกฎการจัดการฟอนต์ระบบทั่วไป และกำหนดพฤติกรรมสำหรับข้อความ ASCII ผ่าน [PDF export options](https://reference.aspose.com/slides/th/php-java/aspose.slides/pdfoptions/).

**ฉันสามารถตรวจจับว่าไฟล์ถูกป้องกันด้วยรหัสผ่านหรือไม่ ก่อนที่จะโหลดจริงได้หรือไม่?**

ใช่ โดยใช้ [factory-based inspection API](https://reference.aspose.com/slides/th/php-java/aspose.slides/presentationfactory/) คุณสามารถสอบถามไฟล์งานนำเสนอเพื่อระบุว่ามันถูกป้องกันด้วยรหัสผ่านหรือไม่โดยไม่ต้องเปิดไฟล์เต็ม.

**มีกลไกการสำรองฟอนต์และการสนับสนุนฟอนต์แบบกำหนดเองหรือไม่?**

ใช่ ไลบรารีนี้รองรับการ[loading](/slides/th/php-java/custom-font/)และการ[embedding](/slides/th/php-java/embedded-font/)ฟอนต์แบบกำหนดเองและให้กฎการ [fallback rules](/slides/th/php-java/fallback-font/) ของฟอนต์เพื่อป้องกันการขาด glyph ระหว่างการเรนเดอร์และการแปลง.

**ฉันสามารถส่งออกสไลด์เป็น XPS ได้หรือไม่ และมีตัวเลือกเพื่อปรับแต่งผลลัพธ์ XPS หรือไม่?**

ใช่ รองรับการ[Export to XPS](/slides/th/php-java/convert-powerpoint-to-xps/) และคุณสามารถปรับ [save options](https://reference.aspose.com/slides/th/php-java/aspose.slides/xpsoptions/) ที่เกี่ยวข้องเพื่อควบคุมคุณภาพและเนื้อหาของเอกสาร XPS.
