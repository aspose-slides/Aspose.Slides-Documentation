## **การประเมิน Aspose.Slides**

คุณสามารถดาวน์โหลด Aspose.Slides เพื่อการประเมินได้อย่างง่ายดาย แพ็กเกจการประเมินมีเนื้อหาเดียวกับแพ็กเกจที่ซื้อ เวอร์ชันการประเมินจะกลายเป็นเวอร์ชันที่ได้รับใบอนุญาตเมื่อคุณเพิ่มบรรทัดโค้ดไม่กี่บรรทัดเพื่อใช้ใบอนุญาต

เวอร์ชันการประเมินของ Aspose.Slides (โดยไม่มีการระบุใบอนุญาต) ให้ฟังก์ชันการทำงานของผลิตภัณฑ์ทั้งหมด แต่จะใส่ลายน้ำการประเมินที่ด้านบนของเอกสารเมื่อเปิดหรือบันทึก นอกจากนี้คุณยังถูกจำกัดให้ใช้งานเพียงสไลด์เดียวเมื่อสกัดข้อความจากสไลด์การนำเสนอ

## **คำถามที่พบบ่อย**

**ฉันสามารถทดสอบการนำเสนอหลายไฟล์พร้อมกันในหลายเธรดในโหมดการประเมินได้หรือไม่?**

ใช่ คุณสามารถประมวลผลเอกสารต่าง ๆ พร้อมกันได้; คุณไม่ควรแชร์อ็อบเจกต์การนำเสนอเดียวกัน [ข้ามเธรด](/slides/th/php-java/multithreading/) โหมดการประเมินไม่มีผลต่อเรื่องนี้

**ฉันจำเป็นต้องติดตั้ง Microsoft PowerPoint เพื่อประเมินไลบรารีบนเซิร์ฟเวอร์หรือใน CI หรือไม่?**

ไม่ Aspose.Slides เป็นเครื่องยนต์แบบสแตนด์อโลนและไม่จำเป็นต้องติดตั้ง PowerPoint ทั้งในโหมดการประเมินหรือการใช้งานจริง

**ฉันสามารถทดสอบการแปลง PPT/PPTX เป็น PDF และภาพในโหมดการประเมินอย่างเต็มที่ได้หรือไม่?**

ใช่ ตัว[ตัวแปลง](/slides/th/php-java/convert-presentation/)ทำงาน; ผลลัพธ์จะมีลายน้ำ

**ฉันสามารถใช้ใบอนุญาตชั่วคราวสำหรับการทดสอบโหลดโดยไม่มีลายน้ำได้หรือไม่?**

ใช่ ใบอนุญาตชั่วคราว 30 วันจะลบข้อจำกัดของโหมดการประเมินและทำให้สามารถทดสอบโดยไม่มีลายน้ำได้
