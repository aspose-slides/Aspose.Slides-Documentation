[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) คือรูปแบบไฟล์เอกสารการนำเสนอที่สามารถสร้าง อ่าน แก้ไข และบันทึกโดยเวอร์ชันต่าง ๆ ของ Microsoft PowerPoint ซึ่งเป็นรูปแบบไบนารีสำหรับเอกสารการนำเสนอที่พัฒนาโดย Microsoft

{{% /alert %}} 

## **PPT ใน Aspose.Slides for PHP via Java**
Aspose.Slides for PHP via Java สามารถอ่านไฟล์ PPT ที่สร้างโดยซอฟต์แวร์ต่อไปนี้

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

ในทำนองเดียวกัน ไฟล์ PPT ที่สร้างโดย Aspose.Slides for PHP via Java สามารถอ่านได้โดยซอฟต์แวร์ชุดข้างต้น

## **การสนับสนุนที่ครอบคลุมสำหรับ PPT**
Aspose.Slides for PHP via Java ให้การสนับสนุนเกือบทุกคุณลักษณะที่รูปแบบไฟล์เอกสาร PPT รองรับ ไม่เพียงแต่ครอบคลุมคุณลักษณะพื้นฐานและขั้นสูงที่ต่างเวอร์ชันของ Microsoft PowerPoint มีให้สำหรับการจัดการเอกสาร PPT เท่านั้น แต่ยังรวมถึงคุณลักษณะที่แม้ Microsoft PowerPoint จะไม่รองรับด้วย ข้อได้เปรียบหลักของการใช้ไลบรารี API ของ Aspose.Slides for PHP via Java คือความง่ายในการใช้งานเพื่อจัดการคุณลักษณะเหล่านี้

นอกจากงานพื้นฐานที่เกี่ยวข้องกับการสร้าง อ่าน และเขียนไฟล์เอกสาร PPT แล้ว ยังมีคุณลักษณะหลายอย่างที่ Aspose.Slides for PHP via Java ให้บริการ

- นำเข้ารูปแบบไฟล์ Microsoft Office อื่นเป็น[วัตถุ OLE เข้าไปในเอกสาร PPT](/slides/th/php-java/manage-ole/).
- [ส่งออกเอกสาร PPT เป็น PDF](/slides/th/php-java/convert-powerpoint-to-pdf/).
- ส่งออกสไลด์ในเอกสาร PPT เป็นรูปแบบ SVG
- เรนเดอร์สไลด์เป็นรูปภาพใด ๆ ที่รองรับโดย Java Framework
- ตั้งค่าขนาดของสไลด์ในเอกสาร PPT
- จัดการแอนิเมชันบนรูปร่าง
- จัดการการแสดงสไลด์โชว์
- [จัดรูปแบบข้อความบนสไลด์](/slides/th/php-java/text-formatting/).
- ดึงข้อความจากเอกสาร PPT
- [จัดการตารางบนสไลด์](/slides/th/php-java/powerpoint-table/).
- คัดลอกมาสตันท์โดยอัตโนมัติด้วย[ฟีเจอร์การโคลน](/slides/th/php-java/clone-slides/).
