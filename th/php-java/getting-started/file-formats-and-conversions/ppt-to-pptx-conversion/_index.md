การแปลงจาก PPT ไปยัง PPTX ได้รับการสนับสนุนจาก Aspose.Slides for PHP via Java คุณลักษณะการนำเสนอส่วนใหญ่ — สไลด์มาสเตอร์ โครงสร้าง ฯลฯ — จะถูกเก็บไว้เมื่อตัวแปลงจากรูปแบบหนึ่งเป็นอีกรูปแบบหนึ่ง แต่มี [a few limitations](/slides/th/php-java/ppt-to-pptx-conversion/).

## **คุณสมบัติที่รองรับในการแปลง**
Aspose.Slides for PHP via Java ให้การสนับสนุนบางส่วนสำหรับการแปลงไฟล์รูปแบบ PPT ไปยัง PPTX การสนับสนุนการแปลงเพิ่งถูกนำเสนอใน Aspose.Slides for PHP via Java ดังนั้นจึงมีข้อจำกัดบางประการและทำงานได้ดีที่สุดกับการนำเสนอแบบง่าย ข้อดีหลักที่ Aspose.Slides for PHP via Java มอบให้เมื่อแปลง PPT เป็น PPTX คือความง่ายในการใช้ API หากต้องการดูตัวอย่างโค้ด อ่านเกี่ยวกับ [Converting PPT to PPTX](). ด้านล่างนี้เป็นรายการที่อธิบายว่าคุณสมบัติใดได้รับการรองรับและคุณสมบัติใดไม่ได้รับการรองรับสำหรับการแปลง PPT เป็น PPTX

## **คุณสมบัติที่รองรับ**
คุณสมบัติต่อไปนี้ได้รับการรองรับสำหรับการแปลง:

- แปลงโครงสร้างของมาสเตอร์, เลย์เอาต์และสไลด์
- แปลงแผนภูมิ
- กลุ่มรูปร่าง
- แปลงออโต้ชเปิลรวมถึงสี่เหลี่ยมผืนผ้าและวงรี
- รูปร่างที่มีรูปทรงกำหนดเอง
- สไตล์การเติมพื้นผิวและรูปภาพสำหรับออโต้ชเปิล
- แปลง placeholder
- แปลงเส้นและโพลีไลน์
- รูปแบบเส้นและการเติม
- สไตล์การเติมแบบไล่สี
- กรอบ OLE, ตาราง, เฟรมวิดีโอและออดิโอ เป็นต้น
- คุณสมบัติของแอนิเมชันและการแสดงสไลด์
- แปลงข้อความในเฟรมข้อความและที่เก็บข้อความ
