## **เกี่ยวกับ TIFF**

รูปแบบไฟล์ [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) ซึ่งเป็นที่รู้จักในการจัดเก็บหลายภาพในเอกสารเดียว ถูกสร้างขึ้นโดย Aldus เดิม format นี้ได้รับการสนับสนุนอย่างกว้างขวางโดยแอปพลิเคชันการสแกน, โทรสาร และการจัดการภาพอื่นๆ

## **TIFF ใน Aspose.Slides สำหรับ PHP ผ่าน Java**

เอกสารใดๆ ที่สามารถโหลดใน Aspose.Slide สำหรับ Java ได้ สามารถแปลงเป็นเอกสาร TIFF โดยตรงโดย Aspose.Slides สำหรับ PHP ผ่าน Java ได้โดยไม่ต้องใช้ส่วนประกอบของบุคคลที่สาม นอกจากนี้ คุณสามารถกำหนดขนาดของภาพในเอกสาร TIFF ที่ได้ตามต้องการ คุณสามารถค้นหาข้อมูลเกี่ยวกับการส่งออกเอกสารการนำเสนอเป็นเอกสาร TIFF ผ่าน Aspose.Slides สำหรับ PHP ผ่าน Java ได้ใน [หัวข้อนี้](/slides/th/php-java/convert-powerpoint-to-tiff/).
