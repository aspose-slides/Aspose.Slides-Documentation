{{% alert color="primary" %}} 

Portable Document Format (PDF) คือรูปแบบไฟล์ที่สร้างโดย Adobe Systems เพื่อใช้แลกเปลี่ยนเอกสารระหว่างองค์กร จุดประสงค์ของรูปแบบนี้คือทำให้เนื้อหาและเค้าโครงคงที่ ไม่ว่าจะแสดงบนแพลตฟอร์มใดก็ตาม Aspose.Slides for PHP via Java ช่วยให้คุณแปลงไฟล์งานนำเสนอเป็น PDF.

{{% /alert %}} 

## **PDF ใน Aspose.Slides for PHP via Java**
งานนำเสนอใด ๆ ที่สามารถโหลดเข้า Aspose.Slides for PHP via Java ได้สามารถแปลงเป็น PDF ที่สอดคล้องกับ [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) หรือ [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA) ตามตัวเลือกของคุณ Aspose.Slides for PHP via Java ส่งออกงานนำเสนอเป็น PDF และในหลาย ๆ กรณี PDF ที่ได้จะดูเหมือนงานนำเสนอเดิมอย่างแม่นยำ

Aspose.Slides รองรับคุณลักษณะของงานนำเสนอต่อไปนี้เมื่อแปลงเป็น PDF:

- รูปภาพ, กล่องข้อความและรูปร่างอื่น ๆ.
- ข้อความและการจัดรูปแบบ.
- ย่อหน้าและการจัดรูปแบบ.
- ไฮเปอร์ลิงก์.
- ส่วนหัวและส่วนท้าย.
- จุดแทนรายการ.
- ตาราง.

คุณสามารถส่งออกงานนำเสนอเป็น PDF โดยตรงด้วย Aspose.Slides for PHP via Java: ไม่จำเป็นต้องใช้ส่วนประกอบอื่นใดเพิ่มเติม ยิ่งไปกว่านั้น คุณสามารถปรับแต่งการส่งออกงานนำเสนอเป็น PDF ด้วยตัวเลือกต่าง ๆ ตามที่อธิบายใน [Converting to PDF](/slides/th/php-java/convert-powerpoint-to-pdf/).
