The [XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) เป็นภาษาการบรรยายหน้ากระดาษและฟอร์แมตเอกสารคงที่ที่พัฒนาโดย Microsoft ตั้งแต่แรกเช่นเดียวกับ PDF, XPS ถูกออกแบบให้รักษาความเที่ยงตรงของเอกสารและให้การแสดงผลของเอกสารที่ไม่ขึ้นกับอุปกรณ์

{{% /alert %}} 

## **XPS ใน Aspose.Slides for PHP via Java**
เอกสารพรีเซนเทชันใด ๆ ที่สามารถโหลดโดย Aspose.Slides for PHP via Java สามารถแปลงเป็นฟอร์แมต XPS ได้ Aspose.Slides for PHP via Java ใช้เอ็นจิ้นการจัดวางหน้าและการเรนเดอร์คุณภาพสูงเพื่อสร้างผลลัพธ์ในรูปแบบเอกสาร XPS แบบคงที่
คุณสามารถเรียนรู้เกี่ยวกับการส่งออกเอกสารพรีเซนเทชันเป็นเอกสาร XPS ผ่าน Aspose.Slides for PHP via Java ได้ใน [Converting to XPS](https://docs.aspose.com/slides/th/php-java/convert-powerpoint-to-xps/).
