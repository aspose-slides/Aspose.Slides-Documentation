**ยินดีต้อนรับสู่ Aspose.Slides สำหรับ C++**

Aspose.Slides for C++ คือ API การจัดการ Microsoft PowerPoint® ที่ทำให้แอปพลิเคชัน C++ สามารถอ่านและเขียนเอกสาร PowerPoint® ได้โดยไม่ต้องใช้ Microsoft PowerPoint® Aspose.Slides for C++ เป็นส่วนประกอบแรกและเพียงเดียวที่ให้ความสามารถในการจัดการเอกสาร PowerPoint® Aspose.Slides for C++ มีคุณลักษณะสำคัญหลายอย่าง เช่น การจัดการข้อความ, รูปร่าง, ตาราง & การเคลื่อนไหว, การเพิ่มเสียงและวิดีโอลงในสไลด์, การแสดงตัวอย่างสไลด์, การส่งออกสไลด์เป็น SVG, PDF และอื่น ๆ
