## **ภาพรวม**

Aspose.Slides รองรับไฟล์การนำเสนอจาก Microsoft PowerPoint 97 จนถึง Office 365 รวมถึง Microsoft PowerPoint สำหรับ Mac บทความนี้แสดงรายการเวอร์ชันของ PowerPoint ที่ไลบรารีสนับสนุนและให้ตารางรูปแบบไฟล์ที่สามารถโหลด บันทึก หรือทั้งสองอย่างได้

บทความนี้ยังตอบคำถามทั่วไปเกี่ยวกับการปฏิบัติตามมาตรฐาน PDF การฝังฟอนต์ ไฟล์ที่มีการป้องกันด้วยรหัสผ่าน ฟอนต์ที่กำหนดเอง การสำรองฟอนต์ และตัวเลือกการส่งออก XPS

## **เวอร์ชัน Microsoft PowerPoint ที่รองรับ**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint สำหรับ Mac
- Office 365

## **คำถามที่พบบ่อย**

**ฉันสามารถบันทึกการนำเสนอเป็น PDF ที่ตรงกับมาตรฐานการเก็บถาวรและการเข้าถึง (PDF/A และ PDF/UA) ได้หรือไม่?**

ใช่. Aspose.Slides รองรับการส่งออกเป็น PDF พร้อมระดับการปฏิบัติตามเช่น PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b รวมถึง PDF/UA ผ่านการตั้งค่า [compliance](https://reference.aspose.com/slides/th/cpp/aspose.slides.export/pdfoptions/set_compliance/) ใน [PDF export options](https://reference.aspose.com/slides/th/cpp/aspose.slides.export/pdfoptions/).

**ไลบรารีนี้รองรับการฝังฟอนต์เมื่อส่งออกเป็น PDF พร้อมการควบคุมละเอียดเกี่ยวกับสิ่งที่ฝังหรือไม่?**

ใช่ คุณสามารถควบคุมได้ว่าฟอนต์จะถูกฝังเต็มหรือเป็นแบบย่อย (เฉพาะ glyph ที่ใช้) ระบุวิธีการจัดการฟอนต์ระบบที่ทั่วไป และกำหนดพฤติกรรมสำหรับข้อความ ASCII ผ่าน [PDF export options](https://reference.aspose.com/slides/th/cpp/aspose.slides.export/pdfoptions/).

**ฉันสามารถตรวจสอบได้ว่าไฟล์มีการป้องกันด้วยรหัสผ่านหรือไม่ก่อนที่จะโหลดจริงหรือไม่?**

ใช่. ด้วยการใช้ [factory-based inspection API](https://reference.aspose.com/slides/th/cpp/aspose.slides/presentationfactory/) คุณสามารถสอบถามไฟล์การนำเสนอเพื่อพิจารณาว่ามีการป้องกันด้วยรหัสผ่านโดยไม่ต้องเปิดไฟล์เต็มที่.

**มีกลไกการสำรองฟอนต์และการสนับสนุนฟอนต์ที่กำหนดเองหรือไม่?**

ใช่. ไลบรารีนี้สนับสนุนการ [loading](/slides/th/cpp/custom-font/) และการ [embedding](/slides/th/cpp/embedded-font/) ฟอนต์ที่กำหนดเอง และให้ [fallback rules](/slides/th/cpp/fallback-font/) สำหรับฟอนต์เพื่อป้องกันการขาด glyph ระหว่างการเรนเดอร์และการแปลง.

**ฉันสามารถส่งออกสไลด์เป็น XPS ได้หรือไม่ และมีตัวเลือกสำหรับปรับแต่งผลลัพธ์ XPS หรือไม่?**

ใช่. [Export to XPS](/slides/th/cpp/convert-powerpoint-to-xps/) ได้รับการสนับสนุน และคุณสามารถปรับ [save options](https://reference.aspose.com/slides/th/cpp/aspose.slides.export/xpsoptions/) ที่เกี่ยวข้องเพื่อควบคุมคุณภาพและเนื้อหาของเอกสาร XPS.
