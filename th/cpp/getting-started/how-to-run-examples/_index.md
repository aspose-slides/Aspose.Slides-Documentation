## **ข้อกำหนดซอฟต์แวร์**
กรุณาตรวจสอบว่าคุณตรงตามข้อกำหนดต่อไปนี้ก่อนดาวน์โหลดและเรียกใช้ตัวอย่าง

1. Visual Studio 2015 หรือเวอร์ชันที่สูงกว่า

## **ดาวน์โหลด Aspose.Slides จาก GitHub**
ตัวอย่างทั้งหมดของ Aspose.Slides สำหรับ C++ ได้รับการโฮสต์บน [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-C)

- คุณสามารถโคลนรีโปซิทอรีโดยใช้ไคลเอนต์ GitHub ที่คุณชื่นชอบหรือดาวน์โหลดไฟล์ ZIP จาก [ที่นี่](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/refs/heads/master.zip).
- คลายเนื้อหาไฟล์ ZIP ไปยังโฟลเดอร์ใดก็ได้บนคอมพิวเตอร์ของคุณ ตัวอย่างทั้งหมดอยู่ในโฟลเดอร์ **Examples**.
- มีไฟล์โซลูชัน Visual Studio สำหรับ C++.
- โครงการถูกสร้างใน Visual Studio 2015 แต่ไฟล์โซลูชันรองรับ Visual Studio 2015 และเวอร์ชันที่สูงกว่า.
- เปิดไฟล์โซลูชันใน Visual Studio และสร้างโครงการ.
- ดาวน์โหลด Aspose.Slides for C++ รุ่นล่าสุดจาก [ส่วนดาวน์โหลด](https://downloads.aspose.com/slides/th/cpp).
- คลายไฟล์ที่ดาวน์โหลดและคัดลอกไดเรกทอรี **Include** และ **Bin** ไปยังโฟลเดอร์ **Examples**.
- เปิดไฟล์ **main.cpp** ตัวอย่างทั้งหมดถูกเรียกจากที่นี่.
- ยกเลิกคอมเมนต์ตัวอย่างที่คุณต้องการรันในโครงการ.
- สร้างและคอมไพล์โซลูชัน

หากคุณมีปัญหาในการตั้งค่า หรือเรียกใช้ตัวอย่าง โปรดติดต่อเราผ่านฟอรั่มของเรา

## **ร่วมสนับสนุน**
หากคุณต้องการเพิ่มหรือปรับปรุงตัวอย่าง เราขอสนับสนุนให้คุณมีส่วนร่วมในโครงการ ตัวอย่างและโครงการแสดงผลทั้งหมดในรีโปซิทอรีนี้เป็นซอร์สเปิดและสามารถใช้ได้อย่างอิสระในแอปพลิเคชันของคุณ

เพื่อมีส่วนร่วม คุณสามารถฟอร์กรีโปซิทอรี แก้ไขซอร์สโค้ดและสร้าง pull request เราจะตรวจสอบการเปลี่ยนแปลงและรวมเข้าไปในรีโปซิทอรีหากเป็นประโยชน์
