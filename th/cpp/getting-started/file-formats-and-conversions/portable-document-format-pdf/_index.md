## **เกี่ยวกับ PDF**
รูปแบบเอกสารพกพา (PDF) คือรูปแบบไฟล์ที่สร้างโดย Adobe System เพื่อแลกเปลี่ยนเอกสารระหว่างองค์กรต่าง ๆ จุดประสงค์ของรูปแบบนี้คือทำให้เนื้อหาในเอกสารสามารถแสดงผลได้โดยไม่ขึ้นกับแพลตฟอร์มที่ใช้ดู

## **PDF ใน Aspose.Slides for C++**
เอกสารพรีเซนเทชันใด ๆ ที่สามารถโหลดเข้า Aspose.Slides for C++ ได้สามารถแปลงเป็นเอกสาร PDF ที่อาจสอดคล้องกับ [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A) หรือ [PDF /A-1b](https://en.wikipedia.org/wiki/PDF/A) ตามที่คุณเลือก Aspose.Slides for C++ จะส่งออกเอกสารพรีเซนเทชันเป็น PDF ในลักษณะที่ส่วนใหญ่แล้วเอกสาร PDF ที่ส่งออกจะดูคล้ายกับเอกสารพรีเซนเทชันต้นฉบับเกือบทั้งหมด
โซลูชันของ Aspose รองรับคุณสมบัติดังต่อไปนี้ของเอกสารพรีเซนเทชันขณะแปลงเป็นเอกสาร PDF:

- รูปภาพ, กล่องข้อความและรูปทรงอื่น ๆ
- ข้อความและการจัดรูปแบบ
- ย่อหน้าและการจัดรูปแบบ
- ลิงก์
- หัวกระดาษและท้ายกระดาษ
- สัญลักษณ์หัวข้อย่อย
- ตาราง

คุณสามารถส่งออกเอกสารพรีเซนเทชันเป็นเอกสาร PDF โดยตรงด้วยส่วนประกอบ Aspose.Slides for C++ อย่างเดียว นั่นหมายความว่า คุณไม่จำเป็นต้องใช้ส่วนประกอบของบุคคลที่สามหรือ Aspose.Pdf สำหรับวัตถุประสงค์นี้ นอกจากนี้คุณยังสามารถปรับแต่งการส่งออกพรีเซนเทชันเป็น PDF ด้วยตัวเลือกต่าง ๆ ตามที่อธิบายใน [หัวข้อนี้](http://docs.aspose.com/display/slidesnet/Converting+to+PDF+File).
