## **Microsoft PowerPoint (PPT)**
### **About PPT**
[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) เป็นรูปแบบไฟล์เอกสารการนำเสนอที่สามารถสร้าง อ่าน แก้ไข และเขียนได้โดยเวอร์ชันต่าง ๆ ของ Microsoft PowerPoint รูปแบบไบนารีนี้พัฒนาโดย Microsoft
### **PPT in Aspose.Slides for C++**
Aspose.Slides for C++ สามารถอ่านไฟล์ PPT ที่สร้างโดยซอฟต์แวร์ด้านล่างนี้

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

ในทำนองเดียวกัน ไฟล์ PPT ที่สร้างโดย Aspose.Slides for C++ สามารถอ่านได้โดยชุดซอฟต์แวร์ดังกล่าวข้างต้น
### **Comprehensive Support for PPT**
Aspose.Slides for C++ ให้การสนับสนุนคุณสมบัติส่วนใหญ่ที่เกี่ยวกับรูปแบบไฟล์เอกสาร PPT ไม่เพียงแต่ครอบคลุมคุณสมบัติพื้นฐาน/ขั้นสูงที่แต่ละเวอร์ชันของ Microsoft PowerPoint ให้ไว้สำหรับการจัดการเอกสาร PPT เท่านั้น แต่ยังรวมถึงคุณสมบัติบางอย่างที่แม้ Microsoft PowerPoint จะไม่สนับสนุนอีกด้วย ข้อดีหลักของการใช้ไลบรารี API Aspose.Slides for C++ คือความง่ายในการจัดการคุณสมบัติเหล่านั้น

นอกจากงานพื้นฐานที่เกี่ยวกับการสร้าง อ่าน และเขียนไฟล์เอกสาร PPT แล้ว ยังมีคุณสมบัติหลายอย่างที่ Aspose.Slides for C++ มีให้ เช่น

- นำเข้ารูปแบบไฟล์ MS Office อื่นเป็นวัตถุ OLE ในเอกสาร PPT
- ส่งออกเอกสาร PPT เป็น PDF, TIFF, XPS
- ส่งออกสไลด์ในเอกสาร PPT เป็นรูปแบบ SVG
- เรนเดอร์สไลด์เป็นรูปแบบภาพใด ๆ ที่ C++ Framework สนับสนุน
- ตั้งค่าขนาดของสไลด์ในเอกสาร PPT
- จัดการแอนิเมชันบนรูปร่าง
- จัดการการแสดงสไลด์
- จัดรูปแบบข้อความบนสไลด์
- สแกนข้อความจากเอกสาร PPT
- จัดการตารางบนสไลด์
- คัดลอกมาสเตอร์อัตโนมัติด้วยคุณสมบัติการโคลน

ไฟล์ PPT ที่สร้างโดย Aspose.Slides for C++ และเปิดใน Microsoft PowerPoint
## **PresentationML (PPTX, XML)**
### **About PresentationML**
PresentationML คือชื่อของตระกูลรูปแบบที่อิง XML สำหรับเอกสารการนำเสนอ Office OpenXML (OOXML) เป็นรูปแบบที่อิง XML ที่แนะนำในแอปพลิเคชัน Microsoft Office 2007 Office OpenXML เป็นรูปแบบคอนเทนเนอร์สำหรับหลายภาษามาร์กอัปที่อิง XML พิเศษ PresentationML เป็นภาษามาร์กอัปที่ Microsoft Office PowerPoint 2007 ใช้เพื่อจัดเก็บเอกสารของมัน
### **PresentationML in Aspose.Slides for C++**
เอกสาร OOXML PresentationML มาในรูปแบบไฟล์ PPTX ซึ่งเป็นแพ็คเกจ XML ที่บีบอัดตามสเปก [OOXML ECMA-376](https://www.ecma-international.org/publications-and-standards/standards/ecma-376/) Aspose.Slides for C++ รองรับการสร้าง อ่าน แก้ไข และเขียนเอกสาร PresentationML อย่างกว้างขวาง นอกจากนี้ Aspose.Slides for C++ ยังสามารถส่งออกเอกสาร PresentationML ไปเป็นรูปแบบเอกสารที่ใช้งานทั่วไปหลายรูปแบบ เช่น PDF, TIFF และ XPS การทำเช่นนี้เป็นไปได้เพราะ Aspose.Slides for C++ ได้ออกแบบโดยมุ่งเน้นการจัดการเอกสารการนำเสนออย่างครอบคลุมและ PresentationML โดยพื้นฐานถือเป็นการนำเสนอภายในของเอกสารในรูปแบบแพ็คเกจ XML ที่บีบอัด

เอกสาร PPTX ที่สร้างโดย Aspose.Slides for C++ และเปิดใน Microsoft PowerPoint

ดูเอกสาร PPTX ที่สร้างโดย Aspose.Slides for C++ ในแอปพลิเคชัน Zip
### **PresentationML is Open, Why Use Aspose.Slides for C++**
เนื่องจาก PresentationML เป็นแบบอิง XML จึงเป็นไปได้อย่างมากที่จะสร้างแอปพลิเคชันสำหรับประมวลผลและสร้างเอกสาร PresentationML โดยใช้คลาส XML โดยไม่ต้องพึ่งพาไลบรารีคลาสของบุคคลภายนอกเช่น Aspose.Slides for C++ อย่างไรก็ตาม มีหลายข้อได้เปรียบของการใช้ Aspose.Slides for C++ แทนคลาส XML ขณะทำงานกับเอกสาร PresentationML

สเปก OOXML มีความยาวหลายพันหน้า หมายความว่าการจัดการเอกสาร PresentationML อย่างถูกต้อง คุณต้องใช้เวลาและความพยายามอย่างมากเพื่อทำความเข้าใจรูปแบบของเอกสารเหล่านั้น ในทางกลับกัน ขณะใช้ Aspose.Slides for C++ คุณเพียงแค่ใช้คลาสและเมธอด/พร็อพเพอร์ตีที่เกี่ยวข้องในการทำงานที่ดูซับซ้อนหากทำผ่านคลาส XML

ต่อไปนี้คือบางคุณสมบัติที่แม้ไม่สามารถใช้ได้เมื่อจัดการเอกสาร PresentationML ผ่านคลาส XML

- ส่งออกเอกสาร PPT เป็น PDF, TIFF, XPS
- ส่งออกสไลด์ในเอกสาร PPT เป็น SVG
- เรนเดอร์สไลด์เป็นรูปแบบภาพใด ๆ ที่ C++ Framework สนับสนุน
- คัดลอกมาสเตอร์อัตโนมัติจากการนำเสนอแหล่งโดยใช้คุณสมบัติการโคลน
- ใช้การป้องกันบนรูปร่าง

ให้ถือเป็นตัวอย่างของเอกสาร PresentationML ที่มีสไลด์เดียวพร้อมกล่องข้อความที่มีข้อความ “Hello World” เพื่ออ่านข้อความผ่านคลาส XML คุณจะต้องเขียนโปรแกรมที่สามารถแยกข้อความง่าย ๆ นี้จากส่วนที่ต่อไปนี้

## **PPT to PPTX conversion**
### **About Conversion**
Aspose.Slides ปัจจุบันยังรองรับการแปลง PPT เป็น PPTX
### **Features Supported in Conversion**
Aspose.Slides for C++ ให้การสนับสนุนบางส่วนสำหรับการแปลงการนำเสนอรูปแบบไฟล์ PPT ไปเป็นการนำเสนอรูปแบบไฟล์ PPTX เนื่องจากคุณสมบัติการแปลงการนำเสนอที่กล่าวถึงเพิ่งถูกนำมาใช้ใน Aspose.Slides for C++ ดังนั้นขณะนี้ยังมีความสามารถจำกัดและทำงานได้เฉพาะรูปแบบการนำเสนออย่างง่าย ข้อดีหลักของไลบรารี API Aspose.Slides for C++ สำหรับการแปลงการนำเสนอ PPT ไปเป็น PPTX คือความง่ายในการใช้ API เพื่อบรรลุเป้าหมายที่ต้องการ โปรดไปที่ this[link]() เพื่อส่วนโค้ดตัวอย่างสำหรับรายละเอียดเพิ่มเติม ส่วนต่อไปนี้แสดงอย่างชัดเจนว่าคุณสมบัติใดสนับสนุนและไม่สนับสนุนขณะแปลงการนำเสนอรูปแบบ PPT ไปเป็น PPTX
### **Supported Features**
คุณสมบัติต่อไปนี้ได้รับการสนับสนุนในระหว่างการแปลง

- การแปลงโครงสร้างของมาสเตอร์ เลย์เอาต์ และสไลด์
- การแปลงโครงสร้างของมาสเตอร์ เลย์เอาต์ และสไลด์
- การแปลงแผนภูมิ
- รูปทรงกลุ่ม
- การแปลง Auto‑shapes รวมถึงสี่เหลี่ยมและวงรี อย่างไรก็ตามค่าอัตราส่วนอาจไม่ถูกต้อง
- รูปทรงที่มีเรขาคณิตกำหนดเอง บางครั้งอาจไม่ถูกแปลง
- สไตล์การเติมพื้นผิวเทกเจอร์และรูปภาพสำหรับ Auto‑shapes บางครั้งอาจไม่ถูกแปลง
- การแปลง Placeholder
- การแปลงข้อความในกรอบข้อความและตัวถือข้อความ อย่างไรก็ตามหัวข้อย่อย การจัดแนว และการตั้งตารางยังไม่ได้ทำอย่างเต็มที่
### **Unsupported Features**
คุณสมบัติดังต่อไปนี้ไม่สนับสนุนในการแปลง

- สไลด์ที่มีโน้ต เนื่องจากการอ่านโน้ตยังไม่ได้ดำเนินการใน PPTX หาก PPT มีโน้ต จะไม่สามารถบันทึกเป็น PPTX ได้ในขณะนี้
- การแปลงเส้นและโพลีไลน์
- รูปแบบเส้นและการเติม
- สไตล์การเติมแบบไล่สี
- กรอบ OLE ตาราง วิดีโอ และเฟรมเสียง ฯลฯ
- แอนิเมชันและคุณสมบัติการสไลด์โชว์อื่น ๆ จะถูกข้าม
  คุณลักษณะใหม่หรือที่ขาดหายจะถูกเพิ่มในรุ่นต่อไปของ Aspose.Slides for C++.

## **Portable Document Format (PDF)**
### **About PDF**
[รูปแบบเอกสารพกพา (PDF)](https://en.wikipedia.org/wiki/PDF) คือรูปแบบไฟล์ที่สร้างโดย Adobe System เพื่อแลกเปลี่ยนเอกสารระหว่างองค์กรต่าง ๆ วัตถุประสงค์ของรูปแบบนี้คือทำให้เนื้อหาเอกสารสามารถแสดงผลได้เหมือนเดิมไม่ว่าจะแสดงบนแพลตฟอร์มใด
### **PDF in Aspose.Slides for C++**
เอกสารการนำเสนอใด ๆ ที่สามารถโหลดเข้ามาใน Aspose.Slides for C++ สามารถแปลงเป็นเอกสาร PDF ที่อาจสอดคล้องกับ [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A) หรือ [PDF /A-1b](https://en.wikipedia.org/wiki/PDF/A) ตามที่คุณเลือก Aspose.Slides for C++ ส่งออกเอกสารการนำเสนอเป็น PDF ในลักษณะที่ส่วนใหญ่ PDF ที่ส่งออกแล้วจะมีลักษณะคล้ายกับเอกสารการนำเสนอเดิมมากที่สุด โซลูชันของ Aspose รองรับคุณสมบัติดังต่อไปนี้ของเอกสารการนำเสนอขณะแปลงเป็น PDF

- รูปภาพ กล่องข้อความและรูปร่างอื่น ๆ
- ข้อความและการจัดรูปแบบ
- ย่อหน้าและการจัดรูปแบบ
- ไฮเปอร์ลิงก์
- ส่วนหัวและส่วนท้าย
- จุดแสดงหัวข้อ
- ตาราง

คุณสามารถส่งออกเอกสารการนำเสนอเป็น PDF โดยตรงด้วยส่วนประกอบ Aspose.Slides for C++ เท่านั้น ไม่จำเป็นต้องใช้ส่วนประกอบของบุคคลภายนอกหรือ Aspose.Pdf สำหรับวัตถุประสงค์นี้ อีกทั้งคุณยังสามารถปรับแต่งการส่งออกการนำเสนอเป็น PDF ด้วยตัวเลือกต่าง ๆ ตามที่อธิบายไว้ใน [this topic](/slides/th/cpp/convert-powerpoint-to-pdf/).

เอกสารการนำเสนอที่แปลงเป็นเอกสาร PDF ผ่าน Aspose.Slides for C++
## **XML Parser Specification (XPS)**
### **About XPS**
[XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) เป็นภาษาการอธิบายหน้าและรูปแบบเอกสารคงที่ที่พัฒนาโดย Microsoft เช่นเดียวกับ PDF XPS เป็นรูปแบบเอกสารที่มีการจัดวางคงที่ออกแบบเพื่อรักษาความเที่ยงตรงของเอกสารและให้ลักษณะการแสดงผลเป็นอิสระจากอุปกรณ์
### **XPS in Aspose.Slides for C++**
เอกสารการนำเสนอใด ๆ ที่สามารถโหลดด้วย Aspose.Slides for C++ สามารถแปลงเป็นรูปแบบ XPS ได้ Aspose.Slides for C++ ใช้เครื่องยนต์การจัดวางหน้าและการเรนเดอร์ที่คงความเที่ยงตรงเพื่อผลิตผลลัพธ์ในรูปแบบเอกสาร XPS ที่คงที่ ควรกล่าวถึงว่า Aspose.Slides for C++ สร้าง XPS โดยตรงโดยไม่ต้องพึ่งพาคลาส Windows Presentation Foundation (WPF) ที่บรรจุมาพร้อมกับ C++ Framework 3.5 ทำให้ Aspose.Slides for C++ สามารถผลิตเอกสาร XPS บนเครื่องที่รัน C++ Framework รุ่นก่อน 3.5 ได้ คุณสามารถเรียนรู้การส่งออกเอกสารการนำเสนอเป็น XPS ผ่าน Aspose.Slides for C++ ใน [this topic](https://docs.aspose.com/slides/th/cpp/convert-powerpoint-to-xps/).

เอกสารการนำเสนอที่แปลงเป็นเอกสาร XPS ผ่าน Aspose.Slides for C++
