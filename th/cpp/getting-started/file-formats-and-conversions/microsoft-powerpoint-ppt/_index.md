## **เกี่ยวกับ PPT**
[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint) คือรูปแบบไฟล์เอกสารการนำเสนอที่สามารถสร้าง อ่าน แก้ไข และเขียนได้โดยรุ่นต่างๆ ของ Microsoft PowerPoint นี่เป็นรูปแบบไบนารีสำหรับเอกสารการนำเสนอที่พัฒนาโดย Microsoft. 

## **PPT ใน Aspose.Slides for C++**
Aspose.Slides for C++ สามารถอ่านไฟล์ PPT ที่สร้างโดยซอฟต์แวร์ที่ระบุด้านล่าง. 

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

เช่นเดียวกัน ไฟล์ PPT ที่สร้างโดย Aspose.Slides for C++ สามารถอ่านได้โดยซอฟต์แวร์ชุดด้านบน. 

## **การสนับสนุน PPT อย่างครอบคลุม**
Aspose.Slides for C++ ให้การสนับสนุนคุณลักษณะเกือบทั้งหมดที่เกี่ยวกับรูปแบบไฟล์เอกสาร PPT ไม่เพียงแต่ครอบคลุมคุณลักษณะพื้นฐาน/ขั้นสูงที่ต่างรุ่นของ Microsoft PowerPoint ให้สำหรับการจัดการเอกสาร PPT เท่านั้น แต่ยังรวมถึงคุณลักษณะบางอย่างที่แม้ Microsoft PowerPoint จะไม่รองรับด้วย ข้อดีหลักของการใช้ไลบรารี API Aspose.Slides for C++ คือความง่ายในการจัดการคุณลักษณะเหล่านี้. 

นอกเหนือจากงานพื้นฐานที่เกี่ยวกับการสร้าง อ่าน และเขียนไฟล์เอกสาร PPT แล้ว ยังมีคุณลักษณะหลายอย่างที่ Aspose.Slides for C++ ให้บริการเช่น: 

- นำเข้าไฟล์ฟอร์แมต MS Office อื่นเป็น OLE Object ในเอกสาร PPT
- ส่งออกเอกสาร PPT ไปยังรูปแบบ PDF, TIFF, XPS
- ส่งออกสไลด์ในเอกสาร PPT ไปยังรูปแบบ SVG
- เรนเดอร์สไลด์เป็นรูปแบบภาพใดก็ได้ที่รองรับโดย C++ Framework
- กำหนดขนาดของสไลด์ในเอกสาร PPT
- จัดการแอนิเมชันบนรูปร่าง
- จัดการการแสดงสไลด์
- จัดรูปแบบข้อความบนสไลด์
- สแกนข้อความจากเอกสาร PPT
- จัดการตารางบนสไลด์
- คัดลอกมาสเตอร์อัตโนมัติด้วยฟีเจอร์การโคลน
