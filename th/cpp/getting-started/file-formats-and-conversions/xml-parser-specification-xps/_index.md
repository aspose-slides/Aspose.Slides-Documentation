## **เกี่ยวกับ XPS**
The [XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) เป็นภาษาการบรรยายหน้าและรูปแบบเอกสารคงที่ที่พัฒนาโดย Microsoft เดิม เช่นเดียวกับ PDF, XPS เป็นรูปแบบเอกสารแบบจัดวางคงที่ที่ออกแบบมาเพื่อรักษาความแม่นยำของเอกสารและให้ลักษณะการแสดงเอกสารที่ไม่ขึ้นกับอุปกรณ์
## **XPS ใน Aspose.Slides for C++**
เอกสารงานนำเสนอใด ๆ ที่สามารถโหลดโดย Aspose.Slides for C++ สามารถแปลงเป็นรูปแบบ XPS ได้ Aspose.Slides for C++ ใช้เครื่องยนต์จัดวางหน้าและการเรนเดอร์ที่มีความแม่นยำสูงเพื่อสร้างผลลัพธ์ในรูปแบบเอกสาร XPS แบบจัดวางคงที่
ควรจะกล่าวว่า Aspose.Slides for C++ สร้าง XPS โดยตรงโดยไม่ต้องพึ่งพาชั้น Windows Presentation Foundation (WPF) ที่บรรจุใน C++ Framework 3.5 ทำให้ Aspose.Slides for C++ สามารถผลิตเอกสาร XPS บนเครื่องที่ใช้ C++ Framework รุ่นก่อนหน้าเวอร์ชัน 3.5 คุณสามารถเรียนรู้วิธีส่งออกเอกสารงานนำเสนอเป็นเอกสาร XPS ผ่าน Aspose.Slides for C++ ใน [หัวข้อนี้](/slides/th/net/).
