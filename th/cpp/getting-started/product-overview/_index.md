## **ยินดีต้อนรับสู่ Aspose.Slides สำหรับ C++!**

Aspose.Slides สำหรับ C++ เป็น API การจัดการ Microsoft PowerPoint® ที่ช่วยให้แอปพลิเคชัน C++ สามารถอ่านและเขียนเอกสาร PowerPoint® ได้โดยไม่ต้องใช้ Microsoft PowerPoint® Aspose.Slides สำหรับ C++ เป็นคอมโพเนนต์แรกและ唯一ที่ให้ฟังก์ชันการจัดการเอกสาร PowerPoint® Aspose.Slides สำหรับ C++ มีคุณลักษณะสำคัญหลายอย่าง เช่น การจัดการข้อความ รูปร่าง ตารางและแอนิเมชัน การเพิ่มเสียงและวิดีโอลงในสไลด์ การแสดงตัวอย่างสไลด์ การส่งออกสไลด์เป็น SVG, PDF และอื่น ๆ

## **คำอธิบายผลิตภัณฑ์**
Aspose.Slides สำหรับ C++ พัฒนาด้วย Native C++ ปัจจุบันคอมไพล์สำหรับสภาพแวดล้อม Windows x64 แต่จะคอมไพล์สำหรับสภาพแวดล้อม Linux ด้วย

Aspose.Slides สำหรับ C++ ไม่เพียงเปิดไฟล์ PowerPoint จากแหล่งต่าง ๆ เท่านั้น แต่ยังอนุญาตให้คุณบันทึกงานนำเสนอได้หลายวิธี ตัวอย่างเช่น คุณสามารถบันทึกงานนำเสนอเป็น PPT, PDF หรือบันทึกสไลด์เป็นภาพได้ Aspose.Slides สำหรับ C++ ให้คุณเข้าถึงคุณลักษณะเกือบทั้งหมดที่พบใน Microsoft PowerPoint ความหลากหลายของ Aspose.Slides สำหรับ C++ คือคุณสามารถเพิ่มตารางและรูปร่างหลากหลายประเภทลงในสไลด์ รวมถึงเฟรมประเภทต่าง ๆ (เช่น ข้อความ, เสียงและวิดีโอ)

## **ชุดคุณลักษณะที่หลากหลาย**
คุณลักษณะของ Aspose.Slides สำหรับ C++ สามารถแบ่งออกเป็นพื้นที่ต่อไปนี้

1. คุณลักษณะของงานนำเสนอ
1. คุณลักษณะของสไลด์
1. คุณลักษณะของกราฟิกและมัลติมีเดีย
1. คุณลักษณะของตาราง
1. คุณลักษณะของข้อความและย่อหน้า
1. คุณลักษณะขั้นสูง

### **คุณลักษณะของงานนำเสนอ**
- เปิดงานนำเสนอ Microsoft PowerPoint จากแหล่งต่าง ๆ
- บันทึกงานนำเสนอในรูปแบบ Microsoft PowerPoint ต่าง ๆ
- สร้างงานนำตั้งแต่ต้น
- แปลงงานนำเสนอเป็นเอกสาร PDF โดยไม่ใช้คอมโพเนนต์อื่น
- อ่านหรือแก้ไขคุณสมบัติของเอกสารงานนำเสนอ
- ใช้การป้องกันบนงานนำเสนอ
- ส่งออกงานนำเสนอเป็น PDF
- ส่งออกงานนำเสนอเป็น XPS
- ส่งออกงานนำเสนอเป็น SWF
- ส่งออกบันทึกย่อของงานนำเสนอเป็น PDF
- ส่งออกงานนำเสนอเป็น HTML

### **คุณลักษณะของสไลด์**
- สร้างสไลด์ใหม่ตั้งแต่ต้นและเพิ่มลงในงานนำเสนอ
- เข้าถึงสไลด์ใดก็ได้ผ่านตำแหน่งของมันในงานนำเสนอ
- คัดลอกหรือโคลนสไลด์ไปยังงานนำเสนอเดียวกันหรืออื่น ๆ
- แก้ไขและลบสไลด์
- คัดลอกมาสเตอร์พร้อมสไลด์โดยอัตโนมัติ
- แสดงตัวอย่างสไลด์
- รับหรือกำหนดชื่อสไลด์
- บันทึกสไลด์เป็นภาพ
- ส่งออกสไลด์เป็นรูปแบบ SVG
- ทำงานกับมาสเตอร์หลายตัวในงานนำเสนอ
- เติมพื้นหลังของสไลด์ด้วยสไตล์ที่รองรับ
- เพิ่มหรือจัดการบันทึกย่อของสไลด์
- ตั้งค่ามาสเตอร์และสไตล์ของสไลด์ใหม่แบบไดนามิก
- เพิ่มหรือแก้ไขการเปลี่ยนภาพสไลด์โชว์
- เพิ่มการเปลี่ยนภาพแบบ Morph

### **คุณลักษณะของกราฟิก & มัลติมีเดีย**
- สร้างหรือจัดการรูปร่างเช่น เส้น, สี่เหลี่ยม, วงรีและโพลีไลน์
- จัดการสไตล์ของเส้นในรูปร่าง
- รับข้อความแทนของรูปร่าง
- หมุนหรือพลิกรูปร่าง
- เติมรูปร่างด้วยสไตล์ต่าง ๆ เช่น แน่น, ไล่สี, แพทเทิร์น, เทกเจอร์และรูปภาพ
- ใช้แพทเทิร์นที่กำหนดล่วงหน้า 48 แบบและเทกเจอร์ 24 แบบ
- กำหนดลิงก์ให้กับรูปร่าง
- อ่าน, แก้ไขหรือเพิ่มเอฟเฟกต์แอนิเมชันใหม่ให้กับสไลด์และรูปร่าง
- เพิ่มรูปภาพใหม่ลงในงานนำเสนอหรือจัดการรูปที่มีอยู่
- เพิ่มจุดหมายรูปภาพใหม่ลงในงานนำเสนอหรือจัดการจุดหมายรูปที่มีอยู่
- เพิ่มเฟรมเสียงใหม่ลงในงานนำเสนอหรือจัดการเฟรมเสียงที่มีอยู่
- เพิ่มเฟรมวิดีโอใหม่ลงในงานนำเสนอหรือจัดการเฟรมวิดีโอที่มีอยู่

### **คุณลักษณะของตาราง**
- สร้างตารางตั้งแต่ต้นและเพิ่มลงในสไลด์
- ปรับขนาดและย้ายตาราง
- เข้าถึงเซลล์ใดก็ได้ในตาราง
- เปลี่ยนรูปแบบและเส้นขอบของเซลล์ใดก็ได้
- ปรับขนาดคอลัมน์และแถว
- เพิ่มและลบคอลัมน์และแถว
- กำหนดความกว้างและสีของเส้นขอบสำหรับตารางทั้งหมด

### **คุณลักษณะของข้อความ & ย่อหน้า**
- รับและกำหนดข้อความในตัวแทนข้อความและเฟรมข้อความ
- รับประเภทของตัวแทนข้อความ
- เปลี่ยนการจัดรูปแบบข้อความในเฟรมข้อความ
- เพิ่มหรือจัดการแบบอักษรที่ใช้ในงานนำเสนอ
- เปลี่ยนข้อความและการจัดรูปแบบข้อความ
- สร้างหรือจัดการลิงก์ในข้อความ
- รองรับแท็บและแท็บที่มีประสิทธิภาพ
- เพิ่มเฟรมข้อความใหม่ลงในรูปร่างเกือบทุกประเภท
- เข้าถึงและแก้ไขค่าขอบและการห่อข้อความของเฟรมข้อความ
- เข้าถึงและแก้ไขการเยื้องของย่อหน้า

### **คุณลักษณะขั้นสูง**
- เข้าถึงวัตถุ OLE ที่ฝังอยู่ในงานนำเสนอ
- ล็อคงานนำเสนอ

## **แพลตฟอร์มที่รองรับ**
- Windows x64/x86 (Microsoft Visual Studio 2017 หรือใหม่กว่า)
- Linux (Clang 3.9 หรือใหม่กว่า, GCC 6.1 หรือใหม่กว่า)
- macOS (Xcode 13.4 หรือใหม่กว่า)

## **การสนับสนุนทางเทคนิค**
Aspose ให้การสนับสนุนทางเทคนิคฟรีไม่จำกัดสำหรับผลิตภัณฑ์ทั้งหมดของตน การสนับสนุนเปิดให้กับผู้ใช้ทุกคนรวมถึงผู้ประเมิน หากคุณต้องการความช่วยเหลือเกี่ยวกับ Aspose.Slides สำหรับ C++ ให้พิจารณาตัวเลือกต่อไปนี้

- ช่องทางหลักสำหรับการสนับสนุนคือ[Aspose. Forums](https://forum.aspose.com/). โพสต์คำถามของคุณใน[Aspose.Slides forum](https://forum.aspose.com/c/slides/th/11)และจะได้รับคำตอบภายในไม่กี่ชั่วโมง ทีมสนับสนุน Aspose.Slides จะตอบโดยตรง
- โปรดทราบว่า Aspose ไม่ให้การสนับสนุนทางเทคนิคผ่านโทรศัพท์ การสนับสนุนทางโทรศัพท์มีให้เฉพาะสำหรับคำถามเกี่ยวกับการขายและการซื้อเท่านั้น
- เมื่อคาดว่าจะได้รับการตอบกลับในฟอรั่ม โปรดคำนึงถึงความแตกต่างของเขตเวลา

หากคุณมีปัญหาเกี่ยวกับ Aspose.Slides สำหรับ C++ ให้ทำตามขั้นตอนง่าย ๆ นี้เพื่อให้มั่นใจว่าปัญหาจะได้รับการแก้ไขอย่างมีประสิทธิภาพ

- ตรวจสอบให้แน่ใจว่าคุณใช้รุ่นล่าสุดของ Aspose.Slides สำหรับ C++ ก่อนรายงานปัญหา ดูที่[Aspose.Slides download](https://downloads.aspose.com/slides/th/cpp)เพื่อค้นหารุ่นล่าสุด

## **คำถามที่พบบ่อย**

**รองรับระดับการปฏิบัติตามมาตรฐาน PDF สำหรับการจัดเก็บและการเข้าถึง (PDF/A และ PDF/UA) หรือไม่?**

ใช่ คุณสามารถบันทึกเป็น PDF ด้วย PDF/A‑2a/2b/2u, PDF/A‑3a/3b รวมถึง PDF/UA ได้โดยกำหนดค่า[PDF export options](https://reference.aspose.com/slides/th/cpp/aspose.slides.export/pdfoptions/)

**มีกลไกการแทนที่แบบอักษรและการสนับสนุนแบบอักษรกำหนดเองเพื่อให้การแสดงผลถูกต้องหรือไม่?**

ใช่ ไลบรารีอนุญาตให้คุณ[load custom fonts](/slides/th/cpp/custom-font/)และ[define fallback rules](/slides/th/cpp/fallback-font/)เพื่อให้ glyph ที่ขาดหายถูกแทนที่อย่างน่าเชื่อถือระหว่างการแปลงและการแสดงผล

**ฉันสามารถตรวจจับว่าไฟล์ถูกป้องกันด้วยรหัสผ่านโดยไม่ต้องเปิดเต็มได้หรือไม่?**

ใช่ คุณสามารถ[inspect a presentation](/slides/th/cpp/examine-presentation/)เพื่อพิจารณาว่าต้องการรหัสผ่านก่อนโหลดเอกสารเต็มหรือไม่

**จำเป็นต้องมี Microsoft PowerPoint สำหรับการประมวลผลและการแปลงหรือไม่?**

ไม่ Aspose.Slides เป็นเอนจินอิสระ; ไม่จำเป็นต้องมี PowerPoint บนเซิร์ฟเวอร์หรือเครื่องทำงาน

**ปลอดภัยหรือไม่ที่จะประมวลผลงานนำเสนอจากหลายเธรดพร้อมกัน?**

ใช่ คุณสามารถประมวลผลเอกสารต่าง ๆ ในเธรดแยกกันได้; เพียงหลีกเลี่ยงการใช้อินสแตนซ์งานนำเสนอเดียวกันพร้อมกัน[across threads](/slides/th/cpp/multithreading/)

**การทำแมคโครจะคงไว้และฉันสามารถจัดการ VBA ในไฟล์ PPTM/PPSM ได้หรือไม่?**

ใช่ งานนำเสนอที่มีแมคโคร[are supported](/slides/th/cpp/presentation-via-vba/)และคุณสามารถ[inspect and manage VBA projects](https://reference.aspose.com/slides/th/cpp/aspose.slides.vba/)ในไฟล์เหล่านั้น

**ฉันสามารถแปลง PDF หรือ HTML กลับเป็นสไลด์ PowerPoint ได้หรือไม่?**

ใช่ คุณสามารถ[import PDF pages or HTML content](/slides/th/cpp/import-presentation/)เพื่อสร้างหรือเติมสไลด์ในงานนำเสนอ

**รองรับการส่งออก XPS หรือไม่และฉันสามารถควบคุมคุณภาพและเนื้อหาของผลลัพธ์ XPS ได้หรือไม่?**

ใช่ [Export to XPS](/slides/th/cpp/convert-powerpoint-to-xps/)พร้อมให้ใช้ได้ และ[save options](https://reference.aspose.com/slides/th/cpp/aspose.slides.export/xpsoptions/)ช่วยให้คุณปรับคุณภาพและเนื้อหาที่รวมไว้ได้

**ฉันสามารถแปลงสไลด์เป็นภาพและควบคุมคุณภาพของผลลัพธ์ได้หรือไม่?**

ใช่ สไลด์สามารถ[rendered to PNG, JPEG, GIF, BMP, TIFF](/slides/th/cpp/convert-powerpoint-to-png/)และรูปแบบอื่น ๆ ด้วยการควบคุมละเอียดของขนาดและคุณภาพ

**รองรับสูตรคณิตศาสตร์ (MathML/MathText) และสามารถส่งออกได้หรือไม่?**

ใช่ คุณสามารถ[create and edit mathematical text](/slides/th/cpp/powerpoint-math-equations/)และ[export equations](/slides/th/cpp/exporting-math-equations/)รวมถึงไปยัง MathML

**ไลบรารีจัดการการเข้ารหัสและรหัสผ่าน (เปิด, ตั้งค่า, ลบ) อย่างไร?**

[It supports](/slides/th/cpp/password-protected-presentation/)การเปิดงานนำเสนอที่เข้ารหัส การตั้งหรือการลบรหัสผ่านสำหรับการเปิดและการแก้ไข และการตรวจสอบสถานะการป้องกันของไฟล์
