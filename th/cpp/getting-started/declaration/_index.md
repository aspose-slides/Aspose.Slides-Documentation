ส่วนประกอบ Aspose C++ ทั้งหมดต้องการชุดสิทธิ์ Full Trust. เหตุผลคือ ส่วนประกอบ Aspose C++ จำเป็นต้องเข้าถึงการตั้งค่าจดหมายทะเบียนและไฟล์ระบบที่อยู่นอกไดเรกทอรีเสมือนสำหรับการทำงานบางอย่าง เช่นการแยกวิเคราะห์ฟอนต์ เป็นต้น. นอกจากนี้ ส่วนประกอบ Aspose C++ ยังอิงจากคลาสระบบ C++ หลักที่ในหลายกรณีก็ต้องการชุดสิทธิ์ Full Trust ด้วย.
