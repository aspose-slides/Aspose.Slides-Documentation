## **ภาพรวม**

บทความนี้อธิบายวิธีจัดรูปแบบแผนภูมิในงานนำเสนอ PowerPoint ด้วย Aspose.Slides โดยแสดงวิธีปรับแต่งส่วนสำคัญของแผนภูมิเช่น แกน, เส้นกริด, ชื่อเรื่อง, คำอธิบาย, พื้นที่พล็อต, และการเติมสีผนัง เพื่อปรับปรุงลักษณะและความอ่านง่ายของข้อมูลแผนภูมิ

นอกจากนี้ยังสาธิตวิธีตั้งค่าคุณสมบัติตัวอักษรสำหรับข้อความแผนภูมิ, ใช้รูปแบบตัวเลขที่กำหนดไว้ล่วงหน้าและแบบกำหนดเองกับข้อมูลแผนภูมิ, และเปิดใช้งานมุมโค้งสำหรับพื้นที่แผนภูมิ ตัวอย่างเหล่านี้แสดงวิธีควบคุมทั้งสไตล์ภาพและการนำเสนอข้อมูลของแผนภูมิในงานนำเสนอ

## **จัดรูปแบบเอนทิตี้ของแผนภูมิ**
Aspose.Slides for C++ ให้ผู้พัฒนาเพิ่มแผนภูมิคัสตอมลงในสไลด์จากศูนย์ บทความนี้อธิบายวิธีจัดรูปแบบเอนทิตี้แผนภูมิต่าง ๆ รวมถึงแกนหมวดหมู่และแกนค่าของแผนภูมิ

Aspose.Slides for C++ มี API ที่ง่ายสำหรับจัดการเอนทิตี้แผนภูมิต่าง ๆ และจัดรูปแบบโดยใช้ค่าคัสตอม:

1. สร้างอินสแตนซ์ของคลาส **Presentation**.
1. รับอ้างอิงสไลด์โดยใช้ดัชนีของมัน.
1. เพิ่มแผนภูมิกับข้อมูลเริ่มต้นพร้อมประเภทที่ต้องการ (ในตัวอย่างนี้เราจะใช้ ChartType.LineWithMarkers).
1. เข้าถึง Value Axis ของแผนภูมิและตั้งค่าคุณสมบัติดังต่อไปนี้:
   1. ตั้งค่า **Line format** สำหรับเส้นกริดหลักของ Value Axis
   1. ตั้งค่า **Line format** สำหรับเส้นกริดย่อยของ Value Axis
   1. ตั้งค่า **Number Format** สำหรับ Value Axis
   1. ตั้งค่า **Min, Max, Major and Minor units** สำหรับ Value Axis
   1. ตั้งค่า **Text Properties** สำหรับข้อมูล Value Axis
   1. ตั้งค่า **Title** สำหรับ Value Axis
   1. ตั้งค่า **Line Format** สำหรับ Value Axis
1. เข้าถึง Category Axis ของแผนภูมิและตั้งค่าคุณสมบัติดังต่อไปนี้:
   1. ตั้งค่า **Line format** สำหรับเส้นกริดหลักของ Category Axis
   1. ตั้งค่า **Line format** สำหรับเส้นกริดย่อยของ Category Axis
   1. ตั้งค่า **Text Properties** สำหรับข้อมูล Category Axis
   1. ตั้งค่า **Title** สำหรับ Category Axis
   1. ตั้งค่า **Label Positioning** สำหรับ Category Axis
   1. ตั้งค่า **Rotation Angle** สำหรับป้ายกำกับ Category Axis
1. เข้าถึง Legend ของแผนภูมิและตั้งค่า **Text Properties** สำหรับมัน
1. ตั้งค่าให้แสดง Legend ของแผนภูมิโดยไม่ทับซ้อนกับแผนภูมิ
1. เข้าถึง **Secondary Value Axis** ของแผนภูมิและตั้งค่าคุณสมบัติดังต่อไปนี้:
   1. เปิดใช้ **Value Axis** รอง
   1. ตั้งค่า **Line Format** สำหรับ Secondary Value Axis
   1. ตั้งค่า **Number Format** สำหรับ Secondary Value Axis
   1. ตั้งค่า **Min, Max, Major and Minor units** สำหรับ Secondary Value Axis
1. ตอนนี้พล็อตซีรีส์แรกของแผนภูมิบน Secondary Value Axis
1. ตั้งค่าผนังด้านหลังของแผนภูมิให้เป็นสีเติม
1. ตั้งค่าสีเติมของพื้นที่พล็อตของแผนภูมิ
1. เขียนงานนำเสนอที่แก้ไขแล้วเป็นไฟล์ PPTX

## **ตั้งค่ารูปแบบตัวเลข**
Aspose.Slides for C++ มี API ที่ง่ายสำหรับจัดการรูปแบบข้อมูลของแผนภูมิ:

1. สร้างอินสแตนซ์ของคลาส [Presentation](https://reference.aspose.com/slides/th/cpp/aspose.slides/presentation/) .
1. รับอ้างอิงสไลด์โดยใช้ดัชนีของมัน.
1. เพิ่มแผนภูมิกับข้อมูลเริ่มต้นพร้อมประเภทที่ต้องการ (ตัวอย่างนี้ใช้ **ChartType.ClusteredColumn**).
1. ตั้งค่ารูปแบบจำนวนที่กำหนดไว้ล่วงหน้าจากค่าที่เป็นไปได้.
1. ไล่ผ่านเซลล์ข้อมูลของแผนภูมิในแต่ละซีรีส์และตั้งค่ารูปแบบจำนวนของข้อมูลแผนภูมิ.
1. บันทึกงานนำเสนอ.
1. ตั้งค่ารูปแบบจำนวนแบบกำหนดเอง.
1. ไล่ผ่านเซลล์ข้อมูลของแผนภูมิในแต่ละซีรีส์และกำหนดรูปแบบจำนวนข้อมูลแผนภูมิที่แตกต่างกัน.
1. บันทึกงานนำเสนอ.

## **คำถามที่พบบ่อย**

**ฉันสามารถตั้งค่าการเติมสีกึ่งโปร่งใสสำหรับคอลัมน์/พื้นที่ในขณะที่ยังคงขอบสีทึบได้หรือไม่?**

ได้ การตั้งค่าความโปร่งใสของการเติมและเส้นขอบทำแยกกัน ซึ่งเป็นประโยชน์สำหรับการปรับปรุงการอ่านของกริดและข้อมูลในภาพที่มีความหนาแน่นสูง

**ฉันจะจัดการกับป้ายข้อมูลที่ซ้อนทับกันอย่างไร?**

ลดขนาดฟอนต์, ปิดการใช้งานส่วนประกอบป้ายที่ไม่จำเป็น (เช่น หมวดหมู่), ตั้งค่าการเคลื่อนที่/ตำแหน่งของป้าย, แสดงป้ายเฉพาะจุดที่เลือกหากจำเป็น, หรือเปลี่ยนรูปแบบเป็น "ค่า + คำอธิบาย"

**ฉันสามารถใช้การเติมแบบไล่สีหรือแพทเทิร์นกับซีรีส์ได้หรือไม่?**

ได้ ทั้งการเติมแบบสีทึบและแบบไล่สี/แพทเทิร์นมักจะมีให้ใช้ ในการปฏิบัติ ควรใช้ไล่สีอย่างระมัดระวังและหลีกเลี่ยงการผสมที่ลดความคอนทราสต์กับกริดและข้อความ
