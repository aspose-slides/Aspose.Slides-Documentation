## **ภาพรวม**

บทความนี้อธิบายวิธีการเพิ่มเส้นแนวโน้มไปยังแผนภูมิการนำเสนอโดยใช้ Aspose.Slides แสดงวิธีสร้างแผนภูมิ, เพิ่มเส้นแนวโน้มให้กับชุดข้อมูลในแผนภูมิ, และทำงานกับประเภทเส้นแนวโน้มหลายประเภท รวมถึงเส้นแนวโน้มเอ็กซ์โพเนนเชียล, เส้นแนวโน้มเชิงเส้น, เส้นแนวโน้มลอการิทึม, เส้นแนวโน้มค่าเฉลี่ยเคลื่อนที่, เส้นแนวโน้มพหุนาม, และเส้นแนวโน้มกำลัง

บทความนี้ยังอธิบายวิธีการเพิ่มเส้นกำหนดเองลงในแผนภูมิโดยการแทรกรูปทรงเส้น และรวมคำถามที่พบบ่อยสั้น ๆ เกี่ยวกับค่าการฉายเส้นแนวโน้มไปข้างหน้าและถอยหลัง รวมถึงว่าเส้นแนวโน้มจะยังคงอยู่ระหว่างการส่งออกเป็น PDF หรือ SVG หรือเมื่อเรนเดอร์แผนภูมิเป็นภาพหรือไม่

## **เพิ่มเส้นแนวโน้ม**
Aspose.Slides for C++ มี API ง่าย ๆ สำหรับจัดการเส้นแนวโน้มของแผนภูมิหลายประเภท:

1. สร้างอินสแตนซ์ของคลาส [Presentation](https://reference.aspose.com/slides/th/cpp/aspose.slides/presentation/)  
2. รับอ้างอิงของสไลด์โดยใช้ดัชนีของมัน  
3. เพิ่มแผนภูมิพร้อมข้อมูลเริ่มต้นและประเภทที่ต้องการ (ตัวอย่างนี้ใช้ ChartType.ClusteredColumn)  
4. เพิ่มเส้นแนวโน้มเอ็กซ์โพเนนเชียลให้กับชุดข้อมูลแผนภูมิที่ 1  
5. เพิ่มเส้นแนวโน้มเชิงเส้นให้กับชุดข้อมูลแผนภูมิที่ 1  
6. เพิ่มเส้นแนวโน้มลอการิทึมให้กับชุดข้อมูลแผนภูมิที่ 2  
7. เพิ่มเส้นแนวโน้มค่าเฉลี่ยเคลื่อนที่ให้กับชุดข้อมูลแผนภูมิที่ 2  
8. เพิ่มเส้นแนวโน้มพหุนามให้กับชุดข้อมูลแผนภูมิที่ 3  
9. เพิ่มเส้นแนวโน้มกำลังให้กับชุดข้อมูลแผนภูมิที่ 3  
10. เขียนการนำเสนอที่ปรับแต่งแล้วลงไฟล์ PPTX

## **เพิ่มเส้นกำหนดเอง**
Aspose.Slides for C++ มี API ง่าย ๆ สำหรับเพิ่มเส้นกำหนดเองในแผนภูมิ เพื่อเพิ่มเส้นธรรมดา ๆ ลงในสไลด์ที่เลือกของการนำเสนอ โปรดทำตามขั้นตอนด้านล่าง:

- สร้างอินสแตนซ์ของคลาส Presentation  
- รับอ้างอิงของสไลด์โดยใช้ Index ของมัน  
- สร้างแผนภูมิใหม่โดยใช้เมธอด AddChart ของอ็อบเจกต์ Shapes  
- เพิ่ม AutoShape ประเภท Line โดยใช้เมธอด AddAutoShape ของอ็อบเจกต์ Shapes  
- ตั้งค่า Color ของเส้นรูปทรง  
- เขียนการนำเสนอที่ปรับแต่งแล้วเป็นไฟล์ PPTX

## **คำถามที่พบบ่อย**

**'forward' และ 'backward' มีความหมายอย่างไรสำหรับเส้นแนวโน้ม?**

เป็นความยาวของเส้นแนวโน้มที่ถูกฉายไปข้างหน้า/ถอยหลัง: สำหรับแผนภูมิกระจาย (XY) — หน่วยแกน; สำหรับแผนภูมิที่ไม่ใช่กระจาย — จำนวนหมวดหมู่. รับค่าได้เฉพาะค่าที่ไม่เป็นลบเท่านั้น

**เส้นแนวโน้มจะยังคงอยู่เมื่อส่งออกการนำเสนอเป็น PDF หรือ SVG หรือเมื่อเรนเดอร์สไลด์เป็นภาพหรือไม่?**

ใช่. Aspose.Slides แปลงการนำเสนอเป็น [PDF](/slides/th/cpp/convert-powerpoint-to-pdf/)/[SVG](/slides/th/cpp/render-a-slide-as-an-svg-image/) และเรนเดอร์แผนภูมิเป็นภาพ; เส้นแนวโน้มซึ่งเป็นส่วนหนึ่งของแผนภูมิจะถูกเก็บไว้ในขั้นตอนเหล่านี้. มีเมธอดให้ [export an image of the chart](/slides/th/cpp/create-shape-thumbnails/) อีกด้วย
