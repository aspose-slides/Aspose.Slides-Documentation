## **ภาพรวม**

บทความนี้อธิบายวิธีทำงานกับแถบข้อผิดพลาดในแผนภูมิการนำเสนอโดยใช้ Aspose.Slides มันแสดงวิธีเพิ่มแถบข้อผิดพลาดให้กับชุดข้อมูลของแผนภูมิ กำหนดการตั้งค่าแถบข้อผิดพลาดในแนว X และ Y และใช้ประเภทค่าที่แตกต่างกัน เช่น ค่าคงที่ เปอร์เซ็นต์ และค่าที่กำหนดเอง

นอกจากนี้ยังแสดงวิธีกำหนดค่าด้วยแถบข้อผิดพลาดแบบกำหนดเองสำหรับจุดข้อมูลแต่ละจุดในชุดข้อมูลโดยใช้คอลเลกชันจุดข้อมูลที่สอดคล้องกัน อีกทั้งบทความยังมีบันทึกสั้น ๆ เกี่ยวกับพฤติกรรมของแถบข้อผิดพลาดระหว่างการส่งออก ความเข้ากันได้กับมาร์คเกอร์และป้ายกำกับข้อมูล และตำแหน่งที่สามารถค้นหาคลาสอ้างอิง API และ enum ที่เกี่ยวข้อง

## **เพิ่มแถบข้อผิดพลาด**
Aspose.Slides for C++ มี API ที่เรียบง่ายสำหรับจัดการค่าของแถบข้อผิดพลาด ตัวอย่างโค้ดนี้ใช้เมื่อใช้ประเภทค่าที่กำหนดเอง เพื่อระบุค่าให้ใช้คุณสมบัติ **ErrorBarCustomValues** ของจุดข้อมูลเฉพาะในคอลเลกชัน **DataPoints** ของชุดข้อมูล:

1. สร้างอินสแตนซ์ของคลาส [Presentation](https://reference.aspose.com/slides/th/cpp/aspose.slides/presentation/)
1. เพิ่มแผนภูมิบับเบิลบนสไลด์ที่ต้องการ
1. เข้าถึงชุดข้อมูลแรกของแผนภูมิและตั้งค่ารูปแบบแถบข้อผิดพลาด X
1. เข้าถึงชุดข้อมูลแรกของแผนภูมิและตั้งค่ารูปแบบแถบข้อผิดพลาด Y
1. ตั้งค่าค่าและรูปแบบของแถบ
1. เขียนการนำเสนอที่แก้ไขแล้วไปยังไฟล์ PPTX

## **เพิ่มแถบข้อผิดพลาดแบบกำหนดเอง**
Aspose.Slides for C++ มี API ที่เรียบง่ายสำหรับจัดการค่าของแถบข้อผิดพลาดแบบกำหนดเอง ตัวอย่างโค้ดนี้ใช้เมื่อคุณสมบัติ **IErrorBarsFormat.ValueType** มีค่าเท่ากับ **Custom** เพื่อระบุค่าให้ใช้คุณสมบัติ **ErrorBarCustomValues** ของจุดข้อมูลเฉพาะในคอลเลกชัน **DataPoints** ของชุดข้อมูล:

1. สร้างอินสแตนซ์ของคลาส [Presentation](https://reference.aspose.com/slides/th/cpp/aspose.slides/presentation/)
1. เพิ่มแผนภูมิบับเบิลบนสไลด์ที่ต้องการ
1. เข้าถึงชุดข้อมูลแรกของแผนภูมิและตั้งค่ารูปแบบแถบข้อผิดพลาด X
1. เข้าถึงชุดข้อมูลแรกของแผนภูมิและตั้งค่ารูปแบบแถบข้อผิดพลาด Y
1. เข้าถึงจุดข้อมูลแต่ละจุดของชุดข้อมูลแผนภูมิและตั้งค่า Error Bar สำหรับจุดข้อมูลนั้น
1. ตั้งค่าค่าและรูปแบบของแถบ
1. เขียนการนำเสนอที่แก้ไขแล้วไปยังไฟล์ PPTX

## **คำถามที่พบบ่อย**

**แถบข้อผิดพลาดจะเกิดอะไรขึ้นเมื่อส่งออกการนำเสนอเป็น PDF หรือรูปภาพ?**

แถบข้อผิดพลาดจะถูกเรนเดอร์เป็นส่วนหนึ่งของแผนภูมิและจะถูกเก็บไว้ระหว่างการแปลงพร้อมกับการจัดรูปแบบแผนภูมิทั้งหมด โดยสมมติว่าใช้เวอร์ชันหรือเรนเดอร์ที่เข้ากันได้

**สามารถรวมแถบข้อผิดพลาดกับมาร์คเกอร์และป้ายกำกับข้อมูลได้หรือไม่?**

ทำได้ แถบข้อผิดพลาดเป็นองค์ประกอบแยกจากกันและเข้ากันได้กับมาร์คเกอร์และป้ายกำกับข้อมูล; หากองค์ประกอบทับกันอาจต้องปรับการจัดรูปแบบ

**ฉันจะหารายการคุณสมบัติและ enum สำหรับการทำงานกับแถบข้อผิดพลาดใน API ได้ที่ไหน?**

ในเอกสารอ้างอิง API: คลาส [ErrorBarsFormat](https://reference.aspose.com/slides/th/cpp/aspose.slides.charts/errorbarsformat/) และ enum ที่เกี่ยวข้องคือ [ErrorBarType](https://reference.aspose.com/slides/th/cpp/aspose.slides.charts/errorbartype/) และ [ErrorBarValueType](https://reference.aspose.com/slides/th/cpp/aspose.slides.charts/errorbarvaluetype/)
