## **ภาพรวม**

Aspose.Slides ช่วยให้คุณเปรียบเทียบสไลด์, สไลด์เค้าโครง, และสไลด์แม่โดยใช้เมธอด `Equals` ที่มาจากอินเทอร์เฟซ `IBaseSlide` และคลาส `BaseSlide`. เมธอดนี้จะคืนค่า `true` เมื่อสไลด์ที่เปรียบเทียบมีโครงสร้างและเนื้อหาคงที่เหมือนกัน.

## **เปรียบเทียบสองสไลด์**
เมธอด Equals ได้ถูกเพิ่มเข้าไปในอินเทอร์เฟซ IBaseSlide และคลาส BaseSlide. มันคืนค่า true สำหรับสไลด์ / สไลด์เค้าโครง / สไลด์แม่ที่มีโครงสร้างและเนื้อหาคงที่เหมือนกัน.

สองสไลด์จะเท่ากันหากรูปทรง, สไตล์, ข้อความ, การเคลื่อนไหวและการตั้งค่าอื่นๆ ทั้งหมดตรงกัน เป็นต้น การเปรียบเทียบจะไม่คำนึงถึงค่าตัวระบุที่เป็นเอกลักษณ์ เช่น SlideId และเนื้อหาแบบไดนามิก เช่น ค่าการแสดงวันที่ปัจจุบันใน Date Placeholder.

## **คำถามที่พบบ่อย**

**การที่สไลด์ถูกซ่อนมีผลต่อการเปรียบเทียบสไลด์เองหรือไม่?**

[Hidden status](https://reference.aspose.com/slides/th/cpp/aspose.slides/slide/get_hidden/) เป็นคุณสมบัติระดับการนำเสนอ/การเล่น ไม่ใช่เนื้อหาภาพ. ความเท่าเทียมของสไลด์สองสไลด์ถูกกำหนดโดยโครงสร้างและเนื้อหาคงที่; การที่สไลด์ถูกซ่อนเพียงอย่างเดียวไม่ได้ทำให้สไลด์แตกต่างกัน.

**ลิงก์ไฮเปอร์และพารามิเตอร์ของมันจะถูกพิจารณาหรือไม่?**

ใช่. ลิงก์เป็นส่วนหนึ่งของเนื้อหาคงของสไลด์. หาก URL หรือการกระทำของลิงก์แตกต่างกัน จะถือว่าเป็นความแตกต่างในเนื้อหาคง.

**หากแผนภูมิอ้างอิงไฟล์ Excel ภายนอก เนื้อหาของไฟล์นั้นจะถูกพิจารณาหรือไม่?**

ไม่. การเปรียบเทียบทำโดยอิงจากสไลด์เอง. แหล่งข้อมูลภายนอกโดยทั่วไปจะไม่ถูกอ่านในขณะเปรียบเทียบ; มีเพียงสิ่งที่อยู่ในโครงสร้างและสถานะคงของสไลด์เท่านั้นที่พิจารณา.
