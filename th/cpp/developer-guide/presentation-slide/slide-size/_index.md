## **บทนำ**

Aspose.Slides มีเครื่องมือครบถ้วนเพื่อปรับขนาดสไลด์และอัตราส่วนภาพในงานนำเสนอ PowerPoint ซึ่งสำคัญทั้งสำหรับการพิมพ์และการแสดงบนหน้าจอ  

ขนาดสไลด์และอัตราส่วนที่นิยม:

- **Standard (อัตราส่วน 4:3)**: เหมาะสำหรับหน้าจอและอุปกรณ์รุ่นเก่า
- **Widescreen (อัตราส่วน 16:9)**: แนะนำสำหรับโปรเจคเตอร์และจอแสดงผลสมัยใหม่  

ตรวจสอบให้แน่ใจว่ามีความสม่ำเสมอทั่วทั้งงานนำเสนอ เพราะขนาดสไลด์และอัตราส่วนเดียวกันจะใช้กับสไลด์ทั้งหมด เพื่อผลลัพธ์ที่ดีที่สุด ให้ตั้งค่าขนาดสไลด์ตั้งแต่ต้นกระบวนการสร้างงานนำเสนอเพื่อหลีกเลี่ยงปัญหา  

{{% alert color="primary" %}} 
โดยค่าเริ่มต้น งานนำเสนอที่สร้างด้วย Aspose.Slides จะใช้เอสเปกตรา 4:3 มาตรฐาน
{{% /alert %}}

## **เปลี่ยนขนาดสไลด์ในงานนำเสนอ**

ตัวอย่างโค้ดนี้แสดงวิธีการเปลี่ยนขนาดสไลด์ในงานนำเสนอด้วย C++ โดยใช้ Aspose.Slides:

``` cpp
auto pres = System::MakeObject<Presentation>(u"pres-4x3-aspect-ratio.pptx");
pres->get_SlideSize()->SetSize(SlideSizeType::OnScreen16x9, SlideSizeScaleType::DoNotScale);
pres->Save(u"pres-4x3-aspect-ratio.pptx", SaveFormat::Pptx);
```

## **กำหนดขนาดสไลด์ที่กำหนดเองในงานนำเสนอ**

หากคุณพบว่าขนาดสไลด์ทั่วไป (4:3 และ 16:9) ไม่เหมาะกับงานของคุณ คุณอาจต้องการใช้ขนาดสไลด์ที่ระบุหรือไม่ซ้ำใคร ตัวอย่างเช่น หากคุณต้องการพิมพ์สไลด์ขนาดเต็มจากงานนำเสนอบนรูปแบบหน้ากระดาษที่กำหนดเอง หรือหากคุณตั้งใจจะแสดงงานนำเสนอบนหน้าจอประเภทบางชนิด คุณจะได้รับประโยชน์จากการตั้งค่าขนาดที่กำหนดเองสำหรับงานนำเสนอของคุณ  

ตัวอย่างโค้ดนี้แสดงวิธีใช้ Aspose.Slides สำหรับ C++ เพื่อกำหนดขนาดสไลด์ที่กำหนดเองสำหรับงานนำเสนอใน C++:

``` cpp
auto pres = System::MakeObject<Presentation>(u"pres.pptx");
// ขนาดกระดาษ A4
pres->get_SlideSize()->SetSize(780.0f, 540.0f, SlideSizeScaleType::DoNotScale);
pres->Save(u"pres-a4-slide-size.pptx", SaveFormat::Pptx);
```

## **จัดการเนื้อหาสไลด์หลังการปรับขนาด**

หลังจากที่คุณเปลี่ยนขนาดสไลด์สำหรับงานนำเสนอ เนื้อหาของสไลด์ (เช่น รูปภาพหรือวัตถุ) อาจเสียรูปแบบโดยอัตโนมัติ ตามค่าเริ่มต้น วัตถุจะถูกปรับขนาดอัตโนมัติเพื่อให้พอดีกับขนาดสไลด์ใหม่ อย่างไรก็ตาม เมื่อเปลี่ยนขนาดสไลด์ของงานนำเสนอ คุณสามารถกำหนดการตั้งค่าที่กำหนดวิธีที่ Aspose.Slides จัดการกับเนื้อหาบนสไลด์ได้  

ขึ้นอยู่กับสิ่งที่คุณต้องการทำหรือบรรลุ คุณสามารถใช้การตั้งค่าใดก็ได้ต่อไปนี้:

- `DoNotScale`

  หากคุณ **ไม่** ต้องการให้วัตถุบนสไลด์ถูกปรับขนาด ให้ใช้การตั้งค่านี้

- `EnsureFit`

  หากคุณต้องการปรับขนาดให้สไลด์เล็กลงและต้องการให้ Aspose.Slides ลดขนาดวัตถุบนสไลด์เพื่อให้ทั้งหมดพอดีกับสไลด์ (เพื่อหลีกเลี่ยงการสูญเสียเนื้อหา) ให้ใช้การตั้งค่านี้

- `Maximize`

  หากคุณต้องการปรับขนาดให้สไลด์ใหญ่ขึ้นและต้องการให้ Aspose.Slides ขยายวัตถุบนสไลด์ให้สัดส่วนกับขนาดสไลด์ใหม่ ให้ใช้การตั้งค่านี้

ตัวอย่างโค้ดนี้แสดงวิธีใช้การตั้งค่า `Maximize` เมื่อต้องการเปลี่ยนขนาดสไลด์ของงานนำเสนอ:

``` cpp
auto pres = System::MakeObject<Presentation>(u"pres.pptx");
pres->get_SlideSize()->SetSize(SlideSizeType::Ledger, SlideSizeScaleType::Maximize);
```

## **คำถามที่พบบ่อย**

**ฉันสามารถตั้งค่าขนาดสไลด์ที่กำหนดเองโดยใช้หน่วยอื่นนอกจากนิ้ว (เช่น จุดหรือมม.) ได้หรือไม่?**

ใช่ Aspose.Slides ใช้หน่วยจุดภายใน โดย 1 จุดเท่ากับ 1/72 นิ้ว คุณสามารถแปลงหน่วยใด ๆ (เช่น มิลลิเมตรหรือเซนติเมตร) เป็นจุดและใช้ค่าที่แปลงแล้วเพื่อกำหนดความกว้างและความสูงของสไลด์  

**ขนาดสไลด์ที่กำหนดเองขนาดใหญ่มากจะส่งผลต่อประสิทธิภาพและการใช้หน่วยความจำระหว่างการเรนเดอร์หรือไม่?**

ใช่ ขนาดสไลด์ที่ใหญ่ขึ้น (เป็นจุด) ร่วมกับสเกลการเรนเดอร์ที่สูงจะทำให้ใช้หน่วยความจำเพิ่มขึ้นและเวลาประมวลผลยาวนานขึ้น ควรกำหนดขนาดสไลด์ให้เหมาะสมและปรับสเกลการเรนเดอร์ตามความต้องการเพื่อให้ได้คุณภาพผลลัพธ์ที่ต้องการ  

**ฉันสามารถกำหนดขนาดสไลด์ที่ไม่เป็นมาตรฐานหนึ่งขนาดและจากนั้นรวมสไลด์จากงานนำเสนอที่มีขนาดต่างกันได้หรือไม่?**

คุณไม่สามารถ [merge presentations](/slides/th/cpp/merge-presentation/) ขณะมีขนาดสไลด์ต่างกัน — ก่อนอื่นให้ปรับขนาดงานนำเสนอหนึ่งให้ตรงกับอีกงานหนึ่ง เมื่อเปลี่ยนขนาดสไลด์ คุณสามารถเลือกวิธีการจัดการกับเนื้อหาที่มีอยู่ผ่านตัวเลือก [SlideSizeScaleType](https://reference.aspose.com/slides/th/cpp/aspose.slides/slidesizescaletype/) หลังจากปรับขนาดให้ตรงกันแล้ว คุณสามารถรวมสไลด์พร้อมคงรูปแบบได้  

**ฉันสามารถสร้างภาพย่อสำหรับรูปทรงเดี่ยวหรือบริเวณเฉพาะของสไลด์ได้หรือไม่ และภาพย่อเหล่านั้นจะเคารพขนาดสไลด์ใหม่หรือไม่?**

ใช่ Aspose.Slides สามารถเรนเดอร์ภาพย่อสำหรับ [entire slides](https://reference.aspose.com/slides/th/cpp/aspose.slides/slide/getimage/) รวมถึง [selected shapes](https://reference.aspose.com/slides/th/cpp/aspose.slides/shape/getimage/) ภาพที่ได้จะแสดงขนาดสไลด์และอัตราส่วนปัจจุบัน ทำให้กรอบและรูปทรงคงที่
