## **ภาพรวม**

Aspose.Slides ช่วยให้คุณเพิ่มสไลด์ลงในงานนำเสนอ PowerPoint โดยโปรแกรม งานนำเสนอประกอบด้วยสไลด์ Master/Layout และสไลด์ปกติ โดยสไลด์ปกติจะจัดลำดับตามดัชนีเริ่มจากศูนย์ แต่ละสไลด์มี ID ที่ไม่ซ้ำกัน และไฟล์งานนำเสนอที่ไม่มีสไลด์ไม่รองรับ

บทความนี้อธิบายวิธีสร้างวัตถุ `Presentation` การเข้าถึงคอลเลกชันสไลด์ การเพิ่มสไลด์เปล่า การทำงานกับสไลด์ที่เพิ่มใหม่ และการบันทึกงานนำเสนอที่อัปเดต นอกจากนี้ยังครอบคลุมประเด็นที่เกี่ยวข้อง เช่น การแทรกสไลด์ในตำแหน่งเฉพาะ การใช้เลเอาต์ และการทำความเข้าใจสไลด์เปล่าที่มีอยู่ในงานนำเสนอที่สร้างใหม่

## **เพิ่มสไลด์ลงในงานนำเสนอ**
ก่อนจะพูดถึงการเพิ่มสไลด์ลงในไฟล์งานนำเสนอ เรามาพูดคุยเกี่ยวกับข้อเท็จจริงของสไลด์กันก่อน แต่ละไฟล์งานนำเสนอ PowerPoint จะมีสไลด์ Master / Layout และสไลด์ Normal อื่น ๆ หมายความว่าไฟล์งานนำเสนอจะต้องมีสไลด์อย่างน้อยหนึ่งสไลด์หรือมากกว่า สิ่งสำคัญที่ต้องทราบคือไฟล์งานนำเสนอที่ไม่มีสไลด์ไม่รองรับโดย Aspose.Slides for C++ แต่ละสไลด์มี Id ที่ไม่ซ้ำกันและสไลด์ Normal ทั้งหมดจะถูกจัดเรียงตามดัชนีเริ่มจากศูนย์ Aspose.Slides for C++ อนุญาตให้ผู้พัฒนาสามารถเพิ่มสไลด์เปล่าเข้าสู่การนำเสนอได้ เพื่อเพิ่มสไลด์เปล่าในงานนำเสนอ โปรดทำตามขั้นตอนต่อไปนี้：

- สร้างอินสแตนซ์ของ [Presentation](https://reference.aspose.com/slides/th/cpp/aspose.slides/presentation/) คลาส
- สร้างอินสแตนซ์ของ [ISlideCollection](https://reference.aspose.com/slides/th/cpp/aspose.slides/islidecollection/) คลาสโดยตั้งค่าอ้างอิงไปยังคุณสมบัติ Slides (คอลเลกชันของอ็อบเจ็กต์ Slide ที่เป็นเนื้อหา) ที่เปิดเผยโดยวัตถุ Presentation
- เพิ่มสไลด์เปล่าไปยังการนำเสนอที่ตำแหน่งท้ายของคอลเลกชันสไลด์เนื้อหาโดยเรียกใช้เมธอด AddEmptySlide ที่เปิดเผยโดยอ็อบเจ็กต์ ISlideCollection
- ทำบางอย่างกับสไลด์เปล่าที่เพิ่มใหม่
- สุดท้าย เขียนไฟล์การนำเสนอโดยใช้วัตถุ [Presentation](https://reference.aspose.com/slides/th/cpp/aspose.slides/presentation/)

## **FAQ**

**ฉันสามารถแทรกสไลด์ใหม่ในตำแหน่งเฉพาะได้หรือไม่ ไม่ใช่เพียงที่ท้ายรายการ?**

ได้。ไลบรารีสนับสนุนคอลเลกชันสไลด์และการดำเนินการ [insert](https://reference.aspose.com/slides/th/cpp/aspose.slides/slidecollection/insertemptyslide/)/[clone](https://reference.aspose.com/slides/th/cpp/aspose.slides/slidecollection/insertclone/) ซึ่งคุณสามารถเพิ่มสไลด์ที่ตำแหน่งดัชนีที่ต้องการได้ ไม่จำเป็นต้องเป็นเพียงที่ท้ายรายการ

**ธีม/สไตล์จะถูกเก็บรักษาเมื่อเพิ่มสไลด์โดยอิงจากเลเอาต์หรือไม่?**

ได้。เลเอาต์สืบทอดการจัดรูปแบบจากมาสเตอร์ของมัน และสไลด์ใหม่สืบทอดจากเลเอาต์ที่เลือกและมาสเตอร์ที่เชื่อมโยงกับมัน

**สไลด์ใดปรากฏอยู่ในงานนำเสนอใหม่ที่ "ว่างเปล่า" ก่อนที่จะเพิ่มสไลด์?**

งานนำเสนอที่สร้างใหม่จะมีสไลด์เปล่าหนึ่งสไลด์ที่มีดัชนีศูนย์อยู่แล้ว สิ่งนี้สำคัญต้องคำนึงเมื่อคำนวณดัชนีการแทรก

**ฉันจะเลือกเลเอาต์ที่ "เหมาะสม" สำหรับสไลด์ใหม่อย่างไร หากมาสเตอร์มีตัวเลือกหลายแบบ?**

โดยทั่วไปให้เลือก [LayoutSlide](https://reference.aspose.com/slides/th/cpp/aspose.slides/layoutslide/) ที่ตรงกับโครงสร้างที่ต้องการ ([Title and Content, Two Content, etc.](https://reference.aspose.com/slides/th/cpp/aspose.slides/slidelayouttype/)) หากไม่มีเลเอาต์ดังกล่าว คุณสามารถ [เพิ่มลงในมาสเตอร์](/slides/th/cpp/slide-layout/) แล้วใช้ได้
