## **ภาพรวม**

FAQ นี้ให้คำตอบสำหรับคำถามทั่วไปเกี่ยวกับ Aspose.Slides ครอบคลุมรูปแบบไฟล์ที่สนับสนุน, การจัดการข้อยกเว้นเมื่อทำงานกับงานนำเสนอขนาดใหญ่, การเปลี่ยนขนาดสไลด์, การแสดงตัวอย่างสไลด์, การดึงข้อความจากงานนำเสนอ, การจัดรูปแบบเส้นขอบของตาราง, การวางรูปภาพ, และการแก้ไขปัญหาเกี่ยวกับฟอนต์เมื่อต้องแปลงงานนำเสนอเป็น PDF หรือรูปภาพ

## **รูปแบบไฟล์ที่สนับสนุน**

**Q: Aspose.Slides for C++ รองรับรูปแบบไฟล์ใดบ้าง?**

**A**: Aspose.Slides for C++ รองรับรูปแบบไฟล์ที่อธิบายไว้ใน[Supported File Formats](/slides/th/cpp/supported-file-formats/)

## **ข้อยกเว้น**

**Q: ฉันได้รับข้อยกเว้น out of memory ขณะโหลดไฟล์ PPT ขนาดใหญ่ที่มีรูปภาพ มีข้อจำกัดเรื่องขนาดไฟล์ใน Aspose.Slides หรือไม่?**

**A**: ไม่มีสูตรที่เฉพาะเจาะจงสำหรับคำนวณขนาดงานนำเสนอที่ Aspose.Slides รองรับ ควรมีพื้นที่เพียงพอเพื่อเก็บโครงสร้างงานนำเสนอทั้งหมดและรูปภาพในหน่วยความจำ โดยทั่วไปรูปภาพในหน่วยความจำจะใช้พื้นที่มากกว่าในฮาร์ดดิสก์ โดยเฉพาะเมื่อรูปภาพมีเอฟเฟ็กต์เพิ่มเติม

โดยปกติ Aspose.Slides for C++ สามารถจัดการไฟล์งานนำเสนอขนาดประมาณ 300 MB ได้อย่างง่ายดายบนเซิร์ฟเวอร์ที่มี RAM 4 GB

## **การทำงานกับสไลด์**

**Q: ฉันสามารถเปลี่ยนขนาดของสไลด์ในงานนำเสนอได้หรือไม่?**

**A**: คุณสามารถใช้เมธอด `get_SlideSize` ที่เปิดให้ใช้งานโดยคลาส[Presentation](https://reference.aspose.com/slides/th/cpp/aspose.slides/presentation/) เพื่อกำหนดขนาดของสไลด์ในงานนำเสนอ

**Q: มีวิธีกำหนดสไลด์ขนาดต่างกันในงานนำเสนอเดียวกันหรือไม่?**

**A**: เนื่องจากขนาดของสไลด์ถูกกำหนดระดับงานนำเสนอในเอกสาร Microsoft PowerPoint จึงไม่มีวิธีทำเช่นนั้น

**Q: Aspose.Slides for C++ รองรับการแสดงตัวอย่างสไลด์ก่อนบันทึกหรือไม่?**

**A**: คุณสามารถเรนเดอร์สไลด์ของงานนำเสนอเป็นรูปภาพและใช้รูปภาพเหล่านี้เพื่อแสดงตัวอย่างสไลด์ได้

## **การทำงานกับข้อความ**

**Q: สามารถดึงข้อความทั้งหมดจากงานนำเสนอได้หรือไม่?**

**A**: Aspose.Slides for C++ มีคลาส[SlideUtil](https://reference.aspose.com/slides/th/cpp/aspose.slides.util/slideutil/) ภายใต้เนมสเปซ`Aspose::Slides::Util` ซึ่งให้เมธอดต่าง ๆ สำหรับดึงข้อความทั้งหมดจากงานนำเสนอ

**Q: ทำไมขนาดย่อหน้าถึงต่างกันบนระบบปฏิบัติการ Windows และ Linux?**

**A**: การคำนวณขนาดย่อหน้าขึ้นอยู่กับการคำนวณขนาดข้อความที่แทนย่อหน้าที่กำหนด ขนาดข้อความคำนวณจากเมตริกของฟอนต์ที่ระบุในงานนำเสนอ PowerPoint หากฟอนต์ที่ระบุไม่มีอยู่ จะถูกแทนที่ด้วยฟอนต์ที่คล้ายที่สุด แต่ฟอนต์นั้นมีเมตริกที่แตกต่างจากฟอนต์เดิม ดังนั้นการคำนวณขนาดย่อหน้าในระบบต่าง ๆ จะให้ผลลัพธ์ต่างกันขึ้นอยู่กับชุดฟอนต์ที่ติดตั้ง เพื่อให้ได้ผลลัพธ์เดียวกันบนระบบปฏิบัติการต่าง ๆ คุณต้องติดตั้งฟอนต์เดียวกันบนทุกระบบหรือโหลดฟอนต์เหล่านั้นในเวลารันไทม์เป็น[external fonts](/slides/th/cpp/custom-font/)

## **การจัดรูปแบบและรูปภาพ**

**Q: จะตั้งค่าสีของเส้นขอบตารางอย่างไร?**

**A**: คุณสามารถเปลี่ยนสีของเส้นขอบตารางทั้งหมดหรือเฉพาะเส้นขอบรอบตารางทั้งหมดได้ สำหรับการเปลี่ยนสีเส้นขอบทั้งหมดให้ใช้เมธอด`get_CellFormat` จากอินเทอร์เฟซ[ICell](https://reference.aspose.com/slides/th/cpp/aspose.slides/icell/) สำหรับเส้นขอบรอบตารางทั้งหมดให้ทำการวนลูปผ่านเซลล์และเปลี่ยนสีของเส้นขอบภายนอก

**Q: Aspose.Slides for C++ ใช้หน่วยใดในการวางรูปภาพ?**

**A**: พิกัดและขนาดของรูปทรงทั้งหมดบนสไลด์วัดเป็นจุด (72 dpi)

## **การทำงานกับฟอนต์**

**Q: เมื่อแปลง PPT เป็น PDF หรือรูปภาพ ทำไมฟอนต์ถึงต่างกันในเอกสารผลลัพธ์?**

**A**: ปัญหานี้อาจบ่งบอกว่าฟอนต์ที่ใช้ในงานนำเสนอไม่มีในระบบปฏิบัติการที่รันโค้ด คุณควรติดตั้งฟอนต์บนระบบปฏิบัติการหรือโหลดฟอนต์เป็นฟอนต์ภายนอกโดยใช้คลาส[FontsLoader](https://reference.aspose.com/slides/th/cpp/aspose.slides/fontsloader/) ตามตัวอย่างด้านล่าง:
```cpp
auto folders = MakeObject<Array<String>>(1, "path_to_a_folder_with_fonts");
FontsLoader::LoadExternalFonts(folders);
```
