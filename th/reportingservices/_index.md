**ยินดีต้อนรับสู่ Aspose.Slides for Reporting Services**

Aspose.Slides for Reporting Services เป็นโซลูชันเดียวในตลาดที่ทำให้สามารถสร้างรายงาน PPT และ PPS จริงใน Microsoft SQL Server 2005, 2008, 2012, 2016 และ 2017 Reporting Services (32‑บิตและ 64‑บิต) ได้ ทุกคุณสมบัติของรายงาน RDL รวมถึงตาราง, เมทริกซ์, แผนภูมิ และรูปภาพ จะถูกแปลงด้วยความแม่นยำสูงสุดเป็นงานนำเสนอ Microsoft PowerPoint
