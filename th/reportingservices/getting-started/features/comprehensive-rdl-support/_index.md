{{% alert color="primary" %}} 

Aspose.Slides for Reporting Services รองรับข้อกำหนดของ RDL ทั้งหมดอย่างเต็มที่ หมายความว่ามีสองประเด็นที่ยอดเยี่ยม:

- ไม่ต้องออกแบบใหม่ทั้งหมดของรายงานที่มีอยู่ คุณสามารถส่งออกรายงาน RDL ใดๆ เป็นงานนำเสนอ Microsoft PowerPoint และจะปรากฏตรงตามการออกแบบของ RDL อย่างแม่นยำ
- ไม่จำเป็นต้องใช้ตัวออกแบบรายงานเฉพาะ คุณสามารถใช้ตัวออกแบบรายงาน RDL ใดก็ได้และรายงานจะถูกส่งออกตามที่คุณออกแบบไว้

{{% /alert %}} 

Aspose.Slides for Reporting Services รองรับองค์ประกอบ RDL ดังต่อไปนี้:

- หน้า, ส่วนหัว, ส่วนท้าย
- กล่องข้อความ
- รูปภาพ
- รายงานย่อย
- แผนภูมิ
- รายการ
- ตาราง
- เมทริกซ์
- สไตล์
- สี่เหลี่ยม

**ตัวอย่างรายงานที่มีส่วนหัว ส่วนท้าย รูปภาพ รายงานย่อย ตาราง กล่องข้อความ และสี่เหลี่ยมที่ส่งออกเป็นงานนำเสนอ Microsoft PowerPoint (PPT).**
