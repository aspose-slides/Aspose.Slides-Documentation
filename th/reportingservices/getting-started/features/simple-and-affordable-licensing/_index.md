## **Server Subscription**
Server subscription provides following features:

- ต้องมีใบอนุญาตหนึ่งใบสำหรับแต่ละเซิร์ฟเวอร์ทางกายภาพหรือเสมือน
- จำนวนโปรเซสเซอร์หรือคอร์ไม่จำกัด
- จำนวนอินสแตนซ์ของ Microsoft SQL Server ไม่จำกัด
- จำนวนผู้พัฒนาไม่จำกัด
- การออกรุ่นใหม่และการแก้ไขปัญหา (hot fixes) ฟรีเป็นเวลา 1 ปี
- การสนับสนุนทางเทคนิคฟรีเป็นเวลา 1 ปี
- ส่วนลดสำหรับการสั่งซื้อหลายครั้ง
## **Site Subscription**

- ต้องมีใบอนุญาตหนึ่งใบต่อไซต์ทางกายภาพหนึ่งแห่ง (campus)
- จำนวนเซิร์ฟเวอร์ทางกายภาพหรือเสมือนไม่จำกัด

- จำนวนอินสแตนซ์ของ Microsoft SQL Server ไม่จำกัด
