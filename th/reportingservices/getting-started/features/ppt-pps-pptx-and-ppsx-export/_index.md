{{% alert color="primary" %}}

Microsoft SQL Server Reporting Services ไม่มีความสามารถในตัวสำหรับการส่งออกรายงานเป็นงานนำเสนอ Microsoft PowerPoint แต่หลังจากติดตั้ง Aspose.Slides for Reporting Services บนเซิร์ฟเวอร์ของคุณ คุณจะสามารถเข้าถึงรูปแบบการส่งออกเพิ่มเติมสองรูปแบบได้:

- Microsoft PowerPoint Presentation (PPT)
- Microsoft PowerPoint SlideShow (PPS)
- Microsoft PowerPoint 2007 Presentation (PPTX)
- Microsoft PowerPoint 2007 SlideShow (PPSX)

{{% /alert %}}

เพื่อสร้างเอกสารในรูปแบบเหล่านี้ Aspose.Slides for Reporting Services พึ่งพาเวอร์ชันในตัวของ [Aspose.Slides for .NET](http://www.aspose.com/Products/Aspose.Slides/) ซึ่งเป็นส่วนประกอบการประมวลผลงานนำเสนอที่เป็นผู้นำตลาดจาก Aspose. Microsoft PowerPoint ไม่ได้ถูกใช้ในการสร้างเอกสารเหล่านี้.

สำเนา PPT และ PPS ของรายงานเดียวกันจะปรากฏเหมือนกันอย่างสมบูรณ์ เนื่องจากการสนับสนุนระดับสูงสำหรับรูปแบบไฟล์พื้นฐานที่ให้โดย Aspose.Slides.
