{{% alert color="primary" %}} 
Aspose.Slides for Reporting Services ได้เพิ่มการสนับสนุนสำหรับ Power BI แล้ว. 
{{% /alert %}} 

Paginated reports คือรายงานที่จัดรูปแบบให้พอดีกับหน้าอย่างดี รายงานประเภทนี้เหมาะอย่างยิ่งสำหรับการส่งออกและการพิมพ์ Paginated reports ถูกออกแบบให้สร้างจำนวนหน้าตามที่จำเป็นเพื่อพิมพ์หรือส่งออกทุกแถวของข้อมูลในตารางของคุณ. 

ตัวอย่างเช่น หากคุณต้องการสร้างรายงานเพื่อให้เจ้าของรายงานสามารถดูจำนวนสินค้า ราคาขาย และกำไรของแต่ละรายการที่ขายได้ในช่วง 12 เดือนที่ผ่านมา รายงานแบบ paginated คือสิ่งที่คุณต้องการ. 

เพื่อให้คุณสามารถสร้าง paginated reports ได้ Microsoft มี Power BI Report Builder ให้ใช้ใน Power BI หลังจากคุณสร้าง paginated report และบันทึกลงบนเครื่องของคุณ ไฟล์ผลลัพธ์จะมีส่วนขยาย .rdl.
