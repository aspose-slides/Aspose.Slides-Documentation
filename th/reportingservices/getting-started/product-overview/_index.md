## **ยินดีต้อนรับสู่ Aspose.Slides for Reporting Services!**

Aspose.Slides for Reporting Services เป็นโซลูชันเดียวในตลาดที่ทำให้สามารถสร้างรายงาน PPT และ PPS จริงใน Microsoft SQL Server 2005, 2008, 2012, 2016 และ 2017 Reporting Services (แบบ 32-bit และ 64-bit) ได้ ทุกคุณลักษณะของรายงาน RDL รวมถึงตาราง, เมทริกซ์, แผนภูมิและรูปภาพ จะถูกแปลงด้วยความแม่นยำสูงสุดเป็นงานนำเสนอ Microsoft PowerPoint

## **ภาพรวมของผลิตภัณฑ์**

Microsoft SQL Server Reporting Services ไม่มีความสามารถในตัวสำหรับการส่งออกรายงานเป็นงานนำเสนอ Microsoft PowerPoint แต่หลังจากติดตั้ง Aspose.Slides for Reporting Services บนเซิร์ฟเวอร์ของคุณ คุณจะได้รับการเข้าถึงรูปแบบการส่งออกเพิ่มเติม:

- PPT - การนำเสนอ PowerPoint ผ่าน Aspose.Slides
- PPS - สไลด์โชว์ PowerPoint ผ่าน Aspose.Slides
- PPTX - การนำเสนอ PowerPoint 2007 ผ่าน Aspose.Slides
- PPSX - สไลด์โชว์ PowerPoint 2007 ผ่าน Aspose.Slides

Aspose.Slides for Reporting Services สร้างงานนำเสนอบนเซิร์ฟเวอร์โดยไม่ใช้ Microsoft PowerPoint โดยภายใน Aspose.Slides for Reporting Services ใช้ Aspose.Slides for .NET – ส่วนประกอบระดับโลกสำหรับการประมวลผลงานนำเสนอบนเซิร์ฟเวอร์

**Aspose.Slides for Reporting Services ทำให้สามารถส่งออกรายงานใด ๆ ในรูปแบบ PPT, PPS, PPTX หรือ PPSX**
