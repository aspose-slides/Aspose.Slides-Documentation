**การติดตั้ง Aspose.Slides for SharePoint จะเพิ่มตัวเลือก Convert via Aspose.Slides ไปยังเมนูของเอกสาร**

## **การแปลงพรีเซนเทชัน**
เพื่อแปลงเอกสาร Microsoft PowerPoint จากไลบรารีเอกสารของ SharePoint: 

1. เลือกเอกสาร Microsoft PowerPoint ในไลบรารีเอกสาร.
2. คลิกที่ลูกศรลงเพื่อเปิดเมนูและคลิก **Convert via Aspose.Slides.SharePoint**. 

   **เมนูของไฟล์ Presentation 2 แสดงตัวเลือก Convert via Aspose.Slides** 

![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_2.png)




3. เลือกรูปแบบผลลัพธ์ที่ต้องการจากแบบฟอร์ม หากต้องการสามารถเปลี่ยนชื่อไฟล์ผลลัพธ์และโฟลเดอร์ปลายทาง.
4. คลิก **Convert** เพื่อแปลงไฟล์. 

   **แบบฟอร์มการแปลงให้คุณเลือกรูปแบบไฟล์ผลลัพธ์, ชื่อและตำแหน่งปลายทาง** 

![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_3.png)




5. เมื่การแปลงเสร็จสมบูรณ์ จะแสดงข้อความสำเร็จ. 

   **การแปลงสำเร็จ** 

![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_4.png)




6. คลิก **Source Library** (เพื่อไปยังไดเรกทอรีต้นทาง) หรือ **Destination Library** (เพื่อไปยังไดเรกทอรีที่ไฟล์ถูกบันทึกไว้). 

   เอกสารที่แปลงแล้วจะปรากฏในไลบรารีเอกสาร. 

   **เอกสารที่แปลงแล้วแสดงในไลบรารีที่บันทึกไว้** 

![todo:image_alt_text](converting-microsoft-powerpoint-documents-into-other-formats_5.png)
