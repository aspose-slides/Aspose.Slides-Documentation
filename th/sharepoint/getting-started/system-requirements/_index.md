## **ข้อกำหนดของเซิร์ฟเวอร์**
{{% alert color="primary" %}} 

เพื่อทำการติดตั้ง Aspose.Slides for SharePoint จำเป็นต้องใช้ระบบที่ทำงานบนหนึ่งในต่อไปนี้:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft Office SharePoint Server 2010
- Microsoft Office SharePoint Server 2013
- Microsoft Office SharePoint Server 2016
- Microsoft Office SharePoint Server 2019

{{% /alert %}} {{% alert color="primary" %}} 

แนะนำให้ติดตั้ง Service Pack 2 หรือรุ่นที่ใหม่กว่าสำหรับผลิตภัณฑ์เหล่านี้

{{% /alert %}} 
## **ข้อกำหนดของไคลเอนต์**
{{% alert color="primary" %}} 

SharePoint sites เข้าถึงผ่านเว็บเบราว์เซอร์ Aspose.Slides for SharePoint ได้รับการทดสอบกับเบราว์เซอร์ต่อไปนี้:

- Microsoft Internet Explorer 6, 7, 8
- Mozilla FireFox 3.5

{{% /alert %}}
