Aspose.Slides for Python via .NET เป็นไลบรารีคลาสที่ช่วยให้แอปพลิเคชัน Python สามารถอ่านและเขียนเอกสารนำเสนอได้โดยไม่ต้องใช้ Microsoft PowerPoint® หรือซอฟต์แวร์อื่น Aspose.Slides for Python via .NET ช่วยให้คุณจัดการข้อความ, รูปร่าง, ตารางและแอนิเมชัน; เพิ่มเสียงและวิดีโอลงในสไลด์; แสดงตัวอย่างสไลด์; และส่งออกสไลด์เป็นรูปแบบ SVG, PDF และอื่น ๆ

อ้างอิง API ล่าสุดสามารถพบได้ที่ [อ้างอิง API ของ Aspose.Slides for Python via .NET](https://reference.aspose.com/slides/th/python-net/).
