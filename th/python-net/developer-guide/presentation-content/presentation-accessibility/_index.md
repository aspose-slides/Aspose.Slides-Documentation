## **บทนำ**

ความสามารถในการเข้าถึงการนำเสนอทำให้ผู้ใช้เทคโนโลยีเสริม—เช่นหน้าจออ่านเสียง, หน้าจอเบรลล์, หรือการนำทางด้วยแป้นพิมพ์เท่านั้น—สามารถเข้าใจและเรียกดูสไลด์ของคุณได้อย่างมีประสิทธิภาพเทียบเท่ากับผู้ชมที่มองเห็นและใช้เมาส์. แนวปฏิบัติที่ดีมุ่งเน้นที่ลำดับการอ่านที่ชัดเจน, ข้อความแทนภาพที่มีความหมายสำหรับภาพข้อมูล, ความต่างสีที่เพียงพอ, ตัวอักษรที่อ่านง่าย, ข้อความลิงก์ที่อธิบายชัดเจน, และการหลีกเลี่ยงการสื่อความหมายโดยอาศัยสีหรือตำแหน่งอย่างเดียว. เมื่อวางแผนการเข้าถึงตั้งแต่เริ่มต้น ผลลัพธ์คือโครงสร้างที่เป็นระเบียบมากขึ้น, ภาพที่สอดคล้องกันมากขึ้น, และเนื้อหาที่เข้าถึงผู้ชมทุกคนโดยไม่ต้องใช้วิธีแก้ไขพิเศษ.

## **ทำเครื่องหมายเป็นของประดับ**

การทำเครื่องหมายเป็นของประดับเป็นการระบุภาพที่เป็นเพียงการตกแต่งเพื่อให้โปรแกรมอ่านหน้าจอข้ามผ่าน, ลดความรบกวนและให้ความสนใจอยู่ที่เนื้อหาที่มีความหมาย. ใช้กับพื้นหลัง, ลวดลาย, และช่องว่าง—ไม่ควรใช้กับแผนภูมิ, ไอคอน, หรือภาพที่สื่อข้อมูล. Aspose.Slides เปิดเผยฟลักนี้สำหรับการตรวจจับและการตรวจสอบ, ทำให้สามารถตรวจสอบการเข้าถึงอัตโนมัติและทำความสะอาดได้.

![Mark as Decorative](mark_as_decorative.png)

ตัวอย่างโค้ดต่อไปนี้แสดงวิธีกำหนดว่ารูปทรงถูกทำเครื่องหมายเป็นของประดับหรือไม่.

```py
import aspose.slides as slides

with slides.Presentation("sample.pptx") as presentation:
    shape = presentation.slides[0].shapes[0]
    print(f"Is shape decorative: {shape.is_decorative}")
```
