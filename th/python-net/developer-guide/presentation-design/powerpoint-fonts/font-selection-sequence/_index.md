## **ภาพรวม**

เมื่อการนำเสนอถูกโหลด, แสดงผล, หรือแปลงเป็นรูปแบบอื่น, Aspose.Slides จะตรวจสอบว่าฟอนต์ที่ใช้ในงานนำเสนอมีอยู่ในระบบปฏิบัติการหรือไม่ หากฟอนต์ที่จำเป็นไม่มีอยู่, Aspose.Slides จะเลือกฟอนต์ทดแทนที่ใกล้เคียงที่สุดกับฟอนต์ที่ PowerPoint จะใช้

Aspose.Slides จะค้นหาฟอนต์ที่เลือกในระบบปฏิบัติการก่อน หากพบฟอนต์นั้นจะใช้มัน หากไม่พบจะใช้ฟอนต์ทดแทนที่เหมาะสม เมื่อมีการกำหนดกฎการแทนฟอนต์ผ่าน `FontSubstRule` กฎเหล่านั้นก็จะถูกรับพิจารณาด้วย

คุณสามารถเพิ่มฟอนต์ในระยะเวลารันของแอปพลิเคชัน, ใช้ฟอนต์ที่ฝังอยู่ในงานนำเสนอ, หรือโหลดฟอนต์ภายนอกสำหรับเอกสารผลลัพธ์เช่นไฟล์ PDF ได้

## **การเลือกฟอนต์**

มีกฎบางอย่างที่ใช้กับฟอนต์ในงานนำเสนอเมื่อโหลด, แสดงผล, หรือแปลงเป็นรูปแบบอื่น ตัวอย่างเช่นเมื่อคุณพยายามแปลงงานนำเสนอ (สไลด์) เป็นภาพ, ฟอนต์ของงานนำเสนอจะถูกตรวจสอบเพื่อยืนยันว่าฟอนต์ที่เลือกมีอยู่ในระบบปฏิบัติการหรือไม่ หากฟอนต์ถูกยืนยันว่าไม่มีอยู่ จะถูกแทนที่ — ดู [**Font Replacement**](https://docs.aspose.com/slides/th/python-net/font-replacement/) และ [**Font Substitution**](https://docs.aspose.com/slides/th/python-net/font-substitution/)

นี่คือกระบวนการที่ Aspose.Slides ปฏิบัติตามเมื่อจัดการกับฟอนต์:

1. Aspose.Slides ค้นหาฟอนต์ในระบบปฏิบัติการเพื่อหาฟอนต์ที่ตรงกับฟอนต์ที่เลือกในงานนำเสนอ
2. หากพบฟอนต์ที่เลือก Aspose.Slides จะใช้มัน หากไม่พบ Aspose.Slides จะใช้ฟอนต์ทดแทนที่ใกล้เคียงที่สุดกับที่ PowerPoint จะใช้
3. หากมีกฎการแทนฟอนต์ที่ตั้งค่าไว้ผ่าน [FontSubstRule](https://reference.aspose.com/slides/th/python-net/aspose.slides/fontsubstrule/) จะนำไปใช้

Aspose.Slides อนุญาตให้คุณเพิ่มฟอนต์ในระยะเวลารันของแอปพลิเคชันและใช้ฟอนต์เหล่านั้น ดู [**Custom fonts**](https://docs.aspose.com/slides/th/python-net/custom-font/)

เมื่อฟอนต์เพิ่มเติมถูกฝังไว้ในงานนำเสนอ จะถูกเรียกว่า [**Embedded fonts**](https://docs.aspose.com/slides/th/python-net/embedded-font/)

Aspose.Slides อนุญาตให้คุณเพิ่มฟอนต์ที่ใช้เฉพาะกับเอกสารผลลัพธ์เท่านั้น ตัวอย่างเช่นหากงานนำเสนอที่คุณต้องการแปลงเป็น PDF มีฟอนต์ที่หายไปจากระบบและฟอนต์ที่ฝังไว้ คุณสามารถเพิ่มหรือโหลดฟอนต์ที่จำเป็นเป็น **external fonts** ได้

{{% alert title="Note" color="primary" %}} 
เราไม่จัดจำหน่ายฟอนต์ใดๆ ไม่ว่าจะเป็นแบบชำระเงินหรือฟรี API ของเราช่วยให้คุณโหลดฟอนต์ภายนอกและฝังลงในเอกสารได้ แต่การทำเช่นนั้นต้องทำด้วยฟอนต์ที่คุณเลือกและรับผิดชอบเอง

## **FAQ**

**ฉันจะกำหนดได้อย่างไรว่าฟอนต์ใดที่ถูกใช้ในงานนำเสนอจริงก่อนการแปลง?**

Aspose.Slides ให้คุณตรวจสอบฟอนต์ที่ใช้ผ่าน [font manager](https://reference.aspose.com/slides/th/python-net/aspose.slides/presentation/fonts_manager/), เพื่อให้คุณตัดสินใจว่าจะ [embed](/slides/th/python-net/embedded-font/), [replace](/slides/th/python-net/font-replacement/), หรือเพิ่ม [external sources](/slides/th/python-net/custom-font/) หรือไม่ สิ่งนี้ช่วยป้องกันการแทนที่ที่ไม่ต้องการระหว่างการแสดงผลและการส่งออก

**ฉันสามารถเพิ่มไดเรกทอรีฟอนต์เพิ่มเติมโดยไม่ต้องติดตั้งบนระบบปฏิบัติการได้หรือไม่?**

ได้ คุณสามารถลงทะเบียน [external font sources](/slides/th/python-net/custom-font/) เช่น โฟลเดอร์หรือสตรีมในหน่วยความจำสำหรับการแสดงผลและการส่งออก สิ่งนี้ลดการพึ่งพาฟอนต์ของโฮสต์ระบบและทำให้การจัดหน้าเป็นไปอย่างคาดเดาได้

**ฉันจะป้องกันไม่ให้ระบบกลับไปใช้ฟอนต์ที่ไม่เหมาะสมโดยอัตโนมัติเมื่อไม่มี glyph หรือไม่?**

กำหนด [font replacement](/slides/th/python-net/font-replacement/) และกฎ [fallBack](/slides/th/python-net/fallback-font/) อย่างชัดเจนล่วงหน้า โดยวิเคราะห์ฟอนต์ที่ใช้และตั้งค่าลำดับความสำคัญของฟอนต์ทดแทน คุณจะมั่นใจว่าการจัดรูปแบบตัวอักษรสอดคล้องและหลีกเลี่ยงผลลัพธ์ที่ไม่คาดคิด
