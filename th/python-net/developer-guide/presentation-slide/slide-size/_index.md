## **บทนำ**

Aspose.Slides มีเครื่องมือครบวงจรสำหรับปรับขนาดสไลด์และอัตราส่วนของสไลด์ในงานนำเสนอ PowerPoint ซึ่งสำคัญทั้งสำหรับการพิมพ์และการแสดงผลบนหน้าจอ  

ขนาดสไลด์และอัตราส่วนที่นิยม:

- **Standard (4:3 Aspect Ratio)**: เหมาะสำหรับหน้าจอและอุปกรณ์รุ่นเก่า.
- **Widescreen (16:9 Aspect Ratio)**: แนะนำสำหรับโปรเจ็กเตอร์และจอแสดงผลสมัยใหม่.

ตรวจสอบความสม่ำเสมอตลอดการนำเสนอ เนื่องจากขนาดสไลด์และอัตราส่วนเดียวกันจะใช้กับทุกสไลด์ เพื่อผลลัพธ์ที่ดีที่สุด ควรกำหนดขนาดสไลด์ตั้งแต่เริ่มสร้างงานนำเสนอเพื่อหลีกเลี่ยงปัญหา.

{{% alert color="primary" %}} 
โดยค่าเริ่มต้น งานนำเสนอที่สร้างด้วย Aspose.Slides จะใช้แบบอัตราส่วนมาตรฐาน 4:3.
{{% /alert %}}

## **เปลี่ยนขนาดสไลด์ในงานนำเสนอ**

ตัวอย่างโค้ดนี้แสดงวิธีเปลี่ยนขนาดสไลด์ในงานนำเสนอโดยใช้ Python กับ Aspose.Slides:

```py
import aspose.slides as slides

with slides.Presentation(path + "AccessSlides.pptx") as pres:
    pres.slide_size.set_size(slides.SlideSizeType.ON_SCREEN16X9, slides.SlideSizeScaleType.DO_NOT_SCALE)
    pres.save("pres-4x3-aspect-ratio.pptx", slides.export.SaveFormat.PPTX)
```

## **ระบุขนาดสไลด์แบบกำหนดเอง**

หากคุณพบว่าขนาดสไลด์ทั่วไป (4:3 และ 16:9) ไม่เหมาะกับงานของคุณ คุณอาจเลือกใช้ขนาดสไลด์เฉพาะหรือขนาดที่ไม่ซ้ำใคร ตัวอย่างเช่น หากคุณต้องการพิมพ์สไลด์ขนาดเต็มจากงานนำเสนอบนรูปแบบหน้ากระดาษที่กำหนดเอง หรือหากคุณต้องการแสดงงานนำเสนอบนประเภทหน้าจอบางประเภท คุณจะได้รับประโยชน์จากการกำหนดขนาดสไลด์แบบกำหนดเองสำหรับงานนำเสนอของคุณ  

ตัวอย่างโค้ดนี้แสดงวิธีใช้ Aspose.Slides สำหรับ Python ผ่าน .NET เพื่อกำหนดขนาดสไลด์แบบกำหนดเองสำหรับงานนำเสนอใน Python:

```py
import aspose.slides as slides

with slides.Presentation(path + "AccessSlides.pptx") as pres:
    pres.slide_size.set_size(780, 540, slides.SlideSizeScaleType.DO_NOT_SCALE) # ขนาดกระดาษ A4
    pres.save("pres-a4-slide-size.pptx", slides.export.SaveFormat.PPTX)
```

## **จัดการเนื้อหาสไลด์หลังการปรับขนาด**

หลังจากที่คุณเปลี่ยนขนาดสไลด์ของงานนำเสนอ เนื้อหาสไลด์ (เช่น รูปภาพหรือวัตถุ) อาจเกิดการบิดเบือน โดยค่าเริ่มต้น วัตถุจะถูกปรับขนาดอัตโนมัติเพื่อให้พอดีกับขนาดสไลด์ใหม่ อย่างไรก็ตาม เมื่อเปลี่ยนขนาดสไลด์ของงานนำเสนอ คุณสามารถกำหนดการตั้งค่าที่กำหนดว่า Aspose.Slides จะจัดการกับเนื้อหาบนสไลด์อย่างไร  

ขึ้นอยู่กับสิ่งที่คุณต้องการทำหรือบรรลุ คุณสามารถใช้การตั้งค่าเหล่านี้ได้  

- `DO_NOT_SCALE`  
  หากคุณ **ไม่ต้องการ** ให้วัตถุบนสไลด์ถูกปรับขนาด ใช้การตั้งค่านี้.  

- `ENSURE_FIT`  
  หากคุณต้องการปรับขนาดลงสไลด์ที่เล็กลงและต้องการให้ Aspose.Slides ปรับขนาดวัตถุในสไลด์ลงเพื่อให้ทุกอย่างพอดีบนสไลด์ (วิธีนี้จะป้องกันการสูญเสียเนื้อหา) ใช้การตั้งค่านี้.  

- `MAXIMIZE`  
  หากคุณต้องการปรับขนาดขึ้นสไลด์ที่ใหญ่ขึ้นและต้องการให้ Aspose.Slides ขยายวัตถุในสไลด์ให้สัดส่วนกับขนาดสไลด์ใหม่ ใช้การตั้งค่านี้.  

ตัวอย่างโค้ดนี้แสดงวิธีใช้การตั้งค่า `MAXIMIZE` เมื่อเปลี่ยนขนาดสไลด์ของงานนำเสนอ:

```py
import aspose.slides as slides

with slides.Presentation(path + "AccessSlides.pptx") as pres:
   pres.slide_size.set_size(slides.SlideSizeType.LEDGER, slides.SlideSizeScaleType.MAXIMIZE)
```

## **คำถามที่พบบ่อย**

**ฉันสามารถกำหนดขนาดสไลด์แบบกำหนดเองโดยใช้หน่วยอื่นนอกจากนิ้ว (เช่น จุดหรือมิลลิเมตร) ได้หรือไม่?**

ใช่ Aspose.Slides ใช้หน่วยจุดภายใน โดย 1 จุดเท่ากับ 1/72 นิ้ว คุณสามารถแปลงหน่วยใด ๆ (เช่น มิลลิเมตรหรือเซนติเมตร) ไปเป็นจุดและใช้ค่าที่แปลงแล้วกำหนดความกว้างและความสูงของสไลด์  

**ขนาดสไลด์แบบกำหนดเองที่ใหญ่มากจะส่งผลต่อประสิทธิภาพและการใช้หน่วยความจำระหว่างการเรนเดอร์หรือไม่?**

ใช่ การเพิ่มขนาดสไลด์ (เป็นจุด) พร้อมกับสเกลการเรนเดอร์ที่สูงขึ้นจะทำให้การใช้หน่วยความจำเพิ่มขึ้นและเวลาประมวลผลนานขึ้น ควรกำหนดขนาดสไลด์ที่เหมาะสมและปรับสเกลการเรนเดอร์ตามความจำเป็นเพื่อให้ได้คุณภาพผลลัพธ์ที่ต้องการ  

**ฉันสามารถกำหนดขนาดสไลด์ที่ไม่เป็นมาตรฐานหนึ่งขนาดแล้วรวมสไลด์จากงานนำเสนอที่มีขนาดต่างกันได้หรือไม่?**

คุณไม่สามารถ [รวมงานนำเสนอ](/slides/th/python-net/merge-presentation/) ในขณะที่มีขนาดสไลด์ต่างกัน — ให้ปรับขนาดงานนำเสนอหนึ่งให้ตรงกับอีกงานก่อน เมื่อเปลี่ยนขนาดสไลด์ คุณสามารถเลือกวิธีการจัดการกับเนื้อหาที่มีอยู่ผ่านตัวเลือก [SlideSizeScaleType](https://reference.aspose.com/slides/th/python-net/aspose.slides/slidesizescaletype/) หลังจากปรับขนาดให้ตรงกันแล้ว คุณสามารถรวมสไลด์โดยคงรูปแบบไว้ได้  

**ฉันสามารถสร้างภาพย่อสำหรับรูปร่างแต่ละอันหรือพื้นที่เฉพาะของสไลด์ได้หรือไม่ และภาพย่อจะเคารพขนาดสไลด์ใหม่หรือไม่?**

ใช่ Aspose.Slides สามารถเรนเดอร์ภาพย่อสำหรับ [สไลด์ทั้งหมด](https://reference.aspose.com/slides/th/python-net/aspose.slides/slide/get_image/) รวมถึง [รูปร่างที่เลือก](https://reference.aspose.com/slides/th/python-net/aspose.slides/shape/get_image/) ภาพที่ได้จะสะท้อนขนาดสไลด์และอัตราส่วนปัจจุบัน ทำให้กรอบและรูปทรงคงที่
