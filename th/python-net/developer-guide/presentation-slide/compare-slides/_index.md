## **Overview**

Aspose.Slides อนุญาตให้คุณเปรียบเทียบสไลด์, สไลด์เค้าโครง, และสไลด์แม่โดยใช้เมธอด `equals` ที่มีอยู่ในคลาส `BaseSlide` เมธอดนี้จะคืนค่า `True` เมื่อสไลด์ที่เปรียบเทียบมีโครงสร้างและเนื้อหาแบบคงที่เหมือนกัน

## **เปรียบเทียบสไลด์สองสไลด์**
`equals` เมธอดได้ถูกเพิ่มเข้าไปในคลาส [BaseSlide](https://reference.aspose.com/slides/th/python-net/aspose.slides/baseslide/) มันจะคืนค่า true สำหรับสไลด์/เค้าโครงและสไลด์/แม่ที่มีโครงสร้างและเนื้อหาแบบคงที่เหมือนกัน

สองสไลด์ถือว่าเท่ากันหากรูปทรง, สไตล์, ข้อความ, การเคลื่อนไหวและการตั้งค่าอื่น ๆ ทั้งหมดเหมือนกัน เป็นต้น การเปรียบเทียบจะไม่พิจารณาค่าตัวระบุที่เป็นเอกลักษณ์ เช่น SlideId และเนื้อหาแบบไดนามิก เช่น ค่าของวันที่ปัจจุบันในตัวเว้นที่วันที่

```py
import aspose.slides as slides

with slides.Presentation(path + "AccessSlides.pptx") as p1:
    with slides.Presentation(path + "HelloWorld.pptx") as p2:
        for i in range(len(p1.masters)):
            for j in range(len(p2.masters)):
                if p1.masters[i].equals(p2.masters[j]):
                    print("Presentation1 MasterSlide#{0} is equal to Presentation2 MasterSlide#{1}".format(i,j))
```

## **คำถามที่พบบ่อย**

**การที่สไลด์ถูกซ่อนมีผลต่อการเปรียบเทียบสไลด์หรือไม่?**

[สถานะซ่อน](https://reference.aspose.com/slides/th/python-net/aspose.slides/slide/hidden/) เป็นคุณสมบัติระดับการนำเสนอ/การเล่น ไม่ได้เป็นเนื้อหาภาพ. ความเท่าเทียมของสไลด์สองสไลด์จะกำหนดโดยโครงสร้างและเนื้อหาแบบคงที่; การที่สไลด์หนึ่งถูกซ่อนเพียงอย่างเดียวไม่ได้ทำให้สไลด์แตกต่างกัน

**ลิงก์ไฮเปอร์และพารามิเตอร์ของมันจะถูกนำมาพิจารณาหรือไม่?**

ใช่. ลิงก์เป็นส่วนหนึ่งของเนื้อหาแบบคงที่ของสไลด์ หาก URL หรือการกระทำของไฮเปอร์ลิงก์ต่างกัน จะถือว่าเป็นความแตกต่างของเนื้อหาแบบคงที่

**หากแผนภูมิอ้างอิงไฟล์ Excel ภายนอก เนื้อหาของไฟล์นั้นจะถูกนำมาพิจารณาหรือไม่?**

ไม่. การเปรียบเทียบทำบนพื้นฐานของสไลด์เอง แหล่งข้อมูลภายนอกโดยปกติจะไม่ถูกอ่านในขณะเปรียบเทียบ; มีเพียงสิ่งที่อยู่ในโครงสร้างและสถานะแบบคงของสไลด์เท่านั้นที่นำมาพิจารณา
