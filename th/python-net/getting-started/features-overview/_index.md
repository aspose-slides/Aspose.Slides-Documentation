## **แพลตฟอร์มที่รองรับ**
แพลตฟอร์มที่ Aspose.Slides for Python via .NET สามารถใช้ได้บน Windows x64 หรือ x86 และหลายดิสทริบิวชันของ Linux ที่ติดตั้ง Python 3.5 หรือใหม่กว่า มีข้อกำหนดเพิ่มเติมสำหรับแพลตฟอร์ม Linux ที่เป้าหมาย:
- ไลบรารีรันไทม์ GCC-6 (หรือรุ่นที่ใหม่กว่า)
- การพึ่งพาของ .NET Core Runtime. การติดตั้ง .NET Core Runtime เองไม่จำเป็น
- สำหรับ Python 3.5‑3.7: จำเป็นต้องใช้การสร้าง Python แบบ `pymalloc`. ตัวเลือกการสร้าง Python `--with-pymalloc` เปิดใช้งานโดยค่าเริ่มต้น โดยทั่วไปการสร้าง Python แบบ `pymalloc` จะมีสฟิกซ์ `m` ในชื่อไฟล์
- ไลบรารี Python แบบแชร์ `libpython`. ตัวเลือกการสร้าง Python `--enable-shared` ปิดใช้งานเป็นค่าเริ่มต้น บางการแจกจ่าย Python ไม่มีไลบรารี `libpython` แบบแชร์ สำหรับบางแพลตฟอร์ม Linux สามารถติดตั้งไลบรารี `libpython` ผ่านผู้จัดการแพ็กเกจได้ เช่น `sudo apt-get install libpython3.7`. ปัญหาที่พบบ่อยคือไลบรารี `libpython` ถูกติดตั้งในตำแหน่งที่ต่างจากตำแหน่งมาตรฐานของระบบสำหรับไลบรารีแบบแชร์ ปัญหานี้สามารถแก้ได้โดยใช้ตัวเลือกการสร้าง Python เพื่อตั้งค่าที่ตั้งไลบรารีทางเลือกเมื่อคอมไพล์ Python หรือโดยการสร้างลิงก์สัญลักษณ์ไปยังไฟล์ไลบรารี `libpython` ในตำแหน่งมาตรฐานของระบบ สำหรับไลบรารี `libpython` แบบแชร์ ชื่อไฟล์โดยทั่วไปคือ `libpythonX.Ym.so.1.0` สำหรับ Python 3.5‑3.7 หรือ `libpythonX.Y.so.1.0` สำหรับ Python 3.8 หรือใหม่กว่า (เช่น `libpython3.7m.so.1.0`, `libpython3.9.so.1.0`).

หากคุณต้องการการรองรับแพลตฟอร์มเพิ่มเติม ให้มองหาผลิตภัณฑ์ “พี่น้องใกล้เคียง” ของ Aspose.Slides for .NET หรือ Aspose.Slides for Java.

## **FAQ**

**ฉันต้องติดตั้ง Microsoft PowerPoint บนเซิร์ฟเวอร์/พีซี เพื่อให้ไลบรารีทำงานหรือไม่?**

ไม่จำเป็น. PowerPoint ไม่ต้องการ; Aspose.Slides เป็นเอนจินอิสระสำหรับการสร้าง, แก้ไข, แปลง, และเรนเดอร์การนำเสนอ.

**การทำงานของมัลติเทรดเป็นอย่างไร? สามารถทำการประมวลผลในแบบขนานได้หรือไม่?**

ปลอดภัยที่จะประมวลผลเอกสารต่าง ๆ ในเธรดต่าง ๆ; วัตถุ [presentation](https://reference.aspose.com/slides/th/python-net/aspose.slides/presentation/) เดียวกันไม่ควรถูกใช้โดย [multiple threads](/slides/th/python-net/multithreading/) พร้อมกัน.

**รองรับรหัสผ่านไฟล์และการเข้ารหัสหรือไม่?**

ใช่. [คุณสามารถ](/slides/th/python-net/password-protected-presentation/) เปิดงานนำเสนอที่เข้ารหัส ตั้งหรือถอนรหัสผ่านสำหรับเปิดและเขียน และตรวจสอบสถานะการป้องกัน.

**ฉันต้องดูแลแพ็กเกจฟอนท์ในคอนเทนเนอร์ Linux หรือไม่?**

ใช่. แนะนำให้ติดตั้งแพ็กเกจฟอนท์ทั่วไปและ/หรือระบุ [ระบุไดเรกทอรีฟอนท์](/slides/th/python-net/custom-font/) อย่างชัดเจนในแอปพลิเคชันของคุณเพื่อหลีกเลี่ยงการแทนที่ที่ไม่คาดคิด.

**มีข้อจำกัดในรุ่นประเมินหรือไม่?**
