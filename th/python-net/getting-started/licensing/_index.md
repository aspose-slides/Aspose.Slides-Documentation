## **การใช้ลิขสิทธิ์**

ลิขสิทธิ์สามารถโหลดจาก **ไฟล์**, **สตรีม**, หรือ **แหล่งฝัง** ได้.

{{% alert color="primary" %}}
Aspose.Slides มีคลาส[License](https://reference.aspose.com/slides/th/python-net/aspose.slides/license/)เพื่อจัดการลิขสิทธิ์
{{% /alert %}}

{{% alert color="warning" %}}
ใบอนุญาตใหม่สามารถเปิดใช้งาน Aspose.Slides ได้เฉพาะกับเวอร์ชัน 21.4 หรือใหม่กว่า เวอร์ชันก่อนใช้ระบบลิขสิทธิ์แตกต่างและจะไม่รับรู้ใบอนุญาตเหล่านี้
{{% /alert %}}

### **ไฟล์**

วิธีที่ง่ายที่สุดในการตั้งค่าลิขสิทธิ์คือวางไฟล์ลิขสิทธิ์ในโฟลเดอร์เดียวกับ DLL ของคอมโพเนนต์และระบุแค่ชื่อไฟล์ (ไม่รวมเส้นทาง)

โค้ด Python ต่อไปนี้แสดงวิธีตั้งค่าไฟล์ลิขสิทธิ์:

```py
import aspose.slides as slides

# สร้างอินสแตนซ์ของคลาส License. 
license = slides.License()

# ตั้งค่าเส้นทางไฟล์ลิขสิทธิ์.
license.set_license("Aspose.Slides.lic")
```

{{% alert color="warning" %}}
หากคุณวางไฟล์ลิขสิทธิ์ในไดเรกทอรีอื่น เมื่อคุณเรียก[License.set_license()](https://reference.aspose.com/slides/th/python-net/aspose.slides/license/set_license/#str) ชื่อไฟล์ที่ส่วนท้ายของเส้นทางต้องตรงกับชื่อไฟล์ลิขสิทธิ์ของคุณ

เช่น คุณสามารถเปลี่ยนชื่อไฟล์ลิขสิทธิ์เป็น *Aspose.Slides.lic.xml* จากนั้นในโค้ดของคุณ ส่งเส้นทางเต็มไปยังไฟล์นั้น (ลงท้ายด้วย Aspose.Slides.lic.xml) ให้กับเมธอด[License.set_license()](https://reference.aspose.com/slides/th/python-net/aspose.slides/license/set_license/#str)
{{% /alert %}}

### **สตรีม**

คุณสามารถโหลดลิขสิทธิ์จากสตรีม ตัวอย่าง Python ต่อไปนี้แสดงวิธีใช้ลิขสิทธิ์จากสตรีม:

```py
import aspose.slides as slides

# สร้างอินสแตนซ์ของคลาส License.
license = slides.License()

# ตั้งค่าลิขสิทธิ์จากสตรีม.
license.set_license(stream)
```

## **การตรวจสอบลิขสิทธิ์**

เพื่อยืนยันว่าลิขสิทธิ์ได้ถูกใช้ถูกต้อง คุณสามารถตรวจสอบได้ โค้ด Python ต่อไปนี้แสดงวิธีตรวจสอบลิขสิทธิ์:

```py
import aspose.slides as slides

license = slides.License()

license.set_license("Aspose.Slides.lic")

if license.is_licensed():
    print("License is good!")
```
