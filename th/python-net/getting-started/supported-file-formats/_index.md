## **ภาพรวม**

Aspose.Slides รองรับไฟล์งานนำเสนอจาก Microsoft PowerPoint 97 จนถึง Office 365 รวมถึง Microsoft PowerPoint for Mac บทความนี้แสดงรายการเวอร์ชันของ PowerPoint ที่ไลบรารีสนับสนุนและให้ตารางรูปแบบไฟล์ที่สามารถโหลด บันทึก หรือทั้งสองอย่างได้  

บทความนี้ยังตอบคำถามทั่วไปเกี่ยวกับการปฏิบัติตามมาตรฐาน PDF การฝังฟอนต์ ไฟล์ที่ป้องกันด้วยรหัสผ่าน ฟอนต์แบบกำหนดเอง การสำรองฟอนต์ และตัวเลือกการส่งออก XPS  

## **เวอร์ชัน Microsoft PowerPoint ที่รองรับ**

## **คำถามที่พบบ่อย**

**ฉันสามารถบันทึกงานนำเสนอเป็น PDF ที่ตรงตามมาตรฐานการเก็บรักษาและการเข้าถึง (PDF/A และ PDF/UA) ได้หรือไม่?**  
ใช่ Aspose.Slides for Python via .NET รองรับการส่งออกเป็น PDF พร้อมระดับการปฏิบัติตามเช่น PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b รวมถึง PDF/UA ผ่านการตั้งค่า [การปฏิบัติตาม](https://reference.aspose.com/slides/th/python-net/aspose.slides.export/pdfoptions/compliance/) ใน [ตัวเลือกการส่งออก PDF](https://reference.aspose.com/slides/th/python-net/aspose.slides.export/pdfoptions/)  

**ไลบรารีสนับสนุนการฝังฟอนต์เมื่อส่งออกเป็น PDF พร้อมการควบคุมละเอียดว่าฝังอะไรบ้างหรือไม่?**  
ใช่ คุณสามารถควบคุมได้ว่าฟอนต์จะถูกฝังเต็มหรือเป็นส่วนย่อย (เฉพาะ glyph ที่ใช้) ระบุว่าฟอนต์ระบบทั่วไปจะถูกจัดการอย่างไร และกำหนดพฤติกรรมสำหรับข้อความ ASCII ผ่าน [ตัวเลือกการส่งออก PDF](https://reference.aspose.com/slides/th/python-net/aspose.slides.export/pdfoptions/)  

**ฉันสามารถตรวจสอบว่าไฟล์ได้รับการป้องกันด้วยรหัสผ่านหรือไม่ก่อนที่จะโหลดจริงหรือไม่?**  
ใช่ โดยใช้ [API การตรวจสอบแบบ factory](https://reference.aspose.com/slides/th/python-net/aspose.slides/presentationfactory/) คุณสามารถสอบถามไฟล์งานนำเสนอเพื่อพิจารณาว่ามีการป้องกันด้วยรหัสผ่านหรือไม่โดยไม่ต้องเปิดไฟล์ทั้งหมด  

**มีกลไกการสำรองฟอนต์และการสนับสนุนฟอนต์แบบกำหนดเองหรือไม่?**  
ใช่ ไลบรารีสนับสนุนการ [การโหลด](/slides/th/python-net/custom-font/) และ [การฝัง](/slides/th/python-net/embedded-font/) ฟอนต์แบบกำหนดเอง และให้ [กฎการสำรองฟอนต์](/slides/th/python-net/fallback-font/) เพื่อป้องกันไม่ให้ glyph หายระหว่างการแสดงผลและการแปลง  

**ฉันสามารถส่งออกสไลด์เป็น XPS ได้หรือไม่ และมีตัวเลือกในการปรับแต่งผลลัพธ์ XPS หรือไม่?**  
ใช่ การ [ส่งออกเป็น XPS](/slides/th/python-net/convert-powerpoint-to-xps/) ได้รับการสนับสนุน และคุณสามารถปรับ [ตัวเลือกการบันทึก](https://reference.aspose.com/slides/th/python-net/aspose.slides.export/xpsoptions/) ที่เกี่ยวข้องเพื่อควบคุมคุณภาพและเนื้อหาของเอกสาร XPS.
