## **คำอธิบายผลิตภัณฑ์**

Aspose.Slides for Node.js via .NET เป็นไลบรารีคลาสข้ามแพลตฟอร์มที่ช่วยให้แอปพลิเคชันของคุณทำงานประมวลผลเอกสารหลากหลายประเภทโดยไม่ต้องใช้ Microsoft PowerPoint®

## **ทำไมคุณควรใช้ Aspose.Slides for Node.js via .NET**

- Aspose.Slides for Node.js via .NET เป็นคอมโพเนนต์ตัวแรกและตัวเดียวที่ให้ฟังก์ชันที่จำเป็นสำหรับการจัดการเอกสาร PowerPoint®  
- Aspose.Slides for Node.js via .NET มีคุณสมบัติหลักหลายอย่าง เช่น การจัดการข้อความ รูปทรง ตารางและแอนิเมชัน การเพิ่มเสียงและวิดีโอลงสไลด์ การดูตัวอย่างสไลด์ การส่งออกสไลด์เป็น SVG, PDF และรูปแบบอื่น ๆ รวมถึงความสามารถอื่น ๆ อีกมากมาย  
- Aspose.Slides for Node.js via .NET ไม่เพียงแค่เปิดไฟล์ PowerPoint จากแหล่งต่าง ๆ แต่ยังอนุญาตให้คุณบันทึกการนำเสนอในหลายวิธี ตัวอย่างเช่น คุณสามารถบันทึกการนำเสนอเป็น PPT หรือบันทึกสไลด์เป็นรูปภาพได้  
- Aspose.Slides for Node.js via .NET ให้ฟีเจอร์เกือบทั้งหมดที่คุณอาจพบหรือไม่พบใน Microsoft PowerPoint ด้วยความหลากหลายของ Aspose.Slides for Node.js via .NET นอกจากตารางและรูปทรงหลากหลายรูปแบบแล้ว คุณยังสามารถใช้เพื่อเพิ่มกรอบประเภทต่าง ๆ — กรอบข้อความ, กรอบเสียง และกรอบวิดีโอ — ลงในสไลด์ของคุณได้  

## **คุณสมบัติของ Aspose.Slides**

คุณสมบัติของ Aspose.Slides for Node.js via .NET สามารถแบ่งออกเป็นกลุ่มต่อไปนี้:

- คุณสมบัติการนำเสนอ
- คุณสมบัติสไลด์
- คุณสมบัติกราฟิกและสื่อมัลติมีเดีย
- คุณสมบัติตาราง
- คุณสมบัติเข้าถึงข้อความและย่อหน้า
- คุณสมบัตอลดขั้นสูง

### **คุณสมบัติการนำเสนอ**

- เปิดไฟล์ Microsoft PowerPoint จากแหล่งต่าง ๆ  
- บันทึกการนำเสนอในรูปแบบต่าง ๆ ของ Microsoft PowerPoint  
- สร้างการนำเสนอจากศูนย์  
- แปลงการนำเสนอเป็นเอกสาร PDF โดยไม่ต้องใช้คอมโพเนนต์อื่น  
- อ่านหรือแก้ไขคุณสมบัติของเอกสารการนำเสนอ  
- ใช้การป้องกันบนการนำเสนอ  
- ส่งออกการนำเสนอเป็น PDF  
- ส่งออกการนำเสนอเป็น TIFF  
- ส่งออกการนำเสนอเป็น XPS  
- ส่งออกการนำเสนอเป็น SWF  
- ส่งออกการนำเสนอเป็น PDF Note / Tiff Notes  
- ส่งออกการนำเสนอเป็น HTML  

### **คุณสมบัติสไลด์**

- สร้างสไลด์ใหม่จากศูนย์และเพิ่มลงในการนำเสนอ  
- เข้าถึงสไลด์ใด ๆ ผ่านตำแหน่งของมันในการนำเสนอ  
- คัดลอกหรือทำสำเนาสไลด์ไปยังการนำเสนอเดียวกันหรืออีกรายการหนึ่ง  
- แก้ไขและลบสไลด์  
- คัดลอกมาสเตอร์พร้อมสไลด์โดยอัตโนมัติ  
- ดูตัวอย่างสไลด์  
- รับหรือกำหนดชื่อสไลด์  
- บันทึกสไลด์เป็นรูปภาพ  
- ส่งออกสไลด์เป็นรูปแบบ SVG  
- ทำงานกับมาสเตอร์หลายตัวในการนำเสนอ  
- เติมพื้นหลังสไลด์ด้วยสไตล์ที่รองรับใด ๆ  
- เพิ่มหรืจัดการโน้ตสไลด์ใหม่  
- กำหนดมาสเตอร์และสไตล์ของสไลด์ใหม่แบบไดนามิก  
- เพิ่มหรือแก้ไขการเปลี่ยนภาพสไลด์โชว์  
- เพิ่มการเปลี่ยนภาพแบบ Morph  

### **คุณสมบัติกราฟิกและสื่อมัลติมีเดีย**

- สร้างหรือจัดการรูปทรงเช่นเส้น, สี่เหลี่ยม, วงรีและโพลีไลน์  
- จัดการสไตล์เส้นในรูปทรง  
- รับข้อความแทนที่ของรูปทรง  
- หมุนและพลิกรูปทรง  
- เติมรูปทรงในสไตล์ต่าง ๆ — เช่น ของแข็ง, เกรเดียนท์, แพทเทิร์น, เทกซ์처 และภาพ  
- ใช้รูปแบบพรีดีเฟินด์ 48 แบบและเทกซ์처 24 แบบได้ทั้งหมด  
- กำหนดลิงก์ให้กับรูปทรง  
- อ่าน, แก้ไขและเพิ่มเอฟเฟกต์แอนิเมชันใหม่ให้กับสไลด์และรูปทรง  
- เพิ่มรูปภาพใหม่ลงในการนำเสนอและจัดการรูปภาพที่มีอยู่แล้ว  
- เพิ่มรูปหัวข้อภาพใหม่ลงในการนำเสนอและจัดการรูปหัวข้อภาพที่มีอยู่แล้ว  
- เพิ่มกรอบเสียงใหม่ลงในการนำเสนอและจัดการกรอบเสียงที่มีอยู่แล้ว  
- เพิ่มกรอบวิดีโอใหม่ลงในการนำเสนอและจัดการกรอบวิดีโอที่มีอยู่แล้ว  

### **คุณสมบัติตาราง**

- สร้างตารางจากศูนย์และเพิ่มลงในสไลด์  
- ปรับขนาดและย้ายตาราง  
- เข้าถึงเซลล์ใด ๆ ในตาราง  
- เปลี่ยนรูปแบบและเส้นขอบของเซลล์ใด ๆ  
- ปรับขนาดคอลัมน์และแถว  
- เพิ่มและลบคอลัมน์และแถว  
- กำหนดความกว้างและสีของเส้นขอบสำหรับตารางทั้งหมด  

### **คุณสมบัติเข้าถึงข้อความและย่อหน้า**

- รับและกำหนดข้อความในช่องข้อความและกรอบข้อความ  
- รับประเภทของช่องข้อความ  
- เปลี่ยนการจัดรูปแบบข้อความในกรอบข้อความ  
- เพิ่มฟอนต์ใหม่หรือจัดการฟอนต์ที่ใช้แล้วในการนำเสนอ  
- เปลี่ยนข้อความและการจัดรูปแบบข้อความ  
- สร้างหรือจัดการลิงก์ไปยังข้อความ  
- รองรับแท็บและแท็บที่มีประสิทธิภาพ  
- เพิ่มกรอบข้อความใหม่ให้กับรูปทรงเกือบทุกประเภท  
- เข้าถึงและแก้ไขขอบเขตและการตัดคำของกรอบข้อความ  
- เข้าถึงและแก้ไขการเยื้องของย่อหน้า  

### **คุณสมบัติลดขั้นสูง**

- เข้าถึงวัตถุ OLE ที่ฝังอยู่ในการนำเสนอ  
- ล็อคการนำเสนอ  

## **การสนับสนุนทางเทคนิค**

Aspose ให้การสนับสนุนทางเทคนิคฟรีไม่จำกัดสำหรับผลิตภัณฑ์ทั้งหมดของบริษัท การสนับสนุนนี้เปิดให้กับผู้ใช้ทุกคน (รวมถึงผู้ใช้ที่มีแพคเกจทดลอง) หากคุณต้องการความช่วยเหลือเกี่ยวกับ Aspose.Slides for Node.js via .NET ให้พิจารณาตัวเลือกต่อไปนี้:

- ช่องทางหลักสำหรับการสนับสนุนคือ[ฟอรั่ม Aspose](https://forum.aspose.com/). โพสต์คำถามของคุณใน[ฟอรั่ม Aspose.Slides](https://forum.aspose.com/c/slides/th/11)—และทีมสนับสนุน Aspose.Slides จะตอบภายในไม่กี่ชั่วโมง  
- โปรดทราบว่า Aspose ไม่ให้การสนับสนุนทางเทคนิคผ่านโทรศัพท์ การสนับสนุนทางโทรศัพท์มีให้เฉพาะสำหรับคำถามด้านการขายและการสั่งซื้อเท่านั้น  
- เมื่อคาดหวังการตอบกลับในฟอรั่ม โปรดอดทนและพิจารณาความแตกต่างของโซนเวลา  

หากคุณพบปัญหาเกี่ยวกับ Aspose.Slides for Node.js via .NET ให้ทำตามแนวทางต่อไปนี้เพื่อแก้ไขอย่างมีประสิทธิภาพที่สุด:

- ตรวจสอบและยืนยันว่าคุณกำลังใช้เวอร์ชันล่าสุดของ Aspose.Slides for Node.js via .NET ก่อนที่จะรายงานปัญหา บางครั้งปัญหาที่คุณประสบอาจได้รับการแก้ไขแล้วในอัปเดต  

  ดู[ดาวน์โหลด Aspose.Slides](https://releases.aspose.com/slides/th/nodejs-net/)เพื่อรับข้อมูลเกี่ยวกับเวอร์ชันล่าสุดของผลิตภัณฑ์  

- ก่อนรายงานปัญหา คุณอาจต้องตรวจสอบฟอรั่มเอกสารนี้และ API Reference ก่อน บางครั้งคำตอบของคุณอาจมีอยู่แล้ว  

- เมื่อรายงานปัญหา กรุณาแนบเอกสารต้นฉบับและ(ถ้าเป็นไปได้)โค้ดส่วนที่เกี่ยวข้องกับปัญหา หากต้องแนบหลายไฟล์ ให้บรรจุเป็นไฟล์ ZIP  

  คุณสามารถแนบเอกสารของคุณใน Aspose.Forums ได้อย่างปลอดภัย เนื่องจากมีเพียงคุณและนักพัฒนา Aspose เท่านั้นที่สามารถเข้าถึงไฟล์ที่แนบมา  

- สร้างและใช้เธรดหนึ่งสำหรับปัญหาแต่ละรายการ หากคุณเจอปัญหาอื่น คุณต้องสร้างเธรดใหม่เพื่อรายงาน
