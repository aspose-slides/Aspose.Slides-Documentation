## **ประเมิน Aspose.Slides**
คุณสามารถดาวน์โหลด Aspose.Slides เพื่อการประเมินได้อย่างง่ายดาย แพ็กเกจการประเมินจะเหมือนกับแพ็กเกจที่ซื้อแล้ว เวอร์ชันประเมินจะกลายเป็นลิขสิทธิ์เมื่อคุณเพิ่มบรรทัดโค้ดบางส่วนเพื่อใช้ใบอนุญาต

## **ข้อจำกัดของเวอร์ชันประเมิน**
เวอร์ชันประเมินของ Aspose.Slides (โดยไม่มีการระบุใบอนุญาต) มีฟังก์ชันการทำงานทั้งหมดของผลิตภัณฑ์ แต่จะใส่น้ำหนักประเมินที่ด้านบนของเอกสารเมื่อเปิดหรือบันทึก คุณยังถูกจำกัดให้ดึงข้อความจากสไลด์เพียงสไลด์เดียวเท่านั้น

หากคุณต้องการทดสอบ Aspose.Slides โดยไม่จำกัดของเวอร์ชันประเมิน คุณสามารถขอ **ใบอนุญาตชั่วคราว 30 วัน** ได้ โปรดดูที่ [วิธีรับใบอนุญาตชั่วคราว?](https://purchase.aspose.com/temporary-license) สำหรับข้อมูลเพิ่มเติม

## **เกี่ยวกับใบอนุญาต**
คุณสามารถดาวน์โหลดเวอร์ชันประเมินของ Aspose.Slides สำหรับ Node.js ผ่าน .NET ได้จาก [หน้าโหลดดาวน์โหลด](https://releases.aspose.com/slides/th/nodejs-net/) เวอร์ชันประเมินให้ **ความสามารถเดียวกัน** อย่างสมบูรณ์แบบกับเวอร์ชันที่มีลิขสิทธิ์ของ Aspose.Slides นอกจากนี้เวอร์ชันประเมินจะกลายเป็นลิขสิทธิ์เมื่อคุณซื้อใบอนุญาตและเพิ่มบรรทัดโค้ดสองสามบรรทัดเพื่อใช้ใบอนุญาต

ใบอนุญาตเป็นไฟล์ XML แบบข้อความธรรมดาที่มีรายละเอียดเช่น ชื่อผลิตภัณฑ์ จำนวนผู้พัฒนาที่ได้รับสิทธิ์ วันที่หมดอายุการสมัครสมาชิก ฯลฯ ไฟล์นี้ถูกลงนามอย่างดิจิทัล ดังนั้นห้ามแก้ไขไฟล์ แม้แต่การเพิ่มบรรทัดว่างโดยไม่ตั้งใจก็จะทำให้ไฟล์ใช้งานไม่ได้

เพื่อหลีกเลี่ยงข้อจำกัดที่เกี่ยวข้องกับเวอร์ชันประเมิน คุณต้องตั้งค่าใบอนุญาตก่อนใช้ **Aspose.Slides** คุณต้องตั้งค่าใบอนุญาตเพียงครั้งเดียวต่อแอปพลิเคชันหรือกระบวนการ

### **การตั้งค่าใบอนุญาตใน Aspose.Slides สำหรับ Node.js ผ่าน .NET**

ใบอนุญาตสามารถนำไปใช้ได้จากตำแหน่งเหล่านี้:

* เส้นทางที่ระบุอย่างชัดเจน
* สตรีม
* เป็น Metered License – กลไกการให้ใบอนุญาตรูปแบบใหม่

{{% alert color="primary" %}}
ใช้เมธอด **setLicense** เพื่อให้ใบอนุญาตกับคอมโพเนนต์
แม้ว่าการเรียกเมธอด **setLicense** หลายครั้งจะไม่ทำให้เกิดอันตราย แต่ก็เป็นการเสียทรัพยากร (โปรเซสเซอร์) อย่างไม่จำเป็น
{{% /alert %}}

{{% alert color="warning" %}}
ใบอนุญาตใหม่สามารถใช้งาน Aspose.Slides ได้เฉพาะกับเวอร์ชัน 21.4 ขึ้นไป เวอร์ชันก่อนหน้านั้นใช้ระบบการให้ใบอนุญาตที่แตกต่างและจะไม่รับรู้ใบอนุญาตเหล่านี้
{{% /alert %}}

#### **การใช้ใบอนุญาตผ่านไฟล์**

โค้ดสแนปชอตนี้ใช้เพื่อกำหนดไฟล์ใบอนุญาต:

**Node.js**

```javascript
// นำเข้าโมดูล Aspose.Slides เพื่อจัดการไฟล์ PowerPoint
const asposeSlides = require('aspose.slides.via.net');

// ฟังก์ชันนี้ตั้งค่าไลบรารี Aspose.Slides ด้วยใบอนุญาต
function setupAsposeSlidesLicense() {
	
    // เริ่มต้นคลาส License จากโมดูล Aspose.Slides
    var license = new asposeSlides.License();
    
    // ใช้ใบอนุญาตจากไฟล์
    // แทนที่ "your_license_file.lic" ด้วยเส้นทางไปยังไฟล์ใบอนุญาตจริงของคุณ
    license.setLicense("your_license_file.lic");
}

// เรียกใช้ฟังก์ชันเพื่อตั้งค่าใบอนุญาตสำหรับ Aspose.Slides
setupAsposeSlidesLicense();
```
{{% alert color="primary" %}}
เมื่อเรียกเมธอด setLicense ชื่อใบอนุญาตต้องตรงกับชื่อไฟล์ใบอนุญาตของคุณ ตัวอย่างเช่น คุณสามารถเปลี่ยนชื่อไฟล์ใบอนุญาตเป็น "Aspose.Slides.lic.xml" แล้วในโค้ดของคุณต้องส่งชื่อใบอนุญาตใหม่ (Aspose.Slides.lic.xml) ไปยังเมธอด setLicense
{{% /alert %}}
