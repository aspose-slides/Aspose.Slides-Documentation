## **แพลตฟอร์มที่สนับสนุน**
Aspose.Slides for Node.js via .NET เป็น API ที่ไม่ขึ้นกับแพลตฟอร์ม สามารถใช้ได้บนแพลตฟอร์มใดก็ได้ (Windows, Linux, MacOS, เป็นต้น) ที่ติดตั้ง Node.js เวอร์ชัน 7 หรือสูงกว่า Oracle JDK 8 หรือสูงกว่าเป็นข้อกำหนดเพิ่มเติม

## **รูปแบบไฟล์และการแปลง**
Aspose.Slides for Node.js via .NET รองรับรูปแบบเอกสาร PowerPoint ส่วนใหญ่ นอกจากนี้ยังทำให้คุณสามารถส่งออกเป็นรูปแบบไฟล์ที่เป็นที่นิยมที่องค์กรต่าง ๆ ใช้และแชร์กันได้ โปรดดูรายละเอียดต่อไปนี้:

|**คุณลักษณะ**|**คำอธิบาย**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via .NET ให้การประมวลผลที่เร็วที่สุดสำหรับรูปแบบเอกสารพรีเซนเทชันนี้|
|การแปลง PPT เป็น PPTX|Aspose.Slides for Node.js via .NET รองรับการแปลง PPT เป็น PPTX|
|Portable Document Format (PDF)|คุณสามารถส่งออกไฟล์ที่รองรับทั้งหมดเป็นเอกสาร Adobe Portable Document Format (PDF) ด้วยวิธีเดียว|
|XML Parser Specification (XPS)|คุณสามารถส่งออกไฟล์ที่รองรับทั้งหมดเป็นเอกสาร XML Parser Specification (XPS) ด้วยวิธีเดียว|
|Tagged Image File Format (TIFF)|คุณสามารถส่งออกไฟล์พรีเซนเทชันที่รองรับทั้งหมดเป็น Tagged Image File Format (TIFF)|
|การแปลง PPTX เป็น HTML|Aspose.Slides for Node.js via .NET รองรับการแปลง PresentationEx ไปเป็นรูปแบบ HTML|

## **การเรนเดอร์และการพิมพ์**
Aspose.Slides for Node.js via .NET รองรับการเรนเดอร์สไลด์ในพรีเซนเทชันด้วยความแม่นยำสูงไปยังรูปแบบกราฟิกต่าง ๆ โปรดดูรายละเอียดต่อไปนี้:

|**คุณลักษณะ**|**คำอธิบาย**|
| :- | :- |
|.NET Supported Image Formats|ใช้ Aspose.Slides for Node.js via .NET คุณสามารถเรนเดอร์สไลด์และรูปภาพบนสไลด์ไปยังรูปแบบกราฟิกที่ .NET รองรับทั้งหมด เช่น TIFF, PNG, BMP, JPEG, GIF|
|SVG Format|Aspose.Slides for Node.js via .NET ยังมีเมธอดในตัวที่ช่วยให้คุณส่งออกสไลด์พรีเซนเทชันเป็นรูปแบบ Scalable Vector Graphics (SVG)|
|Presentation Printing|เวอร์ชันล่าสุดของ Aspose.Slides for Node.js via .NET มีเมธอดพิมพ์ในตัวพร้อมตัวเลือกต่าง ๆ|

## **คุณสมบัติของเนื้อหา**
Aspose.Slides for Node.js via .NET ให้คุณเข้าถึง แก้ไข หรือสร้างรายการหรือเนื้อหาเกือบทั้งหมดของเอกสารพรีเซนเทชัน โปรดดูรายละเอียดต่อไปนี้:

|**คุณลักษณะ**|**คำอธิบาย**|
| :- | :- |
|Master Slides|Master Slides กำหนดเค้าโครงของสไลด์ปกติ Aspose.Slides for Node.js via .NET ให้คุณเข้าถึงและแก้ไข Master Slides ของเอกสารพรีเซนเทชัน|
|Normal Slides|ด้วย Aspose.Slides for Node.js via .NET คุณสามารถสร้างสไลด์ใหม่หลายประเภทได้; อีกทั้งยังสามารถเข้าถึงและแก้ไขสไลด์ที่มีอยู่ในพรีเซนเทชัน|
|Cloning / Copying Slides|มีเมธอดในตัวจาก Aspose.Slides for Node.js via .NET ที่ช่วยให้คุณโคลนหรือคัดลอกสไลด์ที่มีอยู่ภายในพรีเซนเทชัน คุณยังสามารถใช้สไลด์ที่คัดลอกหรือโคลนจากพรีเซนเทชันหนึ่งไปยังอีกพรีเซนเทชันหนึ่ง เนื่องจากสไลด์สืบทอดเค้าโครงจาก master slide เมธอดโคลนในตัวจึงคัดลอก master โดยอัตโนมัติ|
|Managing Slides sections|เมธอดสำหรับจัดระเบียบสไลด์ในส่วนต่าง ๆ ภายในพรีเซนเทชัน|
|Place Holders and Text Holders|คุณสามารถเข้าถึง place holders และ text holders ในสไลด์ นอกจากนี้คุณยังสามารถสร้างสไลด์พร้อม text holders ตั้งแต่ต้นด้วยเมธอดที่เหมาะสม|
|Header and Footers|Aspose.Slides for Node.js via .NET ช่วยทำงานกับส่วนหัว/ส่วนท้ายในสไลด์|
|Notes in Slides|ด้วย Aspose.Slides for Node.js via .NET คุณสามารถเข้าถึงและแก้ไขโน้ตที่สัมพันธ์กับสไลด์และเพิ่มโน้ตใหม่ได้|
|Finding a Shape|คุณสามารถค้นหา shape เฉพาะจากสไลด์โดยใช้ข้อความแทนที่ (alternative text) ที่สัมพันธ์กับ shape นั้น|
|Backgrounds|Aspose.Slides for Node.js via .NET ให้คุณทำงานกับพื้นหลังที่สัมพันธ์กับ master หรือสไลด์ปกติในพรีเซนเทชัน|
|Text Boxes|สามารถสร้าง Text Box ตั้งแต่ต้นได้และเข้าถึง Text Box ที่มีอยู่แล้ว คุณยังสามารถแก้ไขข้อความโดยไม่เสียรูปแบบเดิม|
|Rectangle Shapes|คุณสามารถสร้างหรือแก้ไขรูปสี่เหลี่ยมโดยใช้ Aspose.Slides for Node.js via .NET|
|Poly Line Shapes|คุณสามารถสร้างหรือแก้ไขรูป poly line โดยใช้ Aspose.Slides for Node.js via .NET|
|Ellipse Shapes|คุณสามารถสร้างหรือแก้ไขรูป Ellipse โดยใช้ Aspose.Slides for Node.js via .NET|
|Group Shapes|Aspose.Slides for Node.js via .NET รองรับ group shapes|
|Auto Shapes|Aspose.Slides for Node.js via .NET รองรับ auto shapes|
|SmartArt|Aspose.Slides for Node.js via .NET ให้การสนับสนุนรูปแบบ SmartArt ใน MS PowerPoint|
|Charts|Aspose.Slides for Node.js via .NET ให้การสนับสนุน Charts ของ MSO ใน PowerPoint|
|Shapes Serialization|Aspose.Slides for Node.js via .NET รองรับรูปแบบ shape มากมาย เมื่อ Aspose.Slides for Node.js via .NET ขาดการสนับสนุน shape ใด shape หนึ่ง คุณสามารถใช้วิธีการ serialization เพื่อทำการแปลง shape นั้นจากสไลด์ที่มีอยู่แล้ว แล้วนำ shape ไปใช้ต่อตามความต้องการของคุณ|
|Picture Frames|คุณสามารถจัดการรูปภาพใน picture frames ด้วย Aspose.Slides for Node.js via .NET|
|Audio Frames|คุณสามารถลิงก์หรือฝังไฟล์เสียงใน audio frames บนสไลด์ด้วย Aspose.Slides for Node.js via .NET|
|Video Frames|คุณสามารถจัดการไฟล์วิดีโอใน video frames ได้ Aspose.Slides for Node.js via .NET ยังรองรับวิดีโอที่ลิงก์และฝัง|
|OLE Frame|คุณสามารถจัดการ OLE Objects ใน OLE frames ด้วย Aspose.Slides for Node.js via .NET|
|Tables|Aspose.Slides for Node.js via .NET รองรับตารางในสไลด์|
|ActiveX Controls|รองรับ ActiveX controls|
|VBA Macros|รองรับการจัดการ VBA macros ภายในพรีเซนเทชัน|
|Text Frame|คุณสามารถเข้าถึงข้อความของ shape ใดก็ได้ผ่าน text frame ที่สัมพันธ์กับ shape นั้น|
|Text Scanning|คุณสามารถสแกนข้อความในพรีเซนเทชันระดับพรีเซนเทชันหรือสไลด์โดยใช้เมธอดสแกนในตัว|
|Animations|คุณสามารถใช้ animation กับ shape ต่าง ๆ|
