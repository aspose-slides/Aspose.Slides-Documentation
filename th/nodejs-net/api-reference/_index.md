Aspose.Slides for Node.js via .NET คือไลบรารีคลาสที่ช่วยให้แอปพลิเคชัน Node.js สามารถอ่านและเขียนเอกสารพรีเซนเทชั่นได้โดยไม่ต้องใช้ Microsoft PowerPoint® หรือโปรแกรมอื่นๆ Aspose.Slides for Node.js via .NET อนุญาตให้คุณจัดการข้อความ รูปร่าง ตารางและแอนิเมชัน เพิ่มเสียงและวิดีโอลงในสไลด์ ดูตัวอย่างสไลด์ ส่งออกสไลด์เป็นรูปแบบ SVG, PDF และทำได้มากกว่านั้นอีกหลายอย่าง

Aspose.Slides for Node.js via .NET ใช้ [อ้างอิง API เดียวกันกับ Aspose.Slides .NET](https://reference.aspose.com/slides/th/nodejs-net/).
