**ยินดีต้อนรับสู่ Aspose.Slides for Node.js via .NET**

![Aspose.Slides for Node.js via .NET Product Logo](aspose_slides-for-nodejs-via-net.png)

Aspose.Slides for Node.js via .NET เป็นไลบรารีคลาสที่ช่วยให้แอปพลิเคชันของคุณสามารถอ่านและเขียนเอกสาร PowerPoint® ได้โดยไม่ต้องใช้ Microsoft PowerPoint®.

Aspose.Slides for Node.js via .NET เป็นส่วนประกอบแรกและส่วนเดียวที่ให้ฟังก์ชันในการจัดการเอกสาร PowerPoint®.

Aspose.Slides for Node.js via .NET มีคุณสมบัติสำคัญหลายอย่าง เช่น การจัดการข้อความ, รูปร่าง, ตารางและแอนิเมชัน, การเพิ่มเสียงและวิดีโอไปยังสไลด์, การแสดงตัวอย่างสไลด์, การส่งออกสไลด์เป็นรูปแบบ SVG, PDF และอื่น ๆ.
