Aspose.Slides for Node.js via Java เป็นไลบรารีคลาสที่ช่วยให้แอปพลิเคชัน Node.js สามารถอ่านและเขียนเอกสารพรีเซนเทชันโดยไม่ต้องใช้ Microsoft PowerPoint® หรือโปรแกรมอื่น ๆ Aspose.Slides for Node.js via Java ช่วยให้คุณจัดการข้อความ รูปร่าง ตารางและแอนิเมชัน เพิ่มไฟล์เสียงและวิดีโอบนสไลด์ ดูตัวอย่างสไลด์ ส่งออกสไลด์เป็นรูปแบบ SVG PDF และทำได้อีกมากมาย

อ้างอิง API ล่าสุดสามารถพบได้ที่ [อ้างอิง API ของ Aspose.Slides for Node.js via Java](https://reference.aspose.com/slides/th/nodejs-java/).
