## **ประเมิน Aspose.Slides**
คุณสามารถดาวน์โหลด Aspose.Slides เพื่อการประเมินได้อย่างง่ายดาย แพคเกจการประเมินเหมือนกับแพคเกจที่ซื้อ เวอร์ชันการประเมินจะกลายเป็นมีใบอนุญาตหลังจากคุณเพิ่มโค้ดไม่กี่บรรทัดเพื่อใช้ใบอนุญาต.

## **ข้อจำกัดของเวอร์ชันการประเมิน**
เวอร์ชันการประเมินของ Aspose.Slides (โดยไม่ได้ระบุใบอนุญาต) ให้ฟังก์ชันการทำงานของผลิตภัณฑ์เต็มรูปแบบ แต่จะใส่ลายน้ำการประเมินที่ด้านบนของเอกสารเมื่อเปิดและบันทึก คุณจะถูกจำกัดให้ใช้งานเพียงสไลด์เดียวเมื่อสกัดข้อความจากสไลด์การนำเสนอ.

{{% alert color="primary" %}} 
หากคุณต้องการทดสอบ Aspose.Slides โดยไม่มีข้อจำกัดของเวอร์ชันการประเมิน คุณสามารถขอ **ใบอนุญาตชั่วคราว 30 วัน** กรุณาอ้างอิงที่ [วิธีรับใบอนุญาตชั่วคราว?](https://purchase.aspose.com/temporary-license) เพื่อข้อมูลเพิ่มเติม.

## **เกี่ยวกับใบอนุญาต**
คุณสามารถดาวน์โหลดเวอร์ชันการประเมินของ Aspose.Slides สำหรับ Node.js ผ่าน Java ได้อย่างง่ายดายจาก [หน้าดาวน์โหลด](https://releases.aspose.com/slides/th/nodejs-java/). เวอร์ชันการประเมินให้ **ความสามารถเดียวกัน** อย่างสมบูรณ์แบบกับเวอร์ชันที่มีใบอนุญาตของ Aspose.Slides นอกจากนี้ เวอร์ชันการประเมินจะกลายเป็นมีใบอนุญาตหลังจากที่คุณซื้อใบอนุญาตและเพิ่มโค้ดไม่กี่บรรทัดเพื่อใช้ใบอนุญาต.

ใบอนุญาตเป็นไฟล์ XML แบบข้อความธรรมดาซึ่งมีรายละเอียดเช่น ชื่อผลิตภัณฑ์ จำนวนผู้พัฒนาที่ได้รับใบอนุญาต วันที่หมดอายุการสมัครสมาชิก เป็นต้น ไฟล์นี้มีลายเซ็นดิจิทัล ดังนั้นห้ามแก้ไขไฟล์ แม้แต่การเพิ่มบรรทัดว่างโดยบังเอิญในเนื้อหาไฟล์ก็จะทำให้ไฟล์ไม่ถูกต้อง.

เพื่อหลีกเลี่ยงข้อจำกัดที่เกี่ยวข้องกับเวอร์ชันการประเมิน คุณต้องตั้งค่าใบอนุญาตก่อนใช้ **Aspose.Slides** คุณต้องตั้งค่าใบอนุญาตเพียงครั้งเดียวต่อแอปพลิเคชันหรือกระบวนการ.

### **การตั้งค่าใบอนุญาตใน Aspose.Slides สำหรับ Node.js ผ่าน Java**
สามารถใช้ใบอนุญาตจากตำแหน่งต่อไปนี้:
* เส้นทางโดยตรง
* สตรีม
* เป็นใบอนุญาตแบบจ่ายตามการใช้งาน – กลไกการให้ใบอนุญาตใหม่

{{% alert color="primary" %}}
ใช้เมธอด **setLicense** เพื่อให้ใบอนุญาตกับคอมโพเนนต์

แม้การเรียกเมธอด **setLicense** หลายครั้งจะไม่เป็นอันตราย แต่จะเป็นการสิ้นเปลืองทรัพยากร (โปรเซสเซอร์).
{{% /alert %}}

{{% alert color="warning" %}}
ใบอนุญาตใหม่สามารถเปิดใช้งาน Aspose.Slides ได้เฉพาะกับเวอร์ชัน 21.4 หรือหลังจากนั้น รุ่นก่อนใช้ระบบใบอนุญาตที่แตกต่างและจะไม่รับรู้ใบอนุญาตเหล่านี้.
{{% /alert %}}

#### **การใช้ใบอนุญาตด้วยไฟล์**
โค้ดสแนปนี้ใช้เพื่อกำหนดไฟล์ใบอนุญาต:

**Node.js**

```javascript
var aspose = aspose || {};

aspose.slides = require("aspose.slides.via.java");

var license = new aspose.slides.License();
license.setLicense("Aspose.Slides.lic");
```

เมื่อเรียกเมธอด setLicense ชื่อใบอนุญาตควรเหมือนกับชื่อไฟล์ใบอนุญาตของคุณ ตัวอย่างเช่น คุณสามารถเปลี่ยนชื่อไฟล์ใบอนุญาตเป็น "Aspose.Slides.lic.xml" จากนั้นในโค้ดของคุณต้องส่งชื่อใบอนุญาตใหม่ (Aspose.Slides.lic.xml) ไปยังเมธอด setLicense.

#### **การใช้ใบอนุญาตจากสตรีม**
โค้ดสแนปนี้ใช้เพื่อใช้ใบอนุญาตจากสตรีม:

**Node.js**

```javascript
var aspose = aspose || {};

aspose.slides = require("aspose.slides.via.java");

var license = new aspose.slides.License();

var fs = require("fs");

var readStream = fs.createReadStream("Aspose.Slides.lic");

license.setLicense(readStream, function(err, list) {
    if(err) { 
        console.error(err); return; 
    }});
```

## **คำถามที่พบบ่อย**

**ฉันสามารถใช้ใบอนุญาตในสภาพแวดล้อมที่ออฟไลน์อย่างสมบูรณ์ (ไม่มีการเชื่อมต่ออินเทอร์เน็ต) ได้หรือไม่?**  
ใช่ การตรวจสอบความถูกต้องของใบอนุญาตทำในเครื่องโดยใช้ไฟล์ใบอนุญาต; ไม่ต้องการการเชื่อมต่ออินเทอร์เน็ต

**จะเกิดอะไรขึ้นหลังจากการสมัครสมาชิกหนึ่งปีหมดอายุ? ไลบรารีจะหยุดทำงานหรือไม่?**  
ไม่ ใบอนุญาตเป็นแบบถาวร: คุณสามารถใช้เวอร์ชันที่ออกก่อนวันที่สิ้นสุดการสมัครของคุณต่อไปได้; แต่คุณจะไม่สามารถใช้เวอร์ชันใหม่กว่าได้หากไม่ได้ต่ออายุ.
