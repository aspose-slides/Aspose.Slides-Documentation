## **แพลตฟอร์มที่รองรับ**
Aspose.Slides for Node.js via Java คือ API ที่ไม่ขึ้นกับแพลตฟอร์ม สามารถใช้ได้บนแพลตฟอร์มใดก็ได้ (Windows, Linux, macOS ฯลฯ) ที่ติดตั้ง Node.js เวอร์ชัน 7 หรือสูงกว่า ส่วน Oracle JDK 8 หรือสูงกว่าเป็นความต้องการเพิ่มเติม

## **รูปแบบไฟล์และการแปลง**
Aspose.Slides for Node.js via Java รองรับรูปแบบไฟล์เอกสาร PowerPoint ส่วนใหญ่ นอกจากนี้ยังสามารถส่งออกเป็นรูปแบบไฟล์ที่องค์กรนิยมใช้และแชร์กันได้ ดูรายละเอียดต่อไปนี้:

|**คุณลักษณะ**|**คำอธิบาย**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via Java ให้การประมวลผลที่เร็วที่สุดสำหรับรูปแบบเอกสารการนำเสนอนี้.|
|PPT to PPTX conversion|Aspose.Slides for Node.js via Java รองรับการแปลงจาก PPT ไปยัง PPTX|
|Portable Document Format (PDF)|คุณสามารถส่งออกไฟล์รูปแบบที่รองรับทั้งหมดเป็นเอกสาร Adobe Portable Document Format (PDF) ด้วยวิธีเดียว|
|XML Parser Specification (XPS)|คุณสามารถส่งออกไฟล์รูปแบบที่รองรับทั้งหมดเป็นเอกสาร XML Parser Specification (XPS) ด้วยวิธีเดียว|
|Tagged Image File Format (TIFF)|คุณสามารถส่งออกไฟล์รูปแบบการนำเสนอที่รองรับทั้งหมดเป็น Tagged Image File Format (TIFF)|
|PPTX To HTML Conversion|Aspose.Slides for Node.js via Java รองรับการแปลง PresentationEx เป็นรูปแบบ HTML|

## **การแสดงผลและการพิมพ์**
Aspose.Slides for Node.js via Java รองรับการเรนเดอร์สไลด์ในงานนำเสนอด้วยความละเอียดสูงเป็นรูปแบบกราฟิกต่าง ๆ ดูรายละเอียดต่อไปนี้:

|**คุณลักษณะ**|**คำอธิบาย**|
| :- | :- |
|Java Supported Image Formats|โดยใช้ Aspose.Slides for Node.js via Java คุณสามารถเรนเดอร์สไลด์และภาพบนสไลด์เป็นรูปแบบกราฟิกที่ Java รองรับทั้งหมด เช่น TIFF, PNG, BMP, JPEG, GIF.|
|SVG Format|Aspose.Slides for Node.js via Java ยังมีเมธอดในตัวที่ให้คุณส่งออกสไลด์การนำเสนอเป็นรูปแบบ Scalable Vector Graphics (SVG).|
|Presentation Printing|เวอร์ชันล่าสุดของ Aspose.Slides for Node.js via Java มีเมธอดพิมพ์ในตัวพร้อมตัวเลือกต่าง ๆ|

## **คำถามที่พบบ่อย**

**Do I need to install Microsoft PowerPoint on the server/PC for the library to work?**

ไม่. PowerPoint ไม่จำเป็น; Aspose.Slides เป็นเอนจินแบบสแตนด์อโลนสำหรับสร้าง, แก้ไข, แปลง และแสดงผลการนำเสนอ

**How does multithreading work? Can processing be parallelized?**

ปลอดภัยที่จะประมวลผลเอกสารต่าง ๆ ในเธรดต่าง ๆ; วัตถุ [presentation](https://reference.aspose.com/slides/th/nodejs-java/aspose.slides/presentation/) เดียวกันไม่ควรถูกใช้โดย [multiple threads](/slides/th/nodejs-java/multithreading/) พร้อมกัน

**Are file passwords and encryption supported?**

ใช่. [You can](/slides/th/nodejs-java/password-protected-presentation/) เปิดการนำเสนอที่เข้ารหัส, ตั้งหรือเอารหัสผ่านสำหรับการเปิดและการเขียนออก, และตรวจสอบสถานะการป้องกัน

**Do I need to care about font packages in Linux containers?**

ใช่. แนะนำให้ติดตั้งฟอนต์แพคเกจทั่วไปและ/หรือระบุไดเรกทอรีฟอนต์อย่างชัดเจน [specify font directories](/slides/th/nodejs-java/custom-font/) ในแอปพลิเคชันของคุณเพื่อหลีกเลี่ยงการแทนที่ที่ไม่คาดคิด

**Are there limitations in the evaluation version?**

ใน [evaluation mode](/slides/th/nodejs-java/licensing/) จะมีการเพิ่มลายน้ำลงในผลลัพธ์และมีข้อจำกัดบางประการ; มี [30-day temporary license](https://purchase.aspose.com/temporary-license/) ให้ใช้สำหรับการทดสอบฟีเจอร์ครบถ้วน

**Is importing external formats into a presentation (PDF/HTML → PPTX) supported?**

ใช่. คุณสามารถเพิ่ม [PDF pages and HTML content](/slides/th/nodejs-java/import-presentation/) ไปยังงานนำเสนอ ทำให้เป็นสไลด์ได้
