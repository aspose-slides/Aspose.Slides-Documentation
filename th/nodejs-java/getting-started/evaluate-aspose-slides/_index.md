## **การประเมิน Aspose.Slides**

คุณสามารถดาวน์โหลด Aspose.Slides เพื่อการประเมินได้อย่างง่ายดาย แพ็คเกจการประเมินเหมือนกับแพ็คเกจที่ซื้อ เวอร์ชันการประเมินจะกลายเป็นลิขสิทธิ์ได้โดยการเพิ่มบรรทัดโค้ดบางบรรทัดเพื่อใช้ไลเซนส์  

เวอร์ชันการประเมินของ Aspose.Slides (โดยไม่มีการระบุไลเซนส์) ให้ฟังก์ชันการทำงานของผลิตภัณฑ์เต็มรูปแบบ แต่จะใส่น้ำฆ่า (watermark) ของการประเมินที่ด้านบนของเอกสารเมื่อเปิดหรือบันทึก คุณยังถูกจำกัดให้ใช้เพียงสไลด์เดียวเมื่อต้องสกัดข้อความจากสไลด์นำเสนอ

## **คำถามที่พบบ่อย**

**ฉันสามารถทดสอบการนำเสนอหลายไฟล์พร้อมกันบนหลายเธรดในโหมดประเมินได้ไหม?**

ใช่ คุณสามารถประมวลผลเอกสารต่าง ๆ ได้พร้อมกัน; ไม่ควรแชร์ออบเจกต์การนำเสนอเดียวกัน [ข้ามเธรด](/slides/th/net/multithreading/). โหมดประเมินไม่ส่งผลต่อสิ่งนี้  

**ฉันต้องติดตั้ง Microsoft PowerPoint เพื่อประเมินไลบรารีบนเซิร์ฟเวอร์หรือใน CI หรือไม่?**

ไม่ Aspose.Slides เป็นเอนจินแบบสแตนด์อโลนและไม่ต้องการ PowerPoint ติดตั้งไม่ว่าจะในโหมดประเมินหรือการผลิต  

**ฉันสามารถทดสอบการแปลง PPT/PPTX เป็น PDF และรูปภาพอย่างเต็มที่ในโหมดประเมินได้ไหม?**

ใช่ [ตัวแปลง](/slides/th/net/convert-presentation/)ทำงาน; ผลลัพธ์จะมีลายน้ำ  

**ฉันสามารถใช้ใบอนุญาตชั่วคราวสำหรับการทดสอบโหลดโดยไม่มีลายน้ำได้ไหม?**

ใช่ ใบอนุญาตชั่วคราว 30 วันจะลบข้อจำกัดของโหมดประเมินและอนุญาตให้ทำการทดสอบโดยไม่มีลายน้ำ
