## **ภาพรวม**

FAQ นี้ให้คำตอบสำหรับคำถามทั่วไปเกี่ยวกับ Aspose.Slides ครอบคลุมรูปแบบไฟล์ที่รองรับ, การจัดการข้อยกเว้นเมื่อต้องทำงานกับงานนำเสนอขนาดใหญ่, การเปลี่ยนขนาดสไลด์, การแสดงตัวอย่างสไลด์, การดึงข้อความจากงานนำเสนอ, การจัดรูปแบบเส้นขอบตาราง, การใส่รูปภาพ, และการแก้ไขปัญหาเรื่องฟอนต์เมื่อแปลงงานนำเสนอเป็น PDF หรือรูปภาพ

## **รูปแบบไฟล์ที่รองรับ**

**Q: Aspose.Slides สำหรับ Node.js ผ่าน Java รองรับรูปแบบไฟล์อะไรบ้าง?**

**A**: Aspose.Slides สำหรับ Node.js ผ่าน Java รองรับรูปแบบไฟล์ที่อธิบายไว้ใน [Supported File Formats](/slides/th/nodejs-java/supported-file-formats/).

## **ข้อยกเว้น**

**Q: ฉันได้รับข้อยกเว้น out of memory ขณะโหลดไฟล์ PPT ขนาดใหญ่ที่มีรูปภาพ มีข้อจำกัดขนาดไฟล์ใน Aspose.Slides หรือไม่?**

**A**: ไม่มีสูตรเฉพาะในการคำนวณขนาดงานนำเสนอที่ Aspose.Slides รองรับ ควรมีพื้นที่เพียงพอเพื่อเก็บโครงสร้างงานนำเสนอทั้งหมดและรูปภาพในหน่วยความจำ ปกติรูปภาพในหน่วยความจำจะใช้พื้นที่มากกว่าฮาร์ดดิสก์ โดยเฉพาะเมื่อรูปภาพมีเอฟเฟกต์เพิ่มเติม  
โดยทั่วไป Aspose.Slides สำหรับ Node.js ผ่าน Java สามารถจัดการไฟล์งานนำเสนอขนาดประมาณ 300 MB ได้อย่างง่ายดายบนเซิร์ฟเวอร์ที่มี RAM 4 GB

## **การทำงานกับสไลด์**

**Q: ฉันสามารถเปลี่ยนขนาดของสไลด์ในงานนำเสนอได้หรือไม่?**

**A**: คุณสามารถใช้เมธอด `getSlideSize` ที่เปิดให้ใช้โดยคลาส [Presentation](https://reference.aspose.com/slides/th/nodejs-java/aspose.slides/presentation/) เพื่อกำหนดขนาดของสไลด์ในงานนำเสนอได้

**Q: มีวิธีกำหนดสไลด์ที่มีขนาดต่าง ๆ ในงานนำเสนอได้หรือไม่?**

**A**: เนื่องจากขนาดของสไลด์ถูกกำหนดระดับงานนำเสนอในเอกสาร Microsoft PowerPoint จึงไม่มีวิธีทำเช่นนี้

**Q: Aspose.Slides สำหรับ Node.js ผ่าน Java รองรับการแสดงตัวอย่างสไลด์ก่อนบันทึกหรือไม่?**

**A**: คุณสามารถเรนเดอร์สไลด์ของงานนำเสนอเป็นรูปภาพและใช้รูปภาพเหล่านี้เพื่อแสดงตัวอย่างสไลด์ได้

## **การทำงานกับข้อความ**

**Q: สามารถดึงข้อความทั้งหมดจากงานนำเสนอได้หรือไม่?**

**A**: Aspose.Slides สำหรับ Node.js ผ่าน Java มีคลาส [SlideUtil](https://reference.aspose.com/slides/th/nodejs-java/aspose.slides/slideutil/) ที่ให้เมธอดต่าง ๆ สำหรับการดึงข้อความทั้งหมดจากงานนำเสนอ

**Q: ทำไมขนาดย่อหน้าจึงแตกต่างกันบนระบบปฏิบัติการ Windows และ Linux?**

**A**: การคำนวณขนาดย่อหน้าขึ้นอยู่กับการคำนวณขนาดข้อความที่แทนย่อหน้าที่กำหนด ขนาดข้อความคำนวณจากเมตริกของฟอนต์ที่ระบุในงานนำเสนอ PowerPoint หากฟอนต์ที่ระบุขาดหายไป จะถูกแทนที่ด้วยฟอนต์ที่คล้ายที่สุด แต่ฟอนต์นั้นมีเมตริกที่แตกต่างจากฟอนต์เดิม ดังนั้นการคำนวณขนาดย่อหน้าในระบบต่าง ๆ จะให้ผลลัพธ์ที่ต่างกันตามชุดฟอนต์ที่ติดตั้ง เพื่อให้ผลลัพธ์เหมือนกันบนระบบปฏิบัติการต่าง ๆ คุณต้องติดตั้งฟอนต์เดียวกันบนระบบเหล่านั้นหรือโหลดฟอนต์ในช่วงรันไทม์เป็น [external fonts](/slides/th/nodejs-java/custom-font/)

## **การจัดรูปแบบและรูปภาพ**

**Q: ฉันจะตั้งค่าสีของเส้นขอบตารางอย่างไร?**

**A**: คุณสามารถเปลี่ยนสีของเส้นขอบตารางทั้งหมดหรือเพียงเส้นขอบรอบตารางทั้งหมดได้ สำหรับการเปลี่ยนสีเส้นขอบทั้งหมด ให้ใช้เมธอด `getCellFormat` จากคลาส [Cell](https://reference.aspose.com/slides/th/nodejs-java/aspose.slides/cell/) สำหรับเส้นขอบรอบตารางทั้งหมด คุณต้องวนผ่านเซลล์และเปลี่ยนสีของเส้นขอบภายนอก

**Q: Aspose.Slides สำหรับ Node.js ผ่าน Java ใช้หน่วยวัดอะไรในการวางรูปภาพ?**

**A**: พิกัดและขนาดของรูปทรงทั้งหมดบนสไลด์วัดเป็นจุด (points) ที่ 72 dpi

## **การทำงานกับฟอนต์**

**Q: เมื่อแปลง PPT เป็น PDF หรือรูปภาพ ทำไมฟอนต์ถึงต่างกันในเอกสารผลลัพธ์?**

**A**: ปัญหานี้อาจบ่งชี้ว่าฟอนต์ที่ใช้ในงานนำเสนอขาดหายไปจากระบบปฏิบัติการที่รันโค้ด คุณควรติดตั้งฟอนต์บนระบบปฏิบัติการหรือโหลดฟอนต์เป็นฟอนต์ภายนอกด้วยคลาส [FontsLoader](https://reference.aspose.com/slides/th/nodejs-java/aspose.slides/fontsloader/) ตามตัวอย่างด้านล่าง  
```javascript
var folders = java.newArray("java.lang.String", ["path_to_a_folder_with_fonts"]));
java.callStaticMethodSync("com.aspose.slides.FontsLoader", "loadExternalFonts", folders);
```
