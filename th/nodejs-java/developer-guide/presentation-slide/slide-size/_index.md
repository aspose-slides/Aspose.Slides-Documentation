## **บทนำ**

Aspose.Slides มีเครื่องมือครบวงจรสำหรับปรับขนาดสไลด์และอัตราส่วนในงานนำเสนอ PowerPoint ซึ่งสำคัญทั้งในการพิมพ์และการแสดงบนหน้าจอ  

ขนาดสไลด์และอัตราส่วนที่นิยม:

- **มาตรฐาน (อัตราส่วน 4:3)**: เหมาะสำหรับหน้าจอและอุปกรณ์รุ่นเก่า
- **จอกว้าง (อัตราส่วน 16:9)**: แนะนำสำหรับเครื่องฉายและจอแสดงผลสมัยใหม่

ให้แน่ใจว่าขนาดสไลด์และอัตราส่วนคงที่ตลอดงานนำเสนอ เนื่องจากขนาดสไลด์และอัตราส่วนเดียวกันจะใช้กับสไลด์ทั้งหมด เพื่อผลลัพธ์ที่ดีที่สุด ควรกำหนดขนาดสไลด์ตั้งแต่เริ่มสร้างงานนำเสนอเพื่อหลีกเลี่ยงปัญหา  

{{% alert color="primary" %}}  
โดยค่าเริ่มต้น งานนำเสนอที่สร้างด้วย Aspose.Slides จะใช้อัตราส่วนมาตรฐาน 4:3  
{{% /alert %}}

## **การเปลี่ยนขนาดสไลด์ในงานนำเสนอ**

ตัวอย่างโค้ดนี้แสดงวิธีการเปลี่ยนขนาดสไลด์ในงานนำเสนอด้วย JavaScript โดยใช้ Aspose.Slides:

```javascript
var pres = new aspose.slides.Presentation("pres-4x3-aspect-ratio.pptx");
try {
    pres.getSlideSize().setSize(aspose.slides.SlideSizeType.OnScreen16x9, aspose.slides.SlideSizeScaleType.DoNotScale);
    pres.save("pres-4x3-aspect-ratio.pptx", aspose.slides.SaveFormat.Pptx);
} finally {
    if (pres != null) {
        pres.dispose();
    }
}
```

## **การกำหนดขนาดสไลด์แบบกำหนดเองในงานนำเสนอ**

หากขนาดสไลด์ทั่วไป (4:3 และ 16:9) ไม่เหมาะกับงานของคุณ คุณอาจต้องการใช้ขนาดสไลด์ที่กำหนดเองหรือเฉพาะเจาะจง ตัวอย่างเช่น หากคุณต้องการพิมพ์สไลด์ขนาดเต็มจากงานนำเสนอบนเค้าโครงหน้ากระดาษแบบกำหนดเอง หรือแสดงงานนำเสนอบนอุปกรณ์บางประเภท การกำหนดขนาดสไลด์ที่กำหนดเองจะช่วยให้คุณได้ผลลัพธ์ที่ต้องการ  

ตัวอย่างโค้ดนี้แสดงวิธีการใช้ Aspose.Slides for Node.js ผ่าน Java เพื่อกำหนดขนาดสไลด์แบบกำหนดเองสำหรับงานนำเสนอใน JavaScript:

```javascript
var pres = new aspose.slides.Presentation("pres.pptx");
try {
    pres.getSlideSize().setSize(780, 540, aspose.slides.SlideSizeScaleType.DoNotScale);// ขนาดกระดาษ A4
    pres.save("pres-a4-slide-size.pptx", aspose.slides.SaveFormat.Pptx);
} finally {
    if (pres != null) {
        pres.dispose();
    }
}
```

## **การจัดการปัญหาเมื่อเปลี่ยนขนาดสไลด์ในงานนำเสนอ**

หลังจากที่คุณเปลี่ยนขนาดสไลด์ของงานนำเสนอแล้ว เนื้อหาในสไลด์ (เช่น ภาพหรือวัตถุต่าง ๆ) อาจบิดเบี้ยวโดยอัตโนมัติ ตามค่าเริ่มต้น วัตถุจะถูกปรับขนาดอัตโนมัติเพื่อให้พอดีกับขนาดสไลด์ใหม่ อย่างไรก็ตาม เมื่อเปลี่ยนขนาดสไลด์ของงานนำเสนอ คุณสามารถระบุการตั้งค่าที่กำหนดว่า Aspose.Slides จะจัดการกับเนื้อหาในสไลด์อย่างไร  

ขึ้นอยู่กับวัตถุประสงค์ของคุณ คุณสามารถใช้การตั้งค่าใดก็ได้ต่อไปนี้:

- `DoNotScale`  
  หากคุณไม่ต้องการให้วัตถุบนสไลด์ถูกปรับขนาด ให้ใช้การตั้งค่านี้  

- `EnsureFit`  
  หากคุณต้องการปรับขนาดให้เล็กลงและต้องการให้ Aspose.Slides ปรับขนาดวัตถุบนสไลด์ให้พอดีกับสไลด์ (เพื่อไม่ให้สูญเสียเนื้อหา) ให้ใช้การตั้งค่านี้  

- `Maximize`  
  หากคุณต้องการปรับขนาดให้ใหญ่ขึ้นและต้องการให้ Aspose.Slides ขยายวัตถุบนสไลด์ให้สัดส่วนกับขนาดสไลด์ใหม่ ให้ใช้การตั้งค่านี้  

ตัวอย่างโค้ดนี้แสดงวิธีการใช้การตั้งค่า `Maximize` เมื่อเปลี่ยนขนาดสไลด์ของงานนำเสนอ:

```javascript
var pres = new aspose.slides.Presentation("pres.pptx");
try {
    pres.getSlideSize().setSize(aspose.slides.SlideSizeType.Ledger, aspose.slides.SlideSizeScaleType.Maximize);
} finally {
    if (pres != null) {
        pres.dispose();
    }
}
```

## **คำถามที่พบบ่อย**

**ฉันสามารถกำหนดขนาดสไลด์แบบกำหนดเองโดยใช้หน่วยอื่นนอกจากนิ้ว (เช่น พอยต์หรือมิลลิเมตร) ได้หรือไม่?**  

ใช่ Aspose.Slides ใช้หน่วยพอยต์เป็นหน่วยภายใน โดย 1 พอยต์เท่ากับ 1/72 นิ้ว คุณสามารถแปลงหน่วยใดก็ได้ (เช่น มิลลิเมตรหรือเซนติเมตร) เป็นพอยต์และใช้ค่าที่แปลงแล้วเพื่อกำหนดความกว้างและความสูงของสไลด์  

**ขนาดสไลด์แบบกำหนดเองที่ใหญ่มากจะส่งผลต่อประสิทธิภาพและการใช้หน่วยความจำระหว่างการเรนเดอร์หรือไม่?**  

ใช่ ขนาดสไลด์ที่ใหญ่ขึ้น (เป็นพอยต์) ร่วมกับสเกลการเรนเดอร์ที่สูงทำให้การใช้หน่วยความจำเพิ่มขึ้นและเวลาประมวลผลยาวนานขึ้น ควรกำหนดขนาดสไลด์ให้เหมาะสมและปรับสเกลการเรนเดอร์เฉพาะเมื่อจำเป็นเพื่อให้ได้คุณภาพผลลัพธ์ตามที่ต้องการ  

**ฉันสามารถกำหนดขนาดสไลด์ที่ไม่เป็นมาตรฐานหนึ่งค่าและจากนั้นรวมสไลด์จากงานนำเสนอที่มีขนาดแตกต่างกันได้หรือไม่?**  

คุณไม่สามารถ [รวมงานนำเสนอ](/slides/th/nodejs-java/merge-presentation/) ได้เมื่อขนาดสไลด์ต่างกัน — ก่อนอื่นให้ปรับขนาดงานนำเสนอหนึ่งให้ตรงกับอีกงานหนึ่ง เมื่อเปลี่ยนขนาดสไลด์ คุณสามารถเลือกวิธีจัดการกับเนื้อหาที่มีอยู่ผ่านตัวเลือก [SlideSizeScaleType](https://reference.aspose.com/slides/th/nodejs-java/aspose.slides/slidesizescaletype/) หลังจากปรับขนาดให้ตรงกันแล้ว คุณสามารถรวมสไลด์ได้โดยคงรูปแบบเดิม  

**ฉันสามารถสร้างภาพย่อสำหรับรูปทรงแต่ละอันหรือบริเวณเฉพาะของสไลด์ และภาพย่อเหล่านั้นจะเคารพขนาดสไลด์ใหม่หรือไม่?**  

ใช่ Aspose.Slides สามารถสร้างภาพย่อสำหรับ [สไลด์ทั้งหมด](https://reference.aspose.com/slides/th/nodejs-java/aspose.slides/slide/#getImage) รวมถึง [รูปทรงที่เลือก](https://reference.aspose.com/slides/th/nodejs-java/aspose.slides/shape/#getImage) ด้วย ภาพที่ได้จะสะท้อนขนาดสไลด์และอัตราส่วนปัจจุบัน ทำให้กรอบและเรขาคณิตคงที่
