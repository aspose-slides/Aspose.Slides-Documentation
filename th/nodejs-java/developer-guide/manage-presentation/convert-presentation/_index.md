## **ภาพรวม**

Aspose.Slides for Node.js via Java สามารถโหลดงานนำเสนอ PowerPoint และ OpenDocument และบันทึกหรือเรนเดอร์เป็นหลายรูปแบบอื่นได้โดยไม่ต้องใช้ Microsoft PowerPoint, OpenOffice หรือ LibreOffice. คุณสามารถแปลงไฟล์ PPT เก่าเป็น PPTX สมัยใหม่, ส่งออกงานนำเสนอเป็นเอกสารแบบจัดรูปแบบคงที่เช่น PDF และ XPS, เผยแพร่สไลด์เป็น HTML, หรือเรนเดอร์สไลด์เป็นไฟล์รูปภาพสำหรับตัวอย่างภาพ, ภาพย่อย, และการเก็บถาวร.

## **การแปลง PowerPoint และ OpenDocument**

Aspose.Slides for Node.js via Java รองรับการแปลงจากรูปแบบการนำเสนอที่ใช้บ่อยเช่น PPT, PPTX, PPS, PPSX, POT, POTX และ ODP. API การแปลงเดียวกันใช้ได้กับไฟล์ PowerPoint และ OpenDocument, ดังนั้นกระบวนการที่บันทึกไฟล์ PPTX เป็น PDF มักจะใช้กับไฟล์ ODP ได้โดยเปลี่ยนเพียงไฟล์อินพุต.

เมื่อแปลงไฟล์ ODP, จำไว้ว่าแอปพลิเคชัน PowerPoint และ OpenDocument ไม่รองรับเลย์เอาต์และคุณลักษณะการจัดรูปแบบทุกอย่างอย่างเท่าเทียมกัน. หากไฟล์ ODP ถูกสร้างใน LibreOffice หรือ OpenOffice Impress, ตรวจสอบผลลัพธ์และใช้ตัวเลือกที่อธิบายใน [แปลงงานนำเสนอ OpenDocument](/slides/th/nodejs-java/convert-openoffice-odp/) เมื่อคุณต้องการคำแนะนำเฉพาะรูปแบบ.

## **การแปลง PPT เป็น PPTX**

PPT เป็นรูปแบบ PowerPoint แบบไบนารีเก่า, ในขณะที่ PPTX เป็นรูปแบบ Office Open XML รุ่นใหม่. Aspose.Slides for Node.js via Java รองรับการแปลง PPT เป็น PPTX ด้วยความแม่นยำสูงโดยคงโครงสร้างการนำเสนอที่ซับซ้อนเช่น มาสเตอร์, เลย์เอาต์, สไลด์, แผนภูมิ, รูปร่างที่กลุ่ม, พื้นที่ใส่, กรอบข้อความ, เนื้อผ้า, และการเติมรูปภาพ.

สำหรับรายละเอียด, ดู [แปลง PPT เป็น PPTX](/slides/th/nodejs-java/convert-ppt-to-pptx/) และ [PPT กับ PPTX](/slides/th/nodejs-java/ppt-vs-pptx/).

## **การส่งออกแบบจัดรูปแบบคงที่**

PDF, XPS, และ TIFF มีประโยชน์เมื่อผลลัพธ์ควรมีลักษณะเดียวกันบนอุปกรณ์ต่าง ๆ และไม่ควรถูกแก้ไขเป็นงานนำเสนอ. บทความเฉพาะสำหรับ PDF, XPS, และ TIFF อธิบายวิธีควบคุมการปฏิบัติตาม, สไลด์ที่ซ่อนไว้, โน้ต, คุณภาพภาพ, การบีบอัด, รูปแบบพิกเซล, และขนาดผลลัพธ์.

## **การส่งออก HTML และรูปภาพ**

การส่งออก HTML และ HTML5 มีประโยชน์สำหรับการดูบนเบราว์เซอร์, การเผยแพร่บนเว็บ, และการแชร์แบบเบา. การส่งออกรูปภาพมีประโยชน์เมื่อแต่ละสไลด์ต้องเป็นตัวอย่าง, ภาพย่อย, หรือทรัพยากรเรสเตอร์แยกกัน. ใช้บทความ PNG, JPG, และ SVG เพื่อขอคำแนะนำการเรนเดอร์ตามรูปแบบ.

## **FAQ**

**ฉันต้องการ Microsoft PowerPoint เพื่อแปลงงานนำเสนอหรือไม่?**

ไม่. Aspose.Slides for Node.js via Java เป็นไลบรารีแยกที่ไม่ต้องการ Microsoft PowerPoint หรือการทำงานอัตโนมัติของ Office.

**ฉันสามารถแปลงหลายงานนำเสนอเป็นชุดได้หรือไม่?**

ใช่. โหลดงานนำเสนอแต่ละไฟล์, บันทึกเป็นรูปแบบที่ต้องการ, และทำลายวัตถุงานนำเสนอหลังการประมวลผล. สำหรับการประมวลผลแบบขนาน, ใช้ตัวอย่างงานนำเสนอแยกกันและปฏิบัติตามคำแนะนำ [การทำงานหลายเธรด](/slides/th/nodejs-java/multithreading/).

**ฉันสามารถส่งออกเฉพาะสไลด์ที่เลือกได้หรือไม่?**

ใช่. วิธีการส่งออกหลายวิธีช่วยให้คุณระบุดัชนีสไลด์หรือเรนเดอร์สไลด์เดี่ยวตามรูปแบบผลลัพธ์. ดูบทความเฉพาะสำหรับรูปแบบเป้าหมาย.

**ฉันสามารถรวมสไลด์ที่ซ่อนเมื่อส่งออกเป็น PDF หรือ XPS ได้หรือไม่?**

ใช่. ใช้การตั้งค่าการส่งออกสไลด์ที่ซ่อนที่อธิบายในบทความ [PDF](/slides/th/nodejs-java/convert-powerpoint-to-pdf/) และ [XPS](/slides/th/nodejs-java/convert-powerpoint-to-xps/).

**ฉันสามารถสร้างเอาต์พุต PDF/A ได้หรือไม่?**

ใช่. มีการตั้งค่าการปฏิบัติตาม PDF สำหรับการส่งออก PDF. ดู [แปลง PowerPoint เป็น PDF](/slides/th/nodejs-java/convert-powerpoint-to-pdf/) สำหรับรายละเอียด.

**ฟอนต์จะถูกจัดการอย่างไรระหว่างการแปลง?**

Aspose.Slides สามารถใช้ฟอนต์ฝัง, ฟอนต์สำรอง, และการตั้งค่าการทดแทนฟอนต์. ดู [ฟอนต์ฝัง](/slides/th/nodejs-java/embedded-font/), [ฟอนต์สำรอง](/slides/th/nodejs-java/fallback-font/), และ [การทดแทนฟอนต์](/slides/th/nodejs-java/font-substitution/).
