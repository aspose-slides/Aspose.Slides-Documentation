## **ภาพรวม**

การเข้าถึงการนำเสนอทำให้ผู้ใช้เทคโนโลยีช่วยเหลือ—เช่นโปรแกรมอ่านหน้าจอ, จอแสดงผลอักษรเบรล, หรือการนำทางด้วยแป้นพิมพ์เท่านั้น—สามารถเข้าใจและเลื่อนผ่านสไลด์ของคุณได้อย่างมีประสิทธิภาพเทียบเท่ากับผู้ชมที่มองเห็นและใช้เมาส์ การปฏิบัติตามแนวทางที่ดีมุ่งเน้นที่ลำดับการอ่านที่ชัดเจน, คำอธิบายภาพสำรองที่มีความหมายสำหรับภาพข้อมูล, คอนทราสต์สีที่เพียงพอ, ตัวอักษรที่อ่านง่าย, ข้อความลิงก์ที่อธิบายได้, และหลีกเลี่ยงการสื่อความหมายโดยใช้สีหรือตำแหน่งอย่างเดียว เมื่อวางแผนการเข้าถึงตั้งแต่ต้น ผลลัพธ์จะเป็นโครงสร้างที่สะอาดตา, ภาพที่สอดคล้องกันมากขึ้น, และเนื้อหาที่เข้าถึงผู้ชมทุกคนโดยไม่ต้องมีวิธีแก้ไข

## **ทำเครื่องหมายว่าเป็นการตกแต่ง**

ทำเครื่องหมายว่าเป็นการตกแต่งใช้สำหรับบ่งชี้ภาพที่เป็นเพียงการตกแต่งเท่านั้นเพื่อให้โปรแกรมอ่านหน้าจอข้ามไป, ลดเสียงรบกวนและทำให้โฟกัสอยู่ที่เนื้อหาที่มีความหมาย ใช้กับพื้นหลัง, ลวดลาย, และช่องว่าง—ไม่ใช้กับแผนภูมิ, ไอคอน, หรือรูปภาพที่สื่อข้อมูล Aspose.Slides เปิดเผยฟล็ากนี้เพื่อการตรวจจับและการตรวจสอบ, ทำให้สามารถทำการตรวจสอบการเข้าถึงและทำความสะอาดโดยอัตโนมัติได้

![ทำเครื่องหมายว่าเป็นการตกแต่ง](mark_as_decorative.png)

ตัวอย่างโค้ดต่อไปนี้แสดงวิธีตรวจสอบว่า shape ถูกทำเครื่องหมายว่าเป็นการตกแต่งหรือไม่

```js
let presentation = new aspose.slides.Presentation("sample.pptx");
try {
    let shape = presentation.getSlides().get_Item(0).getShapes().get_Item(0);
    console.log("Is shape decorative:", shape.isDecorative());
} finally {
    presentation.dispose();
}
```
