## **OpenXML คืออะไร?**
OpenXML SDK คือ .NET SDK จาก Microsoft เพื่อประมวลผลรูปแบบไฟล์ Microsoft Office 2007 และรุ่นใหม่กว่า รูปแบบไฟล์เหล่านี้เป็นแพ็กเกจที่บีบอัดเป็น zip และ Open XML SDK for Office ทำให้การจัดการแพ็กเกจ Open XML รวมถึงองค์ประกอบสคีม่า Open XML ภายในแพ็กเกจเป็นเรื่องง่าย คลาสใน Open XML SDK จะบรรจุงานทั่วไปหลายอย่างที่นักพัฒนาต้องทำกับแพ็กเกจ Open XML ทำให้คุณสามารถทำการดำเนินการที่ซับซ้อนได้ด้วยบรรทัดโค้ดไม่กี่บรรทัด

หลังจากที่คุณติดตั้ง Open XML SDK 2.5 แล้ว ให้เปิดโครงการหรือแอปพลิเคชันที่มีอยู่ใน Visual Studio หรือสร้างโครงการหรือแอปพลิเคชันใหม่ จากนั้นในโครงการหรือแอปของคุณ ให้เพิ่มการอ้างอิงไปยังคอมโพเนนต์ต่อไปนี้

- DocumentFormat.OpenXml
- WindowsBase

## **Aspose.Slides for .NET**
Aspose.Slides for .NET เป็นคอมโพเนนต์การจัดการ PowerPoint® ที่เป็นเอกสิทธิ์ ซึ่งทำให้แอปพลิเคชัน .NET สามารถอ่าน เขียน และจัดการเอกสาร PowerPoint โดยไม่ต้องใช้ Microsoft PowerPoint

Aspose.Slides for .NET เป็นคอมโพเนนต์ .NET ตัวแรกและเดียวที่ให้ฟังก์ชันการจัดการเอกสาร PowerPoint ภายในแอปของคุณเช่นเดียวกับคอมโพเนนต์ .NET ของ Aspose ทุกตัว Aspose.Slides เขียนด้วย C# ที่จัดการโดย .NET มีราคาเหมาะสมและทำงานเร็วเป็นอันมาก

## **บทนำ**
นักพัฒนาที่กำลังมองหาห้องสมุดรูปแบบไฟล์อาจต้องสำรวจคุณลักษณะที่ห้องสมุดต่าง ๆ ในตลาดมีให้ โดยทั่วไปจะพิจารณาปัจจัยสองประการเมื่อเลือกตัวเลือก

- คุณลักษณะ
- ความพยายามที่ต้องใช้ในการใช้งานห้องสมุด

หากคุณเป็นนักพัฒนา .NET ที่ต้องการห้องสมุดรูปแบบไฟล์ที่เร็ว ง่าย และมีน้ำหนักเบาเพื่อประมวลผลไฟล์ คุณอาจต้องเปรียบเทียบ VSTO (Visual Studio tool for Office) กับ Aspose โครงการนี้จะช่วยให้คุณเปรียบเทียบคุณลักษณะและโค้ดในห้องสมุดทั้งสองได้
