SVG—ย่อมาจาก Scalable Vector Graphics—เป็นประเภทหรือรูปแบบกราฟิกมาตรฐานที่ใช้ในการแสดงภาพสองมิติ SVG จัดเก็บภาพเป็นเวกเตอร์ใน XML พร้อมด้วยรายละเอียดที่กำหนดพฤติกรรมหรือรูปลักษณ์ของภาพ  

SVG เป็นหนึ่งในไม่กี่รูปแบบของภาพที่ตอบสนองมาตรฐานระดับสูงเหล่านี้: ความสามารถในการขยายตัว, ความโต้ตอบ, ประสิทธิภาพ, การเข้าถึง, ความสามารถในการโปรแกรมและอื่น ๆ ด้วยเหตุผลเหล่านี้ มักจะใช้ในงานพัฒนาเว็บ  

คุณอาจต้องการใช้ไฟล์ SVG ในสถานการณ์ต่อไปนี้:

- เมื่อคุณวางแผนพิมพ์งานนำเสนอของคุณในรูปแบบขนาดใหญ่มาก รูปภาพ SVG สามารถขยายได้ถึงความละเอียดหรือระดับใด ๆ ก็ได้ คุณสามารถปรับขนาดรูปภาพ SVG ได้หลายครั้งตามต้องการโดยไม่สูญเสียคุณภาพ
- เมื่อคุณต้องการใช้แผนภูมิและกราฟจากสไลด์ของคุณในสื่อหรือแพลตฟอร์มต่าง ๆ ส่วนใหญ่ของโปรแกรมอ่านสามารถตีความไฟล์ SVG ได้
- เมื่อคุณต้องการใช้ขนาดภาพที่เล็กที่สุดเท่าที่จะเป็นไปได้ ไฟล์ SVG มักจะมีขนาดเล็กกว่าภาพความละเอียดสูงในรูปแบบอื่น ๆ โดยเฉพาะรูปแบบที่อิงบิตแมพ (JPEG หรือ PNG)

Aspose.Slides for .NET ช่วยให้คุณส่งออกสไลด์ในงานนำเสนอของคุณเป็นภาพ **SVG** เพื่อสร้างภาพ SVG จากใด ๆ ให้ทำตามขั้นตอนต่อไปนี้:

- สร้างอินสแตนซ์ของคลาส Presentation
- วนรอบทุกสไลด์ในงานนำเสนอ
- เขียนแต่ละสไลด์ออกเป็นไฟล์ SVG ของตนเองโดยใช้ FileStream

{{% alert color="primary" %}} 
คุณอาจต้องการลองใช้[เว็บแอปพลิเคชันฟรี](https://products.aspose.app/slides/th/conversion/ppt-to-svg) ที่เราได้ทำการดำเนินการฟังก์ชันการแปลง PPT เป็น SVG จาก Aspose.Slides for .NET
{{% /alert %}} 

โค้ดตัวอย่างนี้ใน C# แสดงวิธีการแปลง PPT เป็น SVG โดยใช้ Aspose.Slides:

``` csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    for (var index = 0; index < pres.Slides.Count; index++)
    {
        ISlide slide = pres.Slides[index];

        using (FileStream fileStream = new FileStream($"slide-{index}.svg", FileMode.Create, FileAccess.Write))
        {
            slide.WriteAsSvg(fileStream);   
        }
    }
}
```
