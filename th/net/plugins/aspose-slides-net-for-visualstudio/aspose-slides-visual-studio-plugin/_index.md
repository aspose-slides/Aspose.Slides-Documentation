## **คำนำ**
Microsoft Visual Studio เป็นสภาพแวดล้อมการพัฒนาแบบบูรณาการ (IDE) จาก Microsoft ใช้ในการพัฒนาโปรแกรมคอมพิวเตอร์สำหรับ Microsoft Windows รวมถึงเว็บไซต์ แอปพลิเคชันเว็บและบริการเว็บ Visual Studio ใช้แพลตฟอร์มการพัฒนาซอฟต์แวร์ของ Microsoft เช่น Windows API, Windows Forms, Windows Presentation Foundation, Windows Store และ Microsoft Silverlight สามารถผลิตโค้ดเนทีฟและโค้ดที่จัดการได้

### **Aspose.Slides for .NET**
Aspose.Slides for .NET เป็น API การจัดการ PowerPoint® ที่เป็นเอกลักษณ์ ซึ่งทำให้แอปพลิเคชัน .NET สามารถอ่าน, เขียนและจัดการเอกสาร PowerPoint โดยไม่ต้องใช้ Microsoft PowerPoint  

Aspose.Slides for .NET เป็น API PowerPoint .NET ตัวแรกและ唯一ที่ให้ความสามารถในการจัดการเอกสาร PowerPoint ภายในแอปพลิเคชันของคุณเอง เช่นเดียวกับคอมโพเนนต์ .NET ของ Aspose ทั้งหมด Aspose.Slides เขียนด้วย C# ที่จัดการได้, มีราคาไม่แพงและเร็วอย่าง Lightning  

สำหรับรายละเอียดเพิ่มเติมเกี่ยวกับ [Aspose.Slides for .NET API](http://www.aspose.com/.net/powerpoint-component.aspx)

### **Explore Aspose.Slides for .NET Examples Using Visual Studio**
Aspose Visual Studio Plugin - **Explore Aspose.Slides for .NET Examples** เป็นเครื่องมือที่ยอดเยี่ยมในการดาวน์โหลดและสำรวจตัวอย่าง API ของ Aspose.Slides for .NET อย่างรวดเร็ว มันช่วยประหยัดเวลาและความพยายามโดยให้ตัวเลือกง่าย ๆ เพื่อเลือก, ดาวน์โหลดและเปิดโครงการตัวอย่างล่าสุดโดยไม่ต้องออกจาก Visual Studio  

เวอร์ชันนี้ให้คุณสมบัติดังต่อไปนี้:

- รองรับ Visual Studio 2010, Visual Studio 2012 และ Visual Studio 2013
- ใช้งานง่ายจากเมนู **File > Aspose** ใน Visual Studio
- ประหยัดเวลาและลดความซับซ้อนในการเรียนรู้
- ให้คุณเลือกและเปิดตัวอย่าง API ของ Aspose.Slides for .NET ล่าสุดที่โฮสต์บน Github.com
- อนุญาตให้คุณเปิดโครงการตัวอย่าง **C#** หรือ **VB.NET**
- โครงการที่เลือกจะเปิดโดยอัตโนมัติใน Visual Studio พร้อมการอ้างอิงที่จำเป็นทั้งหมด เพื่อให้คุณมีสภาพแวดล้อมพร้อมใช้งานและสำรวจ
- ดาวน์โหลดไลบรารี API ล่าสุดจาก **NuGet** และตัวอย่างจาก **GitHub** โดยอัตโนมัติ

## **ข้อกำหนดระบบและแพลตฟอร์มที่รองรับ**
เพื่อทำการติดตั้งและใช้งาน Aspose Visual Studio Plugin คุณต้องมี Visual Studio เวอร์ชันใดเวอร์ชันหนึ่งติดตั้งอยู่  

- Visual Studio 2010, 2012, 2013  

หากพบปัญหาในการติดตั้งหรือใช้ปลั๊กอินนี้ โปรดติดต่อเราได้เลย

## **การติดตั้ง**
การติดตั้ง Aspose Visual Studio Plugin นั้นง่ายมาก โปรดทำตามขั้นตอนต่อไปนี้  

- ดับเบิลคลิกไฟล์ msi ที่ดาวน์โหลด
- ทำตามขั้นตอน คุณสามารถกด Next ในแต่ละหน้าจอเพื่อทำการติดตั้งด้วยการตั้งค่าเริ่มต้น  

คุณจะเห็นกล่องโต้ตอบแสดงความสำเร็จเมื่อการติดตั้งเสร็จสมบูรณ์  

![todo:image_alt_text](aspose-slides-visual-studio-plugin_2)

**หมายเหตุ:** โปรดแน่ใจว่าคุณได้รีสตาร์ท Visual Studio เพื่อให้การเปลี่ยนแปลงมีผล

## **การใช้งาน**
หลังจากติดตั้งแล้ว คุณสามารถเรียกใช้และใช้งานปลั๊กอินโต้ตอบได้อย่างง่ายดายตามที่อธิบายด้านล่าง:  

**1.** เปิดปลั๊กอินโดยเลือก **File > Aspose** แล้วตามด้วย **New Aspose.Slides Example Project** ใน Visual Studio.  

![todo:image_alt_text](aspose-slides-visual-studio-plugin_3)

**2.** ครั้งแรกที่คุณเปิดปลั๊กอิน จะเห็นหน้าต่างเลือก API ตามภาพด้านล่าง ระบบจะใช้เวลาสองสามนาทีเพื่อดาวน์โหลด API ล่าสุดและไลบรารีจาก **NuGet** รวมถึงซอร์สโค้ดตัวอย่างล่าสุดจาก **GitHub**.  

![todo:image_alt_text](aspose-slides-visual-studio-plugin_4)

**3.** ในหน้าต่าง Select an API to View Examples ให้เลือกหรือเปลี่ยนตำแหน่งที่โครงการตัวอย่างจะถูกสร้าง และเลือกหรือเปลี่ยนประเภทโครงการตัวอย่างเป็น **C#** หรือ **VB.NET** แล้วคลิก **Continue** เพื่อเปิดโครงการตัวอย่างใน Visual Studio.  

![todo:image_alt_text](aspose-slides-visual-studio-plugin_1)

**4.** โครงการตัวอย่างใหม่ตามประเภทที่เลือก **C#** หรือ **VB.NET** ใน Visual Studio.

## **การสนับสนุน, ขยายและมีส่วนร่วม**
### **การสนับสนุน**
ตั้งแต่วันแรกของ Aspose เรารู้ว่าการให้ผลิตภัณฑ์ดีเพียงอย่างเดียวไม่เพียงพอ เราต้องส่งมอบบริการที่ดีเช่นกัน เราเป็นนักพัฒนาด้วยตนเองและเข้าใจว่าการเจอปัญหาทางเทคนิคหรือบั๊กในซอฟต์แวร์ทำให้คุณทำสิ่งที่ต้องทำไม่ได้อย่างไร เราอยู่ที่นี่เพื่อแก้ปัญหา ไม่ใช่สร้างปัญหา  

นี่คือเหตุผลที่เรามีบริการสนับสนุนฟรี ผู้ใช้ผลิตภัณฑ์ของเราทั้งที่ซื้อและที่ใช้รุ่นทดลอง มีสิทธิได้รับความเอาใจใส่และเคารพอย่างเต็มที่จากเรา  

คุณสามารถบันทึกปัญหาหรือข้อเสนอแนะที่เกี่ยวกับ Aspose Visual Studio Plugin ผ่านแพลตฟอร์มต่อไปนี้  

- [Github ](https://github.com/asposeSlides/Aspose.Slides-for-.NET/issues)

### **การขยายและการร่วมมือ**
**Explore Aspose.Slides for .NET Examples** สำหรับ Visual Studio เป็นโครงการโอเพนซอร์สและซอร์สโค้ดของมันพร้อมให้ดาวน์โหลดบนเว็บไซต์โซเชียลโค้ดดิ้งหลักด้านล่าง นักพัฒนาถูกกระตุ้นให้ดาวน์โหลดซอร์สโค้ดและขยายฟังก์ชันตามความต้องการของตนเอง  

#### **โค้ดต้นฉบับ**
คุณสามารถรับโค้ดต้นฉบับล่าสุดได้จากแหล่งต่อไปนี้  

- [Github ](https://github.com/aspose-slides/Aspose.Slides-for-.NET)

#### **วิธีตั้งค่าโค้ดต้นฉบับ**
คุณต้องติดตั้งสิ่งต่อไปนี้เพื่อเปิดและขยายโค้ดต้นฉบับ  

- Visual Studio 2010  

กรุณาตามขั้นตอนง่าย ๆ ด้านล่างเพื่อเริ่มต้น  

1. ดาวน์โหลด/คัดลอกโค้ดต้นฉบับ
1. เปิด Visual Studio 2010 และเลือก **File** > **Open Project**
1. ไปที่โฟลเดอร์โค้ดต้นฉบับล่าสุดที่คุณดาวน์โหลดและเปิด **AsposeVisualStudioPlugin.sln**

#### **ภาพรวมของโค้ดต้นฉบับ**
มีสองโครงการในโซลูชัน  

- **AsposeVisualStudioPlugin** - มีการบูรณาการกับ Visual Studio, การเลือกและดาวน์โหลดอัตโนมัติของ Aspose API และตัวอย่าง
- **Installer** - โครงการติดตั้ง Wix เพื่อสร้างไฟล์ msi สำหรับปลั๊กอิน
