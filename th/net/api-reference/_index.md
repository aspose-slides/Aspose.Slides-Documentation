Aspose.Slides for .NET เป็นคลังคลาสที่ทำให้แอปพลิเคชัน .NET สามารถอ่านและเขียนเอกสารงานนำเสนอโดยไม่ต้องใช้ Microsoft PowerPoint® หรือโปรแกรมอื่น ๆ Aspose.Slides for .NET ช่วยให้สามารถจัดการข้อความ รูปทรง ตารางและแอนิเมชัน เพิ่มเสียงและวิดีโอลงในสไลด์ ดูตัวอย่างสไลด์ ส่งออกสไลด์เป็นรูปแบบ SVG, PDF และอื่น ๆ

สามารถดูเอกสารอ้างอิง API ล่าสุดได้ที่ [Aspose.Slides for .NET API Reference](https://reference.aspose.com/slides/th/net).
