## **ภาพรวม**

FAQ นี้ให้คำตอบสำหรับคำถามทั่วไปเกี่ยวกับ Aspose.Slides ครอบคลุมรูปแบบไฟล์ที่รองรับ, การจัดการข้อยกเว้นเมื่อทำงานกับงานนำเสนอขนาดใหญ่, การเปลี่ยนขนาดสไลด์, การแสดงตัวอย่างสไลด์, การดึงข้อความจากงานนำเสนอ, การจัดรูปแบบขอบตาราง, การวางรูปภาพ, และการแก้ไขปัญหาเกี่ยวกับฟอนต์เมื่อแปลงงานนำเสนอเป็น PDF หรือรูปภาพ.

## **รูปแบบไฟล์ที่รองรับ**

**Q:** Aspose.Slides for .NET รองรับรูปแบบไฟล์ใดบ้าง?  

**A:** Aspose.Slides for .NET รองรับรูปแบบไฟล์ที่อธิบายไว้ใน [รูปแบบไฟล์ที่รองรับ](/slides/th/net/supported-file-formats/).

## **ข้อยกเว้น**

**Q:** ฉันได้รับ OutOfMemoryException ขณะโหลดไฟล์ PPT ขนาดใหญ่ที่มีรูปภาพ มีข้อจำกัดเรื่องขนาดไฟล์ใน Aspose.Slides หรือไม่?  

**A:** ไม่มีสูตรเฉพาะสำหรับการคำนวณขนาดของงานนำเสนอที่ Aspose.Slides รองรับ ควรมีพื้นที่เพียงพอเพื่อรองรับโครงสร้างงานนำเสนอทั้งหมดและรูปภาพในหน่วยความจำ โดยทั่วไปแล้วรูปภาพในหน่วยความจำจะใช้พื้นที่มากกว่าในฮาร์ดดิสก์โดยเฉพาะเมื่อรูปภาพมีเอฟเฟกต์เพิ่มเติม  

โดยทั่วไป Aspose.Slides for .NET สามารถจัดการไฟล์งานนำเสนอได้อย่างง่ายดายประมาณ 300 MB บนเซิร์ฟเวอร์ที่มี RAM 4 GB.

## **การทำงานกับสไลด์**

**Q:** ฉันสามารถเปลี่ยนขนาดของสไลด์ในงานนำเสนอได้หรือไม่?  

**A:** คุณสามารถใช้คุณสมบัติ `SlideSize` ที่เปิดให้ใช้งานโดยคลาส [Presentation](https://reference.aspose.com/slides/th/net/aspose.slides/presentation/) เพื่อกำหนดขนาดของสไลด์ในงานนำเสนอได้  

**Q:** มีวิธีกำหนดสไลด์ขนาดต่างกันในงานนำเสนอหรือไม่?  

**A:** เนื่องจากขนาดของสไลด์ถูกกำหนดระดับงานนำเสนอในเอกสาร Microsoft PowerPoint จึงไม่มีวิธีทำเช่นนั้น  

**Q:** Aspose.Slides for .NET รองรับการแสดงตัวอย่างสไลด์ก่อนบันทึกหรือไม่?  

**A:** คุณสามารถเรนเดอร์สไลด์ของงานนำเสนอเป็นภาพและใช้ภาพเหล่านี้เพื่อแสดงตัวอย่างสไลด์ได้  

## **การทำงานกับข้อความ**

**Q:** สามารถดึงข้อความทั้งหมดจากงานนำเสนอได้หรือไม่?  

**A:** Aspose.Slides for .NET มีคลาส [SlideUtil](https://reference.aspose.com/slides/th/net/aspose.slides.util/slideutil/) อยู่ในเนมสเปซ `Aspose.Slides.Util` ซึ่งให้เมธอดต่าง ๆ สำหรับดึงข้อความทั้งหมดจากงานนำเสนอ  

**Q:** ทำไมขนาดของย่อหน้าถึงแตกต่างกันบนระบบปฏิบัติการ Windows และ Linux?  

**A:** การคำนวณขนาดของย่อหน้าขึ้นอยู่กับการคำนวณขนาดข้อความที่แทนย่อหน้าที่กำหนด การคำนวณขนาดข้อความอ้างอิงจากเมตริกของฟอนต์ที่ระบุในงานนำเสนอ PowerPoint หากฟอนต์ที่ระบุไม่มีอยู่ จะถูกแทนที่ด้วยฟอนต์ที่คล้ายที่สุด แต่ฟอนต์นั้นมีเมตริกที่แตกต่างจากฟอนต์เดิม ดังนั้นการคำนวณขนาดย่อหน้าในระบบที่ต่างกันจะให้ผลต่างกัน ขึ้นกับชุดฟอนต์ที่ติดตั้งไว้ เพื่อให้ได้ผลลัพธ์เดียวกันบนระบบปฏิบัติการต่าง ๆ คุณต้องติดตั้งฟอนต์เดียวกันบนระบบหรือโหลดฟอนต์ในเวลารันเป็น [ฟอนต์ภายนอก](/slides/th/net/custom-font/)

## **การจัดรูปแบบและรูปภาพ**

**Q:** จะตั้งค่าสีของขอบตารางอย่างไร?  

**A:** คุณสามารถเปลี่ยนสีของขอบตารางทั้งหมดหรือเพียงขอบรอบตารางทั้งหมดได้ สำหรับการเปลี่ยนขอบทั้งหมด ให้ใช้คุณสมบัติ `CellFormat` จากอินเทอร์เฟซ [ICell](https://reference.aspose.com/slides/th/net/aspose.slides/icell/) สำหรับขอบของตารางทั้งหมด ให้วนลูปผ่านเซลล์และเปลี่ยนสีของขอบภายนอก  

**Q:** Aspose.Slides for .NET ใช้หน่วยวัดใดในการวางรูปภาพ?  

**A:** พิกัดและขนาดของรูปร่างทั้งหมดบนสไลด์วัดเป็นหน่วยพอยต์ (72 dpi).

## **การทำงานกับฟอนต์**

**Q:** เมื่อแปลง PPT เป็น PDF หรือรูปภาพ ทำไมฟอนต์ถึงต่างกันในเอกสารผลลัพธ์?  

**A:** ปัญหานี้อาจบ่งชี้ว่าฟอนต์ที่ใช้ในงานนำเสนอไม่มีอยู่ในระบบปฏิบัติการที่รันโค้ด คุณควรติดตั้งฟอนต์บนระบบปฏิบัติการหรือโหลดฟอนต์เป็นฟอนต์ภายนอกโดยใช้คลาส [FontsLoader](https://reference.aspose.com/slides/th/net/aspose.slides/fontsloader/) ตามตัวอย่างด้านล่าง:  
```cs
var folders = new string[] { "path_to_a_folder_with_fonts" };
FontsLoader.LoadExternalFonts(folders);
```
