## **บทนำ**

การเข้าถึงงานนำเสนอทำให้ผู้ใช้เทคโนโลยีช่วยเหลือ—เช่น โปรแกรมอ่านหน้าจอ, หน้าจอเบรลล์, หรือการนำทางด้วยแป้นพิมพ์เท่านั้น—สามารถเข้าใจและนำทางสไลด์ของคุณได้อย่างมีประสิทธิภาพเทียบเท่ากับผู้ชมที่มองเห็นและใช้เมาส์ การปฏิบัติที่ดีมุ่งเน้นที่ลำดับการอ่านที่ชัดเจน, ข้อความแทนที่มีความหมายสำหรับภาพที่ให้ข้อมูล, ความแตกต่างสีที่เพียงพอ, ตัวอักษรที่อ่านง่าย, ข้อความลิงก์ที่อธิบาย, และการหลีกเลี่ยงการสื่อความหมายด้วยสีหรือตำแหน่งเพียงอย่างเดียว เมื่อการเข้าถึงถูกวางแผนตั้งแต่เริ่มต้น ผลลัพธ์จะเป็นโครงสร้างที่สะอาดขึ้น, ภาพที่สอดคล้องกันมากขึ้น, และเนื้อหาที่เข้าถึงผู้ชมทุกคนโดยไม่ต้องใช้การแก้ไขเสริม

## **ทำเครื่องหมายว่าเป็นของตกแต่ง**

การทำเครื่องหมายว่าเป็นของตกแต่งจะกำหนดว่าภาพที่เป็นของประดับเพียงอย่างเดียวให้โปรแกรมอ่านหน้าจอข้ามไป ลดเสียงรบกวนและทำให้โฟกัสอยู่ที่เนื้อหาที่มีความหมาย ใช้กับพื้นหลัง, การประดับ, และช่องว่าง—ไม่ใช้กับแผนภูมิ, ไอคอน, หรือภาพที่สื่อข้อมูล Aspose.Slides เปิดเผยฟลักนี้เพื่อการตรวจจับและการตรวจสอบ, ทำให้การตรวจสอบการเข้าถึงอัตโนมัติและการทำความสะอาดเป็นไปได้

![ทำเครื่องหมายว่าเป็นของตกแต่ง](mark_as_decorative.png)

ตัวอย่างโค้ดต่อไปนี้แสดงวิธีตรวจสอบว่ารูปร่างถูกทำเครื่องหมายว่าเป็นของตกแต่งหรือไม่

```cs
using var presentation = new Presentation("sample.pptx");

var shape = presentation.Slides[0].Shapes[0];
Console.WriteLine($"Is shape decorative: {shape.IsDecorative}");
```
