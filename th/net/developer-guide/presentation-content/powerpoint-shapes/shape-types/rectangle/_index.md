## **ภาพรวม**

บทความนี้แสดงวิธีการเพิ่มรูปทรงสี่เหลี่ยมผืนผ้าลงในสไลด์ PowerPoint ด้วย Aspose.Slides โดยครอบคลุมการสร้างสี่เหลี่ยมผืนผ้าง่าย การสร้างสี่เหลี่ยมผืนผ้าที่จัดรูปแบบแล้ว และการบันทึกการนำเสนอที่อัปเดตเป็นไฟล์ PPTX

คุณจะได้เห็นวิธีการกำหนดรูปแบบพื้นฐานของสี่เหลี่ยมผืนผ้า เช่น สีเติมแบบทึบ สีเส้น และความกว้างของเส้น นอกจากนี้ส่วน FAQ ของบทความยังชี้แนะงานที่เกี่ยวข้องกับสี่เหลี่ยมผืนผ้า เช่น มุมโค้ง การเติมภาพ เอฟเฟกต์ภาพ ลิงก์ไฮเปอร์ลิงก์ การล็อครูปทรง ตัวเลือกการส่งออก และคุณสมบัติมีประสิทธิภาพ

## **สร้างสี่เหลี่ยมผืนผ้าง่าย**
เช่นเดียวกับหัวข้อก่อนหน้า ตอนนี้เราจะพูดถึงการเพิ่มรูปทรงสี่เหลี่ยมผืนผ้า เพื่อให้ผู้พัฒนาสามารถเพิ่มสี่เหลี่ยมผืนผ้าง่ายหรือแบบจัดรูปแบบลงในสไลด์โดยใช้ Aspose.Slides for .NET   เพื่อเพิ่มสี่เหลี่ยมผืนผ้าง่ายลงในสไลด์ที่เลือกของการนำเสนอ โปรดปฏิบัติตามขั้นตอนต่อไปนี้:

1. สร้างอินสแตนซ์ของ [Presentation](https://reference.aspose.com/slides/th/net/aspose.slides/presentation)class.
2. ได้รับอ้างอิงของสไลด์โดยใช้ Index ของสไลด์.
3. เพิ่ม IAutoShape ประเภท Rectangle โดยใช้เมธอด AddAutoShape ที่เปิดให้ใช้งานจากอ็อบเจกต์ IShapes.
4. เขียนการนำเสนอที่แก้ไขแล้วเป็นไฟล์ PPTX.

ในตัวอย่างด้านล่าง เราได้เพิ่มสี่เหลี่ยมผืนผ้าง่ายลงในสไลด์แรกของการนำเสนอ

```c#
 // สร้างอินสแตนซ์ของคลาส Presentation ที่แสดงถึงไฟล์ PPTX
 using (Presentation pres = new Presentation())
 {
 
     // ดึงสไลด์แรก
     ISlide sld = pres.Slides[0];
 
     // เพิ่ม autoshape ประเภทสี่เหลี่ยมผืนผ้า
     sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 150, 50);
 
     //เขียนไฟล์ PPTX ลงดิสก์
     pres.Save("RectShp1_out.pptx", SaveFormat.Pptx);
 }
```

## **สร้างสี่เหลี่ยมผืนผ้าแบบจัดรูปแบบ**
เพื่อเพิ่มสี่เหลี่ยมผืนผ้าแบบจัดรูปแบบลงในสไลด์ โปรดปฏิบัติตามขั้นตอนต่อไปนี้:

1. สร้างอินสแตนซ์ของ [Presentation](https://reference.aspose.com/slides/th/net/aspose.slides/presentation)class.
2. ได้รับอ้างอิงของสไลด์โดยใช้ Index ของสไลด์.
3. เพิ่ม IAutoShape ประเภท Rectangle โดยใช้เมธอด AddAutoShape ที่เปิดให้ใช้งานจากอ็อบเจกต์ IShapes.
4. ตั้งค่า Fill Type ของสี่เหลี่ยมผืนผ้าเป็น Solid.
5. ตั้งค่าสีของสี่เหลี่ยมผืนผ้าโดยใช้คุณสมบัติ SolidFillColor.Color ของอ็อบเจกต์ FillFormat ที่เชื่อมโยงกับอ็อบเจกต์ IShape.
6. ตั้งค่าสีของเส้นของสี่เหลี่ยมผืนผ้า.
7. ตั้งค่าความกว้างของเส้นของสี่เหลี่ยมผืนผ้า.
8. เขียนการนำเสนอที่แก้ไขแล้วเป็นไฟล์ PPTX.
   ขั้นตอนข้างต้นถูกแสดงในตัวอย่างด้านล่าง

```c#
// สร้างอินสแตนซ์ของคลาส Presentation ที่แสดงถึงไฟล์ PPTX
using (Presentation pres = new Presentation())
{
    // ดึงสไลด์แรก
    ISlide sld = pres.Slides[0];

    // เพิ่ม autoshape ประเภทสี่เหลี่ยมผืนผ้า
    IShape shp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 150, 50);

    // ตั้งค่าการจัดรูปแบบบางอย่างให้กับรูปทรงสี่เหลี่ยมผืนผ้า
    shp.FillFormat.FillType = FillType.Solid;
    shp.FillFormat.SolidFillColor.Color = Color.Chocolate;

    // ตั้งค่าการจัดรูปแบบบางอย่างให้กับเส้นของสี่เหลี่ยมผืนผ้า
    shp.LineFormat.FillFormat.FillType = FillType.Solid;
    shp.LineFormat.FillFormat.SolidFillColor.Color = Color.Black;
    shp.LineFormat.Width = 5;

    //เขียนไฟล์ PPTX ลงดิสก์
    pres.Save("RectShp2_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

## **คำถามที่พบบ่อย**

**ฉันจะเพิ่มสี่เหลี่ยมผืนผ้าที่มีมุมโค้งได้อย่างไร?**

ใช้ [ประเภทรูปทรงมุมโค้ง](https://reference.aspose.com/slides/th/net/aspose.slides/shapetype/) และปรับรัศมีมุมในคุณสมบัติของรูปทรง; การปัดมุมยังสามารถทำได้ต่อมุมผ่านการปรับเชิงเรขาคณิต

**ฉันจะเติมสี่เหลี่ยมผืนผ้าด้วยภาพ (เทกซ์เจอร์) อย่างไร?**

เลือก [ประเภทการเติมรูปภาพ](https://reference.aspose.com/slides/th/net/aspose.slides/filltype/) ให้แหล่งภาพและกำหนด [โหมดการยืด/การเรียงต่อ](https://reference.aspose.com/slides/th/net/aspose.slides/picturefillmode/)

**สี่เหลี่ยมผืนผ้าสามารถมีเงาและแสงเรืองแสดงได้หรือไม่?**

ได้ [เงาภายนอก/ภายใน, แสงเรืองแสดงและขอบนุ่ม](/slides/th/net/shape-effect/) มีให้ใช้พร้อมพารามิเตอร์ที่ปรับได้

**ฉันสามารถแปลงสี่เหลี่ยมผืนผ้าให้เป็นปุ่มพร้อมไฮเปอร์ลิงก์ได้หรือไม่?**

ได้ [กำหนดไฮเปอร์ลิงก์](/slides/th/net/manage-hyperlinks/) ให้กับการคลิกรูปทรง (กระโดดไปยังสไลด์, ไฟล์, เว็บแอดเดรส หรืออีเมล)

**ฉันจะป้องกันสี่เหลี่ยมผืนผ้าจากการย้ายและการเปลี่ยนแปลงได้อย่างไร?**

[ใช้การล็อครูปทรง](/slides/th/net/applying-protection-to-presentation/): คุณสามารถห้ามการย้าย, ปรับขนาด, การเลือก หรือการแก้ไขข้อความเพื่อรักษาเลย์เอาต์

**ฉันสามารถแปลงสี่เหลี่ยมผืนผ้าเป็นภาพแรสเตอร์หรือ SVG ได้หรือไม่?**

ได้ คุณสามารถ[เรนเดอร์รูปทรง](http://reference.aspose.com/slides/th/net/aspose.slides/shape/getimage/) ไปเป็นภาพด้วยขนาด/สเกลที่กำหนดหรือ[ส่งออกเป็น SVG](https://reference.aspose.com/slides/th/net/aspose.slides/shape/writeassvg/) เพื่อใช้ในรูปแบบเวกเตอร์

**ฉันจะดึงคุณสมบัติจริง (effective) ของสี่เหลี่ยมผืนผ้าที่คำนึงถึงธีมและการสืบทอดได้อย่างรวดเร็วอย่างไร?**

[ใช้คุณสมบัติ effective ของรูปทรง](/slides/th/net/shape-effective-properties/): API จะคืนค่าที่คำนวณแล้วซึ่งรวมเอาสไตล์ธีม, เลย์เอาต์และการตั้งค่าท้องถิ่น ทำให้การวิเคราะห์รูปแบบง่ายขึ้น
