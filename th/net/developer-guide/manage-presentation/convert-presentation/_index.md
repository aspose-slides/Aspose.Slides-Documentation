## **ภาพรวม**

Aspose.Slides for .NET สามารถโหลดไฟล์นำเสนอ PowerPoint และ OpenDocument และบันทึกหรือเรนเดอร์เป็นรูปแบบอื่น ๆ หลากหลายได้โดยไม่ต้องใช้ Microsoft PowerPoint, OpenOffice หรือ LibreOffice คุณสามารถแปลงไฟล์ PPT รุ่นเก่าเป็น PPTX สมัยใหม่ ส่งออกการนำเสนอเป็นเอกสารแบบจัดรูปแบบคงที่เช่น PDF และ XPS เผยแพร่สไลด์เป็น HTML หรือเรนเดอร์สไลด์เป็นไฟล์ภาพสำหรับดูตัวอย่าง รูปย่อ และการจัดเก็บ

การแปลงเอกสารส่วนใหญ่ใช้กระบวนการทำงานทั่วไปเดียวกัน: โหลดไฟล์ต้นฉบับ เลือกรูปแบบเอาต์พุตที่ต้องการ และใช้ตัวเลือกเฉพาะรูปแบบตามความจำเป็น สำหรับรูปแบบภาพแต่ละสไลด์จะถูกเรนเดอร์แยกจากกันแล้วบันทึกเป็นภาพเรสเตอร์หรือเวกเตอร์ บทความที่แนบด้านล่างให้รายละเอียดการดำเนินการสำหรับแต่ละกรณี

## **การแปลง PowerPoint และ OpenDocument**

Aspose.Slides for .NET รองรับการแปลงจากรูปแบบการนำเสนอที่ใช้บ่อยเช่น PPT, PPTX, PPS, PPSX, POT, POTX และ ODP API การแปลงเดียวกันใช้ได้กับไฟล์ PowerPoint และ OpenDocument ดังนั้นกระบวนการที่บันทึกไฟล์ PPTX เป็น PDF มักสามารถใช้กับไฟล์ ODP ได้โดยการเปลี่ยนไฟล์อินพุตเท่านั้น

เมื่อแปลงไฟล์ ODP โปรดจำไว้ว่าแอปพลิเคชัน PowerPoint และ OpenDocument ไม่สนับสนุนทุกการจัดวางและคุณลักษณะการฟอร์แมตในลักษณะเดียวกัน หากไฟล์ ODP ถูกสร้างใน LibreOffice หรือ OpenOffice Impress ให้ตรวจสอบผลลัพธ์และใช้ตัวเลือกที่อธิบายไว้ใน [แปลงการนำเสนอ OpenDocument](/slides/th/net/convert-openoffice-odp/) เมื่อคุณต้องการคำแนะนำเฉพาะรูปแบบ

## **การแปลง PPT เป็น PPTX**

PPT เป็นรูปแบบ PowerPoint แบบไบนารีแบบเก่าในขณะที่ PPTX เป็นรูปแบบ Office Open XML สมัยใหม่ Aspose.Slides for .NET รองรับการแปลง PPT เป็น PPTX อย่างแม่นยำสูงพร้อมคงโครงสร้างการนำเสนอที่ซับซ้อน เช่น มาสเตอร์, เลย์เอาต์, สไลด์, แผนภูมิ, รูปร่างที่กลุ่ม, ตัวแทนที่ไว้, กรอบข้อความ, ลายผิว, และการเติมภาพ

สำหรับรายละเอียด ดูที่ [แปลง PPT เป็น PPTX](/slides/th/net/convert-ppt-to-pptx/) และ [PPT vs PPTX](/slides/th/net/ppt-vs-pptx/).

## **การส่งออกแบบเลย์เอาต์คงที่**

PDF, XPS และ TIFF มีประโยชน์เมื่อผลลัพธ์ต้องแสดงรูปแบบเดียวกันบนอุปกรณ์ต่าง ๆ และไม่ควรแก้ไขเป็นการนำเสนอ ใช้ [PdfOptions](https://reference.aspose.com/slides/th/net/aspose.slides.export/pdfoptions/), [XpsOptions](https://reference.aspose.com/slides/th/net/aspose.slides.export/xpsoptions/), และ [TiffOptions](https://reference.aspose.com/slides/th/net/aspose.slides.export/tiffoptions/) เพื่อควบคุมการปฏิบัติตาม, สไลด์ที่ซ่อน, บันทึก, คุณภาพภาพ, การบีบอัด, รูปแบบพิกเซล, และขนาดผลลัพธ์

## **การส่งออก HTML และภาพ**

การส่งออก HTML และ HTML5 มีประโยชน์สำหรับการดูในเบราว์เซอร์ การเผยแพร่บนเว็บ และการแชร์แบบเบา การส่งออกภาพมีประโยชน์เมื่อแต่ละสไลด์ต้องเป็นตัวอย่าง รูปย่อ หรือสินทรัพย์เรสเตอร์แยก ใช้บทความ PNG, JPG, และ SVG เพื่อคำแนะนำการเรนเดอร์เฉพาะรูปแบบ

## **คำถามที่พบบ่อย**

**ฉันต้องใช้ Microsoft PowerPoint เพื่อแปลงการนำเสนอหรือไม่?**

ไม่ Aspose.Slides for .NET เป็นไลบรารีแบบสแตนด์อโลนและไม่ต้องการ Microsoft PowerPoint หรือการอัตโนมัติของ Office

**ฉันสามารถแปลงการนำเสนอหลายไฟล์พร้อมกันได้หรือไม่?**

ได้ โหลดการนำเสนอแต่ละไฟล์ บันทึกเป็นรูปแบบที่ต้องการ และทำลายวัตถุ `Presentation` หลังจากประมวลผล สำหรับการประมวลผลแบบขนาน ใช้ตัวอย่างการนำเสนอแยกกันและปฏิบัติตามคำแนะนำ [multithreading](/slides/th/net/multithreading/)

**ฉันสามารถส่งออกเฉพาะสไลด์ที่เลือกได้หรือไม่?**

ได้ วิธีส่งออกหลายวิธีอนุญาตให้ส่งผ่านดัชนีสไลด์หรือเรนเดอร์สไลด์เดี่ยวขึ้นอยู่กับรูปแบบเอาต์พุต ดูบทความเฉพาะสำหรับรูปแบบเป้าหมาย

**ฉันสามารถรวมสไลด์ที่ซ่อนได้เมื่อส่งออกเป็น PDF หรือ XPS หรือไม่?**

ได้ ใช้คุณสมบัติ `ShowHiddenSlides` ใน [PdfOptions](https://reference.aspose.com/slides/th/net/aspose.slides.export/pdfoptions/) หรือ [XpsOptions](https://reference.aspose.com/slides/th/net/aspose.slides.export/xpsoptions/)

**ฉันสามารถสร้างผลลัพธ์ PDF/A ได้หรือไม่?**

ได้ การตั้งค่าการปฏิบัติตาม PDF มีให้ผ่าน [PdfOptions.Compliance](https://reference.aspose.com/slides/th/net/aspose.slides.export/pdfoptions/compliance/) และ [PdfCompliance](https://reference.aspose.com/slides/th/net/aspose.slides.export/pdfcompliance/)

**ฟอนท์ถูกจัดการอย่างไรระหว่างการแปลง?**

Aspose.Slides สามารถใช้ฟอนท์ที่ฝังอยู่, การสำรองฟอนท์, และการแทนที่ฟอนท์ได้ ดูที่ [Embedded Font](/slides/th/net/embedded-font/), [Fallback Font](/slides/th/net/fallback-font/), และ [Font Substitution](/slides/th/net/font-substitution/)
