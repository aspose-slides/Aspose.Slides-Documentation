## **ภาพรวม**

Aspose.Slides ให้คุณเพิ่มสไลด์ลงในไฟล์งานนำเสนอ PowerPoint ได้โดยโปรแกรม การนำเสนอประกอบด้วยสไลด์มาสเตอร์/เลเอาต์และสไลด์ปกติ และสไลด์ปกติจะเรียงลำดับตามดัชนีที่เริ่มจากศูนย์ แต่ละสไลด์มี ID ที่ไม่ซ้ำกัน และไฟล์นำเสนอที่ไม่มีสไลด์จะไม่ได้รับการสนับสนุน

บทความนี้อธิบายวิธีสร้างออบเจกต์ `Presentation` การเข้าถึงคอลเลกชันสไลด์ การเพิ่มสไลด์เปล่า การทำงานกับสไลด์ที่เพิ่งเพิ่ม และการบันทึกการนำเสนอที่อัปเดต รวมถึงประเด็นที่เกี่ยวข้อง เช่น การแทรกสไลด์ในตำแหน่งเฉพาะ การใช้เลเอาต์ และการทำความเข้าใจสไลด์เปล่าที่มีในงานนำเสนอที่สร้างใหม่

## **เพิ่มสไลด์ลงในงานนำเสนอ**
ก่อนจะพูดถึงการเพิ่มสไลด์ลงในไฟล์งานนำเสนอ เรามาพูดถึงข้อเท็จจริงบางอย่างเกี่ยวกับสไลด์กันก่อน ไฟล์งานนำเสนอ PowerPoint แต่ละไฟล์ประกอบด้วยสไลด์ Master / Layout และสไลด์ Normal อื่น ๆ ซึ่งหมายความว่าไฟล์งานนำเสนอจะต้องมีสไลด์อย่างน้อยหนึ่งสไลด์หรือมากกว่า การทราบว่าไฟล์งานนำเสนอที่ไม่มีสไลด์ไม่ได้รับการสนับสนุนโดย Aspose.Slides for .NET เป็นสิ่งสำคัญ แต่ละสไลด์มี Id ที่เป็นเอกลักษณ์และสไลด์ Normal ทั้งหมดจะจัดเรียงตามดัชนีที่เริ่มจากศูนย์ Aspose.Slides for .NET อนุญาตให้ผู้พัฒนาเพิ่มสไลด์เปล่าไปยังงานนำเสนอของตน เพื่อเพิ่มสไลด์เปล่าในงานนำเสนอ โปรดทำตามขั้นตอนต่อไปนี้:

- สร้างอินสแตนซ์ของคลาส [Presentation](https://reference.aspose.com/slides/th/net/aspose.slides/presentation)
- สร้างอินสแตนซ์ของคลาส [ISlideCollection](https://reference.aspose.com/slides/th/net/aspose.slides/islidecollection) โดยกำหนดการอ้างอิงไปยังคุณสมบัติ Slides (คอลเลกชันของออบเจกต์ Slide ที่เป็นเนื้อหา) ที่เปิดเผยโดยออบเจกต์ Presentation
- เพิ่มสไลด์เปล่าไปยังงานนำเสนอที่ส่วนท้ายของคอลเลกชันสไลด์เนื้อหาโดยเรียกเมธอด AddEmptySlide ที่เปิดเผยโดยออบเจกต์ ISlideCollection
- ทำงานบางอย่างกับสไลด์เปล่าที่เพิ่งเพิ่ม
- สุดท้าย เขียนไฟล์งานนำเสนอโดยใช้ออบเจกต์ [Presentation](https://reference.aspose.com/slides/th/net/aspose.slides/presentation)

## **คำถามที่พบบ่อย**

**ฉันสามารถแทรกสไลด์ใหม่ในตำแหน่งเฉพาะได้หรือไม่ ไม่ใช่แค่ที่ส่วนท้าย?**

ใช่ ไลบรารีสนับสนุนคอลเลกชันสไลด์และการดำเนินการ [insert](https://reference.aspose.com/slides/th/net/aspose.slides/slidecollection/insertemptyslide/)/[clone](https://reference.aspose.com/slides/th/net/aspose.slides/slidecollection/insertclone/) ดังนั้นคุณสามารถเพิ่มสไลด์ที่ดัชนีที่ต้องการได้ ไม่ใช่แค่ที่ส่วนท้าย

**ธีม/สไตล์จะคงเดิ​มเมื่อติดตั้งสไลด์โดยอิงเลเอาต์หรือไม่?**

ใช่ เลเอาต์จะสืบทอดการจัดรูปแบบจากมาสเตอร์ของมัน และสไลด์ใหม่จะสืบทอดจากเลเอาต์ที่เลือกและมาสเตอร์ที่เชื่อมโยงกับมัน

**สไลด์ใดที่อยู่ในงานนำเสนอ "เปล่า" ใหม่ก่อนทำการเพิ่มสไลด์?**

งานนำเสนอที่สร้างใหม่จะมีสไลด์เปล่าอยู่อย่างหนึ่งที่ดัชนีศูนย์อยู่แล้ว สิ่งนี้สำคัญเมื่อคำนวณดัชนีการแทรก

**ฉันจะเลือกเลเอาต์ "ที่เหมาะสม" สำหรับสไลด์ใหม่อย่างไรหากมาสเตอร์มีหลายตัวเลือก?**

โดยทั่วไปให้เลือก [LayoutSlide](https://reference.aspose.com/slides/th/net/aspose.slides/layoutslide/) ที่ตรงกับโครงสร้างที่ต้องการ ([Title and Content, Two Content, ฯลฯ](https://reference.aspose.com/slides/th/net/aspose.slides/slidelayouttype/)) หากไม่มีเลเอาต์ดังกล่าว คุณสามารถ [add it to the master](/slides/th/net/slide-layout/) แล้วใช้งานได้
