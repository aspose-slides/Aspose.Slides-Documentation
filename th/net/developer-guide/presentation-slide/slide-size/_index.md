## **บทนำ**

Aspose.Slides for .NET มีเครื่องมือที่ครอบคลุมสำหรับปรับขนาดสไลด์และอัตราส่วนภาพในงานนำเสนอ PowerPoint ซึ่งสำคัญทั้งสำหรับการพิมพ์และการแสดงบนหน้าจอ  

ขนาดสไลด์และอัตราส่วนที่นิยม:

- **Standard (4:3 Aspect Ratio)**: เหมาะสำหรับหน้าจอและอุปกรณ์รุ่นเก่า.  
- **Widescreen (16:9 Aspect Ratio)**: แนะนำสำหรับโปรเจกเตอร์และจอแสดงผลสมัยใหม่.  

รักษาความสอดคล้องตลอดการนำเสนอของคุณโดยใช้ขนาดสไลด์และอัตราส่วนภาพเดียวกันสำหรับสไลด์ทั้งหมด เพื่อผลลัพธ์ที่ดีที่สุด ให้ตั้งค่าขนาดสไลด์ตั้งแต่เริ่มสร้างงานนำเสนอเพื่อหลีกเลี่ยงปัญหา.  

{{% alert color="primary" %}} 
โดยค่าเริ่มต้น งานนำเสนอที่สร้างด้วย Aspose.Slides จะใช้รูปแบบอัตราส่วน 4:3 มาตรฐาน. 
{{% /alert %}}

## **วิธีการเปลี่ยนขนาดสไลด์ในงานนำเสนอ**

ตัวอย่างนี้แสดงวิธีการเปลี่ยนขนาดสไลด์ของงานนำเสนอโดยใช้ Aspose.Slides ใน C#:  

```csharp
using (Presentation pres = new Presentation("presentation-4x3.pptx"))
{
    pres.SlideSize.SetSize(SlideSizeType.OnScreen16x9, SlideSizeScaleType.DoNotScale);
    pres.Save("presentation-16x9.pptx", SaveFormat.Pptx);
}
```

## **ระบุขนาดสไลด์ที่กำหนดเอง**

การปรับขนาดสไลด์ให้ตรงกับความต้องการของคุณ เช่น สำหรับรูปแบบกระดาษหรือสเปคหน้าจอที่เฉพาะเจาะจง สามารถเป็นประโยชน์ได้ ต่อไปนี้คือวิธีตั้งค่าขนาดสไลด์ที่กำหนดเองด้วย Aspose.Slides สำหรับ .NET:  

```csharp
using (Presentation pres = new Presentation("presentation.pptx"))
{
    pres.SlideSize.SetSize(780, 540, SlideSizeScaleType.DoNotScale); // ขนาดกระดาษ A4
    pres.Save("presentation-a4.pptx", SaveFormat.Pptx);
}
```

## **จัดการเนื้อหาสไลด์หลังจากการปรับขนาด**

หลังจากปรับขนาดแล้ว เนื้อหาสไลด์อาจเกิดการบิดเบี้ยว คุณสามารถควบคุมวิธีที่ Aspose.Slides จัดการการปรับขนาดนี้ได้:  

- **`DoNotScale`**: รักษาวัตถุไว้ในขนาดต้นฉบับเพื่อหลีกเลี่ยงการย่อขยาย.  
- **`EnsureFit`**: ย่อขนาดวัตถุให้พอดีกับสไลด์ที่เล็กลง เพื่อป้องกันการสูญเสียเนื้อหา.  
- **`Maximize`**: ขยายวัตถุให้เหมาะกับสไลด์ที่ใหญ่ขึ้นเพื่อความสอดคล้องทางสุนทรียะ.  

ตัวอย่างการใช้การตั้งค่า `Maximize` เพื่อปรับขนาดสไลด์:  

```csharp
using (Presentation pres = new Presentation("presentation.pptx"))
{
   pres.SlideSize.SetSize(SlideSizeType.Ledger, SlideSizeScaleType.Maximize);
}
```

## **คำถามที่พบบ่อย**

**ฉันสามารถตั้งค่าขนาดสไลด์ที่กำหนดเองโดยใช้หน่วยอื่นนอกจากนิ้ว (เช่น จุดหรือมิลลิเมตร) ได้หรือไม่?**  

ได้ Aspose.Slides ใช้หน่วยจุดภายในระบบ โดย 1 จุดเท่ากับ 1/72 นิ้ว คุณสามารถแปลงหน่วยใดก็ได้ (เช่น มิลลิเมตรหรือเซนติเมตร) ไปเป็นจุดและใช้ค่าที่แปลงแล้วกำหนดความกว้างและความสูงของสไลด์ได้  

**การใช้ขนาดสไลด์ที่กำหนดเองขนาดใหญ่มากจะส่งผลต่อประสิทธิภาพและการใช้หน่วยความจำขณะเรนเดอร์หรือไม่?**  

ได้ ขนาดสไลด์ที่ใหญ่ขึ้น (เป็นจุด) พร้อมกับสเกลการเรนเดอร์ที่สูงจะทำให้การใช้หน่วยความจำเพิ่มขึ้นและเวลาการประมวลผลยาวนานขึ้น ควรเลือกขนาดสไลด์ที่เหมาะสมและปรับสเกลการเรนเดอร์เฉพาะเมื่อจำเป็นเพื่อให้ได้คุณภาพผลลัพธ์ตามต้องการ  

**ฉันสามารถกำหนดขนาดสไลด์ที่ไม่เป็นมาตรฐานหนึ่งขนาดแล้วรวมสไลด์จากงานนำเสนอที่มีขนาดต่างกันได้หรือไม่?**  

คุณไม่สามารถ [รวมงานนำเสนอ](/slides/th/net/merge-presentation/) ได้หากมีขนาดสไลด์ต่างกัน — ก่อนอื่นให้ปรับขนาดงานนำเสนอหนึ่งให้ตรงกับอีกงานหนึ่ง เมื่อเปลี่ยนขนาดสไลด์คุณสามารถเลือกวิธีที่เนื้อหาเดิมจะถูกจัดการผ่านตัวเลือก [SlideSizeScaleType](https://reference.aspose.com/slides/th/net/aspose.slides/slidesizescaletype/) หลังจากปรับขนาดให้ตรงกันแล้ว คุณสามารถรวมสไลด์โดยคงรูปแบบเดิมไว้ได้  

**ฉันสามารถสร้างภาพย่อยสำหรับรูปร่างแต่ละอันหรือพื้นที่เฉพาะของสไลด์ได้หรือไม่ และภาพเหล่านั้นจะเคารพขนาดสไลด์ใหม่หรือไม่?**  

ได้ Aspose.Slides สามารถเรนเดอร์ภาพย่อยสำหรับ [สไลด์ทั้งหมด](https://reference.aspose.com/slides/th/net/aspose.slides/slide/getimage/) รวมถึง [รูปร่างที่เลือก](https://reference.aspose.com/slides/th/net/aspose.slides/shape/getimage/) ด้วย ภาพที่ได้จะสะท้อนขนาดสไลด์และอัตราส่วนภาพในปัจจุบัน ปรับกรอบและเรขาคณิตให้สอดคล้องกัน
