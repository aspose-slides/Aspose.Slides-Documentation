## **ยินดีต้อนรับสู่ Aspose.Slides for .NET!**

Aspose.Slides เป็น Microsoft PowerPoint® management API ที่ช่วยให้แอปพลิเคชันสามารถอ่าน เขียน สร้าง และแก้ไขไฟล์นำเสนอ PowerPoint ได้โดยไม่ต้องใช้ Microsoft PowerPoint® ไลบรารีรองรับการทำงานกับการนำเสนอ สไลด์ รูปร่าง ตาราง ข้อความ แอนิเมชัน เสียง วิดีโอ และองค์ประกอบการนำเสนออื่น ๆ

ด้วย Aspose.Slides นักพัฒนาสามารถเปิดไฟล์นำเสนอจากแหล่งต่าง ๆ บันทึกในหลายรูปแบบของ PowerPoint ส่งออกไฟล์นำเสนอเป็นรูปแบบเช่น PDF, TIFF, XPS, SWF, HTML และ SVG และบันทึกสไลด์เป็นรูปภาพ API ยังให้คุณสมบัติในการจัดการคุณสมบัติของเอกสาร การประยุกต์ใช้การป้องกัน การแสดงตัวอย่างสไลด์ การแก้ไขเนื้อหาในสไลด์ การทำงานกับบันทึกย่อของสไลด์ และการจัดการองค์ประกอบการนำเสนอขั้นสูงเช่นวัตถุ OLE ที่ฝังอยู่

## **ทำไมคุณควรใช้ Aspose.Slides for .NET**

- Aspose.Slides for .NET เป็นคอมโพเนนต์แห่งแรกและเดียวที่ให้ฟังก์ชันที่จำเป็นในการจัดการเอกสาร PowerPoint®  
- Aspose.Slides for .NET มีฟีเจอร์สำคัญหลากหลาย เช่น การจัดการข้อความ รูปร่าง ตารางและแอนิเมชัน การเพิ่มเสียงและวิดีโอในสไลด์ การแสดงตัวอย่างสไลด์ การส่งออกสไลด์เป็น SVG, PDF และรูปแบบอื่น ๆ รวมถึงความสามารถอื่น ๆ อีกมากมาย  
- Aspose.Slides for .NET ไม่เพียงเปิดไฟล์ PowerPoint จากแหล่งต่าง ๆ แต่ยังให้คุณบันทึกการนำเสนอของคุณได้หลายวิธี เช่น สามารถบันทึกเป็น PPT หรือบันทึกสไลด์เป็นรูปภาพได้  

- Aspose.Slides for .NET ให้คุณใช้คุณสมบัติเ� almost all ที่คุณอาจพบหรือไม่พบใน Microsoft PowerPoint ด้วยความหลากหลายของ Aspose.Slides for .NET นอกจากตารางและรูปทรงหลายประเภทแล้ว คุณยังสามารถใช้มันเพื่อเพิ่มกรอบหลายประเภท—กรอบข้อความ กรอบเสียง และกรอบวิดีโอ—ไปยังสไลด์ของคุณได้  
- Aspose.Slides for .NET พัฒนาโดยใช้ Managed C# คุณสามารถใช้ Aspose.Slides for .NET กับภาษา .NET ใดก็ได้ เช่น C#, VB.NET, J# เป็นต้น Aspose.Slides for .NET ยังสามารถใช้กับแอปพลิเคชันประเภทใดก็ได้—เว็บหรือแอปพลิเคชัน Windows  
- Aspose.Slides for .NET ยังมีเดโมและตัวอย่างการทำงานที่ครบถ้วนเพื่อช่วยให้นักพัฒนาทำความเข้าใจ API ได้ดียิ่งขึ้น  

## **คุณสมบัติของ Aspose.Slides**
คุณสมบัติของ Aspose.Slides for .NET แบ่งเป็นกลุ่มต่อไปนี้:

1. คุณสมบัติการนำเสนอ
1. คุณสมบัติสไลด์
1. คุณสมบัติกราฟิกและมัลติมีเดีย
1. คุณสมบัติตาราง
1. คุณสมบัติข้อความและย่อหน้า
1. คุณสมบัติขั้นสูง

### **คุณสมบัติการนำเสนอ**
- เปิดไฟล์ Microsoft PowerPoint จากแหล่งต่าง ๆ  
- บันทึกไฟล์นำเสนอในรูปแบบ Microsoft PowerPoint ต่าง ๆ  
- สร้างไฟล์นำเสนอใหม่จากศูนย์  
- แปลงไฟล์นำเสนอเป็น PDF โดยไม่ต้องใช้คอมโพเนนต์อื่น  
- อ่านหรือแก้ไขคุณสมบัติของเอกสารนำเสนอ  
- ประยุกต์ใช้การป้องกันบนไฟล์นำเสนอ  
- ส่งออกไฟล์นำเสนอเป็น PDF  
- ส่งออกไฟล์นำเสนอเป็น TIFF  
- ส่งออกไฟล์นำเสนอเป็น XPS  
- ส่งออกไฟล์นำเสนอเป็น SWF  
- ส่งออกไฟล์นำเสนอเป็น PDF Note / Tiff Notes  
- ส่งออกไฟล์นำเสนอเป็น HTML  

### **คุณสมบัติสไลด์**
- สร้างสไลด์ใหม่จากศูนย์และเพิ่มไปยังไฟล์นำเสนอ  
- เข้าถึงสไลด์ใดก็ได้ตามตำแหน่งในไฟล์นำเสนอ  
- คัดลอกหรือโคลนสไลด์ไปยังไฟล์เดียวกันหรือไฟล์อื่น  
- แก้ไขและลบสไลด์  
- คัดลอกมาสเตอร์พร้อมสไลด์โดยอัตโนมัติ  
- แสดงตัวอย่างสไลด์  
- รับหรือกำหนดชื่ิอสไลด์  
- บันทึกสไลด์เป็นภาพ  
- ส่งออกสไลด์เป็นรูปแบบ SVG  
- ทำงานกับมาสเตอร์หลายตัวในไฟล์นำเสนอ  
- เติมพื้นหลังสไลด์ด้วยสไตล์ที่รองรับใดก็ได้  
- เพิ่มหรือจัดการบันทึกย่อของสไลด์  
- ตั้งค่ามาสเตอร์และสไตล์ของสไลด์ใหม่แบบไดนามิก  
- เพิ่มหรือแก้ไขการเปลี่ยนภาพสไลด์โชว์  
- เพิ่มการเปลี่ยนภาพแบบ Morph  

### **คุณสมบัติกราฟิกและมัลติมีเดีย**
- สร้างหรือจัดการรูปร่างเช่น เส้น, สี่เหลี่ยม, วงรี, โพลีไลน์  
- จัดการสไตล์เส้นในรูปร่าง  
- รับข้อความแทนที่ของรูปร่าง  
- หมุนหรือพลิกรูปร่าง  
- เติมรูปร่างในสไตล์ต่าง ๆ เช่น โซลิด, เกรเดียนท์, แพทเทิร์น, เทกซ์เจอร์และภาพ  
- ใช้แพทเทิร์นที่กำหนดไว้ล่วงหน้า 48 แบบและเทกซ์เจอร์ 24 แบบ  
- กำหนดลิงก์ให้กับรูปร่าง  
- อ่าน, แก้ไข และเพิ่มเอฟเฟกต์แอนิเมชันใหม่ให้กับสไลด์และรูปร่าง  
- เพิ่มรูปภาพใหม่ไปยังไฟล์นำเสนอและจัดการรูปภาพที่มีอยู่  
- เพิ่มรูปภาพแบบหัวข้อสัญลักษณ์ใหม่และจัดการรูปภาพที่มีอยู่  
- เพิ่มกรอบเสียงใหม่และจัดการกรอบเสียงที่มีอยู่  
- เพิ่มกรอบวิดีโอใหม่และจัดการกรอบวิดีโอที่มีอยู่  

### **คุณสมบัติตาราง**
- สร้างตารางจากศูนย์และเพิ่มไปยังสไลด์  
- ปรับขนาดและย้ายตำแหน่งตาราง  
- เข้าถึงเซลล์ใดก็ได้ในตาราง  
- เปลี่ยนรูปแบบและขอบของเซลล์ใดก็ได้  
- ปรับขนาดคอลัมน์และแถว  
- เพิ่มและลบคอลัมน์และแถว  
- กำหนดความกว้างและสีของขอบสำหรับตารางทั้งหมด  

### **คุณสมบัติข้อความและย่อหน้า**
- รับและกำหนดข้อความในตัวจัดข้อความและกรอบข้อความ  
- รับประเภทของตัวจัดข้อความ  
- เปลี่ยนรูปแบบข้อความในกรอบข้อความ  
- เพิ่มฟอนต์ใหม่หรือจัดการฟอนต์ที่ใช้ในไฟล์นำเสนอ  
- เปลี่ยนข้อความและการจัดรูปแบบข้อความ  
- สร้างหรือจัดการลิงก์ในข้อความ  
- รองรับแท็บและแท็บที่มีประสิทธิภาพ  
- เพิ่มกรอบข้อความใหม่ให้กับรูปร่างเกือบทุกประเภท  
- เข้าถึงและแก้ไขระยะขอบและการห่อหุ้มห่างข้อความของกรอบข้อความ  
- เข้าถึงและแก้ไขการเยื้องของย่อหน้า  

### **คุณสมบัติขั้นสูง**
- เข้าถึงวัตถุ OLE ที่ฝังอยู่ในไฟล์นำเสนอ  
- ล็อคไฟล์นำเสนอ

## **การประกาศ**
คอมโพเนนต์ Aspose .NET ทั้งหมดต้องการชุดสิทธิ์ Full Trust เหตุผล: คอมโพเนนต์ Aspose .NET จำเป็นต้องมีสิทธิ์ระดับสูงเพื่อเข้าถึงการตั้งค่าจดหมายทะเบียน, ไฟล์ระบบ, และไดเรกทอรีเสมือนเพื่อดำเนินการบางอย่าง เช่น การแยกฟอนต์และงานคล้ายกัน นอกจากนี้คอมโพเนนต์ Aspose .NET ยังอิงจากคลาสหลักของระบบ .NET ซึ่งหลายกรณีก็ต้องการชุดสิทธิ์ Full Trust  

ผู้ให้บริการอินเทอร์เน็ตที่โฮสต์หลายแอปพลิเคชันจากบริษัทต่าง ๆ ส่วนใหญ่บังคับใช้ระดับความปลอดภัย Medium Trust ในกรณีของ NET 2.0 ระดับความปลอดภัยนี้มีข้อจำกัดดังนี้:

- OleDbPermission ไม่พร้อมใช้งาน หมายความว่าคุณไม่สามารถใช้ตัวให้บริการข้อมูล OLE DB แบบจัดการของ ADO.NET เพื่อเข้าถึงฐานข้อมูลได้  
- EventLogPermission ไม่พร้อมใช้งาน หมายความว่าคุณไม่สามารถเข้าถึงบันทึกเหตุการณ์ของ Windows ได้  
- ReflectionPermission ไม่พร้อมใช้งาน หมายความว่าคุณไม่สามารถใช้ Reflection ได้  
- RegistryPermission ไม่พร้อมใช้งาน หมายความว่าคุณไม่สามารถเข้าถึงรีจิสทรีได้  
- WebPermission ถูกจำกัด หมายความว่าแอปพลิเคชันของคุณสื่อสารได้เฉพาะกับที่อยู่หรือช่วงที่กำหนดในองค์ประกอบ <trust>  
- FileIOPermission ถูกจำกัด หมายความว่าคุณสามารถเข้าถึงไฟล์ได้เฉพาะในโครงสร้างไดเรกทอรีเสมือนของแอปพลิเคชัน  

ด้วยเหตุผลข้างต้น คอมโพเนนต์ Aspose .NET สามารถใช้ได้บนเซิร์ฟเวอร์ที่ให้ชุดสิทธิ์ Full Trust เท่านั้น  

## **FAQ**

**รองรับระดับการปฏิบัติตาม PDF สำหรับการเก็บรักษาและการเข้าถึง (PDF/A และ PDF/UA) หรือไม่?**

ใช่ คุณสามารถบันทึกเป็น PDF พร้อม PDF/A-2a/2b/2u, PDF/A-3a/3b และ PDF/UA ได้โดยกำหนด [PDF export options](https://reference.aspose.com/slides/th/net/aspose.slides.export/pdfoptions/)

**มีกลไกการทดแทนฟอนต์และการสนับสนุนฟอนต์ที่กำหนดเองเพื่อให้การแสดงผลถูกต้องหรือไม่?**

ใช่ ไลบรารีอนุญาตให้คุณ [load custom fonts](/slides/th/net/custom-font/) และ [define fallback rules](/slides/th/net/fallback-font/) เพื่อให้ glyph ที่ขาดหายถูกแทนที่อย่างแม่นยำระหว่างการแปลงและการแสดงผล

**ฉันสามารถตรวจจับไฟล์ที่มีการป้องกันด้วยรหัสผ่านโดยไม่ต้องเปิดเต็มไฟล์ได้หรือไม่?**

ใช่ คุณสามารถ [inspect a presentation](/slides/th/net/examine-presentation/) เพื่อตรวจสอบว่าต้องใช้รหัสผ่านก่อนโหลดเอกสารเต็มหรือไม่

**ต้องใช้ Microsoft PowerPoint สำหรับการประมวลผลและแปลงไฟล์หรือไม่?**

ไม่ Aspose.Slides เป็นเอนจินอิสระ; ไม่จำเป็นต้องติดตั้ง PowerPoint บนเซิร์ฟเวอร์หรือเครื่องทำงาน

**ปลอดภัยหรือไม่ที่จะประมวลผลการนำเสนอจากหลายเธรดพร้อมกัน?**

ใช่ คุณสามารถประมวลผลเอกสารหลายไฟล์พร้อมกันในเธรดแยกต่างหาก; เพียงหลีกเลี่ยงการใช้อินสแตนซ์การนำเสนอเดียวกันในหลายเธรดพร้อมกัน [across threads](/slides/th/net/multithreading/)

**การแมโครถูกเก็บรักษาไว้และฉันสามารถจัดการ VBA ในไฟล์ PPTM/PPSM ได้หรือไม่?**

ใช่ การนำเสนอที่มีแมโคร [are supported](/slides/th/net/presentation-via-vba/), และคุณสามารถ [inspect and manage VBA projects](https://reference.aspose.com/slides/th/net/aspose.slides.vba/) ในไฟล์เหล่านั้นได้

**ฉันสามารถแปลง PDF หรือ HTML กลับเป็นสไลด์ PowerPoint ได้หรือไม่?**

ใช่ คุณสามารถ [import PDF pages or HTML content](/slides/th/net/import-presentation/) เพื่อสร้างหรือเติมสไลด์ในไฟล์นำเสนอ

**การส่งออกเป็น XPS รองรับหรือไม่ และสามารถควบคุมคุณภาพและเนื้อหาของเอาต์พุต XPS ได้หรือไม่?**

ใช่ การ [Export to XPS](/slides/th/net/convert-powerpoint-to-xps/) มีให้ใช้งานและ [save options](https://reference.aspose.com/slides/th/net/aspose.slides.export/xpsoptions/) ใหคุณปรับคุณภาพและเนื้อหาที่รวมอยู่ได้

**ฉันสามารถแปลงสไลด์เป็นภาพและควบคุมคุณภาพของเอาต์พุตได้หรือไม่?**

ใช่ สไลด์สามารถ [rendered to PNG, JPEG, GIF, BMP, TIFF](/slides/th/net/convert-powerpoint-to-png/) และรูปแบบอื่น ๆ พร้อมการควบคุมขนาดและคุณภาพอย่างละเอียด

**รองรับสูตรคณิตศาสตร์ (MathML/MathText) และสามารถส่งออกได้หรือไม่?**

ใช่ คุณสามารถ [create and edit mathematical text](/slides/th/net/powerpoint-math-equations/) และ [export equations](/slides/th/net/exporting-math-equations/) รวมถึงส่งออกเป็น MathML ได้

**ไลบรารีจัดการการเข้ารหัสและรหัสผ่าน (เปิด, ตั้งค่า, ลบ) อย่างไร?**

[It supports](/slides/th/net/password-protected-presentation/) การเปิดไฟล์นำเสนอที่เข้ารหัส, การตั้งหรือการลบรหัสผ่านสำหรับการเปิดและการแก้ไข, รวมถึงการตรวจสอบสถานะการป้องกันของไฟล์.
