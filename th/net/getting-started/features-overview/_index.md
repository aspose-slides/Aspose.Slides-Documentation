## **แพลตฟอร์มที่รองรับ**
Aspose.Slides for .NET รองรับแพลตฟอร์มการพัฒนาและการปรับใช้ที่เป็นที่นิยมที่สุด ไปดูรายละเอียดต่อไปนี้:

|**คุณลักษณะ**|**รายละเอียด**|
| :- | :- |
|ASP.NET Web Applications|ใช้ Aspose.Slides for .NET เพื่อสร้างแอปพลิเคชันเว็บ ASP.NET ที่มุ่งเป้าไปที่ .NET Framework เวอร์ชัน 2.0 ถึง 4.6.2|
|Web Services|ใช้ Aspose.Slides for .NET เพื่อปรับใช้ Web Services|
|WinForms Applications|Aspose.Slides for .NET ยังสามารถใช้พัฒนา Windows Forms Applications|

|**คุณลักษณะ**|**รายละเอียด**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/th/net/ppt-vs-pptx/)|Aspose.Slides for .NET ให้การประมวลผลที่เร็วที่สุดสำหรับรูปแบบเอกสารการนำเสนอนี้|
|[PPT to PPTX conversion](/slides/th/net/convert-ppt-to-pptx/)|Aspose.Slides for .NET รองรับการแปลงจาก PPT เป็น PPTX|
|[Portable Document Format (PDF)](/slides/th/net/convert-powerpoint-ppt-and-pptx-to-pdf/)|คุณสามารถส่งออกไฟล์ฟอร์แมตที่รองรับทั้งหมดเป็นเอกสาร Adobe Portable Document Format (PDF) ด้วยวิธีเดียว|
|[XML Parser Specification (XPS)](https://docs.aspose.com/slides/th/net/convert-powerpoint-to-xps/)|คุณสามารถส่งออกไฟล์ฟอร์แมตที่รองรับทั้งหมดเป็นเอกสาร XML Parser Specification (XPS) ด้วยวิธีเดียว|
|[Tagged Image File Format (TIFF)](/slides/th/net/convert-powerpoint-to-tiff/)|คุณสามารถส่งออกไฟล์การนำเสนอที่รองรับทั้งหมดเป็น Tagged Image File Format (TIFF)|

|**คุณลักษณะ**|**รายละเอียด**|
| :- | :- |

## **คำถามที่พบบ่อย**

**Do I need to install Microsoft PowerPoint on the server/PC for the library to work?**  
No. PowerPoint is not required; Aspose.Slides is a standalone engine for creating, editing, converting, and rendering presentations.

**How does multithreading work? Can processing be parallelized?**  
It is safe to process different documents in different threads; the same [presentation](https://reference.aspose.com/slides/th/net/aspose.slides/presentation/) object must not be used by [multiple threads](/slides/th/net/multithreading/) at the same time.

**Are file passwords and encryption supported?**  
Yes. [You can](/slides/th/net/password-protected-presentation/) open encrypted presentations, set or remove an open and write password, and check the protection status.

**Do I need to care about font packages in Linux containers?**  
Yes. It is recommended to install common font packages and/or explicitly [specify font directories](/slides/th/net/custom-font/) in your application to avoid unexpected substitutions.

**Are there limitations in the evaluation version?**  
In [evaluation mode](/slides/th/net/licensing/), a watermark is added to the output and certain limitations apply; a [30-day temporary license](https://purchase.aspose.com/temporary-license/) is available for full-feature testing.

**Is importing external formats into a presentation (PDF/HTML → PPTX) supported?**  
Yes. You can add [PDF pages and HTML content](/slides/th/net/import-presentation/) to a presentation, turning them into slides.
