## **ความปลอดภัยใน Aspose.Slides**

Aspose ใช้แนวปฏิบัติที่ดีที่สุดเมื่อพัฒนาผลิตภัณฑ์ของตน  

* Aspose.Slides for .NET ใช้สำหรับจัดการและแปลงงานนำเสนอในไฟล์รูปแบบอื่น ไม่ได้รันสคริปต์ในงานนำเสนอ Aspose.Slides เพียงทำการวิเคราะห์โครงสร้างของงานนำเสนอและอนุญาตให้โค้ดของผู้ใช้ปลายทางจัดการโมเดลวัตถุได้อย่างสะดวก  

* Aspose.Slides ทำหน้าที่เป็นไลบรารีในบริบทของการวิเคราะห์และตีความเอกสารโดยไม่มีการดำเนินการโค้ดจากระยะไกล ผลิตภัณฑ์ทั้งหมดของ Aspose จะทำงานบนเครื่องของคุณ ไม่ได้ส่งข้อมูลใด ๆ ไปยัง Aspose ข้อยกเว้นเดียวคือเมื่อใช้ [metered license](https://purchase.aspose.com/faqs/licensing/metered) และแม้กระนั้น (หากคุณใช้ใบอนุญาตแบบจ่ายตามการใช้งาน) จะมีการประมวลผลเฉพาะข้อมูลการใช้ API ของคุณเท่านั้น  

* ส่วนประกอบของ Aspose ทำงานในบริบทผู้ใช้เดียวกับแอปพลิเคชันทั่วไป ดังนั้นส่วนประกอบของ Aspose จึงไม่เป็นความเสี่ยงต่อทรัพยากรระบบที่สำคัญ อีกทั้งเมื่อเอกสารถูกเปิดโดยส่วนประกอบของ Aspose แมโครจะไม่ถูกเรียกทำงานโดยอัตโนมัติ  

* ความเสี่ยงที่มีอยู่หรือเกี่ยวข้องกับชุด Microsoft Office ไม่ได้ส่งผลต่อส่วนประกอบของ Aspose ดังนั้นผลิตภัณฑ์ของ Aspose จึงปลอดภัยสูง  

## **คำถามที่พบบ่อย**

**ระบบใดที่ใช้เพื่อตรวจสอบช่องโหว่ในโค้ดของ Aspose?**

เราทำการวิเคราะห์โค้ดแบบสถิตสำหรับแต่ละรุ่นของ Aspose.Slides เราสามารถให้รายงานความปลอดภัยที่พิสูจน์ว่าโค้ด Aspose.Slides ผ่าน OWASP Top 10  

**ระบบใดที่คุณใช้เพื่อตรวจสอบช่องโหว่ในแพ็กเกจภายนอก?**

เราไม่ได้ใช้ระบบเพื่อตรวจสอบช่องโหว่ในแพ็กเกจภายนอก เนื่องจาก Aspose.Slides ไม่ได้มีหรือใช้แพ็กเกจภายนอกใด ๆ Aspose.Slides พึ่งพาเฉพาะ .NET framework เท่านั้น
