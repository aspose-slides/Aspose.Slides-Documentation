라이브러리는 파일을 포함하는 콘텐츠 구조입니다. 라이브러리에는 문서, 스프레드시트 및 프레젠테이션을 비롯한 하나의 파일 유형만 포함될 수도 있고, 여러 가지 다른 유형을 포함할 수도 있습니다. SharePoint 라이브러리는 사용자가 문서를 업로드하고 액세스할 수 있는 단일 위치를 생성합니다. 문서는 SharePoint 사이트에서 사용하는 SQL Server 데이터베이스에 바이너리 형식으로 저장됩니다. 문서는 하이퍼링크 목록 형태로 다른 SharePoint 사용자에게 제공됩니다.
