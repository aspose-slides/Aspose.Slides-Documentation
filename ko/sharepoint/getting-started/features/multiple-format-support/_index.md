{{% alert color="primary" %}} 
Aspose.Slides for SharePoint를 사용하면 SharePoint 문서 라이브러리 내에서 여러 인기 오피스 문서 형식 간에 문서를 변환할 수 있습니다. 변환은 높은 충실도와 정밀도로 수행됩니다. 
{{% /alert %}} 
## **지원 입력 형식**
Aspose.Slides for SharePoint는 다음 입력 형식을 지원합니다: 

- PPT – Microsoft PowerPoint Presentation 97 - 2003
- PPS – Microsoft PowerPoint SlideShow 97 - 2003
- POT – Microsoft PowerPoint Template 97 - 2003
- PPTX – Office Open XML Presentation
- PPSX – Office Open XML SlideShow
- POTX – Office Open XML Template

{{% alert color="primary" %}} 
문서를 생성하기 위해 Aspose.Slides for SharePoint는 Aspose의 유일한 PowerPoint 문서 처리 구성 요소인 [Aspose.Slides for .NET](http://www.aspose.com/categories/.net-components/aspose.slides-for-.net/default.aspx)의 내장 버전에 의존합니다. 
{{% /alert %}}
## **지원 출력 형식**
다음 출력 형식이 Aspose.Slides for SharePoint에서 지원됩니다: 

- PDF – 포터블 문서 형식
- TIFF – 이미지 패키지
- XPS – XML 종이 사양
