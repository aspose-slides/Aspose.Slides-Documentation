{{% alert color="primary" %}} 

무료 Aspose.Words for SharePoint 평가판을 활용하십시오: 시간 제한이 없으며 무료 기술 지원이 제공됩니다. 구매 전에 제품을 평가하도록 권장합니다. 이를 통해 제품이 원하는 대로 작동한다는 것을 확신할 수 있습니다.

{{% /alert %}} 

평가 버전과 유료 버전 모두 동일한 다운로드를 사용합니다. 다운로드 페이지에서 [Aspose.Slides for SharePoint 다운로드](http://www.aspose.com/community/files/73/sharepoint-components/aspose.slides-for-sharepoint/default.aspx)를 간단히 다운로드하고, [install it]() 하면 기본적으로 평가 모드에서 작동합니다. 

평가 모드에서는 내보낸 문서의 각 슬라이드에 평가 워터마크를 삽입합니다. 라이선스를 구매한 경우, 설치된 Aspose.Slides for SharePoint 평가 사본 위에 라이선스 솔루션을 설치하면 라이선스 모드에서 작동합니다.
