## **Aspose.Slides for SharePoint에 오신 것을 환영합니다!**

Aspose.Slides for SharePoint는 Microsoft SharePoint 사이트 내에서 PowerPoint® 문서를 변환할 수 있게 해주는 유연한 솔루션입니다.

## **제품 개요**

Aspose.Slides for SharePoint는 다양한 PowerPoint 문서 형식을 지원합니다:

- PPT – Microsoft PowerPoint 프레젠테이션 97 - 2003
- PPS – Microsoft PowerPoint 슬라이드쇼 97 - 2003
- POT – Microsoft PowerPoint 템플릿 97 - 2003
- PPTX – Office Open XML 프레젠테이션
- PPSX – Office Open XML 슬라이드쇼
- POTX – Office Open XML 템플릿

Aspose.Slides for SharePoint는 다음 제품과 함께 사용하도록 설계되었습니다:

- Windows SharePoint Services 3.0 (WSS)
- Microsoft Office SharePoint Server 2007 (MOSS) Standard
- Microsoft Office SharePoint Server 2007 (MOSS) Enterprise
- Microsoft Office SharePoint Server 2013
- Microsoft Office SharePoint Server 2019

위 제품에 이미 존재하는 요구 사항 외에 다른 시스템 요구 사항은 없습니다.

**SharePoint 문서 라이브러리에서 문서를 변환하려면 Aspose.Slides for SharePoint를 사용하십시오**
