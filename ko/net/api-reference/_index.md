{{% alert color="primary" %}} 

Aspose.Slides for .NET은 Microsoft PowerPoint® 등 사용 없이 .NET 애플리케이션이 프레젠테이션 문서를 읽고 쓸 수 있게 하는 클래스 라이브러리입니다. Aspose.Slides for .NET은 텍스트, 도형, 표 및 애니메이션을 관리하고, 슬라이드에 오디오와 비디오를 추가하며, 슬라이드를 미리 보기하고, SVG, PDF 형식 등으로 슬라이드를 내보낼 수 있습니다.

최신 API 레퍼런스는 [Aspose.Slides for .NET API Reference](https://reference.aspose.com/slides/ko/net)에서 찾을 수 있습니다.

{{% /alert %}}
