## **소개**
Microsoft Visual Studio는 Microsoft에서 제공하는 통합 개발 환경(IDE)입니다. 이는 Microsoft Windows용 컴퓨터 프로그램뿐만 아니라 웹 사이트, 웹 애플리케이션 및 웹 서비스를 개발하는 데 사용됩니다. Visual Studio는 Windows API, Windows Forms, Windows Presentation Foundation, Windows Store, Microsoft Silverlight와 같은 Microsoft 소프트웨어 개발 플랫폼을 사용합니다. 네이티브 코드와 관리 코드 모두를 생성할 수 있습니다.

### **Aspose.Slides for .NET**
Aspose.Slides for .NET는 Microsoft PowerPoint를 사용하지 않고도 .NET 응용 프로그램이 PowerPoint 문서를 읽고, 쓰고, 조작할 수 있게 하는 고유한 PowerPoint® 관리 API입니다.

Aspose.Slides for .NET는 .NET PowerPoint API 중 최초이자 유일하게 PowerPoint 문서를 자체 응용 프로그램에서 관리할 수 있는 기능을 제공합니다. 모든 Aspose .NET 구성 요소와 마찬가지로 Aspose.Slides는 관리형 C#로 작성되었으며, 매우 저렴하고 빠릅니다.

자세한 내용은 [Aspose.Slides for .NET API](http://www.aspose.com/.net/powerpoint-component.aspx) 를 참고하십시오.

### **Visual Studio를 사용한 Aspose.Slides for .NET 예제 탐색**
Aspose Visual Studio 플러그인 - **Explore Aspose.Slides for .NET Examples**는 Aspose.Slides for .NET API 예제를 빠르게 다운로드하고 탐색할 수 있는 훌륭한 도구입니다. 최신 예제 프로젝트를 선택, 다운로드 및 열기 위한 매우 간단한 옵션을 제공함으로써 Visual Studio를 떠나지 않고도 많은 시간과 노력을 절약합니다.

This version provides the following features:

- Visual Studio 2010, Visual Studio 2012 및 Visual Studio 2013을 지원합니다.
- Visual Studio의 **File > Aspose** 메뉴에서 쉽게 실행할 수 있습니다.
- 시간을 절약하고 학습 곡선을 줄여줍니다.
- Github.com에 호스팅된 최신 Aspose.Slides for .NET API 예제를 선택하고 열 수 있습니다.
- **C#** 또는 **VB.NET** 예제 프로젝트를 열 수 있습니다.
- 선택한 프로젝트가 자동으로 Visual Studio에서 열리며, 모든 필요한 참조가 설정되어 즉시 실행하고 탐색할 수 있는 환경을 제공합니다.
- **NuGet**에서 최신 API 라이브러리를, **GitHub**에서 예제를 자동으로 다운로드합니다.

## **설치**

- 다운로드한 msi 파일을 더블 클릭합니다.
- 단계에 따라 진행하고, 각 화면에서 Next를 눌러 기본 설정으로 설치하십시오.

## **지원, 확장 및 기여**
### **지원**
Aspose는 초창기부터 고객에게 좋은 제품만 제공하는 것으로는 충분하지 않다는 것을 알았습니다. 좋은 서비스를 제공해야 했습니다. 우리는 개발자이며, 기술적인 문제나 소프트웨어의 사소한 버그가 작업을 방해할 때 얼마나 답답한지 이해합니다. 우리는 문제를 해결하기 위해 존재합니다.

이 때문에 무료 지원을 제공합니다. 제품을 구매했든 평가판을 사용하든 관계없이 모든 사용자는 우리의 전적인 관심과 존중을 받을 자격이 있습니다.

다음 플랫폼 중 하나를 사용하여 Aspose Visual Studio 플러그인과 관련된 문제나 제안을 등록할 수 있습니다

- [Github ](https://github.com/asposeSlides/Aspose.Slides-for-.NET/issues)

### **확장 및 기여**
**Explore Aspose.Slides for .NET Examples** for Visual Studio는 오픈 소스이며, 아래에 나열된 주요 소셜 코딩 웹사이트에서 소스 코드를 확인할 수 있습니다. 개발자는 소스 코드를 다운로드하고 필요에 따라 기능을 확장하도록 권장됩니다.
#### **소스 코드**
다음 위치 중 하나에서 최신 소스 코드를 받을 수 있습니다

- [Github ](https://github.com/aspose-slides/Aspose.Slides-for-.NET)

#### **소스 코드를 구성하는 방법**
소스 코드를 열고 확장하려면 다음을 설치해야 합니다

- Visual Studio 2010

다음 간단한 단계를 따라 시작하십시오

1. 소스 코드를 다운로드/클론합니다.
2. Visual Studio 2010을 열고 **File** > **Open Project**를 선택합니다.
3. 다운로드한 최신 소스 코드를 찾아 **AsposeVisualStudioPlugin.sln**을 엽니다.

#### **소스 코드 개요**
솔루션에는 두 개의 프로젝트가 포함됩니다

- **AsposeVisualStudioPlugin** – Visual Studio와의 통합, Aspose API 및 예제의 선택 및 자동 다운로드를 포함합니다.
- **Installer** – 플러그인용 msi 설치 파일을 생성하는 Wix 설치 프로젝트.
