## **OpenXML이란?**
OpenXML SDK는 Microsoft에서 제공하는 .NET SDK로 Microsoft Office 2007 이상 파일 형식을 처리합니다. 이러한 파일 형식은 기본적으로 zip 패키지이며, Office용 Open XML SDK는 패키지 내 Open XML 패키지와 기본 Open XML 스키마 요소를 조작하는 작업을 단순화합니다. Open XML SDK의 클래스는 개발자가 Open XML 패키지에서 수행하는 일반적인 작업을 캡슐화하여 몇 줄의 코드로 복잡한 작업을 수행할 수 있게 해줍니다.

Open XML SDK 2.5를 설치한 후, Visual Studio에서 기존 프로젝트나 애플리케이션을 열거나 새 프로젝트/응용 프로그램을 만듭니다. 그런 다음 프로젝트나 애플리케이션에 다음 구성 요소에 대한 참조를 추가합니다:

- DocumentFormat.OpenXml
- WindowsBase
## **Aspose.Slides for .NET**
Aspose.Slides for .NET은 Microsoft PowerPoint를 사용하지 않고도 .NET 애플리케이션이 PowerPoint 문서를 읽고, 쓰고, 조작할 수 있게 해주는 고유한 PowerPoint® 관리 구성 요소입니다.

Aspose.Slides for .NET은 자체 애플리케이션 내에서 PowerPoint 문서를 관리할 수 있는 기능을 제공하는 최초이자 유일한 .NET 구성 요소입니다. 모든 Aspose .NET 구성 요소와 마찬가지로 Aspose.Slides는 관리되는 C#로 작성되었으며, 매우 저렴하고 번개처럼 빠릅니다.

## **소개**
파일 형식 라이브러리를 찾는 개발자는 시장에 나와 있는 라이브러리가 제공하는 기능을 살펴볼 필요가 있습니다. 옵션을 선택할 때 주로 두 가지 요소가 고려됩니다:

- 특징
- 라이브러리 사용에 필요한 노력

.NET 개발자로서 파일을 빠르고 쉽고 가볍게 처리할 수 있는 파일 형식 라이브러리를 찾고 있다면 VSTO(Visual Studio tool for Office)와 Aspose를 비교해볼 필요가 있습니다. 이 경우, 본 프로젝트는 두 라이브러리의 기능과 코드를 비교하는 데 도움이 될 것입니다.
## **시스템 요구 사항 및 지원 플랫폼**
### **시스템 요구 사항**
Visual Studio Tools for Office의 다음 버전 중 하나 이상:

- Microsoft Visual Studio 2005 Tools for the Microsoft Office System
- Microsoft Visual Studio 2005 Tools for the 2007 Microsoft Office System
- Visual Studio 2008 Professional Edition
- Visual Studio 2008 Team Suite Edition
- 하나의 Microsoft Office 버전:
- Microsoft Office Professional 2003 SP1
- 2007 Microsoft Office system
### **지원 운영 체제**
- Windows 2000; Windows Server 2003; Windows Vista; Windows XP
### **지원되는 .NET Framework 버전**
- 오직 .NET Framework 2.0 이상만 지원됩니다.
