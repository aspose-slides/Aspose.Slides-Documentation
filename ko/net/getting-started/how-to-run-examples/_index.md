## **소프트웨어 요구 사항**
예제를 다운로드하고 실행하기 전에 설정이 다음 요구 사항을 충족하는지 확인하십시오: 

- Visual Studio 2010 이상.
- Visual Studio에 NuGet Package Manager가 설치되어 있어야 합니다. 최신 NuGet API 버전이 Visual Studio에 설치되어 있는지 확인하십시오. 

NuGet 패키지 관리자를 설치하는 방법에 대해서는 다음 페이지를 참조하십시오: https://docs.microsoft.com/en-us/nuget/install-nuget-client-tools

1. **Tools** > **Options** > **NuGet Package Manager**를 차례대로 선택합니다.

1. **NuGet Package Manager**를 확장(두 번 클릭)하고 **Package Sources**를 선택합니다. 

1. nuget.org 매개변수가 선택되어 있는지 확인합니다. 

   예제 프로젝트는 NuGet 자동 패키지 복원 기능을 사용하므로 활성화된 인터넷 연결이 필요합니다. 

   예제를 실행하려는 머신에 인터넷 연결이 없는 경우, [설치](https://docs.aspose.com/slides/ko/net/installation/)를 확인하고 Aspose.Slides.dll에 대한 참조를 수동으로 추가하십시오.
## **GitHub에서 Aspose.Slides 다운로드**
모든 Aspose.Slides for .NET 예제는 [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-.NET)에 호스팅됩니다.

선호하는 GitHub 클라이언트를 사용해 리포지토리를 복제하거나 ZIP 파일을 [여기](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/master.zip)에서 다운로드할 수 있습니다.

1. ZIP 파일을 다운로드한 경우, 파일 내용을 컴퓨터의 폴더에 압축 해제해야 합니다. 

   모든 예제는 **Examples** 폴더에 저장됩니다.

   C# Visual Studio 솔루션 파일이 포함되어 있습니다. 프로젝트는 Visual Studio 2013에서 생성되었지만, 솔루션 파일은 Visual Studio 2010 SP1 이상에서 호환됩니다.

2. Visual Studio에서 솔루션 파일을 열고 프로젝트를 빌드합니다.

   첫 실행 시, 종속성이 NuGet을 통해 자동으로 다운로드됩니다.

**Examples** 루트 폴더의 **Data** 폴더에는 C# 예제에서 사용되는 입력 파일이 들어 있습니다. 예제 프로젝트와 함께 **Data** 폴더를 다운로드해야 합니다.

3. RunExamples.cs 파일을 엽니다. 모든 예제가 여기서 호출됩니다.

4. 프로젝트 내에서 실행하려는 예제의 주석을 해제합니다.

설정이나 예제 실행에 문제가 있으면 포럼을 통해 자유롭게 문의하십시오.
## **기여**
예제를 추가하거나 개선하여 프로젝트에 기여할 수 있습니다. 리포지토리의 모든 예제와 쇼케이스 프로젝트는 오픈 소스이며, 여러분(및 다른 사람)도 애플리케이션에서 자유롭게 사용할 수 있습니다.

기여하려면 리포지토리를 포크하고, 소스 코드를 편집한 뒤 풀 요청을 생성하십시오. 변경 사항을 검토하고 유용하다고 판단되면 리포지토리에 추가하겠습니다.
