## **지원 플랫폼**
Aspose.Slides for .NET은 가장 인기 있는 개발 및 배포 플랫폼을 지원합니다. 자세한 내용은 아래를 확인하십시오:

|**기능**|**설명**|
| :- | :- |
|ASP.NET 웹 애플리케이션|Aspose.Slides for .NET을 사용하여 .NET Framework 2.0부터 4.6.2까지를 대상으로 하는 ASP.NET 웹 애플리케이션을 구축합니다.|
|Web Services|Aspose.Slides for .NET을 사용하여 웹 서비스를 배포합니다.|
|WinForms 애플리케이션|Aspose.Slides for .NET을 사용하여 Windows Forms 애플리케이션을 개발할 수도 있습니다.|

## **파일 형식 및 변환**
Aspose.Slides for .NET은 대부분의 PowerPoint 문서 형식을 지원합니다. 또한 조직에서 널리 사용하고 교환하는 인기 있는 형식으로 내보낼 수 있습니다. 자세한 내용은 아래를 확인하십시오:

|**기능**|**설명**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/ko/net/ppt-vs-pptx/)|Aspose.Slides for .NET은 이 프레젠테이션 문서 형식에 대해 가장 빠른 처리를 제공합니다.|
|[PPT to PPTX 변환](/slides/ko/net/convert-ppt-to-pptx/)|Aspose.Slides for .NET은 PPT를 PPTX로 변환하는 것을 지원합니다.|
|[Portable Document Format (PDF)](/slides/ko/net/convert-powerpoint-ppt-and-pptx-to-pdf/)|단일 메서드로 모든 지원 파일 형식을 Adobe Portable Document Format(PDF) 문서로 내보낼 수 있습니다.|
|[XML Parser Specification (XPS)](https://docs.aspose.com/slides/ko/net/convert-powerpoint-to-xps/)|단일 메서드로 모든 지원 파일 형식을 XML Parser Specification (XPS) 문서로 내보낼 수 있습니다.|
|[Tagged Image File Format (TIFF)](/slides/ko/net/convert-powerpoint-to-tiff/)|지원되는 모든 프레젠테이션 파일 형식을 Tagged Image File Format (TIFF)으로 내보낼 수 있습니다.|
|[PPTX to HTML 변환](/slides/ko/net/convert-powerpoint-ppt-and-pptx-to-html/)|Aspose.Slides for .NET은 PresentationEx를 HTML 형식으로 변환하는 것을 지원합니다.|

## **렌더링 및 인쇄**
Aspose.Slides for .NET은 프레젠테이션 문서의 슬라이드를 다양한 그래픽 형식으로 고품질 렌더링을 지원합니다. 자세한 내용은 아래를 확인하십시오:

|**기능**|**설명**|
| :- | :- |
|.NET 지원 이미지 형식|Aspose.Slides for .NET을 사용하면 TIFF, PNG, BMP, JPEG, GIF 및 메타 파일과 같은 .NET 지원 그래픽 형식으로 프레젠테이션 슬라이드와 슬라이드의 이미지를 렌더링할 수 있습니다.|
|SVG 형식|Aspose.Slides for .NET은 또한 프레젠테이션 슬라이드를 Scalable Vector Graphics(SVG) 형식으로 내보내는 내장 메서드를 제공합니다.|
|프레젠테이션 인쇄|최신 버전의 Aspose.Slides for .NET은 다양한 옵션을 갖춘 내장 인쇄 메서드를 제공합니다.|

## **콘텐츠 기능**
Aspose.Slides for .NET을 사용하면 프레젠테이션 문서의 거의 모든 항목이나 콘텐츠에 접근, 수정, 생성할 수 있습니다. 자세한 내용은 아래를 확인하십시오:

|**기능**|**설명**|
| :- | :- |
|마스터 슬라이드|마스터 슬라이드는 일반 슬라이드의 레이아웃을 정의합니다. Aspose.Slides for .NET을 사용하면 프레젠테이션 문서의 마스터 슬라이드에 접근하고 수정할 수 있습니다.|
|일반 슬라이드|Aspose.Slides for .NET을 사용하면 다양한 유형의 새 슬라이드를 생성할 수 있으며, 프레젠테이션의 기존 슬라이드에 접근하고 수정할 수도 있습니다.|
|클론 / 복사 슬라이드|Aspose.Slides for .NET이 제공하는 내장 메서드를 사용하면 프레젠테이션 내에서 기존 슬라이드를 복제하거나 복사할 수 있습니다. 복제 및 복사된 슬라이드를 한 프레젠테이션에서 다른 프레젠테이션으로 사용할 수도 있습니다. 슬라이드는 마스터 슬라이드에서 레이아웃을 상속받기 때문에, 내장 복제 메서드는 복제 시 자동으로 마스터를 복사합니다.|
|슬라이드 섹션 관리|프레젠테이션 내부에서 슬라이드를 서로 다른 섹션으로 구성하는 메서드|
|플레이스홀더 및 텍스트 홀더|슬라이드에서 플레이스홀더와 텍스트 홀더에 접근할 수 있습니다. 또한 적절한 메서드를 사용하여 텍스트 홀더가 있는 슬라이드를 처음부터 만들 수 있습니다.|
|헤더 및 푸터|Aspose.Slides for .NET은 슬라이드에서 헤더/푸터를 관리하는 기능을 제공합니다.|
|슬라이드 노트|Aspose.Slides for .NET을 사용하면 슬라이드와 연결된 노트에 접근하고 수정하며, 새 노트를 추가할 수도 있습니다.|
|도형 찾기|도형과 연결된 대체 텍스트를 사용하여 슬라이드에서 특정 도형을 찾을 수 있습니다.|
|배경|Aspose.Slides for .NET을 사용하면 프레젠테이션의 마스터 또는 일반 슬라이드와 연결된 배경을 작업할 수 있습니다.|
|텍스트 상자|텍스트 상자를 처음부터 생성할 수 있습니다. 기존 텍스트 상자에 접근할 수 있으며, 원래 텍스트 형식을 유지하면서 텍스트를 수정할 수도 있습니다.|

## **FAQ**

**라이브러리를 사용하려면 서버/PC에 Microsoft PowerPoint를 설치해야 합니까?**

아니오. PowerPoint는 필요하지 않으며, Aspose.Slides는 프레젠테이션을 생성, 편집, 변환 및 렌더링하기 위한 독립 실행형 엔진입니다.

**멀티스레딩은 어떻게 작동합니까? 처리를 병렬화할 수 있나요?**

다른 스레드에서 서로 다른 문서를 처리하는 것은 안전합니다; 동일한 [presentation](https://reference.aspose.com/slides/ko/net/aspose.slides/presentation/) 객체를 [multiple threads](/slides/ko/net/multithreading/)와 동시에 사용해서는 안 됩니다.

**파일 비밀번호 및 암호화가 지원됩니까?**

예. [가능합니다](/slides/ko/net/password-protected-presentation/) 암호화된 프레젠테이션을 열고, 열기 및 쓰기 비밀번호를 설정하거나 제거하며, 보호 상태를 확인할 수 있습니다.

**Linux 컨테이너에서 폰트 패키지를 신경써야 합니까?**

예. 일반적인 폰트 패키지를 설치하거나 애플리케이션에서 폰트 디렉터리를 명시적으로 [지정](/slides/ko/net/custom-font/)하는 것이 예기치 않은 대체를 방지하는 데 권장됩니다.

**평가 버전에는 제한 사항이 있습니까?**

[평가 모드](/slides/ko/net/licensing/)에서는 출력에 워터마크가 추가되고 일부 제한이 적용됩니다; 전체 기능 테스트를 위한 [30일 임시 라이선스](https://purchase.aspose.com/temporary-license/)를 사용할 수 있습니다.

**외부 형식(PDF/HTML → PPTX)을 프레젠테이션으로 가져오는 것이 지원됩니까?**

예. [PDF 페이지 및 HTML 콘텐츠](/slides/ko/net/import-presentation/)를 프레젠테이션에 추가하여 슬라이드로 변환할 수 있습니다.
