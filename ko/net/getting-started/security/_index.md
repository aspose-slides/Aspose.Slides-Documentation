## **Aspose.Slides 보안**

Aspose는 제품 개발 시 모범 사례를 적용합니다. 

* Aspose.Slides for .NET은 다양한 형식의 파일에서 프레젠테이션을 조작하고 변환하는 데 사용됩니다. 프레젠테이션 내 스크립트를 실행하지 않습니다. Aspose.Slides는 프레젠테이션 구조를 단순히 구문 분석하고, 최종 사용자의 코드가 객체 모델을 편리하게 조작할 수 있도록 합니다. 
* Aspose.Slides는 원격 코드를 실행하지 않고 문서를 구문 분석하고 해석하는 라이브러리로 동작합니다. 모든 Aspose 제품은 사용자의 컴퓨터에서 실행됩니다. Aspose로 데이터를 전송하지 않습니다. 예외는 [metered license](https://purchase.aspose.com/faqs/licensing/metered)가 적용되는 경우이며, 이 경우에도 (metered 라이선스를 사용하는 경우) API 사용량 정보만 처리됩니다.

* Aspose 구성 요소는 일반 애플리케이션과 동일한 사용자 컨텍스트에서 실행됩니다. 따라서 Aspose 구성 요소가 중요한 시스템 리소스에 위험을 초래하지 않습니다. 또한 Aspose 구성 요소가 문서를 열 때 매크로가 자동으로 실행되지 않습니다. 

* Microsoft Office 제품군과 관련된 위험은 Aspose 구성 요소에 적용되지 않으며, 따라서 Aspose 제품은 매우 안전합니다.

## **FAQ**

**Aspose 코드의 취약점을 모니터링하는 데 어떤 시스템을 사용합니까?**

우리는 Aspose.Slides의 각 릴리스마다 정적 코드 분석을 수행합니다. Aspose.Slides 코드가 OWASP Top 10을 통과함을 증명하는 보안 보고서를 제공할 수 있습니다.

**외부 패키지의 취약점을 모니터링하기 위해 어떤 시스템을 사용합니까?**

Aspose.Slides는 외부 패키지를 포함하거나 사용하지 않으므로 외부 패키지의 취약점을 확인하는 시스템을 사용하지 않습니다. Aspose.Slides는 오직 .NET 프레임워크 자체에만 의존합니다.
