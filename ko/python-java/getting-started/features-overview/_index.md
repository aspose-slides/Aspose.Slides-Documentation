## **지원 플랫폼**
Aspose.Slides for Python via Java는 플랫폼에 독립적인 API입니다. Python 3.7 이상이 설치된 모든 플랫폼(Windows, Linux, macOS 등)에서 사용할 수 있습니다. JDK 8 이상이 또 다른 요구 사항입니다.

## **파일 형식 및 변환**
Aspose.Slides for Python via Java는 대부분의 PowerPoint 문서 형식을 지원합니다. 또한 조직에서 사용하고 공유하는 인기 있는 파일 형식으로 내보낼 수 있습니다. 자세한 내용은 아래를 확인하십시오:

|**기능**|**설명**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Python via Java는 이 프레젠테이션 문서 형식에 대해 가장 빠른 처리를 제공합니다.|
|PPT to PPTX conversion|Aspose.Slides for Python via Java는 PPT를 PPTX로 변환하는 기능을 지원합니다.|
|Portable Document Format (PDF)|단일 메서드로 지원되는 모든 파일 형식을 Adobe Portable Document Format(PDF) 문서로 내보낼 수 있습니다.|
|XML Parser Specification (XPS)|단일 메서드로 지원되는 모든 파일 형식을 XML Parser Specification(XPS) 문서로 내보낼 수 있습니다.|
|Tagged Image File Format (TIFF)|지원되는 모든 프레젠테이션 파일 형식을 Tagged Image File Format(TIFF)으로 내보낼 수 있습니다.|
|PPTX To HTML Conversion|Aspose.Slides for Python via Java는 PresentationEx를 HTML 형식으로 변환하는 기능을 지원합니다.|

## **렌더링 및 인쇄**
Aspose.Slides for Python via Java는 프레젠테이션 슬라이드를 다양한 그래픽 형식으로 고충실도 렌더링하는 것을 지원합니다. 자세한 내용은 아래를 확인하십시오:

|**기능**|**설명**|
| :- | :- |
|Java Supported Image Formats|Aspose.Slides for Python via Java를 사용하면 TIFF, PNG, BMP, JPEG, GIF와 같은 Java가 지원하는 모든 그래픽 형식으로 프레젠테이션 슬라이드와 슬라이드의 이미지를 렌더링할 수 있습니다.|
|SVG Format|Aspose.Slides for Python via Java는 Scalable Vector Graphics(SVG) 형식으로 프레젠테이션 슬라이드를 내보내는 내장 메서드를 제공합니다.|
|Presentation Printing|최신 버전의 Aspose.Slides for Python via Java는 다양한 옵션을 갖춘 내장 인쇄 메서드를 제공합니다.|

## **콘텐츠 기능**
Aspose.Slides for Python via Java를 사용하면 프레젠테이션 문서의 거의 모든 항목 또는 내용을 액세스, 수정 또는 생성할 수 있습니다. 자세한 내용은 아래를 확인하십시오:

|**기능**|**설명**|
| :- | :- |
|Master Slides|마스터 슬라이드는 일반 슬라이드의 레이아웃을 정의합니다. Aspose.Slides for Python via Java를 사용하면 프레젠테이션 문서의 마스터 슬라이드에 액세스하고 수정할 수 있습니다.|
|Normal Slides|Aspose.Slides for Python via Java를 사용하면 다양한 유형의 새 슬라이드를 생성할 수 있으며, 프레젠테이션의 기존 슬라이드에 액세스하고 수정할 수도 있습니다.|
|Cloning / Copying Slides|Aspose.Slides for Python via Java가 제공하는 내장 메서드를 사용하면 프레젠테이션 내 기존 슬라이드를 복제하거나 복사할 수 있습니다. 복제 및 복사된 슬라이드를 한 프레젠테이션에서 다른 프레젠테이션으로도 사용할 수 있습니다. 슬라이드는 마스터 슬라이드에서 레이아웃을 상속하므로 내장 복제 메서드는 복제 시 마스터를 자동으로 복사합니다.|
|Managing Slides sections|프레젠테이션 내부에서 슬라이드를 다양한 섹션으로 조직하는 메서드입니다.|
|Place Holders and Text Holders|슬라이드의 자리 표시자와 텍스트 보관자에 액세스할 수 있습니다. 또한 적절한 메서드를 사용하여 텍스트 보관자가 포함된 슬라이드를 처음부터 생성할 수 있습니다.|
|Header and Footers|Aspose.Slides for Python via Java는 슬라이드의 머리글/바닥글 처리 기능을 제공합니다.|
|Notes in Slides|Aspose.Slides for Python via Java를 사용하면 슬라이드에 연결된 메모에 액세스하고 수정하며 새 메모를 추가할 수 있습니다.|
|Finding a Shape|형태에 연결된 대체 텍스트를 사용하여 슬라이드에서 특정 형태를 찾을 수 있습니다.|
|Backgrounds|Aspose.Slides for Python via Java는 마스터 슬라이드 또는 일반 슬라이드와 연결된 배경을 작업할 수 있게 해줍니다.|
|Text Boxes|텍스트 상자를 처음부터 만들 수 있으며, 기존 텍스트 상자에 액세스하고 원본 텍스트 형식을 유지하면서 텍스트를 수정할 수 있습니다.|
|Rectangle Shapes|Aspose.Slides for Python via Java를 사용하여 사각형 형태를 만들거나 수정할 수 있습니다.|
|Poly Line Shapes|Aspose.Slides for Python via Java를 사용하여 폴리라인 형태를 만들거나 수정할 수 있습니다.|
|Ellipse Shapes|Aspose.Slides for Python via Java를 사용하여 타원 형태를 만들거나 수정할 수 있습니다.|
|Group Shapes|Aspose.Slides for Python via Java는 그룹 형태를 지원합니다.|
|Auto Shapes|Aspose.Slides for Python via Java는 자동 형태를 지원합니다.|
|SmartArt|Aspose.Slides for Python via Java는 MS PowerPoint의 SmartArt 형태를 지원합니다.|
|Charts|Aspose.Slides for Python via Java는 PowerPoint의 MSO 차트를 지원합니다.|
|Shapes Serialization|Aspose.Slides for Python via Java는 많은 형태를 지원합니다. 지원되지 않는 형태가 있을 경우, 기존 슬라이드에서 해당 형태를 직렬화하는 방법을 사용하여 직렬화할 수 있습니다. 이렇게 하면 필요에 따라 해당 형태를 계속 사용할 수 있습니다.|
|Picture Frames|Aspose.Slides for Python via Java를 사용하여 그림 프레임 안의 이미지를 관리할 수 있습니다.|
|Audio Frames|Aspose.Slides for Python via Java를 사용하여 오디오 프레임에 오디오 파일을 연결하거나 삽입할 수 있습니다.|
|Video Frames|비디오 프레임에서 비디오 파일을 처리할 수 있습니다. Aspose.Slides for Python via Java는 연결 및 삽입된 비디오를 모두 지원합니다.|
|OLE Frame|Aspose.Slides for Python via Java를 사용하여 OLE 프레임에 OLE 객체를 관리할 수 있습니다.|
|Tables|Aspose.Slides for Python via Java는 슬라이드 내 표를 지원합니다.|
|ActiveX Controls|ActiveX 컨트롤 지원|
|VBA Macros|프레젠테이션 내 VBA 매크로 관리 지원|
|Text Frame|형태와 연결된 텍스트 프레임을 통해 해당 형태의 텍스트에 액세스할 수 있습니다.|
|Text Scanning|내장 스캔 메서드를 사용하여 프레젠테이션 수준 또는 슬라이드 수준에서 텍스트를 스캔할 수 있습니다.|
|Animations|형태에 애니메이션을 적용할 수 있습니다.|
|Slide Shows|Aspose.Slides for Python via Java는 슬라이드 쇼와 슬라이드 전환을 지원합니다.|

## **포맷팅 기능**
Aspose.Slides for Python via Java를 사용하면 프레젠테이션 슬라이드의 텍스트와 형태를 포맷팅할 수 있습니다. 자세한 내용은 아래를 확인하십시오:

|**기능**|**설명**|
| :- | :- |
|Text Formatting|<p>Aspose.Slides for Python via Java에서는 형태와 연결된 텍스트 프레임을 통해 텍스트를 관리할 수 있습니다. 따라서 텍스트 프레임에 연결된 단락과 구문을 사용하여 텍스트를 포맷팅할 수 있습니다. 이러한 텍스트 요소는 Aspose.Slides for Python via Java를 통해 포맷팅됩니다.</p><p>- 글꼴 종류</p><p>- 글꼴 크기</p><p>- 글꼴 색상</p><p>- 글꼴 음영</p><p>- 단락 정렬</p><p>- 단락 글머리표</p><p>- 단락 방향</p>|
|Shape Formatting|<p>Aspose.Slides for Python via Java에서 슬라이드의 기본 요소는 형태입니다. 다음과 같은 속성으로 형태 요소를 포맷팅할 수 있습니다:</p><p>- 위치</p><p>- 크기</p><p>- 선</p><p>- 채우기(패턴, 그라디언트, 단색 포함)</p><p>- 텍스트</p><p>- 이미지</p>|
