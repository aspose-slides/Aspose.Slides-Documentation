Aspose.Slides를 평가용으로 손쉽게 다운로드할 수 있습니다. 평가 패키지는 구매 패키지와 동일합니다. 라이선스를 적용하는 몇 줄의 코드를 추가하면 평가 버전이 바로 라이선스가 적용된 버전으로 전환됩니다. 

라이선스가 지정되지 않은 Aspose.Slides의 평가 버전은 전체 제품 기능을 제공하지만, 열거나 저장할 때 문서 상단에 평가 워터마크를 삽입합니다. 또한 프레젠테이션 슬라이드에서 텍스트를 추출할 때 한 슬라이드로 제한됩니다.
