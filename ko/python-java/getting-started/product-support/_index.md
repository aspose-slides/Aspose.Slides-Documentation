## **Technical support**
{{% alert color="primary" %}}

Aspose는 모든 제품에 대해 무제한 무료 기술 지원을 제공합니다. 이 지원은 평가 패키지를 포함한 모든 사용자에게 제공됩니다.

{{% /alert %}} 

Aspose.Slides for Python via Java에 대한 도움이 필요할 때는 다음을 참고하세요:

- 지원의 주요 경로는 [Aspose.Forums](https://forum.aspose.com/c/slides/ko/11)입니다. Aspose.Slides 포럼에 질문을 게시하면 몇 시간 내에 답변을 받을 수 있습니다. Aspose.Slides 지원 팀이 포럼에 게시된 질문에 직접 답변합니다.
- Aspose는 전화로 기술 지원을 제공하지 않습니다. 전화 지원은 영업 및 구매 관련 문의에만 제공됩니다.
- 포럼에서 답변을 기다릴 때는 인내심을 가지고 시차를 고려해 주세요.

Aspose.Slides for Python via Java에서 문제가 발생하면 다음 지침을 따라 가장 효율적으로 해결하십시오:

- 문제를 보고하기 전에 최신 Aspose.Slides for Python via Java 버전을 사용하고 있는지 확인하세요. 아마도 겪은 문제가 업데이트에서 이미 해결됐을 수 있습니다.  
  최신 버전 정보를 보려면 [Aspose.Slides download](https://releases.aspose.com/slides/ko/python-java/)를 확인하십시오.
- 문제를 보고하기 전에 포럼, 이 문서 및 API 레퍼런스를 확인해 보세요. 질문에 대한 답변이 이미 제공되었을 수 있습니다.
- 문제를 보고할 때는 원본 문서와 (가능하면) 문제와 관련된 코드 조각을 포함하십시오. 여러 파일을 첨부해야 할 경우 ZIP 패키지에 넣어 주세요. Aspose.Forums에 파일을 첨부해도 **오직** 본인과 Aspose 개발자만 파일에 접근할 수 있으므로 안심하셔도 됩니다.
- 특정 이슈당 하나의 스레드를 생성하여 사용하십시오. 다른 문제가 발생하면 별도의 스레드를 만들어 보고해야 합니다.
