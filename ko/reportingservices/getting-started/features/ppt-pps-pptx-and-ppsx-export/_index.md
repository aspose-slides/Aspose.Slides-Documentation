{{% alert color="primary" %}} 

Microsoft SQL Server Reporting Services는 Microsoft PowerPoint 프레젠테이션으로 보고서를 내보내는 기본 기능이 없지만, 서버에 Aspose.Slides for Reporting Services를 설치하면 두 가지 추가 내보내기 형식을 사용할 수 있습니다: 

- Microsoft PowerPoint Presentation (PPT)
- Microsoft PowerPoint SlideShow (PPS)
- Microsoft PowerPoint 2007 Presentation (PPTX)
- Microsoft PowerPoint 2007 SlideShow (PPSX)

{{% /alert %}} 

이 형식의 문서를 만들기 위해 Aspose.Slides for Reporting Services는 Aspose의 시장 선도 프레젠테이션 처리 구성 요소인 [Aspose.Slides for .NET](http://www.aspose.com/Products/Aspose.Slides/)의 내장 버전에 의존합니다. Microsoft PowerPoint는 문서를 생성하는 데 사용되지 않습니다. 


PPT 및 PPS 복사본은 Aspose.Slides가 제공하는 기본 파일 형식에 대한 높은 지원 덕분에 서로 동일하게 표시됩니다.
