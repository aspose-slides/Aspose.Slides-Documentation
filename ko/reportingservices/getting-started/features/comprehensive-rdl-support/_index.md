{{% alert color="primary" %}} 

Aspose.Slides for Reporting Services는 RDL 사양을 모두 완벽하게 지원합니다. 이는 두 가지 좋은 점을 의미합니다: 

- 기존 보고서를 재설계할 필요가 없습니다. 기존 RDL 보고서를 Microsoft PowerPoint 프레젠테이션으로 내보낼 수 있으며, RDL 디자인 그대로 표시됩니다.
- 특정 보고서 디자이너를 사용할 필요가 없습니다. 어떤 RDL 보고서 디자이너든 사용할 수 있으며, 보고서는 설계한 그대로 내보내집니다.

{{% /alert %}} 

Aspose.Slides for Reporting Services는 다음 RDL 요소를 지원합니다: 

- 페이지, 머리글, 바닥글
- 텍스트 상자
- 이미지
- 하위 보고서
- 차트
- 목록
- 표
- 매트릭스
- 스타일
- 사각형

**머리글, 바닥글, 이미지, 하위 보고서, 표, 텍스트 상자 및 사각형이 포함된 보고서를 Microsoft PowerPoint(PPT) 프레젠테이션으로 내보낸 예시**
