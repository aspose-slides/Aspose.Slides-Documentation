{{% alert color="primary" %}}
Aspose.Slides for Reporting Services는 Power BI 지원을 구현했습니다.
{{% /alert %}}

페이지 매김 보고서는 페이지에 잘 맞도록 형식이 지정된 보고서입니다. 이러한 보고서는 내보내기 작업 및 인쇄에 이상적입니다. 페이지 매김 보고서는 테이블의 모든 데이터 행을 인쇄하거나 내보내기 위해 필요한 만큼의 페이지를 제공하도록 설계되었습니다.

예를 들어, 보고서 소유자가 지난 12개월 동안 판매된 각 항목에 대한 제품 수량, 판매 가격 및 이익을 확인할 수 있는 보고서를 만들고자 한다면, 페이지 매김 보고서가 필요합니다.

페이지 매김 보고서를 만들 수 있도록 Microsoft는 Power BI 내에서 Power BI Report Builder를 제공합니다. 페이지 매김 보고서를 만든 후 컴퓨터에 저장하면 해당 파일은 .rdl 확장자를 갖게 됩니다.
