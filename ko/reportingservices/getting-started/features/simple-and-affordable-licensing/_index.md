## 라이선스 유형
{{% alert color="primary" %}} 현재 제공되는 라이선스는 두 가지 유형이 있습니다: 서버 구독 사이트 구독 {{% /alert %}}
모든 라이선스에는 새 버전이 출시될 때마다 1년 동안 무료 업그레이드와 1년 동안 무제한 무료 기술 지원이 제공됩니다. 

Aspose.Slides for Reporting Services 평가 버전과 정식 버전은 동일한 다운로드 파일입니다. Aspose.Slides for Reporting Services를 다운로드하여 서버에 설치하면 평가 모드로 작동합니다. 


Aspose.Slides for Reporting Services 평가판은 시간 제한이 없으며 평가 사용자에게 무료 기술 지원이 제공되므로 꼭 활용하시기 바랍니다. 


평가 모드에서는 내보낸 보고서에 워터마크가 삽입됩니다. [라이선스를 구매](https://purchase.aspose.com/buy)하면 라이선스를 적절한 폴더에 복사하면 Aspose.Slides for Reporting Services가 라이선스 모드에서 작동하기 시작합니다. 
## **서버 구독**
서버 구독은 다음과 같은 기능을 제공합니다:

- 각 물리 또는 가상 서버마다 하나의 라이선스가 필요합니다.
- 프로세서 또는 코어 수에 제한이 없습니다.
- Microsoft SQL Server 인스턴스 수에 제한이 없습니다.
- 개발자 수에 제한이 없습니다.
- 1년 동안 무료 신규 릴리스 및 핫픽스 제공.
- 1년 동안 무료 기술 지원.
- 다수 구매 시 할인이 적용됩니다.
## **사이트 구독**
사이트 구독은 다음과 같은 기능을 제공합니다:

- 각 물리적 사이트(캠퍼스)마다 하나의 라이선스가 필요합니다.
- 물리 또는 가상 서버 수에 제한이 없습니다.
- 프로세서 또는 코어 수에 제한이 없습니다.
- Microsoft SQL Server 인스턴스 수에 제한이 없습니다.
- 개발자 수에 제한이 없습니다.
- 1년 동안 무료 신규 릴리스 및 핫픽스 제공.
- 1년 동안 무료 기술 지원.
- 다수 구매 시 할인이 적용됩니다.
