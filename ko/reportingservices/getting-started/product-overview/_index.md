## **Aspose.Slides for Reporting Services에 오신 것을 환영합니다!**

Aspose.Slides for Reporting Services는 시장에서 유일하게 Microsoft SQL Server 2005, 2008, 2012, 2016 및 2017 Reporting Services(32비트 및 64비트)에서 실제 PPT 및 PPS 보고서를 생성할 수 있게 해주는 솔루션입니다. 테이블, 매트릭스, 차트 및 이미지 등 모든 RDL 보고서 기능이 최고의 정밀도로 Microsoft PowerPoint 프레젠테이션으로 변환됩니다.

## **제품 개요**

Microsoft SQL Server Reporting Services에는 보고서를 Microsoft PowerPoint 프레젠테이션으로 내보내는 기본 기능이 없지만, 서버에 Aspose.Slides for Reporting Services를 설치하면 추가 내보내기 형식에 접근할 수 있습니다:

- PPT - Aspose.Slides를 통한 PowerPoint 프레젠테이션
- PPS - Aspose.Slides를 통한 PowerPoint 슬라이드쇼
- PPTX - Aspose.Slides를 통한 PowerPoint 2007 프레젠테이션
- PPSX - Aspose.Slides를 통한 PowerPoint 2007 슬라이드쇼

Aspose.Slides for Reporting Services는 Microsoft PowerPoint를 사용하지 않고 서버에서 프레젠테이션을 생성합니다. Aspose.Slides for Reporting Services는 내부적으로 Aspose.Slides for .NET을 사용합니다 – 서버 측 프레젠테이션 처리용 세계적 수준의 구성 요소입니다.

**Aspose.Slides for Reporting Services는 PPT, PPS, PPTX 또는 PPSX 형식으로 모든 보고서를 내보낼 수 있게 합니다.**
