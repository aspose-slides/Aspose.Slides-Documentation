## **Power BI**
Power BI 서버에 Aspose.Slides for Reporting Services를 동일한 설치 프로세스를 통해 설치할 수 있습니다:

See 

* [MSI 설치 프로그램으로 설치](https://docs.aspose.com/slides/ko/reportingservices/install-with-msi-installer/#installation). 설치 프로그램은 서버에 설치된 Power BI를 자동으로 감지하고, 제품을 설치한 다음 작업 환경에 맞게 구성합니다. 
* [수동 설치](https://docs.aspose.com/slides/ko/reportingservices/install-manually/).

{{% alert title="Note" color="warning" %}} 

**Aspose.Slides for Reporting Services**는 호스트 머신에 **.NET Framework 3.5** 설치가 필요합니다. 

{{% /alert %}}

Aspose.Slides for SSRS는 페이지 매김 보고서(cdl)를 지원합니다.
