평가 버전의 **Aspose.Slides for Reporting Services**를 [its Releases page](https://releases.aspose.com/slides/ko/reportingservices/)에서 다운로드할 수 있습니다. 평가 버전은 제품의 라이선스 버전과 동일한 기능을 제공합니다. 평가 패키지는 구매한 패키지와 동일합니다. 평가 버전은 몇 줄의 코드를 추가하여(라이선스를 적용하기 위해) 라이선스를 적용하면 바로 정식 라이선스로 전환됩니다.

## **Aspose.Slides for Reporting Service의 라이선스**

* 평가 버전은 라이선스를 구매하고 몇 줄의 코드를 추가하면(라이선스를 적용하기 위해) 라이선스가 적용됩니다.
* 라이선스는 제품 이름, 라이선스 대상 개발자 수, 구독 만료일 등과 같은 상세 정보를 포함하는 일반 텍스트 XML 파일입니다.
* 라이선스 파일은 디지털 서명되어 있으므로 수정해서는 안 됩니다. 파일 내용에 실수로 줄바꿈을 추가하는 것조차 파일을 무효화시킵니다.
* 평가 버전에 따른 제한을 피하려면 Aspose.Slides for Reporting Service를 사용하기 전에 라이선스를 설정해야 합니다.

라이선스를 컴퓨터에 다운로드한 후 **C:\Program Files\Microsoft SQL Server\<Instance>\Reporting Services\ReportServer\bin** 폴더에 복사하십시오. 이 폴더는 **Aspose.Slides.ReportingServices.dll**이 설치된 위치입니다.

라이선스가 올바르게 설치되었는지 확인하려면 보고서를 Microsoft PowerPoint 프레젠테이션으로 내보내십시오. 문서에 워터마크가 표시되지 않으면 라이선스가 성공적으로 활성화된 것입니다.

![todo:image_alt_text](license-aspose-slides-for-reporting-services_1.png)

유효한 Aspose.Slides.ReportingServices.lic 파일이 *ReportServer\bin* 폴더에 존재하면 평가 워터마크가 표시되지 않습니다.

**평가 모드**

Aspose.Slides for Reporting Services는 평가 모드(라이선스 없음)에서 작업할 때 워터마크를 삽입합니다.

![todo:image_alt_text](license-aspose-slides-for-reporting-services_2.png)

{{% alert color="primary" %}} 

제한 없이 Aspose.Slides for Reporting Services를 테스트하려면 **30일 임시 라이선스**를 요청할 수 있습니다. 자세한 내용은 [How to Get a Temporary License](https://purchase.aspose.com/temporary-license) 페이지를 참조하십시오.
