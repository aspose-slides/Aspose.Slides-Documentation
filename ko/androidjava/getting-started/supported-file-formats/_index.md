## **개요**

Aspose.Slides는 Microsoft PowerPoint 97부터 Office 365까지, 그리고 Microsoft PowerPoint for Mac을 포함한 프레젠테이션 파일을 지원합니다. 이 문서에서는 라이브러리에서 지원하는 PowerPoint 버전을 나열하고 로드, 저장 또는 둘 다 가능한 파일 형식 표를 제공합니다.

이 문서는 PDF 규정 준수, 글꼴 포함, 비밀번호로 보호된 파일, 사용자 정의 글꼴, 글꼴 대체 및 XPS 내보내기 옵션에 관한 일반적인 질문에도 답변합니다.

## **지원하는 Microsoft PowerPoint 버전**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint for MAC
- Office 365

## **자주 묻는 질문**

**아카이브 및 접근성 표준(PDF/A 및 PDF/UA)을 충족하는 PDF로 프레젠테이션을 저장할 수 있나요?**

예. Aspose.Slides는 PDF 내보내기 옵션의 [compliance](https://reference.aspose.com/slides/ko/androidjava/com.aspose.slides/pdfoptions/#setCompliance-int-) 설정을 통해 PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b 및 PDF/UA와 같은 준수 수준으로 PDF 내보내기를 지원합니다.

**PDF로 내보낼 때 글꼴 포함을 지원하며, 포함되는 항목에 대한 세밀한 제어가 가능한가요?**

예. [PDF export options](https://reference.aspose.com/slides/ko/androidjava/com.aspose.slides/pdfoptions/)를 사용하여 글꼴을 전체 포함하거나 서브셋(사용된 글리프만)으로 포함 여부를 제어하고, 일반 시스템 글꼴 처리 방식을 지정하며, ASCII 텍스트에 대한 동작을 구성할 수 있습니다.

**파일을 실제로 로드하기 전에 비밀번호로 보호되었는지 감지할 수 있나요?**

예. [factory-based inspection API](https://reference.aspose.com/slides/ko/androidjava/com.aspose.slides/presentationfactory/)를 사용하면 프레젠테이션 파일을 완전히 열지 않고도 비밀번호 보호 여부를 조회할 수 있습니다.

**글꼴 대체 메커니즘과 사용자 정의 글꼴 지원이 있나요?**

예. 라이브러리는 사용자 정의 글꼴의 [loading](/slides/ko/androidjava/custom-font/) 및 [embedding](/slides/ko/androidjava/embedded-font/)을 지원하며, 렌더링 및 변환 시 누락된 글리프를 방지하기 위해 글꼴 [fallback rules](/slides/ko/androidjava/fallback-font/)을 제공합니다.

**슬라이드를 XPS로 내보낼 수 있으며, XPS 출력 품질을 조정하는 옵션이 있나요?**

예. [Export to XPS](/slides/ko/androidjava/convert-powerpoint-to-xps/)를 지원하며, 관련 [save options](https://reference.aspose.com/slides/ko/androidjava/com.aspose.slides/xpsoptions/)를 조정하여 XPS 문서의 출력 품질 및 내용을 제어할 수 있습니다.
