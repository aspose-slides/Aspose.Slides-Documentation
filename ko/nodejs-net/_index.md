**Aspose.Slides for Node.js via .NET에 오신 것을 환영합니다**

![Aspose.Slides for Node.js via .NET 제품 로고](aspose_slides-for-nodejs-via-net.png)

Aspose.Slides for Node.js via .NET은 Microsoft PowerPoint®를 사용하지 않고도 응용 프로그램이 PowerPoint® 문서를 읽고 쓸 수 있도록 하는 클래스 라이브러리입니다.

Aspose.Slides for Node.js via .NET은 PowerPoint® 문서를 관리하는 기능을 제공하는 최초이자 유일한 구성 요소입니다.

Aspose.Slides for Node.js via .NET은 텍스트, 도형, 표 및 애니메이션 관리, 슬라이드에 오디오와 비디오 추가, 슬라이드 미리 보기, 슬라이드를 SVG 및 PDF 형식으로 내보내는 등 다양한 핵심 기능을 제공합니다.
