Aspose.Slides for Node.js via .NET은 Microsoft PowerPoint® 등을 사용하지 않고도 Node.js 애플리케이션이 프레젠테이션 문서를 읽고 쓸 수 있게 해주는 클래스 라이브러리입니다. Aspose.Slides for Node.js via .NET을 사용하면 텍스트, 도형, 표 및 애니메이션을 관리하고, 슬라이드에 오디오와 비디오를 추가하고, 슬라이드를 미리 보기하고, 슬라이드를 SVG, PDF 형식으로 내보내는 등 다양한 작업을 수행할 수 있습니다.

Aspose.Slides for Node.js via .NET은 [Aspose.Slides .NET와 동일한 API 참조](https://reference.aspose.com/slides/ko/nodejs-net/)를 사용합니다.

{{% /alert %}}
