## **제품 설명**

Aspose.Slides for Node.js via .NET은 Microsoft PowerPoint®를 사용하지 않고도 다양한 문서 처리 작업을 수행할 수 있도록 하는 크로스 플랫폼 클래스 라이브러리입니다.

### **Aspose.Slides for Node.js via .NET을 사용해야 하는 이유**

- Aspose.Slides for Node.js via .NET은 PowerPoint® 문서를 관리하는 데 필요한 기능을 제공하는 최초이자 유일한 구성 요소입니다.  
- Aspose.Slides for Node.js via .NET은 텍스트, 도형, 표 및 애니메이션 관리, 슬라이드에 오디오 및 비디오 추가, 슬라이드 미리 보기, SVG, PDF 등 다양한 포맷으로 슬라이드 내보내기 등 많은 핵심 기능을 제공합니다.  
- Aspose.Slides for Node.js via .NET은 다양한 소스에서 PowerPoint 파일을 열 수 있을 뿐만 아니라 프레젠테이션을 여러 방식으로 저장할 수 있습니다. 예를 들어 PPT 형식으로 저장하거나 슬라이드를 이미지로 저장할 수 있습니다.  
- Aspose.Slides for Node.js via .NET은 Microsoft PowerPoint에서 찾을 수 있는 거의 모든 기능을 제공합니다. 표와 다양한 도형 외에도 텍스트, 오디오, 비디오 프레임 등 다양한 종류의 프레임을 슬라이드에 추가할 수 있습니다.  

## **Aspose.Slides 기능**

Aspose.Slides for Node.js via .NET의 기능은 다음과 같이 구분됩니다:

- 프레젠테이션 기능
- 슬라이드 기능
- 그래픽 및 멀티미디어 기능
- 표 기능
- 텍스트 및 단락 기능
- 고급 기능

### **프레젠테이션 기능**

- 다양한 소스에서 Microsoft PowerPoint 프레젠테이션 열기
- 다양한 Microsoft PowerPoint 형식으로 프레젠테이션 저장
- 처음부터 프레젠테이션 만들기
- 다른 구성 요소 없이 프레젠테이션을 PDF 문서로 변환
- 프레젠테이션 문서 속성 읽기 또는 수정
- 프레젠테이션에 보호 적용
- 프레젠테이션을 PDF로 내보내기
- 프레젠테이션을 TIFF로 내보내기
- 프레젠테이션을 XPS로 내보내기
- 프레젠테이션을 SWF로 내보내기
- 프레젠테이션을 PDF Note / Tiff Notes로 내보내기
- 프레젠테이션을 HTML로 내보내기

### **슬라이드 기능**

- 처음부터 새 슬라이드 만들고 프레젠테이션에 추가
- 프레젠테이션 내 슬라이드 위치를 통해 슬라이드에 접근
- 슬라이드를 같은 프레젠테이션 또는 다른 프레젠테이션에 복사 또는 복제
- 슬라이드 편집 및 삭제
- 마스터와 슬라이드를 자동으로 복사
- 슬라이드 미리 보기
- 슬라이드 이름 조회 또는 설정
- 슬라이드를 이미지로 저장
- 슬라이드를 SVG 형식으로 내보내기
- 프레젠테이션에 여러 마스터 사용
- 지원되는 스타일로 슬라이드 배경 채우기
- 새 슬라이드 노트 추가 또는 기존 노트 관리
- 새 슬라이드 마스터와 스타일을 동적으로 설정
- 슬라이드 쇼 전환 추가 또는 수정
- 모프 전환 추가

### **그래픽 및 멀티미디어 기능**

- 선, 사각형, 타원, 폴리라인 등 도형 생성 또는 관리
- 도형의 선 스타일 관리
- 도형의 대체 텍스트 가져오기
- 도형 회전 및 뒤집기
- 솔리드, 그라디언트, 패턴, 텍스처, 이미지 등 다양한 스타일로 도형 채우기
- 48개의 사전 정의된 패턴 및 24개의 텍스처 중 任意 선택
- 도형에 링크 지정
- 슬라이드와 도형에 새로운 애니메이션 효과 읽기, 수정 및 추가
- 프레젠테이션에 새 그림 추가 및 기존 그림 관리
- 프레젠테이션에 새 그림 글머리표 추가 및 기존 글머리표 관리
- 프레젠테이션에 새 오디오 프레임 추가 및 기존 프레임 관리
- 프레젠테이션에 새 비디오 프레임 추가 및 기존 프레임 관리

### **표 기능**

- 처음부터 표 만들기 및 슬라이드에 추가
- 표 크기 조정 및 이동
- 표의 任意 셀에 접근
- 셀의 서식 및 테두리 변경
- 열과 행 크기 조정
- 열과 행 추가 및 삭제
- 전체 표에 대한 테두리 두께와 색상 설정

### **텍스트 및 단락 기능**

- 텍스트 플레이스홀더와 텍스트 프레임의 텍스트 조회 및 설정
- 플레이스홀더 유형 조회
- 텍스트 프레임의 텍스트 서식 변경
- 새로운 글꼴 추가 또는 프레젠테이션에 이미 사용된 글꼴 관리
- 텍스트 및 텍스트 서식 변경
- 텍스트에 대한 링크 생성 또는 관리
- 탭 및 효과적인 탭 지원
- 거의 모든 도형 유형에 새 텍스트 프레임 추가
- 텍스트 프레임의 여백 및 텍스트 래핑 설정 조회 및 수정
- 단락 들여쓰기 조회 및 수정

### **고급 기능**

- 프레젠테이션에 포함된 OLE 객체에 접근
- 프레젠테이션 잠금

## **기술 지원**

Aspose는 모든 제품에 대해 무제한 무료 기술 지원을 제공합니다. 이 지원은 평가 패키지를 포함한 모든 사용자에게 제공됩니다. Aspose.Slides for Node.js via .NET에 대한 도움이 필요하면 다음을 참고하십시오:

- 주요 지원 채널은 [Aspose 포럼](https://forum.aspose.com/)입니다. [Aspose.Slides 포럼](https://forum.aspose.com/c/slides/ko/11)에 질문을 게시하면 몇 시간 내에 답변을 받을 수 있습니다. Aspose.Slides 지원팀이 포럼에 게시된 질문에 직접 답변합니다.
- Aspose는 전화로 기술 지원을 제공하지 않으며, 전화 지원은 영업 및 구매 문의에만 제공됩니다.
- 포럼에서 답변을 기다릴 때는 시간대 차이를 고려하여 인내심을 가지고 기다려 주세요.

Aspose.Slides for Node.js via .NET에 문제가 발생하면 가장 효율적인 해결 방법은 다음 지침을 따르는 것입니다:

- 문제를 보고하기 전에 먼저 최신 버전의 Aspose.Slides for Node.js via .NET을 사용하고 있는지 확인하십시오. 해당 문제가 이미 업데이트에서 해결되었을 수 있습니다.  

  최신 제품 버전 정보는 [Aspose.Slides 다운로드](https://releases.aspose.com/slides/ko/nodejs-net/)에서 확인할 수 있습니다.

- 문제를 보고하기 전에 포럼, 이 문서 및 API 레퍼런스를 살펴보십시오. 질문에 대한 답변이 이미 제공되어 있을 수 있습니다.

- 문제를 보고할 때는 원본 문서와 (가능한 경우) 문제와 관련된 코드 조각을 포함하십시오. 여러 파일을 첨부해야 하면 ZIP 패키지에 넣어 주세요.  

  첨부 파일은 Aspose.Forums에 안전하게 업로드할 수 있으며, 파일에 접근할 수 있는 사람은 귀하와 Aspose 개발자뿐입니다.

- 특정 이슈당 하나의 스레드만 사용하십시오. 다른 문제가 발생하면 새 스레드를 만들어 보고하십시오.
