## **지원되는 플랫폼**
Aspose.Slides for Node.js via .NET은 플랫폼에 독립적인 API입니다. Node.js 7 이상이 설치된 모든 플랫폼(Windows, Linux, macOS 등)에서 사용할 수 있습니다. Oracle JDK 8 이상도 필요합니다.

## **파일 형식 및 변환**
Aspose.Slides for Node.js via .NET은 대부분의 PowerPoint 문서 형식을 지원합니다. 또한 조직에서 사용하고 공유하는 인기 있는 파일 형식으로 내보낼 수 있습니다. 자세한 내용은 다음을 확인하십시오:

|**기능**|**설명**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via .NET은 이 프레젠테이션 문서 형식에 대해 가장 빠른 처리 속도를 제공합니다.|
|PPT to PPTX 변환|Aspose.Slides for Node.js via .NET은 PPT를 PPTX로 변환하는 기능을 지원합니다.|
|Portable Document Format (PDF)|단일 메서드로 지원되는 모든 파일 형식을 Adobe Portable Document Format (PDF) 문서로 내보낼 수 있습니다.|
|XML Parser Specification (XPS)|단일 메서드로 지원되는 모든 파일 형식을 XML Parser Specification (XPS) 문서로 내보낼 수 있습니다.|
|Tagged Image File Format (TIFF)|지원되는 모든 프레젠테이션 파일 형식을 Tagged Image File Format (TIFF)로 내보낼 수 있습니다.|
|PPTX To HTML 변환|Aspose.Slides for Node.js via .NET은 PresentationEx를 HTML 형식으로 변환하는 기능을 지원합니다.|

## **렌더링 및 인쇄**
Aspose.Slides for Node.js via .NET은 프레젠테이션 슬라이드를 다양한 그래픽 형식으로 고품질 렌더링하는 것을 지원합니다. 자세한 내용은 다음을 확인하십시오:

|**기능**|**설명**|
| :- | :- |
|.NET 지원 이미지 형식|Aspose.Slides for Node.js via .NET을 사용하면 TIFF, PNG, BMP, JPEG, GIF와 같은 .NET이 지원하는 모든 그래픽 형식으로 프레젠테이션 슬라이드와 슬라이드의 이미지를 렌더링할 수 있습니다.|
|SVG 형식|Aspose.Slides for Node.js via .NET은 프레젠테이션 슬라이드를 Scalable Vector Graphics (SVG) 형식으로 내보내는 내장 메서드를 제공합니다.|
|프레젠테이션 인쇄|최신 버전의 Aspose.Slides for Node.js via .NET은 다양한 옵션을 갖춘 내장 인쇄 메서드를 제공합니다.|

## **콘텐츠 기능**
Aspose.Slides for Node.js via .NET을 사용하면 프레젠테이션 문서의 거의 모든 항목 또는 콘텐츠에 접근하고 수정하거나 새로 만들 수 있습니다. 자세한 내용은 다음을 확인하십시오:

|**기능**|**설명**|
| :- | :- |
|마스터 슬라이드|마스터 슬라이드는 일반 슬라이드의 레이아웃을 정의합니다. Aspose.Slides for Node.js via .NET을 사용하면 프레젠테이션 문서의 마스터 슬라이드에 접근하고 수정할 수 있습니다.|
|일반 슬라이드|Aspose.Slides for Node.js via .NET을 사용하면 다양한 유형의 새 슬라이드를 만들 수 있으며, 프레젠테이션에 있는 기존 슬라이드에 접근하고 수정할 수 있습니다.|
|슬라이드 복제/복사|Aspose.Slides for Node.js via .NET이 제공하는 내장 메서드를 사용하면 프레젠테이션 내 기존 슬라이드를 복제하거나 복사할 수 있습니다. 복제·복사된 슬라이드를 한 프레젠테이션에서 다른 프레젠테이션으로도 사용할 수 있습니다. 슬라이드는 마스터 슬라이드에서 레이아웃을 상속하므로 복제 시 마스터도 자동으로 복사됩니다.|
|슬라이드 섹션 관리|프레젠테이션 내부에서 슬라이드를 서로 다른 섹션으로 구성하는 메서드.|
|플레이스홀더 및 텍스트 플레이스홀더|슬라이드의 플레이스홀더와 텍스트 플레이스홀더에 접근할 수 있습니다. 또한 적절한 메서드를 사용해 텍스트 플레이스홀더가 포함된 슬라이드를 처음부터 만들 수 있습니다.|
|머리글 및 바닥글|Aspose.Slides for Node.js via .NET은 슬라이드의 머리글/바닥글을 쉽게 처리할 수 있도록 도와줍니다.|
|슬라이드 노트|Aspose.Slides for Node.js via .NET을 사용하면 슬라이드와 연결된 노트에 접근하고 수정할 수 있으며 새 노트를 추가할 수도 있습니다.|
|도형 찾기|도형에 연결된 대체 텍스트를 사용해 특정 도형을 찾을 수 있습니다.|
|배경|프레젠테이션의 마스터 슬라이드 또는 일반 슬라이드와 연결된 배경을 작업할 수 있습니다.|
|텍스트 상자|텍스트 상자를 처음부터 만들 수 있고, 기존 텍스트 상자에 접근할 수 있으며 원래 형식을 유지한 채 텍스트를 수정할 수 있습니다.|
|사각형 도형|Aspose.Slides for Node.js via .NET을 사용해 사각형 도형을 만들거나 수정할 수 있습니다.|
|폴리라인 도형|Aspose.Slides for Node.js via .NET을 사용해 폴리라인 도형을 만들거나 수정할 수 있습니다.|
|타원 도형|Aspose.Slides for Node.js via .NET을 사용해 타원 도형을 만들거나 수정할 수 있습니다.|
|그룹 도형|Aspose.Slides for Node.js via .NET은 그룹 도형을 지원합니다.|
|오토 도형|Aspose.Slides for Node.js via .NET은 오토 도형을 지원합니다.|
|SmartArt|Aspose.Slides for Node.js via .NET은 MS PowerPoint의 SmartArt 도형을 지원합니다.|
|차트|Aspose.Slides for Node.js via .NET은 PowerPoint의 MSO 차트를 지원합니다.|
|도형 직렬화|Aspose.Slides for Node.js via .NET은 많은 도형을 지원합니다. 지원되지 않는 도형이 있을 경우 기존 슬라이드에서 해당 도형을 직렬화하는 방법을 사용해 직렬화하고, 이후 필요에 따라 재사용할 수 있습니다.|
|그림 프레임|Aspose.Slides for Node.js via .NET을 사용해 그림 프레임 안의 이미지를 관리할 수 있습니다.|
|오디오 프레임|Aspose.Slides for Node.js via .NET을 사용해 슬라이드의 오디오 프레임에 오디오 파일을 링크하거나 삽입할 수 있습니다.|
|비디오 프레임|비디오 프레임에서 비디오 파일을 처리할 수 있습니다. Aspose.Slides for Node.js via .NET은 링크된 비디오와 삽입된 비디오를 모두 지원합니다.|
|OLE 프레임|Aspose.Slides for Node.js via .NET을 사용해 OLE 프레임 안의 OLE 개체를 관리할 수 있습니다.|
|표|Aspose.Slides for Node.js via .NET은 슬라이드의 표를 지원합니다.|
|ActiveX 컨트롤|ActiveX 컨트롤을 지원합니다.|
|VBA 매크로|프레젠테이션 내 VBA 매크로 관리 지원|
|텍스트 프레임|어떤 도형이든 해당 도형에 연결된 텍스트 프레임을 통해 텍스트에 접근할 수 있습니다.|
|텍스트 스캔|내장 스캔 메서드를 사용해 프레젠테이션 전체나 슬라이드 수준에서 텍스트를 스캔할 수 있습니다.|
|애니메이션|도형에 애니메이션을 적용할 수 있습니다.|
|슬라이드 쇼|Aspose.Slides for Node.js via .NET은 슬라이드 쇼와 슬라이드 전환을 지원합니다.|

## **서식 기능**
Aspose.Slides for Node.js via .NET을 사용하면 프레젠테이션 슬라이드의 텍스트와 도형을 서식 지정할 수 있습니다. 자세한 내용은 다음을 확인하십시오:

|**기능**|**설명**|
| :- | :- |
|텍스트 서식|<p>Aspose.Slides for Node.js via .NET에서 텍스트는 도형에 연결된 텍스트 프레임을 통해 관리됩니다. 따라서 텍스트 프레임의 단락 및 구성을 사용해 텍스트를 서식 지정할 수 있습니다. 이 텍스트 요소는 Aspose.Slides for Node.js via .NET을 통해 서식 지정됩니다.</p><p>- 글꼴 유형</p><p>- 글꼴 크기</p><p>- 글꼴 색상</p><p>- 글꼴 음영</p><p>- 단락 정렬</p><p>- 단락 글머리표</p><p>- 단락 방향</p>|
|도형 서식|<p>Aspose.Slides for Node.js via .NET에서 슬라이드의 기본 요소는 도형입니다. 다음과 같이 도형 요소를 서식 지정할 수 있습니다:</p><p>- 위치</p><p>- 크기</p><p>- 선</p><p>- 채우기(패턴, 그라데이션, 단색 포함)</p><p>- 텍스트</p><p>- 이미지</p>|
