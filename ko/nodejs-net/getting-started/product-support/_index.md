## **기술 지원**
{{% alert color="primary" %}}

Aspose는 모든 제품에 대해 무제한 무료 기술 지원을 제공합니다. 이 지원은 모든 사용자에게 제공됩니다(평가 패키지를 포함한 사용자도 포함).

{{% /alert %}} 

Aspose.Slides for Node.js via .NET에 대한 도움이 필요할 때는 다음을 고려하십시오:

- 지원의 주요 경로는 [Aspose.Forums](https://forum.aspose.com/c/slides/ko/11)입니다. Aspose.Slides 포럼에 질문을 게시하면 몇 시간 내에 답변이 제공됩니다. Aspose.Slides 지원팀이 포럼에 게시된 질문에 직접 답변합니다.
- Aspose는 전화로 기술 지원을 제공하지 않는다는 점에 유의하십시오. 전화 지원은 판매 및 구매 문의에만 제공됩니다.
- 포럼에서 답변을 기대할 때는 인내심을 가지고 시차 차이를 고려하십시오.


Aspose.Slides for Node.js via .NET에서 문제가 발생하면 가장 효율적인 방식으로 해결하기 위해 다음 가이드라인을 따르십시오:

- 문제를 보고하기 전에 최신 Aspose.Slides for Node.js via .NET 버전을 사용하고 있는지 확인하십시오. 발생한 문제가 이미 업데이트에서 해결되었을 수 있습니다. 

  See [Aspose.Slides download](https://releases.aspose.com/slides/ko/nodejs-net/) to get information on the latest version.

- 문제를 보고하기 전에 포럼, 이 문서 및 API 레퍼런스를 확인하는 것이 좋습니다. 질문에 대한 답변이 이미 제공되었을 수 있습니다. 

- 문제를 보고할 때는 원본 문서와 (가능하면) 문제와 관련된 코드 조각을 포함하십시오. 여러 파일을 첨부해야 하는 경우 ZIP 패키지에 넣으세요. 첨부 파일은 **only** 귀하와 Aspose 개발자만 접근할 수 있기 때문에 Aspose.Forums에 안전하게 첨부할 수 있습니다.

- 특정 이슈에 대해 하나의 스레드를 만들고 사용하십시오. 다른 문제가 발생하면 해당 문제를 보고하기 위해 새 스레드를 만들어야 합니다.
