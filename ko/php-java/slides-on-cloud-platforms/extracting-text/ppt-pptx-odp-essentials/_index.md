## **소개**

프레젠테이션 파일에서 텍스트를 추출하는 것은 **비즈니스 프로세스 자동화**, **데이터 분석**, 그리고 **문서 워크플로우 효율화**에 필수적입니다. 오늘날 디지털 환경에서 많은 조직이 슬라이드에 포함된 정보에 대해 **신속한 접근**이 필요합니다. **검색 인덱싱**, **콘텐츠 분석**, **접근성** 또는 **현지화**를 위해서든, 신뢰할 수 있는 텍스트 추출은 가치 있는 슬라이드 콘텐츠를 다양한 시스템에서 재사용·처리·분석할 수 있도록 보장합니다.

## **텍스트 추출의 실용적 적용**

- **문서 워크플로우 자동화**: PPTX 및 ODP 파일을 SharePoint, Alfresco, 또는 1C:Document Management와 같은 기업 문서 관리 시스템(DMS)에 원활하게 통합합니다.  
- **검색 인덱싱**: 추출된 텍스트를 인덱싱하여 고속 검색 시스템을 구축하고, 대용량 프레젠테이션 아카이브에서 관련 데이터를 신속하게 검색할 수 있게 합니다.  
- **콘텐츠 분석**: 핵심 문구, 주제 및 트렌드를 자동으로 식별하여 마케팅 및 분석 팀이 예측과 전략적 의사결정을 지원합니다.  
- **접근성 및 현지화**: 자막을 생성하고, 슬라이드를 다국어로 번역하거나, 화면 읽기 소프트웨어와 콘텐츠를 통합하여 접근성을 향상시킵니다.  
- **텍스트 위치 및 시각 분석**: 텍스트 자체를 넘어 레이아웃과 위치를 분석함으로써 슬라이드 구조, 서식 및 기업 가이드라인과의 정렬을 보장합니다.

## **프레젠테이션 형식 개요**

### **PPT (레거시 PowerPoint 형식)**

2007년까지 Microsoft PowerPoint에서 사용된 **PPT**는 **MS Office 97–2003**에서 널리 쓰였습니다. **이진 형식**인 PPT는 최신 XML 기반 형식에 비해 특수 도구 없이는 처리하기가 더 어렵습니다.

**텍스트 추출의 주요 어려움**

- 전용 이진 구조 때문에 공식 Microsoft API 또는 특수 라이브러리가 없으면 **데이터 접근**이 어렵습니다.  
- **텍스트가** 여러 위치(슬라이드, 노트, 주석)에서 나타날 수 있어, 포괄적인 추출 접근 방식이 필요합니다.  
- **인코딩 및 글꼴 충돌**이 커스텀 문자와 작업할 때 발생할 수 있습니다.

### **PPTX (Open XML 사양)**

**PowerPoint 2007**에 도입된 **PPTX**는 **Office Open XML** 기반으로, 텍스트 추출을 단순화하는 XML 기반 표준입니다.

**파일 구조 기본**

- PPTX 파일은 여러 **XML 문서**를 포함하는 **ZIP 압축 파일**입니다.  
- 슬라이드, 노트 섹션 및 메타데이터는 각각 별도의 **XML 파일**에 저장됩니다.

**구조화된 XML에서 텍스트 추출**

PPTX는 명확한 XML 구조 덕분에 보다 효율적인 텍스트 추출을 가능하게 합니다:
- `<a:t>` 태그 안에 **텍스트가 `ppt/slides/ko/slideX.xml`** 에 위치합니다.  
- **노트와 주석**은 `ppt/notesSlides/`에 존재합니다.  
- **서식 유지**를 위해 추가 XML 속성을 파싱해야 할 수도 있습니다.

### **ODP (OpenDocument 프레젠테이션)**

**OpenDocument Format (ODF)** 기반인 **ODP**는 **LibreOffice Impress**와 같은 오픈소스 오피스 스위트에서 흔히 사용됩니다.

**PPTX와의 차이점**

- Open XML이 아닌 **OpenDocument XML**에 기반합니다.  
- 구조는 유사하지만 **다른 태그와 별도 계층 구조**를 사용합니다.  
- 텍스트는 흔히 `<text:p>` 요소 안의 **content.xml**에 저장됩니다.

## **결론**

프레젠테이션 파일 구조에 대한 확실한 이해는 성공적인 텍스트 추출에 필수적입니다. **PPTX와 ODP**는 XML 기반의 투명성을 제공하지만, 오래된 **PPT** 파일은 이진 형태라 추가 작업이 필요합니다. 각 형식에 맞게 설계된 특수 도구와 라이브러리는 추출 과정을 자동화하고 최적화하는 데 도움을 주어, 추출된 데이터를 강력한 인덱싱부터 포괄적인 접근성 솔루션에 이르는 다양한 사용 사례에 활용할 수 있게 합니다.
