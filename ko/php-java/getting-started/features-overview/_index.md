## **지원되는 플랫폼**
Aspose.Slides for PHP via Java는 플랫폼에 독립적인 API입니다. PHP 7 이상이 설치된 모든 플랫폼(Windows, Linux, macOS 등)에서 사용할 수 있습니다. Oracle JDK 7 이상이 또 다른 요구 사항입니다.

## **파일 형식 및 변환**
Aspose.Slides for PHP via Java는 대부분의 PowerPoint 문서 형식을 지원합니다. 또한 조직에서 사용하고 공유하는 일반적인 파일 형식으로 내보낼 수 있습니다. 다음 세부 정보를 확인하십시오:

|**기능**|**설명**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for PHP via Java는 이 프레젠테이션 문서 형식에 대해 가장 빠른 처리를 제공합니다.|
|PPT to PPTX conversion|Aspose.Slides for PHP via Java는 PPT를 PPTX로 변환하는 것을 지원합니다.|
|Portable Document Format (PDF)|단일 메서드로 지원되는 모든 파일 형식을 Adobe Portable Document Format(PDF) 문서로 내보낼 수 있습니다.|
|XML Parser Specification (XPS)|단일 메서드로 지원되는 모든 파일 형식을 XML Parser Specification(XPS) 문서로 내보낼 수 있습니다.|
|Tagged Image File Format (TIFF)|지원되는 모든 프레젠테이션 파일 형식을 Tagged Image File Format(TIFF)으로 내보낼 수 있습니다.|
|PPTX To HTML Conversion|Aspose.Slides for PHP via Java는 PresentationEx를 HTML 형식으로 변환하는 것을 지원합니다.|

## **프레젠테이션 렌더링**
Aspose.Slides for PHP via Java는 프레젠테이션 슬라이드를 다양한 그래픽 형식으로 높은 충실도로 렌더링하는 것을 지원합니다. 다음 세부 정보를 확인하십시오:

|**기능**|**설명**|
| :- | :- |
|Java Supported Image Formats|Aspose.Slides for PHP via Java를 사용하면 TIFF, PNG, BMP, JPEG, GIF와 같은 모든 Java 지원 그래픽 형식으로 프레젠테이션 슬라이드와 슬라이드 이미지를 렌더링할 수 있습니다.|
|SVG Format|Aspose.Slides for PHP via Java는 Scalable Vector Graphics(SVG) 형식으로 프레젠테이션 슬라이드를 내보내는 내장 메서드도 제공합니다.|
|Presentation Printing|최신 버전의 Aspose.Slides for PHP via Java는 다양한 옵션을 갖춘 내장 인쇄 메서드를 제공합니다.|

## **콘텐츠 기능**
Aspose.Slides for PHP via Java를 사용하면 프레젠테이션 문서의 거의 모든 항목이나 콘텐츠에 접근, 수정 또는 생성할 수 있습니다. 다음 세부 정보를 확인하십시오:

|**기능**|**설명**|
| :- | :- |
|Master Slides|마스터 슬라이드는 일반 슬라이드의 레이아웃을 정의합니다. Aspose.Slides for PHP via Java를 사용하면 프레젠테이션 문서의 마스터 슬라이드에 접근하고 수정할 수 있습니다.|
|Normal Slides|Aspose.Slides for PHP via Java를 사용하면 다양한 유형의 새 슬라이드를 만들 수 있으며, 프레젠테이션에 존재하는 슬라이드에 접근하고 수정할 수도 있습니다.|
|Cloning / Copying Slides|Aspose.Slides for PHP via Java가 제공하는 내장 메서드를 사용하면 프레젠테이션 내 기존 슬라이드를 복제하거나 복사할 수 있습니다. 복제 및 복사된 슬라이드를 한 프레젠테이션에서 다른 프레젠테이션으로도 사용할 수 있습니다. 슬라이드는 마스터 슬라이드에서 레이아웃을 상속하므로 복제 메서드는 자동으로 마스터를 복사합니다.|
|Managing Slides sections|프레젠테이션 내 슬라이드를 서로 다른 섹션으로 조직하는 메서드|
|Place Holders and Text Holders|슬라이드에서 플레이스홀더와 텍스트홀더에 접근할 수 있습니다. 또한 적절한 메서드를 사용해 텍스트홀더가 포함된 슬라이드를 처음부터 만들 수 있습니다.|
|Header and Footers|Aspose.Slides for PHP via Java는 슬라이드의 머리글/바닥글 처리를 지원합니다.|
|Notes in Slides|Aspose.Slides for PHP via Java를 사용하면 슬라이드에 연결된 노트를 접근·수정하고 새 노트를 추가할 수 있습니다.|
|Finding a Shape|슬라이드에 연결된 대체 텍스트를 사용해 특정 도형을 찾을 수 있습니다.|
|Backgrounds|Aspose.Slides for PHP via Java는 마스터 슬라이드나 일반 슬라이드에 연결된 배경을 작업할 수 있게 해줍니다.|
|Text Boxes|텍스트 박스를 처음부터 만들 수 있습니다. 기존 텍스트 박스에 접근할 수 있으며, 원본 텍스트 형식을 유지하면서 텍스트를 수정할 수 있습니다.|
|Rectangle Shapes|Aspose.Slides for PHP via Java를 사용해 사각형 도형을 만들거나 수정할 수 있습니다.|
|Poly Line Shapes|Aspose.Slides for PHP via Java를 사용해 폴리라인 도형을 만들거나 수정할 수 있습니다.|
|Ellipse Shapes|Aspose.Slides for PHP via Java를 사용해 타원 도형을 만들거나 수정할 수 있습니다.|
|Group Shapes|Aspose.Slides for PHP via Java는 그룹 도형을 지원합니다.|
|Auto Shapes|Aspose.Slides for PHP via Java는 자동 도형을 지원합니다.|
|SmartArt|Aspose.Slides for PHP via Java는 MS PowerPoint의 SmartArt 도형을 지원합니다.|

|Picture Frames|Aspose.Slides for PHP via Java를 사용해 사진 프레임 안의 이미지를 관리할 수 있습니다.|
|Audio Frames|Aspose.Slides for PHP via Java를 사용해 슬라이드의 오디오 프레임에 오디오 파일을 연결하거나 삽입할 수 있습니다.|
|Video Frames|비디오 프레임에서 비디오 파일을 처리할 수 있습니다. Aspose.Slides for Java via Java는 연결 및 삽입된 비디오도 지원합니다.|
|OLE Frame|Aspose.Slides for PHP via Java를 사용해 OLE 프레임 안의 OLE 객체를 관리할 수 있습니다.|

|ActiveX Controls|ActiveX 컨트롤 지원|

## **FAQ**

**서버/PC에 Microsoft PowerPoint를 설치해야 라이브러리를 사용할 수 있나요?**

아니요. PowerPoint는 필요하지 않으며, Aspose.Slides는 프레젠테이션을 생성·편집·변환·렌더링하기 위한 독립 실행형 엔진입니다.

**멀티스레딩은 어떻게 작동하나요? 처리를 병렬화할 수 있나요?**

다른 스레드에서 서로 다른 문서를 처리하는 것은 안전합니다; 동일한 [프레젠테이션](https://reference.aspose.com/slides/ko/php-java/aspose.slides/presentation/) 객체를 [여러 스레드](/slides/ko/php-java/multithreading/)가 동시에 사용해서는 안 됩니다.

**파일 비밀번호와 암호화가 지원되나요?**

네. [암호가 보호된 프레젠테이션](/slides/ko/php-java/password-protected-presentation/)을 열 수 있고, 열기 및 쓰기 비밀번호를 설정·제거하며 보호 상태를 확인할 수 있습니다.

**Linux 컨테이너에서 폰트 패키지를 신경써야 하나요?**

네. 일반적인 폰트 패키지를 설치하거나 애플리케이션에서 [폰트 디렉터리 지정](/slides/ko/php-java/custom-font/)을 명시적으로 설정하여 예기치 않은 대체를 방지하는 것이 권장됩니다.

**평가 버전에 제한이 있나요?**

[평가 모드](/slides/ko/php-java/licensing/)에서는 출력에 워터마크가 추가되고 일부 제한이 적용됩니다; 전체 기능 테스트를 위한 [30일 임시 라이선스](https://purchase.aspose.com/temporary-license/)를 사용할 수 있습니다.

**외부 형식(PDF/HTML → PPTX)을 프레젠테이션으로 가져오는 것이 지원되나요?**

네. [PDF 페이지 및 HTML 콘텐츠](/slides/ko/php-java/import-presentation/)를 프레젠테이션에 추가해 슬라이드로 변환할 수 있습니다.
