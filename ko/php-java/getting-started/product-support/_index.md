## **기술 지원**
{{% alert color="primary" %}}

Aspose는 모든 제품에 대해 무제한 무료 기술 지원을 제공합니다. 이 지원은 평가 패키지를 포함한 모든 사용자에게 제공됩니다.

{{% /alert %}} 

Aspose.Slides for PHP via Java에 대한 도움이 필요할 때, 다음 사항을 고려하십시오:

- 지원의 주요 경로는 [Aspose.Forums](https://forum.aspose.com/c/slides/ko/11)입니다. Aspose.Slides 포럼에 질문을 게시하면 몇 시간 내에 답변을 받게 됩니다. Aspose.Slides 지원 팀이 포럼에 게시된 질문에 직접 답변합니다.
- Aspose는 전화로 기술 지원을 제공하지 않음을 알려드립니다. 전화 지원은 영업 및 구매 문의에만 제공됩니다.
- 포럼에서 답변을 기대할 때는 인내심을 가지고, 시차 차이를 고려해 주시기 바랍니다.

Aspose.Slides for PHP via Java에서 문제가 발생하면, 가장 효율적인 방법으로 해결하기 위해 다음 가이드라인을 따르십시오:

- 문제를 보고하기 전에 최신 Aspose.Slides for PHP via Java 버전을 사용하고 있는지 확인하십시오. 아마도 겪은 문제가 업데이트에서 이미 해결되었을 수 있습니다.

  최신 버전에 대한 정보를 얻으려면 [Aspose.Slides download](https://downloads.aspose.com/slides/ko/php-java)를 참고하십시오.

- 문제를 보고하기 전에 포럼, 이 문서 및 API 레퍼런스를 확인해 보시기 바랍니다. 질문에 대한 답변이 이미 제공되었을 수도 있습니다.

- 문제를 보고할 때는 원본 문서와 (가능하다면) 문제와 관련된 코드 조각을 포함해 주세요. 여러 파일을 첨부해야 할 경우 ZIP 패키지에 넣으십시오. 첨부 파일은 Aspose.Forums에 안전하게 업로드할 수 있으며, **only** 여러분과 Aspose 개발자만 파일에 접근할 수 있습니다.

- 특정 이슈에 대해서는 하나의 스레드를 생성하고 사용하십시오. 다른 문제가 발생하면 별도의 스레드를 만들어 보고해야 합니다.
