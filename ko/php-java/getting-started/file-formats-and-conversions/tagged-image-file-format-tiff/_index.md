## **TIFF에 대해**

여러 이미지를 하나의 문서에 저장할 수 있는 것으로 알려진 [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF)은 원래 Aldus가 만들었습니다. 이 형식은 스캔, 팩스 및 기타 이미지 처리 응용 프로그램에서 널리 지원됩니다.

## **Aspose.Slides for PHP via Java의 TIFF**

Aspose.Slide for Java에서 로드할 수 있는 모든 문서는 Aspose.Slides for PHP via Java를 사용해 직접 TIFF 문서로 변환할 수 있으며, 별도의 서드 파티 구성 요소가 필요하지 않습니다. 또한 결과 TIFF 문서의 이미지 크기를 선택적으로 정의할 수 있습니다. Aspose.Slides for PHP via Java를 통해 프레젠테이션 문서를 TIFF 문서로 내보내는 방법에 대한 정보는 [이 항목](/slides/ko/php-java/convert-powerpoint-to-tiff/)에서 확인할 수 있습니다.
