Aspose.Slides for PHP via Java에서 PPT를 PPTX로 변환하는 것을 지원합니다. 마스터 슬라이드, 구조 등 대부분의 프레젠테이션 기능은 한 형식에서 다른 형식으로 변환할 때 유지되지만, [몇 가지 제한 사항](/slides/ko/php-java/ppt-to-pptx-conversion/)이 있습니다.

{{% /alert %}} 
## **변환에서 지원되는 기능**
Aspose.Slides for PHP via Java는 PPT 파일 형식을 PPTX로 변환하는 부분적인 지원을 제공합니다. 변환 지원은 Aspose.Slides for PHP via Java에 막 도입된 것이므로 몇 가지 제한이 있으며 간단한 프레젠테이션에 가장 적합합니다. PPT를 PPTX로 변환할 때 Aspose.Slides for PHP via Java가 제공하는 주요 장점은 API 사용이 매우 쉽다는 점입니다. 코드 예제를 보려면 [PPT를 PPTX로 변환]()을 읽어보세요. 아래 목록에서는 PPT를 PPTX로 변환할 때 지원되는 기능과 지원되지 않는 기능을 설명합니다.

## **지원되는 기능**
The following features are supported for conversion:

- 마스터, 레이아웃 및 슬라이드 구조 변환.
- 차트 변환.
- 그룹 도형.
- 사각형 및 타원을 포함한 자동 도형 변환.
- 사용자 지정 기하학을 가진 도형.
- 자동 도형에 대한 텍스처 및 이미지 채우기 스타일.
- 플레이스홀더 변환.
- 선 및 폴리라인 변환.
- 선 및 채우기 형식.
- 그라디언트 채우기 스타일.
- OLE 프레임, 표, 비디오 및 오디오 프레임 등.
- 애니메이션 및 슬라이드쇼 속성.
- 텍스트 프레임 및 텍스트 홀더의 텍스트 변환.
