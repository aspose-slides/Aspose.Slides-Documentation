## **개요**

Aspose.Slides는 Microsoft PowerPoint 97부터 Office 365까지, Microsoft PowerPoint for Mac을 포함한 프레젠테이션 파일을 지원합니다. 이 문서는 라이브러리에서 지원하는 PowerPoint 버전을 나열하고 로드, 저장 또는 둘 다 가능한 파일 형식 표를 제공합니다.

이 문서는 PDF 준수, 글꼴 삽입, 암호 보호 파일, 사용자 정의 글꼴, 글꼴 대체 및 XPS 내보내기 옵션에 대한 일반적인 질문에도 답변합니다.

## **지원되는 Microsoft PowerPoint 버전**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint for MAC
- Office 365

## **자주 묻는 질문**

**보관 및 접근성 표준(PDF/A 및 PDF/UA)을 충족하는 PDF로 프레젠테이션을 저장할 수 있나요?**

예. Aspose.Slides는 PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b와 같은 준수 수준 및 PDF/UA를 포함한 PDF 내보내기를 [compliance](https://reference.aspose.com/slides/ko/php-java/aspose.slides/pdfoptions/setcompliance/) 설정을 통해 지원합니다. 자세한 내용은 [PDF export options](https://reference.aspose.com/slides/ko/php-java/aspose.slides/pdfoptions/)를 참조하십시오.

**라이브러리가 PDF로 내보낼 때 글꼴 삽입을 지원하며, 삽입되는 항목에 대한 세밀한 제어가 가능한가요?**

예. 글꼴을 완전히 삽입하거나 서브셋(사용된 글리프만)으로 삽입하는지 제어할 수 있으며, 일반 시스템 글꼴이 어떻게 처리되는지 지정하고 ASCII 텍스트에 대한 동작을 구성할 수 있습니다. 자세한 내용은 [PDF export options](https://reference.aspose.com/slides/ko/php-java/aspose.slides/pdfoptions/)를 참조하십시오.

**파일을 실제로 로드하기 전에 암호 보호 여부를 감지할 수 있나요?**

예. [factory-based inspection API](https://reference.aspose.com/slides/ko/php-java/aspose.slides/presentationfactory/)를 사용하면 프레젠테이션 파일을 완전히 열지 않고도 암호 보호 여부를 확인할 수 있습니다.

**글꼴 대체 메커니즘과 사용자 정의 글꼴 지원이 있나요?**

예. 라이브러리는 [loading](/slides/ko/php-java/custom-font/) 및 [embedding](/slides/ko/php-java/embedded-font/) 사용자 정의 글꼴을 지원하고, 렌더링 및 변환 중에 누락된 글리프를 방지하기 위한 글꼴 [fallback rules](/slides/ko/php-java/fallback-font/)을 제공합니다.

**슬라이드를 XPS로 내보낼 수 있으며, XPS 출력 품질을 조정하는 옵션이 있나요?**

예. [Export to XPS](/slides/ko/php-java/convert-powerpoint-to-xps/)가 지원되며, 관련 [save options](https://reference.aspose.com/slides/ko/php-java/aspose.slides/xpsoptions/)를 조정하여 XPS 문서의 출력 품질과 내용을 제어할 수 있습니다.
