## **소개**

Aspose.Slides for PHP via Java은 Microsoft PowerPoint와 같은 제3자 제품을 설치할 필요가 없습니다. Aspose.Slides 자체는 Microsoft PowerPoint 프레젠테이션 형식을 포함한 다양한 형식의 문서를 만들고, 수정하고, 변환하고, 렌더링하는 엔진입니다.

## **지원 운영 체제**

## **FAQ**

**변환 및 렌더링을 위해 Microsoft PowerPoint를 설치해야 합니까?**

아니요, PowerPoint는 필요하지 않습니다; Aspose.Slides는 독립형 엔진으로 [생성](/slides/ko/php-java/create-presentation/), 수정, [변환](/slides/ko/php-java/convert-presentation/), 및 [렌더링](/slides/ko/php-java/convert-powerpoint-to-png/) 프레젠테이션을 수행합니다.

**정확한 렌더링을 위해 어떤 폰트가 필요합니까?**

실제로 프레젠테이션에 사용된 폰트 또는 적절한 [대체 폰트](/slides/ko/php-java/font-substitution/)가 있어야 합니다. Linux/macOS에서 일관된 렌더링을 보장하려면 일반적인 폰트 패키지를 설치하는 것이 좋습니다.

**Linux에서 사용자 정의 폰트가 대체 폰트나 누락된 텍스트로 표시되는 이유는 무엇입니까?**

폰트 파일에 일관성이 없거나 손상된 name-table 항목이 있는 경우, Linux 폰트 매칭 스택(Freetype/fontconfig)이 잘못된 레코드를 선택하여 폰트를 찾지 못할 수 있습니다. name-table 레코드가 수정된 폰트 버전을 사용하거나 일관된 대체 폰트를 설치하면 문제가 해결됩니다.
