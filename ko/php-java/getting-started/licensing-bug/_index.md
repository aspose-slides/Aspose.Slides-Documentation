## **개요**

이 문서에서는 Aspose.Slides for PHP via Java 버전 23.2, 23.3, 23.4, 23.5, 23.6 및 23.7에서 발생한 라이선스 버그에 대해 설명합니다. 이 버그로 인해 영구 라이선스가 구독이 만료된 후 일시적으로 사용 불가능해졌습니다.

## **증상**

영구 라이선스 구독이 만료된 후 Aspose.Slides for PHP via Java 버전 23.2 - 23.7은 라이선스가 만료된 것으로 보고 모든 기능을 평가 모드로 전환할 수 있습니다.  
이 동작은 올바르지 않으며 23.2 이전 및 23.7 이후 버전에는 영향을 주지 않습니다.

## **해결 방법**

Aspose.Slides for PHP via Java는 이 문제를 해결하고 수정이 반영된 버전(23.2.1, 23.3.1, 23.4.1, 23.5.1, 23.6.1, 23.7.1)을 출시했습니다.

프로젝트에서 영향을 받는 버전을 사용 중인 경우 수정된 버전으로 전환하십시오.

Aspose.Slides for PHP via Java에서 라이선스 문제가 발생하면:

- 사용 중인 버전을 확인하고 위에 설명된 버그의 영향을 받지 않는지 확인하십시오.
- 문제 해결 단계와 알려진 라이선스 문제에 대한 내용은 [Aspose 문서](https://docs.aspose.com/slides/ko/php-java/getting-started/)를 참조하십시오.
- 추가 지원이 필요하면 [Aspose 지원](https://forum.aspose.com/)에 문의하십시오.

## **영향을 받는 제품 및 버전**

이 버그는 Aspose.Slides for PHP via Java 버전 23.2부터 23.7까지에만 영향을 미칩니다. **다른 Aspose 제품 및 다른 버전의 Aspose.Slides는 영향을 받지 않습니다**.
