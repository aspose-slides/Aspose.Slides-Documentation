## **Aspose.Slides for PHP via Java에 오신 것을 환영합니다!**

Aspose.Slides는 Microsoft PowerPoint®를 사용하지 않고도 애플리케이션에서 PowerPoint 프레젠테이션을 읽고, 쓰고, 만들고, 수정할 수 있게 해 주는 Microsoft PowerPoint® 관리 API입니다. 이 라이브러리는 프레젠테이션, 슬라이드, 도형, 표, 텍스트, 애니메이션, 오디오, 비디오 및 기타 프레젠테이션 요소 작업을 지원합니다.

Aspose.Slides를 사용하면 다양한 소스에서 프레젠테이션을 열고, 여러 PowerPoint 형식으로 저장하며, PDF, TIFF, XPS, SWF, HTML, SVG와 같은 형식으로 내보내고, 슬라이드를 이미지로 저장할 수 있습니다. 또한 문서 속성 관리, 보호 적용, 슬라이드 미리보기, 슬라이드 콘텐츠 편집, 슬라이드 노트 작업, 임베드된 OLE 개체와 같은 고급 프레젠테이션 요소 처리 기능도 제공합니다.

### **Aspose.Slides for PHP via Java를 사용해야 하는 이유**

- Aspose.Slides for PHP via Java는 PowerPoint® 문서를 관리하는 데 필요한 기능을 제공하는 최초이자 유일한 컴포넌트입니다.  
- 텍스트, 도형, 표 및 애니메이션 관리, 슬라이드에 오디오와 비디오 추가, 슬라이드 미리보기, SVG, PDF 등 다양한 형식으로 슬라이드 내보내기 등 많은 핵심 기능을 제공합니다.  
- PowerPoint 파일을 다양한 소스에서 열 수 있을 뿐만 아니라 프레젠테이션을 다양한 방식으로 저장할 수 있습니다. 예를 들어 PPT 형식으로 저장하거나 슬라이드를 이미지로 저장할 수 있습니다.  
- Microsoft PowerPoint에서 제공하는 거의 모든 기능을 사용할 수 있으며, 표와 다양한 도형 외에도 텍스트, 오디오, 비디오 프레임 등 다양한 종류의 프레임을 슬라이드에 추가할 수 있습니다.  

## **Aspose.Slides 기능**

Aspose.Slides for PHP via Java 기능은 다음 그룹으로 구분됩니다.

- Presentation Features.
- Slides Features.
- Graphics and Multimedia Features.
- Table Features.
- Text & Paragraph Features
- Advanced Features

### **Presentation Features**

- 다양한 소스에서 Microsoft PowerPoint 프레젠테이션 열기  
- 다양한 Microsoft PowerPoint 형식으로 프레젠테이션 저장  
- 새 프레젠테이션을 처음부터 만들기  
- 다른 컴포넌트를 사용하지 않고 프레젠테이션을 PDF 문서로 변환  
- 프레젠테이션의 문서 속성 읽기 또는 수정  
- 프레젠테이션에 보호 적용  
- 프레젠테이션을 PDF로 내보내기  
- 프레젠테이션을 TIFF로 내보내기  
- 프레젠테이션을 XPS로 내보내기  
- 프레젠테이션을 SWF로 내보내기  
- 프레젠테이션을 PDF Note / Tiff Notes 형태로 내보내기  
- 프레젠테이션을 HTML로 내보내기  

### **Slide Features**

- 새 슬라이드를 처음부터 만들고 프레젠테이션에 추가  
- 프레젠테이션 내 슬라이드 위치를 통해 임의의 슬라이드에 접근  
- 같은 프레젠테이션 또는 다른 프레젠테이션으로 슬라이드 복사 또는 복제  
- 슬라이드 편집 및 삭제  
- 마스터와 슬라이드를 자동으로 복사  
- 슬라이드 미리보기  
- 슬라이드 이름 가져오기 또는 설정  
- 슬라이드를 이미지로 저장  
- 슬라이드를 SVG 형식으로 내보내기  
- 프레젠테이션에 여러 마스터 사용  
- 지원되는 스타일로 슬라이드 배경 채우기  
- 새 슬라이드 노트를 추가하거나 기존 노트 관리  
- 새 슬라이드 마스터와 스타일을 동적으로 설정  
- 슬라이드 쇼 전환 효과 추가 또는 수정  
- Morph 전환 추가

- 슬라이드와 도형에 새로운 애니메이션 효과를 읽고, 수정하고, 추가  
- 프레젠테이션에 새 그림 추가 및 기존 그림 관리  
- 프레젠테이션에 새 그림 글머리표 추가 및 관리  
- 프레젠테이션에 새 오디오 프레임 추가 및 관리

- 새로운 글꼴 추가 또는 프레젠테이션에 이미 사용된 글꼴 관리  
- 텍스트 및 텍스트 서식 변경  
- 텍스트에 대한 링크 생성 또는 관리

## **Technical support**

Aspose는 모든 제품에 대해 무제한 무료 기술 지원을 제공합니다. 이 지원은 평가판 사용자를 포함한 모든 사용자가 이용할 수 있습니다. Aspose.Slides for PHP via Java에 대해 도움이 필요하면 다음을 참고하십시오.

- 주요 지원 경로는 [Aspose Forums](https://forum.aspose.com/). [Aspose.Slides 포럼](https://forum.aspose.com/c/slides/ko/11)에 질문을 게시하면 몇 시간 이내에 답변을 받을 수 있습니다. Aspose.Slides 지원팀이 포럼에 게시된 질문에 직접 답변합니다.  
- Aspose는 전화로 기술 지원을 제공하지 않습니다. 전화 지원은 영업 및 구매 질문에만 제공됩니다.  
- 포럼에서 답변을 기대할 때는 인내심을 가지고 시차 차이를 고려해 주시기 바랍니다.  

Aspose.Slides for PHP via Java에 문제가 발생했을 때 가장 효율적으로 해결하려면 다음 지침을 따르세요.

- 문제를 보고하기 전에 먼저 최신 Aspose.Slides for PHP via Java 버전을 사용하고 있는지 확인하십시오. 문제가 이미 업데이트에서 해결되었을 수 있습니다.  

  최신 제품 버전 정보는 [Aspose.Slides download](https://downloads.aspose.com/slides/ko/php-java)에서 확인할 수 있습니다.  

- 문제를 보고하기 전에 포럼, 이 문서 및 API Reference를 확인하십시오. 질문에 대한 답변이 이미 제공되었을 수 있습니다.

## **FAQ**

**PDF/A 및 PDF/UA와 같은 보관 및 접근성을 위한 PDF 호환 레벨을 지원합니까?**

예. [PDF 내보내기 옵션](https://reference.aspose.com/slides/ko/php-java/aspose.slides/pdfoptions/)을 구성하면 PDF/A-2a/2b/2u, PDF/A-3a/3b 및 PDF/UA로 저장할 수 있습니다.

**글꼴 대체 메커니즘 및 사용자 정의 글꼴 지원이 있어 올바른 렌더링을 보장합니까?**

예. 라이브러리를 통해 [사용자 정의 글꼴을 로드](/slides/ko/php-java/custom-font/)하고 [대체 규칙을 정의](/slides/ko/php-java/fallback-font/)하여 누락된 글리프가 변환 및 렌더링 시 신뢰성 있게 대체됩니다.

**파일이 암호로 보호되어 있는지 전체를 열지 않고 감지할 수 있습니까?**

예. [프레젠테이션을 검사](/slides/ko/php-java/examine-presentation/)하여 전체 문서를 로드하기 전에 암호가 필요 여부를 판단할 수 있습니다.

**Microsoft PowerPoint가 처리 및 변환에 필요합니까?**

아니오. Aspose.Slides는 독립 실행형 엔진이며 서버나 워크스테이션에 PowerPoint가 설치될 필요가 없습니다.

**여러 스레드에서 프레젠테이션을 안전하게 처리할 수 있습니까?**

예. 서로 다른 문서를 별도 스레드에서 병렬 처리할 수 있지만 동일한 프레젠테이션 인스턴스를 동시에 사용하지 않도록 주의하면 됩니다. 자세한 내용은 [스레드 간 사용](/slides/ko/php-java/multithreading/)을 참고하십시오.

**매크로가 보존되며 PPTM/PPSM 파일에서 VBA를 관리할 수 있습니까?**

예. 매크로가 포함된 프레젠테이션은 [지원](/slides/ko/php-java/presentation-via-vba/)되며, 해당 파일에서 [VBA 프로젝트를 검사 및 관리](https://reference.aspose.com/slides/ko/php-java/aspose.slides/presentation/getvbaproject/)할 수 있습니다.

**PDF 또는 HTML을 PowerPoint 슬라이드로 다시 변환할 수 있습니까?**

예. [PDF 페이지 또는 HTML 콘텐츠를 가져와](/slides/ko/php-java/import-presentation/) 프레젠테이션에 슬라이드를 생성하거나 채울 수 있습니다.

**XPS 내보내기가 지원되며 XPS 출력의 품질과 내용을 제어할 수 있습니까?**

예. [XPS로 내보내기](/slides/ko/php-java/convert-powerpoint-to-xps/)가 가능하며, [저장 옵션](https://reference.aspose.com/slides/ko/php-java/aspose.slides/xpsoptions/)을 통해 출력 품질 및 포함 콘텐츠를 조정할 수 있습니다.

**슬라이드를 이미지로 변환하고 출력 품질을 제어할 수 있습니까?**

예. 슬라이드는 [PNG, JPEG, GIF, BMP, TIFF](/slides/ko/php-java/convert-powerpoint-to-png/) 등 다양한 형식으로 렌더링할 수 있으며 크기와 품질을 세밀하게 제어할 수 있습니다.

**수학 식( MathML/MathText )이 지원되며 내보낼 수 있습니까?**

예. [수학 텍스트를 생성 및 편집](/slides/ko/php-java/powerpoint-math-equations/)하고 [수식 내보내기](/slides/ko/php-java/exporting-math-equations/)를 통해 MathML 등으로 출력할 수 있습니다.

**라이브러리가 암호화 및 비밀번호(열기, 설정, 제거)를 어떻게 처리합니까?**

[암호가 적용된 프레젠테이션 열기](/slides/ko/php-java/password-protected-presentation/), 비밀번호 설정·제거 및 파일 보호 상태 확인을 지원합니다.
