{{% alert color="primary" %}} 

JasperReports와 JasperServer는 Microsoft PowerPoint 프레젠테이션으로 보고서를 내보내는 기본 기능을 제공하지 않지만, Aspose.Slides for JasperReports를 사용하면 추가 내보내기 형식에 접근할 수 있습니다:

- Microsoft PowerPoint 프레젠테이션 (PPT)
- Microsoft PowerPoint 프레젠테이션 (PPTX)
- HTML
- PDF

{{% /alert %}} 

이러한 형식으로 문서를 만들기 위해 Aspose.Slides for JasperReports는 Aspose의 시장 선도 프레젠테이션 처리 라이브러리인 [Aspose.Slides for Java](https://products.aspose.com/slides/ko/java/)의 내장 버전을 사용합니다. Microsoft PowerPoint는 문서 생성에 사용되지 않습니다.
