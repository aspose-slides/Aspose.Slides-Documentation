Aspose.Slides for JasperReports는 다음과 호환됩니다:

**Aspose.Slides for JasperReports 17.5 및 이전 버전:**

- Java 2 Platform Standard Edition 1.4 이상.
- JasperReports 2.0, 3.0 이상.

**Aspose.Slides for JasperReports 17.6 및 이후 버전:**

- Java 2 Platform Standard Edition 6.0 이상.
- JasperReports 3.7.2 이상.
