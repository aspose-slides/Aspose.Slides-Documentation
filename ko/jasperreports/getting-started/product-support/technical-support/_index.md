{{% alert color="primary" %}} 

Aspose는 모든 제품에 대해 무제한 무료 기술 지원을 제공합니다. 지원은 평가판을 포함한 모든 사용자에게 제공됩니다.

{{% /alert %}} 

만약 Aspose.Slides for JasperReports에 대한 도움이 필요하다면, 다음을 고려하십시오:

- 지원의 주요 경로는 [**Aspose.Forum**](https://forum.aspose.com/c/slides/ko/11) 입니다. Aspose.Slides 포럼에 질문을 올리면 몇 시간 안에 답변을 받을 수 있습니다. 질문은 Aspose.Slides 지원팀이 직접 답변합니다.
- 참고로, Aspose는 전화로 기술 지원을 제공하지 않습니다. 전화 지원은 판매 및 구매 문의에만 제공됩니다.
- 포럼에서 답변을 기대할 때는 시간대 차이를 고려해 주세요.

Aspose.Slides for JasperReports에 문제가 있는 경우, 가장 효율적으로 해결되도록 다음 간단한 단계를 따라 주세요:

- 문제를 보고하기 전에 최신 Aspose.Slides for JasperReports 버전을 사용하고 있는지 확인하십시오. 최신 버전은 Aspose.Slides 다운로드 페이지에서 확인할 수 있습니다.
- 문제를 보고하기 전에 포럼, 이 문서 및 API 참조를 확인하십시오. 질문에 이미 답변이 있을 수 있습니다.
- 문제를 보고할 때는 원본 문서와 문제를 일으키는 코드 조각을 포함해 주세요. 여러 파일을 첨부해야 한다면 하나의 zip 파일로 압축하십시오. 첨부 파일은 Aspose.Forums에 안전하게 업로드됩니다. 해당 파일에 접근할 수 있는 사람은 귀하와 Aspose 개발자뿐입니다.
- 한 스레드당 하나의 문제만 보고해 주세요. 다른 문제가 있으면 별도의 스레드에 보고하십시오.
