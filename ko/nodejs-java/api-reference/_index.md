{{% alert color="primary" %}} 

Aspose.Slides for Node.js via Java은 Microsoft PowerPoint® 등 없이도 Node.js 애플리케이션이 프레젠테이션 문서를 읽고 쓸 수 있게 하는 클래스 라이브러리입니다. Aspose.Slides for Node.js via Java을 사용하면 텍스트, 도형, 표 및 애니메이션을 관리하고, 슬라이드에 오디오와 비디오를 추가하고, 슬라이드를 미리 보기하고, 슬라이드를 SVG, PDF 형식으로 내보내는 등 많은 작업을 수행할 수 있습니다.

최신 API 참조는 [Aspose.Slides for Node.js via Java API Reference](https://reference.aspose.com/slides/ko/nodejs-java/)에서 확인할 수 있습니다.
