## **Aspose.Slides 평가**

Aspose.Slides를 평가용으로 쉽게 다운로드할 수 있습니다. 평가 패키지는 구매한 패키지와 동일합니다. 평가 버전은 라이선스를 적용하는 몇 줄의 코드를 추가하면 라이선스가 적용된 버전이 됩니다.  

Aspose.Slides의 평가 버전(라이선스가 지정되지 않음)은 전체 제품 기능을 제공하지만, 열거나 저장할 때 문서 상단에 평가 워터마크를 삽입합니다. 프레젠테이션 슬라이드에서 텍스트를 추출할 때도 슬라이드 하나로 제한됩니다.

## **FAQ**

**평가 모드에서 여러 프레젠테이션을 서로 다른 스레드에서 병렬로 테스트할 수 있나요?**

예. 서로 다른 문서를 병렬로 처리할 수 있습니다; 동일한 프레젠테이션 개체를 [스레드 간](/slides/ko/net/multithreading/)와 같이 공유하면 안 됩니다. 평가 모드는 여기에 영향을 주지 않습니다.

**서버나 CI에서 라이브러리를 평가하려면 Microsoft PowerPoint를 설치해야 합니까?**

아니요. Aspose.Slides는 독립 실행형 엔진이며 평가든 프로덕션이든 PowerPoint가 설치될 필요가 없습니다.

**평가 모드에서 PPT/PPTX를 PDF 및 이미지로 완전하게 변환 테스트할 수 있나요?**

예. [변환기](/slides/ko/net/convert-presentation/)가 작동합니다; 출력물에 워터마크가 포함됩니다.

**워터마크 없이 부하 테스트에 임시 라이선스를 사용할 수 있나요?**

예. 30일 임시 라이선스를 사용하면 평가 모드 제한이 해제되어 워터마크 없이 테스트할 수 있습니다.
