## **개요**

이 문서는 Aspose.Slides에 대한 지원을 받는 방법과 문제를 효과적으로 보고하는 방법을 설명합니다. 주요 지원 채널을 설명하고 문제 제출을 위한 권장 사항을 제공하며 제품 페이지, 블로그, 다운로드, 지원 포럼, 문서 및 API 참조와 같은 유용한 기술 리소스를 안내합니다. 또한 이 문서는 유료 지원, 문제 에스컬레이션 및 이전 라이브러리 버전에 대한 지원에 대한 일반적인 질문에 답변합니다.

## **기술 지원**
{{% alert color="primary" %}}

Aspose는 모든 제품에 대해 무제한 무료 기술 지원을 제공합니다. 이 지원은 평가 패키지 사용자를 포함한 모든 사용자에게 제공됩니다.

{{% /alert %}} 

Aspose.Slides for Node.js via Java에 대한 도움이 필요할 때 다음을 고려하십시오:

- 지원의 주요 경로는 [Aspose.Forums](https://forum.aspose.com/c/slides/ko/11)입니다. Aspose.Slides 포럼에 질문을 게시하면 몇 시간 내에 답변을 받을 수 있습니다. Aspose.Slides 지원 팀이 포럼에 게시된 질문에 직접 답변합니다.
- Aspose는 전화로 기술 지원을 제공하지 않음을 유의하십시오. 전화 지원은 영업 및 구매 문의에만 제공됩니다.
- 포럼에서 답변을 기대할 때는 인내심을 가지고 시차를 고려해 주세요.

Aspose.Slides for Node.js via Java에서 문제가 발생하면 가장 효율적인 방법으로 해결하기 위해 다음 지침을 따르십시오:

- 문제를 보고하기 전에 최신 Aspose.Slides for Node.js via Java 버전을 사용하고 있는지 확인하십시오. 발생한 문제가 업데이트에서 이미 해결되었을 수 있습니다.

  최신 버전에 대한 정보를 확인하려면 [Aspose.Slides download](https://releases.aspose.com/slides/ko/nodejs-java/)를 참조하십시오.

- 문제를 보고하기 전에 포럼, 이 문서 및 API 참조를 확인해 보세요. 질문에 대한 답변이 이미 제공되었을 수 있습니다.

- 문제를 보고할 때는 원본 문서와 (가능한 경우) 문제와 관련된 코드 조각을 포함해 주세요. 여러 파일을 첨부해야 하는 경우 ZIP 패키지에 넣으십시오. 첨부 파일은 **오직** 귀하와 Aspose 개발자만 접근할 수 있으므로 Aspose.Forums에 안전하게 첨부할 수 있습니다.

- 특정 문제에 대해 하나의 스레드만 생성하여 사용하십시오. 다른 문제가 발생하면 별도의 스레드를 생성해 보고해야 합니다.

## **FAQ**

**사용 가능한 유료 지원 등급은 무엇이며 어떻게 다릅니까?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). 포함된 사고 수, 응답 우선순위 및 사용 가능한 커뮤니케이션 채널에 따라 차이가 있습니다.

**버그를 직접 엔지니어링 팀에 에스컬레이션할 수 있나요?**

예. [Paid support](https://helpdesk.aspose.com/)는 문제가 더 깊은 조사가 필요할 때 개발자에게 에스컬레이션을 허용합니다.

**이전 라이브러리 버전이 지원되며, 해당 버전에 대한 수정이 제공되나요?**

이전 버전을 사용할 수 있지만, 수정 사항은 현재 릴리스에만 제공되며 레거시 버전에 대한 패치는 제공되지 않습니다.
