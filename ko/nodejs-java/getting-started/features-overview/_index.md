## **지원되는 플랫폼**
Aspose.Slides for Node.js via Java은 플랫폼에 독립적인 API입니다. Node.js 7 이상이 설치된 모든 플랫폼(Windows, Linux, macOS 등)에서 사용할 수 있습니다. Oracle JDK 8 이상도 필요합니다.

## **파일 형식 및 변환**
Aspose.Slides for Node.js via Java는 대부분의 PowerPoint 문서 형식을 지원합니다. 또한 조직에서 사용하고 공유하는 일반적인 파일 형식으로 내보낼 수 있습니다. 자세한 내용은 다음을 참고하십시오:

|**기능**|**설명**|
| :- | :- |
|Microsoft PowerPoint (PPT)|Aspose.Slides for Node.js via Java는 이 프레젠테이션 문서 형식에 대해 가장 빠른 처리를 제공합니다.|
|PPT to PPTX conversion|Aspose.Slides for Node.js via Java는 PPT에서 PPTX로의 변환을 지원합니다.|
|Portable Document Format (PDF)|지원되는 모든 파일 형식을 단일 메서드로 Adobe Portable Document Format(PDF) 문서로 내보낼 수 있습니다.|
|XML Parser Specification (XPS)|지원되는 모든 파일 형식을 단일 메서드로 XML Parser Specification(XPS) 문서로 내보낼 수 있습니다.|
|Tagged Image File Format (TIFF)|지원되는 모든 프레젠테이션 파일 형식을 Tagged Image File Format(TIFF)으로 내보낼 수 있습니다.|
|PPTX To HTML Conversion|Aspose.Slides for Node.js via Java는 PresentationEx를 HTML 형식으로 변환하는 것을 지원합니다.|

## **렌더링 및 인쇄**
Aspose.Slides for Node.js via Java는 프레젠테이션 슬라이드를 다양한 그래픽 형식으로 고충실도 렌더링을 지원합니다. 자세한 내용은 다음을 참고하십시오:

|**기능**|**설명**|
| :- | :- |
|Java 지원 이미지 형식|Aspose.Slides for Node.js via Java를 사용하면 프레젠테이션 슬라이드와 슬라이드에 포함된 이미지를 TIFF, PNG, BMP, JPEG, GIF와 같은 Java가 지원하는 모든 그래픽 형식으로 렌더링할 수 있습니다.|
|SVG Format|Aspose.Slides for Node.js via Java는 프레젠테이션 슬라이드를 Scalable Vector Graphics(SVG) 형식으로 내보내는 내장 메서드도 제공합니다.|
|Presentation Printing|Aspose.Slides for Node.js via Java의 최신 버전은 다양한 옵션을 갖춘 내장 인쇄 메서드를 제공합니다.|

## **콘텐츠 기능**
Aspose.Slides for Node.js via Java를 사용하면 프레젠테이션 문서의 거의 모든 항목이나 내용을 접근, 수정 또는 생성할 수 있습니다. 자세한 내용은 다음을 참고하십시오:

|**기능**|**설명**|
| :- | :- |
|Master Slides|마스터 슬라이드는 일반 슬라이드의 레이아웃을 정의합니다. Aspose.Slides for Node.js via Java를 사용하면 프레젠테이션 문서의 마스터 슬라이드에 접근하고 수정할 수 있습니다.|
|Normal Slides|Aspose.Slides for Node.js via Java를 사용하면 다양한 유형의 새 슬라이드를 만들 수 있으며, 프레젠테이션에 존재하는 슬라이드에 접근하고 수정할 수도 있습니다.|
|Cloning / Copying Slides|Aspose.Slides for Node.js via Java가 제공하는 내장 메서드를 사용하면 프레젠테이션 내 기존 슬라이드를 복제하거나 복사할 수 있습니다. 복사 및 복제된 슬라이드를 한 프레젠테이션에서 다른 프레젠테이션으로 사용할 수도 있습니다. 슬라이드는 마스터 슬라이드의 레이아웃을 상속하기 때문에 내장 복제 메서드는 복제 시 마스터를 자동으로 복사합니다.|
|Managing Slides sections|프레젠테이션 내 슬라이드를 다양한 섹션으로 구성하는 메서드|
|Place Holders and Text Holders|슬라이드의 자리 표시자와 텍스트 표시자에 접근할 수 있습니다. 또한 적절한 메서드를 사용하여 텍스트 표시자가 포함된 슬라이드를 처음부터 만들 수 있습니다.|
|Header and Footers|Aspose.Slides for Node.js via Java는 슬라이드의 머리글/바닥글 처리를 지원합니다.|
|Notes in Slides|Aspose.Slides for Node.js via Java를 사용하면 슬라이드와 연결된 메모에 접근하고 수정할 수 있으며, 새 메모를 추가할 수도 있습니다.|
|Finding a Shape|형식에 연결된 대체 텍스트를 사용하여 슬라이드에서 특정 도형을 찾을 수도 있습니다.|
|Backgrounds|Aspose.Slides for Node.js via Java를 사용하면 프레젠테이션의 마스터 슬라이드 또는 일반 슬라이드와 연결된 배경을 작업할 수 있습니다.|
|Text Boxes|텍스트 상자를 처음부터 만들 수 있습니다. 기존 텍스트 상자에 접근할 수 있으며, 원본 텍스트 형식을 유지하면서 텍스트를 수정할 수 있습니다.|
|Rectangle Shapes|Aspose.Slides for Node.js via Java를 사용하여 사각형 도형을 만들거나 수정할 수 있습니다.|
|Poly Line Shapes|Aspose.Slides for Node.js via Java를 사용하여 폴리라인 도형을 만들거나 수정할 수 있습니다.|
|Ellipse Shapes|Aspose.Slides for Node.js via Java를 사용하여 타원 도형을 만들거나 수정할 수 있습니다.|
|Group Shapes|Aspose.Slides for Node.js via Java는 그룹 도형을 지원합니다.|
|Auto Shapes|Aspose.Slides for Node.js via Java는 자동 도형을 지원합니다.|
|SmartArt|Aspose.Slides for Node.js via Java는 MS PowerPoint의 SmartArt 도형을 지원합니다.|
|Charts|Aspose.Slides for Node.js via Java는 PowerPoint의 MSO 차트를 지원합니다.|
|Shapes Serialization|Aspose.Slides for Node.js via Java는 다양한 도형을 지원합니다. Aspose.Slides for Node.js via Java가 특정 도형을 지원하지 않을 경우, 기존 슬라이드에서 해당 도형을 직렬화하는 방법을 사용하여 도형을 직렬화할 수 있습니다. 이를 통해 필요에 따라 해당 도형을 계속 사용할 수 있습니다.|
|Picture Frames|Aspose.Slides for Node.js via Java를 사용하여 사진 프레임 안의 이미지를 관리할 수 있습니다.|
|Audio Frames|Aspose.Slides for Node.js via Java를 사용하여 슬라이드의 오디오 프레임에 오디오 파일을 연결하거나 삽입할 수 있습니다.|
|Video Frames|비디오 프레임에서 비디오 파일을 처리할 수 있습니다. Aspose.Slides for Node.js via Java는 연결된 비디오와 삽입된 비디오도 지원합니다.|
|OLE Frame|Aspose.Slides for Node.js via Java를 사용하여 OLE 프레임에서 OLE 객체를 관리할 수 있습니다.|
|Tables|Aspose.Slides for Node.js via Java는 슬라이드의 표를 지원합니다.|
|ActiveX Controls|ActiveX 컨트롤 지원|
|VBA Macros|프레젠테이션 내 VBA 매크로 관리 지원|
|Text Frame|해당 도형에 연결된 텍스트 프레임을 통해 모든 도형의 텍스트에 접근할 수 있습니다.|
|Text Scanning|내장 스캔 메서드를 사용하여 프레젠테이션 전체 또는 슬라이드 수준에서 텍스트를 스캔할 수 있습니다.|
|Animations|도형에 애니메이션을 적용할 수 있습니다.|
|Slide Shows|Aspose.Slides for Node.js via Java는 슬라이드 쇼와 슬라이드 전환을 지원합니다.|

Aspose.Slides for Node.js via Java를 사용하면 프레젠테이션 슬라이드의 텍스트와 도형을 포맷팅할 수 있습니다. 자세한 내용은 다음을 참고하십시오:

|**기능**|**설명**|
| :- | :- |
|Text Formatting|<p>Aspose.Slides for Node.js via Java에서는 도형에 연결된 텍스트 프레임을 통해 텍스트를 관리할 수 있습니다. 따라서 텍스트 프레임에 연결된 단락과 부분을 사용하여 텍스트를 포맷팅할 수 있습니다. 이러한 텍스트 요소는 Aspose.Slides for Node.js via Java를 통해 포맷팅할 수 있습니다.</p><p>- 글꼴 종류</p><p>- 글꼴 크기</p><p>- 글꼴 색상</p><p>- 글꼴 명암</p><p>- 단락 정렬</p><p>- 단락 글머리표</p><p>- 단락 방향</p>|
|Shape Formatting|<p>Aspose.Slides for Node.js via Java에서 슬라이드의 기본 요소는 도형입니다. 다음과 같이 도형 요소를 포맷팅할 수 있습니다:</p><p>- 위치</p><p>- 크기</p><p>- 선</p><p>- 채우기(패턴, 그라디언트, 단색 포함)</p><p>- 텍스트</p><p>- 이미지</p>|

## **자주 묻는 질문**

**라이브러리를 사용하려면 서버/PC에 Microsoft PowerPoint를 설치해야 합니까?**

아니요. PowerPoint는 필요하지 않으며, Aspose.Slides는 프레젠테이션을 생성, 편집, 변환 및 렌더링하기 위한 독립 실행형 엔진입니다.

**멀티스레딩은 어떻게 작동합니까? 처리 병렬화가 가능합니까?**

다른 스레드에서 서로 다른 문서를 처리하는 것은 안전합니다; 동일한 [presentation](https://reference.aspose.com/slides/ko/nodejs-java/aspose.slides/presentation/) 객체를 동시에 [multiple threads](/slides/ko/nodejs-java/multithreading/)에서 사용해서는 안 됩니다.

**파일 비밀번호 및 암호화가 지원됩니까?**

예. [You can](/slides/ko/nodejs-java/password-protected-presentation/) 암호화된 프레젠테이션을 열고, 열기 및 쓰기 비밀번호를 설정하거나 제거하며, 보호 상태를 확인할 수 있습니다.

**Linux 컨테이너에서 폰트 패키지를 신경 써야 합니까?**

예. 예기치 않은 폰트 대체를 방지하기 위해 일반적인 폰트 패키지를 설치하거나 애플리케이션에서 명시적으로 [specify font directories](/slides/ko/nodejs-java/custom-font/)를 지정하는 것이 권장됩니다.

**평가 버전에 제한이 있습니까?**

[evaluation mode](/slides/ko/nodejs-java/licensing/)에서는 출력에 워터마크가 추가되고 일부 제한이 적용됩니다; 전체 기능 테스트를 위해 [30-day temporary license](https://purchase.aspose.com/temporary-license/)를 사용할 수 있습니다.

**프레젠테이션으로 외부 형식(PDF/HTML → PPTX) 가져오기가 지원됩니까?**

예. [PDF pages and HTML content](/slides/ko/nodejs-java/import-presentation/)을 프레젠테이션에 추가하여 슬라이드로 변환할 수 있습니다.
