## **XPS에 대하여**
The [XML 파서 사양](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification) 은 Microsoft에서 원래 개발한 페이지 설명 언어이자 고정 문서 형식입니다. PDF와 마찬가지로, XPS는 문서의 충실성을 보존하고 장치에 독립적인 문서 표시를 제공하도록 설계된 고정 레이아웃 문서 형식입니다.

## **C++용 Aspose.Slides의 XPS**
Aspose.Slides for C++에서 로드할 수 있는 모든 프레젠테이션 문서는 XPS 형식으로 변환할 수 있습니다. Aspose.Slides for C++는 고충실도 페이지 레이아웃 및 렌더링 엔진을 사용하여 고정 레이아웃 XPS 문서 형식으로 출력을 생성합니다.
언급할 가치가 있는 점은 Aspose.Slides for C++가 C++ Framework 3.5에 포함된 Windows Presentation Foundation(WPF) 클래스를 사용하지 않고 직접 XPS를 생성한다는 것입니다. 따라서 Aspose.Slides for C++는 버전 3.5 이전의 C++ Framework를 실행하는 컴퓨터에서도 XPS 문서를 생성할 수 있습니다. Aspose.Slides for C++를 통해 프레젠테이션 문서를 XPS 문서로 내보내는 방법에 대해서는 [이 항목](/slides/ko/net/)에서 확인할 수 있습니다.
