## **Microsoft PowerPoint (PPT)**
### **About PPT**
[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint)은 Microsoft PowerPoint의 다양한 버전에서 생성·읽기·조작·작성할 수 있는 프레젠테이션 문서 파일 형식입니다. 이것은 Microsoft가 개발한 프레젠테이션 문서용 바이너리 형식입니다.
### **PPT in Aspose.Slides for C++**
Aspose.Slides for C++는 아래에 나열된 소프트웨어가 만든 PPT 파일을 읽을 수 있습니다.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

마찬가지로, Aspose.Slides for C++가 만든 PPT 파일도 위 소프트웨어에서 읽을 수 있습니다.
### **Comprehensive Support for PPT**
Aspose.Slides for C++는 PPT 문서 파일 형식과 관련된 거의 모든 기능을 지원합니다. 이는 다양한 Microsoft PowerPoint 버전이 제공하는 기본·고급 기능을 포괄할 뿐만 아니라 Microsoft PowerPoint에서도 지원되지 않는 일부 기능까지 포함합니다. Aspose.Slides for C++ API 라이브러리를 사용하는 주요 장점은 이러한 기능을 손쉽게 활용할 수 있다는 점입니다.

PPT 문서 파일의 생성·읽기·쓰기와 같은 기본 작업 외에도 Aspose.Slides for C++가 제공하는 여러 기능이 있습니다.

- PPT 문서에 OLE 개체로 다른 MS Office 파일 형식 가져오기
- PPT 문서를 PDF, TIFF, XPS 형식으로 내보내기
- PPT 문서의 슬라이드를 SVG 형식으로 내보내기
- C++ Framework에서 지원하는 모든 이미지 형식으로 슬라이드 렌더링
- PPT 문서에서 슬라이드 크기 설정
- 도형의 애니메이션 관리
- 슬라이드 쇼 관리
- 슬라이드의 텍스트 서식 지정
- PPT 문서에서 텍스트 스캔
- 슬라이드의 표 관리
- 클론 기능을 사용한 마스터 자동 복사

## **PresentationML (PPTX, XML)**
### **About PresentationML**
PresentationML은 프레젠테이션 문서용 XML 기반 포맷군을 지칭하는 이름입니다. Office OpenXML(OOXML)은 Microsoft Office 2007 애플리케이션에서 도입된 XML 기반 포맷입니다. Office OpenXML은 여러 전문 XML 마크업 언어를 담는 컨테이너 형식이며, PresentationML은 Microsoft Office PowerPoint 2007이 문서를 저장하는 데 사용하는 마크업 언어입니다.
### **PresentationML in Aspose.Slides for C++**
OOXML PresentationML 문서는 PPTX 파일 형태로 제공되며, 이는 [OOXML ECMA-376](https://www.ecma-international.org/publications-and-standards/standards/ecma-376/) 사양을 따르는 zip 압축 XML 패키지입니다. Aspose.Slides for C++는 PresentationML 문서의 생성·읽기·조작·쓰기 를 광범위하게 지원합니다. 또한 Aspose.Slides for C++는 PresentationML 문서를 PDF, TIFF, XPS 등 널리 사용되는 다양한 문서 형식으로 내보낼 수 있습니다. 이는 Aspose.Slides for C++가 프레젠테이션 문서를 포괄적으로 처리하도록 설계되었고, PresentationML이 기본적으로 zip 압축 XML 패키지 형태로 내부 프레젠테이션을 보관하기 때문입니다.

### **PresentationML is Open, Why Use Aspose.Slides for C++**
PresentationML은 XML 기반이므로 XML 클래스를 사용해 제3자 라이브러리인 Aspose.Slides for C++ 없이도 PresentationML 문서를 처리·생성하는 애플리케이션을 구축할 수 있습니다. 하지만 PresentationML 문서를 다룰 때 Aspose.Slides for C++를 사용하는 데는 여러 장점이 있습니다.

OOXML 사양은 수천 페이지에 이르는 방대한 분량입니다. 따라서 PresentationML 문서를 올바르게 다루려면 해당 문서 형식에 대한 이해에 많은 시간과 노력이 필요합니다. 반면 Aspose.Slides for C++를 사용하면 관련 클래스와 해당 메서드·속성을 그대로 활용해 복잡해 보이는 작업도 손쉽게 수행할 수 있습니다.

XML 클래스로 PresentationML 문서를 처리할 때는 사용할 수 없는 기능들이 Aspose.Slides for C++에는 포함되어 있습니다.

- PPT 문서를 PDF, TIFF, XPS 형식으로 내보내기
- PPT 문서의 슬라이드를 SVG 형식으로 내보내기
- C++ Framework에서 지원하는 모든 이미지 형식으로 슬라이드 렌더링
- 클론 기능을 사용해 원본 프레젠테이션에서 마스터 자동 복사
- 도형에 보호 적용

예를 들어 “Hello World” 텍스트가 들어 있는 단일 슬라이드와 하나의 텍스트 상자를 가진 PresentationML 문서를 생각해 보십시오. XML 클래스를 사용해 텍스트를 읽으려면 다음 조각에서 해당 텍스트를 파싱하는 프로그램을 작성해야 합니다.

``` cpp

<p:sld xmlns:a="http://schemas.openxmlformats.org/drawingml/2006/main" xmlns:r="http://schemas.openxmlformats.org/officeDocument/2006/relationships" xmlns:p="http://schemas.openxmlformats.org/presentationml/2006/main">

  <p:cSld>

    <p:spTree>

      <p:nvGrpSpPr>

        <p:cNvPr id="1" name=""/>

        <p:cNvGrpSpPr/>

        <p:nvPr/>

      </p:nvGrpSpPr>

      <p:grpSpPr>

        <a:xfrm>

          <a:off x="0" y="0"/>

          <a:ext cx="0" cy="0"/>

          <a:chOff x="0" y="0"/>

          <a:chExt cx="0" cy="0"/>

        </a:xfrm></p:grpSpPr><p:sp>

          <p:nvSpPr><p:cNvPr id="4" name="TextBox 3"/>

          <p:cNvSpPr txBox="1"/>

            <p:nvPr/>

          </p:nvSpPr>

          <p:spPr>

            <a:xfrm>

              <a:off x="2819400" y="2590800"/>

              <a:ext cx="1297086" cy="369332"/>

            </a:xfrm>

            <a:prstGeom prst="rect">

              <a:avLst/>

            </a:prstGeom>

            <a:noFill/>

          </p:spPr>

          <p:txBody>

            <a:bodyPr wrap="none" rtlCol="0">

              <a:spAutoFit/>

            </a:bodyPr>
```

## **PPT to PPTX conversion**
### **About Conversion**
Aspose.Slides는 이제 PPT를 PPTX로 변환하는 기능도 지원합니다.
### **Features Supported in Conversion**
Aspose.Slides for C++는 PPT 문서 형식 프레젠테이션을 PPTX 파일 형식 프레젠테이션으로 변환하는 기능을 부분적으로 지원합니다. 이 변환 기능이 막 도입된 단계이므로 현재는 제한된 기능만 제공되며 단순한 형태의 프레젠테이션에만 적용됩니다. PPT 프레젠테이션을 PPTX 형식으로 변환할 때 Aspose.Slides for C++ API 라이브러리가 제공하는 주요 장점은 API 사용이 간편하다는 점입니다. 자세한 내용은 this[link]() 코드 스니펫 섹션을 참고하십시오. 아래 섹션은 PPT 형식 프레젠테이션을 PPTX 형식으로 변환할 때 지원되는 기능과 지원되지 않는 기능을 명확히 보여줍니다.
### **Supported Features**
변환 과정에서 지원되는 기능은 다음과 같습니다.

- 마스터, 레이아웃 및 슬라이드 구조 변환
- 마스터, 레이아웃 및 슬라이드 구조 변환
- 차트 변환
- 그룹 도형
- 사각형 및 타원 등 Auto-shape 변환(단, Auto-shape의 조정값이 잘못될 수 있음)
- 사용자 정의 기하학 도형(때때로 변환되지 않을 수 있음)
- Auto-shape의 텍스처·이미지 채우기 스타일(때때로 변환되지 않을 수 있음)
- 플레이스홀더 변환
- 텍스트 프레임 및 텍스트 홀더 내 텍스트 변환(단, 글머리표·정렬·탭이 완전히 구현되지 않음)
### **Unsupported Features**
변환 과정에서 지원되지 않는 기능은 다음과 같습니다.

- 노트가 포함된 슬라이드(노트 읽기가 PPTX에서 구현되지 않음). PPT에 노트가 있으면 아직 PPTX로 저장할 수 없음*
- 선 및 폴리라인 변환
- 선 및 채우기 형식
- 그라디언트 채우기 스타일
- OLE 프레임, 표, 비디오·오디오 프레임 등
- 애니메이션 및 기타 슬라이드쇼 속성은 건너뜀

새롭거나 누락된 기능은 향후 Aspose.Slides for C++ 릴리스에서 추가될 예정입니다.

## **Portable Document Format (PDF)**
### **About PDF**
[Portable Document Format](https://en.wikipedia.org/wiki/PDF)은 Adobe System이 다양한 조직 간 문서 교환을 위해 만든 파일 형식입니다. 이 형식의 목적은 문서 내용이 표시되는 플랫폼에 관계없이 시각적 모양이 동일하게 유지되도록 하는 것입니다.
### **PDF in Aspose.Slides for C++**
Aspose.Slides for C++에 로드할 수 있는 모든 프레젠테이션 문서는 [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A) 또는 [PDF /A-1b](https://en.wikipedia.org/wiki/PDF/A) 형식으로 변환할 수 있습니다. Aspose.Slides for C++는 프레젠테이션 문서를 PDF로 내보낼 때 대부분 원본 프레젠테이션과 거의 동일한 모습을 유지하도록 처리합니다. PDF 변환 시 Aspose 솔루션이 지원하는 주요 기능은 다음과 같습니다.

- 이미지, 텍스트 상자 및 기타 도형
- 텍스트 및 서식
- 단락 및 서식
- 하이퍼링크
- 머리글 및 바닥글
- 글머리표
- 표

Aspose.Slides for C++ 구성 요소만으로 프레젠테이션 문서를 PDF로 직접 내보낼 수 있습니다. 즉, 이를 위해 별도의 타사 또는 Aspose.Pdf 구성 요소가 필요하지 않습니다. 또한 [this topic](/slides/ko/cpp/convert-powerpoint-to-pdf/)에 설명된 대로 다양한 옵션을 사용해 프레젠테이션 → PDF 내보내기를 맞춤 설정할 수 있습니다.

## **XML Parser Specification (XPS)**
### **About XPS**
[XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification)은 Microsoft가 최초 개발한 페이지 설명 언어이자 고정 문서 형식입니다. PDF와 마찬가지로 XPS는 문서 충실도를 보존하고 장치에 독립적인 문서 모습을 제공하도록 설계된 고정 레이아웃 문서 형식입니다.
### **XPS in Aspose.Slides for C++**
Aspose.Slides for C++가 로드할 수 있는 모든 프레젠테이션 문서는 XPS 형식으로 변환할 수 있습니다. Aspose.Slides for C++는 고충실도 페이지 레이아웃 및 렌더링 엔진을 사용해 고정 레이아웃 XPS 문서 형식으로 출력합니다. 특히 Aspose.Slides for C++는 Windows Presentation Foundation(WPF) 클래스를 사용하지 않고 직접 XPS를 생성하므로 C++ Framework 3.5 이전 버전에서도 XPS 문서를 생성할 수 있습니다. XPS 변환에 대한 자세한 내용은 [this topic](https://docs.aspose.com/slides/ko/cpp/convert-powerpoint-to-xps/)를 참고하세요.

Aspose.Slides for C++를 통해 XPS 문서로 변환된 프레젠테이션 문서
