## **소프트웨어 요구 사항**
예제를 다운로드하고 실행하기 전에 아래 요구 사항을 충족하는지 확인하십시오.

1. Visual Studio 2015 이상.

## **GitHub에서 Aspose.Slides 다운로드**
Aspose.Slides for C++의 모든 예제는 [GitHub](https://github.com/aspose-slides/Aspose.Slides-for-C)에서 호스팅됩니다.

- 원하는 GitHub 클라이언트를 사용해 저장소를 복제하거나 [here](https://github.com/aspose-slides/Aspose.Slides-for-.NET/archive/refs/heads/master.zip)에서 ZIP 파일을 다운로드할 수 있습니다.
- ZIP 파일의 내용을 컴퓨터의 원하는 폴더에 압축 해제하십시오. 모든 예제는 **Examples** 폴더에 있습니다.
- C++ Visual Studio 솔루션 파일이 포함되어 있습니다.
- 프로젝트는 Visual Studio 2015에서 생성되었지만 솔루션 파일은 Visual Studio 2015 이상과 호환됩니다.
- Visual Studio에서 솔루션 파일을 열고 프로젝트를 빌드하십시오.
- [Download Section](https://downloads.aspose.com/slides/ko/cpp)에서 최신 Aspose.Slides for C++ 릴리스를 다운로드하십시오.
- 릴리스를 압축 해제하고 **Examples** 폴더 내부에 **Include** 및 **Bin** 디렉터리를 복사하십시오.
- **main.cpp** 파일을 열면 모든 예제가 여기서 호출됩니다.
- 프로젝트 내에서 실행하려는 예제의 주석을 해제하십시오.
- 솔루션을 빌드하고 컴파일하십시오.

예제를 설정하거나 실행하는 데 문제가 있으면 포럼을 통해 언제든지 문의하십시오.

## **기여**
예제를 추가하거나 개선하고 싶다면 프로젝트에 기여해 주시기 바랍니다. 이 저장소의 모든 예제와 쇼케이스 프로젝트는 오픈 소스로, 자유롭게 자신의 애플리케이션에 사용할 수 있습니다.

기여하려면 저장소를 포크하고, 소스 코드를 수정한 뒤 풀 리퀘스트를 생성하면 됩니다. 변경 사항을 검토한 후 도움이 된다고 판단되면 저장소에 포함하겠습니다.
