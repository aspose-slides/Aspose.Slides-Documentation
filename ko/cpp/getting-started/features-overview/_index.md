## **파일 형식 및 변환**
Aspose.Slides for C++는 대부분의 PowerPoint 문서 형식을 지원하며 조직 간 교환에 널리 사용되는 인기 있는 형식으로도 내보낼 수 있습니다. 다음 표에 이러한 형식이 나열되어 있습니다:

|**기능**|**설명**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/ko/cpp/microsoft-powerpoint-ppt/)|Aspose.Slides for C++는 이 프레젠테이션 문서 형식에 대해 가장 빠른 처리 속도를 제공합니다.|
|[Portable Document Format (PDF)](/slides/ko/cpp/portable-document-format-pdf/)|지원되는 파일 형식은 단일 메서드로 Adobe Portable Document Format(PDF) 문서로 내보낼 수 있습니다.|
|[XML Parser Specification (XPS)](/slides/ko/cpp/xml-parser-specification-xps/)|지원되는 모든 파일 형식은 단일 메서드로 XML Parser Specification(XPS) 문서로 내보낼 수 있습니다.|
|[PresentationML (PPTX, XML)](/slides/ko/cpp/presentationml-pptx-xml/)|Aspose.Slides for C++는 PresentationML 또는 PPTX라고도 알려진 OOXML 프레젠테이션 형식 처리를 지원합니다.|
|[PPTX To HTML Conversion](https://docs.aspose.com/slides/ko/cpp/convert-powerpoint-to-html/)|Aspose.Slides for C++는 프레젠테이션을 HTML 형식으로 변환하는 것을 지원합니다.|

## **렌더링**
Aspose.Slides for C++는 아래에 나열된 다양한 그래픽 형식으로 프레젠테이션 문서의 슬라이드를 고충실도 렌더링을 지원합니다:

|**기능**|**설명**|
| :- | :- |
|C++ Supported Image Formats|Aspose.Slides for C++를 사용하면 프레젠테이션 슬라이드뿐만 아니라 슬라이드의 이미지도 PNG, BMP, JPEG와 같은 C++가 지원하는 모든 그래픽 형식으로 렌더링할 수 있습니다.|
|SVG Format|Aspose.Slides for C++는 프레젠테이션 슬라이드를 Scalable Vector Graphics(SVG) 형식으로 내보내는 내장 메서드도 제공합니다.|

## **콘텐츠 기능**
Aspose.Slides for C++를 사용하면 프레젠테이션 문서의 거의 모든 콘텐츠에 접근, 수정 또는 생성할 수 있습니다.

|**기능**|**설명**|
| :- | :- |
|Master Slides|마스터 슬라이드는 일반 슬라이드의 레이아웃을 정의합니다. Aspose.Slides for C++를 사용하면 프레젠테이션 문서의 마스터 슬라이드에 접근하고 수정할 수 있습니다.|
|Normal Slides|Aspose.Slides for C++를 사용하면 다양한 유형의 새 슬라이드를 만들 뿐만 아니라 프레젠테이션 문서의 기존 슬라이드에 접근하고 수정할 수 있습니다.|
|Cloning / Copying Slides|Aspose.Slides for C++는 프레젠테이션 내뿐만 아니라 한 프레젠테이션에서 다른 프레젠테이션으로 기존 슬라이드를 복제하거나 복사하는 내장 메서드를 제공합니다. 슬라이드는 마스터 슬라이드의 레이아웃을 상속하므로, 복제 시 내장 복제 메서드가 마스터 슬라이드도 자동으로 복사합니다.|
|Place Holders and Text Holders|슬라이드의 자리표시자와 텍스트 자리표시자에도 접근할 수 있습니다. 또한 적절한 메서드를 사용해 텍스트 자리표시자를 포함한 슬라이드를 처음부터 생성할 수 있습니다.|
|Header and Footers|Aspose.Slides for C++는 슬라이드의 머리글/바닥글을 다루는 것도 지원합니다.|
|Notes in Slides|Aspose.Slides for C++를 사용하면 슬라이드와 연결된 노트에 접근하고 수정할 뿐만 아니라 노트를 추가할 수도 있습니다.|
|Finding a Shape|슬라이드에서 해당 도형과 연결된 대체 텍스트를 사용해 특정 도형을 찾을 수 있습니다.|
|Backgrounds|Aspose.Slides for C++는 프레젠테이션의 마스터 또는 일반 슬라이드와 연결된 배경을 작업할 수 있게 합니다.|
|Text Boxes|텍스트 상자는 처음부터 생성할 수 있습니다. 기존 텍스트 상자에 접근하고 텍스트를 원래 형식을 유지하면서 수정할 수도 있습니다.|
|Rectangle Shapes|직사각형 도형은 Aspose.Slides for C++로 생성하거나 수정할 수 있습니다.|
|Poly Line Shapes|폴리라인 도형도 Aspose.Slides for C++로 생성하거나 수정할 수 있습니다.|
|Ellipse Shapes|타원형 도형도 Aspose.Slides for C++로 생성하거나 수정할 수 있습니다.|
|Group Shapes|Aspose.Slides for C++는 그룹 도형도 지원합니다.|
|Auto Shapes|자동 도형도 Aspose.Slides for C++에서 지원됩니다.|
|Shapes Serialization|Aspose.Slides for C++는 많은 도형을 지원하지만, 지원되지 않는 도형의 경우 Aspose.Slides for C++에서 제공하는 직렬화 메서드를 통해 기존 슬라이드에서 해당 도형을 직렬화하여 필요에 따라 향후 Aspose.Slides for C++에서 사용할 수 있습니다.|
|Picture Frames|이미지는 Aspose.Slides for C++를 사용해 사진 프레임에서 관리됩니다.|
|Audio Frames|오디오 파일은 Aspose.Slides for C++를 사용해 슬라이드의 오디오 프레임에 링크하거나 삽입할 수 있습니다.|
|Video Frames|비디오 파일은 Aspose.Slides for C++를 통해 비디오 프레임에서 처리됩니다.|
|OLE Frame|OLE 객체는 Aspose.Slides for C++에 의해 OLE 프레임에서 관리됩니다.|
|Tables|슬라이드의 테이블도 Aspose.Slides for C++에서 지원됩니다.|
|Text Frame|어떤 도형과 연결된 텍스트는 해당 도형의 텍스트 프레임을 통해 접근할 수 있습니다.|
|Text Scanning|프레젠테이션의 텍스트는 내장 스캔 메서드를 통해 프레젠테이션 수준 또는 슬라이드 수준에서 스캔할 수 있습니다.|
|Animations|도형에 애니메이션을 적용할 수 있습니다.|
|Slide Shows|슬라이드 쇼, 슬라이드 전환 및 Morph 전환도 지원됩니다.|

|**기능**|**설명**|
| :- | :- |
|Text Formatting|<p>Aspose.Slides for C++에서 텍스트는 도형에 연결된 텍스트 프레임을 통해 관리됩니다. 따라서 텍스트는 텍스트 프레임에 연결된 단락 및 부분을 사용해 서식 지정할 수 있습니다. 다음 텍스트 요소들은 Aspose.Slides for C++를 통해 서식 지정이 가능합니다.</p><p>- 글꼴 종류</p><p>- 글꼴 크기</p><p>- 글꼴 색상</p><p>- 글꼴 음영</p><p>- 단락 정렬</p><p>- 단락 글머리표</p><p>- 단락 방향</p>|

## **FAQ**

**라이브러리를 사용하기 위해 서버/PC에 Microsoft PowerPoint를 설치해야 합니까?**

아니오. PowerPoint는 필요하지 않으며; Aspose.Slides는 프레젠테이션을 만들고, 편집하고, 변환하고, 렌더링하는 독립형 엔진입니다.

**멀티스레딩은 어떻게 작동합니까? 처리를 병렬화할 수 있습니까?**

It is safe to process different documents in different threads; the same [프레젠테이션](https://reference.aspose.com/slides/ko/cpp/aspose.slides/presentation/) object must not be used by [여러 스레드](/slides/ko/cpp/multithreading/) at the same time.

**파일 비밀번호 및 암호화가 지원됩니까?**

예. [가능합니다](/slides/ko/cpp/password-protected-presentation/) 암호화된 프레젠테이션을 열고, 열기 및 쓰기 비밀번호를 설정하거나 제거하며, 보호 상태를 확인할 수 있습니다.

**Linux 컨테이너에서 폰트 패키지를 신경써야 합니까?**

예. 일반적인 폰트 패키지를 설치하고/또는 애플리케이션에서 [폰트 디렉터리 지정](/slides/ko/cpp/custom-font/)을 명시적으로 설정하여 예기치 않은 대체를 방지하는 것이 권장됩니다.

**평가 버전에는 제한 사항이 있습니까?**

평가 모드([평가 모드](/slides/ko/cpp/licensing/))에서는 출력에 워터마크가 추가되고 특정 제한이 적용됩니다; 전체 기능 테스트를 위한 [30일 임시 라이선스](https://purchase.aspose.com/temporary-license/)가 제공됩니다.

**프레젠테이션에 외부 형식(PDF/HTML → PPTX)을 가져오는 것이 지원됩니까?**

예. [PDF 페이지 및 HTML 콘텐츠](/slides/ko/cpp/import-presentation/)를 프레젠테이션에 추가하여 슬라이드로 변환할 수 있습니다.
