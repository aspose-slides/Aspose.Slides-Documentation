## **레거시 코드 지원**
13.x 이전 버전의 Aspose.Slides for Java로 개발된 레거시 코드를 사용하려면 코드를 약간 수정하면 이전과 같이 동작합니다. 이전 Aspose.Slides for Java에서 Aspose.Slide 및 Aspose.Slides.Pptx 네임스페이스에 있던 모든 클래스가 이제 단일 Aspose.Slides 네임스페이스로 통합되었습니다. 레거시 Aspose.Slides API로 Hello World 프레젠테이션 문서를 생성하는 다음 간단한 코드 조각을 살펴보고, 새로운 통합 API로 마이그레이션하는 방법을 설명하는 단계들을 따라 주세요.
