## **레거시 코드 지원**
Aspose.Slides for Java 14.x.x 이전 버전으로 개발된 레거시 코드를 사용하려면 코드에 약간의 변경을 적용해야 하며, 이렇게 하면 코드는 이전과 동일하게 작동합니다. 이전 Aspose.Slides for Java에서 com.aspose.slides 및 com.aspose.slides.pptx 네임스페이스 아래에 있던 모든 클래스는 이제 단일 com.aspose.slides 네임스페이스로 병합되었습니다. 레거시 Aspose.Slides API를 사용하여 프레젠테이션에서 처음부터 일반 차트를 만드는 간단한 코드 스니펫을 살펴보고, 새로운 병합 API로 마이그레이션하는 단계에 대해 확인하십시오.
