Aspose.Slides for Java 14.3.0 (14.4.0)는 PPT 처리에 대한 완전히 새로운 구현을 제공합니다. 많은 개선 사항이 있으며, PPTX에서 PPT로의 부분 변환을 지원합니다. 하지만 구현되지 않은 기능이 몇 가지 있습니다:

- 일부 도형은 직렬화된 PPT 문서에서 잘못된 기하학을 가지고 있습니다(주석)
- 모든 PPTX 텍스트 서식 기능이 PPT 직렬화에서 지원되는 것은 아닙니다
- 텍스트 언어 및 맞춤법 설정에 대한 정보가 직렬화된 PPT 문서에 없습니다
- 모든 PPTX 테마 기능이 PPT 직렬화에서 지원되는 것은 아닙니다

**Aspose.Slides for Java 8.6.0과 비교했을 때 몇 가지 차이점이 있습니다:**

- OLE/ActiveX PPT 직렬화에서 알려진 문제가 있습니다

**Aspose.Slides for .NET 14.3.0과 비교했을 때 몇 가지 차이점이 있습니다:**

- 현재 Aspose.Slides for Java에서는 프레젠테이션 인쇄 지원을 사용할 수 없습니다
