{{% alert color="primary" %}} 

Aspose.Slides for Java 14.4.0는 PowerPoint 문서 처리를 위한 새로운 결정을 제공합니다. 다음과 같은 제한 사항 및 알려진 문제가 있으며, 향후 릴리스에서 제거될 예정입니다:

- 직렬화된 PPT 문서에서 일부 도형의 기하학이 잘못됩니다(호, 원형 화살표, 설명선).
- PPT 직렬화에서 모든 PPTX 텍스트 서식 기능이 지원되지 않습니다(탭, 들여쓰기 및 단락 서식 제한).
- 텍스트 언어 및 맞춤법 설정 정보가 직렬화된 PPT 문서에 포함되지 않습니다.
- PPT 직렬화에서 모든 PPTX 테마 기능이 지원되지 않습니다(채우기 형식, 선 형식 및 글꼴 직렬화만 지원).
- PPT로 OLE/ActiveX 직렬화 시 알려진 문제가 있습니다.
- 워드아트 직렬화 및 렌더링이 지원되지 않습니다.

{{% /alert %}}
