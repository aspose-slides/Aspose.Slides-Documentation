## **Adding Text Dynamically**
두 방법 모두 다음 단계에 따라 진행됩니다:

1. 프레젠테이션을 생성합니다.
1. 빈 슬라이드를 추가합니다.
1. 텍스트 상자를 추가합니다.
1. 텍스트를 설정합니다.
1. 프레젠테이션을 저장합니다.
