{{% alert color="primary" %}} 

[XML Parser Specification](https://en.wikipedia.org/wiki/Open_XML_Paper_Specification)은 Microsoft에서 처음 개발한 페이지 설명 언어이자 고정 문서 형식입니다. PDF와 마찬가지로 XPS는 문서 충실도를 유지하고 장치에 종속되지 않는 문서 모습을 제공하도록 설계되었습니다. 

{{% /alert %}} 

## **Aspose.Slides for Java의 XPS**
Aspose.Slides for Java에서 로드할 수 있는 모든 프레젠테이션 문서는 XPS 형식으로 변환할 수 있습니다. Aspose.Slides for Java는 고충실도 페이지 레이아웃 및 렌더링 엔진을 사용하여 고정 레이아웃 XPS 문서 형식으로 출력합니다.
Aspose.Slides for Java를 통해 프레젠테이션 문서를 XPS 문서로 내보내는 방법은 [Converting to XPS](https://docs.aspose.com/slides/ko/java/convert-powerpoint-to-xps/)에서 확인할 수 있습니다.
