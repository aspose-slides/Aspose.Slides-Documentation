{{% alert color="primary" %}} 

[PPT](https://en.wikipedia.org/wiki/Microsoft_PowerPoint)는 다양한 버전의 Microsoft PowerPoint에서 생성, 읽기, 조작 및 쓰기가 가능한 프레젠테이션 문서 파일 형식입니다. 이것은 Microsoft에서 개발한 프레젠테이션 문서용 바이너리 형식입니다.

{{% /alert %}} 

## **Aspose.Slides for Java의 PPT**
Aspose.Slides for Java는 아래에 나열된 소프트웨어에서 만든 PPT 파일을 읽을 수 있습니다.

- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003

마찬가지로 Aspose.Slides for Java로 만든 PPT 파일은 위 소프트웨어 집합에서 읽을 수 있습니다.

## **PPT에 대한 포괄적인 지원**
Aspose.Slides for Java는 PPT 문서 파일 형식이 지원하는 거의 모든 기능을 지원합니다. 이는 다양한 Microsoft PowerPoint 버전이 제공하는 기본 및 고급 기능을 포괄할 뿐 아니라 Microsoft PowerPoint에서 지원하지 않는 기능까지 포함합니다. Aspose.Slides for Java API 라이브러리를 사용하는 주요 장점은 이러한 기능을 손쉽게 처리할 수 있다는 점입니다.

PPT 문서 파일의 생성, 읽기 및 쓰기와 관련된 기본 작업 외에도 Aspose.Slides for Java가 제공하는 여러 기능이 있습니다:

- 다른 Microsoft Office 파일 형식을 [OLE 개체로 PPT 문서에 가져오기]().
- [PPT 문서를 PDF로 내보내기](/slides/ko/java/convert-powerpoint-ppt-and-pptx-to-pdf/).
- PPT 문서의 슬라이드를 SVG 형식으로 내보내기.
- Java 프레임워크에서 지원하는 모든 이미지 형식으로 슬라이드 렌더링.
- PPT 문서의 슬라이드 크기 설정.
- 도형의 애니메이션 관리.
- 슬라이드 쇼 관리.
- [슬라이드의 텍스트 서식 지정]().
- PPT 문서에서 텍스트 추출.
- [슬라이드의 표 처리]().
- [복제 기능]을 사용하여 마스터를 자동으로 복사.
