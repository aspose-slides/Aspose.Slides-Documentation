{{% alert color="primary" %}} 

The [Portable Document Format](https://en.wikipedia.org/wiki/PDF) 은 Adobe Systems에서 만든 파일 형식으로 조직 간 문서 교환을 위해 사용됩니다. 이 형식의 목적은 보는 플랫폼에 관계없이 내용과 레이아웃을 동일하게 유지하는 것이었습니다. Aspose.Slides for Java를 사용하면 프레젠테이션 파일을 PDF로 변환할 수 있습니다.

{{% /alert %}} 

## **Aspose.Slides for Java의 PDF**
Aspose.Slides for Java에 로드할 수 있는 모든 프레젠테이션은 선택에 따라 [PDF 1.5](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1a](https://en.wikipedia.org/wiki/PDF/A), [PDF/A-1b](https://en.wikipedia.org/wiki/PDF/A) 또는 [PDF/UA](https://en.wikipedia.org/wiki/PDF/UA) 에 부합하는 PDF로 변환할 수 있습니다. Aspose.Slides for Java는 프레젠테이션을 PDF로 내보내며 대부분의 경우 출력 PDF가 원본 프레젠테이션과 정확히 동일하게 보입니다.

Aspose.Slides는 PDF로 변환할 때 다음 프레젠테이션 기능을 지원합니다:

- 이미지, 텍스트 상자 및 기타 도형.
- 텍스트 및 서식.
- 단락 및 서식.
- 하이퍼링크.
- 머리글 및 바닥글.
- 글머리표.
- 표.

Aspose.Slides for Java를 사용하면 추가 구성 요소 없이 직접 프레젠테이션을 PDF로 내보낼 수 있습니다. 또한 [Converting to PDF](/slides/ko/java/converting-a-presentation/)에 설명된 다양한 옵션을 사용해 프레젠테이션을 PDF로 내보내는 방식을 사용자 지정할 수 있습니다.
