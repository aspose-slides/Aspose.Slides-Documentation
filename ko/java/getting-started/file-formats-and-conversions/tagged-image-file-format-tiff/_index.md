## **TIFF에 대하여**

The [Tagged Image File Format](https://en.wikipedia.org/wiki/TIFF) which is known for storing multiple images in one document was originally created by Aldus. This format is widely supported by scanning, faxing and other image manipulation applications.

## **Java용 Aspose.Slides의 TIFF**

Any document that can be loaded in Aspose.Slide for Java can also be converted to TIFF document directly by Aspose.Slides for Java eliminating requirement of any third party component. Further, you can optionally define the size of the images in the resulting TIFF document. You can find information about exporting the presentation documents to TIFF documents through Aspose.Slides for Java in [this topic](/slides/ko/java/converting-a-presentation/).
