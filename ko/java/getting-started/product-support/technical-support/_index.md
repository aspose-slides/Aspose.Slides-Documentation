## **개요**

본 문서는 Aspose.Slides에 대한 지원을 받는 방법과 문제를 효율적으로 보고하는 방법을 설명합니다. 주요 지원 채널을 설명하고 문제 제출에 대한 권장 사항을 제공하며, 제품 페이지, 블로그, 다운로드, 지원 포럼, 문서 및 API 레퍼런스와 같은 유용한 기술 리소스를 안내합니다. 또한 유료 지원, 문제 에스컬레이션 및 이전 라이브러리 버전 지원에 대한 일반적인 질문에 답변합니다.

## **기술 지원**

{{% alert color="primary" %}} 

Aspose는 모든 제품에 대해 무제한 무료 기술 지원을 제공합니다. 평가판을 포함한 모든 사용자가 지원을 받을 수 있습니다.

{{% /alert %}} 

Aspose.Slides for Java에 도움이 필요하면 다음을 고려하십시오:

- 지원의 주요 경로는 Aspose.Forums입니다. Aspose.Slides 포럼에 질문을 게시하면 몇 시간 이내에 답변을 받을 수 있습니다. 질문은 Aspose.Slides 지원 팀이 직접 답변합니다.
- 참고로, Aspose는 전화로 기술 지원을 제공하지 않습니다. 전화 지원은 판매 및 구매 문의에만 사용할 수 있습니다.
- 포럼에서 답변을 기대할 경우, 시간대 차이를 고려해 주세요.

Aspose.Slides for Java에 문제가 있는 경우, 가장 효율적으로 해결될 수 있도록 다음 간단한 단계를 따르세요:

- 문제를 보고하기 전에 최신 Aspose.Slides for Java 버전을 사용하고 있는지 확인하십시오. 최신 버전은 Aspose.Slides 다운로드에서 확인할 수 있습니다.
- 문제를 보고하기 전에 포럼, 이 문서 및 API 레퍼런스를 살펴보세요. 질문이 이미 답변되었을 수 있습니다.
- 문제를 보고할 때는 원본 문서와 문제를 일으키는 코드 조각을 포함해 주세요. 여러 파일을 첨부해야 할 경우 하나의 zip 파일로 압축하십시오. Aspose.Forums에 문서를 첨부해도 안전합니다. 첨부 파일에 접근할 수 있는 사람은 귀하와 Aspose 개발자뿐이기 때문입니다.
- 스레드당 하나의 문제만 보고해 주세요. 다른 문제가 있는 경우 별도의 스레드에 보고하십시오.

## **자주 묻는 질문**

**사용 가능한 유료 지원 티어는 무엇이며 어떻게 차이가 있나요?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). 포함된 사고 수, 응답 우선순위 및 사용 가능한 커뮤니케이션 채널에 따라 차이가 있습니다.

**버그를 직접 엔지니어링 팀에 에스컬레이션할 수 있나요?**

예. [Paid support](https://helpdesk.aspose.com/)는 문제가 더 깊은 조사가 필요할 때 개발자로 에스컬레이션을 허용합니다.

**이전 라이브러리 버전이 지원되나요? 또한 해당 버전에 대한 수정 사항을 릴리스하나요?**

이전 버전을 사용할 수 있지만, 수정 사항은 현재 릴리스에만 제공됩니다. 레거시 버전에 대한 패치는 제공되지 않습니다.
