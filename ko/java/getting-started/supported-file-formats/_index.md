## **개요**

Aspose.Slides는 Microsoft PowerPoint 97부터 Office 365까지, Mac용 Microsoft PowerPoint를 포함한 프레젠테이션 파일을 지원합니다. 이 문서에서는 라이브러리가 지원하는 PowerPoint 버전을 나열하고 로드, 저장 또는 두 가지 모두 가능한 파일 형식 표를 제공합니다.

이 문서는 PDF 규격 준수, 글꼴 포함, 비밀번호 보호 파일, 사용자 정의 글꼴, 글꼴 대체 및 XPS 내보내기 옵션에 대한 일반적인 질문에도 답변합니다.

## **지원되는 Microsoft PowerPoint 버전**
- Microsoft PowerPoint 97
- Microsoft PowerPoint 2000
- Microsoft PowerPoint XP
- Microsoft PowerPoint 2003
- Microsoft PowerPoint 2007
- Microsoft PowerPoint 2010
- Microsoft PowerPoint 2013
- Microsoft PowerPoint 2016
- Microsoft PowerPoint 2019
- Microsoft PowerPoint for MAC
- Office 365

## **자주 묻는 질문**

**PDF/A 및 PDF/UA와 같은 보관 및 접근성 표준을 충족하는 PDF로 프레젠테이션을 저장할 수 있나요?**

네. Aspose.Slides는 PDF/A-2a, PDF/A-2b, PDF/A-2u, PDF/A-3a, PDF/A-3b 및 PDF/UA와 같은 준수 수준으로 PDF 내보내기를 지원하며, 이는 [compliance](https://reference.aspose.com/slides/ko/java/com.aspose.slides/pdfoptions/#setCompliance-int-) 설정을 통해 [PDF export options](https://reference.aspose.com/slides/ko/java/com.aspose.slides/pdfoptions/)에서 지정할 수 있습니다.

**라이브러리가 PDF로 내보낼 때 글꼴 포함을 지원하며, 포함되는 글꼴을 세밀하게 제어할 수 있나요?**

네. 글꼴을 전체 포함하거나 서브셋(사용된 글리프만)으로 포함할지 제어할 수 있으며, 일반 시스템 글꼴의 처리 방식과 ASCII 텍스트에 대한 동작을 [PDF export options](https://reference.aspose.com/slides/ko/java/com.aspose.slides/pdfoptions/)를 통해 지정할 수 있습니다.

**파일이 비밀번호로 보호되어 있는지 실제로 로드하기 전에 감지할 수 있나요?**

네. [factory-based inspection API](https://reference.aspose.com/slides/ko/java/com.aspose.slides/presentationfactory/)를 사용하면 프레젠테이션 파일을 완전히 열지 않고도 비밀번호 보호 여부를 확인할 수 있습니다.

**글꼴 대체 메커니즘 및 사용자 정의 글꼴 지원이 있나요?**

네. 라이브러리는 [loading](/slides/ko/java/custom-font/) 및 [embedding](/slides/ko/java/embedded-font/) 사용자 정의 글꼴을 지원하고, 렌더링 및 변환 중에 누락된 글리프가 발생하지 않도록 [fallback rules](/slides/ko/java/fallback-font/)를 제공합니다.

**슬라이드를 XPS로 내보낼 수 있으며, XPS 출력 품질을 조정할 옵션이 있나요?**

네. [Export to XPS](/slides/ko/java/convert-powerpoint-to-xps/)가 지원되며, 관련 [save options](https://reference.aspose.com/slides/ko/java/com.aspose.slides/xpsoptions/)를 조정해 XPS 문서의 출력 품질 및 내용을 제어할 수 있습니다.
