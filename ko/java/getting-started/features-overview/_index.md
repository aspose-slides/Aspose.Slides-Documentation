## **지원되는 플랫폼**
Aspose.Slides for Java supports the most popular development and deployment platforms.

|**기능**|**설명**|
| :- | :- |
|Desktop Applications|Aspose.Slides for Java는 Windows Forms 애플리케이션 개발에 사용할 수 있습니다|
|Enterprise Web Applications|Aspose.Slides for Java를 사용하면 웹 애플리케이션을 구축할 수 있습니다. PHP와 함께 Aspose.Slides for Java를 사용하는 지원도 제공됩니다.|
|Linux/Unix|Aspose.Slides for Java는 플랫폼에 독립적인 API이며 리눅스 및 유닉스 환경에서 동작할 수 있습니다.|

## **파일 형식 및 변환**
Aspose.Slides for Java supports most of the Microsoft PowerPoint document formats and exports them to popular formats used widely by organizations.

|**기능**|**설명**|
| :- | :- |
|[Microsoft PowerPoint (PPT)](/slides/ko/java/microsoft-powerpoint-ppt/)|Aspose.Slides for Java는 이 프레젠테이션 문서 형식에 대해 가장 빠른 처리를 제공합니다.|
|[PresentationML (PPTX, XML)](/slides/ko/java/presentationml-pptx-xml/)|Aspose.Slides for Java는 OOXML 프레젠테이션 형식(프레젠테이션ML 또는 PPTX라고도 함) 처리를 지원합니다.|
|[PPT to PPTX conversion](/slides/ko/java/ppt-to-pptx-conversion/)|Aspose.Slides for Java는 PPT에서 PPTX로의 변환을 지원합니다.|
|[Portable Document Format (PDF)](/slides/ko/java/developer-guide/)|지원되는 파일 형식은 단일 메서드로 Adobe Portable Document Format(PDF) 문서로 내보낼 수 있습니다.|
|[XML Parser Specification (XPS)](/slides/ko/java/xml-parser-specification-xps/)|지원되는 모든 파일 형식은 단일 메서드로 XML Parser Specification(XPS) 문서로 내보낼 수 있습니다.|
|[Tagged Image File Format (TIFF)](https://docs.aspose.com/slides/ko/java/convert-powerpoint-to-tiff/)|Aspose.Slides for Java가 지원하는 프레젠테이션 파일 형식은 Tagged Image File Format(TIFF)으로도 내보낼 수 있습니다.|
|[ODP to PPTX Conversion](https://docs.aspose.com/slides/ko/java/convert-odp-to-pptx/)|Aspose.Slides for Java는 OpenDocument Presentation(ODP)을 로드하고 PPTX로 변환하는 것을 지원합니다.|
|[Presentation To HTML Conversion](https://docs.aspose.com/slides/ko/java/convert-powerpoint-to-html/)|Aspose.Slides for Java는 PresentationEx를 HTML 형식으로 변환하는 것을 지원합니다.|

## **프레젠테이션 렌더링**
Aspose .Slides for Java supports high fidelity rendering of slides in the presentations to various graphics formats:

|**기능**|**설명**|
| :- | :- |
|Supported Image Formats|Aspose.Slides for Java를 사용하면 프레젠테이션 슬라이드뿐만 아니라 슬라이드의 이미지도 TIFF, PNG, BMP, JPEG, GIF 및 메타파일과 같은 널리 지원되는 그래픽 형식으로 렌더링할 수 있습니다.|
|SVG Format|Aspose.Slides for Java는 프레젠테이션 슬라이드를 Scalable Vector Graphics(SVG) 형식으로 내보내는 내장 메서드를 제공합니다.|

## **FAQ**

**서버/PC에 Microsoft PowerPoint를 설치해야 라이브러리를 사용할 수 있나요?**

아니요. PowerPoint는 필요하지 않으며; Aspose.Slides는 프레젠테이션을 생성, 편집, 변환 및 렌더링하기 위한 독립 실행형 엔진입니다.

**멀티스레딩은 어떻게 작동하나요? 처리 병렬화가 가능한가요?**

다른 스레드에서 서로 다른 문서를 처리하는 것은 안전합니다; 동일한 [presentation](https://reference.aspose.com/slides/ko/java/com.aspose.slides/presentation/) 개체를 동시에 [multiple threads](/slides/ko/java/multithreading/)에서 사용하면 안 됩니다.

**파일 비밀번호 및 암호화가 지원되나요?**

예. [You can](/slides/ko/java/password-protected-presentation/) 암호화된 프레젠테이션을 열고, 열기 및 쓰기 비밀번호를 설정하거나 제거하며, 보호 상태를 확인할 수 있습니다.

**Linux 컨테이너의 폰트 패키지에 신경써야 하나요?**

예. 예상치 못한 대체를 방지하기 위해 일반 폰트 패키지를 설치하거나 애플리케이션에서 명시적으로 [specify font directories](/slides/ko/java/custom-font/)를 지정하는 것이 권장됩니다.

**평가 버전에 제한이 있나요?**

[evaluation mode](/slides/ko/java/licensing/)에서는 출력에 워터마크가 추가되고 특정 제한이 적용됩니다; 전체 기능 테스트를 위해 [30-day temporary license](https://purchase.aspose.com/temporary-license/)를 사용할 수 있습니다.

**외부 형식(PDF/HTML → PPTX)을 프레젠테이션에 가져오는 것이 지원되나요?**

예. 프레젠테이션에 [PDF pages and HTML content](/slides/ko/java/import-presentation/)를 추가하여 슬라이드로 변환할 수 있습니다.
