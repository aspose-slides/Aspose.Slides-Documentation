{{% alert color="primary" %}} 

Aspose.Slides for Java은 Microsoft PowerPoint® 등 없이도 Java 애플리케이션이 프레젠테이션 문서를 읽고 쓸 수 있게 하는 클래스 라이브러리입니다. Aspose.Slides for Java을 사용하면 텍스트, 도형, 표 및 애니메이션을 관리하고, 슬라이드에 오디오와 비디오를 추가하며, 슬라이드를 미리 보고, SVG 및 PDF 형식으로 슬라이드를 내보내는 등 다양한 작업을 수행할 수 있습니다.

최신 API 참조는 [Aspose.Slides for Java API Reference](https://reference.aspose.com/slides/ko/java)에서 확인할 수 있습니다.
