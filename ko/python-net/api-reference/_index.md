{{% alert color="primary" %}} 

Aspose.Slides for Python via .NET은 Microsoft PowerPoint® 또는 기타 소프트웨어를 사용하지 않고도 Python 애플리케이션이 프레젠테이션 문서를 읽고 쓸 수 있게 해주는 클래스 라이브러리입니다. Aspose.Slides for Python via .NET을 사용하면 텍스트, 도형, 표 및 애니메이션을 관리하고, 슬라이드에 오디오와 비디오를 추가하며, 슬라이드를 미리 보고, SVG, PDF 형식 등으로 내보낼 수 있습니다.

최신 API 레퍼런스는 [Aspose.Slides for Python via .NET API Reference](https://reference.aspose.com/slides/ko/python-net/)에서 확인할 수 있습니다.
