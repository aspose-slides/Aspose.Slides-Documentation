**Aspose.Slides for Python via .NET에 오신 것을 환영합니다**

![Aspose.Slides for Python via .NET Product Logo](aspose_slides-for-python.png)

Aspose.Slides for Python via .NET는 강력한 클래스 라이브러리로, Microsoft PowerPoint®가 없어도 애플리케이션에서 PowerPoint® 프레젠테이션을 읽고 쓸 수 있게 해줍니다.

Python 개발자를 위해 전체 기능을 갖춘 PowerPoint® 문서 관리를 제공하는 최초이자 유일한 구성 요소입니다.

Aspose.Slides for Python via .NET는 텍스트, 도형, 표, 애니메이션 작업; 오디오 및 비디오 추가; 슬라이드 미리 보기; SVG, PDF 등 다양한 형식으로 슬라이드 내보내기와 같은 광범위한 기능을 포함합니다.

## 자주 묻는 질문

### Aspose.Slides for Python via .NET란 무엇인가요?

Aspose.Slides for Python via .NET는 강력한 Python 라이브러리로, Microsoft PowerPoint를 설치하지 않고도 프로그래밍 방식으로 PowerPoint 프레젠테이션(PPT, PPTX, ODP)을 생성, 편집 및 변환할 수 있습니다.

### Aspose.Slides가 지원하는 프레젠테이션 기능은 무엇인가요?

이 라이브러리는 텍스트, 도형, 표, 차트, 애니메이션, 마스터 슬라이드, 오디오, 비디오 등을 관리할 수 있습니다. 또한 슬라이드 미리 보기, 렌더링, 인쇄 및 PDF, SVG, HTML, 이미지와 같은 형식으로 내보내기를 지원합니다.

### Aspose.Slides를 사용해 프레젠테이션을 다른 형식으로 변환할 수 있나요?

예. Aspose.Slides는 높은 정확도와 성능으로 PowerPoint 파일을 PDF, SVG, HTML, JPG, PNG, TIFF 등 다른 형식으로 변환할 수 있습니다.

### Aspose.Slides 사용에 Microsoft PowerPoint가 필요합니까?

아니요. Aspose.Slides는 독립형 API이며 Microsoft Office나 기타 서드파티 소프트웨어가 필요하지 않습니다.

### Aspose.Slides for Python via .NET가 지원하는 플랫폼은 무엇인가요?

이 제품은 크로스 플랫폼이며 Windows, Linux, macOS 환경에서 작동합니다.

### Aspose.Slides for Python을 시작하려면 어떻게 해야 하나요?

PyPi를 통해 설치하고, [개발자 가이드](/slides/ko/python-net/developer-guide/)를 살펴보면 예제, API 참조 및 튜토리얼을 통해 시작할 수 있습니다.
