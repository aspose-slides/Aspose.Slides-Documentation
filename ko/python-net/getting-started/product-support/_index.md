## **개요**

이 문서에서는 Aspose.Slides에 대한 지원을 받는 방법과 문제를 효과적으로 보고하는 방법을 설명합니다. 주요 지원 채널을 설명하고 문제 제출에 대한 권장 사항을 제공하며, 제품 페이지, 블로그, 다운로드, 지원 포럼, 문서 및 API 레퍼런스와 같은 유용한 기술 리소스를 안내합니다. 또한 유료 지원, 문제 에스컬레이션 및 이전 라이브러리 버전에 대한 지원과 같은 일반적인 질문에 답변합니다.

## **기술 지원**
{{% alert color="primary" %}}

Aspose는 모든 제품에 대해 무제한 무료 기술 지원을 제공합니다. 이 지원은 평가 패키지를 포함한 모든 사용자에게 제공됩니다.

{{% /alert %}} 

Aspose.Slides for Python via .NET에 대한 도움이 필요할 때는 다음을 고려하십시오:

- 지원의 주요 경로는 [Aspose.Forums](https://forum.aspose.com/c/slides/ko/11)입니다. Aspose.Slides 포럼에 질문을 게시하면 몇 시간 이내에 답변을 받을 수 있습니다. Aspose.Slides 지원 팀이 포럼에 게시된 질문에 직접 답변합니다.
- Aspose는 전화로 기술 지원을 제공하지 않음을 참고하십시오. 전화 지원은 영업 및 구매 문의에만 제공됩니다.
- 포럼에서 답변을 기대할 때는 인내심을 가지고 시차 차이를 고려해 주시기 바랍니다.

Aspose.Slides for Python via .NET에서 문제가 발생하면, 가장 효율적인 방법으로 해결하기 위해 다음 지침을 따르세요:

- 문제를 보고하기 전에 최신 Aspose.Slides for Python via .NET 버전을 사용하고 있는지 확인하십시오. 문제가 이미 업데이트에서 해결되었을 수도 있습니다.

  최신 버전에 대한 정보는 Aspose.Slides 다운로드에서 확인할 수 있습니다.

- 문제를 보고하기 전에 포럼, 이 문서 및 API 레퍼런스를 살펴보십시오. 질문에 대한 답변이 이미 제공되었을 수 있습니다.

- 문제를 보고할 때는 원본 문서와 가능한 경우 문제와 관련된 코드 조각을 포함하십시오. 파일이 여러 개인 경우 ZIP 패키지에 압축하십시오.

  Aspose.Forums에 파일을 안전하게 첨부할 수 있습니다. **only** 귀하와 Aspose 개발자만 첨부된 파일에 접근할 수 있습니다.

- 특정 문제에 대해서는 하나의 스레드만 사용하십시오. 다른 문제가 발생하면 새 스레드를 생성하여 보고해야 합니다.

**사용 가능한 유료 지원 티어는 무엇이며 어떻게 다릅니까?**

[Developer, Business, and Enterprise](https://helpdesk.aspose.com/kb/faq/1-Introduction-to-Paid-Support-Services). 포함된 사고 수, 응답 우선순위 및 제공되는 커뮤니케이션 채널에 따라 차이가 있습니다.

**버그를 직접 엔지니어링 팀에 에스컬레이션할 수 있나요?**
