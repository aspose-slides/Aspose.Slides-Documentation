## **Aspose.Slides for Python via .NET에 오신 것을 환영합니다!**

Aspose.Slides는 Microsoft PowerPoint® 관리 API로, Microsoft PowerPoint® 없이도 애플리케이션이 PowerPoint 프레젠테이션을 읽고, 쓰고, 만들고, 수정할 수 있게 해줍니다. 이 라이브러리는 프레젠테이션, 슬라이드, 도형, 표, 텍스트, 애니메이션, 오디오, 비디오 및 기타 프레젠테이션 요소 작업을 지원합니다.

Aspose.Slides를 사용하면 개발자는 다양한 소스에서 프레젠테이션을 열고, 여러 PowerPoint 형식으로 저장하며, 프레젠테이션을 PDF, TIFF, XPS, SWF, HTML, SVG와 같은 형식으로 내보내고, 슬라이드를 이미지로 저장할 수 있습니다. API는 문서 속성 관리, 보호 적용, 슬라이드 미리 보기, 슬라이드 내용 편집, 슬라이드 노트 작업, 임베드된 OLE 개체와 같은 고급 프레젠테이션 요소 처리 기능도 제공합니다.

## **Aspose.Slides for Python via .NET를 사용해야 하는 이유**

- Aspose.Slides for Python via .NET는 PowerPoint® 문서를 관리하는 데 필요한 기능을 제공하는 최초이자 유일한 구성 요소입니다.  
- Aspose.Slides for Python via .NET는 텍스트, 도형, 표 및 애니메이션 관리, 슬라이드에 오디오와 비디오 추가, 슬라이드 미리 보기, 슬라이드를 SVG, PDF 등 다양한 형식으로 내보내는 등 주요 기능을 많이 제공합니다.  
- Aspose.Slides for Python via .NET는 다양한 소스에서 PowerPoint 파일을 열 수 있을 뿐만 아니라 프레젠테이션을 다양한 방식으로 저장할 수 있습니다. 예를 들어 PPT 형식으로 저장하거나 슬라이드를 이미지로 저장할 수 있습니다.  

- Aspose.Slides for Python via .NET는 Microsoft PowerPoint에서 찾을 수 있거나 찾기 어려운 거의 모든 기능을 제공합니다. Aspose.Slides for Python via .NET를 사용하면 표와 다양한 도형 외에도 텍스트, 오디오, 비디오 프레임과 같은 다양한 프레임을 슬라이드에 추가할 수 있습니다.  
- Aspose.Slides for Python via .NET는 Managed Python으로 개발되었습니다. Python, VB.NET, J# 등 any .NET 언어와 함께 사용할 수 있으며, 웹 애플리케이션이든 Windows 애플리케이션이든 any 종류의 애플리케이션에서 사용할 수 있습니다.  
- Aspose.Slides for Python via .NET는 API에 대한 이해를 돕는 완전한 데모와 실용 예제를 제공합니다.

## **Aspose.Slides 기능**
Aspose.Slides for Python via .NET 기능은 다음 그룹으로 나눌 수 있습니다:

1. Presentation Features.
1. Slides Features.
1. Graphics and Multimedia Features.
1. Table Features.
1. Text & Paragraph Features
1. Advanced Features

### **Presentation Features**
- 다양한 소스에서 Microsoft PowerPoint 프레젠테이션 열기
- 다양한 Microsoft PowerPoint 형식으로 프레젠테이션 저장
- 처음부터 프레젠테이션 만들기
- 다른 구성 요소 없이 PDF 문서로 변환
- 프레젠테이션 문서 속성 읽기 또는 수정
- 프레젠테이션에 보호 적용
- 프레젠테이션을 PDF로 내보내기
- 프레젠테이션을 TIFF로 내보내기
- 프레젠테이션을 XPS로 내보내기
- 프레젠테이션을 SWF로 내보내기
- 프레젠테이션을 PDF Note / Tiff Notes로 내보내기
- 프레젠테이션을 HTML로 내보내기

### **Slides Features**
- 처음부터 새 슬라이드 만들고 프레젠테이션에 추가
- 프레젠테이션 내 슬라이드 위치를 통해 슬라이드에 접근
- 슬라이드를 같은 프레젠테이션이나 다른 프레젠테이션으로 복사 또는 복제
- 슬라이드 편집 및 삭제
- 마스터와 슬라이드 자동 복사
- 슬라이드 미리 보기
- 슬라이드 이름 가져오기 또는 설정
- 슬라이드를 이미지로 저장
- 슬라이드를 SVG 형식으로 내보내기
- 프레젠테이션에서 여러 마스터 작업
- 슬라이드 배경을 지원되는 스타일로 채우기
- 새 슬라이드 노트 추가 또는 관리
- 새 슬라이드 마스터와 스타일을 동적으로 설정
- 슬라이드 쇼 전환 추가 또는 수정
- Morph 전환 추가

### **Graphics & Multimedia Features**
- 선, 사각형, 타원, 폴리라인과 같은 도형 생성 또는 관리
- 도형의 선 스타일 관리
- 도형의 대체 텍스트 가져오기
- 도형 회전 및 뒤집기
- 다양한 스타일(예: 단색, 그라디언트, 패턴, 텍스처, 이미지)로 도형 채우기
- 미리 정의된 48가지 패턴 및 24가지 텍스처 사용
- 도형에 링크 할당
- 슬라이드와 도형에 새로운 애니메이션 효과를 읽고, 수정하고, 추가
- 프레젠테이션에 새 그림 추가 및 기존 그림 관리
- 프레젠테이션에 새 그림 글머리표 추가 및 기존 글머리표 관리
- 프레젠테이션에 새 오디오 프레임 추가 및 기존 프레임 관리
- 프레젠테이션에 새 비디오 프레임 추가 및 기존 프레임 관리

### **Table Features**
- 처음부터 표 생성 및 슬라이드에 추가
- 표 크기 조정 및 이동
- 표의 모든 셀에 접근
- 셀의 서식 및 테두리 변경
- 열과 행 크기 조정
- 열과 행 추가 및 삭제
- 표 전체에 테두리 두께와 색상 설정

### **Text & Paragraph Features**
- 텍스트 플레이스홀더와 텍스트 프레임의 텍스트 가져오기 및 설정
- 플레이스홀더 유형 가져오기
- 텍스트 프레임 내 텍스트 서식 변경
- 새 글꼴 추가 또는 프레젠테이션에 이미 사용된 글꼴 관리
- 텍스트와 텍스트 서식 변경
- 텍스트에 대한 링크 생성 또는 관리
- 탭 및 효과적인 탭 지원
- 거의 모든 도형 유형에 새 텍스트 프레임 추가
- 텍스트 프레임의 여백 및 텍스트 래핑 설정 접근 및 수정
- 단락 들여쓰기 접근 및 수정

### **Advanced Features**
- 프레젠테이션에 포함된 OLE 개체 접근
- 프레젠테이션 잠금

## **Technical support**
Aspose는 모든 제품에 대해 무제한 무료 기술 지원을 제공합니다. 평가는 물론 모든 사용자가 지원을 이용할 수 있습니다. Aspose.Slides for Python via .NET에 대한 도움이 필요하면 다음을 참고하십시오:

- 주요 지원 경로는 [Aspose Forums](https://forum.aspose.com/)입니다. [Aspose.Slides 포럼](https://forum.aspose.com/c/slides/ko/11)에 질문을 게시하면 몇 시간 내에 답변을 받을 수 있습니다. Aspose.Slides 지원팀이 포럼에 게시된 질문에 직접 답변합니다.
- Aspose는 전화로 기술 지원을 제공하지 않습니다. 전화 지원은 영업 및 구매 문의에만 제공됩니다.
- 포럼에서 답변을 기대한다면 인내심을 가지고 시차 차이를 고려해 주세요.

Aspose.Slides for Python via .NET에 문제가 발생하면 가장 효율적인 해결을 위해 다음 지침을 따르세요:

- 문제를 보고하기 전에 먼저 최신 Aspose.Slides for Python via .NET 버전을 사용하고 있는지 확인하십시오. 문제가 업데이트에서 이미 해결되었을 수 있습니다.  

  최신 제품 버전 정보를 보려면 [Aspose.Slides download](https://pypi.org/project/aspose.slides/)를 확인하세요.

- 문제를 보고하기 전에 포럼, 이 문서 및 API Reference를 확인하십시오. 질문에 대한 답이 이미 제공되었을 수 있습니다.

- 문제를 보고할 때는 원본 문서와 (가능하면) 문제와 관련된 코드 조각을 포함하십시오. 여러 파일을 첨부해야 하면 ZIP 패키지에 넣으세요.

  Aspose.Forums에 문서를 안전하게 첨부할 수 있는데, 첨부 파일에 접근할 수 있는 사람은 본인과 Aspose 개발자뿐입니다.

- 특정 이슈마다 하나의 스레드만 사용하십시오. 다른 문제가 발생하면 별도의 스레드를 만들어 보고하십시오.

## **FAQ**

**PDF/A 및 PDF/UA와 같은 보관 및 접근성을 위한 PDF 적합성 수준을 지원합니까?**

예. [PDF export options](https://reference.aspose.com/slides/ko/python-net/aspose.slides.export/pdfoptions/)를 구성하면 PDF/A‑2a/2b/2u, PDF/A‑3a/3b 및 PDF/UA로 저장할 수 있습니다.

**글꼴 대체 메커니즘 및 사용자 정의 글꼴 지원이 있어 올바른 렌더링을 보장합니까?**

예. 라이브러리는 [custom fonts 로드](/slides/ko/python-net/custom-font/)와 [fallback 규칙 정의](/slides/ko/python-net/fallback-font/)를 지원하여 누락된 글리프를 변환 및 렌더링 시 신뢰하게 대체합니다.

**파일이 암호로 보호되어 있는지 전체를 열지 않고 감지할 수 있습니까?**

예. [presentation 검사](/slides/ko/python-net/examine-presentation/)를 통해 전체 문서를 로드하기 전에 암호가 필요한지 확인할 수 있습니다.

**Microsoft PowerPoint가 처리 및 변환에 필요합니까?**

아니오. Aspose.Slides는 독립 실행형 엔진이며 서버나 워크스테이션에 PowerPoint가 필요하지 않습니다.

**여러 스레드에서 프레젠테이션을 안전하게 처리할 수 있습니까?**

예. 서로 다른 문서를 별도 스레드에서 병렬 처리할 수 있습니다. 동일한 프레젠테이션 인스턴스를 동시에 사용하지 않도록 주의하면 됩니다([across threads](/slides/ko/python-net/multithreading/) 참고).

**매크로가 보존되며 PPTM/PPSM 파일에서 VBA를 관리할 수 있습니까?**

예. 매크로가 포함된 프레젠테이션이 [지원](/slides/ko/python-net/presentation-via-vba/)되며, 해당 파일에서 [VBA 프로젝트를 검사하고 관리](https://reference.aspose.com/slides/ko/python-net/aspose.slides.vba/)할 수 있습니다.

**PDF 또는 HTML을 다시 PowerPoint 슬라이드로 변환할 수 있습니까?**

예. [PDF 페이지 또는 HTML 내용 가져오기](/slides/ko/python-net/import-presentation/)를 통해 슬라이드를 만들거나 기존 프레젠테이션에 채울 수 있습니다.

**XPS 내보내기가 지원되며 XPS 출력 품질 및 내용을 제어할 수 있습니까?**

예. [Export to XPS](/slides/ko/python-net/convert-powerpoint-to-xps/)가 가능하며, [save options](https://reference.aspose.com/slides/ko/python-net/aspose.slides.export/xpsoptions/)를 통해 출력 품질 및 포함 내용을 조정할 수 있습니다.

**슬라이드를 이미지로 변환하고 출력 품질을 제어할 수 있습니까?**

예. 슬라이드를 [PNG, JPEG, GIF, BMP, TIFF](/slides/ko/python-net/convert-powerpoint-to-png/) 등으로 렌더링할 수 있으며 크기와 품질을 세밀하게 제어할 수 있습니다.

**수학 공식(MathML/MathText)이 지원되며 내보낼 수 있습니까?**

예. [수학 텍스트 생성 및 편집](/slides/ko/python-net/powerpoint-math-equations/)과 [수식 내보내기](/slides/ko/python-net/exporting-math-equations/)를 통해 MathML을 포함한 다양한 형식으로 내보낼 수 있습니다.

**라이브러리가 암호화 및 비밀번호(열기, 설정, 제거)를 어떻게 처리합니까?**

[암호가 보호된 프레젠테이션 열기], 열기 및 수정용 비밀번호 설정 또는 제거, 파일 보호 상태 확인을 모두 지원합니다.
